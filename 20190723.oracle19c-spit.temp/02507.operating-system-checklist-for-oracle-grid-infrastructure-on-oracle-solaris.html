Operating System Checklist for Oracle Grid Infrastructure on Oracle Solaris

Use this checklist to check minimum operating system requirements for Oracle Database.

Table 1-2 Operating System General Checklist for Oracle Grid Infrastructure on Oracle Solaris

Item Task
Operating system general requirements

  • OpenSSH installed manually, if you do not have it installed already as part of a default Oracle Solaris installation

  • The following Oracle Solaris on SPARC (64-Bit) kernels are supported:

    Oracle Solaris 11.4 (Oracle Solaris 11.4.2.0.1.3.0) or later SRUs and updates
    Oracle Solaris 11.3 SRU 31 (Oracle Solaris 11.3.31.6.0) or later SRUs and updates

Review the system requirements section for a list of minimum package requirements.
