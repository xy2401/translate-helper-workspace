Recommended Postinstallation Tasks

Oracle recommends that you complete these tasks after installation.

  • Setting Up Additional User Accounts
    You can set up additional user accounts to manage your database.
  • Updating the Oracle ORAchk Health Check Tool
    Update the existing version of your Oracle ORAchk utility to perform proactive heath checks for the Oracle software stack.
