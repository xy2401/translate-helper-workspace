Description of the illustration cc12_server_pool_overrides.gif

The Server Pool Directive Override section contains a table that has five main columns. Three of the main columns, Min, Max, and Importance, are split into two subcolumns: Current Value and Override. The five main columns are: Server Pool, Min, Max, Current (server pool size), and Importance. The table has three rows which have the following values:

  • backoffice, current min:1, override min:1, current max:-1, override max:-1, current size: 2, current importance:10, override importance: 10.

  • Free, current min:0, override min:null, current max:-1, override max:null, current size: 0, current importance: 0, override importance: 0

  • online, current min:1, override min:1, current max:-1, override max:-1, current size: 2, current importance:20, override importance: 20

End of description.
