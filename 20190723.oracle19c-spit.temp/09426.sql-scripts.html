B SQL Scripts

This appendix describes SQL scripts that are required for optimal operation of the Oracle Database.

The SQL scripts are described in the following sections:

  • Creating the Data Dictionary

  • Creating Additional Data Dictionary Structures

  • The "NO" Scripts

  • Upgrade Scripts

  • Java Scripts

Note:

Comments within the SQL scripts themselves contain more detailed information and examples.

Note:

To run scripts in a pluggable database (PDB), connect to the PDB before running the script. For more information about running scripts in a multitenant container database (CDB), see Oracle Multitenant Administrator's Guide.
