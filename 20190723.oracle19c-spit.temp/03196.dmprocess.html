Description of the illustration dmprocess.png

This graphic identifies four phases in the data mining process: Problem Definition, Data Gathering and Preparation, Model Building and Evaluation, and Knowledge Deployment.

Problem Definition points to Data Gathering and Preparation. This phase includes:

  • Data Access

  • Data Sampling

  • Data Transformation

Data Gathering and Preparation points to Model Building and Evaluation. This phase includes:

  • Create Model

  • Test Model

  • Evaluate and Interpret Model

Model Building and Evaluation points to Knowledge Deployment. This phase includes:

  • Model Apply

  • Customer Reports

  • External Applications

Knowledge Deployment points back to Problem Definition, illustrating the iterative nature of the data mining process.
