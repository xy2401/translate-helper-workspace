6.183 DOCUMENT_LINKS

DOCUMENT_LINKS provides system information about Oracle XML DB document links in Oracle XML DB Repository documents.

When an XML document that includes XLink or XInclude links is added to the repository, these links can be mapped to document links, which are tracked using view DOCUMENT_LINKS.

Column Datatype NULL Description

SOURCE_ID

RAW(16)

The source resource OID

TARGET_ID

RAW(16)

The target resource OID

TARGET_PATH

VARCHAR2(4000)

This column is always NULL. It is reserved for future use.

LINK_TYPE

VARCHAR2(8)

The document link type: Hard or Weak

LINK_FORM

VARCHAR2(8)

Whether the original link was of form XLink or XInclude

SOURCE_TYPE

VARCHAR2(17)

Whether the link is contained in Resource Content or Resource Metadata
