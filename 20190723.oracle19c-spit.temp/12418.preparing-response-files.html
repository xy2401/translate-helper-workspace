Preparing Response Files

Review this information to prepare response files for use during silent mode or response file mode installations.

  • Editing a Response File Template
    For Oracle Database Client, response files are located in the $ORACLE_HOME/response directory.
  • Recording Response Files
    You can use Oracle Universal Installer (OUI) in interactive mode to record response files, which you can then edit and use to complete silent mode or response file mode installations.
