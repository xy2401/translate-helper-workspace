Preface

This guide provides specific use cases for Fleet Patching and Provisioning to simplify lifecycle management tasks in your standard operating environment.

Audience

This guide is intended for anyone responsible for provisioning, patching, and upgrading their Oracle Database software using Fleet Patching and Provisioning. Additional installation guides for Oracle Database, Oracle Real Application Clusters, and Oracle Clusterware are available at:

http://docs.oracle.com
