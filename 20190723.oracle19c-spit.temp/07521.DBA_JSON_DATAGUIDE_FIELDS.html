5.139 DBA_JSON_DATAGUIDE_FIELDS

DBA_JSON_DATAGUIDE_FIELDS extracts the path and type information from all the data guides in the database, which are the data guides returned by the DBA_JSON_DATAGUIDE view. Its columns are the same as those in ALL_JSON_DATAGUIDE_FIELDS.
