Get job details

get

/rhp-restapi/rhp/jobs/{jobId}

Return the configuration of the Job resource associated to the given job Id.

Request

Supported Media Types
  • application/json
Path Parameters
  • jobId: string
    Unique integer job ID value for the scheduled job.
