The screen shows a database filter field, and a list field. The list field is in the form of a five-column table, with the following columns, in order: a selection box column; a source database column, which shows database names; a version column, which shows the database release for the named database in that row; a type column, which shows the type of database; an Oracle home column, which shows the Oracle home path of the database in that row. Below the list field are two text entry boxes. The first text entry box prompts you to provide the SYSDBA username for the database you have selected. The second text entry box prompts you to provide the SYSDBA password for the database that you select.

At the bottom of the page are Help, Back, Next, and Cancel buttons.
