The Prerequisite Checks window provides information about the success or failure of prerequisite checks and automatic fixups.

The Prerequsite Check window shows two selection buttons: Check Again, Fix & Check Again.

There are two drop-down list boxes. The Show drop-down list enables you to select a specific error or warning to view. The Show All Containers drop-down list enables you to see information about individual PDBs in a CDB. Under the buttons, a table is provided with the following columns:

Validation, which describes validation errors.

Severity, which describes the nature of the validation error.

Fixable, which indicates if you can fix the error by using a generated fixup script.

These options are described in more detail in the text that follows the images.

Help, Back, Next, and Cancel buttons are at the bottom of the window.
