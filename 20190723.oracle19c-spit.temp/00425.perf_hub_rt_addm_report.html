Description of the illustration perf_hub_rt_addm_report.gif

This screen shot shows the Performance Hub page, with the ADDM tab displayed. The title of the page is Performance Hub: Real Time - Last Hour, which indicates that the Real Time - Last Hour option was selected in the Select Time Period field. To the right of the Select Time Period field is the Hide Time Picker button. To the right of the Hide Time Picker button is the PerfHub Report icon. To the right of the PerfHub Report icon is the AWR Report icon. The time that the Performance Hub page was last refreshed appears to the right of the AWR Report button.

Below the UI elements at the top of the page is the time picker, which shows the CPU, User I/O, and Wait values from 12:05 PM to 1:05 PM of the current day.

Below the time picker, the Summary, Activity, Workload, Monitored SQL, and ADDM tabs appear. The ADDM tab is displayed.

The ADDM Tasks table shows the ADDM tasks for the selected time period. The Task Name, Task ID, Time, Task Analysis Type, Number of Findings, and Average Active Sessions columns appear in the ADDM Tasks table. The Search field at the top right of the table enables you to search for a particular string in the table.

The Real-Time ADDM Reports table at the bottom of the tab shows the Real-Time ADDM reports for the selected time period. The Report Name, Report ID, Generation Time, and Reasons columns appear in the Real-Time ADDM Reports table. The Search field at the top right of the table enables you to search for a particular string in the table.
