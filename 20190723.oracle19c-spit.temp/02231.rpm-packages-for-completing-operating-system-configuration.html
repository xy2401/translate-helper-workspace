RPM Packages for Completing Operating System Configuration

To complete basic operating system configuration, use the RPM packages that Oracle creates for your Linux distribution.

Oracle Linux Preinstallation RPM with ULN Support

Oracle Preinstallation RPM for your Oracle Linux 7 kernel (oracle-database-preinstall-19c).

Unbreakable Linux Network (ULN) customers can obtain the Oracle Preinstallation RPM by using yum.

Oracle Linux Preinstallation RPM Without ULN Support

Oracle Linux packages on the Oracle Linux yum server website:

http://yum.oracle.com/repo/OracleLinux/OL6/latest/x86_64//
