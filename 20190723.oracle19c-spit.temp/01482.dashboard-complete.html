Description of the illustration dashboard-complete

The screenshot shows the Quality of Service Management Dashboard. The screenshot shows the four sections of the dashboard, which are General, Performance Overview, Recommendations, and Resource Wait Times Breakdowns.

The General section of the dashboard lists the status of QoS Management (enabled or disabled), the current active policy, and links to the performance class performance charts, and the current recommendation, if available.

The Performance Overview section has a chart showing the metrics for the defined performance classes. This chart is described in detail in a separate topic.

The Recommendations section lists the current recommendation, if available. This screenshot shows a recommendation to promote the salescart_pc from Consumer Group 2 to Consumer Group 0, which will help salescart_pc get the resources it needs to meet its performance objectives.

The last section on the dashboard page is the Resource Wait Times Breakdown, which displays the resource wait times by performance class.

End of description.
