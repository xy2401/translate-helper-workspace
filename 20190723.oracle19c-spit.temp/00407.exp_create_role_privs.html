Description of the illustration exp_create_role_privs.gif

This is a screenshot of the Privilege page of the Create Role wizard. The available system privileges and roles table on the left shows the available system privileges and roles that can be assigned to the role. The selected system privileges and roles table on the right shows the system privileges and roles that are currently selected for the role.Between the two tables are a right arrow button and a left arrow button, which are used to move one or more selected system privileges or roles between the two tables. The selected system privileges and roles table includes the CREATE TABLE, CREATE VIEW, CREATE PROCEDURE, CREATE TRIGGER, CREATE SEQUENCE, and CREATE SYNONYM system privileges. At the bottom of the page is a left arrow button, Show SQL button, OK button, Cancel button, and right arrow button. The left arrow button enables you to move to the previous page in the Create Role wizard. The right arrow button enables you to move to the next page in the Create Role wizard.
