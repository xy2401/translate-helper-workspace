6.175 DBMS_METADATA_TRANSFORM_PARAMS

DBMS_METADATA_TRANSFORM_PARAMS documents the valid transform parameters for each transform.

Column Datatype NULL Description

NOT NULL

TRANSFORM

VARCHAR2(128)

NOT NULL

Transform name

VARCHAR2(128)

NOT NULL

Parameter name

DATATYPE

VARCHAR2(11)

BOOLEAN, TEXT, NUMBER or UNSPECIFIED

VARCHAR2(2000)

Default value of transform parameter

VARCHAR2(4000)

Description of the transform parameter
