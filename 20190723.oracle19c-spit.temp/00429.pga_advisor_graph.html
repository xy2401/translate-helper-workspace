Description of the illustration pga_advisor_graph.gif

This is a screenshot of the PGA Advisor graph on the Memory Management page in EM Express. The PGA Advisor graph appears on the Memory Management page when either automatic shared memory management or manual shared memory management is enabled for the database. The horizontal axis of the graph is labeled pga_aggregate_target (MB) and shows possible values for the PGA_AGGREGATE_TARGET initialization parameter. The horizontal axis starts with a value of 0 and displays values for the PGA_AGGREGATE_TARGET initialization parameter in 40 MB increments up to 240 MB. The vertical axis of the graph is labeled Estimated Cache Hit Percentage. The vertical axis starts with a value of 0 and increases in increments of 20% to 100%. An orange line on the graph plots possible values for the PGA_AGGREGATE_TARGET initialization parameter. The current setting for the PGA_AGGREGATE_TARGET initialization parameter is shown by a blue dot near the 40 MB value.
