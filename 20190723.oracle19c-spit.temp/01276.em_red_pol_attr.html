Description of the illustration em_red_pol_exp_bldr.png

Image em_red_pol_exp_bldr.png shows the Policy Expression Builder, which as the following components from top to bottom, left to right:

  • The label Policy Expression Builder

  • A menu listing the environment for the expression, which in this case is Oracle Database Environment. To the right is the Add button, which is enabled when you select when the policy is in effect.

  • An option listing "Policy is in effect when" with menus enabling you to select when the policy is in effect. This example says, "Policy is in effect when session user is not SCOTT."

  • The Policy Expression field, with the Edit check box to the right. When you select the Edit check box, the field expands so that you can manually build an expression.

  • The OK button, and to the right of it, the Cancel button.
