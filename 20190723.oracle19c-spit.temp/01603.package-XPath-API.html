13 Package XPath for XML C APIs

XPath methods process XPath related types and interfaces.

The following table summarizes the methods available through the XPath package for XML C APIs.

   ub4 ctxsize);

xsl
 
IN

baseuri

ctxnode

ctxpos

ctxsize
