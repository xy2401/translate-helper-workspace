Recommended Postinstallation Tasks

Oracle recommends that you complete these tasks after installation.

  • Creating a Backup of the root.sh Script
    Oracle recommends that you back up the root.sh script after you complete an installation.
  • Setting Language and Locale Preferences for Client Connections
    Configure client applications connecting to an Oracle Database according to your locale preferences and your I/O device character set.
  • Recompiling All Invalid Objects
    Oracle recommends that you run the utlrp.sql script after you install, patch, or upgrade a database, to identify and recompile invalid objects.
  • Updating the Oracle ORAchk Health Check Tool
    Update the existing version of your Oracle ORAchk utility to perform proactive heath checks for the Oracle software stack.
  • Enabling Data Analytics Accelerators on SPARC for Oracle Database
    The microprocessors for the SPARC M7 and T7 series servers include Data Analytics Accelerator (DAX) coprocessors. These coprocessors perform query-related operations directly through the hardware, which improves Oracle Database performance.
