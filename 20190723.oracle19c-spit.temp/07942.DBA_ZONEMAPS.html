6.169 DBA_ZONEMAPS

DBA_ZONEMAPS describes all the zone maps in the database. Its columns are the same as those in ALL_ZONEMAPS.

Note:

This view is intended for use with Oracle Exadata release 12.1.2.1.1 or later.
