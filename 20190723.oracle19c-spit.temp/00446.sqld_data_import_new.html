Description of the illustration sqld_data_import_new.gif

This is a screenshot of the Column Definition page in the Data Import Wizard in SQL Developer. The Match By field has a value of Name. Below the Match By field are the Source Data Columns list on the left and the Target Table Columns section on the right. COLUMN5 is selected at the bottom of the Source Data Columns list. The Target Table Columns section shows the Name field with PO_DATE_RECEIVED selected. The Data Type shows a value of DATE. The Format column is empty. The Nullable? field is enabled, and the Default field and Comment field are empty.
