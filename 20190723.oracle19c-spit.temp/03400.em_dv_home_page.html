Description of the Illustration em_dv_home_page.png

The image em_dv_home_page.png shows the Home page of the Database Vault Administrator pages. From top to bottom, left to right, are the following components:

  • The Home tab, which is selected, and the Administration tab.

  • To the right of the Home tab is a Page Refreshed status, which gives the last time the page was refreshed. To the right of this time is a Refresh button.

  • The General area, which has the following:

    • From top to bottom, the Status, Realms, Command Rules, Attempted Violations, Database Vault Policy Changes, and Logged in as settings

    • To the right of Status is a Disable button, which if the status is Disabled, becomes an Enabled button

    • To the right of the Logged in as setting is the Change Password button

  • To the right of the General area is the Attempted Violations area, which contains:

    • Following the Attempted Violations label and to the right is the Time Series button, which changes the pie chart style to a graph style. To the right of this button is a Vew Data label. Following the View Data label is list that enables you to select the time span, such as Last 24 hours.

    • Top 5 Attempted Violations, in pie chart formation. Following this label is a menu for you to select Database Vault components, such as Realms. Following this pie chart is a label saying Oracle Enterprise Manager, which is where the violations took place.

    • To the right of the 5 Attempted Violations is the Top 5 Violators pie chart. Following the label is a menu for you to seelct the type of violator, such as Users. Following the pie chart is a listing of violating users.

  • Following the General area is the Database Vault Policy Propagation area, which contains the Database Vault Policy Propagation link, with this label “(Use this feature to securely propagate Database Vault Policies to multiple databases)”.

  • Following the Database Vault Policy Propagation area is the Database Vault Reports area. It contains the following links:

    • Configuration Issues Reports

    • Enforcement Audit Reports

    • Configuration Changes Audit Reports

    • Simulation Mode Reports

  • Following the Database Vault Reports area is the Alerts table, which has these columns: Severity, Category, Name, Message, Alert Triggered.
