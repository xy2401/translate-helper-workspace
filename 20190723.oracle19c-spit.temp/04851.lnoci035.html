Description of the illustration lnoci035.eps

This illustration shows six boxes connected by downward pointing arrows illustrate basic object operational flow. The boxes in order of flow are labeled as follows:

  1. Initialize OCI in Object mode

  2. Pin Object (brings the object into the client-side cache)

  3. Operate on Object in Cache

  4. Mark Object as Dirtied

  5. Flush Changes to Object

  6. Refresh Object
