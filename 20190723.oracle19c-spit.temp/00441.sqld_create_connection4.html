Description of the illustration sqld_create_connection4.gif

This is a screenshot of the New / Select Database Connection dialog box being used to create a new database connection for a user in a non-CDB. The database connections table in the upper left corner shows the current database connection and details for SQL Developer. On the right side at the top, the Connection Name field has a value of jonathan, the Username field has a value of jonathan, and the Password field has the password for jonathan entered (and obscured by dots). On the Oracle tab, the Connection Type field has a value of Basic, the Role field has a value of default, the Hostname field has a value of jonathan_db_host.example.com, the Port field has a value of 1521, and the SID field has a value of orcl.
