Description of the illustration adxdk080.gif

Arrows lead from an oval labeled “file, string buffer, or URL xml input” to ovals labeled “DOMBuilder.addDOMBuilder Listener()” and “DOMBuilder.parse()”. A dotted arrow leads from oval “file, string buffer, or URL xml input” to an oval labeled “perform other tasks”, which has an associated callout “see the list of available methods”. A dotted arrow leads from oval “perform other tasks” to an oval labeled “.DOMBuilderListener()”. A line (not an arrow) connects oval “.DOMBuilderListener()” with an oval labeled “.DOMBuilderStarted()”. A line connects oval “.DOMBuilderStarted()” with an oval labeled “.DOMBuilderError()”. An arrow leads from oval “.DOMBuilderListener()” to an oval labeled “DOMBuilderListener.DOMBuilderOver()”, which has an associated callout “async call”. An arrow leads from oval “DOMBuilderListener.DOMBuilderOver()” to an oval labeled “DOMBuilder.getDocument()”. An arrow leads from oval “DOMBuilder.getDocument()” to an oval labeled “DOM Document”.
