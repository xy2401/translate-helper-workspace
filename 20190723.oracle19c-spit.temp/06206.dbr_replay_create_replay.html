Description of the illustration dbr_replay_create_replay.gif

In the Replay Task Summary section, the Name field has a value of NewEmpReplay, the Description field is blank, and the Owner field has a value of SYSMAN.

The Captured Workloads table contains one row of entries for the following columns: Name, Database Name, Database Version, Database Time (hh:mm:ss), Workload Duration (hh:mm:ss), Storage Host, and Storage Location. The Workload Analyzer Report column has the eyeglasses icon as a link for accessing the report.

The Replays table has no rows for the following columns: Name, Owner, Status, Database Name, Database Version, Creation Date, and Description.

The user controls above this table are View, Create, Delete, Compare, and Detach. The cursor is placed over Create, and it is highlighted.

End of description.
