Description of the illustration em_red_pol_edit.png

Image em_red_pol_edit.png shows the Data Redaction page. From top to bottom, left to right, are the following components:

  • The label Data Redaction, followed by text explaining what Oracle Data Redaction does

  • The tabs Policies and Formats, with the Policies tab selected

  • The label Search Data Redaction Policies followed by the fields Schema, Table/View, and Policy Name. By default, the % sign is in the fields, which sets the search to all items. Following the search fields is a Go button.

  • The label Policies, followed by, from left to right, the buttons Create, Edit, View, Enable, Disable, Delete

  • Following the buttons is a table with the columns Schema, Table/View, Policy Name, Enabled, Restricted Column.
