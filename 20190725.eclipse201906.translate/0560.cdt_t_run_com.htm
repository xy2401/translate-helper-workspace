指定运行配置的位置

创建运行配置时，它会在.launch以扩展名org.eclipse.debug.core保存。您可以指定存储运行配置的备用位置。您还可以指定访问它的方式以及运行应用程序时要打开的透视图。

要指定运行配置的位置：

  1. 在“C / C ++项目”视图中，选择一个项目。
  2. 单击运行>运行或运行>调试 。
  3. 在“ 配置”框中，展开“ C / C ++ Local” 。
  4. 单击“ 常用”选项卡。
  5. 要将 .launch保存到工作区中的项目，并且能够将其提交到CVS，请单击“ 共享” 。
  6. 在“ 文件夹选择”窗口中，选择一个项目，然后单击“ 确定” 。
  7. 要指定运行时打开的透视图，请从“ 运行模式”列表中选择透视图。
  8. 要指定运行时打开的透视图，请从“ 调试模式”列表中选择透视图。
  9. 单击“运行” ，或根据需要执行以下操作：
    • 要设置应用程序运行时要使用的环境变量和值，请参阅设置环境变量 。
    • 要选择调试应用程序时要使用的调试器，请参阅定义调试设置 。
    • 要指定调试C或C ++应用程序时使用的源文件的位置，请参阅指定源文件的位置 。
    • 要指定运行配置的存储位置，访问方式以及运行应用程序时打开的透视图，请参阅选择运行/调试配置 。
