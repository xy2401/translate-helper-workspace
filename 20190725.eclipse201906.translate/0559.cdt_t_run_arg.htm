指定执行参数

您可以指定应用程序使用的执行参数以及运行配置的工作目录。

要指定执行参数：

  1. 在“C / C ++项目”视图中，选择一个项目。
  2. 单击运行>运行或运行>调试 。
  3. 在“ 配置”框中，展开“ C / C ++ Local” 。
  4. 选择运行或调试配置。
  5. 单击“ 参数”选项卡。
  6. 在“ C / C ++程序参数”框中，键入要传递给命令行的参数。
  7. 要在工作空间中指定本地目录或其他项目，请清除“ 使用默认工作目录”复选框。
  8. 单击“运行”或根据需要执行以下操作：
    • 要设置应用程序运行时要使用的环境变量和值，请参阅设置环境变量
    • 要选择调试应用程序时要使用的调试器，请参阅创建运行配置
    • 要指定调试C或C ++应用程序时使用的源文件的位置 ，请参阅指定源文件的位置
    • 要指定运行配置的存储位置，访问方式以及运行应用程序时要打开的透视图，请参阅指定运行配置的位置
