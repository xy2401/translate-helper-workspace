嵌入式映射

使用嵌入式映射指定实体的持久字段或属性，该实体的值是可嵌入类的实例。

  1. 在JPA Structure视图中 ，右键单击要映射的字段。

  2. 选择Map as> Embedded 。JPA详细信息视图（用于属性）显示所选字段的属性。

    JPA详细信息，嵌入式映射


  3. 完成嵌入式映射区域中的每个字段。

Eclipse在该字段中添加以下注释：

@Embedded
@AttributeOverride(
    column=@Column(
        table="<COLUMN_TABLE>", 
        name = "<COLUMN_NAME>"
    )
)
