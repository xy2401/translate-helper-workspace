Java编辑器面包屑

Java编辑器痕迹导航栏显示当前插入符号位置及其所有父元素的元素。面包屑嵌入到编辑器中并显示在编辑器的顶部。

要启用面包屑 ，请在工具栏中调用Toggle Java Editor Breadcrumb ，或者在Java编辑器中按Alt + Shift + B.

breadcrumb允许对breadcrumb中显示的任何元素调用操作。可以通过上下文菜单或通过键绑定来调用操作。要在痕迹导轨中选择元素，请单击它或在Java编辑器中按Alt + Shift + B ，然后使用向左和向右箭头键。

使用面包屑还可以导航到工作空间中的其他元素：单击其中一个箭头或任何元素以打开下拉窗口。下拉列表显示打开下拉列表的元素的所有兄弟节点。要打开元素，请单击它。要使用键盘打开下拉列表，请选择一个元素，然后按向下箭头键。

每个视角都会记住面包屑的启用。例如，可以在调试透视图中启用面包屑，并在Java透视图中禁用面包屑。
