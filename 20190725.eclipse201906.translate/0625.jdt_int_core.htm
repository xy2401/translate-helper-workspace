JDT核心

JDT Core（ org.eclipse.jdt.core ）是定义核心Java元素和API的插件。在开发Java特定功能时，应始终将此插件列为先决条件。

JDT Core包使您可以访问Java模型对象和无头Java IDE基础结构。JDT核心软件包包括：

  • org.eclipse.jdt.core - 定义描述Java模型的类。
  • org.eclipse.jdt.core.compiler - 为编译器基础结构定义API。
  • org.eclipse.jdt.core.dom - 支持抽象语法树（AST），可用于检查编译单元的结构，直到语句级别。
  • org.eclipse.jdt.core.dom.rewrite - 支持重写抽象语法树（AST），可用于将编译单元的结构操作到语句级别。
  • org.eclipse.jdt.core.eval - 支持评估剪贴簿或调试器内的代码片段。
  • org.eclipse.jdt.core.formatter - 支持编译单元，类型，语句，表达式等的格式。
  • org.eclipse.jdt.core.index - 支持生成可由JDT搜索引擎使用的索引。
  • org.eclipse.jdt.core.jdom - 支持可用于遍历Java编译单元结构的Java文档对象模型（DOM）。不推荐使用此包中的所有类型。
  • org.eclipse.jdt.core.search - 支持在工作区的Java模型中搜索与特定描述匹配的Java元素。
  • org.eclipse.jdt.core.util - 提供用于操作.class文件和Java模型元素的实用程序类。

自3.0版以来， org.eclipse.jdt.core.jdom已弃用。编译单元结构的操作应该使用org.eclipse.jdt.core.dom来完成。
