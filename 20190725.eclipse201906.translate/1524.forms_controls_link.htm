超链接控制

Hyperlink是一个自定义窗口小部件，用于在UI窗体的上下文中使用时补充标准SWT窗口小部件集。Hyperlink是一个可选的文本控件，其作用类似于Web浏览器超链接：

	Hyperlink link = toolkit.createHyperlink(form.getBody(), "Click here.",
				SWT.WRAP);
		link.addHyperlinkListener(new HyperlinkAdapter() {
			public void linkActivated(HyperlinkEvent e) {
				System.out.println("Link activated!");
			}
		});
		link.setText("A sample link");

当用户与它们交互时，超链接会触发HyperlinkEvent对象。通过添加HyperlinkListener ，客户端可以捕获鼠标何时进入和退出链接，以及激活它（通过鼠标单击或“Enter”键）。

表单工具包创建的超链接会自动插入到超链接组中 。对于属于该组的所有链接， HyperlinkGroup管理常规超链接属性，如普通和悬停前景色，下划线样式等。

由于许多超链接与小图像组合在一起，因此UI Forms提供了一个名为ImageHyperlink的子类，它增加了在一个可单击控件中组合文本和图像的功能。当需要超链接图像（不带文本）时，也可以使用此类。如果未设置图像，则ImageHyperlink行为与Hyperlink相同。
