支持的正则表达式

搜索文本时，您可以使用Pattern类定义的正则表达式。
此外，\ R可用于匹配任何形式的行分隔符（Unix，Windows，Classic Mac OS）。请注意\ R在字符组（[]）中不起作用。

替换字符串允许以下构造：

$ I 插入捕获组 i。
\一世 插入捕获组 i（与$ i相同）。
\ x 引号字符x （除非\ x有特殊含义）
\ r 插入文档的默认行分隔符
\ x hh 插入十六进制字符hh
你好 插入Unicode字符hhhh
\ n 插入换行符（通常，你应该使用\ R代替）
\ r 插入一个回车字符（通常，你应该使用\ R代替）
\ t 插入制表符
\F 插入换页符
\一个 插入警报（哔哔）字符
。\ E 插入转义字符
\ c X. 插入X的控制字符
\C 对于替换字符串的其余部分，保留匹配的大小写（全部小写，全部大写，大写）
