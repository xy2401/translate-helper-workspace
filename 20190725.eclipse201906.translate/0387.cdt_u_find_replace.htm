C / C ++查找/替换

Ctrl + F （或编辑>查找/替换 ）显示“ 查找/替换”对话框。您可以在此处指定要搜索的文本以及用于替换它的文本。

您可以指定：

  • 方向（从当前光标位置向前或向后）
  • 范围（ 所有文件或选定行仅在突出显示的区域内搜索）
  • 搜索是区分大小写还是全字 。您还可以指定搜索是否包装在文件末尾。

如果在“ 查找”字段中使用文本关闭“ 查找/替换”对话框，则可以使用Ctrl + K （或“ 编辑”>“查找下一个” ）或“ Ctrl + Shift + K” （或“ 编辑”>“查找上一个” ）转到该文本的下一个匹配项。“下一步”和“上一页”的说明不受“ 查找/替换”对话框中“ 方向”设置的影响。

注意：搜索目前不支持通配符。

增量查找

您还可以从“ 查找/替换”对话框中选择“ 增量查找 ”。选择此选项后，您在“ 查找”字段中键入的每个字母都会使编辑器焦点移动到您键入的文本的第一个完整位置。您也可以通过按Ctrl + J （ 编辑>增量查找 ）来使用增量查找 。在这种情况下，您键入的文本将显示在Eclipse窗口底部的“状态行”中。

注意： “ 查找/替换”对话框中的设置不会影响“状态行”中增量查找的操作。
