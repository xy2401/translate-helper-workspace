C / C ++项目属性，C / C ++索引器

您可以选择要用于项目的C / C ++ Indexer。索引器对于搜索和相关功能（如内容辅助）是必需的。

启用项目特定设置
如果您需要此项目的特定索引设置，请选中此框。其他常用设置（在“首选项”中定义）将被应用。在这种情况下，以下所有控件都被禁用。
选择索引器
指定要用于此项目的索引器; No Indexer禁用索引。每个索引器都有自己的一组选项。
构建索引器的配置
索引源来自指定配置，或来自活动索引。由于索引需要花费大量时间，因此不建议使用活动配置，因为在每次活动配置更改后将触发重新索引。
