注册备忘单向导

关于本向导

此向导允许您使用Eclipse工作台注册简单或复合备忘单，以使其可通过“ 帮助”菜单访问。

此向导执行的基础操作涉及为Cheat Sheet Content扩展点（ org.eclipse.ui.cheatsheets.cheatSheetContent ）生成与XML相关的标记，并将其添加到属于插件项目的plugin.xml文件中被登记的备忘单包含在内。

可以在注册期间为备忘单指定简要描述和问题域类别。该向导只会注册一次备忘单。此向导的后续使用允许用户更新原始注册的备忘单描述和类别值。

要注册正在编辑的当前备忘单，请单击“ 简单 备忘 单编辑器”或“ 复合 备忘 单编辑器 ”标题区域中的“ 注册此备忘单”超链接。

向导动作

该向导提供以下操作：

标签 类型 名称 描述
新... 按键 创建新类别 单击此按钮可创建新的问题域类别。

在生成的“ 创建新类别”对话框中，在“ 名称”字段中输入类别名称，以指定要注册的备忘单所属的问题域。

新类别将添加到“ 类别”组合框中。

如何访问已注册的备忘单

要在Eclipse工作台中访问已注册的备忘单，请执行以下步骤：

  1. 启动一个新的运行时Eclipse工作台* 。
  2. 在运行时Eclipse工作台中，单击“ 帮助”>“备忘单...”以打开“ 备忘单选择”对话框。
  3. 在对话框中，选择以下选项： 从列表中选择备忘单
  4. 在树查看器中，展开用于对注册的备忘单进行分类的问题域类别。如果未指定，请展开“ 其他”类别。
  5. 选择已注册的备忘单
  6. 单击“ 确定” ，将在“备忘单”视图中打开备忘单。

注意* ：无法通过用于编辑它的（主机）Eclipse工作台中的“ 帮助”菜单访问已注册的备忘单。
