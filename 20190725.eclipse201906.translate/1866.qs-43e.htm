保存观点

本教程演示了如何向透视图添加新视图并重新排列视图。Workbench还允许保存此布局以供将来使用。

  1. 在快捷方式栏中单击“资源”透视图。资源透视图现在处于活动状态。

  2. 拖动“大纲”视图并使用其中一个导航视图堆叠它。

  3. 选择Window> Perspective> Save Perspective As ...

  4. “保存透视图”对话框允许重新定义现有透视图或创建新透视图。

    单击确定以更新资源透视图， 单击是以更新后续确认对话框。如果重置透视图或打开新透视图，将使用新的透视布局。

  5. 在“资源”透视图中，移动“大纲”视图，使其现在与“任务”视图堆叠在一起。

  6. 选择Window> Perspective> Reset Perspective 。请注意，“大纲”视图与导航视图堆叠在一起。最初在Workbench首次启动时它位于导航视图下方，但由于透视图是在导航视图和Outline堆叠的情况下保存的，因此它现在将其视为初始布局。

  7. 选择“ 窗口”>“新建窗口”以打开显示资源透视图的第二个窗口。观察它使用新保存的布局。

  8. 关闭第二个窗口。

虽然Resource透视图已更改，但有一种方法可以恢复原始布局。要将Resource透视图还原为其原始布局：

  1. 打开“ 首选项”对话框

  2. 展开“ 常规”并选择“透视” 。

  3. 选择“ 资源”透视图，然后单击右侧的“ 还原 ”。

  4. 现在已撤消对已保存透视状态的任何更改。要更新正在使用的Resource透视图的当前副本，还可以从Workbench的菜单栏中选择Window> Perspective> Reset Perspective ....
