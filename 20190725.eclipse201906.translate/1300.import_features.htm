功能导入

PDE提供了一个功能导入向导，用于将文件系统中的功能导入工作区。该向导位于顶级菜单中的文件>导入...>插件开发>功能 。

导入位置

默认情况下，导入要素的位置是指定的目标平台 窗口>首选项...>插件开发>目标平台首选项页面。

Target Platform ...按钮是一个快捷方式，用于打开Target Platform首选项页面以进行检查或修改。

如果您选择从文件系统上的任意位置导入功能，并且您不希望在首选项中更改目标平台的位置，请取消选中“从目标平台中选择功能”复选框。然后，使用“ 浏览...”按钮导航并输入所选目录。最后，确保按“ 重新加载”按钮刷新对话框。

导入格式和目标

导入要素时，会在工作区中为每个要素创建项目。项目名称是功能ID和“ -feature ”的串联。

如果PDE检测到您的工作区已设置为基于功能的自托管（详见“ 技巧和窍门”部分），则这些功能将导入到功能/目录中，该功能/目录是工作区所在的插件/目录的兄弟。位于。
