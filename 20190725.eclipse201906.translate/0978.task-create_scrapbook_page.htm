创建Java剪贴簿页面

剪贴簿允许在调试器的控制下运行，检查和显示Java表达式。断点和异常的行为与常规调试会话中的行为相同。

代码在剪贴簿页面上编辑。为每个剪贴簿页面启动VM，其中正在评估表达式。第一次在打开剪贴簿页面后评估表达式时，将启动VM。页面的VM将保持活动状态，直到页面关闭，显式终止（在调试器中或通过编辑器工具栏中的“ 停止评估”按钮），或者在评估System.exit（）时 。

有几种方法可以打开 新的Java剪贴簿页面向导。

  • 创建扩展名为.jpage的文件
  • 从菜单栏中，选择“ 文件” >“ 新建” >“ 其他...” 。然后选择Java > Java Run / Debug > Scrapbook Page 。然后单击下一步 。
一旦打开New Java Scrapbook Page向导：
  1. 在“ 输入或选择文件夹”字段中，键入或单击“ 浏览”以选择新页面的容器。
  2. 在“ 文件名”字段中，键入新页面的名称。如果您不自行输入，则会自动添加.jpage扩展名。
  3. 完成后单击“ 完成” 。新剪贴簿页面将在编辑器中打开。
