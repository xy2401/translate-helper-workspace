C / C ++编辑器

独立调试器提供了一个编辑器，为编辑C / C ++代码提供特定支持。打开C / C ++源文件时，将自动调用此C / C ++编辑器。虽然您可以修改文件的内容，但是如果保存这些更改，则需要在独立调试器之外重建项目，以将可执行文件中的调试信息与使用的源文件同步。

C / C ++编辑器包括以下功能：

  • 语法突出显示
  • 内容/代码帮助
  • 代码折叠
  • 集成调试功能

您可以从Window> Preferences> C / C ++> Editor preferences对话框中自定义Editor视图的某些操作。
