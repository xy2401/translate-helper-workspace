强制回归

选择“ 强制返回”命令以使用指定值从当前方法返回。

您可以强制从方法中提前返回（仅在Java SE 6虚拟机上进行调试时可用）。这将从当前堆栈帧返回一个值，而不执行该方法中的任何其他指令，并释放由synchronized块获取的任何锁。通过选择表达式和强制返回 （ Alt + Shift + F ）来创建返回值。可以从Java编辑器的上下文菜单，顶级“运行”菜单，“ 显示视图 ”以及“ 变量视图”的详细信息窗格中获取此操作。

强制从非void方法提前返回需要计算表达式。例如，如果某个方法返回false，则可以通过在“ 显示视图”中选择一个表达式并调用“ 强制返回”来返回值true。在以下示例中， elementCount不等于零，并且将返回false（请参阅调试hover显示elementCount的值）。

在Debug Shell View中 ，我们可以输入我们想要返回的值，选择它并使用Force Return命令强制方法isEmpty()返回该值（在下面的示例中，我们将强制isEmpty()返回价值true ）。
