编辑观点

C / C ++编辑器提供了用于编辑C / C ++相关文件的专用功能。

与编辑器相关联的是一个特定于C / C ++的Outline视图，它显示了活动C，C ++或makefile的结构。它会在您编辑这些文件时更新。

该编辑器包括以下功能：

  • 语法突出显示
  • 内容/代码帮助
  • 集成调试功能

调用C / C ++编辑器的最常用方法是使用弹出菜单或单击文件（单击或双击，具体取决于用户首选项）从Project Explorer或C / C ++ Project视图打开文件。

C / C ++编辑器本身不包含工具栏，但依赖于主工具栏 ，“ 编辑”菜单 ，“ 搜索”菜单和键绑定操作的使用。
