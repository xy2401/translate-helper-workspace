删除书签

本节将演示如何删除已创建的书签。

  1. 在“书签”视图中，选择JanesText.txt（我们创建的第二个书签）并执行以下操作之一：

    • 从书签的上下文菜单中选择删除。
    • 点击键盘上的Delete键。

    请注意，书签已从“书签”视图中删除。

  2. 应该有一个剩余的书签。此书签与文件JanesFile.txt中的一行相关联。还有另外两种方法可以删除此书签。
    • 在JanesFile.txt编辑器的标记栏中使用“ 删除书签” 。请记住，首次创建书签时，标记栏中使用了添加书签 。

    • 删除书签使用从书签视图书签的弹出式菜单中删除 （如上面完成）。

    这是第二种方法。

  3. 确保在JanesFile.txt上打开一个编辑器。

    虽然编辑器实际上不需要打开，但是当删除书签时，编辑器更新将是可见的。

  4. 在“书签”视图中，选择JanesFile.txt（其余书签）。按Delete键。请注意，书签将从“书签”视图和JanesFile.txt编辑器中删除。
