创建一个新的Runnable JAR文件

要在工作台中创建新的可运行JAR文件：

  1. 从菜单栏的“ 文件”菜单中，选择“ 导出” 。
  2. 展开Java节点并选择Runnable JAR文件 。单击下一步 。
  3. 在里面 在Runnable JAR File Specification页面中，选择一个'Java Application'启动配置，用于创建可运行的JAR。
  4. 在“ 导出目标”字段中，键入或单击“ 浏览”以选择JAR文件的位置。
  5. 选择适当的库处理策略。
  6. （可选）您还可以创建ANT脚本以快速重新生成以前创建的可运行JAR文件。
