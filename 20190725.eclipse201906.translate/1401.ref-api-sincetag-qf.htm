自标签快速修复

此快速修复可用于快速添加标签或帮助纠正现有标签的问题。

可以在以下两个位置之一使用此快速修复：

  1. 问题观点 。
  2. 编辑问题所在的资源 。

问题观点

  • 打开“ 问题”视图。
  • 选择自标签问题。

  • 使用Ctrl + 1从“问题”视图触发快速修复（或使用上面显示的上下文菜单）。

编辑资源

  • 打开API问题所在的资源。
  • 对相应的问题使用Ctrl + 1 。


一旦更新了自标记问题，就会重建相应的资源，并从“问题”视图和任何打开的编辑器中删除问题。
