创建一个新的JAXB项目

使用此过程可以创建新的JPA项目。

  1. 从Navigator或Project Explorer中，选择File> New> Project 。将出现“选择向导”对话框。


    小费：

    您还可以选择JPA透视图，然后选择“ 文件”>“新建”>“JAXB项目” 。

    选择“创建JAXB项目”向导


  2. 选择JAXB Project ，然后单击Next 。将出现“ 新建JAXB项目”页面 。

    JAXB项目页面


  3. 完成New JAXB Project页面上的字段以指定项目名称和位置，目标运行时和预定义的配置。


    注意：

    Java SE开发不需要Target运行时 。

  4. 单击下一步 。将出现Java源页面。

    Java源页面


  5. 单击“ 添加文件夹”以将现有Java源文件添加到项目中。

  6. 单击下一步 。出现JAXB Facet页面 。

    JAXB Facet


  7. 填写JAXB Facet页面上的字段以指定您的供应商特定平台，JPA实现库和数据库连接。

    单击“ 管理库”以创建或更新JPA用户库。单击“ 下载库”以获取其他JPA实现库。


    注意：

    根据您的JAXB实现（例如，Generic或EclipseLink），在创建JAXB项目时可能会有不同的选项。

  8. 单击完成 。Dali创建了新的JAXB项目。

    您现在应该打开JPA Development透视图 。
