自定义Java编辑器

JDT工具实现了功能齐全的Java编辑器。它们还定义了API，因此您可以将Java文本编辑功能分配给您自己的编辑器，或使用插件特定功能增强Java编辑器。

Java文本编辑

JavaSourceViewerConfiguration描述了显示Java代码的查看器的配置。给定源代码查看器，此类提供对实用程序类的实例的访问，这些实例类向查看器添加Java特定行为，例如着色，令牌扫描，内容辅助等。

Java文本盘旋

当您实现提供与Java代码相关的其他功能的插件时，您可能希望在编辑器中实现新的悬停行为，以便在用户将鼠标悬停在Java代码上时显示有关模型元素的信息。例如，当用户将鼠标悬停在编辑器中的变量名称上时，调试器使用自定义悬停来显示变量值。

扩展点org.eclipse.jdt.ui.javaEditorTextHovers允许您注册自己的悬停实现。您负责实现IJavaEditorTextHover并在扩展定义中注册实现类。您还为悬停类型提供标签，以便用户可以从编辑器首选项页面启用和禁用不同类型的悬停。
