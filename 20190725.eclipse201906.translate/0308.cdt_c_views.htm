C / C ++透视图中的视图

C / C ++视图是帮助您执行创建C / C ++程序所涉及的任务的面板。C / C ++透视图在单个Eclipse窗口中显示这些面板。

改变观点：
  • 您可以向透视图添加视图。从菜单栏中选择“ 窗口”>“显示视图”>“其他”，然后从“显示视图”对话框中选择新视图。
  • 要将当前透视图重置为其原始布局，请从菜单栏中选择“ 窗口”>“重置透视图” 。

以下视图通常用于C / C ++透视图：

Project Explorer
显示工作空间目录下的所有文件系统文件。
安慰
显示程序的输出以及构建工具的输出。
编辑
C / C ++编辑器视图提供了用于编辑C / C ++相关文件的专用功能。
制作目标
使您可以选择要在工作区中构建的制作目标。
大纲
显示当前在编辑器中打开的文件的结构。
问题
如果在构建期间遇到任何错误，它们将显示在“问题”视图中。
属性
显示当前在视图或编辑器中选择的项目的属性。
任务
列出要跟踪的任务，可以是要做的事情安排，也可以是已完成事情的历史记录。
