Java编辑器

Java编辑器提供了编辑Java代码的专用功能。

与编辑器相关联的是特定于Java的Outline视图，它显示了活动Java编译单元的结构。它在用户编辑编译单元时更新。

编辑器还可以显示面包屑导航栏。面包屑显示当前光标位置处元素的父链。当用户编辑编辑单元或更改选择时，它也会更新。

可以在二进制CLASS文件上打开Java编辑器。如果包含CLASS文件的JAR文件具有源附件，则编辑器将显示相应的源。

该编辑器包括以下功能：

  • 语法突出显示
  • 内容/代码帮助
  • 代码格式
  • 进口援助
  • 快速解决
  • 集成调试功能

可以将Java编辑器配置为仅显示整个编译单元或单个Java元素。要更改设置，请使用工具栏按钮仅显示所选元素的来源 。

调用Java编辑器的最常用方法是使用弹出菜单或单击文件（单击或双击，具体取决于用户首选项）从Package Explorer打开Java文件。您还可以通过从其他视图中打开Java元素（如类型，方法或字段）来打开编辑器。
