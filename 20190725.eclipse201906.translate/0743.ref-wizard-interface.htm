新的Java接口向导

该 New Java Interface向导可帮助您在Java项目中创建新的Java接口。

Java接口选项
选项 描述 默认
源文件夹 输入新接口的源文件夹。键入有效的源文件夹路径或单击“ 浏览”以通过对话框选择源文件夹。 启动向导时选择的元素的源文件夹。
包 输入包以包含新界面。您可以选择此选项或下面的Enclosing Type选项。键入有效的包名称或单击“ 浏览”以通过对话框选择包。 启动向导时选择的元素包。如果选择的源文件夹只包含一个包，则使用该包。如果源文件夹不包含任何包，则使用项目名称（如果它是有效的包名称）。
封闭式 选择此选项可选择用于包含新接口的类型。您可以选择此选项或上面的“包”选项。在字段中键入有效名称，或单击“ 浏览”以通过对话框选择类型。 启动向导时选择的编译单元的类型或主要类型或<blank>
名称 键入新接口的名称。 <空白>
修饰符

为新界面选择一个或多个访问修饰符。

  • public，package，private或protected（private和protected仅在指定封闭类型时可用）
  • static（仅在指定封闭类型时可用）
 上市
扩展接口 单击“添加”以选择新接口扩展的接口。 <空白>
你想添加评论吗？ 选中后，向导会在适当的位置向新类添加注释。 不要添加评论
