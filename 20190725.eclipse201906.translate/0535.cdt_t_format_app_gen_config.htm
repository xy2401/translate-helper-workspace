为Formatter应用程序生成配置文件

为格式化程序应用程序生成配置文件涉及修改C / C ++项目的代码格式化程序设置，并将org.eclipse.cdt.core.prefs复制到该项目的.settings目录之外。

  1. 选择一个C / C ++项目，打开弹出菜单，然后选择“ 属性” 。
  2. 选择“ C / C ++常规”>“格式化程序”页面，然后选中“ 启用项目特定设置” 。
  3. 如上所述选择或编辑配置文件。
  4. 完成后单击“ 确定” 。
  5. 使用文件管理器或命令行将workspace/YourCProject/.settings/org.eclipse.cdt.core.prefs复制到新位置。
