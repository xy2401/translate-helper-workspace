C / C ++工具栏

C / C ++工具栏图标

图标命令描述
创建新的 创建新项目，文件夹或文件。
保存 保存当前编辑器的内容。如果编辑器不包含未保存的更改，则禁用。
打印 打印当前编辑器的内容。
全部构建 构建所有项目。
新的C / C ++项目 创建新的C / C ++项目。
创建文件夹 在当前项目中创建一个文件夹。
创建文件 在当前项目中创建一个文件。
创建C ++类 在当前项目中创建C ++类。
建立选定 构建所选项目的活动配置。
管理配置 显示“管理构建配置”对话框。
调试 启动“调试”对话框。
跑 启动“运行”对话框
外部工具 启动“外部工具”对话框
打开元素 打开“打开元素”选择对话框以在编辑器中打开元素。“打开元素”选择对话框显示工作空间中存在的所有元素。
搜索 启动“C / C ++搜索”对话框
添加到工作集 将所选元素添加到工作集。
从工作集中删除 从工作集中删除选定的元素。
修改工作集 修改工作台工作集。
切换标记出现次数 在C / C ++编辑器中打开和关闭标记出现 。
显示空白字符 在编辑器中显示空白字符。
显示所选元素 仅显示所选元素的来源
下一个注释 转到编辑器中的下一个注释。
以前的注释 转到编辑器中的上一个注释。
转到上次编辑位置 将编辑器视图返回到编辑的最后一行，如果上次编辑的文件已关闭，则将重新打开。
背部 浏览打开的文件。
向前 向前浏览打开的文件。
