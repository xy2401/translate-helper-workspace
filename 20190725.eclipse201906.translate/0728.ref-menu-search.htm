搜索操作

搜索菜单命令：

名称

功能

搜索...

打开搜索对话框（ Ctrl + H ）

文件...

在“文件搜索”页面上打开搜索对话框

Java的...

在Java搜索页面上打开搜索对话框

文本

在所选范围内搜索所选文本：

  • 工作区（ Ctrl + Alt + G ）
  • 项目
  • 等级制度
  • 工作集

参考

查找所选范围内所选Java元素的所有引用：

  • 工作区（ Ctrl + Shift + G ）：
  • 项目
  • 等级制度
  • 工作集

声明

查找所选范围中所选Java元素的所有声明：

  • 工作区（ Ctrl + G ）
  • 项目
  • 等级制度
  • 工作集

注意：严格的声明搜索只会返回一个声明。由于这不是很有趣，宣言搜索放宽如下：

  • 方法声明：层次结构搜索在层次结构中查找覆盖和重写的方法。其他搜索查找具有相同名称和签名的方法，但忽略声明类型和返回类型。
  • 字段声明：忽略声明类型和字段的类型。

实施者

在所选范围中查找所选接口的所有实现者：

  • 工作区
  • 项目
  • 工作集
阅读权限

查找所选范围内所选字段的所有读取访问：

  • 工作区
  • 项目
  • 等级制度
  • 工作集
写入权限

查找所选范围内所选字段的所有写访问：

  • 工作区
  • 项目
  • 等级制度
  • 工作集

文件中的出现次数

在其文件中查找所选出现的所有Java元素（ Ctrl + Shift + U ）

  • 标识符：解析为所选元素的所有标识符的出现次数
  • 实现方法：由所选的超级接口声明实现的所有方法
  • 抛出异常：可能抛出所选声明异常的所有语句
  • 方法退出：所有可以退出所选返回类型的方法的语句
  • 中断/继续目标：所选中断和继续语句的目标

参考测试

查找引用当前所选类型的所有JUnit测试

搜索范围子菜单：

范围

 可用性

描述

工作区

 所有元素

在完整工作区中搜索

项目

 所有元素

搜索包含所选元素的项目

等级制度

 类型和成员

搜索类型的层次结构

工作集

 所有元素

在工作集中搜索

可以使用工作集对话框配置自定义范围。搜索范围子菜单在搜索中使用后，现有工作集将显示在搜索范围子菜单中。

还可以通过所有Java视图的上下文菜单进行Java搜索。搜索上下文菜单也可在Java编辑器中使用。仅当可以将当前选定的文本解析为Java元素时，才执行搜索。

所选Java元素的类型定义了哪些搜索上下文菜单可用。Java编辑器不会根据选择限制可用Java搜索列表。
