CDT概述

C / C ++开发工具包（CDT）是一组Eclipse插件，为Eclipse工作台提供C和C ++扩展。有关Eclipse的更多信息，请参阅“ Workbench用户指南”>“概念”>“Workbench” 。

CDT提供了一个C / C ++ IDE，它简化了许多可以从命令行使用的相同工具。CDT还可以与许多外部实用程序通信并解释其响应，例如：

  • 构建（例如make）。
  • 编译（如gcc）。有关更多信息，请访问http://gcc.gnu.org 。
  • 调试（如gdb）。有关更多信息，请参阅http://sources.redhat.com/gdb/ 。
注意：虽然make，gcc和gdb是文档中使用的示例，但几乎可以使用任何类似的工具或实用程序集。

CDT作为Eclipse工作台的C / C ++透视图打开。C / C ++透视图由编辑器和以下视图组成：

Project Explorer
显示工作空间目录下的所有文件系统文件。
安慰
显示程序的输出，以及构建和外部工具链的输出。
编辑
C / C ++编辑器视图提供了用于编辑C / C ++相关文件的专用功能。
制作目标
使您可以选择要在工作区中构建的制作目标。
大纲
显示当前在编辑器中打开的文件的结构。
问题观点
如果在构建期间遇到任何错误，它们将显示在“问题”视图中。
属性
显示当前在视图或编辑器中选择的项目的属性。
搜索
显示搜索文件或文本的结果。
任务
列出要跟踪的任务，可以是要做的事情安排，也可以是已完成事情的历史记录。

有关更多信息，请参阅“ Workbench用户指南”>“概念”>“透视图” 。

CDT更新

“ 安装/更新”向导提供有关当前Eclipse安装的信息，并提供管理更新的框架。有关更多信息，请参阅“ Workbench用户指南”>“任务”>“更新和安装软件” 。

要查看可用于您安装的工具集的更新列表，请单击“ 帮助”>“检查更新” 。
