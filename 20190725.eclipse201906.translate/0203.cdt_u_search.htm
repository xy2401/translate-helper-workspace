C / C ++搜索

搜索字符串
指定搜索字符串。
搜索功能支持以下通配符：
  • 使用*表示任何一系列字符
  • 使用 ？表示任何单个字符
  • 使用\ *查找星号字符（*）


区分大小写
搜索将返回与搜索字符串中指定的大小写精确匹配的结果。
 
搜索
指定要包含在搜索中的元素类型：

搜索 描述
类/结构 在搜索中包含类和结构。
功能 搜索命名空间中的全局函数或函数（不是类，结构或联合的成员的函数）。
变量 搜索不是类，结构或联合的成员的变量。
联盟 搜索工会。
方法 搜索作为类，结构或联合的成员的方法。
领域 搜索属于类，结构或联合的成员的字段。
列举 搜索枚举。
枚举 搜索枚举器。
命名空间 搜索命名空间。
的typedef
 搜索typedef。
宏
 搜索宏。
任何元素 包括搜索中的所有元素。

限制于
指定要搜索的上下文类型：

限制于 描述
声明 将搜索限制为声明。
定义 将搜索限制为定义（对于函数，方法，变量和字段）。
参考 将搜索限制为引用。
所有事件 包括搜索中的声明，定义和引用。

范围

范围 可用性 描述
工作区 所有元素 在完整工作区中搜索
精选资源 所有元素 搜索在“ 项目”视图中选择的资源
附上项目
 所有元素
 搜索所选资源的封闭项目（以及包含路径）
工作集 所有元素 在工作集中搜索
