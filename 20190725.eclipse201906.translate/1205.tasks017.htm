多对一映射

使用多对一映射来定义与具有多对一多重性的另一个实体类的单值关联。

  1. 在JPA Structure视图中 ，选择要映射的字段。

  2. 右键单击该字段，然后选择Map As> Many-to-One 。JPA详细信息视图（用于属性）显示所选的属性。

    JPA详细信息，多对一映射


  3. 完成“ 多对一映射”区域中的每个字段。

  4. 完成JPA详细信息视图中的其余区域（对于属性） 。

Eclipse在该字段中添加以下注释：

@JoinTable(joinColumns=@JoinColumn(name="<JOIN_COLUMN>"), 
    name = "<JOIN_TABLE_NAME>")
@ManyToOne(
    targetEntity=<TARGET_ENTITY>, 
    fetch=<FETCH_TYPE>, 
    =<_TYPE>
)
