选择二进制解析器

选择正确的二进制解析器对于确保C / C ++ Projects视图的准确性以及成功运行和调试程序非常重要。Windows用户应选择PE Windows Parser。UNIX和Linux用户应该选择Elf Parser。为开发环境选择正确的解析器并构建项目时，可以在C / C ++ Projects视图中查看.o文件的组件，并在C / C ++编辑器中查看.o文件的内容。您还可以在定义运行/调试配置时轻松浏览可执行文件。

如果使用CDT生成的makefile来构建项目，则在创建期间根据选定的工具链设置适当的二进制解析器。

如果使用自己的makefile进行构建，则在项目创建期间从Preferences中获取二进制解析器的默认设置。

您可以在“首选项”窗口中修改这些设置，以便在将来的项目中使用。

使用用户提供的Makefile为将来的项目设置属性：

  1. 单击窗口>首选项> C / C ++>新建CDT项目向导> Makefile项目
  2. 单击二进制解析器选项卡。
  3. 在Binary Parser列表中，单击：
    • Elf Parser ，如果您是Solaris，UNIX或Linux用户。
    • PE Windows Parser ，如果您是Windows用户。
    • Cygwin PE Parser ，如果您使用的是Cygwin。
  4. 单击确定 。

要为现有项目更改二进制解析器：

  1. 在“C / C ++项目”视图中，右键单击项目，然后选择“ 属性” 。
  2. 从列表中选择C / C ++ Build> Settings 。
  3. 单击二进制解析器选项卡。
