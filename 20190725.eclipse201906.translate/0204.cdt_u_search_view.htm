搜索视图

任何比赛都在报道 搜索视图。

完成搜索并在“ 搜索”视图中显示结果后，您可以将焦点放在该视图上，并在“ 搜索”菜单上获得更多选项。

AC / C ++搜索也可以通过以下视图中所选资源和元素的上下文菜单进行：

  • Project Explorer
  • C / C ++项目
  • 大纲视图
  • 搜索结果视图

搜索上下文菜单也可在C / C ++编辑器中使用。仅当当前选定的文本可以解析为C / C ++元素时，才执行搜索。

所选C / C ++元素的类型定义了哪些搜索上下文菜单可用。C / C ++编辑器不会根据选择限制可用的C / C ++搜索列表。
