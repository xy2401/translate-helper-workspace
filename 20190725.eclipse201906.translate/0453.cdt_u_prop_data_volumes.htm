C / C ++项目属性，构建，设置，数据卷对话框

当用户从“容器设置数据卷”表中选择“添加...”按钮时，将显示“ 数据卷”对话框。与属性选项卡类似，此对话框仅在安装了可选的Docker C / C ++启动功能时可用。

可以在Docker容器中从本地文件系统，另一个Container或创建为空的数据卷中指定数据卷，其中将执行构建。

数据卷对话框
组 描述
集装箱路径 这定义了数据卷将在Docker容器中安装的路径。
没有外部安装 使用此选项可在Docker Container中创建一个未安装到任何外部目录的新空目录。
挂载主机目录或主机文件 使用此选项指定将使用给定Container路径安装在Docker Container中的本地系统路径。
路径 这是要在Docker Container中装入的目录或文件的本地路径。这可以手动指定，也可以通过目录/文件按钮指定。
目录 使用此按钮打开本地目录浏览器，以便在Docker容器中装入本地目录时填写“路径”文本字段。
文件 在Docker Container中装载本地文件时，使用此按钮调出本地文件浏览器以填充“路径”文本字段。
只读访问权限 如果安装了本地目录或文件，并且您不希望Container修改它，请选中此按钮。
装入数据卷容器 如果安装另一个Container的卷，请选中此按钮。
容器 这是一个包含主动运行容器的下拉菜单，用于在选择上述选项时使用数据卷。

适用于： CDT项目。
