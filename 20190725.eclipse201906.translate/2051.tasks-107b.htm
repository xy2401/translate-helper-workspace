启用CVS资源装饰

启用后，CVS将使用指示资源的CVS状态的图标和标签装饰来装饰Workbench视图中的资源。

要启用CVS装饰：

  1. 打开 常规>外观>标签装饰首选项页面。此页面允许启用和禁用Workbench中不同插件定义的所有装饰。
  2. 单击CVS框以启用CVS装饰。
  3. 单击确定 。

注意：在某些情况下，装饰可能不会立即出现，因为它们的计算成本很高。在这些情况下，它们是在后台计算的。

要配置CVS装饰：

  1. 打开 Team> CVS> Label Decorations首选项页面。此页面允许配置装饰在CVS项目，文件夹和文件上的显示方式。
  2. 要配置文本标签装饰，请单击“ 文本”选项卡，然后：
    1. 选择要更改装饰的资源类型（文件，文件夹，项目）。
    2. 在新装饰元素应该到达的位置单击该资源类型的装饰字符串。
    3. 单击该资源类型的“ 添加变量”按钮。
    4. 选择要添加的CVS信息变量。例如，对于文件，您可以选择added_flag ，它将插入放置光标的装饰中。字符串{added_flag}将被添加到装饰字符串中，该装饰字符串将使用添加的资源标签（可以在添加字段的标签装饰中设置）来装饰新添加的资源。
  3. 对要添加的任何其他装饰重复上述操作。
  4. 可以通过从装饰字符串中删除装饰变量名称（即{added_flag} ）来删除装饰。以类似的方式，可以添加间距。
  5. 要配置图标装饰，请单击“ 图标”选项卡，只需启用/禁用图标复选框。
  6. 单击“ 确定”或“ 应用”以使新装饰生效。

注意 ：在“ 常规”选项卡下，可以选择禁用文件夹的深脏装饰。这将使装饰器确定更有效，但会在其中一个导航视图中更难找到脏资源。在“ 同步视图”选项卡下，可以将“同步”视图配置为显示资源的同步状态，作为“结构比较”窗格中显示的文本标签的一部分。

您还可以根据项目的CVS状态更改项目的颜色和字体。设置颜色和字体：

  1. 打开 Team> CVS> Label Decorations首选项页面。
  2. 选中“ 启用字体和颜色装饰 ”复选框。
  3. 接下来，打开 常规>外观>颜色和字体首选项页面。
  4. 在CVS类别中有各种CVS状态，您可以为其设置颜色和字体。根据需要设置它们。
  5. 你可以回到 Team> CVS> Label Decorations首选项页面以预览更改。
  6. 单击“ 确定”或“ 应用”以使新装饰生效。
