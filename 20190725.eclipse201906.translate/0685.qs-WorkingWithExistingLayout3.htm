具有嵌套测试的产品

文件系统上的布局

  • 产品的Java源文件布局在包目录中。
  • 测试的源文件在嵌套包目录中布局。

定义相应项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“Product1”。单击下一步 。

  4. 在下一页上，选择“Product1”源文件夹。

    单击详细信息窗格或按钮中的链接其他源链接在视图栏中。

  5. 在Link Source中单击Browse ....并选择D:\Product1\JavaSourceFiles目录。

    在文件夹名称字段中键入“src”。

  6. 单击“ 完成”以验证并关闭对话框。

  7. 展开“src”源文件夹。选择空包“tests”并使用“用作源文件夹”弹出菜单项或按钮将其设置为源文件夹在视图栏中。

  9. 单击完成 。
  10. 您现在有一个带有两个源文件夹的Java项目：“src”和“src / tests”，它们分别包含D:\Product1\JavaSourceFiles目录和D:\Product1\JavaSourceFiles\tests目录。
