附加或删除断点操作

您可以将一个或多个断点操作附加到单个断点。例如，当断点被击中时，您既可以记录消息又可以播放声音。操作按它们在此断点列表的操作中出现的顺序执行。

要从断点连接或删除断点操作，请执行以下操作：

  1. 在“ 断点”视图中，右键单击断点，然后选择“ 属性” 。
  2. 在“ 用于C / C ++断点的属性”对话框中，单击“ 操作” 。
  3. 在“ 可用操作”列表中：
    • 选择一个操作，然后单击“ 附加”将其链接到断点。根据需要重复。
    • 要添加新的断点操作，请单击“ 新建” 。
    • 要编辑选定的断点操作，请单击“ 编辑” 。
    • 要删除所选操作，请单击“ 删除” 。

      注意 ：删除断点操作会将其从工作区中删除。

  4. 在此断点列表的操作中，您可以：
    • 单击“ 删除”以从列表中删除所选操作。
    • 单击向上可在列表中向上移动所选操作。
    • 单击“ 向下”可在列表中向下移动所选操作。
  5. 单击确定 。
