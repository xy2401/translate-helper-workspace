CVS存储库视图

CVS Repositories视图是CVS Repository Exploring透视图的一部分，它显示了已添加到Workbench的CVS存储库位置。扩展位置会显示该主存储库中的主干（HEAD），项目版本和分支。您可以进一步扩展项目版本和分支以显示其中包含的文件夹和文件。

此视图的弹出菜单还允许您指定新的存储库位置。使用CVS存储库视图检出从存储库到Workbench的资源，配置视图显示的分支和版本，查看资源历史记录以及比较资源版本。

存储库视图中显示两种类型的文件夹。在大多数情况下，它们可以以相同的方式处理。也就是说，可以从存储库视图等检出它们。但是，在某些情况下，即使仍然可以检出模块，也无法在存储库视图中展开模块。

一个CVS模块，映射到CVS服务器上的文件夹。
CVS模块在存储库的CVSROOT / modules文件中定义。

与Workbench中的其他视图一样，CVS存储库视图也有自己的工具栏。工具栏按钮用于刷新视图，导航和创建新的存储库位置。工具栏还包含一个下拉菜单，允许您通过工作集过滤视图。

工具栏

家

此命令重绘视图，将存储库位置显示为视图显示的根。

背部

此命令重绘视图，显示在上一个发出的Go Into命令之前显示的根。

进入

此命令重绘视图，使所选条目的子项成为视图显示的根。

刷新视图

此命令将刷新“存储库”视图的内容。

全部收缩

此命令将折叠“存储库”视图中的所有扩展条目。

添加CVS存储库

此命令将打开向导以帮助您指定新的存储库位置。

下拉式菜单

标题栏中的下拉菜单允许通过工作集过滤存储库视图。它还允许隐藏CVSROOT / modules文件中定义的CVS模块。

上下文菜单

从“存储库”视图的上下文菜单中，您可以执行许多有趣的操作。

新>存储库位置

此命令将打开向导以帮助您指定新的存储库位置（与添加CVS存储库工具栏项相同）。

新>日期标记

此命令在存储库位置和日期类别条目上可用，将打开一个对话框，该对话框允许指定要添加到“存储库”视图中位置条目的“日期”类别的日期标记。

查看

此命令将选定的CVS模块检出到Workbench中的项目，其名称与远程模块（对于服务器上的文件夹的远程模块）相同，或者检查具有模块指定名称的项目（如果模块在模块中定义）服务器上的CVSROOT / modules文件）。

结帐为......

此命令打开“检出”向导，以便允许配置所选远程模块如何签出到Workbench中。

标记为版本...

此命令根据当前分支内容对所选资源进行版本控制。

标记为现有...

此命令根据当前分支内容对所选资源进行版本化，如果需要，可从先前标记的资源中移动标记。

相比

此命令将比较两个选定的资源。

与之比较...

此命令将所选文件夹与同一文件夹的分支或版本进行比较。

展开全部

此命令将在一次传递中获取所选项的所有子文件夹。

刷新视图

此命令将刷新存储库文件夹树。

粘贴连接

根据剪贴板的内容创建新的存储库位置。剪贴板中必须有一个有效的存储库位置字符串（例如：pserver：dev.eclipse.org：/ cvsroot / eclipse）才能粘贴新连接。

复制到剪贴板

将所选存储库的标识字符串复制到剪贴板。

丢弃位置

此命令将删除选定的存储库位置。请注意，先前映射到此存储库的所有项目必须首先取消映射。

配置分支和版本

此命令将打开向导，以帮助您发现存储库中存在的所选文件夹的分支和版本标记，以便将它们添加到存储库视图以允许浏览具有这些标记的资源。

刷新分支......

此命令（在存储库位置条目上可用）允许您刷新所选项目的存储库视图中显示的已知分支和版本的列表。此操作使用为每个项目定义的自动刷新文件。如果特定项目的操作失败，请使用项目上的“配置分支和版本”选择一个或多个相应的刷新文件。

添加到分支列表...

此命令将所选项目添加到存储库视图中指定分支下显示的项目列表中。此命令仅修改存储库视图，不会以任何方式影响存储库。如果要将项目添加到分支，可以在执行此操作后执行Tag with Existing 。

打开

此命令在编辑器中打开所选文件。由于存储库中的文件修订是不可变的，因此编辑器以只读状态打开，因此它不可修改。

显示注释

此命令显示带有注释的文件内容，该注释标识文件中每行代码的作者。

显示历史

此命令在“历史记录”视图中显示所选文件的修订历史记录。

属性

该命令在存储库位置条目上可用，允许修改该位置的任何属性。它还允许分配显示名称和服务器编码，用于转换文件路径并在客户端和服务器之间提交注释（但不影响文件内容）。
