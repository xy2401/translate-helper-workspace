更换

假设在进一步反思之后，Jane很明显刚收到的file1.txt的修订版本不正确，需要更早版本。可以使用存储库中的早期版本的资源替换Workbench资源。要回滚到早期版本：

  1. 在其中一个导航视图中，选择file1.txt 。
  2. 从文件的上下文菜单中，选择替换为>历史....
  3. 历史记录视图将打开并显示可用于所选文件的修订。确保历史记录视图处于“远程修订”模式。（如果不是，您可以通过单击“历史记录”视图工具栏中的“ 远程修订”按钮将其置于“远程”模式。）在表格中选择修订版1.3 ，然后从弹出菜单中选择“ 获取内容” 。
  4. 观察导航视图更新以显示资源的CVS状态。请注意，修改后的文件前面带有“>”，表示file1.txt已更改（通过将其替换为早期版本）。
  5. 现在假设这个旧版本不像最初看起来那么好，事实上存储库中的修订更好。

    不要选择Team> Synchronize with Repository ，而是从HEAD中选择Replace With> Latest 。

    观察到file1.txt已更新为与存储库中的最新修订版具有相同的内容，并且已删除了前导指示符“>”，因为它现在与存储库的修订版本相同。

现在，您在历史记录视图中显示了文件的修订版，可以通过选择表中的修订版并选择“相互比较”来直接比较修订版本。
