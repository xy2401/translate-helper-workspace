C / C ++项目属性，配置管理

所有特定于CDT的属性页都具有统一控件来处理配置。


配置管理属性

选项 描述
组态 指定所选项目的配置类型。通过调试配置，您可以查看程序执行时“内部”发生的情况。要调试应用程序，必须使用为调试编译的可执行文件。这些可执行文件包含其他调试信息，使调试器可以在源代码和从原始源生成的二进制文件之间建立直接关联。发布配置为工具提供了选项集，以创建具有最佳性能的应用程序。
管理配置 打开“管理配置”对话框，您可以在其中根据为所选项目指定的工具链设置哪些配置处于活动状态。此外，您还可以创建新配置，重命名现有配置或删除不再需要的配置。

注意：您在此对话框中指定的“ 配置”选项（与CDT 3.x的行为不同）仅影响这些属性页。这意味着所选配置不会成为活动配置（用于构建项目的配置）。
