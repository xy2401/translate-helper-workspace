更改API工具选项

可以按项目或每个工作区更改选项。

  1. 首先，您需要打开首选项页面。根据所需范围执行以下操作之一：
    • 对于项目范围，右键单击项目并选择“ 属性” 。然后转到插件开发> API错误/警告
    • 对于工作区范围，请转到 插件开发> API错误/警告首选项页面。
  2. 选择要为其修改选项的选项卡：
    • API使用 ：用于与API使用相关的选项（不支持的标签，API限制，API泄漏）
    • API兼容性 ：用于与兼容性问题相关的选项。它们按每种元素排序
    • 版本管理 ：用于与无效，缺失，格式错误的@since标记或不正确的插件版本相关的选项）
    • 分析选项 - 用于一般选项
  3. 通过单击“ 恢复默认值”，可以将页面的所有选项重置为默认值。
  4. 使用“ 设置所有为”组内的按钮将页面的所有选项修改为特定值。
