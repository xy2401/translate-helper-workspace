C / C ++项目属性，构建，设置，错误分析器

使用“ 错误分析程序”属性选项卡可以自定义在构建输出日志中检测错误消息的解析器列表。

错误分析器选项
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
错误解析器 列出CDT当前已知的所有错误解析器。选择要使用的解析器，并根据需要选择相应的行来编辑解析器的选项。
加... 将错误解析器添加到列表中。
编辑... 编辑列表中的当前错误解析器。
删除 删除列表中选定的错误解析器。
提升 将所选解析器移动到列表中的较高位置。注意：顺序仅适用于所选解析器：它们以与用户定义的顺序相同的顺序应用于错误日志。对于未经检查的解析器，不会保留该顺序，因此您不必移动它们。
下移 将所选解析器移动到列表中较低的位置。注意：顺序仅适用于所选解析器：它们以与用户定义的顺序相同的顺序应用于错误日志。对于未经检查的解析器，不会保留该顺序，因此您不必移动它们。

适用于： CDT项目。
