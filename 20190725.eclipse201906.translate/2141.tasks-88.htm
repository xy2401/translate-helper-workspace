设置本地历史首选项

要指示应在Workbench中为每个资源保留的本地历史记录级别：

  1. 打开 常规>工作区>本地历史记录首选项页面。
  2. 在“ 限制历史记录大小”字段中，指定是否要使用以下选项限制本地历史记录大小。
  3. 在“ 保留文件的天数”字段中，键入要为任何一个Workbench资源保留记录的天数。例如，如果键入7，则将保留过去七天保存状态的历史记录。
  4. 在“ 每个文件的最大条目数”字段中，键入要为任何一个Workbench资源保留的状态数。请注意，当您超过每个文件的条目数时，将丢弃最旧的更改，以便为较新的更改腾出空间。
  5. 在“ 最大文件大小（MB）”字段中，键入应保留本地历史记录的资源的最大文件大小（以MB为单位）。如果资源的大小超过分配的最大文件大小，则不会为该资源保留本地历史记录。
  6. 单击“ 确定”设置首选项并关闭“本地历史记录首选项”页面。
