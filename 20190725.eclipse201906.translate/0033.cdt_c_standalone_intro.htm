介绍

独立调试器是一个Eclipse应用程序，它将来自CDT（C / C ++开发工具）项目的Eclipse插件打包到可以从命令行脚本启动的应用程序中。调试器删除了一些可在完整CDT集中找到的与构建和项目设置相关的插件。这简化了仅对调试感兴趣的最终用户的UI，并且对使用Eclipse构建应用程序本身不感兴趣。

此外，独立调试器有助于简化调试，因为它隐藏了Eclipse特定的概念，例如Eclipse项目和启动配置。虽然这些概念是隐藏的，但它们仍然在下面使用，有经验的Eclipse用户仍然可以访问它们。此处的后续文档将尝试指出独立调试器自动执行此类概念的位置以及如何仍可访问这些概念。
