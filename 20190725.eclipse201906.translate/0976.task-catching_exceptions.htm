捕获Java异常

通过指定异常断点抛出异常时，可以暂停执行线程。可以在异常未被捕获，被捕获或两者兼有的位置暂停执行。

  1. 从Breakpoints View或工作台Run菜单中选择Add Java Exception Breakpoint 。
  2. 将显示一个列出所有可用异常的对话框（请参见下图）。



  3. 键入要捕获的异常的名称，或从列表中选择它。
  4. 在页面底部，使用复选框指定执行在抛出异常的位置暂停的方式。
    • 如果希望执行在抛出异常并被catch子句捕获的位置暂停，请选择Caught 。
    • 如果希望执行在未捕获异常的位置暂停，请选择“ 未捕获”。

注意：可以启用和禁用异常断点，并且像常规断点一样具有命中计数。
