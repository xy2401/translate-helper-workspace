语法着色首选项

使用 语法着色首选项面板，用于指定如何呈现C / C ++源代码。每个元素类别（代码，注释和预处理器）都包含可以使用自己的颜色和样式呈现的语言元素列表。

请注意，可以在常规“ 文本编辑器”首选项页面上配置常规文本编辑器设置（如背景颜色）。可以在常规颜色和字体首选项页面上配置字体
