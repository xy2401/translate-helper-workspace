创建API问题过滤器

过滤器可用于删除无法轻易纠正的已知或预期的 API问题。使用Create API Problem Filter快速修复可以轻松创建问题过滤器 。

使用快速修复程序，我们可以看到如何添加新的过滤器。

问题观点

  • 打开“ 问题”视图。
  • 选择API Tools问题。
  • 使用Ctrl + 1从“问题”视图触发快速修复。
  • 选择“完成”以创建新过滤器。

编辑资源

  • 打开API问题所在的资源。
  • 对相应的问题使用Ctrl + 1 。
  • 其中一个快速解决方案是为相应的问题创建问题过滤器。
  • 选择它并为您添加问题过滤器。

创建问题过滤器后，将重建相应的资源，并从“问题”视图和任何打开的编辑器中删除问题。
