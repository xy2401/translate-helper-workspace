属性 描述 默认

柱

包含属性值的数据库列。该字段对应于@Column注释。

默认情况下，假定Column的名称与属性完全相同，并始终包含在INSERT和UPDATE语句中。

名称

包含属性值的数据库列。

表

包含所选列的数据库表的名称。

细节

插入

指定列是否始终包含在SQL INSERT语句中。

真正

可更新

指定此列是否始终包含在SQL UPDATE语句中。

真正

独特

设置列的UNIQUE约束。

假

可空

指定列是否允许空值。

真正

长度

255

精确

设置列值的精度。

0

规模

0

列定义

生成列的DDL时使用的SQL片段。

取

定义如何从数据库加载数据：

  • 渴望 - 数据在实际需要之前加载。

  • 延迟 - 仅在事务需要时加载数据。

急于
