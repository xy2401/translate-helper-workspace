通过项目模板创建项目

从CDT 4.0开始，可以根据IDE供应商定义的模板创建项目。这意味着您可以轻松创建包含自定义源代码的项目，只需单击几下即可使用。生成的项目的某些方面可以由用户定制为新项目向导的一部分。

要从模板创建项目：

  1. 单击文件>新建> C项目或文件>新建> C ++项目 。


  2. 向导将打开，如下所示。
    展开您感兴趣的项目类型。有关项目类型的更多信息，请参阅选择项目类型。这将显示可用作新项目基础的模板项目。CDT附带了一些简单的模板，但ISV可以使用org.eclipse.cdt.core.templates扩展点添加更多模板。
    在“ 项目名称”框中，键入名称。该示例使用“HelloWorld”


  3. 单击下一步 。
  4. 下一页取决于您选择的模板类型。它将包含许多输入字段，可能带有默认值，这些字段将输入到您创建的模板中。


  5. 可以跳过最后一页“选择配置”，或者允许以与非基于模板的项目相同的方式进一步进行项目配置。


  6. 要创建项目，请单击“ 完成” 。
  7. 在Project Explorer中，您将看到已创建新项目。您将看到源文件已使用您在向导中先前输入的值进行了自定义。


  8. 有时，您在会话之间会记住您输入模板向导的值，以便您不需要每次都输入它们。如果要更改这些，可以使用“ 窗口”>“首选项”>“C / C ++”>“模板默认值”
