将变量，表达式或寄存器添加到“内存”视图

您可以从Memory视图向内存监视器添加寄存器，变量名称或表达式（例如原始地址，例如0x00000234）。

要从“内存”视图添加新的内存监视器：

  1. 单击内存视图添加内存监视器按钮（ ）。此按钮位于“ 监视器”窗格中。
  2. 在“监视内存”对话框中，在字段中输入地址或表达式（表达式必须求值为地址）。此条目不需要区分大小写。或者，如果您在调试此应用程序时先前已监视过地址或表达式，请从下拉列表中选择它。根据调试器用户界面所连接的调试引擎，有效表达式的示例包括寄存器名称，变量和HEX地址。

    注意：将寄存器添加到存储器监视器时，寄存器视图是查看应用程序中所有寄存器名称的便捷位置。您可以从视图中记下寄存器的名称，然后在添加内存监视器时使用它。

  3. 单击确定 。

添加新内存监视器后，您可以选择要在“ 渲染”窗格中显示的内存格式。

您还可以从“内存”视图“ 渲染”窗格中添加新的内存监视器：

  1. 单击Memory视图Add Rendering（s）按钮（ ）。此按钮位于“ 渲染”窗格中。
  2. 在“添加内存呈现”对话框中，从下拉列表中选择现有内存监视器 - 或单击“ 添加”以添加新内存监视器。如果添加新的内存监视器，将通过“监视内存”对话框提示您输入要监视的地址或表达式。单击“ 确定”返回“添加内存渲染”对话框。
  3. 执行以下操作之一：
    • 选择要为内存监视器显示的内存呈现，然后单击“ 确定” 。
    • 单击“ 取消” ，然后从“ 渲染”窗格中选择要显示的内存渲染。
