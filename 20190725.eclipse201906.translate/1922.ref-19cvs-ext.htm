使用外部程序与使用内部连接方法 此页面允许您配置ext连接方法以使用外部程序或其他连接方法连接到服务器。提供了后一个选项，以允许自定义连接方法（如extssh）与外部CVS客户端工具保持兼容。 使用外部程序
CVS_RSH 使用此选项配置将被调用以连接到远程CVS服务器的程序。使用以下调用模式调用RSH命令：

CVS_RSH参数CVS_SERVER

 SSH
参数 使用此选项配置传递给CVS_RSH程序的参数。默认参数模式为{host} -l {user}。它可以使用{host}，{user}，{password}和{port}变量进行定制。 -l {host} {user}
CVS_SERVER 使用此选项可配置要运行的远程CVS服务器程序的名称。仅当远程CVS服务器二进制名称不同于默认值时才更改此设置。 CVS
连接类型 如果启用了使用其他连接方法的选项，请使用此选项设置要用于使用ext连接方法的存储库位置的连接方法。 PSERVER
