命令细节部分

关于本节

本部分允许您执行以下操作：

  • 测试可执行的工作台命令
  • 编辑可执行工作台命令的适用参数
  • 将可执行工作台命令的字符串表示形式复制到剪贴板。

通过在“ 命令”部分中选择命令节点来激活此部分。

部分字段

本节包含以下字段：

标签 类型 名称 描述 需要
命令ID 文本 Workbench命令ID 此不可编辑的字段显示工作台命令ID。 N / A
命令参数（0或更多） 文字或组合框 命令参数 工作台命令可以具有0个或更多可配置参数。为每个命令参数生成一个输入字段。

在这些字段中输入适用的值，以相应地修改工作台命令的行为。 		N / A

部分行动

本节提供以下操作：

标签 类型 名称 描述
执行 超链接 执行Workbench命令 单击此链接以从Command Composer对话框中执行workbench命令。

注意：此操作不适用于无法在对话框中执行的工作台命令。
复制到剪贴板 超链接 将命令字符串复制到剪贴板 单击此链接可将可执行工作台命令的字符串表示复制到剪贴板。

字符串表示可用于将工作台命令直接嵌入到简单的备忘单和用户帮助文档中。
