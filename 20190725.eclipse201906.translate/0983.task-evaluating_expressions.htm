评估表达式

当VM挂起线程（由于命中断点或单步执行代码）时，您可以在堆栈帧的上下文中计算表达式。

  1. 选择要在其中执行评估的堆栈帧。评估上下文将是视图中当前选定的变量。如果未选择任何变量，则所选堆栈帧将成为上下文。
  2. 可以在以下区域输入和评估表达式：
    • 调试Shell视图
    • 详细信息窗格 （在“ 表达式视图和变量”视图中 ）
    • Java编辑器在显示源时它不是只读的
  3. 选择要评估的表达式，然后从上下文弹出菜单中选择“ 显示” ，“ 检查”或“ 执行 ”。显示或检查评估的结果显示在弹出窗口中。请注意， Execute不显示结果 - 表达式刚刚执行。
  4. 单击弹出窗口外部或按Esc键可以关闭结果弹出窗口。通过按弹出窗口底部显示的键序列，可以将结果持久保存到Debug Shell视图（如果选择了Display ）或Expressions视图（如果选择了Inspect ）。例如，要将Inspect评估的结果移动到“表达式”视图，请按Ctrl-Shift-I 。请注意，在“ 显示 ”视图中使用“ 显示”操作时，结果将写入“调试Shell”视图而不是弹出窗口

注意：无法在已手动挂起的线程中执行评估。
