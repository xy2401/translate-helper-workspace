版本化项目

现在项目已经完成，现在是时候对它进行版本化了。在提交文件资源时，存储库会自动为其分配版本号，或者更准确的“版本号”。在提交文件时，提供的唯一信息是提交注释。

要版本化项目，请执行以下操作：

  1. 选择项目JanesTeamProject 。
  2. 在其中一个导航视图中，选择Team> Tag as Version ...
  3. “标记资源”对话框打开后，输入版本名称A1 ，然后单击“ 确定” 。
  4. 选择CVS存储库视图。
  5. 展开版本和JanesTeamProject 。请注意，在JanesTeamProject下，现在有一个版本号为A1的项目版本。仔细查看folder1的内容，请注意各个文件的版本号与Workbench中的版本号相匹配。
