导出并安装到正在运行的主机中

PDE提供了几种工具，用于在工作区中启动插件和功能。Eclipse Application Launcher可用于创建Eclipse的第二个运行实例，该实例运行工作区中的代码。这通常被称为自托管 。自托管是一个功能强大的调试和测试工具。但是，有时需要在主机（当前运行的Eclipse实例）中测试代码。要测试主机中的代码，需要导出并安装到正在运行的Eclipse实例中。

要导出插件和功能并将其安装到主机Eclipse中，请执行以下操作：

  1. 在工作区中开发插件和功能
  2. 打开导出向导 文件>导出...>插件开发>可部署的插件和片段或 文件>导出...>插件开发>可部署的功能
  3. 选择要导出和安装的插件或功能
  4. 选择目标选项卡上的最后一个选项安装到主机。存储库 。然后选择一个目录来创建存储库
  5. 命中完成 。将运行导出操作，然后执行安装操作。
  6. 如果操作成功完成，系统将提示您重新启动。选择立即重启
  7. 重启后，您的插件将安装并运行。您可以使用“关于”对话框上的“ 安装详细信息”按钮查看已安装的内容（可通过转至“帮助”>“关于Eclipse SDK”获取）
