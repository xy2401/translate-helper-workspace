内容标签

“ 编辑目标向导”中的“ 内容”选项卡用于设置要包含在目标平台中的插件子集。该选项卡显示目标定义位置中可用的所有插件的列表。仅包含此选项卡上选中的插件。有多种方法可以组织和过滤列表。目标内容也可以在目标编辑器的内容页面上编辑 。

选择/取消选择将选中或取消选中所选插件。

选择全部/取消选择全部将选中或取消选中目标定义中的所有插件。

Add Required采用当前检查的插件集并确定它们所需的所有插件。这还将包括在此目标上设置的任何隐式依赖项 。将在列表中检查所需的插件。

使用“ 显示”选项以及列表顶部的过滤器文本框来过滤列表中显示的内容。显示插件将在目标中显示任何非源插件，而显示源包将显示任何源。即使从列表中过滤了插件，它也可能包含在目标中。您可以在列表下方看到包含的插件数量。

Group By选项用于将列表组织到不同的组中。选项是：

  1. 无 - 没有按名称排序的分组，插件
  2. 文件路径 - 由插件存储在本地文件系统中的目录路径分组
  3. 位置 - 按位置选项卡上设置的位置分组

管理具有功能的内容

在右侧，您可以选择两种不同的模式来管理目标的内容。默认情况下，使用插件列表管理目标。选择功能会将目标内容更改为功能列表。

您可以按照与插件管理相同的方式选择功能。“添加必需”将选择当前所选功能在其包含或要求条目中具有的任何功能。如果目标中存在不属于某个功能的插件 ，则它们将列在“ 其他插件”项下，并且可以单独选择。
