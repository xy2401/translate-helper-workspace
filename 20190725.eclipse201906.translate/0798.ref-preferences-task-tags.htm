Java任务标签首选项

可以在上配置任务标签 Java>编译器>任务标签首选项页面。当标记列表不为空时，只要遇到Java源代码中任何注释中的相应标记之一，编译器就会发出任务标记。生成的任务消息将包括标记，范围直到下一行分隔符，注释结束或非空标记，并将被修剪。如果同一行代码带有多个标记，则会单独报告它们。

有关如何在源代码中启用任务标记的信息，请参阅“ 编译器”首选项页面 。

行动

描述

新...

添加新的任务标记。在结果对话框中，指定新任务标记的名称和优先级。

去掉

删除所选的任务标记。

编辑...

允许您编辑选定的任务标记。在结果对话框中，编辑任务标记的名称和/或优先级。

默认

将当前选定的任务标记设置为默认任务标记。默认任务标记是在代码模板首选项页面上指定的代码模板中使用的标记。默认任务标记以粗体显示。

可以使用选项区分大小写的任务标记名称在首选项页面的底部指定任务标记的区分大小写 。
