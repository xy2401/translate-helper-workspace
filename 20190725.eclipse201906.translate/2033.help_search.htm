搜索帮助

帮助系统包括一个搜索引擎，可以对文档运行简单或复杂的查询，以帮助您查找所需的信息。

要搜索帮助：

  1. 从主菜单中，选择“ 帮助”>“搜索”
  2. 输入您要搜索的单词或短语
  3. 单击GO或按Enter键。结果列表将显示如下
  4. 要在结果列表中查看主题的内容，请单击它

或者，您可以使用窗口顶部的“ 搜索”字段从“帮助”窗口进行搜索 。

在帮助视图中优化搜索结果

如果搜索结果太多，则您要查找的信息可能不会显示在前10或15个结果中。然后，您可以优化搜索以减少结果数量。

要优化搜索：

  1. 单击“ 搜索范围”链接。扩大搜索范围部分
  2. 单击“ 高级设置”链接。搜索范围首选项对话框将打开
  3. 从列表中选择“ 本地帮助 ”
  4. 选择仅搜索以下主题按钮以缩小搜索范围
  5. 在工作集内容树中，选择要将搜索范围缩小到的主题
  6. 单击“ 确定”以激活更改并返回“帮助”视图中的搜索页面
  7. 再次单击GO 。将出现新的结果列表

更改结果的外观

工具栏上的两个按钮可用于更改结果的显示方式。该 显示结果类别按钮，按下时，将导致结果按书分组。该 显示结果描述按钮，按下时，将显示说明。

突出显示搜索字词

默认情况下，选择搜索结果时，将突出显示用于查找文档的搜索词。通过使用 突出显示“搜索条件”工具栏按钮，您可以打开和关闭此功能。此按钮在帮助窗口和帮助视图中都可用，每个按钮都会记住按钮的状态，以显示后续搜索结果。

查询语法

按照以下搜索表达式规则搜索本地帮助内容：

  • 除非另有说明，否则所有搜索词之间都有隐含的AND。换句话说，将返回包含所有搜索词的主题。例如：
    Java project

    返回包含单词Java和单词project的主题，但不返回仅包含其中一个单词的主题。

  • 在可选术语之前使用OR。例如：
    applet OR application

    返回包含单词applet或单词application （或两者）的主题。

  • 在您要从搜索结果中排除的字词之前使用NOT。例如：
    servlet NOT ejb

    返回包含单词servlet的主题，不包含单词ejb 。注意：不仅用作二元运算符（即“NOT servlet”不是有效表达式）。

  • 使用 ？用于单字符通配符，*用于多字符通配符。例如：
    par?

    返回包含part或park但不参与的主题 。另一方面：

    par*

    返回包含part ， park ， participation ， pardon等主题。注意：搜索引擎不接受在第一个字符位置使用通配符的条款。

  • 在您希望视为短语的术语周围使用双引号。例如：
    "creating projects"

    返回包含创建项目的整个短语的主题。不返回创建和项目不连续的主题。

  • 标点符号用作术语分隔符。例如：
    plugin.xml

    返回包含插件和xml的主题的匹配，这可能比您想要的更广泛。如果您只想查找包含plugin.xml的主题，请使用双引号，如下所示：

    "plugin.xml"
  • 搜索引擎忽略了字符大小写。例如：
    Workbench

    返回包含“workbench”，“Workbench”，“WorkBench”和“WORKBENCH”的主题。

  • 以下停用词是常见的英语单词，如果它们出现在搜索表达式中，将被忽略（未搜索）：a，and，as，as，at，be，but，by，in，into，is，it，no ，不，，或，等等，那些，它们，它们，那些，它们，它们，它们，将来，将来，将会与之相伴。
  • 搜索引擎进行“模糊”搜索和词干。如果输入create ，它将返回包含创建 ， 创建 ， 创建者等主题的匹配。为了防止搜索引擎阻塞术语，请将它们用双引号括起来。

扩展搜索范围

如果您无法在本地帮助中找到信息，则可以将搜索范围扩展到远程信息中心或搜索引擎。

要启用搜索引擎：

  1. 单击“ 搜索范围”链接。扩大搜索范围部分。显示搜索引擎列表。
  2. 选择包含您要查找的信息的那些。

除了提供的搜索引擎，您还可以定义其他搜索引擎。

要定义新的搜索引擎：

  1. 单击“ 搜索范围”链接。扩大搜索范围部分。
  2. 单击“ 高级设置”链接。搜索范围首选项对话框打开
  3. 单击新建 。
  4. 选择搜索引擎类型。
  5. 单击确定 。
  6. 提供名称和说明
  7. 选择下面的引擎特定设置和范围。对于使用URL访问的远程搜索引擎，请填写完整的URL以查询引擎。在搜索表达式的位置使用{expression} 。

定义多个搜索范围

默认情况下，更改搜索范围会修改名为“default”的搜索范围。您可以定义多个搜索范围。它们将被保存，允许快速将搜索范围更改为其中一个。

要定义新的搜索范围：

  1. 单击“ 搜索范围”链接旁边的当前搜索范围名称。。出现“搜索范围集”对话框
  2. 单击新建 。
  3. 输入名称，然后确认。
  4. 选择新创建的搜索范围。
  5. 单击确定 。新的搜索范围变为最新。

对搜索范围的更改会影响当前搜索范围。

搜索索引生成

第一次搜索联机帮助时，帮助系统可能会启动索引生成过程。此过程构建要使用的搜索引擎的索引。可能需要几分钟，具体取决于文档量以及是否安装了预构建索引。搜索结果将在索引过程完成后提供。

每次添加或修改文档集时（例如，安装新功能或更新现有功能时），都会更新索引以反映新信息集。
