创建一个新的JPA项目

使用此过程可以创建新的JPA项目。

  1. 从Navigator或Project Explorer中，选择File> New> Project 。将出现“选择向导”对话框。


    小费：

    您还可以选择JPA透视图，然后选择“ 文件”>“新建”>“JPA项目” 。

    选择“创建JPA项目”向导


  2. 选择JPA Project ，然后单击Next 。将出现“ 新JPA项目”页面 。

    JPA项目页面


  3. 完成“ 新建JPA项目”页面上的字段以指定项目名称和位置，目标运行时和预定义配置。


    注意：

    Java SE开发不需要Target运行时 。

  4. 单击下一步 。将出现Java源页面。

    Java源页面


  5. 单击“ 添加文件夹”以将现有Java源文件添加到项目中。

  6. 单击下一步 。出现JPA Facet页面 。

    JPA Facet


  7. 填写JPA Facet页面上的字段以指定您的供应商特定平台，JPA实施库和数据库连接。

    单击“ 管理库”以创建或更新JPA用户库。单击“ 下载库”以获取其他JPA实现库。

    如果Dali派生的模式不正确，请选择“ 覆盖连接的默认架构” 。使用此选项，您可以选择默认值和验证的开发时间架构。

    如果清除“ 创建映射文件（orm.xml）”选项（默认情况下已选中），则可以稍后使用“ 创建ORM映射文件”向导将映射文件添加到项目中。


    注意：

    如果服务器运行时未提供JPA实现，则必须显式选择JPA实现库。

    要确保应用程序的可移植性，必须明确列出持久性单元中包含的托管持久性类。如果服务器支持EJB 3.0，则将自动发现持久化类。

    根据您的JPA实现（例如，Generic或EclipseLink），在创建JPA项目时可能会有不同的选项。

  8. 单击完成 。Dali创建了新的JPA项目。

    您现在应该打开JPA Development透视图 。
