启动时更改活动透视图

您可以控制在启动程序和暂停程序时哪个透视图变为活动状态。对于每种启动配置类型，可针对其支持的每种启动模式配置该设置。

要在程序启动时激活特定透视图，请执行以下操作：

  1. 打开 运行/调试>透视首选项页面。
  2. 在启动首选项时，选择“ 打开关联透视图”的“ 始终”选项。这将导致与程序关联的透视图在启动时变为活动状态。

要在程序挂起时激活特定透视图，请执行以下操作：

  1. 打开 运行/调试>透视首选项页面。
  2. 当应用程序挂起首选项时，选择“ 打开关联透视图”的“ 始终”选项。这将导致与程序关联的透视图在程序挂起时变为活动状态。

要将特定透视图与程序相关联，请执行以下操作：

  1. 打开 运行/调试>透视首选项页面。
  2. 选择要与透视关联的应用程序类型（例如， Java Application ）。
  3. 对于每种启动模式，使用组合框选择所需的透视图。这将导致您选择根据首选项设置激活的视角（即启动程序时和/或暂停时）。

如果指定的透视图在需要激活时未打开，则会创建该透视图。
