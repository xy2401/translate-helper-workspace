导入和导出首选项

可以将工作台导入和导出首选项文件，以便轻松共享个人或组首选项。

导入向导可用于将首选项从文件系统导入Workbench。

要导入首选项文件：

  1. 选择 文件>导入 。
  2. 在Import向导中，选择General> Preferences ，然后单击Next 。
  3. 单击“ 浏览...”并在文件系统上找到“首选项”文件。
  4. 单击全部导入以接受文件中定义的所有首选项。
  5. 如果您只想导入少数首选项，请从列表中选择它们。
  6. 单击完成 。

 

导出向导可用于将首选项从Workbench导出到文件系统。

要导出首选项文件：

  1. 选择 文件>导出 。
  2. 在Export向导中，选择General> Preferences ，然后单击Next 。
  3. 单击全部导出以将所有首选项添加到文件。
  4. 如果您只想导出少数首选项，请从列表中选择它们。
  5. 单击“ 浏览...”并在文件系统上找到首选项文件。
  6. 单击完成

注意：如果未对原始首选项设置进行任何更改，则首选项文件将为空。
