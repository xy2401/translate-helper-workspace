Java剪贴簿

剪贴簿允许在调试器的控制下运行，检查和显示Java表达式。

注意：剪贴簿页面上提供了内容辅助（例如代码帮助）。

Java剪贴簿页面按钮

命令

名称

描述

运行代码段

运行表达式会计算表达式但不显示结果。

显示

显示在剪贴簿编辑器中显示将表达式计算为字符串的结果。

检查

检查显示在“表达式”视图中计算表达式的结果。

终止

此命令终止用于计算表达式的Java VM。

设置导入声明

此命令将导入声明设置为用于评估代码的上下文
