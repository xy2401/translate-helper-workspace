C / C ++项目属性，管理配置对话框

管理为项目定义的配置。选择一个按钮以添加，重命名，删除配置或将其设置为活动状态。

主窗格
显示给定项目的已定义配置列表。
设置为活动
设置所选配置激活。
新
显示可用于创建新配置的新配置对话框。
去掉
从列表中删除所选配置，提示用户确认操作。
改名
显示可用于重命名配置名称和描述的重命名配置对话框。
好
关闭管理配置对话框
