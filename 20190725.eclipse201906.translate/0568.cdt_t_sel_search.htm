选择搜索C / C ++元素

建议您对成功编译的程序执行搜索，以确保搜索结果的准确性。请务必熟悉用于完成有效搜索的正确搜索语法。确保正确定义包含路径和符号也很重要。有关更多信息，请参阅为C / C ++索引器设置包含路径和宏 。

有关以下内容的更多信息，请参阅C / C ++搜索 ：

  • 你可以搜索什么
  • 如何限制您的搜索
  • 如何在搜索中使用通配符
  • 语法示例

要在项目中搜索元素：

  1. 突出显示要搜索的元素。
  2. 右键单击并从上下文菜单中选择“ 搜索” 。
  3. 选择所有声明或所有引用 。
  4. 选择工作区或工作集....
  5. 搜索结果将显示在“搜索视图”中

有关更多信息，请参阅：

  • Workbench用户指南>概念>视图>搜索视图
  • Workbench用户指南>任务>导航和查找资源
