项目菜单操作

名称 功能 键盘快捷键
打开项目 打开当前选定的已关闭项目，使其内容可用于工作区  
关闭项目 关闭当前选定的项目  
全部构建 在工作区中构建所有打开的项目 按Ctrl + B
构建配置 通过以下方式配置构建：  
设置活动（子菜单）
  
管理...
  
按工作集构建（子菜单）
  
通过工作集设置活动（子菜单）
  
管理工作集......
  
建设项目 构建当前选定的项目  
建立工作集 构建当前工作集，而不是整个项目  
清洁 调用makefile中定义的make clean  
自动构建 切换是否在每个文件保存时重建项目（对于非常大的项目，应关闭此功能。）  
制作目标（子菜单） 创建，编辑或运行Make Target  
属性 显示“ 属性”对话框。在该对话框中，您可以在Info ， 外部工具构建器 ， C / C ++构建 （仅管理） 文件类型 ， 索引器选项 ， C / C ++生成项目 （仅标准）， C / C ++项目路径 （仅标准）中显示资源的属性， 包括路径和符号 （仅限标准）和项目参考 。
