JDT UI

JDT UI（ org.eclipse.jdt.ui ）是实现操作Java元素的Java特定用户界面类的插件。JDT UI中的包实现了工作台的Java特定扩展。JDT UI包包括：

  • org.eclipse.jdt.ui - 提供用于在用户界面中呈现Java元素的支持类。此包公开了用于从工作台注册表检索Java用户界面部件的常量，以及从Java首选项中检索首选项设置的常量。编程接口ITypeHierarchyViewPart和IPackagesViewPart用于与Java视图交互。
  • org.eclipse.jdt.ui.actions - 提供操作和操作组，以使用JDT特定功能填充工具栏，全局菜单栏和上下文菜单。您应该使用操作组来填充菜单和工具栏，而不是直接添加操作。这可以防止您丢失新添加的操作或使用过时的菜单结构。
  • org.eclipse.jdt.ui.cleanup - 提供通过org.eclipse.jdt.ui.cleanUps扩展点提供代码清理所需的接口和类。
  • org.eclipse.jdt.ui.dialogs - 提供访问和扩展JDT对话框的类和接口，特别是“打开类型”对话框。
  • org.eclipse.jdt.ui.jarpackager - 提供生成JAR文件的类和接口。
  • org.eclipse.jdt.ui.refactoring - 为运行重命名重构提供支持
  • org.eclipse.jdt.ui.search - 提供类和接口，以便为Java搜索贡献参与者
  • org.eclipse.jdt.ui.text - 提供用于呈现Java文本的支持类。
  • org.eclipse.jdt.ui.text.folding - 提供接口以实现Java编辑器的代码折叠策略。
  • org.eclipse.jdt.ui.text.java - 提供实现代码完成处理器的接口。
  • org.eclipse.jdt.ui.text.java.correction - 提供更正提议的默认实现，可用于提供快速修复和快速帮助。
  • org.eclipse.jdt.ui.text.java.hover - 提供在Java编辑器中呈现文本悬停的实现。
  • org.eclipse.jdt.ui.wizards - 提供用于创建和配置Java元素的向导页面。
