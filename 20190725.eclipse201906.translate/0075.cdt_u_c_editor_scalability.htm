可伸缩性首选项

使用 可伸缩性首选项面板，用于配置处理大型源文件的选项。

可伸缩性首选项选项
选项 描述
当可伸缩性模式打开时提醒我 如果启用，则在使用超出指定行限制的C / C ++编辑器打开大文件时会弹出信息对话框。
当文件中的行数大于时，启用可伸缩性模式选项 如果文件中的行数超过指定的数量，则C / C ++编辑器将启用可伸缩性模式，并且下面的所有可伸缩性模式选项都将生效。
启用所有可伸缩性模式选项 如果启用，则也会启用以下所有选项。
禁用编辑器实时解析 如果启用，将禁用键入时解析。这也会禁用其他功能，如大纲视图，语义突出显示，折叠等。
禁用语义突出显示 如果启用，则不会对C / C ++标识符进行着色。
禁用语法着色 如果启用，则不会对关键字，注释和文字进行着色。
禁用基于解析的内容辅助提议 如果启用，则无法提供需要解析文件的内容辅助提议。
禁用内容辅助自动激活 如果启用，则不会在触发序列上自动激活内容辅助，例如“。”，“::”或“ - >”。
