浏览帮助主题

您可以使用“ 帮助”窗口或“ 帮助”视图浏览帮助主题。你使用哪一个只是一个偏好的问题;视图与工作台完美集成，适用于快速帮助查找。另一方面，“帮助”窗口具有更多的空间，可能更适合长时间阅读。

使用“帮助”窗口

“帮助”窗口是一个独立于工作台的窗口，专门用于浏览，搜索和打印帮助。要打开窗口，请从菜单中选择“ 帮助”>“帮助内容 ”。这将打开帮助窗口，其中显示内容视图，其中显示了目录。

要导航“帮助”窗口：

  1. 通过单击以展开子主题，在目录中找到要阅读的主题。
  2. 大多数主题提供了底部相关主题的链接列表。请点击以下链接了解更多信息。
  3. 使用 回去吧 前进按钮来回导航。这些行为与Web浏览器中的行为相同。
  4. 使用 主页按钮返回帮助主页，如上图所示。

最大化帮助框架

“帮助”窗口的两个主框架可以最大化以占用整个窗口。要最大化框架，请单击 最大化工具栏中的按钮，或双击视图的标题标题。要将视图恢复为原始大小，请单击 还原按钮或再次双击标题标题。

打印帮助

要从“帮助”窗口打印主题：

  1. 在导航框架中选择主题。
  2. 点击帮助工具栏中的“ 打印”按钮。
  3. 选择所需的打印机设置，然后单击“ 打印” 。

使用“帮助”视图

“ 帮助”视图提供与“帮助”窗口相同的功能，但与工作台位于同一窗口中，而不是单独的窗口。

在目录中显示

在阅读主题时，您可以通过单击以查找主题在目录中的位置 在工具栏中的目录按钮中显示。这将显示突出显示主题的目录。
