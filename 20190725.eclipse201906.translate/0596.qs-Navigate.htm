导航到Tcl元素的声明

在本节中，您将学习如何轻松找到引用的Tcl变量的“声明”（通常是变量的set命令）。

  1. 创建一个新项目，如本教程的第一部分所示。

  2. 在该项目中创建一个新的Tcl文件。然后复制并粘贴以下代码：
    设置变量1
    proc modify {arg} {
    设置b 0
    放“之前：$ arg”
    设置arg $ b
    放“after：$ arg”
    }
    修改$ var

  3. 将游标设置为最后一行的$var变量引用。

  4. 选择导航>打开声明菜单项。

    现在在编辑器中选择变量的声明。
    请注意，您也可以使用F3键或左键单击保持Ctrl键而不是使用菜单项的变量引用来实现相同的效果。

您可以以相同的方式导航到Tcl过程声明。
