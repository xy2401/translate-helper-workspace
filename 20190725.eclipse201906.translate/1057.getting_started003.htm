创建一个新的JPA项目

本快速入门介绍了如何创建新的JPA项目。

  1. 选择文件>新建>项目 。将出现“选择向导”对话框。


    小费：

    您还可以选择JPA透视图，然后选择“ 文件”>“新建”>“JPA项目” 。

  2. 选择JPA Project ，然后单击Next 。将出现“ 新JPA项目”页面 。

  3. 输入项目名称 （例如QuickStart ）。

  4. 如果需要，请选择目标运行时 （例如Apache Tomcat ）和配置，例如Apache Tomcat的默认配置 ，然后单击下一步 。将出现Java源页面。


    注意：

    Java SE开发不需要目标运行时。

  5. 如果您有现有的Java源文件，请将它们添加到类路径中，然后单击“ 下一步” 。出现JPA Facet页面 。

  6. 在JPA Facet对话框中，选择特定于供应商的JPA平台（或选择Generic 2.0 ），JPA实现库（例如EclipseLink），数据库连接（或创建新连接），定义Dali应如何管理持久类，然后单击完成 。


    小费：

    如果您需要的模式不是Dali从连接信息中获取的模式，请选择“ 覆盖连接的默认模式”，在某些情况下可能不正确。使用此选项，您可以选择默认值和验证的开发时间架构。

Eclipse将项目添加到工作台并打开JPA透视图。
