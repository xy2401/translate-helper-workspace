新文件创建向导

除了清单文件之外，在插件开发的各个方面和不同阶段还需要几种其他类型的文件：

为了帮助您创建这些文件，PDE提供了专门的新文件创建向导 文件>新建>其他...>从顶级菜单插件开发 。

插件开发文件

  • 类别定义 - 指定在导出期间应用于要素的类别
  • 扩展点模式 - 指定扩展点的语法
  • 产品配置 - 管理定义，标记和构建Eclipse产品所需的所有数据
  • 目标定义 - 定义目标平台的所有方面

用户帮助文件

  • 备忘单 - 为Eclipse产品创建备忘单教程
  • 上下文帮助 - 创建上下文帮助xml文件以将UI组件关联到帮助文档
  • 帮助目录 - 创建目录以获取帮助文档
