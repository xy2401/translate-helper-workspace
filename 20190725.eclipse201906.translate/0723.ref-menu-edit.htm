编辑动作

编辑Java编辑器时显示的菜单命令：

名称

功能

 键盘快捷键

解开

还原编辑器中的最后一个更改

 Ctrl + Z.

重做

恢复未完成的更改

 Ctrl + Y.

切

将当前选定的文本或元素复制到剪贴板并删除该元素。在元素上，在粘贴剪贴板之前不执行删除。

Ctrl + X.

复制

将当前选定的文本或元素复制到剪贴板

 Ctrl + C.

复制合格的名称

将当前所选元素的完全限定名称复制到剪贴板

  

糊

将当前内容作为文本粘贴到编辑器，或作为当前所选元素的兄弟或子元素。

Ctrl + V.
删除 删除当前文本或元素选择。 删除
全选 选择所有编辑器内容。 Ctrl + A.
展开选择到
  • Enclosing Element：在代码中选择封闭的表达式，块或方法。此操作了解Java语法。当代码有语法错误时，它可能无法正常工作。（向上箭头）
  • 下一个元素：选择当前和下一个元素。（向右箭头）
  • 上一个元素：选择当前和以前的元素。（向左箭头）
  • 恢复上一个选择：在调用“ 扩展选择到”后恢复先前的选择。（箭头向下）
 Alt + Shift +箭头键
查找/替换 打开“查找/替换”对话框。仅限编辑。 Ctrl + F.
找下一个 查找当前所选文本的下一个匹配项。仅限编辑。 Ctrl + K.
找到上一个 查找当前所选文本的上一个匹配项。仅限编辑。 Ctrl + Shift + K.
增量查找下一个 启动增量查找模式。调用后，按照状态栏中的说明输入搜索文本。仅限编辑。 Ctrl + J
增量查找上一个 启动增量查找模式。调用后，按照状态栏中的说明输入搜索文本。仅限编辑。 Ctrl + Shift + J.
添加书签 将书签添加到当前文本选择或所选元素。  
添加任务 将用户定义的任务添加到当前文本选择或所选元素。  
智能插入模式 切换插入模式。禁用智能插入模式时，将禁用键入自动缩进，关闭括号等辅助工具。 Ctrl + Shift + Insert
显示工具提示说明 显示将出现在当前光标位置的悬停值。显示的对话框是可滚动的，不会缩短说明。 F2
内容辅助 在当前光标位置打开内容辅助对话框。默认情况下，内容辅助支持五种不同类别的提案。
  1. 默认 - 联合所有提案
  2. 输入Proposals （例如java.lang。字符串，列表）
  3. 其他Java提案（例如方法或字段名称）
  4. 模板建议（例如'迭代数组'模板）
  5. 单词提案（提出已经输入的单词）

请参阅Java编辑器首选项页面，以配置代码帮助的行为和默认提议类别的内容。

有关可用模板，请参阅模板首选项页面

 Ctrl +空格
单词完成 根据在任何打开的编辑器中找到的所有单词，为当前字符串提出单词补全。适用于所有基于文本的编辑器。 Alt + /
快速解决 如果光标位于问题指示附近，则打开一个包含可能解决方案的对话框。 Ctrl + 1
设置编码 切换当前显示的文本内容的编码。
