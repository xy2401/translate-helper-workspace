创建JPA实体

使用此过程使用Create JPA Entity向导创建JPA实体 ：

  1. 从Navigator或Project Explorer中，选择JPA项目，然后选择File> New> Other 。将出现“选择向导”对话框。

    选择“创建JPA实体向导”



    小费：

    您还可以选择JPA透视图，然后选择“ 文件”>“新建”>“JPA实体” 。

  2. 选择JPA> JPA Entity ，然后单击Next 。将显示“ 实体类”页面 。

    实体类页面


    完成“ 实体类”页面上的字段，如下所示：

    • 在“ 项目”字段中选择JPA项目。

    • 在“ 源文件夹”字段中，选择或输入JPA项目的src文件夹的位置。

    • 在Java Package字段中选择或输入此实体的类包的名称。

    • 在“ 类名”字段中输入Java类的名称 。

    • 如果需要，请输入或选择超类。

    • 如果需要，请按如下所示完成“继承”部分（这些属性是可选的）：

      • 接受Entity选项（默认值）以使用@Entity选项创建Java类。

      • 或者，选择Mapped superclass （如果您定义了超类）。

      • 选择继承 ，然后选择一个JSR 220继承映射策略（SINGLE_TABLE，TABLE_PER_CLASS，JOINED）。

      • 选择“ 添加到XML中的实体映射”以在orm.xml创建XML映射，而不是注释。

  3. 单击“ 下一步”以进入“ 实体属性”页面 ，您可以在其中定义实体的持久字段。

    实体属性页面


    或者，单击“ 完成”以完成实体。

  4. 完成页面如下：

    1. 如果需要，请输入实体的新名称。这样做会导致为name表示法添加@Entity属性（ @Entity(name="EntityName") ）。

    2. 接受使用默认值 （默认设置）以使用映射表名称的默认值。输入不同的名称会导致添加@Table表示法，并将其name属性定义为新表（ @Table(name="TableName") ）。


      注意：

      如果在“ 实体类”页面上选择了“ 映射的超类” ，则“实体名称”相关选项不可用

    3. 单击“ 添加”，将持久性字段添加到实体。将出现“实体字段”对话框。

      实体字段对话框


    4. 从“ 类型”列表中选择持久性类型。您可以使用“ 浏览”功能检索其他类型。

    5. 输入字段名称，然后单击“ 确定” 。对每个字段重复此过程。

    6. 如果需要，请选择“ 密钥”以将字段指定为主键。

    7. 选择基于字段的访问类型（默认）或基于属性的访问类型。

  5. 单击完成 。Eclipse将实体添加到项目中。
