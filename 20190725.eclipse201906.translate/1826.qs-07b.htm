使用“文件”菜单

您可以使用Workbench菜单栏上的File> New菜单创建新资源。首先创建一个简单的项目，如下所示：

  1. 从菜单栏中选择File> New> Project ...

  2. 在New Project向导中，选择General > Project，然后单击Next 。

  3. 在“ 项目名称”字段中，键入您的名称作为新项目的名称，例如“JaneQuser”。

  4. 选中此框以使用新项目的默认位置。完成后单击“ 完成” 。

     

如果你偷看导航视图，你会发现它现在包含我们刚刚创建的简单项目。

使用相同的步骤创建名为JaneQUser2的第二个项目，但不是单击“ 完成” ，而是单击“ 下一步” 。此时，您可以指定JaneQuser2依赖的项目的其他项目。由于我们要创建两个独立项目，因此我们不会选择Project References表中的任何项目。单击“ 完成”以创建第二个简单项目。
