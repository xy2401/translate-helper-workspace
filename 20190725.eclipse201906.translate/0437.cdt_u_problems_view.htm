问题视图（C / C ++）

使用 问题视图显示构建期间遇到的任何错误。

错误从C ++编译器传递出来。Problems视图列出了错误，文件名和文件夹。如果选择错误，则相关文件将在C / C ++编辑器视图中打开，并且光标和文本突出显示将显示遇到错误的行。

问题视图工具栏选项

下表列出了“问题”视图中显示的工具栏选项。

图标命令描述
查看菜单 单击以打开特定于视图的项目菜单，包括：
  • 显示 - 指定显示哪些错误或警告
  • 分组依据 - 指定错误分组
  • 排序依据 - 指定错误的排序方式
  • 新问题视图 - 创建新的问题视图
  • 配置内容... - 打开“配置内容”对话框
  • 列... - 打开“配置列”对话框
  • 首选项 - 打开“首选项”对话框
