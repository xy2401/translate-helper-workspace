使用记忆

您可以检查和更改进程内存。

Memory视图支持与C和C ++语言相同的寻址。您可以使用以下表达式来寻址内存：

  • 0x0847d3c
  • (&y)+1024
  • *ptr

您可以将输出配置为显示十六进制，ascii和/或有符号/无符号十进制。您还可以设置渲染列的大小。您可以单独配置每个内存监视器渲染。

您可以将“内存”视图自定义为显示的颜色和字体。

监控进程内存：

  1. 在“调试”视图中，选择一个调试会话。选择线程或堆栈帧会自动选择关联的会话。
  2. 在Memory Monitors窗格的上下文菜单中选择Add Memory Monitor 。出现Memory Monitor对话框。
  3. 键入指定要监视的内存部分的地址或表达式，然后按“确定” 。监视器显示在监视器列表中，“内存渲染”窗格显示从指定地址开始的内存位置的内容。

要在不同的渲染中查看内存：

  1. 在Memory Renderings窗格的上下文菜单中选择Add Rendering 。将出现“ 添加内存渲染”对话框。
  2. 从列表中选择渲染，然后按“确定” 。

要更改进程内存：

警告：更改进程内存可能导致程序崩溃。

  1. 在“调试”视图中，选择一个调试会话。选择线程或堆栈帧会自动选择关联的会话。
  2. 在“内存监视器”窗格中，添加新监视器或选择现有监视器。
  3. 在“内存渲染”窗格中添加新渲染或选择现有渲染。
  4. 为内存键入新值。
