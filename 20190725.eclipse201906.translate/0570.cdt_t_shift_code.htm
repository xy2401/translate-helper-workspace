向右或向左移动代码行

您可以在C / C ++编辑器中向左或向右移动代码行。您可以在C / C ++编辑器首选项窗口中更改选项卡宽度。有关更多信息，请参阅自定义C / C ++编辑器 。

要将代码行向右或向左移动：

  1. 在C / C ++编辑器中，选择要移动的行的全长。
  2. 执行以下操作之一：
    • 要将文本向右移动，请按Tab键 。
    • 要将文本向右移动，请单击“ 编辑”>“右移” 。
    • 要将文本移到左侧，请按Shift + Tab 。
    • 要将文本移动到左侧，请单击“ 编辑”>“向左移动” 。
