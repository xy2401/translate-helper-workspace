项目参考

项目引用是表达CDT项目之间依赖关系的强大方式，允许您编写使用CDT的managedbuild构建的模块化软件。

两个主页控制项目参考：

  • 项目参考页面下
    右键单击>属性> C / C ++常规>路径和符号
  • 导出设置页面下
    右键单击>属性> C / C ++常规>导出设置
    如果缺少此页面，请在“窗口”>“首选项”>“C / C ++”>“属性页设置”下启用它

导出设置页面为您提供了对给定项目配置导出的路径，预处理程序符号和库文件的精细控制。

当工作空间中的另一个项目的配置引用具有导出设置的配置时，这些设置将自动传播到从属项目配置中。CDT还确保将引用的项目构建为构建依赖项目的先决条件。

此功能可防止您需要在工作区中的所有相关项目之间手动维护和更新构建设置。只需配置导出的设置，设置参考，设置就会传播并保持同步。
