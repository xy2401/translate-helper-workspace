设置API基准

将插件转换为使用API工具后，需要设置基线以便为插件执行兼容性分析。

要创建API基线，请执行以下操作：

  1. 打开 插件开发> API Baselines首选项页面。
  2. 选择Add Baseline ...按钮以打开API基准向导 。
    • 为您的基线提供有意义的名称
    • 单击Browse ...并选择要选择的Eclipse SDK的根文件夹作为基准
    • 在文件夹对话框中单击确定
    • 初始化基线后，单击“ 完成”
  3. 单击“ 确定” ，即可完成设置。如果您的工作区包含一些现有项目，系统将提示您进行重建。
