FormText控件

使用在表单中混合在一起的图像，超链接和文本片段可以实现高度抛光的结果。但是，当需要将这些元素的混合作为一个整体文本的一部分时，很难做到。为了解决这个问题，UI Forms提供了一个基本的文本控件，可以执行以下操作：

  • 渲染纯文本包装文本

  • 呈现纯文本，但将任何以http：//开头的段转换为动态超链接

  • 使用XML标记渲染文本

在所有模式中， FormText控件都能够呈现字符串或输入流。

渲染普通文本（标签模式）

	FormText rtext = toolkit.createFormText(form.getBody(), true);
	String data = "Here is some plain text for the text to render.";
	rtext.setText(data, false, false);

第二个参数设置为false意味着我们将按原样处理输入文本，第三个参数如果找到则不会尝试扩展URL。

自动将URL转换为超链接

仍然可以正常处理文本，但会自动将带有http：//协议的段转换为超链接：

	FormText rtext = toolkit.createFormText(form.getBody(), true);
	String data = "Here is some plain text for the text to render; "+
                       "this text is at http://www.eclipse.org web site.";
	rtext.setText(data, false, true);

与Hyperlink控件类似， FormText接受实现HyperlinkListener侦听器。将通知这些侦听器有关控件内的超链接段的事件。
