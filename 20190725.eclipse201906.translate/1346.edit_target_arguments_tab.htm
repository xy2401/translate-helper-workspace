参数选项卡

“ 编辑目标向导”中的“ 参数”选项卡用于设置新PDE 启动配置的默认程序和vm参数。也可以在目标编辑器的环境页面上编辑参数 。

按变量将打开变量选择向导 。您可以浏览所有已知变量并获取每个变量的描述。在VM Arguments组下，您也可以按Import 。这将打开Import Arguments Dialog 。您可以使用它来导入已添加到目标定义的任何安装位置的配置数据中使用的vm参数。
