定义调试设置

选择调试应用程序时要使用的调试器。

要选择调试器：

  1. 在“C / C ++项目”视图中，选择一个项目。
  2. 单击运行>运行或运行>调试 。
  3. 在“ 配置”框中，展开“ C / C ++ Local” 。
  4. 选择运行或调试配置。
  5. 单击Debugger选项卡..
  6. 从Debugger列表中选择一个调试器 。
  7. 要让程序运行直到您手动中断它，或直到它遇到断点，请清除启动时在main（）处停止复选框。
  8. 在“ 调试器选项”框中指定调试选项。
  9. 单击“运行”或根据需要执行以下操作：
    • 要指定应用程序使用的执行参数以及运行配置的工作目录，请参阅指定执行参数 。
    • 要设置环境变量和应用程序运行时要使用的值，请参阅设置环境变量
    • 要指定调试C或C ++应用程序时使用的源文件的位置 ，请参阅指定源文件的位置
    • 要指定运行配置的存储位置，访问方式以及运行应用程序时要打开的透视图，请参阅指定运行配置的位置
