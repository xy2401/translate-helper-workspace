使用内容辅助

在本节中，您将使用内容辅助来完成编写新方法。如果尚未打开文件，请在Java编辑器中打开文件junit.samples / VectorTest.java ，并在“大纲”视图中选择testSizeIsThree()方法。如果文件不包含此类方法，请参阅添加新方法以获取有关如何添加此方法的说明。

  1. 用以下行替换TODO注释。

    assertTrue(fFull.size() == 3);
    Vector v = new Vector();
    for (int i=0; i<3; i++)
      v.addElement(new Object());
    assert

  2. 将光标放在单词assert的末尾，按Ctrl+Space激活内容辅助。将出现带有提案列表的内容辅助窗口。滚动列表以查看可用选项。

  3. 在内容辅助窗口仍处于活动状态时，在assert后在源代码中键入字母“t”（中间没有空格）。该列表变窄，仅显示以“assert”开头的条目。单击列表中的各个项目以查看每个项目的任何可用Javadoc帮助。

  4. 从列表中选择assertTrue(boolean) ，然后按Enter 。插入assertTrue(boolean)方法的代码。
  5. 完成该行，使其如下所示：

    assertTrue(v.size() == fFull.size());

  6. 保存文件。
