调试UI实用程序类

除了定义用于显示调试UI的一般框架之外，调试UI插件还包括实现调试器UI的有用实用方法的类。

DebugUITools对许多这些实用程序进行分组，并包含以下方法：

  • 在启动特定启动配置之前保存和/或构建工作空间
  • 代表调试UI客户端存储和检索图像
  • 查找与给定调试模型关联的调试模型表示
  • 检索调试插件首选项存储
  • 有关调试下当前程序的信息，例如当前上下文，进程或控制台
  • 使用各种默认值打开启动配置对话框
  • 启用或禁用步骤过滤器的使用

IDebugView为调试视图提供了通用功能。它提供对底层查看器及其调试模型表示的访问。通常，客户端应该扩展AbstractDebugView，而不是从头开始实现接口。AbstractDebugView提供了许多有用的功能：

  • 在操作注册表中存储操作
  • 底层查看器上下文菜单的通用处理
  • 删除键和双击功能的一般实现
  • 在视图中显示错误消息的机制
