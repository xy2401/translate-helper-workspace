C / C ++项目视图

注意：此视图不再受支持。建议使用Project Explorer视图。

该 C / C ++项目视图在树结构中仅显示与C和C ++项目文件相关的元素，类似于Project Explorer视图。在此视图中，您可以执行以下操作：

  • 浏览C / C ++源文件的元素
  • 在编辑器视图中打开文件
  • 在新窗口中打开项目
  • 创建新项目，类，文件或文件夹
  • 管理现有文件（剪切，粘贴，删除，移动或重命名）
  • 从本地历史记录恢复已删除的文
  • 导入或导出文件和项目

在“ C / C ++项目”视图中选择的文件会影响其他视图中显示的信息。

右键单击Project Explorer视图中的任何资源，打开一个弹出菜单，允许您执行复制，移动，创建新资源，相互比较资源或执行团队操作等操作。

背部 此命令显示当前显示之前显示的层次结构。例如，如果您进入资源，则结果显示中的Back命令会将视图返回到您激活Go Into命令的同一层次结构。此按钮的悬停帮助会告诉您它将从何处开始。此命令类似于Web浏览器中的“后退”按钮。
向前 此命令显示当前显示后立即显示的层次结构。例如，如果刚刚选择了“后退”命令，则在结果显示中选择“前进”命令会将视图返回到激活“后退”命令的同一层次结构中。此按钮的悬停帮助会告诉您它将从何处开始。此命令类似于Web浏览器中的“前进”按钮。
向上 此命令显示当前最高级别资源的父级的层次结构。此按钮的悬停帮助会告诉您它将从何处开始。
