分枝

在尚未准备好将更改放入主开发流程的情况下，创建分支并向该分支释放资源非常有用。它对于为现有版本创建增量补丁也很有用。

要创建分支：

  1. 从项目的上下文菜单中，选择Team> Branch 。

  2. 在文本区域中输入新分支的分支名称。
  3. 对话框将提出默认版本标记名称。如果您不喜欢建议的名称，可以在文本区域中进行更改。
  4. 如果您想立即开始在分支机构工作，请确保选中该复选框。
  5. 单击确定 。

注意：版本名称很重要;如果您想稍后合并更改，则需要使用此选项。它标识创建分支的点。
