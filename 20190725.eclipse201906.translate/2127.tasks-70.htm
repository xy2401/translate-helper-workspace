将资源与本地历史进行比较

要将非托管Workbench资源与本地历史记录中的状态进行比较：

  1. 在其中一个导航视图中，选择要与本地历史记录状态进行比较的资源。
  2. 从资源的弹出菜单中，选择“ 比较” >“ 本地历史记录” 。将打开“与本地历史记录比较”页面
  3. 在“ 本地历史记录”列表中选择一个州。文本比较编辑器打开。
  4. 单击“ 选择下一个更改”和“ 选择上一个更改”按钮以浏览本地历史记录中的状态与Workbench资源之间所做的更改。
  5. 完成后单击“ 确定” 。

要将CVS管理的Workbench资源与本地历史记录中的状态进行比较：

  1. 在其中一个导航视图中，选择要与本地历史记录状态进行比较的资源。
  2. 从资源的弹出菜单中，选择Compare With > History ....历史记录视图打开。
  3. 通过单击历史记录视图中的“本地修订”工具栏按钮，将“历史记录视图”置于“ 本地修订”模式。这会筛选出除本地历史记录修订版之外的所有修订版。
  4. 从历史记录视图中选择修订。文本比较编辑器打开。
  5. 单击“ 选择下一个更改”和“ 选择上一个更改”按钮以浏览本地历史记录中的状态与Workbench资源之间所做的更改。
  6. 完成后单击“ 确定” 。
