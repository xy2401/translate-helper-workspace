内容/代码协助

在Java编辑器中按Ctrl + Space代码完成。这将打开可用代码完成列表。以下段落中列出了使用代码帮助的一些提示：

  • 您可以使用鼠标或键盘（向上箭头，向下箭头，向上翻页，向下翻页，主页，结束，回车）来导航和选择列表中的行。
  • 如果在内容辅助列表中选择一行，则可以查看该行的Javadoc信息。
  • 单击或按下列表中选定行的Enter键会将选择内容插入编辑器中。
  • 您可以在Javadoc注释中访问专门的内容辅助功能。

在Java>编辑器>代码辅助首选项页面中配置内容辅助的行为。
