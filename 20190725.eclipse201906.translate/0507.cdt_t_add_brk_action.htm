添加断点操作

要添加新的断点操作：

  1. 单击窗口>首选项 。
  2. 展开“ C / C ++”>“调试” ，然后单击“ 断点操作” 。



  3. 在“ 断点操作”页面中，单击“ 新建” 。



  4. 在“ 新建断点操作”对话框中，在“ 操作名称”框中键入操作的名称 。
  5. 从“ 操作类型”列表中选择一个操作。
  6. 设置动作的属性：
    • 要进行声音操作，请选择声音文件。
    • 对于日志操作，键入要在“日志操作消息”控制台中显示的消息。
    • 对于恢复操作，请以秒为单位设置“ 恢复时间”。
    • 对于外部工具操作，单击“选择”以选择程序。
  7. 单击确定 。

新断点操作显示在工作空间列表中任何断点的可用操作中。
