设置发现选项

请注意，此功能已被弃用，不再完全受支持

对于大多数make项目，您将需要解析构建的输出以填充路径和符号表。

要填充表格：

  1. 右键单击项目，然后选择“ 属性” 。
  2. 展开C / C ++ Build并选择Discovery选项 。
  3. 选择“ 自动发现路径和符号”选项以启用它。
  4. 对于“C / C ++属性”窗口的“发现选项”页面上的扫描仪配置，您有以下选项：

    每个项目都有各种发现配置文件 - 所有发现的路径和符号都与项目相关联;所有文件都使用此信息。

    启用生成扫描仪信息命令
    允许从扫描仪检索信息。




    对于这两个配置文件，可以选择基于先前构建的输出来发现路径和符号。若要激活发现，请单击“ 浏览”以选择生成日志文件，然后单击“ 加载”按钮。
