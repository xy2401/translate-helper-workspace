另一个用户进行更改

既然Fred在他的Workbench中有项目，他需要修改几个文件并与存储库同步以提交它们。

  • Fred将对file1.txt和file2.txt进行更改
  • Fred将同步并将其传出的更改提交给这两个文件。

弗雷德应该如下进行：

  1. 修改file1.txt ，如下所示
    原创内容：
    This is the contents
of file 1.
    新内容（更改以粗体显示）：
    This is the contents
Fred-update
of file 1.
  2. 修改file2.txt如下
    原创内容：
    File2 is a small file
with simple text.
    新内容（更改以粗体显示）：
    File2 is a (Fred was here) small file
with simple text.
  3. 观察Project Explorer更新资源的CVS状态。请注意，Fred已更改的两个文件前面带有“>”。
  4. 要将更改提交到存储库，Fred可以：
    • 选择这两个文件，然后选择Team> Synchronize with Repository，或者，
    • 选择项目，然后选择Team> Synchronize with Repository

    通常选择项目是最容易的事情。我们这样做。选择项目，然后从其上下文菜单中选择Team> Synchronize with Repository 。当系统要求您切换到“团队同步”透视图时，请选择“ 是” 。

  5. 当同步视图打开时，Fred可以浏览他所做的更改，最后应通过在Synchronize View中选择JanesTeamProject并将其更改提交到file1.txt和file2.txt ，然后从上下文菜单中选择Commit 。在提交更改之前，您必须输入提交注释。
