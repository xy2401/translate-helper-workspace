Project Explorer视图

Project Explorer视图提供Workbench中资源的分层视图。在此处，您可以打开文件进行编辑，也可以选择资源进行导出等操作。

右键单击Project Explorer视图中的任何资源，打开一个弹出菜单，允许您执行复制，移动，创建新资源，相互比较资源或执行团队操作等操作。要查看每个菜单项的功能描述，请将选择突出显示移动到该菜单项，然后按上下文相关帮助键（例如，Microsoft Windows上的F1）。

默认情况下，Project Explorer视图包含在Resources透视图中。要将其添加到当前透视图，请单击 窗口>显示视图>其他...>常规>项目浏览器 。

工具栏

Project Explorer视图的工具栏包含以下按钮：

全部收缩
折叠视图中所有资源的树扩展状态。
与编辑链接
切换Project Explorer视图选择是否链接到活动编辑器。选择此选项后，更改活动编辑器将自动将Project Explorer选项更新为正在编辑的资源。
菜单
提供菜单项，允许您对Project Explorer视图的内容进行排序或过滤，以及选择工作集。
