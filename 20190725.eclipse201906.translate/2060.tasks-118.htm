版本控制存储库中的项目

您可以在存储库中对项目进行版本控制，而无需将项目添加到Workbench。

  1. 切换到 CVS Repository探索透视图或添加 CVS存储库查看当前透视图。
  2. 在“CVS存储库”视图中，展开存储库位置并查找要进行版本控制的项目。
  3. 右键单击项目，然后选择Tag as Version 。
  4. 输入您要创建的版本的名称。
  5. 单击“ 确定”关闭对话框并创建版本。

提示：您可以通过单击“标记资源”对话框中的“ 详细信息”按钮来浏览项目的现有版本。单击“ 从存储库刷新”应填充“现有版本”列表。在命名新版本之前查看现有版本有时会很有帮助。

提示：如果要应用的版本标记已存在于某些要进行版本控制的资源上，则可以执行Tag with Existing 。此操作将现有标记移动到所选资源，并可用于移动版本和分支标记。如果要应用的标记未显示在列表中，则可以单击“ 从存储库刷新” 。如果找不到该标记，则可以单击“ 配置标记” ，打开一个对话框，允许您在存储库中的特定文件上搜索标记。
