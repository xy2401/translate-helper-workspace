Java编辑器模板首选项

对话框中将显示以下字段和按钮：

选项 描述
名称 模板的名称。
上下文 上下文确定模板的使用位置和可用的预定义模板变量集。Java编辑器为Java和Javadoc区域定义上下文。
  • Javadoc ：提案仅在Javadoc评论中提供
  • Java ：Proposal仅在Java代码中提供
  • Java类型成员 ：仅在完成Java类型成员时提供Proposal
  • Java语句 ：仅在完成Java语句时提供提议
  • SWT ：只有在构建路径上使用SWT在Java代码中完成时才提供建议
  • SWT类型成员 ：仅在构建路径上使用SWT完成Java类型成员时提供提议
  • SWT语句 ：仅在构建路径上使用SWT完成Java语句时提供提议
自动插入 如果选中，代码帮助将自动插入模板（如果它是插入符号位置上唯一可用的提议）。
描述 模板的描述，在选择模板时显示给用户。
图案 模板模式。
插入变量...... 显示预定义的上下文特定模板变量的列表。
