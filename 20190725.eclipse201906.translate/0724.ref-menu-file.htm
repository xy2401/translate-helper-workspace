功能

 键盘快捷键

新

创建Java元素或新资源。在“ 窗口”>“自定义透视图”中配置子菜单中显示的元素。在Java透视图中，默认情况下，可以使用创建项目 ， 包 ， 类 ， 接口 ， 枚举 ， 注释 ， 源文件夹 ，Java工作集，JUnit测试，文件和文件夹的操作。

Alt + Shift + N.

打开文件

在编辑器中从文件系统打开文件。

 

关

关闭当前编辑器。如果编辑器包含未保存的数据，则会显示保存请求对话框。

Ctrl + W

关闭所有

关闭所有编辑器。如果编辑器包含未保存的数据，则会显示保存请求对话框。

Ctrl + Shift + W

保存

保存当前编辑器的内容。如果编辑器不包含未保存的更改，则禁用。

Ctrl + S.

另存为

以新名称保存当前编辑器的内容。

 
保存全部 使用未保存的更改保存所有编辑器的内容。如果没有编辑器包含未保存的更改则禁用 Ctrl + Shift + S.
还原 将当前编辑器的内容还原为已保存文件的内容。如果编辑器不包含未保存的更改，则禁用。  
移动 移动资源。在Java Elements上禁用。要移动Java元素，请使用Refactor> Move （更新对文件的所有引用）或Edit > Cut / Paste （不更新引用）。  
改名 重命名资源。在Java Elements上禁用。要重命名Java元素，请使用Refactor> Rename （更新对文件的所有引用）。  
刷新 使用本地文件系统刷新所选元素的内容。从没有特定选择启动时，此命令将刷新所有项目。 F5
将行分隔符转换为 将选定内容中所有文本文件的行分隔符转换为选定的分隔符类型。  
打印 打印当前编辑器的内容。编辑器具有焦点时启用。 Ctrl + P.
切换工作区 切换到另一个工作区。  
重新开始 重新启动应用程序。  
进口 打开导入向导对话框。JDT不提供任何导入向导。  
出口 打开导出向导对话框。JDT提供JAR文件导出向导和Javadoc生成向导 。  
属性 打开选择元素的属性页面。在Java项目上打开了Java Build Path页面和Javadoc Location页面 。对于JAR存档，请在此处配置JAR的源附件和Javadoc位置 。 Alt + Enter
出口 退出Eclipse
