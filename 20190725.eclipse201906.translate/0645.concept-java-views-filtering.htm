在Java视图中过滤

应用视图过滤器

所有Java视图都允许过滤显示的Java元素。可用过滤器集取决于视图。

过滤元素：

  1. 在视图工具栏上，单击“ 菜单”按钮，然后选择“ 过滤器...” 。
  2. 选择或清除要应用于视图的过滤器 （阅读过滤器说明以了解所选过滤器的功能）。
  3. （可选）您可以选择用于过滤视图的模式：
    • 选择对话框顶部的“ 名称过滤器模式”复选框。
    • 在下面的文本字段中，指定所需的模式（将隐藏与其中一个模式匹配的名称）。

过滤会员

几个Java视图（例如， 大纲 ， 类型层次结构 ， 成员 ）提供成员（字段，类型和方法）的过滤。过滤器可用作工具栏按钮或视图菜单项，具体取决于视图。有3个成员过滤器：

  • 隐藏字段 ：激活时，此过滤器会导致从视图中删除所有字段。
  • 隐藏静态字段和方法 ：激活此过滤器后，将从视图中删除所有静态成员。
  • 隐藏非公共成员 ：激活此过滤器后，将从视图中删除所有非公共成员。
此外， Package Explorer视图可以显示或隐藏编译单元内的所有元素。

要在Package Explorer视图中显示或隐藏成员，请选中或清除“ 在Package Explorer中显示成员”复选框 Java>外观首选项页面。

过滤工作集

几个Java视图（例如Package Explorer ， Type Hierarchy ）提供了一个或多个工作集的过滤。视图菜单中提供了过滤器。只有选定工作集中包含的元素才会显示在视图中。
