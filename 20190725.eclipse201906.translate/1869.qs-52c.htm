简单比较

在其中一个导航视图中，选择file1.txt和file2.txt，从上下文菜单中选择Compare With> Each Other 。

将出现一个对话框，指示两个文件相同。

编辑file1.txt如下：

  • 删除第1行“ 这是第1行”。
  • 将第3行改为“这是一条更好的第3行”。
  • 插入第4a行（第5行之前），内容为“这是第4a行，它是新的”

文件（file1.txt）现在应该如下所示：

这是第2行。
这是一个更好的线3。
这是第4行。
这是第4a行，它是新的
这是第5行。

选择File> Save （或按Ctrl + S） 保存文件内容。

要比较文件，请再次选择file1.txt和file2.txt，然后在导航视图上下文菜单中选择Compare With> Each Other 。
