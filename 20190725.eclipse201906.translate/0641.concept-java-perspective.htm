Java Perspectives

Java开发工具为工作台提供了以下视角：

Java的

设计用于处理Java项目的透视图。它由一个编辑区和以下视图组成：

  • Package Explorer
  • 大纲
  • 问题
  • 的Javadoc
  • 宣言

Java浏览

用于浏览Java项目结构的透视图。它由一个编辑区和以下视图组成：

  • 项目
  • 包
  • 类型
  • 会员

Java类型层次结构

设计用于探索类型层次结构的透视图。它可以在类型，编译单元，包，项目或源文件夹上打开，并由“类型层次结构”视图和编辑器组成。

调试

用于调试Java程序的透视图。它包括一个编辑区和以下视图。

  • 调试
  • 断点
  • 变量
  • 表达式
  • 大纲
  • 安慰
  • 任务
