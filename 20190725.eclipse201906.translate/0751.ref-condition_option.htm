断点启用条件

选择“ 条件”选项以启用为断点提供自定义条件的功能。

每个断点都可以有一个唯一的条件来确定何时命中断点。

断点的条件可以是任何计算为true或false的逻辑表达式。表达式在断点位置的范围内进行计算，这意味着在编写表达式时，不能在断点位置范围之外引用类等。

请考虑以下示例：

public class Person {

	String name = "";
	int age = 0;

	public Person(String name, int age) {
		this.name = name;
		this.age = age;		//breakpoint here
  }
};

如果我们采用上面的例子，在指定的位置放置一个断点，并转到断点属性，我们可以添加条件。在这种情况下，我们仅限于包含断点的类成员和Object提供的成员。

例如，有效条件可能是：

	age == 56

意味着断点只会在年龄等于56时暂停。

条件可以添加到“ 断点视图”详细信息窗格中的断点或使用下面显示的“ 断点属性” 。
