从表生成动态实体

使用EclipseLink JPA时，可以从数据库表中创建动态实体。这种动态持久性提供了对JPA的所有好处的关系数据库的访问， 而无需编写或维护Java类。

Dali根据需要在运行时动态创建类。

要生成动态实体：

  1. 在Project Explorer中右键单击JPA项目，然后选择JPA Tools> Generate Dynamic Entities from Tables 。


    将出现“ 从表生成动态实体”向导的 “选择表”页面。

    选择表格


  2. 在“ 选择表”页面上，选择要从中生成实体的表，然后单击“ 下一步” 。

    将出现“ 表关联”页面。

    表关联


  3. 在“ 表关联”页面上，指定应生成哪些表关联。使用“ 创建新关联”向导可以创建其他关系。

    单击下一步 。将显示“ 自定义默认实体生成”页面。

    自定义默认值


  4. 在“ 自定义默认实体生成”页面上，指定生成实体时要使用的默认信息，然后单击“ 下一步” 。

    将出现“ 自定义单个实体”页面。

    自定义单个实体


  5. 使用“ 自定义单个实体”页面可以自定义特定的生成实体。

  6. 单击“ 完成”以完成向导并生成实体。

Dali使用VIRTUAL访问类型生成动态实体，如示例：带有动态实体的示例eclipselink-orm.xml文件中所示。

使用动态实体示例eclipselink-orm.xml文件

<?xml version="1.0" encoding="UTF-8"?>
<entity-mappings version="2.4" xmlns="http://www.eclipse.org/eclipselink/xsds/persistence/orm" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://www.eclipse.org/eclipselink/xsds/persistence/orm http://www.eclipse.org/eclipselink/xsds/eclipselink_orm_2_4.xsd">
    <entity class="quickstart.demo.model.Inventory" access="VIRTUAL">
        <attributes>
            <id name="itemSku" attribute-type="long">

            </id>
        </attributes>
    </entity>
    <entity ...">
    </entity>
</entity-mappings>
