在Eclipse中使用APT的无头建筑

有关从命令行直接或使用ant构建工作空间的说明

APT插件可以与Eclipse一起使用，以直接从命令行或通过ant及其内置的ant任务构建预配置的工作空间。

从命令行构建

在安装了安装了APT插件的Eclipse的情况下，您可以使用以下命令调用命令行构建器：
eclipsec -nosplash -application org.eclipse.jdt.apt.core.aptBuild -data% WORKSPACE％
这将在无头模式下将Eclipse作为控制台应用程序调用，并按照命令行提供的方式构建工作空间。将在构建中观察所提供工作空间的任何配置。

用Ant构建

同样，您需要安装Eclipse并安装APT插件。鉴于此，可以使用build.xml文件中的以下xml调用“apt”任务：
<project name =“test_eclipse”default =“build”basedir =“。”> <taskdef name =“apt”classname =“org.eclipse.jdt.apt.core.build。JdtApt“/> <target name =”build“> <apt workspace =”％WORKSPACE％“eclipseHome =”％ECLIPSE_HOME％“/> </ target> </ project>
