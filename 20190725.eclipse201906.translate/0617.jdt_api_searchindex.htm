Java搜索的索引

JDT搜索为所有源和从属JAR创建索引。当JAR被添加到类路径中时，它被索引。此操作可能需要一些时间，具体取决于JAR的大小。为了提高性能，可以预先构建索引，并在JAR添加到类路径时为其指定索引文件。这仅适用于JAR。

生成索引

索引可以通过以下方式预先生成：

  • API JavaIndexer＃generateIndexForJar 。
  • 命令行应用程序JavaIndexer
  • Ant任务eclipse.buildJarIndex

指定索引

可以将索引指定为要添加的库的类路径属性 。索引文件必须是有效的URL格式。如果未找到索引文件，JDT将生成索引文件并将其用于搜索。
