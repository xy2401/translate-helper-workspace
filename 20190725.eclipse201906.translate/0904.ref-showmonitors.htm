显示监视器

选择“ 显示监视器”命令[ ]更改是否显示暂停线程的监视器信息。

注意：底层VM必须支持监视器信息的显示。如果使用IBM或SUN VM，则任何大于1.4的版本都将支持监视器信息。

请考虑以下代码示例：

/**
 * Hello World
 */
public class HelloWorld {


	 public static void main(String[] args) {
		Object mutex = new Object();
		synchronized (mutex) {
			System.out.println("Hello World!");
		}
	 }
}

如果上面的代码片段在选择了“ 显示监视器”的支持VM上运行，则监视器信息将表示为“键”，并在任何堆栈帧之前显示为挂起线程的子级。

但是，如果我们在选择了“ 显示监视器”的不受支持的VM上运行相同的代码段，则会警告我们VM不支持监视器信息。
