可以使用以下设置来设置以下首选项 运行/调试首选项页面。

这里的首选项对所有调试器都是通用的，并且管理样式和提示选项

描述

默认

显示源代码时重用编辑器

单步执行应用程序时，调试器会在编辑器中显示源代码。启用此选项后，调试器将重用其打开的编辑器以显示来自不同源文件的源。这可以防止调试器打开过多的编辑器。当此选项关闭时，调试器将为每个需要显示的源文件打开一个新编辑器。

命中断点时激活调试视图 遇到断点时，此选项会引起对调试视图的注意。如果视图已经打开，它将被激活。如果视图尚未打开，它将自动打开。 上
在“运行到行”操作期间跳过断点 此选项控制在执行“运行到行”操作时是否忽略断点。当该选项打开时，调试器不会在调用“运行到行”操作时遇到的断点处挂起。选项关闭时，断点行为正常。 离
删除所有断点时提示确认 此选项控制在尝试删除所有断点时是否提示您进行确认 上

改变了颜色值 此选项允许您在变量视图，表达式视图，内存视图中，在运行程序变量的任何位置更改已更改值的颜色 红色
更改值背景颜色 此选项允许您更改已更改变量的选择颜色，例如在显示列的变量视图中 黄色
