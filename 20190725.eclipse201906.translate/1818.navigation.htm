使用键盘导航用户界面

用户界面可使用键盘导航。Tab键用于遍历窗口中的控件（例如，对话框，或当前透视图的视图和编辑器以及Workbench窗口的主控件）。要选择使用Tab键的视图（例如编辑器），请使用Ctrl + Tab。当焦点位于编辑器或视图选项卡上时，按左/右箭头键将移至上一个/下一个选项卡，然后按Enter键将焦点移至编辑器或视图中的当前选择。

菜单

大多数菜单都为每个条目分配了助记符，允许您通过键入带下划线的字母而不是鼠标来选择它们。您还可以使用箭头键在菜单和子菜单中移动来选择项目。

可通过以下方式使用键盘访问各种可用菜单：

  • F10访问主菜单栏上的菜单。
  • Shift + F10会弹出当前视图的上下文菜单。（注意：此快捷方式实际上取决于您的窗口管理器，但对于大多数人来说，它应该是Shift + F10。）
  • 如果有当前视图，Ctrl + F10将打开当前视图的下拉菜单。对于编辑器，Ctrl + F10将打开编辑器区域左侧标记栏的菜单。
  • Alt +助记符将激活特定条目的Workbench菜单（例如，Alt + W将关闭Window菜单）。
  • 仅限Microsoft Windows：按Alt键将重点关注菜单栏。

控制

助记符被分配给对话框，首选项页面和属性页面中的大多数控件标签（例如，按钮，复选框，单选按钮等）。要访问与标签关联的控件，请使用Alt键以及标签中带下划线的字母。

导航背景

为包，导航器视图，Workbench首选项和属性对话框保存导航上下文。首选项和属性对话框的选定页面将保存在对话框的调用之间，但不会保存在工作台调用之间。

骑自行车的编辑，观点和观点

要在编辑器，视图和透视图之间切换，工作台提供了一个由Ctrl和一个功能键调用的循环功能。所有这些循环功能都可以回忆起最后选择的东西，以便在两个项目之间来回快速循环。骑车功能是

  • Ctrl + F6 - 循环到编辑器
  • Ctrl + F7 - 循环查看
  • Ctrl + F8 - 循环到透视

如果使用“StickyKeys”输入，则可能需要启用“保持下一个/上一个编辑器，查看和透视对话框打开”选项 一般偏好页面。

此外，Ctrl + E可用于激活编辑器下拉菜单，Ctrl + PageUp和Ctrl + PageDown可用于在打开的编辑器之间切换，或在同一上下文中的视图之间切换。

加速器

Eclipse中的许多操作都分配了一个加速器。有关加速器的其他信息，请参阅按键 。

快速访问

要快速访问视图，命令，首选项页面等UI元素，可以使用“快速访问”对话框 窗口>导航>快速访问并默认绑定到Ctrl + 3 。开始在过滤器字段中输入以查看匹配项。例如，要打开帮助视图，请键入Ctrl + 3，然后键入“help”。首批比赛之一将是打开帮助视图;其他匹配显示与帮助相关的命令和首选项页面。您可以使用箭头键为给定的过滤字符串选择不同的匹配项。按Enter选择突出显示的条目。这将执行命令，或打开视图，透视图或向导等。

关注嵌入式链接

对于嵌入在对话框中的链接（例如首选项页面之间的链接或“关于”对话框中的超链接），请按“空格”键以激活当前选定的链接。

更改视图和编辑器的大小

您可以使用“大小”系统菜单中的命令调整大多数视图和编辑器的大小。默认情况下，活动部件的系统菜单绑定到Alt +“ - ” 。选择“尺寸”子菜单（左，右，上或下）后，所选部件的一侧将突出显示，并可使用箭头键移动。

帮助系统

您可以使用以下组合键通过键盘导航帮助系统：

  • 在框架内按Tab键（页面）将转到下一个链接，按钮或主题节点。
  • 要展开/折叠树节点，请按向右/向左箭头。
  • 要移至下一个主题节点，请按向下箭头或Tab键
  • 要移至上一个主题节点，请按向上箭头或Shift + Tab
  • 要显示所选主题，请按Enter键。
  • 要向上或向下滚动，请按Home或End。
  • 要返回按Alt +向左箭头;前进按Alt +向右箭头。
  • 要转到下一帧，或按Ctrl + Tab键（Ctrl + F6，如果使用Mozilla或基于Mozilla的浏览器）。
  • 要移至上一帧，请按Shift + Ctrl + Tab。（Shift-Ctrl + F6，如果使用Mozilla或基于Mozilla的浏览器）。
  • 要移至显示主题内容的框架，请按Alt + K（在Windows或Internet Explorer上使用嵌入式帮助浏览器时）。
  • 要移至“目录”选项卡，请按Alt + C.
  • 要移至“搜索结果”选项卡，请按Alt + R.
  • 要在标签之间移动，请按向右/向左箭头。
  • 要切换视图，请选择一个选项卡，然后按Enter键。
  • 要切换并移动到视图，请选择一个选项卡，然后按向上箭头。
  • 要移至搜索条目字段，请按Alt + S.
  • 要打印当前页面或活动框架，请按Ctrl + P.
  • 要在当前页面或活动框架中查找字符串，请按Ctrl + F（在Windows上使用嵌入式帮助浏览器或Internet Explorer时）。

帮助系统弹出对话框中的大多数控件标签都会为其分配助记符。要访问与标签关联的控件，请使用Alt键以及带下划线的字母。
