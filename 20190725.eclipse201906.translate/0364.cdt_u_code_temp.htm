模板首选项

使用 模板首选项面板用于操作CDT中预定义的任何常用模板或创建自己的模板。要修改，删除，导出，导入或创建自己的模板，请单击窗口>首选项> C / C ++>模板 。

模板首选项选项
选项 描述
新... 打开“ 新建模板”对话框以创建新模板。
编辑... 打开“ 编辑模板”对话框以编辑选定的模板。
去掉 从列表中删除选定的模板。
恢复已删除 恢复上次删除操作删除的模板。
恢复默认值 删除对所选模板进行的最后编辑，将其恢复为未修改状态。
进口... 导入模板。
出口... 导出所选模板。
使用代码格式化程序 允许将当前选定的代码样式应用于模板。
