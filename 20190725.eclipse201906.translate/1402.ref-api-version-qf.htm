版本编号快速修复

此快速修复程序用于快速轻松地更正插件的版本编号问题。

可以在以下两个位置之一使用此快速修复：

  1. 问题观点 。
  2. 相应清单文件的清单编辑器 。

问题观点

  • 打开“ 问题”视图。
  • 选择版本编号问题。

  • 使用Ctrl + 1从“问题”视图触发快速修复（或使用上面显示的上下文菜单）。

清单编辑器

  • 使用版本问题打开清单上的清单编辑器 。
  • 导航到MANIFEST.MF选项卡。
  • 在相应的版本问题上使用Ctrl + 1 。
