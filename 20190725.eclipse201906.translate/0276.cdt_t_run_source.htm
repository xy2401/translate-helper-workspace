指定源文件的位置

您可以指定在调试C或C ++应用程序时使用的源文件的位置。默认情况下，此信息来自您的项目。

要指定源文件的位置：

  1. 在“C / C ++项目”视图中，选择一个项目。
  2. 单击运行>运行或运行>调试 。
  3. 在“ 配置”框中，展开“ C / C ++ Local” 。
  4. 选择运行或调试配置。
  5. 单击“ 源”选项卡。
  6. 要将源容器添加到源位置列表：
    • 单击“ 添加”以打开“ 添加源”对话框
    • 选择容器类型。
    • 从所选类型的可用容器列表中选择一个容器。
  7. 您可以通过选择容器并单击“ 删除”或“ 编辑”按钮来删除或修改源容器。
  8. 您可以通过选择容器并单击向上或向下按钮来更改源容器的顺序。
  9. 要在源位置搜索重复，请选中“ 在路径上搜索重复的源文件”复选框。
