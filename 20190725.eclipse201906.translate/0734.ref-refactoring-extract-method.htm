提取方法错误

当您尝试提取方法时，您可能会收到以下一个或多个常见错误：

  • Selected block references a local type declared outside the selection声明的本地类型本地类型声明不是选择的一部分，但是由选择用于提取的语句之一引用。扩展包含本地类型声明的选择，或者减少未选择对本地类型声明的引用的选择。
  • A local type declared in the selected block is referenced outside the selection 。选择包括本地类型声明，但类型也在所选语句之外引用。扩展包含对本地类型的所有引用的选择，或者减少未选择本地类型声明的选择。
  • Ambiguous return value: selected block contains more than one assignment to local variable赋值在所选块内找到多个局部变量赋值。要么减少只选择一个赋值的选择，要么扩展选择，使得至少除了1之外的所有引用都被选择覆盖。
  • Ambiguous return value: expression access to local and return statement selected所选语句生成多个返回值。例如，如果选择了表达式并且也修改了表达式的参数，则会出现这种情况。要解决此问题，请将选择范围扩展为涵盖已修改参数的读取访问权限。
  • Selection contains a break statement but the corresponding break target isn't selected要解决此问题，请将选择范围扩展为包括中断/继续目标，或者减少选择不包含中断/继续语句的选择。
  • Selection contains a continue statement but the corresponding continue target isn't selected要解决此问题，请将选择范围扩展为包括中断/继续目标，或者减少选择不包含中断/继续语句的选择。
  • Selection starts inside a comment无法提取Selection starts inside a comment部分内容。扩展它涵盖整个注释的选择，或者减少根本没有涵盖注释的选择。
  • Selection ends inside a comment无法提取Selection ends inside a comment部分内容。扩展它涵盖整个注释的选择，或者减少根本没有涵盖注释的选择。
  • Cannot extract selection that ends in the middle of a statement选择调整选择以使其完全覆盖一组语句或表达式。用户可以使用“ 编辑”菜单中的“ 展开选择”将选择范围扩展到有效范围。
