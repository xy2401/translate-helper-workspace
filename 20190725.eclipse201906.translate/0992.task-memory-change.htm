更改内存位置的内容

要在Memory视图中更改内存监视器中内存位置的内容：

  1. 在“ 监视器”窗格中，选择包含要编辑的内存位置的内存监视器。内存将显示在“ 渲染”窗格中，您将在其中执行所有其他步骤。如果添加了多个渲染，请选择包含要编辑的渲染的选项卡。
  2. 向下滚动到要更改的内存位置。或者，在监视器中单击鼠标右键，然后选择“ 转到地址”弹出菜单项。这将打开渲染底部的“转到地址”部分，您可以在其中输入要跳转到的地址。
  3. 选择包含要更改的值的行，然后双击要更改的值。

    提示：如果渲染当前处于焦点，则无需双击要更改的值即可对其进行编辑。相反，您只需开始键入更改，编辑器就会激活。

  4. 输入该内存位置的有效值。
  5. 按Enter键提交更改。调试器检查有效值。
