从CVS存储库中签出项目

可以使用Checkout向导或从CVS Repositories视图将项目从CVS存储库签出到Workbench。

使用结帐向导结账

它可以从“导入”菜单，“新建”>“项目”菜单和“CVS存储库浏览”透视图的工具栏中获得。从CVS存储库视图执行Checkout As时也会打开它。

  1. 启动结帐向导 。它可以从“ 导入”菜单，“ 新建”>“项目”菜单和“CVS存储库浏览”透视图的工具栏中获得。从CVS存储库视图执行Checkout As时也会打开它。（ 注意 ：如果禁用CVS功能，则只能从“ 导入”菜单中使用“检出”向导）。
  2. 从已知存储库列表中选择所需的存储库，或者，如果所需的存储库不在此列表中，请选择创建新的存储库位置，然后单击“ 下一步” 。
  3. 如果输入新的存储库位置，请输入存储库信息，完成后单击“ 下一步” 。（注意 ：此页面的格式与创建CVS存储库位置向导的格式相同。）
  4. 从现有模块列表中选择一个或多个模块，或键入要检出的模块的名称。对于从服务器获取现有模块列表不切实际或不受支持的情况，支持输入名称。如果存储库配置为在其根目录下承载单个项目，则还可以输入点（。）作为模块名称。指定要检出的模块时，单击“ 下一步” 。
  5. 可选 ：选择是将一个或多个所选模块签出为新项目还是现有项目或文件夹。如果选择了一个模块且它不包含.project文件（Workbench项目配置文件），您还可以选择使用New Project向导配置项目（例如，为了使项目成为Java项目，例如）。单击下一步 。
  6. 可选 ：对于单个项目，您可以将项目的位置配置为默认位置或工作空间外的自定义位置。对于多个项目，您可以配置所有项目应位于的父文件夹。单击下一步 。
  7. 可选 ：选择要签出的标记，然后单击“ 完成” 。

从CVS存储库视图中签出

要从CVS存储库视图签出项目到Workbench：

  1. 切换到 CVS Repository探索透视图或添加 CVS存储库查看当前透视图。
  2. 在“CVS存储库”视图中，展开存储库位置。
  3. 展开HEAD并选择要作为项目添加到Workbench的文件夹。如果您要查找特定版本的文件夹：
    1. 展开“ 版本”类别，找到要添加到Workbench的文件夹名称。
    2. 展开文件夹以显示其版本。

    如果您要在分支中查找最新文件夹：

    1. 展开“ 分支”类别。
    2. 展开包含要添加到Workbench的文件夹的分支。
  4. 从所选文件夹的弹出菜单中，选择以下选项之一：
    1. 退房退房每个选定的文件夹为在本地工作区具有相同名称在资源库中的文件夹中的一个项目。
    2. 签出为...将选定的文件夹签出到本地工作区中的自定义配置项目中。注意：选择多个文件夹时，此操作仅允许为新项目指定自定义父位置。
  5. 如果在单个项目中选择了Check Out As ... ，则会显示两个可能的对话框之一，具体取决于存储库中的文件夹是否包含.project文件。
    1. 如果有.project文件，该对话框将接受自定义项目名称和位置。
    2. 否则，该对话框将是New Project向导，它允许完全自定义项目（例如Java项目）。

提示：可以从CVS存储库中检出任何文件夹，包括非根文件夹。
