C / C ++项目属性：预处理器包括路径，宏等 - 条目选项卡

可以在项目属性页“预处理器包含路径，宏等”，“条目”选项卡上检查和设置包含C / C ++索引器的路径和预处理器宏。此选项卡显示Scanner Discovery使用的语言设置提供程序 ，以提供包含路径，宏和其他语言设置 。

请注意，也可以在单个资源（例如文件或文件夹）上设置条目。打开文件或文件夹属性以检查特定于资源的条目。文件夹或项目上的条目将应用于其下的所有子文件夹和C / C ++文件 - 除非在较低级别上覆盖。

“条目”选项卡显示每种语言的语言设置提供程序列表。可以扩展提供者节点以显示由提供者定义的包含路径和宏。用户可以在“用户设置条目”提供程序下添加自定义条目。

条目选项卡控件
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
语言 列出工具链定义的所有语言。选择一种语言以查看适用的提供商列表。
设置条目 列出所选语言的所有提供程序以及它们提供的条目。展开提供程序节点以查看条目。
显示内置值 禁用复选框“显示内置值”以过滤掉视图中的内置设置条目。内置编译器条目需要提供给C / C ++索引器，但在构建期间不需要提供给编译器。
加... 向提供者添加条目。仅在提供程序支持该操作时启用。
编辑... 编辑选定的设置条目。仅在提供程序支持该操作时启用。
删除 删除所选的设置条目。仅在提供程序支持该操作时启用。
提升 将所选条目在列表中向上移动。仅在提供程序支持该操作时启用
下移 将所选条目移动到列表中较低的位置。仅在提供程序支持该操作时启用
