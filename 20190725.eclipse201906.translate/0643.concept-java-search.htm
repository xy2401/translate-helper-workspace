Java搜索

Java搜索支持允许您查找Java元素（包，类型，方法，字段）的声明，引用和出现。索引支持搜索，该索引在后台保持最新，因为更改了与Java元素相对应的资源。Java搜索在独立于其构建状态的工作空间上运行。例如，可以在关闭自动构建时执行搜索。

可以从Java元素的弹出菜单或Java搜索对话框中启动以下搜索：

命令

描述

参考

查找对所选Java元素的所有引用

声明

查找所选Java元素的所有声明

实施者

查找所选Java接口的所有实现者

读取权限

查找对所选Java字段的所有读取访问

写访问权限

查找对所选Java字段的所有写访问

匹配类型引用的位置

在指定位置查找所有类型引用（在强制类型中，作为字段类型，...）

文件中的出现次数

在其文件中查找所选出现的所有Java元素

搜索范围定义为：

工作区 - 工作区中的所有项目和文件都包含在此搜索中

封闭项目 - 包含当前所选元素的项目

层次结构 - 此搜索中仅包含所选元素的类型层次结构

工作集 - 此搜索中包含属于所选工作集的资源
