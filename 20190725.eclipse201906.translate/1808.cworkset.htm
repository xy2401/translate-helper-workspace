工作集

工作集用于在视图中显示或在一组元素上进行操作的组元素。

导航视图使用工作集来限制显示的资源集。如果在导航器中选择了工作集，则仅显示工作集中包含的资源，资源子节点和资源父节点。可以通过“ 过滤器”视图菜单项基于工作集过滤问题视图，任务视图和书签视图。使用搜索工具时，您还可以使用工作集来限制搜索的元素集。

不同的视图提供了指定工作集的不同方法。Eclipse SDK中的视图默认使用Window Working Set 。窗口工作集最初是空的。要修改窗口工作集，需要通过Window> Perspective> Customize Perspective ...启用Window Working Set命令组。然后，您可以通过Window> Working Sets菜单或相应的工具栏按钮修改窗口工作集。支持工作集的视图通常使用以下工作集选择对话框来管理现有工作集和创建新工作集：

工作集也可以是手动构建工作流程的一部分。禁用autobuild后，将启用Project> Build Working Set菜单。从这里，您可以有选择地构建您选择的工作集。

注意：新创建的资源不会自动包含在活动工作集中。如果它们是现有工作集元素的子元素，则它们隐式包含在工作集中。如果要在创建它们之后包含其他资源，则必须将它们显式添加到工作集中。
