设置方法断点

处理没有源代码的类型（二进制类型）时使用方法断点。

  1. 在“ 大纲视图”中打开该类，然后选择要添加方法断点的方法。
  2. 从方法的弹出菜单中，选择Toggle Method Breakpoint 。
  3. 断点视图中将显示断点 。如果该类存在source，则断点也会出现在文件编辑器的标记栏中，用于所选方法。
  4. 启用断点时，在执行方法之前，执行方法中的任何行之前，线程执行将暂停。

方法断点也可以设置为在方法退出时中断。在“断点”视图中，选择断点并在其上下文菜单中切换“ 退出”项。

方法断点可以像行断点一样被删除 ， 启用和禁用 。
