定义最小的介绍配置

创建新的介绍配置的第一个阶段是添加必要的扩展点并创建最少量的内容。这个阶段往往是最困难的。以下是创建最小介绍配置并将其显示在产品中所需的步骤。

创建插件项目并添加扩展点

创建一个新的插件项目“org.eclipse.intro.minimal”，该项目依赖于org.eclipse.ui.intro。在插件清单编辑器中添加扩展名org.eclipse.ui.intro。编辑plugin.xml，直到看起来像这样：

<？xml version =“1.0”encoding =“UTF-8”？> <？eclipse版本=“3.2”？> <plugin> <extension point =“org.eclipse.ui.intro”> <intro class =“org.eclipse.ui.intro.config。CustomizableIntroPart“icon =”$ nl $ / icons / image_obj.gif“id =”org.eclipse.intro.minimal“label =”Minimal Intro“/> </ extension> <extension point =”org.eclipse.ui.intro .config“> <config content =”$ nl $ / introContent.xml“id =”org.eclipse.intro.minimal.config“introId =”org.eclipse.intro.minimal“> <presentation home-page-id = “root”standby-page-id =“standby”> <implementation kind =“html”> </ implementation> </ presentation> </ config> </ extension> </ plugin>

创建介绍配置文件

使用这些内容在插件项目org.eclipse.intro.minimal中创建文件introcontent.xml。

<？xml version =“1.0”encoding =“utf-8”？> <introContent> <！ - 根页 - > <page id =“root”style =“html / style.css”style-id =“page”> <title style-id =“intro-header”>最小简介</ title> <link url =“http：//org.eclipse.ui.intro/switchToLaunchBar”label =“转到工作台”id =“workbench”/> <link label =“链接到教程”url =“ http：//org.eclipse.ui.intro/showPage？id = tutorials“id =”tutorials“/> </ page> <！ - 备用页面 - > <page id =”standby“style =”html / style.css“style-id =”page“> <title style-id =”intro-header“>待机页面</ title> </ page> <！ - 教程页面 - > <page id =”tutorials“ style =“html / style.css”style-id =“page”> <title style-id =“intro-header”>教程</ title> <text>正在构建的页面</ text> </ page> < / introContent>

定义产品绑定

在此步骤中，我们需要确定正在使用的产品并创建介绍产品绑定并修改其plugin.xml文件。如果您使用的是Eclipse SDK，那么它将位于插件项目org.eclipse.sdk中，该项目可以在Eclipse安装的plugins目录中找到。以下说明假设您使用的是Eclipse SDK，稍作修改，这些对于RCP或其他基于Eclipse的产品同样有效。打开plugin.xml并找到以下部分：

<extension point =“org.eclipse.ui.intro”> <introProductBinding introId =“org.eclipse.ui.intro.universal”productId =“org.eclipse.sdk.ide”> </ introProductBinding> </ extension>

将introId更改为“org.eclipse.intro.minimal”，以便扩展代码如下所示。

<extension point =“org.eclipse.ui.intro”> <introProductBinding introId =“org.eclipse.intro.minimal”productId =“org.eclipse.sdk.ide”> </ introProductBinding> </ extension>

测试和自定义

从工作台启动一个新的Eclipse应用程序。您的介绍屏幕可能已经显示（取决于Eclipse上次退出时的状态）。如果未显示，请从工作台菜单中选择帮助/最小简介。通过编辑文件html / style.css，您可以修改介绍的外观。您现在可以通过编辑intro.xml文件或添加配置扩展来扩展您的介绍。
