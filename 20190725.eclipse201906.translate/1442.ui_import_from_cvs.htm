从存储库导入项目

插件导入向导允许您通过从活动目标平台，目录或目标定义导入插件来在工作空间中创建项目。如果要导入的插件包含适当的源参考信息，则导入向导允许您从存储库导入与这些捆绑包关联的项目。

例如，Eclipse SDK中的插件包含Eclipse-SourceReference：标识CVS存储库的标头，以及插件源自的插件。您可以导入构建插件的项目的特定版本，也可以从HEAD导入关联的项目。

要从存储库导入插件，请执行以下操作：

  1. 打开导入向导： 文件>导入...>插件开发>插件和片段
  2. 在“ 导入自”部分中，选择插件源，例如活动目标平台。

  3. 在“ 导入为”部分中， 从存储库中选择“ 项目” 。

  4. 按Next 。将显示目标平台中可从存储库导入的插件列表。选择要导入的插件，然后按“ 下一步” 。
  5. 下一页允许您导入关联项目的特定版本（用于构建目标平台的版本），或者从HEAD导入。选择所需选项，然后按“ 完成” 。项目将被导入并在工作区中显示为项目。

您还可以从插件视图从存储库导入插件。选择要导入的插件，然后使用“ 存储库...”操作中的“ 导入为”>“项目” 。
