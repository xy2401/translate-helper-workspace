控制台视图

该 控制台视图显示进程的输出，并允许您为进程提供键盘输入。有许多可用的控制台，请参阅Open Console下拉列表，了解可供您使用的控制台。

输出控制台显示几种不同类型的文本，每种文本都有不同的颜色：

  • 标准输出
  • 标准错误
  • 标准输入

控制台视图工具栏

下表列出了Console视图中显示的工具栏选项。请注意，各种控制台之间的工具栏选项可能不同。

图标 名称 描述
下一个错误 在“控制台”视图中显示下一个错误。
以前的错误 在“控制台”视图中显示上一个错误。
在编辑器中显示错误 切换以使所选错误与编辑器视图同步（默认）。
复制构建日志 将构建输出复制到文件。
滚动锁定 切换Scroll Lock。
清除控制台 清除当前控制台。
Pin控制台 强制控制台视图保持在窗口区域中的其他视图之上。
显示所选控制台 如果打开了多个控制台，则可以从列表中选择要显示的控制台。
打开控制台 打开一个新控制台以显示以下日志：
  • 新的控制台视图
  • 程序输出控制台
  • Windows程序输出控制台
  • CVS和其他基于已安装功能的

控制台视图上下文菜单

当您在控制台视图中右键单击（或当您按下Shift + F10时是在控制台视图），你看到的上下文菜单依赖于可见的控制台视图以下一个或多个选项：

名称 描述
明确 清除当前控制台。
编辑器选项：剪切，复制，粘贴，全选 这些选项执行标准编辑操作。哪些选项可用取决于焦点在Console视图中的位置。例如，您无法将文本粘贴到程序输出中，但可以将文本粘贴到文件的底部。
去线 打开一个对话框，将焦点移动到您指定的行。该对话框还指示控制台缓冲区中的总行数。
查找/替换... 打开“ 查找/替换”对话框，该对话框仅对“ 控制台”视图中的文本进行操作 。
打开链接 选择以在任何支持的控制台视图中跟踪检测到的超链接。例如，CVS控制台。
滚动锁定 切换Scroll Lock。
