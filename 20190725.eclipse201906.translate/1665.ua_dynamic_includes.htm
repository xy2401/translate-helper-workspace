包括

使用的包含包括文档中的其他文档的一部分。标记指定要包括的文档和元素，并且在向用户显示文档之前，include元素将替换为所引用的元素。

例如，您可能希望在您提供的每个帮助文档的末尾显示版权。您可以将其复制并粘贴到每个文档中，但如果版权发生变化（例如版权年份），则必须修改每个文档。

相反，您可以将版权放在名为copyright.xhtml的单独XHTML文件中，并为其分配唯一标识符，如下所示：

<p id =“copyright” >版权所有2006我的公司。版权所有。</ p>

然后对于每个文档，只需包含版权段落：

<p>这是我的文件。它最终应该有版权。</ p> <include path =“my.plugin.id/path/copyright.xhtml/copyright”/>

在将文档发送到要显示的浏览器之前， include元素将被版权段落替换 。
