调试视图

Debug View允许您在工作台中管理程序的调试或运行。它显示正在调试的每个目标的挂起线程的堆栈帧。程序中的每个线程都显示为树中的节点。它显示您正在运行的每个目标的进程。

如果线程被挂起，则其堆栈帧将显示为子元素。

执行控制命令允许您调试程序，启动，停止和单步执行代码。

掉到相框

此命令允许您退回并重新输入指定的堆栈帧。此功能类似于“向后运行”并在中途重新启动程序。

要退回并重新输入指定的堆栈帧，请选择要“ 放入 ”的堆栈帧，然后选择“ Drop to Frame” 。

一些注意事项适用于此功能：

  • 您不能删除堆栈上的本机方法。
  • 全局数据不受影响，并将保留其当前值。例如，不会清除包含元素的静态向量。
