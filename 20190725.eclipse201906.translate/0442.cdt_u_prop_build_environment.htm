C / C ++项目属性，环境

使用Environment属性页面为工作区中的所有项目自定义构建环境;它允许您控制构建使用的环境变量。

环境偏好选项
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
要设置的环境变量 显示当前环境变量列表及其对应值。这些是构建时使用的环境变量值。
变量 指定环境变量的名称。
值 指定环境变量的值。
将变量附加到本机环境 在执行期间将变量追加到本机环境。
用指定的环境替换本机环境 用指定的变量替换本机环境，然后在完成后恢复本机环境。
加... 打开“ 新建变量”对话框以创建新的环境变量和值。您创建的自定义环境变量在列表中以粗体显示。

单击“ 变量”以按名称选择变量以包含在值中。选择“ 添加到所有配置”以使此新环境变量可用于所选项目的所有配置;否则，该变量仅适用于当前选定的配置。
选择... 打开“ 选择变量”对话框，您可以从中选择系统变量列表。

编辑... 修改所选环境变量的名称和值。
去掉 从列表中删除选定的环境变量。
取消定义 取消定义当前选定的变量;但是，某些变量（例如PATH变量）不能是未定义的。
