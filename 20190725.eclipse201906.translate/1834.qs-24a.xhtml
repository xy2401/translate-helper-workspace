导出文件

可以通过以下方式从Workbench导出文件：

  • 拖放到文件系统（在大多数平台上），或
  • 复制并粘贴到文件系统，或
  • 使用 导出向导 。

拖放或复制粘贴

操作系统的文件系统资源管理器可用于将文件夹或文件的副本从Workbench导出到文件系统。

  1. 打开操作系统的文件系统资源管理器。

  2. 将文件JanesFile.txt从导航视图之一拖动到文件系统资源管理器。

  3. 根据您拖动到的位置，您可能需要在拖动时按住Ctrl或Shift键以确保复制文件。在拖动光标上查找一个小加号，以了解是否正在复制或移动文件。

  4. 也可以通过在Project Explorer中选择文件并选择“ 编辑”>“复制”来实现导出

    将其粘贴到文件系统资源管理器中。

导出向导

导出向导可用于从Workbench导出到文件系统。

  1. 在导航视图中选择项目JaneQuser。

  2. 从弹出菜单中，选择“ 导出” 。

  3. 在Export向导中，选择File system ，然后单击Next 。

  4. 展开JaneQUser项目，然后单击JanesFolder。在右侧窗格中，确保仅选择JanesINIFile.ini。请注意，左窗格中的文件夹和项目现在有一个灰色的复选框，表示将导出其中一些但不是全部的内容。

    “ 过滤器类型”按钮可用于过滤要导出的资源类型。

    注意：要仅导出JanesINIFile.ini，只需在导航视图中选择它，然后选择“ 文件”>“导出” 。导出向导将自动确保它是唯一选择导出的文件。

  5. 在“ 目录”字段中，键入或浏览以在文件系统中选择要驻留的导出资源的位置。

    如果输入了不存在的目录的名称，则在选择“ 完成”后 ，“导出”向导将提供创建它的名称。

  6. 在“ 选项”区域中，选项包括：

    • 在没有警告的情况下覆盖现有资源
    • 为文件创建目录结构或仅创建选定的目录

  7. 完成后单击“ 完成” 。
