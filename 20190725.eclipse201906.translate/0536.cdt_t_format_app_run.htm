运行Formatter应用程序

运行formatter应用程序就像运行org.eclipse.cdt.core一样简单。命令行中的CodeFormatter应用程序：

    eclipse -application org.eclipse.cdt.core.CodeFormatter [ OPTIONS ] [ -config <configFile> ] <files>

在MacOS上调用时，指向配置文件或源文件的路径可以是相对的，但它们将从eclipse.ini文件的位置计算。这是MacOS上Eclipse启动器的限制。在所有其他平台上，相对路径是相对于当前用户目录计算的。

<文件> 要格式化的C / C ++源/头文件和/或目录。只有源/头文件结尾（如.c .cpp或.h）将在给定目录中格式化。
-config <configFile> 使用指定属性文件中的格式样式。有关详细信息，请参阅为Formatter应用程序生成配置文件 。

OPTIONS

描述

-救命

显示帮助信息。

-安静

仅打印错误消息。

-verbose

关于格式化工作要冗长。
