创建CVS存储库位置

先决条件 ：必须已在主机上配置CVS服务器，才能在Workbench中创建有效的存储库位置。

要创建新的存储库位置：

  1. 选择打开CVS Repositories视图 窗口>显示视图>其他...> CVS> CVS存储库 。或者，CVS存储库视图也显示在 CVS Repository探索视角。
  2. 在工具栏上，单击“ 添加CVS存储库” （或从“CVS存储库视图”的上下文菜单中选择“ 新建”>“存储库位置” ）。将打开“添加CVS存储库”向导。
  3. 输入标识和连接到存储库位置所需的信息：
    1. 在“ 主机”字段中，键入主机的地址。（例如： mymachine.com) 。
    2. 在Repository path字段中，键入主机上存储库的路径（例如/home/repo, d:/repo.)
    3. 在“ 用户”字段中，键入要在其下连接到存储库的用户名。
    4. 在“ 密码”字段中，键入上述用户名的密码。
    5. 从“ 连接类型”列表中，选择CVS服务器的身份验证协议。Eclipse CVS客户端有三种连接方法：
      • pserver - 一种CVS特定的连接方法。
      • extssh - Eclipse附带的SSH 2.0客户端
      • pserverssh2 - 通过ssh2提供pserver连接。
      • ext - CVS ext连接方法，它使用SSH等外部工具连接到存储库。ext使用的工具在Team> CVS> EXT Connection Method首选项页面中配置。
    6. 如果主机使用自定义端口，请启用“ 使用端口”并输入端口号。
  4. 可选：如果要在关闭此向导时对指定主机的指定用户进行身份验证，请选择“完成时验证连接” 。（如果未选择此选项，则在尝试访问存储库的内容时，稍后将对用户名进行身份验证。）
  5. 可选：如果要在会话之间保存密码 ，请选择“ 保存密码” ，这样下次启动Eclipse时就不必再次输入密码。要以安全的加密形式保留登录凭据，请使用Secure Storage 。
  6. 单击完成 。存储库位置已创建。

注意 ：在Eclipse 3.0之前， extssh连接方法仅支持SSH 1.0。在3.0或更高版本中，此连接方法也已升级为支持SSH 2.0。

提示 ：如果要使用extssh连接方法但希望保持本地工作空间（沙箱）与CVS命令行客户端兼容，则可以使用ext连接方法并将其配置为使用Eclipse内部的extssh Team> CVS> Ext Connection Method首选项页面。
