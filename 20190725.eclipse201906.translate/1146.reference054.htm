连接

下表列出了persistence.xml编辑器的“连接”页面中可用的属性。

属性 描述 默认

交易类型

指定此持久性单元的连接是否使用以下事务类型之一：

  • 默认 - 选择以使用容器使用的容器。

  • JTA （Java Transaction API） - Java EE服务器的事务。

  • 资源本地 - 由持久性单元引用的JDBC驱动程序的本机操作。

JTA

批量写作

指定批量写入的使用以优化具有多个写入操作的事务。

在部署时将此属性的值设置为会话。

注意 ：在Java SE和Java EE环境中使用时，此属性适用。

以下是oracle.toplink.config的有效值。BatchWriting：

  • JDBC -使用JDBC批处理编写。

  • 缓冲 - 不要使用JDBC批处理写入也不使用本机平台批处理写入。

  • OracleJDBC -使用JDBC批处理编写和Oracle本机平台批处理编写。

  • 无 - 不使用批量写入（将其关闭）。

没有

声明缓存

指定查询是否缓存其JDBC语句。如果启用，您还可以设置要缓存的语句数。

50

原生SQL

指定Dali是否包含特定于平台（即“本机”）的SQL语句。如果为false，则Dali使用通用SLQ。

假

数据库

JTA数据源名称

如果选择JTA作为事务类型，则输入持久性单元的默认JTA数据源的名称。

非JTA数据源名称

如果选择“ 资源本地”作为事务类型，则输入非JTA数据源的名称。

此属性不适用于使用Generic平台的项目。

EclipseLink连接池

定义连接池驱动程序，URL，用户名和密码。

对于使用Generic平台的项目，请注意这些属性。

绑定参数

控制查询是否使用参数绑定。

注意：在Java SE环境中使用时，此属性适用。

此属性不适用于使用Generic平台的项目。

真正

读取连接

JDBC读取连接池中允许的最大和最小连接数。

注意 ：在Java SE环境中使用时，这些属性适用。

这些属性不适用于使用Generic平台的项目

最低：2

最多2

写连接

JDBC读取连接池中允许的最大和最小连接数。

注意 ：在Java SE环境中使用时，这些属性适用。

这些属性不适用于使用Generic平台的项目

最低：5

最多10

独家连接

仅当“ 事务类型”为“ 本地资源”时，这些字段才可用

独家连接模式

指定Dali何时执行通过写连接的读取。

  • 始终 - 如果读取隔离的实体，则创建独占的隔离客户端会话，否则创建独占客户端会话。

  • 隔离 - 如果读取隔离的实体，则创建独占的隔离客户端会话，否则引发错误。

  • 事务性 - 如果某些或所有实体需要隔离缓存，则创建隔离的客户端会话，否则创建客户端会话。

交易

懒惰的连接获取

指定Dali是否懒惰地获取写连接。

真正
