来源行动

源菜单命令：

名称

功能

 键盘快捷键

切换评论

注释或取消注释包含当前选择的所有行。

Ctrl + /

添加阻止评论

在包含当前选择的所有行周围添加块注释。

Ctrl + Shift + /

删除块注释

从包含当前选择的所有行中删除块注释。

Ctrl + Shift + \
生成元素注释 向所选元素添加注释。请参阅“ 代码模板”首选项页面以指定生成的注释的格式。可用于类型，字段，构造函数和方法。 Alt + Shift + J.

向右移动

增加当前选择行的缩进级别。仅在选择涵盖多行或单个整行时激活。

 

向左移动

减少当前选定行的缩进级别。仅在选择涵盖多行或单个整行时激活。

 

正确的缩进

更正当前文本选择表示的行的缩进。

Ctrl + I

格式

使用代码格式化程序格式化当前文本选择。格式化选项在Code Formatter首选项页面上配置。

Ctrl + Shift + F.

格式元素

使用代码格式化程序格式化包含当前文本选择的Java元素。“格式元素”操作适用于方法和类型级别。格式化选项在Code Formatter首选项页面上配置。

 
添加导入 为当前选定的类型引用创建导入声明。如果类型引用符合条件，则将尽可能删除资格。如果引用的类型名称无法唯一映射到当前项目的类型，系统将提示您指定正确的类型。添加导入尝试按照“ 组织导入”首选项页面中指定的导入顺序进行操作 Ctrl + Shift + M.
组织进口 在当前打开或选择的编译单元中组织导入声明。将删除不必要的导入声明，并按照“ 组织导入”首选项页面中的指定对所需的导入声明进行排序。组织导入可以在不完整的源上执行，并且当引用的类型名称无法唯一地映射到当前项目中的类型时将提示您。
您还可以通过调用包上的操作或选择一组编译单元来组织多个编译单元。 		Ctrl + Shift + O.

对会员排序

根据“ 成员排序顺序”首选项页面中指定的排序顺序对类型的成员进行排序

 

清理

执行各种更改，以便根据“ 清理”首选项页面中指定的设置清理代码

 
覆盖/实施方法 打开“ 覆盖方法”对话框 ，允许您覆盖或实现当前类型的方法。可用于类型内的类型或文本选择。  
生成Getter和Setter 打开Generate Getters and Setters对话框 ，允许您为当前类型的字段创建Getters和Setter。可用于字段和类型，也可用于类型内的文本选择。  
生成委托方法 打开“生成委派方法”对话框，该对话框允许您为当前类型的字段创建方法委托。适用于带字段的字段和类型。  
生成hashCode（）和equals（） 打开Generate HashCode和Equals对话框，允许您启动和控制hashCode的生成并等于当前类型中的方法。  
生成toString（） 打开Generate toString（）对话框 ，该对话框允许您启动和控制当前类型中toString（）方法的生成。  
使用字段生成构造函数 添加构造函数，用于初始化当前所选类型的字段。可用于类型，字段或类型内的文本选择。  
从Superclass添加构造函数 为当前选定的类型添加超类中定义的构造函数。可用于类型内的类型或文本选择。  
环绕着 使用代码模板包围所选语句。在“ 模板”首选项页面上创建自己的模板。此外，您可以使用“ 编辑”菜单中的“ 扩展选择”来获取有效的选择范围。 Alt + Shift + Z.
外化字符串 打开Externalize字符串向导。此向导允许您通过访问属性文件的语句替换代码中的所有字符串。  
找到破碎的外部字符串 在选定的属性文件，包，项目或项目集中搜索损坏的外部化字符串。
