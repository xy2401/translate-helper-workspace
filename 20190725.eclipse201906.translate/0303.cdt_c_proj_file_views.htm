项目文件视图

项目文件和元素显示在C / C ++项目视图和ProjectExplorer视图中。

C / C ++项目视图

在树结构中仅显示与C和C ++项目相关的元素。在此视图中，您可以执行以下操作：

  • 浏览C / C ++源文件的元素
  • 构建目标
  • 创建新项目，类，文件或文件夹
  • 导入或导出文件和项目
  • 管理现有文件（剪切，粘贴，删除，移动或重命名）
  • 在编辑器视图中打开文件
  • 在新窗口中打开项目
  • 重构
  • 从本地历史记录恢复已删除的文
  • 搜索
在“C / C ++项目”视图中选择的文件会影响其他视图中显示的信息。

Project Explorer视图

Project Explorer视图提供了工作台中所有资源的分层视图，而不仅仅是您的C / C ++资源。在此视图中，您可以打开文件进行编辑，也可以选择资源进行导出等操作。

右键单击Project Explorer视图中的任何资源，打开一个弹出菜单，您可以从中执行复制，移动，创建新资源，比较资源或执行团队操作等操作。有关每个菜单项的功能的说明，请选择一个项目并按F1。

Project Explorer视图由Eclipse Platform提供。有关详细信息，请参阅“ Workbench用户指南” 。
