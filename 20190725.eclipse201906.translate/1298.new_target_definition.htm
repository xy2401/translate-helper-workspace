目标定义

目标定义指定目标的所有方面 - 包括其位置，组成插件和环境。

“ 新建目标定义”向导位于从顶级菜单中选择文件>新建>其他...>插件开发>目标定义 。

文件名和位置

唯一的限制是目标定义必须具有.target文件扩展名。目标定义可以放在任何项目和任何文件夹中。命名文件的灵活性允许您在同一文件夹中存储多个目标定义。

初始文件内容

可以使用以下四种方法之一初始化目标定义：

  1. Nothing ：此选项创建一个空目标定义。
  2. 默认值 ：此选项使用所有部分的默认设置创建最小目标定义。
  3. 当前目标 ：此选项使用当前目标平台的设置初始化文件 窗口>首选项...>插件开发>目标平台首选项页面。
  4. 模板 ：此选项可创建现有目标定义的副本。如果要创建扩充或稍微修改现有目标的目标定义，则此选项特别有用。
