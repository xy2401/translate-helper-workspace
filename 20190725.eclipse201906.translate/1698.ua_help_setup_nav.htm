生成的导航主题

如果目录中的主题未指定href属性，或者toc元素未指定topic元素，则“帮助”将自动生成包含主题标题的主题，标题以表示子主题列表的开头，以及一个无序的子主题链接列表（仅限直接子主题）。

产品可以使用CSS自定义这些生成主题的外观。帮助将在当前运行的产品插件的根目录中查找book.css文件，并将使用此样式表来设置生成的文档的样式。您可以依赖具有以下类的元素存在于生成的文档中，并使用它们来设置样式：

  • NavTitle ：包含主题标题的标题的类。
  • NavListTitle ：主题链接列表前的标题类。
  • NavList ：包含直接子主题链接的列表类。

例如，要隐藏列表项目符号，您将使用以下CSS规则：

UL.NavList {
   list-style-type: none;
}
