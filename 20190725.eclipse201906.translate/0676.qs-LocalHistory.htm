使用当地历史

在本节中，您将使用本地历史记录功能切换到以前保存的单个Java元素的版本。

  1. 在Java编辑器中打开junit.samples / VectorTest.java文件，然后在Outline视图中选择方法testCapacity（） 。
  2. 更改方法的内容，以便'for'语句读作：

    for (int i= 0; i < 99; i++)

    按Ctrl+S保存文件。
  3. 在Outline视图中，选择方法testCapacity() ，然后从其上下文菜单中选择Replace With> Element from Local History 。
  4. 在“ 从本地历史记录替换Java元素”对话框中，“本地历史记录”列表显示方法的各种已保存状态。“Java源比较”窗格显示所选历史记录资源与现有工作台资源之间差异的详细信息。

  5. 在顶部窗格中，选择以前的版本，然后单击“ 替换”按钮 。在Java编辑器中，该方法将替换为所选的历史版本。
  6. 保存文件。
  7. 除了用前一个版本替换方法的版本之外，您还可以恢复已删除的Java元素。再次，在Outline视图中选择方法testCapacity（） ，然后从其上下文菜单中选择Delete 。单击是确认生成的对话框并保存文件。
  8. 现在，在Outline视图中选择VectorTest类型，然后从其上下文菜单中选择Restore from Local History ....选择并检查“可用Java元素”窗格中的方法testCapacity（） 。和以前一样，“本地历史记录”窗格显示本地历史记录中保存的版本。

  9. 在“本地历史记录”窗格中，选择较早的工作版本，然后单击“还原” 。
