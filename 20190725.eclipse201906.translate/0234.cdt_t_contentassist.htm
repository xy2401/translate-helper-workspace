使用内容辅助

使用内容辅助将项目的C / C ++元素和模板插入代码中。您可以在源代码中插入模板，而不是重新输入常用的代码片段。

要插入模板或元素：

  1. 在C / C ++编辑器中，至少键入模板或元素的第一个字母，然后键入Ctrl + Space 。
    列表显示模板后跟以您键入的字母组合开头的元素。

  2. 执行以下操作之一：
    • 继续输入。该清单缩短了。如果列表中只有一个项目，则会自动插入。
    • 双击列表中的项目以将其插入到代码中。
    • 按Esc键关闭内容辅助窗口。

要设置内容辅助首选项：

  1. 单击窗口>首选项 。
  2. 展开C / C ++和编辑器 。
  3. 选择内容辅助 。
  4. 请执行下列操作：
    • 要在打开内容辅助时插入元素，并且它是列表中的唯一项目，请选中“ 自动插入单个提议”复选框。
    • 要自动插入所有提案的公共前缀，请选中“ 自动插入公共前缀”复选框。
    • 要显示按字母顺序排列的建议，而不是由相关性，选择按字母顺序复选框提出建议 。
    • 要更改“。”，“ - >”或“::”触发器的内容辅助的自动激活 ，请选中相应的复选框。
  5. 单击确定 。
