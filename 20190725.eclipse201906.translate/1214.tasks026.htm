将JPA元数据转换为XML

从版本3.2开始，Dali可以将元数据（例如转换器，查询和生成器）转换为XML映射文件。这允许您在XML映射文件中维护持久性单元（例如查询和生成器）的全局元数据。

  1. 在Explorer视图中右键单击项目，然后选择JPA Tools> Move to XML >特定元数据（例如Java Converters ， Java Queries或Java Generators ）。

    将元数据移动到XML


    将出现“ JPA元数据转换”对话框页面。

  2. 输入XML映射文件的文件名和位置，然后单击“ 完成” 。

Dali生成XML文件。
