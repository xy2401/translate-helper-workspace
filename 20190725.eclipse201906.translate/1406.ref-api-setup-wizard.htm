API工具安装向导

在任何插件项目可以使用API工具之前，必须首先将其设置为使用工具，这可以使用API工具设置向导完成。

要激活向导，请使用以下步骤：

  1. 打开插件项目的上下文菜单，然后选择PDE Tools
  2. 在生成的弹出菜单上选择API Tools Setup ...

选项

描述

默认
选择一个项目 此文本字段允许您按名称过滤项目的当前列表。'*'匹配任何内容和'？'匹配任何角色 空
项目 此列表包含当前工作空间中可以设置为使用API工具的所有项目。要转换，项目必须是Java插件项目。  
全选 选择列表中的所有项目。  
取消全选 取消选择列表中的所有项目。  
更新完成后删除component.xml

安装完成后，此选项将从插件项目中删除component.xml文件。在API Tools之前使用了component.xml文件来描述插件的API。

如果找到component.xml文件，安装向导将使用它来将API限制 （Javadoc标记）添加到插件中的相应源文件。可以在向导的预览页面上查看所有建议的更改。

 上

向导的第一页允许您选择项目并设置其他选项。

在预览页面上，您可以看到将对项目和任何源文件进行的所有更改。
