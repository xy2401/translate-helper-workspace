提高Java代码质量

Eclipse Java编译器执行的检查和分析比Java语言规范要求的更多。这样做是为了帮助您提高Java代码的质量。由于不同的用户对哪些警告感兴趣有不同的视图，因此这种行为是高度可配置的。有关可用选项，请参阅Java编译错误/警告首选项。

一些警告和错误应该是显而易见的，并且通常对每个人都有效。在某些情况下，您可能希望配置编译器以匹配您的代码样式和质量目标。如果您在某种程度上调整代码样式以使编译器可以更好地分析，则某些分析最有用。
