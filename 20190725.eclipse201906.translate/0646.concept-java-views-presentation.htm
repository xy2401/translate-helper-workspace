Java视图的表示选项

显示和隐藏方法返回类型

几个Java视图（例如Outline ， Members ）提供了方法，还可以显示它们的返回类型和类型参数。

要在Java视图中显示方法返回类型，请选中或清除“ 显示方法返回类型”或“ 显示方法类型参数”复选框 Java>外观首选项页面。

显示完整或压缩包名称

Package Explorer和Packages视图可以显示完整或压缩的包名称。

要显示完整的包名称：

  • 清除压缩所有包名称段，除了最后一个段复选框 Java>外观首选项页面。
要显示压缩包名称：

  • 选择压缩所有包名称段，除了最后一个段复选框 Java>外观首选项页面。

    压缩模式控制每个包名称段的显示字符数。始终显示包名称的最后一段。

    压缩模式为“。”表示仅显示分隔周期表示段。压缩模式中的数字（n）表示包名称段的前n个字符。示例是了解压缩模式的最佳方法。使用示例压缩模式， org.eclipse.jdt包将显示如下：

    。 ..jdt

    0 jdt

    2~或〜.ec~.jdt

    3~ org.ecl~.jdt
