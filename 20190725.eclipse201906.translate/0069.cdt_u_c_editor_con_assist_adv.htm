高级偏好

使用 “ 首选项”窗口中的“ 高级”面板用于配置“内容辅助”（ Ctrl + Space ）命令的行为。

高级内容辅助首选项选项
选项 描述
默认提案种类 指定内容辅助使用的提议类型和分配的键绑定。
内容辅助循环 指定内容辅助将循环的提议的顺序和类型。
向上 在循环列表中向上移动选定的提案项目。
下 在循环列表中向下移动选定的提案项。
每种提案类型超时（ms） 指定允许代码完成提议类型计算提案的毫秒数。如果需要更多时间，则认为此提议类型有缺陷。在这种情况下，将打开一个对话框，并向用户显示相应的消息
如果值为0，则每个完成提议可以在被假定为错误之前进行计算。
