在文件中搜索文本

其中一个导航视图的弹出菜单中的“ 转到”选项允许您通过搜索资源名称在Workbench中快速查找资源。

您也可以为一个包含在在工作台文件中的信息做上下文搜索。要查找包含特定字符串的所有文件：

  1. 在主工具栏上，单击 搜索按钮 。
  2. 在“ 包含文本”字段中键入搜索字符串，或使用下拉列表选择以前输入的搜索表达式。
    使用\作为包含特殊字符* ，？的搜索字符串的转义字符?或\ ，（例如： d:\\directory\\filename.ext ）。
  3. 完成输入搜索选项（例如，将搜索范围限定为指定的文件类型），然后单击“ 搜索” 。
  4. “搜索”视图显示搜索结果。右键单击“搜索”视图中的任何项目以打开弹出菜单，该菜单允许您从列表中删除项目，将搜索结果复制到剪贴板或重新运行搜索。要打开其中一个列出的文件，请双击它或从其弹出菜单中选择“ 打开” 。

如果关闭“搜索”视图，可以稍后通过选择返回 窗口>显示视图>其他...>常规>搜索 。
