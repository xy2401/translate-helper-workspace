C / C ++索引器

C / C ++索引器使用解析器创建源文件和头文件的数据库，为C / C ++搜索，导航功能和内容辅助的部分提供基础。

索引器在后台线程上运行，并对资源更改事件做出反应，例如：

  • C / C ++项目创建/删除
  • 源文件创建/删除
  • 文件导入
  • 源文件内容更改

可以通过使用源文件夹自定义索引器的行为，甚至可以完全关闭它。此可自定义行为可在每个项目的基础上使用（即，可以为工作区中的每个项目设置不同的索引器设置）。

CDT支持其他索引器的贡献，2个索引器提供默认CDT版本：

  • 快速C / C ++索引器：提供最快的索引功能 - 声明和交叉引用信息。这是推荐的索引器。
  • 完整的C / C ++索引器：以性能为代价提供更加准确的索引功能 - 声明和交叉引用信息。
