C / C ++项目属性，托管生成文件，C / C ++生成，自定义生成步骤

自定义选定的构建配置，允许为所选文件指定用户定义的构建命令步骤。除了构建命令之外，用户还可以指定任何其他输入文件依赖项，构建步骤生成的输出文件，执行命令时要在构建输出中显示的描述性消息，以及关于如何应用自定义构建步骤相对于配置中定义的其他工具（如果有）的规则。无论项目性质如何，都可以为任何文件类型指定自定义构建步骤。

从构建中排除
排除使用所选构建配置构建的选定文件。
组态：
从提供的列表中指定所选的构建配置。
资源配置设置，自定义生成步骤
为所选文件定义自定义生成步骤。
自定义构建步骤适用性
禁用所选文件的自定义生成步骤，或应用自定义生成步骤，覆盖为配置定义的任何其他工具。
附加输入文件名
指定此构建命令所需的以分号分隔的其他输入文件列表。默认情况下包含所选文件。路径被解释为相对于项目目录。
输出文件名
指定此构建命令生成的以分号分隔的输出文件列表。路径被解释为相对于构建目录。
命令
指定以分号分隔的命令列表，这些命令将执行以执行此构建步骤。
描述
指定在执行此构建步骤时将在构建输出中回显的公告字符串。
恢复默认值
将所有更改返回到所选配置的设置。
应用
应用任何更改。
