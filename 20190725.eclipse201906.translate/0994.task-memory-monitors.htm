使用内存监视器

要从Memory视图中查看内存的内容：

  1. 在“ 监视器”窗格中，选择包含要查看的内存位置的内存监视器。内存将显示在“ 渲染”窗格中，您将在其中执行所有其他步骤。如果添加了多个渲染，请选择包含要查看的渲染的选项卡。
  2. 如果需要，可通过选择“ 切换拆分窗格”按钮拆分“ 渲染”窗格（ ）。默认情况下，“内存”视图仅显示一个渲染窗格。单击“ 切换拆分窗格”时 ，将打开第二个渲染并显示为拆分窗格。
  3. 如有必要，请使用渲染中的滚动条查看当前渲染所显示的内存监视器基址上方或下方的内存位置。或者，您可以在渲染中右键单击并选择“ 转到地址”弹出菜单项或按Ctrl + G.这将打开渲染底部的一个部分，您可以在其中执行以下操作：
    • 选择“ 转到地址”下拉菜单项，然后输入要跳转到的地址。将定位渲染，以便输入的地址可见并被选中。
    • 选择“ 转到偏移”下拉菜单项，然后输入偏移量。将定位渲染，以便可以看到和选择表达式的地址（基址）加上输入的偏移量。负值将使渲染从基地址返回。
    • 选择Jump Memory Units下拉菜单项。此函数采用当前选定的地址并添加您指定的内存单元数。选择结果地址。负值将从当前地址返回渲染。

    对于所有这些条目，您可以通过选择“ 输入为十六进制”复选框将它们输入为HEX（如果未选中此复选框，则输入将为十进制）。在字段中输入后，按Enter键或单击“ 确定”转到渲染中的位置。要关闭此部分，请单击“ 取消”或按Ctrl + G.

    注意：如果输入前缀为0x ，则输入也被视为HEX。

  4. 如果需要，可通过单击其标题单元格的左侧或右侧并拖动它来更改列的宽度来更改任何列的宽度 - 或者在渲染内右键单击并从弹出菜单中选择“ 调整大小以适合 ”以便重新调整所有列的大小，以便可以查看其中的所有文本。或者，您可以在渲染内右键单击，然后从弹出菜单中选择“ 格式 ”。这将打开“格式”对话框。在此对话框中，您可以设置每行的单位数和每列的单位数。在进行这些设置时，对话框中的“ 预览”窗口将显示您正在设置的渲染布局。要将这些设置保存为默认布局，请单击“ 另存为默认值” 。
  5. 您还可以隐藏“内存”视图的元素以便于查看：
    • 您可以通过取消选择“ 切换内存监视器窗格”切换来隐藏“ 监视器”窗格。
    • 您可以通过隐藏的渲染和选择隐藏地址栏里面右键单击地址栏。要在隐藏地址列时还原地址列，请在渲染内右键单击，然后从弹出菜单中选择“ 显示地址列 ”。

如果您正在进行内存渲染并远离最初设置为监视的地址，则选择“ 重置为基址”弹出菜单项将光标置于内存监视器的基址。或者，您可以通过右键单击监视器并选择“ 重置” （或者，您可以选择多个监视器并选择此操作）来重置内存监视器的所有渲染。重置监视器时，默认情况下，可见渲染将重置为基址。要将当前“内存”视图中的所有渲染重置为基址，请修改“内存”视图首选项。
