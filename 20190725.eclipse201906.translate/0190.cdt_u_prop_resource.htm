C / C ++项目属性，资源

显示所选类型的资源信息。



资源属性

属性 描述
路径 指定工作空间中所选资源类型的位置。例如，类似于文件夹，项目映射到文件系统中的目录。
类型 指示所选资源的类型： 文件夹 ， 项目或文件 。
地点 具体说明文件系统中所选资源的位置。
最后修改 显示上次修改所选资源的日期。
文本文件编码 设置备用文本编码。由于文本文件的编码方式不同（取决于区域设置和平台），因此请使用主机操作系统区域设置的默认文本文件编码。但是，如果要使用源自其他源的文本文件（例如，使用与本机编码方案不同的文本文件，以便您可以轻松地与其他团队交换文件），请选择“ 其他”和“从列表中选择一个合适的。
继承自容器 启用后，所选资源将继承为其容器资源指定的文本编码。
其他 启用后，所选资源将使用不同于其容器资源的替代文本编码。如果要处理源自其他源（使用与本机编码方案不同的编码方案的文本文件）的文本文件，则可以启用此选项，以便您可以轻松地与其他人交换文件。
新文本文件行分隔符 指定要用于创建的新文本文件的行尾字符。
继承自容器 启用后，所选资源将继承为其容器资源指定的新文本文件的结束字符行。
其他 启用后，所选资源将使用备用行尾字符替换为其容器资源指定的新文本文件之外的新文本文件。例如，您可以将文本文件编码选项设置为UTF-8，然后将新文件的行结束字符设置为Unix ，以便文本文件以不特定于Windows操作系统的格式保存，并且文件可以在各种开发人员系统之间轻松共享。

适用于：任何项目，文件夹和文件，无论项目类型如何。
