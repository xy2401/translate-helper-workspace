安全存储

安全存储以加密形式保存数据。在某些操作系统上，它使用您的操作系统帐户信息来提供单点登录体验。

安全存储本身不包含加密代码。它使用Java虚拟机和/或操作系统提供的算法。

经常问的问题

  • 使用什么加密？

    默认情况下，从Java虚拟机请求56位DES算法。

    该 常规>安全性>安全存储首选项页面允许您更改加密算法。请注意，新算法仅适用于新创建的安全存储。兼容算法列表取决于运行此应用程序的Java虚拟机。

  • 如何更改密码？

    您可以通过更改密码...按钮更改密码 常规>安全性>安全存储首选项页面。

解决问题

  • 丢失安全存储密码或无效的安全存储密码

    如果您已指定密码恢复问题/答案，则可以使用 常规>安全>安全存储首选项页面以恢复主密码。恢复主密码后，强烈建议您使用更改密码...按钮更改主密码 常规>安全性>安全存储首选项页面。

    恢复的密码不会显示，但会在应用程序关闭之前存储在内存缓存中。

  • 如果一切都失败了

    如果您遇到无法找到解决方法的安全存储问题，可以使用“ 删除”按钮删除安全存储。 常规>安全性>安全存储首选项页面。当然，这将删除安全存储的所有内容。强烈建议您在删除安全存储后重新启动应用程序。
