环境标签

“ 编辑目标向导”中的“ 环境”选项卡用于设置影响目标编译和运行方式的环境设置。也可以在目标编辑器的环境页面上编辑环境设置。

“ 目标环境”部分有四个设置，用于描述为此目标构建的系统。如果留空，则使用当前运行环境的设置。更改这些设置将影响软件站点位置的解析方式以及如何构建和导出插件。

您可以在Java Runtime Environment部分中为此目标定义指定JRE或执行环境。如果在此处指定特定的JRE或EE，则将此目标设置为活动目标平台时，将更改工作空间的默认JRE。
