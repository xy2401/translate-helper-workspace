定义自定义IntroURL操作

使用org.eclipse.ui.intro.configExtension扩展点，插件可以提供自己的自定义操作，这些操作可以用作页面中链接元素的url值。例如，请考虑以下链接：

？HTTP：//org.eclipse.ui.intro/runAction pluginId = org.eclipse.pde.ui之间＆安培;类= org.eclipse.pde.ui.internal.samples.ShowSampleAction和ID = org.eclipse.sdk.samples。 swt.examples

这个IntroURL将运行一个名为ShowSampleAction的动作类，它位于插件“org.eclipse.pde.ui”的包“org.eclipse.pde.ui.internal.samples”中。要运行的示例的id是“org.eclipse.sdk.samples.swt.examples”。

要定义此简介URL的自定义版本，您可以使用以下标记：

   <extension point="org.eclipse.ui.intro.configExtension">
      <action
            name="myCommand"
            replaces="runAction?pluginId=org.eclipse.pde.ui&amp;class=org.eclipse.pde.ui.internal.samples.ShowSampleAction">
      </action>
   </extension>
使用上述扩展名，您现在可以使用以下URL运行相同的操作：

HTTP：//org.eclipse.ui.intro/myCommand ID = org.eclipse.sdk.samples.swt.examples

操作“myCommand”将替换为replaceces属性的值，任何剩余的URL参数将附加到末尾。替换完成后，生成的URL将扩展回：

？HTTP：//org.eclipse.ui.intro/runAction pluginId = org.eclipse.pde.ui之间＆安培;类= org.eclipse.pde.ui.internal.samples.ShowSampleAction和ID = org.eclipse.sdk.samples。 swt.examples
