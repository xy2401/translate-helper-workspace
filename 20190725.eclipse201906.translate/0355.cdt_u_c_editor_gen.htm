编辑首选项

使用编辑器首选项面板，用于控制C / C ++编辑器中文本突出显示的外观。

选项 描述
智能插入符号在标识符中的定位 将CamelCase标识符视为由单独的单词组成。
键入时报告问题 发现的问题将立即在编辑器中突出显示。
突出显示匹配括号 当光标位于括号旁边时，将突出显示匹配的括号。
突出显示非活动代码 当光标位于其中时，突出显示非活动代码。
外观颜色选项 列出可以指定颜色的项目。
颜色 指定显示所选项目的颜色。
文档工具评论 指定应使用哪个文档工具来确定编辑器的表示和行为。这样可以实现编辑器功能，例如内容辅助，文档注释生成和语法突出显示。此处选择的特定工具将用于与项目无关的文件，或者未设置项目级别首选项的文件。
