文件菜单操作

名称 功能 键盘快捷键
新（子菜单） 创建新项目，文件夹或文件 ALT + SHIFT + N
打开文件... 打开现有文件  
关 关闭当前编辑器（如果编辑器包含未保存的数据，则会显示保存请求对话框。） 按Ctrl + W
关闭所有 关闭所有编辑器（如果任何编辑器包含未保存的数据，将为它们显示保存请求对话框。） 按Ctrl + Shift + W
保存 保存当前编辑器的内容（如果编辑器不包含未保存的更改，则禁用。） 按Ctrl + S
另存为... 以新名称保存当前编辑器的内容  
保存全部 保存所有打开的编辑器窗口的内容（如果没有编辑器包含未保存的更改，则禁用。） 按Ctrl + Shift + S
还原 将当前编辑器的内容还原为最近保存的状态（如果编辑器不包含未保存的更改，则禁用。）  
移动... 移动资源  
改名... 重命名资源 F2
刷新 根据本地文件系统刷新所选元素（如果未选择任何内容，则刷新所有打开的项目。） F5
将行分隔符转换为（子菜单） 将当前编辑器的行分隔符更改为Windows（\ r \ n）或UNIX（\ n）格式  
打印... 打印当前编辑器的内容（当编辑器具有焦点时启用。） 按Ctrl + P
切换工作区（子菜单） 更改活动工作区（子菜单包含最近使用的工作区列表，以及其他... ，这将打开工作区选择对话框）  
重新开始 重新启动Eclipse  
进口... 打开“ 导入”对话框，允许选择“导入”向导  
出口 打开“ 导出”对话框，该对话框允许选择“导出”向导  
属性 打开所选元素的属性页面 Alt + Enter键
出口 退出Eclipse
