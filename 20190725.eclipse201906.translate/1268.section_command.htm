指挥科

关于本节

本部分允许您执行以下操作：

  • 将可执行工作台命令添加到步骤或子步骤
  • 从步骤或子步骤中删除可执行工作台命令
  • 编辑添加到步骤或子步骤的可执行工作台命令的属性

通过在“ 内容”部分中选择步骤或子步骤节点，可以访问此部分。

部分字段

本节包含以下字段：

标签 类型 名称 描述 需要
命令 组合框 Workbench命令 从此字段中选择最近添加的工作台命令，以将命令添加到步骤或子步骤。选择<无>以从步骤或子步骤中删除命令。

注意：无法将命令添加到包含子步骤的步骤。

此字段全局填充并与最近添加到所有步骤和子步骤的所有命令同步。

使用“ 浏览...”按钮选择新的工作台命令。 		没有
参数 表 Workbench命令参数 此不可编辑字段显示在命令组合框中选择的工作台命令的适用参数。

每行代表一个命令参数或键值对。第一列列出命令参数键，第二列列出命令参数值。

要编辑所选工作台命令的参数，请单击“ 浏览...”按钮。 		没有

部分行动

本节提供以下操作：

标签 类型 名称 描述
浏览... 按键 浏览工作台命令 单击此按钮可启动Command Composer对话框 。

此对话框允许您浏览，测试和选择要添加到步骤或子步骤的新可执行工作台命令。该对话框还允许您编辑所选工作台命令的适用参数。
