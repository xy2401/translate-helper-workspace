API弃用报告转换Ant任务

目的

此任务将apitooling.apideprecation Ant任务创建的XML报告转换为HTML文件。

用法

描述

Ant任务的名称是： apitooling.apideprecation_reportconversion 。要使用，jar文件apitooling-ant.jar必须位于Ant类路径上。 

<apitooling.apideprecation_reportconversion xmlfile = “...” htmlfile = “...” debug = “...” />

参数

属性 描述 需要
XMLFILE 此属性指定要转换的xml文件的位置。

使用绝对路径指定位置。 		是
HTMLFILE 此属性指定要生成的html文件的位置。
如果省略，则html文件名将是给定的xml文件名，其中扩展名将替换为html。

使用绝对路径指定位置。 		没有
调试 设置调试值。

可能的值为： true ， false
默认值为false 。 		没有

例子

 <apitooling.apideprecation_reportconversion xmlfile = “ /eclipse/deprecation/deprecation.xml ” debug = “true” />

这将运行在文件夹/eclipse/deprecation创建deprecation.html文件的任务。

如果启用了调试，则会在Ant控制台中显示某些调试跟踪。
