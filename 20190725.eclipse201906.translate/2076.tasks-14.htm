更改构建顺序

默认情况下，Workbench通过将项目引用解释为先决条件关系来计算构建顺序。或者，您可以明确定义项目的构建顺序。

Build Order首选项页面允许您禁用Use default build order选项，以便您可以访问项目列表并操纵它的顺序。

要定义Workbench执行构建项目的顺序：

  1. 打开 常规>工作区>构建顺序首选项页面。
  2. 确保清除“ 使用默认构建顺序”复选框。如果选中此选项，则会禁用此页面上的按钮，并按项目列表的顺序执行构建。
  3. （可选）单击“ 添加项目”和“ 删除项目”按钮以在列表中添加和删除项目。
  4. 在列表中选择一个或多个项目，然后单击向上或向下以设置首选项目构建顺序。
  5. 单击确定以关闭“首选项”对话框。
