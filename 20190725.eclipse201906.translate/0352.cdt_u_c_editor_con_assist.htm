内容辅助首选项

使用 内容辅助面板，用于在C / C ++编辑器中自定义内容辅助功能。

内容辅助首选项选项
选项 描述
自动插入单个提案 当内容辅助功能只找到一个提案时，会在代码中插入一个元素。
自动插入公共前缀 插入公共前缀。
完成提案过滤器 您的Eclipse CDT发行版可能已提供完成建议过滤器。完成提案过滤器可以对内容辅助功能提供的提案进行排序和排序，以最好地满足您的需求。从可用过滤器列表中选择所需的内容辅助过滤器。
启用“。”作为触发器 启用“。”作为自动完成的触发器。
启用“ - >”作为触发器 允许使用“ - >”作为自动完成的触发器。
启用“::”作为触发器 允许使用“::”作为自动完成的触发器。
延迟（ms） 指定激活内容辅助之前的毫秒数。
