代码模板首选项

该 Java>代码样式>代码模板首选项页面允许您配置新生成的代码和注释的格式。

代码和评论

代码和注释页面包含生成代码的操作使用的代码模板。模板包含应用模板时替换的变量。某些变量在所有模板中都可用，有些变量特定于模板。

代码模板

模板名称

描述

新的Java文件

创建新Java文件时由New Type向导使用。

模板可以指定添加注释的位置。请注意，模板可以包含变量$ {typecomment}和$ {filecomment} ，它们将由Types Types Files注释模板的评估替代。

类体

创建新Java类时由New Class向导使用。

界面体

创建新的Java接口时，由New Interface向导使用。

枚举身体

创建新的Java枚举时，由New Enum向导使用。

注释体

创建新的Java注释时，由New Annotation向导使用。

方法体

当创建具有主体的新方法仍然需要一些代码来完成其功能时，使用“方法主体”模板。

它包含变量$ {body_statement} ，它解析为return语句或/和超级调用。

构造体

当创建具有body的新方法或构造函数时，将使用“构造函数体”模板。

它包含解析超级调用的变量$ {body_statement} 。

吸气体

创建新的getter方法时使用'Getter body'模板。

它包含变量$ {body_statement} ，该变量解析为相应的return语句。

二传手身体

创建新的setter方法时使用'Setter body'模板。

它包含变量$ {body_statement} ，它解析为适当的赋值语句。

抓住块体

创建catch块体时使用'Catch块体'模板。

它可以使用变量$ {exception_type}和$ {exception_var} 。
