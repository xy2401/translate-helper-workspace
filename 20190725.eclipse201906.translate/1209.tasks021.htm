版本映射

使用版本映射指定用于乐观锁定的字段。如果实体与多个表关联，则应仅使用主表的版本映射。每个持久性实体应该只有一个版本映射。版本映射只能与以下属性类型一起使用：

  • INT

  • 整数

  • 简短，短

  • 很久很久

  • 时间戳

要创建版本映射：

  1. 在JPA Structure视图中 ，选择要映射的字段。

  2. 右键单击该字段，然后选择“ 映射为”>“版本” 。JPA详细信息视图（用于属性）显示所选的属性。

    JPA详细信息，版本映射


  3. 完成“ 版本映射”区域中的每个字段。

  4. 完成JPA详细信息视图中的其余区域（对于属性） ：

    • 输入信息

    • 转换器

Eclipse在该字段中添加以下注释：

@Version @Column（table =“ <COLUMN_TABLE> ”，name =“< COLUMN_NAME >”）
