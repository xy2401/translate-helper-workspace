同步类

在处理Java项目中的类时，需要更新persistence.xml文件以反映更改。

使用此过程同步persistence.xml文件：

  1. 右键单击Project Explorer中的persistence.xml文件，然后选择JPA Tools> Synchronize Class List 。


    注意：

    如果选择了此功能，则必须在JPA Facet页面 的persistence.xml选项中列出带注释的类 。通常，您不必在容器中使用此功能。

    同步persistence.xml文件


    Dali将必要的<class>元素添加到persistence.xml文件中。

  2. 使用persistence.xml编辑器继续编辑persistence.xml文件。
