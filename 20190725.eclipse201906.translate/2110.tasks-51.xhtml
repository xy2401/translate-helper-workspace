将编辑器与文件类型相关联

要将编辑器与Workbench中的各种文件类型相关联：

  1. 打开 常规>编辑器>文件关联首选项页面。
  2. 从“ 文件类型”列表中选择文件类型 ，或单击“ 添加”以添加列表中尚未包含的类型。
  3. 在“ 关联的编辑器”列表中，选择要与该文件类型关联的编辑器。要将编辑器添加到列表中：
    1. 单击添加 。将打开“编辑器选择”对话框。
    2. 选择“ 内部编辑器”或“ 外部程序” ，具体取决于所需的编辑器是为Workbench构建还是在Workbench外部运行。
    3. 如果选择“ 外部程序” ，则可以单击“浏览”按钮浏览文件系统。
    4. 从列表中选择编辑器，然后单击“ 确定” 。
  4. 单击“ 确定”完成将编辑器与所选文件类型相关联。

将内部编辑器与文件类型相关联时，该编辑器将在Workbench的编辑器区域中打开。例如，如果双击Project Explorer中的文件或“书签”或“任务”视图中的条目，则会在编辑器区域中打开该文件。

支持OLE文档模式的编辑器也可以在Workbench的编辑器区域中运行。

提示 ：您可以选择覆盖默认编辑器选择，方法是从其中一个导航视图中的任何资源的弹出菜单中选择“ 打开方式” 。
