文件系统上的布局

  • 在本节中，您将创建一个新的Java项目，并在单独的文件夹中组织您的源。这将为您处理更复杂的布局做好准备。
  • 假设您想将源代码放在一个文件夹中，并将测试放在另一个文件夹中：

定义相应项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“MyProject”。

  4. 在项目布局组中，将选择更改为创建单独的源和输出文件夹，然后编辑配置默认...以将源文件夹名称从“src”修改为“sources”。

  5. 单击“ 确定”返回“新建Java项目”向导，然后单击“ 下一步” 。

  6. 要向项目添加其他源文件夹，请单击“ 详细信息”窗格或按钮中的“ 创建新源文件夹”链接在视图栏中。

  7. 在“ 新建源文件夹”对话框的“ 文件夹名称”字段中，键入“tests”。

  8. 单击“ 完成”以验证并关闭对话框。

  10. 单击完成
  11. 您现在有一个包含两个源文件夹的Java项目： sources和tests 。您可以开始向这些文件夹添加类，也可以使用拖放操作来复制它们。
