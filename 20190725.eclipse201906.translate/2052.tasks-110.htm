发现分支和版本标签

先决条件：在您发现存储库的分支和版本标记之前，您必须在CVS存储库视图中具有有效的存储库位置。

要在单个存储库位置中发现多个文件夹的现有分支标记：

  1. 切换到 CVS Repository探索透视图或添加 CVS存储库查看当前透视图。
  2. 在“CVS存储库”视图中，选择存储库位置。
  3. 从CVS Repositories视图的弹出菜单中，选择Refresh Branches 。“刷新分支”对话框将打开（根据所选的列出的远程项目所选的存储库位置可能不同）：

  4. 在文件夹列表中，选择要刷新其标签的文件夹。或者，在对话框的底部，您可以选择工作集，并且将检查名称与工作集中项目名称匹配的所有远程文件夹。
  5. 单击“ 完成”以发现已检查项目的标记。这些标记将添加到存储库视图中。

注意：此操作可能会长时间运行，尤其是当所选文件夹的数量很大时。

提示 ：如果上述操作不会导致发现任何标记，则存储库中的文件夹可能没有.project文件。.project文件是标记发现过程使用的默认文件。可以使用“ 配置分支和版本” （参见下文）为各个项目更改该文件。

要发现单个项目文件夹的现有分支和版本标记：

  1. 切换到 CVS Repository探索透视图或添加 CVS存储库查看当前透视图。
  2. 在“CVS存储库”视图中，展开存储库位置。
  3. 展开HEAD并选择要发现其标记的根文件夹。
  4. 从“CVS存储库”视图的弹出菜单中，选择“ 配置分支和版本” 。将打开与下面类似的“标记配置”对话框。

  5. 在左侧的顶部窗格（标题为标签的浏览文件 ）中，展开一个或多个项目以查找已知具有应添加到CVS存储库视图的标记的文件。
  6. 在左上方窗格中选择一个文件。如果所选文件上定义了标签，则它们将显示在右上方窗格中（标题为在所选文件中找到的新标签 ）。
  7. 根据需要选中或取消选中右窗格中的标签。
  8. 单击“ 添加选中的标签” ，将选中的标签添加到记忆标签列表中。
  9. 单击“ 确定”将记住的标记添加到选定的存储库位置或项目。

注意 ：您还可以通过在文件窗格中选择文件并单击对话框右下角的“ 添加所选文件”来设置自动刷新文件。
