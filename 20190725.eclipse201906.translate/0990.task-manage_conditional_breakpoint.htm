管理条件断点

条件表达式可以应用于行断点或监视点，以便断点在以下某种情况下暂停执行线程：

  • 当表达式的结果为真时
  • 当表达式的结果发生变化时

条件表达式可以包含任意Java代码，并且可以包含多个语句，允许断点条件实现跟踪等功能。例如，条件可以执行print语句，然后返回硬编码值以永不挂起（“ System.out.println(...); return false; ”）。

要在断点上设置条件：

  1. 找到要应用启用条件的断点 （在“ 断点”视图中或在编辑器标记栏中）。
  2. 从断点的弹出菜单中，选择Breakpoint Properties ....将打开“断点属性”对话框。
  3. 在属性对话框中，选中“ 启用条件”复选框。
  4. 在Condition字段中输入断点条件的表达式。
  5. 执行以下操作之一：
    • 如果希望断点在每次条件计算结果为true时停止，请选择条件为'true'选项。提供的表达式必须是布尔表达式。
    • 如果希望断点仅在条件更改结果时停止，请选择条件更改选项的值 。
  6. 选择“ 确定”关闭对话框并提交更改。启用断点时，如果断点条件的计算结果为true，则在执行该行代码之前，线程执行将暂停。

条件断点在断点图标上有一个问号覆盖。
