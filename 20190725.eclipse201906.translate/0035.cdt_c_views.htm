C / C ++视图

C / C ++视图是帮助您执行管理C / C ++程序所涉及的任务的面板。可以将这些视图添加到Eclipse窗口的透视图中。

改变观点：
  • 您可以向透视图添加视图。从菜单栏中选择“ 窗口”>“显示视图”，然后从“显示视图”对话框中的一个视图类别中选择一个新视图。
  • 要将当前透视图重置为其原始布局，请从菜单栏中选择“ 窗口”>“重置透视图” 。

以下视图通常用于C / C ++透视图：

Project Explorer
显示工作空间目录下的所有文件系统文件。
安慰
显示程序的输出以及构建工具的输出。
编辑
C / C ++编辑器视图提供了用于编辑C / C ++相关文件的专用功能。
大纲
显示当前在编辑器中打开的文件的结构。
问题
如果在构建期间遇到任何错误，它们将显示在“问题”视图中。
属性
显示当前在视图或编辑器中选择的项目的属性。
任务
列出要跟踪的任务，可以是要做的事情安排，也可以是已完成事情的历史记录。
