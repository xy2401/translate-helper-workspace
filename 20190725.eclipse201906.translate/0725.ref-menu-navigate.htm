导航动作

导航菜单命令：

名称

功能

 键盘快捷键

进入

将视图输入设置为当前选定的元素。Package Explorer视图支持。

 

去
  • 后退：在历史记录中将视图输入设置回输入：仅在存在历史记录时启用（使用Go Into ）
  • 转发：在历史记录中将视图输入设置为输入转发：仅在历史存在时启用（使用“进入” ，“ 转到”>“返回” ）
  • Up One Level：将当前视图的输入设置为其输入的父元素
  • 键入：浏览类型并在当前视图中显示它。Package Explorer视图支持
  • 包：浏览包并在当前视图中显示它。Package Explorer视图支持
  • 资源：浏览资源并在当前视图中显示它。
 

公开宣言

尝试解析当前代码选择中引用的元素，并打开声明引用的文件。

F3

打开“类型层次结构”

尝试解析当前代码选择中引用的元素，并在“ 类型层次结构”视图中打开该元素。在元素上调用，打开元素的类型层次结构。在Java编辑器和支持Java元素的视图中受支持。

F4

打开调用层次结构

尝试解析当前代码选择中引用的方法，并在“ 调用层次结构”视图中打开该元素。

按Ctrl + Alt + H

开放实施

如果只有一个实现者，则打开当前所选方法或类型的实现，否则在其上打开快速类型层次结构。

 

开放超级实施

打开一个编辑器，用于围绕当前光标位置的当前所选方法或方法的超级实现。如果未选择任何方法或方法没有超级实现，则不会打开编辑器。

打开类型 打开“打开类型”对话框以在编辑器中打开类型。“打开类型”选择对话框显示工作空间中存在的所有类型。 CTRL + SHIFT + T
打开类型层次结构 打开“打开类型”对话框以在编辑器和“ 类型层次结构”视图中打开类型。“打开类型”选择对话框显示工作空间中存在的所有类型。 按Ctrl + Shift + H
