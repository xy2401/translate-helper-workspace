编辑目标向导

该向导用于编辑目标定义 。当编辑目标平台首选项页面上的目标定义或创建新定义时，它将打开。该向导提供与目标编辑器相同的功能。

可以为目标定义指定一个名称，该名称将在目标平台首选项页面上显示时使用。其他目标设置分为多个选项卡：

  1. 位置 - 提供将提供插件包含在目标中的目录，安装和站点
  2. 内容 - 允许在位置中找到的插件子集包含在目标中。提供了许多过滤选项，用于快速编辑内容
  3. 环境 - 影响插件构建和运行方式的操作系统，窗口系统，执行环境和其他环境设置
  4. 参数 - 指定使用此目标启动时要使用的默认程序和vm参数
  5. 隐式依赖项 - 将一个或多个插件添加为隐式依赖项。在PDE中，当有一个选项可以添加所需的插件时（例如在此向导的“内容”选项卡上），任何标记为隐式依赖项的插件将始终视为必需

按Finish将保存对目标定义的更改并关闭向导。

解决目标

在可以编辑目标的内容之前，可以在将目标定义设置为活动目标平台之前，必须解析它。解析目标定义后，将搜索定义的位置以查找插件。根据目标中的位置类型，这可能涉及搜索目录，读取安装的元数据，读取功能或从远程站点下载。在解析过程中，编辑目标向导将被禁用，并将显示进度。您可以取消该工作，但是您将无法编辑目标的内容。

有关最新目标故障排除帮助，请参阅目标定义Wiki页面 。
