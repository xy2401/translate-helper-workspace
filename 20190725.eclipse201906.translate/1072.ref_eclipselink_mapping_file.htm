新的EclipseLink映射文件对话框

指定EclipseLink映射文件（ eclipselink-orm.xml ）的位置和属性。

属性 描述

项目

选择要添加映射文件的项目。

源文件夹

单击“ 浏览”并选择要添加映射文件的源文件。默认值为../ <PROJECT> /src 。

文件路径

输入映射文件的文件名和路径。默认值为META-INF/eclipselink-orm.xml 。

默认访问权限

选择实体对实例变量的访问是基于字段的还是基于属性的，如JPA规范中所定义。

添加到持久性单元

指定是否应将此映射文件添加到持久性单元（ persistence.xml ）。

持久单位

选择要添加映射文件的持久性单元。
