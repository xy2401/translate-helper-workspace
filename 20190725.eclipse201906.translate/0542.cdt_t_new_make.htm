创建一个makefile

如果您已创建Standard Make C / C ++项目，则需要提供makefile。

构建项目时，make的输出显示在Console视图中。Makefile操作显示在Make Targets视图中。

要创建一个makefile：

  1. 在“ C ++项目”视图中，右键单击项目，然后选择“ 新建”>“文件” 。
  2. 在“ 文件名”框中，键入makefile 。
  3. 在项目列表中，验证是否选择了正确的项目。
  4. 单击完成 。
  5. C / C ++编辑器打开。在C / C ++编辑器中键入makefile指令。
  6. 单击文件>保存 。
