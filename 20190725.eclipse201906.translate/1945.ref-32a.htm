CVS历史记录视图

此视图提供存储库中资源的所有修订的列表以及本地历史记录中资源的所有修订。在此视图中，您可以比较修订版，加载修订版，将相应的工作区文件还原为修订版，使用现有标记标记，显示注释并在修订版上打开编辑器。

以下是CVS历史视图的外观：

列

历史记录视图可以显示本地和远程修订。本地历史版本具有与之相关的最少信息 - 只有时间戳。因此，对于“本地历史记录”修订，大多数列项将显示为空白。另一方面，远程修订将具有更完整的信息集。以下是列：

调整

由于CVS历史记录视图可以显示远程和本地修订。对于远程修订：

  • 该列显示远程修订图标：
  • 该列显示历史记录中的修订号。
对于本地历史修订：
  • 该列显示本地历史记录修订图标：

分行

此列显示可用修订的分支列表。（仅限远程修订）

标签

与修订关联的标记。选择修订行将在视图的左下窗格中列出标记。（仅限远程修订）

修订时间

此列显示历史记录中修订的创建日期和时间。

作者

此列显示创建和创作版本的人员的姓名。（仅限远程修订）

评论

此列显示在提交时为此修订提供的注释（如果有）。选择修订行将在视图的右下方窗格中显示完整注释。此外，将鼠标悬停在虚线注释上将导致显示带有完整注释的工具提示。（仅限远程修订）

工具栏

刷新

此命令刷新视图的内容，从服务器获取资源的最新历史信息。

与编辑和选择链接

启用后，视图将显示活动编辑器或活动选择的资源的历史记录。

销

启用后将固定视图及其内容。任何新的历史请求都将打开历史记录视图的新实例。

导航历史

启用后，您可以从下拉菜单中选择过去查看过历史记录的资源。首次打开“历史记录”视图时，将禁用该操作。

按日期进行的集团修订

启用后，所有历史记录项将分类为以下日期类别之一：

  • 今天
  • 昨天
  • 这个月
  • 以前

本地和远程修订

启用后，将在“历史记录”视图中显示资源的本地和远程修订。

本地修订

启用后，将仅在“历史记录”视图中显示资源的本地修订。

远程修订

启用后，将仅在“历史记录”视图中显示资源的远程修订。

全部收缩

如果视图处于按日期分组模式，则“全部折叠”将折叠所有日期类别。

比较模式

启用后，双击（或单击一次，具体取决于“首选项”>“常规”中设置的“打开模式”策略）将打开比较编辑器。禁用时，单击修订将打开该修订。

下拉式菜单

下拉菜单包含以下项目：

  • 换行注释 ：将在评论查看器中包含注释。
  • 显示评论查看器 ：显示评论查看器。
  • 显示标签查看器 ：显示标签查看器。
  • 显示搜索字段 ：显示搜索字段。
  • 过滤.... ：打开过滤器对话框。
  • 删除过滤器 ：从历史记录视图中删除任何过滤。

上下文菜单

从CVS资源历史记录视图的上下文菜单中，您可以执行许多有趣的操作。

打开

此命令将在只读编辑器中打开所选修订的内容。（使用的编辑器是注册为文件类型的默认编辑器的编辑器）。 注意 ：如果要打开的修订版是文件的当前版本，则它将在常规编辑器中打开。

打开用

此菜单允许选择编辑器以打开所选修订的内容。 注意 ：如果要打开的修订版是文件的当前版本，则它将在常规编辑器中打开，否则将使用只读编辑器。

比较电流与修正/相互比较

compare命令根据历史记录视图中选择的修订数量而有所不同。对于单个选择， Compare Current With Revision将比较当前版本的资源与所选修订版本。对于两个选项，“相互比较”将比较所选的修订。

获取内容

此命令将所选修订的内容加载到其历史记录显示在视图中的文件的本地副本中。然后可以提交本地文件，使HEAD（或分支）中的最新修订内容与所选修订的内容相匹配。

获得粘性修订

此命令将在显示其历史记录的本地文件上加载所选修订。很少需要此操作，主要用于更改一个或多个资源上的标记。加载的修订版不能随后提交但可以标记。

标记为现有...

此命令允许将现有标记移动到所选文件修订版。

显示注释

此命令显示带有注释的文件内容，该注释标识文件中每行代码的作者。

刷新视图

此命令刷新视图的内容。
