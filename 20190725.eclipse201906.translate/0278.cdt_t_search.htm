搜索C / C ++元素

独立调试器提供了搜索应用程序源代码的功能。为此，调试器使用用于构建应用程序的包含路径和宏来索引源代码。如果您的应用程序是使用默认的包含路径构建的，并且没有其他用户定义的标志，那么您无需做任何准确的搜索工作。如果您的应用程序在构建中指定了其他包含路径和用户定义的标志，则需要使用-g3标志来包含宏信息，否则您需要为独立调试器提供构建的副本登录。可以使用-b选项或通过File> New Executable ...对话框在命令行上提供构建日志。如果未正确设置，搜索，类型层次结构和悬停显示等各种操作可能不准确。

有关以下内容的更多信息，请参阅C / C ++搜索 ：

  • 你可以搜索什么
  • 如何限制您的搜索
  • 如何在搜索中使用通配符
  • 语法示例

启动C / C ++搜索有三种主要方式：

  • 使用“ C / C ++搜索”对话框
  • 在“ 编辑器”视图中选择元素
  • 在“ C / C ++项目”视图中选择元素或从“ 大纲”视图中选择元素


使用“C / C ++搜索”对话框

  1. 在“ 搜索字符串”字段中输入搜索字符串（可选标记为区分大小写）。
    注意：以前的搜索查询（来自同一工作会话）会被记住，可以通过下拉列表进行访问。
  2. 选择“ 搜索”元素。
  3. 选择限制到 。
  4. 选择范围 。
  5. 按搜索 。

结果显示在“ 搜索”视图中。

在“编辑器”视图中选择元素

  1. 在编辑器中选择所需的元素。
  2. 右键单击并选择“ 所有声明”或“ 所有引用”以及要搜索的范围。

结果显示在“ 搜索”视图中。

在C / C ++项目或大纲视图中选择元素

  1. 在树中选择所需的元素。
  2. 右键单击并选择“ 所有声明”或“ 所有引用”以及要搜索的范围。

结果显示在“ 搜索”视图中。
