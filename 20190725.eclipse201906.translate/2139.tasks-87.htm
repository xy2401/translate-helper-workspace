用本地历史记录替换资源

要使用本地历史记录中的状态替换非托管Workbench资源，请执行以下操作：

  1. 在其中一个导航视图中，选择要用本地历史状态替换的资源。
  2. 从资源弹出菜单中，选择替换为 > 本地历史记录 。将打开“从本地历史记录替换”页
  3. 从“ 本地历史记录”列表中选择一个州。文本比较编辑器打开。
  4. 单击“ 选择下一个更改”和“ 选择上一个更改”按钮以浏览本地历史记录和Workbench资源中的状态之间所做的更改。
  5. 选择要替换的状态，然后单击“ 替换” 。

要将CVS管理的Workbench资源替换为本地历史记录中的状态：

  1. 在其中一个导航视图中，选择要用本地历史状态替换的资源。
  2. 从资源弹出菜单中，选择替换为 > 历史记录....历史记录视图打开。
  3. 单击“本地修订”工具栏按钮，将“历史记录视图”置于“ 本地修订”模式。这将仅在历史记录视图中显示本地历史记录修订。
  4. 选择要替换当前修订的修订，然后从上下文菜单中选择“ 获取内容 ”。

提示 ：您可以配置常规首选项以指定保留文件的天数，或每个文件要保留的条目数，或者保留文件的最大文件大小。 常规>工作区>本地历史记录首选项页面。
