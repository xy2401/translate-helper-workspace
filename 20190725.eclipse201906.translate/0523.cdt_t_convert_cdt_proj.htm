转换CDT 1.x项目

如何将CDT 1.x项目更新为当前CDT项目格式取决于项目是Standard Make项目还是Managed Make项目。有关标准制作项目，请参阅下面的转换为C / C ++项目向导 。

更新托管生成项目

对于Managed Make项目，Managed Build系统将在尝试读取Managed Build项目信息并发现项目需要更新时提示您转换项目。系统将提示您更新项目。

单击是 ，项目将更新。CDT 1.x或CDT 2.x无法再加载更新的托管构建信息。选择否 ，项目将不会更新。您将能够查看托管构建设置，但在关闭项目或退出Eclipse时不会保存任何更改。

转换为C / C ++项目向导

选择“ 文件”>“新建”>“转换为C / C ++项目” 。如果该选项不可用，您可以在此处找到添加它的说明。

从Convert to C / C ++ Make Project Wizard中选择要转换的项目，然后单击Finish 。

注意：您可能需要为CDT 1.x Standard Make项目手动启用Path Discovery，具体取决于CDT项目的配置方式。有关详细信息，请参阅Set Discovery Options 。
