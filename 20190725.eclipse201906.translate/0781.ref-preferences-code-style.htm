Java代码样式首选项

该 Java>代码样式首选项页面允许配置命名约定，样式规则和注释设置。必须生成新代码时使用这些首选项。

子页面提供更多代码样式设置：

  • 清理
  • 代码模板
  • 格式化
  • 组织进口

选项是：

命名约定

该列表定义了字段（静态和非静态），参数和局部变量的命名约定。对于每种变量类型，可以配置前缀或后缀或两者的列表。
命名约定由创建字段，参数和局部变量的所有操作和“快速修复”提议使用，特别是源操作 。

代码约定

以下设置指定新生成的代码的外观。可以指定getter方法的名称以及字段访问，方法注释，注释和异常变量的格式。
