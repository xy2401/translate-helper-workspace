设置源文件夹

请注意，源文件夹目前只能与Standard Make项目一起使用。Managed Make项目将整个项目视为源文件夹。

源文件夹是一种减少项目索引范围的方法。您可以标记作为日常工作的一部分的文件夹或您工作的子系统的一部分。源文件夹中的所有文件都将被编入索引，因此可以搜索。请注意，源文件夹中的文件引入的任何文件也将被编入索引。

要设置源文件夹：

  1. 右键单击该项目，然后选择“ 属性”>“C / C ++项目路径”
  2. 单击“ 源”选项卡
    默认情况下，整个项目是一个源文件夹，这意味着所有内容都将被编入索引。这对于较小的项目来说是合理的，但绝对不建议用于大型项目。
  3. 单击“ 添加文件夹”以添加文件夹。对话框将说明已将嵌套过滤器添加到嵌套文件夹中。您将看到您添加的文件夹将从项目文件夹中排除（为了避免包含两次文件夹）。重复，直到添加完所有文件夹。
    注意：不要忘记删除项目文件夹（默认情况下显示），否则将对所有内容编制索引。
  4. 单击确定 。您在C / C ++项目中的视图现在将发生变化。您应该看到使用“C”指定的源文件夹，并且非源文件夹中的所有其他源和头文件图标将更改为“空心”C或H.
