共同源代码树中的兄弟产品

文件系统上的布局

  • 产品的源文件放在一个大的目录中，该目录是在Eclipse之外管理的版本和配置。
  • 源目录包含两个兄弟目录Product1和Product2 。

定义相应项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“Product1”。单击下一步 。

  4. 在下一页上，选择“Product1”源文件夹。

    单击详细信息窗格或按钮中的链接其他源链接在视图栏中。

  5. 在Link Source对话框中，单击Browse ....并选择D:\AllProducts\Product1\JavaSourceFiles目录。

  6. 单击“ 完成”以验证并关闭对话框。

  7. 您的项目源设置现在如下所示：

  8. 单击完成 。

  9. 对“Product2”重复步骤2到8。

  10. 您现在有两个Java项目，分别包含“Product1”和“Product2”的源。
