在Package Explorer视图中打开一个Type

您可以在项目的类路径中包含的任何类型上打开Package Explorer。

  1. 从菜单栏中，选择“ 导航”>“转到”>“类型”。 将转到“转到类型”对话框。
  2. 在“ 选择类型”字段中，根据需要使用通配符开始键入表达式以缩小可用类型列表。键入时，列表将被过滤以仅显示与当前表达式匹配的类型。
  3. 在“ 匹配项目”列表中，选择一种类型。提示：您可以按向下键移动到第一种类型。
  4. 完成后单击“ 确定” 。所选类型显示在Package Explorer中。

注意： “转到类型”对话框维护最近打开的类型的历史记录。这些在打开对话框时显示，并在您开始键入过滤器表达式时保持在分隔线上方。

注意：如果应用了Package Explorer过滤器，则可能无法显示。
