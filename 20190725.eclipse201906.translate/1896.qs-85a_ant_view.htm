使用Ant视图

Eclipse提供了一个标准视图Ant视图 ，它允许您使用Ant构建文件。此视图是树形结构，将Ant构建文件显示为顶级条目，将目标和内部目标显示为子级。此视图的主要优点是，您可以在一个位置使用所有Ant构建文件，而不是在其中一个导航视图中搜索它们。

  1. 通过选择“ 窗口”>“显示视图”>“Ant”，从工作台菜单中打开Ant视图 。
  2. 默认情况下，Ant视图为空。有三种方法可以将Ant构建文件添加到此视图中：
    • 单击添加构建文件[ ] 。这将打开一个对话框，您可以在其中显式选择要添加的Ant构建文件
    • 单击“ 使用搜索添加构建文件”[ ] 。这将打开一个搜索对话框，您可以在其中指定Ant构建文件的文件名模式，并在整个工作区或指定的工作集中进行搜索。
    • 将构建文件从其他视图拖放到Ant视图 。

      一旦添加到Ant视图，Ant构建文件条目将保留在工作台调用的视图中，直到显式删除或关闭相关项目。
  3. 单击“ 使用搜索添加构建文件” 。假设您只记得您要使用的构建文件以H开头。输入H*.xml作为构建文件名。确保为范围选择了Workspace ，然后单击“ 搜索” 。找到HelloWorld.xml文件并将其放在Ant视图中 。
  4. 展开顶级条目以查看默认目标Hello和内部目标World 。



  5. 选择World internal target并单击Run the Selected Target [ ] 。请注意，只有World目标才会被执行。
  6. 选择顶级HelloWorld构建文件，然后单击“运行所选构建文件的默认目标” ] 。请注意，只执行默认目标Hello 。
  7. 要编辑构建文件，请打开HelloWorld文件上的上下文菜单，然后选择Open With> Ant Editor 。
  8. 要编辑默认启动配置，请从上下文菜单中选择Run As> Ant Build ....
  9. 将显示“运行Ant”启动配置对话框。在这里，您可以从Ant视图修改构建文件的运行方式 。
  10. 选择HelloWorld文件，然后单击“ 删除”按钮。构建文件将从视图中删除。
    注意 ：这不会从工作区中删除该文件。
