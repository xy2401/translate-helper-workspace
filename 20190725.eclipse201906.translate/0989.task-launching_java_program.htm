启动Java程序

启动Java程序的最简单方法是使用Java Application启动配置运行它。此启动配置类型使用从工作台首选项和程序的Java项目派生的信息来启动该程序。

  1. 在Package Explorer中 ，选择要使用要启动的main方法的Java编译单元或类文件。
  2. 按运行 [ 工作台工具栏中的]按钮或从工作台菜单栏中选择“运行”>“运行” 。另外，在Package Explorer弹出菜单中选择Run As> Java应用程序 ，或者在工作台菜单栏中选择运行>运行方式> Java应用程序 ，或者在运行工具栏上的下拉菜单中选择Run As> Java应用程序按钮。
  3. 您的程序现已启动，文本输出显示在控制台视图中 。

您还可以通过选择项目而不是编译单元或类文件来启动Java程序。系统将提示您从定义main方法的类中选择一个类。（如果在项目中只找到一个带有main方法的类，则会启动该类，就像您选择它一样。）
