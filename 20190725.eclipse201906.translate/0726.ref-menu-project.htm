名称

功能

 键盘快捷键

打开项目

显示一个对话框，可用于选择已关闭的项目并将其打开

  

关闭项目

关闭当前选定的项目

  

全部构建

在工作区中构建所有项目。这是一个增量构建，这意味着构建器会分析自上次构建以来的更改，并最大限度地减少已更改文件的数量。

Ctrl + B.

建设项目

构建当前选定的项目。这是一个增量构建，这意味着构建器会分析自上次构建以来的更改，并最大限度地减少已更改文件的数量。

 

建立工作集

构建当前所选工作集中包含的项目。这是一个增量构建，这意味着构建器会分析自上次构建以来的更改，并最大限度地减少已更改文件的数量。

 

清洁...

显示一个对话框，可在其中选择要清理的项目。

 
自动构建 如果选中，则保存时会自动重建所有已修改的文件。这是一个增量构建，这意味着构建器会分析自上次构建以来的更改，并最大限度地减少已更改文件的数量。  
生成Javadoc ...... 在当前选定的项目上打开Generate Javadoc向导。  
属性 打开当前所选项目的属性页面。
