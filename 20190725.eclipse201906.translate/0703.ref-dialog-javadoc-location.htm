Javadoc位置对话框

此对话框允许您为JAR或Java项目定义Javadoc文档的位置。

您可以通过以下方式访问此对话框：

  • 选择JAR或Java项目，打开上下文菜单，然后选择“属性”>“Javadoc位置”或使用“ 文件”菜单中的“ 属性 ”
  • 在Javadoc生成向导的“标准doclet设置”页面上，选择“ 配置”

Javadoc可以附加到JAR，类文件夹或Java项目。对于项目，它记录了所有源文件夹的元素，对于JAR和类文件夹，记录了JAR中包含的元素。该位置使用

  • 在打开附加的Javadoc 导航菜单来打开一个元素的连接的Javadoc
  • 上下文帮助（F1）指向Javadoc位置
  • Javadoc导出向导链接到其他文档或作为项目文档的默认目标

有效位置是指向包含API文档的index.html和package-list文件的文件夹的URL。例如：

文件：/// L：/JAVA/JDK1.2/DOCS/API/
http://download.oracle.com/javase/6/docs/api/

选项

描述

默认

Javadoc URL

指定生成的Javadoc文档的位置。您可以在本地文件系统中浏览 Javadoc位置（将生成file：// URL）

 <空>

验证

通过尝试使用给定的URL访问index.html和package-list文件来验证当前位置。如果验证成功，您可以直接打开文档。

 

JAR和类文件夹的Javadoc也可以位于本地文件系统上可用的存档内。在这种情况下，必须指定存档的位置以及存档中文档的位置。

选项

描述

默认

存档位置

指定包含生成的Javadoc文档的存档的位置。您可以在本地文件系统中浏览 Javadoc位置

 <空>

存档中的路径

指定存档中包含生成的Javadoc文档的路径。您可以浏览以查看存档的内容。

<空>

验证

通过尝试使用给定的URL访问index.html和package-list文件来验证当前位置。如果验证成功，您可以直接打开文档。
