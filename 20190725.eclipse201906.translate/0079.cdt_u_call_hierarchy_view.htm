调用层次结构视图

该 “调用层次结构”视图显示所选C / C ++函数的已加载调用方和被调用方。右键单击源文件中的函数名称，然后选择“ 打开调用层次结构”以打开“调用层次结构”视图（如果没有打开），并显示函数的调用者。

调用层次结构视图工具栏命令

“调用层次结构”视图工具栏命令包括：

命令 名称 描述
显示下一个参考 选择下一个功能。
显示上一个参考 选择上一个功能。
隐藏变量，常量和枚举器 切换以显示或隐藏视图中的所有变量，常量和枚举器。
显示来电者 显示所选功能的搜索范围内的所有呼叫者。
显示被叫 显示当前所选功能调用的所有成员。
显示历史列表 此菜单显示先前显示的呼叫层次结构的历史记录。
刷新视图内容 刷新视图以反映函数的当前状态。
固定调用层次结构视图 固定当前视图并使用户能够同时打开多个“调用层次结构”视图。
查看菜单 提供以下命令：
  • 使用工作集
  • 显示来电者或被叫者
  • 显示或隐藏文件名
  • 激活或停用过滤器

调用层次结构视图上下文菜单命令

“调用层次结构”视图上下文菜单命令包括：

名称 描述
