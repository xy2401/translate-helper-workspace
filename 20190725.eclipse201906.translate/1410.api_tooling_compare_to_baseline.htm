与API基准相比

您工作区中启用的任何API Tools项目都可以与工作区中设置的任何一个API基线进行比较。除了选择要与基线进行比较的项目之外，您还可以在启用API工具的项目中选择（或多选）以下任何内容：

  • 源文件夹
  • 包裹碎片
  • 编制单位

注意：必须从Package Explorer中进行所有选择，因为比较向导仅适用于Java元素。


要与API基准进行比较，请执行以下操作：

  1. 使用上述标准选择要比较的元素。
  2. 在比较向导中选择要比较的API基线
  3. 单击“ 完成” ，即可完成设置。比较的任何结果都将显示在API工具视图中。
