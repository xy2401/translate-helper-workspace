创建C / C ++文件

文件在C / C ++编辑器中编辑，默认情况下，该编辑器位于Project Explorer或C / C ++ Projects视图右侧的编辑器区域中。

C / C ++编辑器左边缘的标记栏显示错误，警告，书签，断点和任务的图标。

有关标记栏的更多信息，请参见“ 工作台用户指南”>“参考”>“用户界面信息”>“视图和编辑器”>“编辑器”区域 。

要创建C ++文件：

  1. 在Project Explorer或C ++ Projects视图中，右键单击项目，然后选择New> File 。
  2. 在项目列表中，验证是否选择了正确的项目。
  3. 在“ 文件名”框中，键入名称，后跟相应的扩展名。
  4. 单击完成 。


    该文件将在C / C ++编辑器中打开。

  5. 在编辑器视图中输入您的代码。
  6. 键入CTRL + S以保存文件。
