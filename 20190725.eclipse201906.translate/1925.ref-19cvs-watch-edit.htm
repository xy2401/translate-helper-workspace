选项

描述

默认
配置要在结帐时使用Watch / edit的项目 使用此选项可指示从存储库中签出的文件应设置为只读。 残
在编辑时启用临时监视 使用此选项可在用户开始编辑您当前正在编辑的文件时收到通知。 残
在编辑器中修改只读文件时 使用此选项可配置在打开的编辑器或其他工具中修改只读文件时发生的情况。选项是：
  • 向服务器发送cvs编辑通知：在使文件可写之前向服务器发出cvs编辑通知。如果文件中存在其他编辑器，则会提示您，可以继续或取消。
  • 在后台向服务器发送cvs编辑通知：在使文件可写后，在后台向服务器发出cvs编辑通知。这使您可以在将编辑通知发送到服务器时不间断地键入内容。
  • 编辑文件而不通知服务器：在不通知服务器的情况下使文件成为只读文件。
 在后台向服务器发送cvs编辑通知
在将CVS编辑通知发送到服务器之前 使用此选项可配置需要将cvs编辑通知发送到服务器时发生的情况。选项是：
  • 始终提示：始终提示确认。
  • 仅在有其他编辑器时才会提示：向用户显示当前编辑器的列表，并允许您确认或取消编辑。
  • 从不提示：在不提示的情况下发送编辑通知
 只有在有其他编辑器时才会提示
更新已编辑的文件 使用此选项可配置编辑文件之前发生的情况。选项是：
  • 始终在编辑前更新：始终在编辑前更新文件。
  • 如果过期则提示更新：仅在文件过期时提示确认更新。
  • 永不更新：切勿在编辑前更新文件。
 永远不要更新
