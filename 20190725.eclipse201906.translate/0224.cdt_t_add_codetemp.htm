创建和编辑模板

内容辅助使用模板使您能够快速使用常用的代码段。

要创建模板：

  1. 单击窗口>首选项 。
  2. 展开C / C ++ ，展开编辑器并选择模板 。
  3. 单击新建 。




  4. 执行以下操作:(如果需要）

    • 在“ 名称”字段中，键入模板名称。
    • 在“ 上下文”下拉列表中，选择新模板的上下文类型;无论是C / C ++还是评论 。
    • 在“ 描述”字段中，键入新模板的描述。
    • 在“ 模式”字段中，键入此模板的代码。
    • 单击“ 插入变量”将列表中的变量添加到您在“ 模式”字段中输入的代码中。
  5. 单击确定 。
    新模板显示在“模板”列表中。

要编辑模板：

  1. 单击窗口>首选项 。
  2. 展开C / C ++ ，展开编辑器并选择模板 。
  3. 单击编辑 。“编辑模板”对话框打开。




  4. 请执行下列操作：

    • 在“ 名称”字段中，更改模板名称以基于当前模板创建新模板。
    • 在“ 上下文”下拉列表中，选择新模板的上下文类型;无论是C / C ++还是评论 。
    • 在“ 描述”字段中，更改代码模板的描述以反映您的更改。
    • 在Pattern字段中，编辑代码。
    • 单击“ 插入变量”将列表中的变量添加到“ 模式”字段中编辑的代码中。
  5. 单击确定 。
