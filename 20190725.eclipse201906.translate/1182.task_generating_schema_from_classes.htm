从类生成架构

使用JAXB Classes中的Generate Schema向导为一组JAXB映射类创建XML模式（.xsd）。

  1. 从Navigator或Project Explorer中，选择File> New> Other 。将出现“选择向导”对话框。


    小费：

    您还可以从特定项目或包中访问JAXB Classes向导中的Generate Schema。

    从JAXB类向导中选择Schema


  2. 从JAXB Classes中选择JAXB> Schema ，然后单击Next 。将出现“ 从JAXB类向导 生成架构”的“从类生成架构”页面。

    从类生成模式页面


  3. 选择要从中生成架构的项目，包或类，然后单击“ 完成” 。
