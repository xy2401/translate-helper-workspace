创建异常断点过滤

可以为Java异常断点设置类型名称和包名称筛选。这可以从异常断点的Filtering属性页面完成，如下图所示。

使用Filtering属性页面有四种方法可以为Java异常断点添加过滤：

  1. 您可以选择特定线程来限制断点（必须在调试会话中完成）
  2. 您可以使用“ 添加类”按钮通过“ 类型选择”对话框选择类
  3. 您可以使用“ 添加包”按钮使用筛选的“ 类型选择”对话框选择包
  4. 您可以使用“ 添加”按钮定义自己的模式以匹配类和/或包过滤器

定义自己的过滤器模式

使用上面提到的Add按钮，您可以定义自己的模式，以用作Java异常断点的过滤器。一旦按下，你就可以了
在Selected Locations列表的新空间中输入任何表达式，如下图所示。

创建自己的模式时需要遵循一些规则。

  1. 您的模式最后只能包含'*'
  2. 您的模式必须完全合格，例如abc我的课
  3. 您的图案中不能包含空格

例子

  • abc * - 将匹配包abc中的所有内容
  • ABC我的* -将匹配以'My'开头的abc包中的任何内容
  • 我的* - 将匹配以“我的”开头的默认包中的任何内容
