UI表单

UI Forms是一个基于SWT和JFace的可选Rich Client插件，它支持在所有Eclipse UI类别中创建可移植的Web样式用户界面。它提供了一些精心挑选的自定义小部件，布局和支持类，以实现所需的Web外观。基于SWT，它们在支持SWT的所有平台上具有固有的可移植性。

UI Forms打破了Eclipse工作台UI类别（编辑器，视图，向导，对话框）中可以预期哪些类别的小部件的既定期望。Eclipse表单可以出现在任何和所有表单中，从而扩展了UI开发人员使用最合适的任务概念的可能性，无论他们身在何处。

Eclipse Forms添加以下内容以使Web样式用户界面成为可能：

  • “表单”的概念，适合包含在内容区域中，例如视图和编辑器
  • 一个管理颜色，超链接组和表单其他方面的工具包，以及作为许多SWT控件的工厂
  • 一个新的布局管理器，它布局类似于HTML表格布局算法的控件
  • 一组自定义控件，设计适合表单（超链接，图像超链接，可滚动复合，部分）
  • 一个多页面编辑器，每个页面都是一个表单（例如PDE清单编辑器）
