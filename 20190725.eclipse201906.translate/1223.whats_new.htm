支持JSF 2.0

项目配置

JSF Tools现在支持将Dynamic Web Project配置为JSF 2.0项目。该工具允许用户下载以下JSF 2.0实现库：
  • JSF 2.0（Mojarra 2.0.2-FCS）
  • JSF 2.0（Apache Myfaces JSF Core-2.0 API 2.0.0）




XHTML文件创建

该工具提供了一组HTML模板，可帮助创建Facelets XHTML页面



XHTML源编辑

网页编辑器（WPE）源选项卡已得到增强，可支持编辑XHTML文件。XHTML文件的源代码编辑器具有与JSF-JSP页面的源代码编辑器中相似的功能。 以下是此版本中添加的功能列表，以支持JSF 2.0。

内容辅助可用于库名称空间。可以通过Facelet taglib工具显式定义命名空间，也可以通过资源加载工具隐式地通过组合检测来定义命名空间。





内容对标签和标签属性有帮助

Content Assist可用于JSF Facelets库中的新标记和标记属性





EL表达的内容辅助

关于EL表达式的内容辅助提议包括新的隐式变量



复合组件支持

库名称空间的内容辅助包括通过资源加载工具发现的复合组件，例如在Web应用程序根目录中名为“resources”的目录的子目录下定义的组件。

本节包含对JavaServer Faces Tooling for Release WTP 3.0所做的以下重要更改的说明：

支持JSF的备用视图描述语言

    有关Alternative View Handler支持的信息，请参见此处

    标记注册表视图

    Tag Registry View允许JSF用户浏览每个项目中声明的JSF标记，包括有关它们在运行时创建的组件，转换器和验证器的信息。

    组件树视图

    组件树视图允许用户浏览视图（即JSP页面或Facelet XHTML）在运行时的外观设计时近似值。未来，越来越多的基于组件的功能也将使用此组件树。


WPE的Source选项卡中的超链接支持

    Web页面编辑器（WPE）的“源”选项卡支持在标记属性的表达式语言（EL）中引用的Managed Bean变量，Managed Bean属性和Managed Bean方法上的Hyperlink。用户可以（按住Ctrl键单击）超链接以导航到Managed Bean的源。




在WPE的“源”选项卡中悬停“帮助”支持

    网页编辑器（WPE）的“源”选项卡为标记属性的表达式语言（EL）中定义的所有符号提供悬浮式帮助，包括对托管Bean和资源包的引用。




网页编辑器（WPE）属性页面的增强功能

    网页编辑器（WPE）中的选项卡式属性页面已得到增强。此更改对于项目的采用者非常重要。有关详细信息，请参阅WPE属性页的增强功能 。

支持Apache MyFaces Trinidad

    现在，Web页面编辑器，源编辑器和“属性”视图中支持Apache MyFaces Trinidad标记。






EL内容辅助和悬停帮助的改进

    感谢Matthias Fuessel为我们的EL支持提供改进：

新的EL框架增强功能

    感谢IBM的Xiaonan Jiang推动新的框架要求并提供原型。

    装饰变量和属性解析器

    为了更好地支持JSF变量和属性解析器的装饰性，我们添加了将装饰设计时间解析器注册到默认应用程序管理器的功能。仅当在当前Web项目的配置中找到与其关联的运行时解析程序时，才会调用装饰解析程序。有关更多信息，请参阅： 此处

    符号创建API的增强功能

    我们重构了几种内部符号创建方法，并将它们与Matthias Fuessel和Xiaonan Jiang （IBM）的贡献相结合，创建了一组新的扩展点来简化常见JSF符号的创建。有关详细信息，请参见此处

    我们还在名为JSFSymbolFactory的新工厂上提供了符号创建方法。


新的EL验证首选项

    EL验证问题的新验证首选项允许您更改错误，警告和忽略之间的类型分配问题的严重性。


本节包含对JavaServer Faces Tooling for Release 1.0进行的以下重要更改的说明：

支持JSF 1.2

    JSF Tools支持JavaServer Faces 1.1和1.2版本。




网页编辑器

    现在可以使用增强的网页编辑器。在现有JSP编辑支持的基础上，它在分屏窗格布局，弹出/可拆卸调色板，增强属性表，预览页面等中提供Faces JSP的同步同步视觉和源编辑。



    窗格的布局以及是否显示两者都是可配置的。



    可以在编辑器的维基页面找到更多计划的功能和想法。

面部配置编辑器

    Faces Configuration Editor是一个全面的多页面编辑器，用于编辑Faces配置资源文件。它包括一个图形编辑器，可以直观地定义导航规则。


改进了HTML DataTable标记变量支持

    现在，使用h：dataTable标记上的var属性声明的EL变量现在可以被JSF EL框架识别。附加支持的数量取决于用于支持变量的“value”属性，为带有通用列表的标量或数组对象提供的最完整。


JSF库现在是常规的Classpath容器

    JSF库使用此版本中引入的依赖项功能的简单发布 。使用标准Java Build Path属性页面对项目中的JSF库进行引用，并使用J2EE Module Dependencies属性页面进行发布/导出，并在Project和Package Explorer视图中显示为JDT Libraries。 打开旧工作区时，将自动进行JSF库注册表的升级。但是，实际参考资料需要手动升级。有关这方面的信息， 请访问http://wiki.eclipse.org/index.php/JSF_Library_Migration 。JSF库引用仍在JSF Facet安装时创建，但如果您需要在项目创建后添加，修改或删除引用，您现在将使用标准的Java Build Path和J2EE Module Dependencies属性页。
