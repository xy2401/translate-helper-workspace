要求和安装

在安装Dali之前，请确保您的环境满足以下最低要求：

  • Eclipse 3.7（ http://www.eclipse.org/downloads ）

  • Java运行时环境（JRE）1.5（ http://java.com ）

  • Eclipse Web Tools Platform（WTP）3.3（ http://www.eclipse.org/webtools ）

  • Java Persistence API（JPA）for Java EE 5。例如，JPA的EclipseLink实现可以从以下位置获得： http://www.eclipse.org/eclipselink/ ： http://www.eclipse.org/eclipselink/

有关其他安装信息，请参阅http://www.eclipse.org/webtools/dali/gettingstarted_main.html 。

Dali作为WTP 3.4的一部分。无需其他安装或配置。

辅助功能

Dali支持Eclipse中的标准辅助功能，包括：

  • 使用键盘导航用户界面。

  • 指定编辑器的常规辅助功能首选项。

有关详细信息，请参阅Workbench用户指南 中的Eclipse中的辅助功能 。
