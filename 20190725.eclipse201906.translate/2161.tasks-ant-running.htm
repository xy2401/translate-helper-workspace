运行Ant构建文件

要在Workbench中运行Ant构建文件：

  1. 在其中一个导航视图中，选择一个XML文件。
  2. 从文件的弹出菜单中，选择Run Ant ...。将打开启动配置对话框。
  3. 从“ 目标”选项卡中选择一个或多个目标。选择项目的顺序是它们的运行顺序。订单显示在选项卡底部的“ 目标执行顺序”框中。您可以通过单击“ 订单...”按钮来更改目标的顺序 。
  4. （可选）在其他选项卡上配置选项。例如，在Main选项卡上，在Arguments字段中键入任何必需的参数 。
  5. 单击运行 。

Ant构建文件将在选定的目标上运行。除非您在Main选项卡上禁用了Capture输出选项，否则控制台会在构建文件运行时显示任何适用的执行结果。

这些步骤创建了持久启动配置。新创建的配置将显示在“ 运行”>“外部工具”下的启动历史记录中，并将在通过单击打开的启动配置对话框中可用 运行>外部工具>外部工具....
