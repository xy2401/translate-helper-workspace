命令

描述

切换Java编辑器面包屑

此按钮启用Java编辑器痕迹。

分别记住每个视角的启用。

切换标记出现次数

在Java编辑器中打开和关闭标记出现 。

切换块选择模式

此按钮在编辑器中启用块（aka列）选择模式。

显示空白字符

使用此按钮可以在编辑器中显示空白字符。

仅显示所选元素的来源

此按钮启用了对编译单元源的分段视图的显示。

仅当您自定义透视图以显示编辑器演示文稿操作时，才会显示此按钮。

例如，如果在“大纲”视图中选择了一个方法，则“仅显示所选元素的源”选项仅导致该方法显示在编辑器中，而不是整个类。

关 ：整个编译单元显示在编辑器中，选中的Java元素在标记栏中突出显示，并带有范围指示器。

On ：编辑器中仅显示选定的Java元素，该元素链接到“大纲”或“类型层次结构”视图中的选择。

转到下一个问题

此命令导航到活动编辑器中的下一个问题标记。

转到上一个问题

此命令导航到活动编辑器中的上一个问题标记。
