编辑菜单操作

名称 功能 键盘快捷键
解开 恢复编辑器中的最后一次更改 按Ctrl + Z
重做 重新应用先前使用撤消恢复的更改 按Ctrl + Y
切 将当前选定的文本复制到剪贴板并将其从编辑器中删除 按Ctrl + X
复制 将当前选定的文本复制到剪贴板 按Ctrl + C
糊 将剪贴板的内容作为文本粘贴到当前活动的编辑器窗口 按Ctrl + V
删除 删除当前选定的文本 删除
全选 选择当前活动编辑器窗口中的所有文本 按Ctrl + A
展开选择到

通过以下方式扩展选择（或从光标开始创建选择）：

 Alt + Shift + 箭头键
封闭元素
将封闭表达式或块添加到选择中（它依赖于嵌入式解析器，因此在出现语法错误时可能会失败。）
 Alt + Shift +向上
下一个元素
将以下元素添加到选择中
 Alt + Shift +右键
上一个元素
将前面的元素添加到选择中
 ALT + SHIFT +左键
恢复上次选择
在调用Expand Selection To后，恢复先前的选择。
Alt + Shift +向下
查找/替换... 打开基本的编辑器内查找/替换对话框 按Ctrl + F
找到单词 如果它包含多个单词，则查找包含光标的单词或当前选择的第一个单词的下一个出现（为Visual Studio兼容性添加。）  
找下一个 如果没有选择，则查找当前所选文本的下一个匹配项，或搜索的最后一个文本 按Ctrl + K
找到上一个 如果没有选择，则查找当前所选文本的上一次出现或搜索的最后一个文本 按Ctrl + Shift + K
增量查找下一个 启动增量查找模式（将搜索随后输入的内容。） 按Ctrl + J
增量查找上一个 向后看，开始增量查找模式 CTRL + SHIFT + J
添加书签... 添加指向当前光标位置或所选文本的书签  
添加任务... 添加位于当前行的用户定义任务  
智能插入模式 切换智能插入模式（即，在中定义的自动完成功能） C / C ++>编辑器>打字 ） 按Ctrl + Shift + Insert键
显示工具提示说明 显示适合光标位置的工具提示。 F2
单词完成 使用下一个可用的完成来完成包含光标的单词（如果重复调用，它将循环查看可能性。） Alt + /
快速解决 显示适合光标位置的“快速修复”对话框。 按Ctrl + 1
内容辅助 显示适合光标位置的“内容辅助”对话框（如果重复调用，它将循环显示可用的提议类型。） Ctrl +空格键
参数提示 在工具提示中显示可能的签名（当光标位于参数列表中时启用） 按Ctrl + Shift +空格
设置编码... 显示一个对话框，可在其中指定活动编辑器的编码
