C / C ++开发人员可以使用的透视图

透视图是Workbench窗口中的视图布局（开发工具）。每种类型的透视图都是视图，菜单和工具栏的组合，使您可以执行特定任务。例如，C / C ++透视图具有组织的视图，以帮助您开发C / C ++程序; Debug透视图具有允许您调试这些程序的视图。

要打开C / C ++透视图，请选择Window> Open Perspective> Other ...并选择C / C ++ 。

选择/打开视图：
  • 您可以向透视图添加视图。从菜单栏中选择“ 窗口”>“显示视图”>“其他”，然后从“ 显示视图”对话框中选择新视图。
  • 要将当前透视图重置为其原始布局，请从菜单栏中选择“ 窗口”>“重置透视图” 。

C / C ++开发工具为工作台提供了以下视角：

C / C ++透视图

此透视图适用于使用C / C ++项目。默认情况下，它由编辑器区域和以下视图组成：

  • Project Explorer
  • 安慰
  • 属性
  • 问题
  • 制作目标
  • 大纲
  • 任务

在调试会话期间可能有一些其他视图可能很有用。您可以通过单击“ 窗口”>“显示视图”>“C / C ++”来添加它们，以便在C / C ++透视图中显示它们。

调试透视图

此透视图已调整为调试C / C ++程序。默认情况下，它包含编辑器区域和以下视图：

  • 调试
  • 变量
  • 断点
  • 大纲
  • 安慰
  • 任务
  • 问题
  • 可执行文件

在调试会话期间可能有一些其他视图可能很有用。您可以通过单击Window> Show View> Debug来添加它们，以在Debug透视图中显示它们。

  • Java的
  • Java浏览
  • 插件开发。
  • CVS资源库探索
