创建C ++文件

您可以开始编写HelloWorld程序的代码。您创建的.cpp文件将保存在刚刚创建的项目文件夹中创建Makefile项目 。

文件在位于C / C ++ Projects视图右侧的C / C ++编辑器中编辑。C / C ++编辑器的左边距（称为标记栏）显示项目的图标，例如书签，断点以及编译器错误和警告。

有关的更多信息：

  • 编辑器区域和标记栏，请参见“ 工作台用户指南”>“参考”>“用户界面信息”>“视图和编辑器”>“编辑器”区域
  • 标记栏图标，请参见“ 工作台用户指南”>“参考”>“用户界面信息”>“图标和按钮”>“编辑器区域标记栏”

要创建C ++文件：

  1. 在Project Explorer视图中，右键单击HelloWorld项目文件夹，然后选择New> Source File 。
  2. 在源文件：字段中，键入main.cpp 。
    默认情况下，源文件夹应该是您的项目文件夹。
    选择的模板可能是默认C / C ++源模板。
  3. 单击完成 。
  4. 注释模板可能显示在其他空文件的顶部。在编辑器中键入完全如下所示的代码。或者您可以从此帮助文件中粘贴它。 
    #include <iostream>
using namespace std;

int main () {
    // Say HelloWorld five times
    for (int index = 0; index < 5; ++index)
      cout << "HelloWorld!" << endl;
    char input = 'i';
    cout << "To exit, press 'm' then the 'Enter' key." << endl;
    cin  >> input;
    while(input != 'm') {
        cout << "You just entered '" << input << "'. "
             << "You need to enter 'm' to exit." << endl;
        cin  >> input;
    }
    cout << "Thank you. Exiting." << endl;
    return 0;
}
  5. 单击文件>保存 。

您的新.cpp文件显示在Project Explorer视图中。您的项目现在包含main.cpp。在构建HelloWorld项目之前，必须先创建一个makefile。
