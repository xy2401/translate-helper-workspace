转换项目的C或C ++性质

使用“ 转换为C / C ++项目”向导为C ++项目指定C特性，反之亦然。

为项目分配C或C ++特性

  1. 单击文件>新建>其他 。
  2. 单击C或C ++ 。
  3. 单击转换为C / C ++生成项目 。
  4. 单击下一步 。
  5. 在候选转换列表中，选择要转换的项目。
  6. 在“ 转换为C或C ++”组中，单击“ C项目”或“ C ++项目” 。
  7. 在“ 项目选项”组中，如果启用了“ 指定项目类型”选项，则可以指定“ 项目类型”和“ 工具链 ”。
  8. 单击完成 。
