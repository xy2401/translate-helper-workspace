运行Ant构建文件

任何扩展名为.xml文件都可以作为Ant构建文件运行。当然，并非所有这些文件都是Ant构建文件，但如果尝试将非Ant .xml文件错误地作为Ant构建文件运行，则不会造成任何损害。

在本节中，将使用在编辑Ant构建文件任务中创建的HelloWorld.xml文件来涵盖在Eclipse中运行Ant构建文件的基本机制。

  1. 在其中一个导航视图中选择HelloWorld.xml ，然后从其上下文菜单中选择Run As> Ant Build ....
  2. 将在此Ant构建文件的启动配置上打开启动配置对话框。

  3. 此对话框允许配置运行Ant构建文件的方式的许多方面，但现在集中在“ 目标”选项卡上，该选项卡允许选择要运行的Ant目标及其顺序。选择两个目标并将订单保留为默认值。
  4. 单击“ 运行”按钮。
  5. 运行Ant构建文件，并将输出发送到Console视图 。
