将资源嵌套在输出目录中

文件系统上的布局

  • 产品的Java源文件在sources和deliverables目录中布局。
  • 所有Java类文件都在deliverables目录中列出。
  • 项目需要使用位于deliverables/libraries目录中的一些库：

定义相应项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“产品”。单击下一步 。

  4. 在下一页上，在“ 默认输出文件夹”字段中键入“产品/可交付物”。

    选择“产品”源文件夹。

    单击详细信息窗格或按钮中的链接其他源链接在视图栏中。

  5. 在Link Source中单击Browse ....并选择D:\Product\sources目录。

  6. 单击“ 完成”以验证并关闭对话框。

  7. 再次选择“产品”，然后单击“ 详细信息”窗格或按钮中的“ 链接其他源”链接在视图栏中。

  8. 在Link Source中单击Browse ....并选择D:\Product\deliverables目录。

  9. 展开“产品/可交付物”源文件夹。选择“libraries”包并使用从构建路径链接中排除“库”或“ 排除”弹出菜单项将其从构建路径中排除 。

  10. 您的项目源设置现在如下所示：

  11. 选择“ 库”选项卡。

    单击Add JARs ....

  12. 展开“Product”层次结构以选择“libraries”目录中的jar文件

    单击确定 。

  13. 单击“ 完成”以完成项目创建。

  14. 您现在拥有一个带有“sources”文件夹的Java项目和一个包含嵌套库资源的输出文件夹。
