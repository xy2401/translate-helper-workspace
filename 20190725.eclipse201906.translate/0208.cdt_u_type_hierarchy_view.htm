键入层次结构视图

该 “类型层次结构”视图显示所选类型或方法的树形显示。“ 类型层次结构”视图由两个窗格组成：

  • 键入“层次结构树”窗格
  • 成员列表窗格（可选）

键入层次结构视图工具栏

类型层次结构树显示给定类型的超类型，子类型或两者，具体取决于工具栏中的选择。

键入层次结构工具栏选项
图标 命令 描述

显示类型层次结构

此命令在“层次结构”视图中的完整上下文（即超类和子类）中显示类型。

显示超类型层次结构

此命令在“层次结构”视图中显示超类型。树以选定的类型开始，并显示遍历层次结构的结果。

注意：所选类型始终位于左上角的顶层。

显示子类型层次结构

此命令在“层次结构”视图中显示所选类型的子类型。树以选定的类型开始，并显示遍历层次结构的结果

注意：所选类型始终位于左上角的顶层。

显示历史列表

此菜单显示先前显示的类型层次结构的历史记录。

刷新

此命令通过在索引数据库上运行新查询来更新视图内容。

取消

此命令取消当前运行的查询。

查看菜单

显示管理“类型层次结构”视图的其他选项，包括：

  • 选择工作集......
  • 取消选择工作集
  • 编辑活动工作集...
  • 键入层次结构
  • 超类型层次结构
  • 子类型层次结构
  • 布局
  • 显示文件名

成员列表窗格工具栏按钮

成员列表窗格在类型层次结构树窗格中显示当前所选类型的成员。

键入层次结构成员选项
命令 描述

显示所有继承的成员

显示或隐藏基类继承的所有方法和字段。设置此选项后，定义方法的类型名称将附加到方法名称。

隐藏字段

显示或隐藏字段。

隐藏静态字段和方法

显示或隐藏静态字段和方法。

隐藏非公开会员

显示或隐藏静态字段和方法。
