Project Explorer视图

该 Project Explorer视图以类似于C / C ++ Projects视图的树结构显示，但不限于C和C ++项目。在此视图中，您可以执行以下操作：

  • 浏览C / C ++源文件的元素
  • 在编辑器视图中打开文件
  • 在新窗口中打开项目
  • 创建新项目，类，文件或文件夹
  • 管理现有文件（剪切，粘贴，删除，移动或重命名）
  • 从本地历史记录恢复已删除的文
  • 导入或导出文件和项目

在Project Explorer视图中选择的文件会影响其他视图中显示的信息。

右键单击Project Explorer视图中的任何资源，打开一个弹出菜单，允许您执行复制，移动，创建新资源，相互比较资源或执行团队操作等操作。

Project Explorer工具栏

下表列出了Project Explorer视图中显示的工具栏选项。

图标 名称 描述
最小化 最小化视图。
最大化 最大化视图。
全部收缩 此命令折叠视图中所有资源的树扩展状态。
与编辑链接 此命令切换视图选择是否链接到活动编辑器。选择此选项后，更改活动编辑器将自动将选择更新为正在编辑的资源。
菜单 单击黑色倒置三角形图标以打开特定于视图的项目菜单。

顶级元素
选择是将工作集还是项目显示为顶级元素。选择工作集可以在大型工作空间中轻松分组项目。

选择工作集
打开“ 选择工作集”对话框以允许为视图选择工作集。

取消选择工作集
取消选择当前工作集。

编辑活动工作集
打开“ 编辑工作集”对话框以允许更改当前工作集。

包装介绍
选择以平面或分层形式显示包。

自定义视图
此命令允许自定义视图筛选器和内容模块。前一个将允许您禁止显示某些类型的文件，而后者将允许在视图中显示全新类型的内容。
