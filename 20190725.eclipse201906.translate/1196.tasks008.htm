创建orm.xml文件

如果在创建JPA项目时选择不创建orm.xml文件，则可以使用“ 创建ORM映射文件”向导创建一个。

使用此过程创建orm.xml文件：

  1. 从Navigator或Project Explorer中，选择File> New> Other 。将出现“选择向导”对话框。

    选择向导对话框


  2. 选择JPA ORM Mapping File ，然后单击Next 。将出现“ 创建ORM映射文件”向导的 “映射文件”页面。

    如果您使用的是EclipseLink，则可以选择EclipseLink> EclipseLink ORM Mapping File 。

    新映射文件位置页面


  3. 选择映射文件的名称和位置，然后单击“ 下一步” 。将出现“ 映射文件选项”页面。

    新映射文件选项页面


  4. 在Mapping File Options页面中定义属性，然后单击Finish 。orm.xml文件出现在所选JPA项目的src目录中。您可以使用JPA Details视图或XML Editor管理orm.xml文件。另请参阅JPA Details视图（对于orm.xml） 。
