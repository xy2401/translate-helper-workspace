源菜单操作

名称 功能 键盘快捷键
切换评论 如果没有选择，则在选择中的每一行或包含光标的行中添加或删除单行，C ++样式，注释标记 按Ctrl + /
添加阻止评论 在选择周围添加块注释 按Ctrl + Shift + /
删除块注释 删除选择中任何块注释周围的注释标记 按Ctrl + Shift + \
向右移动 如果没有选择，则增加选区中每一行的缩进，或包含光标的行  
向左移动 如果没有选择，则减少选区中每行的缩进或包含光标的行 Shift + Tab键
正确的缩进 如果没有选择，则将简单缩进规则应用于当前选择或包含光标的行 按Ctrl + I
格式 适用完整 C / C ++>代码样式>格式化程序首选项，指向活动编辑器的全部内容，或者选择是否存在 按Ctrl + Shift + F
添加包含 添加一个include语句，并可选地为当前选定的名称添加“using”声明 Ctrl + Shift + N
组织包括 添加所选文件的所有必需包含  
排序行 对所选行进行排序 按Ctrl + Alt + S
实施方法...... 打开“实施方法”对话框  
生成Getters和Setter ...... 打开Generate Getters and Setters对话框，允许您为当前类型的字段创建Getters和Setter。可用于字段和类型，也可用于类型内的文本选择  
环绕着 使用代码模板（如中所定义）围绕选定的语句 C / C ++>编辑器>模板 ） Alt + Shift + Z
