在C / C ++ Projects视图中显示C / C ++文件组件

文件组件显示在“ C / C ++项目”视图和“ 大纲”视图中。您可以在“C / C ++项目”视图中显示或隐藏所有文件组件。

显示文件组件

  1. 单击窗口>首选项 。
  2. 在“ 首选项”对话框中，从列表中选择“ C / C ++”>“外观 ”。
  3. 选中“ 显示翻译单元成员”复选框。
  4. 单击确定 。
  5. 在“ C / C ++项目”视图中，双击文件组件。
    该组件在C / C ++编辑器中突出显示。

还可以过滤C / C ++ Projects视图以显示某些类型的文件组件。有关更多信息，请参阅“C / C ++项目”视图中的“按类型隐藏文件” 。
