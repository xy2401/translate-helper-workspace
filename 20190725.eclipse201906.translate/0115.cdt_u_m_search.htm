搜索菜单操作

搜索菜单命令打开搜索对话框。常规“ 搜索”对话框上有专门的选项卡，可帮助您搜索：

  • 文件或文件中的文本
  • C / C ++文件中的元素
  • 在线帮助中的文字
  • 插件。
名称 功能 键盘快捷键
C / C ++ ... 在C / C ++搜索页面上打开搜索对话框  
搜索... 打开当前编辑器的搜索对话框 按Ctrl + H
文件... 在“文件搜索”页面上打开搜索对话框  
文本 在给定范围（工作空间，项目，文件或工作集）中打开全文搜索子菜单
