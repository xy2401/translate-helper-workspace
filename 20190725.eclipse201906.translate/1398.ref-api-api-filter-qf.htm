添加API问题过滤器快速修复

此快速修复可用于快速创建API问题筛选器，以解决无法轻松纠正的已知或预期的API问题。

可以在以下两个位置之一使用此快速修复：

  1. 问题观点 。
  2. 编辑问题所在的资源 。

问题观点

  • 打开“ 问题”视图。
  • 选择API Tools问题。

  • 使用Ctrl + 1从“问题”视图触发快速修复（或使用上面显示的上下文菜单）。

编辑资源

  • 打开API问题所在的资源。
  • 对相应的问题使用Ctrl + 1 。


创建问题过滤器后，将重建相应的资源，并从“问题”视图和任何打开的编辑器中删除问题。
