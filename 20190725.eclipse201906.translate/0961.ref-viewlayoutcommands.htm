查看布局命令

视图布局命令会影响视图中详细信息窗格的方向。此外，“ 变量视图”还具有允许您更改列设置的命令。

这些命令适用于：

  • 表达式视图
  • 注册视图
  • 变量视图

这些命令在“布局”菜单下可用，该菜单显示在下面的“变量”视图中。

详细信息窗格显示命令

命令

名称

描述

可用性
显示列 如果列应显示在视图中，则更改。变量仅查看。 查看操作
  选择列... 允许您选择列可见时显示的列。变量仅查看。 查看操作
横 显示视图右侧的详细信息窗格，水平对齐视图的各个部分。 查看操作

垂直

 显示视图底部的详细信息窗格，垂直对齐视图的各个部分。 查看操作
只读 隐藏详细信息窗格。 查看操作
