新的JAXB项目页面

此表列出了“ 创建新JPA项目”向导的“新建JPA项目”页面上可用的属性。

属性 描述 默认

项目名

Eclipse JPA项目的名称。

项目地点

用于保存项目的工作空间的位置。

取消选中“ 使用默认位置”选项，然后单击“ 浏览”以选择新位置。

当前工作空间

目标运行时

为项目选择预定义的目标。

单击“ 新建运行时”以使用“新建服务器运行时”向导创建新环境。

<无>

JAXB版本

为项目选择Java Architecture for XML Binding（JAXB）版本。

2.2

配置

选择具有预定义构面的项目配置。

选择“ 修改”以手动选择此项目的构面。

工作集

指定此项目是否应包含在现有工作集中。下拉字段显示先前所选工作集的列表。

选择“ 将项目添加到工作集” ，然后选择要在其中添加此项目的工作集 。
