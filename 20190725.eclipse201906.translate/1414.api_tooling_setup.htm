设置API工具

配置现有插件

本节介绍如何启用现有的API分析插件。

  1. 选择要转换的项目。
  2. 右键单击并选择“ PDE工具” >“ API工具设置”...
  3. 打开API Tools安装向导后，检查项目是否已选中。
  4. 如果项目包含文件component.xml ，则可以取消选中该复选框以在转换后保留它。
  5. 如果要预览更改，请单击“ 下一步”，否则单击“ 完成” 。
  6. 您的插件现已转换为使用API工具。

配置新插件

本节介绍如何为API分析启用新插件。

  1. 单击文件>新建>项目...
  2. 选择Plug-in Development> Plug-in Project ，然后单击Next 。
  3. 像对任何其他插件一样填写第一页，然后单击“ 下一步” 。
  4. 选中“ Enable API Analysis ”选项。

  5. 完成插件设置
  6. 单击完成 。
