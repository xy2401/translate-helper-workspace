评论代码

您可以在任何C / C ++编辑器视图中注释掉一行或多行代码。在注释一行或多行代码时，前导字符//被添加到每行的开头。您还可以使用字符/* */来阻止多行代码注释。

提示：如上所述，当您注释和取消注释代码时，已经注释掉的行上的字符/* */不会受到影响。

要切换//对代码的评论：

  1. 在C / C ++编辑器中，选择要注释掉的代码行。如果未选择任何行，则将在当前光标位置添加（或删除）注释。
  2. 右键单击并选择“ 源”>“切换注释” 。（ CTRL+/ ）

阻止评论/* */代码：

  1. 在C / C ++编辑器中，选择要注释掉的多行代码。
  2. 要注释掉多个代码行，请右键单击并选择“ 源”>“添加块注释” 。（ CTRL+SHIFT+/ ）
  3. 要取消注释多个代码行，请右键单击并选择“ 源”>“删除块注释” 。（ CTRL+SHIFT+\ ）
