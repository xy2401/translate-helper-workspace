Java元素装饰

覆盖指标

Java视图可以显示特殊的装饰图标，以指示从超类型覆盖或实现方法的方法。

要显示或隐藏覆盖指示符，请选中或清除中的Java方法覆盖指示符复选框 常规>外观>标签装饰首选项页面。

Java类型指示器

包浏览器可以指示Java文件中包含的Java类型（Enum，Interface，Annotation）的类型。

注意：

  • 如果Java文件包含多个顶级类型，则会根据其主要类型进行装饰。
  • 除抽象类外，没有为类显示装饰。

要显示或隐藏Java类型指示符，请选中或清除中的Java类型指示符复选框 常规>外观>标签装饰首选项页面。
