C / C ++项目属性，重构历史

允许您浏览当前所选项目的整个重构历史记录。

重构<project>的历史记录：
重构历史记录对话框显示工作区的整个重构历史记录。深入查看单个重构并选择它以在下面的详细信息视图中查看有关执行的重构的更多详细信息。此对话框还可用于从重构历史中删除某些重构。选择整年，月，周或日，然后按“删除”从重构历史记录中删除选定的重构。为了能够更好地从重构历史中选择重构，有两种方法可以对重构进行分组：按项目分组：重构按项目分组。每个重构都与包含元素的项目相关联，该元素用作记录重构的输入。注意：某些重构实例无法与项目唯一关联。这些重构与整个工作区相关联。按日期分组：重构按日期分组。重构历史对话框将重构作为历史记录，最近最近执行的重构。
