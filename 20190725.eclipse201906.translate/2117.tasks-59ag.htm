将资源导出到存档文件

您可以使用“导出”向导来将资源从Workbench导出到文件系统中的存档文件 。

  1. 在其中一个导航视图中，选择要导出的资源。
  2. 从主菜单栏中选择 文件>导出....“导出”向导将打开。
  3. 选择General> Archive File ，然后单击Next 。
  4. 默认情况下，您选择的资源将与其子项一起导出。（可选）使用左窗格和右窗格中的复选框选择要导出的资源集，并使用“ 选择类型”等按钮来过滤要导出的文件类型。
  5. 指定要将所选资源导出到的存档文件的路径和名称。
  6. 单击完成。
