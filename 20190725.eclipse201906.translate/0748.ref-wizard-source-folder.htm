新源文件夹向导

该 “新建源文件夹”向导可帮助您为Java项目创建新的源文件夹。

请注意，新的源文件夹不能嵌套在现有的源文件夹或输出文件夹中。您可以选择将排除过滤器添加到其他嵌套源文件夹，或者向导将建议使用新创建的条目替换嵌套类路径条目。该向导还会建议更改输出位置。

新的源文件夹选项
选项 描述 默认
项目名 输入包含新源文件夹的项目。键入有效的项目名称或单击“ 浏览”以通过对话框选择项目。 向导启动时选择的元素项目。
文件夹名称 键入新源文件夹的路径。路径相对于所选项目。 <空白>
更新其他源文件夹中的排除过滤器以解决嵌套问题 选择修改现有源文件夹的排除过滤器以解决嵌套问题。例如，如果存在现有的源文件夹src并且创建了文件夹src / inner ，则源文件夹src将更新为具有排除过滤器src / inner 。 离
