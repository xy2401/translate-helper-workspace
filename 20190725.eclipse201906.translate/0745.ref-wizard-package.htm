新的Java包向导

该 New Java Package向导可帮助您创建与新Java包对应的文件夹以及可选的包级文档文件。它还可用于将包级别文档添加到现有包中。

默认包的相应文件夹始终存在，因此不必创建。

Java包选项
选项 描述 默认
源文件夹 键入或浏览以选择新包的容器（项目或文件夹）。 启动向导时选择的元素的源文件夹。
名称 输入新包的名称 项目名称，如果它是有效的包名称，并且源文件夹不包含任何其他包，否则为<blank>
创建package-info.java

是否应在包中创建package-info.java文件。

package-info.java文件包含包级别文档和注释。例如：

/ ** *此包包含JAXB类。* / @XmlSchema（namespace =“http://www.example.com”，elementFormDefault = XmlNsForm。QUALIFIED）包com.acme.jaxb;

有关更多信息，请参阅Java语言规范第7.4.1节。命名包 。
