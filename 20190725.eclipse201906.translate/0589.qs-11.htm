搜索工作台

在本节中，您将在工作台中搜索Tcl元素。

在“搜索”对话框中，您可以执行文件，文本或Tcl搜索。Tcl搜索操作代码的结构。文件搜索按名称和/或文本内容对文件进行操作。Tcl搜索更快，因为代码结构有一个底层索引结构。文本搜索允许您在注释和字符串中查找匹配项。

从工作台执行Tcl搜索

  1. 复制http.tcl文件的内容（来自http1.0标准包）并将其粘贴到示例项目中的新Tcl文件中。
  2. 在Tcl透视图中，单击“ 搜索” （ ）工作台工具栏中的按钮或使用菜单栏中的搜索> Tcl 。
  3. 如果尚未选中，请选择“ Tcl搜索”选项卡。
  4. 在“ 搜索字符串”字段中，键入http_config 。在“ 搜索”区域中，选择“ 方法” ，然后在“ 限制到”区域中，选择“声明” 。
    验证Scope是否设置为Workspace 。

    然后单击搜索 。搜索时，您可以随时单击取消以停止搜索。将显示部分结果。
  5. 在Tcl透视图中，“搜索”视图显示搜索结果，并在编辑器中突出显示第一个匹配项。

    你可能会显示下一场比赛 （ ）并显示上一场比赛 （ ）导航到每个匹配的按钮（在这个例子中有一个匹配）。
    请注意，如果找到匹配项的文件当前未打开，则在“搜索”视图中选择匹配项时，将在编辑器中打开该文件。
  6. 使用“搜索”视图按钮导航到搜索匹配时，文件将在编辑器中的匹配位置打开。搜索匹配在垂直标尺中用搜索标记标记，如上面的屏幕截图所示。

执行文件搜索

  1. 在Tcl透视图中，单击工作台工具栏中的“ 搜索”按钮，或从菜单栏中选择“ 搜索”>“文件 ”。
  2. 如果尚未选中，请选择“ 文件搜索”选项卡。
  3. 在“ 包含”文本字段中，键入http_config 。确保“ 文件名模式”字段设置为* 。范围应设置为Workspace 。然后单击搜索 。

  4. 要查找给定文件名模式的所有文件，请将“包含文本”字段留空。

查看以前的搜索结果

在“搜索结果”视图中，单击“ 上一个搜索结果”工具栏按钮旁边的箭头，以查看包含最新搜索列表的菜单。您可以从此菜单中选择项目以查看以前的搜索。可以通过选择“ 清除历史记录”清除列表。

“ 上一个搜索结果”按钮将显示一个对话框，其中包含当前会话中所有先前搜索的列表。

从此对话框中选择上一个搜索将允许您查看该搜索。
