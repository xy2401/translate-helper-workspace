外观偏好

使用 “ 首选项”窗口的“ 外观”面板用于自定义查看器中C元素的外观。单击窗口>首选项> C / C ++>外观打开。

外观选项
选项 描述
显示翻译单位成员 选择此选项可在Project Explorer和C / C ++ Projects视图中显示翻译单元成员。
组包含Project Explorer和C / C ++ Projects视图中的指令 选择此选项可在Project Explorer和C / C ++ Projects视图中对include指令进行分组。
组在“大纲”视图中包含指令 选择此选项可在“大纲”视图中对include指令进行分组。
“大纲”视图中的组命名空间 选择此选项可在“大纲”视图中对名称空间声明进行分组。
“大纲”视图中的组方法定义 选择此选项可在“大纲”视图中对同一类型的方法定义进行分组。
在Project Explorer和C / C ++ Project视图中组宏定义。 选择此选项可在Project Explorer和C / C ++ Project视图中对相同类型的方法定义进行分组。
“大纲”视图中的组宏定义。 选择此选项可在“大纲”视图中对宏定义进行分组。
在Project Explorer和C / C ++ Projects视图中对源文件之前的头文件进行排序 选择此选项可在Project Explorer和C / C ++ Projects视图中分隔标题和源文件。
在项目顶部显示源根 选择此选项可将项目视图顶部的源根分组。
