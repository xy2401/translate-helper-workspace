文件菜单

使用“ 文件”菜单可以创建，保存，关闭，打印，导入和导出Workbench资源并退出Workbench。

新 （Shift + Alt + N）
使您可以创建新资源。在创建新文件之前，必须创建一个用于存储文件的项目。
打开文件
使您可以打开文件进行编辑 - 包括不驻留在工作区中的文件。
关闭 （Ctrl + W）
关闭活动编辑器。在文件关闭之前，系统会提示您保存更改。
全部关闭 （Shift + Ctrl + W）
关闭所有开放编辑。在文件关闭之前，系统会提示您保存更改。
保存 （Ctrl + S）
保存活动编辑器的内容。
另存为
使您可以将活动编辑器的内容保存在另一个文件名或位置下。
全部保存 （Shift + Ctrl + S）
保存所有打开的编辑器的内容。
还原
用以前保存的内容替换活动编辑器的内容。
移动
使您可以将当前选定的资源移动到其他项目或使用这些资源创建Java包。
重命名 （F2）
使您可以更改当前所选资源的名称。
刷新 （F5）
使用文件系统中的内容刷新资源。
将行分隔符转换为
更改所选文件的行分隔符。在您再次更改分隔符之前，更改会立即生效并保持不变 - 您无需保存文件。
打印 （Ctrl + P）
打印活动编辑器的内容。
切换工作区
打开Workspace Launcher ，您可以从中切换到其他工作区。这将重新启动Workbench。
进口
启动“ 导入”向导，该向导使您可以向Workbench添加资源。
出口
启动“ 导出”向导，该向导使您可以从Workbench导出资源。
属性 （Alt + Enter）
打开当前所选资源的“ 属性”对话框。你可以学习：
  • 文件系统中资源的路径
  • 上次修改的日期
  • 文件是可写还是可执行，以及它的编码是什么
  • 项目的资源是否继承其编码和行分隔符，或者它们是否设置为特定值。
最近的文件列表
包含Workbench中最近访问的文件的列表。只需单击文件名，即可从“ 文件”菜单中打开任何这些文件。您可以从“ 编辑器”首选项页面控制此列表中的文件数。
出口
关闭并退出Workbench。
