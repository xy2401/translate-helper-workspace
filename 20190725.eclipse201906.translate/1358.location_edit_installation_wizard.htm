安装位置向导

“ 安装位置向导”用于在目标定义中创建和编辑安装位置。要创建新的安装位置，请使用“ 位置”选项卡或“ 定义”页面中的“ 添加位置向导” 。

选择包含Eclipse安装或基于Eclipse的产品的文件夹。如果安装使用特殊配置目录，则可以输入特定位置而不是使用默认位置。选择目录时，您可以使用下拉列表查看以前使用过的文件夹。您还可以在路径中使用变量（例如$ eclipse_home） ，以便更轻松地共享目标定义。

选择安装目录后，可以按“ 完成”将位置添加到目标，也可以按“ 下一步” 预览位置内容。
