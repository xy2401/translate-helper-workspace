创建链接资源

文件夹和文件可以链接到项目位置之外的文件系统中的位置。这些特殊文件夹和文件称为链接资源 。

要创建链接文件夹：

  1. 在其中一个导航视图中，右键单击要在其中创建链接文件夹的项目或文件夹。
  2. 从弹出菜单中选择 新建>文件夹 。
  3. 指定将在工作台中显示的文件夹的名称。此名称可以与文件系统中的文件夹名称不同。
  4. 单击高级 。
  5. 检查链接到备用位置（链接文件夹） 。
  6. 输入文件系统路径，或单击“ 浏览”以选择文件系统中的文件夹。
  7. 单击完成 。

要创建链接文件，请按照上述相同步骤操作，但选择除外 新建>文件而不是上下文菜单中的新建>文件夹 。

也可以相对于变量指定链接的资源位置。这使得与其他团队成员共享包含链接资源的项目变得更加容易，因为它避免了硬编码的绝对文件系统路径，这些路径可能因机器而异。

要定义相对于路径变量的链接资源，请在上面的步骤5之后执行以下操作：

  1. 单击“ 变量”按钮。
  2. 在结果对话框中，选择现有路径变量或创建新路径变量。
  3. 如果所选变量定义链接资源的确切路径，请单击“ 确定” 。否则，单击“ 扩展”以指定路径变量描述的位置下方的文件或文件夹，然后单击“ 确定” 。
  4. 单击完成 。

提示： 常规>工作区>链接资源首选项页面还允许您定义路径变量。

请注意，创建链接资源后，您将能够更改在上面的步骤6.或8中输入的链接目标路径。通过选择链接资源的“ 文件”>“属性”>“资源”属性页中的“ 编辑...”按钮。
