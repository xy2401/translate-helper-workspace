元素集合映射

属性 描述 默认

目标类

作为集合的元素类型的类（基本或可嵌入）。

取

定义如何从数据库加载数据：

  • 渴望 - 数据在实际需要之前加载。

  • 延迟 - 仅在事务需要时加载数据。

懒

加入抓取

要使用的提取类型：

  • 内部 - 提供相关对象的内部联接提取。

    注意 ：内部连接不允许null或空值

  • 外部 - 提供相关对象的外部联接提取

    注意 ：外部连接允许空值或空值。

没有

收集表

名称

用于映射的数据库表的名称。

架构

集合表的数据库模式。

目录

集合表的数据库目录。

覆盖默认值

0
