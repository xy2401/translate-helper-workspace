创建一个makefile

出于本教程的目的，您被指示创建一个C ++项目，该项目要求您创建一个makefile。

要创建一个makefile：

  1. 在Project Explorer视图中，右键单击HelloWorld项目文件夹，然后选择New> File 。
  2. 在“ 文件名”框中，键入makefile 。
  3. 单击完成 。
  4. 在编辑器中键入下面的gnu make说明。行使用制表符缩进，而不是空格。


    all：hello.exe

    清洁：
    rm main.o hello.exe

    hello.exe：main.o
    g ++ -g -o hello main.o

    main.o：
    g ++ -c -g main.cpp
     

  5. 单击文件>保存 。

您的新makefile以及main.cpp文件将显示在Project Explorer视图中。您的项目现在包含main.cpp和makefile。您现在可以构建HelloWorld项目。

新点击此处查看插图 。
