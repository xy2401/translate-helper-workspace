为工作台分配默认JRE

默认JRE用于在所有项目中编译和启动Java程序，除非您专门覆盖项目构建路径或启动配置上的默认JRE 。

以下是更改默认JRE的方法：

  1. 打开 Java>已安装的JRE首选项页面。
  2. 选中要在工作台中指定为JRE的JRE行的框。如果要指定为默认的JRE未显示在列表中，则必须添加它 。
  3. 单击确定 。
注意 ：如果启用了自动构建，则更改默认JRE可能会导致构建（ 项目>自动构建或在 常规>工作区首选项页面）。
