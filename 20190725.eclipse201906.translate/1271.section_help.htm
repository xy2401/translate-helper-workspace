帮助科

关于本节

此可折叠部分允许您执行以下操作：

  • 使用帮助文档链接或帮助上下文ID作为输入，为步骤或简介添加帮助链接。
  • 从步骤或简介中删除帮助链接。
  • 编辑用作输入的帮助文档链接或帮助上下文ID，以向步骤或简介添加帮助链接。

通过在“ 内容”部分中选择步骤或简介节点，可以访问此部分。

部分字段

本节包含以下字段：

标签 类型 名称 描述 需要
类型 组合框 帮助链接类型 选择<无>以从步骤或简介中删除帮助链接。因此， 值输入字段将不再可见。

选择“ 帮助上下文ID”以使“ 值”输入字段在此字段下可见，并准备接受帮助上下文ID作为输入。

选择“ 帮助文档链接”以使“ 值”输入字段在此字段下可见，并准备接受帮助文档链接作为输入。 		没有
值 文本 帮助文档链接值或帮助上下文ID值 如果在类型字段中选择帮助上下文ID，在此字段中指定一个有效的帮助上下文ID来帮助链接添加到台阶或介绍。

在“ 备忘单”视图中，单击使用帮助上下文ID创建的步骤或简介的帮助链接将在“ 帮助”视图中打开该帮助上下文ID。

有效帮助上下文ID的格式如下： <plugin-id>。<help-context-id>

如果帮助文档链接在类型字段中选择，在该字段指定一个有效的帮助文档链接到一个帮助链接添加到台阶或介绍。

在“ 备忘单”视图中，单击使用帮助文档链接创建的步骤或简介的帮助链接将打开帮助信息中心中的帮助文档链接。

有效帮助文档链接的格式如下： / <plugin-id> / <path> / <html-file> 		没有
