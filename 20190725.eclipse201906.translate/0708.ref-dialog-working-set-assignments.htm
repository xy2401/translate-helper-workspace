工作集分配对话框

“工作集分配”对话框提供了从现有工作集列表中选中或取消选中一个或多个工作集的方法，以便将它们分配给先前选定的资源。

要打开此对话框，请从Package Explorer的上下文菜单中选择Assign Working Sets ....

选项

描述

默认

选择工作集

现有工作集的列表。如果“仅显示包资源管理器”工作集可用并已选中，则此列表将仅显示配置为在“包资源管理器”中显示的工作集。可以为所选资源检查或取消选中一个或多个工作集。

N / A

全选

选择所有工作集。

N / A

取消全选

取消选择所有工作集。

N / A

新...

创建一个新的工作集。

N / A

仅显示Package Explorer工作集

仅当Package Explorer将工作集显示为顶级元素时，此选项才可见。如果选中，则对话框仅显示当前配置为在Package Explorer中显示的工作集。

上

配置Package Explorer工作集

打开“ 配置工作集”对话框。仅当Package Explorer将工作集显示为顶级元素时，此选项才可用。
