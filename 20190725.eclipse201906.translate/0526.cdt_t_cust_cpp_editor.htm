自定义C / C ++编辑器

您可以更改许多C / C ++编辑器首选项。

要自定义C / C ++编辑器首选项：

  1. 单击窗口>首选项 。
  2. 展开C / C ++ ，然后单击C / C ++ Editor 。
  3. 要为编辑器设置常规首选项，请单击“ 常规”选项卡。
  4. 在常规页面上设置您的首选项，然后单击应用 。有关“常规”首选项的说明，请单击此处 。
  5. 要自定义C / C ++编辑器的可编辑区域中文本的颜色 ，请单击“ 颜色”选项卡。
  6. 在“颜色”页面上设置首选项，然后单击“ 应用”有关颜色首选项的说明，请单击此处 。
  7. 要自定义“内容辅助”首选项，请单击“ 内容辅助”选项卡。有关更多信息，请参阅使用内容辅助
  8. 单击确定 。
