设置CVS关键字替换模式

CVS使用文件的关键字替换模式来区分二进制文件和ASCII文件，并指示在提交和签出文件时要进行的关键字替换类型。

要设置CVS关键字替换模式：

  1. 在其中一个导航视图中，选择文件或包含需要更改关键字替换模式的文件夹。注意 ：确保将要提交的任何新文件添加到CVS版本控制中，因为只能为已经在CVS控制下的文件设置关键字替换模式。
  2. 从弹出菜单中，选择Team> Change ASCII / Binary Property 。将打开“设置关键字替换模式”向导。在org.eclipse.jdt.core项目上打开了以下对话框：

  3. 该向导包含选择中找到的所有创建的列表。在此页面上，您可以执行以下操作：
    • 您可以输入文件模式以过滤文件列表。
    • 您可以单独更改文件的模式。
    • 您可以使用页面底部的下拉列表选择多个文件并更改所有模式。
    • 您可以选择多个文件，然后单击“ 建议” ，根据您的首选项设置确定模式。
    • 您可以选中“ 仅显示将要更改的文件”选项，以仅查看向导中已更改模式的文件。
  4. 完成为任何文件指定新模式后，输入要与任何文件提交关联的提交注释。如果将文件类型从二进制更改为ASCII，则由于行终止符调整而导致文件内容发生更改，因此需要提交文件。单击“ 完成”以应用更改。
