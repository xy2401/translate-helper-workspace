从文件系统导入资源

您可以使用导入向导将资源从本地文件系统导入现有项目 。

  1. 从主菜单栏中选择 文件>导入....导入向导将打开。
  2. 选择General> File System ，然后单击Next 。
  3. 单击向导下一页上的“ 浏览”按钮，选择要添加资源的目录。
  4. 在导入选择窗格中，使用以下方法准确选择要添加的资源：
    • 展开左窗格中的层次结构，然后选中或清除表示所选目录中文件夹的复选框。然后在右侧窗格中，选择或清除单个文件的复选框。
    • 单击“ 筛选类型”以筛选特定类型文件的当前选择。
    • 单击全选以选择目录中的所有资源，然后浏览并取消选择您不想添加的资源。
    • 单击“ 取消全选”以取消选择目录中的所有资源，然后选择要添加的各个资源。
  5. 指定将成为导入目标的Workbench项目或文件夹。
  6. 完成指定导入选项后，单击“ 完成”。

提示：您还可以通过从文件系统中拖动文件夹和文件并将其拖放到其中一个导航视图中，或通过复制和粘贴来导入文件夹和文件。
