在公共源树中重叠产品

文件系统上的布局

  • 产品的Java源文件都保存在一个主目录中。
  • 产品分为四个兄弟姐妹包装Product1 ， Product2 ， Product3和Product4 。

定义相应的“Product1”和“Product2”项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“Product1”。单击下一步 。

  4. 在下一页上，选择“Product1”源文件夹。

    单击详细信息窗格或按钮中的链接其他源链接在视图栏中。

  5. 在Link Source中单击Browse ....并选择D:\AllJavaSourceFiles目录。

    在文件夹名称中键入“src”。

  6. 单击“ 完成”以验证并关闭对话框。

  7. 展开“src”源文件夹。选择最后三个包，并使用“ 从构建路径中 排除”链接或“ 排除”弹出菜单项将其从构建路径中排除 。

  8. 您的项目源设置现在如下所示：

  9. 单击完成 。

  10. 复制“Product1”项目并将其粘贴为“Product2”。

    编辑“Product2”项目属性，然后转到Java Build Path页面。

    在“ 源”选项卡中，展开“Product2 / src”源文件夹，选择“已排除” ，然后单击“ 编辑...” 。

  11. 在“ 包含和排除模式”中 ，选择“com / xyz / product2”，然后单击“ 编辑...” 。

  12. 将“com / xyz / product2”更改为“com / xyz / product1”。

  13. 单击“ 确定”以输入更改。

    单击“ 完成”以验证并关闭“ 包含和排除模式”对话框。

    再次单击“ 确定 ”以验证“Product2”项目属性更改。

  14. 您现在有两个Java项目，分别包含“product1”，“product2”的源代码。
