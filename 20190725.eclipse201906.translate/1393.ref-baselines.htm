API基线首选项

可以使用以下设置来设置以下首选项 插件开发> API Baselines首选项页面。

API基线定义了您希望将开发工作区捆绑包与二进制兼容性，包版本号和@since标记进行比较的状态。例如，如果您正在为Eclipse 3.4开发bundle，则可以使用Eclipse 3.3作为基准。


选项

描述

默认
添加基线...... 允许您添加新的API基准。  
编辑... 允许您编辑选定的API基准。  
去掉 允许您删除选定的API基准。  
缺少API基线 此首选项用于在API构建器运行时通知您，并且您没有默认的API基准集。 错误
