故障排除

检查你的ID

这是配置CNF时最常见的问题。确保ID与您期望的完全匹配。

它是通配符还是正则表达式？

确保导航器内容扩展或过滤器的includes和excludes ID实际上是正则表达式而不是通配符。因此，如果您正在寻找com.mycompany.myplugin* （这是一个通配符），请确保使用com.mycompany.myplugin.* （这是匹配的正则表达式）。前者会找到类似com.mycompany.mypluginnnnn ，但找不到com.mycompany.myplugin.function1 。

追踪

CNF提供的跟踪有助于解决问题。要启用跟踪，请转到“ 启动配置”对话框中的“ 跟踪”选项卡，然后选择“ 启用跟踪” 。然后检查org.eclipse.ui.navigator插件，并在右侧检查调试 ， 并执行下列操作之一：

  • debug / setup - 显示配置的处理。
  • debug / dnd - 显示拖放处理的所有方面。
  • debug / resolution - 显示对象到其导航器内容扩展的分辨率。
  • debug / viewermap - 显示将Navigator Content Extensions与查看器中的活动元素相关联的低级别映射。

如果需要支持，请在报告问题时包括跟踪。
