了解Java Persistence API

Java EE EJB 3.0规范的Java Persistence API（JPA）部分简化了Java持久性。它提供了一种对象关系映射方法，允许您以声明的方式定义如何以标准，可移植的方式将Java对象映射到关系数据库表。JPA既可以在Java EE应用程序服务器内部工作，也可以在Java Standard Edition（Java SE）应用程序中的EJB容器外部工作。根据JPA规范编写的应用程序是可伸缩的，事务性的和安全的。
