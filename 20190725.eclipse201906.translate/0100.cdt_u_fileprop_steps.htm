C / C ++文件属性，构建，设置，构建步骤选项卡

使用“ 构建步骤”属性面板自定义选定的构建配置，允许在正常构建处理执行之前和之后立即生成用户定义的构建命令步骤的规范，以及在构建输出中显示的描述性消息。

设置，构建步骤选项卡
选项 描述
自定义构建步骤适用性 选择“应用”或“禁用”自定义构建步骤。如果是apply，自定义构建步骤将覆盖标准构建步骤。
其他输入文件名 定义要在自定义生成步骤中使用的其他文件名。
输出文件名 定义自定义生成步骤生成的输出文件的名称。
命令 指定在自定义生成步骤期间执行的命令或以分号分隔的命令列表。
描述 指定与后构建步骤关联的可选描述性文本，该步骤将在执行构建后步骤命令之前立即显示在构建输出中。
