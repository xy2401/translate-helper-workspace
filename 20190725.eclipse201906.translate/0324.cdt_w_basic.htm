教程：创建一个简单的应用程序

在本教程中，您将使用CDT创建一个简单的“Hello World”应用程序。本教程描述了创建新C ++项目的过程，其中构建由CDT自动管理并运行程序。

注意 ：在早期版本的CDT中，有两种不同的项目类型：托管make（自动生成makefile）和Standard make（需要用户的makefile构建）。现在使用CDT，您只需选择一个项目类型，即可确定要使用的构建系统。

要使用CDT创建简单的“Hello World”应用程序，请执行以下常规步骤：

  1. 创建一个项目
  2. 查看代码并构建项目
  3. 运行应用程序

步骤1： 创建一个项目

  1. 选择文件>新建>项目 。


  2. 选择要创建的项目类型。在本教程中，展开C ++文件夹并选择C ++ Project ，然后单击Next 。



  3. “ C ++项目”向导将打开。



    默认情况下，CDT会根据您为本教程选择的C ++项目向导的语言支持，过滤它在这些列表中显示的工具链和项目类型 。


    1. 在“ 项目名称”字段中，键入项目的名称 ，例如HelloWorld 。


    2. 从Project type列表中，展开Executable并选择Hello World C ++ Project 。此项目类型在C ++中提供简单的Hello World应用程序，并且CDT自动创建makefile。


    3. 从工具链列表中选择所需的工具链 。


      工具链是一组用于构建项目的工具（如编译器，链接器和汇编程序）。其他工具（如调试器）可以与工具链相关联。可以使用多个工具链，具体取决于系统上安装的编译器。


    4. 单击下一步 。

  4. 指定新项目的基本属性 ，例如作者，版权和源信息，然后单击“ 下一步” 。



  5. “ 选择配置”页面根据之前选择的项目类型和工具链显示配置列表。



    可选 ：如果要更改默认项目设置，请单击“ 高级设置”以打开新项目的“ 项目属性”对话框，允许您更改任何项目特定设置，例如包含路径，编译器选项和库。

  6. 单击完成 。

注意如果当前未将C ++透视图设置为默认值，系统将提示您确定是否要将此项目与C / C ++透视图关联。单击是 。

使用默认设置和基于所选项目类型和工具链的一整套配置创建项目。您现在应该在Project Explorer视图中看到新项目。

             



第2步： 查看代码并构建项目

  1. 在Project Explorer视图中，双击为项目创建的.cpp文件，例如HelloWorld.cpp 。你会在项目“ src ”文件夹中找到它。

    此文件在默认编辑器中打开。它包含您之前选择的Hello World示例项目的C ++模板代码。此外， Outline视图还填充了从代码创建的对象。



    注意：您可以指定其他编辑器，并在“ 窗口”>“首选项”中添加或修改现有代码模板。

    可选 ：您可以在此文件中键入其他代码，然后单击文件>保存或按CTRL + S 保存更改。

    接下来，您需要在运行之前构建项目。

  2. 按CTRL + B构建项目，或在Project Explorer视图中选择项目，然后选择Project> Build Project 。


    注意：如果构建生成任何错误或警告，您可以在“ 问题”视图中查看这些错误或警告。如果遇到困难，请参阅构建C / C ++项目主题和开始之前
    。

  3. 在Console视图中阅读构建消息。该项目应该成功建立。

    您还将看到Outline视图也已填充了从您的代码创建的对象。如果从“ 大纲”视图中选择一个项目，则会突出显示编辑器中的相应文本。


第3步： 运行应用程序

要运行您的应用程序：

  1. 在C / C ++ Perspective中，单击Run> Run Configurations ....


  2. 选择C / C ++应用程序 。


  3. 单击新建 。

    将创建一个新的运行配置。它的名称和可执行文件的路径由项目提供（在我们的例子中是'Hello World'）。

  4. 单击运行 。


    现在，您应该看到在Console视图中运行的Hello World应用程序。控制台还会显示标题栏中正在运行的应用程序。


  5. 您可以单击其他视图以查看它们包含的信息。
