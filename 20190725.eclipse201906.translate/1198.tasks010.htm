创建查询

命名查询可以提高应用程序性能，因为它们只需准备一次，并且它们（以及它们所有相关的支持对象）之后可以有效地重用，使它们非常适合复杂且经常执行的操作。命名查询使用JPA查询语言在任何底层数据库上进行可移植执行;命名本机查询使用底层数据库本机的SQL语言。

使用此过程将@NamedQuery和@NamedNativeQuery注释添加到实体。

要创建命名查询：

  1. 在Project Explorer中选择实体。

  2. 在JPA Details视图中，展开Queries区域。

    JPA详细信息，查询选项卡


  3. 单击“ 添加”以添加新查询。出现“ 添加查询”对话框

    添加查询对话框


  4. 输入查询的名称，选择查询类型（ 命名查询或命名本机查询 ），然后单击“ 确定” 。 “ 查询”区域将展开以显示其他字段。

    JPA详细信息，查询选项卡


  5. 在“ 查询”字段中输入查询。

  6. 在“ 查询”页面上填写该字段的其余部分。

  7. 要添加“查询”提示，请单击“ 添加” 。输入提示的名称和值 。
