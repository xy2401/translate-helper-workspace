C / C ++文件属性，路径和符号，符号

使用“ 路径和符号”属性面板“ ＃符号”选项卡可以修改预处理程序符号列表。

路径和符号，符号选项卡
选项 描述
语言 （左侧窗格） 显示与此文件相关的语言。
符号/价值 与给定语言相关的符号列表。
加... 添加新符号。
编辑... 修改选定的符号值。
删除 删除所选符号。
导出/取消导出 切换是否导出所选符号。
显示内置值 检查显示系统（内置）包括。
导入设置... 导入符号设置。
导出设置... 导出符号设置。
