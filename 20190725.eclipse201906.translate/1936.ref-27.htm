Project Explorer视图

此视图提供Workbench中工件的分层视图，该视图由Workbench的特定配置自定义。

仅安装了Java开发工具（JDT）的典型Project Explorer看起来类似于Package Explorer：

要将Project Explorer视图添加到当前透视图，请单击 窗口>显示视图>其他...>常规>项目浏览器 。

 

工具栏

全部收缩

此命令折叠视图中所有资源的树扩展状态。

与编辑链接

此命令切换Project Explorer选择是否链接到活动编辑器。选择此选项后，更改活动编辑器将自动将Project Explorer选项更新为正在编辑的资源。


菜单

单击黑色倒置三角形图标以打开特定于Project Explorer的项目菜单。在视图内右键单击以打开上下文菜单。

选择工作集

打开“ 选择工作集”对话框以允许为Project Explorer选择工作集。

取消选择工作集

取消选择当前工作集。

编辑活动工作集

打开“ 编辑工作集”对话框以允许更改当前工作集。

除了这些菜单项之外，Project Explorer菜单还显示了在视图中选择的最近使用的工作集列表。


上下文菜单

新

此命令允许您在Workbench中创建新资源。从子菜单中选择要创建的资源类型。

复制

此命令将所选资源复制到剪贴板。

糊

此命令将剪贴板上的资源粘贴到选定的项目或文件夹中。如果选择了资源，则剪贴板上的资源将粘贴为所选资源的兄弟。

删除

此命令从工作空间中删除所选资源。

移动

此命令将所选资源移动到另一个位置。将出现一个对话框，提示您将资源移动到的目标位置。

改名

此命令允许您为所选资源指定新名称。

进口

此命令将打开导入向导，并允许您选择要导入Workbench的资源。

出口

此命令将打开导出向导，并允许您将资源导出到外部位置。

刷新

此命令刷新Workbench对所选资源及其子项的视图。例如，当您在Workbench外部为现有项目创建新文件并希望该文件显示在Project Explorer视图中时，将使用此方法。

关闭项目

选择打开的项目时，将显示关闭项目命令。此命令将关闭所选项目。

关闭不相关的项目

此命令将关闭与所选项目无关的任何项目。

打开项目

选择已关闭的项目时，可以看到打开的项目命令。此命令将打开所选项目。

球队

Team子菜单中的菜单项与版本控制管理相关，由与项目关联的版本控制管理系统确定。Eclipse为不受版本控制管理的项目提供特殊菜单项Share Project ....此命令将显示一个向导，允许您选择与已添加到Eclipse的任何版本控制管理系统共享项目。Eclipse附带了对CVS的支持。

与之比较

Compare With子菜单上的命令允许您执行以下类型的比较之一：

  • 将两个或三个选定的资源相互比较
  • 将所选资源与远程版本进行比较（如果项目与版本控制管理系统相关联）。
  • 将所选资源与本地历史状态进行比较

选择要执行的比较类型后，您将看到比较编辑器或比较对话框。在比较编辑器中，您可以浏览和复制比较资源之间的各种更改。在比较对话框中，您只能浏览更改。

用。。。来代替

“替换为”子菜单上的命令允许您将所选资源替换为本地历史记录中的另一个状态。如果项目受版本控制管理，则版本控制管理系统可能还提供其他项目。

属性

此命令显示所选资源的属性。显示的属性类型取决于所选的资源类型。资源属性可能包括（但不限于）：

  • 相对于其所在项目的路径
  • 资源类型
  • 使用链接资源时的绝对文件系统路径或路径变量名称
  • 为链接资源使用路径变量时已解析的路径变量
  • 资源规模
  • 上次修改日期
  • 只读状态
  • 派生资源状态
  • 执行参数，如果它是可执行资源
  • 程序发射器，如果它是可发射的
  • 项目依赖项，如果有的话
