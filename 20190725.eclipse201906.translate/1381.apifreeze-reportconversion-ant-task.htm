API Tools API冻结报告转换Ant任务

目的

此任务将apitooling.apifreeze Ant任务创建的XML报告转换为HTML文件。

用法

描述

Ant任务的名称是： apitooling.apifreeze_reportconversion 。要使用，jar文件apitooling-ant.jar必须位于Ant类路径上。 

<apitooling.apifreeze_reportconversion
	xmlfile="..."
	htmlfile="..."
	debug="..."
/>

参数

属性 描述 需要
XMLFILE 此属性指定要转换的xml文件的位置。

使用绝对路径指定位置。
这必须是文件位置，而不是目录。 		是
HTMLFILE 此属性指定要生成的html文件的位置。

如果未设置，则通过替换“.html”中的“.xml”从xml文件名中检索html文件名。

使用绝对路径指定位置。
这必须是文件位置，而不是目录。 		没有
调试 设置调试值。

可能的值为： true ， false
默认值为false 。 		没有

例子

 <apitooling.apifreeze_reportconversion
  xmlfile=="D:\report\report.xml"
  debug="true"
 />

这将在文件夹report.html运行创建D:\report的任务，因为未设置htmlfile属性。

由于启用了调试，因此一些调试跟踪将显示在Ant控制台中。
