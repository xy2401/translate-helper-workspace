检测现有布局

文件系统上的布局

  • 产品的源文件放在一个目录“src”中。
  • 类文件位于另一个目录“bin”中。

定义相应项目的步骤

  1. 点击 Window> Open Perspective> Other ...> Java改为Java透视图。

  2. 点击 File> New> Other ...> Java Project打开New Java Project向导。

  3. 在项目名称字段中键入“产品”。

  4. 在“ 项目布局”组中，将选择更改为“ 创建单独的源和输出文件夹

    取消选择使用默认位置 。

    单击Browse ...并选择驱动器Product上的D:目录。

  5. 单击下一步 。

    确保检测到源文件夹和输出文件夹。

    警告：如果Scrub输出文件夹时清除项目首选项的输出文件夹部分选中Java> Compiler> Building首选项页面，单击Finish将在生成类文件之前清除文件系统中的“bin”目录。

  6. 单击完成 。
  7. 您现在有一个带有“src”文件夹的Java项目，该文件夹包含“Product”目录的源代码。

注意：此解决方案在“Product”目录中创建“.project”文件和“.classpath”文件。如果您不希望在“Product”目录中包含这些文件，则应使用链接文件夹，如共同源树部分中的Sibling产品中所示。
