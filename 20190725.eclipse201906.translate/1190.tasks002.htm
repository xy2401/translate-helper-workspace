从Builder XML创建数据库Web服务

使用此过程可以创建新的JPA项目。

  1. 从Navigator或Project Explorer中，选择File> New> Project 。将出现“选择向导”对话框。

    从Builder XML向导中选择Web服务


  2. 从Builder XML中选择Database Web Services> Web Services ，然后单击Next 。将显示“ Web动态”页面 。

    Web动态项目页面


  3. 选择Dynamic Web Project，然后单击Next 。将出现“ 选择构建器XML文件”页面 。

    选择构建器XML文件页面


  4. 选择XML文件，然后单击“ 下一步” 。出现“ 驱动程序文件”页面

    驱动程序文件页面


  5. 单击“ 添加”以将其他JAR文件添加到项目中。

  6. 单击完成 。Dali添加了新的JAXB项目。

    您现在应该打开JPA Development透视图 。
