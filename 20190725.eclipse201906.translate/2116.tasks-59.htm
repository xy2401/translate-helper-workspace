将资源导出到文件系统

您可以使用“导出”向导来将资源从Workbench导出到文件系统 。

  1. 在其中一个导航视图中，选择要导出的资源。
  2. 从主菜单栏中选择 文件>导出....“导出”向导将打开。
  3. 选择General> File System ，然后单击Next 。
  4. 默认情况下，将导出您选择的资源及其所有子项。（可选）使用左窗格和右窗格中的复选框选择要导出的资源集，并使用“ 过滤器类型”等按钮来过滤要导出的文件类型。
  5. 单击向导下一页上的“ 浏览”按钮，选择要将资源导出到的目录。
  6. 指定文件系统中将成为导出目标的目录。
  7. 单击完成 。

提示：您还可以通过将文件夹和文件从其中一个导航视图拖动到文件系统并将其拖放到文件系统中，或通过复制和粘贴来导出文件夹和文件。
