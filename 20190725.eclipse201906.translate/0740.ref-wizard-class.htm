新的Java类向导

该 New Java Class向导可帮助您在Java项目中创建新的Java类。

Java类选项

包 输入包以包含新类。您可以选择此选项或下面的Enclosing Type选项。键入有效的包名称或单击“ 浏览”以通过对话框选择包。 启动向导时选择的元素包。如果选择的源文件夹只包含一个包，则使用该包。如果源文件夹不包含任何包，则使用项目名称（如果它是有效的包名称）。

您想要创建哪些方法存根？

选择要在此类中创建的方法存根：

  • public static void main（String [] args）：将主方法存根添加到新类。
  • 来自超类的构造函数：从新类的超类复制构造函数，并将这些存根添加到新类中。
  • 继承的抽象方法：从超类或需要实现的接口方法添加到任何抽象方法的新类存根。
