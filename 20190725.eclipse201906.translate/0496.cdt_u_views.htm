选择视图和编辑器

要查看所有视图的列表，请从菜单栏中选择“ 窗口”>“显示视图”>“其他” 。

以下视图包含C / C ++项目视图：

基本观点

安慰
显示应用程序的输出。
Project Explorer
显示launchdir / workspace目录下的文件系统。
大纲
显示源文件中的函数和头文件。在编辑器中打开源文件以查看其大纲。
问题
显示问题。
属性
显示Project Explorer视图中突出显示的资源的名称，路径，大小，权限和上次修改日期。
搜索
显示文件或文本搜索的结果。

C次观看

C / C ++项目
显示您当前的项目。这与Project Explorer视图类似。

发表意见

制作目标
列出您的项目。要构建项目，请双击它。

编辑观点

“ 窗口”>“显示视图”或“ 窗口”>“显示视图”>“其他”下未列出“ 编辑器”视图，只要从“C / C ++项目”或“项目浏览器”视图中打开可编辑文件，它就会打开。
