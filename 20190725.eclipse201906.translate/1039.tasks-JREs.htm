使用JRE

您可以根据需要安装任意数量的Java Runtime Environments（JRE）。JRE用于运行和调试java程序。您的JRE在Installed JREs首选项页面上进行管理。

JRE定义包括：

  • JRE的类型（例如标准VM或标准1.xx VM）
  • 一个名字
  • JRE的安装位置
  • 包含Java系统类的系统库（如java.lang。宾语）。（可选）系统库可以与以下内容相关联：
    • 将源文件包含在JRE的类文件类的源
    • 一个javadoc位置（URL）
    • 外部注释 。
  • VM构建，运行和调试应用程序所需的其他信息。

您可以切换工作台的默认JRE。在构建，运行和调试应用程序时，默认使用默认JRE。或者，项目可以指定应该构建和运行的特定JRE。
