评估片段

在本节中，您将使用Java剪贴簿评估Java表达式。Java剪贴簿页面允许您在将Java代码片段放入程序之前对其进行试验。

  1. 在File菜单中，选择New> Other> Java> Java Run / Debug> Scrapbook Page 。系统将提示您输入页面的文件夹目标。

  2. 在Enter或选择文件夹字段中，键入或浏览以下以选择JUnit项目根目录。
  3. 在“ 文件名”字段中，键入MyScrap 。
  4. 完成后单击“ 完成” 。使用jpage文件扩展名为您创建剪贴簿页面资源。（如果您不自行输入，则会自动添加jpage文件扩展名。）剪贴簿页面会在编辑器中自动打开。
  5. 在编辑器中，键入System.get ，然后使用内容辅助（ Ctrl+Space ）将片段完成为System.getProperties() 。

  6. 选择刚刚键入的整行，然后从上下文菜单中选择“ 显示 ”。您还可以从工具栏中选择“评估所选文本的显示结果” 。

  7. 评估完成后，评估结果将在剪贴簿页面中显示并突出显示。
  8. 您可以通过选择文本，然后从上下文菜单中检查检查评估的结果（或选择检查从工具栏中所选文本评估结果 ）。
  9. 完成剪贴簿页面中的代码片段评估后，可以关闭编辑器。如果要保留代码段以备将来使用，请在页面中保存更改。
