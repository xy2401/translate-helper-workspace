API基线向导

API基准向导可用于：

  • 向工作区添加新的API基准。
  • 编辑现有API基准。

该向导可以从中使用 插件开发> API Baselines首选项页面首选项页面，通过选择Add Baseline ...按钮（添加新基线）或Edit ...按钮（编辑现有基线）。

选项

描述

默认
名称 您可以为基准选择任何名称，但不能包含系统限制字符。例如，在Windows上，您不能在文件名中使用“/”。 空
地点 API基线的位置是Eclipse SDK安装的路径，通常是安装的“ plugins ”文件夹的路径。 空
浏览... 选择“浏览”按钮将允许您浏览Eclipse SDK安装的位置。  
重启 选择“重置”按钮将重新扫描指定的位置并重新创建API基准。
