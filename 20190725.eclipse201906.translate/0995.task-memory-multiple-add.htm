添加多个内存渲染并删除渲染

将变量，表达式或注册表添加到“内存”视图时，可以多次执行此操作，每次添加新的（或相同的）渲染。或者，添加内存监视器并进行渲染后，可以转到“ 渲染”窗格并单击“ 添加渲染”按钮（ ）。这将提示您使用对话框选择要添加到视图的渲染。在此对话框中，您可以使用键盘Shift或Ctrl键选择多个渲染 - 这样做会导致为所选的每种渲染格式打开渲染。添加多个内存渲染时，它们由制表符分隔。

您还可以通过选择“ 切换拆分窗格”按钮来拆分“ 内存渲染”窗格（ ）。拆分“ 渲染”窗格时，可以并排查看两个渲染。

当您为内存监视器提供多个内存渲染时，可以将渲染设置为彼此链接。为此，请选择“ 链接内存渲染窗格”按钮（ ）。当渲染渲染时，它们彼此同步（例如，如果在一次渲染中更改列大小，则另一种渲染中的列大小也会更改 - 或者如果在一次渲染中滚动或移动光标，则另一种渲染渲染将滚动或遵循相同的光标移动）。链接内存渲染仅适用于当前的内存视图。如果您打开了多个内存视图，则它们不会相互链接。

要删除渲染，请在“ 渲染”窗格中选择它，然后单击“ 删除渲染” （ ）。删除受监视表达式，变量或寄存器的所有内存渲染时，将使用内存呈现选择列表填充“ 渲染”窗格。从此列表中，您可以选择要用于内存呈现的数据格式，然后单击“ 添加渲染” 。
