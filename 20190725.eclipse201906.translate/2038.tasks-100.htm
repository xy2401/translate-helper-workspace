创建项目的版本

先决条件 ：您要版本的项目必须首先位于工作区中。

提示 ：通常最好在版本控制之前将资源与CVS存储库同步。这将确保没有可能意外地从版本中排除的未完成的传入或传出更改。

制作项目的版本：

  1. 在其中一个导航视图中，选择要版本化的项目。
  2. 从项目的上下文菜单中，选择Team> Tag as Version 。
  3. 输入您要创建的版本的名称。
  4. 单击“ 确定”关闭“标记资源”对话框并创建版本。

提示：您可以通过单击“标记资源”对话框中的“ 详细信息”按钮来浏览项目的现有版本。单击“ 从存储库刷新”应填充“现有版本”列表。在命名新版本之前查看现有版本有时会很有帮助。
