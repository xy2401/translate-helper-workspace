安装和更新CDT

您可以使用软件更新管理器使用Internet连接直接从Eclipse工作台快速安装或更新CDT。

要获得最新的CDT功能：

  1. 选择“ 帮助”>“软件更新”>“查找并安装” 。

    单击此处查看插图（显示在单独的窗口中） 。


  2. 在“ 功能更新”窗口中，选择“ 搜索要安装的新功能”，然后单击“ 下一步” 。

    注意：如果要搜索当前已安装的功能的更新，可以选择“ 搜索当前安装的功能的更新 ”选项，然后按照步骤使用该方法进行更新。

    点击这里查看插图 。


  3. 如果以前未更新，则必须输入Eclipse CDT安装站点的位置。单击“ 新建远程站点...” 。

    点击这里查看插图 。


  4. 在“新建更新站点”对话框中，键入Eclipse更新站点的名称和URL。

    点击这里查看插图 。


  5. 单击确定 。


  6. 从要包含在搜索列表中的站点，通过单击相应的复选框选择刚刚创建的更新站点，然后单击完成 。

    点击这里查看插图 。


  7. “搜索结果”对话框显示可从更新站点安装的功能。展开并选择所需的任何其他功能。取消选择任何不适合您的选项。


  8. 单击下一步 。

    点击这里查看插图 。


  9. 您必须接受Eclipse.org软件用户协议才能继续安装CDT更新。如果您接受这些条款，请选择我接受许可协议中的条款 ，然后单击下一步 。

    点击这里查看插图 。


    您可以查看选择安装的功能的摘要列表。


  10. 可选：单击“ 更改位置”以选择要安装更新的位置。通常，这是您安装Eclipse的目录。


  11. 单击完成 。

    点击这里查看插图 。


  12. 您现在将开始下载CDT组件。您必须通过单击“ 安装”为所选的每个功能验证是否要安装这些功能。

    点击这里查看插图 。


    现在，您必须重新启动Eclipse。


  13. 选择“ 是”以完成功能安装过程。

    点击这里查看插图 。
