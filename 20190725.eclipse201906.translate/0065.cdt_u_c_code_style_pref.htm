代码样式首选项

使用 “代码样式”首选项面板，用于配置智能输入功能的全局代码样式配置文件，如自动缩进和格式设置。单击窗口>首选项> C / C ++>代码样式进行更改。

代码样式首选项选项
选项 描述
配置项目特定设置... 打开“ 项目特定配置”对话框以选择要应用代码样式设置的项目。
编辑... 显示一个对话框，显示存储在活动配置文件中的设置。只能修改用户定义的配置文件。
去掉 删除所选的配置文件。此操作仅适用于用户定义的配置文件。
新... 显示用于创建新配置文件的对话框。该对话框要求您输入新配置文件的名称。此外，您可以选择内置或用户定义的现有配置文件，以基于您的新配置文件。
进口... 从文件系统导入配置文件。
