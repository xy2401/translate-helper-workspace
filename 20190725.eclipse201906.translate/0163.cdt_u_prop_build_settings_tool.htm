C / C ++项目属性，构建，设置，工具设置选项卡

使用“ 工具设置”属性选项卡可以自定义构建配置中使用的工具和工具选项。

工具设置选项
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
（左窗格） 显示工具及其选项类别的列表。从列表中选择所需的工具以修改其选项。
（右窗格） 显示可以为所选工具修改的选项。此选项列表会根据您在左窗格中为特定工具选择的选项类别而更改。

适用于： CDT项目和源文件
