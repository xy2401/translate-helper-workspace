本教程将帮助您开始在Workbench中使用C / C ++开发工具包（CDT）。

首先，您将验证是否为C / C ++开发正确配置了工作台。假设：

  • 您的Workbench具有默认设置。要将当前透视图重置为其原始布局，请从菜单栏中选择“ 窗口”>“重置透视图” 。
  • 您熟悉基本的Workbench功能，例如视图和透视图 。

设置全局编辑器操作和外观

配置全局编辑器相关首选项的主要位置是常规>编辑器首选项页面（也可通过“ 窗口”>“首选项”菜单项访问）

设置C / C ++编辑器的操作和外观

C / C ++ - 特定的编辑器首选项 C / C ++>编辑。 它们包括以下内容：

  • 编辑器 - 控制文本突出显示的外观
  • 内容辅助 - 自定义内容辅助功能
  • 折叠 - 自定义折叠行为
  • 悬停 - 自定义悬停行为
  • Mark Occurrences - 设置注释
  • 保存操作 - 配置保存文件时执行的操作
  • 可伸缩性 - 配置处理大型源文件的行为
  • 语法着色 - 指定如何呈现C / C ++源代码
  • 模板 - 配置新生成的代码存根，注释和文件的格式
  • 键入 - 在键入时配置格式输入
