Ant运行时首选项

可以更改以下首选项 Ant>运行时首选项页面。

类路径选项

在Classpath选项卡上，您可以向Ant类路径添加定义任务和类型的其他类。可以将新任务和类型添加到Ant Home或Global Entries部分。“贡献条目”部分无法修改。

选项 描述 默认
添加罐子...... 允许您将当前工作空间中的jar文件添加到Ant Home和/或Global部分的Ant类路径中。  
添加外部罐子...... 允许您将本地文件系统中的jar文件添加到Ant Home和/或Global部分的Ant类路径中。  
新增文件夹... 允许您将本地文件系统中的文件夹添加到Ant Home和/或Global部分的Ant类路径中。  
添加变量... 允许您将新的类路径变量添加到Ant Home和/或Global部分。  
蚂蚁家 允许您将新文件夹添加到Ant Home位置。  
去掉 允许您从Ant类和/或全局部分的Ant类路径中删除选定的条目元素。  
向上 允许您将选定的条目元素向上移动到其列表中的一个空格。  
下 允许您将选定的条目元素向下移动到其列表中的一个空格。

任务选项

在“任务”选项卡上，您可以添加在Ant类路径中的一个类中定义的任务。

选项 描述 默认
添加任务... 允许您向运行时添加新的Ant任务定义。  
编辑任务... 允许您编辑现有Ant任务。

注意：您只能编辑未贡献的任务（即黄色装饰任务已提供且无法编辑）。		  
去掉 允许您从运行时删除现有Ant任务。

注意：您只能删除未提供的任务（即黄色装饰的任务已被提供且无法删除）。

类型选项

在“类型”选项卡上，您可以添加在类路径中的一个类中定义的类型。

选项 描述 默认
添加类型... 允许您添加在Ant类路径中的一个类中定义的新类型。  
编辑类型... 允许您编辑现有类型。  
去掉 允许您删除现有类型。

属性选项

在“属性”选项卡上，您可以添加将传递给Ant的属性和属性文件。

选项 描述 默认
添加属性... 允许您向运行时添加新的Ant属性。  
编辑属性... 允许您编辑现有的Ant属性。

注意：您只能编辑未提供的属性（即黄色装饰属性已提供且无法编辑）。		  
去掉 允许您从运行时中删除现有Ant属性。

注意：您只能删除未提供的属性（即黄色装饰属性已提供且无法删除）。		  
添加文件... 允许您从要在运行时中使用的工作空间添加*.properties文件。  
添加外部... 允许您从本地文件系统添加*.properties文件以在运行时中使用。  
去掉 允许您从运行时删除现有的*.properties文件。
