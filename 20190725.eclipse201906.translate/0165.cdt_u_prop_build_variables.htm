变量属性（C / C ++项目属性）

为工作区中的所有项目自定义Eclipse CDT构建环境变量;它允许您控制构建使用的环境变量，以便构建所选配置。


变量首选项

选项 描述
要设置的环境变量 显示当前环境变量列表及其对应值。这些是运行应用程序时使用的环境变量值。
组态 点击此处查看说明。
管理配置 点击此处查看说明。
（变量表） 按名称 ， 类型和值列出CDT构建变量和系统变量。自定义构建变量由使用粗体字体突出显示的名称标识。
名称 指示变量的名称，用户定义的变量或系统变量。
类型 显示变量的类型。
值 指定变量的值。
显示系统变量 选择后，系统变量将包含在“变量”表中;否则，它们被排除，以便只有用户定义的变量出现在Variables列表中。
加 创建一个新变量和相应的值。
编辑 修改所选变量的名称和值。
删除 从列表中删除选定的变量。注意：某些变量是只读的，无法删除。
