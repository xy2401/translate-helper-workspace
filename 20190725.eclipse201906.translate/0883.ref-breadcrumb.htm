调试面包屑

Debug Breadcrumb是当前活动的调试上下文及其父项的紧凑表示。

要启用Debug Breadcrumb，请将Debug View布局更改为Breadcrumb（在Debug View菜单中选择Layout-> Breadcrumb ）。或者，调整调试视图的大小，使视图的高度足够小，只适合一行文本。这将导致视图自动切换到Breadcrumb布局。

Debug Breadcrumb允许对面包屑中显示的任何元素调用操作。可以通过上下文菜单或通过键绑定来调用操作。要选择面包屑中的元素，请单击它或激活“调试”视图并使用向左和向右箭头键。

您还可以使用痕迹更改活动的调试上下文。单击其中一个箭头或任何元素顶部打开一个下拉窗口。下拉列表显示打开下拉列表的元素的所有兄弟节点。单击元素以使其成为新的活动调试上下文。要使用键盘打开下拉列表，请选择和元素，然后按向下键。
