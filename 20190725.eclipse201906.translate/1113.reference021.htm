实体

下表列出了一个JPA的详细信息视图中提供的实体信息领域的实体 。

属性 描述 默认

表

此实体的默认数据库表信息。“ 属性覆盖 ”（ Attribute Overrides）区域中的信息可以覆盖这些字段。

名称

与实体关联的主数据库表的名称。

目录

包含表的数据库目录。

如orm.xml所定义。

架构

包含表的数据库架构。

如orm.xml所定义。

名称

该实体的名称。默认情况下，类名用作实体名称。

访问

指定实体的访问实例变量。

  • 属性 - 通过属性访问器方法访问的持久状态。属性访问器方法必须是公共或私有的 。

  • 字段 - 直接访问实例变量。所有非瞬态实例变量都是持久的。

注意 ：此字段仅用于显示，基于orm.xml的属性：如果仅注释类的方法，则使用属性访问类型。在所有其他情况下，使用字段访问类型。

领域

ID类

单击“ 浏览”并选择实体的主键。单击表示为超链接的字段名称可以创建新类。
