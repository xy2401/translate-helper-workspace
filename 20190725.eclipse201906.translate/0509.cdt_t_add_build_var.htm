添加和编辑构建变量

使用“ 定义新建立变量”对话框添加本地构建变量或“ 编辑现有构建变量”对话框以编辑先前定义的变量。两者都可以从 “构建变量”首选项面板。

定义新的构建变量

  1. 单击窗口>首选项> C / C ++>构建>构建变量
  2. 单击Build Variables首选项面板中的Add ...以打开Define a New Build Variables对话框
  3. 输入变量名称
  4. 选择变量类型
  5. 输入变量值
  6. 单击确定

定义新的构建变量/编辑现有构建变量对话框选项
选项 描述
变量名 输入变量名称。
类型 从可用列表中选择变量类型。
值 输入或编辑变量的值。

编辑现有构建变量

  1. 单击窗口>首选项> C / C ++>构建>构建变量
  2. 在Build Variables首选项面板中选择要编辑的本地构建变量
  3. 单击编辑...以打开“ 编辑现有构建变量”对话框
  4. 更改变量类型（可选）
  5. 编辑变量值（可选）
  6. 单击确定
