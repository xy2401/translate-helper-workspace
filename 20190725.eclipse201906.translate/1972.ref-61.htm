帮助菜单

此菜单提供有关使用Workbench的帮助。

欢迎

此命令将打开欢迎内容。

帮助内容

此命令在帮助窗口或外部浏览器中显示帮助内容。帮助内容包含与Workbench和已安装功能相关的帮助手册，主题和信息。

搜索

此命令显示“搜索”页面上打开的帮助视图。

动态帮助

此命令显示打开到“相关主题”页面的帮助视图。

关键辅助......

此命令将显示键绑定列表

技巧和窍门 ...

此命令将打开您可能未发现的有趣生产力功能列表。

备忘单 ...

此命令将打开备忘单选择对话框。

检查更新

此命令将检查已安装软件的更新。

安装新软件......

此命令允许您下载和安装新软件。

关于Eclipse SDK

此命令显示有关产品，已安装功能和可用插件的信息。
