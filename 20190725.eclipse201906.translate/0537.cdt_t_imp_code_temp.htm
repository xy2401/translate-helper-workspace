导入和导出模板

您可以导入和导出模板。

注意：模板必须是.xml文件，格式如下：

<?xml version="1.0" encoding="UTF-8" ?> 
<templates>
<template name="author" description="author name" context="C" enabled="true">author ${user}</template> 
</templates>

要导入模板

  1. 单击窗口>首选项 。
  2. 展开C / C ++ ，展开编辑器 ，然后单击模板 。
  3. 单击导入 。
  4. 选择要导入的模板文件。
  5. 单击确定 。
    模板列表将更新为包含您导入的模板。

导出模板

  1. 单击窗口>首选项 。
  2. 展开C / C ++ ，展开编辑器 ，然后单击模板 。
  3. 选择要导出的模板，然后单击“ 导出...”按钮。“导出模板”对话框打开。
  4. 在“ 文件名”框中，键入要保存模板文件的路径。
  5. 单击保存 。
    包含您导出的模板的templates.xml文件保存在文件系统中。
