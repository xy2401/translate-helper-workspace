选择要运行或调试的应用程序

您需要指定要为此运行配置运行或调试的项目或程序。

要选择要运行的应用程序：

  1. 在“C / C ++项目”视图中，选择一个项目。
  2. 单击运行>运行或运行>调试 。
  3. 在“ 配置”框中，展开“ C / C ++ Local” 。
  4. 选择运行或调试配置。
  5. 单击主选项卡。
  6. 请执行下列操作：
    • 在“ 名称”框中，为此新启动配置的新描述性名称键入一个描述性名称。
    • 在“ 项目”框中，键入包含要运行的应用程序的项目的名称。
    • 在“ C / C ++应用程序”框中，键入要运行的可执行文件的名称。
  7. 单击“运行”或根据需要执行以下操作：
    • 要指定应用程序使用的执行参数，并指定运行配置的工作目录，请参阅指定执行参数 。
    • 要设置应用程序运行时要使用的环境变量和值，请参阅设置环境变量
    • 要选择调试应用程序时要使用的调试器 ，请参阅选择调试器
    • 要指定调试C或C ++应用程序时使用的源文件的位置 ，请参阅指定源文件的位置
    • 要指定运行配置的存储位置，访问方式以及运行应用程序时要打开的透视图，请参阅指定运行配置的位置
