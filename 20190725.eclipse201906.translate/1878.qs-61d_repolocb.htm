指定存储库位置

必须先指定可用的存储库，然后才能与其他人共享项目。

  1. 打开CVS Repository Exploring透视图。左上角的视图显示了当前正在使用的所有CVS存储库。如您所见，视图为空，这意味着仍需要指定存储库。

  2. 在CVS Repositories视图的上下文菜单中，选择New> Repository Location 。

  3. 在CVS存储库位置向导中，需要填写存储库的位置和登录信息。您可能需要存储库管理员的帮助才能填写必要的信息。

  4. 在“ 主机”字段中，键入主机的地址（例如，“teamsamples.com”）。

  5. 在“ 存储库路径”字段中，键入主机地址处的存储库路径（例如，“/ home / cvsroot / repositoryName”）。

  6. 在“ 用户”字段中，键入要连接的用户名。

  7. 在“ 密码”字段中，键入密码。

  8. 在“ 连接类型”字段中，选择存储库的CVS连接类型（默认为pserver）。

  9. 保持使用默认端口 。

  10. 默认情况下，选中“ 完成时验证连接” 选项 。

  11. 完成后单击“ 完成” 。

    由于检查完成后验证位置 ， 现在，向导将尝试通过连接到存储库来验证信息。这样做可能会提示输入密码。注意：存储库连接仅用于验证信息。

  12. 观察CVS存储库视图现在显示新的存储库位置。
