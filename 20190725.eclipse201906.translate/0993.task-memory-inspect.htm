在Memory视图中检查内存

使用Memory视图，您可以查看特定地址的内存内容。可以从引用变量或寄存器的表达式获取地址。监视内存时，可以将监视器设置为以各种数据格式呈现。

Memory视图由两部分组成：

  • “ 监视器”窗格位于视图的左侧。视图的这一部分包含您为监视添加的表达式，变量和寄存器的列表。
  • “ 渲染”窗格位于视图的右侧（如果视图设置为水平方向）。您可以使用它来设置要为受监控内存显示的数据格式（或格式）。

从“ 监视器”窗格监视地址或表达式时，“ 渲染”窗格将填充调试器提供的一组默认渲染。从“ 渲染”窗格监视地址或表达式时，窗格将填充一个渲染列表，您可以从中选择要显示的渲染。

您可以在“内存”视图中监视多个变量，表达式和寄存器 - 并且可以向“ 渲染”窗格添加多个渲染。在“ 监视器”窗格中，列出了已添加的每个变量，表达式或寄存器。在“ 渲染”窗格中，仅显示“内存”视图中当前所选监视器的内存渲染（多个渲染由制表符或拆分窗格分隔）。

您可以在“内存”视图中设置两个窗格，以水平方向（并排）或垂直方向（从上到下）显示。要设置视图的布局，请单击“内存”视图向下箭头图标，然后从菜单中选择“ 布局 ”。这将打开一个子菜单，您可以从中选择要显示的方向。
