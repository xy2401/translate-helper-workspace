大纲视图

“大纲”视图通过列出结构元素，显示当前在编辑器区域中打开的结构化C / C ++文件的大纲。

“大纲”视图按发生顺序显示源文件中的以下元素：

  • 类
  • 命名空间
  • 包括
  • 枚举
  • 枚举
  • 现场私人
  • 现场保护
  • 现场公众
  • 包括
  • 方法私有
  • 方法受保护
  • 方法公开
  • 结构
  • 的typedef
  • 联盟
  • 变量
  • 功能
  • 宏定义

您也可以按字母顺序对列表进行排序。在“大纲”视图中选择元素时，C / C ++编辑器会突出显示所选项目和标记栏（左边距）。例如，要在C / C ++编辑器中移动到main（）的开头，请在“大纲”视图中单击main() 。

有关标记栏的详细信息，请参阅“ 工作台用户指南”>“参考”>“用户界面信息”>“视图和编辑器”>“编辑器”区域 。

过滤大纲视图

您可以通过选择显示或隐藏以下项来筛选“大纲”视图：

  • 字段
  • 静态成员
  • 非公开成员

您可以在“大纲”视图中选择一个元素，然后执行以下操作：

  • 打开“C / C ++搜索”窗口框。将填充“搜索”字符串框，并选择元素类型。
  • 完成基于文本的搜索，工作空间或所选元素的指定工作集。
  • 在编辑器中打开选定的.h文件。
  • 重命名重构
