手动构建

手动构建允许您选择构建的范围，以及构建或重建项目的选项。您可以在控制台中查看make实用程序的输出。

增量构建
要逐步构建所有打开的项目，可以选择“ 项目”>“全部构建”或键入CTRL + B.
建立个人项目
要构建单个项目，请单击Project> Build Project 。
重建项目
要重建项目，请右键单击该项目，然后选择“ 重建项目” 。
注意：这将重建此项目引用的项目，但不会重建引用此项目的项目。
自动构建
无论何时保存任何项目文件（例如makefile） ，都会执行Build All。
提示：对于C / C ++项目，应关闭此功能。如果“ 自动构建”旁边有复选标记，则表示该功能已启用。要将其关闭，请选择“ 自动构建” 。

提示：如果收到错误消息：

Exec error:Launching failed

然后CDT无法找到构建命令（通常是make ）。您的路径配置不正确，或者您的系统上没有安装make 。

提示： C / C ++项目的菜单栏项目Project> Build Working Set子菜单只是创建了一个指向makefile中定义的build all目标的链接，并且与Incremental Build没有什么不同。
