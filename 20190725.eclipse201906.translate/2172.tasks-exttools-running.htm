运行外部工具

Workbench提供了一种运行不属于它的工具的机制。要配置外部工具：

  1. 点击 运行>外部工具>外部工具....
  2. 选择程序配置
  3. 单击“ 新建”按钮。
  4. 输入外部工具的名称（例如，我的外部工具）。
  5. 单击“ 浏览文件系统”按钮。
  6. 找到要运行的工具（例如，在Windows上，它通常是扩展名为.exe或.bat的文件）。
  7. （可选）在“ 参数”字段中，输入该工具的必要参数。
  8. （可选）在“ 工作目录”字段中，输入该工具的工作目录。
  9. 单击运行 。

这些步骤创建了持久启动配置。新创建的配置将显示在“ 运行”>“外部工具”下的启动历史记录中，并将在通过单击打开的启动配置对话框中可用 运行>外部工具>外部工具....

也可以设置和运行外部工具来构建项目。

  1. 选择所需的项目。
  2. 从弹出菜单中选择“ 属性” 。
  3. 单击列表中的“ 构建器 ”，然后按上述方法配置该工具。
