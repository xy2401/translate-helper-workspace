类路径变量首选项

可配置的变量

可以在Java构建路径中使用类路径变量来避免对本地文件系统的引用。使用变量条目，类路径只包含一个变量，并且可以在团队中共享构建路径。这些变量的值是在。上配置的 Java>构建路径>类路径变量首选项页面。

 

命令

描述

新...

添加一个新的变量条目。在结果对话框中，指定新变量的名称和路径。您可以单击“ 文件”或“ 文件夹”按钮以浏览路径。

编辑...

允许您编辑选定的变量条目。在结果对话框中，编辑变量的名称和/或路径。您可以单击“ 文件”或“ 文件夹”按钮以浏览路径。

去掉

删除选定的变量条目。

保留的类路径变量

某些类路径变量在内部设置，无法在Classpath变量首选项中更改：

  • JRE_LIB：包含当前使用的JRE的运行时JAR文件的存档。
  • JRE_SRC：当前使用的JRE的源存档。
  • JRE_SRCROOT：当前使用的JRE的源归档中的根路径。

这些变量已弃用，不应再使用。而是使用JRE系统库 。
