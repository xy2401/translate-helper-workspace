Package Explorer视图

默认情况下在Java透视图中显示的Package Explorer视图显示工作台中Java项目的Java元素层次结构。它为您提供了导航器中显示的资源的Java特定视图。元素层次结构源自项目的构建路径。

对于每个项目，其源文件夹和引用的库都显示在树中。您可以打开并浏览内部和外部JAR文件的内容。在JAR中打开Java元素将打开CLASS文件编辑器，如果有JAR文件的源附件，则会显示其相应的源。

顶级元素>项目

在Package Explorer中将项目显示为顶级元素。仅显示属于当前活动工作集的项目。以下操作可用于配置活动工作集：

  • 选择工作集：打开“ 选择工作集”对话框以允许选择应显示元素的工作集。
  • 取消选择工作集：取消选择活动工作集。调用此操作后将显示所有元素
  • 编辑活动工作集：打开“ 编辑工作集”向导以编辑当前活动的工作集
 

顶级元素>工作集

将工作集显示为Package Explorer中的顶级元素。

“ 配置工作集...”操作可用于打开允许配置可见工作集的对话框。

过滤器...

打开Java元素过滤器对话框。
