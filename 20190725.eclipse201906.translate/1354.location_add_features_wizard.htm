添加功能位置向导

从添加位置向导打开添加功能 位置向导 。选择包含功能的文件夹（它还可以包含名为“功能”的文件夹）。目录中的所有功能都将添加到列表中。检查要添加到目标的功能。按下完成后，将为每个所选要素添加目标位置。功能中的插件将添加到目标中。

选择目录时，您可以使用下拉列表查看以前使用过的文件夹。您还可以在路径中使用变量（例如$ eclipse_home） ，以便更轻松地共享目标定义。

列表下方的选项使用特定版本的功能可以在版本中提供更大的灵活性。选中后，列表中选定的确切要素版本将添加到目标，并且必须始终可供目标正确解析 。通过关闭此选项，目标将使用目录中找到的最新版本的功能。如果更改目录的内容，只要找到某个版本的功能，目标仍可以正确解析 。

将功能添加到目标后，编辑它将打开编辑功能位置向导
