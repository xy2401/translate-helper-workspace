C / C ++项目属性，构建，设置，二进制解析器

使用二进制解析器属性选项卡可确保Project Explorer和C / C ++项目视图的准确性，并成功运行和调试程序。为开发环境选择正确的解析器并构建项目后，可以使用C / C ++ Projects视图查看目标文件的符号。

二进制解析器选项
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
二进制解析器 （顶部窗格） 列出CDT当前已知的所有二进制解析器。选择要使用的解析器，并根据需要选择相应的行来编辑解析器的选项。
二进制解析器选项 在上面的列表中显示当前所选解析器的参数。根据您选择的解析器，列表中的选项将有所不同（特别是，某些选项可能根本没有选项）。
提升 将所选解析器移动到列表中的较高位置。注意：顺序仅适用于所选解析器：它们以与用户定义的顺序相同的顺序应用于二进制文件。对于未经检查的解析器，不会保留该顺序，因此您不必移动它们。
下移 将所选解析器移动到列表中较低的位置。注意：顺序仅适用于所选解析器：它们以与用户定义的顺序相同的顺序应用于二进制文件。对于未经检查的解析器，不会保留该顺序，因此您不必移动它们。

适用于： CDT项目。
