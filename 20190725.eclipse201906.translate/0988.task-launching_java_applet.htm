启动Java小程序

如果Java程序是作为applet构建的，则可以使用Java Applet启动配置。此启动配置使用从工作台首选项和程序的Java项目派生的信息来启动该程序。

  1. 在Package Explorer中 ，选择包含要启动的applet的Java编译单元或类文件。
  2. 按运行 [ 工作台工具栏中的]按钮或从工作台菜单中选择“运行”>“运行” 。另外，在Package Explorer弹出菜单中选择Run As> Java小程序 ，在工作台菜单栏中选择运行>运行方式> Java小程序 ，或者在运行工具栏按钮的下拉菜单中选择Run As> Java小程序 。
  3. 您的程序现已启动。

您还可以通过选择项目而不是编译单元或类文件来启动Java小程序。系统将提示您从扩展Applet的类中选择一个类。（如果在项目中只找到一个applet类，则启动该类，就像选择它一样。）
