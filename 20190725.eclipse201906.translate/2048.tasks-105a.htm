从存储库中还原已删除的文件

CVS插件允许您从CVS存储库中删除文件。如果删除托管文件并将删除提交到服务器，则会从当前分支或版本中删除该文件。虽然该文件已从当前分支中删除，但该文件及其先前的修订版实际上仍在服务器上。CVS插件提供了一个工具，可帮助将文件还原到先前已从CVS存储库中删除的工作区：

  1. 选择由CVS管理的文件夹，然后从上下文菜单中选择Team> Restore from Repository 。
  2. 将搜索存储库以查找已删除的文件，该列表将显示在对话框中。
  3. 在最左侧窗格中选择一个文件，以显示存储库中可用文件的修订版。
  4. 在最右侧窗格中选择一个修订，以查看底部文本窗格中的内容。
  5. 检查要还原的修订版本。
  6. 在为要恢复的每个文件选中修订后，单击“ 完成” 。
  7. 如果要将文件实际还原到当前分支还原文件，则必须添加，然后将文件提交回存储库。如果文件名未更改，则将为还原的文件维护修订历史记录。
