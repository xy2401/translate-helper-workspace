为Formatter应用程序生成配置文件

为格式化程序应用程序生成配置文件涉及修改Java项目的代码格式化程序设置，并将org.eclipse.jdt.core.prefs复制到该项目的.settings目录之外。

  1. 选择Java项目，打开弹出菜单，然后选择“ 属性” 。
  2. 选择Java Code Style> Formatter页面，然后选中Enable project specific settings 。
  3. 如上所述选择或编辑配置文件。
  4. 完成后单击“ 确定” 。
  5. 使用文件管理器或命令行将workspace/YourJavaProject/.settings/org.eclipse.jdt.core.prefs复制到新位置。
  6. 如果要格式化的文件使用1.5构造，则必须在刚刚创建的首选项文件中添加以下行：
    org.eclipse.jdt.core.compiler.compliance = 1.5
    org.eclipse.jdt.core.compiler.codegen.targetPlatform = 1.5
    org.eclipse.jdt.core.compiler.source = 1.5
