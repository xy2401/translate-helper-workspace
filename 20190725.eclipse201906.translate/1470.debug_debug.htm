调试程序

定义用于运行程序的启动配置时 ，可以指定程序支持哪些模式（运行，调试，配置文件等）。如果您支持调试模式，那么您需要实现一个调试模型和UI，允许用户在调试时与您的程序进行交互。核心平台调试插件提供以下支持：

  • 通用调试模型
  • 调试事件和监听器
  • 断点管理
  • 表达管理

调试UI插件提供了一个框架，用于在UI中显示调试模型。它还包括用于实现常见UI任务的实用程序类。

由于很难以任何有意义的细节讨论泛型调试，因此我们将从Java调试器的角度审查平台调试模型和UI类。

如何编写Eclipse调试器为使用简单的下推自动机（PDA）汇编语言构建调试器提供了一个开始完成的示例。
