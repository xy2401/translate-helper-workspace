输入首选项

使用 键入首选项面板以在键入时帮助格式化输入。

键入首选项选项
选项 描述
“字符串” 如果启用，则自动关闭字符串引号。
（括号）和[方括号]括号 如果启用，则自动关闭括号和方括号。
<Angle>括号 如果启用，则自动关闭尖括号。
大括号{} 如果启用，则自动关闭花括号。
Tab键调整当前行的缩进 如果启用，则在按Tab键时缩进当前行。
调整缩进 调整粘贴文本的缩进以匹配当前缩进级别。
自动换行 允许在超过最大行长度时自动换行字符串文字。
粘贴到文字字符串时转义文本 如果启用，粘贴到现有字符串文字中时，粘贴字符串中的特殊字符将被转义。
新的线条和大括号 如果启用，将根据“ 代码样式”首选项缩进新行或括号。如果禁用，则应用默认的文本编辑器缩进策略（从前一行复制缩进）
