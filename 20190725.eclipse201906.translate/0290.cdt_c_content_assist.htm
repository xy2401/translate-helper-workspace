内容辅助

内容辅助是CDT中内置的一组工具，可以减少创建代码时必须键入的击键次数。Content Assist插件由几个组件组成，这些组件根据当前上下文，范围和前缀预测开发人员将键入的内容。

代码完成

内容辅助提供文档中任何位置的代码完成。对于当前项目，将显示以您输入的字母组合开头的元素的列表，并按以下顺序确定每个提案的相关性：

  • 字段
  • 变量
  • 方法
  • 功能
  • 类
  • 结构
  • 工会
  • 命名空间
  • 枚举
当您调用内容辅助时（例如，当您键入Ctrl + Space时 ），您会触发代码完成功能，但在键入时会自动激活。 ， - >或:: 。

您可以通过指向弹出窗口查看列表中每个项目的签名。然后，您可以选择列表中的项目将其直接插入代码中。

模板

您可以为经常使用的代码段创建和保存模板，这些代码段将根据范围插入。内容辅助功能还可以快速访问代码模板。

在C / C ++编辑器中输入字母组合，并键入CTRL + SPACE （或右键单击并单击“ 内容辅助” ）时，将显示以您键入的字母组合开头的代码元素和模板的列表。

然后，您可以从列表中选择一个模板，并将其直接插入到您的代码中。
