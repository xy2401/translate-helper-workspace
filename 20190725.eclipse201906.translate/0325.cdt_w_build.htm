建立一个项目

make输出和构建进度信息显示在Console视图中。Make Targets视图显示makefile操作， Problems视图显示编译警告或错误。

要建立一个项目：

  1. 在Project Explorer视图中，选择您的项目。对于本教程，您可以选择之前创建的HelloWorld项目。


  2. 单击“ 项目”>“生成项目” ，或单击生成图标在工具栏上。

    如果您看到错误消息，例如：


       (Cannot run program "make": Launching failed)


    然后CDT无法找到构建命令，通常是make 。您的路径配置不正确，或者您的系统上没有安装make 。



  3. 您可以在Console视图中看到build命令的输出和结果。如果当前不可见，请单击其选项卡以使视图向前移动。如果由于某种原因它不存在，您可以通过选择Window> Show View> Console来打开它。
