C / C ++属性，构建，设置，构建工件选项卡

使用“ 构建工件”属性选项卡指定由所选构建配置构建的构建工件信息，例如类型和名称。

构建工件选项
组 描述
组态 点击此处查看说明。
管理配置...... 点击此处查看说明。
神器类型 显示所选工件的类型。选择由当前选定的构建配置（例如可执行文件 ， 静态库和共享库 ）构建的工件类型。
工件名称 指示工件的名称。默认情况下，名称与项目名称相同。
神器扩展 指定指定工件类型的文件扩展名。
输出前缀 表示要添加到输出结果的前缀。

适用于： CDT项目。
