用户库首选项

在上显示用户库设置的首选项 Java>构建路径>用户库首选项页面。

用户库是一组JAR文件。可以通过构建路径属性页面将用户库添加到项目构建路径。“用户库”首选项页面允许定义，编辑，导入，导出或删除用户库。

选项

描述
新... 创建一个新的用户库
编辑... 编辑当前选定的库，JAR或JAR属性
添加JAR ... 将一个或多个JAR添加到当前选定的用户库
去掉.. 删除当前选定的用户库或JAR
向上 将当前选定的JAR向上移动到用户库的JAR列表中。如果多个JAR包含具有相同限定名称的类型，则该顺序可能很重要。
下 将当前选定的JAR向下移动到用户库的JAR列表中。如果多个JAR包含具有相同限定名称的类型，则该顺序可能很重要。
进口... 从导入文件添加新库
出口... 将库导出到导入文件
