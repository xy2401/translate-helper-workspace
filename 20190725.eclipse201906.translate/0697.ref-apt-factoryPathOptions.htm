高级工厂路径选项对话框

从Factory Path首选项窗格中可以访问Advanced Factory Path Options对话框。它用于查看插件或jar文件中的各个处理器并配置其他选项。通常不需要从默认值更改此对话框上的选项。

以批处理模式运行此容器的处理器
选择此选项后，此jar中的处理器和处理器工厂将以增强与Sun（TM）命令行工具apt的兼容性的模式运行。在这种模式下，处理器类在构建开始时加载，在整个Java类型集上执行，然后卸载，就像在命令行运行apt时一样。在此模式下运行的处理器仅在完整构建期间运行，而不是在增量构建期间运行。此选项仅可用于jar文件;对于插件，它总是设置为false。此选项仅适用于使用Java 5 Mirror API的处理器。它不会影响使用Java 6注释处理API的处理器。
