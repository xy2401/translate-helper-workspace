评论

注释是源文件中已标记为由编译器忽略的行。当前的C / C ++编译器支持两种注释样式：

  • /* text */
  • // text

评论

您可以通过在该行的开头插入前导字符//迅速注释掉的代码一行或多行。为此，请选择要注释掉的代码行（或行），然后按CTRL + / （斜杠）。

取消注释

要取消注释，请选择代码行（或行），然后按CTRL + \ （反斜杠）。

提示：在注释和取消注释代码时，已注释掉的行上的字符/* */不会受到影响。

多行评论

您可以使用内容辅助功能在功能之前插入多行注释。键入com + Ctrl + Space ，并在光标位置输入以下代码：

/*
 * author userid
 *
 * To change this generated comment edit the template variable "comment":
 * Window > Preferences > C/C++ > Editor > Templates.
 */
要更改默认注释，请单击窗口>首选项> C>模板 。有关更多信息，请参阅“ 内容辅助”部分。
