1.108 DNFS_BATCH_SIZE

DNFS_BATCH_SIZE控制启用Direct NFS客户端时可由Oracle进程排队的异步I / O数。

属性 描述

参数类型

整数

默认值

4096

可修改

没有

可在PDB中修改

没有

价值范围

0 - 4096

基本

没有

在NFS服务器无法处理大量未完成的异步I / O请求的环境中，请使用此参数来限制Oracle前台进程发出的I / O数。此参数的建议设置是从128开始，并根据NFS服务器性能增加或减少它。
