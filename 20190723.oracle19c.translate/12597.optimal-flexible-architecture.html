B最佳灵活架构

Oracle Optimal Flexible Architecture（OFA）规则是一组配置指南，旨在确保组织良好的Oracle安装，从而简化管理，支持和维护。

  • 关于最优灵活架构标准
    Oracle Optimal Flexible Architecture（OFA）规则可帮助您组织数据库软件并配置数据库，以允许不同用户拥有的不同版本的多个数据库共存。
  • 关于多个Oracle Homes支持
    Oracle数据库支持多个Oracle主目录。您可以在同一系统上，在不同的Oracle主目录中多次安装此版本或更早版本的软件。
  • 关于Oracle Inventory目录和安装
    您指定为Oracle Inventory目录（oraInventory）的目录存储系统上安装的所有软件的清单。
  • Oracle Base Directory命名约定
    Oracle Base目录是Oracle数据库安装所有者的数据库主目录，以及Oracle Grid Infrastructure所有者的日志文件位置。
  • Oracle主目录命名约定
    默认情况下，Oracle Universal Installer使用这些Oracle Optimal Flexible Architecture约定来配置Oracle主目录。
