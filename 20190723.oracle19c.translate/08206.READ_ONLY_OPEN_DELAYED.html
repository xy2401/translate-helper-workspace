1.276 READ_ONLY_OPEN_DELAYED

READ_ONLY_OPEN_DELAYED确定何时访问只读表空间中的数据文件。

属性 描述

参数类型

布尔

默认值

假

可修改

没有

可在PDB中修改

没有

价值范围

是的|假

基本

没有

值

  • 真正

    仅当尝试读取存储在其中的数据时才首次访问数据文件。

  • 假

    在数据库打开时访问数据文件。

当数据库的大部分内容存储在只读表空间中时，您可以使用此参数来加速非常大型数据库的某些操作（主要是打开数据库）。考虑将此参数设置为true以用于此类数据库，尤其是在只读数据的某些部分存储在慢速访问设备或分层存储上时。
