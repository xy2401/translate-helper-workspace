9.86 V $ STREAMS_APPLY_SERVER

V$STREAMS_APPLY_SERVER显示有关每个应用服务器及其活动的信息。 应用服务器从应用协调器接收应用进程的消息。对于收到的每条消息，应用服务器应用消息或将消息发送到适当的应用处理程序。应用服务器是应用进程，出站服务器或入站服务器的子组件。
柱 数据类型 描述

SID

数

应用服务器会话的会话ID

SERIAL＃

数

应用服务器会话的序列号

应用＃

数

申请流程编号。应用过程是Oracle后台进程，以ap为前缀。

APPLY_NAME

VARCHAR2（128）

申请流程的名称

SERVER_ID

数

应用服务器的并行执行服务器编号

州

VARCHAR2（20）

应用服务器的状态：

  • INITIALIZING - 启动

  • IDLE - 不做任何工作

  • RECORD LOW-WATERMARK - 执行管理作业，维护有关应用进度的信息，该信息在ALL_APPLY_PROGRESS和DBA_APPLY_PROGRESS数据字典视图中使用

  • ADD PARTITION - 执行管理作业，添加用于记录正在进行的事务的信息的分区

  • DROP PARTITION - 执行管理作业，清除用于记录正在进行的事务的信息的行

  • EXECUTE TRANSACTION - 应用交易

  • WAIT COMMIT - 等待提交事务，直到应用具有较低提交SCN的所有其他事务。仅当COMMIT_SERIALIZATION apply process参数设置为DEPENDENT_TRANSACTIONS以外的值且PARALLELISM apply process参数设置为大于1的值时，才可以使用此状态。

  • WAIT DEPENDENCY - 等待在事务中应用逻辑更改记录（LCR），直到应用具有依赖性的另一个事务。仅当PARALLELISM apply process参数设置为大于1的值时，才可以使用此状态。

  • ROLLBACK TRANSACTION - 回滚事务

  • TRANSACTION CLEANUP - 清理已应用的事务，包括从应用进程的队列中删除LCR

  • WAIT FOR CLIENT -等待XStream的客户端应用程序请求更多的LCR

  • WAIT FOR NEXT CHUNK -等待下一组的LCR的一个大交易

XIDUSN

数

事务ID撤消当前正在应用的事务的段号

XIDSLT

数

当前正在应用的交易的交易ID槽号

XIDSQN

数

当前正在应用的事务的事务ID序列号

COMMITSCN

数

提交当前正在应用的事务的系统更改号（SCN）

DEP_XIDUSN

数

事务ID撤消此应用服务器应用的事务所依赖的事务的段号

DEP_XIDSLT

数

此应用服务器所应用的事务所依赖的事务的事务ID槽号

DEP_XIDSQN

数

此应用服务器应用的事务所依赖的事务的事务ID序列号

DEP_COMMITSCN

数

提交此应用服务器所依赖的事务的系统更改号（SCN）

MESSAGE_SEQUENCE

数

应用服务器正在应用的当前消息的编号。每个事务开始时，此值将重置为1 。

TOTAL_ASSIGNED

数

自上次启动应用进程以来分配给应用服务器的事务总数

TOTAL_ADMIN

数

自上次启动应用进程以来应用服务器完成的管理作业总数。有关管理作业的类型，请参阅此视图中的STATE信息。

TOTAL_ROLLBACKS

数

分配给此服务器的已回滚的事务数

TOTAL_MESSAGES_APPLIED

数

自上次启动应用进程以来此应用服务器应用的消息总数

APPLY_TIME

日期

应用最后一条消息的时间

APPLIED_MESSAGE_NUMBER

数

应用的最后一条消息的编号

APPLIED_MESSAGE_CREATE_TIME

日期

应用上次捕获的消息的源数据库的创建时间。此列中未记录有关用户排队消息的信息。

ELAPSED_DEQUEUE_TIME

数

自上次启动应用过程以来经过的时间（以百分之一秒为单位）出列消息

ELAPSED_APPLY_TIME

数

自上次启动应用过程以来应用消息所用的时间（以百分之一秒为单位）

COMMIT_POSITION

RAW（64）

提交交易的位置。此列仅填充用作XStream出站服务器或入站服务器的应用进程。

DEP_COMMIT_POSITION

RAW（64）

承诺奴隶所依赖的交易的位置。此列仅填充用作XStream入站服务器的应用进程。

LAST_APPLY_POSITION

RAW（64）

对于入站服务器，应用最后一条消息的位置;对于出站服务器，发送到XStream客户端应用程序的最后一条消息的位置。此列仅填充用作XStream出站服务器或入站服务器的应用进程。

TRANSACTION_ID

VARCHAR2（128）

从站正在应用的事务ID。此列仅填充用作XStream入站服务器的应用进程。

DEP_TRANSACTION_ID

VARCHAR2（128）

从属依赖的事务的事务ID。此列仅填充用作XStream入站服务器的应用进程。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

TOTAL_LCRS_RETRIED

数

此服务器重试的LCR总数

LCR_RETRY_ITERATION

数

此服务器重试此事务的迭代

TOTAL_TXNS_RETRIED

数

此服务器重试的总事务数

TXN_RETRY_ITERATION

数

此服务器的总重试迭代次数

TOTAL_TXNS_RECORDED

数

此服务器在错误队列中记录的事务总数
