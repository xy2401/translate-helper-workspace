15 Oracle XML DB和Oracle Data Provider for .NET

Oracle Data Provider for Microsoft .NET（ODP.NET）是Oracle数据库的数据提供程序的实现。它使用Oracle本机API，可以从任何.NET应用程序快速可靠地访问Oracle数据和功能。

它还使用和继承Microsoft .NET Framework类库中提供的类和接口。ODP.NET本身支持.NET的以下LOB数据类型： BLOB ， CLOB ， NCLOB和BFILE 。

15.1 Oracle XML DB和ODP.NET XML

ODP.NET通过Oracle XML DB在数据库中本机支持XML数据。

ODP.NET：

  • 将XML数据本地存储在Oracle数据库中作为XMLType 。

  • 将关系数据和对象关系数据作为XML数据从Oracle数据库访问到Microsoft .NET环境，并使用Microsoft .NET框架处理XML。

  • 使用XML数据保存对数据库服务器的更改。

  • 提供以下特定于XML的类：

    • OracleXmlType

    • OracleXmlStream

    • OracleXmlQueryProperties

    • OracleXmlSaveProperties

  • 增强OracleCommand ， OracleConnection和OracleDataReader 。
