说明adxdb035.eps的说明

该图在顶部有“Oracle XML DB数据存储选项”字样。紧接着这些词语的是“您的存储选项会影响性能和数据保真度”。两行连接到两个框下方。一个框标记为“XMLType Tables”，另一个框标记为“XMLType Views”。紧接在“XMLType Views”框右侧的是“如果现有关系数据使用XMLType视图”。

标有“XMLType Tables”的框有两行连接到它下面的两个框，从左到右标记：“Binary XML Storage”和“Object-Relational Storage”。标有“XMLType Views”的框下方是“可以使用：定义视图”。这些单词有两行连接下面两个框，标记如下，从左到右：“SQL / XML Operators”和“Object Types”。

标有“SQL / XML Operators”的框在它下面连接到标有“Relational Tables”的框。标有“对象类型”的框在其下方连接到三个框，标记如下，从左到右：“对象表”，“对象视图”和“对象构造函数”。框“对象构造函数”在它下面连接到标记为“关系表”的框。
