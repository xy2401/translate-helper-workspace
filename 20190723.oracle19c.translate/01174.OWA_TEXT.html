220.1 OWA_OPT_LOCK类型

有三种OWA_OPT_LOCK锁定类型。

  • MULTI_LINE数据类型

  • ROW_LIST数据类型

  • VC_ARR数据类型

220.1.1 MULTI_LINE数据类型

此数据类型是包含大量文本的PL / SQL记录。行字段，类型为OWA_TEXT 。 VC_ARR DATA TYPE ，包含记录中的文本数据。 

TYPE multi_line IS RECORD（行vc_arr，num_rows INTEGER，partial_row BOOLEAN）;

220.1.2 ROW_LIST数据类型

这是用于保存要处理的数据的数据类型。

TYPE row_list IS RECORD（rows int_arr，num_rows INTEGER）; int_arr IS DEFINED AS：TYPE int_arr是BINARY_INTEGER的INTEGER INDEX表;

220.1.3 VC_ARR数据类型

这是MULTI_LINE DATA TYPE的一个组件，用于保存大量文本。

TYPE vc_arr是VARCHAR2的表（32767）INDEX BY BINARY_INTEGER;

220.2.1 ADD2MULTI程序

此过程将内容添加到现有的Multi_Line数据类型

句法

OWA_TEXT.ADD2MULTI（流IN IN VARCHAR2，mline IN OUT multi_line，

继续在BOOLEAN DEFAULT TRUE）;

参数

表220-2 ADD2MULTI过程参数

参数 描述

流

要添加的文字。

多线

OWA_TEXT.MULTI_LINE DATA TYPE 。此参数的输出包含流 。

继续

如果为TRUE ，则过程在前一个最后一行中附加流 （假设它小于32K）。如果为FALSE ，则该过程将流放入新行。

220.2.2 NEW_ROW_LIST功能和过程

此函数或过程创建新的OWA_TEX.ROW_LIST DATA TYPE 。

函数版本不使用参数并返回新的空row_list 。过程版本将row_list数据类型创建为输出参数。

句法

OWA_TEXT.NEW_ROW_LIST返回ROW_LIST; OWA_TEXT.NEW_ROW_LIST（rlist OUT row_list）;

参数

表220-3 NEW_ROW_LIST过程参数

参数 描述

RLIST

这是包含新row_list数据类型的输出参数

返回值

函数版本返回新的row_list数据类型。
