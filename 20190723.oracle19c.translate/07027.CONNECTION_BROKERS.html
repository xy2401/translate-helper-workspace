1.49 CONNECTION_BROKERS

CONNECTION_BROKERS用于指定连接代理类型，每种类型的连接代理数以及每个代理的最大连接数。

属性 描述

参数类型

串

句法

CONNECTION_BROKERS = broker_description [，...]

句法

broker_description :: =

（（type_clause）（brokers_clause）[（broker_options）]）

句法

type_clause :: =

TYPE = {DEDICATED | EMON | POOLED}

代理代理类型。这是一个必需参数。

句法

brokers_clause :: =

BROKERS= integer

指示指定类型的代理数。这是一个必需参数。指定的整数值必须大于或等于0。如果设置为0，代理将不接受新连接，但现有连接将照常继续。

句法

broker_options :: =

CONNECTIONS= integer

每个代理的最大连接数。这是一个可选参数。如果指定的整数值大于平台允许的最大值，则使用平台允许的最大值。

默认值

初始化参数的默认值将包含DEDICATED和EMON代理类型。某些类型的代理仅在启用某些功能时启动，因此只有在需要代理时才会应用它们的配置。

Oracle RAC

应在所有实例上设置相同的值

在这些情况下使用连接代理：

  • DEDICATED代理由多进程和多线程Oracle功能使用。有关这些功能的更多信息，请参见Oracle数据库概念 。

    当USE_DEDICATED_BROKER初始化参数设置为true时，也会使用DEDICATED代理。有关USE_DEDICATED_BROKER初始化参数的详细信息，请参阅“ USE_DEDICATED_BROKER ” 。

  • 当客户注册了12 c AQ队列的通知时，将使用EMON代理。它默认不启用。有关AQ队列的更多信息，请参见“ Oracle数据库高级队列用户指南” 。

  • 只要数据库驻留连接池（DRCP）池处于活动状态，就会使用POOLED代理。有关DRCP池的更多信息，请参阅Oracle数据库概念 。

    默认情况下，POOLED代理不会出现在参数中。在这种情况下，使用DRCP字段。如果在参数中指定POOLED代理，则该规范将覆盖DRCP字段，然后进一步尝试设置DRCP字段将引发错误。Oracle建议使用该参数而不是DRCP字段。

为了更容易修改列表，使用TYPE索引单个条目，因此不必重新指定整个列表。例如，要调整DEDICATED经纪人的数量：

alter system set connection_brokers ='（（TYPE = DEDICATED）（BROKERS = 2））'
