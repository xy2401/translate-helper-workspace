2.191 ALL_JAVA_RESOLVERS

ALL_JAVA_RESOLVERS显示有关当前用户可访问的Java类的解析程序的信息。

相关观点

  • DBA_JAVA_RESOLVERS显示有关数据库中所有Java类的解析程序的信息。

  • USER_JAVA_RESOLVERS显示有关当前用户拥有的Java类的解析程序的信息。此视图不显示OWNER列。
