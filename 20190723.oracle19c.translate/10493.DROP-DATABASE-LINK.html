DROP DATABASE LINK

目的

使用DROP DATABASE LINK语句从数据库中删除数据库链接。

也可以看看：

CREATE DATABASE LINK的信息，创建数据库链接

先决条件

私有数据库链接必须位于您自己的架构中。要删除PUBLIC数据库链接，您必须具有DROP PUBLIC DATABASE LINK系统特权。

句法

drop_database_link :: =


说明drop_database_link.eps的说明

语义

上市

您必须指定PUBLIC以删除PUBLIC数据库链接。

DBLINK

指定要删除的数据库链接的名称。

删除数据库链接的限制

您不能在另一个用户的模式中删除数据库链接，也不能使用模式名称限定dblink ，因为数据库链接的名称允许使用句点。因此，Oracle数据库将整个名称（例如ralph.linktosales解释为模式中数据库链接的名称，而不是模式ralph名为linktosales的数据库链接。
