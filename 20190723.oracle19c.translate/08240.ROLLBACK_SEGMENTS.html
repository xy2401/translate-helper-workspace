1.296 ROLLBACK_SEGMENTS

ROLLBACK_SEGMENTS按名称为此实例分配一个或多个回滚段。

属性 描述

参数类型

串

句法

ROLLBACK_SEGMENTS =

（segment_name [，segment_name] ...）

默认值

如果未指定此参数，则默认情况下实例使用公共回滚段，除非UNDO_MANAGEMENT初始化参数设置为AUTO 。在这种情况下，将忽略ROLLBACK_SEGMENTS参数并使用自动撤消管理。

可修改

没有

可在PDB中修改

是

价值范围

除SYSTEM之外的DBA_ROLLBACK_SEGS列出的任何回滚段名称

基本

没有

Oracle RAC

多个实例必须具有不同的值。

如果设置此参数，则实例将获取此参数中指定的所有回滚段，即使回滚段的数量超过实例所需的最小数量（计算为TRANSACTIONS / TRANSACTIONS_PER_ROLLBACK_SEGMENT ）。

您无法动态更改此参数的值，但可以更改其值，然后重新启动实例。虽然此参数通常指定专用回滚段，但如果它们尚未使用，则还可以指定公共回滚段。

要查找数据库中每个回滚段的名称，段ID号和状态，请查询数据字典视图DBA_ROLLBACK_SEGS 。

当UNDO_MANAGEMENT设置为AUTO ，将忽略ROLLBACK_SEGMENTS 。
