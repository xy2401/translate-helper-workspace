13.27异常处理程序

异常处理程序处理引发的异常。

异常处理程序出现在匿名块，子程序，触发器和包的异常处理部分中。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

exception_handler :: =


说明exception_handler.eps的描述

（ statement :: = ）

语义

exception_handler

例外

预定义异常的名称（请参阅表11-3 ）或用户定义的异常（请参阅“ 异常声明 ” ）。

如果PL / SQL引发指定的异常，则运行关联的语句。

其他

指定未在块的异常处理部分中明确指定的所有异常。如果PL / SQL引发了这样的异常，则运行相关的语句。

注意：

Oracle建议OTHERS异常处理程序中的最后一个语句是RAISE或RAISE_APPLICATION_ERROR过程的调用。

如果您不遵循此惯例，并且启用了PL / SQL警告，则会获得PLW-06009。

在块的异常处理部分中， WHEN OTHERS异常处理程序是可选的。它只能出现一次，作为块的异常处理部分中的最后一个异常处理程序。
