监听器配置和listener.ora文件

在Oracle RAC环境中，Oracle建议您让Oracle代理管理Oracle数据库的Oracle侦听器。

注意：

如果启用GNS，则无需手动配置侦听器。

  • Oracle RAC数据库的本地监听器
    安装了Oracle Grid Infrastructure后，本地侦听器或默认侦听器位于Grid主目录中。listener.ora文件位于Grid_home /network/admin目录中。
  • Oracle RAC数据库的远程监听器
    远程侦听器是驻留在一台计算机上的侦听器 ，它将连接重定向到另一台计算机上的数据库实例。例如，SCAN侦听器是远程侦听器。
  • 管理Oracle RAC数据库的多个侦听器
    查看以了解如何使用SRVCTL和TNS_ADMIN来管理侦听器
  • Oracle数据库如何使用监听器文件（listener.ora）
    listener.ora文件是listener.ora器的配置文件。它可以包括接受连接请求的协议地址，数据库列表以及它正在侦听的其他服务，以及监听器使用的控制参数。
