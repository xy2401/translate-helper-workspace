第五部分在多租户环境中使用Oracle功能

您可以在多租户环境中使用Oracle数据库功能。

本部分包含以下章节：

  • 备份和恢复CDB和PDB
    您可以备份和恢复多租户容器数据库（CDB）和可插拔数据库（PDB）。
  • 在多租户环境中使用数据库实用程序
    您可以在多租户环境中使用Oracle Data Pump，DBNEWID和Oracle LogMiner等实用程序。
  • 使用Oracle Resource Manager进行PDB
    使用PL / SQL包过程来管理Oracle Resource Manager（资源管理器），以便将资源分配给多租户容器数据库（CDB）中的可插拔数据库（PDB）。
  • 将Oracle Scheduler与CDB一起使用
    您可以使用Oracle Scheduler在多租户容器数据库（CDB）中计划作业。
  • 将Oracle Database Vault与CDB一起使用
    您可以在多租户容器数据库（CDB）中使用Oracle Database Vault。
  • 将XStream与CDB一起使用
    您可以在多租户容器数据库（CDB）中使用Oracle Database XStream。
