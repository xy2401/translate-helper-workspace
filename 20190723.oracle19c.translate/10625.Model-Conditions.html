例

以下示例将2000年的每个产品的销售额设置为0：

现状条件

is_present_condition :: =

IS PRESENT条件只能在SELECT语句的model_clause中使用。使用此条件来测试在执行model_clause之前引用的单元格是否存在。


插图描述is_present_condition.eps

如果单元格在执行model_clause之前存在，则条件返回TRUE如果不存在则返回FALSE 。

model_clause和Model Expressions以获取信息

例

在以下示例中，如果存在1999年鼠标垫的销售额，则2000年鼠标垫的销售额将设置为1999年鼠标垫的销售额。否则，2000年鼠标垫的销售额将设置为0。

SELECT country，prod，year，s FROM sales_view_ref MODEL PARTITION BY（country）DIMENSION BY（prod，year）MEASURES（sale s）IGNORE NAV UNIQUE DIMENSION RULES UPSERT序列顺序（s ['Mouse Pad'，2000] = CASE when s ['鼠标垫'，1999]现在是['鼠标垫'，1999] ELSE 0 END）按国家，生产年份订购;国家年度---------- -----------------------------------  - ------- --------- France Mouse Pad 1998 2509.42 France Mouse Pad 1999 3678.69 France Mouse Pad 2000 3678.69 France Mouse Pad 2001 3269.09 France Standard Mouse 1998 2390.83 France Standard Mouse 1999 2280.45 France Standard Mouse 2000 1274.31法国标准鼠标2001 2164.54德国鼠标垫1998 5827.87德国鼠标垫1999 8346.44德国鼠标垫2000 8346.44德国鼠标垫2001 9535.08德国标准鼠标1998 7116.11德国标准鼠标1999 6263.14德国标准鼠标2000 2637.31德国标准鼠标2001 6456.13选择16行。
