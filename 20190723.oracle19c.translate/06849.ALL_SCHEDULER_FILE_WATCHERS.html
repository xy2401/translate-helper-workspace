3.42 ALL_SCHEDULER_FILE_WATCHERS

ALL_SCHEDULER_FILE_WATCHERS显示有关当前用户可访问的Scheduler文件监视请求的信息。

相关观点

  • DBA_SCHEDULER_FILE_WATCHERS显示有关数据库中所有Scheduler文件监视请求的信息。

  • USER_SCHEDULER_FILE_WATCHERS显示有关当前用户拥有的Scheduler文件监视请求的信息。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

文件监视请求的所有者

FILE_WATCHER_NAME

VARCHAR2（128）

文件监视请求的名称

启用

VARCHAR2（5）

指示此文件监视请求是启用（ TRUE ）还是禁用（ FALSE ）

DESTINATION_OWNER

VARCHAR2（128）

VARCHAR2（128）

不是NULL

VARCHAR2（512）
