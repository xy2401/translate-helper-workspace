TO_CHAR（号码）

句法

to_char_number :: =


to_char_number.eps插图的描述

目的

TO_CHAR （number）使用可选的数字格式fmt将n转换为VARCHAR2数据类型的值。值n可以是NUMBER ， BINARY_FLOAT或BINARY_DOUBLE BINARY_FLOAT 。如果省略fmt ，则n将转换为VARCHAR2值，其长度足以保持其有效数字。

如果n为负数，则在应用格式后应用符号。因此TO_CHAR(-1, '$9')返回 - $ 1，而不是$ -1。

有关数字格式的信息，请参阅“ 格式模型 ” 。

'nlsparam'参数指定由数字格式元素返回的这些字符：

  • 十进制字符

  • 组分隔符

  • 当地货币符号

  • 国际货币符号

这个论点可以有这样的形式：

'NLS_NUMERIC_CHARACTERS = '' 危险品 'NLS_CURRENCY = '' 文本 'NLS_ISO_CURRENCY = 领土 '

字符d和g代表十进制字符和组分隔符。它们必须是不同的单字节字符。在引用的字符串中，必须在参数值周围使用两个单引号。货币符号有十个字符。

如果省略'nlsparam'或任何一个参数，则此函数使用会话的默认参数值。

  • “ 数据库全球化支持指南”中的附录C，用于排序规则派生规则，用于定义分配给此函数的字符返回值的排序规则

例子

以下语句使用隐式转换将字符串和数字组合成一个数字：

SELECT TO_CHAR（'01110'+ 1）FROM DUAL; TO_C ---- 1111

将此示例与TO_CHAR（字符）的第一个示例进行比较。

在下一个示例中，输出是空白填充到货币符号的左侧。在可选的数字格式fmt中， L表示本地货币符号， MI表示尾随减号。有关数字格式元素的完整列表，请参阅表2-15 。该示例显示会话中的输出，其中会话参数NLS_TERRITORY设置为AMERICA 。

SELECT TO_CHAR（-10000，'L99G999D99MI'）“金额”FROM DUAL;金额-------------- $ 10,000.00-

在下一个示例中， NLS_CURRENCY指定用作L数字格式元素的本地货币符号的字符串。NLS_NUMERIC_CHARACTERS逗号指定为用作D数字格式元素的小数分隔符的字符，将NLS_NUMERIC_CHARACTERS指定为用作G数字格式元素的组分隔符的字符。这些特征预计在许多国家，例如在德国。 

SELECT TO_CHAR（-10000，'L99G999D99MI'，'NLS_NUMERIC_CHARACTERS ='',.''NLS_CURRENCY =''AusDollars'''）“金额”来自DUAL;金额------------------- AusDollars10.000,00-

在下一个示例中， NLS_ISO_CURRENCY指示数据库使用POLAND区域的国际货币符号作为C编号格式元素：

SELECT TO_CHAR（-10000，'99G999D99C'，'NLS_NUMERIC_CHARACTERS ='',.''NLS_ISO_CURRENCY = POLAND'）“金额”FROM DUAL;金额----------------- -10.000,00PLN

TO_CHAR（数字）功能：示例

以下语句创建一个名为empl_temp的表，并使用员工详细信息填充它：

CREATE TABLE empl_temp（employee_id NUMBER（6），first_name VARCHAR2（20），last_name VARCHAR2（25），email VARCHAR2（25），hire_date DATE DEFAULT SYSDATE，job_id VARCHAR2（10），clob_column CLOB）; INSERT INTO empl_temp VALUES（111，'John'，'Doe'，'example.com'，'10-JAN-2015'，'1001'，'有经验的员工'）; INSERT INTO empl_temp VALUES（112，'John'，'Smith'，'example.com'，'12-JAN-2015'，'1002'，'Junior Employee'）; INSERT INTO empl_temp VALUES（113，'Johnnie'，'Smith'，'example.com'，'12-JAN-2014'，'1002'，'Mid-Career Employee'）; INSERT INTO empl_temp VALUES（115，'Jane'，'Doe'，'example.com'，'2015年1月15日'，'1005'，'执行员工'）;

以下语句将数字数据转换为数据库字符集：

SELECT To_char（employee_id）“NUM_TO_CHAR”FROM empl_temp WHERE employee_id IN（111,112,113,115）; NUM_TO_CHAR -------------------- 111 112 113 115
