1.87 DB_PERFORMANCE_PROFILE

DB_PERFORMANCE_PROFILE指定数据库或可插拔数据库（PDB）的性能配置文件。

属性 描述

参数类型

串

句法

DB_PERFORMANCE_PROFILE = string

默认值

没有默认值。

可修改

没有

可在PDB中修改

是

基本

没有

Oracle RAC

应在所有实例上设置相同的值。

性能配置文件支持更轻松地管理数百个数据库或PDB。

在Oracle Exadata上，常规数据库或CDB的性能配置文件将下推到Exadata存储单元。性能配置文件用于管理Exadata I / O资源管理器。

CDB资源计划可以使用DBMS_RESOURCE_MANAGER.CREATE_CDB_PROFILE_DIRECTIVE过程为性能概要文件指定不同的资源控制。具有匹配性能配置文件的PDB将使用指定的控件。

DB_PERFORMANCE_PROFILE的值最多为30个字符，不区分大小写。
