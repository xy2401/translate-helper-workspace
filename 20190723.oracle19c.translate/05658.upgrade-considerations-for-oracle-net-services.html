B.1匿名访问Oracle Internet Directory

目录命名（LDAP）的典型用户需要匿名访问Oracle Internet Directory以进行名称查找。

Oracle Internet目录设置

如果升级11g或更高版本的Oracle Internet Directory软件版本，则Oracle Internet Directory的默认设置将更改为禁止匿名访问该目录。目录管理员必须配置目录以在升级目录后启用匿名绑定以释放11 g 。此外，在Oracle Internet Directory中配置匿名绑定的方式在Oracle Database 10g和Oracle Database 11g之间发生了更改。
