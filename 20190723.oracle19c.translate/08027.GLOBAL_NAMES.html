1.130 GLOBAL_NAMES

GLOBAL_NAMES指定数据库链接是否需要与其连接的数据库具有相同的名称。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

如果GLOBAL_NAMES值为false ，则不执行检查。如果您使用或计划使用分布式处理，则Oracle建议您将此参数设置为true以确保对网络环境中的数据库和链接使用一致的命名约定。
