TO_CHAR（角色）

句法

to_char_char :: =


to_char_char.eps插图的描述

目的

TO_CHAR （字符）将NCHAR ， NVARCHAR2 ， CLOB或NCLOB数据转换为数据库字符集。返回的值始终为VARCHAR2 。

使用此函数将字符LOB转换为数据库字符集时，如果要转换的LOB值大于目标类型，则数据库将返回错误。

也可以看看：

“ 数据库全球化支持指南”中的附录C，用于排序规则派生规则，用于定义分配给此函数的字符返回值的排序规则

例子

以下示例将简单字符串解释为字符数据：

SELECT TO_CHAR（'01110'）FROM DUAL; TO_CH ----- 01110

将此示例与TO_CHAR（数字）的第一个示例进行比较。

以下示例将pm.print_media表中的一些CLOB数据转换为数据库字符集：

TO_CHAR（字符）功能：示例

以下语句创建一个名为empl_temp的表，并使用员工详细信息填充它：

CREATE TABLE empl_temp（employee_id NUMBER（6），first_name VARCHAR2（20），last_name VARCHAR2（25），email VARCHAR2（25），hire_date DATE DEFAULT SYSDATE，job_id VARCHAR2（10），clob_column CLOB）; INSERT INTO empl_temp VALUES（111，'John'，'Doe'，'example.com'，'10-JAN-2015'，'1001'，'有经验的员工'）; INSERT INTO empl_temp VALUES（112，'John'，'Smith'，'example.com'，'12-JAN-2015'，'1002'，'Junior Employee'）; INSERT INTO empl_temp VALUES（113，'Johnnie'，'Smith'，'example.com'，'12-JAN-2014'，'1002'，'Mid-Career Employee'）; INSERT INTO empl_temp VALUES（115，'Jane'，'Doe'，'example.com'，'2015年1月15日'，'1005'，'执行员工'）;

以下语句将CLOB数据转换为数据库字符集：

SELECT To_char（clob_column）“CLOB_TO_CHAR”FROM empl_temp WHERE employee_id IN（111,112,115）; CLOB_TO_CHAR --------------------经验丰富的员工初级员工执行员工
