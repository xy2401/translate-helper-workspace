1.255 PARALLEL_THREADS_PER_CPU

PARALLEL_THREADS_PER_CPU描述了CPU在并行执行期间可以处理的并行执行进程或线程的数量。

属性 描述

参数类型

整数

默认值

1

可修改

改变系统

可在PDB中修改

没有

价值范围

任何非零数字

基本

没有

注意：

此参数适用于独占模式以及Oracle Real Application Clusters环境中的并行执行。

此参数用于确定其他并行执行相关参数的默认值，例如PARALLEL_MAX_SERVERS 。它还用于确定SQL语句的默认并行度，并确定自动并行度中并行度的上限。在大多数情况下，默认值就足够了。
