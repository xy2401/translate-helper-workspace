前言

Java是面向对象的编程语言，提供平台独立性和自动存储管理技术。它使您可以创建应用程序和小程序。Oracle数据库为开发和部署Java应用程序提供支持。

听众

Oracle数据库Java开发人员指南适用于Java和非Java开发人员。对于不熟悉Java编程的PL / SQL开发人员，本手册简要概述了Java和面向对象的概念。对于Java和PL / SQL开发人员，本手册讨论以下内容：

  • Java和数据库概念如何合并

  • 如何开发，加载和运行Java存储过程

  • Oracle JVM

  • 用于管理数据库中的Java对象的数据库概念

  • Oracle数据库和Java安全策略

要使用本文档，您需要了解Oracle数据库，SQL和PL / SQL。 Java和面向对象编程的先前知识可能会有所帮助。
