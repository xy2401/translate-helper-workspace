安装Oracle Messaging Gateway

Oracle Messaging Gateway随Oracle Database企业版一起安装。但是，您可能需要CSD或Fix Pack。

如果您需要IBM WebSphere MQ的CSD或Fix Pack，请访问以下网站以获取更多信息：

https://www.ibm.com/support/

注意：

Oracle Messaging Gateway不支持在IBM System：Linux on System z上集成高级队列和TIBCO Rendezvous。
