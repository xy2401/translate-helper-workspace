9.48 V $ SQL_BIND_METADATA

V$SQL_BIND_METADATA描述与绑定变量相关的元数据。

对于查询此视图的会话所拥有的每个游标中的每个不同绑定变量， V$SQL_BIND_METADATA描述：

  • 如果绑定变量是用户定义的，则绑定客户端提供的元数据

  • 如果CURSOR_SHARING参数设置为FORCE且绑定变量是系统生成的，则基于底层文字的元数据。

柱 数据类型 描述

地址

RAW（4 | 8）

拥有此绑定变量的子游标的内存地址

位置

数

绑定位置

数据类型

数

绑定数据类型的内部标识符。从Oracle Database 12 c开始，表示PL / SQL数据类型的数字可以出现在此列中。

最长长度

数

绑定值的最大长度

ARRAY_LEN

数

最大数组元素数（仅适用于数组绑定）

BIND_NAME

VARCHAR2（128）

用户定义的或系统生成的绑定变量名称（如果使用）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
