什么是强认证？

您使用身份验证来证明尝试登录数据库的用户的身份。

在分布式环境中对身份验证用户身份是必不可少的，否则对网络安全性几乎没有信心。密码是最常用的身份验证方式。Oracle数据库支持使用支持各种第三方身份验证服务的Oracle身份验证适配器进行强身份验证，包括带有数字证书的SSL

集中身份验证和单点登录

单点登录使用户可以使用单个密码访问多个帐户和应用程序。

集中身份验证还为用户提供单点登录（SSO）的好处。

在单点登录中，用户只需登录一次，然后可以自动连接到任何其他服务，而无需再次提供用户名和密码。单点登录消除了用户记住和管理多个密码的需要，减少了登录多个服务所花费的时间。

集中式网络身份验证的工作原理

集中式网络身份验证系统与Oracle服务器，身份验证服务器以及连接到Oracle服务器的用户一起使用。

图18-2显示了集中式网络身份验证服务通常如何运行。

图18-2网络验证服务如何验证用户


“图18-2网络验证服务如何验证用户”的说明

以下步骤描述了集中式网络验证过程的工作原理。

  1. 用户（客户端）请求认证服务并提供标识信息，例如令牌或密码。

  2. 验证服务器验证用户的身份并将票证或凭证传递回客户端，其中可能包括到期时间。

  3. 客户端将这些凭据与服务请求并发传递到Oracle服务器，例如连接到数据库。

  4. 服务器将凭据发送回认证服务器以进行认证。

  5. 身份验证服务器检查凭据并通知Oracle服务器。

  6. 如果身份验证服务器接受了凭据，则Oracle服务器会对用户进行身份验证。如果身份验证服务器拒绝了凭据，则身份验证将失败，并且服务请求将被拒绝。

关于Kerberos

Oracle数据库对Kerberos的支持提供了Oracle用户的单点登录和集中身份验证的优势。

Kerberos是一个受信任的第三方身份验证系统，它依赖于共享机密。它假设第三方是安全的，并提供单点登录功能，集中密码存储，数据库链接身份验证和增强的PC安全性。它通过Kerberos身份验证服务器执行此操作。有关配置和使用此适配器的信息，请参阅配置Kerberos身份验证 。

注意：

Kerberos的Oracle身份验证提供数据库链接身份验证（也称为代理身份验证）。Kerberos也是Enterprise User Security支持的身份验证方法。

父主题： 支持的强身份验证方法

关于远程身份验证拨入用户服务（RADIUS）

RADIUS是一种客户端/服务器安全协议，最广为人知的是启用远程身份验证和访问。

Oracle数据库在客户端/服务器网络环境中使用此标准，以支持使用支持RADIUS协议的任何身份验证方法。RADIUS可以与各种身份验证机制一起使用，包括令牌卡和智能卡。

  • 智能卡。符合RADIUS标准的智能卡是一种类似信用卡的硬件设备，具有内存和处理器。它由位于客户端工作站的智能卡读卡器读取。

  • 令牌卡。令牌卡（安全ID或RADIUS兼容）可以通过几种不同的机制提高易用性。某些令牌卡动态显示与身份验证服务同步的一次性密码。服务器可以通过联系身份验证服务在任何给定时间验证令牌卡提供的密码。其他令牌卡具有键盘并且在挑战 - 响应的基础上操作。在这种情况下，服务器提供用户输入令牌卡的挑战（数字）。令牌卡提供用户输入并发送到服务器的响应（加密地从挑战中导出的另一个号码）。

    您可以通过RADIUS适配器使用SecurID令牌。

关于安全套接字层

安全套接字层（SSL）是用于保护网络连接的行业标准协议。

SSL提供身份验证 ，数据加密和数据完整性 。

SSL协议是公钥基础结构（PKI）的基础 。对于身份验证，SSL使用符合X.509v3标准的数字证书以及公钥和私钥对 。

您可以使用Oracle Database SSL来保护任何客户端和任何服务器之间的通信。您可以配置SSL以仅为服务器，仅客户端或客户端和服务器提供身份验证。您还可以结合Oracle数据库支持的其他身份验证方法（数据库用户名和密码，RADIUS和Kerberos）配置SSL功能。

为了支持您的PKI实现，除了SSL之外，Oracle数据库还包括以下功能：

  • Oracle钱包，您可以在其中存储PKI凭据

  • Oracle Wallet Manager，可用于管理Oracle钱包

  • 证书撤销列表（CRL）的证书验证

  • 硬件安全模块支持

Oracle数据库本机网络加密和强身份验证限制

Oracle应用程序支持Oracle数据库本机网络加密和强身份验证。

但是，由于Oracle数据库本机网络加密和强身份验证要求Oracle Net Services安全地传输数据，因此当Oracle Financial，人力资源和制造应用程序在Microsoft Windows上运行时，这些外部身份验证功能不受支持。

使用Oracle Display Manager（ODM）的这些产品部分不利用Oracle数据库本机网络加密和强身份验证，因为ODM不使用Oracle Net Services。
