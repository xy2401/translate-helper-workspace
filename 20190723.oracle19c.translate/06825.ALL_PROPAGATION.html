3.18 ALL_PROPAGATION

ALL_PROPAGATION显示有关当前用户可访问的源队列的传播的信息。

相关视图

DBA_PROPAGATION显示有关数据库中所有传播的信息。

柱 数据类型 空值 描述

PROPAGATION_NAME

VARCHAR2（128）

不是NULL

传播的名称

SOURCE_QUEUE_OWNER

VARCHAR2（128）

传播源队列的所有者

SOURCE_QUEUE_NAME

VARCHAR2（128）

传播的源队列的名称

DESTINATION_QUEUE_OWNER

VARCHAR2（128）

传播的目标队列的所有者

DESTINATION_QUEUE_NAME

VARCHAR2（128）

传播的目标队列的名称

DESTINATION_DBLINK

VARCHAR2（128）

用于将事件从源队列传播到目标队列的数据库链接

RULE_SET_OWNER

VARCHAR2（128）

传播正规则集的所有者

RULE_SET_NAME

VARCHAR2（128）

传播正规则集的名称

NEGATIVE_RULE_SET_OWNER

VARCHAR2（128）

传播否定规则集的所有者

NEGATIVE_RULE_SET_NAME

VARCHAR2（128）

传播否定规则集的名称

QUEUE_TO_QUEUE

VARCHAR2（5）

指示传播是否为队列到队列传播（ TRUE ）（ FALSE ）。队列到队列的传播始终具有自己的独占传播作业，以将消息从源队列传播到目标队列。

状态

VARCHAR2（8）

传播状况：

  • 禁用

  • 启用

  • ABORTED

错误信息

VARCHAR2（4000）

传播最后遇到的错误消息

ERROR_DATE

日期

传播最后遇到错误的时间

ORIGINAL_PROPAGATION_NAME

VARCHAR2（128）

克隆传播的原始传播

ORIGINAL_SOURCE_QUEUE_OWNER

VARCHAR2（128）

原始传播的源队列所有者

ORIGINAL_SOURCE_QUEUE_NAME

VARCHAR2（128）

原始传播的源队列名称

ACKED_SCN

数

确认目标队列中捕获的消息的订户的SCN用于传播

AUTO_MERGE_THRESHOLD

数

合并阈值以将传播合并回原始源队列
