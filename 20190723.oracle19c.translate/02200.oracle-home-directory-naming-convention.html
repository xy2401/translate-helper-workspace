Oracle主目录命名约定

默认情况下，Oracle Universal Installer使用这些Oracle Optimal Flexible Architecture约定来配置Oracle主目录。

Oracle homes的目录模式语法是/ pm / s / u / product / v / type _ [ n ]。下表描述了此语法中使用的变量：

变量 描述

下午

装入点名称。

小号

标准目录名称。

ü

目录所有者的名称。

v

该软件的版本。

类型

安装类型。例如：数据库（ dbhome ），客户端（ client ）或Oracle Grid Infrastructure（ grid ）

ñ

一个可选的计数器，使您可以在同一Oracle基本目录中多次安装相同的产品。例如：数据库1和数据库2（ dbhome_1 ， dbhome_2 ）

例如，以下路径是此系统上首次安装Oracle数据库的典型路径：

/u01/app/oracle/product/19.0.0/dbhome_1
