29 SDO_OLS包（OpenLS）

MDSYS.SDO_OLS包中包含Spatial和Graph OpenLS支持的子程序。

该要使用本章中的子程序，您必须了解OpenLS支持中有关OpenLS的概念和用法信息。

本章的其余部分提供了有关子程序的参考信息，按字母顺序列出。

  • SDO_OLS.MakeOpenLSClobRequest
  • SDO_OLS.MakeOpenLSRequest

29.1 SDO_OLS.MakeOpenLSClobRequest

格式

SDO_OLS.MakeOpenLSClobRequest（请求IN CLOB）返回CLOB;

描述

使用CLOB对象提交OpenLS请求，并将结果作为CLOB对象返回。

参数

请求

以CLOB对象的形式的OpenLS请求。

使用说明

要将输入请求指定为XMLType对象以返回XMLType对象，请使用SDO_OLS.MakeOpenLSRequest函数。

有关OpenLS支持的信息，请参阅OpenLS支持 。

例子

以下示例请求指定类别（即具有指定的SIC_code值）的最近的业务到指定位置（经度：-122.4083257，纬度：37.788208）。 

DECLARE请求CLOB;结果CLOB; BEGIN请求：= TO_CLOB（'<？xml version =“1.0”encoding =“UTF-8”？> <XLS xmlns =“http://www.opengis.net/xls”xmlns：gml =“http://www.opengis.net/gml”xmlns：xsi =“http://www.w3.org/ 2001 / XMLSchema-instance“version =”1.1“> <RequestHeader clientName =”someName“clientPassword =” password “/> <Request requestID =”123“maximumResponses =”100“version =”1.1“methodName =”DirectoryRequest“> < DirectoryRequest> <POILocation> <Nearest nearestCriterion =“Proximity”> <Position> <gml：Point xmlns：gml =“http://www.opengis.net/gml”> <gml：pos dimension =“2”srsName =“ 4326“>  -  122.4083257 37.788208 </ gml：pos> </ gml：Point> </ Position> </最近> </ POILocation> <POIProperties> <POIProperty name =”SIC_code“value =”1234567890“/> </ POIProperties > </ DirectoryRequest> </ Request> </ XLS>'）; result：= SDO_OLS.makeOpenLSClobRequest（request）;结束; /

29.2 SDO_OLS.MakeOpenLSRequest

格式

SDO_OLS.MakeOpenLSRequest（请求IN XMLTYPE）返回XMLTYPE;

描述

使用XMLType对象提交OpenLS请求，并将结果作为XMLType对象返回。

参数

请求

OpenLS请求以XMLType对象的形式。

使用说明

要将输入请求指定为CLOB并返回CLOB，请使用SDO_OLS.MakeOpenLSClobRequest函数。

有关OpenLS支持的信息，请参阅OpenLS支持 。

例子

以下示例请求指定类别（即具有指定的SIC_code值）的最近的业务到指定位置（经度：-122.4083257，纬度：37.788208）。 

SELECT SDO_OLS.makeOpenLSRequest（XMLTYPE（'<？xml version =“1.0”encoding =“UTF-8”？> <XLS xmlns =“http://www.opengis.net/xls”xmlns：gml =“http://www.opengis.net/gml”xmlns：xsi =“http://www.w3.org/ 2001 / XMLSchema-instance“version =”1.1“> <RequestHeader clientName =”someName“clientPassword =” password “/> <Request requestID =”123“maximumResponses =”100“version =”1.1“methodName =”DirectoryRequest“> < DirectoryRequest> <POILocation> <Nearest nearestCriterion =“Proximity”> <Position> <gml：Point xmlns：gml =“http://www.opengis.net/gml”> <gml：pos dimension =“2”srsName =“ 4326“>  -  122.4083257 37.788208 </ gml：pos> </ gml：Point> </ Position> </最近> </ POILocation> <POIProperties> <POIProperty name =”SIC_code“value =”1234567890“/> </ POIProperties > </ DirectoryRequest> </ Request> </ XLS>'））“OpenLS响应”FROM DUAL;
