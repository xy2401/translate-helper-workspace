8.157 V $ RECOVERY_FILE_STATUS

V$RECOVERY_FILE_STATUS为每个RECOVER语句的每个数据文件包含一行。 此视图仅包含执行恢复的Oracle进程的有用信息。当Recovery Manager指示服务器进程执行恢复时，只有Recovery Manager可以在此视图中查看相关信息。对于所有其他Oracle用户， V$RECOVERY_FILE_STATUS将为空。
柱 数据类型 描述

FILENUM

数

要恢复的文件的编号

文件名

VARCHAR2（513）

正在恢复的数据文件的文件名

状态

VARCHAR2（13）

恢复状况：

  • 在恢复中

  • 当前

  • 没有恢复

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
