38.13 PREDICTION_BOUNDS

句法


说明predict_bounds.eps的说明

mining_attribute_clause :: =


mining_attribute_clause.eps插图的描述

目的

PREDICTION_BOUNDS应用广义线性模型（GLM）来预测选择中每行的类或值。该函数返回具有UPPER和LOWER字段的对象的每个预测的上限和LOWER 。

GLM可以执行回归分析或二进制分类：

  • 回归的界限是指预测的目标值。UPPER和LOWER的数据类型是目标的数据类型。

  • 二进制分类的界限是指预测目标类或指定class_value的概率。UPPER和LOWER的数据类型是BINARY_DOUBLE 。

如果使用岭回归建立模型，或者如果在构建期间发现协方差矩阵是单数的，则PREDICTION_BOUNDS为两个边界返回NULL 。

confidence_level是范围（0,1）中的数字。默认值为0.95。您可以通过为confidence_level指定NULL来指定class_value同时将confidence_level保留为默认NULL 。

在对分区模型进行评分时， PREDICTION_BOUNDS函数的语法可以使用可选的GROUPING提示。请参阅GROUPING提示 。

mining_attribute_clause

mining_attribute_clause标识要用作评分预测变量的列属性。该子句的行为与PREDICTION函数的描述相同。（请注意，对分析语法的引用不适用。）请参阅“ mining_attribute_clause ” 。

  • 有关广义线性模型的信息，请参阅Oracle数据挖掘概念

例

以下示例返回预测年龄的客户分布，其置信度为98％且大于24且小于46。

SELECT count（cust_id）cust_count，cust_marital_status FROM（SELECT cust_id，cust_marital_status FROM mining_data_apply_v WHERE PREDICTION_BOUNDS（glmr_sh_regr_sample，0.98 USING *）。LOWER> 24 AND PREDICTION_BOUNDS（glmr_sh_regr_sample，0.98 USING *）。UPPER <46）GROUP BY cust_marital_status; CUST_COUNT CUST_MARITAL_STATUS -------------- -------------------- 46 NeverM 7 Mabsent 5 Separ。35离婚。72已婚
