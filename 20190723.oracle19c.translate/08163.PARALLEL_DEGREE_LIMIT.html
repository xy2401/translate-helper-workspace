1.244 PARALLEL_DEGREE_LIMIT

PARALLEL_DEGREE_LIMIT限制优化程序使用的并行度，以确保并行服务器进程不会泛滥系统。

属性 描述

参数类型

串

句法

PARALLEL_DEGREE_LIMIT = { CPU | IO | integer }

默认值

中央处理器

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

通过自动并行度，Oracle自动决定语句是否应该并行执行以及语句应该使用多大程度的并行性。优化程序根据语句的资源要求自动确定语句的并行度。但是， PARALLEL_DEGREE_LIMIT强制优化程序使用的并行度限制。

值

  • 中央处理器

    最大并行度受系统中CPU数量的限制。用于计算限制的公式是PARALLEL_THREADS_PER_CPU * CPU_COUNT *可用实例的数量（默认情况下，群集上的所有已打开实例，但可以使用PARALLEL_INSTANCE_GROUP或服务规范进行约束）。这是默认值。

  • IO

    优化程序可以使用的最大并行度受系统的I / O容量限制。通过将总系统吞吐量除以每个进程的最大I / O带宽来计算该值。必须在系统上运行DBMS_RESOURCE_MANAGER.CALIBRATE_IO过程才能使用IO设置。此过程将计算总系统吞吐量和每个进程的最大I / O带宽。

  • 整数

    此参数的数值指定优化程序在自动并行度处于活动状态时可为SQL语句选择的最大并行度。仅当PARALLEL_DEGREE_POLICY设置为ADAPTIVE ， AUTO或LIMITED ，才会启用自动并行度。
