1.101 DG_BROKER_CONFIG_FILE n

DG_BROKER_CONFIG_FILE n （其中n = 1,2）指定Data Guard代理配置文件的名称。

属性 描述

参数类型

串

句法

DG_BROKER_CONFIG_FILE[1 | 2] = filename

默认值

依赖于操作系统

可修改

改变系统

可在PDB中修改

没有

价值范围

一个文件名

基本

没有

属于Data Guard代理配置的每个数据库都有两个代理配置文件，其中包含描述配置状态和属性的条目（例如，属于配置的站点和数据库，每个的角色和属性）数据库，以及配置的每个元素的状态）。提供两个文件以始终保持配置的最后已知良好状态。

如果未明确定义DG_BROKER_CONFIG_FILE n ，则在实例启动时将其设置为特定于操作系统的默认值。仅当Data Guard代理未运行时，才能更改此参数。有关如何停止和启动代理的信息，请参阅“ DG_BROKER_START ” 。
