3.32 ALL_RULE_SET_RULES

ALL_RULE_SET_RULES描述了当前用户可访问的规则集中的规则。

相关观点

  • DBA_RULE_SET_RULES描述了数据库中所有规则集中的规则。

  • USER_RULE_SET_RULES描述了当前用户拥有的规则集中的规则。此视图不显示RULE_SET_OWNER列。

柱 数据类型 空值 描述

RULE_SET_OWNER

VARCHAR2（128）

不是NULL

规则集的所有者

RULE_SET_NAME

VARCHAR2（128）

不是NULL

规则集的名称

RULE_OWNER

VARCHAR2（128）

不是NULL

规则的所有者

RULE_NAME

VARCHAR2（128）

不是NULL

规则的名称

RULE_SET_RULE_ENABLED

VARCHAR2（8）

指示是否在规则集（ ENABLED ）中启用规则（ DISABLED ）（ DISABLED ）

RULE_SET_RULE_EVAL_CTX_OWNER

VARCHAR2（128）

将规则添加到规则集时指定的评估上下文的所有者（如果有）

RULE_SET_RULE_EVAL_CTX_NAME

VARCHAR2（128）

将规则添加到规则集时指定的评估上下文的名称（如果有）

RULE_SET_RULE_COMMENT

VARCHAR2（4000）

将规则添加到规则集时指定的注释（如果有）
