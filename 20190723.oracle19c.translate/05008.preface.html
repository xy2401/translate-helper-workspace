前言

本文档是Pro * C / C ++的综合用户指南和参考。它向您展示了如何使用数据库语言SQL和Oracle的过程扩展PL / SQL以及Pro * C / C ++来操作Oracle数据库中的数据。它探索了从底层概念到高级编程技术的全方位主题，并提供了代码示例。

本前言包含以下主题：

  • 目标受众

  • 文档可访问性

  • 相关文件

  • 约定

目标受众

Pro * C / C ++程序员指南适用于执行或有兴趣了解以下任务的程序员，系统分析员，项目经理和其他Oracle用户：

  • 在Oracle环境中设计和开发软件应用程序。

  • 转换现有软件应用程序以在Oracle环境中运行。

  • 管理软件应用程序的开发。

要使用本文档，您需要具备C和C ++应用程序编程的应用知识，并熟悉结构化查询语言（SQL）的使用。
