getExpandCharRefs（）

检查是否在DOM数据中扩展了字符引用。默认情况下，字符引用将替换为它们表示的字符。但是，保存文档时，这些字符实体不会再次出现。为了确保它们能够通过加载和保存，它们不应该被扩展。
