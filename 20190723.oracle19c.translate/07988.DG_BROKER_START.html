1.102 DG_BROKER_START

DG_BROKER_START使Oracle能够确定是否应启动Oracle Data Guard代理（DMON）进程。

属性 描述

参数类型

布尔

默认值

假

可修改

改变系统......SID = '*'

可在PDB中修改

没有

价值范围

是的|假

基本

没有

DMON是一个非致命的Oracle后台进程，只要该实例存在，只要此参数设置为true ，就会存在。

如果永远不会在Oracle Data Guard代理配置中配置数据库，则可以不指定此参数并接受默认值false 。如果数据库是Oracle Data Guard代理配置的一部分，则可以通过在初始化参数文件中将此参数设置为true来简化代理的自动启动。
