DROP SEQUENCE

目的

使用DROP SEQUENCE语句从数据库中删除序列。

您还可以使用此语句通过删除然后重新创建序列来重新启动序列。例如，如果序列的当前值为150，并且您希望重新启动序列值为27，则可以删除序列，然后使用相同的名称重新创建序列，并将START WITH值设置为27。

也可以看看：

创建序列和更改序列以获取有关创建和修改序列的更多信息

先决条件

序列必须位于您自己的架构中，或者您必须具有DROP ANY SEQUENCE系统特权。

句法

drop_sequence :: =


说明drop_sequence.eps的描述

语义

模式

指定包含序列的模式。如果省略schema ，则Oracle数据库假定序列位于您自己的模式中。

SEQUENCE_NAME

指定要删除的序列的名称。

例子

删除序列：示例

以下语句删除了用户oe拥有的序列customers_seq ，该序列是在“ 创建序列：示例 ”中创建的 。要发出此语句，您必须以用户oe身份连接或具有DROP ANY SEQUENCE系统特权：

DROP SEQUENCE oe.customers_seq;
