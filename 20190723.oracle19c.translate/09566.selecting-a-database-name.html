选择数据库名称

数据库名称由各种字符串组成，并且只能包含允许的字符。选择数据库名称时，请查看以下准则。

数据库名称输入字段设置以下Oracle初始化参数值：

  • DB_NAME

  • DB_UNIQUE_NAME

  • DB_DOMAIN

在Oracle RAC环境中，数据库名称（ DB_UNIQUE_NAME ）部分是一个不超过30个字符的字符串，可以包含字母数字，下划线（_），美元（$）和磅（＃）字符，但必须以字母字符开头。数据库名称中不允许使用其他特殊字符。数据库的DB_NAME参数设置为数据库名称的前8个字符。

全局数据库名称（ DB_DOMAIN ）的域部分不能超过128个字符。不允许使用下划线（_）的域名。DB_UNIQUE_NAME.DB_DOMAIN的整体值必须在企业内是唯一的。

注意：

对于Oracle Real Applications Cluster（Oracle RAC）数据库，可插拔数据库（PDB）名称在群集中必须是唯一的。

数据库名称和ORACLE_SID

Oracle服务标识符（SID）前缀是数据库名称的前8个字符。SID前缀只能包含字符az，AZ和0-9。SID前缀不能包含操作系统特殊字符，因此如果在数据库名称的前8个字符中使用特殊字符，则在SID前缀中省略这些特殊字符。每个数据库都有一个SID前缀。数据库的SID前缀在群集中必须是唯一的。

对于Oracle RAC数据库，每个实例都有一个唯一标识符ORACLE_SID ，它由SID前缀和实例编号组成。ORACLE_SID前缀最多可包含12个字符。Oracle RAC数据库实例的ORACLE_SID生成方式不同，具体取决于您选择如何管理数据库。如果选择策略管理的数据库，则Oracle将以name_#格式生成SID，其中name是DB_UNIQUE_NAME的前八个字母数字字符，而#是实例编号。如果选择管理员管理的数据库，则Oracle Database Configuration Assistant将使用格式name#生成实例名称的默认SID，其中name是DB_UNIQUE_NAME的前八个字母数字字符，而#是实例编号。但是，在安装或数据库创建期间，您可以为SID指定非默认值。对于每个实例，实例编号会自动添加到此字符串的末尾。

对于Oracle RAC One Node数据库，实例名称为ORACLE_SID_1 ，它由附加到SID前缀的_1组成。在线重定位期间，将启动第二个实例ORACLE_SID_2 ，它将成为重定位完成后的唯一实例。下一次在线重定位使用ORACLE_SID_1作为新实例。
