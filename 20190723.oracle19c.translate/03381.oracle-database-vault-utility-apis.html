20 Oracle Database Vault Utility API

Oracle Database Vault在DBMS_MACUTL PL / SQL包中提供了一组实用程序API。

  • DBMS_MACUTL常量
    您可以使用DBMS_MACUTL PL / SQL包中提供的一组常量。
  • DBMS_MACUTL包过程和函数
    DBMS_MACUTL PL / SQL包可以执行诸如查找时间值或用户是否具有适当权限等任务。

DBMS_MACUTL常量

您可以使用DBMS_MACUTL PL / SQL包中提供的一组常量。

  • DBMS_MACUTL常量列表
    DBMS_MACUTL PL / SQL包提供了与Oracle Database Vault PL / SQL包一起使用的常量（字段）。
  • 示例：使用DBMS_MACUTL常量创建领域
    在Oracle Database Vault中创建对象时，常量可用于回答简单的“是”或“否”设置。
  • 示例：使用DBMS_MACUTL常量创建规则集
    常量可用于设置选项，例如使用的审核类型或失败选项。
  • 示例：使用DBMS_MACUTL常量创建因子
    常量可用于设置特定于因素的信息，例如身份或标签。

DBMS_MACUTL包过程和函数

DBMS_MACUTL PL / SQL包可以执行诸如查找时间值或用户是否具有适当权限等任务。

  • CHECK_DVSYS_DML_ALLOWED过程
    CHECK_DVSYS_DML_ALLOWED过程检查用户是否可以发出数据修改语言（DML）命令来访问DVSYS对象。
  • GET_CODE_VALUE函数
    GET_CODE_VALUE函数查找代码组中代码的值，然后返回VARCHAR2值。
  • GET_SECOND功能
    GET_SECOND函数以Oracle SS（秒）格式（ GET_SECOND返回秒，然后返回NUMBER值。
  • GET_MINUTE函数
    GET_MINUTE函数以NUMBER值返回Oracle MI（分钟）格式（00-59）中的分钟。
  • GET_HOUR功能
    GET_HOUR函数以NUMBER值返回Oracle HH24（小时）格式（00-23）中的小时。
  • GET_DAY功能
    GET_DAY函数以NUMBER值返回Oracle DD（日）格式（01-31）中的日期。
  • GET_MONTH函数
    GET_MONTH函数以NUMBER值返回Oracle MM（月）格式（01-12）中的月份。
  • GET_YEAR函数
    GET_YEAR函数以NUMBER值返回Oracle YYYY（年）格式（0001-9999）中的年份。
  • IS_ALPHA功能
    IS_ALPHA函数返回一个BOOLEAN值，指示字符是否为字母。
  • IS_DIGIT功能
    IS_DIGIT函数检查返回一个BOOLEAN值，指示字符是否为数字。
  • IS_DVSYS_OWNER函数
    IS_DVSYS_OWNER函数返回BOOLEAN值，指示用户是否有权管理Oracle Database Vault配置。
  • IS_OLS_INSTALLED函数
    IS_OLS_INSTALLED函数返回BOOLEAN值，指示是否已安装Oracle Label Security。
  • IS_OLS_INSTALLED_VARCHAR函数
    IS_OLS_INSTALLED_VARCHAR函数返回BOOLEAN值，指示是否已安装Oracle Label Security。
  • USER_HAS_OBJECT_PRIVILEGE函数
    USER_HAS_OBJECT_PRIVILEGE函数返回BOOLEAN值，指示用户或角色是否可以通过单个指定的对象权限授予访问对象。
  • USER_HAS_ROLE函数
    USER_HAS_ROLE函数返回BOOLEAN值，指示用户是否具有直接或间接（通过其他角色）的角色权限。
  • USER_HAS_ROLE_VARCHAR函数
    USER_HAS_ROLE_VARCHAR函数返回VARCHAR2值，指示用户是否具有角色权限，直接或间接（通过其他角色）。
  • USER_HAS_SYSTEM_PRIVILEGE函数
    USER_HAS_SYSTEM_PRIVILEGE函数返回BOOLEAN值，指示用户是直接还是间接（通过角色）具有系统特权。

GET_SECOND功能

它对基于时间数据的规则表达式很有用。

句法

DBMS_MACUTL.GET_SECOND（p_date IN DATE DEFAULT SYSDATE）返回号码;

参数

表20-4 GET_SECOND参数

参数 描述

p_date

SS格式的日期（例如， 59 ）。

如果未指定日期，则Oracle Database Vault将使用Oracle Database SYSDATE函数检索为数据库所在的操作系统设置的当前日期和时间。

例

SET SERVEROUTPUT ON DECLARE秒号; BEGIN秒：= DBMS_MACUTL.GET_SECOND（TO_DATE（'03-APR-2009 6:56 PM'，'dd-mon-yyyy hh：mi PM'））; DBMS_OUTPUT.PUT_LINE（'Seconds：'|| seconds）;结束; /

此示例使用固定的日期和时间，返回以下内容：

秒：56

IS_DVSYS_OWNER函数返回BOOLEAN值，指示用户是否有权管理Oracle Database Vault配置。

如果用户已获得授权，则IS_DVSYS_OWNER返回TRUE 。

句法

DBMS_MACUTL.IS_DVSYS_OWNER（p_user IN VARCHAR2 DEFAULT USER）返回BOOLEAN;

参数

表20-12 IS_DVSYS_OWNER参数

参数 描述

p_user

用户检查。

要查找现有用户，请查询以下视图：

  • DBA_USERS ：查找当前数据库实例的可用用户。请参见Oracle数据库参考 。

  • DBA_DV_REALM_AUTH ：查找特定用户或角色的授权。请参阅DBA_DV_REALM_AUTH视图 。

  • DBA_DV_ROLE ：查找权限管理中使用的现有安全应用程序角色。请参阅DBA_DV_ROLE视图 。

例

SET SERVEROUTPUT开启如果DBMS_MACUTL.IS_DVSYS_OWNER（'PSMITH'）那么DBMS_OUTPUT.PUT_LINE（'PSMITH有权管理Database Vault。“）; ELSE DBMS_OUTPUT.PUT_LINE（'PSMITH无权管理Database Vault。“）;万一;结束; /

IS_OLS_INSTALLED函数

IS_OLS_INSTALLED函数返回BOOLEAN值，指示是否已安装Oracle Label Security。

如果安装了Oracle Label Security，则IS_OLS_INSTALLED返回TRUE 。

句法

DBMS_MACUTL.IS_OLS_INSTALLED（）返回BOOLEAN;

参数

没有

例

如果DBMS_MACUTL.IS_OLS_INSTALLED（）那么设置SERVEROUTPUT，那么DBMS_OUTPUT.PUT_LINE（'OLS已安装'）; ELSE DBMS_OUTPUT.PUT_LINE（'未安装OLS'）;万一;结束; /

USER_HAS_OBJECT_PRIVILEGE函数

USER_HAS_OBJECT_PRIVILEGE函数返回BOOLEAN值，指示用户或角色是否可以通过单个指定的对象权限授予访问对象。

如果用户或角色具有对象权限，则USER_HAS_OBJECT_PRIVILEGE返回TRUE 。

句法

DBMS_MACUTL.USER_HAS_OBJECT_PRIVILEGE（p_user VARCHAR2，p_object_owner VARCHAR2，p_object_name VARCHAR2，p_privilege VARCHAR2）RETURNS BOOLEAN;

参数

表20-13 USER_HAS_OBJECT_PRIVILEGE参数

参数 描述

p_user

要检查的用户或角色。

要查找现有用户，请查询以下视图：

  • DBA_USERS ：查找当前数据库实例的可用用户。请参见Oracle数据库参考 。

  • DBA_ROLES ：在当前数据库实例中查找可用角色。请参见Oracle数据库参考 。

  • DVA_DV_REALM_AUTH ：查找特定用户或角色的授权。请参阅DBA_DV_REALM_AUTH视图 。

  • DBA_DV_ROLE ：查找权限管理中使用的现有安全应用程序角色。请参阅DBA_DV_ROLE视图 。

p_object_owner

对象所有者，例如架构。

要查找可用用户，请查询Oracle Database Reference中描述的DBA_USERS视图。

要查找特定用户的授权，请查询他们的DVA_DV_REALM_AUTH视图。

p_object_name

对象名称，例如p_object_owner参数中指定的模式中的p_object_owner 。

要查找可用对象，请查询Oracle数据库参考中描述的ALL_OBJECTS视图。

要查找由现有领域保护的对象，请查询它们DBA_DV_REALM_OBJECT视图。

p_privilege

对象权限，例如UPDATE 。

要查找不包括PUBLIC权限的数据库帐户的权限，请查询它们的DBA_DV_USER_PRIVS视图。

要查找数据库帐户的所有权限，请查询DBA_DV_USER_PRIVS_ALL 。视图

例

SET SERVEROUTPUT开启如果DBMS_MACUTL.USER_HAS_OBJECT_PRIVILEGE（'SECTOR2_APP_MGR'，'OE'，'ORDERS'，'UPDATE'）那么DBMS_OUTPUT.PUT_LINE（'SECTOR2_APP_MGR具有OE.ORDERS表的UPDATE特权'）; ELSE DBMS_OUTPUT.PUT_LINE（'SECTOR2_APP_MGR没有OE.ORDERS表的UPDATE权限。“）;万一;结束; /

USER_HAS_SYSTEM_PRIVILEGE函数

USER_HAS_SYSTEM_PRIVILEGE函数返回BOOLEAN值，指示用户是直接还是间接（通过角色）具有系统特权。

如果用户具有指定的系统特权，则USER_HAS_SYSTEM_PRIVILEGE返回TRUE 。

句法

DBMS_MACUTL.USER_HAS_SYSTEM_PRIVILEGE（p_privilege IN VARCHAR2，p_user IN VARCHAR2 DEFAULT USER）RETURN BOOLEAN;

参数

表20-16 USER_HAS_SYSTEM_PRIVILEGE参数

参数 描述

p_privilege

要检查的系统特权。

要查找不包括PUBLIC权限的数据库帐户的权限，请查询DBA_DV_USER_PRIVS视图中描述的DBA_DV_USER_PRIVS视图 。

要查找数据库帐户的所有权限，请使用DBA_DV_USER_PRIVS_ALL ，如DBA_DV_USER_PRIVS_ALL View中所述 。

p_user

用户检查。

要查找现有用户，请查询以下视图：

  • DBA_USERS ：查找当前数据库实例的可用用户。请参见Oracle数据库参考 。

  • DBA_DV_REALM_AUTH ：查找特定用户或角色的授权。请参阅DBA_DV_REALM_AUTH视图 。

例

SET SERVEROUTPUT开启如果DBMS_MACUTL.USER_HAS_SYSTEM_PRIVILEGE（'EXECUTE'，'PSMITH'）那么DBMS_OUTPUT.PUT_LINE（'用户PSMITH具有EXECUTE ANY PRIVILEGE权限。“）; ELSE DBMS_OUTPUT.PUT_LINE（'用户PSMITH没有EXECUTE ANY PRIVILEGE权限。“）;万一;结束; /
