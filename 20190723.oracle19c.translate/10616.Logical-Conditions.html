逻辑条件

逻辑条件组合两个组件条件的结果以基于它们产生单个结果或者反转单个条件的结果。表6-4列出了逻辑条件。

表6-4逻辑条件

条件类型 手术 例子 
不

如果以下条件为FALSE则返回TRUE 。如果为TRUE ，则返回FALSE 。如果它是UNKNOWN ，那么它仍然是UNKNOWN 。

 
SELECT * FROM employees WHERE NOT（job_id IS NULL）ORDER BY employee_id; SELECT * FROM employees WHERE NOT（薪水在1000和2000之间）ORDER BY employee_id;
 
和

返回TRUE如果两个组件的条件是TRUE 。返回FALSE如果其中一个是FALSE 。否则返回UNKNOWN 。

 
SELECT * FROM employees WHERE job_id ='PU_CLERK'AND department_id = 30 ORDER BY employee_id;
 
要么

返回TRUE如果任一组件条件为TRUE 。返回FALSE如果两者都FALSE 。否则返回UNKNOWN 。

 
SELECT * FROM employees WHERE job_id ='PU_CLERK'ORr department_id = 10 ORDER BY employee_id;
