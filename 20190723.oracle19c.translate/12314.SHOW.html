12.42 SHOW

句法

SHO [W] 选项

其中option代表以下条款或条款之一：

system_variable ALL BTI [TLE] CON_ID CON_NAME EDITION ERR [ORS] [{ANALYTIC VIEW |属性维度| HIERARCHY |功能|程序|包装|包装身体| TRIGGER |查看| TYPE | TYPE BODY |尺寸| JAVA CLASS} [  schema 。] name ] HISTORY LNO LOBPREF [ETCH] PARAMETER [S] [ parameter_name ] PDBS PNO RECYC [LEBIN] [ original_name ] REL [EASE] REPF [OTER] REPH [EADER] ROWPREF [ETCH] SGA SPOO [L] SPPARAMETER [ S] [ parameter_name ] SQLCODE STATEMENTC [ACHE] TTI [TLE] USER XQUERY

展会 SQL * Plus系统变量或当前SQL * Plus环境的值 。SHOW SGA需要DBA特权登录。

条款

system_variable

表示SET命令设置的任何系统变量。

所有

按字母顺序列出除ERRORS和SGA之外的所有SHOW选项的设置。 

CON_ID

显示连接到统一数据库时连接到的Container的ID。如果在连接到非Consolidated Database时发出，则此命令返回0。 

CON_NAME

显示连接到统一数据库时连接到的Container的名称。对于非统一数据库，它将返回“Non Consolidated”。 

版

显示现有数据库的edition属性。

BTI [TLE]

显示当前的BTITLE定义。

 ERR [ORS] [{解析视图|属性维度| HIERARCHY |功能|程序|包装|包装身体| TRIGGER |查看| TYPE | TYPE BODY |尺寸| JAVA CLASS} [  架构 。] 名称 ]

显示存储过程的编译错误（包括存储的函数，过程和包）。使用CREATE命令创建存储过程后，如果存储过程有任何编译错误，则会显示一条消息。要查看错误，请使用SHOW ERRORS。

指定不带参数的SHOW ERRORS时，SQL * Plus会显示最近创建或更改的存储过程的编译错误。指定类型（分析视图，属性维，层次结构，函数，过程，包，包体，触发器，视图，类型，类型主体，维或java类）和PL / SQL存储过程的名称时，SQL * Plus显示该存储过程的错误。有关编译错误的更多信息，请参阅PL / SQL用户指南和参考。

schema包含命名对象。如果省略模式 ，则SHOW ERRORS假定该对象位于当前模式中。

SHOW ERRORS输出显示错误的行号和列号（LINE / COL）以及错误本身（ERROR）。LINE / COL和ERROR的默认宽度分别为8和65。您可以使用COLUMN命令更改默认宽度。

历史

显示使用SET HISTORY命令设置的当前命令历史记录状态。 

LNO

显示当前行号（显示当前页面和/或假脱机输出中的位置）。 

LOBPREFETCH

显示SQL * Plus一次从数据库中预取的LOB数据量的当前设置。有关设置SQL * Plus一次从数据库中预取的LOB数据量的更多信息，请参阅SET LOBPREFETCH 。

参数[ parameter_name ]

显示器一个或多个初始化参数的当前值。您可以在命令后使用字符串查看名称包含该字符串的参数子集。例如，如果输入：

显示参数计数
NAME TYPE VALUE ------------------------------ ----- ----- db_file_multiblock_read_count integer 12 spin_count integer 0

SHOW PARAMETERS命令在命令后面没有任何字符串，显示所有初始化参数。

您的输出可能会有所不同，具体取决于您连接的Oracle数据库服务器的版本和配置。您需要SELECT ON V_ $ PARAMETER对象权限才能使用PARAMETERS子句，否则您将收到一条消息

ORA-00942：表或视图不存在
PDBS

显示您所连接的统一数据库中可插拔数据库的名称，ID，模式和限制状态。如果已连接到非合并数据库，则该命令不可用。

PDBS选项仅在使用DBA权限登录时可用。对于非DBA用户，尝试使用PDBS选项将返回错误SP2-0382：SHOW PDBS命令不可用。

PNO

显示当前页码。 

RECYC [LEBIN] [ original_name ]

显示可以使用FLASHBACK BEFORE DROP命令恢复的回收站中的对象。您不需要记住列名，也不需要解释查询中可读性较差的输出：

SELECT * FROM USER_RECYCLEBIN

该查询返回按以下顺序显示的四列：

列名称 描述

原始名称

显示创建对象时使用的原始名称。

RECYCLEBIN NAME

显示用于标识recyclebin中对象的名称。

对象类型

显示对象的类型。

下降时间

显示删除对象的时间。

可以使用COLUMN命令格式化输出列。

对于DBA，该命令列出了自己的对象，因为它们具有自己的user_recyclebin视图。

发布]

显示SQL * Plus正在访问的Oracle数据库的版本号。 

REPF [OOTER]

显示当前的REPFOOTER定义。 

REPH [EADER]

显示当前的REPHEADER定义。 

ROWPREFETCH

显示SQL * Plus一次从数据库中预取的行数的当前设置。有关设置SQL * Plus一次从数据库中预取的行数的详细信息，请参阅SET ROWPREFETCH 。

SPOO [L]

显示是否正在假脱机输出。 

SGA

显示有关当前实例的系统全局区域的信息。您需要SELECT ON V_ $ SGA对象权限，否则您将收到一条消息

ORA-00942：表或视图不存在
SPPARAMETERS [ parameter_name ]

至于SHOW PARAMETERS，除了显示SHOW SPPARAMETERS 所有实例的初始化参数的当前值。您可以在命令后使用字符串查看名称包含该字符串的参数子集。

SHOW SPPARAMETERS命令在命令后面没有任何字符串，显示所有实例的所有初始化参数。

您的输出可能会有所不同，具体取决于您连接的Oracle数据库服务器的版本和配置。您需要SELECT ON V_ $ PARAMETER对象权限才能使用SPPARAMETERS子句。

SQLCODE

显示SQL.SQLCODE的值（最近操作的SQL返回码）。 

STATEMENTCACHE

显示语句高速缓存大小的当前设置。有关设置语句高速缓存大小的更多信息，请参阅SET STATEMENTCACHE 。

TTI [TLE]

显示当前的TTITLE定义。 

用户

显示您当前用于访问SQL * Plus的用户名。如果以“/ AS SYSDBA”连接，则显示SHOW USER命令

USER是“SYS”
XQUERY

显示XQUERY设置，BASEURI，CONTEXT，NODE和ORDERING的当前值。

xquery BASEURI“public / scott”语境“doc（”test.xml“）”NODE byreference ORDERING ordered

未设置任何值时，将显示以下输出：

xquery BASEURI“”CONTEXT“”NODE默认ORDERING默认值
