9.4加载Java类

编写Java类之后，使用loadjava工具将Java存储过程上载到Oracle数据库，如下所示：

> loadjava -u HR @ myPC：1521：orcl -v -r -t POManager.java密码： 密码初始化完成加载：POManager创建：POManager解析器：解析器（（“*”HR）（“*”public）（“* “ - ））解决：POManager

-v选项启用详细模式， -r选项编译上载的Java源文件并解析类中的外部引用， -t选项告诉loadjava工具使用客户端JDBC Thin驱动程序连接到数据库。
