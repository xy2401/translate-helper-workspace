4.131 DBA_AUTOTASK_WINDOW_CLIENTS

DBA_AUTOTASK_WINDOW_CLIENTS显示属于MAINTENANCE_WINDOW_GROUP的窗口，以及每个维护任务窗口的“已启用”或“已禁用”状态。

DBA_AUTOTASK_WINDOW_CLIENTS主要由Enterprise Manager使用。

柱 数据类型 空值 描述

WINDOW_NAME

VARCHAR2（128）

不是NULL

维护窗口的名称

WINDOW_NEXT_TIME

TIMESTAMP（6）带时区

除非禁用窗口，否则下一个计划窗口打开时间

WINDOW_ACTIVE

VARCHAR2（5）

指示窗口当前是否处于活动状态（打开）（ TRUE ）（ FALSE ）

AUTOTASK_STATUS

VARCHAR2（8）

自动维护任务子系统的状态：

  • 启用

  • 禁用

OPTIMIZER_STATS

VARCHAR2（8）

优化程序统计信息收集的状态：

  • 启用

  • 禁用

SEGMENT_ADVISOR

VARCHAR2（8）

细分顾问的状态：

  • 启用

  • 禁用

SQL_TUNE_ADVISOR

VARCHAR2（8）

SQL Tuning Advisor的状态：

  • 启用

  • 禁用
