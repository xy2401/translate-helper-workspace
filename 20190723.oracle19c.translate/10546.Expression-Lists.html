表达列表

表达式列表是其他表达式的组合。

expression_list :: =


说明expression_list.eps的描述

表达式列表可以出现在比较和成员条件以及查询和子查询的GROUP BY子句中。比较或成员条件中的表达式列表有时称为行值构造函数或行构造函数 。

比较和成员条件出现在WHERE子句的条件中。它们可以包含一个或多个逗号分隔的表达式或一组或多组表达式，其中每个集合包含一个或多个逗号分隔的表达式。在后一种情况下（多组表达式）：

  • 每组都用括号括起来

  • 每个集合必须包含相同数量的表达式

  • 每个集合中的表达式数量必须与比较条件中运算符之前或成员条件中IN关键字之前的表达式数量相匹配。

逗号分隔的表达式列表可以包含不超过1000个表达式。逗号分隔的表达式列表可以包含任意数量的集合，但每个集合可以包含不超过1000个表达式。

以下是条件中的一些有效表达式列表：

（10,20,40）（'SCOTT'，'BLAKE'，'TAYLOR'）（（'Guy'，'Himuro'，'GHIMURO'），（'Karen'，'Colmenares'，'KCOLMENA'））

在第三个示例中，每个集合中的表达式数量必须等于条件第一部分中的表达式数量。例如：

SELECT * FROM employees WHERE（ first_name，last_name，email ）IN （（'Guy'，'Himuro'，'GHIMURO'），（'Karen'，'Colmenares'，'KCOLMENA'））

SELECT department_id，MIN（工资）min，MAX（工资）max FROM员工GROUP BY department_id，工资ORDER BY department_id，min，max; SELECT department_id，MIN（工资）min，MAX（工资）max FROM employees GROUP BY（department_id，salary）ORDER BY department_id，min，max;
