网络服务配置文件（sqlnet.ora）

安装数据库软件后，Oracle Universal Installer启动Oracle Net Configuration Assistant（NETCA）。NETCA创建Oracle Net Services概要文件或sqlnet.ora文件。

在Oracle Grid Infrastructure安装中， sqlnet.ora文件默认位于以下目录中：

Grid_home / network / admin

对于Oracle RAC数据库实例的本地侦听器， sqlnet.ora文件的缺省位置是$ORACLE_HOME/network/admin目录。在此目录中，有一个默认的sqlnet.ora文件。此外，您可以在子目录sample找到示例sqlnet.ora文件。

在安装Oracle RAC软件期间，NETCA在sqlnet.ora文件中创建以下条目，其中$ORACLE_BASE是Oracle RAC安装的Oracle基本目录的路径：

NAMES.DIRECTORY_PATH =（TNSNAMES，EZCONNECT）ADR_BASE = $ ORACLE_BASE

参数NAMES.DIRECTORY_PATH指定用于解析连接标识符以连接描述符的命名方法的优先级顺序。

ADR_BASE参数指定在为数据库启用自动诊断存储库（ADR）时将跟踪和日志记录事件存储到的基本目录。
