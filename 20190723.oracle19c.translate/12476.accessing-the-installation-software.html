访问安装软件

您可以从Oracle Technology Network网站或Oracle Software Delivery Cloud门户下载Oracle数据库软件。在某些情况下，Oracle数据库软件也可能在安装介质上可用。

要从硬盘安装软件，您必须下载并解压缩，或者从安装介质中复制它（如果有）。

  • 下载Oracle软件
    选择要用于下载软件的方法。
  • 从OTN下载安装存档文件
    从Oracle技术网（OTN）下载安装归档文件。
  • 从Oracle Software Delivery Cloud Portal下载软件
    您可以从Oracle Software Delivery Cloud下载软件作为媒体包。
  • 将软件复制到硬盘
    Oracle建议您将安装软件复制到硬盘，以使安装更快地运行。
