3使用Oracle预安装RPM自动配置Oracle Linux

使用Oracle预安装RPM简化操作系统配置，为Oracle软件安装做好准备。

Oracle建议您安装Oracle Linux 7并使用Oracle Preinstallation RPM配置Oracle数据库和Oracle Grid Infrastructure安装的操作系统。

  • 使用Oracle预安装RPM的Oracle Linux配置概述
    使用Oracle预安装RPM来简化操作系统配置，并确保您具有所需的内核包。
  • 使用ULN支持安装Oracle预安装RPM
    使用此过程为Oracle软件订阅Unbreakable Linux Network（ULN）Oracle Linux通道。
  • 从坚不可摧的Linux网络安装Oracle预安装RPM
    使用此过程订阅Oracle Linux通道，并添加分发Oracle预安装RPM的Oracle Linux通道。
  • 从DVD或映像安装新的Oracle Linux安装
    使用此过程安装新的Oracle Linux安装并使用Oracle预安装RPM执行系统配置：
  • 使用Oracle Linux Yum Server支持安装Oracle Linux
    使用此过程安装Oracle Linux并使用Oracle Linux yum服务器配置Linux安装以获取安全勘误或错误修复更新。
  • 配置Oracle Ksplice以执行Oracle Linux更新
    如果您拥有Oracle Linux Premier Support订阅和访问密钥（可在Unbreakable Linux Network（ULN）上使用），则可以将Oracle Ksplice配置为对您的操作系统执行零停机更新。
  • 配置其他操作系统功能
    Oracle建议您在开始安装之前配置操作系统，并使用其他功能，例如IPMI或其他编程环境。
