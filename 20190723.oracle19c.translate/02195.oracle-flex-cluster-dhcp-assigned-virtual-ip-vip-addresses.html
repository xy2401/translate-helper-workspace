Oracle Flex Cluster DHCP分配的虚拟IP（VIP）地址

为所有群集节点配置群集节点VIP名称。

DHCP分配的VIP地址的要求

如果要配置DHCP分配的VIP，则在安装期间，按如下方式配置群集节点VIP名称：

  • 自动分配的名称：选择“ 配置节点动态网络分配的虚拟IP”选项，以允许安装程序自动为通过DHCP生成的VIP地址分配名称。地址通过DHCP分配，并由GNS解决。 Oracle Clusterware使用client ID nodename -vip发送DHCP请求，而不使用MAC地址。您可以使用cluvfy comp dhcp命令验证DHCP地址的可用性。
