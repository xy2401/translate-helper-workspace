3.108 ALL_TAB_HISTOGRAMS

ALL_TAB_HISTOGRAMS描述了当前用户可访问的表和视图的直方图。

ALL_TAB_HISTOGRAMS视图可能包含单桶直方图，实际上它表示Oracle数据库软件的“无直方图”。因此，不应该询问是否在特定列上指示直方图的存在或不存在。而是在ALL_TAB_COL_STATISTICS视图中查询列HISTOGRAM的值。

相关观点

  • DBA_TAB_HISTOGRAMS描述数据库中所有表和视图的直方图。

  • USER_TAB_HISTOGRAMS描述了当前用户拥有的所有表和视图的直方图。此视图不显示OWNER列。

注意：

仅当使用DBMS_STATS包收集有关表的统计信息时，才会填充这些视图。有关更多信息，请参阅Oracle Database PL / SQL包和类型参考 。

对于在除全局临时表之外的任何表上收集的统计信息，该值为SHARED 。

对于全局临时表，可能的值为：

  • SESSION - 表示统计信息是特定于会话的

  • SHARED - 表示统计信息在所有会话中共享
