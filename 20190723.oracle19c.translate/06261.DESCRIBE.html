2.17描述

目的

使用DESCRIBE命令列出表或视图的列定义。

此命令在RMAN中提供SQL * Plus DESCRIBE命令的功能。

先决条件

要访问另一个模式中的表或视图，您必须对该对象具有READ或SELECT权限，或者以AS SYSDBA模式连接。SYSBACKUP权限不授予对用户表或视图的访问权限。

使用说明

描述为表或视图中的每列提供此信息：

  • 名称

  • 是否允许空值（ NULL或NOT NULL ）

  • 数据类型以及适用的精度或比例

句法

describecmd :: =


插图描述describecmd.eps

语义

语法元素 描述

目录

将对象标识为驻留在包含恢复目录的数据库中，

模式

表或视图的模式位置，仅当它不在您的登录模式中时才需要。

表

表或视图的名称。

DBLINK

存在对象的数据库的数据库链接名称，仅当它不在登录数据库中时才需要。

例子

此示例描述了V$CONTROLFILE表：
