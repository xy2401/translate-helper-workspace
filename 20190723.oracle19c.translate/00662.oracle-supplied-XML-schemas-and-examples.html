甲 Oracle提供XML模式和实施例

此处提供了完整列表，包括Oracle XML DB提供的XML模式，采购订单XML模式和各种示例中使用的XSLT样式表，以及用于将XML内容加载到Oracle XML DB并初始化和终止的XML语言（OCI）示例。 XML上下文。

  • XDBResource.xsd：Oracle XML DB资源的XML模式
    提供了Oracle XML DB提供的XML模式XDBResource.xsd的完整列表，该模式用于表示Oracle XML DB资源。
  • XDBResConfig.xsd：资源配置的XML模式
    提供了Oracle XML DB提供的用于配置存储库资源的XML模式的完整列表。可以在Oracle XML DB Repository的路径/sys/schemas/PUBLIC/xmlns.oracle.com/xdb/XDBResConfig.xsd访问它。
  • acl.xsd：ACL的XML模式
    提供了Oracle数据库提供的用于表示访问控制列表（ACL）的XML模式。
  • xdbconfig.xsd：用于配置Oracle XML DB的XML架构
    提供了文件xdbconfig.xsd完整列表，其中包含用于配置Oracle XML DB的XML架构。
  • xdiff.xsd：用于比较就地演进模式的XML模式
    提供了xdiff.xsd完整列表，这是Oracle XML DB提供的XML模式，文档指定为过程DBMS_XMLSCHEMA.inPlaceEvolve的diffXML参数必须符合该模式。
  • 采购订单XML模式
    提供了各种示例中使用的采购订单XML模式的完整列表。
  • XSLT样式表示例，PurchaseOrder.xsl
    完整列表提供了XSLT样式表PurchaseOrder.xsl ，它在各种示例中使用。
  • 使用C（OCI）加载XML数据
    提供了将XML数据插入XMLType表的C程序的完整列表。
  • 初始化和终止XML上下文（OCI）
    示例显示如何使用OCI函数OCIXmlDbInitXmlCtx()和OCIXmlDbFreeXmlCtx()来初始化和终止XML上下文。它使用C DOM API构造XML文档并将其保存到数据库中。
