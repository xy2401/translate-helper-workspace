说明adfns063.eps的描述

该图显示：

  • 代表Oracle数据库磁盘存储的圆柱体，包含PL / SQL子程序和Java方法

  • 表示Oracle Server流程执行的框，包含PL / SQL解释器，Java虚拟机和SQL引擎

  • 表示外部进程执行的两个框：一个包含DLL，另一个包含外部C进程

使用这些对象之间的箭头显示交互：数据库柱面和Oracle Server Process Execution框之间的双箭头，此框和External C Process框之间的双箭头，以及从DLL框指向外部C进程的箭头框。
