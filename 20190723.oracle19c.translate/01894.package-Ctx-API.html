getEncoding（）

返回XML上下文使用的数据编码。通常，数据编码由用户选择，因此不需要该功能。但是，如果未指定数据编码，并且允许使用默认值，则可以使用此函数返回该默认编码的名称。

句法

oratext * getEncoding（）const;

返回

(oratext *)数据编码的名称

getErrHandler（）

此成员函数返回用户在创建上下文时提供的错误处理程序，如果没有提供则返回NULL 。

句法

ErrorHandler * getErrHandler（）const;

返回

(ErrorHandler *)指向Error Handler对象的指针，或NULL

getMemAllocator（）

此成员函数返回用户在创建上下文时提供的内存分配器或默认内存分配器。重要的是，此内存分配器用于所有C级内存分配

句法

MemAllocator * getMemAllocator（）const;

返回

(MemAllocator*)指向内存分配器对象的指针

isSimple（）

返回一个标志，指出上下文的数据编码是否“简单”，每个字符的单字节，如ASCII或EBCDIC。

句法

boolean isSimple（）const;

返回

(boolean)数据编码的TRUE是“简单”，否则为FALSE

isUnicode（）

返回一个标志，指出上下文的数据编码是否为Unicode，UTF-16，每个字符都有两个字节。

句法

boolean isUnicode（）const;

返回

(boolean)如果数据编码是Unicode，则为TRUE ，否则为FALSE
