功能表达

您可以使用任何内置SQL函数或用户定义函数作为表达式。一些有效的内置函数表达式是：

长度（'BLAKE'）ROUND（1234.567 * 43）SYSDATE

也可以看看：

关于SQL函数和聚合函数以获取有关内置函数的信息

用户定义的函数表达式指定对以下内容的调用：

  • Oracle提供的包中的函数（请参阅Oracle数据库PL / SQL包和类型参考 ）

  • 用户定义的包或类型或独立的用户定义函数中的函数（请参阅关于用户定义的函数 ）

  • 用户定义的函数或运算符（请参阅CREATE OPERATOR ， CREATE FUNCTION和Oracle Database Data Cartridge开发人员指南 ）

一些有效的用户定义函数表达式是：

circle_area（radius）payroll.tax_rate（empno）hr.employees.comm_pct@remote（dependents，empno）DBMS_LOB.getlength（column_name）my_function（a_column）

在用作表达式的用户定义函数中，支持位置，命名和混合表示法。例如，以下所有表示法都是正确的：

CALL my_function（arg1 => 3，arg2 => 4）...

CALL my_function（3,4）......CALL my_function（3，arg2 => 4）...

用户定义函数表达式的限制

您不能将对象类型或XMLType参数传递给远程函数和过程。
