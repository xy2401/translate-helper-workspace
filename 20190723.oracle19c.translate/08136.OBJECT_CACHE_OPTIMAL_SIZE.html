1.217 OBJECT_CACHE_OPTIMAL_SIZE

OBJECT_CACHE_OPTIMAL_SIZE指定（以字节为单位）当缓存大小超过最大大小时会话对象缓存减少的大小。

属性 描述

参数类型

整数

默认值

10240000 （10M）

可修改

ALTER SESSION ， ALTER SYSTEM ...DEFERRED

可在PDB中修改

是

价值范围

10 KB到操作系统相关的最大值

基本

没有

对象缓存是客户端上的一个内存块，它允许应用程序存储整个对象并在它们之间导航，而无需往返服务器。在服务器上，它用于缓存由于客户端请求而请求的常用对象，以帮助提高性能。
