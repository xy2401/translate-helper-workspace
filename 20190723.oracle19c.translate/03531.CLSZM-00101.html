22 CLSZM-00101至CLSZM-00617

CLSZM-00101：操作被拒绝。

原因：安全策略拒绝了该操作。

操作：以具有执行操作权限的用户身份调用操作。

CLSZM-00102：程序没有找到类型的对象string名为“ string ”。

原因：不存在具有所请求名称的对象。

操作：指定存在的名称，或首先创建它。

CLSZM-00103：程序无法创建名称为' string '的string对象，该对象已存在。

原因：使对象失败，因为它已经存在该名称。

操作：指定尚不存在的名称。

CLSZM-00104：给定名称' string '太长。最大长度为string 。

原因：请求的名称长于允许的名称。

行动：提供较短的名称。

CLSZM-00105：程序无法向ACL添加其他条目。最大条目数是string 。

原因：程序无法添加到ACL的条目。

操作：考虑使用角色来简化ACL。

CLSZM-00106：程序无法将角色string添加到角色string因为它会导致循环。

原因：添加子角色会导致角色包含自身。

行动：简化角色包含。

CLSZM-00107：程序无法销毁繁忙的string对象。string对象仍在使用它。

原因：在其他对象中有对象的引用，并且销毁它会导致不一致。

操作：首先删除引用，或者执行递归销毁。

CLSZM-00108：程序对名称为“ string ”的string对象的操作失败，因为它已从存储库中删除。

原因：该对象已被其他进程或用户删除，因此无法继续执行此操作。

行动：了解删除的原因，并考虑替代行动。
