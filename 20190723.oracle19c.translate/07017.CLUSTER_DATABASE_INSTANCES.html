1.41 CLUSTER_DATABASE_INSTANCES

CLUSTER_DATABASE_INSTANCES是一个Oracle RAC参数，用于指定作为群集数据库的一部分配置的实例数。

属性 描述

参数类型

整数

默认值

如果CLUSTER_DATABASE设置为false ，则为1

如果CLUSTER_DATABASE设置为true ，则配置Oracle RAC实例的数量

可修改

没有

可在PDB中修改

没有

价值范围

任何非零值

基本

没有

Oracle RAC

多个实例应具有相同的值。

您必须为每个实例设置此参数。通常，您应将此参数设置为Oracle RAC环境中的实例数。适当设置此参数可以改善内存使用。

注意：

在Oracle Database 19c版本19.1中不推荐使用CLUSTER_DATABASE_INSTANCES参数，并且可能在将来的发行版中不支持该参数。从Oracle Database 19c版本19.1开始，可配置的Oracle RAC实例的数量来自Oracle Clusterware。
