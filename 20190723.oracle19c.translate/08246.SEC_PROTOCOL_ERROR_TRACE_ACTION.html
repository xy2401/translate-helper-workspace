1.300 SEC_PROTOCOL_ERROR_TRACE_ACTION

SEC_PROTOCOL_ERROR_TRACE_ACTION指定数据库服务器在从可能的恶意客户端接收到错误数据包时所执行的日志记录类型，而不是客户端接收错误。

属性 描述

参数类型

串

句法

SEC_PROTOCOL_ERROR_TRACE_ACTION = {无|追踪|日志| ALERT}

默认值

跟踪

可修改

改变系统

可在PDB中修改

没有

基本

没有

值

  • 没有

    数据库服务器不会将其记录在任何跟踪文件中。

  • 跟踪

    警报日志文件中会生成一个简短条目，并生成详细的事件文件。服务器跟踪文件还将包含有关协议错误和事件文件的条目。

  • LOG

    服务器跟踪文件将包含有关协议违规的条目。

  • 警报

    警报日志文件和服务器跟踪文件中有关于协议违规的简短条目。
