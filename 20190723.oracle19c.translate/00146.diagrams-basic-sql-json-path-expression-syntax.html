注意：

  • 数组索引是从零开始的，因此integer是一个非负整数（0,1,2,3，...）。

  • 范围的第一个integer （ integer to integer ）必须小于第二个integer 。

  • 数组元素必须按索引按升序指定，不得重复。

如果违反任何这些语法规则，则会引发编译时错误。
