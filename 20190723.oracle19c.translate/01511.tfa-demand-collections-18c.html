此图像描绘了按需收集发生的事件流。

  • dba或sysadmin使用tfactl summary请求tfactl summary ，Oracle Trace File Analyzer提供系统，更改，事件，运行状况和分析的摘要

  • 如果有任何问题并且需要更多信息，则DBA使用tfactl请求进一步分析，Oracle Trace File Analyzer使用支持工具执行分析

  • 如果仍需要更多帮助，则DBA使用tfactl diagcollect请求集合，Oracle Trace File Analyzer收集并打包所有相关诊断

  • 然后，DBA将程序包上载到Oracle Support以帮助解决问题
