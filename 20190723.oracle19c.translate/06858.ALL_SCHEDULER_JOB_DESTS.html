3.50 ALL_SCHEDULER_JOB_DESTS

ALL_SCHEDULER_JOB_DESTS显示有关当前用户在其每个目的地可访问的作业状态的信息。

相关观点

  • DBA_SCHEDULER_JOB_DESTS显示有关每个目标的数据库中所有作业的状态的信息。

  • USER_SCHEDULER_JOB_DESTS显示有关当前用户在其每个目标上拥有的作业状态的信息。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

调度程序作业的所有者

JOB_NAME

VARCHAR2（128）

调度程序作业的名称

JOB_SUBNAME

VARCHAR2（128）

调度程序作业的子名称

CREDENTIAL_OWNER

VARCHAR2（128）

用于远程目标的凭据的所有者

CREDENTIAL_NAME

VARCHAR2（128）

用于远程目标的凭据的名称

DESTINATION_OWNER

VARCHAR2（128）

指向目标的目标对象的所有者

目的地

VARCHAR2（128）

目标对象的名称或目标本身的名称

JOB_DEST_ID

数

分配给此目的地的作业的数字ID

启用

VARCHAR2（5）

指示父作业是启用（ TRUE ）还是禁用（ FALSE ）

REFS_ENABLED

VARCHAR2（5）

指示此目标及其代理是启用（ TRUE ）还是禁用（ FALSE ）

州

VARCHAR2（15）

这个目的地的工作状态：

  • 禁用

  • RUNNING

  • CHAIN_STALLED

  • 排定

  • 重新计划

  • 准备好跑

  • 已完成

  • 破碎

  • 失败

  • SUCCEEDED

  • 远程

  • 已停止

NEXT_START_DATE

TIMESTAMP（6）带时区

此目的地的下一个工作开始时间

RUN_COUNT

数

此作业在此目标上运行的次数

RETRY_COUNT

数

在此目的地重试此作业的次数

FAILURE_COUNT

数

此作业在此目的地失败的次数

LAST_START_DATE

TIMESTAMP（6）带时区

上次这个工作在这个目的地开始了

LAST_END_DATE

TIMESTAMP（6）带时区

上次这个工作在这个目的地结束了
