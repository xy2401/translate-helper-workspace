1.68 DB_BLOCK_BUFFERS

DB_BLOCK_BUFFERS指定缓冲区高速缓存中的数据库缓冲区数。

属性 描述

参数类型

整数

默认值

0

可修改

没有

可在PDB中修改

没有

价值范围

50到特定于操作系统的最大值

基本

没有

Oracle RAC

多个实例可以具有不同的值，您可以根据需要更改值。

注意：

DB_BLOCK_BUFFERS不能与动态DB_CACHE_SIZE参数组合使用;在同一参数文件中组合这些参数将产生错误。

DB_BLOCK_BUFFERS是导致实例的SGA的总内存需求的几个参数之一。

此参数与DB_BLOCK_SIZE一起确定缓冲区高速缓存的总大小。有效使用缓冲区缓存可以大大减少数据库的I / O负载。只有在首次创建数据库时才能指定DB_BLOCK_SIZE ，因此使用DB_BLOCK_BUFFERS来控制缓冲区高速缓存的大小。

注意：

不推荐使用DB_BLOCK_BUFFERS初始化参数。它是为了向后兼容而维护的。

Oracle建议您改用DB_CACHE_SIZE 。
