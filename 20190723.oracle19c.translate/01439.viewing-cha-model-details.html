3.5查看Oracle Cluster Health Advisor模型的详细信息

使用chactl query model命令查看模型详细信息。

  1. 您可以使用chactl query model命令随时查看Oracle Cluster Health Advisor模型的详细信息。
    例如：
    $ chactl查询模型-name weekday型号：工作日目标类型：CLUSTERWARE版本：OS12.2_V14_0.9.8 OS校准：Linux amd64校准目标名称：MYCLUSTER校准日期：2016-07-05 01:13:49校准时间范围：开始= 2016-07-03 20：50：00，end = 2016-07-04 15:00:00校准KPI：未指定

    您还可以重命名，导入，导出和删除模型。
