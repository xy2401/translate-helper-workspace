13.58标量变量声明

标量变量存储没有内部组件的值。价值可以改变。标量变量声明指定变量的名称和数据类型，并为其分配存储空间。

声明还可以分配初始值并强制使用NOT NULL约束。

您可以通过名称引用标量变量。

  • 句法

  • 语义

  • 例子

句法

variable_declaration :: =


插图变量描述variable_declaration.eps

（ 表达式:: = ）

语义

variable_declaration

变量

您要声明的变量的名称。

数据类型

标量数据类型的名称，包括大小，精度，字符或字节语义的任何限定符。

有关标量数据类型的信息，请参阅“ PL / SQL数据类型 ” 。

不是NULL

对变量强加NOT NULL约束。有关此约束的信息，请参阅“ NOT NULL约束 ” 。

表达

在详细说明声明时要赋值给变量的值。 expression和variable必须具有兼容的数据类型。
