7.154 V $ CR_BLOCK_SERVER

V$CR_BLOCK_SERVER显示有关缓存融合中使用的全局缓存服务进程（LMS）的统计信息。

柱 数据类型 描述

CR_REQUESTS

数

由于远程CR块请求而服务的CR块数

CURRENT_REQUESTS

数

由于远程CR块请求而服务的当前块数

CR_REQUESTS + CURRENT_REQUESTS =服务的全局高速缓存CR时钟（来自V$SYSSTAT ）。

DATA_REQUESTS

数

数据块的当前或CR请求数

UNDO_REQUESTS

数

撤消块的CR请求数

TX_REQUESTS

数

撤消段头块的CR请求数

OTHER_REQUESTS

数

其他类型块的CR请求数

DATA_REQUESTS + UNDO_REQUESTS + TX_REQUESTS + OTHER_REQUESTS = LMS进程处理的请求总数

CURRENT_RESULTS

数

返回到请求实例的块中没有更改的请求数

PRIVATE_RESULTS

数

返回到请求实例的块的更改请求的数量，并且只有请求的事务可以使用生成的CR块

ZERO_RESULTS

数

返回到请求实例的块的更改请求数。只有零XID事务才能使用该块。

DISK_READ_RESULTS

数

请求实例必须从磁盘读取请求的块的请求数

FAIL_RESULTS

数

失败的请求数;请求交易必须重新发出请求

陈旧

数

请求块的磁盘读取过期的请求数

FAIRNESS_DOWN_CONVERTS

数

接收请求的实例已对块上的X锁进行下转换的次数，因为它没有修改块

FAIRNESS_CLEARS

数

“公平计数器”被清除的次数。此计数器跟踪块在服务后被修改的次数。

FREE_GC_ELEMENTS

数

从另一个实例收到请求的次数，并且X锁没有缓冲区

刷新

数

LMS进程刷新日志的次数

FLUSHES_QUEUED

数

LMS进程排队的刷新次数

FLUSH_QUEUE_FULL

数

刷新队列已满的次数

FLUSH_MAX_TIME

数

最大冲洗时间

LIGHT_WORKS

数

引起轻工规则的次数。此规则可防止LMS进程在响应数据，撤消或撤消段头块的CR请求时进入磁盘。此规则可以阻止LMS进程完成对CR请求的响应。

错误

数

LMS进程发出错误信号的次数

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

注意：

此视图包含供Oracle支持服务使用的内部诊断信息。它可随时更改，恕不另行通知。
