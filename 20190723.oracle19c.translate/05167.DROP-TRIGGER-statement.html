14.19 DROP TRIGGER声明

DROP TRIGGER语句从数据库中删除数据库触发器。

话题

  • 先决条件

  • 句法

  • 语义

  • 例

  • 相关话题

先决条件

触发器必须位于您的架构中，或者您必须具有DROP ANY TRIGGER系统特权。要在另一个用户的架构中删除DATABASE上的触发器，您还必须具有ADMINISTER DATABASE TRIGGER系统特权。

句法

drop_trigger :: =


说明drop_trigger.eps的说明

语义

模式

包含触发器的模式的名称。默认值：您的架构。

触发

要删除的触发器的名称。

例

例14-36删除触发器

此语句删除模式hr的salary_check触发器：

DROP TRIGGER hr.salary_check;
