打开影响所有平台的错误

本节介绍Oracle Database 19c中影响所有平台的已知错误。

  • Analytic Workspace Manager已知错误
    这些是Oracle Database 19c中的Analytic Workspace Manager错误。
  • Oracle ASM集群文件系统（Oracle ACFS）已知错误
    这些是Oracle Database 19c中的Oracle自动存储管理（Oracle ASM）群集文件系统（Oracle ACFS）错误。
  • Oracle ASM动态卷管理器（Oracle ADVM）已知错误
    这些是Oracle Database 19c中的Oracle自动存储管理（Oracle ASM）动态卷管理器（Oracle ADVM）错误。
  • Oracle数据泵已知错误
    这些是Oracle Database 19c中的Oracle Data Pump错误。
  • Oracle数据库企业版已知错误
    这些是Oracle Database 19c中的Oracle数据库企业版错误。
  • Oracle Grid Infrastructure已知错误
    这些是Oracle Database 19c中的Oracle Grid Infrastructure错误。
  • Oracle Universal Installer已知错误
    这些是Oracle Database 19c中的Oracle Universal Installer（OUI）错误。
