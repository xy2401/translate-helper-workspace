259.2.1 FORMAT_MESSAGE函数

此函数格式化由GET_MESSAGE函数检索的消息并返回格式化的消息。如果函数失败，则返回NULL结果。

下表显示了可在格式字符串中使用的特殊字符。

特殊字符 描述

'％s' 的

替换下一个字符串参数

'％d'

替换下一个整数参数

'%%'

表示特殊字符%

句法

UTL_LMS.FORMAT_MESSAGE（格式IN VARCHAR2 CHARACTER SET ANY_CS，args IN VARCHAR2 CHARACTER SET ANY_CS DEFAULT NULL）RETURN VARCHAR2 CHARACTER SET格式％CHARSET;

参数

表259-2 FORMAT_MESSAGE过程参数

参数 描述

格式

指定要格式化的字符串

ARGS

指定参数列表

例子

DECLARE s varchar2（200）;我pls_integer; BEGIN i：= utl_lms.get_message（26052，'rdbms'，'ora'，'french'，s）; dbms_output.put_line（'格式化之前，消息是：'|| s）; dbms_output.put_line（'格式化消息是：'|| utl_lms.format_message（s，9，'my_column_name'）; END; /

以下是未格式化的消息：

输入％d non pris en charge pour l'expression SQL sur la colonne％s。

以下是格式化的消息：

类型9非暴力充电倾注l表达式SQL sur la colonne my_column_name。

259.2.2 GET_MESSAGE函数

此函数检索Oracle错误消息。用户可以使用lmsgen实用程序定义特定于用户的错误消息。

成功时返回0 。它失败时返回-1 。

也可以看看：

有关lmsgen实用程序的详细信息，请lmsgen “Oracle数据库全球化支持指南”

句法

UTL_LMS.GET_MESSAGE（errnum IN PLS_INTEGER，产品IN VARCHAR2，设施IN VARCHAR2，语言IN VARCHAR2，消息OUT NOCOPY VARCHAR2CHARCTER SET ANY_CS）RETURN PLS_INTEGER;

参数

表259-3 GET_MESSAGE函数参数

参数 描述

差错编号

指定错误号。

示例： '972' （对于ORA-00972)

产品

指定错误消息适用的产品

示例：' rdbms '

设施

指定错误消息前缀

示例： 'ora'

语言

指定消息的语言。该参数不区分大小写。默认值为NULL ，这会导致GET_MESSAGE使用NLS_LANGUAGE会话参数的值。

信息

指定检索到的消息的输出缓冲区

使用说明

如果language参数设置为NULL ，则NLS_LANGUAGE会话参数的值将用作默认值。

例子

DECLARE s varchar2（200）;我pls_integer; BEGIN i：= utl_lms.get_message（601，'rdbms'，'oci'，'french'，s）; dbms_output.put_line（'OCI  -  00601是：'|| s）;结束 /

以下输出结果：

OCI  -  00601是：Echec du processus de nettoyage。
