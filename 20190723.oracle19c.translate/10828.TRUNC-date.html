TRUNC（约会）

句法

trunc_date :: =


截图说明trunc_date.eps

目的

TRUNC （日期）函数返回date ，截断date的时间部分为格式模型fmt指定的单位。此函数对NLS_CALENDAR会话参数不敏感。它根据公历的规则运作。即使您为date指定了不同的日期时间数据类型，返回的值也始终为数据类型DATE 。如果省略fmt ，则使用默认格式模型' DD '，返回的值是截断date的日期，时间为午夜。有关在fmt使用的允许格式模型，请参阅“ ROUND和TRUNC日期函数 ” 。

例子

以下示例截断日期：

SELECT TRUNC（TO_DATE（'27 -OCT-92'，'DD-MON-YY'），'YEAR'）“新年”来自DUAL;新年--------- 01-JAN-92

使用TRUNC格式化日期：示例

在以下示例中， TRUNC函数返回输入日期，其中截断日期的时间部分按格式模型中的指定：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual union SELECT timestamp'2015-03- 03 23:45:00'd FROM dual union SELECT timestamp'2015-04-11 12:34:56'd FROM dual）SELECT d“Original Date”，trunc（d）“Nearest Day，Time Removed”，trunc（ d，'ww'）“最近一周”，截断（d，'iw'）“周开始”，截断（d，'mm'）“月初”，截断（d，'年'）“开始年“从日期;

在以下示例中，输入日期值被截断， TO_CHAR函数用于获取截断日期值的分钟组件：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual union SELECT timestamp'2015-03- 03 23:45:00'from FROM union SELECT timestamp'2015-04-11 12:34:56'd FROM dual）SELECT d“Original Date”，trunc（d）“Date with Time of Removed”，to_char（trunc （d，'mi'），'dd-mon-yyyy hh24：mi'）“最近的分钟”，截断（d，'iw'）“周开始”，截断（d，'mm'）“月初“，截断（d，'年'）”年初“从日期开始;

以下语句更改当前会话的日期格式：

ALTER SESSION SET nls_date_format ='dd-mon-yyyy hh24：mi';

在以下示例中，数据以新的日期格式显示：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual union SELECT timestamp'2015-03- 03 23:44:32来自双联盟SELECT timestamp'2015-04-11 12:34:56'd FROM dual）SELECT d“Original Date”，trunc（d）“Date，time removed”，to_char（trunc （d，'mi'），'dd-mon-yyyy hh24：mi'）“最近的分钟”，截断（d，'iw'）“周开始”，截断（d，'mm'）“月初“，截断（d，'年'）”年初“从日期开始;
