14.1更改功能声明

ALTER FUNCTION语句显式重新编译独立函数。

显式重新编译消除了对隐式运行时重新编译的需要，并防止了相关的运行时编译错误和性能开销。

注意：

此声明不会更改现有函数的声明或定义。要重新声明或重新定义独立函数，请使用带有OR REPLACE子句的“ CREATE FUNCTION语句 ” 。

例

例14-1重新编译函数

要显式重新编译示例用户oe拥有的函数get_bal ，请发出以下语句：

ALTER FUNCTION oe.get_bal COMPILE;

如果数据库在重新编译get_bal时没有遇到编译错误，则get_bal变为有效。数据库随后可以运行它，而无需在运行时重新编译它。如果重新编译get_bal导致编译错误，那么数据库将返回错误，并且get_bal仍然无效。

该数据库还使依赖于get_bal所有对象无效。如果您随后引用其中一个对象而未先显式重新编译它，则数据库会在运行时隐式重新编译它。
