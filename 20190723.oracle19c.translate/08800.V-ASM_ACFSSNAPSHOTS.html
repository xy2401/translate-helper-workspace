7.50 V $ ASM_ACFSSNAPSHOTS

V$ASM_ACFSSNAPSHOTS显示每个已安装的Oracle ACFS的快照信息。

柱 数据类型 描述

FS_NAME

VARCHAR2（1024）

文件系统挂载点（ V$ASM_FILESYSTEM外键）

VOL_DEVICE

VARCHAR2（256）

Oracle ADVM设备的名称

SNAP_NAME

VARCHAR2（1024）

快照的名称

CREATE_TIME

日期

创建快照的时间

类型

VARCHAR2（2）

快照类型。可能的类型是只读（RO）或读/写（RW）。

家长

VARCHAR2（1024）

用于创建快照的父名称。如果使用装入点作为基础创建快照，则父名称将为NULL。如果使用现有快照作为基础创建快照，则父名称将是传递给acfsutil snap create -p操作的快照的名称。

LINK Foot 1

VARCHAR2（1024）

如果已设置链接，则链接的路径。如果未设置，则为NULL。

ADDITIONAL_STORAGE Footref 1

数

相当于“存储添加到快照”输出

QUOTA Footref 1

数

配额价值。如果未设置，则为NULL。

REPL Footref 1

VARCHAR2（5）

指示快照是否用于复制（ TRUE ）（ FALSE ）

STATE Footref 1

VARCHAR2（14）
快照的状态：

  • AVAILABLE - 默认

  • DELETE_WAITING - 映射到“删除等待最后关闭”

  • DELETE_IN_PROGRESS - 映射到“正在删除”

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。

脚注1此列从Oracle Database 19c版本19.1开始提供。
