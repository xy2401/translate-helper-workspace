140 PROTL-00001到PROTL-00604

PROTL-00001：无法初始化ocrconfig

原因：无法初始化并启动ocrconfig。

操作：检查日志文件以获取有关失败的详细信息。

PROTL-00002：导出文件' string '已存在

原因：请求的导出文件存在于给定的路径中。

操作：更改导出文件的位置或删除旧的导出文件。

PROTL-00003：无法创建导出文件' string '

原因：无法创建或打开用于导出本地注册表数据的文件。

操作：检查所请求路径的访问权限或磁盘空间。

PROTL-00004：无法从本地注册表检索数据

原因：无法从本地注册表中检索必要的信息。

操作：从日志文件中检查本地注册表的可用性以及失败的详细信息。

PROTL-00006：无法访问指定的Oracle Local Registry导入文件“ string ”。

原因：尝试从指示的导入文件导入Oracle Local Registry内容失败，因为导入文件不存在或用户没有打开导入文件的权限。

操作：确保导入文件存在且其权限正确，然后重试导入命令。

PROTL-00007：无法打开指定的导入文件' string '

原因：无法访问指定的导入文件。

操作：检查导入文件的访问权限。

PROTL-00008：无法将指定文件中的数据导入本地注册表

原因：无法从导入文件中读取数据并导入到本地注册表。

操作：从日志文件中检查本地注册表的可用性以及失败的详细信息。

PROTL-00010：无法打开指定的源本地注册表

原因：无法访问指定的源本地注册表。

操作：检查源本地注册表的访问权限和位置，以及日志文件以获取更多详细信息。

PROTL-00011：无法打开指定的目标本地注册表

原因：无法访问指定的目标本地注册表。

操作：检查目标本地注册表的访问权限和位置。还要检查日志文件以获取更多详细信息。

PROTL-00012：要升级的Oracle Local Registry具有不受支持的版本

原因：要升级的Oracle Local Registry具有不受支持的版本。

操作：将 Oracle Local Registry升级到支持的版本。

PROTL-00013：Oracle Local Registry是当前版本

原因： Oracle Local Registry处于当前版本。

行动：无需采取任何行动。

PROTL-00015：未实现的功能

原因：当前版本中尚未实现请求的功能。

操作：在下一个版本中再次尝试该功能。

PROTL-00016：内部错误

原因：这是ocrconfig的一般内部错误。

操作：联系Oracle支持服务。

PROTL-00017：致命错误

原因：这是ocrconfig的一般致命错误。

操作：联系Oracle支持服务。

PROTL-00018：无法打开指定的备份文件' string '

原因：文件不存在或用户没有打开文件的权限。

操作：确保文件存在且其权限正确。还要检查日志文件以获取更多详细信息。

PROTL-00019：Oracle高可用性服务运行时无法继续

原因： Oracle高可用性服务正在运行。

操作：使用'crsctl stop crs'命令关闭本地节点上的Oracle High Availability Service。

PROTL-00020：没有足够的权限继续进行。需要特权用户

原因：用户不是特权用户。

操作：使用特权用户再次运行该命令。

PROTL-00021：参数无效

原因：输入参数无效。

操作：检查日志文件以获取更多信息。

PROTL-00023：无法备份Oracle Local Registry

原因：尝试备份Oracle本地注册表失败。

操作：确认注册表存在（'ocrcheck -local'）并且集群件堆栈正在运行（'crsctl check crs'）。确保备份位置有足够的可用空间用于ocrcheck报告的OLR大小。如有必要，请使用'ocrconfig -local -backuploc'指定其他位置。

PROTL-00024：目前没有OLR的自动备份。

原因：命令'ocrconfig -local -showbackup'确定当时没有可用的Oracle本地注册表（OLR）的自动备份。

操作：每天创建一次OLR的自动备份。等待24小时，然后重试'showbackup'操作。或者，使用'ocrconfig -local -manualbackup'命令进行手动备份。

PROTL-00025：Oracle本地注册表的手动备份不可用

原因：尚未创建Oracle Local Registry的手动备份。

操作：可以使用'ocrconfig -local -manualbackup'命令创建手动备份。

PROTL-00035：无法访问已配置的OLR位置

原因：配置的Oracle本地注册表（OLR）位置不存在，没有所需的权限，或者没有足够的磁盘空间。

操作：确保该位置存在，其权限正确，并且具有足够的磁盘空间。另请检查Clusterware警报日志文件以获取更多详细信息。

PROTL-00039：指定的文件' string '格式无效

原因：指定的文件不是有效的Oracle本地注册表导出/导入文件。

操作：指定使用'ocrconfig -local -export'命令创建的有效Oracle本地注册表文件。

PROTL-00040：指定的文件' string '格式无效

原因：指定的文件不是有效的Oracle本地注册表备份文件。

操作：使用'ocrconfig -local -showbackup'命令列出所有可用备份。指定上述命令列出的有效Oracle Local Registry备份文件。

PROTL-00041：无法访问指定的备份位置“ string ”

原因：备份位置不存在或未授予所需权限。

操作：确保该位置存在并授予所需的文件创建权限。

PROTL-00042：指定的备份位置“ string ”不是目录

原因：指定的备份位置不是目录。

操作：指定目录。

PROTL-00049：Oracle本地注册表（OLR）完整性检查失败。

原因：已配置的Oracle Local Registry位置的完整性检查失败。

操作：使用“ocrconfig -local -showbackup”命令检查备份的可用性并还原兼容的备份。如果无法解决问题，请与Oracle支持服务联系。

PROTL-00052：无法检索Oracle Local Registry的备份位置

原因：尝试显示Oracle本地注册表的备份位置失败，因为无法访问Oracle本地注册表。

操作：检查Oracle Local Registry的可访问性。

PROTL-00054：无法将Oracle本地注册表备份文件复制到指定的目标“ string ”

原因：尝试将Oracle本地注册表备份文件复制到指定位置失败，因为它无法由用户写入或没有足够的空间。

操作：确保目标可由用户写入，并且文件有足够的可用空间。

PROTL-00056：为'ocrconfig -local -copy'命令指定的文件名[ string ]无效。

原因：尝试复制Oracle本地注册表文件失败，因为指定的目标文件名参数未指定完整的路径名。

操作：重新发出命令，指定目标文件的完整路径名。

PROTL-00057：'ocrconfig -local -manualbackup -filename'命令中指定的备份文件已存在[ string ]。

原因：请求的备份文件已存在于给定路径中。

操作：更改“ocrconfig -local -manualbackup -filename”命令的备份文件位置，或删除旧的备份文件。

PROTL-00058：成功将Oracle Local Registry内容导出到文件' string '

原因： Oracle Local Registry内容已成功导出到用户请求的指定文件。

行动：没有

PROTL-00059：无法显示Oracle Local Registry的手动备份。

原因：尝试显示OLR手动备份位置失败，因为正在进行手动备份。

操作：在进行中的备份完成后，使用“ocrconfig -local -showbackup manual”命令重试该操作以显示手动备份位置。

PROTL-00061：创建的Oracle本地注册表备份文件是“ string ”。

原因：无，这是一条信息性消息。

行动：没有

PROTL-00062：无法删除Oracle本地注册表备份文件“ string ”。

原因：可能无法从发出命令的节点访问Oracle本地注册表备份，或者用户可能没有相应的权限来删除备份文件。

操作：检查随附的错误并采取相应措施。

PROTL-00302：无法初始化ocrdump

原因：无法初始化并启动ocrdump。

操作：检查日志文件以获取有关失败的详细信息。

PROTL-00303：转储文件已存在[ string ]

原因：请求的转储文件存在于给定的路径中。

操作：更改转储文件的位置或删除旧的转储文件。

PROTL-00304：无法创建转储文件[ string ]

原因：无法创建或打开文件以转储本地注册表数据。

操作：检查所请求路径的访问权限或磁盘空间。

PROTL-00305：无法写入转储文件[ string ]

原因：转储本地注册表数据时无法写入转储文件。

操作：检查所请求路径的访问权限或磁盘空间。

PROTL-00306：无法检索本地注册表数据

原因：无法读取本地注册表数据。

操作：检查日志文件以获取有关失败的详细信息。

PROTL-00307：本地注册表中不存在指定的密钥

原因：无法从本地注册表中检索指定的密钥。

操作：检查日志文件以获取密钥名称。

PROTL-00309：无法访问Oracle Local Registry备份文件。

原因： Oracle Local Registry备份文件不存在或用户没有打开文件的权限。

操作：确保文件存在且其权限正确。

PROTL-00310：由于权限，并非所有密钥都被转储。

原因：尝试转储用户没有访问权限的一个或多个Oracle Local Registry密钥。

操作：以其他用户身份登录并重试。

PROTL-00312：指定的文件“ string ”格式无效。

原因： ocrdump命令的file参数指定的文件不是有效的Oracle Local Registry备份文件。

操作：使用'ocrconfig -local -showbackup'命令列出所有可用备份。重新发出ocrdump命令，指定ocrconfig命令列出的有效Oracle本地注册表备份文件。

PROTL-00601：初始化ocrcheck失败

原因：无法初始化并启动ocrcheck。

操作：检查日志文件以获取有关失败的详细信息。

PROTL-00602：无法从本地注册表中检索数据

原因：无法从本地注册表中检索必要的信息。

操作：从日志文件中检查本地注册表的可用性以及失败的详细信息。

PROTL-00603：Oracle本地注册表在逻辑上已损坏

原因：在Oracle本地注册表中检测到逻辑损坏。

操作：检查日志文件以获取更多信息。

PROTL-00604：无法检索本地注册表的已配置位置

原因：无法检索本地注册表的位置。

操作：检查日志文件以获取有关失败的详细信息。
