x86-64 Linux平台的操作系统要求

x86-64上的此版本支持本节中列出的Linux发行版和软件包。

确定Linux发行版的要求，并确保在开始安装之前安装了受支持的内核和所需的软件包。

注意：

  • Oracle数据库安装需要X Window系统（例如， libx ）。libx包是默认Linux安装的一部分。如果在具有减少的软件包集的系统上执行安装，则必须确保安装了libx或类似的X Window System软件包。

  • 适用于Oracle Linux的Unbreakable Enterprise Kernel可以安装在运行Oracle Linux或Red Hat Enterprise Linux的x86-64服务器上。从Oracle Linux 5 Update 6开始，Unbreakable Enterprise Kernel是默认的系统内核。Oracle Linux 5更新7及更高版本提供了包含Unbreakable Enterprise Kernel的Oracle Linux x86（32位）版本。

  • 仅当您打算使用32位客户端应用程序访问64位服务器时，才需要这些要求列表中的32位软件包。

  • 对于Oracle数据库或Oracle Grid Infrastructure安装，Oracle Database 12c第2版（12.2）及更高版本不需要编译器包gcc和gcc-c++ 。

本指南中包含的特定于平台的硬件和软件要求是本指南发布时的最新要求。但是，由于在本指南发布后可能会对新平台和操作系统软件版本进行认证，请查看My Oracle Support网站上的认证表，以获取最新的认证硬件平台和操作系统版本列表：
