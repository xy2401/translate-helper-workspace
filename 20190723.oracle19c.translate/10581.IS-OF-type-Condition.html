是条件类型

使用IS OF type条件根据对象实例的特定类型信息测试对象实例。

is_of_type_condition :: =


插图描述is_of_type_condition.eps

你必须有EXECUTE对引用的所有类型的权限type ，所有type小号必须属于同一类型的家庭。

如果expr为null，则此条件的计算结果为null。如果expr不为null，则在以下任一情况下，条件的计算结果为true（如果指定NOT关键字，则为false）：

  • 最具体的expr类型是type列表中指定的type之一的子类型，并且您尚未ONLY类型指定，或者

  • 最具体的expr type在type列表中明确指定。

expr经常采用VALUE函数的形式和相关变量。

以下示例使用示例表oe.persons ，该表构建在Substitutable Table和Column Examples中的类型层次结构上。该示例使用IS OF type条件将查询限制为特定的子类型：

SELECT * FROM people p WHERE VALUE（p）IS OF TYPE（employee_t）; NAME SSN ---------------------------- Joe 32456 Tim 5678 SELECT * FROM people p WHERE VALUE（p）IS OF（仅限part_time_emp_t） ; NAME SSN ---------------------------- Tim 5678
