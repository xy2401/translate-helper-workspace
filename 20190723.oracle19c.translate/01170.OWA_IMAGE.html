216.1 OWA_IMAGE概述

如果您有任何目标链接调用PL / SQL网关的图像映射，请使用此程序包。

216.2 OWA_IMAGE类型

此数据类型（ point ）包含坐标的X和Y值，因此提供用户在imagemap上单击的坐标。

它被定义为：

TYPE POINT是VARCHAR2的表（32767）INDEX BY BINARY_INTEGER

216.3 OWA_IMAGE变量

TYPE POINT此包变量（ null_point ）用于默认点参数。此变量的X和Y字段均为NULL 。

216.4 OWA_IMAGE示例

此示例显示了OWA_IMAGE使用。

CREATE或REPLACE PROCEDURE process_image（OWA_IMAGE.POINT中的my_img）AS x integer：= OWA_IMAGE.GET_X（my_img）; y integer：= OWA_IMAGE.GET_Y（my_img）; BEGIN / *处理坐标* / END

216.5.1 GET_X功能

此函数返回用户单击图像映射的点的X坐标。

句法

OWA_IMAGE.GET_X（p IN点）RETURN INTEGER;

参数

表216-2 GET_X功能参数

用户点击的点。

返回值
