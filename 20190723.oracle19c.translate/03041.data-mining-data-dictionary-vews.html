以下示例描述了ALL_MINING_MODEL_SETTINGS并显示了示例查询。设置会影响模型行为。设置可以特定于算法或挖掘功能，或者它们可以是通用的。

例28-4 ALL_MINING_MODEL_SETTINGS

描述ALL_MINING_MODEL_SETTINGS名称是否为空？输入----------------------------------------- -------- ---------------------------- OWNER NOT NULL VARCHAR2（128）MODEL_NAME NOT NULL VARCHAR2（128）SETTING_NAME NOT NULL VARCHAR2（30）SETTING_VALUE VARCHAR2（4000）SETTING_TYPE VARCHAR2（7）
