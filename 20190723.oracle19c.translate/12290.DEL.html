12.18 DEL

句法

DEL [ n | n m | n * | n最后| * | * n | *最后|持续]

删除缓冲区的一行或多行。

条款

术语 描述 
ñ

删除行 n 。 

为N×M

删除行 从n到m 。 

n *

删除当前行的行n 。 

ñLAST

从最后一行删除第n行。 

*

删除该当前线。 

* n

通过第n行删除当前行。 

*最后

删除最后一行的当前行。

 
持续

删除最后一个线。

输入DEL，不带子句删除缓冲区的当前行。

用法

DEL使缓冲区的下一行（如果有）成为当前行。您可以多次输入DEL以删除多个连续行。

注意：

DEL是SQL * Plus命令，DELETE是SQL命令。有关SQL DELETE命令的详细信息，请参阅DELETE 。

例子

假设SQL缓冲区包含以下查询：

SELECT LAST_NAME，DEPARTMENT_ID from EMP_DETAILS_VIEW WHERE JOB_ID ='SA_MAN'ORDER BY DEPARTMENT_ID;

要使包含WHERE子句的行成为当前行，您可以输入

列表3
3 * WHERE JOB_ID ='SA_MAN'

其次是

DEL

SQL缓冲区现在包含以下行：

选择LAST_NAME，DEPARTMENT_ID来自EMP_DETAILS_VIEW按DEPARTMENT_ID订购

要删除缓冲区的第三行，请输入

DEL 3

SQL缓冲区现在包含以下行：

从EMP_DETAILS_VIEW中选择LAST_NAME，DEPARTMENT_ID
