121 PRGP-01001至PRGP-01029

PRGP-01001：没有集群被移动到目标路径{0}。

原因：尝试恢复或中止将群集移动到指定的目标路径被拒绝，因为尚未为指定的目标路径启动移动操作。

行动：没有

PRGP-01002：没有数据库被移动到目标路径{0}。

原因：尝试从指定的目标路径恢复或中止移动数据库被拒绝，因为尚未为指示的目标路径指定移动操作。

行动：没有

PRGP-01003：没有从Oracle主目录“{0}”移动数据库。

原因：尝试修补Oracle数据库被拒绝，因为'-excludedblist'选项导致没有数据库被识别为需要从指定的Oracle主目录移动的数据库。

操作：重试该操作，确保选项'-excludedblist'的输入不会排除指定Oracle主目录中的所有数据库的修补。

PRGP-01004：缺少将数据库从集群“{1}”上的Oracle Home“{0}”移出的信息

原因：尝试修补Oracle数据库被拒绝，因为指定的Oracle主目录涉及并行“rhpctl移动数据库”操作，并且用户未指定足够的信息来唯一标识特定操作。

操作：重试该操作，确保选项'-dbname'或选项'-excludedblist'的输入唯一标识特定操作。

PRGP-01005：无法指定数据库“{0}”以及数据库“{1}”

原因：尝试修补Oracle数据库被拒绝，因为选项'-dbname'的输入不正确。

操作：重试该操作，确保选项'-dbname'的输入不包含违规值。

PRGP-01006：选项'-excludedblist'的值匹配正在进行的多个'rhpctl移动数据库'操作：“{0}”

原因：尝试修补Oracle数据库被拒绝，因为选项'-excludedblist'的输入不正确。

操作：重试该操作，确保选项'-excludedblist'的输入在正在进行的指示操作中标识特定的'rhpctl移动数据库'操作。

PRGP-01013：指定的源主目录“{0}”不是活动的Cluster Ready Services（CRS）主目录。

原因：尝试移动或升级Oracle Grid Infrastructure home被拒绝，因为指定的源home不是活动的Cluster Ready Services（CRS）主目录。

操作：重试操作，指定活动的，已配置的Oracle Grid Infrastructure主目录作为源。

PRGP-01014：无法将Oracle Grid Infrastructure置于主页，因为源和目标主页不能相同

原因：尝试移动Oracle Grid Infrastructure主页被拒绝，因为指定的源和目标主页是相同的。

操作：指定不同的源和目标主页并重试该操作。

PRGP-01015：此RHP服务器群集上未配置GNS服务器和GNS客户端。

原因： 'rhpctl add workingcopy'命令被拒绝，因为在Rapid Home Provisioning（RHP）服务器群集上既未配置网格命名服务（GNS）服务器也未配置GNS客户端。

操作：使用RHP服务器群集中的“srvctl add gns”命令配置GNS服务器或GNS客户端，然后重试该命令。

PRGP-01016：仅对STANDALONE群集支持零影响修补，而检测到的群集为{0}。

原因：尝试还原或移动Oracle Grid Infrastructure主页被拒绝，因为指定的集群类不支持零影响修补。

操作：重试还原或移动操作，而不指定'-tgip'选项。

PRGP-01017：不支持为零影响修补指定多个节点。

原因：尝试还原或移动Oracle Grid Infrastructure主页被拒绝，因为一次只支持一个节点的零影响修补，并为'-batch'选项指定了多个节点。

操作：重试命令，仅为“-batch”选项指定一个节点，或者不指定“-tgip”选项。

PRGP-01018：Oracle Restart不支持零影响修补。

原因：尝试恢复或移动Oracle Grid Infrastructure主页被拒绝，因为Oracle Restart不支持零影响修补。

操作：重试还原或移动操作，而不指定'-tgip'选项。

PRGP-01019：仅在Oracle Linux平台上支持零影响修补，而检测到的平台为{0}。

原因：尝试恢复或移动Oracle Grid Infrastructure主页被拒绝，因为指定平台不支持零影响修补。

操作：重试还原或移动操作，而不指定'-tgip'选项。

PRGP-01025：钱包文件中未找到任何客户。

原因：尝试从钱包文件导入客户端凭据被拒绝，因为钱包文件没有客户端凭据。

操作：使用包含客户端凭据的有效钱包文件重试导入操作。

PRGP-01026：指定的钱包文件“{0}”不存在。

原因：尝试从钱包文件读取失败，因为钱包文件不存在。

操作：使用包含客户端凭据的有效钱包文件重试导入操作。

PRGP-01027：指定的钱包文件“{0}”已存在。

原因：尝试写钱包文件失败，因为钱包文件已存在。

操作：使用其他钱包文件名重试导出操作。

PRGP-01028：在阅读钱包文件时找到无效的客户名称“{0}”

原因：尝试从钱包文件中读取客户端失败，因为客户端名称无效。

操作：使用其他钱包文件重试该操作。

PRGP-01029：删除工作副本部分失败。

原因：尝试删除工作副本部分失败，因为无法连接到Rapid Home Provisioning Client（RHPC）。仅从Rapid Home Provisioning Server（RHPS）中删除了工作副本元数据。

操作：使用操作系统命令从Rapid Home Provisioning Client（RHPC）手动删除工作副本存储数据。
