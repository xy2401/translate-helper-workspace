2.79 ALL_CODE_ROLE_PRIVS

ALL_CODE_ROLE_PRIVS描述与当前用户拥有或可访问的程序单元关联的所有角色。

相关观点

  • DBA_CODE_ROLE_PRIVS描述与数据库中的程序单元关联的所有角色。

  • USER_CODE_ROLE_PRIVS描述与当前用户拥有的程序单元关联的所有角色。此视图不显示OWNER列。
