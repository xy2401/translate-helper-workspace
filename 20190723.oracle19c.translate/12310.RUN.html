12.38跑

句法

跑]

列出并执行SQL命令或者当前存储在PL / SQL块中 SQL缓冲区。

用法

RUN导致SQL缓冲区的最后一行成为当前行。

斜杠命令（/） 功能与RUN类似 ，但不会在屏幕上的SQL缓冲区中列出该命令。SQL缓冲区始终包含输入的最后一个SQL语句或PL / SQL块。

例子

假设SQL缓冲区包含以下脚本：

从EMP_DETAILS_VIEW WHERE SALARY> 12000中选择DEPARTMENT_ID

要运行脚本，请输入

跑
1 SELECT DEPARTMENT_ID 2 FROM EMP_DETAILS_VIEW 3 WHERE SALARY> 12000 DEPARTMENT_ID ------------- 90 90 90 80 80 20 6行选择。
