确认主机名解析

检查以确保服务器的主机名是可解析的。

通常，要安装Oracle数据库的计算机已连接到网络。确保计算机主机名可以通过域名系统（DNS），网络信息服务（NIS）或集中维护的TCP / IP主机文件（例如/etc/hosts ）进行解析。使用ping命令确保您的计算机主机名可解析。例如：
ping myhostname使用32字节数据ping myhostname.example.com [192.0.2.2]：从192.0.2.2回复：bytes = 32 time = 138ms TTL = 56
