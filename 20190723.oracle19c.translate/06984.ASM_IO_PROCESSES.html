1.19 ASM_IO_PROCESSES

ASM_IO_PROCESSES指定要在Oracle IOServer实例中启动的I / O工作进程数。

属性 描述

参数类型

整数

默认值

20

可修改

改变系统

可在PDB中修改

没有

价值范围

1 - 32

基本

没有

Oracle RAC

多个实例可以具有不同的值。

此参数仅适用于Oracle IOServer实例，该实例在Oracle Grid Infrastructure主目录之外运行。

在大多数情况下，默认值应该有效。但是，在I / O负载较重的情况下，在I / O等待之外发布进程可能会有一些延迟。在这种情况下，略高于默认值的值可能是合适的。
