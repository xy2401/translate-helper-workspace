8.143 V $ PX_PROCESS_SYSSTAT

V$PX_PROCESS_SYSSTAT包含有关运行并行执行的会话的信息。

柱 数据类型 描述

统计

VARCHAR2（30）

统计名称：

  • Servers In Use的服务器 - 当前正在执行并行操作的PX服务器数

  • Servers Available - 可用于执行并行操作的PX服务器数

  • Servers Started - 系统必须创建PX服务器进程的次数

  • Servers Shutdown - PX服务器进程已关闭的次数。如果最近没有使用PX服务器进程，它将被关闭。

    如果此值很大，则考虑增加参数。这将通过避免PX服务器进程创建的延迟来提高性能。

  • Servers HWM - 最大并发PX服务器进程数

    如果此数字等于PARALLEL_MAX_SERVERS初始化参数，则考虑增加参数。这可以允许您增加吞吐量，特别是如果您的系统未充分利用且V$SYSSTAT统计“并行操作降级为串行”很大。

  • Servers Cleaned Up - PMON必须清理PX服务器的次数。这应该仅在并行操作的异常终止期间发生。

    如果这个数字很大，那么你应该确定原因。

  • 服务器会话数 - 所有PX服务器创建的会话总数

  • Memory Chunks Allocated - PX服务器分配的大内存块数

  • Memory Chunks Freed - 释放的大内存块数

  • Memory Chunks Current - 当前正在使用的大内存块数

  • Memory Chunks HWM - 同时分配的最大块数

  • Buffers allocated -的次数的消息已分配缓冲区

  • Buffers freed - Buffers freed消息缓冲区的次数

  • Buffers Current - 当前正在使用的消息缓冲区数

  • Buffers HWM - 同时分配的最大消息缓冲区数

值

数

统计数据的价值

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
