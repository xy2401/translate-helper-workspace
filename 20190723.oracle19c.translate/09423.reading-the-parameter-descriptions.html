1.5阅读参数说明

本节介绍初始化参数说明中记录的属性。

PARAMETER_NAME

属性 描述

参数类型

指定参数的类型：

  • Boolean参数接受true或false作为其值。

  • 字符串参数接受任何字符序列作为其值，具体取决于参数的语法。

  • 整数参数接受4字节值，范围从0到2 32 - 1。

  • 参数文件参数接受初始化参数文件规范作为其值。

  • 大整数参数接受8字节值，范围从0到2 64 - 1。您可以将大整数的值指定为整数以及可选的修饰符（如K，M或G），它们分别表示千字节，兆字节或千兆字节。

    例如，1000,100 KB，50 MB和2 GB是大整数的有效规范。

句法

对于字符串和大整数参数，指定用于指定参数的有效语法。

默认值

如果未显式指定值，则指定参数所采用的值。

对于大多数初始化参数，通过在V$PARAMETER视图中查询参数的DEFAULT_VALUE列，可以获得为Default value属性记录的值 。

可修改

指定是否可以为当前会话（通过ALTER SESSION语句）或当前实例中的所有会话（通过ALTER SYSTEM语句） ALTER SYSTEM ：

  • ALTER SESSION覆盖当前会话的实例范围的参数设置。您只能通过发出另一个ALTER SESSION语句来恢复该会话的实例范围设置。

  • ALTER SYSTEM可用于更改任何初始化参数的服务器参数文件（SPFILE）中的值。这种更改仅在后续实例中生效。参数描述仅指示可以为当前实例修改的那些参数。

可在PDB中修改

指定是否可以在PDB中修改参数（是）或不是（否）

对于大多数初始化参数，通过在V$PARAMETER视图中查询参数的ISPDB_MODIFIABLE列，可以获得PDB属性中Modifiable记录的值。

价值范围

指定此参数可以采用的有效值范围，显示为最小值和最大值。不适用于所有参数。

基本

指定参数是否是基本初始化参数

Oracle RAC

指定必须如何为Oracle Real Application Clusters环境中的多个实例指定此参数的值。不适用于所有参数。
