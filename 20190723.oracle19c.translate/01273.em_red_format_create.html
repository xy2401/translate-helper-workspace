em_red_format_create.png插图的描述

图像em_red_format_create.png显示“创建”对话框，其中包含以下组件，从上到下，从左到右：

  • 从上到下，以下内容：格式名称字段，描述字段，敏感列类型菜单，编辑功能菜单，描述当前所选编校功能类型的标签

  • 功能属性区域，根据REGEX编辑功能的选择，从上到下具有以下内容：

    • Pattern（指定要搜索的正则表达式模式）。示例：'\ d \ d \ d \ d \ d \ d678'代表'012345678'这样的数字

    • 替换字符串（例如：使用'XXXXXX \ 3'（替换字符串）来编辑'012345678'（实际值）匹配'（\ d \ d \ d）（\ d \ d \ d）（\ d \ d \ d） ）'（正则表达式）到'XXXXXX678'（编辑值）。请注意，替换字符串中的'\ 3'保留模式中第三组括号中的实际数据。）

    • 位置（指定字符串搜索的起始位置。默认值为1，表示从列数据的第一个字符开始搜索。）

    • 出现（指定如何执行搜索和替换操作。零意味着它取代所有出现的事件。正整数'n'将替换第n个字符串。）

    • 匹配参数菜单（指定REGEX编校功能的匹配参数。）

  • 在右下角，从左到右，按钮确定并取消
