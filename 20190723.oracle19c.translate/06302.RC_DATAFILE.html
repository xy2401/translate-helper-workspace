5.30 RC_DATAFILE

此视图列出有关在恢复目录中注册的所有数据文件的信息。它对应于V$DATAFILE视图。

如果删除了表空间，则数据文件显示为已删除。

柱 数据类型 描述

DB_KEY

数

恢复目录中此数据库的主键。使用此列可以与几乎任何其他目录视图连接。

DBINC_KEY

数

目标数据库化身的主键。使用此列与RC_DATABASE_INCARNATION 。

DB_NAME

VARCHAR2（8）

此记录所属的数据库化身的DB_NAME 。

CON_ID

数

当前容器的ID：

  • 0：包含与整个多租户容器数据库（CDB）或传统Oracle数据库（非CDB）相关的数据的行

  • 1：包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

PDB_NAME

VARCHAR2（31）

恢复目录中可插拔数据库（PDB）的名称。

PDB_KEY

数

恢复目录中此PDB的主键。

PDBINC_KEY

数

PDB化身的主要关键。

TS#

数

数据文件所属的表空间的编号。如果删除并重新创建表空间，则TS#可能在同一个版本中存在多次。

TABLESPACE_NAME

VARCHAR2（30）

表空间名称。如果删除并重新创建表空间，则该名称可能在同一个版本中存在多次。

文件＃

数

数据文件的绝对文件号。如果删除并重新创建数据文件，则相同的数据文件编号可以在同一个版本中多次存在。

CREATION_CHANGE＃

数

创建数据文件时的SCN。

CREATION_TIME

日期

数据文件创建的时间。

DROP_CHANGE＃

数

删除数据文件时记录的SCN。如果发现具有相同文件号的新数据文件，则将DROP_CHANGE#设置为数据文件的CREATION_CHANGE# ;否则该值设置为RC_CHECKPOINT.CKP_SCN 。

DROP_TIME

日期

删除数据文件的时间。如果发现具有相同文件号的新数据文件，则将DROP_TIME设置为数据文件的CREATION_TIME ;否则该值设置为RC_CHECKPOINT.CKP_TIME 。

BYTES

数

数据文件的大小（以字节为单位）。

积木

数

块中数据文件的大小。

BLOCK_SIZE

数

数据块的大小（以字节为单位）。

名称

VARCHAR2（1024）

数据文件名。

STOP_CHANGE＃

数

对于脱机或只读数据文件，SCN值使得在相同或更大SCN的重做流中的更改不适用于此文件。

STOP_TIME

日期

对于脱机正常或只读数据文件，应用于此数据文件的重做流中没有更改的时间。

只读

数

如果文件是只读的，则为1 ;否则为0 。

的RFile＃

数

该表空间中此数据文件的相对文件号。

INCLUDED_IN_DATABASE_BACKUP

VARCHAR2（3）

指示此表空间是否包含在整个数据库备份中： YES或NO 。仅当在拥有此数据文件的表空间上运行CONFIGURE EXCLUDE时才会出现NO值。

AUX_NAME

VARCHAR2（1024）

表示CONFIGURE AUXNAME设置的数据文件的辅助名称。

ENCRYPT_IN_BACKUP

VARCHAR2（3）

如果此数据文件已配置为在备份时透明加密，则为YES ;否则为NULL 。

SITE_KEY

数

与此文件关联的Data Guard数据库的主键。Data Guard环境中的每个数据库都具有唯一的SITE_KEY值。您可以使用SITE_KEY与联接RC_SITE以获得DB_UNIQUE_NAME的数据库。

DB_UNIQUE_NAME

VARCHAR2（512）

此记录所属的数据库化身的DB_UNIQUE_NAME 。Data Guard环境中的所有数据库共享相同的DBID，但DB_UNIQUE_NAME值不同。当特定文件的数据库名称不可知时，此列中的值为null。例如，Oracle Database 11g之前由RMAN版本管理的数据库的行为空。

FOREIGN_DBID

数

此数据文件来自的外部DBID。如果此文件不是外部数据库文件，则该值为0 。

FOREIGN_CREATION_CHANGE＃

数

创建外部数据文件的SCN。如果此文件不是外部数据库文件，则该值为0 。

FOREIGN_CREATION_TIME

日期

外国数据文件的创建时间。如果此文件不是外部数据库文件，则该值为0 。

PLUGGED_READONLY

VARCHAR2（3）

如果这是一个传输的只读外国文件，则为YES ;否则NO 。

PLUGIN_CHANGE＃

数

将外部数据文件传输到数据库的SCN。如果此文件不是外部数据库文件，则该值为0 。

PLUGIN_RESETLOGS_CHANGE＃

数

RESETLOGS操作的SCN用于传输此外来文件的化身。如果此文件不是外部数据库文件，则该值为0 。

PLUGIN_RESETLOGS_TIME

日期

RESETLOGS操作的时间，用于传输此外来文件的化身。如果此文件不是外部数据库文件，则该值为0 。

CREATION_THREAD

数

此行描述的数据文件的线程号。

CREATION_SIZE

数

创建时数据文件的大小。
