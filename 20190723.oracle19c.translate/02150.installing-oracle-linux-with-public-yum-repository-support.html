使用Oracle Linux Yum Server支持安装Oracle Linux

使用此过程安装Oracle Linux并使用Oracle Linux yum服务器配置Linux安装以获取安全勘误或错误修复更新。

  1. 从Oracle Store获取Oracle Linux DVD，或从Oracle Software Delivery Cloud下载Oracle Linux：

    Oracle商店：

    https://shop.oracle.com

    Oracle Software Delivery Cloud网站：

    https://edelivery.oracle.com/linux

  2. 从ISO或DVD映像安装Oracle Linux。

  3. 以root.身份登录root.

  4. 使用Oracle Linux yum服务器网站上的说明从以下URL下载Linux发行版的yum存储库文件：

    http://yum.oracle.com/

    例如：

    #cd /etc/yum.repos.d/ wget http://yum.oracle.com/public-yum-ol7.repo

    确保已public_olrelease_latest文件（例如， public_ol7_latest于Oracle Linux 7的public_ol7_latest ），因为这是包含Oracle预安装RPM的存储库。

  5. （可选）编辑repo文件以启用其他存储库。例如，通过使用文本编辑器在文件中设置enabled=1来enabled=1存储库public_ol7_latest 。

  6. 运行命令yum repolist以验证已注册的通道。

  7. 启动终端会话并以root身份输入以下命令，具体取决于您的平台。例如：

    Oracle Linux 7：

    #yum install oracle-database-preinstall-19c

    注意：

    如果您希望yum跳过包确认提示，请使用-y选项。

    您应该看到输出，表明您已订阅Oracle Linux通道，并且正在安装这些软件包。

    Oracle Linux自动创建标准（非角色分配）Oracle安装所有者和组，并根据Oracle安装的要求设置其他内核配置设置。如果计划使用作业角色分离，则根据您的要求创建扩展的数据库用户和组集。

    安装后，根据需要运行命令yum update ，以获取Oracle Linux安装的最新安全勘误和错误修复。
