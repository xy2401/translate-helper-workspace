关于Oracle Grid Infrastructure安装

您可以将Oracle Grid Infrastructure安装为Oracle独立群集，Oracle域服务群集或Oracle成员群集。

从Oracle Grid Infrastructure 12c第2版（12.2）开始，安装介质将替换为Oracle Grid Infrastructure安装程序的zip文件。将zip文件解压缩到目标主路径后运行安装向导。

在安装过程中的任何时候，如果您对要求执行的操作或安装期间需要提供的输入有疑问，请单击安装程序窗口中的“ 帮助”按钮。

在开始安装之前，您应该可以使用网络信息，存储信息以及操作系统用户和组，并且应该准备好运行根脚本。对于Oracle成员集群安装，请确保已按照本指南中的说明创建了成员集群清单文件。

作为在第一个节点上拥有用于集群（ grid ）的Oracle Grid Infrastructure软件的用户，请为集群安装Oracle Grid Infrastructure。请注意，安装程序使用Secure Shell（SSH）将二进制文件从此节点复制到其他节点。在安装过程中，在“群集节点信息”窗口中，指定群集中的节点时，可以单击“ SSH连接” ，安装程序会为您配置指定节点之间的SSH连接。

注意：

这些安装说明假定您的系统上尚未安装任何Oracle软件。
