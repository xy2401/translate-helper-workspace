1.304 SESSION_CACHED_CURSORS

SESSION_CACHED_CURSORS指定要缓存的会话游标数。

属性 描述

参数类型

整数

默认值

50

可修改

ALTER SESSION，ALTER SYSTEM ......DEFERRED

可在PDB中修改

是

价值范围

0到操作系统相关

基本

没有

Oracle RAC

多个实例可以具有不同的值。

对同一SQL（包括递归SQL）或PL / SQL语句的重复解析调用会导致该语句的会话游标移动到会话游标缓存中。Oracle使用最近最少使用的算法来删除会话游标缓存中的条目，以便在需要时为新条目腾出空间。
