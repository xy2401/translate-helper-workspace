3.103 ALL_TAB_COL_STATISTICS

ALL_TAB_COL_STATISTICS显示从ALL_TAB_COLUMNS提取的列统计信息和直方图信息。

相关观点

  • DBA_TAB_COL_STATISTICS显示从“ DBA_TAB_COLUMNS ”中提取的此类信息。

  • USER_TAB_COL_STATISTICS显示从“ USER_TAB_COLUMNS ”中提取的此类信息。此视图不显示OWNER列。

表的所有者

VARCHAR2（128）

COLUMN_NAME

VARCHAR2（128）

列名

NUM_DISTINCT

数

列中的不同值的数量

数

如果COLUMN_NAME上有直方图，则此列显示直方图中跨越少于2个端点的值的选择性。它不表示跨越2个或更多端点的值的选择性。

如果COLUMN_NAME上没有直方图，则此列的值为1 / NUM_DISTINCT 。

VARCHAR2（3）

指示统计信息是否由用户直接输入（ YES ）（ NO ）

笔记

VARCHAR2（99）

描述统计信息的一些其他属性。例如：

  • 值INCREMENTAL表示全局统计信息源自概要，即全局统计信息以递增方式维护。

  • 值STATS_ON_CONVENTIONAL_LOAD表示通过在线统计信息收集传统DML获得统计信息。 脚1

AVG_COL_LEN

数

直方图

VARCHAR2（15）

表示直方图的存在/类型：

  • 没有

  • 频率

  • 高度平衡

  • HYBRID

  • TOP频

范围

VARCHAR2（7）

对于在除全局临时表之外的任何表上收集的统计信息，该值为SHARED 。

对于全局临时表，可能的值为：

  • SESSION - 表示统计信息是特定于会话的

  • SHARED - 表示统计信息在所有会话中共享

有关使用DBMS_STATS程序包的GLOBAL_TEMP_TABLE_STATS首选项来控制是否收集全局临时表的会话或共享统计信息的信息，请参阅Oracle数据库PL / SQL程序包和类型参考 。

脚注1从Oracle Database 19c版本19.1开始，可以使用STATS_ON_CONVENTIONAL_LOAD值。
