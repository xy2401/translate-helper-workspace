1.62 DATA_GUARD_MAX_IO_TIME

DATA_GUARD_MAX_IO_TIME设置在Oracle Data Guard环境中执行常规I / O操作时将进程视为挂起之前可以经过的最大秒数。常规I / O操作包括读，写和状态操作。

也可以看看：

“ DATA_GUARD_MAX_LONGIO_TIME ” ，使您可以在Oracle Data Guard环境中设置长I / O操作的超时，例如打开和关闭操作
