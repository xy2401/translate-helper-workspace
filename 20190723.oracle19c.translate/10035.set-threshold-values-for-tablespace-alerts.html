设置表空间警报的阈值

升级后，已升级的Oracle数据库表空间警报的阈值将设置为null，从而禁用警报。

您必须在数据库中标识要监视的表空间，并且必须为这些表空间设置适当的阈值。

从Oracle Database 18c及更高版本开始，在新创建的Oracle数据库安装中，以下值用作默认值：

  • 85％完全警告

  • 97％完全关键
