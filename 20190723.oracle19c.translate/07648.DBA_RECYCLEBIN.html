5.267 DBA_RECYCLEBIN

DBA_RECYCLEBIN显示有关数据库中所有回收站的信息。

相关视图

USER_RECYCLEBIN显示有关当前用户拥有的回收站的信息。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

对象的原始所有者的名称

OBJECT_NAME

VARCHAR2（128）

不是NULL

对象的新名称

ORIGINAL_NAME

VARCHAR2（128）

对象的原始名称

操作

VARCHAR2（9）

对象执行的操作：

  • DROP - 对象被删除

  • TRUNCATE - 对象被截断

    注意： Oracle数据库当前仅支持从回收站中恢复已删除的对象。截断的对象无法恢复。

类型

VARCHAR2（25）

对象的类型：

  • 表

  • 正常指数

  • BITMAP INDEX

  • 嵌套表

  • 高球

  • LOB INDEX

  • DOMAIN INDEX

  • IOT TOP INDEX

  • 物联网溢出部分

  • 物联网映射表

  • 触发

  • 表分区

  • 表复合分区

  • 索引分区

  • 索引复合分区

  • LOB分区

  • LOB复合分区

TS_NAME

VARCHAR2（30）

对象所属的表空间的名称

CREATETIME

VARCHAR2（19）

用于创建对象的时间戳

DROPTIME

VARCHAR2（19）

删除对象的时间戳

DROPSCN

数

将对象移动到回收站的事务的系统更改号（SCN）

PARTITION_NAME

VARCHAR2（128）

已删除的分区的名称

CAN_UNDROP

VARCHAR2（3）

指示对象是否可以取消（ YES ）与否（ NO ）

CAN_PURGE

VARCHAR2（3）

指示是否可以清除对象（ YES ）或不清除（ NO ）
