1.2.3 Oracle Trace File Analyzer简介

Oracle跟踪文件分析器是一种用于目标诊断收集的实用程序，它还简化了Oracle Clusterware，Oracle Grid Infrastructure和Oracle Real Application Clusters（Oracle RAC）系统的诊断数据收集，以及单实例非群集数据库。

默认情况下启用Oracle跟踪文件分析器：

  • 提供全面的首次故障诊断收集

  • 有效地收集，打包和传输诊断数据到Oracle支持

  • 减少客户与Oracle之间的往返行程

Oracle Trace File Analyzer可缩短获取正确诊断数据所需的时间，从而最终节省您的业务资金。
