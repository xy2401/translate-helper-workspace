安装操作系统包

了解如何安装前面列出的最新版本的Oracle Linux和SUSE Linux Enterprise Server操作系统软件包。

您必须使用YUM或YaST等软件包管理器从相应的操作系统供应商存储库安装最新版本的操作系统软件包，具体取决于您的操作系统。

确保启用适当的通道或存储库以包含这些包。

例如：

在Oracle Linux或Red Hat Enterprise Linux上，要使用YUM安装最新的bc软件包，请运行以下命令：

$ yum install bc

在SUSE Linux Enterprise Server上，要使用YaST安装最新的bc软件包，请运行以下命令：

$ yast --install bc
