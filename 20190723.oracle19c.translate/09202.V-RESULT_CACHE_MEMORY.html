8.169 V $ RESULT_CACHE_MEMORY

V$RESULT_CACHE_MEMORY显示所有内存块及其状态。

柱 数据类型 描述

ID

数

唯一块标识符（即块编号）

CHUNK

数

块所属的块（ID的高27位）

OFFSET

数

块内的块偏移（ID的低5位）

自由

VARCHAR2（3）

指示块是否空闲（ YES ）否（ NO ）

OBJECT_ID

数

内存块所属的缓存对象;如果内存块未分配给缓存对象，则为NULL（ FREE = YES ）

位置

数

块在缓存对象中的位置;如果内存块未分配给缓存对象，则为NULL（ FREE = YES ）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
