13.10收集方法调用

集合方法是PL / SQL子程序，它返回有关集合的信息或对集合进行操作。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

collection_method_call :: =


插图collection_method_call.eps的描述

语义

collection_method_call

采集

要调用其方法的集合的名称。

计数

返回集合中元素数量的函数，在“ COUNT集合方法 ”中进行了说明 。

删除

从“ DELETE收集方法 ”中解释的从集合中删除元素的过程。

对DELETE的限制

如果collection是varray，则无法使用DELETE指定索引。

指数

数据类型为PLS_INTEGER或可以隐式转换为PLS_INTEGER数字表达式（有关后者的信息，请参阅“ s ” ）。

EXISTS

如果集合的index th元素存在则返回TRUE函数，否则返回FALSE ，在“ EXISTS Collection Method ”中解释。

延伸

将元素添加到集合末尾的过程，在“ EXTEND集合方法 ”中进行了说明 。

对EXTEND的限制

如果collection是关联数组，则不能使用EXTEND 。

第一

返回集合中第一个索引的函数，在“ FIRST和LAST Collection Methods ”中进行了解释 。

持续

返回集合中最后一个索引的函数，在“ FIRST和LAST集合方法 ”中进行了说明 。

限制

返回集合可以具有的最大元素数的函数。如果集合没有最大大小，则LIMIT返回NULL 。有关示例，请参阅“ LIMIT收集方法 ” 。

下一个

返回集合的后续现有元素的索引的函数（如果存在）。否则， NEXT返回NULL 。有关更多信息，请参阅“ PRIOR和NEXT收集方法 ” 。

PRIOR

返回集合的前一个现有元素的索引的函数（如果存在）。否则， NEXT返回NULL 。有关更多信息，请参阅“ PRIOR和NEXT收集方法 ” 。

修剪

从“ TRIM收集方法 ”中解释的从集合末尾删除元素的过程。

对TRIM的限制

如果collection是关联数组，则无法使用TRIM 。

数

要从集合末尾删除的元素数。默认值：一。
