关于管理旧版数据库版本的磁盘组

在Oracle Grid Infrastructure安装上安装早期Oracle数据库时，使用Oracle ASM Configuration Assistant（Oracle ASMCA）创建和修改磁盘组。

Oracle Database 11g第2版之前的版本使用Oracle数据库配置助手（Oracle DBCA）在Oracle ASM上执行管理任务。从Oracle Database 11g第2版（11.2）开始，Oracle ASM作为Oracle Grid Infrastructure安装的一部分安装。您无法再使用Oracle DBCA在Oracle ASM上执行管理任务。
