为独立服务器配置Oracle Grid Infrastructure的软件二进制文件

使用这些步骤为Oracle Restart的独立服务器安装配置和激活仅限软件的Oracle Grid Infrastructure。

使用Oracle自动存储管理进行配置

  1. 安装软件二进制文件。请参阅“ Oracle数据库安装指南 ”中的“为独立服务器安装Oracle Grid Infrastructure的软件二进制文件” 。

  2. 运行gridSetup.sh以启动Oracle Grid Infrastructure安装向导。

    请参阅“ Oracle数据库安装指南 ”中的“使用新数据库安装为独立服务器安装Oracle Grid Infrastructure ” 。

配置不使用Oracle自动存储管理

  1. 以root身份登录并使用以下语法运行位于Grid_home路径中的roothas.sh脚本：

    #cd Grid_home / crs / install＃。/ roothas.sh

    其中， Grid_home是Oracle Grid Infrastructure主目录。

    例如：

    #cd /u01/app/oracle/product/19.0.0/grid/crs/install#./roothas.sh

  2. 将目录更改为路径Grid_home/ oui/bin 。

  3. 以Oracle Restart软件所有者用户身份登录，并使用以下命令语法，其中Grid_home是独立服务器主目录的Oracle Grid Infrastructure的路径。

    $ ./runInstaller -updateNodeList ORACLE_HOME = Grid_home -defaultHomeName CLUSTER_NODES = CRS = TRUE

    例如：

    $ ./runInstaller -updateNodeList ORACLE_HOME = / u01 / app / oracle / product / 19.0.0 / grid -defaultHomeName CLUSTER_NODES = CRS = TRUE

  4. 使用SRVCTL实用程序和Oracle Network Configuration Assistant将侦听器添加到Oracle Restart配置。

注意：

此过程不适用于以前版本的Oracle Restart升级。
