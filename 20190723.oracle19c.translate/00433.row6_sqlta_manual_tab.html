Description of the illustration row6_sqlta_manual_tab.gif

This is a screen shot of the Manual tab on the SQL Tuning Advisor page in EM Express. The top left side of the page includes the page name, SQL Tuning Advisor and the Automatic Runs field, which has a value of Show All. To the right of the Automatic Runs field is the Configuration icon. The top right side of the page shows when the page was last refreshed, and the Auto Refresh interval currently selected.

The section below the page name includes a text description of the SQL Tuning Advisor.

The Manual tab includes a description of the table of SQL tuning tasks that appears below. Just above the table is the Actions menu and the View Result and Drop icons. To the far right is the Search field for the table.

The columns in the table (from left to right) are Status, Name, Description, User, Start Time, Duration, and Expires In (days).

In this screen shot, the first tuning task is selected in the tuning tasks table.
