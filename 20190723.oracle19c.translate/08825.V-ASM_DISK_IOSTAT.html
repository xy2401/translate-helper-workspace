7.62 V $ ASM_DISK_IOSTAT

V$ASM_DISK_IOSTAT显示有关每个Oracle自动存储管理（Oracle ASM）客户端的磁盘I / O统计信息的信息。

如果从数据库实例查询此视图，则仅显示该实例的行。

柱 数据类型 描述

INSTNAME

VARCHAR2（64）

DB / ADVM实例客户端的标识符

DBNAME

VARCHAR2（8）

唯一数据库名称（ DB_UNIQUE_NAME ）

CLUSTERNAME

VARCHAR2（31）

集群的名称

GROUP_NUMBER

数

包含磁盘的磁盘组的编号

DISK_NUMBER

数

分配给其磁盘组中的磁盘的编号

FAILGROUP

VARCHAR2（30）

磁盘所属的故障组的名称

SITE_NAME

VARCHAR2（30）

这是磁盘所属站点的名称

READS

数

磁盘的I / O读取请求总数

WRITES

数

磁盘的I / O写请求总数

READ_ERRS

数

磁盘的I / O读取请求失败总数

WRITE_ERRS

数

磁盘的I / O写入请求失败总数

READ_TIMEOUT

数

超时的读I / O数

WRITE_TIMEOUT

数

超时的写I / O数

READ_TIME

数

如果TIMED_STATISTICS初始化参数设置为TRUE ，则磁盘读取请求的总I / O时间（以秒为单位）（如果设置为FALSE则为0 ）。

WRITE_TIME

数

如果TIMED_STATISTICS初始化参数设置为TRUE则为磁盘写入请求的总I / O时间（以秒为单位）（如果设置为FALSE则为0 ）

bytes_read缓存

数

从磁盘读取的总字节数

BYTES_WRITTEN

数

从磁盘写入的总字节数

HOT_READS

数

磁盘上热区域的读取次数

HOT_WRITES

数

磁盘上热区域的写入次数

HOT_BYTES_READ

数

从磁盘上的热区读取的字节数

HOT_BYTES_WRITTEN

数

写入磁盘上热区的字节数

COLD_READS

数

从磁盘上的冷区读取的数量

COLD_WRITES

数

磁盘上冷区写入的次数

COLD_BYTES_READ

数

从磁盘上的冷区读取的字节数

COLD_BYTES_WRITTEN

数

写入磁盘上冷区的字节数

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。
