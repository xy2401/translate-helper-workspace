1.352 USER_DUMP_DEST

USER_DUMP_DEST指定服务器将代表用户进程编写调试跟踪文件的目录的路径名。

属性 描述

参数类型

串

句法

USER_DUMP_DEST = { pathname | directory }

默认值

依赖于操作系统

可修改

改变系统

可在PDB中修改

没有

价值范围

任何有效的本地路径，目录或磁盘

基本

没有

注意：

不推荐使用USER_DUMP_DEST初始化参数。

例如，此目录可能设置如下：

  • 在MS-DOS上： C:\ORACLE\UTRC

  • 在UNIX上： /oracle/utrc

  • 在VMS上： DISK$UR3:[ORACLE.UTRC]

注意：

Oracle Database 11g第1版（11.1）中引入的可诊断性基础结构忽略此参数，该基础结构将跟踪和核心文件放在由DIAGNOSTIC_DEST初始化参数控制的位置。
