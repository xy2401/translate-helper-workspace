2.144 ALL_FILE_GROUPS

ALL_FILE_GROUPS显示有关当前用户可访问的文件组的顶级元数据。

相关观点

  • DBA_FILE_GROUPS显示有关数据库中所有文件组的顶级元数据。

  • USER_FILE_GROUPS显示有关当前用户拥有的文件组的顶级元数据。此视图不显示FILE_GROUP_OWNER列。

柱 数据类型 空值 描述

FILE_GROUP_OWNER

VARCHAR2（128）

不是NULL

文件组的所有者

FILE_GROUP_NAME

VARCHAR2（128）

不是NULL

文件组的名称

KEEP_FILES

VARCHAR2（1）

不是NULL

值Y或N表示在清除版本时是否应删除文件

MIN_VERSIONS

数

不是NULL

如果违反此条件，Autopurge不应删除版本

MAX_VERSIONS

数

不是NULL

当违反此条件时，Autopurge将删除最旧版本

RETENTION_DAYS

数

不是NULL

如果这样做不违反MIN_VERSIONS Autopurge将删除早于此版本的版本

CREATED

TIMESTAMP（6）带时区

不是NULL

创建文件组的时间

评论

VARCHAR2（4000）

关于文件组的评论

DEFAULT_DIRECTORY

VARCHAR2（128）

默认目录对象的名称
