251 UTL_COLL

The UTL_COLL包让PL / SQL程序中使用定位器收集查询和更新。

  • UTL_COLL子程序摘要

251.1摘要UTL_COLL子程序的

UTL_COLL包有一个子程序，即IS_LOCATOR函数。

表251-1 UTL_COLL包子程序

IS_LOCATOR功能

确定集合项是否实际上是定位器

251.1.1 IS_LOCATOR函数

此函数确定集合项是否实际上是定位器。

句法

UTL_COLL.IS_LOCATOR（coln IN STANDARD）返回BOOLEAN;

编译指示

断言WNDS ， WNPS和RNPS编译指示

参数

表251-2 IS_LOCATOR函数参数

参数 描述

科隆

嵌套表或varray项。

返回值

表251-3 IS_LOCATOR函数返回值

回报价值 描述

1

收集项目确实是一个定位器。

0

集合项不是定位器。

例子

创建或替换类型list_t作为表格VARCHAR2（20）; / CREATE OR REPLACE TYPE phone_book_t AS OBJECT（pno number，ph list_t）; / CREATE TABLE phone_book OF phone_book_t NESTED TABLE ph STORE AS nt_ph; CREATE TABLE phone_book1 OF phone_book_t NESTED TABLE ph STORE AS nt_ph_1 RETURN LOCATOR; INSERT INTO phone_book VALUES（1，list_t（'650-633-5707'，'650-323-0953'））; INSERT INTO phone_book1 VALUES（1，list_t（'415-555-1212'））;创建或替换程序chk_coll是plist list_t; plist1 list_t; BEGIN SELECT ph INTO plist FROM phone_book WHERE pno = 1; SELECT ph INTO plist1 FROM phone_book1 WHERE pno = 1; IF（UTL_COLL.IS_LOCATOR（plist））那么DBMS_OUTPUT.PUT_LINE（'plist是一个定位器'）; ELSE DBMS_OUTPUT.PUT_LINE（'plist不是定位器'）;万一; IF（UTL_COLL.IS_LOCATOR（plist1））那么DBMS_OUTPUT.PUT_LINE（'plist1是定位器'）; ELSE DBMS_OUTPUT.PUT_LINE（'plist1不是定位器'）;万一;结束chk_coll; SET SERVEROUTPUT ON EXECUTE chk_coll;
