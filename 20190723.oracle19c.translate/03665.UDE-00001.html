155 UDE-00001至UDE-00054

UDE-00001：用户请求取消当前操作\ n

原因：用户已请求取消当前作业。

行动：不需要采取任何行动。

UDE-00002：用户名或密码无效

原因：指定了无效的用户名或密码。

操作：使用有效的用户名和密码重试。

UDE-00003：所有允许的登录尝试都失败

原因：重复尝试使用无效的用户名或密码登录。

操作：关闭该实用程序，然后重新启动并使用有效的用户名和密码重试。

UDE-00004：操作生成ORACLE错误number

原因：操作意外产生了指定的错误。有关更多信息，请参阅该错误的说明。

操作：请参阅报告的错误说明以了解可能的操作。

UDE-00005：读取输入时遇到意外的文件结尾\ n

原因：在读取用户输入时遇到文件结尾。

操作：如果要重定向输出到导出，请检查文件是否有错误。

UDE-00006：无法分配大小number内存

原因：无法分配必要的内存。

操作：记录后面的消息并将其作为导出内部错误报告给Oracle客户支持。

UDE-00007：尝试释放未分配的内存

原因：尝试释放未分配的内存。

操作：联系Oracle全球支持

UDE-00010：请求多个作业模式， string和string 。

原因：指定了多个作业模式。

操作：仅指定一个有效模式，然后重试。

UDE-00011：参数string与参数string不兼容

原因：指定了无效的参数组合。

操作：请参阅文档以获取有效模式集，然后重试。

UDE-00012：表模式导出仅允许来自一个模式的对象

原因：选择了多个模式进行导出。

操作：重试导出仅指定一个架构。

UDE-00013：内部expdp错误。

原因：致命的内部数据泵客户端错误。

操作：联系Oracle客户支持

UDE-00014：参数值' string '无效。

原因：无法解析指定的参数或参数值。

操作：请参阅文档以获取有效的参数值。

UDE-00015：参数' string '的上下文或作业状态无效。

原因：指定参数的作业状态或上下文无效。

操作：请参阅文档以了解正确的参数用法。

UDE-00016：' string '参数需要一个值。

原因：指定的参数没有值。

操作：请参阅文档以了解正确的参数用法。

UDE-00017：数据泵作业已在执行。

原因：请求的操作的数据泵作业状态不正确。

操作：请参阅文档以了解正确用法。

UDE-00018：数据泵客户端与数据库版本string不兼容

原因：导出实用程序的版本高于数据库版本，因此不兼容。

操作：使用相同版本的Export实用程序作为数据库。

UDE-00019：' string '参数列表太长。

原因：请求的参数列表最多超过4000个字符。

操作：减少指定数量的参数值并重新运行。

UDE-00020：Data Pump客户端与Data Pump API不兼容。

原因：导出实用程序的版本高于数据泵服务器，因此不兼容。

操作：使用与服务器相同版本的Export实用程序。

UDE-00022：连接字符串\ n中指定的不受支持的管理权限

原因：连接字符串指定了数据泵不支持的管理权限。

操作：指定数据泵支持的管理权限。这些是SYSDBA和SYSBACKUP。

UDE-00050：无法设置参数处理。

原因：参数管理器初始化期间发生意外错误。

操作：记录后面的消息并将其作为导出内部错误报告给Oracle客户支持。

UDE-00051：无法将NLS_NUMERIC_CHARACTERS设置为所需的默认值。

原因：导出无法将NLS_NUMERIC_CHARACTERS设置为“。”，

操作：记录随附的消息并将其作为导出内部错误报告给Oracle客户支持。

UDE-00052：操作的Data Pump客户端不正确： string

原因：导出客户端尝试附加到导入作业。

操作：附加到现有作业时使用适当的客户端

UDE-00053：无法识别的客户端命令' string '

原因：无效的交互式客户端命令。

操作：请参阅文档以获取有效的客户端命令。

UDE-00054：已弃用的传统模式参数' string '在或上指定： string

原因：指定的传统模式参数没有相应的数据泵映射。

操作：查看错误消息中引用的位置（（“命令行”，“参数文件”或“环境变量”）并删除参数。
