2.15在版本之间应用补丁

使用命令列表来管理修补程序。

句法

orachk -applypatch orachk_ bug_id .zip exachk -applypatch exachk_ bug_id .zip
orachk -querypatch all exachk -querypatch all orachk -querypatch bug_id exachk -querypatch bug_id
orachk -rollbackpatch bug_id exachk -rollbackpatch bug_id

表2-6管理补丁

命令 描述

-applypatch

为指定的错误ID应用新修补程序。

-querypatch

列出所有已安装修补程序的详细信息或指定的错误ID。

-rollbackpatch

将应用的修补程序回滚到之前的状态，即应用修补程序的状态。
