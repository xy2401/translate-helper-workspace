说明awm_mod.gif的描述

Analytic Workspace Manager主窗口：此屏幕截图显示Analytic Workspace Manager中的主窗口。左侧窗格显示包含以下文件夹的导航树：

  • Databases / rel12c（全局） - OLAP11g / Schemas / GLOBAL包含以下文件夹：

    • Analytic Workspaces / GLOBAL（附加RW）包含语言和这些子文件夹：

      • 外形尺寸

      • 立方体

      • 测量文件夹

      • OLAP DML程序

    • 维护脚本

    • 维护报告

  • Databases / rel12c（全局） - OLAP11g / Schemas / Data Security角色

  • 数据库/ rel12c（全局） - OLAP11g / Reports

由于在导航树中选择了GLOBAL（附加的RW），因此右窗格显示GLOBAL分析工作区的属性表。
