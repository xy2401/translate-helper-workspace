确定Oracle Home是否为只读

运行orabasehome命令以确定您的Oracle主目录是读/写还是只读Oracle主目录。

如果orabasehome命令的输出与$ ORACLE_HOME相同，那么您的Oracle主目录处于读/写模式。如果输出显示路径ORACLE_BASE/homes/HOME_NAME ，则表示您的Oracle主目录处于只读模式。
  1. 设置ORACLE_HOME环境变量：

    Bourne，Bash或Korn shell：

    $ ORACLE_HOME = / u01 / app / oracle / product / 19.0.0 / dbhome_1 $ export ORACLE_HOME

    C shell：

    ％setenv ORACLE_HOME /u01/app/oracle/product/19.0.0/dbhome_1
  2. 转到bin目录并运行orabasehome命令：
    $ cd $ ORACLE_HOME / bin $ ./orabasehome / u01 / app / oracle / homes / OraDB19Home1

    在此示例中，Oracle主目录处于只读模式。
