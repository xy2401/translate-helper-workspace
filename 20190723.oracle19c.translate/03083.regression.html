3.1关于回归

回归是一种数据挖掘功能，可以预测连续体中的数值。利用回归技术可以预测利润，销售，抵押贷款利率，房屋价值，面积，温度或距离。例如，回归模型可用于根据位置，房间数量，批量和其他因素预测房屋的价值。

回归任务以一个已知目标值的数据集开始。例如，可以基于一段时间内许多房屋的观测数据来开发预测房屋价值的回归模型。除了价值，数据还可以跟踪房屋的年龄，面积，房间数量，税收，学区，购物中心的距离等。房屋价值可以是目标，其他属性是预测变量，每个房屋的数据构成一个案例。

在模型构建（训练）过程中，回归算法估计目标的值作为构建数据中每个案例的预测变量的函数。预测变量和目标之间的这些关系在模型中进行了总结，然后可以将其应用于目标值未知的不同数据集。

通过计算测量预测值与预期值之间差异的各种统计数据来测试回归模型。回归项目的历史数据通常分为两个数据集：一个用于构建模型，另一个用于测试模型。

回归建模在趋势分析，业务规划，营销，财务预测，时间序列预测，生物医学和药物响应建模以及环境建模方面具有许多应用。

3.1.1回归如何工作？

您不需要了解回归分析中使用的数学来开发和使用质量回归模型进行数据挖掘。但是，理解一些基本概念是有帮助的。

回归分析旨在确定函数的参数值，使函数最适合您提供的一组数据观察。以下等式用符号表示这些关系。它表明回归是估计连续目标（ y ）的值作为一个或多个预测变量（ x 1 ， x 2 ，...， x n ）的函数（ F ）的过程，一组参数（θ） 1，θ2，...，θn），以及差错（e的度量）。 

y = F （ x ，θ）+ e

3.1.1.2多元线性回归

术语多元线性回归是指具有两个或更多个预测因子（ x 1 ， x 2 ，...， x n ）的线性回归。当使用多个预测变量时，回归线无法在二维空间中可视化。然而，可以简单地通过扩展单预测器线性回归的等式来计算线，以包括每个预测器的参数。 

Y =θ1个 +θ2×1个 +θ3×2 + .....θ 的N×N-1 + E
3.1.1.3回归系数

在多元线性回归，回归参数通常被称为系数。构建多元线性回归模型时，算法会计算模型使用的每个预测变量的系数。系数是预测器x对目标y的影响的度量。有许多统计数据可用于分析回归系数，以评估回归线与数据的匹配程度。

3.1.1.5多元非线性回归

术语多元非线性回归是指具有两个或更多个预测变量（ x 1 ， x 2 ，...， x n ）的非线性回归。当使用多个预测变量时，非线性关系不能在二维空间中可视化。

3.1.1.6置信区间

回归模型预测评分数据中每个案例的数字目标值。除了预测，一些回归算法可以识别置信界限，即预测值可能存在的区间的上下边界。

当建立模型以使用给定置信度进行预测时，产生置信区间以及预测。例如，模型预测房屋的价值为500,000美元，95％的信心值为475,000美元至525,000美元。

3.2测试回归模型

通过将回归模型应用于具有已知目标值的测试数据并将预测值与已知值进行比较来测试回归模型。

测试数据必须与用于构建模型的数据兼容，并且必须以与构建数据相同的方式准备。通常，构建数据和测试数据来自相同的历史数据集。一部分记录用于构建模型;其余记录用于测试模型。

测试指标用于评估模型预测这些已知值的准确程度。如果模型运行良好并满足业务要求，则可以将其应用于新数据以预测未来。

3.2.1回归统计

通常使用均方根误差和平均绝对误差用于评估回归模型总体质量的统计数据。根据算法使用的回归方法，也可以使用不同的统计数据。

3.3回归算法

Oracle Data Mining支持三种用于回归广义线性模型（GLM）， 神经网络 （NN）和支持向量机（SVM）的算法。

广义线性模型（GLM）和支持向量机（SVM）算法特别适用于挖掘具有非常高维度（许多属性）的数据集，包括事务和非结构化数据。

  • 广义线性模型（GLM）

    GLM是一种流行的线性建模统计技术。Oracle Data Mining实现了GLM for Regression和二进制分类。GLM提供广泛的系数统计和模型统计以及行诊断。GLM也支持置信区间。

  • 神经网络

    神经网络是强大的算法，可以学习任意非线性回归函数。

  • 支持向量机（SVM）

    SVM是一种功能强大，最先进的线性和非线性回归算法。Oracle Data Mining实现了SVM用于回归，分类和异常检测。SVM回归支持两个核：非线性回归的高斯核和线性回归的线性核。

注意：

Oracle Data Mining使用线性内核SVM作为默认值回归算法。
