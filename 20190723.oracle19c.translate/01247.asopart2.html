第二部分使用Oracle数据编辑

第II部分介绍了如何使用Oracle Data Redaction。

  • Oracle数据编辑简介
    Oracle Data Redaction能够实时编辑敏感数据。
  • Oracle数据编辑功能和功能
    Oracle Data Redaction提供了多种编辑不同类型数据的方法。
  • 配置Oracle数据编校策略
    Oracle Data Redaction策略定义了如何根据表列类型和要使用的编辑类型来编辑列中的数据。
  • 在Oracle Enterprise Manager中管理Oracle数据编校策略
    Oracle Enterprise Manager云控制（云控制）可以管理Oracle Data Redaction策略和格式。
  • 将Oracle数据编辑与Oracle数据库功能结合使用
    Oracle Data Redaction可以与其他Oracle功能一起使用，但某些Oracle功能可能对Oracle Data Redaction有限制。
  • Oracle Data Redaction的安全注意事项
    Oracle提供了使用Oracle Data Redaction的指南。
