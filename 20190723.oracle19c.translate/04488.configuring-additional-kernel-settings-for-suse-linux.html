配置SUSE Linux的其他内核设置

SUSE Linux Enterprise Server系统上需要其他内核参数设置配置。

仅在SUSE Linux Enterprise Server系统上执行以下步骤：
  1. 输入以下命令以使系统在重新启动时读取/etc/sysctl.conf文件：
    ＃/ sbin / chkconfig boot.sysctl on
  2. 输入oinstall组的GID作为参数/proc/sys/vm/hugetlb_shm_group 。

    例如，oinstall组GID为501 ：

    #echo 501> / proc / sys / vm / hugetlb_shm_group
    使用vi将以下文本添加到/etc/sysctl.conf ，并启用boot.sysctl脚本以在系统重新启动时运行：
    vm.hugetlb_shm_group = 501

    注意：

    只能将一个组定义为vm.hugetlb_shm_group 。
