前言

本指南介绍了使用SecureFiles和大对象（LOB）数据类型以及数据库文件系统（DBFS）支持应用程序开发的数据库功能。本指南中的信息适用于所有平台，不包括特定于系统的信息。

听众

Oracle Database SecureFiles和Large Objects Developer's Guide适用于使用LOB和DBFS开发新应用程序的程序员，以及之前已实现此技术并且现在希望利用新功能的程序员。

多媒体和非结构化数据的高效安全存储变得越来越重要，本指南是Oracle Application Developers文档集中此主题的关键资源。

功能覆盖范围和可用性

Oracle Database SecureFiles和Large Objects Developer's Guide包含描述Oracle Database 12 c第2版（12.2）的SecureFiles LOB和BasicFiles LOB特性和功能的信息。

使用LOB的先决条件

Oracle数据库包括在应用程序中使用LOB所需的所有资源;但是，存在一些限制，在“ LOB规则和限制 ”和“ 分区索引组织表中的LOB 限制 ”中有所描述。
