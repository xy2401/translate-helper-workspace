9.126 V $ VERSION

V$VERSION显示Oracle数据库的版本号。数据库组件具有与数据库相同的版本号，因此版本号仅返回一次。

柱 数据类型 描述

旗帜

VARCHAR2（80）

组件名称和版本号

BANNER_FULL

VARCHAR2（160）

Oracle Database 18c中引入了新的2行横幅格式。横幅显示数据库版本和版本号。

BANNER_LEGACY

VARCHAR2（80）

Oracle Database 18c之前使用的旧版1行横幅。此列显示与BANNER列相同的值。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
