1.275 RDBMS_SERVER_DN

RDBMS_SERVER_DN指定Oracle服务器的专有名称（DN）。它用于从企业目录服务检索企业角色。

属性 描述

参数类型

X.500杰出名称

默认值

没有默认值。

可修改

没有

可在PDB中修改

没有

价值范围

所有X.500可分辨名称格式值

基本

没有

注意：

Oracle Database 12 c第1版（12.1.0.2）中不推荐使用RDBMS_SERVER_DN参数。它由LDAP_DIRECTORY_ACCESS初始化参数替换。

如果您不想将目录用于企业用户和权限管理，但更喜欢单独使用SSL身份验证，请不要设置此参数。
