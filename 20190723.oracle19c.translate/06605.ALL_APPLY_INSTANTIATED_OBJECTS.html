2.39 ALL_APPLY_INSTANTIATED_OBJECTS

ALL_APPLY_INSTANTIATED_OBJECTS显示有关已为其设置实例化SCN的当前用户可访问的对象的信息。

相关视图

DBA_APPLY_INSTANTIATED_OBJECTS显示有关已为其设置实例化SCN的对象的信息。

柱 数据类型 空值 描述

SOURCE_DATABASE

VARCHAR2（128）

不是NULL

对象源自的数据库的名称

SOURCE_OBJECT_OWNER

VARCHAR2（128）

不是NULL

源数据库中对象的所有者

SOURCE_OBJECT_NAME

VARCHAR2（128）

不是NULL

源数据库中对象的名称

SOURCE_OBJECT_TYPE

VARCHAR2（11）

源数据库中对象的类型

INSTANTIATION_SCN

数

对象的实例化SCN。仅应用过程应用此SCN之后提交的更改。

IGNORE_SCN

数

SCN低于其，无法设置实例化SCN。此值对应于准备对象以进行实例化时源数据库中的SCN值。

APPLY_DATABASE_LINK

VARCHAR2（128）

应用更改的数据库链接。如果为null，则更改将应用于本地数据库。

SOURCE_ROOT_NAME

VARCHAR2（128）

源根数据库的全局名称
