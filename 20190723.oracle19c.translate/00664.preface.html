前言

本手册介绍了Oracle XML DB，以及如何使用它来存储，生成，操作，管理和查询数据库中的XML数据。

在介绍了Oracle XML DB的核心，即XMLType框架和Oracle XML DB Repository之后，本手册简要介绍了在规划Oracle XML DB应用程序时要考虑的设计标准。它提供了使用Oracle XML DB的方式和位置示例。

然后，本手册介绍了使用Oracle XML DB存储和检索XML数据的方法，用于处理XMLType数据的API以及查看，生成，转换和搜索现有XML数据的方法。本手册的其余部分讨论了如何使用Oracle XML DB Repository，包括版本控制和安全性，如何使用协议，SQL，PL / SQL或Java访问和操作存储库资源，以及如何使用Oracle Enterprise管理Oracle XML DB应用程序经理。它还向您介绍了XML消息传递和Oracle数据库高级队列XMLType支持。

关于案例敏感性的提醒

SQL不区分大小写。XML区分大小写。混合使用SQL和XML时请记住这一点。

在检查本书中的示例时，请记住以下内容：

  • SQL不区分大小写，但SQL代码中的名称是隐式大写的，除非将它们用双引号（ " ）括起来。

  • XML区分大小写。您必须使用正确的大小写引用XML代码中的SQL名称：大写SQL名称必须写为大写。

例如，如果在SQL中创建名为my_table的表而不使用双引号，则必须在XML代码中将其引用为“ MY_TABLE ”。
