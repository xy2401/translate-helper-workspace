272 ANYDATASET TYPE

ANYDATASET TYPE包含给定类型的描述以及该类型的一组数据实例。如果需要， ANYDATASET可以持久存储在数据库中，或者它可以用作接口参数来传递自描述数据集，所有这些数据都属于某种类型。

  • 施工

  • ANYDATASET TYPE子程序摘要

272.1 ANDDATASET TYPE结构

ANYDATASET需要按值依次构建。

对于每个数据实例（ ANYDATASET的类型），必须调用ADDINSTANCE函数。这会向ANYDATASET添加一个新的数据实例。随后，可以调用SET*来完整地设置每个值。

通过调用PIECEWISE.可以将构造/访问MODE更改为属性/集合元素PIECEWISE.

  • 如果ANYDATASET的类型是TYPECODE_OBJECT ，则将使用后续SET*调用设置各个属性。同样在访问。

  • 如果当前数据值的类型是集合类型，则将使用后续SET*调用设置单个集合元素。同样在访问。这个调用非常相似ANYDATA.PIECEWISE对该类型的呼叫ANYDATA 。

请注意，不支持分段构造和访问对象类型或集合类型的嵌套（非顶级）属性。

应该调用ENDCREATE来完成构建过程（在此之前不能进行访问调用）。

272.2.4 GET *成员函数

这些函数获取当前数据值（应该是适当的类型）。

当前数据值的类型取决于用于访问它的MODE （取决于调用PIECEWISE调用的方式）。如果尚未调用PIECEWISE ，则完整访问实例，并且数据值的类型应与ANYDATASET的类型匹配。

如果已调用PIECEWISE ，则逐个访问实例。数据值的类型应与当前位置的属性（或集合元素）的类型匹配。

272.2.6 GETINSTANCE成员函数

此函数获取ANYDATASET的下一个实例。只允许对ANYDATASET的实例进行顺序访问。

调用此函数后，可以在ANYDATASET上调用GET*函数来访问当前实例。如果在执行GET*调用之前调用了PIECEWISE则可以访问各个属性（或集合元素）。

在完全创建ANYDATASET之前调用此函数是错误的。

句法

成员函数GETINSTANCE（自我输入NOCOPY ANYDATASET）返回PLS_INTEGER;

272.2.9 PIECEWISE会员程序

此过程设置构造MODE ，一次访问数据值为属性（如果数据值为TYPECODE_OBJECT ）。

它设置构造MODE ，一次访问数据值作为集合元素（如果数据值是集合TYPE ）。一旦完成此调用，后续的SET*和GET*调用将依次获取单个属性或集合元素。

句法

成员程序分段（自我进出心跳任意数据）;

参数

表272-10 PIECEWISE过程参数

参数 描述

自

ANYDATASET正在建设中。

例外

使用说明

在进行此调用之前，当前数据值必须是对象或集合类型。不支持分段构造或访问嵌入对象类型属性或嵌套集合。

272.2.10 SET *会员程序

此过程设置当前数据值。

当前数据值的类型取决于我们构造的MODE （取决于我们如何调用PIECEWISE调用）。如果未调用PIECEWISE ，则当前数据的类型应为ANYDATASET的类型。如果已调用PIECEWISE则类型应为当前位置的属性类型。
