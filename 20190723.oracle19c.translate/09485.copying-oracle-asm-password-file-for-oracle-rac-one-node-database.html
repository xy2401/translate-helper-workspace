为Oracle RAC One节点数据库复制Oracle ASM密码文件

安装Oracle RAC One Node数据库后，将Oracle ASM密码文件（如果已配置）复制到配置为故障转移的所有候选节点，除非您使用存储在Oracle ASM中的共享密码文件。

如果使用Database Configuration Assistant（DBCA）安装Oracle RAC One Node数据库，则无需执行此任务，因为DBCA会将Oracle ASM密码文件复制到共享位置。
