关于数据库安装的群集节点选择

选择群集节点以安装Oracle RAC或Oracle RAC One Node数据库。

您可以使用群集节点托管以只读模式运行的Oracle RAC数据库实例，这些实例将成为Oracle RAC Reader节点。Oracle RAC Reader节点通过在集群节点上分配一组运行联机事务处理（OLTP）工作负载和一组只读数据库实例的读/写实例来促进Oracle Flex Cluster体系结构。
