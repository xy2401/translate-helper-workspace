12.11改变

句法

C [HANGE] sepchar 旧 [ sepchar [ new [ sepchar ]]]

更改缓冲区中当前行的第一次出现的指定文本。

条款

sepchar

表示任何非字母数字字符，例如“/”或“！”。使用未出现在旧版或新版中的sepchar 。 

旧

表示您要更改的文本。CHANGE忽略搜索旧的情况 。例如，

CHANGE / aq / aw

找到第一次出现的“aq”，“AQ”，“aQ”或“Aq”并将其更改为“aw”。SQL * Plus完全按照您的指定插入新文本。

如果old以“...”为前缀，则它匹配所有内容，包括第一次出现的旧内容 。如果它以“...”为后缀，则它匹配第一次出现的old以及该行后面的所有内容。如果它包含一个嵌入的“...”，它匹配一切从老通过老以下部分的前半部分。 

新

表示您希望替换旧文本的文本。如果省略new和（可选）第二个和第三个sepchars，CHANGE将从缓冲区的当前行中删除old 。

用法

改变变化缓冲区当前行上第一次出现的指定文本到新指定的文本。当前行在LIST输出中标有星号（*）。

您还可以使用CHANGE修改生成Oracle数据库错误的缓冲区中的行。SQL * Plus将缓冲区的当前行设置为包含错误的行，以便您可以进行修改。

要重新输入整行，可以键入行号，后跟行的新内容。如果指定的行号大于缓冲区中的行数并使用文本后跟数字，则SQL * Plus会在缓冲区末尾的新行中添加文本。如果为行号指定零（“0”）并在文本后跟零，则SQL * Plus会在缓冲区的开头插入该行（该行变为第1行）。

例子

输入3，以便缓冲区的当前行包含以下文本：

3
3 * WHERE JOB_ID在（'CLERK'，'SA_MAN'）

输入以下命令：

更改/秘书/ SH_CLERK /

缓冲区中的文本更改如下：

3 * WHERE JOB_ID在（'SH_CLERK'，'SA_MAN'）

或输入以下命令：

改变''CLERK'，....../ 'SH_CLERK'/

原始行更改为

3 * WHERE JOB_ID在（'SH_CLERK'）

或输入以下命令：

改变/（...）/（ 'SA_MAN'）/

原始行更改为

3 * WHERE JOB_ID在（'SA_MAN'）

您可以使用行号替换整行的内容。这个条目

3 WHERE JOB_ID在（'SH_CLERK'）

导致缓冲区的第二行被替换为

JOB_ID在哪里（'SH_CLERK'）

请注意，输入行号后跟字符串将替换该行，而不管行号后面的文本是什么。例如，

2更改/旧/新/

将更改缓冲区的第二行

2 * C / OLD / NEW /
