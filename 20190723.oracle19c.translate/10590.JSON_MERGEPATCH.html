目的

JSON_MERGEPATCH是一种标准化功能，用于使用称为合并补丁的JSON文档修改目标JSON文档。该功能在RFC 7396中描述。

target_expr计算为JSON值目标文档。

patch_expr计算为JSON值补丁文档。

JSON_MERGEPATCH针对目标文档JSON_MERGEPATCH补丁文档以生成结果文档。如果目标或补丁文档为NULL，则结果也为NULL。

JSON_query_returning_clause指定运算符的返回类型。默认返回类型是VARCHAR2(4000) 。

PRETTY关键字指定应格式化结果以提供人类可读性。

ASCII关键字指定应使用JSON转义序列输出非ASCII字符。

TRUNCATE关键字指定应截断结果文档以适合指定的返回类型。

on_error_clause可选地控制处理目标和补丁文档期间发生的错误。

  • NULL ON ERROR - 发生错误时返回null。这是默认值。
  • ERROR ON ERROR - 发生错误时返回相应的Oracle错误。
