前言

Oracle C ++调用接口（OCCI）是一个应用程序编程接口（API），它允许用C ++编写的应用程序与一个或多个Oracle数据库服务器进行交互。OCCI使您的程序能够执行Oracle数据库服务器可能实现的所有数据库操作，包括SQL语句处理和对象操作。

听众

“ Oracle C ++调用接口程序员指南”适用于执行或有兴趣了解以下任务的程序员，系统分析员，项目经理和其他Oracle用户：

  • 在Oracle环境中设计和开发数据库应用程序。

  • 转换现有数据库应用程序以在Oracle环境中运行。

  • 管理数据库应用程序的开发。

要使用本文档，您需要对面向对象编程概念有基本的了解，熟悉结构化查询语言（SQL）的使用，以及使用C ++进行应用程序开发的应用知识。
