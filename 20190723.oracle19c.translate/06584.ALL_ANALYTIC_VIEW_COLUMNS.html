2.17 ALL_ANALYTIC_VIEW_COLUMNS

ALL_ANALYTIC_VIEW_COLUMNS描述了当前用户可访问的分析视图的列。

相关观点

  • DBA_ANALYTIC_VIEW_COLUMNS描述数据库中所有分析视图的列。

  • USER_ANALYTIC_VIEW_COLUMNS描述了当前用户拥有的分析视图列。此视图不显示OWNER列。

柱 数据类型 空值 描述

不是NULL

ORDER_NUM

数

不是NULL

列的顺序，首先是层次结构列，然后是度量列。层次结构的列组合在一起，按顺序在分析视图定义的HIERARCHIES子句中列出。在层次结构中，属性首先按属性维度定义的ATTRIBUTES子句中的定义顺序列出，然后是分析视图的DIMENSION BY子句中的分层属性。

ORIGIN_CON_ID

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
