130 DBMS_RANDOM

DBMS_RANDOM包提供内置随机数生成器。DBMS_RANDOM不适用于加密。

本章包含以下主题：

  • 不推荐使用的子程序

  • 安全模型

  • 操作说明

  • DBMS_RANDOM子程序摘要

130.1 DBMS_RANDOM已弃用的子程序

Oracle Database 11g不推荐使用这些子程序。Oracle建议您不要在新应用程序中使用已弃用的过程。对不推荐使用的功能的支持仅用于向后兼容。

  • INITIALIZE程序

  • 随机功能

  • 终止程序

130.2 DBMS_RANDOM安全模型

该软件包应作为SYS安装。默认情况下，使用当前用户名，当前时间到第二个以及当前会话初始化程序包。Oracle建议需要执行此程序包的用户应明确授予EXECUTE权限，并且不应依赖PUBLIC EXECUTE权限。

130.3 DBMS_RANDOM操作说明

这些操作说明适用于DBMS_RANDOM。

  • DBMS_RANDOM.RANDOM在[-2 ^^ DBMS_RANDOM.RANDOM ^^ 31）中生成整数。

  • DBMS_RANDOM.VALUE在[0,1）中生成38位精度的数字。

DBMS_RANDOM可以显式初始化，但在调用随机数生成器之前不需要初始化。如果未执行显式初始化，它将自动使用日期，用户ID和进程ID进行初始化。

如果这个包用相同的种子播种两次，然后以相同的方式访问，它将在两种情况下产生相同的结果。

在某些情况下，例如测试时，您可能希望每次运行时随机数序列相同。在这种情况下，通过调用DBMS_RANDOM.SEED一个重载，您可以使用常量值为生成器设定种子。要为每次运行生成不同的输出，只需省略对“种子”的调用，系统就会为您选择合适的种子。

130.4.1初始化程序

不推荐使用的过程初始化生成器。

注意：

版本11gR1不推荐使用此过程，虽然目前支持，但不应使用它。

句法

DBMS_RANDOM.INITIALIZE（val IN BINARY_INTEGER）;

编译指示

PRAGMA restrict_references（初始化，WNDS）;

参数

130.4.3随机函数

不推荐使用的过程会生成一个随机数。

注意：

Release 11gR1不推荐使用此函数，虽然目前支持，但不应使用它。

句法

DBMS_RANDOM.RANDOM RETURN binary_integer;

编译指示

PRAGMA restrict_references（随机，WNDS）;

返回值

表130-4 RANDOM功能参数

参数 描述

BINARY_INTEGER

返回大于或等于-power（2,31）且小于power（2,31）的随机整数

130.4.4 SEED程序

此过程重置种子。

句法

DBMS_RANDOM.SEED（val IN BINARY_INTEGER）; DBMS_RANDOM.SEED（val IN VARCHAR2）;

编译指示

PRAGMA restrict_references（种子，WNDS）;

参数

表130-5 SEED程序参数

参数 描述

VAL

种子编号或用于生成随机数的字符串

使用说明

种子可以是长度达2000的字符串。

130.4.5 STRING功能

此函数获取随机字符串。

句法

DBMS_RANDOM.STRING选择IN CHAR，len IN NUMBER）RETURN VARCHAR2;

编译指示

PRAGMA restrict_references（字符串，WNDS）;

参数

表130-6 STRING功能参数

参数 描述

选择

指定返回字符串的外观：

  • 'u'，'U' - 以大写字母字符返回字符串

  • 'l'，'L' - 以小写字母字符返回字符串

  • 'a'，'A' - 以大小写混合字符字符返回字符串

  • 'x'，'X' - 以大写字母数字字符返回字符串

  • 'p'，'P' - 以任何可打印字符返回字符串。

否则返回的字符串是大写字母字符。

130.4.6终止程序

完成程序包后，请调用TERMINATE过程。

注意：

版本11gR1不推荐使用此过程，虽然目前支持，但不应使用它。

句法

DBMS_RANDOM.TERMINATE;

130.4.7 VALUE函数

基本函数获得一个随机数，大于或等于0且小于1，小数点右边有38位数（38位精度）。或者，您可以获得随机的Oracle编号x，其中x大于或等于low且小于high 。

句法

DBMS_RANDOM.VALUE返回号码; DBMS_RANDOM.VALUE（低数字，高数字）返回数字;

参数

表130-8 VALUE函数参数

参数 描述

低

生成随机数的范围中的最小数字。生成的数量可能等于low

高

生成随机数的最高数字。产生的数量将小于high

返回值

数
