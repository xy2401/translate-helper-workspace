36.1 OCI SODA功能介绍

C API的简单Oracle文档（SODA）是Oracle调用接口（OCI）的一部分。

SODA for C允许OCI程序员使用抽象存储和查询Javascript Object Notation（JSON）数据并访问数据库中的JSON功能：数据库，集合和文档。数据库包含集合和集合包含文档。API为文档集提供了创建，读取，更新和删除（CRUD）操作。
