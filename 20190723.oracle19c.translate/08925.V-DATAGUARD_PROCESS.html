7.164 V $ DATAGUARD_PROCESS

V$DATAGUARD_PROCESS为当前正在运行的每个Oracle Data Guard进程显示一行。

柱 数据类型 描述

名称

VARCHAR2（5）

正在报告其信息的进程的名称。一些可能的值包括：

  • ARC n - 存档过程

  • DTS - 数据传输过程

  • FAL - 文件/宣布过程

  • LGWR - 日志编写过程

  • MRP0 - 分离的恢复服务器进程

  • NSS n - SYNC重做传输过程

  • ORA - 前景过程

  • RFS - 远程文件服务器

  • RMI - 远程消息进程

  • TMON - 重做传输过程监视器

  • TT nn - 重做运输奴隶程序

PID

VARCHAR2（24）

操作系统进程的进程标识符

类型

VARCHAR2（3）

指示哪个Oracle子系统创建了该进程。可能的值：

  • KSB

  • KSV

  • 净

  • UNK

角色

VARCHAR2（23）

这个过程的作用。一些可能的值包括：

  • 测试

  • async ORL multi

  • 异步ORL单

  • async SRL multi

  • async SRL单

  • 日志作家

  • 同步

  • 存档重做

  • 存档本地

  • 档案差距

  • RFS异步

  • RFS同步

  • RFS档案

  • RFS差距

  • RFS SMON

  • 数据传输

  • 数据接收

  • 重做运输监视器

  • 心跳重做告密者

  • 进程杀死

  • 岗位角色转换

  • 差距经理

  • 更新TMI

  • RFS ping

  • FAL GAP

  • FAL宣布

  • 故障转移

  • 切换

  • 远程故障转移

  • 远程切换

  • 重做运输计时器

  • 宣布请求

  • 管理恢复

  • 复苏

  • 控制文件更新

  • 未知

PROC_TIME

TIMESTAMP（0）

进程开始或注册以包含在此固定视图中的时间戳

TASK_TIME

TIMESTAMP（0）

请求进程的第一个任务的时间戳

TASK_DONE

VARCHAR2（1）

指示进程执行的任务是否完成（ Y ）（ N ）

行动

该过程的当前行动。可能的值包括：

  • ATTACHED

  • 未知

组＃

数

进程正在运行的日志的组编号

RESETLOG_ID

数

重置日志ID（分支）进程正在操作的日志

线＃

数

进程正在运行的线程号

序列＃

数

进程正在运行的序列号

块＃

数

该进程正在运行的起始块号

BLOCK_COUNT

数

进程正在运行的块数

DELAY_MINS

数

存档的重做日志延迟间隔（以分钟为单位）

DEST_ID

数

进程当前正在操作的目标ID

DEST_MASK

数

进程将在其上运行的所有目标ID的掩码，其中：

  • 位0是LOG_ARCHIVE_DEST1
  • 位1是LOG_ARCHIVE_DEST2
  • 位2是LOG_ARCHIVE_DEST3

等等。

DBID

数

进程正在运行的重做的数据库ID

DGID

数

进程必须与之通信的Data Guard ID

INSTANCE

数

进程必须在指定的DGID上与之通信的实例编号

STOP_STATE

VARCHAR2（7）

指示请求进程停止的方法：

  • 现在 - 请求立即停止并退出

  • 很快 - 请求停在下一个日志并退出

  • COMMIT - 请求写入剩余的重做并退出

  • 不适用 - 没有要求停止

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
