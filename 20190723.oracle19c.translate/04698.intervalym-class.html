13.16 IntervalYM类

IntervalYM支持SQL标准数据类型年 - 月间隔。

前导字段精度由输入的小数位数决定。

13.16.2 fromText（）

此方法将间隔初始化为inpstr的值。使用环境中设置的NLS参数解释字符串。

NLS参数从env 。如果env为NULL ，则从与实例关联的环境中拾取NLS参数（如果有）。

句法

void fromText（const string＆inpStr，const string＆nlsParam =“”，const Environment * env = NULL）;
参数 描述 
inpStr

输入字符串表示“年 - 月”形式的年月间隔。

 
nlsParam

NLS参数字符串。如果指定了nlsParam ，则确定要用于转换的NLS参数。如果未指定nlsParam ，则从envp中获取NLS参数。 

ENV

使用NLS参数的环境。
