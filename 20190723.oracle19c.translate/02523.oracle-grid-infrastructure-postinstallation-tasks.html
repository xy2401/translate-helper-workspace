10 Oracle Grid Infrastructure安装后任务

安装Oracle Grid Infrastructure后完成配置任务。

安装Oracle Grid Infrastructure后，您需要完成一些配置任务。此外，Oracle建议您在安装后立即完成其他任务。在使用这些产品之前，还必须完成特定于产品的配置任务。

注意：

本章仅介绍基本配置。有关更详细的配置和调整信息，请参阅特定于产品的管理和调整指南。

  • 必需的安装后任务
    完成初始安装后，下载并应用软件版本所需的修补程序。
  • 推荐的安装后任务
    Oracle建议您在安装后完成这些任务。
  • 关于Oracle数据库的默认SGA权限的更改
    从Oracle Database 12c第2版（12.2.0.1）开始，默认情况下，读取和写入系统全局区域（SGA）的权限仅限于Oracle软件安装所有者。
  • 将早期Oracle数据库版本与Oracle Grid Infrastructure一起使用
    有关将早期Oracle数据库版本与Oracle Grid Infrastructure 19c安装一起使用的信息，请查看以下主题：
  • 安装后修改Oracle Clusterware二进制文件
    安装后，如果需要修改Oracle Clusterware配置，则必须解锁Grid home。查看有关解锁Grid主页的信息。
