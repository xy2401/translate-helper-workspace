5.5从UCP借用标记的连接

启用池的数据源提供两个getConnection方法，用于从池中借用标记的连接。方法如下所示：

public Connection getConnection（java.util。属性标签）抛出SQLException; public Connection getConnection（String user，String password，java.util。属性标签）抛出SQLException;

这些方法要求将标签作为Properties对象传递给getConnection方法。以下示例演示如何与标签property1, value建立连接。 

String pname =“property1”; String pvalue =“value”;属性label = new Properties（）; label.setProperty（pname，pvalue）; Connection conn = pds.getConnection（label）;
