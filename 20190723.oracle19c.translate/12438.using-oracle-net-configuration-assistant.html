使用Oracle Net Configuration Assistant

完成Oracle数据库客户端安装后，以独立模式运行Oracle Net Configuration Assistant，以配置侦听器，命名方法，网络服务名称和目录服务器使用情况。

Oracle建议您准备好有关安装Oracle数据库的计算机主机名的信息。
以独立模式启动Oracle Net Configuration Assistant：
  1. 从$ORACLE_HOME/bin目录运行netca 。
  2. 根据需要响应配置提示和屏幕。屏幕因您选择的选项而异。在配置过程中的任何时候，如果您对要求执行的操作有疑问，请单击“ 帮助”。
