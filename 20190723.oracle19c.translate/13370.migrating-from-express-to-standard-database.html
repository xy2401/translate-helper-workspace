从Oracle Database Express Edition（Oracle数据库XE）迁移到Oracle数据库

您必须从Oracle Database Express Edition升级到Oracle Database Enterprise Edition，然后升级到当前的Oracle Database版本。

Oracle Database Express Edition（Oracle Database XE）是Oracle数据库的入门级版本。

要将Oracle Database 11g第2版（11.2）Express Edition（Oracle数据库XE）升级到Oracle Database 12c第2版或更高版本，必须先从Oracle Database XE升级到Oracle Database 12c第1版（12.1.0.2）企业版，并且然后升级到以后的Oracle Database Enterprise Edition版本。

有关更多信息，请参阅“Oracle Database Express Edition（XE）”Oracle在线论坛：

http://forums.oracle.com
