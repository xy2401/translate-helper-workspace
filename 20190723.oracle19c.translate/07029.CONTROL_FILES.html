1.52 CONTROL_FILES

CONTROL_FILES指定一个或多个控制文件名称，以逗号分隔。

属性 描述

参数类型

串

句法

CONTROL_FILES = filename [, filename ] ...

注意：控制文件名可以是OMF（Oracle托管文件）名称。使用CREATE CONTROLFILE REUSE语句重新创建控制文件时会发生这种情况。

默认值

依赖于操作系统

可修改

没有

可在PDB中修改

没有

价值范围

1到8个文件名

基本

是

Oracle RAC

多个实例必须具有相同的值。

每个数据库都有一个控制文件 ，其中包含描述数据库结构的条目（例如其名称，其创建的时间戳以及其数据文件和重做文件的名称和位置）。

Oracle建议您在不同设备上复用多个控制文件，或在操作系统级别镜像该文件。
