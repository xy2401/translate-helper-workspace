14.1数据库Web服务概述

Web服务支持Web上的应用程序到应用程序交互，无论平台，语言或数据格式如何。关键成分，包括可扩展标记语言（XML），简单对象访问协议（SOAP），Web服务描述语言（WSDL），REpresentational State Transfer（REST），Web应用程序描述语言（WADL），以及通用描述，发现和集成（UDDI）已在整个软件行业中得到采用。Web服务通常指的是在中间层应用程序服务器中实现和部署的服务。但是，在异构和断开连接的环境中，越来越需要通过Web服务接口访问存储过程以及数据和元数据。

Oracle数据库可以通过PL / SQL包和数据库中部署的Java类访问Web服务。从数据库中使用外部Web服务以及与SQL引擎集成，可实现企业信息集成。
