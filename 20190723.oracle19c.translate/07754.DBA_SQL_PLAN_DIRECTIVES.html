5.363 DBA_SQL_PLAN_DIRECTIVES

DBA_SQL_PLAN_DIRECTIVES显示有关系统中SQL计划指令的信息。

柱 数据类型 空值 描述

DIRECTIVE_ID

数

不是NULL

SQL计划指令的标识符

类型

VARCHAR2（16）

SQL计划指令的类型：

  • DYNAMIC SAMPLING ：SQL计划指令

  • DYNAMIC SAMPLING RESULT ：动态采样查询结果。此值仅出现在Oracle Database 12 c第2版（12.2.0.1）及更高版本中。

  • UNKNOWN ：未知

启用

VARCHAR2（3）

指示是否启用SQL计划指令。可能的值：

  • YES ：启用SQL计划指令。

  • NO ：未启用SQL计划指令。

州

VARCHAR2（10）

SQL计划指令的状态。可能的值包括：

  • SUPERSEDED ：此值表示相应的一列或多个组具有扩展名或直方图，或者存在可用于该指令的另一个SQL计划指令。

  • USABLE ：此值表示SQL计划指令可用于优化程序。

AUTO_DROP

VARCHAR2（3）

如果为YES ，则SQL计划指令在SPD_RETENTION_WEEKS之外的未使用时将被删除

原因

VARCHAR2（36）

创建SQL计划指令的原因

CREATED

TIMESTAMP（6）

SQL计划指令的创建时间戳

最后修改

TIMESTAMP（6）

最近修改SQL计划指令的时间戳

最后使用

TIMESTAMP（9）

最近使用SQL计划指令的时间戳
