类：ServerJDBCGeorasterTheme

OM .server 。ServerJDBCGeorasterTheme

新的ServerJDBCGeorasterTheme（名字）

构造一个新的动态GeoRaster主题（JDBC主题）。例如：


    var jdbcLandsat = new OM.server。ServerJDBCGeorasterTheme（ 'theme_jdbc_goeraster'）;

在实例化ServerJDBCTheme之后，可以添加其他属性。例如：

    jdbcLandsat.setDataSourceName（ 'mvdemo'）;
    jdbcLandsat.setSRID（ '8307'）;
    jdbcLandsat.setGeorasterColumnName（ '的GeoRaster'）;
    var sql ='从pci_image中选择georaster';
    jdbcLandsat.setQuery（SQL）;

可以将这样的主题添加到服务器映射请求实例中。
例如：

  • var req = new OM.server。ServerMapRequest（基本URL）;

  • req.addTheme（jdbcLandsat）;

参数：
名称 类型 描述
名称 串

JDBC主题的名称;主题
将使用此类中的属性动态构建。

setImageOperations（）

使用其参数设置图像操作。
一些操作，如标准化，均衡和线性拉伸，可以使用统计数据。
默认情况下，它们基于正在渲染的图像区域的统计信息，因此每个磁贴应具有自己的统计信息，
它会产生一个没有统一模式的地图。
为了在切片上生成统一的图像，使用PLSQL函数计算GeoRaster对象的统计信息，
然后在此类上设置全局统计信息的使用。

参数：
类型 描述
