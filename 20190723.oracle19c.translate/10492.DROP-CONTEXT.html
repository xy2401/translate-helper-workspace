DROP CONTEXT

目的

使用DROP CONTEXT语句从数据库中删除上下文命名空间。

删除上下文命名空间不会使该命名空间下为用户会话设置的任何上下文无效。但是，当用户下次尝试设置该上下文时，上下文将无效。

也可以看看：

有关上下文的更多信息，请参阅“ 创建上下文”和“ Oracle数据库安全指南”

先决条件

您必须具有DROP ANY CONTEXT系统特权。

句法

drop_context :: =


插图说明drop_context.eps

语义

命名空间

指定要删除的上下文命名空间的名称。您不能删除内置命名空间USERENV 。

也可以看看：

SYS_CONTEXT有关USERENV命名空间的信息

例子

删除应用程序上下文：示例

以下语句删除在CREATE CONTEXT中创建的上下文 ：

DROP CONTEXT hr_context;
