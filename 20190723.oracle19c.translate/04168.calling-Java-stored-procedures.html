9.6调用Java存储过程

发布Java类之后，从顶级和数据库触发器，SQL数据操作语言（DML）语句和PL / SQL块调用Java存储过程。使用点表示法在po_mgr包中引用这些存储过程。

从匿名PL / SQL块，您可以通过库存零件启动新的采购订单系统，如下所示：

BEGIN po_mgr.add_stock_item（2010，'camshaft'，245.00）; po_mgr.add_stock_item（2011，'连杆'，122.50）; po_mgr.add_stock_item（2012，'crankshaft'，388.25）; po_mgr.add_stock_item（2013，'cylinder head'，201.75）; po_mgr.add_stock_item（2014，'cylinder sleeve'，73.50）; po_mgr.add_stock_item（2015，'engine bearning'，43.85）; po_mgr.add_stock_item（2016，'flywheel'，155.00）; po_mgr.add_stock_item（2017年，'冻结插头'，17.95）; po_mgr.add_stock_item（2018，'head gasket'，36.75）; po_mgr.add_stock_item（2019，'lifter'，96.25）; po_mgr.add_stock_item（2020，'oil pump'，207.95）; po_mgr.add_stock_item（2021，'piston'，137.75）; po_mgr.add_stock_item（2022，'活塞环'，21.35）; po_mgr.add_stock_item（2023，'pushrod'，110.00）; po_mgr.add_stock_item（2024，'摇臂'，186.50）; po_mgr.add_stock_item（2025，'valve'，68.50）; po_mgr.add_stock_item（2026，'valve spring'，13.25）; po_mgr.add_stock_item（2027，'water pump'，144.50）;承诺;结束;

注册您的客户，如下：

BEGIN po_mgr.add_customer（101，'A-1 Automotive'，'4490 Stevens Blvd'，'San Jose'，'CA'，'95129'，'408-555-1212'）; po_mgr.add_customer（102，'AutoQuest'，'2032 America Ave'，'Hayward'，'CA'，'94545'，'510-555-1212'）; po_mgr.add_customer（103，'Bell Auto Supply'，'305 Cheyenne Ave'，'Richardson'，'TX'，'75080'，'972-555-1212'）; po_mgr.add_customer（104，'CarTech Auto Parts'，'910 LBJ Freeway'，'Dallas'，'TX'，'75234'，'214-555-1212'）;承诺;结束;

输入各个客户下的采购订单，如下所示：
