1.153 INSTANCE_TYPE

INSTANCE_TYPE指定实例是数据库实例，Oracle自动存储管理（Oracle ASM）实例还是Oracle ASM代理实例。

属性 描述

参数类型

串

句法

INSTANCE_TYPE = {RDBMS | ASM | ASMPROXY}

默认值

RDBMS

可修改

没有

可在PDB中修改

没有

基本

没有

Oracle RAC

多个实例必须具有相同的值。

值

  • RDBMS

    该实例是一个数据库实例。

  • ASM

    该实例是Oracle ASM实例。

  • ASMPROXY

    该实例是Oracle ASM代理实例。
