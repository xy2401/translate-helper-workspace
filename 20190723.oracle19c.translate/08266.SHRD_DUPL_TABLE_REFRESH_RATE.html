1.316 SHRD_DUPL_TABLE_REFRESH_RATE

SHRD_DUPL_TABLE_REFRESH_RATE显示重复的表刷新率（以秒为单位）。

属性 描述

参数类型

整数

默认值

60

可修改

改变系统

可在PDB中修改

没有

价值范围

20至4294967295（UB4MAXVAL）

基本

没有

Oracle RAC

所有实例都应使用相同的实例。

您可以更改此参数的值以更改重复的表刷新率。

复制表是在分片数据库中的所有分片上具有相同内容的表。使用以特定速率刷新的物化视图实现重复表。
