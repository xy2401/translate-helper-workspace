打开最低安装的SSH要求

Oracle Grid Infrastructure安装需要OpenSSH。

OpenSSH应包含在Linux发行版最小安装中。要确认已安装SSH软件包，请输入以下命令：

#rpm -qa | grep ssh

如果没有看到SSH软件包列表，请为Linux发行版安装这些软件包。
