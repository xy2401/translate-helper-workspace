BITMAP_COUNT

句法



插图描述bitmap_count.eps

目的

BITMAP_COUNT是一个标量函数，它返回输入位图的1位计数。

参数expr的类型为BLOB 。

它返回一个NUMBER表示在其输入中设置的位数。

如果expr为NULL，则返回0。

限制

参数必须是BLOB类型。该参数被预期是通过产生一个位图BITMAP_CONSTRUCT_AGG或递归，通过BITMAP_OR_AGG 。 ORA-62578任何其他输入结果：
62578,00000，“输入不是由BITMAP COUNT DISTINCT相关运算符生成的有效位图。” // *原因：尝试传递一个不是由BITMAP COUNT DISTINCT运算符生成的位图。// *操作：仅将通过BITMAP_CONSTRUCT_AGG或BITMAP_OR_AGG构造的位图传递给BITMAP COUNT DISTINCT相关运算符。
