创建和使用GNS客户端数据文件

生成GNS客户端数据文件，并将该文件复制到运行Oracle Grid Infrastructure安装的GNS客户端集群成员节点。

在GNS服务器集群成员上，运行以下命令，其中path_to_file是您创建的GNS客户端数据文件的名称和路径位置：

srvctl export gns -clientdata path_to_file -role client

例如：

$ srvctl export gns -clientdata / home / grid / gns_client_data -role client

将GNS客户端数据文件复制到运行GNS客户端群集安装的GNS客户端节点上的安全路径。Oracle安装用户必须具有访问该文件的权限。Oracle建议不授予其他用户访问GNS客户端数据文件的权限。在安装过程中，系统会提示您提供该文件的路径。

srvctl add gns -clientdata path_to_file

例如：

$ srvctl添加gns -clientdata / home / grid / gns_client_data
