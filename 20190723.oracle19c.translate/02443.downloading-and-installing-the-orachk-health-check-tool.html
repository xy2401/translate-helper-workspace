更新Oracle ORAchk运行状况检查工具

更新现有版本的Oracle ORAchk实用程序，以对Oracle软件堆栈执行主动健康检查。

Oracle ORAchk取代了RACCheck实用程序。Oracle ORAchk将运行状况检查覆盖范围扩展到整个Oracle软件堆栈，并识别和解决Oracle用户报告的主要问题。Oracle ORAchk主动扫描Oracle产品和部署的已知问题，包括：

  • 独立的Oracle数据库

  • Oracle Grid Infrastructure

  • Oracle Real Application Clusters

  • 最高可用性架构（MAA）验证

  • 升级准备验证

  • Oracle GoldenGate

Oracle正在继续根据客户要求扩展检查。

由于Oracle ORAchk已经在Python中重写，因此不再需要在Microsoft Windows上使用Cygwin。

Oracle ORAchk需要Python 3.6软件和库。运行时提供了确切需要哪些库的详细信息。

Oracle ORAchk预先安装在$ORACLE_HOME/suptools/orachk目录中的Oracle数据库中。Oracle建议您更新到最新版本的Oracle ORAchk。有关更多信息，请参见Oracle ORAchk和EXAchk用户指南 。

您还可以从My Oracle Support下载并运行最新的独立版Oracle ORAchk。有关下载，配置和运行Oracle ORAchk实用程序的信息，请参阅My Oracle Support说明1268927.2：

https://support.oracle.com/epmos/faces/DocContentDisplay?id=1268927.2&parent=DOCUMENTATION&sourceId=USERGUIDE
