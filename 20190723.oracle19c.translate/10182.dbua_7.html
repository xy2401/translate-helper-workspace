“配置管理”页面为您提供了两个选项，您可以选择向Enterprise Manager注册：1）配置企业管理器（EM）数据库Express。2）注册Enterprise Manager Cloud Control。

如果选择使用EM Database Express进行配置，则此选择下方会有一个字段，您可以在其中输入EM Database Express端口号，并选中一个复选框，以便将EM Database Express端口配置为全局端口。

如果选择向Enterprise Manager Cloud Control注册，则可以使用输入字段为CMS主机，CMS端口，企业管理器管理员用户名，企业管理器管理员密码，DBSNMP用户密码和ASMSNMP用户密码。这些输入字段在下文中详细描述。

帮助，后退，下一步和取消按钮位于页面底部。
