使用Oracle ASM的网络文件系统的准则

在Oracle Grid Infrastructure安装期间，您可以选择在网络文件系统上配置Oracle ASM。

要在网络文件系统（NFS），Oracle Clusterware和Oracle RAC上配置Oracle ASM，文件系统必须符合以下要求：

  • 要使用NFS文件系统，它必须位于支持的NAS设备上。通过以下URL登录My Oracle Support，然后单击“认证”以查找有关支持的NAS设备的最新信息：

    https://support.oracle.com/

  • 用于执行安装的用户帐户（ oracle或grid ）必须具有写入权限才能在指定的路径中创建文件。

注意：

服务器和存储供应商都必须支持所有存储产品。
