5.2 CREATE_MODEL过程

DBMS_DATA_MINING包中的CREATE_MODEL过程使用指定的数据创建具有指定名称和挖掘函数的挖掘模型。可以使用配置设置和用户指定的转换创建模型。 

PROCEDURE CREATE_MODEL（model_name IN VARCHAR2，mining_function IN VARCHAR2，data_table_name IN VARCHAR2，case_id_column_name IN VARCHAR2，target_column_name IN VARCHAR2 DEFAULT NULL，settings_table_name IN VARCHAR2 DEFAULT NULL，data_schema_name IN VARCHAR2 DEFAULT NULL，settings_schema_name IN VARCHAR2 DEFAULT NULL，xform_list IN TRANSFORM_LIST DEFAULT NULL）;

5.2.1选择挖掘函数

解释为CREATE_MODEL提供挖掘功能。

挖掘函数是CREATE_MODEL过程的必需参数。数据挖掘功能指定了可以建模和解决的一类问题。

数据挖掘功能实现有监督或无监督学习。监督学习使用一组独立属性来预测依赖属性或目标的值。无监督学习不区分依赖属性和独立属性。监督功能是预测性的。无监督功能是描述性的。

注意：

在数据中挖掘术语， 函数是通过给定的数据挖掘方法要解决的一般类型的问题。在SQL语言术语中， 函数是返回值的运算符。

在Oracle Data Mining文档中，术语函数或挖掘函数是指数据挖掘函数;术语SQL函数或SQL数据挖掘函数是指SQL函数评分（应用数据挖掘模型）。

分类

分类是预测挖掘功能。一个分类模型使用历史数据来预测分类目标。

分类模型可以使用朴素贝叶斯， 神经网络， 决策树， Logistic回归， 随机森林，支持向量机，或显式语义分析。默认为朴素贝叶斯。

分类功能也可用于异常检测。在这种情况下，使用具有空目标的SVM算法（ 一类SVM）。

聚类

聚类是一种描述性挖掘功能。聚类模型识别数据集内的自然分组。

聚类模型可以使用 K均值， O-Cluster，或期望最大化。默认值为k -Means。

特征提取

特征提取是一种描述性挖掘功能。一个特征提取模型创建一组优化属性。

特征提取模型可以使用非负矩阵分解，奇异值分解（也可用于主成分分析）或显式语义分析。默认值为非负矩阵分解。

回归

回归是一种预测性挖掘函数。一个回归模型使用历史数据来预测数值目标。

回归模型可以使用支持向量机或线性回归。默认为支持向量机。

时间序列

时间序列是预测挖掘函数。时间序列模型预测在用户指定的时间窗口上按时间排序的一系列历史数值数据的未来值。时间序列模型使用指数平滑算法。默认值为指数平滑。

5.2.3提供转换

您可以选择将xform_list参数中的构建数据转换为CREATE_MODEL 。转换指令嵌入在模型中，并在模型应用于新数据时重新应用。

5.2.3.1创建转换列表

以下是创建转换列表的方法：

  • DBMS_DATA_MINING_TRANSFORM的STACK接口。

    STACK接口提供一组预定义的转换，您可以将这些转换应用于属性或属性组。例如，您可以为所有分类属性指定受监督的分箱。

  • 该SET_TRANSFORM手续DBMS_DATA_MINING_TRANSFORM 。

    SET_TRANSFORM过程将指定的SQL表达式应用于指定的属性。例如，以下语句将country_id的转换指令附加到名为my_xforms的转换列表中。在算法处理开始之前，转换指令将country_id除以10。反向转换将country_id乘以10。 

    dbms_data_mining_transform。SET_TRANSFORM（my_xforms，'country_id'，NULL，'country_id / 10'，'country_id * 10'）;

    反向转换应用于模型详细信息中。如果country_id是受监督模型的目标，则反向变换也将应用于评分目标。

5.2.3.2转换列表和自动数据准备

了解转换列表和自动数据准备（ADP）之间的交互。

CREATE_MODEL的转换列表参数与控制的PREP_AUTO设置进行交互 ADP：

  • 启用ADP并指定转换列表后，您的转换将应用自动转换并嵌入到模型中。您指定的转换将在自动转换之前执行。

  • 当ADP关闭并指定转换列表时，您的转换将应用并嵌入到模型中，但不会执行系统生成的转换。

  • 启用ADP并且未指定转换列表时，将应用系统生成的转换并将其嵌入到模型中。

  • 当ADP关闭且您未指定转换列表时，模型中不会嵌入转换;您必须单独准备用于构建，测试和评分模型的数据集。
