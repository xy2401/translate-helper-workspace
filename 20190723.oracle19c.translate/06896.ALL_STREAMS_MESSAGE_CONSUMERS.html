3.89 ALL_STREAMS_MESSAGE_CONSUMERS

ALL_STREAMS_MESSAGE_CONSUMERS显示有关当前用户可访问的复制邮件客户端的信息。

相关视图

DBA_STREAMS_MESSAGE_CONSUMERS显示有关数据库中所有Replication消息传递客户端的信息。

柱 数据类型 空值 描述

STREAMS_NAME

VARCHAR2（128）

不是NULL

消息传递客户端的名称

QUEUE_NAME

VARCHAR2（128）

不是NULL

队列名称

QUEUE_OWNER

VARCHAR2（128）

不是NULL

队列的所有者

RULE_SET_NAME

VARCHAR2（128）

正规则集的名称

RULE_SET_OWNER

VARCHAR2（128）

积极规则集的所有者

NEGATIVE_RULE_SET_NAME

VARCHAR2（128）

否定规则集的名称

NEGATIVE_RULE_SET_OWNER

VARCHAR2（128）

负面规则集的所有者

NOTIFICATION_TYPE

VARCHAR2（9）

通知操作的类型：

  • 程序

  • 邮件

  • HTTP

NOTIFICATION_ACTION

VARCHAR2（256）

通知行动

NOTIFICATION_CONTEXT

ANYDATA

通知操作的上下文
