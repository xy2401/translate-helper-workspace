升级Oracle Grid Infrastructure后转换为Oracle Extended Cluster

升级Oracle Grid Infrastructure后，请查看此信息以转换为Oracle Extended Cluster。Oracle Extended Cluster使您可以在群集上部署Oracle RAC数据库，其中某些节点位于不同的站点中。

确保已按照本章所述升级到Oracle Grid Infrastructure 19c。
  1. 以root用户身份登录第一个节点，然后运行命令：
    rootcrs.sh -converttoextended -first -sites list_of_sites -site node_site

    list_of_sites是扩展集群中以逗号分隔的站点列表， node_site是包含该站点的节点。

    例如：

    rootcrs.sh -converttoextended -first -sites newyork，newjersey，conn -site newyork
  2. 以root用户身份在所有其他节点上运行以下命令：
    rootcrs.sh -converttoextended -site node_site

    node_site是包含站点的节点。

    例如：

    rootcrs.sh -converttoextended -site newjersey
  3. 迁移关联的节点和存储后删除默认站点。
    crsctl删除集群站点site_name

    例如：

    [root @ node4] #crsctl删除集群站点mycluster
