Real Application Security管理员和开发人员指南
HTML PDF
描述如何在数据库上实现Real Application Security。Real Application Security是Oracle Database 12c中的一项新功能。Real Application Security是一种数据库授权模型，可为多层应用程序提供端到端安全性。它提供了一个保护数据库和应用程序用户社区的集成解决方案。此外，它还推进了Oracle数据库的安全架构，以满足为Internet开发的应用程序的现有和新兴需求。
