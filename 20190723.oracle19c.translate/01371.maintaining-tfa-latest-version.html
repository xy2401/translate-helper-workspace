17维护Oracle Trace File Analyzer到最新版本

Oracle大约每三个月发布一个新版本的Oracle Trace File Analyzer。

应用标准版本更新修订（RUR）会自动更新Oracle跟踪文件分析器。但是，发布更新修订（RUR）不包含其余的Oracle数据库支持工具包更新。从My Oracle Support说明1513912.1下载最新版本的Oracle跟踪文件分析器和Oracle数据库支持工具包。

升级与首次安装类似。以root installTFA platform使用installTFA platform脚本。如果已安装Oracle跟踪文件分析器，则安装程序将更新现有安装。已安装时，群集升级不需要SSH。群集升级使用主机之间的现有守护程序安全套接字通信。 

$ ./installTFA platform

如果您无法以root身份安装，请将Oracle Trace File Analyzer安装为Oracle主人。使用-extractto和–javahome选项：

$ ./installTFA platform jre_home dir -javahome jre_home
