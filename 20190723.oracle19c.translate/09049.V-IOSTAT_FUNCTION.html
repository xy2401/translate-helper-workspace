8.19 V $ IOSTAT_FUNCTION

V$IOSTAT_FUNCTION显示数据库函数（例如LGWR和DBWR）的磁盘I / O统计信息。

柱 数据类型 描述

FUNCTION_ID

数

功能ID

FUNCTION_NAME

VARCHAR2（18）

功能名称：

  • RMAN

  • DBWR

  • LGWR

  • 拱

  • XDB

  • Streams AQ

  • Data Pump

  • 复苏

  • Buffer Cache Reads

  • Direct Reads

  • Direct Writes

  • Smart Scan

  • Archive Manager

  • 其他

SMALL_READ_MEGABYTES

数

通过单块读取请求读取的兆字节数

SMALL_WRITE_MEGABYTES

数

通过单块写入请求写入的兆字节数

LARGE_READ_MEGABYTES

数

通过多块读取请求读取的兆字节数

LARGE_WRITE_MEGABYTES

数

通过多块写入请求写入的兆字节数

SMALL_READ_REQS

数

单块读取请求的数量

SMALL_WRITE_REQS

数

单个块写入请求的数量

LARGE_READ_REQS

数

多块读取请求的数量

LARGE_WRITE_REQS

数

多块写入请求的数量

NUMBER_OF_WAITS

数

按功能划分的同步I / O等待数

等待时间

数

总同步I / O等待时间（以毫秒为单位）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
