锁定和解锁用户帐户

要暂时拒绝访问特定用户帐户的数据库，可以锁定用户帐户。如果用户然后尝试连接，则数据库显示错误消息并且不允许连接。如果要为该用户再次允许数据库访问，可以解锁用户帐户。您可以使用Oracle Enterprise Manager Database Express（EM Express）来锁定和解锁用户帐户。

要锁定或解锁用户帐户：

  1. 在EM Express中，转到“用户”页面，如“查看用户帐户”主题中所述。

  2. 单击所需的用户帐户。

  3. 从“ 操作”菜单中，选择“ 更改帐户” 。

    将出现“更改帐户”页面。

  4. 执行以下操作之一：

    • 要锁定帐户，请启用“ 帐户锁定”选项，然后单击“ 确定” 。

    • 要解锁帐户，请禁用“ 帐户锁定”选项，然后单击“ 确定” 。
