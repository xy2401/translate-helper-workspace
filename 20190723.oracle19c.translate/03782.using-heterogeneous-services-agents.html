4使用异构服务代理

本章介绍如何使用异构服务（HS）代理。有关安装和配置代理程序的信息，请参阅Oracle数据库网关安装和配置指南。

  • 初始化参数
    使用初始化参数配置网关。
  • 使用批量提取优化数据传输
    您可以使用批量提取来优化数据传输。
  • 使用并行负载优化数据负载
    DBMS_HS_PARALLEL PL / SQL包在访问异构目标时启用并行处理。
  • 注册代理商
    注册是Oracle将代理信息存储在数据字典中的操作。
  • Oracle数据库服务器SQL构造处理
    当需要翻译或后处理语句时，异构服务和网关会重写SQL语句。
  • 在非Oracle数据库上执行用户定义的函数
    您可以在远程非Oracle数据库中执行用户定义的函数。
  • 使用同义词提供数据位置和网络透明度
    您可以使用Oracle数据库服务器的同义词功能提供完整的数据位置透明性和网络透明性。
  • 将数据从Oracle数据库服务器复制到非Oracle数据库系统
    异构服务支持回调链接。
  • 将数据从非Oracle数据库系统复制到Oracle数据库服务器
    CREATE TABLE语句允许您将数据从非Oracle数据库复制到Oracle数据库。
  • 异构服务数据字典视图
    您可以使用异构服务数据字典视图来访问有关异构服务的信息。
  • 异构服务动态性能视图
    您可以使用动态性能视图来访问Oracle数据库服务器中的信息。
