12.47商店

句法

STORE SET file_name [。 EXT] [CRE [ATE | REP [LACE] |附加]]

保存脚本中当前SQL * Plus环境的属性。

条款

有关STORE命令语法中其他术语和子句的信息，请参阅SAVE 。 

组

保存系统变量的值。

用法

此命令创建一个脚本，该脚本可以使用START ， @（at sign）或@@（sign at sign）命令执行。

如果要以与STORE命令子句相同的名称（即CREATE，REPLACE或APPEND）存储文件，则必须将名称放在单引号中或指定文件延期。

例子

要将当前SQL * Plus系统变量存储在名为DEFAULTENV的文件中，并使用默认的命令文件扩展名，请输入

商店设置DEFAULTENV

要将当前SQL * Plus系统变量附加到名为DEFAULTENV且扩展名为OLD的现有文件，请输入

STORE SET DEFAULTENV.OLD APPEND
