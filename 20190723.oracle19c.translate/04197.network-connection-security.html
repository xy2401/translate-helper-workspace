11.1网络连接安全性

网络安全的两个主要方面是身份验证和数据机密性。身份验证和数据机密性的类型取决于您通过Oracle Net或Java数据库连接（JDBC）连接连接数据库的方式。下表提供了Oracle Net和JDBC连接的安全性说明：

连接安全 描述

Oracle Net

在允许用户连接之前，数据库可能需要身份验证和授权。Oracle Net数据库连接安全性可能需要以下一项或多项：

  • 用于客户端验证的用户名和密码。对于每个连接请求，必须提供在Oracle Net中配置的用户名和密码。

  • 高级网络选项，用于加密，kerberos或secureId。

  • SSL用于证书身份验证。

J DBC

所需的JDBC连接安全性类似于Oracle Net数据库连接所需的约束。
