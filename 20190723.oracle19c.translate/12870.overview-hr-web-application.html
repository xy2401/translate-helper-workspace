3 HR Web应用程序概述

人力资源Web应用程序旨在让您访问与AnyCo Corporation的所有员工相关的信息。

能够访问此应用程序的两种类型的用户是：

  • HRStaff

  • HRAdmin

HRStaff和HRAdmin帐户具有不同的权限。

HRStaff具有对应用程序的只读访问权限，并且没有更新/删除员工记录的权限。HRStaff只能列出员工和按员工ID搜索。

HRAdmin可以完全控制应用程序并具有读写权限。HRAdmin是唯一可以访问应用程序所有功能的用户，例如更新/删除员工记录 ，或为所有员工提供薪资增量 。

  • 按员工ID搜索

    使用primary key （即员工ID）搜索特定员工。
