13.17.1听众（）

Listener类构造函数。

句法 描述 
监听器（const Connection * conn）;

创建一个Listener对象。 

监听器（const Connection * conn vector <Agent>＆aglist，int waitTime = 0）;

创建一个Listener对象，并代表它在队列中侦听的Agent列表。还设定等待时间;默认：没有等待。

参数 描述 
康恩

新Listener对象的连接。 

aglist

Listener对象代表队列等待的代理列表;这个Listener客户。 

等待时间

在队列中等待客户感兴趣的消息的时间;很快。

13.17.4 listen（）

在前一个setTimeOutForListen（）调用指定的时间内，代表指定的Agent s侦听Message s。返回有Message的Agent 。

请注意，这是一个阻止调用。在此呼叫之前，请完成以下步骤：

  • 通过setAgentList（）调用注册每个代理侦听器。

  • 对setTimeOutForListen（）进行阻塞调用，该调用在列表中某个Agent的Message到达时返回。如果在等待时间到期之前没有Message到达，则呼叫返回错误。

句法

代理听（）;
