F Fleet Patching and Provisioning Control（RHPCTL）命令参考

使用Fleet Patching and Provisioning Control（RHPCTL）实用程序来管理群集中的Fleet Patching和Provisioning。

本附录包含Fleet Patching和Provisioning命令的参考信息，包括实用程序使用信息和RHPCTL命令的完整列表。

RHPCTL概述

RHPCTL是一个命令行实用程序，您可以使用该实用程序执行Fleet Patching和Provisioning操作以及管理Fleet Patching和Provisioning Server以及Clients。RHPCTL使用以下语法：

rhpctl 命令 对象 [ 参数 ]

在RHPCTL语法中：

  • command是一个动词，例如add ， delete或query

  • object （也称为名词）是RHPCTL执行命令的目标或对象，例如客户端或图像。

  • parameters扩展了前面命令组合的使用，以包括命令的附加参数。将参数指定为-keyword value 。如果value字段包含以逗号分隔的列表，则不要在列表中的项之间使用空格。

您可以使用RHPCTL命令执行多个Fleet Patching和Provisioning操作，包括：

  • Fleet Patching和Provisioning Client操作，例如创建Fleet Patching和Provisioning Client配置。

  • 角色操作，例如添加和删除角色，以及为用户授予和撤消角色。

  • 站点操作，例如获取Fleet Patching和Provisioning Server的配置信息。

  • 图像操作，例如添加，删除和导入图像。

  • 图像系列操作，例如添加和删除图像系列。

  • 工作副本操作，例如添加和删除工作副本。

使用RHPCTL帮助

为所有人寻求帮助 RHPCTL命令，从命令行输入：

rhpctl -help

要查看命令语法和每个RHPCTL命令的参数列表，请从命令行输入：

rhpctl 命令 （或动词 ） 对象 （或名词 ）-help

rhpctl删除审计

句法

rhpctl删除审计[-to timestamp]

使用说明

（可选）您可以使用YYYY-MM-DD格式指定要删除审核记录的日期。否则，此命令将删除所有审核记录。

句法

句法

rhpctl add client -client cluster_name [-toclientdata path ] [-targetnode node_name {-sudouser sudo_user_name -sudopath sudo_binary_location | -root | -cred cred_name } | -auth plugin_name [-arg1 name1 ： value1 ...]][-maproles role = user_name [， role = user_name [，...]]][ version ]

参数

参数 描述
客户端client_name

指定要在其中创建客户端的群集的名称。
-toclientdata path

（可选）您可以指定由Fleet Patching and Provisioning Server（特定于客户端群集）创建的XML文件的路径，其中包含客户端配置其与服务器的连接所需的信息。

-maproles role = user_name [，...]

您可以指定已定义的内置角色或角色，也可以为每个角色分配多个用途。使用逗号分隔多个角色和用户。
- version

  • 您只能在Fleet Patching and Provisioning Server上运行此命令。

例子

要将客户端添加到Fleet Patching和Provisioning Server：

$ rhpctl add client -client ClientCluster3 -toclientdata Grid_home / RHPserver / info

rhpctl删除客户端

从配置中删除特定的Fleet Patching和Provisioning Client。

句法

rhpctl delete client -client cluster_name [-force]

使用说明

  • 指定要从配置中删除的客户端群集的名称。

  • 在运行此命令或使用-force选项之前，必须停止Fleet Patching and Provisioning Client。

例

要删除Fleet Patching和Provisioning Client ClientCluster3 ：
$ rhpctl delete client -client ClientCluster3

参数

表F-6 rhpctl update client命令参数

-targetnode node_name
