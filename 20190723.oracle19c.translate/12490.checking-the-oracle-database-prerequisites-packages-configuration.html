检查Oracle数据库先决条件包安装

使用此过程可收集有关Oracle数据库先决条件组程序包配置的信息。

  1. 要检查是否已安装oracle-rdbms-server-18c-preinstall ：
    $ pkg list oracle-rdbms-server-18c-preinstall
  2. 要检查最新版本的oracle-rdbms-server-18c-preinstall ：
    $ pkg list -n oracle-rdbms-server-18c-preinstall
  3. 在安装oracle-rdbms-server-18c-preinstall ：
    1. 使用-n选项检查错误：
      $ pkg install -nv oracle-rdbms-server-18c-preinstall

      注意：

      使用-n选项检查安装错误。如果-n未显示任何错误，则在安装oracle-rdbms-server-18c-preinstall时省略-n选项。
    2. 如果没有错误，则以root身份登录，然后安装组包：
      #pkg install oracle-rdbms-server-18c-preinstall
  4. 要查看oracle-rdbms-server-18c-preinstall安装的软件包：
    $ pkg contents -ro type，fmri -t依赖oracle-rdbms-server-18c-preinstall

    此命令的示例输出：

    TYPE FMRI group x11 / diagnostic / x11-info-clients group x11 / library / libxi group x11 / library / libxtst group x11 / session / xauth require compress / unzip require developer / assembler require developer / build / make
