4.3 completedTimeSpec

目的

使用completedTimeSpec子句指定备份或副本何时完成。

使用说明

所有日期字符串必须是：

  • 根据当前有效的全球技术日期格式规范进行格式化。

  • 由返回DATE值的SQL表达式创建，如以下示例所示：

    • ' SYSDATE-30 '

    • TO_DATE（'09 / 30/2013 08:00:00'，'MM / DD / YYYY HH24：MI：SS'）

    TO_DATE函数指定独立于当前Global Technology环境变量设置的日期。

句法

completedTimeSpec :: =


插图说明completedtimespec.eps

语义

语法元素 描述

AFTER ' date_string '

指定备份完成之前的时间（请参阅示例4-7 ）。

BEFORE ' date_string ' BEFORE

指定备份完成之前的时间（请参见示例4-9 ）。

BETWEEN ' date_string ' AND ' date_string '

指定备份完成的时间范围（请参阅示例4-8 ）。BETWEEN ' date1 ' AND ' date2 '等于AFTER ' date1 ' BEFORE ' date2 ' 。
