Oracle Internet Directory自助服务控制台

Oracle Internet Directory自助服务控制台是一种基于Delegated Administration Services的工具。这是一个自助服务应用程序，允许管理访问目录中管理的应用程序数据。此工具随时可用于Oracle Internet Directory。

“ 委派管理的Oracle身份管理指南”讨论了委派管理服务和Oracle Internet Directory自助服务控制台工具。

Oracle Net Configuration Assistant

Oracle Net Configuration Assistant是一个基于向导的工具，具有图形用户界面。其主要用途是配置基本的Oracle Net网络组件，例如侦听器名称和协议地址，以及配置Oracle主目录服务器的用法。后一种用途是使该工具对配置企业用户安全性很重要的原因。

如果使用域名系统（DNS）发现（自动域名查找）在网络上查找Oracle Internet Directory，则无需此助手。请注意，建议使用DNS发现。

在使用目录注册数据库之前，必须执行以下两个任务之一：

  • 在网络上配置Oracle Internet Directory的DNS发现。

    也可以看看：

    有关DNS服务器发现的信息，请参阅Oracle Internet Middle Directory的Oracle融合中间件管理员指南

  • 如果未在网络上配置DNS发现，请使用Oracle Net Configuration Assistant为Oracle主目录创建ldap.ora文件。

您的数据库最初使用ldap.ora文件在您的网络上找到正确的Oracle Internet Directory服务器。此配置文件包含目录服务器的主机名，端口号和身份管理领域信息。

数据库注册完成后，通过存储在数据库钱包中的数据库DN确定域。

Oracle企业管理器

企业用户安全使用Oracle企业管理器来管理企业用户，管理组， 企业域 S和企业的作用 Š存储在Oracle互联网目录。您可以使用Oracle Enterprise Manager提供的基于Web的用户界面来管理企业用户安全性。

企业用户是在LDAP兼容目录（例如Oracle Internet Directory）中集中配置和管理的用户，用于数据库访问。企业域是包含数据库，企业角色（分配给企业用户的访问权限）和代理权限（使企业用户能够像其他用户一样连接到数据库）的目录构造。

用户迁移工具

用户迁移实用程序是一个命令行工具，使您可以将数据库用户批量迁移到Oracle Internet Directory，并将其作为企业用户进行存储和管理。此工具分两个阶段执行批量迁移：在第一阶段，它使用数据库用户信息填充表。在第二阶段，数据库用户信息将迁移到该目录。

安装Oracle数据库客户端时，此工具将自动安装在以下位置：

$ORACLE_HOME / rdbms / bin / umu

该实用程序的基本语法如下：

umu parameter_keyword_1 = value1：value2
parameter_keyword_2 = value parameter_keyword_3 = value1：value2：value3 ...parameter_keyword_n =值

请注意，当参数采用多个值时，它们用冒号（:)分隔。
