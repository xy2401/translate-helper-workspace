1.150 INSTANCE_MODE

INSTANCE_MODE指示实例是读写，只读还是大部分读取。

属性 描述

参数类型

串

句法

INSTANCE_MODE = {READ-WRITE |只读| READ-MOSTLY}

默认值

读写

可修改

没有

基本

没有

Oracle RAC

请参阅下面的初始化参数说明中的Oracle RAC限制。

READ-WRITE实例是常规Oracle实例。

READ-ONLY实例是一个只能以只读模式打开的Oracle实例。

READ-MOSTLY实例是一个执行极少数据库写入的Oracle实例。

在不同的Oracle Real Application Clusters（Oracle RAC）实例上设置参数时，这些限制适用：

  1. 在任何给定时间， INSTANCE_MODE设置为READ-WRITE ，必须至少有一个实例处于活动状态。如果INSTANCE_MODE设置为READ-WRITE的唯一实例关闭，则Oracle将自动关闭所有其他实例。

  2. 将INSTANCE_MODE设置为READ-ONLY的实例不能是在Oracle RAC集群中的所有实例之间打开数据库或任何可插拔数据库（PDB）的第一个实例。

  3. Oracle不支持将INSTANCE_MODE设置为READ-ONLY的实例与在同一Oracle RAC集群中将INSTANCE_MODE设置为READ-MOSTLY的实例共存。
