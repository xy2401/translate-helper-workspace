设置Oracle用户环境变量

在每个节点上的安装所有者用户配置文件中，设置环境变量ORACLE_BASE和ORACLE_HOME以及ORACLE_SID ;还要将ORACLE_HOME/bin添加到路径环境变量中。

如下所示，设置环境变量，如下例所示：

export ORACLE_BASE = / u01 / app / oracle export ORACLE_HOME = / u01 / app / oracle / product / 19.0.0 / dbhome_1 export PATH = $ PATH：$ ORACLE_HOME / bin export ORACLE_SID = sales1

如果未设置环境变量ORACLE_HOME和ORACLE_SID ，并且您尝试使用SQL * Plus或其他工具，则会收到一条错误消息，要求您设置这些变量。
