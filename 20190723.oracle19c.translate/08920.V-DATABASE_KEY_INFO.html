7.159 V $ DATABASE_KEY_INFO

V$DATABASE_KEY_INFO提供用于加密数据块的默认数据库密钥的信息。当此类数据与加密表空间或加密列具有相关性时，Oracle使用数据库密钥加密SYSTEM，UNDO和TEMP表空间中的敏感信息

V$DATABASE_KEY_INFO反映存储在控制文件中的数据库密钥信息。通常它与系统表空间中记录的系统表空间键相同。如果系统表空间已加密，它也将显示在V$ENCRYPTED_TABLESPACES 。如果系统表空间未加密，则这是提供此信息的唯一视图。

柱 数据类型 描述

ENCRYPTIONALG

VARCHAR2（7）

加密演算法：

  • 没有

  • 3DES168

  • AES128

  • AES192

  • AES256

AES128是默认的加密算法。

ENCRYPTEDKEY

RAW（48）

数据库密钥的加密版本

MASTERKEYID

RAW（16）

用于加密数据库密钥的主密钥的ID

MASTER_ACTIVATED

VARCHAR2（3）

指示是否已为Oracle密钥库中的此数据库或容器设置（ YES ）或（ NO ）主密钥。换句话说，指示是否已为此数据库或容器激活透明数据加密（TDE）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
