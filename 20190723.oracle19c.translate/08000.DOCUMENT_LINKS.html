6.183 DOCUMENT_LINKS

DOCUMENT_LINKS在Oracle XML DB存储库文档中提供有关Oracle XML DB文档链接的系统信息。

当包含XLink或XInclude链接的XML文档添加到存储库时，这些链接可以映射到文档链接，使用查看DOCUMENT_LINKS进行跟踪。

柱 数据类型 空值 描述

SOURCE_ID

RAW（16）

源资源OID

TARGET_ID

RAW（16）

目标资源OID

TARGET_PATH

VARCHAR2（4000）

此列始终为NULL。它留作将来使用。

LINK_TYPE

VARCHAR2（8）

文档链接类型：硬或弱

LINK_FORM

VARCHAR2（8）

原始链接是XLink还是XInclude形式

SOURCE_TYPE

VARCHAR2（17）

链接是否包含在“资源内容”或“资源元数据”中
