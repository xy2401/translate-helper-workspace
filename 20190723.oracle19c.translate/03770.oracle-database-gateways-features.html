3 Oracle数据库网关的功能

本章介绍Oracle数据库网关提供的主要功能。

注意：

并非所有异构服务网关都提供这些功能。不仅必须对这些功能进行通用支持，而异构服务提供这些功能，还必须为驱动程序添加支持。请参阅相应的网关文档，以确定您的网关是否支持特定功能。

  • SQL和PL / SQL支持
    转换SQL语句并根据功能映射数据类型。PL / SQL调用映射到非Oracle系统存储过程。
  • 异构复制
    可以使用物化视图在非Oracle系统和Oracle数据库之间复制数据。
  • Passthrough SQL
    直通SQL功能使您可以直接将语句发送到非Oracle系统，而无需先由Oracle数据库进行解释。
  • 结果集支持
    各种关系数据库使存储过程能够返回结果集（一组或多组行）。
  • 数据字典翻译
    数据字典是有关由系统的各个用户创建的数据库对象的信息的集合。
  • 日期时间数据类型
    异构服务通用代码支持SQL和存储过程中的Oracle日期时间数据类型。异构服务不支持数据字典转换中的这些数据类型或涉及数据字典转换的查询。
  • 两阶段提交协议
    异构服务提供了实现两阶段提交协议的基础结构。
  • 分段LONG数据类型
    现在，网关在代理和Oracle数据库之间分段管理数据，从而消除了大量内存和网络带宽需求。
  • SQL * Plus DESCRIBE命令
    您可以使用SQL * Plus DESCRIBE命令描述非Oracle系统对象。
  • 分布式环境中SQL的约束
    本节介绍分布式环境中SQL的一些约束。
  • Oracle的优化程序和异构服务
    Oracle的优化器可以与异构服务一起使用。
