1.225 OPTIMIZER_ADAPTIVE_REPORTING_ONLY

OPTIMIZER_ADAPTIVE_REPORTING_ONLY控制自适应优化的仅报告模式。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

TRUE |假

基本

没有

当OPTIMIZER_ADAPTIVE_REPORTING_ONLY设置为FALSE ，仅报告模式关闭，并且照常启用自适应优化。

当OPTIMIZER_ADAPTIVE_REPORTING_ONLY设置为TRUE ，自适应优化将以仅报告模式运行。使用此设置，将收集自适应优化所需的信息，但不会采取任何操作来更改计划。例如，自适应计划将始终选择默认（优化程序选择）计划，但会收集有关在非报告模式下适应的计划的信息。可以在自适应计划报告中查看此信息。

此参数仅影响已启用的自适应优化。
