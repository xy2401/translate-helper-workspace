1.256 PDB_FILE_NAME_CONVERT

PDB_FILE_NAME_CONVERT在处理CREATE PLUGGABLE DATABASE语句时以及在处理CREATE DATABASE语句的ENABLE PLUGGABLE DATABASE子句时，如果未指定file_name_convert_clause且未启用Oracle Managed Files，则将现有文件的名称映射到新文件名。

属性 描述

参数类型

串

句法

PDB_FILE_NAME_CONVERT = ' string1 ' , ' string2 ' , ' string4 ' , ... ' , ' string3 ' , ...

哪里：

  • string1是现有文件名的模式

  • string2是新文件名的模式

  • string3是现有文件名的模式

  • string4是新文件名的模式

您可以根据需要使用尽可能多的现有和新替换字符串对。您可以使用单引号或双引号。

以下是可接受的示例设置：

PDB_FILE_NAME_CONVERT ='/ dbs / t1 /'，'/ dbs / t1 / s _'，'dbs / t2 /'，'dbs / t2 / s_'

默认值

没有默认值

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

Oracle RAC

您必须为每个实例设置此参数，并且多个实例必须具有相同的值。

此初始化参数中指定的文件名模式不能与Oracle Managed Files管理的文件或目录匹配。
