Oracle Grid Infrastructure升级选项

升级到Oracle Grid Infrastructure 19c时，升级到Oracle Flex Cluster配置。

此版本的Oracle Grid Infrastructure支持的升级路径包括：

  • Oracle Grid Infrastructure从版本11.2.0.3和11.2.0.4升级到Oracle Grid Infrastructure 19c。

  • Oracle Grid Infrastructure从版本12c第1版（12.1）升级到Oracle Grid Infrastructure 19c。

  • Oracle Grid Infrastructure从版本12c第2版（12.2）升级到Oracle Grid Infrastructure 19c。

  • Oracle Grid Infrastructure从版本18c升级到Oracle Grid Infrastructure 19c。

Oracle Grid Infrastructure 11g，Oracle Grid Infrastructure 12c第1版（12.1），Oracle Grid Infrastructure 12c第2版（12.2）和Oracle Grid Infrastructure 18c到Oracle Grid Infrastructure 19c的升级选项包括：

  • Oracle Grid Infrastructure滚动升级涉及升级单个节点，而不会在群集中的其他节点上停止Oracle Grid Infrastructure

  • 通过降低集群并升级整个集群，Oracle Grid Infrastructure进行非滚动升级

注意：

  • 升级到Oracle Grid Infrastructure 19c时，升级到Oracle独立群集配置。

  • 您可以使用Oracle ASM或共享文件系统在Oracle Standalone Cluster部署中存储OCR和投票文件。如果OCR和表决文件的存储不是其他集群类型上的Oracle ASM，则需要在升级到Oracle Grid Infrastructure 19c之前将OCR和表决文件迁移到Oracle ASM。
