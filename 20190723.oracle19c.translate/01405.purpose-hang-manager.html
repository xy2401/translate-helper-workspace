12解决数据库和数据库实例挂起

Hang Manager通过解决挂起并保持资源可用来保留数据库性能。

  • Hang Manager Architecture
    Hang Manager在数据库中自动作为DIA0任务运行。
  • Hang Manager的可选配置
    您可以调整灵敏度，并控制Hang Manager使用的日志文件的大小和数量。
  • 挂起管理器诊断和日志记录
    Hang Manager自动解决挂起并不断在数据库警报日志中记录解决方案，并在跟踪文件中记录诊断。
