B Oracle Locator

Oracle Locator（也称为Locator）是Oracle数据库标准版的一项功能。Locator提供Oracle Spatial and Graph中提供的核心功能和服务。

它提供了支持基于Internet和无线服务的应用程序和基于合作伙伴的GIS解决方案所需的重要功能。定位器不是为需要复杂空间数据管理的地理信息系统（GIS）应用程序而设计的。如果需要线性参考，高级空间函数或空间和图形Web服务等功能，请使用Oracle Spatial and Graph而不是Locator。

与Spatial和Graph一样，Locator不是最终用户应用程序，而是应用程序开发人员的一组空间功能。

Locator适用于Oracle数据库的Standard Edition One，Standard Edition和Enterprise Edition，以及Oracle Database XE（Express Edition）。Spatial and Graph是一种定价选项，仅适用于Oracle数据库企业版。Spatial和Graph包括Locator的所有功能以及Locator不具备的其他功能。

虽然表B-2中不允许使用Oracle Locator的功能的最终列表，但您可以按照安装和卸载定位器或手动和空间和图形中的说明删除大多数这些功能。有关Spatial和Graph and Locator的许可信息 ，请参阅Oracle数据库许可信息 。

注意：

一些仅可用于Spatial和Graph的底层组件可以安装在Locator中以支持某些Locator功能。（ 表B-2列出了不允许与Oracle Locator一起使用的功能。）

Locator的安装取决于Oracle Multimedia的成功和正确安装。Oracle Multimedia随Oracle Database一起安装和配置。在安装Oracle Multimedia期间，安装了Locator。

通常，Locator包括Oracle Spatial和Graph的数据类型，运算符和索引功能，以及Spatial和Graph的一组有限的子程序（函数和过程）。定位器功能包括以下内容：

  • 描述和支持任何类型几何的对象类型（SDO_GEOMETRY）

  • 一种空间索引功能，可用于在几何数据上创建空间索引

  • 空间运算符（在空间运算符中描述），使用空间索引执行空间查询

  • 几何子程序（SDO_GEOM包，在SDO_GEOM包（几何）中描述）

  • 空间聚合函数（在空间聚合函数中描述）

  • 协调系统对几何和图层转换的支持（SDO_CS包，在SDO_CS包（坐标系转换）中描述 ）

  • 调整子程序（SDO_TUNE包，在SDO_TUNE包（调整）中描述）

  • 空间实用程序功能（SDO_UTIL包，在SDO_UTIL包（实用程序）中描述）

  • Spatial和Graph Java API的oracle.spatial.geometry （ sdoapi.jar ）和oracle.spatial.util （ sdoutil.jar ）包中的类（在Oracle Spatial and Graph Java API Reference中描述 ）

有关空间概念，SDO_GEOMETRY对象类型，对SQL多媒体类型注释文本的支持以及空间数据的索引和加载的信息，请参阅本指南中的第1章到第5章。有关Locator支持的功能的参考和用法信息，请参阅表B-1中列出的章节或章节。

B.1手动安装和卸载定位器或空间和图形

要安装空间和图形或使用特定于空间和图形但不使用定位器的任何功能，您必须满足空间和图形的许可要求。有关Spatial和Graph and Locator的许可信息 ，请参阅Oracle数据库许可信息 。

注意：

如果使用Oracle数据库安装和配置Oracle多媒体，则还会安装Oracle Locator。

脚本可用于执行与定位器和空间和图形相关的某些安装和卸载操作：

  • md dins .sql手动卸载Spatial和Graph，只留下Locator，因此只“安装”Locator。这将删除仅适用于Spatial和Graph的大多数功能。

  • md inst .sql手动安装Spatial和Graph，以便您可以使用Spatial and Graph和Locator提供的所有功能。

如果只想安装定位器功能，但已安装Spatial和Graph，请运行mddins.sql脚本，如下所示：

  1. 确保已安装Oracle Multimedia。

  2. 以具有SYSDBA权限的SYS用户身份（SYS AS SYSDBA）连接到数据库，并在出现提示时输入SYS帐户密码。

  3. 启动SQL * Plus，并输入以下语句：

    • Linux： @$ORACLE_HOME/md/admin/mddins.sql

    • Windows： @%ORACLE_HOME%\md\admin\mddins.sql

如果您需要能够使用Spatial和Graph，则可以运行md inst .sql脚本，如下所示：

  1. 以具有SYSDBA权限的SYS用户身份（SYS AS SYSDBA）连接到数据库，并在出现提示时输入SYS帐户密码。

  2. 启动SQL * Plus，并输入以下语句：

    • Linux： @$ORACLE_HOME/md/admin/mdinst.sql

    • Windows： @%ORACLE_HOME%\md\admin\mdinst.sql
