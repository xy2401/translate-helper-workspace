将修补程序应用于Oracle Grid Infrastructure

升级Oracle Grid Infrastructure 19c后，可以通过从My Oracle Support下载来安装单个软件修补程序。

  • 关于单个Oracle Grid Infrastructure补丁
    下载Oracle ASM个人（一次性）补丁，并使用OPatch Utility将其应用于Oracle Grid Infrastructure。
  • 关于Oracle Grid Infrastructure软件修补程序级别
    查看此主题以了解如何为Oracle ASM和Oracle Clusterware应用修补程序。
  • 将Oracle Grid Infrastructure修补为软件补丁级别
    从Oracle Grid Infrastructure 12c第1版（12.1）开始，可以使用名为Rolling Patch的新集群状态。就这种静默状态允许的Oracle ASM操作而言，此模式类似于现有的“滚动升级”模式。
  • 在Oracle Grid Infrastructure安装或升级期间应用修补程序
    从Oracle Grid Infrastructure 18c开始，您可以在Oracle Grid Infrastructure安装或升级期间下载并应用版本更新（RU）和一次性补丁。
