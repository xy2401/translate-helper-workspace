1打包用于C ++的OracleXml API

OracleXml是Oracle实现的所有XML C ++接口的命名空间。它包括类XmlException ，XML中所有异常的根，以及以下命名空间：

  • Ctx - TCtx相关声明的命名空间，在C ++的 Ctx API包中描述

  • Dom - 与DOM相关的声明的命名空间，在Package Dom API for C ++中描述

  • IO - 输入和输出源声明的命名空间，在C ++的包IO API中描述

  • Parser - 解析器和模式验证器声明的命名空间，在Package Parser API for C ++中描述

  • SOAP - 简单对象访问协议声明的命名空间，在Package SOAP API for C ++中描述

  • Tools - Tools :: Factory相关声明的命名空间，在C ++的Package Tools API中描述

  • XPath - 与XPath相关的声明的命名空间，在Package XPath API for C ++中描述

  • XPointer - XPointer相关声明的命名空间，在Package XPointer API for C ++中描述

  • Xsl - 与XSLT相关的声明的命名空间，在C ++的Package Xsl API中描述
