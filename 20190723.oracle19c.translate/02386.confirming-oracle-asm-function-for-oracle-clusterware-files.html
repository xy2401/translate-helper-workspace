确认Oracle Clusterware文件的Oracle ASM功能

安装Oracle Grid Infrastructure后确认Oracle ASM正在运行。

安装Oracle Grid Infrastructure后，Oracle Clusterware文件存储在Oracle ASM上。使用以下命令语法作为Oracle Grid Infrastructure安装所有者（ grid ）以确认您的Oracle ASM安装正在运行：

srvctl status asm

例如：

srvctl status asm ASM在node1，node2，node3，node4上运行

注意：

要管理Oracle ASM或Oracle Net 11g第2版（11.2）或更高版本的安装，请在Oracle Grid Infrastructure主目录中使用srvctl二进制文件作为集群（Grid home）。如果安装了Oracle Real Application Clusters或Oracle Database，则无法使用数据库主目录中的srvctl二进制文件来管理Oracle ASM或Oracle Net。
