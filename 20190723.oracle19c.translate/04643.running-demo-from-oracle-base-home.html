将演示目录复制到Oracle Base Home

在只读模式ORACLE_HOME中，必须将本主题中列出的demo目录从ORACLE_HOME复制到ORACLE_BASE_HOME。

Oracle数据库包含各种demo目录，其中包含各种示例和产品演示，您可以使用它们来了解Oracle数据库的产品，选项和功能。在只读模式ORACLE_HOME中，您不能在ORACLE_HOME中使用这些demo目录，因为在使用它们时会对这些demo目录执行写入。

默认情况下，许多演示目录不可用。您必须安装Oracle数据库示例才能查看和使用示例和产品演示。

将相应的demo目录复制到ORACLE_BASE_HOME中的相应位置。现在，您可以使用此demo目录的副本。

您必须将以下demo目录从ORACLE_HOME复制到ORACLE_BASE_HOME：

  • JDBC /演示

  • ODBC /演示

  • ORD / HTTP /演示

  • 预补偿/演示

  • RDBMS /演示

  • SQLJ /演示

  • SQLPLUS /演示

  • XDK /演示

您还必须为odbc/demo ， precomp/demo ， rdbms/demo和xdk/demo demo目录创建符号链接。请参阅本主题中的“创建符号链接”部分。

复制演示目录

例如，要将rdbms/demo目录从ORACLE_HOME复制到ORACLE_BASE_HOME，请执行以下操作：

  1. 以Oracle软件所有者用户（ oracle ）身份登录。

  2. 检查rdbms/demo目录是否已复制到ORACLE_BASE_HOME。

    $ ls -l -d $（orabasehome）/ rdbms / demo

  3. 如果尚未复制rdbms/demo目录，则创建并复制它。 

    $ mkdir -p $（orabasehome）/ rdbms $ cp -r $ ORACLE_HOME / rdbms / demo $（orabasehome）/ rdbms / demo

同样，将之前列出的所有demo目录从ORACLE_HOME复制到ORACLE_BASE_HOME。

创建符号链接

您必须为odbc/demo ， precomp/demo ， rdbms/demo和xdk/demo demo目录创建符号链接。

对于rdbms/demo ，将$ORACLE_HOME/rdbms/demo替换为该副本的符号链接。

  1. 确保符号链接尚不存在。

    $ ls -l -d $ ORACLE_HOME / rdbms / demo

  2. 如果$ORACLE_HOME/rdbms/demo仍然是原始的demo目录，请重命名它并用符号链接替换它。 

    $ cd $ ORACLE_HOME / rdbms $ mv demo demo.installed $ ln -s $（orabasehome）/ rdbms / demo $ ORACLE_HOME / rdbms / demo

对于odbc/demo ，将$ORACLE_HOME/odbc/demo替换$ORACLE_HOME/odbc/demo副本的符号链接。

  1. 确保符号链接尚不存在。

    $ ls -l -d $ ORACLE_HOME / odbc / demo

  2. 如果$ORACLE_HOME/odbc/demo仍然是原始的demo目录，请重命名它并用符号链接替换它。 

    $ cd $ ORACLE_HOME / odbc $ mv demo demo.installed $ ln -s $（orabasehome）/ odbc / demo $ ORACLE_HOME / odbc / demo

对于precomp/demo ，将$ORACLE_HOME/precomp/demo替换$ORACLE_HOME/precomp/demo副本的符号链接。

  1. 确保符号链接尚不存在。

    $ ls -l -d $ ORACLE_HOME / precomp / demo

  2. 如果$ORACLE_HOME/precomp/demo仍然是原始的demo目录，请重命名它并用符号链接替换它。 

    $ cd $ ORACLE_HOME / precomp $ mv demo demo.installed $ ln -s $（orabasehome）/ precomp / demo $ ORACLE_HOME / precomp / demo

该xdk/demo目录中，需要在一个符号链接$ORACLE_HOME/xdk/include指向$(orabasehome)/xdk/include你复制后的xdk/demo目录。

  1. 确保符号链接尚不存在：

    $ ls -l -d $ ORACLE_HOME / xdk / include

  2. 如果符号链接不存在，则运行以下命令：

    $ ln -s $ ORACLE_HOME / xdk / include $（orabasehome）/ xdk / include

注意：

在plsql/demo目录中， ncmpdemo.sql在只读模式下不可用。

复制init.ora文件

将init.ora文件从ORACLE_HOME复制到ORACLE_BASE_HOME。

  1. 以Oracle软件所有者用户（ oracle ）身份登录。

  2. 检查ORACLE_BASE_HOME中是否存在init.ora文件。

    $ ls $（orabasehome）/init.ora

    如果ORACLE_BASE_HOME中存在init.ora文件，则将此init.ora文件更新为与$ORACLE_HOME/init.ora文件同步。

  3. 如果ORACLE_BASE_HOME中不存在init.ora文件，则从ORACLE_HOME复制它。

    cp $ ORACLE_HOME / init.ora $（orabasehome）/init.ora
