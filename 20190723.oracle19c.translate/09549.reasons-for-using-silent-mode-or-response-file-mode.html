使用静默模式或响应文件模式的原因

有关在静默模式或响应文件模式下运行安装程序的用例，请查看本节。

模式 用途

无声

使用静默模式进行以下安装：

  • 完成无人参与安装，使用操作系统实用程序（如at 。

  • 在多个系统上完成几个类似的安装，无需用户交互

  • 在没有安装X Window System软件的系统上安装软件。

安装程序会在您用于启动它的终端上显示进度信息，但不会显示任何安装程序屏幕。

响应文件

使用响应文件模式在多个系统上完成类似的Oracle软件安装，为某些安装程序提示提供默认答案，但不是所有安装程序提示。
