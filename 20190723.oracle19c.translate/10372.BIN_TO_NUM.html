BIN_TO_NUM

句法


说明bin_to_num.eps的描述

目的

BIN_TO_NUM将位向量转换为其等效数。该函数的每个参数表示位向量中的一个位。此函数将任何数值数据类型或可以隐式转换为NUMBER任何非数字数据类型作为参数。每个expr必须计算为0或1。此函数返回Oracle NUMBER 。

BIN_TO_NUM在数据仓库应用程序中非常有用，可以使用分组集从物化视图中选择感兴趣的组。

例子

以下示例将二进制值转换为数字：

SELECT BIN_TO_NUM（1,0,1,0）FROM DUAL; BIN_TO_NUM（1,0,1,0）------------------- 10

下一个示例将三个值转换为单个二进制值，并使用BIN_TO_NUM将该二进制转换为数字。该示例使用PL / SQL声明来指定原始值。这些通常来自实际数据源。 

SELECT order_status FROM orders WHERE order_id = 2441; ORDER_STATUS ------------ 5 DECLARE仓库NUMBER：= 1;地面NUMBER：= 1;保险号码：= 1;结果NUMBER; BEGIN SELECT BIN_TO_NUM（仓库，地面，投保）INTO结果来自DUAL; UPDATE命令SET order_status = result WHERE order_id = 2441;结束; / PL / SQL过程成功完成。SELECT order_status FROM orders WHERE order_id = 2441; ORDER_STATUS ------------ 7
