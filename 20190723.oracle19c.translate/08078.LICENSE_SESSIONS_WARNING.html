1.164 LICENSE_SESSIONS_WARNING

LICENSE_SESSIONS_WARNING指定并发用户会话数的警告限制。

属性 描述

参数类型

整数

默认值

0

可修改

改变系统

可在PDB中修改

没有

价值范围

0到LICENSE_MAX_SESSIONS参数的值

基本

没有

Oracle RAC

多个实例可以具有不同的值。

注意：

Oracle不再按并发会话数提供许可。因此，不推荐使用LICENSE_MAX_SESSIONS和LICENSE_SESSIONS_WARNING初始化参数。

当达到LICENSE_SESSIONS_WARNING指定的此限制时，其他用户可以连接，但Oracle会在警报日志中为每个新连接写入一条消息。在达到限制后连接的具有RESTRICTED SESSION权限的用户会收到一条警告消息，指出系统已接近其最大容量。

如果此参数设置为零，则在接近并发使用（会话）限制时不会发出警告。如果将此参数设置为非零数字，则还应设置LICENSE_MAX_SESSIONS （请参阅“ LICENSE_MAX_SESSIONS ” ）。
