C XDK for Java XSU错误消息

列出了在执行XML SQL实用程序（XSU）接口期间使用Oracle XML Developer Kit（XDK）for Java的应用程序的错误消息。

通用错误消息

通用错误消息的范围是XSUE-0000到XSUE-0099。

XSUE-0000：内部错误 - 异常捕获字符串

XSUE-0001：内部错误 - 字符串

XSUE-0002： 字符串不是标量列。row id属性只能从标量列中获取值。

XSUE-0003： string不是有效的列名。

XSUE-0004：此对象已关闭。如果您希望在调用之间不隐式关闭对象，请参阅字符串方法。

XSUE-0005：省略了行集封闭标记和行封闭标记;因此，结果可以包含至多一行，其中只包含一列未标记为XML属性的列。

XSUE-0006：省略行封闭标记或行集封闭标记;因此，为了获得格式良好的XML文档，结果只能包含具有多个列的单个行或具有一个列的多个行，而这些列未标记为XML属性。

XSUE-0007：解析sqlname失败 - 参数无效。

XSUE-0008：XML标记名称中不允许使用字符串 。

XSUE-0009： 字符串类不支持此方法。请改用字符串 。

XSUE-0010：绑定名称的数量不等于绑定值的数量。

XSUE-0011：绑定值的数量与SQL语句中的绑定数量不匹配。

XSUE-0012：sql查询中不存在绑定名称标识符字符串 。

XSUE-0013：绑定标识符必须是非零长度。

XSUE-0014：提供的根节点为空。

XSUE-0015：指定了无效的LOB定位符。

XSUE-0016：文件字符串不退出。

XSUE-0017：无法创建oracle.sql。除oracle.sql之外的类型的STRUCT对象。结构（即ADT）。

XSUE-0018：Null不是有效的DocumentHandler。

XSUE-0019：空字符串和空字符串不是有效的命名空间别名。

XSUE-0020：要使用此方法，您必须在子类中覆盖它。

XSUE-0021：您使用的是旧版本的gss库;因此，不支持sql-xml名称转义。

XSUE-0022：无法从opaque基本类型创建XMLType对象： string

查询错误消息

查询错误消息的范围是XSUE-0100到XSUE-0199。

XSUE-0100：指定了无效的上下文句柄。

XSUE-0101：在结果集的第一行中有一个嵌套游标，其父游标为空;当这种情况发生时，我们无法生成dtd。

XSUE-0102： 字符串不是有效的IANA编码。

XSUE-0103：结果集是“TYPE_FORWARD_ONLY”（不可滚动）;因此，xsu无法重新定位读取点。此外，由于调用者已将结果集传递给xsu，因此xsu无法重新创建结果集。

XSUE-0104：输入字符对格式良好的XML： string无效

DML错误消息

数据操作语言（DML）错误消息的范围是XSUE-0200到XSUE-0299。

XSUE-0200：XML元素标记字符串与目标数据库对象的任何列/属性的名称不匹配。

XSUE-0201：NULL是无效的列名。

XSUE-0202：列字符串 ，指定为键列，在表字符串不不退出。

XSUE-0203：指定为要更新的列的列字符串在表字符串中不存在。

XSUE-0204：无效的REF元素 - 字符串 - 缺少属性字符串 。

XSUE-0206：必须在调用update例程之前指定键值。使用字符串函数。

XSUE-0207：UpdateXML：没有要更新的列。XML文档必须包含一些要更新的非键列。

XSUE-0208：键列数组必须为非空。

XSUE-0209：键列数组必须为非空。

XSUE-0210：没有要修改的行 - 缺少行封闭标记。指定正确的行封闭标记。

XSUE-0211：XML文档中处理ROW元素字符串中遇到的字符串 。

XSUE-0212：已成功处理字符串 XML行。

XSUE-0213：回滚了所有先前的XML行更改。
