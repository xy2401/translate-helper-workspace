D.14 chactl set maxretention

使用chactl set maxretention命令设置诊断数据的最长保留时间。

默认和最短保留时间为72小时。如果Oracle Cluster Health Advisor存储库没有足够的空间，则所有目标的保留时间都会减少。

注意：

如果保留时间少于24小时，Oracle Cluster Health Advisor将停止监控。

句法

chactl set maxretention -time retention_time [-help]

以小时为单位指定保留时间。

例子

要将最长保留时间设置为80小时：

$ chactl set maxretention -time 80 max retention成功设置为80小时

错误信息

错误： 指定的时间小于允许的最小值

描述：如果最大保留时间的输入值小于最小值，则返回此消息。
