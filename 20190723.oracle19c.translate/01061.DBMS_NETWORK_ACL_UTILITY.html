111 DBMS_NETWORK_ACL_UTILITY

DBMS_NETWORK_ACL_UTILITY包提供实用程序功能，以便于评估管理到网络主机的TCP连接的访问控制列表（ACL）分配。

111.1 DBMS_NETWORK_ACL_UTILITY安全模型

DBMS_NETWORK_ACL_UTILITY包上的EXECUTE被授予PUBLIC 。

111.2 DBMS_NETWORK_ACL_UTILITY示例

此程序包中的CONTAINS_HOST函数指示域或子网是否包含给定的主机或IP地址。

它可以与DBMS_NETWORK_ACL_ADMIN包中的CHECK_PRIVILEGE_ACLID函数一起使用，以确定影响用户访问网络主机的权限的权限分配。CONTAINS_HOST函数的返回值也可用于按优先顺序排列ACL分配。

例1

例如，SCOTT允许连接到www.hr.example.com ：

SELECT host，lower_port，upper_port，acl，DECODE（DBMS_NETWORK_ACL_ADMIN.CHECK_PRIVILEGE_ACLID（aclid，'SCOTT'，'connect'），1，'GRANTED'，0，'DENIED'，NULL）特权FROM（SELECT host，acl，aclid， lower_port，upper_port，DBMS_NETWORK_ACL_UTILITY.CONTAINS_HOST（'www.hr.example.com'，host）优先级FROM dba_network_acls）WHERE优先级> 0 ORDER BY优先级DESC，lower_port nulls LAST; HOST LOWER_PORT UPPER_PORT ACL PRIVILEGE -------------------- ---------- ---------- ----- --------------- --------- www.hr.example.com 80 80 /sys/acls/www.xml GRANTED www.hr.example.com 3000 3999 /sys/acls/www.xml GRANTED www.hr.example.com /sys/acls/www.xml GRANTED * .hr.example.com /sys/acls/all.xml * .example.com / sys / acls /all.xml

例2

例如， SCOTT允许为www.hr.example.com进行域名解析：

SELECT host，acl，DECODE（DBMS_NETWORK_ACL_ADMIN.CHECK_PRIVILEGE_ACLID（aclid，'SCOTT'，'resolve'），1，'GRANTED'，0，'DENIED'，null）特权FROM（SELECT host，acl，aclid，DBMS_NETWORK_ACL_UTILITY.CONTAINS_HOST（ 'www.hr.example.com'，host）优先级从dba_network_acls WHERE lower_port IS NULL AND upper_port IS NULL）WHERE优先级> 0 ORDER BY优先级DESC; HOST ACL PRIVILEGE ---------------------- ------------------------- --- --------- www.hr.example.com /sys/acls/hr-www.xml GRANTED * .hr.example.com /sys/acls/hr-domain.xml * .example .com /sys/acls/corp-domain.xml

请注意，“resolve”权限仅在分配时没有任何端口范围的ACL中生效（当lower_port和upper_port为NULL ）。因此，该示例在查询中不包含lower_port和upper_port列。

111.3.1 CONTAINS_HOST函数

此函数确定给定主机是否等于或包含在给定主机，域或子网中。它处理相同IP地址或子网的不同表示。例如，IPv4映射的IPv6地址被认为等于它所代表的IPv4本地地址。在评估主机或域时，它不执行域名解析。

句法

DBMS_NETWORK_ACL_UTILITY.CONTAINS_HOST（主机IN VARCHAR2，域IN VARCHAR2）返回号码;

参数

表111-2 CONTAINS_HOST函数参数

参数 描述

主办

网络主机

域

网络主机，域或子网

返回值

如果给定主机等于或包含在相关主机，域或子网中，则返回非NULL值：

  • 如果domain是主机名，则返回其域的级别+ 1

  • 如果domain是域名，则返回域级别

  • 如果domain是IP地址或子网，则返回IP地址或子网的有效地址位数

  • 如果domain是通配符“*”，则返回0

返回的非NULL值表示域或子网对ACL分配的优先级。值越高，优先级越高。如果主机不等于或包含在给定的主机，域或子网中，则将返回NULL 。

例子

SELECT host，acl，precedence FROM（select host，acl，DBMS_NETWORK_ACL_UTILITY.CONTAINS_HOST（'192.0.2.3'，host）precedence FROM dba_network_acls）WHERE precedence> 0 ORDER BY precedence DESC; HOST ACL PRECEDENCE ---------------------- ------------------------- -  ---------- 192.0.2.3 /sys/acls/hr-www.xml 32 :: ffff：192.0.2.0/120 /sys/acls/hr-domain.xml 24 :: ffff：192.0 .0.0 / 104 /sys/acls/corp-domain.xml 8

111.3.2 DOMAIN_LEVEL函数

此函数返回给定主机名，域或子网的域级别。

句法

DBMS_NETWORK_ACL_UTILITY.DOMAIN_LEVEL（主机IN VARCHAR2）返回号码;

参数

表111-3 DOMAIN_LEVEL函数参数

参数 描述

主办

网络主机，域或子网

返回值

给定主机，域或子网的域级别。

使用说明

请注意，此功能无法处理IPv6地址和子网以及CIDR表示法中的子网。

例子

SELECT host，acl，domain_level FROM（select host，acl，DBMS_NETWORK_ACL_UTILITY.DOMAIN_LEVEL（host）domain_level FROM dba_network_acls）order by domain_level desc; HOST ACL DOMAIN_LEVEL ---------------------- ------------------------- --- ------------ www.hr.example.com /sys/acls/hr-www.xml 4 * .hr.example.com /sys/acls/hr-domain.xml 3 * .example.com /sys/acls/corp-domain.xml 2

111.3.3域功能

对于给定的主机，此函数返回其ACL分配的域确定用户是否具有访问给定主机的权限。当给出主机的IP地址时，返回子网。

句法

DBMS_NETWORK_ACL_UTILITY.DOMAINS（主机IN VARCHAR2）返回DOMAIN_TABLE PIPELINED;

参数

表111-4 DOMAINS函数参数

参数 描述

主办

网络主机

返回值

给定主机的域或子网。

使用说明

请注意，此功能无法处理IPv6地址。它也不能为IPv4地址生成任意数量的前缀位的子网。

例子

select * from table（dbms_network_acl_utility.domains（'www.hr.example.com'））; DOMAINS -------------------------- www.hr.example.com * .hr.example.com * .example.com * .com *

111.3.4 EQUALS_HOST功能

此函数确定两个给定的主机，域或子网是否相等。它处理相同IP地址或子网的不同表示。例如，IPv4映射的IPv6地址被认为等于它所代表的IPv4本地地址。比较两个主机或域时，它不执行域名解析。

句法

DBMS_NETWORK_ACL_UTILITY.EQUALS_HOST（host1 IN VARCHAR2，host2 IN VARCHAR2）返回号码;

参数

表111-5 EQUALS_HOST功能参数

参数 描述

host 1

要比较的网络主机，域或子网

host 2

要比较的网络主机，域或子网

返回值

如果两个主机，域或子网相等，则为1。否则为0。

例子

SELECT host，acl FROM dba_network_acls WHERE DBMS_NETWORK_ACL_UTILITY.EQUALS_HOST（'192.0.2。*'，host）= 1;主机ACL ---------------------- -------------------------- -  :: ffff：192.0.2.0/120 /sys/acls/hr-domain.xml
