图形语法图

语法图是说明有效SQL语法的图。要读取图表，请按箭头所示方向从左向右跟踪。

命令和其他关键字出现在矩形内的大写字母中。完全按照矩形所示键入它们。参数在椭圆内显示为小写。变量用于参数。标点符号，运算符，分隔符和终止符出现在圆圈内。

如果语法图有多个路径，则可以选择任何路径。例如，在以下语法中，您可以指定NOPARALLEL或PARALLEL ：

parallel_clause :: =


parallel_clause.eps插图的描述

如果您可以选择多个关键字，运算符或参数，则您的选项将显示在垂直列表中。例如，在以下语法图中，您可以指定堆栈中的四个参数中的一个或多个：

physical_attributes_clause :: =
