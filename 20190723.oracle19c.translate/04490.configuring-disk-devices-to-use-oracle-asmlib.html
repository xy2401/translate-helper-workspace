配置磁盘设备以使用Oracle ASMLIB

配置要在Oracle自动存储管理磁盘组中使用的磁盘设备。

要配置要在Oracle自动存储管理磁盘组中使用的磁盘设备，请执行以下步骤：

  1. 如果要在Oracle自动存储管理磁盘组中使用IDE，SCSI或RAID设备，请执行以下步骤：

    1. 安装或配置要用于磁盘组的磁盘设备，然后重新启动系统。

    2. 输入以下命令以标识要使用的磁盘的设备名称：

      ＃/ sbin / fdisk -l

      根据磁盘类型，设备名称可能会有所不同。

      表A-3基于磁盘类型的设备名称格式

      磁盘类型 设备名称格式 描述

      IDE磁盘

      		 
      / dev / hd xn

      在此示例中， x是标识IDE磁盘的字母， n是分区号。例如， /dev/hda是第一个IDE总线上的第一个磁盘。

      SCSI磁盘

      		 
      / dev / sd xn

      在此示例中， x是标识IDE磁盘的字母， n是分区号。例如， /dev/hda是第一个IDE总线上的第一个磁盘。

      RAID磁盘

      		 
      / dev / rd / c x d y p z / dev / ida / c x d y p z

      根据RAID控制器，RAID设备可以具有不同的设备名称。在所示的示例中， x是标识控制器的数字， y是标识磁盘的数字， z是标识分区的数字。例如， /dev/ida/c0d1是第一个控制器上的第二个逻辑驱动器。

      要在磁盘组中包含设备，可以指定整个驱动器设备名称或分区设备名称。

      注意：

      Oracle建议您在每个磁盘上创建一个要使用的整个磁盘分区。

    3. 使用fdisk或parted在磁盘设备上创建单个整个磁盘分区。

  2. 输入与以下内容类似的命令，将磁盘标记为Oracle自动存储管理磁盘：

    ＃/ usr / sbin / oracleasm createdisk DISK1 / dev / sdb1

    在此示例中， DISK1是分配给磁盘的名称。

    注意：

    • 您指定的磁盘名称可以包含大写字母，数字和下划线字符。它们必须以大写字母开头。

    • 要在安装期间使用Oracle自动存储管理库驱动程序创建数据库，必须将磁盘发现字符串更改为ORCL:* 。

    • 如果在Oracle ASM中使用多路径磁盘驱动程序，请确保为磁盘指定正确的逻辑设备名称。

  3. 要使磁盘在群集中的其他节点上可用，请在每个节点上以root身份输入以下命令：

    ＃/ usr / sbin / oracleasm scandisks

    此命令标识连接到节点的标记为Oracle ASM磁盘的共享磁盘。
