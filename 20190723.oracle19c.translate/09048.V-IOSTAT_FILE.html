8.18 V $ IOSTAT_FILE

V$IOSTAT_FILE显示有关数据库文件（包括数据文件，临时文件和其他类型的数据库文件）的磁盘I / O统计信息。

为每个文件提供数据文件和临时文件的I / O统计信息。所有其他文件类型（例如，控制文件，日志文件，存档日志等）将其统计信息合并到视图中的一个条目中。

柱 数据类型 描述

文件编号

数

文件标识号

FILETYPE_ID

数

文件类型（例如，日志文件，数据文件等）

FILETYPE_NAME

VARCHAR2（28）

对于数据文件或临时文件，文件的名称。对于所有其他文件，将显示相应的字符串（例如， ARCHIVELOG ）。

SMALL_READ_MEGABYTES

数

读取的单个块兆字节数

SMALL_WRITE_MEGABYTES

数

LARGE_READ_MEGABYTES

数

读取的多块兆字节数

LARGE_WRITE_MEGABYTES

数

写入的多块兆字节数

SMALL_READ_REQS

数

单块读取请求的数量

SMALL_WRITE_REQS

数

单个块写入请求的数量

SMALL_SYNC_READ_REQS

数

同步单块读取请求的数量

LARGE_READ_REQS

数

多块读取请求的数量

LARGE_WRITE_REQS

数

多块写入请求的数量

SMALL_READ_SERVICETIME

数

单个块读取请求的总服务时间（以毫秒为单位）

SMALL_WRITE_SERVICETIME

数

单个块写入请求的总服务时间（以毫秒为单位）

SMALL_SYNC_READ_LATENCY

LARGE_READ_SERVICETIME

数

多块读取请求的总服务时间（以毫秒为单位）

LARGE_WRITE_SERVICETIME

数

多块写入请求的总服务时间（以毫秒为单位）

ASYNCH_IO

VARCHAR2（9）

指示异步I / O是否可用于文件（ ASYNC_ON ）（ ASYNC_OFF ）

ACCESS_METHOD

VARCHAR2（11）

用于访问文件的I / O库。可能的值包括：

  • OS_LIB - 操作系统调用用于访问文件

  • ODM_LIB - Oracle Disk Manager库用于访问该文件

  • ASM_MANAGED - 通过ASM管理和访问该文件

  • DNFS_LIB - 通过Direct NFS库访问该文件

RETRIES_ON_ERROR

数

出错时的读取重试次数

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
