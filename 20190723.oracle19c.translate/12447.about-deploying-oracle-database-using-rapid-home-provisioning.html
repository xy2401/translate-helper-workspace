关于使用Oracle Fleet补丁和配置部署Oracle数据库

您可以使用Oracle Fleet Patching and Provisioning（Oracle FPP）来配置Oracle数据库软件。

从Oracle Database 19c开始，Rapid Home Provisioning重命名为Oracle Fleet Patching and Provisioning（Oracle FPP）。

通过Oracle Fleet Patching和Provisioning，您可以创建，存储和管理Oracle家庭模板，作为Oracle软件的图像（称为黄金映像），例如数据库，中间件和应用程序。您可以制作任何黄金映像的工作副本，然后您可以将该工作副本提供给数据中心或云计算环境中的任何节点。

您可以使用Oracle Fleet Patching和Provisioning来配置，修补和升级在Oracle Restart上运行的单实例数据库，群集或单个独立节点上。这些可能位于安装或未安装Oracle Grid Infrastructure的节点上。

注意：

Microsoft Windows和HP-UX Itanium系统不支持Oracle Fleet Patching和Provisioning。
