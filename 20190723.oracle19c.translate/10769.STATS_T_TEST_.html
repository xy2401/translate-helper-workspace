STATS_T_TEST_ *

t -test函数是：

  • STATS_T_TEST_ONE ：单样本t检验

  • STATS_T_TEST_PAIRED ：双样本，配对t检验（也称为交叉t检验）

  • STATS_T_TEST_INDEP ：具有相同方差的两个独立组的t检验（合并方差）

  • STATS_T_TEST_INDEPU ：具有不等方差的两个独立组的t检验（未计算的方差）

句法

stats_t_test :: =


说明stats_t_test.eps的描述

目的

t检验衡量均值差异的重要性。您可以使用它来比较两组的平均值或一组具有常数的平均值。每个t -test函数都有两个表达式参数，尽管第二个表达式对于单样本函数（ STATS_T_TEST_ONE ）是可选的。每个t -test函数都有一个可选的第三个参数，它允许您指定函数返回的NUMBER值的含义，如表7-9所示。对于此参数，您可以指定文本文字，或指定为常量字符值的绑定变量或表达式。如果省略第三个参数，则默认值为'TWO_SIDED_SIG' 。

如果第三个参数指定为'STATISTIC'或'ONE_SIDED_SIG' ，则两个独立的STATS_T_TEST_ *函数可以采用第四个参数（ expr3 ）。在这种情况下， expr3指示expr1哪个值是高值，或者拒绝区域是上尾的值。

吨的观测值的意义在于，t的值会被偶然-若干0和1之间获得的概率。值越小，均值之间的差异越显着。片面意义始终是对上尾的尊重。对于单样本和配对t检验，高值是第一个表达式。对于独立的t -test，高值是expr3指定的值。

的自由度取决于导致吨的观测值t检验的类型。例如，对于单样本t检验（ STATS_T_TEST_ONE ），自由度是样本中观察数减1。

STATS_T_TEST_ONE

在STATS_T_TEST_ONE函数中， expr1是样本， expr2是与样本均值进行比较的常量均值。仅对于此t测试， expr2是可选的;常量mean默认为0。该功能通过由平均值的标准误差除以样本均值和已知平均值之间的差获得t的值（而不是装置的差的标准误差，作为用于STATS_T_TEST_PAIRED ）。

STATS_T_TEST_ONE示例

以下示例确定平均清单价格与常数值60之间差异的显着性：

SELECT AVG（prod_list_price）group_mean，STATS_T_TEST_ONE（prod_list_price，60，'STATISTIC'）t_observed，STATS_T_TEST_ONE（prod_list_price，60）two_sided_p_value FROM sh.products; GROUP_MEAN T_OBSERVED TWO_SIDED_P_VALUE ---------- ---------- ----------------- 139.545556 2.32107746 .023158537

STATS_T_TEST_PAIRED

在STATS_T_TEST_PAIRED函数中， expr1和expr2是正在比较其平均值的两个样本。该功能通过由装置的差的标准误差除以样品的平均值之差求出t的值（而不是平均值的标准误差，作为用于STATS_T_TEST_ONE ）。

STATS_T_TEST_INDEP和STATS_T_TEST_INDEPU

在STATS_T_TEST_INDEP和STATS_T_TEST_INDEPU函数中， expr1是分组列， expr2是值的样本。合并差异版本（ STATS_T_TEST_INDEP ）测试具有相似差异的两个分布的均值是相同还是不同。即使已知两个分布具有显着不同的方差，非合并方差版本（ STATS_T_TEST_INDEPU ） STATS_T_TEST_INDEPU测试平均值是相同还是不同。

在使用这些功能之前，建议确定样品的方差是否显着不同。如果是，则数据可能来自具有不同形状的分布，并且均值的差异可能不是非常有用。您可以执行f -test来确定方差的差异。如果它们没有显着差异，请使用STATS_T_TEST_INDEP 。如果它们明显不同，请使用STATS_T_TEST_INDEPU 。有关执行f - test的信息，请参阅STATS_F_TEST 。
