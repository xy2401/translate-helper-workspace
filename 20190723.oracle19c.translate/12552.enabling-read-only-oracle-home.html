启用只读Oracle主目录

在执行仅限软件的Oracle数据库安装后，将Oracle主目录配置为只读Oracle主目录。

要启用只读Oracle主目录：

  1. 执行仅限软件的Oracle数据库安装。

  2. 运行roohctl -enable脚本。

  3. 运行Oracle数据库配置助手（Oracle DBCA）以创建数据库。

纯软件数据库安装

  1. 以您希望拥有软件二进制文件的Oracle安装所有者用户帐户（ oracle ）身份登录。

  2. 将Oracle数据库安装映像文件（ db_home.zip ）下载到您选择的目录中。例如，您可以将图像文件下载到/tmp目录。

  3. 创建Oracle主目录并将已下载的映像文件解压缩到此Oracle主目录。例如：

    $ mkdir -p /u01/app/oracle/product/19.0.0/dbhome_1 $ chown oracle：oinstall /u01/app/oracle/product/19.0.0/dbhome_1 $ cd /u01/app/oracle/product/19.0。 0 / dbhome_1 $ unzip -q /tmp/db_home.zip

    注意：

    确保您创建的Oracle主目录路径符合Oracle Optimal Flexible Architecture建议。此外，仅在您创建的此Oracle主目录中解压缩安装映像文件。

  4. 在Oracle主目录中，运行runInstaller命令以启动Oracle数据库安装程序。

  5. 在“选择配置选项”屏幕中，选择“仅设置软件”。

  6. 选择您的安装类型。安装屏幕因您选择的安装选项而异。根据需要响应配置提示。

注意：

如果您对安装期间要求提交的信息有任何疑问，请单击“ 帮助” 。

运行roohctl脚本

  1. 转到bin目录

    $ cd /u01/app/oracle/product/19.0.0/dbhome_1/bin

  2. 运行roohctl脚本以启用只读Oracle主目录。 

    $ ./roohctl -enable

  3. 在Oracle Real Application Clusters（Oracle RAC）安装上，在群集的每个节点上运行前面的roohctl脚本。或者，使用nodelist选项运行roohctl脚本并提供集群节点列表：

    $ ./roohctl -enable -nodelist comma_separated_list_of_nodes

运行Oracle数据库配置助手

  1. 确保您仍在bin目录中并运行Oracle DBCA。

    $ ./dbca

  2. 在“选择数据库操作”屏幕中，选择“ 创建数据库”。

  3. 配置屏幕取决于您选择的选项。根据需要响应提示。

注意：

如果您对在数据库创建期间要求提交的信息有任何疑问，请单击“ 帮助” 。
