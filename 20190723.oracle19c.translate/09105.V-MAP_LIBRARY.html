8.71 V $ MAP_LIBRARY

V$MAP_LIBRARY显示外部进程动态加载的所有映射库的列表。

柱 数据类型 描述

LIB_IDX

数

与库对应的索引

LIB_NAME

VARCHAR2（256）

绝对的库名称

供应商名称

VARCHAR2（64）

实现库的供应商的名称

PROTOCOL_NUM

数

库支持的映射协议

VERSION_NUM

VARCHAR2（32）

版本号

路径名

VARCHAR2（4000）

路径名称

MAP_FILE

VARCHAR2（1）

指示库是否支持映射文件（ Y ）或不支持（ N ）

FILE_CFGID

VARCHAR2（13）

文件支持的配置ID类型：

  • NONE - 不支持

  • 持久性

  • 非持久

MAP_ELEM

VARCHAR2（1）

指示库是否支持映射元素（ Y ）或不支持（ N ）

ELEM_CFGID

VARCHAR2（13）

元素支持的配置ID类型：

  • NONE - 不支持

  • 持久性

  • 非持久

MAP_SYNC

VARCHAR2（1）

指示是否需要显式同步库，以便将来的映射反映最近的更改（ Y ）或不反映（ N ）。请注意，如果需要显式同步库，则不支持配置ID。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
