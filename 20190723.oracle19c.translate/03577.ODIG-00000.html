68 ODIG-00000至ODIG-00013

ODIG-00000：没有错误。

原因：操作成功。

行动：无需采取任何行动。

ODIG-00001：没有为选项' string '指定值

原因：没有为选项指定值。

操作：指定值。

ODIG-00002：无效参数：' string '

原因：指定了无效参数。

操作：指定有效参数。

ODIG-00003：无效选项：' string '

原因：指定了无效选项。

操作：指定有效选项。

ODIG-00004：为选项' string '指定的值无效：' string '

原因：为选项指定的值不是预期的类型。

操作：指定有效值。

ODIG-00005：无效的查询类型：' string '。支持的类型： string 。

原因：查询类型无效。

操作：指定有效的查询类型。

ODIG-00007：线程生成失败。

原因：用于查询的线程的生成失败。

操作：在常规使用中不应出现此错误。如果出现，请联系Oracle客户支持。

ODIG-00008：服务器地址或名称' string '无效： string

原因：为DNS服务器指定的地址或名称无效。

操作：指定有效的DNS名称。有关详细信息，请参阅RFC 1034。

ODIG-00009：未指定名称。

原因：未指定要查询的名称。

操作：指定名称。

ODIG-00010：内部错误： string

原因：发生意外错误。

操作：联系Oracle支持服务。

ODIG-00011：为选项' string '指定的值无效： number

原因：为该选项指定的值必须大于零。

操作：指定有效值。

ODIG-00012：未能获得GNS域名

原因：尝试获取网格命名服务（GNS）域的名称时发生错误。

操作：检查附带的错误消息以获取有关错误的更多信息。

ODIG-00013：选项' string '和' string '不兼容。

原因：查询同时指定了DNSSEC选项和DNS查询选项，这是不允许的。

操作：指定DNSSEC选项或DNS查询选项，但不能同时指定两者。
