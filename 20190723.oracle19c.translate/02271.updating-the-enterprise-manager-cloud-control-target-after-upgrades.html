升级后更新企业管理器云控制目标

升级Oracle Grid Infrastructure后，使用新的Grid主路径更新企业管理器目标。

  1. 登录企业管理器云控制。

  2. 导航到“ 目标”菜单，然后导航到“ 群集”页面。

  3. 单击已升级的群集目标。

  4. 单击Cluster ，然后单击Target Setup ，然后从菜单中单击Monitoring Configuration 。

  5. 使用新的Grid主路径更新Oracle Home的值。

  6. 保存更新。
