1.168 LOCK_NAME_SPACE

LOCK_NAME_SPACE指定分布式锁管理器（DLM）用于生成锁名称的命名空间。

属性 描述

参数类型

串

句法

LOCK_NAME_SPACE = namespace

默认值

没有默认值。

可修改

没有

可在PDB中修改

没有

价值范围

最多8个字母数字字符。不允许使用特殊字符。

基本

没有

注意：

不推荐使用LOCK_NAME_SPACE参数。它仅为了向后兼容而保留。

如果备用数据库或克隆数据库在与主数据库相同的集群上具有相同的数据库名称，请考虑设置此参数。

如果备用数据库与主数据库位于同一文件系统上，则将备用参数文件中的LOCK_NAME_SPACE设置为不同的值，如下所示：
