DROP DIMENSION

目的

使用DROP DIMENSION语句删除命名的维度。

此语句不会使使用维度中指定的关系的实体化视图无效。但是，已经通过查询重写重写的请求可能无效，并且对这些视图的后续操作可能执行得更慢。

也可以看看：

  • CREATE DIMENSION和ALTER DIMENSION有关创建和修改尺寸的信息

  • Oracle Database Concepts有关维度的一般信息

先决条件

维度必须位于您自己的架构中，或者您必须具有DROP ANY DIMENSION系统特权才能使用此语句。

句法

drop_dimension :: =


说明drop_dimension.eps的描述

语义

模式

指定维度所在的架构的名称。如果省略schema ，则Oracle数据库假定维度位于您自己的模式中。

尺寸

指定要删除的维的名称。维度必须已存在。

例子

删除维度：示例

此示例删除sh.customers_dim维度：

DROP DIMENSION customers_dim;
