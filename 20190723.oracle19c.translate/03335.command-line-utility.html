10适用于DRDA的Oracle数据库提供程序的命令行实用程序

适用于DRDA的Oracle数据库提供程序提供命令行实用程序。

命令行实用程序

命令行实用程序drdactl使用户能够控制Oracle DRDA应用程序服务器。它控制AS的启动，关闭，状态和操作更改。可以使用命令参数调用drdactl实用程序以立即执行，或者不使用参数（该实用程序将提示输入命令）。

开始

此命令启动指定的实例。

句法

START { instance_name }

停

此命令将停止指定的运行实例。

句法

STOP { instance_name }

状态

此命令显示指定的运行实例的当前状态以及任何当前连接的会话信息。

句法

状态{ instance_name } {详情}

跟踪

对于指定的会话ID，此命令禁用或启用指定级别的跟踪。

句法

TRACE {<instance_name>} <OFF | Level> {SESSION {<session_id> | ALL}}

暂停

此命令将使用指定的会话ID暂停会话。

句法

PAUSE { instance_name } SESSION < session_id >

恢复

此命令将使用指定的会话ID恢复会话。

句法

RESUME { instance_name } SESSION < session_id >

RELOAD

此命令使服务器重新加载实例配置。

句法

RELOAD { instance_name }

出口

此命令退出实用程序并返回操作系统。

句法

出口
