XSL接口

描述了XSL接口。

XslException Interface-Root接口，用于所有与XSLT相关的异常。

Transformer Interface-Basic XSLT处理器。您可以使用此接口来调用所有XSLT处理器。

CompTransformer接口 - 扩展XSLT处理器。您只能将此接口用于创建中间二进制字节码的处理器（目前只有基于XVM的处理器）。

Compiler接口-XSLT编译器。它用于将XSLT编译为二进制字节码的编译器。

XSLT for C ++ DOM接口使用

描述了XSLT for C ++ DOM的基本用法。

XMLParser.xmlparse()有两个输入：

  • 可扩展标记语言（XML）文档

  • 要应用于XML文档的XSLT样式表

通过调用工具工厂来创建XSLT处理器以创建特定的XSLT转换器或编译器。

通过调用setXSL()成员函数将XSLT样式表提供给转换器。

XML实例文档作为参数提供给转换成员函数。

生成的文档（XML，HTML，矢量标记语言（VML）等）通常被发送到应用程序以进行进一步处理。它作为文档对象模型（DOM）树或作为XML（SAX）事件的简单API序列发送。如果用户提供SAX事件处理程序，则会生成SAX事件。

应用程序通过调用其析构函数来终止XSLT处理器。
