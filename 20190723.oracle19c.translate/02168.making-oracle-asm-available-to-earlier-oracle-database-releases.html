使Oracle ASM适用于早期的Oracle数据库版本

要将Oracle ASM与Oracle Database 12c第2版（12.2）之前的Oracle数据库版本一起使用，必须固定所有群集节点。

安装Oracle Grid Infrastructure 18c或更高版本后，如果要使用Oracle ASM为早于Oracle Database 12c第2版（12.2）的Oracle数据库版本提供存储服务，则必须使用以下命令来固定节点：

$ crsctl pin css -n node1 node2

此设置更新Oracle ASM条目的oratab文件。

您可以使用以下命令检查固定节点：

$ ./olsnodes -t -n

注意：

重新启动Oracle ASM以加载更新的oratab文件。
