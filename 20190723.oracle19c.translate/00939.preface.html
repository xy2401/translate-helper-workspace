前言

Oracle数据库网关APPC为Oracle应用程序提供了对几乎任何支持APPC的系统的无缝访问，包括通过远程过程调用（RPC）处理的IBM大型机数据和服务。

目标受众

如果您负责以下任务，请阅读本指南：

  • 确定硬件和软件要求

  • 安装，配置或管理APPC的Oracle数据库网关

  • 使用SNA通信协议或TCP / IP通信协议开发通过Oracle Database Gateway for APPC访问远程主机数据库的应用程序

  • 确定安全要求

  • 确定和解决问题

在使用本指南管理网关之前，您应该了解操作系统和Oracle数据库网关的基础知识。
