BITMAP_CONSTRUCT_AGG

句法



插图描述bitmap_construct_agg.eps

目的

BITMAP_CONSTRUCT_AGG是一个聚合函数， BITMAP_CONSTRUCT_AGG位位置进行操作并返回所有输入位位置集的位图表示。它基本上维护一个位图，并将所有输入位置置于其中。它返回位图的表示。

参数expr的类型为NUMBER 。

返回类型是BLOB类型。

如果expr为NULL，则该函数返回NULL。

限制

  • 参数必须是NUMBER类型。如果输入值无法转换为自然数，则会引发错误ORA-62575 ：
    62575,00000，“无效值已传递给BITMAP COUNT DISTINCT相关运算符。” // *原因：尝试将无效值传递给BITMAP COUNT DISTINCT运算符。// *操作：仅将自然数值传递给BITMAP_CONSTRUCT_AGG。
  • 如果位图超过BLOB的最大值，您将看到错误ORA-62577 ：
    62577,00000，“位图大小超过其SQL数据类型的最大大小。” // *原因：尝试构造一个大于其最大SQL类型大小的位图。// *动作：将BITMAP_CONSTRUCT_AGG的输入分解为更小的范围。
