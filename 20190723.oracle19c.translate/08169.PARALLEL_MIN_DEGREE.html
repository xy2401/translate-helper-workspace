1.250 PARALLEL_MIN_DEGREE

PARALLEL_MIN_DEGREE控制由自动并行度计算的最小并行度。

属性 描述

参数类型

串

句法

PARALLEL_MIN_DEGREE = [n |中央处理器 ]

默认值

1

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

Oracle RAC

可以在不同的实例上设置不同的值。

PARALLEL_MIN_DEGREE的值是对应于由自动并行度计算的并行度的下限的数字，或者是由CPU_COUNT * PARALLEL_THREADS_PER_CPU函数计算的字符串值CPU 。

PARALLEL_MIN_DEGREE的默认值为1。

注意：

PARALLEL_MIN_DEGREE对这两种情况都没有影响：

  • 当PARALLEL_MIN_DEGREE的值大于CPU_COUNT的值时

  • 当对象是Oracle拥有的时，例如在字典表上创建的字典表或视图

  • “ PARALLEL_DEGREE_LIMIT ”

  • 有关优化程序如何自动确定语句并行度的信息，请参见“Oracle数据库VLDB和分区指南”
