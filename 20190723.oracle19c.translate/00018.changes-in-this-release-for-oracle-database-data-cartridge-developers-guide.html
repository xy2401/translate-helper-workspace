Oracle Database 19c，版本19.1中的更改

本节列出了Oracle Database Data Cartridge Developer for Oracle Database 19c版本19.1中的更改。

有关此版本中新功能的完整列表，请参阅“ Oracle数据库新增功能指南” 。

不受支持的功能

Oracle多媒体：从Oracle Database 19c开始，不支持Oracle多媒体功能。将Oracle SecureFiles与开源或第三方产品一起用于元数据提取以及图像处理和转换，而不是Oracle多媒体。
