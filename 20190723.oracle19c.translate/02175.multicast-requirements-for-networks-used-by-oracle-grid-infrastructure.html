Oracle Grid Infrastructure使用的网络的多播要求

对于每个集群成员节点，Oracle mDNS守护程序在所有接口上使用多播来与集群中的其他节点进行通信。

Oracle Grid Infrastructure使用的网络的多播要求

专用互连需要多播。因此，您必须至少为群集启用多播：

  • 跨越为专用互连定义的广播域

  • 在IP地址子网范围224.0.0.0/24和可选230.0.1.0/24上

您无需在路由器之间启用多播通信。
