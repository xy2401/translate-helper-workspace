C OCI功能服务器Round-Trips

本附录提供有关在各种OCI调用期间发生的服务器往返的信息。

此信息可以帮助程序员确定在应用程序中完成特定任务的最有效方法。服务器往返被定义为从客户端到服务器并返回到客户端的行程。

使用客户端SQL时，OCI提供：

  • 增强的阵列数组操作语言（DML）功能

  • 能够将提交请求与运行相关联以减少往返次数

  • 使用透明预取缓冲区优化查询以减少往返次数

通常情况下，存在权衡。例如，对于查询，权衡通常是数组大小和预取大小与基础提取的数量。
