使用ULN支持安装Oracle预安装RPM

使用此过程为Oracle软件订阅Unbreakable Linux Network（ULN）Oracle Linux通道。

要获得Unbreakable Linux Network（ULN）支持，请订阅Oracle Linux通道，并添加分发Oracle预安装RPM的Oracle Linux通道：
  1. 使用Unbreakable Linux Network（ULN）注册您的服务器。默认情况下，您已注册适用于您的操作系统和硬件的Oracle Linux Latest频道。
  2. 登录坚不可摧的Linux网络：
    https://linux.oracle.com
  3. 启动终端会话并以root身份输入以下命令，具体取决于您的平台。例如：
    • Oracle Linux 7：
      #yum install oracle-database-preinstall-19c

      注意：

      如果您希望yum跳过包确认提示，请使用-y选项。

    您应该看到输出，表明您已订阅Oracle Linux通道，并且正在安装这些软件包。

    Oracle Linux自动创建标准（非角色分配）Oracle安装所有者和组，并根据Oracle安装的要求设置其他内核配置设置。
  4. 检查RPM日志文件以查看系统配置更改。例如，在Oracle Linux 7上：
    / var / log / oracle-database-preinstall-19c / backup / timestamp /orakernel.log
  5. 在群集中的所有其他服务器上重复步骤1到4。
