1.104 DISK_ASYNCH_IO

DISK_ASYNCH_IO控制数据文件，控制文件和日志文件的I / O是否是异步的（即，并行服务器进程是否可以在表扫描期间与CPU处理重叠I / O请求）。

属性 描述

参数类型

布尔

默认值

真正

可修改

没有

可在PDB中修改

没有

价值范围

是的|假

基本

没有

如果您的平台支持异步I / O到磁盘，Oracle建议您将此参数设置为其默认值。但是，如果异步I / O实现不稳定，则可以将此参数设置为false以禁用异步I / O.如果您的平台不支持磁盘的异步I / O，则此参数无效。

如果将DISK_ASYNCH_IO设置为false ，则可以增加DB_WRITER_PROCESSES或使用DBWR_IO_SLAVES来模拟异步I / O.
