12.1 SQL * Plus命令摘要

命令 描述 
@（在牌子上）

在指定的脚本中运行SQL * Plus语句。可以从本地文件系统或Web服务器调用该脚本。

 
@@（双重标志）

运行脚本。此命令类似于@（at sign）命令它对于运行嵌套脚本很有用，因为它在与调用脚本相同的路径中查找指定的脚本。

 
/（斜线）

执行SQL命令或PL / SQL块。

 
接受

读取一行输入并将其存储在给定的替换变量中。

 
附加

将指定的文本添加到缓冲区中当前行的末尾。

 
存档日志

显示有关重做日志文件的信息。

 
属性

指定“对象类型”列的给定属性的显示特性，并列出单个属性或所有属性的当前显示特征。

 
打破

指定格式在报表中的位置和方式，或列出当前的中断定义。

 
BTITLE

在每个报告页面的底部放置和格式化指定的标题，或列出当前的BTITLE定义。

 
更改

更改缓冲区中当前行的文本。

 
明确

重置或删除指定选项的当前子句或设置，例如BREAKS或COLUMNS。

 
柱

指定给定列的显示特性，或列出单个列或所有列的当前显示特性。

 
计算

使用各种标准计算在所选行的子集上计算和打印汇总行，或列出所有COMPUTE定义。

 
CONNECT

将给定用户连接到Oracle数据库。

 
复制

将查询结果复制到同一数据库或另一个数据库中的表。

 
限定

指定替换变量并为其指定CHAR值，或列出单个变量或所有变量的值和变量类型。

 
DEL

删除一行缓冲区。

 
描述

列出指定表，视图或同义词的列定义或指定功能过程的规范。

 
断开

提交对数据库的挂起更改并将当前用户从Oracle数据库中记录下来，但不会退出SQL * Plus。

 
编辑

对指定文件的内容或缓冲区的内容调用操作系统文本编辑器。

 
执行

执行单个PL / SQL语句。

 
出口

终止SQL * Plus并将控制权返回给操作系统。

 
得到

将操作系统文件加载到缓冲区中。

 
救命

访问SQL * Plus命令行帮助系统。

 
历史

回顾当前SQL * Plus会话中发出的命令，SQL * Plus命令和SQL或PL / SQL语句的历史记录。

 
主办

执行操作系统命令而不离开SQL * Plus。

 
INPUT

在缓冲区中的当前行之后添加一个或多个新行。

 
LIST

列出缓冲区的一行或多行。

 
密码

允许更改密码而不回显输入设备上的密码。

 
暂停

显示指定的文本，然后等待用户按Return键。

 
打印

显示绑定变量的当前值。

 
提示

将指定的消息发送到用户的屏幕。

 
出口

终止SQL * Plus并将控制权返回给操作系统QUIT与EXIT相同。

 
恢复

在一个或多个表空间，一个或多个数据文件或整个数据库上执行介质恢复。

 
备注

在脚本中开始发表评论。

 
REPFOOTER

在每个报告的底部放置和格式化指定的报告页脚，或列出当前的REPFOOTER定义。

 
REPHEADER

在每个报告的顶部放置和格式化指定的报告标题，或列出当前的REPHEADER定义。

 
跑

列出并运行当前存储在SQL缓冲区中的SQL命令或PL / SQL块。

 
保存

将缓冲区的内容保存在操作系统文件（脚本）中。

 
组

设置系统变量以更改当前会话的SQL * Plus环境。

 
节目

显示SQL * Plus系统变量或当前SQL * Plus环境的值。

 
关掉

关闭当前运行的Oracle数据库实例。

 
SPOOL

将查询结果存储在操作系统文件中，并可选择将文件发送到打印机。

 
开始

在指定的脚本中运行SQL语句。可以从SQL * Plus命令行中的本地文件系统或Web服务器调用该脚本。

 
启动

启动Oracle数据库实例，并可选择安装并打开数据库。

 
商店

在操作系统脚本中保存当前SQL * Plus环境的属性。

 
定时

记录经过一段时间的计时数据，列出当前计时器的标题和计时数据，或列出活动计时器的数量。

 
TTITLE

在每个报告页面的顶部放置和格式化指定的标题，或列出当前的TTITLE定义。

 
UNDEFINE

删除显式（使用DEFINE命令）或隐式（使用START命令的参数）定义的一个或多个替换变量。

 
变量

声明可以在PL / SQL中引用的绑定变量，或列出单个变量或所有变量的当前显示特性。

 
在OSERROR

如果操作系统命令生成错误，则退出SQL * Plus。

 
更好的SQLERROR

如果SQL命令或PL / SQL块生成错误，则退出SQL * Plus。

 
XQUERY

运行XQuery 1.0语句。
