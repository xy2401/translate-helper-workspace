54 NDFN-00020至NDFN-00025

NDFN-00020：缺少IPMI用户ID

原因：未指定必需的用户名参数。

操作：使用正确的参数重新发出命令。

NDFN-00021：缺少节点列表

原因：未提供所需的节点列表。

操作：使用正确的参数重新发出命令。

NDFN-00022：无法识别的命令选项：' string '

原因：操作类型丢失或拼写错误。

操作：使用正确的参数重新发出命令。

NDFN-00023：无密码读取。重新发出请求

原因：在查找行尾之前，未读取所需的管理员密码以响应密码提示。

操作：重新发出命令，并确保在提示后立即键入密码。

NDFN-00024：捕获I / O异常： string

原因：尝试读取密码时遇到意外的I / O异常。异常字符串应指明确切原因。

操作：修复根本原因后重新发出命令。

NDFN-00025：节点string string失败

原因：无法为指定节点完成指示的操作。附加的较低级别错误消息应指示根本原因。

操作：修复指示的条件后重新发出命令。
