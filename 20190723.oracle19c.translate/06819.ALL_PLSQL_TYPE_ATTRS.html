3.11 ALL_PLSQL_TYPE_ATTRS

ALL_PLSQL_TYPE_ATTRS描述用户可访问的PL / SQL类型的属性。

相关观点

  • DBA_PLSQL_TYPE_ATTRS描述数据库中所有PL / SQL类型的属性。

  • USER_PLSQL_TYPE_ATTRS描述了用户自己的PL / SQL类型的属性。此视图不显示OWNER或CHAR_USED列。

柱 数据类型 空值 描述

TYPE_NAME

VARCHAR2（136）

包裹名字

VARCHAR2（128）

包含该类型的包的名称

ATTR_NAME

VARCHAR2（128）

属性的名称

ATTR_TYPE_MOD

VARCHAR2（7）
