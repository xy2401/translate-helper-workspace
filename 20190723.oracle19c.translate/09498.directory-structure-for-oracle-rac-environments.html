B Oracle RAC环境的目录结构

了解Oracle Real Application Clusters（Oracle RAC）软件环境的目录结构。

  • 了解Oracle RAC目录结构
    使用Oracle RAC安装Oracle Database 19c时，除Oracle Inventory目录，Oracle自动存储管理主目录（如果适用）和Oracle Clusterware主目录之外的所有子目录都位于顶级Oracle基本目录下。Oracle主目录和admin目录也位于Oracle基本目录下。
  • Oracle RAC的目录结构
    查看此主题以了解适用于Oracle RAC的示例最佳灵活体系结构（OFA）兼容数据库的分层目录树。
