2.5.1启动和停止守护进程

启动和停止守护程序并强制守护程序停止运行状况检查。

要启动和停止守护程序：

  1. 要启动守护程序，请使用–d start选项，如下所示：
    $ ./orachk -d start
    $ ./exachk -d start

    这些工具会提示您在启动期间提供所需的信息。

  2. 要停止守护程序，请使用–d stop选项，如下所示：
    $ ./orachk -d stop
    $ ./exachk -d stop

    如果运行stop命令时运行状况检查运行正在进行，则守护程序将指示并继续运行。

  3. 要强制守护程序停止运行状况检查，请使用–d stop_client选项：
    $ ./orachk -d stop_client
    $ ./exachk -d stop_client
守护程序停止运行状况检查，然后确认它何时完成。如有必要，请使用–d stop选项停止守护程序。
