253 UTL_ENCODE

UTL_ENCODE包提供将RAW数据编码为标准编码格式的功能，以便可以在主机之间传输数据。

您可以使用UTL_ENCODE函数对电子邮件文本的主体进行编码。该包还包含编码功能的解码对应功能。这些函数遵循已发布的编码标准，以适应发送端或接收端的非Oracle实用程序。

本章包含以下主题：

  • UTL_ENCODE子程序摘要

253.1.3 MIMEHEADER_DECODE函数

该函数接受“编码字”作为输入。

它使用以下形式：

=？<字符集>？<编码>？<编码文字>？= =？ISO-8859-1？Q＆这是一些编码文本？=

<encoded text>封装在mime头文件标签中，该标签为MIMEHEADER_DECODE函数提供有关如何解码字符串的信息。mime标头元数据标签从输入字符串中剥离， <encoded text>转换为基本数据库字符集，如下所示：

  • 如果这是UTF16平台，请将编码文本从UTF16转换为ASCII

  • 如果这是EBCDIC平台，请将编码文本从EBCDIC转换为ASCII

  • 如果这是ASCII或UTF8平台，则不需要转换

使用quoted-printable或base64解码对字符串进行解码，如编码字中的<encoding>元数据标记所指定。生成的转换和解码文本将作为VARCHAR2字符串返回给调用者。

句法

UTL_ENCODE.MIMEHEADER_DECODE（buf IN VARCHAR2 CHARACTER SET ANY_CS）RETURN data VARCHAR2 CHARACTER SET buf％CHARSET;

参数

表253-6 MIMEHEADER_DECODE函数参数

参数 描述

BUF

带有mime标头格式标签的编码文本数据。

返回值

表253-7 MIMEHEADER_DECODE函数返回值

返回 描述

数据

带有mime标头格式标签的编码文本数据

例子

V2：= utl_encode.mimeheader_decode（'=？ISO-8859-1？Q＆这是一些编码文本？='）;

253.1.4 MIMEHEADER_ENCODE函数

此函数将“编码字”作为输出返回。

输出采用以下形式：

=？<字符集>？<编码>？<编码文字>？= =？ISO-8859-1？Q＆这是一些文字？=

buf输入参数是要编码的文本，并成为<encoded text> 。

对于quoted-printable encode或base64编码，<encoding>值分别为“Q”或“B”。ENCODING输入参数接受UTL_ENCODE.QUOTED_PRINTABLE或UTL_ENCODE.BASE64或NULL作为有效值。如果为NULL ，则选择quoted-printable encoding作为默认值。

<charset>值被指定为输入参数encode_charset 。如果为NULL ，则选择数据库字符集作为默认值。

所述的MimeHeader编码处理包括转换buf输入字符串到由指定的字符集encode_charset参数。转换后的字符串被编码为quoted-printable或base64编码格式。附加并添加了mime标头标记。

最后，字符串将转换为数据库的基本字符集：

  • 如果这是UTF16平台，请将编码文本转换为UTF16

  • 如果这是EBCDIC平台，请将编码文本转换为EBCDIC

  • 如果这是ASCII或UTF8平台，则不需要转换。

句法

UTL_ENCODE.MIMEHEADER_ENCODE（buf IN VARCHAR2 CHARACTER SET ANY_CS，encode_charset IN VARCHAR2 DEFAULT NULL，编码IN PLS_INTEGER DEFAULT NULL）RETURN string VARCHAR2 CHARACTER SET buf％CHARSET;

253.1.7 TEXT_DECODE函数

此函数将输入文本转换为encode_charset参数指定的目标字符集，如果不是NULL 。

编码后的文本将转换为数据库的基本字符集，如下所示：

  • 如果这是UTF16平台，请将编码文本从UTF16转换为ASCII

  • 如果这是EBCDIC平台，请将编码文本从EBCDIC转换为ASCII

  • 如果这是ASCII或UTF8平台，则不需要转换

对于每个encoding参数，您可以使用quoted-printable或base64格式进行encoding 。如果为NULL ，则选择quoted-printable作为默认解码格式。如果encode_charset不为NULL ，则将字符串从指定的字符集转换为数据库字符集。生成的解码和转换后的文本字符串将返回给调用者。

句法

UTL_ENCODE.TEXT_DECODE（buf IN VARCHAR2 CHARACTER SET ANY_CS，encode_charset IN VARCHAR2 DEFAULT NULL，编码IN PLS_INTEGER DEFAULT NULL）RETURN string VARCHAR2 CHARACTER SET buf％CHARSET;

253.1.10 UUENCODE功能

此函数读取RAW输入字符串并将其编码为相应的uuencode格式字符串。

该函数的输出是累积的，因为它可以用于编码大数据流，通过将数据流分成可接受大小的RAW值，编码并连接成单个编码的字符串。

句法

UTL_ENCODE.UUENCODE（r IN RAW，输入IN PLS_INTEGER DEFAULT 1，文件名IN VARCHAR2 DEFAULT NULL，权限IN VARCHAR2 DEFAULT NULL）返回原始状态;

编译指示

编译指示RESTRICT_REFERENCES（uuencode，WNDS，RNDS，WNPS，RNPS）;
