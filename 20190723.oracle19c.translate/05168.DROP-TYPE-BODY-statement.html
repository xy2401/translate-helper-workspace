14.21 DROP TYPE BODY声明

DROP TYPE BODY语句删除ADT， VARRAY类型或嵌套表类型的主体。

删除类型体时，类型规范仍然存在，您可以重新创建类型体。在重新创建主体之前，您仍然可以使用该类型，尽管您无法调用其成员函数。

先决条件

类型主体必须位于架构中，或者您必须具有DROP ANY TYPE系统特权。

句法

drop_type_body :: =


插图说明drop_type_body.eps

语义

模式

包含该类型的架构的名称。默认值：您的架构。

TYPE_NAME

要删除的类型主体的名称。

对type_name的限制

只有在没有类型或表依赖关系的情况下，才可以删除类型主体。

例

例14-38掉落ADT主体

该语句删除ADT主体data_typ1 。有关创建此ADT的示例，请参阅“ CREATE TYPE语句 ” 。

DROP TYPE BODY data_typ1;
