关于Oracle Grid Infrastructure软件修补程序级别

查看此主题以了解如何为Oracle ASM和Oracle Clusterware应用修补程序。

Oracle Grid Infrastructure的软件修补程序级别表示应用于Oracle Grid Infrastructure软件版本的所有一次性修补程序集，包括Oracle ASM。该版本是版本号，格式为主要版本，次要版本和补丁集版本号。例如，版本号为19.1.0.1，主要版本为19，次要版本为1，0.0为补丁集编号。使用一次性补丁时，主要版本和次要版本保持不变，但每次应用或回滚临时补丁时补丁级别都会更改。

与Oracle Grid Infrastructure的标准升级一样，在群集正常运行的任何给定时间点，群集中的所有节点都必须具有相同的软件版本和补丁级别。由于一次性补丁可以作为滚动升级应用，因此特定软件版本上的所有可能补丁级别都是相互兼容的。
