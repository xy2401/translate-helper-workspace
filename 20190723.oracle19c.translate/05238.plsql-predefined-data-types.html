E PL / SQL预定义数据类型

本附录按数据类型族分组STANDARD包预定义的数据类型和子类型。

常量

此常量定义可能的最大名称长度。

ORA_MAX_NAME_LEN常数PLS_INTEGER：= 128;

BFILE数据类型系列

类型BFILE是BFILE_BASE;

BLOB数据类型系列

BLOB类型是BLOB_BASE;子类型“BINARY LARGE OBJECT”是BLOB;

BOOLEAN数据类型系列

类型BOOLEAN是（FALSE，TRUE）;

CHAR数据类型系列

type VARCHAR2是新的CHAR_BASE;类型MLSLABEL是新的CHAR_BASE;类型UROWID是新的CHAR_BASE; DBMS_ID和DBMS_QUOTED_ID定义SQL，PL / SQL和用户的对象中的标识符长度。子类型DBMS_ID是VARCHAR2（ORA_MAX_NAME_LEN）;子类型DBMS_QUOTED_ID是VARCHAR2（ORA_MAX_NAME_LEN + 2）; DBMS_ID_30和DBMS_QUOTED_ID_30定义限制为30个字节的SQL对象的长度。子类型DBMS_ID_30是VARCHAR2（30）;子类型DBMS_QUOTED_ID_30是VARCHAR2（32）;子类型VARCHAR是VARCHAR2;子类型STRING是VARCHAR2;子类型LONG是VARCHAR2（32760）;子类型RAW是VARCHAR2;子类型“LONG RAW”是RAW（32760）;子类型ROWID是VARCHAR2（256）;子类型CHAR是VARCHAR2;子类型CHARACTER是CHAR;子类型“CHARACTER VARYING”是VARCHAR;子类型“CHAR VARYING”是VARCHAR;子类型“NATIONAL CHARACTER”是CHAR CHARACTER SET NCHAR_CS;子类型“NATIONAL CHAR”是CHAR CHARACTER SET NCHAR_CS;子类型“NCHAR”是CHAR CHARACTER SET NCHAR_CS;子类型“NVARCHAR2”是VARCHAR2 CHARACTER SET NCHAR_CS;

CLOB数据类型系列

类型CLOB是CLOB_BASE;子类型“CHARACTER LARGE OBJECT”是CLOB;子类型“CHAR LARGE OBJECT”是CLOB;子类型“NATIONAL CHARACTER LARGE OBJECT”是CLOB CHARACTER SET NCHAR_CS;子类型“NCHAR LARGE OBJECT”是CLOB CHARACTER SET NCHAR_CS;子类型“NCLOB”是CLOB CHARACTER SET NCHAR_CS;

日期数据类型系列

类型DATE是DATE_BASE;
类型TIMESTAMP是新的DATE_BASE;
类型“TIMESTAMP WITH TIME ZONE”是新的DATE_BASE;输入“INTERVAL YEAR TO MONTH”是新的DATE_BASE;输入“INTERVAL DAY TO SECOND”是新的DATE_BASE;键入“TIMESTAMP WITH LOCAL TIME ZONE”是新的DATE_BASE;子类型TIME_UNCONSTRAINED是TIME（9）;子类型TIME_TZ_UNCONSTRAINED是TIME（9）WITH TIME ZONE;子类型TIMESTAMP_UNCONSTRAINED是TIMESTAMP（9）;子类型TIMESTAMP_TZ_UNCONSTRAINED是TIMESTAMP（9）WITH TIME ZONE;子类型YMINTERVAL_UNCONSTRAINED是间隔年份（9）到月份;子类型DSINTERVAL_UNCONSTRAINED是INTERVAL DAY（9）TO SECOND（9）;子类型TIMESTAMP_LTZ_UNCONSTRAINED是TIMESTAMP（9）WITH LOCAL TIME ZONE;

NUMBER数据类型系列

类型NUMBER是NUMBER_BASE;子类型FLOAT是NUMBER; -  NUMBER（126）子类型REAL是FLOAT; -  FLOAT（63）子类型“DOUBLE PRECISION”是FLOAT;子类型INTEGER是NUMBER（38,0）;子类型INT是INTEGER;子类型SMALLINT是NUMBER（38,0）;子类型DECIMAL是NUMBER（38,0）;子类型NUMERIC是DECIMAL;子类型DEC是DECIMAL;子类型BINARY_INTEGER是INTEGER范围'-2147483647'.. 2147483647;子类型NATURAL是BINARY_INTEGER范围0..2147483647;子类型NATURALN是NATURAL不为空;子类型POSITIVE是BINARY_INTEGER范围1..2147483647;子类型POSITIVEN为POSITIVE不为空;子类型SIGNTYPE是BINARY_INTEGER范围'-1'.. 1; - 对于SIGN函数，子类型PLS_INTEGER是BINARY_INTEGER;类型BINARY_FLOAT是NUMBER;类型BINARY_DOUBLE是NUMBER;子类型SIMPLE_INTEGER是BINARY_INTEGER NOT NULL;子类型SIMPLE_FLOAT是BINARY_FLOAT NOT NULL;子类型SIMPLE_DOUBLE是BINARY_DOUBLE NOT NULL;
