了解群集配置选项

查看这些主题以了解Oracle Grid Infrastructure 19c中可用的群集配置选项。

  • 关于Oracle Standalone Clusters
    Oracle独立群集在本地托管所有Oracle Grid Infrastructure服务和Oracle ASM，并且需要直接访问共享存储。
  • 关于Oracle Cluster Domain和Oracle Domain Services Cluster
    Oracle Cluster Domain是Oracle Clusterware 12c第2版中引入的新集群部署体系结构的一种选择。
  • 关于Oracle成员集群
    Oracle成员集群使用Oracle域服务集群中的集中服务，可以托管数据库或应用程序。Oracle成员集群可以有两种类型 - Oracle数据库的Oracle成员集群或应用程序的Oracle成员集群。
  • 关于Oracle Extended Clusters
    Oracle扩展群集由位于多个位置（称为站点）的节点组成。
