5.59 DBA_HIST_SQL_BIND_METADATA

DBA_HIST_SQL_BIND_METADATA显示有关SQL游标使用的绑定变量的元数据的历史信息。

柱 数据类型 空值 描述

DBID

数

不是NULL

快照的数据库ID

SQL_ID

VARCHAR2（13）

不是NULL

库高速缓存中父游标的SQL标识符

名称

VARCHAR2（128）

绑定变量的名称

位置

数

不是NULL

绑定变量在SQL语句中的位置

DUP_POSITION

数

如果绑定是通过名称执行的，并且绑定变量是重复的，则此列将给出主绑定变量的位置

数据类型

数

绑定数据类型的内部标识符。从Oracle Database 12 c开始，表示PL / SQL数据类型的数字可以出现在此列中。

DATATYPE_STRING

VARCHAR2（15）

绑定数据类型的文本表示。Oracle数据库12c的开始，一个PL / SQL-仅数据类型的文本表示可以显示在此列。如果实际数据类型是PL / SQL子类型，则将显示数据类型的名称，而不是子类型。

CHARACTER_SID

数

国家字符集标识符

精确

数

精度（用于数字绑定）

规模

数

比例（用于数字绑定）

最长长度

数

最大绑定长度

CON_DBID

数

采样会话的PDB的数据库ID

CON_ID

数

CON_DBID标识的容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
