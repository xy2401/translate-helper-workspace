第III部分与XML数据之间的关系数据

您可以通过多种方式将现有关系数据视为XML数据。

  • 从关系数据生成XML数据
    Oracle XML DB提供了从数据库中的关系数据生成（构造）XML数据的功能。有SQL / XML标准函数和Oracle特定的函数和包，用于从关系内容生成XML数据。
  • XML数据的关系视图
    XML数据的关系数据库视图提供对XML内容的传统关系访问。
  • XMLType视图
    您可以在关系数据和对象关系数据上创建XMLType视图。
