6.182 DM_USER_MODELS

DM_USER_MODELS显示有关用户架构中的模型的信息。

柱 数据类型 空值 描述

名称

VARCHAR2（128）

不是NULL

模型的名称

FUNCTION_NAME

VARCHAR2（30）

型号功能：

  • association - 关联是一种描述性挖掘功能。关联模型识别关系及其在数据集中出现的概率。

  • attribute_importance - 属性重要性是一种预测挖掘功能。属性重要性模型识别属性在预测给定结果中的相对重要性。

  • classification - 分类是一种预测挖掘功能。分类模型使用历史数据来预测新的离散或分类数据。

    classification功能也可用于异常检测。在这种情况下，使用具有空目标的SVM算法（One-Class SVM）。

  • clustering - 群集是一种描述性挖掘功能。聚类模型识别数据集内的自然分组。

  • feature_extraction - 特征提取是一种描述性挖掘功能。特征提取模型创建基于模型的优化数据集。

  • regression - 回归是一种预测性挖掘函数。回归模型使用历史数据来预测新的连续数字数据。

ALGORITHM_NAME

VARCHAR2（30）

模型使用的算法：

  • algo_name - 指定模型使用的算法的设置。

  • asso_max_rule_length - 指定关联模型使用的规则的最大长度的设置。

  • asso_min_confidence - 指定关联模型的最小置信度的设置。

  • asso_min_support - 指定对关联模型的最低支持的设置。

  • clas_cost_table_name - 设置，指定分类模型的成本矩阵表的名称。

  • clas_priors_table_name - 设置，指定NB和ABN模型的先验概率表的名称。决策树是唯一不使用先验的分类算法。

    对于SVM分类模型，此设置指定权重表的名称。

  • clus_num_clusters - 指定群集模型的群集数的设置。

  • feat_num_features - 指定要素选择模型的要素数的设置。
