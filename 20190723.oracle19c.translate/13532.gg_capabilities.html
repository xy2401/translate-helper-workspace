Part IV Oracle GoldenGate Capabilities in Oracle Database

You can manage Oracle GoldenGate capabilities, such as conflict detection and resolution and procedural replication, in Oracle Database.

  • Oracle GoldenGate Automatic Conflict Detection and Resolution
    You can configure and manage Oracle GoldenGate automatic conflict detection and resolution in Oracle Database.
  • Oracle GoldenGate Procedural Replication
    Procedural replication is configured mainly within Oracle GoldenGate. However, you can display the procedural replication configuration and complete some configuration tasks within the database.
