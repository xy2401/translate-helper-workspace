关于Oracle Grid Infrastructure仅限软件的安装

手动安装和配置Oracle Grid Infrastructure的软件二进制文件。

Oracle建议只有高级用户才能执行纯软件安装，因为此安装方法不提供安装验证，此安装选项需要手动安装后步骤才能为独立服务器软件启用Oracle Grid Infrastructure。

执行纯软件安装需要执行以下步骤：

  1. 安装软件二进制文件。

  2. 配置软件二进制文件。
