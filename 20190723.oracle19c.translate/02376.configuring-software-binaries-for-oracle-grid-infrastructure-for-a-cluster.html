为群集配置Oracle Grid Infrastructure的软件二进制文件

通过在GUI模式下启动Oracle Grid Infrastructure配置向导来配置软件二进制文件。

  1. 以Oracle Grid Infrastructure安装所有者身份登录到群集节点，并将目录更改为Grid_home 。
  2. 启动Oracle Grid Infrastructure配置向导：
    $ ./gridSetup.sh
  3. 根据配置提供信息。OUI验证信息并在所有集群节点上配置安装。
  4. 完成提供信息后，OUI会显示“摘要”页面，列出您为群集提供的信息。验证摘要是否具有群集的正确信息，然后单击“ 安装”以开始配置本地节点。
    完成本地节点的配置后，OUI会将Oracle Grid Infrastructure配置文件复制到其他集群成员节点。
  5. 出现提示时，运行root脚本。
  6. 确认所有root脚本都已运行后，OUI将检查群集配置状态，并根据需要启动其他配置工具。
