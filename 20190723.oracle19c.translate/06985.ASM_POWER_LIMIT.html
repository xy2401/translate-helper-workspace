1.20 ASM_POWER_LIMIT

ASM_POWER_LIMIT指定Oracle ASM实例上用于磁盘重新平衡的最大功率。

属性 描述

参数类型

整数

默认值

1

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

没有

价值范围

0到11 英尺1

Oracle RAC

多个实例可以具有不同的值。

脚注1

从Oracle Database 11g第2版（11.2.0.2）开始，如果是COMPATIBLE 。 ASM磁盘组属性设置为11.2.0.2或更高，则值范围为0到1024。

注意：

  • 此参数只能在Oracle自动存储管理（Oracle ASM）实例中指定。

  • 在Oracle Exadata或Oracle数据机环境中，您无法将此参数设置为0。

限制越高，重新平衡就越快。较低的值将花费更长的时间，但消耗更少的处理和I / O资源。

如果未指定重新平衡操作的POWER子句，则默认功率将为ASM_POWER_LIMIT的值。
