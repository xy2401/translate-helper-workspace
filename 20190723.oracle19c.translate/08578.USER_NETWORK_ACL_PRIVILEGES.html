6.520 USER_NETWORK_ACL_PRIVILEGES

USER_NETWORK_ACL_PRIVILEGES描述当前用户访问网络主机的网络权限的状态。

注意：

在Oracle Database 12 c第1版（12.1）中不推荐使用此USER_NETWORK_ACL_PRIVILEGES视图。Oracle建议您使用USER_HOST_ACES视图。

柱 数据类型 空值 描述

主办

VARCHAR2（1000）

不是NULL

网络主机

LOWER_PORT

NUMBER（5）

端口范围的下限

UPPER_PORT

NUMBER（5）

端口范围的上限

特权

CHAR（128）

网络权限

状态

VARCHAR2（7）

特权状态：

  • 被拒绝

  • GRANTED
