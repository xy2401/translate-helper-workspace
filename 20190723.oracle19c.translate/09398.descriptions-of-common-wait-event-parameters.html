C.2公共等待事件参数的描述

本节介绍一些常见的等待事件参数。

块＃

这是Oracle需要等待的块的块编号。块编号相对于文件的开头。要查找此块所属的对象，请发出以下SQL语句：

从dba_extents中选择segment_name，segment_type，owner，tablespace_name，其中file_id = file＃和block＃在block_id和block_id + blocks  -  1之间;

块

从文件读取或写入的块数。块大小取决于文件类型：

  • 数据库文件的块大小为DB_BLOCK_SIZE

  • 日志文件和控制文件的块大小等于平台的物理块大小

打破？

如果此参数的值等于0，则会向客户端发送重置。非零值表示已将中断发送到客户端。

类

块的类描述了如何使用块的内容。例如，类1表示数据块，类2表示排序块。使用此SQL查询来查看块类：

SELECT view_definition FROM V_ $ FIXED_VIEW_DEFINITION WHERE view_name ='GV $ WAITSTAT';

DBA

首字母“dba”表示数据块地址，它由文件号和块号组成。

司机身份

当前正在使用的驱动程序的断开功能的地址。

文件＃

以下查询返回数据库文件的名称：

select * from v $ datafile where file＃= file＃ ;

ID1

enqueue或global lock的第一个标识符（ id1 ）从P2或P2RAW获取其值。标识符的含义取决于名称（P1）。

ID2

enqueue或global lock的第二个标识符（ id2 ）从P3或P3RAW获取其值。标识符的含义取决于名称（P1）。

乐

相对索引号为V$GC_ELEMENT 。

命名空间

对象命名空间的名称，因为它显示在V$DB_OBJECT_CACHE视图中。

要求

“请求”的I / O数。这与块数不同，因为一个请求可能包含多个块。

会议＃

非活动会话的编号。使用以下SQL语句查找有关会话的更多信息：
