APPROX_PERCENTILE_DETAIL

句法


附图说明approx_percentile_detail.eps

目的

APPROX_PERCENTILE_DETAIL计算expr值的近似百分位数信息，并返回一个称为详细信息的BLOB值，该值包含特殊格式的信息。

expr的可接受数据类型取决于您使用DETERMINISTIC子句指定的算法。有关更多信息，请参阅DETERMINISTIC子句。

此函数通常与SELECT语句中的GROUP BY子句一起使用。它计算每组行中expr近似百分位数信息，并返回每个组的单个详细信息。

APPROX_PERCENTILE_DETAIL返回的详细信息可用作APPROX_PERCENTILE_AGG函数的输入，该函数可用于执行详细信息的聚合，或TO_APPROX_PERCENTILE函数，可将细节转换为指定的百分位值。您可以将这三个函数一起使用，以执行一次资源密集型近似百分位计算，存储生成的详细信息，然后对这些详细信息执行有效的聚合和查询。例如：

  1. 使用APPROX_PERCENTILE_DETAIL函数执行近似百分位数计算，并将结果详细信息存储在表格或物化视图中。这些可能是高度细粒度的百分位细节，例如城市的收入百分位信息。

  2. 使用APPROX_PERCENTILE_AGG函数聚合上一步中获取的详细信息，并将结果详细信息存储在表或物化视图中。这些可能是较低粒度的细节，例如州的收入百分位信息。

  3. 使用TO_APPROX_PERCENTILE函数将存储的详细信息值转换为百分位值。您可以使用TO_APPROX_PERCENTILE函数查询APPROX_PERCENTILE_DETAIL函数或APPROX_PERCENTILE_AGG函数创建的详细信息值。

确定性

此子句允许您控制用于计算近似百分位值的算法类型。

  • 如果指定DETERMINISTIC ，则此函数计算确定性近似百分位数信息。在这种情况下， expr必须求值为数值或可以隐式转换为数值的值。

  • 如果省略DETERMINSTIC ，则此函数计算非确定性近似百分位信息。在这种情况下， expr必须求值为numeric或datetime值，或者是可以隐式转换为numeric或datetime值的值。

例子

本节中的示例演示了如何一起使用APPROX_PERCENTILE_DETAIL ， APPROX_PERCENTILE_AGG和TO_APPROX_PERCENTILE函数来执行一次资源密集型近似百分位计算，存储生成的详细信息，然后对这些详细信息执行有效的聚合和查询。

APPROX_PERCENTILE_DETAIL：示例

以下语句查询表sh 。 customers和sh 。 sales给每个客户的产品的货币金额的sales 。APPROX_PERCENTILE_DETAIL函数返回客户所在的每个城市的详细信息，名为city_detail 。返回的详细信息存储在名为amt_sold_by_city_mv的物化视图中。

CREATE MATERIALIZED VIEW amt_sold_by_city_mv ENABLE QUERY REWRITE AS SELECT c.country_id country，c.cust_state_province state，c.cust_city city，APPROX_PERCENTILE_DETAIL（s.amount_sold）city_detail FROM customers c，sales s WHERE c.cust_id = s.cust_id GROUP BY c.country_id ，c.cust_state_province，c.cust_city;

APPROX_PERCENTILE_AGG：示例

以下语句使用APPROX_PERCENTILE_AGG函数读取存储在amt_sold_by_city_mv详细信息，并创建汇总的详细信息，其中包含销售给每个州的客户的产品的金额。这些聚合的详细信息存储在名为amt_sold_by_state_mv的物化视图中。

CREATE MATERIALIZED VIEW amt_sold_by_state_mv AS SELECT country，state，APPROX_PERCENTILE_AGG（city_detail）state_detail FROM amt_sold_by_city_mv GROUP BY country，state;

以下声明与之前的声明类似，不同之处在于它创建的汇总详细信息包含每个国家/地区向客户销售的产品的近似货币金额。这些聚合的详细信息存储在名为amt_sold_by_country_mv的物化视图中。

CREATE MATERIALIZED VIEW amt_sold_by_country_mv AS SELECT country，APPROX_PERCENTILE_AGG（city_detail）country_detail FROM amt_sold_by_city_mv GROUP BY country;
