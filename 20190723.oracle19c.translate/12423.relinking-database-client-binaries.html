安装后重新链接Oracle数据库客户端二进制文件

安装Oracle Database Client后，如果需要，可以使用relink as_installed选项修改二进制文件。

例如，您可能希望每次应用操作系统修补程序或操作系统升级后重新链接Oracle数据库客户机二进制文件。

警告：

在重新链接可执行文件之前，必须关闭正在重新链接的Oracle主目录中运行的所有可执行文件。此外，关闭与Oracle共享库链接的应用程序。
  1. 以Oracle数据库客户机所有者用户（ oracle ）身份登录。
  2. 设置ORACLE_HOME环境变量
    $ ORACLE_HOME = / u01 / app / oracle / product / 19.0.0 / client_1
  3. 转到$ORACLE_HOME/bin目录：
    $ cd $ ORACLE_HOME / bin
  4. 使用as_installed选项运行relink脚本以重新链接二进制文件。
    $ ./relink as_installed
    重新链接已完成，日志文件在$ORACLE_HOME/install目录下生成。
