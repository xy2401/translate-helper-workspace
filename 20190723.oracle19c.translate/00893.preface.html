前言

Oracle Database Gateway for APPC通过远程过程调用（RPC）处理为Oracle应用程序提供对IBM大型机数据和服务的无缝访问。

此网关版本支持的UNIX平台是：

  • POWER Systems上的IBM AIX（64位）

  • Linux x86-64

  • SPARC上的Oracle Solaris（64位）

有关经过认证的硬件平台和操作系统版本的最新列表，请参阅Oracle数据库安装指南和My Oracle Support网站上的认证表。My Oracle Support Web站点位于

https://support.oracle.com

目标受众

如果您负责以下任务，请阅读本指南：

  • 确定硬件和软件要求。

  • 为APPC安装，配置或管理Oracle Database Gateway。

  • 使用SNA通信协议或TCP / IP for IMS Connect通信协议开发通过网关访问远程主机数据库的应用程序。

  • 确定安全要求。

  • 确定和解决问题。

在使用本指南管理网关之前，您应该了解适用于您的平台和Oracle数据库网关的操作系统的基础知识。

约定

本文档中使用以下文本约定：

惯例 含义

黑体字

粗体字表示与动作相关联的图形用户界面元素，或在文本或术语表中定义的术语。

斜体

斜体类型表示您为其提供特定值的书名，重点或占位符变量。

等宽

等宽字体类型表示段落中的命令，URL，示例中的代码，屏幕上显示的文本或您输入的文本。

命令语法

UNIX命令语法以monospace字体显示。美元字符（$），数字符号（＃）或百分号（％）是UNIX命令提示符。不要将它们作为命令的一部分输入。本指南中使用以下命令语法约定：

惯例 描述

反斜杠

反斜杠是UNIX命令延续字符。它用于命令示例，这些示例太长而无法放在一行上。输入显示的命令（带反斜杠）或在没有反斜杠的单行中输入：

dd if = / dev / rdsk / c0t1d0s6 = = dev / rst0 bs = 10b \ count = 10000

大括号{}

大括号表示所需物品：

。DEFINE {macro1}

括号[]

括号表示可选项：

cvtcrt termname [ outfile ]

省略号...

省略号表示任意数量的类似项目：

CHKVAL fieldname value1 value2 ... valueN

斜体

斜体类型表示变量。替换变量的值：

LIBRARY_NAME

垂直线|

垂直线表示括号或括号内的选择：

文件大小 [K | M]
