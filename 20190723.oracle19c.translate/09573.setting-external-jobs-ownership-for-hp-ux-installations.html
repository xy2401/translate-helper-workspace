为HP-UX安装设置外部作业所有权

仅在HP-UX平台上，将外部作业所有权设置为低权限用户。

完成以下过程以将外部作业所有权设置为低权限用户extjob ：
  1. 以root身份登录。
  2. 转到Oracle数据库Oracle主目录：
    #cd $ ORACLE_HOME / rdbms / admin /
  3. 使用文本编辑器打开externaljob.ora ，找到run_user和run_group参数。
    仅以root身份修改externaljob.ora 。
  4. 将run_user设置为外部作业用户（ extjob ），并将run_group设置为低权限组，例如other组，例如：
    run_user = extproc run_group = other
  5. 保存文件。
