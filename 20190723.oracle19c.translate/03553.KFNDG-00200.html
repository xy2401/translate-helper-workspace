44 KFNDG-00200至KFNDG-00701

KFNDG-00200：语法无效

原因：指定了无效的renamedg语法。

操作：尝试使用renamedg -help获取更多信息。

KFNDG-00201：参数无效

原因：为renamedg指定了无效的键或值。

操作：尝试使用renamedg -help获取更多信息。

KFNDG-00202：参数阶段的值无效（ string ）

原因：指定了参数PHASE的值不正确。

操作：参数PHASE的有效值为ONE，TWO，BOTH。使用正确的值重试。

KFNDG-00203：磁盘组名称（ string ）超出允许的最大长度number

原因：指定了无效的磁盘组名称。

操作：磁盘组名称最多可包含32个字符。

KFNDG-00204：磁盘组名称string包含无效字符

原因：指定了无效的磁盘组名称。

操作：磁盘组名称只能包含字母，数字和下划线（_）字符。

KFNDG-00205：无法解释配置文件名

原因：指定了无效的配置文件名。

操作：验证配置文件名和文件夹权限，然后重试renamedg。

KFNDG-00301：新磁盘组（ string ）已存在，忽略重命名

原因： NEWDGNAME中指定的磁盘组已存在。

行动：没有

KFNDG-00302：disk（ string ： string ）不包含有效标头

原因：无法找到或损坏设备头。

操作：验证提到的设备是否为ASM磁盘设备。

KFNDG-00303：无法更新string磁盘组名称

原因：磁盘中的权限不足或写入错误。

操作：验证磁盘权限并重试重命名。另外，请参阅日志以获取其他信息。

KFNDG-00304：无法访问设备string heartbeat块： string

原因：磁盘的心跳块已损坏。

操作：使用ASM实用程序验证磁盘头，重试renamedg。

KFNDG-00305：无法创建配置文件

原因：权限不足或路径无效。

操作：验证文件夹权限并重试该命令。

KFNDG-00306：无法写入配置文件

原因：权限不足。

操作：检查文件权限并重试重命名。

KFNDG-00307：无法读取配置文件

原因：权限不足或磁盘读取错误。

操作：检查文件权限并重试重命名。

KFNDG-00308：无法处理配置文件条目： string

原因：配置文件已损坏。

操作：检查文件信息并重试重命名。

KFNDG-00401：分配单位大小冲突（ number ， number ）

原因：磁盘组和磁盘的分配单元大小不匹配。

操作：验证设备是否已损坏并重试重命名。

KFNDG-00402：ASM元数据块大小冲突（ number ， number ）

原因：磁盘组和磁盘的元数据块大小不匹配。

操作：验证设备是否已损坏并重试重命名。

KFNDG-00403：未发现磁盘组string所有磁盘

原因：指定磁盘组中存在Offine磁盘。

操作：有关更多详细信息，请参阅Oracle数据库存储管理员指南。

KFNDG-00404：磁盘组string包含脱机磁盘，忽略重命名

原因：无法重命名具有脱机磁盘的磁盘组。

操作：有关更多详细信息，请参阅Oracle数据库存储管理员指南。

KFNDG-00405：指定的磁盘组string似乎已挂载

原因：已装入磁盘组。

操作：卸载磁盘组并重试重命名。

KFNDG-00406：CSS正在使用磁盘组（ string ）

原因：群集同步服务（CSS）正在使用要重命名的磁盘组。

操作：请参阅crsctl命令以从磁盘组重新定位表决磁盘。

KFNDG-00407：找不到磁盘组string磁盘

原因：没有为磁盘组发现磁盘。

操作：尝试使用不同的发现字符串重命名。

KFNDG-00408：无法发现disk（ string ： string ）

原因：磁盘已被删除或移动。

行动：没有

KFNDG-00409：磁盘（ string ）不属于磁盘组（ string ）

原因：磁盘不是指定磁盘组的一部分。

行动：没有

KFNDG-00410：无法发现文件夹string设备

原因：权限不足或发现字符串无效。

操作：请参阅日志以获取其他详细信

KFNDG-00600：内部错误string

原因：执行renamedg时发生内部错误。

操作：联系Oracle支持服务。

KFNDG-00601：致命错误string

原因：执行renamedg时发生致命错误。

操作：联系Oracle支持服务。

KFNDG-00603：无法初始化设备发现

原因：与通用服务层通信时出错。

操作：联系Oracle支持服务。

KFNDG-00700：发生系统相关错误（ number ， string ， string ， string ）

原因：发生内部系统错误。

操作：联系Oracle支持服务。

KFNDG-00701：错误number在执行I / O

原因： I / O故障。

操作：联系Oracle支持服务。
