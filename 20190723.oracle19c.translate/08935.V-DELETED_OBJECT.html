7.174 V $ DELETED_OBJECT

V$DELETED_OBJECT显示有关已删除的归档日志，数据文件副本和控制文件中的备份片段的信息。

此视图的唯一目的是优化恢复目录重新同步操作。删除存档日志，数据文件副本或备份片段时，相应的记录将被标记为已删除。

柱 数据类型 描述

RECID

数

删除了对象记录ID

邮票

数

删除了对象记录戳记

类型

VARCHAR2（26）

标识已删除对象的类型：

  • 存档日志

  • 备份PIECE

  • 数据文件复制

  • 代理复制

  • 备份件可用

  • BACKUP PIECE EXPIRED

  • PROXY COPY AVAILABLE

  • PROXY COPY EXPIRED

  • BACKUP PIECE UNAVAILABLE

  • PROXY COPY UNAVAILABLE

  • 数据文件复制可用

  • DATAFILE COPY EXPIRED

  • 数据文件无法复制

  • 存档日志可用

  • 已归档日志已过期

  • 归档日志无法使用

  • 备份设置保留选项

  • BACKUP SET KEEP UNTIL

  • PROXY COPY KEEP OPTIONS

  • PROXY COPY KEEP UNTIL

  • 数据文件复制保留选项

  • DATAFILE COPY KEEP UNTIL

  • RESTRE上的DATAFILE重命名

OBJECT_RECID

数

记录已删除对象的ID

OBJECT_STAMP

数

记录已删除对象的时间戳

OBJECT_DATA

数

显示与此已删除对象相关的其他内部信息。仅供内部Oracle使用。

SET_STAMP

数

设置已删除对象的戳记

SET_COUNT

数

设置已删除对象的计数

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
