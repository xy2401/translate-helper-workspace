47 LCD-00100至LCD-00219

LCD-00100：内部错误，参数[ number ]

原因：发生内部错误

操作：立即向系统管理员报告

LCD-00111：值不在合法范围内[ string ]

原因：参数的值在界外

操作：有关参数的界限，请参阅手册

LCD-00112：非法整数基数规范[ string ]

原因：自我解释

行动：有效字符为'd'，'h'，'D'和'H'

LCD-00113：整数转换错误或负整数[ string ]

原因：为整数参数分配了错误或负整数

行动：自我解释

LCD-00114：非法布尔响应[ string ]

原因：您对布尔参数的值大于TRUE或FALSE

操作：布尔参数仅采用TRUE或FALSE

LCD-00122：无法识别的关键字[ string ]

原因：您拼错了参数名称

操作：正确拼写参数名称

LCD-00128：位置参数值无效[ string ]

原因：输入了无效的位置参数值

操作：删除无效的位置参数

LCD-00161：ORACLE错误（可能的语法错误）参数[ string ]

原因：可能的语法错误。其他消息如下。

操作：更正错误参数。

LCD-00201：无法打开指定的文件名[ string ]

原因：指定的文件不存在

操作：创建适当的参数文件

LCD-00203：缺少关键字[ string ]

原因：预计会出现关键字但未找到关键字

操作：添加关键字后跟'='符号和参数值

LCD-00204：左父，没有指定参数[ string ]

原因：已启动参数列表但未指定参数

操作：添加适当的值列表并关闭参数列表

LCD-00205：不平衡括号[ string ]

原因：自我解释

行动：平衡括号

LCD-00206：关键字[ string ]后输入的位置参数

原因：缺少'='符号

操作：所有关键字必须后跟'='符号

LCD-00207：遇到嵌套括号[ string ]

原因：只允许使用一组括号

操作：删除嵌套的括号

LCD-00208：值列表中的意外关键字[ string ]

原因：找到关键字而不是值

操作：为给定参数插入适当的值

LCD-00209：字符串末尾的关键字缺失值[ string ]

原因：指定的关键字没有值

操作：为关键字指定有效值

LCD-00210：非法赋值运算符[ string ]

原因：您使用“=”以外的其他符号为关键字指定值

操作：仅使用“=”符号

LCD-00211：意外分隔符[ string ]

原因：在值之间找到了无效的分隔符

操作：逗号或空格是有效的分隔符

LCD-00212：失控引用字符串[ string ]

原因：参数值在左侧但不在右侧引用

操作：引用两侧的参数值

LCD-00215：嵌套参数文件太深[ string ]

原因：文件参数的间接层次太多。最高为3。

操作：将间接级别降低到3以下

LCD-00217：处理文件参数失败[ string ]

原因：处理指定的文件参数时发生错误

行动：具体细节将出现在其他消息中

LCD-00218：文件错误[ string ]

原因：特定文件中发生错误。

操作：请参阅其他消息以了解特定错误。

LCD-00219：嵌套包含文件[ string ]太大

原因：参数文件的大小有限制，并且已超出。此限制现在为8192个字节。

操作：从文件中删除字符以使其变小，或使用其他级别的嵌套。
