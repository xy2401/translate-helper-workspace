4.304 DBA_HIST_COLORED_SQL

DBA_HIST_COLORED_SQL显示已标记为AWR SQL捕获的SQL ID。

如果使用DBMS_WORKLOAD_REPOSITORY.ADD_COLORED_SQL过程对SQL语句进行着色，则AWR将始终捕获有色SQL ID的SQL统计信息。可以使用DBMS_WORKLOAD_REPOSITORY.REMOVE_COLORED_SQL过程从着色中删除SQL语句。

数

不是NULL

数据库ID

SQL_ID

VARCHAR2（13）

不是NULL

彩色SQL语句的SQL ID

CREATE_TIME

日期

不是NULL

SQL ID被着色的时间

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
