更改第一个节点以进行安装和升级

如果第一个节点无法访问，则可以强制另一个节点成为安装或升级的第一个节点。

在安装期间，如果root.sh无法在第一个节点上完成，请使用-force选项在另一个节点上运行以下命令：

root.sh -force -first

升级：

rootupgrade.sh -force -first
