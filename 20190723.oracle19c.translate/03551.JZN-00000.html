JZN-00000：正常，成功完成

原因：操作成功完成。

行动：无需采取任何行动。

原因：处理已到达可用输入的末尾。

JZN-00002：参数无效

原因：输入参数不是缓冲区，文件或流。

操作：这是JavaScript Object Notation（JSON）工具的调用者的编程错误。联系Oracle支持服务。

JZN-00003：未指定输入

原因：在未指定输入的情况下请求了解析或解码操作。

操作：在尝试解析或解码输入之前，将缓冲区，文件或流指定为输入。

JZN-00004：无法打开输入文件

原因：找不到请求的输入文件，或者无法读取。

操作：确保文件存在且读取权限可用。

原因： JavaScript Object Notation（JSON）解析器遇到语法无效的JSON。

操作：更正JSON输入文档。

JZN-00006：转义序列无效

原因： JavaScript Object Notation（JSON）解析器在反斜杠字符后遇到无效的字符序列。

操作：更正JSON输入并删除或更正无效的转义序列。

原因： JavaScript Object Notation（JSON）解析器检测到重复的成员密钥。

JZN-00008：Parse已在进行中

原因：尝试在解析操作正在进行时开始解析输入。
