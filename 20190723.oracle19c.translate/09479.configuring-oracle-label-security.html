配置Oracle Label Security

安装后，必须在使用之前在数据库中配置Oracle Label Security。

您可以通过两种方式配置Oracle Label Security：使用Oracle Internet Directory集成，或者不使用Oracle Internet Directory集成。

表4-1安装后配置Oracle Label Security的选项

组态 需求

与Oracle Internet Directory集成

要通过Oracle Internet Directory集成配置Oracle Label Security，必须在您的环境中安装Oracle Internet Directory，并且必须在目录中注册Oracle数据库。

没有Oracle Internet Directory集成

如果在没有Oracle Internet Directory集成的情况下配置Oracle Label Security（OLS），则无法将其配置为在以后使用Oracle Internet Directory。要在以后的数据库上使用Oracle Internet Directory配置Oracle Label Security，必须删除数据库上的OLS选项，然后使用Oracle Internet Directory集成选项配置OLS。
