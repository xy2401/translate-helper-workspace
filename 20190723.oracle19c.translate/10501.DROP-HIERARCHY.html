DROP HIERARCHY

目的

使用DROP HIERARCHY语句删除层次结构。HIERARCHY对象是分析视图的一个组件。

先决条件

要在您自己的模式中删除层次结构，您必须具有DROP HIERARCHY系统特权。要在另一个用户的模式中删除层次结构，您必须具有DROP ANY HIERARCHY系统特权。

句法

drop_hierarchy :: =


插图说明drop_hierarchy.eps

语义

模式

指定层次结构所在的模式。如果未指定架构，则Oracle数据库会在您自己的架构中查找层次结构。

hierarchy_name

指定要删除的层次结构的名称。

例

以下语句删除指定的层次结构对象：

DROP HIERARCHY product_hier;
