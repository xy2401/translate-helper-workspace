13.41祈求者的权利和定义者的权利条款

指定存储的PL / SQL子程序的AUTHID属性。AUTHID属性会影响设备在运行时发出的SQL语句的名称解析和权限检查。

invoker_rights_clause可以出现在以下SQL语句中：

  • ALTER TYPE语句

  • 创建功能声明

  • CREATE PACKAGE声明

  • CREATE PROCEDURE声明

  • CREATE TYPE语句

  • CREATE TYPE BODY声明

语义

invoker_rights_clause

当它出现在包声明中时，它指定包中函数和过程的AUTHID属性，以及包规范中声明的显式游标。

当它出现在独立函数声明中时，它指定函数的AUTHID属性。

当它出现在独立过程声明中时，它指定过程的AUTHID属性。

invoker_rights_clause只能在子程序声明中出现一次。

当它出现在ADT中时，它指定ADT的成员函数和过程的AUTHID属性。

对invoker_rights_clause的限制

以下限制适用于类型：

  • 此子句仅对ADT有效，而不适用于嵌套表或VARRAY类型。

  • 如果要创建子类型，则可以明确指定此子句。但是，子类型继承其超类型的AUTHID属性，因此您不能指定与为超类型指定的值不同的值。

  • 如果使用AUTHID DEFINER创建超类型，则必须在与超类型相同的模式中创建子类型。
