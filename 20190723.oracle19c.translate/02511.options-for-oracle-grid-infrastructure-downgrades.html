Oracle Grid Infrastructure的选项降级

您可以将Oracle Grid Infrastructure 19c降级到早期版本。

降级选项包括以下早期版本：

  • Oracle Grid Infrastructure降级为Oracle Grid Infrastructure 18c。

  • Oracle Grid Infrastructure降级到Oracle Grid Infrastructure 12c第2版（12.2）。

  • Oracle Grid Infrastructure降级到Oracle Grid Infrastructure 12c第1版（12.1）。

  • Oracle Grid Infrastructure降级到Oracle Grid Infrastructure 11g第2版（11.2）。由于Oracle Grid Infrastructure 19c中的所有群集配置均为Oracle Flex Clusters，因此在降级到Oracle Grid Infrastructure 11g第2版（11.2）时，会将Oracle Flex群集配置降级为标准群集配置。

注意：

当您将Oracle Grid Infrastructure降级到早期版本（例如从Oracle Grid Infrastructure 19c降级到Oracle Grid Infrastructure 18c）时，已经在Oracle Grid Infrastructure中注册的更高版本的RAC数据库将在降级后无法启动。
