8 JSON数据的复制

您可以使用Oracle GoldenGate复制包含包含JSON数据的列的表。

请注意，Oracle GoldenGate要求复制的表具有非虚拟主键列; 主键列不能是虚拟的 。

JSON数据上的所有索引也将被复制。但是，您必须在副本数据库上执行用于维护JSON搜索索引的任何Oracle Text操作。以下是此类程序的示例：

  • CTX_DDL.sync_index

  • CTX_DDL.optimize_index

也可以看看：

  • 有关Oracle GoldenGate的信息，请访问Oracle GoldenGate

  • 有关CTX_DDL.sync_index信息，请CTX_DDL.sync_index Oracle Text Reference

  • 有关CTX_DDL.optimize_index信息，请CTX_DDL.optimize_index Oracle Text Reference
