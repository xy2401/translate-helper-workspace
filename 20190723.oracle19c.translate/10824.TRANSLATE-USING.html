翻译......使用

句法


说明translate_using.eps的说明

目的

TRANSLATE ......USING将char转换为为数据库字符集和国家字符集之间的转换指定的字符集。

注意：

TRANSLATE ......USING功能主要支持ANSI兼容性。Oracle建议您根据需要使用TO_CHAR和TO_NCHAR函数将数据转换为数据库或国家字符集。TO_CHAR和TO_NCHAR可以将比TRANSLATE更多种类的数据类型作为参数...USING ，仅接受字符数据。

char参数是要转换的表达式。

  • 指定USING CHAR_CS参数会将char转换为数据库字符集。输出数据类型为VARCHAR2 。

  • 指定USING NCHAR_CS参数会将char转换为国家字符集。输出数据类型为NVARCHAR2 。

此函数类似于Oracle CONVERT函数，但如果输入或输出数据类型用作NCHAR或NVARCHAR2 ，则必须使用该函数而不是CONVERT 。如果输入包含UCS2代码点或反斜杠字符（\），则使用UNISTR函数。
