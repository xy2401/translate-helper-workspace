操作系统一般要求

如果您没有安装OpenSSH作为默认Linux安装的一部分，则手动安装。

本指南中列出的受支持的内核和发行版列表中的Linux内核。

Linux x86-64操作系统要求

支持以下Linux x86-64内核：

具有Unbreakable Enterprise Kernel 4的Oracle Linux 7.4：4.1.12-112.16.7.el7uek.x86_64或更高版本
具有Unbreakable Enterprise Kernel 5的Oracle Linux 7.4：4.14.35-1818.1.6.el7uek.x86_64或更高版本
带有Red Hat兼容内核的Oracle Linux 7.4：3.10.0-693.5.2.0.1.el7.x86_64或更高版本

Red Hat Enterprise Linux 7.4：3.10.0-693.5.2.0.1.el7.x86_64或更高版本

SUSE Linux Enterprise Server 12 SP3：4.4.103-92.56-default或更高版本

查看系统要求部分以获取最低包装要求列表。
适用于Oracle Linux的Oracle预安装RPM

如果您使用的是Oracle Linux，那么Oracle建议您为Linux版本运行Oracle预安装RPM，以便为Oracle数据库和Oracle Grid Infrastructure安装配置操作系统。
