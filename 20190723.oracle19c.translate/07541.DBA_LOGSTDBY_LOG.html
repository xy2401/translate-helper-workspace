5.162 DBA_LOGSTDBY_LOG

DBA_LOGSTDBY_LOG显示有关为逻辑备用数据库注册的日志的信息。

此视图仅适用于逻辑备用数据库。

指示字典构建的开头是否在此存档日志中（ YES ）或否（ NO ）

DICT_END

VARCHAR2（3）

指示字典构建的结尾是否在此存档日志中（ YES ）或否（ NO ）

应用

VARCHAR2（8）

主要指示SQL Apply是否已完全应用给定的外部归档日志：

  • YES - SQL Apply已完全应用外部存档日志，不再需要它

  • CURRENT - SQL Apply当前正在应用外部归档日志中包含的更改

  • NO - SQL Apply尚未开始应用外部归档日志中包含的任何更改

  • FETCHING - SQL Apply在从此外部归档日志中读取重做记录时遇到损坏，并且当前正在使用自动间隙解析从主数据库重新获取日志的新副本

  • CORRUPT - 从此外部归档日志读取重做记录时，SQL Apply遇到损坏，并且重新获取归档日志的新副本无法解决问题。SQL Apply不会自动重新获取此归档日志的新副本，并且需要用户干预才能手动注册外部归档日志的新副本。
