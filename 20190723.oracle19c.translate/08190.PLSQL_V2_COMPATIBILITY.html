1.267 PLSQL_V2_COMPATIBILITY

PLSQL_V2_COMPATIBILITY用于指定PL / SQL版本8中是否允许PL / SQL版本2允许的异常行为。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

注意：

不推荐使用PLSQL_V2_COMPATIBILITY参数。它仅为了向后兼容而保留。

PL / SQL版本2允许版本8不允许的一些异常行为。如果要保留该行为以实现向后兼容性，请将PLSQL_V2_COMPATIBILITY设置为true 。如果将其设置为false ，则会强制执行PL / SQL V8行为，并且不允许使用版本2行为。
