2.122 ALL_DIRECTORIES

ALL_DIRECTORIES描述当前用户可访问的所有目录。

相关视图

DBA_DIRECTORIES描述数据库中的所有目录。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

目录的所有者（总是SYS ）

DIRECTORY_NAME

VARCHAR2（128）

不是NULL

目录的名称

目录路径

VARCHAR2（4000）

目录的操作系统路径名

ORIGIN_CON_ID

VARCHAR2（256）

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行来自root，则n = 1）
