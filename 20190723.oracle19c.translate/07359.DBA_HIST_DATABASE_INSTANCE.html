4.313 DBA_HIST_DATABASE_INSTANCE

DBA_HIST_DATABASE_INSTANCE显示Workload Repository中的数据库和实例。

柱 数据类型 空值 描述

DBID

数

不是NULL

数据库ID

INSTANCE_NUMBER

数

不是NULL

实例编号

启动时间

TIMESTAMP（3）

不是NULL

实例的启动时间

平行

VARCHAR2（3）

不是NULL

指示实例是否在Oracle Real Application Clusters（Oracle RAC）环境中运行（ YES ）或否（ NO ）

版

VARCHAR2（17）

不是NULL

数据库版本

DB_NAME

VARCHAR2（9）

数据库的名称

INSTANCE_NAME

VARCHAR2（16）

实例的名称

主机名

VARCHAR2（64）

主机名称

LAST_ASH_SAMPLE_ID

数

不是NULL

活动会话历史记录的最后一个样本ID

PLATFORM_NAME

VARCHAR2（101）

运行实例的平台

CDB

VARCHAR2（3）

可能的值是：

  • 如果数据库是CDB，则为YES

  • 如果数据库不是CDB，则为NO

版

VARCHAR2（7）

数据库的版本。

可能的值包括：

  • CORE EE ：核心企业版

  • EE ：企业版

  • PO ：个人版

  • XE ：Express Edition

DB_UNIQUE_NAME

VARCHAR2（30）

唯一数据库名称

DATABASE_ROLE

VARCHAR2（16）

数据库的当前角色：

  • SNAPSHOT STANDBY

  • LOGICAL STANDBY

  • 物理待命

  • 主

  • 远期同步

CDB_ROOT_DBID

数

采样会话的CDB根的数据库ID

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

STARTUP_TIME_TZ

TIMESTAMP（3）带时区

实例的启动时间

也可以看看：

“ DB_UNIQUE_NAME ”
