ROUND（日期）

句法

round_date :: =


说明round_date.eps的描述

目的

ROUND返回舍入到格式模型fmt指定的单位的date 。此函数对NLS_CALENDAR会话参数不敏感。它根据公历的规则运作。即使您为date指定了不同的日期时间数据类型，返回的值也始终为数据类型DATE 。如果省略fmt ，则将date舍入到最近的一天。date表达式必须解析为DATE值。

也可以看看：

“ ROUND和TRUNC Date Functions ”用于fmt允许的格式模型

例子

以下示例将日期舍入到下一年的第一天：

SELECT ROUND（TO_DATE（'27 -OCT-00'），'YEAR'）“新年”FROM DUAL;新年--------- 01-JAN-01
