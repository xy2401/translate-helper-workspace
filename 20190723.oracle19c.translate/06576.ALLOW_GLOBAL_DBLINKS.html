1.10 ALLOW_GLOBAL_DBLINKS

ALLOW_GLOBAL_DBLINKS指定是否允许数据库链接的LDAP查找。

属性 描述

参数类型

布尔

句法

ALLOW_GLOBAL_DBLINKS = {FALSE | TRUE}

默认值

假

可修改

改变系统

可在PDB中修改

没有

基本

没有

Oracle RAC

必须在所有实例上使用相同的值。

可以设置以下值：

  • FALSE ：数据库不允许LDAP查找数据库链接的定义。

  • TRUE ：数据库允许LDAP查找数据库链接的定义。
