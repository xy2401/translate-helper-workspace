13.38 IF声明

IF语句运行或跳过一个或多个语句的序列，具体取决于BOOLEAN表达式的值。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

if_statement :: =


说明if_statement.eps的说明

（ boolean_expression :: = ， statement :: = ）

语义

逻辑表达式

值为TRUE ， FALSE或NULL表达式。

始终计算第一个boolean_expression 。仅当前面表达式的值为FALSE才会计算boolean_expression 。

如果计算boolean_expression并且其值为TRUE ，则在相应的THEN运行之后的语句。不评估后续表达式，并且不运行与它们关联的语句。

其他

如果没有boolean_expression的值为TRUE ，则ELSE运行后的语句。
