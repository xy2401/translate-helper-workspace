早期版本的自动侦听器迁移

查看此信息，以便从早期数据库版本迁移侦听器。

如果您的系统具有Oracle Grid Infrastructure 11g第2版（11.2），12c第1版（12.1），12c第2版（12.2）或18c安装，并且您安装Oracle Grid Infrastructure 19c以与Oracle Grid共存或升级Oracle Grid基础架构11.2,12.1,12.2或18c安装，然后大多数安装类型自动将现有侦听器迁移到19c Oracle主目录。在迁移期间，升级过程使用与现有侦听器相同的TCP / IP端口配置并启动默认Oracle Net Listener，并使用IPC密钥值。

在Oracle Clusterware升级期间，默认侦听器（ LISTENER_ nodename ）将迁移到Oracle Grid Infrastructure主目录（Grid home）。Oracle Database Configuration Assistant始终使用默认侦听器。

注意：

在迁移期间，客户端应用程序可能无法连接到已注册到正在迁移的侦听器的任何数据库。
