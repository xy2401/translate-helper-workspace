3.172 AUDIT_UNIFIED_ENABLED_POLICIES

AUDIT_UNIFIED_ENABLED_POLICIES描述了数据库中启用的所有审核策略。

注意：

此视图仅在启用了统一审核的Oracle数据库中填充。

  • 有关统一审计的详细信息，请参见“ Oracle数据库安全指南” 。

  • 有关迁移到统一审计的详细信息，请参阅“ Oracle数据库升级指南” 。

柱 数据类型 空值 描述

POLICY_NAME

VARCHAR2（128）

审计策略的名称

ENABLED_OPTION

VARCHAR2（15）

已启用审核策略选项。可能的值：

  • BY USER ：用于在用户上启用的策略

  • EXCEPT USER ：对于在用户上启用的策略

  • BY GRANTED ROLE ：对于在角色上启用的策略

  • INVALID ：对于未在用户或角色上启用的策略

实体名称

VARCHAR2（128）

启用审核策略的数据库实体（用户名或角色名称）。

在所有数据库用户上启用审核策略后，此列中将显示ALL USERS 。

ENTITY_TYPE

VARCHAR2（7）

数据库实体类型。可能的值：

  • USER ：表示已对某个用户启用该策略。

  • ROLE ：表示对一个或多个角色启用了策略。

成功

VARCHAR2（3）

指示审核策略是否已启用审核成功事件（ YES ）或否（ NO ）

失败

VARCHAR2（3）

指示是否启用审核策略以审核不成功事件（ YES ）或否（ NO ）
