13.22废除Pragma

DEPRECATE pragma将PL / SQL元素标记为已弃用。编译器发出使用pragma DEPRECATE或不推荐使用的元素的警告。

相关警告告诉用户一个已弃用的元素，可能需要更改其他代码以考虑弃用。

deprecate_pragma

DEPRECATE pragma只能出现在包规范，对象规范，顶级过程或顶级函数的声明部分中。

可以弃用这些类型的PL / SQL元素：

  • 子程序

  • 包

  • 变量

  • 常量

DEPRECATE pragma只能出现在PL / SQL单元的声明部分中。它必须在声明要弃用的项目后立即出现。

DEPRECATE pragma适用于在pragma之前的声明中命名的PL / SQL元素。

当DEPRECATE pragma应用于包规范，对象规范或子程序时，pragma必须紧跟在终止定义的声明部分的关键字IS或AS之后。

当DEPRECATE pragma应用于包或对象规范时，也不推荐对规范中声明的所有元素（可以弃用的类型）的引用。

如果DEPRECATE编译指示适用于子程序声明，则只有该子程序受到影响;不推荐使用具有相同名称的其他重载。

如果在使用DEPRECATE编译指示时出现可选的自定义消息，则自定义消息将添加到为不推荐使用的元素的任何引用而发出的警告。

DEPRECATE pragma中的标识符必须在其应用的声明中命名该元素。

在类型派生期间继承了Deprecation。不推荐使用其父项已弃用的子对象类型。仅推荐使用继承的属性和方法。

如果不弃用基类型但不推荐使用单个方法或属性，并且从此类型派生类型并且继承了不推荐使用的类型或方法，则通过派生类型对这些类型或方法的引用将导致编译器发出警告。

除了具有deprecation pragma或其主体的单元之外的任何地方出现的已弃用元素的引用将导致PL / SQL编译器对引用的元素发出警告。对匿名块中不推荐使用的元素的引用不会导致编译器发出警告;只有命名实体中的引用才会引发警告。

如果在另一个已弃用的实体的定义中引用了已弃用的实体，则不会发出警告。

当较旧的客户端代码引用已弃用的实体时，它将失效并重新编译。没有发出警告。

当SQL代码直接引用不推荐的元素时，没有任何效果。

对PL / SQL静态SQL语句中不推荐使用的元素的引用可能会导致PL / SQL编译器发出警告。但是，这些参考可能无法检测到。

pls_identifier

不推荐使用的PL / SQL元素的标识符。

character_literal

可选的编译时警告消息。

DEPRECATE Pragma编译警告

使用DEPRECATE pragma时以及引用不推荐使用的项时，PL / SQL编译器会发出警告。

  • 6019 - 该实体已弃用，可在将来的版本中删除。不要使用已弃用的实体。

  • 6020 - 引用的实体已弃用，可在将来的版本中删除。不要使用已弃用的实体。如果有任何警告，请遵循警告中的具体说明。

  • 6021 - 错位的pragma。pragma DEPRECATE应在声明被弃用的实体后立即执行。在声明正在弃用的实体后立即放置pragma。

  • 6022 - 不能弃用此实体。弃用仅适用于可在包或类型规范中声明的实体以及顶级过程和函数定义。删除pragma。

可以使用PLSQL_WARNINGS参数或DBMS_WARNING包管理DEPRECATE编译指示警告。

示例13-9启用弃用警告

此示例显示如何设置PLSQL_WARNINGS参数以在会话中启用这些警告。

ALTER SESSION SET PLSQL_WARNINGS ='ENABLE：（6019,6020,6021,6022）';

例13-10 PL / SQL包的弃用

此示例显示了整个PL / SQL包的弃用。在包装及其主体外部使用时，将对包装包1的任何参考以及程序foo和bar发出警告。

实时SQL：

您可以在PL / SQL包的弃用中在Oracle Live SQL上查看和运行此示例

PACKAGE pack1 AS PRAGMA DEPRECATE （pack1）;程序foo;程序吧; END pack1;

示例13-21元素名称和DEPRECATE Pragma参数不匹配

此示例显示如果pragma的参数与声明中的名称不匹配，则会忽略该pragma，并且编译器不会发出警告。

实时SQL：

您可以在元素名称和DEPRECATE Pragma参数不匹配的 Oracle Live SQL上查看和运行此示例

如果使用不匹配的名称应用标识符，则编译器会发出有关错误放置的pragma的警告。该pragma没有任何效果。

创建包装pkg17是PRAGMA DEPRECATE （“pkg17”）;结束pkg17;
