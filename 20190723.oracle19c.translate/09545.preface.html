设置Java Access Bridge以实现Java辅助功能

安装Java Access Bridge，以便Microsoft Windows系统上的辅助技术可以使用Java Accessibility API。

Java Access Bridge是一种技术，它使实现Java Accessibility API的Java应用程序和applet对Microsoft Windows系统上的辅助技术可见。

有关使用Java Access Bridge所需的最低支持辅助技术版本的信息，请参阅Java平台标准版可访问性指南 。另请参阅本指南以获取安装和测试说明以及有关如何使用Java Access Bridge的说明。
