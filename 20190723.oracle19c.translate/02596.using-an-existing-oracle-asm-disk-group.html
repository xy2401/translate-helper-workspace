使用现有Oracle ASM磁盘组

使用Oracle Enterprise Manager Cloud Control或Oracle ASM命令行工具（ asmcmd ）标识现有磁盘组，并确定磁盘组中是否有足够的可用空间。

  1. 连接到Oracle ASM实例并在必要时启动实例：
    $ $ ORACLE_HOME / bin / asmcmd ASMCMD>启动
  2. 输入以下命令之一以查看现有磁盘组，其冗余级别以及每个磁盘组中的可用磁盘空间量：
    ASMCMD> lsdg

    要么

    $ ORACLE_HOME / bin / asmcmd -p lsdg

    lsdg命令仅列出有关已装入磁盘组的信息。

  3. 从输出中，标识具有适当冗余级别的磁盘组，并记下它包含的可用空间。
  4. 如有必要，请安装或识别满足安装存储要求所需的其他磁盘设备。

注意：

如果要将设备添加到现有磁盘组，则Oracle建议您使用与该磁盘组中的现有设备具有相同大小和性能特征的设备。
