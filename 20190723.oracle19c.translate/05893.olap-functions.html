2 OLAP函数

OLAP函数扩展了SQL分析函数的语法。SQL语言开发人员和DBA都熟悉这种语法，因此您可以比专有的OLAP语言和API更容易地采用它。使用OLAP功能，您可以创建所有标准计算度量，包括排名，共享，先前和未来期间，期间至今，并行期间，移动聚合和累积聚合。

本章介绍OLAP功能。它包含以下主题：

2.3 AVERAGE_RANK

AVERAGE_RANK根据表达式的值对维度的成员进行AVERAGE_RANK 。该函数返回维度成员的序列号。

AVERAGE_RANK将相同的平均等级分配给相同的值。例如，对于一系列五维成员， AVERAGE_RANK可以返回AVERAGE_RANK 。

回报价值

数

句法

AVERAGE_RANK（）OVER（ rank_clause ）

rank_clause :: =

{ DIMENSION dimension_id | HIERARCHY hierarchy_id} ORDER BY order_by_clause [ ， order_by_clause] ...[ 在十层 | 父母 | ANCESTOR在 { DIMENSION LEVEL dim_level_id | HIERARCHY LEVEL hier_level_id}}]

order_by_clause :: =

表达[ ASC | DESC ] [ NULLS {第一个 | 最后 }]

参数

dimension_id

使用默认层次结构计算值的维度。

hierarchy_id

计算值的层次结构。如果使用dimension_id则使用默认层次结构。

ORDER BY

提供排名的基础。您可以提供其他ORDER BY子句来打破ORDER BY任何关系。

表达式

提供用作排名基础的值。

ASC | DESC

将排名从最小到最大（升序）或从最大到最小（降序）排序。

NULLS {FIRST |持续}

确定具有空值的成员是首先列出还是最后列出。

WITHIN

选择要排名的一组相关维成员。

LEVEL将所有成员排在同一级别。

PARENT将同一级别的成员与同一父级进行排名。

ANCESTOR将所有成员排在同一级别，并在指定级别对同一祖先进行排名。

dim_level_id

dimension_id级别的名称。

hier_level_id

hierarchy_id级别的名称。

2.4 AVG

AVG返回通过时间维度计算的一组值的平均值。使用此功能可以创建累积平均值和移动平均值。

GREGORIAN关系在时间维度上叠加格里高利历。这些关系可用于计算财务和非标准层次结构。

回报价值

数

句法

AVG（ value_expr ）OVER（ window_clause ）

window_clause :: =

[{ DIMENSION dimension_id | HIERARCHY hierarchy_id}] BETWEEN preceding_boundary | following_boundary [ WITHIN { LEVEL | 父母 | GREGORIAN { 年份 | QUARTER | 月 | WEEK } | ANCESTOR在 { DIMENSION LEVEL dim_level_id | HIERARCHY LEVEL hier_level_id}}]

preceding_boundary :: =

{ UNBOUNDED PRECEDING |表达PRECEDING } 和 { CURRENT MEMBER |表达{ PRECEDING | 跟随 } | 无限制跟随 }

following_boundary :: =

{ CURRENT MEMBER |表达式跟随 } 和 {表达式跟随 | 无限制跟随 }

参数

value_expr

要计算其值的维表达式。

dimension_id

使用默认层次结构计算值的时间维度。

hierarchy_id

计算值的层次结构。如果使用dimension_id则使用默认层次结构。

dim_level_id

dimension_id级别的名称。

hier_level_id

hierarchy_id级别的名称。

边界

preceding_boundary和following_boundary标识由维度或层次结构标识的组内的一系列时间段。

UNBOUNDED从第一个句点开始，或以组的最后一个句点结束。

CURRENT MEMBER在当前时间段开始或结束计算。

表达式

标识与开始或结束范围的当前时间段的特定距离处的时段的数值。

WITHIN

标识计算中使用的时间段范围。以下是关键字的说明。

范围 描述

水平

计算同一级别的所有时间段。（默认）

家长

使用同一父级计算同一级别的时间段。

GREGORIAN年

计算同一公历年内的时间段。

GREGORIAN QUARTER

计算同一格里高利季度内的时间段。

GREGORIAN MONTH

计算同一公历月内的时间段。

GREGORIAN WEEK

计算同一格里高利周内的时间段。

祖先

包括同一级别的时间段和指定级别的相同祖先。
