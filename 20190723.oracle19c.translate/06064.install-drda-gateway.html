12为DRDA安装Oracle Database Gateway

本节提供有关DRDA的Oracle Database Gateway的硬件和软件要求以及安装过程的信息。

要安装网关，请按照下列步骤操作：

  1. 确保系统满足Oracle Database Gateway for DRDA的系统要求中指定的所有硬件和软件要求。

  2. 运行Oracle Universal Installer。

    有关运行Oracle Universal Installer的更多信息，请参阅单步执行Oracle Universal Installer。

    Oracle Universal Installer是一个菜单驱动的实用程序，它通过提示您执行操作项来指导您完成网关的安装。操作项及其显示顺序取决于您的平台。

    有关适用于DRDA的Oracle数据库网关的安装过程的说明，请参阅表12-3 。

*最小交换空间为1 GB（或RAM的两倍）。在具有2 GB或更多RAM的系统上，交换空间可以是RAM大小的一到两倍。在具有1 GB或更多内存的AIX系统上，不要将交换空间增加超过2 GB。

对于大多数安装，建议第一个用户至少支持256 MB的实内存以支持用于DRDA的Oracle数据库网关。

每次并发使用网关的实际内存总需求取决于以下因素：

  • 数量每个用户打开并发TCP / IP连接

  • 在网关和网关之间传输的数据项数远程交易程序

  • 其他因素，例如配置的网络缓冲区大小

检查硬件要求

要确保系统满足最低要求，请按照下列步骤操作：

  1. 要确定物理RAM大小，请输入以下命令之一：
    操作系统 命令

    POWER Systems上的IBM AIX（64位）

    ＃ /usr/sbin/lsattr -E -l sys0 -a realmem

    Linux x86 64位

    #grep MemTotal / proc / meminfo

    SPARC上的Oracle Solaris（64位）

    ＃/ usr/sbin/prtconf | grep "Memory size"

    x86-64上的Oracle Solaris（64位）

    ＃/ usr/sbin/prtconf | grep "Memory size"

    HP-UX Itanium

    ＃/ usr / contrib / bin / machinfo | grep -i记忆

    如果系统中安装的物理RAM的大小小于所需的大小，则必须先安装更多内存，然后再继续。

  2. 要确定已配置交换空间的大小，请输入以下命令之一：
    操作系统 命令

    POWER Systems上的IBM AIX（64位）

    ＃/ usr / sbin / lsps -a

    Linux x86 64位

    #grep SwapTotal / proc / meminfo

    SPARC上的Oracle Solaris（64位）

    ＃/ usr / sbin / swap -s

    x86-64上的Oracle Solaris（64位）

    ＃/ usr / sbin / swap -s

    HP-UX Itanium

    ＃/ usr / sbin / swapinfo -a

    如有必要，请参阅操作系统文档以获取有关如何配置其他交换空间的信息。

  3. 要确定/tmp目录中可用的磁盘空间量，请输入以下命令：
    操作系统 命令

    POWER Systems上的IBM AIX（64位）

    #df -k / tmp

    Linux x86 64位

    #df -k / tmp

    SPARC上的Oracle Solaris（64位）

    #df -k / tmp

    x86-64上的Oracle Solaris（64位）

    #df -k / tmp

    HP-UX Itanium

    #bdf / tmp
  4. 要确定系统上可用的磁盘空间量，请输入以下命令：
    操作系统 命令

    POWER Systems上的IBM AIX（64位）

    #df -k

    Linux x86 64位

    #df -k

    SPARC上的Oracle Solaris（64位）

    #df -k

    x86-64上的Oracle Solaris（64位）

    #df -k

    HP-UX Itanium

    #bdf

检查软件要求

要确保系统满足最低要求，请按照下列步骤操作：

  • 要确定安装了POWER Systems（64位）上的哪个版本的IBM AIX，请输入以下命令：

    #oslevel -r

  • 要确定安装了Linux x86 64位的分发版和版本，请输入以下命令：

    #cat / proc / version

  • 要确定安装了SPARC（64位）上的Oracle Solaris版本，请输入以下命令：

    #uname -r

  • 要确定安装了x86-64（64位）上的Oracle Solaris版本，请输入以下命令：

    #uname -r

  • 要确定安装了哪个版本的HP-UX Itanium，请输入以下命令：

    #uname -a

逐步完成Oracle Universal Installer

使用以下命令启动安装程序：

$ ./runInstaller
