“添加”页面的标题下方会显示三个输入字段。第一个是Capture Name，其值为“NewEmpCapture”。第二个字段是目标数据库，它是只读的，其值为“数据库”。第三个字段是数据库版本，其值为“12.1.0.1.0”。 “捕获名称”，“数据库目标”和“数据库版本”字段右侧是“捕获描述”文本字段，该字段为空白。

这些字段下面有两个部分：左侧是Database Credential，右侧是Database Host Credential。“数据库凭据”部分的文本“请提供访问数据库的凭据”。对于Credential字段，选择Preferred选项。“首选凭证名称”列表为空白。“凭据详细信息”字段包含文本“未设置默认首选凭据”。

“数据库主机凭据”部分的文本“请提供访问数据库名称的凭据”。对于Credential字段，选择Preferred选项。“首选凭证名称”列表为空白。“凭据详细信息”字段包含文本“未设置默认首选凭据”。

Database Host Credential部分下面是Database Capture Intermediate Storage Root Location输入字段，其值为“/ scratch / destination_dir”。浏览按钮位于此字段的右侧。字段下方是信息文本“此根目录用于在捕获过程中临时存储捕获的工作负载”。在“信息”文本下，“使用中间存储位置”作为最终存储位置选项，该选项已启用。

“确定”和“取消”按钮显示在弹出窗口的右下角。
