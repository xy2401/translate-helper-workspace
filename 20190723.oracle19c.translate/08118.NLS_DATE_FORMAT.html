1.202 NLS_DATE_FORMAT

NLS_DATE_FORMAT指定用于TO_CHAR和TO_DATE函数的默认日期格式。 此参数的默认值由NLS_TERRITORY确定。
属性 描述

参数类型

串

句法

NLS_DATE_FORMAT = " format "

默认值

源自NLS_TERRITORY

可修改

改变会话

可在PDB中修改

是

价值范围

任何有效的日期格式掩码，但不超过固定长度

基本

没有

此参数的值可以是任何有效的日期格式掩码，并且该值必须用双引号括起来。例如：

NLS_DATE_FORMAT =“MM / DD / YYYY”

注意：

此初始化参数NLS_DATE_FORMAT值用于初始化此参数的会话值，该值是SQL查询处理引用的实际值。如果客户端使用Oracle JDBC驱动程序或客户端是基于OCI并且定义了NLS_LANG客户端设置（环境变量），则此初始值将被客户端值覆盖。因此，通常忽略初始化参数值。
