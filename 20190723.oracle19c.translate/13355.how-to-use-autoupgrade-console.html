如何使用AutoUpgrade控制台

AutoUpgrade控制台提供一组命令来监视AutoUpgrade作业的进度。运行AutoUpgrade实用程序时，控制台默认启动，并由参数console和noconsole启用或禁用。

在控制台模式下，AutoUpgrade控制台使您可以运行命令来监视AutoUpgrade作业在系统上运行时的特定方面。

注意：

您一次只能运行一个AutoUpgrade实例。

在此示例中，AutoUpgrade使用noconsole模式下的配置文件在noconsole模式下运行。 

java -jar autoupgrade.jar -config autoupgrade.cfg -mode analyze -noconsole

使用noconsole模式会关闭用户输入的要求，因此适合与脚本一起使用。

在此示例中，AutoUpgrade在Analyze模式下运行，并且使用-console选项再次打开控制台：

java -jar autoupgrade.jar -config autoupgrade.cfg -mode analyze -console

控制台用户输入再次恢复。
