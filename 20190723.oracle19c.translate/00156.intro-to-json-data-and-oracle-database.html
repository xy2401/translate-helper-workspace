第一部分 JSON数据和Oracle数据库简介

开始了解JSON数据以及如何将SQL和PL / SQL与存储在Oracle数据库中的JSON数据结合使用。

基于JSON文档形式的持久应用程序数据的无架构开发使您可以快速响应不断变化的应用程序需求。您可以更改和重新部署应用程序，而无需更改其使用的存储架构。

SQL和关系数据库为复杂的数据分析和报告提供灵活的支持，以及坚如磐石的数据保护和访问控制。对于NoSQL数据库通常不是这种情况，NoSQL数据库通常与过去使用JSON的无模式开发相关联。

Oracle数据库为SQL和关系数据库提供了JSON数据的所有好处，您可以以与任何其他类型的数据库数据相同的方式存储和操作它们。

  • Oracle数据库中的JSON
    Oracle数据库本身支持JavaScript对象表示法（JSON）数据，具有关系数据库功能，包括事务，索引，声明性查询和视图。
  • JSON数据
    JavaScript Object Notation（JSON）在标准ECMA-404（JSON数据交换格式）和ECMA-262（ECMAScript语言规范，第三版）中定义。ECMAScript的JavaScript方言是一种在Web浏览器和Web服务器中广泛使用的通用编程语言。
