Oracle正则表达式中受Perl影响的扩展

Oracle数据库正则表达式函数和条件接受许多常用的受Perl影响的运算符，尽管它们不是POSIX标准的一部分。表D-3列出了那些运算符。有关示例的更完整描述，请参阅Oracle数据库开发指南 。

表D-3 Oracle正则表达式中受Perl影响的运算符

操作者 描述

\ d

一个数字字符。

\ d

非数字字符。

\ W

一个字的字符。

\ W

非字符。

\ S

一个空白字符。

\ S

非空白字符。

\一个

仅匹配字符串的开头，或者在字符串末尾的换行符之前匹配。

\ž

仅匹配字符串末尾的匹配项。

*？

匹配前面的模式元素0次或更多次（不一致）。

+？

匹配前面的模式元素1次或更多次（不一致）。

??

匹配前面的模式元素0或1次（不一致）。

{N}？

正好匹配前面的模式元素n次（nongreedy）。

{N，}？

匹配前面的模式元素至少n次（nongreedy）。

{N，M}？

匹配前面的模式元素至少n但不超过m次（nongreedy）。
