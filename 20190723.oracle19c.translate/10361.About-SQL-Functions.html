关于SQL函数

SQL函数内置于Oracle数据库中，可用于各种适当的SQL语句。不要将SQL函数与用PL / SQL编写的用户定义函数混淆。

如果使用SQL函数所期望的数据类型之外的数据类型的参数调用SQL函数，则Oracle会在执行SQL函数之前尝试将参数转换为期望的数据类型。

SQL函数中的空值

给定null参数时，大多数标量函数返回null。您可以使用NVL函数在出现null时返回值。例如，表达NVL(commission_pct,0)如果返回0 commission_pct为空或值commission_pct如果它不为空。

有关聚合函数如何处理空值的信息，请参阅聚合函数 。

SQL函数的语法

在SQL函数的语法图中，参数由其数据类型指示。当参数function出现在SQL语法中时，请将其替换为本节中描述的函数之一。函数按其参数的数据类型及其返回值进行分组。

注意：

将SQL函数应用于LOB列时，Oracle数据库会在SQL和PL / SQL处理期间创建临时LOB。您应确保临时表空间配额足以存储应用程序的这些临时LOB。

SQL函数可能是排序规则敏感的，这意味着它执行的字符值比较或匹配由排序规则控制。函数使用的特定排序规则是根据函数参数的排序规则确定的。

如果SQL函数的结果具有字符数据类型，则排序规则派生规则定义要与结果关联的排序规则。
