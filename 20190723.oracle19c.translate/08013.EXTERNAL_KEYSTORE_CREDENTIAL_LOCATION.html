1.119 EXTERNAL_KEYSTORE_CREDENTIAL_LOCATION

EXTERNAL_KEYSTORE_CREDENTIAL_LOCATION指定安全外部密钥库的位置。

属性 描述

参数类型

串

句法

EXTERNAL_KEYSTORE_CREDENTIAL_LOCATION =文件位置

默认值

没有默认值

可修改

没有

可在PDB中修改

没有

基本

没有

Oracle RAC

可以在每个实例上设置此参数。Oracle建议，如果实例具有共享位置，则在共享位置使用此参数的目录，并在所有Oracle RAC实例上设置相同的值。如果数据库没有共享位置，则数据库的每个实例都将拥有自己的目录，并且应该按实例设置该值。

TDE密钥库凭证可以存储在安全的外部密钥库中，以自动执行透明数据加密（TDE）操作。

要在新配置的PDB中自动设置主密钥，请设置EXTERNAL_KEYSTORE_CREDENTIAL_LOCATION以便CDB知道安全外部密钥库的位置。

使用此参数设置文件规范时，将在该位置查找安全外部密钥库。如果在该位置找不到安全外部密钥库，则忽略此参数。
