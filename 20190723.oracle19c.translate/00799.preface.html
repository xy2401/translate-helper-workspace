前言

本文档描述了Oracle XML Developer's Kit（XDK）。它提供了有关各种语言组件的详细信息，包括可扩展标记语言（XML），Java，C和C ++。

听众

本文档适用于使用XDK的语言组件在数据库或数据库外部的文档中生成和存储XML数据的应用程序开发人员。尽可能提供示例和示例应用程序。本文档假定您熟悉XML以及Java，C或C ++。

例子

本文档中的许多示例都使用Oracle数据库示例数据库模式，或者随软件提供。

有关示例模式的信息，请参阅Oracle数据库示例模式 。

可以在以下目录中找到随软件提供的示例：

  • $ ORACLE_HOME / XDK /演示/ JAVA /

  • $ ORACLE_HOME / XDK /演示/ C /

  • $ ORACLE_HOME / XDK / JAVA /采样/
