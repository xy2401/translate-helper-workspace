5为Oracle Grid Infrastructure和Oracle RAC配置网络

检查是否具有用于群集安装的Oracle Grid Infrastructure所需的网络硬件和Internet协议（IP）地址。

  • 关于Oracle Grid Infrastructure网络配置选项
    确保您具有用于群集安装的Oracle Grid Infrastructure所需的网络硬件和Internet协议（IP）地址。
  • 了解网络地址
    将每个接口标识为公共或私有接口，或者作为您不希望Oracle Grid Infrastructure或Oracle Flex ASM集群使用的接口。
  • 网络接口硬件最低要求
    查看这些要求以确保您拥有Oracle Grid Infrastructure群集的最低网络硬件技术。
  • 专用IP接口配置要求
    专用接口的要求取决于您使用的是单个还是多个接口。
  • IPv4和IPv6协议要求
    Oracle Grid Infrastructure和Oracle RAC支持RFC 2732指定的标准IPv6地址表示法以及RFC 4193定义的全局和站点本地IPv6地址。
  • Oracle Grid Infrastructure IP名称和地址要求
    查看有关Oracle Grid Infrastructure IP名称和地址要求的此信息。
  • Oracle Grid Infrastructure使用的网络的广播要求
    广播通信（ARP和UDP）必须在配置为供Oracle Grid Infrastructure使用的所有公共和专用接口上正常工作。
  • Oracle Grid Infrastructure使用的网络的多播要求
    对于每个集群成员节点，Oracle mDNS守护程序在所有接口上使用多播来与集群中的其他节点进行通信。
  • 域名委派到网格命名服务
    如果要为标准群集配置网格命名服务（GNS），则在安装Oracle Grid Infrastructure之前，必须配置DNS以向GNS发送对GNS服务的子域的任何名称解析请求。
  • Oracle Flex群集的配置要求
    了解Oracle Flex Clusters及其配置要求。
  • 网格命名服务群集配置示例
    查看此示例以了解网格命名服务配置。
  • 手动IP地址配置举例
    如果您选择不使用GNS，则在安装之前必须配置公共，虚拟和专用IP地址。
  • 网络接口配置选项
    在安装过程中，系统会要求您确定Oracle Universal Installer（OUI）在群集节点上检测到的每个网络适配器（或网络接口）的计划使用情况。
