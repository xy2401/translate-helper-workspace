8为Oracle Grid Infrastructure配置存储

完成这些过程以为群集配置Oracle Grid Infrastructure的Oracle自动存储管理（Oracle ASM）。

集群的Oracle Grid Infrastructure为Oracle数据库提供系统支持。Oracle ASM是Oracle数据库文件的卷管理器和文件系统，支持单实例Oracle数据库和Oracle Real Application Clusters（Oracle RAC）配置。Oracle自动存储管理还支持满足您的应用程序需求的通用文件系统，包括Oracle数据库二进制文件。Oracle自动存储管理是Oracle推荐的存储管理解决方案。它提供了传统卷管理器和文件系统的替代方案。

注意：

Oracle ASM和共享文件系统是Oracle Cluster Registry（OCR）和Oracle Clusterware投票文件支持的存储管理解决方案。OCR是包含配置信息和群集状态的文件。安装Oracle Clusterware期间，安装程序会自动初始化OCR。Database Configuration Assistant使用OCR存储其创建的群集数据库的配置。
