第VI部分 Oracle XML DB存储库

Oracle XML DB Repository允许您对数据进行版本控制，实现和管理安全性，并使用API来访问和操作存储库资源。

  • 访问Oracle XML DB存储库数据
    在Oracle XML DB Repository中有多种方法可以访问和操作数据，包括使用标准协议，如FTP和HTTP（S）/ WebDAV; Oracle XML DB资源应用程序接口（API）;并且存储库视图RESOURCE_VIEW和PATH_VIEW 。
  • 配置Oracle XML DB存储库
    Oracle XML DB Repository的整体配置适用于所有存储库资源。它不包括配置用于处理事件或管理XLink和XInclude处理的参数。您使用资源配置文件来配置资源。
  • 将XLink和XInclude与Oracle XML DB一起使用
    您可以将XLink和XInclude与Oracle XML DB Repository中的资源一起使用。但是不推荐使用XLink。
  • 使用RESOURCE_VIEW和PATH_VIEW进行存储库访问
    预定义的公共视图RESOURCE_VIEW和PATH_VIEW提供对Oracle XML DB存储库数据的访问。您可以使用Oracle SQL函数under_path和equals_path根据其路径名查询资源，并使用函数path和depth来返回资源路径名和深度。
  • 资源版本
    可以对Oracle XML DB资源库资源进行版本控制。保留对受版本控制的资源的所有更改的记录。
  • PL / SQL访问Oracle XML DB存储库
    PL / SQL包DBMS_XDB_CONFIG和DBMS_XDB_REPOS一起为PL / SQL提供Oracle XML DB资源应用程序接口（API）。您可以使用前者配置Oracle XML DB及其存储库。您使用后者在存储库上执行其他非配置操作。
  • 存储库访问控制
    Oracle数据库提供经典的数据库安全性，例如数据库用户的行级和列级安全访问。它还为Oracle XML DB Repository中的资源提供了细粒度的访问控制。您可以创建，设置和修改访问控制列表（ACL）。
  • 使用协议的存储库访问
    您可以使用FTP和HTTP（S）/ WebDAV协议访问Oracle XML DB存储库数据。
  • 用户定义的存储库元数据
    您可以创建自己的元数据以与存储在Oracle XML DB Repository中的XML数据相关联。
  • Oracle XML DB存储库事件
    您可以使用Oracle XML DB Repository以存储库资源的形式存储和访问任何类型的数据。您可以从任何应用程序访问存储库数据。有时您的应用程序需要在发生特定存储库操作时执行某些操作。您可以使用存储库事件执行此操作。
  • Java中的Oracle XML DB应用程序指南
    提供了使用Java编写Oracle XML DB应用程序的设计指南。这包括为Oracle XML DB编写和配置Java servlet的准则。
  • 使用URI进行数据访问
    您可以在数据库中生成和存储URI，并使用它们来检索它们所针对的数据库数据。您可以使用这种方式使用三种URI：DBUris，XDBUris和HTTPUris。
  • 原生Oracle XML DB Web服务
    您的应用程序可以使用本机Oracle XML DB Web服务访问Oracle数据库。
