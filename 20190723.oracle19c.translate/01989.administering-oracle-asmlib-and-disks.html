管理Oracle ASMLIB和磁盘

查看此信息以管理Oracle自动存储管理库驱动程序和磁盘。

要管理Oracle自动存储管理库驱动程序和磁盘，请使用带有不同选项的/usr/sbin/oracleasm初始化脚本，如下表所述：

表B-2使用ORACLEASM的磁盘管理任务

任务 命令示例 描述

配置或重新配置ASMLIB

oracleasm configure -i

如有必要，使用configure选项重新配置Oracle自动存储管理库驱动程序。

要查看命令选项，请输入不带-i标志的oracleasm configure 。

更改ASMLIB的系统重新启动加载选项

oracleasm configure -e

选项是-e启用和-d禁用。

使用-e和-d选项可在系统启动时更改Oracle自动存储管理库驱动程序的操作。-e选项导致Oracle自动存储管理库驱动程序在系统启动时加载。

在不重新启动系统的情况下加载或卸载ASMLIB

oracleasm init

选项是init加载和exit卸载。

使用init和exit选项加载或卸载Oracle自动存储管理库驱动程序，而无需重新启动系统。

标记磁盘以与ASMLIB一起使用

oracleasm createdisk VOL1 / dev / sda1

使用createdisk选项标记磁盘设备与Oracle自动存储管理库驱动器使用，并给它一个名称，其中labelname是要用来标记设备名称， devicepath是路径设备：

oracleasm createdisk labelname devicepath

取消标记已命名的磁盘设备

oracleasm deletedisk VOL1

使用deletedisk选项取消标记已命名的磁盘设备，其中diskname是磁盘的名称：

oracleasm deletedisk diskname

警告：请勿使用此命令取消标记Oracle自动存储管理磁盘组正在使用的磁盘。在取消标记之前，必须从Oracle自动存储管理磁盘组中删除该磁盘。

确定ASMLIB是否正在使用磁盘设备

oracleasm querydisk

使用querydisk选项确定Oracle自动存储管理库驱动程序是否正在使用磁盘设备或磁盘名称，其中diskname_devicename是要查询的磁盘或设备的名称：

oracleasm querydisk diskname_devicename

列出Oracle ASMLIB磁盘

oracleasm listdisks

使用listdisks选项列出标记的Oracle ASM库驱动程序磁盘的磁盘名称。

识别标记为ASMLIB磁盘的磁盘

oracleasm scandisks

使用scandisks选项可以使群集节点能够识别哪个共享磁盘已在另一个节点上标记为ASMLIB磁盘。

重命名ASMLIB磁盘

oracleasm renamedisk VOL1 VOL2

使用renamedisk选项通过使用以下语法更改Oracle ASM库驱动程序磁盘或设备的标签，其中manager指定管理器设备， label_device指定要重命名的磁盘，由OracleASM标签名称或设备指定path和new_label指定要用于磁盘的新标签：

oracleasm renamedisk [-l manager ] [-v] label_device new_label

使用-v标志为调试提供详细输出。

警告：在重新标记磁盘之前，必须确保所有Oracle数据库和Oracle ASM实例已停止使用该磁盘。如果您不这样做，那么您可能会丢失数据。
