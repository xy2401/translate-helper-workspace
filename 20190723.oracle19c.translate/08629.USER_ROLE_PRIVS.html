6.571 USER_ROLE_PRIVS

USER_ROLE_PRIVS描述授予当前用户的角色。

柱 数据类型 空值 描述

用户名

VARCHAR2（128）

用户名或PUBLIC

GRANTED_ROLE

VARCHAR2（128）

授予用户的角色的名称

ADMIN_OPTION

VARCHAR2（3）

指示授权是否使用ADMIN OPTION （ YES ）（ NO ）（ NO ）

DELEGATE_OPTION

VARCHAR2（3）

指示授权是否使用DELEGATE OPTION （ YES ）（ NO ）

DEFAULT_ROLE

VARCHAR2（3）

指示角色是否被指定为用户的DEFAULT ROLE （ YES ）或否（ NO ）

OS_GRANTED

VARCHAR2（3）

指示操作系统是否授予角色（ YES ）（ NO ）;如果OS_ROLES初始化参数为true则会发生

共同

VARCHAR2（3）

表示授予的方式。可能的值：

  • 如果角色被普遍授予，则为YES （使用CONTAINER=ALL ）

  • 如果角色是在本地授予的，则为NO （未使用CONTAINER=ALL ）

遗传

VARCHAR2（3）

指示授权是从其他容器继承（ YES ）还是不是（ NO ）
