产品支持

这些是Oracle Database 19c支持的产品或功能：

  • SPARC上的Oracle Solaris支持

    Oracle Solaris on SPARC（64位）支持Oracle Database 19c。

  • 数据库智能闪存缓存支持

    Oracle Solaris支持Database Smart Flash Cache。

  • Oracle ACFS和Oracle ADVM支持

    虽然Oracle ADVM支持Oracle自动存储管理磁盘组中的原始磁盘，但不支持通过raw创建的Oracle ADVM设备专用文件; Oracle ADVM仅支持块设备专用文件。

    有关支持的平台和发行版的最新信息，请参阅My Oracle Support上的说明1369107.1， 网址为https://support.oracle.com

错误21800407

在英语和日语环境中安装Oracle数据库时，背景字体错误地显示为Oracle Universal Installer（OUI），Database Configuration Assistant（DBCA）和Database Upgrade Assistant（DBUA）页面的黄色。

解决方法：

在安装Oracle数据库之前，请运行以下命令：

export _JAVA_OPTIONS =' - Dsun.java2d.xrender = false'
