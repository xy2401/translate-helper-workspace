1.89 DB_RECOVERY_FILE_DEST_SIZE

DB_RECOVERY_FILE_DEST_SIZE指定（以字节为单位）在快速恢复区域中创建的目标数据库恢复文件要使用的总空间的硬限制。

属性 描述

参数类型

大整数

句法

DB_RECOVERY_FILE_DEST_SIZE = integer [K | M | G]

默认值

0

可修改

改变系统......SID = '*'

可在PDB中修改

没有

基本

是

Oracle RAC

您必须为每个实例设置此参数，并且多个实例必须具有相同的值。

请注意，每个Oracle文件的块0和OS块头都不包含在此大小中。在计算快速恢复区域所需的实际磁盘使用量时，允许额外10％的此数据。
