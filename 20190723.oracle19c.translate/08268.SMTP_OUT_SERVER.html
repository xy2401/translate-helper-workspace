1.318 SMTP_OUT_SERVER

SMTP_OUT_SERVER指定UTL_MAIL向其发送出站电子邮件的SMTP主机和端口。

属性 描述

参数类型

串

句法

SMTP_OUT_SERVER = server_clause [, server_clause ] ...

句法

server_clause :: =

host_name [: port ]

默认值

没有默认值。

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

可以指定多个服务器，以逗号分隔。

如果列表中的第一个服务器不可用，则UTL_MAIL尝试第二个服务器，依此类推。

如果未指定SMTP_OUT_SERVER ，则SMTP服务器名称默认为DB_DOMAIN ，端口号默认为25 ，SMTP域默认为DB_DOMAIN的后缀。
