4.111 DBA_AUDIT_MGMT_CONFIG_PARAMS

DBA_AUDIT_MGMT_CONFIG_PARAMS显示有关DBMS_AUDIT_MGMT PL / SQL程序包使用的当前配置的审计跟踪属性的信息。

注意：

无论是否为数据库启用了Oracle Database 12 c之前的审核或统一审核，此视图都会填充到启用了审核的任何Oracle数据库中。

  • 有关统一审计的详细信息，请参见“ Oracle数据库安全指南” 。

  • 有关迁移到统一审计的详细信息，请参阅“ Oracle数据库升级指南” 。

柱 数据类型 空值 描述

PARAMETER_NAME

VARCHAR2（1024）

不是NULL

物业名称

PARAMETER_VALUE

VARCHAR2（4000）

物业的价值

AUDIT_TRAIL

VARCHAR2（28）

为其配置属性的审计跟踪：

  • 标准审计跟踪

  • FGA审计跟踪

  • 标准和FGA审计跟踪

  • OS AUDIT TRAIL

  • XML审计跟踪

  • 操作系统和XML审计跟踪

  • 所有审计跟踪

  • 统一的审计轨道

注意：

在只读数据库（包括Oracle Active Data Guard物理备用数据库）中，调用DBMS_AUDIT_MGMT.SET_AUDIT_TRAIL PL / SQL过程时不会填充此视图。如果在数据库处于只读模式时使用该过程，请使用DBMS_AUDIT_MGMT.GET_AUDIT_TRAIL检查属性的值。
