此图说明了在从Oracle 9.2.0.8或10.1.0.5到Oracle Database 11.2.0.3或更高版本的数据库升级中使用SQL Performance Analyzer的工作流程。左上角是一个标有“Oracle Database 9.2.0.8或10.1.0.5 - Production”的数据库。标有“1”的箭头从该数据库指向标记为“SQL Trace（会话子集）”的多个文档。标记为“2”的第二个箭头从该数据库指向标记为“映射表”的单个文档。标记为“3”的两个箭头从两组文档指向标有“Oracle Database 12.2”的左下角的第二个数据库。

从标记为“Oracle Database 11.2.0.3或更高版本 - 测试”的第三个数据库，标记为“5”的另一个箭头指向标记为“Oracle Database 12.2”的第二个数据库。此箭头进一步标有“Plan + Stats”文本。标有“6”的行连接到“Oracle Database 12.2”数据库中的“比较Perf，查看报告”文本。
