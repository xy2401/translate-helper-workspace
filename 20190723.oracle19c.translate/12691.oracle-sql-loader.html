第二部分 SQL * Loader

本部分讨论的主题包括SQL * Loader及其功能，以及数据加载概念（包括对象支持）。

  • SQL * Loader概念
    这些部分描述了SQL * Loader概念。
  • SQL * Loader命令行参考
    您可以使用命令行参数来启动SQL * Loader。
  • SQL * Loader控制文件参考
    SQL * Loader控制文件是一个文本文件，其中包含SQL * Loader作业的数据定义语言（DDL）指令。
  • SQL * Loader字段列表参考
    SQL * Loader控制文件的字段列表部分提供有关正在加载的字段的信息，例如位置，数据类型，条件和分隔符。
  • 加载对象，LOB和集合
    您可以使用SQL * Loader以各种格式加载列对象，并加载对象表，REF列，LOB和集合。
  • 常规和直接路径负载
    SQL * Loader提供了传统的路径加载方法和直接路径加载方法。
  • SQL * Loader Express
    SQL * Loader express模式允许您快速轻松地使用SQL * Loader加载简单数据类型。
