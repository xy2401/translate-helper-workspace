配置Oracle数据库安全设置

使用Oracle Database Configuration Assistant（DBCA）命令在安装后更改安全设置。

要在安装后启用或禁用数据库安全性配置，必须使用命令行Oracle数据库配置助手（DBCA）选项。根据设计，DBCA图形用户界面（GUI）无法启用或禁用安全配置。例如，要在安装后启用安全设置，请使用以下格式的命令，其中myRACdb1.example.com是本地数据库实例的名称，或者是初始化参数DB_UNIQUE_NAME的值集：

dbca -configureDatabase -sourceDB myRACdb1.example.com -SID -enableSecurityConfiguration true
