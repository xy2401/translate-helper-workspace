1.265 PLSQL_DEBUG

PLSQL_DEBUG指定是否将编译PL / SQL库单元以进行调试。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

注意：

不推荐使用PLSQL_DEBUG参数。它仅为了向后兼容而保留。

值

  • 真正

    将编译PL / SQL库单元以进行调试

  • 假

    将编译PL / SQL库单元以进行正常执行

当PLSQL_DEBUG设置为true ，PL / SQL库单元始终编译为INTERPRETED以便可调试。

更改此参数的值时，它对已编译的PL / SQL库单元没有影响。每个库单元持久存储此参数的值。
