5.57 RC_STORED_SCRIPT

此视图列出有关存储在恢复目录中的脚本的信息。

视图包含每个存储脚本的一行。用于脚本管理的RMAN命令（如LIST SCRIPT NAMES和LIST SCRIPT提供了更方便的查看此信息的方法。

柱 数据类型 描述

DB_KEY

数

拥有此存储脚本的数据库的主键。使用此列可以与几乎任何其他目录视图连接。

DB_NAME

VARCHAR2（8）

此记录所属的数据库化身的DB_NAME 。

SCRIPT_NAME

VARCHAR2（100）

存储脚本的名称。

SCRIPT_COMMENT

VARCHAR2（255）

用户在创建此脚本时输入的注释。如果未输入注释，则为NULL 。
