更改功能

目的

使用PL / SQL定义函数。因此，本节提供了一些常规信息，但有关语法和语义的详细信息，请参阅Oracle Database PL / SQL语言参考 。

使用ALTER FUNCTION语句重新编译无效的独立存储函数。显式重新编译消除了对隐式运行时重新编译的需要，并防止了相关的运行时编译错误和性能开销。

此声明不会更改现有函数的声明或定义。要重新声明或重新定义函数，请使用带有OR REPLACE子句的CREATE FUNCTION语句。请参见CREATE FUNCTION 。

先决条件

该函数必须位于您自己的架构中，或者您必须具有ALTER ANY PROCEDURE系统特权。

句法

alter_function :: =

语义

模式

指定包含该函数的模式。如果省略schema ，则Oracle数据库假定该函数位于您自己的模式中。

FUNCTION_NAME

指定要重新编译的函数的名称。
