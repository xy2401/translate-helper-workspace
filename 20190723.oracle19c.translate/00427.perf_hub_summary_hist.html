这是在选择时间段字段中选择历史 - 全部时，性能中心页面的屏幕截图。

Performance Hub标题位于页面的左上角，表示从Select Time Period字段中选择Historical - All。“选择时间段”字段右侧是“隐藏时间选择器”图标。隐藏时间选择器图标右侧是PerfHub报告图标。“PerfHub报告”图标右侧是AWR报告图标。最右边是页面刷新字段和页面上次刷新的时间。

时间选择器显示在页面顶部的用户界面元素下方。它由选定时间段的性能数据组成。

时间选择器下方有八个选项卡，分别是摘要，活动，工作负载，受监控的SQL，ADDM，数据库时间，资源和系统统计信息选项卡。将显示“摘要”选项卡。

“摘要”选项卡左上角是“配置”部分，其中显示了数据库名称，ID，实例名称，版本，主机名和平台名称的值。

“配置”部分右侧是“等待类”图表。它显示其他，网络，配置，提交，应用程序，并发，系统I / O，用户I / O和CPU类别的所有等待类的平均活动会话度量标准。

在“配置”部分下方，将显示“每秒加载配置文件”图表。它显示了逻辑读取，I / O块，I / O字节，事务，解析计数，用户调用，执行计数和登录的性能数据。

“每秒加载配置文件”图表右侧是“活动会话”图表。它显示其他人，等待，用户I / O和CPU的活动会话数据。

“活动会话”图表右侧是“主机CPU”图表。它显示实例和其他进程的CPU使用情况。

主机CPU图表右侧是每秒I / O请求数图表。它显示了写请求和读请求。

“每秒I / O请求数”图表右侧是“内存”图表。它显示了PGA Allocated，Log Buffer，Java Pool，Shared Pool和Buffer Cache的内存使用情况。
