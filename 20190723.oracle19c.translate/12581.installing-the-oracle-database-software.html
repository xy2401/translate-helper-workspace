安装Oracle数据库软件

这些主题说明了如何运行Oracle Universal Installer来执行大多数数据库安装。

注意：

  • 如果您计划使用Oracle Restart或Oracle ASM，则必须在安装和创建数据库之前为独立服务器安装Oracle Grid Infrastructure。否则，您必须使用Oracle Restart手动注册数据库。

  • 在启动数据库安装之前，可能必须关闭现有的Oracle进程。

  • 您可以使用静默或响应文件安装方法安装Oracle数据库，而无需GUI。此方法对于执行Oracle数据库的多个安装非常有用。

  • 安装向导用于创建映像的安装选项
    在启动Oracle数据库或Oracle Grid Infrastructure安装的安装向导之前，请确定是否要使用任何可用的映像创建选项。
  • 运行Oracle数据库安装向导以安装Oracle数据库
    提取数据库映像文件并使用runInstaller命令开始安装。
