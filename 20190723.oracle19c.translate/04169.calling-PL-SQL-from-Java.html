8.5从Java调用PL / SQL

Java数据库连接（JDBC）使您可以调用PL / SQL存储的函数和过程。例如，您要调用以下存储函数，该函数返回指定银行帐户的余额：

功能余额（acct_id NUMBER）返回号码是acct_bal NUMBER; BEGIN SELECT bal INTO acct_bal FROM accts WHERE acct_no = acct_id;返回acct_bal;结束;

在JDBC程序中，对balance函数的调用可以写成如下：

...CallableStatement cstmt = conn.prepareCall（“{？= CALL余额（？）}“）; cstmt.registerOutParameter（1，类型。浮动）; cstmt.setInt（2，acctNo）; cstmt.executeUpdate（）; float acctBal = cstmt.getFloat（1）; ...
