1.272 PROCESSOR_GROUP_NAME

PROCESSOR_GROUP_NAME指定此实例应运行的处理器组的名称。

属性 描述

参数类型

串

句法

PROCESSOR_GROUP_NAME = string

默认值

没有默认值

可修改

没有

可在PDB中修改

没有

基本

没有

Oracle RAC

对于Oracle RAC数据库，建议将每个数据库实例的操作系统处理器组命名为相同，以便可以使用相同的PROCESSOR_GROUP_NAME值。但是，您可以选择使用不同的名称。在这种情况下，您将为每个数据库实例显式设置PROCESSOR_GROUP_NAME 。

PROCESSOR_GROUP_NAME指示数据库实例在指定的操作系统处理器组中自行运行。所有Oracle进程都将绑定到该组中的CPU，并且只能在这些CPU上运行。对于NUMA系统，数据库实例分配的所有SGA和PGA内存将从组内的NUMA节点分配。

Linux和Solaris操作系统支持此初始化参数。

注意：

Oracle建议仅为USE_DEDICATED_BROKER初始化参数也设置为TRUE数据库设置此参数。
