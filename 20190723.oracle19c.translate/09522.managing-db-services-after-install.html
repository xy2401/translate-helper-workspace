安装后管理数据库服务

使用服务器控制实用程序（SRVCTL），或使用Oracle Enterprise Manager Database Express或Oracle Enterprise Manager云控制来管理和监视Oracle RAC数据库的数据库服务。

您无法使用Oracle数据库配置助手（DBCA）来管理Oracle RAC数据库的数据库服务。

Oracle Enterprise Manager Cloud Control可单独在Oracle Enterprise Manager Cloud Control安装介质上以及以下网站的Oracle Technology Network网站上提供：

http://www.oracle.com/technetwork/indexes/documentation/index.html
