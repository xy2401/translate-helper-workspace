目的

NTH_VALUE返回analytic_clause定义的窗口中第n行的measure_expr值。返回的值具有measure_expr的数据类型。

  • { RESPECT | IGNORE } NULLS确定measure_expr空值是否包含在计算中或从计算中消除。默认值为RESPECT NULLS 。

  • n确定要返回度量值的第n行。 n可以是常量，绑定变量，列或涉及它们的表达式，只要它解析为正整数即可。如果数据源窗口少于n行，则该函数返回NULL 。如果n为null，则该函数返回错误。

  • FROM { FIRST | LAST }确定计算是从窗口的第一行还是最后一行开始。默认值为FROM FIRST 。

如果省略analytic_clause的windowing_clause ，则默认为RANGE BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW 。此默认值有时会返回NTH_VALUE的意外值...FROM LAST ...，因为窗口中的最后一个值位于窗口的底部，这是不固定的。它会随着当前行的变化而不断变化。对于预期的结果，指定windowing_clause为RANGE BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING 。或者，您可以将windowing_clause指定为RANGE BETWEEN CURRENT ROW AND UNBOUNDED FOLLOWING 。
