6.479 USER_JSON_COLUMNS

USER_JSON_COLUMNS提供有关用户所有者的JavaScript Object Notation（JSON）列的信息。它的列（ OWNER除外）与ALL_JSON_COLUMNS列相同。

此视图中将显示用户拥有的每个具有AND条件的IS JSON检查约束的列。此视图使用户能够查找他或她拥有的所有JSON列。

例如，如果检查约束将IS JSON条件与使用逻辑条件OR的另一个条件组合在一起，则该列不会在此视图中列出。在这种情况下，不确定列中的数据是否为JSON数据。例如，以下约束不能确保列jcol中的数据是JSON数据：

jcol是json OR length（jcol）<1000
