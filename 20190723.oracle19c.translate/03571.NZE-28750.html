NZE-28750：未知错误

原因：发生了未指定类型的Oracle Security Server错误。

操作：启用跟踪以确定此错误的确切原因。如果需要，请联系Oracle客户支持。

NZE-28751：内存分配失败

原因：由于系统内存不足，内存分配失败。

操作：终止其他进程以释放内存或向系统添加内存。

NZE-28752：数据源初始化失败

原因：数据源无法初始化。通常，此错误是由系统资源不足引起的。

操作：启用跟踪以确定此错误的确切原因。如果需要，请联系Oracle客户支持。

NZE-28753：未能终止使用数据源

原因：系统无法终止其数据源的使用。此错误可能是由内存或磁盘损坏等引起的。

操作：启用跟踪以确定此错误的确切原因。如果需要，请联系Oracle客户支持。

NZE-28755：对象检索失败

原因：系统无法从文件或数据库中检索信息。

操作：检查数据源是否存在，或检查以确保存在正确的信息。

NZE-28756：回调无法分配内存

原因：调用者提供的函数无法分配内存。此错误可能是由系统内存耗尽引起的。

操作：终止其他进程以释放内存或向系统添加内存。

NZE-28757：系统尝试分配0字节的内存

原因：系统尝试分配0字节的内存，因为没有可用的内存可供分配。此错误可能是由系统内存不足引起的。

操作：终止其他进程以释放内存或向系统添加内存。

NZE-28758：内存释放失败

原因：内存片段的释放失败。这可能是由于应用程序内部或Oracle Security Server库中的内存损坏。

操作：启用跟踪以确定此错误的确切原因。如果需要，请联系Oracle客户支持。

NZE-28759：无法打开文件
