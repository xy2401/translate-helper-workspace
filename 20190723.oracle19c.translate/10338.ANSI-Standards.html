ANSI标准

美国国家标准协会（ANSI）的以下文件与SQL有关：

  • INCITS / ANSI / ISO / IEC 9075-1：2016，信息技术 - 数据库语言-SQL-第1部分：框架（SQL /框架）

  • INCITS / ANSI / ISO / IEC 9075-2：2016，信息技术 - 数据库语言 - SQL - 第2部分：基础（SQL / Foundation）

  • INCITS / ANSI / ISO / IEC 9075-3：2016，信息技术 - 数据库语言 - SQL - 第3部分：调用级接口（SQL / CLI）

  • INCITS / ANSI / ISO / IEC 9075-4：2016，信息技术 - 数据库语言 - SQL - 第4部分：持久存储模块（SQL / PSM）

  • INCITS / ANSI / ISO / IEC 9075-9：2016，信息技术 - 数据库语言 - SQL - 第9部分：外部数据管理（SQL / MED）

  • INCITS / ANSI / ISO / IEC 9075-10：2016，信息技术 - 数据库语言 - SQL - 第10部分：对象语言绑定（SQL / OLB）

  • INCITS / ANSI / ISO / IEC 9075-11：2016，信息技术 - 数据库语言-SQL-第11部分：信息和定义模式（SQL / Schemata）

  • INCITS / ANSI / ISO / IEC 9075-13：2016，信息技术 - 数据库语言-SQL-第13部分：使用Java编程语言的SQL例程和类型（SQL / JRT）

  • INCITS / ANSI / ISO / IEC 9075-14：2016，信息技术 - 数据库语言 - SQL - 第14部分：与XML相关的规范（SQL / XML）

这些标准与下一节中列出的相应ISO标准相同。
