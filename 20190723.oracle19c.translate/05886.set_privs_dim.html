在尺寸上设置对象安全性：此屏幕截图显示“设置尺寸PRODUCT对象安全性”对话框。尺寸名称为<ALL DIMENSIONS>。用户或角色是PUBLIC。选择“将选择权限扩展到维度和层次结构视图”选项。已选中“扩展分析工作区GLOBAL的所有维度权限”选项。未选择With Grant Option选项。可用权限是INSERT，UPDATE，DELETE和ALTER。 Selected Privilege是SELECT。在“SQL命令”列表中，可以看到以下语句：GRANT SELECT ON GLOBAL.CUSTOMER_MARKET_VIEW to PUBLIC，GRANT SELECT ON GLOBAL.PRODUCT TO PUBLIC，GRANT SELECT ON GLOBAL.PRODUCT_VIEW to PUBLIC，GRANT SELECT ON GLOBAL.PRODUCT_PRIMARY_VIEW to PUBLIC，GRANT SELECT ON GLOBAL.TIME TO PUBLIC，GRANT SELECT GLOBAL.TIME_VIEW to PUBLIC，GRANT SELECT GLOBAL.TIME_CALENDAR_VIEW to PUBLICGRANT SELECT GLOBAL.TIME_FISCAL_VIEW to PUBLIC。
