8.129 V $ PROCESS_MEMORY_DETAIL

V$PROCESS_MEMORY_DETAIL提供有关每个自动捕获的快照的动态PGA内存使用情况的详细信息。

柱 数据类型 描述

PID

数

Oracle进程标识符

SERIAL＃

数

Oracle进程序列号

类别

VARCHAR2（15）

分类名称。类别包括：

  • SQL

  • PL / SQL

  • OLAP

  • JAVA

  • Freeable

  • 其他

名称

VARCHAR2（26）

PGA内存分配评论。出于性能原因，可以将小分配组合在一起，并将NAME设置为Miscellaneous 。

HEAP_NAME

VARCHAR2（15）

包含分配的堆或堆（如果相同名称）的名称

BYTES

数

从堆中使用给定堆名和给定分配注释分配的PGA内存字节数

ALLOCATION_COUNT

数

具有给定堆名称的堆内部进程中发现的注释的分配数

HEAP_DESCRIPTOR

RAW（4 | 8）

如果所有分配都来自一个堆，那么这就是该堆的堆描述符的地址。否则，此列为NULL 。

PARENT_HEAP_DESCRIPTOR

RAW（4 | 8）

如果所有分配都来自一个堆，那么这是该堆的父堆描述符的地址。否则，此列为NULL 。如果堆没有父级，则值为零。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
