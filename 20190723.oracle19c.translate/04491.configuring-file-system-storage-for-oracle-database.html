8配置Oracle数据库的文件系统存储

完成这些过程以使用Oracle Database的文件系统存储。

为了获得最佳的数据库组织和性能，Oracle建议您将数据文件和Oracle数据库软件安装在不同的磁盘中。

如果计划在网络文件系统（NFS）协议设备上放置存储，则Oracle建议您使用Oracle Direct NFS（dNFS）来利用Oracle Direct NFS客户端内置的性能优化。

  • 配置Oracle数据库的NFS缓冲区大小参数
    将NFS缓冲区大小参数rsize和wsize的值设置为至少32768。
  • 检查直接NFS客户端的TCP网络协议缓冲区
    检查TCP网络缓冲区大小，确保它足以满足服务器的速度。
  • 为直接NFS客户端创建oranfstab文件
    Direct NFS使用配置文件oranfstab来确定可用的挂载点。
  • 启用和禁用NFS的直接NFS客户端控制
    默认情况下，Direct NFS Client安装在具有单实例Oracle数据库安装的禁用状态。在启用Direct NFS之前，必须配置oranfstab文件。
