1.264 PLSQL_CODE_TYPE

PLSQL_CODE_TYPE指定PL / SQL库单元的编译模式。

属性 描述

参数类型

串

句法

PLSQL_CODE_TYPE = {解释| NATIVE}

默认值

INTERPRETED

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

值

  • INTERPRETED

    PL / SQL库单元将被编译为PL / SQL字节码格式。这些模块由PL / SQL解释器引擎执行。

  • NATIVE

    PL / SQL库单元（可能除了顶级匿名PL / SQL块）将被编译为本机（机器）代码。这些模块将在本地执行，而不会产生任何解释器开销。

更改此参数的值时，它对已编译的PL / SQL库单元没有影响。每个库单元持久存储此参数的值。

如果PL / SQL库单元是本机编译的，则该库单元的所有后续自动重新编译都将使用本机编译。
