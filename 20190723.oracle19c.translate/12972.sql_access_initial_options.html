屏幕截图显示了SQL Access Advisor：Initial Options页面。

页面标题下方是文本“选择一组初始选项”。这两个选项是：验证访问结构的使用（索引，物化视图，分区等）仅限和推荐新访问结构（已选中）。与推荐新访问结构相关联的是先前保存的任务或模板的继承选项复选框。它被选中了。

取消和继续按钮位于最右侧。

“概述”部分具有以下文本：SQL Access Advisor评估工作负载源中的SQL语句，并可以建议索引，分区，物化视图和物化视图日志，从而提高整个工作负载的性能。

提示说，“您正在选择向导的起点。可以在向导中更改所有选项。

“任务和模板”部分具有“视图”列表，其值为“仅模板”。下面是“视图选项”按钮。

该部分中的表具有以下列：选择，名称，描述，上次修改和类型。“选择”列中的值是选项。行的值如下。

第1行:(已选中），SQLACESS_EMTASK，默认企业管理器任务模板，2012年1月11日下午1:27:47 CST，默认模板。

第2行:(未选中），SQLACCESS_GENERAL，通用数据库模板，2012年1月11日下午1:27:46 CST，模板。

第3行:(未选中），SQLACCESS_OLTP，OLTP数据库模板，2012年1月11日下午1:27:46 CST，模板。

第4行:(未选中），SQLACCESS_WAREHOUSE，数据仓库数据库模板，2012年1月11日下午1:27:46 CST，模板。

描述结束。
