12.6附录

句法

一个[PPEND] 文字

其中text表示要追加的文本。

添加指定的文本到当前行的末尾在SQL缓冲区中 。

要使用空格将文本与前面的字符分开，请在APPEND和文本之间输入两个空格。

要附加以分号结尾的文本，请以两个分号结束命令（SQL * Plus将单个分号解释为可选的命令终止符）。

例子

要将逗号分隔符，空格和列名CITY附加到缓冲区的第一行，请通过列出该行来使该行成为当前行，如下所示：

1
1 * SELECT DEPARTMENT_ID

现在输入APPEND：

APPEND，CITY 1
1 * SELECT DEPARTMENT_ID，CITY

要在行中附加分号，请输入

APPEND ;;

SQL * Plus将第一个分号附加到该行，并将第二个分号解释为APPEND命令的终止符。
