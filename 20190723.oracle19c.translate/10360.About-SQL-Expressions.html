关于SQL表达式

表达式是一个或多个值，运算符和SQL函数的组合，它们计算为值。表达式通常假定其组件的数据类型。

此简单表达式的计算结果为4，数据类型为NUMBER （与其组件的数据类型相同）：

2 * 2

以下表达式是使用函数和运算符的更复杂表达式的示例。表达式将7天添加到当前日期，从总和中删除时间组件，并将结果转换为CHAR数据类型：

TO_CHAR（TRUNC（SYSDATE + 7））

您可以在以下表达式中使用

  • SELECT语句的选择列表

  • WHERE子句和HAVING子句的条件

  • CONNECT BY ， START WITH和ORDER BY子句

  • INSERT语句的VALUES子句

  • UPDATE语句的SET子句

例如，您可以在此UPDATE语句SET子句中使用表达式代替带引号的字符串'Smith' ：

SET last_name ='Smith';

此SET子句的表达式为INITCAP （ last_name ），而不是带引号的字符串' Smith '：

SET last_name = INITCAP（last_name）;

表达式有多种形式，如以下语法所示：

expr :: =
