13.5基本LOOP声明

随着基本LOOP语句的每次迭代，其语句运行并且控制返回到循环的顶部。当LOOP内的语句将控制转移到循环外或引发异常时， LOOP语句结束。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

basic_loop_statement :: =


说明basic_loop_statement.eps的说明

（ statement :: = ）

语义

basic_loop_statement

声明

为了防止无限循环，至少有一个语句必须在循环外转移控制。可以在循环外传递控制的语句是：

  • “ CONTINUE Statement ” （当它将控制转移到封闭标记循环的下一次迭代时）

  • “ 退出声明 ”

  • “ GOTO声明 ”

  • “ 提升声明 ”

标签

标识basic_loop_statement的标签（请参阅“ statement :: = ”和“ label ” ）。CONTINUE ， EXIT和GOTO语句可以引用此标签。

标签提高了可读性，尤其是在嵌套LOOP语句时，但LOOP是确保END LOOP语句中的标签与同一LOOP语句开头的标签匹配（编译器不检查）。
