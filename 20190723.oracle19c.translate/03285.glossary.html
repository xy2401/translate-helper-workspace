词汇表

适配器

一种实时专有工具，用于从另一个数据库访问存储在一个数据库中的数据。适配器通常用于转换SQL，映射数据类型以及促进SQL语句，触发器和存储过程的集成。

自定义SQL翻译

用户可以使用SQL Translation Profile注册其客户特定的SQL语句翻译的方案。在翻译非Oracle语句期间，配置文件首先查找自定义翻译。然后，如果未找到匹配项，则调用SQL Translator。

数据集成

不同数据库之间的数据交换，可以在实时事务中异步进行，也可以作为批处理进行同步。

数据集成框架

用于在不同数据库之间进行数据交换的一组工具和过程。传统框架包括许多夜间流程，例如大批量提取和提要，以及批量加载数据。较新的框架可以包括近乎实时发生的小型日常流程和馈送。

数据库模式迁移

识别和转换非Oracle模式中的表，列和其他对象以符合Oracle数据库所需的命名，大小和其他约定的过程。

错误翻译

用户可以使用SQL转换配置文件注册特定于供应商的错误代码和消息转换的方案。在SQL执行期间，客户端应用程序依赖于特定于供应商的错误代码和消息。发生错误时，将返回已转换的错误代码和消息，而不是Oracle错误代码和消息。

移民

修改非Oracle应用程序（包括其所有组件（如体系结构，数据，SQL代码和客户端））以使用Oracle RDBMS而非专有数据库管理系统的过程。

迁移存储库

Oracle数据库中的数据存储，Oracle SQL Developer使用该数据存储在迁移期间管理源和目标模式模型的元数据。多个迁移存储库可用于同时从多个数据库迁移到Oracle数据库。

Oracle数据库网关

一组Oracle产品，使用一致的API同步支持与非Oracle系统的数据集成。

Oracle GoldenGate

一种Oracle产品，支持存储在SQL Server，Sybase，DB2，Oracle和其他数据库中的各种数据源之间的模块化事务级数据集成。

Oracle SQL * Loader

一个快速，灵活且免费的Oracle实用程序，支持将平面文件中的数据加载到Oracle数据库中。它支持多种数据格式和许多不同的编码。它还支持并行数据加载。

Oracle SQL Developer迁移向导

一种Oracle工具，支持以批处理模式将第三方数据库迁移到Oracle数据库。迁移包括数据，模式，对象，触发器和存储过程。

SQL方言

由数据库供应商实现的SQL的变体或扩展。将客户端应用程序从第三方数据库迁移到Oracle时，必须将所有非Oracle SQL语句转换为Oracle SQL。由于这些非Oracle SQL语句嵌入在客户端应用程序的源代码中，因此查找和翻译它们是一项耗时的手动任务。此版本增强了Oracle数据库以接受来自外部供应商的非Oracle SQL语句，并在执行之前在运行时自动翻译它们。

SQL翻译配置文件

一种数据库模式对象，用于指导非Oracle SQL语句如何转换为Oracle SQL方言。此架构还包含错误代码，SQLSTATE和SQL执行期间发生错误时要返回的错误消息的转换。

将具有非Oracle SQL语句的客户端应用程序迁移到Oracle时，用户会创建SQL转换配置文件并对其进行配置以转换应用程序的SQL语句和错误。在运行时，应用程序在Oracle数据库中设置转换配置文件以转换其SQL语句和错误。

SQL转换器

SQL转换程序是由Oracle或第三方供应商提供的软件组件，可以安装在Oracle数据库中。它在Oracle Database SQL编译器处理客户端程序之前对其进行转换。如果由转换的SQL语句执行导致错误，则Oracle Database SQL编译器会生成Oracle错误消息。

SQLSTATE

ANSI SQL标准定义的状态参数。它是一个5个字符的字符串，表示SQL操作的状态。其中一些值是：

  • 00 XXX ：不合格的成功完成

  • 01 XXX ：警告

  • 02 XXX ：没有数据

  • 07 XXX ：动态SQL错误

  • 08 XXX ：连接例外

  • 09 XXX ：触发动作例外
