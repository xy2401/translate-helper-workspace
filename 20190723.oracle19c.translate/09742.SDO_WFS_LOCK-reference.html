37 SDO_WFS_LOCK包（WFS）

MDSYS.SDO_WFS_LOCK包中包含WFS支持的子程序，用于注册和取消注册要素表。

注册功能表可启用WFS事务锁定表;取消注册功能表会禁用WFS事务锁定的表。

要使用本章中的子程序，您必须了解Web要素服务（WFS）支持中有关Web要素服务（WFS）的概念和用法信息。

本章的其余部分提供了有关子程序的参考信息，按字母顺序列出。

  • SDO_WFS_LOCK.EnableDBTxns
  • SDO_WFS_LOCK.RegisterFeatureTable
  • SDO_WFS_LOCK.UnRegisterFeatureTable

37.1 SDO_WFS_LOCK.EnableDBTxns

格式

SDO_WFS_LOCK.EnableDBTxns（）;

描述

在WFS表上启用数据库事务。

参数

没有。

使用说明

此过程在会话结束时将覆盖WFS表上针对数据库事务的WFS-T标准限制，以便具有当前会话ID的任何事务都可以对WFS表执行更新和删除操作。在允许当前事务修改行之前，Oracle数据库触发器仍会检查WFS锁定。因此，如果WFS事务对某行具有锁定，则触发器将不允许执行该操作。但是，如果当前行上没有WFS锁定，则触发器将允许当前事务修改该行。

在使用Oracle Workspace Manager对WFS表进行版本启用时，必须调用此过程来执行某些操作，如将WFS与Oracle Workspace Manager一起使用中所述 。但是，即使不将Workspace Manager与WFS表一起使用，也可以使用此过程。

有关Web功能服务支持的信息，请参阅Web功能服务（WFS）支持 。

例子

以下示例在WFS表上为当前会话的剩余部分启用数据库事务。

BEGIN SDO_WFS_LOCK.enableDBTxns;结束; /

37.2 SDO_WFS_LOCK.RegisterFeatureTable

格式

SDO_WFS_LOCK.RegisterFeatureTable（用户名IN VARCHAR2，表名IN VARCHAR2）;

描述

注册功能表;也就是说，启用WFS事务锁定的功能表。

参数

用户名

拥有要注册的要素表的数据库用户的名称。

tablename

要注册的要素表的名称。

使用说明

此过程确保实现WFS事务语义的必要约束与功能表相关联。

有关Web功能服务支持的信息，请参阅Web功能服务（WFS）支持 。

要取消注册尚未启用版本的表，请使用SDO_WFS_LOCK.UnRegisterFeatureTable过程。

例子

以下示例注册名为COLA_MARKETS_CS的功能表，该功能表由用户WFS_REL_USER拥有。

BEGIN SDO_WFS_LOCK.registerFeatureTable（'wfs_rel_user'，'cola_markets_cs'）;结束; /

37.3 SDO_WFS_LOCK.UnRegisterFeatureTable

格式

SDO_WFS_LOCK.UnRegisterFeatureTable（用户名IN VARCHAR2，表名IN VARCHAR2）;

描述

取消注册功能表;也就是说，禁用WFS事务锁定的功能表。

参数

用户名

拥有要取消注册的要素表的数据库用户的名称。

tablename

要注销的要素表的名称。

使用说明

此过程为功能表禁用实现WFS事务语义的约束。

必须先在调用SDO_WFS_LOCK.RegisterFeatureTable过程中注册要素表。

有关Web功能服务支持的信息，请参阅Web功能服务（WFS）支持 。

例子

以下示例取消注册名为COLA_MARKETS_CS的功能表，该功能表由用户WFS_REL_USER拥有。

BEGIN SDO_WFS_LOCK.unRegisterFeatureTable（'wfs_rel_user'，'cola_markets_cs'）;结束; /
