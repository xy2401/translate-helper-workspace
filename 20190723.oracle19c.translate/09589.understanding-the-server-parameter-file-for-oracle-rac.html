了解Oracle RAC的服务器参数文件

创建数据库时，Oracle数据库会在您指定的文件位置中创建SPFILE。此位置可以是Oracle ASM磁盘组或群集文件系统。

群集数据库中的所有实例在启动时都使用相同的SPFILE。由于SPFILE是二进制文件，因此不要使用编辑器直接编辑SPFILE。而是使用Oracle Enterprise Manager或ALTER SYSTEM SQL语句更改SPFILE参数设置。
