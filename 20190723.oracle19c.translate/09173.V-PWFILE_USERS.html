8.140 V $ PWFILE_USERS

V$PWFILE_USERS列出密码文件中的所有用户，并指示是否已授予用户SYSDBA ， SYSOPER ， SYSASM ， SYSBACKUP ， SYSDG和SYSKM权限。

柱 数据类型 描述

用户名

VARCHAR2（128）

密码文件中包含的用户的名称

SYSDBA

VARCHAR2（5）

指示用户是否可以使用SYSDBA权限（ TRUE ）连接（ FALSE ）

SYSOPER

VARCHAR2（5）

指示用户是否可以使用SYSOPER权限（ TRUE ）连接（ FALSE ）

SYSASM

VARCHAR2（5）

指示用户是否可以使用SYSASM权限（ TRUE ）连接（ FALSE ）

SYSBACKUP

VARCHAR2（5）

指示用户是否可以使用SYSBACKUP权限（ TRUE ）连接（ FALSE ）

SYSDG

VARCHAR2（5）

指示用户是否可以使用SYSDG权限（ TRUE ）连接（ FALSE ）

SYSKM

VARCHAR2（5）

指示用户是否可以使用SYSKM权限（ TRUE ）连接（ FALSE ）

帐户状态

VARCHAR2（30）

帐户状态：

  • 打开

  • 过期

  • 过期（GRACE）

  • 锁定（定时）

  • 已锁定

  • 已过期和已锁定（已定时）

  • 过期和锁定

  • 过期（GRACE）和锁定

PASSWORD_PROFILE

VARCHAR2（128）

密码配置文件名称

上次登录

TIMESTAMP（9）带时区

上次用户登录的时间

LOCK_DATE

日期

帐户状态为LOCKED帐户被锁定的日期

到期日

日期

帐户到期日期

EXTERNAL_NAME

VARCHAR2（1024）

显示外部认证用户的证书DN或主要名称

AUTHENTICATION_TYPE

VARCHAR2（8）

表示用户的认证机制：

  • EXTERNAL: CREATE USER user1 IDENTIFIED EXTERNALLY;

  • GLOBAL: CREATE USER user2 IDENTIFIED GLOBALLY;

  • PASSWORD: CREATE USER user3 IDENTIFIED BY password3 ;

共同

VARCHAR2（3）

如果使用CONTAINER=ALL授予管理权限（例如， SYSDBA ），则此列的值为YES 。否则，该列的值为NO 。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
