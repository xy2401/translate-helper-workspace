从PDB $ SEED或XML创建PDB

岗位

/数据库/ PDBS /

从PDB $ SEED或数据库实例可访问的XML元数据文件创建可插拔数据库。仅在数据库和Oracle REST数据服务位于同一主机上的拓扑中，才支持使用Oracle透明数据加密。此服务需要在池配置中设置db.cdb.adminUser凭据。

  • 应用程序/ JSON

根架构：架构
输入： object
显示来源
  • adminName（可选）：字符串
    仅在创建PDB时指定管理员用户名。
  • adminPwd（可选）：字符串
    仅在创建PDB时指定管理员用户名。
  • asCopy（可选）：布尔值
    由于Copy是PLUG操作专用的。
  • copyAction（可选）：string
    允许值： [ "COPY", "NOCOPY", "MOVE" ]
    PLUG运营的独家财产。
  • fileNameConversions：string
    与创建和插件操作相关。如Oracle Multitenant数据库文档中所定义。值可以是文件名转换模式或NONE。
  • getScript（可选）：boolean
    如果已定义，则响应将包含一个JSON对象，其中包含为执行而生成的脚本信息。当此属性设置为true时，不会创建数据库。
  • 方法：字符串
    允许值： [ "CREATE", "PLUG" ]
    此属性定义操作类型。
  • pdb_name：string
    在创建/插入操作的情况下，新的PDB名称。
  • reuseTempFile（可选）：boolean
    与创建和插件操作相关。对于临时文件重用是真的。
  • sourceFileNameConversions（可选）：string
    与插件操作相关。如Oracle Multitenant数据库文档中所定义。值可以是源文件名转换模式或NONE。
  • tdeImport（可选）：布尔值
    TDE导入插件操作。
  • tdeKeystorePath（可选）：string
    如果tdeImport标志设置为true，则需要TDE密钥库路径。可用于插件操作。
  • tdePassword（可选）：字符串
    适用时的TDE密码（可选）。可用于创建和插件操作。
  • tdeSecret（可选）：字符串
    如果tdeImport标志设置为true，则需要TDE密钥。可用于插件操作。
  • tempSize（可选）：字符串
    与创建和克隆操作相关。Oracle Multitenant数据库文档中定义的临时表空间的总大小。请参阅“数据库SQL语言参考”文档中的size_clause说明。
  • totalSize（可选）：字符串
    与创建和插件操作相关。Oracle Multitenant数据库文档中定义的总大小。请参阅“数据库SQL语言参考”文档中的size_clause说明。
  • unlimitedStorage（可选）：布尔值
    与创建和插件操作相关。适用于无限存储。即使设置为true，如果方法是CREATE，则必须在请求中指定totalSize和tempSize。
  • xmlFileName（可选）：string
    PLUG运营的独家财产。
