2.204 ALL_LOG_GROUPS

ALL_LOG_GROUPS描述当前用户可访问的表上的日志组定义。

相关观点

  • DBA_LOG_GROUPS描述数据库中所有表的日志组定义。

  • USER_LOG_GROUPS描述了当前用户拥有的表上的日志组定义。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

日志组定义的所有者

LOG_GROUP_NAME

VARCHAR2（128）

不是NULL

日志组定义的名称

TABLE_NAME

VARCHAR2（128）

不是NULL

定义日志组的表的名称

LOG_GROUP_TYPE

VARCHAR2（28）

日志组的类型：

  • 主要关键记录

  • 独特的钥匙记录

  • 外国钥匙记录

  • 所有栏目记录

  • 用户登录组

总是

VARCHAR2（11）

Y表示每次更新行时都会记录日志组; N表示每次更新成员列时都会记录日志组

GENERATED

VARCHAR2（14）

指示补充日志组的名称是否是系统生成的（ GENERATED NAME ）（ USER NAME ）
