4.14 DBA_ADVISOR_DIR_DEFINITIONS

DBA_ADVISOR_DIR_DEFINITIONS提供了基本指令的定义。

柱 数据类型 空值 描述

ID

数

不是NULL

指令的唯一ID。指令管理引擎自动生成ID号。无论域名和指令名称如何，标识符在所有指令中都是唯一的。

ADVISOR_ID

数

不是NULL

所有者顾问的标识号。

ADVISOR_NAME

VARCHAR2（128）

不是NULL

此指令所属的顾问的名称。

DIRECTIVE_NAME

VARCHAR2（128）

不是NULL

在域中进一步对此指令进行分类的任何值。域和名称构成指令的唯一键。

DOMAIN_NAME

VARCHAR2（128）

不是NULL

域名或命名空间名称。

描述

VARCHAR2（256）

不是NULL

描述指令目的的可选描述。

类型

数

不是NULL

进一步描述了该指令的使用。可能的值是：

  1. 过滤器 - Xpath过滤器

  2. 单值 - 评估返回单个字符串值

  3. 多个值 - 评估返回一个到多个字符串值

  4. 条件 - 评估基于输入键返回单个值，类似于CASE或SWITCH语句

TYPE_NAME

VARCHAR2（15）

TYPE列的解码版本。

TASK_STATUS

VARCHAR2（9）

任务未处于初始状态时指令实例的状态。可能的值是：

  • IMMUTABLE

  • MUTABLE

INSTANCES

VARCHAR2（8）

指示指令是否允许多个实例。可能的值是：

  • 单

  • 多

元数据

CLOB

不是NULL

用于处理指令的DTD。
