Oracle数据库可能的客户端和服务器配置

选择客户端/服务器配置以运行预编译器和OCI应用程序。

您的预编译器和OCI应用程序在客户端/服务器环境中的客户端上运行，其中Oracle数据库服务器是服务器。您可以在您的环境中使用以下一个或多个客户端/服务器配置

不同计算机上的Oracle数据库客户端和服务器

客户端软件和服务器软件位于不同的计算机上，它们通过网络连接。客户端和服务器环境是分开的。

同一台计算机上不同Oracle位置的Oracle数据库客户端和服务器

客户端软件和服务器软件位于同一台计算机上，但它们安装在不同的Oracle主目录中。同样，客户端和服务器环境是分开的。

同一Oracle位置中的Oracle数据库客户端和服务器

客户端软件和服务器软件安装在同一台计算机上的同一Oracle主目录中。在这种情况下，服务器软件的任何升级也是客户端软件的升级。

Oracle数据库客户端和服务器软件的软件升级类型

检查以确定版本是主要版本还是维护版本。

Oracle数据库客户端和服务器软件有两种类型的升级：Oracle数据库的主要版本和Oracle数据库的维护版本。

Oracle数据库主要版本升级

在主要版本中，升级会更改版本号的第一个数字。例如，从Oracle数据库11克升级到Oracle数据库12c为一个主要的数据库版本升级。

Oracle数据库维护版本升级

在维护版本中，升级会更改版本号的第二个数字。例如，从Oracle Database 12 c第1版（12.1）升级到Oracle Database 12 g第 2版（12.2）是数据库维护版本升级。但是，Oracle可以在维护版本中引入新功能。
