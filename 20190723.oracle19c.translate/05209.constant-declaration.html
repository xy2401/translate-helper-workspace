13.14不断宣言

常量保持不变的值。常量声明指定常量的名称，数据类型和值，并为其分配存储。声明也可以强加NOT NULL约束。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

constant_declaration :: =


插图说明constant_declaration.eps

（ datatype :: = ， expression :: = ）

语义

constant_declaration

不变

您要声明的常量的名称。

数据类型

可以使用初始值声明变量的数据类型。

不是NULL

对常量强加NOT NULL约束。

有关此约束的信息，请参阅“ NOT NULL约束 ” 。

表达

常数的初始值。 expression必须具有与数据类型兼容的数据类型 。当详细说明constant_declaration时 ， 表达式的值被赋值为constant 。
