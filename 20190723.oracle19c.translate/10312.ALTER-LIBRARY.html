ALTER LIBRARY

目的

ALTER LIBRARY语句显式重新编译库。显式重新编译消除了对隐式运行时重新编译的需要，并防止了相关的运行时编译错误和性能开销。

注意：

此语句不会更改现有库的声明或定义。要重新声明或重新定义库，请使用带有OR REPLACE子句的“ CREATE LIBRARY ” 。

先决条件

如果库位于SYS架构中，则必须以SYSDBA身份连接。否则，库必须位于您自己的架构中，或者您必须具有ALTER ANY LIBRARY系统特权。

语义

模式

指定包含库的架构。如果省略schema ，则Oracle数据库假定该过程位于您自己的模式中。

LIBRARY_NAME

指定要重新编译的库的名称。
