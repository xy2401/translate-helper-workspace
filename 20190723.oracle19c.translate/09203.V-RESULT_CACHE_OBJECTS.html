8.170 V $ RESULT_CACHE_OBJECTS

V$RESULT_CACHE_OBJECTS显示所有对象（缓存结果和依赖项）及其属性。

柱 数据类型 描述

ID

数

缓存对象的标识符（也是第一个块的ID）

类型

VARCHAR2（10）

缓存对象的类型：

  • 结果

  • 依赖

状态

VARCHAR2（9）

对象的状态：

  • New结果 - 结果仍在建设中

  • Published - 结果可供使用

  • Bypass - 结果将被绕过使用

  • Expired - 结果已超出到期时间

  • Invalid - 结果不再可用

BUCKET_NO

数

对象的内部哈希桶

HASH

数

对象的哈希值

名称

VARCHAR2（387）

名称（例如，SQL前缀或PL / SQL函数名称）

NAMESPACE

VARCHAR2（10）

命名空间：

  • SQL

  • PLSQL

  • 关键矢量

CREATION_TIMESTAMP

日期

创建对象的时间

CREATOR_UID

数

创建对象的UID

DEPEND_COUNT

数

依赖项数（ TYPE = Result ）或依赖项（ TYPE = Dependency ）

BLOCK_COUNT

数

缓存对象中的块总数

SCN

数

构建SCN（ TYPE = Result ）或失效SCN（ TYPE = Dependency ）

COLUMN_COUNT

数

缓存结果Foot 1中的列数

PIN_COUNT

数

此结果Footref 1上的活动扫描数

SCAN_COUNT

数

在缓存的结果Footref 1上启动的扫描总数

ROW_COUNT

数

缓存结果Footref 1中的总行数

ROW_SIZE_MAX

数

最大行的大小（以字节为单位） Footref 1

ROW_SIZE_MIN

数

最小行的大小（以字节为单位） Footref 1

ROW_SIZE_AVG

数

行的平均大小（以字节为单位） Footref 1

BUILD_TIME

数

构建缓存结果Footref 1所花费的时间（以百分之一秒为单位）

LRU_NUMBER

数

LRU列表位置（值越小，使用越近） Footref 1

OBJECT_NO

数

依赖项对象Foot 2的字典对象编号

废票

数

对象使其依赖Footref 2无效的次数

SPACE_OVERHEAD

数

结果Footref 1的开销（以字节为单位）

SPACE_UNUSED

数

结果Footref 1的未使用空间（以字节为单位）

的cache_id

VARCHAR2（387）

结果的CacheId （对象名称，如果它是依赖项）

CACHE_KEY

VARCHAR2（387）

结果的CacheKey （对象名称，如果它是依赖项）

校验

数

校验和的结果对象。校验和是在结果缓存对象中的所有块减去对象头的情况下计算的。

EDITION_ID

数

显示计算结果时正在使用的版本的对象ID

DB_LINK

VARCHAR2（3）

可能的值：

  • YES ：如果结果缓存对象引用远程数据库对象

  • NO ：如果结果缓存对象未引用远程数据库对象

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

脚注1

这些列仅对TYPE = Result有效;否则，它们是NULL。

脚注2

这些列仅对TYPE = Dependency有效;否则，它们是NULL。
