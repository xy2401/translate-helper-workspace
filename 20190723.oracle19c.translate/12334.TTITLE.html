12.49 TTITLE

句法

TTI [TLE] [ printspec [ text | 变量 ] ...][ON | 关 ]

其中printspec表示用于放置和格式化文本的以下一个或多个子句：

BOLD CE [NTER] COL n格式文本 LE [FT] R [IGHT] S [KIP] [ n ] TAB n

地点和格式指定的标题在每个报告页面的顶部。输入没有子句的TTITLE列出其当前定义。如果引号中的单个单词或字符串跟随TTITLE命令，则使用旧形式的TTITLE。

有关旧形式TTITLE的说明，请参阅TTI [TLE]文本（过时的旧表单） 。

条款

这些条款和条款也适用于BTITLE命令。

文本

标题文字。如果要在一行上放置多个单词，请在单引号中输入文本。

变量

替换变量或以下任何一种系统维护值，SQL.LNO（当前行号），SQL.PNO（当前页码），SQL.RELEASE（当前Oracle数据库版本号），SQL.SQLCODE（当前错误代码）或SQL.USER（当前用户名）。

要打印其中一个值，请在标题中引用相应的变量。您可以使用FORMAT子句格式化变量 。

在执行TTITLE之前扩展SQL * Plus替换变量（和变量）。结果字符串存储为TTITLE文本。在每个结果页面的后续执行期间，变量的扩展值本身可以被解释为具有意外结果的替换变量。

您可以通过不对每个结果页面上要替换的变量使用＆前缀来避免在TTITLE命令中进行此双重替换。如果要使用替换变量在TTITLE中插入不变的文本，请将其括在引号中，以便仅替换一次。

关闭

变成了标题 off（抑制其显示）而不影响其定义。

上

打开标题（恢复其显示）。定义顶部标题时，SQL * Plus会自动将TTITLE设置为ON。

COL n

缩进至当前行的列n （如果已传递列n ，则向后）。这里“列”表示打印位置，而不是表格列。 

S [KIP] [ n ]

跳到新行的开头n次;如果省略n ，一次;如果为n输入零，则返回当前行的开头。 

TAB n

向前跳过n列（如果为n输入负值，则向后）。此上下文中的“列”表示打印位置，而不是表格列。 

LE [FT] | CE [NTER] |对]

左对齐 ，居中和右对齐分别是当前行的数据。SQL * Plus将以下数据项作为一个组对齐，直到printspec的末尾或下一个LEFT，CENTER，RIGHT或COL命令。CENTER和RIGHT使用SET LINESIZE值来计算后面数据项的位置。 

胆大

以粗体打印数据。SQL * Plus通过在三个连续行上重复数据来表示终端上的粗体打印。在某些操作系统上，SQL * Plus可能会指示您的打印机在三个连续行而不是粗体上打印粗体文本。 

格式文字

指定一个格式模型，用于确定以下数据项的格式，直到下一个FORMAT子句或命令的结尾。格式模型必须是文本常量，例如A10或$ 999。有关格式化和有效格式模型的详细信息，请参阅COLUMN命令。

如果格式模型的数据类型与给定数据项的数据类型不匹配，则FORMAT子句对该项无效。

如果在给定数据项之前没有适当的FORMAT模型，则SQL * Plus将使用SET NUMFORMAT指定的格式打印NUMBER值，如果尚未使用SET NUMFORMAT，则使用默认格式。SQL * Plus根据默认格式打印DATE值。

输入没有子句的TTITLE列出当前 TTITLE 定义。

用法

如果在第一次出现文本之前没有输入printspecact子句，则TTITLE左对齐文本。如果有效的printspec子句（LEFT，SKIP，COL等）紧跟在命令名后面，则SQL * Plus会以新的形式解释TTITLE。

有关在顶部标题中打印列和DATE值的信息，请参阅COLUMN 。

您可以在printspec中使用任意数量的常量和变量。SQL * Plus按照您指定的顺序显示它们，定位和格式化每个常量或变量， 由其前面的printspec子句指定。

使用TTITLE指定的标题长度不能超过2400个字符。

在单引号标题文本字符串中不会识别连续字符（连字符）。要识别，延续字符必须出现在引号之外，如下所示：

TTITLE CENTER''>'五月份'总结报告'

例子

要将“每月分析”定义为顶部标题并将其左对齐，将日期居中，以右对齐页码使用三位数格式，并在下一行的中心显示“数千个数据”，输入

TTITLE LEFT'每月分析'中心'2003年1月01日' - 右''页面：'格式999 SQL.PNO跳过中心 - '数千人'
每月分析2003年1月01日页数：1数千个数据

要在不更改其定义的情况下取消顶部标题显示，请输入

TTITLE OFF
