13.17游标FOR LOOP语句

游标FOR LOOP语句隐式声明其循环索引为指定游标返回的行类型的记录变量，然后打开游标。

每次迭代时，游标FOR LOOP语句将结果集中的一行提取到记录中。当没有更多要获取的行时，游标FOR LOOP语句将关闭游标。如果循环内的语句将控制转移到循环外部或引发异常，则游标也会关闭。

语义

cursor_for_loop_statement

记录

循环索引的名称，游标FOR LOOP语句隐式声明为cursor或select_statement返回的类型的%ROWTYPE记录变量。

record是游标FOR LOOP语句的本地。循环内的语句可以引用record及其字段。它们只能通过别名引用虚拟列。循环外的语句不能引用record 。运行游标FOR LOOP语句后， record未定义。

光标

输入光标FOR LOOP时未打开的显式游标（不是游标变量）的名称。

actual_cursor_parameter

与cursor的形式参数对应的实际参数。

select_statement中

SQL SELECT语句（不是PL / SQL SELECT INTO语句）。对于select_statement ，PL / SQL声明，打开，从中提取和关闭隐式游标。但是，因为select_statement不是一个独立的语句，所以隐式游标是内部的 - 你不能用名称SQL引用它。

标签

标识cursor_for_loop_statement标签（请参阅“ statement :: = ”和“ label ” ）。CONTINUE ， EXIT和GOTO语句可以引用此标签。

标签提高了可读性，尤其是在嵌套LOOP语句时，但LOOP是确保END LOOP语句中的标签与同一LOOP语句开头的标签匹配（编译器不检查）。
