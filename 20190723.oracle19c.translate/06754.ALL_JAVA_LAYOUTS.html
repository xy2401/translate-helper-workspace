2.188 ALL_JAVA_LAYOUTS

ALL_JAVA_LAYOUTS显示有关当前用户可访问的存储Java类的类布局信息。

相关观点

  • DBA_JAVA_LAYOUTS显示有关数据库中所有存储的Java类的类布局信息。

  • USER_JAVA_LAYOUTS显示有关当前用户拥有的存储Java类的类布局信息。此视图不显示OWNER列。
