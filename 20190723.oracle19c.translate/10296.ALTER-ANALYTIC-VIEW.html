ALTER ANALYTIC VIEW

目的

使用ALTER ANALYTIC VIEW语句重命名或编译分析视图。对于其他更改，请使用CREATE OR REPLACE ANALYTIC VIEW 。

先决条件

要更改自己架构中的分析视图，您必须具有ALTER ANALYTIC VIEW系统特权。要在另一个用户的模式中更改分析视图，您必须在分析视图上授予ALTER ANY ANALYTIC VIEW系统特权或ALTER ANY TABLE 。

句法

alter_analytic_view :: =


说明alter_analytic_view.eps的说明

语义

模式

指定分析视图所在的模式。如果未指定架构，则Oracle数据库会在您自己的架构中查找分析视图。

analytic_view_name

指定分析视图的名称。

重命名

指定RENAME TO以更改分析视图的名称。对于new_av_name ，请为分析视图指定新名称。

COMPILE

指定COMPILE以编译分析视图。

例

以下语句更改分析视图的名称：

ALTER ANALYTIC VIEW sales_av RENAME TO mysales_av;
