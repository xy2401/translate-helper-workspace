2.233 ALL_OBJECTS

ALL_OBJECTS描述当前用户可访问的所有对象。

相关观点

  • DBA_OBJECTS描述数据库中的所有对象。

  • USER_OBJECTS描述当前用户拥有的所有对象。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

不是NULL

对象的所有者

OBJECT_NAME

VARCHAR2（128）

不是NULL

对象的名称

数

共享

VARCHAR2（13）

价值观：

  • METADATA LINK - 如果对象是元数据链接的，或者是根目录中对象的元数据链接

  • DATA LINK - 如果对象是数据链接的，或者是根目录中对象的数据链接

  • NONE - 如果以上都不适用

EDITIONABLE

VARCHAR2（1）

价值观：

  • Y - 用于标记为版本的对象

  • N - 对于标记为NONEDITIONABLE的对象

  • NULL - 对于类型不可在数据库中编辑的对象

VARCHAR2（1）
