SYS_DBURIGEN

句法


说明sys_dburigen.eps的说明

目的

SYS_DBURIGen将一个或多个列或属性以及可选的rowid作为其参数，并将数据类型为DBURIType的URL DBURIType成为特定的列或行对象。然后，您可以使用URL从数据库中检索XML文档。

引用的所有列或属性必须位于同一个表中。它们必须执行主键的功能。它们实际上不需要匹配表的主键，但它们必须引用唯一值。如果指定多个列，则除最终列之外的所有列都标识数据库中的行，并且指定的最后一列标识行中的列。

默认情况下，URL指向格式化的XML文档。如果您希望URL仅指向文档的文本，请指定可选的“ text() ”。

注意：

在此XML上下文中，小写text是关键字，而不是语法占位符。

如果包含列或属性的表或视图没有在查询上下文中指定的模式，则Oracle数据库会将表或视图名称解释为公共同义词。

也可以看看：

Oracle XML DB Developer's Guide ，提供有关数据库中DBURIType数据类型和XML文档的信息

例子

以下示例使用SYS_DBURIGen函数生成数据类型DBURIType的URL到示例表hr.employees中employee_id = 206的行的email列：

SELECT SYS_DBURIGEN（employee_id，email）FROM employees WHERE employee_id = 206; SYS_DBURIGEN（EMPLOYEE_ID，EMAIL）（URL，SPARE）--------------------------------------- ----------------------------- DBURITYPE（'/ PUBLIC / EMPLOYEES / ROW [EMPLOYEE_ID ='''206''] / EMAIL' ， 空值）
