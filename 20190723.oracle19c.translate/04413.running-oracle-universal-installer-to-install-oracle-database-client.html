运行安装向导以安装Oracle数据库客户端

使用runInstaller命令启动Oracle Database Client安装。

在开始安装之前，请准备好有关用户组和存储路径的所有信息。

Oracle建议您在安装期间使用My Oracle Support凭据。
在安装过程中，系统会要求您以root用户身份运行配置脚本。您可以在出现提示时以root身份手动运行这些脚本，也可以使用root权限委派选项（如sudo提供配置信息和密码。
  1. 以Oracle安装所有者用户帐户（ oracle ）身份登录。
  2. 在安装介质上或已下载安装二进制文件的位置，运行runInstaller命令以启动Oracle安装向导。

    例如：

    • 在安装媒体上：

      的/ dev / DVD-RW /媒体/ runInstaller的
    • 在硬盘上：
      $ cd / home / oracle_sw / $ ./runInstaller

    注意：

    仅从Oracle主目录运行runInstaller命令。请勿使用驻留在$ORACLE_HOME/oui/bin/或任何其他位置的runInstaller命令来安装Oracle数据库，Oracle数据库客户端或Oracle Grid Infrastructure。
  3. 选择您的安装类型。

    安装屏幕因您选择的安装选项而异。根据需要响应配置提示。

    注意：

    在安装过程中的任何时候，如果您对要求执行的操作有疑问，请单击“ 帮助” 。
