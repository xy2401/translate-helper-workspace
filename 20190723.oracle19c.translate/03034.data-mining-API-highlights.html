27.1 Data Mining API的重点

了解Data Mining应用程序编程接口（API）的优势。

数据挖掘是许多应用领域中的一项有价值的技术。作为优化运营和保持竞争优势的工具，私营部门越来越不可或缺。数据挖掘在公共部门和科学研究中也具有重要应用。但是，数据挖掘应用程序开发的复杂性以及管理和保护大型数据存储所固有的复杂性可能会限制数据挖掘技术的采用。

Oracle Data Mining非常适合应对这些挑战。数据挖掘引擎在数据库内核中实现，Oracle数据库的强大管理功能可用于管理和保护数据。在支持全系列数据挖掘算法和过程的同时，API还具有简化数据挖掘应用程序开发的功能。

Oracle Data Mining API包含对Oracle SQL的扩展，Oracle SQL是数据库的本机语言。API具有以下优点：

  • 在SQL查询的上下文中进行评分。评分可以动态执行或通过应用数据挖掘模型执行。

  • 自动数据准备（ADP）和嵌入式转换。

  • 模型透明度。特定于算法的查询返回有关用于创建模型的属性的详细信息。

  • 得分透明度。可以使用分数返回关于预测，聚类或特征提取操作的细节。

  • 简单的例行程序预测分析。

    Oracle Data Miner
