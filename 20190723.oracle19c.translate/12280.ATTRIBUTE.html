12.8 ATTRIBUTE

句法

ATTR [IBUTE] [ type_name.attribute_name [ option ...]]

其中option表示以下子句之一：

ALI [AS] 别名 CLE [AR] FOR [MAT] 格式 LIKE { type_name.attribute_name | 别名 } ON |关闭

指定显示“对象类型”列的给定属性的特征，例如NUMBER数据的格式。列和属性不应具有相同的名称，因为它们共享公共命名空间。

还列出了单个属性或所有属性的当前显示特性。

输入ATTRIBUTE 后跟type_name.attribute_name ，没有其他子句列出仅指定属性的当前显示特性。输入ATTRIBUTE 没有子句列出所有当前属性显示特征。

条款

type_name.attribute_name

标识对象类型type_name的给定对象的属性集内的数据项（通常是属性的名称）。

如果选择相同对象类型的对象，则该type_name.attribute_name的ATTRIBUTE命令将应用于您在该会话中引用的所有此类对象。 

ALI [AS] 别名

为type_name.attribute_name分配指定的别名，该别名可用于引用其他ATTRIBUTE命令中的type_name.attribute_name 。 

明确]

重置attribute_name的显示特征。格式规范必须是文本常量，例如A10或$ 9,999，而不是变量。 

FOR [MAT] 格式

指定列的显示格式。格式规范必须是文本常量，例如A10或$ 9,999，而不是变量。 

 LIKE { type_name.attribute_name | 别名 }

副本另一个属性的显示特征。LIKE仅复制当前ATTRIBUTE命令中未由另一个子句定义的特征。 

ON |关闭

控制列的显示特征的状态。关闭禁用属性的特征而不影响特征的定义。上恢复特征。

用法

您可以为一个或多个属性输入任意数量的ATTRIBUTE命令。为每个属性设置的所有属性特征在会话的其余部分保持有效，直到您关闭该属性，或直到您使用该属性 CLEAR COLUMN命令。因此，ATTRIBUTE 您输入的命令可以控制多个SQL SELECT命令的属性的显示特性。

当您输入多个ATTRIBUTE 对于相同属性的命令，SQL * Plus统一应用它们的子句。如果多个ATTRIBUTE命令将同一子句应用于同一属性，则输入的最后一个将控制输出。

例子

要使对象类型EMPLOYEE_TYPE的LAST_NAME属性为20个字符宽，请输入

ATTRIBUTE EMPLOYEE_TYPE.LAST_NAME格式A20

要格式化对象类型EMPLOYEE_TYPE的SALARY属性，使其显示数百万美元，舍入为美分，使用逗号分隔数千，并在值为零时显示$ 0.00，输入

ATTRIBUTE EMPLOYEE_TYPE.SALARY FORMAT $ 9,999,990.99
