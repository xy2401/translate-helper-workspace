说明adxdb006.eps的说明

该图显示了三种XML体系结构：非本机XML处理，独立数据和内容服务器以及Oracle XML DB。

  • 非原生XML处理 - 该图具有标记为“RDBMS（结构化数据和元数据）”的圆柱体和标记为“多媒体和文档内容”的文件系统图标。这些对象中的每一个都连接到标有“XML Processing and Repository Layer”的框。此框还连接到标有“Application Server”的框。Application Server和RDBMS均连接到标有“Applications”的框。

  • 独立的数据和内容服务器 - 该图有两个圆柱体，标记为“RDBMS（结构化数据）”和“XML存储库（多媒体，文档内容和XML，元数据）”。每个气缸都连接到标有“Application Server”的盒子。Application Server和RDBMS均连接到标有“Applications”的框。

  • Oracle XML DB - 该图有一个标记为“Oracle XML DB（多媒体和文档内容，结构化数据，XML，元数据）”的圆柱体。圆柱体连接到标有“Oracle Application Server”的盒子和标有“Applications”的盒子。这两个盒子也相互连接。
