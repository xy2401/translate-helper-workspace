14.16 DROP LIBRARY声明

DROP LIBRARY语句从数据库中删除外部过程库。

话题

  • 先决条件

  • 句法

  • 语义

  • 例

  • 相关话题

先决条件

您必须具有DROP ANY LIBRARY系统特权。

句法

drop_library :: =


说明drop_library.eps的描述

语义

LIBRARY_NAME

要删除的外部过程库的名称。

例

示例14-33删除库

以下语句删除了在“ CREATE LIBRARY Statement ”中创建的ext_lib库：

DROP LIBRARY ext_lib;
