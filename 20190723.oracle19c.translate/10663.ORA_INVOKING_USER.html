ORA_INVOKING_USER

句法


插图说明ora_invoking_user.eps

目的

ORA_INVOKING_USER返回调用当前语句或视图的数据库用户的名称。此函数考虑了语句中引用的中间视图的BEQUEATH属性。如果从定义者的权限上下文中调用此函数，则它返回定义者权限对象的所有者的名称。如果调用用户是Real Application Security用户，则它返回用户XS$NULL 。

此函数返回VARCHAR2值。
