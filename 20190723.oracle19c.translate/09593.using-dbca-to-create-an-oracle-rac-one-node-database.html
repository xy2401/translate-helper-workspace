使用DBCA创建Oracle RAC One节点数据库

如果已选择仅在群集节点上安装Oracle RAC软件，则可以使用Oracle数据库配置助手（DBCA）配置Oracle RAC One节点。

安装Oracle Real Application Clusters（Oracle RAC）软件后，启动Oracle Database Configuration Assistant（DBCA）。
  1. 从“数据库操作”页面中，选择“ 创建数据库 ”选项。
  2. 在“创建模式”页面上，选择“ 高级配置” 。
  3. 在“部署类型”页面上，选择“ Oracle RAC One节点数据库” 。

    选择一个节点会在单个节点上部署Oracle RAC One Node。Oracle建议您选择群集中希望Oracle RAC One Node能够进行故障转移的所有节点。

    创建管理员管理的Oracle RAC One Node数据库时，请注意，虽然数据库仅在您安装二进制文件的一个节点池上启动，但所有候选服务器都放在通用服务器池中。如果服务器尚未处于Generic或Free，则可能导致停止在候选服务器上运行的资源。

使用DBCA创建Oracle RAC One Node数据库时，会自动配置故障转移服务。
