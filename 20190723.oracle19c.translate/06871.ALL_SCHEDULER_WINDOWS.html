3.67 ALL_SCHEDULER_WINDOWS

ALL_SCHEDULER_WINDOWS显示有关当前用户可访问的调度程序窗口的信息。

相关视图

DBA_SCHEDULER_WINDOWS显示有关数据库中所有Scheduler窗口的信息。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

“调度程序”窗口的所有者

WINDOW_NAME

VARCHAR2（128）

不是NULL

“调度程序”窗口的名称

RESOURCE_PLAN

VARCHAR2（128）

与窗口关联的资源计划

SCHEDULE_OWNER

VARCHAR2（4000）

窗口日程表的所有者

SCHEDULE_NAME

VARCHAR2（4000）

窗口日程表的名称

时间安排：SCHEDULE_TYPE

VARCHAR2（8）

窗口时间表的类型：

  • ONCE - 重复间隔为NULL

  • NAMED - 命名时间表

  • CALENDAR - 用作计划的Oracle日历表达式

开始日期

TIMESTAMP（6）带时区

窗口的开始日期（对于内联计划）

REPEAT_INTERVAL

VARCHAR2（4000）

窗口的日历字符串（用于内联计划）

结束日期

TIMESTAMP（6）带时区

窗口将不再打开的日期（对于内联计划）

期限

间隔日（3）至第二（0）

窗口的持续时间

WINDOW_PRIORITY

VARCHAR2（4）

相对于其他窗口的作业优先级：

  • 高

  • 低

NEXT_START_DATE

TIMESTAMP（6）带时区

计划启动窗口的下一个日期

LAST_START_DATE

TIMESTAMP（6）带时区

窗口打开的最后日期

启用

VARCHAR2（5）

指示窗口是启用（ TRUE ）还是禁用（ FALSE ）

活性

VARCHAR2（5）

指示窗口是否打开（ TRUE ）（ FALSE ）

MANUAL_OPEN_TIME

TIMESTAMP（6）带时区

如果手动打开，则打开窗口的时间，否则为NULL

MANUAL_DURATION

间隔日（3）至第二（0）

手动打开窗口的持续时间，否则为NULL
