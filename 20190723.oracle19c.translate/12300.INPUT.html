12.28输入

句法

我[NPUT] [ 文字 ]

其中text表示您要添加的文本。

添加一个或多个新线缓冲区中当前行之后的文本。

要添加单行，请在命令INPUT后输入行的文本，使用空格将文本与命令分开。要使用一个或多个空格开始该行，请在INPUT和文本的第一个非空白字符之间输入两个或多个空格。

要添加多行，请输入没有文本的INPUT。INPUT 提示您输入每一行。要离开INPUT，请输入空（空）行或句点。

用法

如果在命令提示符处输入的行号大于缓冲区中的行数，并使用带有文本的数字，则SQL * Plus会在新行中添加文本在缓冲区的末尾。如果为行号指定零（0）并使用文本跟随零，则SQL * Plus将插入该行在缓冲区的开头（该行变为第1行）。

例子

假设SQL缓冲区包含以下命令：

从EMP_DETAILS_VIEW中选择LAST_NAME，DEPARTMENT_ID，SALARY，COMMISSION_PCT

要向查询添加ORDER BY子句，请输入

列表2
2 *来自EMP_DETAILS_VIEW
LAST_NAME输入订单

LIST 2确保第2行是当前行。INPUT在当前行之后添加一个包含ORDER BY子句的新行。SQL缓冲区现在包含以下行：

1选择LAST_NAME，DEPARTMENT_ID，SALARY，COMMISSION_PCT 2来自EMP_DETAILS_VIEW 3 *由LAST_NAME订购

要添加两行WHERE子句，请输入

列表2
2 *来自EMP_DETAILS_VIEW
INPUT 3 WHERE JOB_ID ='SA_MAN'4 AND COMMISSION_PCT = .25 5

在您输入空行或句点之前，INPUT会提示您输入新行。SQL缓冲区现在包含以下行：

SELECT LAST_NAME，DEPARTMENT_ID，SALARY，COMMISSION_PCT FROM EMP_DETAILS_VIEW WHERE JOB_ID ='SA_MAN'AND COMMISSION_PCT = .25 ORDER BY LAST_NAME
