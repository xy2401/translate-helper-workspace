5.329 DBA_SCHEDULER_JOB_ROLES

DBA_SCHEDULER_JOB_ROLES按数据库角色显示有关数据库中所有Scheduler作业的信息。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

调度程序作业的所有者

JOB_NAME

VARCHAR2（128）

不是NULL

调度程序作业的名称

JOB_SUBNAME

VARCHAR2（128）

调度程序作业的子名称（用于运行链式步骤的作业）

JOB_CREATOR

VARCHAR2（128）

调度程序作业的创建者

DATABASE_ROLE

VARCHAR2（16）

数据库角色的名称

PROGRAM_OWNER

VARCHAR2（4000）

与作业关联的程序的所有者

PROGRAM_NAME

VARCHAR2（4000）

与作业关联的程序的名称

工作类型

VARCHAR2（16）

内联作业操作类型：

  • PLSQL_BLOCK

  • STORED_PROCEDURE

  • EXECUTABLE

  • 链

JOB_ACTION

VARCHAR2（4000）

内联工作行动

JOB_CLASS

VARCHAR2（128）

与作业关联的作业类的名称

SCHEDULE_OWNER

VARCHAR2（4000）

作业使用的计划的所有者（可以是窗口或窗口组）

SCHEDULE_NAME

VARCHAR2（4000）

作业使用的计划的名称（可以是窗口或窗口组）

时间安排：SCHEDULE_TYPE

VARCHAR2（12）

作业使用的计划类型：

  • IMMEDIATE - 开始日期和重复间隔为NULL

  • ONCE - 重复间隔为NULL

  • PLSQL - 用作计划的PL / SQL表达式

  • CALENDAR - 用作计划的Oracle日历表达式

  • EVENT -事件时间表

  • NAMED - 命名时间表

  • WINDOW - 用作计划的窗口

  • WINDOW_GROUP - 用作计划的窗口组

开始日期

TIMESTAMP（6）带时区

作业的原始预定开始日期（对于内联计划）

REPEAT_INTERVAL

VARCHAR2（4000）

内联计划PL / SQL表达式或日历字符串

结束日期

TIMESTAMP（6）带时区

作业将不再运行的日期（对于内联计划）

LAST_START_DATE

TIMESTAMP（6）带时区

作业运行的最后日期

启用

VARCHAR2（5）

指示作业是启用（ TRUE ）还是禁用（ FALSE ）

州

VARCHAR2（15）

目前的工作状态：

  • 禁用

  • 重新计划

  • 排定

  • RUNNING

  • 已完成

  • 破碎

  • 失败

  • 远程

  • SUCCEEDED

  • CHAIN_STALLED

评论

VARCHAR2（240）

对工作的评论
