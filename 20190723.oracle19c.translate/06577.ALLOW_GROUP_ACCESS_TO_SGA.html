1.11 ALLOW_GROUP_ACCESS_TO_SGA

ALLOW_GROUP_ACCESS_TO_SGA控制对UNIX平台上的共享内存的group访问。

属性 描述

参数类型

布尔

默认值

假

可修改

没有

可在PDB中修改

没有

价值范围

是的|假

基本

没有

Oracle RAC

必须在所有实例上使用相同的值。

默认值为false ，这意味着仅使用owner访问权限创建数据库共享内存。在Oracle Database 12c第2版（12.2.0.1）之前的Oracle数据库版本中，创建了具有owner和group访问权限的数据库共享内存。

当此参数设置为true ，将创建具有owner和group访问权限的数据库共享内存。此行为授予DBA管理数据库外部共享内存的权限，但也允许DBA读取和写入共享内存，这可能不适合某些安装。
