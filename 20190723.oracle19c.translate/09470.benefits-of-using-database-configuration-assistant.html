关于Database Configuration Assistant

了解Oracle数据库配置助手（DBCA）为创建Oracle RAC数据库提供的功能。

DBCA使您可以创建策略管理和管理员管理的数据库。使用DBCA，您还可以创建特定于站点的表空间作为数据库创建的一部分。如果您的数据文件要求与DBCA模板提供的要求不同，则使用DBCA创建数据库并稍后修改数据文件。您还可以在数据库创建过程中运行用户指定的脚本。

DBCA还为各种Oracle高可用性功能（如群集管理工具）配置Oracle RAC环境。DBCA还会启动支持您定义的配置所需的任何数据库实例。

您可以使用DBCA从Oracle提供的模板或您创建的模板创建数据库。模板包含针对特定类型的工作负载优化的设置。

Oracle提供以下两种工作负载类型的模板：

  • 通用或交易处理

  • 数据仓库

对于更复杂的环境，您可以选择“ 自定义数据库”选项。此选项不使用模板并导致更广泛的访谈，这意味着创建数据库需要更长的时间。

单击“ 显示详细信息”以查看每种类型数据库的配置。选择适合您的数据库支持的工作负载类型的模板。如果您不确定选择哪个，请选择默认的通用或事务处理模板。
