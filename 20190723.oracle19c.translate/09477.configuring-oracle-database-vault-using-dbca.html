使用DBCA配置Oracle Database Vault

您可以使用Oracle Database Configuration Assistant（DBCA）在安装后配置Oracle Database Vault，也可以选择不配置Oracle Database Vault。

使用DBCA安装Oracle Database Vault：

  1. 安装Oracle RAC后，创建数据库。
  2. 启动DBCA并选择配置数据库选项。
  3. 在组件列表中，选择Oracle Label Security和Oracle Database Vault 。
  4. 提供所需的Oracle Database Vault用户标识和密码以继续进行配置。要启用单独的Oracle Database Vault管理员，请选择配置DV_ACCTMGR用户。
  5. 完成后，重新启动每个数据库实例以完成软件配置。
