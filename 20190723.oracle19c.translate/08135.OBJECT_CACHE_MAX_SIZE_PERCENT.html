1.216 OBJECT_CACHE_MAX_SIZE_PERCENT

对象缓存是客户端上的一个内存块，它允许应用程序存储整个对象并在它们之间导航，而无需往返服务器。OBJECT_CACHE_MAX_SIZE_PERCENT指定会话对象高速缓存可以超过最佳大小的最佳高速缓存大小的百分比。
属性 描述

参数类型

整数

默认值

10

可修改

ALTER SESSION ， ALTER SYSTEM ...DEFERRED

可在PDB中修改

是

价值范围

0到操作系统相关的最大值

基本

没有

最大尺寸等于最佳尺寸加上该百分比和最佳尺寸的乘积。当缓存大小超过此最大大小时，系统将尝试将缓存缩小到最佳大小。
