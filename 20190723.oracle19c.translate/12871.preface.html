前言

本前言讨论了Oracle Database 2 Day + Java Developer's Guide的目标读者和惯例。它还包括一系列相关的Oracle文档，您可以参考这些文档以获取更多信息。

听众

本指南适用于使用Java访问和修改Oracle数据库中的数据的应用程序开发人员。本指南说明了如何使用简单的Java数据库连接（JDBC）应用程序执行这些任务。本指南使用Oracle JDeveloper集成开发环境（IDE）来创建应用程序。任何对Java编程感兴趣的人都可以阅读本指南，但它至少假设了以下一些先验知识：

  • Java的

  • Oracle PL / SQL

  • Oracle数据库
