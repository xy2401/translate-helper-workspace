1.243 PARALLEL_ADAPTIVE_MULTI_USER

PARALLEL_ADAPTIVE_MULTI_USER设置为true ，启用自适应算法，旨在提高使用并行执行的多用户环境中的性能。

属性 描述

参数类型

布尔

默认值

假

可修改

改变系统

可在PDB中修改

没有

价值范围

是的|假

基本

没有

该算法根据查询启动时的系统负载自动降低所请求的并行度。有效并行度基于默认的并行度，或者表或提示的度数除以减少因子。

该算法假定系统已针对单用户环境中的最佳性能进行了调整。

表和提示使用默认的并行度。

注意：

PARALLEL_ADAPTIVE_MULTI_USER初始化参数在Oracle Database 12 c第2版（12.2.0.1）中已弃用，可能会在将来的发行版中删除。Oracle建议您使用并行语句排队功能。
