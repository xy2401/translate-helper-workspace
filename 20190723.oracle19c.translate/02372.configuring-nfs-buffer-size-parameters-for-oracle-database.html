配置Oracle数据库的NFS缓冲区大小参数

将NFS缓冲区大小参数rsize和wsize的值设置为至少32768。

例如，要为Oracle数据库数据文件装入点使用值为32768的rsize和wsize缓冲区设置，请将装入点参数设置为类似于以下值的值：
nfs_server：/ vol / DATA / oradata / home / oracle / netapp nfs \ rw，bg，hard，nointr，rsize = 32768，wsize = 32768，tcp，actimeo = 0，vers = 3，timeo = 600
直接NFS客户端以wtmax粒度向NFS服务器发出写入问题。
