ASMCMD-08001：磁盘组“ string ”不存在，或者未安装

原因：操作失败，因为指定的磁盘组不存在或当前的Oracle自动存储管理（Oracle ASM）实例未装入。

操作：验证指定的磁盘组名称是否有效，磁盘组是否存在，以及磁盘组是否由当前Oracle ASM实例装入。

ASMCMD-08002：进入“ string ”在目录中不“存在string ”

原因：指定的路径包含不存在的元素。

操作：指定现有的Oracle ASM文件路径。

ASMCMD-08003：当前实例类型不允许的命令

原因： ASMCMD已连接到具有Oracle自动存储管理（Oracle ASM）以外的实例类型的实例。

操作：确保连接到INSTANCE_TYPE参数为Oracle ASM的实例。

ASMCMD-08004：Oracle ASM文件路径' string '包含无效的别名

原因：指定的路径包含无效的别名。

操作：指定有效路径。

ASMCMD-08005：目录' string '不明确

ASMCMD-09531：外部冗余磁盘组不能包含站点。

原因：标记site与外部冗余磁盘组不兼容。

操作：将命令应用于外部磁盘组时，请勿使用site标记。

ASMCMD-09532：当Oracle集群堆栈向下命令不能使用

原因： Oracle Clusterware堆栈关闭时执行了该命令。

操作：使用'crsctl start crs'命令启动Oracle Clusterware并重试。

ASMCMD-09533：资源ora.driver.afd不存在

原因：当OHASD资源ora.driver.afd不存在时执行该命令。

操作：使用“ASMCMD afd_state”命令验证AFD的状态。

ASMCMD-09534：未提供发现字符串。

原因：未提供发现字符串，并且不允许执行戳记操作。

操作：提供一个发现字符串，将“--dscstr”选项添加到命令“asmcmd stamp”。

ASMCMD-09535：未提供故障组标签。

原因：尝试在不提供故障组标签的情况下标记站点标签。

操作：提供故障组标签，将“--failgroup”选项添加到命令“asmcmd stamp”。

ASMCMD-09536：网站标签“ string ”无效。

原因：提供了无效的站点标签。

操作：指定一个站点名称，其长度至少为一个字符但不超过15个字符。站点名称由字母数字字符和连字符（a..z，A ..Z，0..9， - ）。它不能以数字字符开头。它不能以连字符（ - ）开头或结尾。

ASMCMD-09537：失败组标签' string '无效。

原因：提供了无效的故障组标签。

操作：指定作为有效SQL标识符的故障组标签。它必须至少有一个字符长度，但长度不得超过30个字符。它必须以字母开头，由字母，数字和下划线（_）字符组成。

ASMCMD-09538：磁盘标签' string '无效。

原因：提供了无效的磁盘标签。

操作：指定作为有效SQL标识符的磁盘标签。它必须至少有一个字符长度，但长度不得超过23个字符。它必须以字母开头，由字母，数字和下划线（_）字符组成。

ASMCMD-09539：未使用字符串' string '发现磁盘。

原因：提供的发现字符串与任何磁盘都不匹配。

操作：提供有效的发现字符串。

ASMCMD-09540：磁盘由AFD或ASMLIB管理。

原因：尝试在由Oracle ASM筛选器驱动程序（AFD）或ASMLIB管理的磁盘上标记磁盘标签。

操作：为不由AFD或ASMLIB管理的磁盘提供发现字符串，或者不提供磁盘标签，然后重试该命令。

ASMCMD-09541：磁盘string属于已装入的磁盘组string 。

原因：尝试标记属于已装入磁盘组的磁盘。

操作：卸载磁盘组并重试该命令。

ASMCMD-09542：磁盘string是磁盘组string成员，ASMCMD未连接到实例。

原因：尝试标记作为ASM磁盘组成员的磁盘，并且ASMCMD未连接到ASM实例以验证是否已装入该磁盘组。

操作：将ASMCMD连接到ASM实例并重试该命令。

ASMCMD-09543：无法重命名磁盘的站点和故障组

原因：尝试重命名磁盘的站点和故障组。不允许该操作，因为这些选项是互斥的，因为只有在磁盘组冗余是外部的情况下才能重命名故障组，并且如果磁盘组冗余是外部的，则无法重命名站点。

操作：仅提供以下两个选项之一：“site”或“failgroup”。

ASMCMD-09544：成员群集不支持该命令。

原因：尝试运行成员群集不支持的命令。

操作：在存储服务器上重新运行该命令。

ASMCMD-09545：已安装的AFD驱动程序对于此操作系统不正确。

原因：发出了检索Oracle ASM筛选器驱动程序（AFD）状态的请求。安装的AFD驱动程序和当前运行的操作系统不兼容。

操作：联系Oracle支持服务。

ASMCMD-09546：执行命令的权限不足。该命令需要Oracle Grid Infrastructure用户。

原因：尝试由不是Oracle Grid Infrastructure用户的用户执行命令。

操作：以Oracle Grid Infrastructure用户身份重新运行该命令。

ASMCMD-09601：成员群集创建失败，因为指定的访问模式无效。

原因：尝试创建Oracle数据库机的成员集群时，提供了无效的访问模式。

操作：通过指定正确的访问模式重新运行命令。
