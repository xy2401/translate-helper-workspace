13.5 BatchSQLException类

BatchSQLException类提供了处理批处理错误的方法。由于BatchSQLException类派生自SQLException类 ， BatchSQLException除了表13-6中汇总的方法外，所有BatchSQLException实例都支持SQLException所有方法。

也可以看看：

“ 优化C ++应用程序性能 ”中的“ 迭代修改行 ”一节。

表13-6 BatchSQLException方法摘要

方法 摘要

getException（）

返回异常。

getFailedRowCount（）

返回插入或更新失败的行数。

getRowNum（）

返回具有插入或更新错误的行的编号

13.5.1 getException（）

返回与指定index匹配的异常。

句法

SQLException getSQLException（unsigned int index）const;
参数 描述 
指数

批处理进程返回的错误列表的索引。

13.5.2 getFailedRowCount（）

返回语句插入或更新失败的行数。

句法

unsigned int getFailedRowCount（）const;

13.5.3 getRowNum（）

返回与指定index匹配的错误行的编号。

句法

unsigned int getRowNum（unsigned int index）const;
参数 描述 
指数

批处理进程返回的错误列表的索引。
