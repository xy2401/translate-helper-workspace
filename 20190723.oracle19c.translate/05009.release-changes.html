此版本的Pro * C / C ++程序员指南中的更改

本前言列出了Pro * C / C ++程序员指南中的更改。

Pro * C / C ++版本18c，版本18.1中的更改

新功能

以下功能是此版本中的新增功能：

  • 在流量控制器模式下支持Oracle Connection Manager

    Traffic Director模式中的Oracle Connection Manager是一个代理，位于支持的数据库客户端和数据库实例之间，以提高高可用性，连接多路复用和负载平衡。

    有关详细信息，请参阅关于在Traffic Director模式下使用Oracle Connection Manager 。

Pro * C / C ++ 12c第2版（12.2）的变化

新功能

以下功能是此版本中的新增功能：

  • Pro * C / C ++现在支持128字节的标识符长度。在以前的版本中，标识符长度限制为30个字节。

  • Pro * C / C ++现在支持Oracle Instant Client - Basic Light版本。

  • Pro * C / C ++现在引入了一个新的命令行选项“trim_password”，以防止由包含尾随空格的密码字符串引起的身份验证问题。

  • 诸如Transaction Guard之类的高可用性功能在OCI层中实现。有关更多信息，请参阅“调用接口程序员指南”。
