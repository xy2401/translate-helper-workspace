11.3在Oracle Real Application Clusters（Oracle RAC）环境中启用Memory Guard

为Oracle Real Application Clusters（Oracle RAC）或Oracle RAC One Node数据库安装Oracle Grid Infrastructure时，会自动启用Memory Guard。

运行srvctl命令查询Memory Guard的状态，如下所示：
srvctl status qosmserver

示例11-1验证Memory Guard是否在节点上运行

以下示例显示qosmserver上Memory Guard状态的示例输出。 

$ srvctl status qosmserver启用QoS Management Server。QoS Management Server正在节点nodeABC上运行
