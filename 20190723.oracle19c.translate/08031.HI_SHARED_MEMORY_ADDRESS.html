1.134 HI_SHARED_MEMORY_ADDRESS

HI_SHARED_MEMORY_ADDRESS指定系统全局区域（SGA）运行时的起始地址。

属性 描述

参数类型

整数

默认值

0

可修改

没有

可在PDB中修改

没有

基本

没有

在链接时指定SGA起始地址的平台上将忽略此参数。

在64位平台上，使用HI_SHARED_MEMORY_ADDRESS指定64位地址的高32位。使用SHARED_MEMORY_ADDRESS指定地址的低32位（请参阅“ SHARED_MEMORY_ADDRESS ” ）。如果两个参数均为0或未指定，则SGA地址默认为特定于平台的位置。
