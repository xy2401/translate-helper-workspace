1.302 SERIAL_REUSE

SERIAL_REUSE指定哪些类型的游标使用串行可重用内存功能。 此功能在SGA中分配专用游标内存，以便执行相同游标的会话可以重复使用（串行，不同时）。
属性 描述

参数类型

串

句法

SERIAL_REUSE = {disable |所有|选择| dml | plsql |力量

默认值

禁用

可修改

没有

可在PDB中修改

没有

基本

没有

注意：

不推荐使用SERIAL_REUSE参数。它仅为了向后兼容而保留。

值

  • 禁用

    禁用所有SQL语句类型的选项。此值将覆盖列表中包含的任何其他值。

  • 所有

    为DML和SELECT语句启用选项。相当于设置select ， dml和plsql 。

  • 选择

    启用SELECT语句的选项。

  • DML

    启用DML语句选项。

  • PLSQL

    目前没有任何效果（尽管PL / SQL包支持使用PL / SQL pragma的串行重用内存选项）。

注意：

如果CURSOR_SPACE_FOR_TIME设置为true ，那么的值SERIAL_REUSE被忽略，就好像它是设置为处理disable 。
