用户

句法


说明user.eps的说明

目的

USER返回会话用户的名称（登录的用户）。在附加或分离Real Application Security会话期间，这可能会在数据库会话期间发生更改。对于企业用户，此函数返回架构。对于其他用户，它返回数据库用户名。如果Real Application Security会话当前附加到数据库会话，则它返回用户XS$NULL 。

此函数返回VARCHAR2值。

Oracle数据库将此函数的值与空白填充的比较语义进行比较。

在分布式SQL语句中， UID和USER函数一起标识本地数据库上的用户。您不能在CHECK约束条件下使用这些函数。

例子

以下示例返回会话用户和用户的UID：

SELECT USER，UID FROM DUAL;
