在Oracle Solaris上验证操作系统版本

要检查软件以确定它们是否满足安装的最低版本要求，请执行以下步骤：

  1. 要确定安装了哪个版本的Oracle Solaris：
    $ uname -r
    5.11

    在此示例中，显示的版本是Oracle Solaris 11（5.11）。如有必要，请参阅操作系统文档以获取有关升级操作系统的信息。

  2. 要确定发布级别：
    $ cat / etc / release
    Oracle Solaris 11.4 SPARC

    在此示例中，显示的发行版级别为Oracle Solaris 11.4 SPARC。

  3. 要确定有关操作系统版本的详细信息，例如更新级别，SRU和构建：
    1. 在Oracle Solaris 11上
      $ pkg列出整个
      姓名（出版商）版本IFO整个（solaris）0.5.11-0.175.3.1.0.5.0 i--
