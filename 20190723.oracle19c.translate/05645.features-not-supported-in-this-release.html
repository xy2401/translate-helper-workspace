A.1不支持的功能概述

本节介绍Oracle数据库不再支持的功能和配置文件。

  • Oracle Net Connection Pooling
    在Oracle Database 12 c第2版（12.2）中，不再支持Oracle Net连接池
  • Oracle名称
    此版本不支持Oracle Names。
  • Oracle Net Listener密码
    在Oracle Database 12 c第2版（12.2）中，不再支持Oracle Net Listener密码功能。

父主题： 此版本不支持的功能

A.1.1 Oracle网络连接池

在Oracle Database 12 c第2版（12.2）中，不再支持Oracle Net连接池

Oracle Net Connection Pooling的弃用

它在Oracle Database 11g发行版中已被弃用。请参阅My Oracle Support说明1469466.1。

父主题： 不支持的功能概述

A.1.2 Oracle名称

此版本不支持Oracle Names。

命名方法

自Oracle Database 11g以来，Oracle Names一直不被支持作为命名方法。您必须迁移到目录命名。

父主题： 不支持的功能概述

A.1.3 Oracle Net Listener密码

在Oracle Database 12 c第2版（12.2）中，不再支持Oracle Net Listener密码功能。

Oracle Net Listener密码支持

在Oracle Database 12 c第2版（12.2）中，不再支持Oracle Net Listener密码功能。这不会导致安全性丢失，因为通过本地操作系统身份验证强制执行身份验证。
