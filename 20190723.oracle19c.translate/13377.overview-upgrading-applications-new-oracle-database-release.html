在新的Oracle数据库发行版上升级应用程序的概述

您无需修改不使用新Oracle数据库版本中可用功能的现有应用程序。

在新版本的Oracle数据库中运行的现有应用程序的功能与早期版本中的相同，并实现相同或增强的性能。

升级到新的Oracle数据库版本后，可以使用许多新功能和增强功能。其中一些功能提供了附加功能和功能，而其他功能则提供了改进的性能。在升级应用程序之前，您应该查看这些新功能以确定要使用哪些功能。
