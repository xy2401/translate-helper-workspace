1.186 MAX_DATAPUMP_PARALLEL_PER_JOB

MAX_DATAPUMP_PARALLEL_PER_JOB指定每个Oracle Data Pump作业允许的最大并行进程数。

属性 描述

参数类型

串

句法

MAX_DATAPUMP_PARALLEL_PER_JOB = { integer |自动}

默认值

50

价值范围

1到1024 ，或AUTO

可修改

改变系统

可在PDB中修改

是

基本

没有

Oracle RAC

不同的实例可以使用不同的值。

当此参数的值为AUTO ，Oracle Data Pump将其值派生为SESSIONS初始化参数值的50％。

注意：

从Oracle Database 19c版本19.1开始提供此参数。
