1.152 INSTANCE_NUMBER

INSTANCE_NUMBER指定一个唯一编号，该编号将实例映射到使用存储参数FREELIST GROUPS创建的每个数据库对象的一个空闲列表组。

属性 描述

参数类型

整数

默认值

0 （零）

可修改

没有

可在PDB中修改

没有

价值范围

1到创建数据库时指定的最大实例数

基本

是

Oracle RAC

您必须为每个实例设置此参数，并且所有实例必须具有不同的值。

INSTANCE_NUMBER是Oracle RAC参数，可以在并行模式或独占模式下指定。

ALTER TABLE ...的INSTANCE参数ALTER TABLE ...ALLOCATE EXTENT语句为特定的空闲列表组分配范围。如果将INSTANCE_NUMBER设置为为INSTANCE参数指定的值，则实例将该范围用于插入和扩展行的更新。

此参数的实际最大值是CREATE DATABASE语句中指定的最大实例数。绝对最大值取决于操作系统。
