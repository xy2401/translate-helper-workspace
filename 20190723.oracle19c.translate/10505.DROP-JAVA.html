DROP JAVA

目的

使用DROP JAVA语句删除Java源，类或资源架构对象。

也可以看看：

  • 创建Java有关创建Java对象

  • 有关解析Java源代码，类和资源的更多信息，请参见“Oracle数据库Java开发人员指南”

先决条件

Java源，类或资源必须位于您自己的架构中，或者您必须具有DROP ANY PROCEDURE系统特权。您还必须具有Java类的EXECUTE对象特权才能使用此命令。

句法

drop_java :: =


说明drop_java.eps的描述

语义

JAVA来源

指定SOURCE以删除Java源模式对象以及从中派生的所有Java类模式对象。

JAVA CLASS

指定CLASS以删除Java类架构对象。

JAVA资源

指定RESOURCE以删除Java资源架构对象。

OBJECT_NAME

指定现有Java类，源或资源架构对象的名称。将object_name括在双引号中以保留较低或大小写混合的名称。
