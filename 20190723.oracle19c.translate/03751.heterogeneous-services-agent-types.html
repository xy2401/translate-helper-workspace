异构服务代理的类型

有两种类型的异构服务代理。

  • Oracle数据库网关
    Oracle数据库网关是专为访问特定的非Oracle系统而设计的网关。
  • 用于ODBC代理的Oracle数据库网关
    除了适用于各种非Oracle数据库系统的Oracle数据库网关外，还有Oracle Database Gateway for ODBC代理。

Oracle数据库网关

Oracle数据库网关是专为访问特定的非Oracle系统而设计的网关。

Oracle提供了访问多个商用非Oracle系统的网关。例如，用于Sybase的Oracle数据库网关旨在访问Sybase数据库。

使用Oracle数据库网关，您可以访问分布式数据库系统中的任何位置，而无需了解数据的位置或数据的存储方式。当Oracle数据库向您返回查询结果时，它们将呈现给您，就好像它们被查询的数据存储全部驻留在Oracle分布式数据库的远程实例中一样。

用于ODBC代理的Oracle数据库网关

除了适用于各种非Oracle数据库系统的Oracle数据库网关外，还有Oracle Database Gateway for ODBC代理。

Oracle Database Gateway for ODBC代理仅包含通用代码。您有责任提供必要的驱动程序。Oracle Database Gateway for ODBC使您可以使用ODBC驱动程序访问具有ODBC接口的非Oracle系统。

要使用Oracle Database Gateway for ODBC访问特定的非Oracle系统，必须为非Oracle系统配置ODBC驱动程序。Oracle不提供这些驱动程序。但是，如果非Oracle系统支持ODBC协议，则可以使用Oracle Database Gateway for ODBC访问可使用ODBC驱动程序访问的任何非Oracle系统。

Oracle Database Gateway for ODBC有一些限制。特别是，当与特定目标相比时，功能和性能受到限制。
