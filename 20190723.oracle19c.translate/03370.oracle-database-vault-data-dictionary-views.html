关于Oracle Database Vault数据字典视图

Oracle Database Vault提供了一组DBA样式的数据字典视图，可通过DV_SECANALYST角色或DV_ADMIN角色进行访问。

这些视图提供对DVSYS和LBACSYS模式中各种基础Oracle Database Vault表的DVSYS ，而不会暴露可能存在的主键和外键列。这些视图旨在供数据库管理用户报告Oracle Database Vault配置的状态，而无需执行获取存储在核心表或相关表中的代码的标签所需的连接。
