4.25 DBA_ADVISOR_OBJECT_TYPES

DBA_ADVISOR_OBJECT_TYPES显示有关数据库中所有顾问程序使用的对象类型的信息。

除了常规数据库对象类型（例如TABLE和INDEX ）之外，还定义了以下类型：

  • 系统

  • I / O

  • SGA

  • PGA

  • 共享池

  • BUFFER CACHE

  • 图书馆CACHE

  • 处理

  • SESSION

  • ENQUEUE

  • LATCH

  • ROLLBACK SEGMENT

  • 文件

  • 参数

  • 光标

  • SQL

  • SQL WORKLOAD
