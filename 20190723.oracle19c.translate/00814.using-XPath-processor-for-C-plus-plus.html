31使用XPath XP ++处理器

解释了如何将XPath处理器用于C ++。

注意：

在xml.hpp使用统一的C ++应用程序编程接口（API），用于Oracle XML Developer's Kit（XDK）应用程序。在较旧的，nonunified C ++ API oraxml.hpp已被废弃，仅支持向后兼容。它将在以后的版本中删除。

XPath接口

描述了XPath接口。

Processor接口 - 任何XPath处理器必须符合的基本XPath处理器接口。

CompProcessor接口扩展的XPath处理器，它增加了使用预编译为内部二进制表示的XPath表达式的能力。在此版本中，此接口表示Oracle虚拟机接口。

Compiler接口-XPath编译器成二进制表示。

NodeSetRef接口 - 定义XPath表达式求值返回时对节点集的引用。

XPathException XPath编译器和处理器的接口异常。

用于XPath 1.0对象的XPathObject接口接口。
