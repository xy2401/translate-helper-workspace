10用于C ++的Xsl API包

C ++的Xsl包包含与XSL转换相关的类型和方法。它们包括Xsl数据类型， Compiler方法， CompTransformer方法， Transformer方法和XSLException方法。

定义

typedef enum XslTrIdType {XslTrCXml = 1，XvmTrCXml = 2} XslTrIdType;
