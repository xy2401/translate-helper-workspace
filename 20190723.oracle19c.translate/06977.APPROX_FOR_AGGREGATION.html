1.12 APPROX_FOR_AGGREGATION

APPROX_FOR_AGGREGATION使用近似查询处理替换聚合查询的精确查询处理。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

Oracle RAC

不同的实例可以使用不同的值。

数据分析应用程序大量使用聚合函数和分析函数查询。聚合函数和分析函数需要对大量数据进行排序，并且精确的查询应答需要大量内存，并且可能非常耗时。通过近似查询处理，聚合函数和分析函数查询的结果比精确查询处理更快地返回。近似查询处理对于可接受的可容忍量的错误以获得更快的查询结果的情况非常有用。

可以设置的值是：

  • false ：

    近似查询处理不用于聚合查询和分析查询。

  • true ：近似查询处理用于聚合查询和分析查询。
