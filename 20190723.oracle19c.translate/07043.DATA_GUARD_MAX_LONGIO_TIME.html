1.63 DATA_GUARD_MAX_LONGIO_TIME

DATA_GUARD_MAX_LONGIO_TIME设置在Oracle Data Guard环境中执行长I / O操作时将进程视为挂起之前可以经过的最大秒数。长I / O操作包括打开和关闭操作。

属性 描述

参数类型

整数

默认值

240

可修改

改变系统

可在PDB中修改

没有

价值范围

10到7200

基本

没有

Oracle RAC

不同的实例可以使用不同的值。

注意：

从Oracle Database 19c版本19.1开始提供此参数。
