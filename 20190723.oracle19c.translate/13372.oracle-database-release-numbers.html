关于Oracle数据库版本号

Oracle数据库版本按五个数字段分类，表示发布信息。

注意：

从Oracle Database 18c开始，Oracle以发布更新（更新）和发布更新修订（修订版）的形式提供季度更新。Oracle不再发布补丁集。有关更多信息，请参阅My Oracle Support说明2285040.1。

Oracle数据库版本在version和version_full版本中发布。

version发行version以major release version .0.0.0.0的形式指定。主要版本基于第一次发布Oracle数据库版本的年份的最后两位数字。例如，2018年首次发布的Oracle数据库版本的主要发行版本为18，因此其version发布版本为18.0.0.0.0 。

version_full版本是version发行version的更新version并根据主要版本，季度版本更新版本（更新）和季度版本更新版本（版本）进行指定。version_full版本按五个以句点分隔的数字段进行分类，如以下示例所示：

图1-2 Oracle数据库版本号示例


“图1-2 Oracle数据库版本号示例”的说明

  • 第一个数字：此数字表示主要版本。它还表示首次发布Oracle数据库版本的年份的最后两位数字。

  • 第二个数字：该数字表示版本更新版本（更新）。

  • 第三个数字：此数字表示版本更新修订版本（修订版）。

  • 第四个数字：该数字表示增量版本。此术语可适用于将来版本中的更新。

  • 第五个数字：该数字留作将来使用。

注意：

前三个数字主要标识Oracle数据库版本。

警告：

Oracle强烈建议您在开始升级之前和开始降级之前，将最新版本更新（更新）或捆绑补丁或补丁集更新应用于源数据库和目标数据库。
