关于Oracle数据库网络配置选项

您可以使数据库客户端连接到与多个IP地址关联的服务器，并且可以在没有网络连接的服务器上安装Oracle数据库，并在安装后设置数据库服务。

通常，要安装Oracle数据库的计算机是运行单个数据库实例的服务器，具有可在网络上解析的单个主机名。Oracle Universal Installer使用主机名和Oracle数据库实例信息自动设置网络服务。数据库使用连接描述符为客户端提供数据库服务，该连接描述符解析为运行数据库实例的主机名。

但是，您可以在具有以下非标准配置的服务器上配置Oracle数据库：

  • 多宿主计算机 ：具有多个Oracle数据库安装的服务器

  • 多个别名计算机 ：具有多个别名的服务器，以便多个主机名解析为计算机

  • 非联网计算机 ：安装时没有网络连接的服务器
