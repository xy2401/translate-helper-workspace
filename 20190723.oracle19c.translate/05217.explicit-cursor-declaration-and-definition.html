13.30明确的光标声明和定义

显式游标是指向私有SQL区域的命名指针，该区域存储用于处理特定查询或DML语句的信息 - 通常是返回或影响多行的语句。

您可以使用显式游标一次检索一个结果集的行。

在使用显式游标之前，必须声明并定义它。您可以先声明它（使用cursor_declaration ），然后在同一个块，子程序或包中使用cursor_definition定义它，或者同时声明和定义它（使用cursor_definition ）。

显式游标声明和定义也分别称为游标规范和游标体 。

注意：

包规范中声明的显式游标受包的AUTHID子句的影响。有关更多信息，请参阅“ CREATE PACKAGE语句 ” 。

语义

cursor_declaration

光标

您现在声明的显式游标的名称，稍后将在同一个块，子程序或包中定义。 cursor可以是除保留字SQL之外的任何标识符。 Oracle建议不要为游标提供与数据库表相同的名称。

显式游标名称遵循与变量相同的作用域规则（请参阅“ 标识符的范围和可见性 ” ）。

cursor_definition

定义前面声明的显式游标或者都声明并定义显式游标。

光标

您之前声明并现在定义的显式游标的名称，或者您声明和定义的显式游标的名称。 cursor可以是除保留字SQL之外的任何标识符。 Oracle建议不要为游标提供与数据库表相同的名称。

select_statement中

SQL SELECT语句（不是PL / SQL SELECT INTO语句）。如果游标具有形式参数，则每个参数必须出现在select_statement 。select_statement还可以在其范围内引用其他PL / SQL变量。

对select_statement的限制

这个select_statement不能有WITH子句。

cursor_parameter_dec

游标参数声明。

参数

您声明的正式游标参数的名称。此名称可以出现在select_statement中可以显示常量的任何位置。

在

无论是否指定IN ，正式游标参数都具有IN子程序参数的特征，如表8-1所示 。当光标打开时，形参的值是其实际参数或默认值的值。

数据类型

参数的数据类型。

对数据类型的限制

此datatype不能具有约束（例如， NOT NULL ，或者数字的精度和比例，或字符串的长度）。

表达

指定正式游标参数的默认值。expression的数据类型和形式游标参数必须兼容。

如果OPEN语句未指定正式游标参数的实际参数，则该语句将计算expression并将其值分配给正式游标参数。

如果OPEN语句确实为正式游标参数指定了实际参数，则该语句将实际参数的值分配给形式游标参数，并且不计算expression 。

行类型

游标返回的行的数据类型。此行的列必须与select_statement返回的行的列匹配。

db_table_or_view

数据库表或视图的名称，在详细说明时必须可访问。

光标

先前声明的显式游标的名称。

cursor_variable

先前声明的游标变量的名称。

记录

先前声明的记录变量的名称。

RECORD_TYPE

使用数据类型说明符RECORD定义的用户定义类型的名称。
