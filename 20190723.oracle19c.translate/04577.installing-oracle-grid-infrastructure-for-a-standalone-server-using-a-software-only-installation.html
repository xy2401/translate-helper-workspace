  • 为独立服务器安装Oracle Grid Infrastructure的软件二进制文件
    使用此过程为独立服务器执行仅用于软件的Oracle Grid Infrastructure安装。
  • 为独立服务器配置Oracle Grid Infrastructure的软件二进制文件
    使用这些步骤为Oracle Restart的独立服务器安装配置和激活仅限软件的Oracle Grid Infrastructure。
