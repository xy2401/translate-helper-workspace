安装OCFS2

OCFS2安装由两部分组成，内核模块和工具模块。

受支持的OCFS2内核模块版本取决于Oracle Linux 7提供的Unbreakable Enterprise Kernel的版本。运行以下命令以安装最新版本的OCFS2内核模块：

#yum install kernel-uek ocfs2

OCFS2版本1.8.6-9是此版本支持的OCFS2工具模块版本。安装OCFS2内核模块后，运行以下命令，安装OCFS2工具模块：

#yum install ocfs2-tools-1.8.6-9

注意：

每个群集节点应运行相同版本的OCFS2模块和兼容版本的Unbreakable Enterprise Kernel。
