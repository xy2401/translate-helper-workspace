SAVEPOINT

目的

使用SAVEPOINT语句为系统更改编号（SCN）创建名称，以后可以回滚。

也可以看看：

  • 有关保存点的信息，请参见Oracle数据库概念

  • ROLLBACK有关回滚事务的信息

  • SET TRANSACTION有关设置当前事务特征的信息

先决条件

没有。

句法

savepoint :: =


说明savepoint.eps的说明

语义

保存点

指定要创建的保存点的名称。

保存点名称在给定事务中必须是不同的。如果创建第二个保存点，其标识符与早期保存点相同，则会删除较早的保存点。创建保存点后，您可以继续处理，提交工作，回滚整个事务或回滚到保存点。

例子

创建保存点：示例

要在示例表hr.employees更新Banda和Greene的工资，请检查总部门工资是否超过314,000，然后重新输入Greene的工资：

UPDATE员工SET薪水= 7000 WHERE last_name ='Banda'; SAVEPOINT banda_sal; UPDATE员工SET薪水= 12000 WHERE last_name ='Greene'; SAVEPOINT greene_sal; SELECT SUM（薪水）来自员工; ROLLBACK TO SAVEPOINT banda_sal; UPDATE员工SET薪水= 11000 WHERE last_name ='Greene';承诺;
