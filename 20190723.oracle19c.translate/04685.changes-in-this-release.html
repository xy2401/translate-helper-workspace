Oracle Database Release 18c版本18.1中的更改

以下是本书对Oracle Database 18c版本18.1的更改。

新功能

以下是Oracle C ++调用接口程序员指南 Oracle数据库版本18c版本18.1中的更改。

本节包括以下主题：

  • OCCI支持流量控制器模式下的Oracle Connection Manager

    流量控制器模式中的Oracle连接管理器是放置在数据库客户端和数据库实例之间的代理。Oracle Database 11g第2版（11.2）及更高版本中支持的客户端可以在Traffic Director模式下连接到Oracle Connection Manager。流量控制器模式中的Oracle连接管理器为计划内和计划外数据库服务器中断，连接多路复用支持和负载平衡提供了改进的高可用性（HA）。

    流量控制器模式中的Oracle连接管理器可以使用客户端oraaccess.xml配置文件为在客户端和数据库实例之间提供代理的一个或多个服务配置代理驻留连接池。此功能为计划内和计划外中断提供了改进的高可用性和性能。

    有关详细信息，请参阅关于在Traffic Director模式下使用Oracle Connection Manager 。
