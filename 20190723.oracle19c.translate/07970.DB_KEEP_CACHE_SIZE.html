1.84 DB_KEEP_CACHE_SIZE

DB_KEEP_CACHE_SIZE指定KEEP缓冲池的大小。

属性 描述

参数类型

大整数

句法

DB_KEEP_CACHE_SIZE = integer [K | M | G]

默认值

0 （默认情况下未配置DB_KEEP_CACHE_SIZE ）

可修改

改变系统

可在PDB中修改

没有

价值范围

最小值： 0 （大于零的值会自动修改为粒度大小*处理器组数，或4 MB * CPU数，以较大者为准）

最大值：取决于操作系统

基本

没有

KEEP缓冲池中缓冲区的大小是主块大小（由DB_BLOCK_SIZE初始化参数定义的块大小）。
