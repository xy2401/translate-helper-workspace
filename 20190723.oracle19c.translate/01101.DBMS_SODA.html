150 DBMS_SODA

DBMS_SODA包是一个实现简单Oracle文档访问（ SODA ）的PL/SQL包。SODA允许您将Oracle数据库用作NoSQL文档存储。SODA提供的核心抽象是文档集合。DBMS_SODA包允许您从PL/SQL创建，列出和删除文档集合，以及对文档执行CRUD（创建，替换，更新，删除）操作。所有DDL函数都封装在此包中。

150.1 DBMS_SODA安全模型

具有SODA_APP角色的用户可以使用此包。

所有SODA类型（包和类型）都是SYS类型。PUBLIC被授予本章所述的DBMS_SODA EXECUTE特权。

150.2.1 CREATE_COLLECTION函数

使用集合名称和元数据创建集合。使用元数据中指定的设置并自动分配非元数据，并返回集合对象。如果省略元数据参数或将其设置为NULL ，则会使用默认元数据创建集合。返回的集合打开以进行read和/或write操作。如果集合已存在，则该函数将打开并返回集合对象。

句法

DBMS_SODA.CREATE_COLLECTION（collection_Name IN NVARCHAR2，metadata IN VARCHAR2 DEFAULT NULL，create_Mode IN PLS_INTEGER DEFAULT CREATE_MODE_DDL）RETURN SODA_Collection_T;

参数

表150-2 CREATE_COLLECTION参数

参数 描述

COLLECTION_NAME

集合的名称。

collection_Name的值区分大小写。

元数据

VARCHAR2格式的集合的元数据。

create_Mode
有效值为：
  • DBMS_SODA.CREATE_MODE_DDL （默认）。首先尝试为集合创建新表。如果表已存在，则尝试将该现有表映射到该集合。执行最小化检查以确保表格形状与提供的集合元数据匹配（如果不匹配，则返回错误）。
  • DBMS_SODA.CREATE_MODE_MAP 。尝试将现有表映射到集合。执行最小化检查以确保表格形状与提供的集合元数据匹配（如果不匹配，则返回错误）。

返回值

该函数返回表示集合的Soda_Collection_T对象。

例外

  • Descriptor Error - 如果输入描述符无效

  • Error - 创建集合时发生错误

150.2.2 DROP_COLLECTION函数

从用户的架构中删除现有集合。这也会删除集合中的所有文档。

句法

DBMS_SODA.DROP_COLLECTION（collection_Name IN NVARCHAR2）返回号码;

参数

表150-3 DROP_COLLECTION参数

参数 描述

COLLECTION_NAME

集合的名称。

collection_Name的值区分大小写。

返回值

此函数返回以下值：

  • 1 - 如果集合成功删除

  • 0如果集合不存在

例外

如果在删除集合时发生错误，例如，由于对集合的未提交写入或权限问题。

150.2.3 LIST_COLLECTION_NAMES函数

此函数返回用户架构中的集合名称列表，作为NVARCHAR2的表。

句法

DBMS_SODA.LIST_COLLECTION_NAMES（）返回SODA_CollName_List_T;

返回值

此函数返回集合名称列表，作为NVARCHAR2(255)的表。如果架构中没有集合，则集合列表为空。

例外

Error - 列出集合名称时发生错误。

150.2.4 OPEN_COLLECTION功能

打开现有集合以进行read和/或write操作。

句法

DBMS_SODA.OPEN_COLLECTION（collection_Name IN NVARCHAR2）RETURN SODA_Collection_T;

参数

表150-4 OPEN_COLLECTION参数

参数 描述

COLLECTION_NAME

集合的名称。

collection_Name的值区分大小写。

返回值

此函数返回以下值：

  • 一个打开的集合对象

  • 如果集合不存在，则为NULL

例外

Error - 创建集合时发生错误
