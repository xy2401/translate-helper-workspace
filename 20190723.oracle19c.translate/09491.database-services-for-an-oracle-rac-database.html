Oracle RAC数据库的数据库服务

每个数据库由一个或多个服务表示。服务由服务名称标识，例如sales.example.com 。客户端使用服务名称来标识它必须访问的数据库。

在安装期间，Oracle RAC数据库配置了一个与数据库同名的默认数据库服务。此服务可用于执行数据库管理任务。为客户端和应用程序与数据库的连接创建其他服务。

服务名称可以与多个数据库实例相关联，并且实例可以与多个服务相关联。侦听器充当客户端和数据库实例之间的中介，并将连接请求路由到适当的实例。连接到服务的客户端不必指定要连接到的实例。
