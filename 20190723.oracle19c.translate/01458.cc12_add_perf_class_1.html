说明cc12_add_perf_class_1.gif的说明

屏幕截图显示了Add Performance Class页面。此页面上的描述性文本显示“性能类名称用于标识流经系统的工作请求集合以及希望设置的性能目标”。此文本下方是标记为Performance Class Name的文本输入字段。在屏幕截图中，已在此字段中输入值NewClass。

本页的下一部分标题为“分类器”。本节的描述性文本指出“分类器是一组规则，用于根据由数据库会话参数组成的布尔表达式对传入请求进行分类。允许使用逗号分隔的多个值。“描述性文本下方是两个按钮，删除分类器和添加分类器。

按钮下方是两个超链接：全选和选择无。在这些超链接下面是一系列文本输入字段，您可以在其中指定有关新Performance Class的信息。第一个字段是Select选项。下一个字段是“服务”字段，您可以在其中键入数据库服务的名称，或单击搜索图标以搜索可用的服务名称。接下来的字段是Module，Action，UserName和Program。对于这些字段中的每个字段，您都可以输入文本，然后选择“入设置”或“未设置”选项。

描述结束。
