180 DBMS_WARNING

DBMS_WARNING包提供了一种操作PL / SQL警告消息行为的方法，特别是通过读取和更改PLSQL_WARNINGS初始化参数的设置来控制哪些类型的警告被抑制，显示或视为错误。此包提供查询，修改和删除当前系统或会话设置的界面。

本章包含以下主题：

  • 安全模型

  • DBMS_WARNING子程序摘要

180.1 DBMS_WARNING安全模型

请注意，对于以下所有接口，如果scope参数的值为SYSTEM ，则用户必须具有ALTER SYSTEM权限。

180.2.1 ADD_WARNING_SETTING_CAT过程

您可以使用此过程中提供的值修改当前会话或系统的警告设置。如果尚未设置warning_category或warning_value的值，则该值将添加到现有参数设置中，或覆盖现有值。

调用此函数的效果与在指定的类别上添加限定符（ ENABLE / DISABLE / ERROR ）到当前会话或系统设置的结尾相同。

句法

DBMS_WARNING.ADD_WARNING_SETTING_CAT（warning_category IN VARCHAR2，warning_value IN VARCHAR2，范围IN VARCHAR2）;

参数

表180-2 ADD_WARNING_SETTING_CAT过程参数

参数 描述

warning_category

类别的名称。允许的值为ALL ， INFORMATIONAL ， SEVERE和PERFORMANCE 。

warning_value

类别的价值。允许的值为ENABLE ， DISABLE和ERROR 。

范围

指定是在会话上下文还是系统上下文中执行更改。允许的值为SESSION或SYSTEM 。

180.2.2 ADD_WARNING_SETTING_NUM过程

您可以使用此过程中提供的值修改当前session或system警告设置。如果已设置该值，则将覆盖现有值。

调用此函数的效果与在指定的类别上添加限定符（ ENABLE / DISABLE / ERROR ）到当前会话或系统设置的结尾相同。

句法

DBMS_WARNING.ADD_WARNING_SETTING_NUM（warning_number IN NUMBER，warning_value IN VARCHAR2，范围IN VARCHAR2）;

参数

表180-3 ADD_WARNING_SETTING_NUM过程参数

参数 描述

warning_number

警告号码。允许的值都是有效的警告号。

warning_value

类别的价值。允许的值为ENABLE ， DISABLE和ERROR 。

范围

指定是在会话上下文还是系统上下文中执行更改。允许的值为SESSION或SYSTEM 。

示例180-1启用弃用警告

此示例显示如何使用DBMS_WARNING包为会话启用DEPRECATE编译指示警告。
BEGIN DBMS_WARNING.ADD_WARNING_SETTING_NUM（6019，'ENABLE'，'SESSION'）; DBMS_WARNING.ADD_WARNING_SETTING_NUM（6020，'ENABLE'，'SESSION'）; DBMS_WARNING.ADD_WARNING_SETTING_NUM（6021，'ENABLE'，'SESSION'）; DBMS_WARNING.ADD_WARNING_SETTING_NUM（6022，'ENABLE'，'SESSION'）;结束;

表180-6 GET_WARNING_SETTING_NUM函数参数

180.2.6 GET_WARNING_SETTING_STRING函数

此函数返回当前会话的整个警告字符串。

句法

DBMS_WARNING.GET_WARNING_SETTING_STRING RETURN VARCHAR2;

使用说明

如果您对v$parameter或v$paramater2固定表没有SELECT或READ权限，或者您想自己解析警告字符串，然后使用SET_WARNING_SETTING_STRING修改和设置新值，请使用此SET_WARNING_SETTING_STRING 。

180.2.7 SET_WARNING_SETTING_STRING过程

此过程使用新值替换以前的设置。

警告字符串可能包含类别和警告编号的混合，使用与发出ALTER SESSION或SYSTEM SET PLSQL_WARNINGS命令时在'='右侧使用的语法相同的语法。这与ALTER SESSION或ALTER SYSTEM命令具有相同的效果。

句法

DBMS_WARNING.SET_WARNING_SETTING_STRING（warning_value IN VARCHAR2，范围IN VARCHAR2）;

参数

表180-7 SET_WARNING_SETTING_STRING过程参数

参数 描述

warning_value

将构成新值的新字符串。

范围

这将指定是在会话上下文或系统上下文中完成更改。允许的值为SESSION或SYSTEM 。
