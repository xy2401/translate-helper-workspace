149 RDJ-00001至RDJ-00137

RDJ-00001：OK ADDRESS

原因：操作成功。

行动：不需要采取进一步行动。

RDJ-00002：操作无效：

原因：指定的操作不正确。

操作：指定有效的操作。

RDJ-00003：解析“ string ”\ n string出错

原因：解析地址时出错。

操作：使用错误中的其他信息更正地址。

RDJ-00004：跟踪级别无效

原因：指定的跟踪级别不正确。

操作：指定有效的跟踪级别。

RDJ-00005：操作： string失败。错误： string

原因： RD操作因指定的错误而失败。

操作：重新尝试启用跟踪的操作以确定发生了什么。

RDJ-00006：无效的端口号： number

原因：为服务指定的端口号无效。

操作：指定大于零的端口号。

RDJ-00008：指定的编号无效：

原因：指定的数字包含非数字字符。

操作：指定仅包含数字字符的数字。

RDJ-00009：地址格式无效：

原因：为地址指定的格式不正确。

操作：指定有效的格式。

RDJ-00011：结果处理器可能不为空

原因：为要处理操作结果的类指定了null对象。

操作：指定非null对象。

RDJ-00012：无效的实例唯一性

原因：为实例指定的唯一性值无效。

操作：指定有效的唯一性值。

RDJ-00013：无效的服务实例属性

原因：为服务实例指定的属性无效。

操作：以正确的格式指定属性。

RDJ-00015：无效的服务实例标记：“ string ”。可用标签：

原因：为服务实例指定的标记值不正确。

操作：指定有效的标记值。

RDJ-00017：域名类型无效：“ string ”。可用的域类型：

原因：为域指定的类型无效。

操作：指定有效的域类型。

RDJ-00048：未知命令：“ string ”

原因：指定了无效的命令。

操作：指定有效命令。

RDJ-00049：未知选项：“ string ”

原因：指定了无效选项。

操作：指定有效选项。

RDJ-00052：无效的标志值：

原因：为标志指定的值无效。

操作：指定有效的标志值。

RDJ-00053：删除跟踪输出文件失败：

原因：该工具尝试删除跟踪文件，以便创建新文件。但是，尝试失败了。

操作：检查随附的异常，了解发生的情况。

RDJ-00054：删除错误日志文件失败：

原因：该工具尝试删除错误日志，以便它可以创建一个新错误日志。但是，尝试失败了。

操作：检查随附的异常，了解发生的情况。

RDJ-00055：跟踪文件已存在：

原因：指定的跟踪文件已存在。

操作：删除文件或指定不存在的文件。

RDJ-00056：错误日志已存在：

原因：指定的错误日志已存在。

操作：删除文件或指定不存在的文件。

RDJ-00057：跟踪文件是一个目录：

原因：指定的跟踪文件实际上是一个目录。

操作：指定文件名。

RDJ-00058：错误日志是一个目录：

原因：指定的错误日志实际上是一个目录。

操作：指定文件名。

RDJ-00059：无法创建跟踪文件“ string ”

原因：该工具无法创建跟踪文件。

操作：检查附带的异常以获取有关错误的详细信息。

RDJ-00060：无法创建错误日志“ string ”。检查父目录的权限：

原因：该工具无法创建错误日志。

操作：检查附带的异常以获取有关错误的详细信息。

RDJ-00061：无法创建文件流：

原因：该工具无法为文件创建Java流。

操作：检查附带的异常以获取有关错误的详细信息。

RDJ-00062：无法创建打印流：

原因：工具尝试为文件创建Java打印流时发生错误。

操作：检查附带的异常以获取有关错误的详细信息。

RDJ-00063：无效生存时间：

原因：为生存时间指定了非数字值。

操作：指定数值。

RDJ-00065：选项类型必须大于零：

原因：为选项指定的类型小于或等于零。

操作：指定大于零的类型。

RDJ-00134：未知类型：

原因：指定的类型无效。

操作：指定已知类型。

RDJ-00135：无效的流类型：

原因：指定的流类型无效。

操作：指定已知类型。

RDJ-00136：参数计数无效：

原因：指定的参数数量无效。

操作：指定有效数量的参数。

RDJ-00137：SRVMContext init失败。

原因：尝试设置进程环境失败，因为资源发现工具初始化CLuSterware Backbone（CLSB）以启用跟踪文件创建时发生错误。

操作：确定是否设置了环境变量ORACLE_HOME和ORACLE_BASE。请参阅“Oracle Grid Infrastructure安装指南”以适当地设置它们并重试该操作。
