2.143 ALL_FILE_GROUP_VERSIONS

ALL_FILE_GROUP_VERSIONS显示当前用户可访问的文件组的顶级版本信息。

相关观点

  • DBA_FILE_GROUP_VERSIONS显示数据库中所有文件组的顶级版本信息。

  • USER_FILE_GROUP_VERSIONS显示当前用户拥有的所有文件组的顶级版本信息。此视图不显示FILE_GROUP_OWNER列。
