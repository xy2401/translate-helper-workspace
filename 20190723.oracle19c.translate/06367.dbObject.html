4.6 dbObject

目的

使用dbObject子句识别数据库或数据库的子集。

句法

dbObject :: =


说明dbobject.eps的说明

（ datafileSpec :: = ）

语义

语法元素 描述

数据库

指定整个数据库。在多租户容器数据库（CDB）中，指定整个CDB。

DATABASE ROOT

仅指定CDB中的根。

PLUGGABLE DATABASE pdb_name

指定CDB中的一个或多个可插拔数据库（PDB）。使用逗号分隔的列表指定多个PDB。

DATAFILE datafileSpec

通过文件名或绝对数据文件号指定一个或多个数据文件的列表。

另请参见： datafileSpec
TABLESPACE tablespace_name

指定一个或多个表空间。

TABLESPACE pdb_name : tablespace_name

指定PDB中表空间的名称。使用逗号分隔的列表指定多个表空间。多个PDB可以具有相同名称的表空间。名称前面的限定符唯一标识表空间。 pdb-name是包含tablespace_name的PDB的名称。如果省略pdb_name ， pdb_name root用作默认容器。
