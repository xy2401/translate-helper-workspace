40 DBMS_CSX_ADMIN

DBMS_CSX_ADMIN包提供了一个接口，用于在传输包含二进制XML数据的表空间时自定义设置。

本章包含以下主题：

  • 概观

  • 安全模型

  • 常量

  • DBMS_CSX_ADMIN摘要

40.1 DBMS_CSX_ADMIN概述

在传输包含二进制XML数据的表空间时，DBA可以使用此包来自定义设置。为了运行可传输的表空间作业，不需要使用该包。

默认情况下，所有二进制XML表都将使用默认令牌表集，该表集将在目标数据库上传输期间进行复制。为了避免传输潜在的大型令牌表集的成本，DBA可以选择为给定的表空间注册一组新的令牌表。该包提供令牌表集注册和查找的例程。

40.2 DBMS_CSX_ADMIN安全模型

由XDB拥有， DBMS_CSX_ADMIN包必须由SYS或XDB创建。 EXECUTE特权被授予SYS或XDB或DBA 。此包中的子程序使用当前用户的权限执行。

40.4.1 GETTOKENTABLEINFO程序和功能

此过程已超载。将描述特定形式的功能以及语法声明。

给定表名和所有者，过程的第一个重载将返回令牌表集的全局唯一标识符（GUID），其中可以找到此表的令牌映射。该过程还返回令牌表的名称，以及令牌表集是否为默认表。

给定表的对象编号，过程的第二个重载将返回表使用的标记表集的GUID，以及这是否是默认标记表集。

句法

DBMS_CSX_ADMIN.GETTOKENTABLEINFO（所有者名称为IN VARCHAR2，表名为IN VARCHAR2，guid OUT RAW，qnametable OUT VARCHAR2，nmspctable OUT VARCHAR2，级别OUT NUMBER，tabno OUT NUMBER）;
DBMS_CSX_ADMIN.GETTOKENTABLEINFO（tabno IN NUMBER，guid OUT RAW）;返回布尔;

参数

给定表空间编号，此过程返回此表空间的GUID和标记表名称。

40.4.3 NAMESPACEIDTABLE函数

此过程返回默认名称空间ID标记表。

句法

DBMS_CSX_ADMIN.NAMESPACEIDTABLE RETURN VARCHAR2;

40.4.4 PATHIDTABLE功能

此过程返回默认的路径ID令牌表。这用于在对具有XML索引的XML表执行EXPLAIN PLAN之前为用户授予对默认路径ID令牌表的权限。

句法

DBMS_CSX_ADMIN.PATHIDTABLE RETURN VARCHAR2;

40.4.5 QNAMEIDTABLE函数

此过程返回默认的qname-ID标记表。

句法

DBMS_CSX_ADMIN.QNAMEIDTABLE RETURN VARCHAR2;
