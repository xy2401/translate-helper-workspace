6.148 DBA_XS_AUDIT_TRAIL

DBA_XS_AUDIT_TRAIL描述了Oracle Database Real Application Security特有的所有审计记录。

柱 数据类型 空值 描述

用户身份

VARCHAR2（128）

审核其操作的数据库用户的名称

行动

数

数字审计跟踪操作类型代码。操作类型的相应名称位于ACTION_NAME列中。

ACTION_NAME

VARCHAR2（64）

与ACTION列中的数字代码对应的操作类型的名称

OBJ_OWNER

VARCHAR2（128）

受操作影响的对象的所有者

OBJ_NAME

VARCHAR2（128）

受操作影响的对象的名称

值return_code

数

操作生成的Oracle错误代码

XS_USER_NAME

VARCHAR2（128）

Real Application Security用户的名称

XS_SESSIONID

RAW（33）

真实应用程序安全会话的标识

XS_INACTIVITY_TIMEOUT

数

Real Application Security会话的非活动超时

XS_ENTITY_TYPE

VARCHAR2（32）

Real Application Security实体的类型。可能的值包括USER ， ROLE ， ROLESET ， SECURITYCLASS ， ACL ， DATASECURITY和NSTEMPLATE 。

XS_TARGET_PRINCIPAL_NAME

VARCHAR2（128）

Real Application Security操作中的目标主体名称。可能的操作包括设置验证器，设置密码，添加代理，删除代理，切换用户，分配用户，创建会话，授予角色。

XS_PROXY_USER_NAME

VARCHAR2（128）

Real Application Security代理用户的名称。

XS_DATASEC_POLICY_NAME

VARCHAR2（128）

启用或禁用Real Application Security数据安全策略的名称

XS_SCHEMA_NAME

VARCHAR2（128）

启用中的架构名称，禁用数据安全策略和全局回调操作

XS_CALLBACK_EVENT_TYPE

VARCHAR2（32）

Real Application Security全局回调事件类型

XS_PACKAGE_NAME

VARCHAR2（128）

全局回调的Real Application Security回调包名称

XS_PROCEDURE_NAME

VARCHAR2（128）

VARCHAR2（128）

已启用的角色

XS_COOKIE

XS_NS_ATTRIBUTE

VARCHAR2（4000）

Real Application Security会话命名空间属性的名称

XS_NS_ATTRIBUTE_OLD_VAL

VARCHAR2（4000）

Real Application Security会话命名空间属性的旧值

XS_NS_ATTRIBUTE_NEW_VAL

VARCHAR2（4000）
