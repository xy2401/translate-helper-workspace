5.377 DBA_SQLTUNE_RATIONALE_PLAN

DBA_SQLTUNE_RATIONALE_PLAN显示数据库中所有SQL语句的执行计划中的基本原理和操作之间的关联。

相关视图

USER_SQLTUNE_RATIONALE_PLAN显示当前用户拥有的SQL语句的执行计划中的基本原理和操作之间的关联。

柱 数据类型 空值 描述

TASK_ID

NUMBER（38）

不是NULL

调整任务标识符

EXECUTION_NAME

VARCHAR2（128）

不是NULL

与此条目（行）关联的任务执行的名称

RATIONALE_ID

NUMBER（38）

不是NULL

理由标识符

OBJECT_ID

NUMBER（38）

不是NULL

顾问框架对象标识符

OPERATION_ID

NUMBER（38）

不是NULL

操作标识符

PLAN_ATTRIBUTE

VARCHAR2（34）

执行计划的类型：

  • Original - 查询的原始计划

  • Original with adjusted cost - 与Original相同，但调整后的成本

  • Using SQL profile - Using SQL profile规划

  • Using new indices - 应用Using new indices计划
