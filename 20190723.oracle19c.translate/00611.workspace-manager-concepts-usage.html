第一部分概念和使用信息

本文档包含三个部分：

  • 参考信息提供有关Workspace Manager PL / SQL API（DBMS_WM包）和静态数据字典视图的参考信息。

第一部分是为了有效学习Workspace Manager而组织的。它首先介绍了基本概念和技术，然后介绍了更高级的材料（例如Workspace Manager事件和有效时间支持）。第一部分包含以下章节：

  • Workspace Manager简介
    Oracle Workspace Manager（通常称为工作区管理器）提供了一种基础结构，使应用程序能够创建工作空间并将不同版本的表行值分组到不同的工作空间中。
  • 工作区管理器事件
    某些应用程序可能对了解正在执行的Workspace Manager操作感兴趣，并且可能希望基于此操作采取某些操作。可以将多种类型的Workspace Manager操作捕获为事件。
  • 工作区管理器有效时间支持
    本章介绍了对启用版本的表的有效时间支持，也称为有效约会。
