Oracle Base Directory命名约定

Oracle Base目录是Oracle数据库安装所有者的数据库主目录，以及Oracle Grid Infrastructure所有者的日志文件位置。

使用语法/pm/h/u命名Oracle基本目录，其中pm是字符串安装点名称， h是从一小组标准目录名称中选择的， u是目录所有者的名称。

您可以将同一Oracle基本目录用于多个安装。如果不同的操作系统用户在同一系统上安装Oracle软件，则必须为每个安装所有者创建单独的Oracle基本目录。为便于管理，Oracle建议您为每个Oracle软件安装所有者创建一个唯一的所有者，以分隔日志文件。

由于所有Oracle安装所有者都写入中央Oracle清单文件，并且该文件mountpoint与初始Oracle安装位于相同的安装点路径中，因此Oracle建议您对所有Oracle安装所有者使用相同的/ pm/h路径。
