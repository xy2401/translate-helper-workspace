了解网络地址

将每个接口标识为公共或私有接口，或者作为您不希望Oracle Grid Infrastructure或Oracle Flex ASM集群使用的接口。

在安装过程中，系统会要求您确定OUI在群集节点上检测到的每个网络接口的计划用途。将每个接口标识为公共或私有接口，或者作为您不希望Oracle Grid Infrastructure或Oracle Flex ASM集群使用的接口。公共接口上配置公共和虚拟IP地址。私有地址在私有接口上配置。

  • 关于公共IP地址
    公共IP地址使用动态主机配置协议（DHCP）动态分配，或者在域名系统（DNS）或主机文件中静态定义。
  • 关于私有IP地址
    Oracle Clusterware使用标记为私有的接口进行节点间通信。
  • 关于虚拟IP地址
    虚拟IP（VIP）地址在网格命名服务（GNS），DNS或主机文件中注册。
  • 关于网格命名服务（GNS）虚拟IP地址
    GNS虚拟IP地址是在域名系统（DNS）中配置的静态IP地址。
  • 关于SCAN
    在安装Oracle Grid Infrastructure期间，将为SCAN创建多个Oracle Clusterware资源。
  • 关于共享SCAN
    共享SCAN提供跨多个群集共享SCAN VIP的功能，从而减少部署Oracle群集时必须分配的IP地址数量。
