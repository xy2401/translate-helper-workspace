XMLCOLATTVAL

句法


插图描述xmlcolattval.eps

目的

XMLColAttVal创建一个XML片段，然后展开生成的XML，以便每个XML片段都具有带有属性name的name column 。

您可以使用AS子句将name属性的值更改为列名以外的值。您可以通过指定c_alias （字符串文字）或指定EVALNAME value_expr 。在后一种情况下，将计算值表达式，并将结果（必须是字符串文字）用作别名。如果初始化参数MAX_STRING_SIZE = STANDARD ，则别名最多为4000个字符;如果MAX_STRING_SIZE = EXTENDED则MAX_STRING_SIZE = 32767个字符。有关详细信息，请参阅“ 扩展数据类型 ” 。

您必须为value_expr指定一个值。如果value_expr为null，则不返回任何元素。

对XMLColAttVal的限制

您不能为value_expr指定对象类型列。

例子

以下示例为员工子集创建Emp元素，其中嵌套employee_id ， last_name和salary元素作为Emp的内容。每个嵌套元素都命名为column并且具有name属性，其列名称为属性值：

SELECT XMLELEMENT（“Emp”，XMLCOLATTVAL（e.employee_id，e.last_name，e.salary））“Emp Element”来自员工e WHERE employee_id = 204; Emp Element ------------------------------------------------ -------------------- <Emp> <column name =“EMPLOYEE_ID”> 204 </ column> <column name =“LAST_NAME”> Baer </ column> < column name =“SALARY”> 10000 </ column> </ Emp>

请参阅XMLFOREST的示例以比较这两个函数的输出。
