使用CVU运行OUI并使用修复脚本

Oracle Universal Installer与Cluster Verification Utility（CVU）完全集成，可自动执行许多CVU先决条件检查。运行安装程序时，Oracle Universal Installer会运行所有先决条件检查并创建修订脚本。您可以在不启动安装的情况下运行OUI直到“摘要”屏幕。

您也可以手动运行CVU命令以检查系统准备情况。有关更多信息，请参阅：

“Oracle Clusterware管理和部署指南”
确保在安装期间不运行cron作业 如果安装程序在每日cron作业启动时运行，那么如果您的cron作业正在执行清理，则可能会遇到无法解释的安装问题，并且在安装完成之前删除临时文件。Oracle建议您在运行每日cron作业之前完成安装，或者禁用执行清理的每日cron作业，直到安装完成为止。
确定客户端安装类型

安装Oracle数据库客户端时，可以选择以下安装类型之一：

  • Instant Client：使您只能安装Oracle调用接口（OCI），Oracle C ++调用接口（OCCI），Pro * C或Java数据库连接（JDBC）OCI应用程序所需的共享库。与其他Oracle数据库客户端安装类型相比，此安装类型所需的磁盘空间要少得多。有关Oracle Database Instant Client的详细信息，请参阅以下URL：

    http://www.oracle.com/technetwork/database/features/instant-client/index.html

  • 管理员：使应用程序能够连接到本地系统或远程系统上的Oracle数据库实例。它还提供了可以管理Oracle数据库的工具。

  • 运行时：使应用程序能够连接到本地系统或远程系统上的Oracle数据库实例。

  • 自定义：允许您从Administrator和Runtime组件列表中选择单个组件。

获取My Oracle Support帐户信息。

在安装过程中，您需要My Oracle Support用户名和密码来配置安全更新，下载软件更新和其他安装任务。您可以通过以下URL注册My Oracle Support：

https://support.oracle.com/
确定您是否需要32位客户端软件

64位Oracle数据库客户端软件不包含任何32位客户端二进制文件。如果在64位平台上需要32位客户端二进制文件，则将32位二进制文件从相应的32位客户端软件安装到单独的Oracle主目录中。

64位Oracle数据库客户端预安装要求也适用于32位Oracle数据库客户端。

有关更多信息，请参阅My Oracle Support说明883702.1：

https://support.oracle.com/rs?type=doc&id=883702.1

Oracle数据库客户端和Oracle数据库互操作性

有关Oracle Database Client与Oracle Database版本之间的互操作性的信息，请参阅My Oracle Support说明207303.1：
