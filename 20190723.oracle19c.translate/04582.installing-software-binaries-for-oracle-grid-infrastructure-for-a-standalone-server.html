为独立服务器安装Oracle Grid Infrastructure的软件二进制文件

使用此过程为独立服务器执行仅用于软件的Oracle Grid Infrastructure安装。

  1. 以Oracle Restart软件所有者用户（ oracle ）身份登录。
  2. 下载Oracle Grid Infrastructure以获取独立服务器安装映像文件，创建Grid主目录，并提取此Grid主目录中的映像文件。

    例如：

    $ mkdir -p /u01/app/oracle/product/19.0.0/grid $ chown oracle：oinstall /u01/app/oracle/product/19.0.0/grid $ cd /u01/app/oracle/product/19.0。 0 / grid $ unzip -q /tmp/grid_home.zip

    注意：

    确保您创建的Grid主目录路径符合Oracle Optimal Flexible Architecture建议。此外，仅在您创建的此Grid主目录中解压缩安装映像文件。
  3. 确保完成所有存储和服务器预安装要求。使用以下命令验证服务器是否满足安装要求：
    $ cd /u01/app/oracle/product/19.0.0/grid $ runcluvfy.sh stage -pre hacfg
  4. 运行gridSetup.sh以启动Oracle Grid Infrastructure安装向导：
    $ Grid_home /gridSetup.sh

    其中， Grid_home是Oracle Grid Infrastructure主目录。

    注意：

    您必须从Oracle Grid Infrastructure介质为独立服务器安装Oracle Grid Infrastructure。
  5. 在“选择配置选项”屏幕中，选择“ 仅设置软件”选项，以便为独立服务器执行仅适用于软件的Oracle Grid Infrastructure安装。单击下一步。
  6. 根据需要响应设置Oracle Grid Infrastructure的提示。单击帮助以获取信息。
  7. Oracle Grid Infrastructure安装向导会提示您运行root.sh脚本，如果需要，还会运行orainstRoot.sh脚本。
  8. root.sh脚本输出提供有关如何继续的信息，具体取决于您计划在此安装中完成的配置。请注意此信息。
