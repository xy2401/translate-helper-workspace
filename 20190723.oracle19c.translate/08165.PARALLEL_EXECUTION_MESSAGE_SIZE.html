1.246 PARALLEL_EXECUTION_MESSAGE_SIZE

PARALLEL_EXECUTION_MESSAGE_SIZE指定用于并行执行的消息的大小（以前称为并行查询，PDML，并行恢复，复制）。

属性 描述

参数类型

整数

默认值

依赖于操作系统

可修改

没有

可在PDB中修改

没有

价值范围

最低：2148

最大值：65536，但某些操作系统的值可能较小

基本

没有

Oracle RAC

多个实例必须具有相同的值。

在大多数平台上，默认值如下：

  • 如果COMPATIBLE设置为11.2.0或更高，则为16384字节

  • 如果COMPATIBLE小于11.2.0则为2148字节

默认值足以满足大多数应用程序的需要。较大的值需要更大的共享池。较大的值会以更高的内存使用成本获得更好的性能。因此，复制不会因增加大小而受益。
