SQL * Plus命令

SQL * Plus是一个命令行工具，提供对Oracle RDBMS的访问。 SQL * Plus使您能够：

  • 输入SQL * Plus命令以配置SQL * Plus环境

  • 启动和关闭Oracle数据库

  • 连接到Oracle数据库

  • 输入并执行SQL命令和PL / SQL块

  • 格式化和打印查询结果

SQL * Plus可在多个平台上使用。

表A-1中显示的命令是命令行界面中提供的SQL * Plus命令。并非显示所有命令或命令参数。

启动数据库

db_options具有以下语法：

[FORCE] [RESTRICT] [PFILE = filename] [QUIET] [MOUNT [ dbname ] | [OPEN [ open_db_options ] [ dbname ]] | NOMOUNT]

open_db_options具有以下语法：

阅读{仅| | WRITE [RECOVER]} |恢复

其中cdb_options具有以下语法：

root_connection_options | pdb_connection_options

root_connection_options具有以下语法：

PLUGGABLE DATABASE pdbname [FORCE] | [限制] [OPEN { open_pdb_options }]

其中pdb_connection_options具有以下语法：

[FORCE] | [限制] [OPEN { open_pdb_options }]

open_pdb_options具有以下语法：

READ WRITE |只读

upgrade_options具有以下语法：

[PFILE = filename ] {UPGRADE | DOWNGRADE} [QUIET]
