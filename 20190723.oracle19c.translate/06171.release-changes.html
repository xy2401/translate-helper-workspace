Oracle Database 12 c第1版（12.1）中的更改

以下是Oracle数据库测试指南（适用于Oracle Database 12 c第1版（12.1））中的更改。

新功能

以下功能是此版本中的新增功能：

  • SQL性能分析器快速检查（SPA快速检查）

    在某些Oracle企业管理器云控制数据库管理页面上，SPA快速检查可以在进行更改之前验证系统更改对数据库工作负载的影响。

    请参阅使用SPA快速检查 。

  • SQL Performance Analyzer支持多租户架构

    通过重新映射其CDB标识符，可以将从非CDB传输到多租户容器数据库（CDB）的SQL调整集用作SQL Performance Analyzer的输入源。

    请参阅使用API在分析任务中重新映射多租户容器数据库标识符 。

  • 工作量情报

    Workload Intelligence包含一套Java程序，使您可以分析存储在捕获的工作负载中的数据。

    请参阅使用Workload Intelligence 。

  • 统一数据库重播

    通过Consolidated Database Replay，您可以整合从一个或多个系统捕获的多个工作负载，并在单个测试系统（例如单个Oracle Exadata计算机）上同时重放这些工作负载。

    请参阅使用Consolidated Database Replay 。

  • 工作量扩大规模

    数据库重放使您可以在各种用例和场景中执行纵向扩展测试和压力测试。

    请参阅使用Workload Scale-Up 。

其他变化

以下是此版本中的其他更改：

  • 新书

    Oracle数据库实际应用程序测试用户指南现在是Oracle数据库测试指南的一部分 。

  • 新的一部分

    第3部分“测试数据管理”已添加到Oracle数据库测试指南中，以描述Oracle数据库的测试数据管理功能。
