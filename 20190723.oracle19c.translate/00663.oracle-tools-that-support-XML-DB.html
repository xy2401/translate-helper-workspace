第VII部分支持Oracle XML DB的Oracle工具

您可以将各种Oracle工具与Oracle XML DB一起使用。其中包括用于管理Oracle XML DB，加载XML数据和交换XML数据的工具。

  • 管理Oracle XML DB
    Oracle XML DB的管理包括安装，升级和配置它。
  • 如何加载XML数据
    将XML数据加载到Oracle XML DB的主要方法是使用SQL * Loader。
  • 导出和导入Oracle XML DB数据
    您可以使用Oracle Data Pump导出和导入XMLType表以与Oracle XML DB一起使用。
  • 使用Oracle数据库高级排队的XML数据交换
    您可以使用Oracle数据库高级队列（AQ）交换XML数据，该队列提供数据库集成的消息队列。AQ使用消息启用和管理应用程序之间的异步通信。它支持点对点和发布/订阅通信模型。
