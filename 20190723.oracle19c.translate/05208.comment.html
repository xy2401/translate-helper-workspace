13.12评论

注释是PL / SQL编译器忽略的源程序文本。它的主要目的是记录代码，但您也可以使用它来禁用过时或未完成的代码片段（也就是说，您可以将代码转换为注释）。PL / SQL包含单行和多行注释。

句法

comment :: =


说明comment.eps的描述

语义

评论

-

将该行的其余部分转换为单行注释。包装到下一行的任何文本都不是注释的一部分。

警告：

不要在PL / SQL块中放置单行注释，以便Oracle预编译器程序动态处理。Oracle预编译器程序忽略行尾字符，这意味着单行注释在块结束时结束。

/ *

开始发表评论，可以跨越多行。

* /

结束评论。

文本

任何文字。

对文本的限制

在多行注释中， text不能包含多行注释分隔符/*或*/ 。因此，一条多行注释不能包含其他多行注释。但是，多行注释可以包含单行注释。
