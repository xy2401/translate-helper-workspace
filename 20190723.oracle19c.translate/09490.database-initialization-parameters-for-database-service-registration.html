数据库服务注册的数据库初始化参数

Oracle Database 19c数据库服务自动向LOCAL_LISTENER和REMOTE_LISTENER参数中指定的侦听器注册。

在注册期间，侦听器注册（LREG）进程将诸如服务名称，实例名称和工作负载信息之类的信息发送给侦听器。当监听器在Oracle实例启动后启动，并且监听器可用于服务注册时，直到下次Oracle Database LREG进程启动其发现例程时才会进行注册。默认情况下，LREG发现例程每60秒启动一次。要覆盖60秒延迟，请使用SQL语句ALTER SYSTEM REGISTER 。该声明迫使LREG立即注册该服务。

注意：

Oracle建议您在启动侦听器后立即创建脚本以运行ALTER SYSTEM REGISTER语句。如果在注册实例并且当前已注册所有服务时或在侦听器关闭时运行此语句，则该语句无效。
