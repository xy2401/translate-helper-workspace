5.2多级集合类型

多级集合类型是集合类型，其元素本身是直接或间接的另一种集合类型。

可能的多级集合类型是：

  • 嵌套表类型的嵌套表

  • varray类型的嵌套表

  • 嵌套表类型的Varray

  • Varray类型的Varray

  • 嵌套表或用户定义类型的varray，其属性是嵌套表或varray类型

与单级集合类型一样，多级集合类型：

  • 可以用作关系表中的列，也可以用作对象表中的对象属性。

  • 要求源和目标都具有相同的已声明数据类型以进行分配。

5.2.1多级集合类型的嵌套表存储表

要使用嵌套表的多级嵌套表集合，必须指定嵌套表存储子句。

嵌套表类型列或对象表属性需要存储表来存储其所有嵌套表的行，如“ 存储嵌套表的元素 ”中所述 。

使用嵌套表的多级嵌套表集合，必须为内部集合和外部嵌套表集指定嵌套表存储子句（ STORE AS ）。您必须拥有与集合中嵌套表级别一样多的嵌套表存储子句。

每个嵌套表存储表都包含一个可由NESTED_TABLE_ID引用的NESTED_TABLE_ID ，该列将存储表中的行NESTED_TABLE_ID父表中的关联行。父表本身就是一个嵌套表，它有两个系统提供的ID列：

  • 系统提供的ID列，可由NESTED_TABLE_ID引用，该列将其行键回到其父表中的行。

  • 系统提供的ID列，由嵌套表子NESTED_TABLE_ID中的NESTED_TABLE_ID列隐藏和引用。

如果未指定具有NESTED_TABLE_ID列的主键，则数据库会自动在NESTED_TABLE_ID列上创建b树索引以获得更好的性能。

5.2.1.1创建多级嵌套表存储

您可以创建嵌套表的嵌套表。

例5-10创建了多级集合类型nt_country_typ ，这是嵌套表的嵌套表。该示例为公司区域系统建模，其中每个区域都有一个国家的嵌套表集合，每个国家/地区都有一个嵌套的表位集合。这个例子需要regions ， countries和locations对甲骨文的表HR示例模式。

在示例5-10中 ，SQL语句创建表region_tab ，其中包含列countries ，其类型为多级集合nt_country_typ 。此多级集合是具有嵌套表属性locations的对象类型的嵌套表。为外部countries嵌套表和内部locations嵌套表提供单独的嵌套表子句。

在示例5-10中，您可以按名称引用内部嵌套表locations ，因为此嵌套表是对象的命名属性。但是， 如果内部嵌套表不是对象的属性，则它没有名称 。为此案例提供了关键字COLUMN_VALUE 。见例5-11

示例5-10多级嵌套表存储

- 需要HR示例模式CREATE TYPE location_typ AS OBJECT（location_id NUMBER（4），street_address VARCHAR2（40），postal_code VARCHAR2（12），city VARCHAR2（30），state_province VARCHAR2（25））; / CREATE TYPE nt_location_typ AS TABLE of location_typ; - 嵌套表类型/ CREATE TYPE country_typ AS OBJECT（country_id CHAR（2），country_name VARCHAR2（40）， locations nt_location_typ）; - 内部嵌套表/ CREATE TYPE nt_country_typ AS TABLE OF country_typ; - 多级集合类型/ CREATE TABLE region_tab（region_id NUMBER，region_name VARCHAR2（25）， countries nt_country_typ） - 外部嵌套表NESTED TABLE countries STORE AS nt_countries_tab（ NESTED TABLE locations STORE AS nt_locations_tab）;

5.2.5多级集合的构造函数

               

                  
通过调用相应类型的构造函数来创建多级集合类型，就像单级集合和其他对象类型一样。

                  
多级集合类型的构造函数是系统定义的函数，其名称与类型相同，并返回它的新实例。构造函数参数具有对象类型的属性的名称和类型。

                  
例5-15显示了多级集合类型nt_country_typ的构造函数调用。nt_country_typ构造函数调用country_typ构造函数，该构造函数调用nt_location_typ ，后者调用location_typ构造函数。
                  

                  

                     
注意：

                     
 nt_country_typ是一个多级集合，因为它是一个嵌套表，其中包含另一个嵌套表作为属性。
