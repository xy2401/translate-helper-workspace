3.71 ALL_SEQUCES

ALL_SEQUENCES描述当前用户可访问的所有序列。

相关观点

  • DBA_SEQUENCES描述数据库中的所有序列。

  • USER_SEQUENCES描述当前用户拥有的所有序列。此视图不显示SEQUENCE_OWNER列。

柱 数据类型 空值 描述

SEQUENCE_OWNER

VARCHAR2（128）

不是NULL

序列的所有者

SEQUENCE_NAME

VARCHAR2（128）

不是NULL

序列名称

MIN_VALUE

数

序列的最小值

MAX_VALUE

数

序列的最大值

INCREMENT_BY

数

不是NULL

序列递增的值

CYCLE_FLAG

VARCHAR2（1）

指示序列是否在达到极限（ Y ）时回绕（ N ）

ORDER_FLAG

VARCHAR2（1）

指示是否按顺序（ Y ）生成序列号（ N ）

CACHE_SIZE

数

不是NULL

要缓存的序列号的数量

LAST_NUMBER

数

不是NULL

写入磁盘的最后序列号。如果序列使用缓存，则写入磁盘的编号是序列缓存中的最后一个编号。此数字可能大于使用的最后一个序列号。

对于会话序列，应忽略此列中的值。

SCALE_FLAG

VARCHAR2（1）

指示这是否是可伸缩序列（ Y ）或不是（ N ）

EXTEND_FLAG

VARCHAR2（1）

指示此可伸缩序列的生成值是否超出MAX_VALUE或MIN_VALUE （ Y ）或不是（ N ）

SHARDED_FLAG

脚1

VARCHAR2（1）

指示这是否是分片序列（ Y ）或不是（ N ）

SESSION_FLAG

VARCHAR2（1）

指示序列值是会话专用（ Y ）还是不是（ N ）

KEEP_VALUE

VARCHAR2（1）

指示在失败（ Y ）之后是否在重放期间保留序列值（ N ）

脚注1此列从Oracle Database 19c版本19.1开始提供。
