将Oracle Clusterware降级到12c版本2（12.2）

在升级成功或失败后，将所有Oracle Grid Infrastructure群集类型降级到12c版本2（12.2）的降级程序。

  • 将Oracle独立群集降级到12c版本2（12.2）
    成功升级后，使用此过程将Oracle独立群集降级到Oracle Grid Infrastructure 12c第2版（12.2）。
  • 将Oracle域服务集群降级到12c版本2（12.2）
    成功升级后，使用此过程将Oracle域服务集群降级到Oracle Grid Infrastructure 12c第2版（12.2）。
  • 将Oracle成员集群降级到12c版本2（12.2）
    成功升级后，使用此过程将Oracle成员集群降级到Oracle Grid Infrastructure 12c第2版（12.2）。
  • 升级失败时将Oracle Grid Infrastructure降级到12c版本2（12.2）
    如果在CVU升级后检查成功之前升级Oracle Grid Infrastructure失败，则可以运行gridSetup.sh并将Oracle Grid Infrastructure降级到早期版本。
