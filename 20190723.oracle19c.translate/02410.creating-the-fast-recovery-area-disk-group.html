创建快速恢复区磁盘组

创建快速恢复区磁盘组的过程。

  1. 转至Oracle Grid Infrastructure主目录bin目录，然后启动Oracle ASM Configuration Assistant（ASMCA）。

    例如：

    $ cd /u01/app/19.0.0/grid / bin $ ./asmca

    ASMCA打开主窗口。

  2. 点击左侧面板中的磁盘组打开磁盘组选项卡。
  3. 单击“ 创建”以创建新磁盘组。
    将打开“ 创建磁盘组”窗口。
  4. 根据提示提供快速恢复区域的配置信息：

    在“磁盘组名称”字段中，输入快速恢复区域组的描述性名称。例如： FRA 。

    在“冗余”部分中，选择要使用的冗余级别。例如： 正常

    在“选择成员磁盘”字段中，选择要添加到快速恢复区域的符合条件的磁盘，然后单击“ 确定” 。

  5. 快速恢复区磁盘组创建完成后，单击“ 退出”并单击“ 是”以确认关闭ASMCA应用程序。
