配置和管理工具管理Oracle Net Services的加密，完整性（校验和）和强身份验证方法。

关于Oracle Net Manager

Oracle Net Manager为本地客户端或服务器主机上的Oracle主目录配置Oracle Net Services。

虽然您可以使用Oracle Net Manager（用于配置Oracle Net Services的图形用户界面工具，例如命名，侦听器和常规网络设置），但它还允许您配置以下使用Oracle Net协议的功能：

  • 强身份验证（Kerberos，RADIUS和安全套接字层）

  • 本地网络加密（RC4，DES，Triple-DES和AES）

  • 校验和数据完整性（MD5，SHA-1，SHA-2）

关于Oracle Wallet Manager

电子钱包所有者和安全管理员使用Oracle Wallet Manager来管理和编辑其Oracle钱包中的安全凭证。

钱包是受密码保护的容器，用于存储身份验证和签名凭据，包括SSL所需的私钥，证书和可信证书。您可以使用Oracle Wallet Manager执行以下任务：

  • 创建公钥和私钥对

  • 存储和管理用户凭据

  • 生成证书请求

  • 存储和管理证书颁发机构证书（ 根密钥证书和证书链 ）

  • 从LDAP目录上载和下载钱包

  • 创建钱包以存储硬件安全模块凭据

注意：

在以前版本的Oracle数据库中，您可以使用Oracle Wallet Manager为透明数据加密配置钱包。在此版本中，您可以使用ADMINISTER KEY MANAGEMENT SQL语句。有关更多信息，请参见Oracle数据库高级安全性指南 。

父主题： 公钥基础结构凭据管理工具

关于orapki实用程序

orapki实用程序管理证书吊销列表（CRL），创建和管理Oracle钱包，并创建签名证书。

此命令行实用程序的基本语法如下：

orapki module command -option_1 argument ... -option_n argument

例如，以下命令列出了安装在machine1.us.example.com并使用端口389的Oracle Internet Directory实例中CRL子树中的所有CRL ：

orapki crl list -ldap machine1.us.example.com:389

注意：

使用orapki配置透明数据加密已被弃用。而是使用ADMINISTER KEY MANAGEMENT SQL语句。

也可以看看：

  • 证书吊销列表管理有关如何使用orapki管理目录中的CRL的信息

  • 管理公钥基础结构（PKI）元素，以获取有关所有可用orapki命令的参考信息

  • 有关使用ADMINISTER KEY MANAGEMENT SQL语句的信息，请参见“Oracle数据库高级安全性指南”
