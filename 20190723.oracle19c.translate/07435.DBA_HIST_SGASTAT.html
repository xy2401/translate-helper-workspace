5.54 DBA_HIST_SGASTAT

DBA_HIST_SGASTAT显示有关系统全局区域（SGA）的详细历史信息。

该视图包含V$SGASTAT快照。

柱 数据类型 空值 描述

SNAP_ID

数

不是NULL

唯一快照ID

DBID

数

不是NULL

快照的数据库ID

INSTANCE_NUMBER

数

不是NULL

快照的实例编号

名称

VARCHAR2（64）

SGA组件组

POOL

VARCHAR2（12）

指定NAME中的内存所在的池：

  • shared pool - 从共享池分配内存

  • large pool - 从大型游泳池分配内存

  • java pool - 从Java池分配内存

  • streams pool - 内存从Streams池分配

BYTES

数

内存大小（以字节为单位）

CON_DBID

数

采样会话的PDB的数据库ID

CON_ID

数

CON_DBID标识的容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

也可以看看：

“ V $ SGASTAT ”
