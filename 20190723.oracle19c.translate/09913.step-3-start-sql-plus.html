第3步：启动SQL * Plus

启动SQL * Plus。

  1. 执行以下操作之一：

    • 确保PATH环境变量包含ORACLE_HOME / bin。

    • 将目录更改为ORACLE_HOME / bin。确保PATH环境变量包含点（“。”）。

  2. 输入以下命令（在UNIX和Linux上区分大小写）：
    sqlplus / nolog

    您还可以通过指定其完整路径来运行sqlplus命令：

    ORACLE_HOME / bin / sqlplus / nolog
