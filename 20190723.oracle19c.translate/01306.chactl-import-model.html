D.13 chactl导入模型

使用chactl import model命令导入Oracle Cluster Health Advisor模型。

句法

chactl导入模型-name model_name -file model_file [-force] [-help]

导入时，如果存在与要导入的模型同名的现有模型，则使用-force选项进行覆盖。

实施例D-1实施例

$ chactl import model -name weekday -file /tmp//weekday.mod
