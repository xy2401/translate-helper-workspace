1.75 DB_CREATE_ONLINE_LOG_DEST_ n

DB_CREATE_ONLINE_LOG_DEST_ n （其中n = 1,2,3，... 5）指定Oracle管理的控制文件和联机重做日志的默认位置。

属性 描述

参数类型

串

句法

DB_CREATE_ONLINE_LOG_DEST_[1 | 2 | 3 | 4 | 5] = directory | disk group

默认值

没有默认值。

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

是

如果指定了多个DB_CREATE_ONLINE_LOG_DEST_ n参数，则控制文件或联机重做日志将跨多个其他DB_CREATE_ONLINE_LOG_DEST_ n参数的位置进行多路复用。在每个位置创建每个联机重做日志的一个成员，并在每个位置创建一个控制文件。

如果其中一个位置失败，则指定至少两个参数可为控制文件和联机重做日志提供更高的容错能力。

如果将文件系统目录指定为默认位置，则该目录必须已存在; Oracle不会创建它。该目录必须具有允许Oracle在其中创建文件的适当权限。Oracle为文件生成唯一的名称，因此创建的文件是Oracle托管文件。
