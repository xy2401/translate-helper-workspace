13.31流类

您使用Stream来读取或写入流数据（通常为LONG ）。

  • 可读Stream用于从结果集获取流数据或从存储过程调用获取OUT绑定变量。必须完全读取可读Stream直到达到数据结束，或者应关闭它以丢弃任何不需要的数据。

  • 可写入Stream用于向参数化语句（包括可调用语句）提供流数据（通常为LONG ）。

表13-45 Stream类使用的枚举值

属性 选项 
状态

  • READY_FOR_READ表示Stream已准备好进行读取操作

  • READY_FOR_WRITE表示Stream已准备好进行写操作

  • INACTIVE表示Stream不可用于就绪或写入操作

13.31.1 readBuffer（）

从Stream读取数据。size参数指定要读取的最大字节字符数。返回从Stream对象读取的数据量。 -1表示流上的数据结束。

句法

virtual int readBuffer（char * buffer，unsigned int size）= 0;
参数 描述 
缓冲

指向数据缓冲区的指针;必须由调用者分配和释放。

 
尺寸

指定要读取的字节数。

13.31.2 readLastBuffer（）

从Stream读取最后一个缓冲区。它也可以称为顶级丢弃未读数据。size参数指定要读取的最大字节字符数。返回从Stream对象读取的数据量; -1表示流上的数据结束。

句法

virtual int readLastBuffer（char * buffer，unsigned int size）= 0;
参数 描述 
缓冲

指向数据缓冲区的指针;必须由调用者分配和释放。

 
尺寸

指定要读取的字节数。

13.31.3 writeBuffer（）

将数据从缓冲区写入流。数据量由size决定。

句法

virtual void writeBuffer（char * buffer，unsigned int size）= 0;
参数 描述 
缓冲

指向数据缓冲区的指针。

 
尺寸

指定要写入的chars数。

13.31.4 writeLastBuffer（）

此方法将最后一个数据缓冲区写入流。它也可以被调用来写入最后一块数据。写入的数据量由大小决定。

句法

virtual void writeLastBuffer（char * buffer，unsigned int size）= 0;
参数 描述 
缓冲

指向数据缓冲区的指针。

 
尺寸

指定要写入的字节数。
