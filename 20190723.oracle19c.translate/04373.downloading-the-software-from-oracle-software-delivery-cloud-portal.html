从Oracle Software Delivery Cloud Portal下载软件

您可以从Oracle Software Delivery Cloud下载软件作为媒体包。

Media Pack是该软件的电子版本，也可通过CD-ROM或DVD供Oracle客户使用。
  1. 使用任何浏览器访问Oracle Software Delivery Cloud门户：
    https://edelivery.oracle.com/
  2. 通过在线表单输入信息（姓名，公司，电子邮件地址和国家/地区）来完成导出验证过程。
  3. 在“介质包搜索”页面中，指定产品包和平台以标识要下载的介质包。如果您不知道产品包的名称，则可以使用许可证列表进行搜索。
  4. （可选）从“ 结果”列表中选择要下载的相关产品。
  5. 在搜索结果页面中，单击自述文件以下载并查看自述文件以获取下载说明和产品信息。
  6. 查看自述文件后，从搜索结果中选择要下载的媒体包，以下载媒体包的各个zip文件，然后按照此页面中的“ 下载说明”说明进行操作。下载并解压缩zip文件的内容后，即可安装该软件。

    注意：

    使用可下载文件列表打印页面。它包含一个部件号列表及其在安装过程中可能引用的相应描述。
  7. 下载文件后，单击“ 查看摘要”以验证MD5或SHA-1校验和是否与媒体下载页面上列出的值匹配。
