前言

Oracle OLAP Java API开发人员指南将Java程序员介绍给Oracle OLAP Java API，它是Oracle OLAP的Java应用程序编程接口。通过Oracle OLAP，OLAP Java API可以访问存储在Oracle数据库中的数据，尤其是分析工作区中的数据。用于创建和维护分析工作区以及查询，操作和呈现数据的OLAP Java API功能特别适用于执行联机分析处理（OLAP）操作的应用程序。

前言包含以下主题：

  • 听众

  • 文档可访问性

听众

Oracle OLAP Java API开发人员指南适用于负责创建执行以下一项或多项操作的应用程序的Java程序员：

  • 实现Oracle OLAP元数据模型。

  • 定义，构建和维护分析工作区。

  • 使用Oracle OLAP执行分析。

要使用本手册，您应熟悉Java，关系数据库管理系统，数据仓库，OLAP概念和Oracle OLAP。
