识别Oracle软件所有者用户帐户

首次在系统上安装Oracle软件时，必须至少创建一个软件所有者用户帐户。使用现有的Oracle软件用户帐户，或为您的安装创建Oracle软件所有者用户帐户。

要使用现有用户帐户，请从系统管理员处获取现有Oracle安装所有者的名称。确认现有所有者是Oracle Inventory组的成员。
例如，如果您知道Oracle Inventory组的名称是oinstall ，则应将Oracle软件所有者列为oinstall的成员。
$ grep“oinstall”/ etc / group oinstall：x：54321：oracle
然后，您可以使用ID命令验证要使用的Oracle安装所有者是否将Oracle Inventory组作为其主要组。例如： $ id oracle
uid = 54321（oracle）gid = 54321（oinstall）groups = 54321（oper），54322（dba）

创建操作系统组后，请根据操作系统身份验证规划创建或修改Oracle用户帐户。
