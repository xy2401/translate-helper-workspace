C OLAP_API_SESSION_INIT PL / SQL包

OLAP_API_SESSION_INIT PL / SQL包提供了维护OLAP API的初始化参数表的过程。

本章包含以下主题：

  • OLAP API的初始化参数

  • 查看配置表

  • OLAP_API_SESSION_INIT子程序的摘要

C.1 OLAP API的初始化参数

OLAP_API_SESSION_INIT包中包含用于维护初始化参数的配置表的过程。当OLAP API打开会话时，它会为具有指定角色的任何用户执行表中列出的ALTER SESSION命令。只有OLAP API使用此表;没有其他类型的应用程序执行存储在其中的命令。

此功能提供了在数据库初始化文件或init.ora文件中设置这些参数的替代方法，这将改变所有用户的环境。

在安装过程中，将使用ALTER SESSION命令填充该表，这些命令已显示为可增强OLAP API的性能。除非新设置更有益，否则您无需对表进行更改。

可以通过ALL_OLAP_ALTER_SESSION视图别名查询表中的信息，本章中也对此进行了描述。

注意：

该软件包由SYS用户拥有。必须先明确授予执行权限，然后才能使用它。

C.3.1 ADD_ALTER_SESSION过程

此过程为具有特定数据库角色的OLAP API用户指定ALTER SESSION参数。它向OLAP$ALTER_SESSION表添加一行。

句法

ADD_ALTER_SESSION（role_name IN VARCHAR2，session_parameter IN VARCHAR2）;

参数

role_name和session_parameter在OLAP$ALTER_SESSION作为一行添加。

表C-3 ADD_ALTER_SESSION过程参数

session_parameter

例

以下示例在OLAP$ALTER_SESSION中插入一行，该OLAP_DBA具有OLAP_DBA角色的用户启用查询重写。 

执行olap_api_session_init.add_alter_session（'OLAP_DBA'， 'SET QUERY_REWRITE_ENABLED = TRUE'）;行插入SELECT * FROM all_olap_alter_session WHERE role ='OLAP_DBA'; ROLE CLAUSE_TEXT ------------ ------------------------------------ ------------------------ OLAP_DBA ALTER SESSION SET QUERY_REWRITE_ENABLED = TRUE

C.3.2 CLEAN_ALTER_SESSION程序

此过程将删除当前未在数据库中定义的任何角色的所有ALTER SESSION参数。它会删除OLAP$ALTER_SESSION表中针对这些角色的所有孤立行。

句法

CLEAN_ALTER_SESSION（）;

例

以下示例删除所有孤立的行。

执行olap_api_session_init.clean_alter_session（）;

C.3.3 DELETE_ALTER_SESSION过程

此过程为具有特定数据库角色的OLAP API用户删除以前定义的ALTER SESSION参数。它从OLAP$ALTER_SESSION表中删除一行。

句法

DELETE_ALTER_SESSION（role_name IN VARCHAR2，session_parameter IN VARCHAR2）;

参数

role_name和session_parameter一起唯一标识OLAP$ALTER_SESSION的行。
