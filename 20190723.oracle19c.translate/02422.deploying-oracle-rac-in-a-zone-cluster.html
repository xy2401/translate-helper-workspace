在区域群集中部署Oracle RAC

在此部署方案中，您可以在区域群集中安装Oracle Real Application Clusters（Oracle RAC）。区域群集是Oracle Solaris非全局区域的群集。

  1. 按照以下链接中的说明规划和创建区域群集：

    • 规划Oracle Solaris Cluster配置

    • 创建区域群集

  2. 以root身份，在区域群集中创建Oracle RAC基础结构，如下所示：
    1. 从全局群集节点运行clsetup实用程序，然后按照向导中的步骤操作。
    2. 从全局群集节点，验证Oracle RAC框架资源：
      #clresource status -Z zone_name
  3. 准备环境，然后安装和配置Oracle Grid Infrastructure和Oracle数据库。
    有关配置用户，组和环境的信息，请参阅为Oracle Grid Infrastructure和Oracle数据库配置用户，组和环境。
    有关安装Oracle Grid Infrastructure的信息，请参阅安装Oracle Grid Infrastructure。
    有关安装Oracle RAC数据库的信息，请参阅“ Oracle Real Application Clusters安装指南” 。
  4. 创建Oracle Solaris Cluster资源，链接它们并将它们联机，如配置Oracle RAC数据库实例的资源中所述。
