运行RPM软件包以安装Oracle数据库

执行以下步骤以使用RPM软件包安装和配置Oracle数据库。

通过ULN支持安装Oracle数据库RPM

  1. 以root身份登录。

  2. 如果您使用的是Oracle Linux并订阅了Unbreakable Linux Network（ULN），则可以通过单个yum命令安装Oracle数据库。

    #yum -y install oracle-database-ee-19c

    此命令将下载并安装Oracle Preinstallation RPM和Oracle Database RPM软件包。

    现在已完成Oracle数据库软件的安装，继续“创建和配置Oracle数据库”部分。

手动安装Oracle Database RPM

  1. 以root身份登录。

  2. 下载并安装Oracle预安装RPM：

    1. 在Oracle Linux上

      #yum -y install oracle-database-preinstall-19c

    2. 在Red Hat Enterprise Linux上

      #curl -o oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm https://yum.oracle.com/repo/OracleLinux/OL7/latest/x86_64/getPackage/oracle-database-preinstall-19c -1.0-1.el7.x86_64.rpm #yum -y localinstall oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

      成功安装后，您可以删除下载的RPM文件：

      #rm oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

  3. 从以下位置访问基于Oracle Database RPM的安装的软件下载页面：

    • Oracle技术网

      http://www.oracle.com/technetwork/indexes/downloads/index.html

    • 坚不可摧的Linux网络（ULN）

      https://linux.oracle.com/

  4. 将执行基于RPM的安装所需的.rpm文件下载到您选择的目录中。例如，将oracle-database-ee-19c-1.0-1.x86_64.rpm文件下载到/tmp目录。

  5. 使用yum localinstall命令安装数据库软件。 

    #cd / tmp＃yum -y localinstall oracle-database-ee-19c-1.0-1.x86_64.rpm

    其中， oracle-database-ee-19c-1.0-1.x86_64.rpm是Oracle Database RPM的完全限定名称。

Oracle数据库软件的安装现已完成。

创建和配置Oracle数据库

要使用默认设置创建示例数据库，请执行以下步骤：

  1. 以root.身份登录root.

  2. 要配置示例Oracle数据库实例，请运行以下服务配置脚本：

    #etet/init.d/oracledb_ORCLCDB-19c configure

    注意：

    您可以通过编辑/etc/sysconfig/oracledb_ORCLCDB-19c.conf文件来修改配置参数。

    此脚本使用一个可插拔数据库（ ORCLPDB1 ）创建容器数据库（ ORCLCDB ），并在默认端口（1521）配置侦听器。

    查看屏幕上显示的状态信息。
