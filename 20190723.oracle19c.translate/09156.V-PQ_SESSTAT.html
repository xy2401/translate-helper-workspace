8.123 V $ PQ_SESSTAT

V$PQ_SESSTAT列出并行查询的会话统计信息。 运行查询或DML操作后，可以使用从V$PQ_SESSTAT派生的信息来查看所使用的从进程数以及会话和系统的其他信息。
柱 数据类型 描述

统计

VARCHAR2（30）

统计名称：

  • Queries Parallelized - Queries Parallelized运行的查询数

  • DDL Parallelized - DDL Parallelized运行的DDL操作数

  • DML Parallelized - DML Parallelized运行的DML操作数

  • DFO Trees - 执行的DFO树的数量

  • DOP - 用于最后一个语句的并行度

  • Server Threads数 - 使用的PX服务器数

  • Allocation Height - 每个实例请求的服务器数量

  • Allocation Width - 请求的实例数

  • Local Msgs Sent - 已Local Msgs Sent的本地（实例内）消息数

  • Distr Msgs Sent - Distr Msgs Sent的远程（实例间）消息数

  • Local Msgs Recv'd - 收到的本地（实例内）消息数

  • Distr Msgs Recv'd - 收到的远程（实例间）消息数

  • Slave Sets - 用于最后一个语句的从设备总数。对于并行计划中的多个并行程序，它是累积计数。

LAST_QUERY

数

上次操作的统计值

SESSION_TOTAL

数

整个会话的统计值到此时的值。

对于DOP ， Server Threads ， Allocation Height ， Allocation Width和Slave Set统计信息，不维护整个会话到此时的统计值。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
