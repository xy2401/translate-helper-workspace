升级后更新企业管理器代理基本目录

升级Oracle Grid Infrastructure后，在运行代理的每个集群成员节点上更新企业管理器代理基本目录。

Agent Base目录是创建Management Agent主目录的目录。Management Agent主目录位于路径Agent_Base_Directory / core / EMAgent_Version中 。例如，如果Agent Base目录是/u01/app/emagent ，则管理代理主目录创建为/u01/app/emagent/core/13.1.1.0 。

  1. 导航到Management Agent主目录中的bin目录。

  2. 在/u01/app/emagent/core/13.1.1.0/bin目录中，使用文本编辑器打开文件emctl 。

  3. 找到参数CRS_HOME ，并将参数更新为新的Grid主路径。

  4. 使用企业管理器代理在群集的每个节点上重复步骤1-3。
