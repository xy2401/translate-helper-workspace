XMLPATCH

句法


说明xmlpatch.eps的描述

目的

XMLPatch函数是XmlPatch C API的SQL接口。此函数使用指定的更改修补XML文档。返回修补的XMLType文档。

  • 对于第一个参数，请指定输入XMLType文档的名称。

  • 对于第二个参数，请指定包含要应用于第一个文档的更改的XMLType文档。更改应符合Xdiff XML架构。您可以从Oracle XML Developer的Kit Java方法diff()提供XML输出。

也可以看看：

Oracle XML Developer's Kit程序员指南 ，了解有关使用此函数的更多信息，包括示例和Oracle Database XML C API参考，以获取有关C的XML API的信息
