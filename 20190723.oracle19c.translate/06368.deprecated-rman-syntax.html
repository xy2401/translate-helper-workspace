B不推荐使用的RMAN语法

本附录描述了不推荐使用的Recovery Manager语法，并描述了首选语法（如果存在）。

为了向后兼容，后续版本继续支持不推荐使用的RMAN语法。例如， SET AUXNAME命令替换了Oracle8 i中的SET CLONENAME命令， CONFIGURE AUXNAME命令替换了Oracle9 i中的SET AUXNAME命令。但是，您可以继续在所有后续RMAN版本中运行SET CLONENAME和SET AUXNAME 。
