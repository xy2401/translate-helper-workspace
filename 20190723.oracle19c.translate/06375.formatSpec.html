4.12 formatSpec

目的

使用formatSpec子条款为备份片或映像副本指定文件名格式或自动存储管理磁盘组。如果没有为FORMAT参数指定值，则RMAN会在快速恢复区域中创建备份（如果已启用），或者在特定于平台的目录中创建（例如， ?/dbs UNIX上的/dbs ）如果未启用快速恢复区域。在任何一种情况下，RMAN都使用变量％U来命名备份。

小费：

Oracle数据库SQL语言参考 ，了解如何创建和命名自动存储管理器磁盘组

使用说明

只要每个备份片或副本具有唯一名称，就允许在平台上作为顺序文件名有效的任何名称。如果备份到磁盘，则允许任何有效的磁盘文件名，前提是它是唯一的。

您不能将Oracle Managed Files文件名指定为备份的格式。例如，如果+DISK1/datafile/system.732.609791431是OMF文件名，则无法在FORMAT参数中指定此文件名。

环境变量在FORMAT参数中无效。

目标实例以特定于端口的方式处理整个format_string以派生最终的备份片段名称。“ 语义 ”中列出的替换变量在FORMAT字符串中可用，以帮助生成唯一的文件名。此信息的格式因平台而异。

您最多可以指定四个FORMAT字符串。仅当BACKUP COPIES ， SET BACKUP COPIES或CONFIGURE时，RMAN才使用第二个，第三个和第四个值...BACKUP COPIES生效。在为每个备份片段选择格式时，RMAN使用副本1的第一个格式值，副本2的第二个格式值，依此类推。如果格式值的数量超过副本数，则不使用额外的格式。如果格式值的数量小于副本数，则RMAN将重用格式值，从第一个开始。

在以下任何位置指定format_string ，按优先顺序列出：

  1. backupSpec子句

  2. BACKUP命令

  3. ALLOCATE CHANNEL命令

  4. CONFIGURE CHANNEL命令

如果在多个位置指定，则RMAN按所示顺序搜索FORMAT参数。

语义

formatSpec

下表列出了在格式字符串中有效的RMAN替换变量。

语法元素 描述

％一个

指定数据库的激活ID。

％b

指定剥离目录路径的文件名。它仅在生成映像副本时对SET NEWNAME和备份有效如果用作生成备份片段的备份的格式规范，则会产生错误。

％C

指定一组双工备份片段中备份片的副本号。如果未双工备份，则此变量对于备份集为1，对于代理副本为0。如果启用了命令，则变量显示副本编号。%c的最大值为256。

％d

指定数据库的名称（请参见示例4-23 ）。

％d

以格式DD从公历中指定月中的当前日期。

％E

指定存档的日志序列号。

％F

指定绝对文件编号（请参见例4-23 ）。

％F

将DBID，日，月，年和序列组合成唯一且可重复生成的名称。此变量转换为c-IIIIIIIIII-YYYYMMDD-QQ ，其中：

  • IIIIIIIIII代表DBID。 DBID以十进制格式打印，以便可以轻松地与目标数据库关联。

  • YYYYMMDD是生成备份当天的公历中的时间戳

  • QQ是以十六进制数表示的序列，以00开头，最大为'FF'（256）

注意 ： %F仅在CONFIGURE CONTROLFILE AUTOBACKUP FORMAT命令中有效。

％H

指定存档的重做日志线程编号。

％一世

指定DBID。

％M

以格式MM指定公历中的月份。

％N

指定表空间名称。此替换变量仅在将数据文件备份为映像副本时有效。

％N

指定数据库的名称，在右侧填充x字符，总长度为8个字符。例如，如果prod1是数据库名称，则填充名称为prod1xxx 。

％p

指定备份集中的件号。对于每个备份集，此值从1开始，并在创建每个备份片时递增1。

注意：如果指定PROXY ，则%p变量必须在％U中显式或隐式包含在FORMAT字符串中。

％S

指定备份集编号。此编号是控制文件中的计数器，该计数器针对每个备份集递增。计数器值从1开始，并且在控制文件的生命周期内是唯一的。如果还原备份控制文件，则可能会导致重复的值。此外， CREATE CONTROLFILE将计数器初始化为1 。

％T

指定备份集时间戳，它是一个4字节值，派生为自固定参考时间以来经过的秒数。您可以使用%s和%t的组合来形成备份集的唯一名称。

％T

以这种格式指定公历中的年，月和日： YYYYMMDD 。

％u

指定由备份集或映像副本号的压缩表示以及创建备份集或映像副本的时间构成的8个字符的名称。

％U

指定系统生成的唯一文件名（默认）。

%U的含义对于图像副本和备份片段是不同的。对于备份片段， %U指定%u_%p_%c的便捷速记，以保证生成的备份文件名中的唯一性。对于数据文件的映像副本， %U表示以下内容：

数据-D-％d_id-％I_TS-％N_FNO-％F_％U

对于归档重做日志的映像副本， %U表示以下内容：

拱D_％d-ID-％I_S-％e_T-％h_A-％A_％U

对于控制文件的映像副本， %U表示以下内容：

CF-D_％d-ID-％I_％U

％Y

以此格式指定年份： YYYY 。

%%

指定百分比（ % ）字符。例如， %%Y转换为字符串%Y 。
