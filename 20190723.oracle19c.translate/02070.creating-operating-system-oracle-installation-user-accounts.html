创建操作系统Oracle安装用户帐户

在开始安装之前，请创建Oracle软件所有者用户帐户，并配置其环境。

Oracle软件所有者用户帐户需要资源设置和其他环境配置。为防止发生意外，Oracle建议您为安装的每个Oracle软件程序创建一个软件安装所有者帐户。

  • 创建Oracle软件所有者用户
    如果Oracle软件所有者用户（ oracle或grid ）不存在，或者您需要新的Oracle软件所有者用户，请按照本节中的说明创建它。
  • 修改Oracle所有者用户组
    如果您已创建Oracle软件安装所有者帐户，但它不是要指定为OSDBA，OSOPER，OSDBA for ASM，ASMADMIN或其他系统特权组的组的成员，则修改该用户的组设置在安装之前。
  • 识别现有用户和组ID
    要创建相同的用户和组，必须在创建它们的节点上标识分配它们的用户标识和组ID，然后在其他集群节点上创建具有相同名称和ID的用户和组。
  • 在其他群集节点上创建相同的数据库用户和组
    Oracle软件所有者用户和Oracle Inventory，OSDBA和OSOPER组必须存在且在所有群集节点上都相同。
