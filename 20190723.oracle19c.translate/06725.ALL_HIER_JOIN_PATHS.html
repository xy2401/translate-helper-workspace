2.158 ALL_HIER_JOIN_PATHS

ALL_HIER_JOIN_PATHS描述了当前用户可访问的所有层次结构的连接路径。

相关观点

  • DBA_HIER_JOIN_PATHS描述数据库中的所有层次结构连接路径。

  • USER_HIER_JOIN_PATHS描述当前用户架构中所有层次结构的连接路径。此视图不显示OWNER列。

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
