说明disk_details.gif的说明

将显示“磁盘详细信息”页面。

屏幕截图的右上角是标签值对。标签是从目标收集的最新数据，价值是美国中部标准时间2012年1月6日下午1:05:56。值旁边是刷新按钮。

标签 - 值对下面是查看数据列表，显示实时：手动刷新。

此页面包含三个部分：所有磁盘的磁盘I / O总数，所有磁盘和顶部磁盘设备的最大平均磁盘I / O服务时间（毫秒）（按％忙排序）。

“在所有磁盘上制作的磁盘I / O总量”图表的y轴为0到1500，x轴为11:46 AM到1:00 pm。该图表测量所有磁盘上的每秒总I / O.该图表显示了一条参差不齐的线条，从11:46左右的大约100到大约12:17的800，到12:50的1000，然后在1:10下降到200。x轴下方是2012年1月6日。日期下方是一个链接：所有磁盘上的磁盘I / O总量。图表下方是标签值对。标签是每秒在所有磁盘上进行的总I / O，值为153.07。

所有磁盘中的最大平均磁盘I / O服务时间（毫秒）图表的y轴为0到7.5，x轴为12:16到1:06。该图表测量所有磁盘中的最大平均磁盘I / O服务时间（以毫秒为单位）。该图表显示了一条参差不齐的线，从12:16的2到12:35的6左右，然后是1:06的2。x轴下方是2012年1月6日。日期下方是一个链接：最大平均磁盘I / O服务时间（毫秒）之间。图表下方是标签值对。标签是所有磁盘中的最大平均磁盘I / O服务时间（毫秒），值为1.79。

顶部磁盘设备（按％繁忙排序）部分包括一个表。表的列包括：状态，设备，忙（％），平均未完成请求，每秒读取数，每秒写入数，块读取数（每秒512字节）。在该页面的屏幕截图中看不到的其他列是：写入的块（每秒512字节，平均等待时间（ms），平均服务时间（ms））。

表中可以看到两行。列中的值如下：

第1行:(复选标记图标），xvda，12.14,19.88,31.11,45.6,907.08。

第2行:(复选标记图标），xvda1,10.93,21.4,27.96,32.12,774.04。
