7为XML C API打包SAX

SAX是基于事件的XML解析的标准接口，由XML-DEV邮件列表的成员协作开发。要使用SAX， xmlsaxcb结构将使用函数指针进行初始化，并传递给其中一个XmlLoadSax调用。还提供了指向用户定义的上下文结构的指针，并将其传递给每个SAX函数。

对于基于事件的模式验证API，请参阅C的Package Event API 。

下表总结了通过SAX接口为XML C API提供的方法。

XmlSaxXmlDecl（）

此事件标记XML声明。XmlSaxStartDocument事件始终是第一个;如果注册了这个回调并且存在XMLDecl ，那么它将是第二个事件。编码标志表示是否指定了编码。由于文档自己的编码规范可能被覆盖（或错误），并且输入将被转换为数据编码，因此不提供文档中指定的实际编码。对于独立标志，如果未指定，则返回-1 ，否则返回0表示FALSE ，1表示TRUE 。
