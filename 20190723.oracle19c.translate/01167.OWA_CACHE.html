213.1 OWA_CACHE常量

OWA_CACHE定义了在指定参数值时要使用的几个常量。

  • system_level CONSTANT VARCHAR(6) := 'SYSTEM';

  • user_level CONSTANT VARCHAR(4) := 'USER';

213.2.1禁用程序

此过程禁用此特定请求的缓存。

句法

OWA_CACHE.DISABLE;

213.2.2 GET_ETAG功能

此函数返回与缓存内容关联的标记。它仅用于验证技术。

句法

OWA_CACHE.GET_ETAG RETURN VARCHAR2;

返回值

缓存命中的标记，否则为NULL 。

213.2.3 GET_LEVEL函数

这将返回缓存级别。它仅用于验证技术模型。

句法

OWA_CACHE.GET_LEVEL RETURN VARCHAR2;

返回值

缓存命中的缓存级别字符串（' USER '或' SYSTEM '），否则为NULL 。

213.2.4 SET_CACHE过程

此过程为验证模型缓存类型设置缓存标头。

句法

OWA_CACHE.SET_CACHE（p_etag IN VARCHAR2，p_level IN VARCHAR2）;

参数

表213-2 SET_CACHE过程参数

参数 描述

p_etag

与此内容相关联的etag

p_level

缓存级别（' USER'或' SYSTEM' ）。

例外

如果，则抛出VALUE_ERROR

  • p_etag大于55

  • p_level不是' USER '或' SYSTEM '

213.2.5 SET_EXPIRES过程

此过程为expires模型缓存类型设置缓存标头。

句法

OWA_CACHE.SET_EXPIRES（p_expires IN NUMBER，p_level IN VARCHAR2）;

参数

表213-3 SET_EXPIRES过程参数

参数 描述

p_expires

此内容有效的分钟数。

p_level

缓存级别（' USER'或' SYSTEM' ）。

例外

如果，则抛出VALUE_ERROR

  • p_expires为负数或零

  • p_level不是' USER '或' SYSTEM '

  • p_expires > 525600（1年）

213.2.6 SET_NOT_MODIFIED过程

此过程为未修改的缓存命中设置标头。它仅用于验证技术。

句法

OWA_CACHE.SET_NOT_MODIFIED;

例外

如果未传入etag ，则抛出VALUE_ERROR

213.2.7 SET_SURROGATE_CONTROL过程

此过程为Web缓存设置代理控制标头的标头

句法

OWA_CACHE.SET_SURROGATE_CONTROL（p_value IN VARCHAR2）;

参数

表213-4 SET_SURROGATE_CONTROL过程参数

参数 描述

P_VALUE

要作为Surrogate-Control标头传递的值。

例外

如果If p_value的长度大于55，则抛出VALUE_ERROR 。
