2.25主持人

目的

使用HOST命令从RMAN中调用操作系统命令行子shell。

句法

host :: =


插图描述host.eps

先决条件

在RMAN提示符下或在RUN命令的大括号内执行此命令。

语义

语法元素 描述

主办

显示命令提示符并在退出子shell后继续（请参阅示例2-104 ）。

    ' command '

在指定的字符串中运行该命令，然后继续（参见示例2-105 ）。
