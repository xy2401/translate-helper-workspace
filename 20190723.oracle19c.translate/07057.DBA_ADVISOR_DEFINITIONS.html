4.13 DBA_ADVISOR_DEFINITIONS

DBA_ADVISOR_DEFINITIONS显示数据库中所有顾问程序的属性。

视图包含每个任务的一行，表示任务的当前状态以及执行特定数据，例如进度监视和完成状态。

属性

数

不是NULL

属性：

  • 位0： -指示是否顾问在运行COMPREHENSIVE模式（ 1 ）或否（ 0 ）

  • 位1： -指示是否顾问在运行LIMITED模式（ 1 ）或否（ 0 ）

  • 位2： - 指示顾问是否可恢复（ 1 ）或不（ 0 ）

  • 位3： - 指示顾问程序是否接受用户指令（ 1 ）或不接受（ 0 ）
