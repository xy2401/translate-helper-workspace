16 CLSS-00001至CLSS-03202

CLSS-00001：skgxn未激活

原因： skgxn服务是必需的，但不是活动的

操作：验证集群件skgxn是否处于活动状态，并在必要时重新启动它

CLSS-00002：无法确定本地节点号。键入string

原因： skgxn服务不可用，或者OCR配置不完整/不准确。

操作：验证集群件skgxn是否处于活动状态，并在必要时重新启动它。此外，验证群集软件安装是否成功，以及是否在节点列表中输入了本地节点。

CLSS-00003：无法确定群集名称。键入string

原因：无法从配置中确定群集的名称。

操作：验证集群件安装是否成功以及OCR注册表是否可用。

CLSS-00004：无法分配内存资源

原因：内存分配例程失败。

操作：验证系统是否有足够的内存资源来使用该产品。

CLSS-00099：对于string ，错误string ，logfile打开失败

原因：无法打开日志文件，错误类型错误

操作：验证安装期间指定的日志目录是否存在，以及ocssd可执行文件是否有权在该目录中创建文件

CLSS-00100：密钥string配置数据访问失败

原因：密钥指示的配置数据访问失败，错误代码错误

操作：验证集群件安装是否成功以及是否可以访问OCR注册表

CLSS-00101：Oracle Cluster Repository与节点string不匹配。 （ string ！= string ）

原因：远程节点正在访问另一组配置。在存在活动集群节点时尝试重新安装可能会导致此情况。

操作：验证本地和远程节点是否正在访问同一Oracle Cluster Repository。

CLSS-00102：初始化失败：[ string ] [ string ] [ string ] [ string ] [ string ] [ string ]

原因：发生一般初始化失败。

操作：联系Oracle客户服务以获取更多信息。

CLSS-00103：无法读取节点string配置信息

原因：找到指定节点的配置信息不完整。

操作：验证Oracle Cluster Repository的完整性，并在必要时从备份中还原它。

CLSS-00200：配置中未列出的本地节点string

原因： OCR配置信息未将此节点列为群集的成员

操作：验证集群件安装是否成功以及安装后是否尚未添加节点

CLSS-01200：skgxn errror：类别string ，操作string ，loc string

原因： skgxn服务遇到错误，错误消息中的信息是供应商提供的诊断信息

操作：将此错误报告给提供skgxn的供应商

CLSS-02000：重新配置成功，具有string节点的化身string

原因：配置已成功完成

行动：没有

CLSS-02001：本地节点号string ，主节点号string

原因：有关此化身的信息性消息

行动：没有

CLSS-02200：接收活动节点string节点删除请求。

原因：用户尝试在关闭之前从配置中删除节点。

操作：关闭指定的节点并重试删除节点。

CLSS-02201：端点已在使用中： string

原因：指定的端点已在使用中。另一个过程是使用端点。

操作：关闭其他进程，然后重试启动尝试。

CLSS-02202：投票设备访问花费了过多的时间。 （ string秒）

原因：对投票设备的读取或写入花费了不可接受的时间。

操作：请与磁盘硬件供应商联系，以获取有关提高磁盘可用性的信息。

CLSS-02203：无法访问投票设备： string

原因： CSS守护程序使用投票设备收到故障。

操作：检查投票设备上的权限。如果此故障发生的时间晚于启动时间，则丢失了对投票设备的访问权限。

CLSS-02204：未配置投票设备。 [ string ] [ string ]

原因：配置期间未提供投票设备。

操作：投票设备有助于确保数据完整性，并且是CSS守护程序所必需的。配置投票设备并重新启动集群。

CLSS-02205：远程节点string通过投票设备通知我们的驱逐：原因string

原因：由于网络故障或本地CSS守护程序中的软件故障，群集的远程节点驱逐了本地节点。

操作：验证本地节点与指定远程节点之间的专用网络的稳定性。

CLSS-02206：供应商节点监视器逐出的本地节点

原因：操作系统供应商的节点监视器驱逐了本地节点。

操作：检查供应商节点监视器的日志以获取相关信息。

CLSS-02207：错误string从供应商节点监视器获取状态

原因： CSS守护程序收到与供应商节点监视器交互的故障。本地节点可能已被驱逐或供应商节点监视器可能已被关闭。

操作：检查供应商节点监视器的日志以获取相关信息。

CLSS-03000：重新配置成功，带string节点的化身string

原因：配置已成功完成

行动：没有

CLSS-03001：本地节点号string ，主节点号string

原因：有关此化身的信息性消息

行动：没有

CLSS-03200：等待嵌套重新配置超时

原因：当前重新配置挂起。网络故障可能已经发生。

操作：本地节点将自行终止，以使远程节点进入嵌套的重新配置。这可能允许群集继续运行。

CLSS-03201：尝试开始嵌套重新配置失败。

原因：无法取消先前的重新配置以开始新的重新配置

行动：没有

CLSS-03202：与master不同步：[ string ] [ string ] [ string ] [ string ]

原因：本地节点和主节点之间的一致性检查失败。

操作：本地守护程序将自行终止，以确保正常协调的所有群集都已正确协调。
