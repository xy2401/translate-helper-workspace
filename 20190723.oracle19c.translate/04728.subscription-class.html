13.32订阅类

订阅类封装了注册订阅者通知所需的信息和操作。

表13-47订阅类使用的枚举值

属性 选项 
介绍

  • PRES_DEFAULT表示事件通知应采用默认格式。

  • PRES_XML表示事件通知应为XML格式。 

协议

  • PROTO_CBK表示客户端通过默认系统协议接收通知。

  • PROTO_MAIL表示客户端通过电子邮件接收通知，如xyz@oracle.com 。数据库不检查电子邮件是否有效。

  • PROTO_SERVER表示客户端通过数据库中调用的PL / SQL过程接收通知，如schema.procedure 。订户必须具有该过程的适当权限。

  • PROTO_HTTP表示客户端通过HTTP URL接收通知，如http://www.oracle.com:80 。数据库不检查URL是否有效。 

命名空间

  • NS_ANONYMOUS表示注册是在匿名命名空间中进行的。

  • NS_AQ表示注册是在Oracle Database Advanced Queuing命名空间中进行的。

13.32.21 setSubscriptionName（）

设置订阅的名称。所有订阅都由订阅名称标识，订阅名称由指定长度的字节序列组成。

如果名称空间是NS_AQ ，则订阅名称为：

  • 在单个使用者队列上注册时的SCHEMA.QUEUE

  • SCHEMA.QUEUE:CONSUMER_NAME在多SCHEMA.QUEUE:CONSUMER_NAME队列中注册时的SCHEMA.QUEUE:CONSUMER_NAME

句法

void setSubscriptionName（const string＆name）;
参数 描述 
名称

订阅名称。
