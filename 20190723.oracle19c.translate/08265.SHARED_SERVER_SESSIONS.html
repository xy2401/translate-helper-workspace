1.314 SHARED_SERVER_SESSIONS

SHARED_SERVER_SESSIONS指定允许的共享服务器会话总数。

属性 描述

参数类型

整数

默认值

没有默认值。

可修改

改变系统

可在PDB中修改

没有

价值范围

如果指定了SHARED_SERVER_SESSIONS ，那么它应该小于SESSIONS 。如果未指定SHARED_SERVER_SESSIONS ，则只要存在空闲会话槽，就可以创建共享服务器会话。

基本

没有

通过设置此参数，您可以为专用服务器保留用户会话。
