Linux x86-64上的Oracle ACFS和Oracle ADVM支持

Oracle Linux，Red Hat Enterprise Linux和SUSE Linux Enterprise Server支持Oracle ACFS和Oracle ADVM。

表7-2支持Oracle ACFS和Oracle ADVM的平台

平台/操作系统 核心

Oracle Linux 7

所有与Oracle Linux兼容的内核

红帽企业Linux 7

所有与Red Hat Enterprise Linux兼容的内核

SUSE Linux Enterprise Server 12

所有SUSE Linux Enterprise Server 12兼容的内核

Linux容器

不支持

注意：

如果在Oracle ACFS的强制模式下使用安全增强型Linux（SELinux），请确保使用SELinux默认上下文安装Oracle ACFS文件系统。有关上下文装入选项的信息，请参阅Linux供应商文档。

重要：

您必须将修补程序应用于某些Linux内核版本才能成功安装Oracle Grid Infrastructure。有关更多信息，请参阅以下注释：

  • 有关更多信息，请参阅My Oracle Support说明1369107.1以及支持Oracle ACFS和Oracle ADVM的平台和发行版的完整列表：

    https://support.oracle.com/rs?type=doc&id=1369107.1

  • 针对当前版本和支持信息，Oracle产品的补丁集更新（My Oracle Support说明854428.1）：

    https://support.oracle.com/rs?type=doc&id=854428.1
