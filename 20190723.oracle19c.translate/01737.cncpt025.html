图像cncpt025.gif显示了银行交易示例的SQL语句。该声明分为四个部分：

  • 减少储蓄账户

  • 增量检查帐户

  • 在交易日志中记录

  • 结束交易

递减储蓄账户的SQL语句如下：

UPDATE savings_accounts SET balance = balance  -  500 WHERE account = 3209;

增量检查帐户的SQL语句如下：

UPDATE checking_accounts SET balance = balance + 500 WHERE account = 3208;

Transaction in Transaction Journal中的SQL语句如下：

INSERT INTO期刊VALUES（journal_seq。NEXTVAL，'1B'3209,3208,500），

End Transaction的SQL语句如下：

承诺工作;
