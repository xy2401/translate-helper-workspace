5.350 DBA_SEGMENTS_OLD

DBA_SEGMENTS_OLD列出了有关为所有数据库段分配的存储的信息。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

细分所有者的用户名

SEGMENT_NAME

VARCHAR2（128）

该段的名称（如果有）

PARTITION_NAME

VARCHAR2（128）

分区的名称

段类型

VARCHAR2（18）

细分类型： INDEX PARTITION, TABLE PARTITION, TABLE, CLUSTER, INDEX, ROLLBACK, DEFERRED ROLLBACK, TEMPORARY, CACHE, LOBSEGMENT和LOBINDEX

TABLESPACE_NAME

VARCHAR2（30）

包含段的表空间的名称

HEADER_FILE

数

包含段头的文件的ID

HEADER_BLOCK

数

包含段头的块的ID

BYTES

数

段的大小（以字节为单位）

积木

数

段的大小，在Oracle块中

EXTENTS

数

分配给段的区数

INITIAL_EXTENT

数

在创建时为段的初始范围请求的大小（以字节为单位）。（如果请求的大小超过5个块，Oracle会将范围大小舍入为5个块的倍数。）

NEXT_EXTENT

数

要分配给段的下一个范围的大小（以字节为单位）

MIN_EXTENTS

数

段中允许的最小区数

MAX_EXTENTS

数

段中允许的最大区数

PCT_INCREASE

数

增加下一个要分配的范围大小的百分比

FREELISTS

数

分配给该段的进程空闲列表的数量

FREELIST_GROUPS

数

分配给此段的空闲列表组的数量

RELATIVE_FNO

数

段头的相对文件号

BUFFER_POOL

VARCHAR2（7）

对象的缓冲池
