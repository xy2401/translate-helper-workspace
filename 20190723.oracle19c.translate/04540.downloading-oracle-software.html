下载Oracle软件

选择要用于下载软件的方法。

您可以从Oracle技术网（OTN）或Oracle Software Delivery Cloud门户下载安装文件的试用版，并将其解压缩到硬盘上。确保您查看并了解许可条款。

  • 从OTN下载安装存档文件
    从Oracle技术网（OTN）下载安装归档文件。
  • 从Oracle Software Delivery Cloud Portal下载软件
    您可以从Oracle Software Delivery Cloud下载软件作为媒体包。
