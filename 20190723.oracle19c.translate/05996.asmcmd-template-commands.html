ASMCMD模板管理命令

本主题提供ASMCMD模板管理命令的摘要。

表10-89列出了ASMCMD模板管理命令及其简要说明。

表10-89 ASMCMD模板管理命令摘要

命令 描述

chtmpl

更改模板的属性。

lstmpl

列出磁盘组模板。

mktmpl

将模板添加到磁盘组。

rmtmpl

从磁盘组中删除模板。

chtmpl

目的

更改模板的属性。

语法和说明

chtmpl -G diskgroup {[--striping {coarse | fine}] [--redundancy {high | mirror | unprotected}] [--primary {hot | cold}] [ -  secondary {hot | cold}]} template

表10-90列出了chtmpl命令的语法选项。

表10-90 chtmpl命令的选项

选项 描述

-G diskgroup

磁盘组的名称。

模板

要更改的模板的名称。

--striping { coarse | fine }

条带规格， coarse或fine 。

--redundancy { high | mirror | unprotected }

冗余规范， high ， mirror或unprotected 。

--primary { hot | cold

主要范围（ hot区或cold区）的数据放置规范。

--secondary { hot | cold

次要范围的数据放置规范，无论是hot区还是cold区。

至少需要以下选项之一：-- --striping ，-- --redundancy ， - --primary或--secondary 。

有关更新模板的信息，请参阅“ 修改磁盘组模板 ” 。

例

以下是更新data磁盘组的mytemplate模板的chtmpl命令mytemplate 。redundancy属性设置为high，条带属性设置为fine。

例10-93使用ASMCMD chtmpl命令

ASMCMD [+]> chtmpl -G data --redundancy high --striping fine mytemplate

lstmpl

目的

列出所有磁盘组的所有模板或指定磁盘组的模板。

语法和说明

lstmpl [--suppressheader] [-l] [-G diskgroup ] [ pattern ]

表10-91列出了lstmpl命令的语法选项。

表10-91 lstmpl命令的选项

选项 描述

-G diskgroup

指定磁盘组名称。

--suppressheader

禁止列标题。

-l

显示所有细节。

图案

显示与模式表达式匹配的模板。

有关模板属性的信息，请参阅“ 模板属性 ” 。

mktmpl

目的

将模板添加到磁盘组。

语法和说明

mktmpl -G diskgroup [--striping {coarse | fine}] [--redundancy {high | mirror | unprotected}] [--primary {hot | cold}] [ -  secondary {hot | cold}] template

表10-92列出了mktmpl命令的语法选项。

表10-92 mktmpl命令的选项

选项 描述

-G diskgroup

磁盘组的名称。

模板

要创建的模板的名称。

--striping { coarse | fine }

条带规格， coarse或fine 。

--redundancy { high | mirror | unprotected }

冗余规范， high ， mirror或unprotected 。

--primary { hot | cold

主要范围（ hot区或cold区）的数据放置规范。

--secondary { hot | cold

次要范围的数据放置规范，无论是hot区还是cold区。

有关添加模板的信息，请参阅将模板添加到磁盘组 。

例

以下是将mytemplate模板添加到data磁盘组的mktmpl命令mytemplate 。新模板的冗余设置为镜像，条带设置为粗略。

例10-95使用ASMCMD mktmpl命令

ASMCMD [+]> mktmpl -G data --redundancy mirror --striping coarse mytemplate
