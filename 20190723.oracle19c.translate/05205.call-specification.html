13.7呼叫规范

调用规范声明Java方法或C语言子程序，以便可以从PL / SQL调用它。您还可以使用SQL CALL语句来调用此类方法或子程序。调用规范告诉数据库在调用时调用哪个Java方法或哪个命名子程序在哪个共享库中调用。它还告诉数据库为参数和返回值进行的类型转换。

调用规范可以出现在以下SQL语句中：

  • ALTER TYPE语句

  • 创建功能声明

  • CREATE PROCEDURE声明

  • CREATE TYPE语句

  • CREATE TYPE BODY声明

语义

call_spec

将C过程或Java方法名称，参数类型和返回类型映射到它们的SQL副本。

调用规范可以出现在PL / SQL独立子程序，包规范和主体以及类型规范和主体中。它们不能出现在PL / SQL块中。

java_declaration

串

标识方法的Java实现。

c_declaration

LIBRARY lib_name

标识由“ CREATE LIBRARY语句 ” 创建的库 。

外部

声明C子程序的不推荐方式，仅支持向后兼容性。如果C调用规范包含默认参数或受约束的PL / SQL类型，则在C调用规范中使用EXTERNAL ，否则使用LANGUAGE C语法。
