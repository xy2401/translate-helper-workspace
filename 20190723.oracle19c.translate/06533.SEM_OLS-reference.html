11 SEM_OLS包子程序

SEM_OLS软件包包含使用Oracle Label Security（OLS）与RDF数据的三级安全性相关的子程序（函数和过程）。

要使用本章中的子程序，您应该了解RDF语义图概述和RDF数据细粒度访问控制中的概念和用法信息。

本章提供有关子程序的参考信息，按字母顺序列出。

  • SEM_OLS.APPLY_POLICY_TO_APP_TAB
  • SEM_OLS.REMOVE_POLICY_FROM_APP_TAB

11.1 SEM_OLS.APPLY_POLICY_TO_APP_TAB

格式

SEM_OLS.APPLY_POLICY_TO_APP_TAB（policy_name IN VARCHAR2，schema_name IN VARCHAR2，table_name IN VARCHAR2，谓词IN VARCHAR2 DEFAULT NULL，network_owner IN VARCHAR2 DEFAULT'MDSYS'，network_name IN VARCHAR2 DEFAULT NULL）;

描述

将OLS策略应用于应用程序表。

参数

policy_name

现有OLS策略的名称。

架构名称

包含应用程序表的模式的名称。

table_name

应用程序表的名称。

谓词

与基于标签的谓词结合使用的附加谓词。

network_owner

语义网络的所有者。（ 见表1-1 ）

network_name

语义网络的名称。（ 见表1-1 ）

使用说明

使用三级安全性时，OLS将应用于网络中的每个语义模型。也就是说，标签安全性应用于相关的内部表和所有应用程序表;无需手动将策略应用于现有语义模型的应用程序表。但是，如果在应用OLS策略后需要创建其他模型，则必须使用SEM_OLS.APPLY_POLICY_TO_APP_TAB过程在创建模型之前将OLS应用于应用程序表。

您必须具有以下条件才能执行此过程：SA_POLICY_ADMIN程序包的EXECUTE特权和policy_DBA角色。

在执行此过程之前，您必须执行SEM_RDFSA.APPLY_OLS_POLICY过程，为rdfsa_options参数指定SEM_RDFSA.TRIPLE_LEVEL_ONLY 。

要从应用程序表中删除OLS策略，请使用SEM_OLS.REMOVE_POLICY_FROM_APP_TAB过程。

有关OLS支持的信息，请参阅RDF数据的细粒度访问控制 。

有关语义网络类型和选项的信息，请参阅语义网络 。

例子

以下示例将名为defense的OLS策略应用于MY_SCHEMA.MY_APP_TABLE应用程序表。 

开始sem_ols.apply_policy_to_app_table（policy_name =>'defense'，schema_name =>'my_schema'，table_name =>'my_app_table'）;结束; /

11.2 SEM_OLS.REMOVE_POLICY_FROM_APP_TAB

格式

SEM_OLS.REMOVE_POLICY_FROM_APP_TAB（policy_name IN VARCHAR2，schema_name IN VARCHAR2，table_name IN VARCHAR2，check_model IN BOOLEAN DEFAULT），network_owner IN VARCHAR2 DEFAULT'MDSYS'，network_name IN VARCHAR2 DEFAULT NULL）;

描述

从应用程序表中永久删除或分离OLS策略。

参数

policy_name

现有OLS策略的名称。

架构名称

包含应用程序表的模式的名称。

table_name

应用程序表的名称。

check_model

network_owner

语义网络的所有者。（ 见表1-1 ）

network_name

语义网络的名称。（ 见表1-1 ）

使用说明

如果您删除了语义模型，并且不再需要保护应用程序表，则可以使用此过程。

您必须具有以下条件才能执行此过程：SA_POLICY_ADMIN程序包的EXECUTE特权和policy_DBA角色。

在执行此过程之前，您必须执行SEM_RDFSA.APPLY_OLS_POLICY过程，为rdfsa_options参数指定SEM_RDFSA.TRIPLE_LEVEL_ONLY 。

如果存在关联模型，则会生成异常。在这种情况下，如果要执行此过程，则必须先删除模型。

有关OLS支持的信息，请参阅RDF数据的细粒度访问控制 。

有关语义网络类型和选项的信息，请参阅语义网络 。

例子

以下示例从MY_SCHEMA.MY_APP_TABLE应用程序表中删除名为defense的OLS策略。 

开始sem_ols.remove_policy_from_app_table（policy_name =>'defense'，schema_name =>'my_schema'，table_name =>'my_app_table'）;结束; /
