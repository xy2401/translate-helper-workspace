关于传输和升级数据库（完全可传输导出/导入）

您可以使用基于文件或非基于文件的模式来传输数据。

Oracle Data Pump的完全可传输导出/导入功能提供了两个选项。

  • 使用基于文件的Oracle Data Pump导出/导入

  • 使用基于非文件的网络模式Oracle Data Pump导入
