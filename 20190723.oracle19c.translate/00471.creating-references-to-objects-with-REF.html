6.9使用REF创建对象的引用

您可以使用REF修改器创建视图以获取对象的引用。

示例6-7使用REF查询对象的引用

- 需要Ex。6-2，例如6-4和Ex。6-6创建或替换视图emp_view OF emp_t WITH OBJECT IDENTIFIER（empno）AS SELECT e.empno，e.empname，e.salary， REF （d）FROM emp e，dept_view d WHERE e.deptno = d.deptno;

在例6-7中 ， dept_view连接deptno键上的emp表。

使用MAKE_REF运算符（如示例6-6 ）而不是REF修饰符的优点是，使用前者，您可以创建循环引用。例如，您可以创建一个具有对员工部门的引用的员工视图，而部门视图则包含对在该部门中工作的员工的引用列表。

与合成对象一样，您也可以选择持久存储的引用作为视图列，并在查询中无缝使用它们。但是，不能持久存储对视图对象的对象引用。
