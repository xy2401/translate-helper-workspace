1 Oracle数据库客户端安装核对表

使用清单来查看系统要求，以及规划和执行Oracle Database Client安装。

Oracle建议您使用清单作为安装规划过程的一部分。使用清单可以帮助您确认服务器硬件和配置是否满足此版本的最低要求，并可帮助您成功安装。

  • Oracle数据库客户端安装的服务器硬件清单
    使用此核对表检查Oracle Database Client安装的硬件要求。
  • Oracle Solaris上的Oracle数据库操作系统清单
    使用此核对表可以检查Oracle数据库的最低操作系统要求。
  • Oracle数据库客户端的服务器配置清单
    使用此核对表可以检查Oracle Database Client安装的最低服务器配置要求。
  • Oracle数据库安装的Oracle用户环境配置清单
    使用此核对表来规划Oracle数据库管理的操作系统用户，组和环境。
  • Oracle数据库客户端的存储核对表
    使用此核对表可以查看存储最低要求并协助进行配置规划。
  • Oracle数据库客户端的安装程序规划核对表
    使用此核对表可帮助您在启动Oracle Universal Installer之前做好准备。
