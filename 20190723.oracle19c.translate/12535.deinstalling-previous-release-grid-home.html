卸载以前的版本Grid Home

使用此过程卸载以前版本的Grid home。

对于以前版本的升级，如果要卸载以前版本的Grid home，请执行以下步骤：
  1. 以root用户身份登录。
  2. 手动更改先前版本Grid home的权限。
    #chmod -R 775 /u01/app/oracle/product/18.0.0/grid#chown -R oracle：oinstall /u01/app/oracle/product/18.0.0/grid#chown oracle / u01 / app / oracle /产品/ 18.0.0

    在这个例子中：

    • /u01/app/oracle/product/18.0.0/grid是以前版本的Oracle Grid Infrastructure，用于独立服务器主页

    • oracle是Oracle Grid Infrastructure安装所有者用户

    • oinstall是Oracle Inventory组的名称（OINSTALL组）

    • /u01/app/oracle/product/18.0.0是上一个Grid主目录的父目录。

  3. 以Oracle Grid Infrastructure软件所有者用户（ oracle ）身份登录并运行deinstall命令。
