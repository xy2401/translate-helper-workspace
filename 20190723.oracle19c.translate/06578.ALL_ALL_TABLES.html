2.12 ALL_ALL_TABLES

ALL_ALL_TABLES描述了当前用户可访问的对象表和关系表。

相关观点

  • DBA_ALL_TABLES描述数据库中的所有对象表和关系表。

  • USER_ALL_TABLES描述了当前用户拥有的对象表和关系表。此视图不显示OWNER列。

HYBRID Foot 1

VARCHAR2（3）

指示表是否是混合分区表（ YES ）或否（ NO ）。混合分区表可以包含存储在段中的分区和外部存储的分区的混合。

CELLMEMORY Foot 2

VARCHAR2（24）

存储单元闪存缓存中的列式压缩值。可能的值：

  • ENABLED ：Oracle Exadata Storage将自动决定是否以柱状形式进行缓存

  • DISABLED ：禁止Oracle Exadata存储以列式形式缓存

  • NO CACHECOMPRESS ：Oracle Exadata存储将以HCC格式缓存（无重新压缩）

  • FOR QUERY ：Oracle Exadata存储将以INMEMORY查询高格式重新压缩和缓存

  • FOR CAPACITY ：Oracle Exadata存储将以INMEMORY容量低格式重新压缩和缓存

INMEMORY_SERVICE

VARCHAR2（12）

指示如何在各种实例上填充IM列存储。可能的值是：

  • DEFAULT ：在使用PARALLEL_INSTANCE_GROUP初始化参数指定的所有实例上填充数据。如果未设置该参数，则会在所有实例上填充数据。这是默认值。

  • NONE ：未在任何实例上填充数据。

  • ALL ：无论PARALLEL_INSTANCE_GROUP初始化参数的值如何，都会在所有实例上填充数据。

  • USER_DEFINED ：仅在用户指定的服务处于活动状态的实例上填充数据。与此对应的服务名称存储在INMEMORY_SERVICE_NAME列中。

INMEMORY_SERVICE_NAME

VARCHAR2（1000）

指示应在其上填充IM列存储的服务的服务名称。仅当相应的INMEMORY_SERVICE为USER_DEFINED时，此列才具有值。在所有其他情况下，此列为空。
