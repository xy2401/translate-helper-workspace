38.11 ORA_DM_PARTITION_NAME

句法


插图说明ora_dm_partition_name.eps

mining_attribute_clause :: =


mining_attribute_clause.eps插图的描述

目的

ORA_DM_PARTITION_NAME是一个单行函数，可与其他现有函数一起使用。此函数返回与输入行关联的分区的名称。在非分区模型上使用ORA_DM_PARTITION_NAME ，结果为NULL 。

在对分区模型进行评分时， ORA_DM_PARTITION_NAME函数的语法可以使用可选的GROUPING提示。请参阅GROUPING提示 。

mining_attribute_clause

mining_attribute_clause标识要用作评分预测变量的列属性。使用分析语法调用函数时，这些预测变量也用于构建瞬态模型。mining_attribute_clause行为与PREDICTION函数的描述相同。请参阅mining_attribute_clause 。
