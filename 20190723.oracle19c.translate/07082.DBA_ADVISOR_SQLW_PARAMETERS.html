4.37 DBA_ADVISOR_SQLW_PARAMETERS

DBA_ADVISOR_SQLW_PARAMETERS显示数据库中的所有工作负载参数及其当前值。

相关视图

USER_ADVISOR_SQLW_PARAMETERS显示当前用户拥有的工作负载参数及其当前值。此视图不显示OWNER列。

PARAMETER_TYPE

VARCHAR2（10）

参数的数据类型：

  • NUMBER - 数字值

  • STRING - 字符串值。如果字符串包含特殊字符，则它将用单引号括起来。

  • STRINGLIST - 以逗号分隔的字符串元素列表。如果字符串元素包含逗号或其他特殊字符，则该元素将用单引号括起来。

  • TABLE - 单表引用。引用包含模式名称，后跟可选的表名称。

    如果省略表名或字符为% ，则表名称将被解释为通配符。支持SQL引用的标识符。

  • TABLELIST - 一个或多个以逗号分隔的表引用的列表。引用包含模式名称，后跟可选的表名称。

    如果省略表名或字符为% ，则表名称将被解释为通配符。支持SQL引用的标识符。
