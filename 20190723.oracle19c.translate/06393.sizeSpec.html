4.19 sizeSpec

目的

sizeSpec子条款指定数据的大小。

句法

sizeSpec :: =


插图说明sizespec.eps

语义

语法元素 描述

integer [G | K | M]

指定以千兆字节（ G ），千字节（ K ）或兆字节（ M ）为单位的数据大小。

例子

示例4-36设置备份件的最大尺寸

此ALLOCATE命令在MAXPIECESIZE参数中指定写入磁带的备份片不能超过800 MB。

RUN {ALLOCATE CHANNEL c1 DEVICE TYPE sbt PARMS'SBT_LIBRARY = / usr / local / oracle / backup / lib / libobk.so，ENV =（OB_DEVICE_1 = stape1）'MAXPIECESIZE 800M; }
