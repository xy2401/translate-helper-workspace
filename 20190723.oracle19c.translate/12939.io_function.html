说明io_function.gif的描述

显示I / O图表。

顶部是同步单块读取的延迟。

将显示折线图。y轴单位是毫秒。值范围为0到10。x轴单位是一天中的小时，以3小时为间隔显示。x轴值的范围从早上5:00到凌晨2:00。图表跟踪延迟信息。该图表显示活动很少，2点钟时飙升约2。

I / O细分选项包括：I / O功能（显示为选中），I / O类型和用户组。

显示了I / O功能标记为每秒I / O兆字节的折线图。y轴单位是每秒兆字节。y轴的值范围为0到50。可以显示在图表中的信息类型的图例包含以下内容：Archive Manager，Others，XDB，Streams AQ，Data Pump，Recovery，RMAN，ARCH，LGWR，DBWR，Direct Writes，Direct Reads，Buffer Cache Reads。该图表显示了其他人，LGWR和DBWR的活动，其他人在凌晨2点和LGWR凌晨4点激增。

显示了通过I / O功能标记的每秒I / O请求的折线图。y轴单位是每秒I / O.y轴的值范围为0到1200。可以显示在图表中的信息类型的图例包含以下内容：Archive Manager，Others，XDB，Streams AQ，Data Pump，Recovery，RMAN，ARCH，LGWR，DBWR，Direct Writes，Direct Reads，Buffer Cache Reads。该图表显示LGWR和DBWR的活动，LGWR在4:00出现最大涨幅。
