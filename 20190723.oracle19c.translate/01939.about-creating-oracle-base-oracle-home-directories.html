关于创建Oracle Home和Oracle Base Directory

在每个群集节点上创建Grid home和Oracle基本主目录。

Oracle建议您手动创建Oracle Grid Infrastructure Grid主目录和Oracle基本主目录，特别是如果您为群集和Oracle数据库软件所有者提供单独的Oracle Grid Infrastructure，以便您可以单独为Oracle Grid Infrastructure安装所有者分隔日志文件Oracle基础，并防止在Oracle基本路径下意外放置Grid主页。例如：

#mkdir -p /u01/app/19.0.0/grid#mkdir -p / u01 / app / grid #mkdir -p / u01 / app / oracle #chown -R grid：oinstall / u01＃chown oracle：oinstall / u01 / app / oracle＃chmod -R 775 / u01 /

注意：

不支持在群集文件系统上为群集二进制文件放置Oracle Grid Infrastructure。

如果计划在共享OCFS2位置安装Oracle RAC主目录，则必须将OCFS2升级到至少1.4.1版，该版本支持共享可写maps 。

Oracle建议您在每个集群成员节点上本地安装Oracle Grid Infrastructure。使用共享网格主页可防止滚动升级，并为群集创建单点故障。
