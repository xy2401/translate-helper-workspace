7.128 V $ CLEANUP_PROCESS

V$CLEANUP_PROCESS提供有关PMON进程的信息。

柱 数据类型 描述

名称

VARCHAR2（5）

清理过程的名称（PMON，CLMN，CL **）

PADDR

RAW（8）

清理过程的进程指针（可以与V$PROCESS ）

SADDR

RAW（8）

清理过程的会话指针（可以与V$SESSION一起加入）

州

VARCHAR2（4）

清理过程状态：

  • IDLE ：目前没有进行清理

  • BUSY ：目前正在进行清理

DEAD_IN_CLEANUP

RAW（8）

清理中指向树根的指针（可以在V$DEAD_CLEANUP与ROOT_ADDR V$DEAD_CLEANUP ）

CLEANUP_TIME

数

如果STATE = BUSY ，则在当前清理尝试中花费的时间（以秒为单位）。否则， 0 。

TIME_SINCE_LAST_CLEANUP

数

如果STATE = IDLE ，则自上次需要执行清理以来的时间（以秒为单位）。否则， 0 。

NUM_CLEANED

数

清理过程清理的树数。每次尝试根树时增加1。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个多租户容器数据库（CDB）相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
