选择Oracle RAC的安装选项

查看主题以选择安装Oracle RAC的选项。

  • 关于数据库安装的群集节点选择
    选择群集节点以安装Oracle RAC或Oracle RAC One Node数据库。
  • 选择数据库名称
    数据库名称由各种字符串组成，并且只能包含允许的字符。选择数据库名称时，请查看以下准则。
  • 数据库密码的要求
    要保护数据库，请使用满足Oracle建议密码要求的密码，甚至是预定义用户帐户的密码。
  • 关于自动内存管理安装选项
    确定是否要在安装期间配置自动内存管理。
  • 关于安装期间的字符集选择
    在创建数据库之前，请确定要使用的字符集。
  • 安装后管理数据库服务
    使用服务器控制实用程序（SRVCTL），或使用Oracle Enterprise Manager Database Express或Oracle Enterprise Manager云控制来管理和监视Oracle RAC数据库的数据库服务。
