12.22执行

句法

EXEC [UTE] 声明

where 语句表示PL / SQL语句。

执行单个PL / SQL语句。当您要执行引用存储过程的PL / SQL语句时，EXECUTE命令通常很有用。有关PL / SQL的更多信息，请参阅Oracle Database PL / SQL语言参考 。

用法

如果由于PL / SQL语句而导致EXECUTE命令无法放在一行上，请使用SQL * Plus连续符（连字符）。

命令和PL / SQL语句的长度不能超过SET LINESIZE定义的长度。

您可以使用SET FEEDBACK OFF禁止打印消息“PL / SQL过程成功完成”。
