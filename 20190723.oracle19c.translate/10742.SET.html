目的

SET通过消除重复项将嵌套表转换为集合。该函数返回一个嵌套表，其元素彼此不同。返回的嵌套表与输入嵌套表的类型相同。

嵌套表的元素类型必须具有可比性。有关非标量类型可比性的信息，请参阅“ 比较条件 ” 。

例子

以下示例从customers_demo表中选择cust_address_ntab嵌套表列的唯一元素：

SELECT customer_id，SET（cust_address_ntab）地址FROM customers_demo ORDER BY customer_id;
