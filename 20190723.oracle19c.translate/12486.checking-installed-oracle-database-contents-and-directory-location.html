检查已安装的Oracle数据库内容和目录位置

使用以下步骤检查Oracle数据库安装的内容和目录位置：

  1. 转到$ORACLE_HOME/oui/bin 。
  2. 启动Oracle Universal Installer。
    $ ./runInstaller
  3. 单击“ 已安装的产品”以在“ 欢迎”屏幕上显示“清单”对话框。
  4. 从列表中选择Oracle数据库产品以检查已安装的内容。
  5. 单击“ 详细信息”以查找有关已安装产品的其他信
  6. 单击“ 关闭”以关闭“清单”对话框。
  7. 单击“ 取消”关闭Oracle Universal Installer，然后单击“ 是”进行确认。
