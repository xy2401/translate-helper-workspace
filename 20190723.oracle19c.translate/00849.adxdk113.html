XML文档既可以输入到XML Parser，也可以通过Java Architecture for XML Binding（JAXB）或C ++ Class Generator生成Java或C ++类。Java应用程序可以输出XML。

可以选择将XML Schema Validator应用于XML Parser输出。

XML Schema也可以输入到XML Schema Validator。输出类以及XML Schema Validator输出被输入到SAX，DOM或XML Compressor。DOM树输出被输入到XSLT处理器，XSLT处理器还接收来自XSL样式表的输入。

XSLT处理器输出转换后的XML文档。

压缩的XML是来自XML Compressor的另一个输出。
