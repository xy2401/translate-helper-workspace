12.36 REPFOOTER

句法

REPF [OTER] [PAGE] [ printspec [ text | 变量 ] ...]| [ON | 关 ]

其中printspec表示用于放置和格式化文本的以下一个或多个子句：

COL n S [KIP] [ n ] TAB n LE [FT] CE [NTER] R [IGHT] BOLD FORMAT text

地点和格式指定的报告页脚位于每个报告的底部，或列出当前的REPFOOTER定义。

输入REPFOOTER，不带任何条款列出当前的REPFOOTER 定义。

条款

见REPHEADER关于在REPFOOTER命令语法术语和条款的附加信息的命令。

用法

如果在文本或变量之前没有输入printspecact子句，REPFOOTER会左对齐文本或变量。

您可以在printspec中使用任意数量的常量和变量。SQL * Plus按照您指定的顺序显示常量和变量，定位和格式化每个常量或变量，这些常量或变量由其前面的printspec子句指定。

注意：

如果SET EMBEDDED为ON，则报告页脚将被禁止。

例子

要将“END EMPLOYEE LISTING REPORT”定义为单独页面上的报告页脚并将其居中，请输入：

REPFOOTER PAGE CENTER'END EMPLOYEE LISTING REPORT'TTITLE RIGHT'页面：'格式999 SQL.PNO SELECT LAST_NAME，SALARY FROM EMP_DETAILS_VIEW WHERE SALARY> 12000;
LAST_NAME SALARY ------------------------- ---------- King 24000 Kochhar 17000 De Haan 17000 Russell 14000 Partners 13500 Hartstein 13000 ----------总和98500页数：2结束员工上市报告选择了6行。

要在不更改其定义的情况下禁止报表页脚，请输入

REPFOOTER OFF
