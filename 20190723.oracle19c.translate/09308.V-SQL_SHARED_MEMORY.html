9.65 V $ SQL_SHARED_MEMORY

V$SQL_SHARED_MEMORY显示有关游标共享内存快照的信息。 存储在共享池中的每个SQL语句都有一个或多个与之关联的子对象。每个子对象都有几个部分，其中一个是上下文堆，其中包含查询计划等。
柱 数据类型 描述

SQL_TEXT

VARCHAR2（1000）

此行正在显示信息的共享游标子对象的SQL文本

SQL_FULLTEXT

CLOB

作为CLOB列公开的SQL语句的全文。可以使用此列检索SQL语句的全文，而不是使用V$SQLTEXT动态性能视图进行连接。

HASH_VALUE

数

共享池中上述SQL文本的哈希值

SQL_ID

VARCHAR2（13）

共享池中上述SQL文本的SQL标识符

HEAP_DESC

RAW（4 | 8）

此行中描述的子游标的上下文堆的描述符的地址

结构体

VARCHAR2（64）

如果此行中描述的内存块是使用“X：Y”形式的注释分配的，那么这是注释的“X”部分

功能

VARCHAR2（64）

与STRUCTURE列类似，这是注释的“Y”字段

CHUNK_COM

VARCHAR2（16）

分配此内存块时提供的整个注释字段

CHUNK_PTR

RAW（4 | 8）

分配的内存块的起始地址

CHUNK_SIZE

数

为此块分配的内存量

ALLOC_CLASS

VARCHAR2（8）

这块内存所属的内存类。它通常是FREEABLE或PERMANENT 。

CHUNK_TYPE

数

回调函数表的索引，它告诉服务器如何重新创建这块内存，如果需要根据LRU算法从共享池中删除它

SUBHEAP_DESC

RAW（4 | 8）

如果此上下文堆的父堆本身是子堆，那么这是父堆的描述符的地址

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
