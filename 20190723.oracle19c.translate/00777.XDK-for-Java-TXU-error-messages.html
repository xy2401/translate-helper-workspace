B XDK for Java TXU错误消息

列出了在执行TXU接口期间使用Oracle XML Developer Kit（XDK）for Java的应用程序的错误消息。

DLF错误消息

数据加载格式（DLF）错误消息的范围为TXU-0100至TXU-0199。

TXU-0100：未找到查询字符串中的参数字符串

原因：查询中没有参数的占位符

操作：提供一个参数，其id可以作为关联查询中的关联占位符找到

TXU-0101：不兼容属性的山坳，在字符串中的查询字符串常量并存

原因：属性'col'和'constant'不能共存

操作：删除'col'或'constant'属性

TXU-0102：找不到节点字符串

原因：该文档缺少预期的节点

操作：提供缺少的节点

TXU-0103：元素字符串缺少内容

原因：该元素没有数据

行动：提供内容

TXU-0104：与SQL 字符串元素串缺乏山坳或常量属性

原因：该元素缺少'col'或'constant'的必需属性

操作：提供'col'或'constant'属性

TXU-0105：SQL异常字符串在处理SQL 字符串

原因： SQL执行期间发生错误

操作：解决SQL语句中的错误

TXU-0106：SQL 字符串选择的列字符串没有数据

原因： SQL查询未返回任何数据

操作：提供数据或修改查询

TXU-0107：不支持数据类型字符串

原因：尝试处理不受支持的数据类型

操作：将数据类型更改为支持的数据类型

TXU-0108：缺少列字符串的 maxsize属性

原因：指定了size-unit属性，但未指定maxsize。

操作：也提供maxsize属性

TXU-0109： 字符串的字符串文本长度超过了允许的最大字符串的

原因：文本数据的长度太长

操作：缩短数据以使其符合限制，或者放大maxsize属性并确保数据库列足够大

TXU-0110：行字符串不宣柱串

原因：数据部分中的列未在列部分中声明

操作：将列名修改为已声明的列名

TXU-0111：缺乏行字符串 字符串列数据

原因：声明了列但数据丢失。

操作：提供name属性与列名匹配的col元素

TXU-0112：未申报的查询参数字符串列字符串

原因：查询参数引用未声明的列

操作：指定声明的列

TXU-0113：不兼容的属性字符串对列串查询

原因：带有查询的列不能具有指定的属性

操作：删除属性或查询

TXU-0114：行字符串 DLF解析错误（ 字符串 ），在字符串 字符串

原因：报告格式错误

行动：纠正错误的部分

TXU-0115：指定的日期字符串字符串格式无效

原因：指定的日期字符串与指定的formatstring不匹配。

操作：确保日期字符串采用适当的日期格式

TransX信息消息

TransX信息性消息在TXU-0200到TXU-0299的范围内。

TXU-0200： 字符串重复行

原因：数据库中存在重复的行

操作：此消息显示在DuplicateRowException上，以通知应用程序存在于数据库中的一个或多个重复行的存在

TransX错误消息

TransX错误消息的范围是TXU-0300到TXU-0399。

TXU-0300：找不到文档字符串

原因：无法找到该文档

操作：修改文档位置或在该位置提供文档

TXU-0301：无法读取文件字符串

原因：读取文件时发生I / O错误

操作：解决I / O问题

TXU-0302：找不到存档字符串

原因：找不到存档文件

操作：确保CLASSPATH正确包含TransX且仅包含一次

TXU-0303：模式字符串中的字符串找不到

原因：无法找到DLF的架构定义

操作：获取TransX存档的完整副本

TXU-0304：找不到字符串的归档路径

原因：无法确定存档的路径

操作：确保CLASSPATH正确包含TransX且仅包含一次

TXU-0305：上串呼吁字符串中没有数据库连接

原因：未尝试数据库连接尝试操作

操作：首先打开连接

TXU-0306：给出空表名;拒绝访问

原因：未提供表名

操作：指定表名

TXU-0307：无法确定查找键是否为字符串

原因：数据字典已损坏

操作：恢复数据字典

TXU-0308：找不到二进制文件字符串

原因：文件名无效

行动：提供一个好的文件名

TXU-0309： 字符串的文件大小超过允许的最大2,000字节

原因：文件太大

操作：减小文件大小

断言错误消息

断言错误消息的范围为TXU-0400至TXU-0499。

TXU-0400： 字符串上缺少SQL语句元素

原因：内部断言未成功

操作：联系Oracle客户支持

TXU-0401：缺少节点字符串

原因：内部断言未成功

操作：联系Oracle客户支持

TXU-0402：无效的标志字符串

原因：内部断言未成功

操作：联系Oracle客户支持

TXU-0403：内部错误字符串

原因：内部断言未成功

操作：联系Oracle客户支持

TXU-0404：意外的异常字符串

原因：内部断言未成功

操作：联系Oracle客户支持
