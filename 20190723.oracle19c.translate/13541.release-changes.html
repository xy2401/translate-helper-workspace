不受支持的功能

此版本不支持以下功能：

Oracle Database 12 c第2版（12.2.0.1）中的更改

以下是适用于Oracle Database 12 c第2版（12.2.0.1）的Oracle数据库XStream指南中的更改。

  • 新功能

父主题： Oracle Database XStream指南的此发行版中的更改

新功能

以下功能是此版本中的新增功能：

  • DML冲突处理程序

    XStream提供了预构建的DML冲突处理程序。

    请参阅“ 在冲突检测和解决方案中管理XStream ” 。

  • 支持多租户环境中的应用程序容器和应用程序公共对象

    XStream可以复制对应用程序根和应用程序PDB所做的更改。

    请参阅“ XStream Out和多租户环境 ”和“ XStream In和多租户环境 ” 。

  • 改进了对用户定义类型的支持

    XStream现在支持对象类型，REF和varrays。

    请参阅“ 捕获进程捕获的数据类型 ”和“ 入站服务器应用的数据类型 ” 。

  • LCRID版本

    LCRID是用于指定XStream Out的LCR位置的原始值。从Oracle Database 12 c第2版（12.2.0.1）开始，LCRID已经过版本控制。

    参见“ LCRID和LCR的位置 ” 。

  • XStream支持多租户容器数据库（CDB）中的不同字符集

    当CDB的根的字符集是AL32UTF8时，CDB中的任何可插拔数据库（PDB）都可以使用与根和其他PDB不同的字符集。

  • XStream支持数据库对象的长标识符

    在此版本中，某些数据库对象的标识符长度最多为128个字节。XStream出站服务器可以捕获对具有长标识符的数据库对象所做的更改，而XStream入站服务器可以将更改应用于具有长标识符的数据库对象。

  • LCR跟踪

    您可以使用message_tracking_frequency apply参数或DBMS_XSTREAM_ADM包中的SET_MESSAGE_TRACKING过程通过流跟踪LCR。

    请参阅“ 更改无法在XStream中访问客户端应用程序 ” 。

  • 新的捕获过程参数

    为XStream Out配置添加了以下新的捕获过程参数：

    • annotateddlsize

    有关这些应用参数的信息，请参见Oracle Database PL / SQL包和类型参考 。

  • 新的应用参数

    为XStream入站服务器添加以下新的应用参数：

    • batchsql_mode

    • cdgranularity

    • message_tracking_frequency

    有关这些应用参数的信息，请参见Oracle Database PL / SQL包和类型参考 。

  • 查看XStream组件的事件历史记录

    DBA_REPLICATION_PROCESS_EVENTS视图显示XStream组件的事件历史记录。

    请参阅“ 监视XStream Out组件的事件历史记录 ”和“ 监视组件中XStream的事件历史记录 ” 。
