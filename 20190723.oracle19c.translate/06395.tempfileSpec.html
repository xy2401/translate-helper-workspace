4.20 tempfileSpec

目的

使用tempfileSpec子句按名称或绝对文件号指定临时文件。

句法

tempfileSpec :: =


tempfilespec.eps插图的描述

语义

语法元素 描述

' filename '

使用完整路径或相对文件名指定数据文件。如果指定相对文件名，则目标数据库将以特定于平台的方式限定文件名。

您可以指定绝对路径名称或相对于Oracle主目录的路径名称。双引号和单引号都有效（尽管图中仅显示单引号）。使用问号（ ? ）代表Oracle主目录和Oracle SID的at符号（ @ ）。

整数

按绝对文件编号指定数据文件。从V$TEMPFILE视图或REPORT SCHEMA输出中获取文件编号。

例子

示例4-37按文件名指定临时文件

此示例将临时文件/disk1/oradata/prod/temp01.dbf重命名为/disk2/temp01.dbf ， /disk2/temp01.dbf文件名指定。执行此示例时必须装入数据库。 

SHUTDOWN IMMEDIATE STARTUP MOUNT RUN {SWITCH TEMPFILE'/disk1/oradata/prod/temp01.dbf'TO'/disk2/temp01.dbf'; ALTER DATABASE OPEN;
