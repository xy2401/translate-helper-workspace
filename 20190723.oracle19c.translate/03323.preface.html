前言

用于DRDA的Oracle数据库网关为用户提供对DB2的透明访问。

目标受众

本指南适用于负责安装，配置和管理网关的任何人，也适用于应用程序开发人员。

如果您负责编写通过网关访问DRDA数据库的应用程序，请阅读本指南。

在使用本指南安装或管理网关之前，您必须了解Oracle Database Gateway的基础知识以及您正在使用的操作系统。

文档可访问性

有关Oracle对辅助功能的承诺的信息，请访问Oracle辅助功能计划网站http://www.oracle.com/pls/topic/lookup?ctx=acc&id=docacc 。

访问Oracle支持

已购买支持的Oracle客户可通过My Oracle Support获得电子支持。有关信息，请访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=info或访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=trs听力受损。

印刷约定

本指南中使用以下印刷约定：

惯例 描述

等宽

等宽字体类型表示命令，目录名称，用户名，路径名和文件名。

斜体

斜体类型表示变量，包括文件名的可变部分。它也用于强调和书籍。

大写

大写字母表示结构化查询语言（SQL）保留字，初始化参数和环境变量。

胆大

粗体显示屏幕名称和字段。

SQL * Plus提示

SQL * Plus提示符SQL>出现在SQL语句和SQL * Plus命令示例中。在提示符下输入您的回复。请勿在响应中输入提示“SQL>”的文本。

SQL * Plus提示

SQL * Plus提示符SQL>,出现在SQL语句和SQL * Plus命令示例中。在提示符下输入您的回复。请勿在响应中输入提示“ SQL> ”的文本。

存储测量

存储测量使用以下缩写：

  • KB，千字节，等于1,024字节

  • MB，兆字节，等于1,048,576字节

  • GB，千兆字节，等于1,073,741,824字节
