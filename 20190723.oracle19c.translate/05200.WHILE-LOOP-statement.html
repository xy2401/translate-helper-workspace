13.67 WHILE LOOP声明

WHILE LOOP语句在条件为TRUE时运行一个或多个语句。 WHILE LOOP语句在条件变为FALSE或NULL时结束，或者当循环内的语句将控制转移到循环外或引发异常时结束。

话题

  • 句法

  • 语义

  • 例子

  • 相关话题

句法

while_loop_statement :: =


说明while_loop_statement.eps的描述

（ boolean_expression :: = ， statement :: = ）

语义

while_loop_statement

逻辑表达式

值为TRUE ， FALSE或NULL表达式。

boolean_expression在循环的每次迭代开始时计算。如果其值为TRUE ，则LOOP运行后的语句。否则，控制转移到WHILE LOOP语句之后的语句。

声明

要防止无限循环，至少有一个语句必须将boolean_expression的值boolean_expression为FALSE或NULL ，在循环外部传递控制或引发异常。可以在循环外传递控制的语句是：

  • “ CONTINUE Statement ” （当它将控制转移到封闭标记循环的下一次迭代时）

  • “ 退出声明 ”

  • “ GOTO声明 ”

  • “ 提升声明 ”

标签

标识while_loop_statement标签（请参阅“ statement :: = ”和“ label ” ）。CONTINUE ， EXIT和GOTO语句可以引用此标签。

标签提高了可读性，尤其是在嵌套LOOP语句时，但LOOP是确保END LOOP语句中的标签与同一LOOP语句开头的标签匹配（编译器不检查）。
