1.163 LICENSE_MAX_USERS

LICENSE_MAX_USERS指定您可以在数据库中创建的最大用户数。

属性 描述

参数类型

整数

默认值

0

可修改

改变系统

可在PDB中修改

没有

价值范围

0到用户许可证的数量

基本

没有

Oracle RAC

Oracle建议多个实例具有相同的值。

达到LICENSE_MAX_USERS指定的限制时，无法创建更多用户。但是，您可以增加限制。

注意：

Oracle不再按并发会话数提供许可。因此，不推荐使用LICENSE_MAX_SESSIONS和LICENSE_SESSIONS_WARNING初始化参数。
