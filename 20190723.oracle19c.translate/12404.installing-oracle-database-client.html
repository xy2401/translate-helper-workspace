5安装Oracle数据库客户端

Oracle Database Client安装软件有多种介质，可以使用多个选项进行安装。

Oracle数据库客户端软件可在安装介质上获得，也可以从Oracle Technology Network网站或Oracle Software Delivery Cloud门户下载。在大多数情况下，您使用Oracle Universal Installer（OUI）提供的图形用户界面（GUI）来安装软件。但是，您也可以使用Oracle Universal Installer完成静默模式安装，而无需使用GUI。

注意：

您不能使用早期Oracle版本中的Oracle Universal Installer来安装此版本的组件。

  • 关于基于映像的Oracle数据库客户端安装
    从Oracle Database 19c开始，通过基于映像的安装简化了Oracle Database Client软件的安装和配置。
  • 访问安装软件
    您可以从Oracle Technology Network网站或Oracle Software Delivery Cloud门户下载Oracle数据库软件。在某些情况下，Oracle数据库软件也可能在安装介质上可用。
  • 关于安装期间的字符集选择
    在创建数据库之前，请确定要使用的字符集。
  • 以不同语言运行安装程序
    介绍如何以其他语言运行安装程序。
  • 安装Oracle数据库客户端软件
    这些主题说明如何运行安装向导以执行大多数数据库客户端安装。
  • 安装后重新链接Oracle数据库客户端二进制文件
    安装Oracle Database Client后，如果需要，可以使用relink as_installed选项修改二进制文件。
