8.105 V $选项

V$OPTION显示Oracle数据库选项和功能。 通常，虽然并非总是如此，但选项必须单独许可，而功能随产品提供，并且基于正在运行的产品（例如，Enterprise Edition）启用。
柱 数据类型 描述

参数

VARCHAR2（64）

选项（或功能）的名称

值

VARCHAR2（64）

指示是否安装了选项（或功能）（ TRUE ）（ FALSE ）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
