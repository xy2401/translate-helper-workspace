适用于Oracle Net Services的C LDAP模式

本附录描述了Oracle Net Services对象的目录服务器中定义的Oracle模式对象类和属性。它没有描述为将来的功能保留或由其他Oracle产品使用的对象类和属性。

  • 结构对象类
    Oracle模式支持Oracle Net目录命名查找的结构对象类。
  • 属性
    它列出了用于对象类的属性。此列表可能会发生变化。

C.1结构对象类

Oracle模式支持Oracle Net目录命名查找的结构对象类。

表C-1 Oracle Net结构对象类

对象类 属性 描述

  • orclNetDescName

  • orclVersion

orclNetAddress

  • orclNetAddressString

  • orclNetProtocol

  • orclVersion

指定侦听器协议地址。

orclNetAddressAux1

  • orclNetHostname

指定要将属性添加到orclNetAddress条目的辅助对象类。

orclNetAddressList

  • orclNetAddrList

  • orclNetFailover

  • orclNetLoadBalance

  • orclNetSourceRoute

  • orclVersion

指定协议地址列表。

  • orclNetAddrList

  • orclNetInstanceName

  • orclNetFailover

  • orclNetLoadBalance

  • orclNetSdu

  • orclNetServiceName

C.2属性

它列出了用于对象类的属性。此列表可能会发生变化。

表C-2 Oracle Net Services的LDAP模式属性

属性 描述

orclCommonContextMap

允许在目录服务器中映射多个默认Oracle Context。

orclNetAddrList

标识一个或多个侦听器协议地址。

orclNetAddressString

定义侦听器协议地址。

orclNetConnParamList

用于连接数据参数的占位符。

orclNetDescList

标识一个或多个连接描述符。

orclNetDescName

标识连接描述符或连接描述符列表。

orclNetFailover

打开协议地址列表的连接时故障转移。

orclNetFailoverModeString

如果第一个侦听器在运行时失败，则指示Oracle Net故障转移到另一个侦听器。根据配置，会话或正在进行的任何SELECT语句将自动进行故障转移。

orclNetHostname

指定主机名。

orclNetInstanceName

指定要访问的实例名称。

orclNetInstanceRole

指定与Oracle Real Application Clusters（Oracle RAC）配置的主实例或辅助实例的连接。

orclNetLoadBalance

为协议地址列表打开客户端负载平衡。

orclNetProtocol

标识orclAddressString属性中使用的协议。

orclNetReceiveBufSize

指定会话接收操作的缓冲区空间限制。

orclNetSdu

指定会话数据单元（SDU）大小。

orclNetSendBufSize

指定会话发送操作的缓冲区空间限制。

orclNetServiceName

指定CONNECT_DATA部分中的数据库服务名称。

orclNetSourceRoute

指示Oracle Net按顺序使用每个地址，直到到达目的地。

orclSid

指定连接描述符的CONNECT_DATA部分中的Oracle系统标识符（SID）。

orclVersion

指定用于创建条目的软件版本。
