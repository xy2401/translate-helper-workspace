前言

Oracle Data Guard是当今最有效的解决方案，可以保护任何企业的核心资产 - 其数据，并使其即使面对灾难和其他灾难也能全天候提供。本指南介绍了Oracle Data Guard技术和概念，并帮助您配置和实现备用数据库。

听众

Oracle Data Guard概念和管理适用于管理Oracle数据库系统的备份，还原和恢复操作的数据库管理员（DBA）。

要使用此文档，您应该熟悉关系数据库概念和基本备份和恢复管理。您还应该熟悉运行Oracle软件的操作系统环境。
