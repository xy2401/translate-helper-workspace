7 Oracle数据库和Oracle Grid Infrastructure支持的存储选项

在安装规划过程中查看支持的存储选项。

  • Oracle数据库支持的存储选项
    下表显示了Oracle Database二进制文件和文件支持的存储选项：
  • 关于独立服务器的Oracle Grid Infrastructure
    如果您计划使用Oracle自动存储管理（Oracle ASM），则必须在安装数据库之前安装Oracle Restart。
  • 关于升级现有Oracle自动存储管理实例
    Oracle自动存储管理（Oracle ASM）升级在Oracle Grid Infrastructure升级期间执行。
  • 关于管理旧版数据库版本的磁盘组
    在Oracle Grid Infrastructure安装上安装早期Oracle数据库时，使用Oracle ASM Configuration Assistant（Oracle ASMCA）创建和修改磁盘组。
  • Oracle ACFS和Oracle ADVM
    Oracle自动存储管理群集文件系统（Oracle ACFS）扩展了Oracle ASM技术，以支持单实例和群集配置中的所有应用程序数据。
  • Oracle数据库的文件系统选项
    如果在文件系统上安装Oracle数据库文件，则Oracle Database Configuration Assistant会在计算机上安装的文件系统上的目录中创建数据库文件。

  • 关于直接NFS客户端安装到NFS存储设备
    Direct NFS Client直接在Oracle软件中集成NFS客户端功能，以优化Oracle和NFS服务器之间的I / O路径。这种集成可以显着提高性能。
