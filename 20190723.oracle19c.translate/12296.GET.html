12.24 GET

句法

GET [FILE] file_name [。 分机 ] [ 列表 | NOLIST]

加载一个操作系统文件进入SQL缓冲区。

条款

文件

用于指定以下参数是要加载的脚本的名称的关键字。通常省略此可选关键字。

如果要使用名称文件加载脚本，因为它是命令关键字，您需要将名称文件放在单引号中。

file_name [。 分机 ]

代表您要加载的文件（通常是脚本 ）。 

LIST

列出加载后的文件内容。这是默认值。 

NOLIST

禁止列表。

用法

如果未指定文件扩展名，则SQL * Plus将采用默认的命令文件扩展名（通常为SQL）。有关更改默认扩展名的信息，请参阅 SET SUF [FIX] {SQL |文字} 。

如果您指定的文件名包含单词列表或单词文件，则名称必须是双引号。SQL * Plus搜索当前工作目录中的文件。

操作系统文件应包含单个SQL语句或PL / SQL块。声明不应以分号结束。如果从操作系统文件将SQL * Plus命令或多个SQL语句或PL / SQL块加载到SQL缓冲区中，则在RUN或RUN时发生错误斜杠（/）命令用于执行缓冲区。

GET命令可用于加载使用SAVE命令创建的文件。有关更多信息，请参阅SAVE 。
