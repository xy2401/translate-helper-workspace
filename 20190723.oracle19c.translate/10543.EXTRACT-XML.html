提取（XML）

注意：

不推荐使用EXTRACT （XML）函数。它仍然支持向后兼容性。但是，Oracle建议您使用XMLQUERY函数。有关更多信息，请参阅XMLQUERY 。

句法

extract_xml :: =


说明extract_xml.eps的描述

目的

EXTRACT （XML）类似于EXISTSNODE函数。它应用VARCHAR2 XPath字符串并返回包含XML片段的XMLType实例。您可以通过省略初始斜杠来指定带有初始斜杠或相对XPath_string的绝对XPath_string 。如果省略初始斜杠，则相对路径的上下文默认为根节点。如果要处理的XML使用名称空间前缀，则需要可选的namespace_string 。此参数必须解析为VARCHAR2值，该值指定前缀的默认映射或命名空间映射，Oracle数据库在评估XPath表达式时使用该映射。

例子

以下示例提取示例表oe.warehouses warehouse_spec列的XML路径的/Warehouse/Dock节点的oe.warehouses ：

SELECT仓库名称，EXTRACT（warehouse_spec，'/仓库/码头'）“库存数量”来自仓库WHERE warehouse_spec IS NOT NULL ORDER BY仓库名称; WAREHOUSE_NAME码头数量------------------------- --------------------- ----新泽西旧金山<码头> 1 </码头>华盛顿州西雅图<码头> 3 </码头>德克萨斯州南湖<码头> 2 </码头>

将此示例与EXTRACTVALUE的示例进行比较，该示例返回XML片段的标量值。
