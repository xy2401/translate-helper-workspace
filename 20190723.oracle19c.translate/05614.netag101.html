netag101.eps插图的描述

此图描绘了JDBC驱动程序使用的堆栈通信层。JDBC OCI驱动程序堆栈自上而下包含以下层：

  • Java客户端应用程序

  • JDBC OCI驱动程序（使用OCI）

  • 演示文稿 - TTC

  • Oracle Net Foundation Layer

  • Oracle协议支持

  • 传输协议（TCP / IP，带有SSL，SDP和命名管道的TCP / IP）

JDBC Thin Driver堆栈自上而下包含以下层：

  • Java Applet /应用程序

  • JDBC瘦驱动程序

  • 演示文稿 - Java TTC

  • JavaNet

  • TCP / IP网络协议
