7.108 V $ BACKUP_SPFILE_DETAILS

V$BACKUP_SPFILE_DETAILS显示有关备份集中备份的所有可V$BACKUP_SPFILE_DETAILS SP文件的信息。

柱 数据类型 描述

SESSION_KEY

数

会话标识符

SESSION_RECID

数

会话重新开始

SESSION_STAMP

数

会话戳

BS_KEY

数

唯一的备份集标识符

SET_STAMP

数

备份集邮票

SET_COUNT

数

备份集计数

MODIFICATION_TIME

日期

备份集的修改时间

文件大小

数

备份的SPFILE的大小（以字节为单位）

FILESIZE_DISPLAY

VARCHAR2（4000）

与FILESIZE列的值相同，但转换为用户可显示的格式，例如n M， n G， n T， n P等

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
