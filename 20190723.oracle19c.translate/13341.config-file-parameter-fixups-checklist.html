本地配置文件参数修正清单示例

如果要在升级期间包含或排除单个数据库的特定修订，则可以使用本地配置文件核对表。

在此示例中，有一个名为sales4.checklist.cfg的本地清单文件，该文件为数据库sales4提供了sales4清单。该文件的一部分包含以下设置：

[checkname] DICTIONARY_STATS [stage] PRECHECKS [fixup_available]是[runfix]是[严重性]推荐

您可以更改DICTIONARY_STATS的默认修正，以通过更改\的runfix选项来排除对数据库sales4执行修正。 检查从YES到NO ：

[checkname] DICTIONARY_STATS [stage] PRECHECKS [fixup_available] YES [runfix] NO [severity] RECOMMEND
