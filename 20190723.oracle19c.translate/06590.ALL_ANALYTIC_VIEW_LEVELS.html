2.24 ALL_ANALYTIC_VIEW_LEVELS

ALL_ANALYTIC_VIEW_LEVELS描述当前用户可访问的分析视图中层次结构中的所有级别。

相关观点

  • DBA_ANALYTIC_VIEW_LEVELS描述数据库中所有分析视图中的级别。

  • USER_ANALYTIC_VIEW_LEVELS描述当前用户拥有的分析视图中的级别。此视图不显示OWNER列。

ORIGIN_CON_ID

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
