关于Oracle Solaris的Oracle数据库先决条件包

使用Oracle数据库先决条件组程序包可简化操作系统配置并确保您具有所需的程序包。

从Oracle Solaris 11.2开始，对于Oracle Database 18c和更高版本的数据库，使用Oracle数据库先决条件组软件包group/prerequisite/oracle/oracle-rdbms-server-18c-preinstall来确保Oracle数据库和Oracle所需的所有必需软件包系统上存在Grid Infrastructure安装。

即使使用任何服务器软件包组安装Oracle Solaris，也可以安装oracle-rdbms-server-18c-preinstall ，例如solaris-minimal-server ， solaris-small-server ， solaris-large-server或solaris-desktop 。Oracle建议您安装solaris-minimal-server组软件包，然后安装oracle-rdbms-server-18c-preinstall 。

使用Oracle Solaris和Oracle数据库先决条件组软件包配置服务器包括以下步骤：

  1. 安装适用于Oracle数据库的推荐Oracle Solaris版本。

  2. 安装Oracle数据库先决条件组软件包oracle-rdbms-server-18c-preinstall 。

  3. 创建角色分配的组和用户。

  4. 为每个群集节点候选者完成网络接口配置。

  5. 根据每个标准或核心节点群集候选所需的完整系统配置进行共享存储访问。

完成这些步骤后，您可以继续安装Oracle数据库，Oracle Grid Infrastructure或Oracle RAC。
