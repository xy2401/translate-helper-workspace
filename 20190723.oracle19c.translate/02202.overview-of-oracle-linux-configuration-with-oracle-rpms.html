使用Oracle预安装RPM的Oracle Linux配置概述

使用Oracle预安装RPM来简化操作系统配置，并确保您具有所需的内核包。

Oracle建议您安装Oracle Linux 7并使用Oracle Preinstallation RPM配置Oracle数据库和Oracle Grid Infrastructure安装的操作系统。

Oracle Linux发行版和数据库发行版的Oracle预安装RPM会自动安装安装Oracle Grid Infrastructure和Oracle Database所需的任何其他软件包，并自动配置服务器操作系统，包括设置内核参数和其他基本操作系统安装要求。有关Oracle Linux和Oracle预安装RPM的更多信息，请参阅：

http://docs.oracle.com/en/operating-systems/linux.html

使用Oracle Linux和Oracle预安装RPM配置服务器包括以下步骤：

  1. 安装Oracle Linux。

  2. 使用Oracle Unbreakable Linux Network（ULN）注册Linux发行版，或者使用适用于Oracle Linux发行版的Oracle Linux yum服务器为您的系统下载和配置yum存储库。

  3. 使用适用于Oracle Grid Infrastructure和Oracle Database版本的RPM安装Oracle Preinstallation RPM，并更新Linux版本。

  4. 创建角色分配的组和具有相同名称和ID号的用户。

  5. 为每个群集节点候选者完成网络接口配置。

  6. 根据每个标准或核心节点群集候选所需的完整系统配置进行共享存储访问。

完成这些步骤后，您可以继续安装Oracle Grid Infrastructure和Oracle数据库。
