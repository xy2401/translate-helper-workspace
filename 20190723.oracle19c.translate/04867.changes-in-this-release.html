Pro * COBOL程序员指南的此发行版中的更改

本前言列出了Pro * COBOL程序员指南中的更改。

Pro * COBOL Release 18c，版本18.1中的更改

新功能

以下功能是此版本中的新增功能：

  • 在流量控制器模式下支持Oracle Connection Manager

    Traffic Director模式中的Oracle Connection Manager是一个代理，位于支持的数据库客户端和数据库实例之间，以提高高可用性，连接多路复用和负载平衡。

    有关详细信息，请参阅关于在Traffic Director模式下使用Oracle Connection Manager 。

Pro * COBOL 12c Release 2（12.2）的变化

以下是Pro * COBOL for Oracle Database 12c第2版（12.2）中的更改。

新功能

  • Pro * COBOL现在支持128字节的标识符长度。在以前的版本中，标识符长度限制为30个字节。

  • Pro * COBOL现在支持Oracle Instant Client - Basic Light版本。

  • Pro * COBOL现在引入了一个新的命令行选项“trim_password”，以防止由包含尾随空格的密码字符串引起的身份验证问题。

  • Pro * COBOL现在支持以下平台的Micro Focus Visual COBOL 2.2 Update 2编译器：

    • Linux x64

    • Windows 64位和32位

    • Solaris x86（32位和64位）

    • Solaris SPARC（32位和64位）
