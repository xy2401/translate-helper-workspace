第V部分 XML模式和对象关系XMLType

涵盖了XML Schema的使用和XMLType数据的对象关系存储。

  • 选择XMLType存储和索引
    应用程序的重要设计选择包括要使用的XMLType存储模型以及要使用的索引方法。
  • XML模式存储和查询：基本
    XML Schema是用于描述XML文档的内容和结构的标准。您可以注册，更新和删除与Oracle XML DB一起使用的XML架构。您可以定义用于基于XML模式的数据的存储结构，并将XML模式数据类型映射到SQL数据类型。
  • XML模式存储和查询：对象关系存储
    基于XML Schema的数据的高级技术包括使用对象关系存储;注释XML模式;将Schema数据类型映射到SQL;使用complexType扩展和限制;创建，指定关系约束，分区基于XML Schema的数据，存储XMLType数据，使用复杂或大型模式，以及调试模式注册。
  • 用于对象关系存储的XPath重写
    对于以对象方式存储的XMLType数据，涉及各种SQL函数的XPath表达式参数的查询通常可以自动重写为针对高度优化的基础SQL表的查询。
  • XML Schema Evolution
    在向Oracle XML DB注册XML模式后，可以使用XML模式演变来更新XML模式。
