使用cURL

cURL是一个开源的命令行工具，用于使用URL语法传输数据，支持各种协议，包括HTTP和HTTPS。

本文档中的示例使用cURL来演示如何访问数据库REST API。

要使用cURL访问REST API：

  • 按照快速入门中的说明安装cURL 。

  • 在命令窗口中，将cURL环境变量CURL_CA_BUNDLE设置为本地CA证书包的位置。有关使用cURL验证CA证书的信息，请参阅： http ： //curl.haxx.se/docs/sslcerts.html 。

  • 调用cURL并根据需要指定下表中定义的一个或多个命令行选项以指示其执行

    cURL选项 描述
    -d， - data @ file.json 在本地计算机上标识JSON格式的请求文档。
    -F， - form @ file.json 在本地计算机上标识JSON格式的表单数据。
    -H 定义以下一个或两个：

    • 请求文档的内容类型

    • 自定义标头X-ID-TENANT-NAME ，用于标识标识域
    -一世 显示响应标头信息。
    -u， - 用户名：密码 指定<ProductName>帐户的用户名和密码。
    -X 指示请求的类型（例如，GET，POST等）。

以下是cURL命令的示例语法：

curl -i -X GET -u <用户名>：<密码> -H <请求标头>：<值> https：// <子域>。<域> .com / <路径> / <资源路径>
