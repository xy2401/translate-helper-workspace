命名空间：gv

OM 。 GV

包含共享的各种配置属性的静态类
在同一页面上的多个Map实例之间。

在大多数情况下，应用程序应该只对这些公共属性进行更改
在它创建OM.Map的第一个实例之前。

会员

<static> unitFactors

定义距离单位及其转换因子的常数

方法

<static> addCSTransformFunction（fromSrid，toSrid，func）

添加自定义坐标系转换功能。

参数：
名称 类型 描述
fromSrid INT

原始SRID。
toSrid INT

目标SRID。
FUNC 功能

变换功能。
返回：

空值

添加customSpatial Reference System定义。

参数：
名称 类型 描述
Ť 串

自定义SRS的类型，可能的值是'GEODETIC'和'PROJECTED'
ü 串

当前srid的单位。
小号 数

自定义SRS的srid。

<static> setBaseMapViewerURL（mvUrl）

将完整URL设置为基本MapViewer实例。基础MapViewer就在哪里
API将解析对服务器端预定义样式，主题和切片层的引用。例如，如果a
添加预定义的矢量图层时未明确指定MapViewer URL，
那么API将尝试从基础MapViewer加载图层。
它也是客户端API将发送空间坐标转换请求的地方
客户端不支持特定转换。
默认值设置为document.location.protocol +“//"+document.location.host+"/mapviewer”。

参数：
名称 类型 描述
mvUrl 串

基础MapViewer的完整URL。它必须开始
使用http或https并以/ mapviewer结尾。

<static> setHttpMethod（method）

默认情况下，Oracle Maps将使用http POST发送请求。
如果需要，使用此方法将其更改为GET。

参数：
名称 类型 描述
方法 串

“POST”或“GET”

<static> setLogLevel（level）

设置全局日志记录级别。必须是以下列出的值之一
他们越来越重视或严重：
“最好”，“更精细”，“精细”，“信息”，“配置”，“警告”和“严重”。

以低于设置级别的优先级记录的内部日志消息将不会
显示在浏览器控制台中。

参数：
名称 类型 描述
水平 串

要使用的全局日志记录级别。

<static> setProxyPath（path）
