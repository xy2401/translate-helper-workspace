1.157 JAVA_SOFT_SESSIONSPACE_LIMIT

JAVA_SOFT_SESSIONSPACE_LIMIT指定（以字节为单位）会话中Java内存使用的软限制 ，作为在用户的会话持续时间Java状态使用过多内存时发出警告的方法。

属性 描述

参数类型

整数

默认值

0

可修改

没有

可在PDB中修改

没有

价值范围

0到2 GB - 1

基本

没有

Java会话空间是将Java状态从一个数据库调用保存到另一个数据库的内存。当用户的会话持续时间Java状态超过JAVA_SOFT_SESSIONSPACE_LIMIT指定的大小时，Oracle会生成一条警告，该警告将进入跟踪文件。
