使用ASMCA管理磁盘组以进行早期数据库发布

在Oracle Grid Infrastructure安装上安装早期Oracle数据库和Oracle RAC数据库时，使用Oracle ASM Configuration Assistant（ASMCA）创建和修改磁盘组。

从Oracle Database 11g第2版开始，Oracle ASM作为Oracle Grid Infrastructure安装的一部分与Oracle Clusterware一起安装。您无法再使用Database Configuration Assistant（DBCA）在Oracle ASM上执行管理任务。
