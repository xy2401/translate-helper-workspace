1.1关于Oracle Objects

Oracle对象类型是用户定义的类型，可以将实际实体（如客户和采购订单）建模为数据库中的对象。

可以从任何内置数据库类型和任何先前创建的对象类型，对象引用和集合类型创建新对象类型。对象类型可以处理复杂数据，例如图像，音频和视频。Oracle数据库将用户定义类型的元数据存储在可用于SQL，PL / SQL，Java和其他语言的模式中。

对象类型和相关的面向对象特性（如varrays和嵌套表）提供了组织和访问数据库中数据的更高级方法。在对象层下面，数据仍然存储在列和表中，但您可以根据使数据有意义的真实实体来处理数据。查询数据库时，您可以简单地选择已创建的实体，例如客户和采购订单，而不是在查询数据库时考虑列和表。

您可以开始使用面向对象的功能，同时继续以关系方式处理大多数数据，或者完全使用面向对象的方法。

对象类型也称为用户定义类型或ADT。Oracle Database PL / SQL语言参考通常将它们称为ADT。
