最佳灵活架构文件路径示例

查看符合Optimal Flexible Architecture的安装的分层文件映射示例。

此示例显示了具有三个Oracle主目录和三个数据库的最佳灵活体系结构兼容安装，以及集群安装和Oracle Grid Infrastructure的独立服务器安装之间的部署路径差异示例。数据库文件分布在三个挂载点： /u02 ， /u03和/u04 。

注意：

  • Grid主页是用于独立服务器部署（Oracle Restart）的Oracle Grid Infrastructure的Grid主机的示例，或用于群集部署的Oracle Grid Infrastructure的Grid主目录（Oracle Clusterware）。您可以拥有Oracle Restart部署或Oracle Clusterware部署。您不能同时部署这两个选项。

  • Oracle自动存储管理（Oracle ASM）作为Oracle Grid Infrastructure安装的一部分包含在内。Oracle建议您使用Oracle ASM提供更高的冗余和吞吐量。

/ U01 /应用程序/ oraInventory的

/ U01 /应用程序/ ORACLE /

用户oracle Oracle基目录。服务器上可以安装许多Oracle数据库，以及许多Oracle数据库软件安装所有者。

Oracle安装所有者拥有的Oracle软件主目录应位于Oracle软件安装所有者的Oracle基本目录中，除非该Oracle软件是为集群部署的Oracle Grid Infrastructure。
