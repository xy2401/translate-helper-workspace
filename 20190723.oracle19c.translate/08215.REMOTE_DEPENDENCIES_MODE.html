1.280 REMOTE_DEPENDENCIES_MODE

REMOTE_DEPENDENCIES_MODE指定Oracle应如何处理远程PL / SQL存储过程的依赖关系。

属性 描述

参数类型

串

句法

REMOTE_DEPENDENCIES_MODE = {TIMESTAMP |签名}

默认值

TIMESTAMP

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

值

  • TIMESTAMP

    运行该过程的客户端将服务器端过程中记录的时间戳与本地过程的当前时间戳进行比较，并仅在时间戳匹配时执行该过程。

  • 签名

    只要签名被认为是安全的，Oracle就允许执行该过程。此设置允许运行客户端PL / SQL应用程序而无需重新编译。
