前言

约定

网关和Oracle环境的输入和输出示例以特殊字体显示：

C：\> mkdir \ ORACLE \ your_name

所有输出都会显示。对于输入，约定列表及其含义如下：

  • example tex t：单词或短语，例如mkdir和ORACLE ，必须完全拼写并在所示的字母大小写中输入。在此示例中， mkdir必须以小写字母输入， ORACLE必须以大写字母输入。

  • 斜体文本 ：斜体大写或小写，例如your_name ，表示必须替换单词或短语，例如实际目录名称。

  • bold italic TEXT BOLD text或bold italic TEXT ：粗体字或短语是指文件或目录结构，例如目录，路径或文件ID。

  • ...：省略号表示前面的项目可以重复。您可以输入任意数量的类似项目。

  • {}：大括号表示需要一个附带的参数。不要自己进入牙箍。

  • |：垂直线分隔选择。

  • []：方括号括起可选子句，您可以从中选择一个或不选择。不要自己输入括号。

除非另有说明，否则必须如图所示输入其他标点符号，例如逗号，引号或管道符号（|）。目录名称，文件ID等显示在示例中的必需字母大小写中。当这些名称出现在文本中时使用相同的约定，并且名称以粗体突出显示 。使用斜体字的指示出现在斜体文件ID的那些部分可以变化。

网关命令，文件ID保留字，MS-DOS命令，关键字和环境变量在示例和文本中以大写字母显示。必须始终输入保留字，如图所示;他们在Oracle系统中保留了意义。

SQL * Plus提示

SQL * Plus提示符SQL>出现在SQL语句和SQL * Plus命令示例中。在提示符下输入您的回复。请勿在响应中输入提示SQL>的文本。

MS-DOS提示

MS-DOS命令示例中出现MS-DOS提示符C:\> 。在提示符下输入您的回复。请勿在回复中输入提示文本C:\> 。

PGAU提示

PGAU提示符PGAU>出现在PGAU命令示例中。在提示符下输入您的回复。请勿在回复中输入提示文本PGAU> 。

目录名称

在本文档中，引用了与产品相关的文件所在的目录。%ORACLE_HOME%用于表示Oracle主目录。这是Oracle产品的默认位置。如果已安装到%ORACLE_HOME%以外的位置，请将所有对%ORACLE_HOME%引用替换为您使用的驱动器和路径规范。
