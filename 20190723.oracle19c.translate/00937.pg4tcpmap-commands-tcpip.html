准备填充PGA_TCP_IMSC表

如果您的网关正在使用对IMS Connect的TCP / IP支持，那么你必须使用 pg4tcpmap工具提示 PGAINIT提供所需的TCP / IP参数作为输入。

必须在执行任何PL / SQL网关语句之前运行pg4tcpmap工具才能填充 PGA_TCP_IMSC表，它利用相应的TIP。

请注意，如果pg4tcpmap工具共享相同的IMS Connect属性，则pg4tcpmap为其他IMS事务重新运行pg4tcpmap工具。

在您的网关配置期间，在基于UNIX的系统上执行Microsoft Windows上的%ORACLE_HOME%\dg4appc\admin\pgaimsc.sql脚本或$ORACLE_HOME/dg4appc/admin/pgaimsc.sql脚本时，会创建PGA_TCP_IMSC表。如果您需要有关创建PGA_TCP_IMSC表的更多信息，那么根据您的平台，请参阅适用于POWER Systems（64位）上的IBM AIX的Oracle数据库网关APPC安装和配置指南的第13章，Linux x86-64，Oracle SPARC上的Solaris（64位）和HP-UX Itanium或适用于Microsoft Windows的APPC安装和配置指南的Oracle Database Gateway的第10章。

概观

在PGAINIT过程调用中，用户必须指定侧面配置文件名称和TP名称。这些参数的值将插入到名为的表中 PGA_TCP_IMSC 。

在运行网关映射工具之前配置用户标识和密码

在执行pg4tcpmap工具之前，必须为PGA_TCP_IMSC表所在的Oracle数据库配置有效的用户标识和密码以及TNSNAMES别名。您必须在中指定用户标识，密码和数据库 PGA_TCP_USER ， PGA_TCP_PASS和PGA_TCP_DB参数分别位于Microsoft Windows的网关初始化文件%ORACLE_HOME%\dg4appc\admin\init sid .ora和基于UNIX的系统的$ORACLE_HOME/dg4appc/admin/initsid.ora中。

在运行pg4tcpmap工具之前

在运行网关之前，请按照以下步骤准备运行pg4tcpmap工具。

  1. 为Oracle数据库设置ORACLE_HOME和ORACLE_SID 。
  2. 确保已在Oracle数据库中创建用户PGAADMIN ，并且您可以与数据库通信。问题：

    Microsoft Windows上的%ORACLE_HOME%\dg4appc\admin\pgacr8au.sql

    要么，

    基于UNIX的系统上的$ORACLE_HOME/dg4appc/admin/pgacr8au.sql

  3. 该 init sid .ora文件必须包含适当的参数。设置以下参数：

    • PGA_TCP_USER

    • PGA_TCP_PASS

    • PGA_TCP_DB

    • 如果要启用跟踪，还需要设置以下参数：

      • TRACE_LEVEL = 255

      • LOG_DESTINATION = <有效目录>

      有关跟踪的信息，请参阅故障排除 。

  4. 确保已创建PGA_TCP_IMSC表。问题：

    Microsoft Windows上的%ORACLE_HOME%\dg4appc\admin\pgaimsc.sql

    要么，

    基于UNIX的系统上的$ORACLE_HOME/dg4appc/admin/pgaimsc.sql

pg4tcpmap工具命令

pg4tcpmap工具有两个命令：

  • 一个命令在PGA_TCP_IMSC表中插入一行;

  • 另一个命令从表中删除一行，用户必须将谓词指定为“Side Profile Name”。

将行插入PGA_TCP_IMSC表

对于Microsoft Windows，从网关Oracle主目录%ORACLE_HOME%\bin目录发出以下命令：

C：\> pg4tcpmap

对于基于UNIX的系统，请从网关Oracle home $ORACLE_HOME/bin目录发出以下命令：

$ pg4tcpmap

将显示网关版本号，版权信息以及以下文本：

此工具获取IMS Connect TCP / IP信息，例如主机名和端口号，并将它们映射到您的TIP。您可以使用此工具插入或删除IMS Connect TCP / IP信息。如果要插入行，请键入“I”如果要删除行，请键入“D”

输入<i> ，然后只需输入所需的参数。

从PGA_TCP_IMSC表中删除行

对于Microsoft Windows，从网关Oracle主目录%ORACLE_HOME%\bin目录发出以下命令：

C：\> pg4tcpmap

对于基于UNIX的系统，请从网关Oracle home $ORACLE_HOME/bin目录发出以下命令：

$ pg4tcpmap

将显示网关版本号，版权信息以及以下文本：

此工具获取IMS Connect TCP / IP信息，例如主机名和端口号，并将它们映射到您的TIP。您可以使用此工具插入或删除IMS Connect TCP / IP信息。如果要插入行，请键入“I”如果要删除行，请键入“D”

输入<d> ，pg4tcpmap工具将询问您要删除的侧面配置文件名称。

如果该行不存在，您将收到ORA-1403错误消息。

注意：

不要使用SQL * Plus来更新PGA_TCP_IMSC表。如果表中存在问题或数据不正确， %ORACLE_HOME%\dg4appc\admin\pgaimsc.sql在Microsoft Windows上使用%ORACLE_HOME%\dg4appc\admin\pgaimsc.sql或在基于UNIX的系统上使用$ORACLE_HOME/dg4appc/admin/pgaimsc.sql重新创建表及其索引。
