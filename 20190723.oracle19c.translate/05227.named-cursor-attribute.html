13.43命名游标属性

每个命名游标（显式游标或游标变量）都有四个属性，每个属性都返回有关DML语句执行的信息。

注意：

您只能在过程语句中使用游标属性，而不能在SQL语句中使用游标属性。

语义

named_cursor_attribute

％开了

如果游标打开，则named_cursor %ISOPEN的值为TRUE如果未打开，则为FALSE 。

％FOUND

named_cursor %FOUND具有以下值之一：

  • 如果光标未打开，则为INVALID_CURSOR

  • 如果游标已打开但未尝试获取，则为NULL 。

  • 如果最近的提取返回一行， TRUE 。

  • 如果最近的提取没有返回一行， FALSE 。

％未找到

named_cursor %NOTFOUND具有以下值之一：

  • 如果游标未打开，则为INVALID_CURSOR 。

  • 如果游标已打开但未尝试获取，则为NULL 。

  • 如果最近的fetch返回一行， FALSE 。

  • 如果最近的提取没有返回一行， TRUE 。

％ROWCOUNT

named_cursor %ROWCOUNT具有以下值之一：

  • 如果游标未打开，则为INVALID_CURSOR 。

  • 如果游标打开，则到目前为止获取的行数。

named_cursor

explicit_cursor

显式游标的名称。

cursor_parameter

正式游标参数的名称。

cursor_variable

游标变量的名称。

： host_cursor_variable

在PL / SQL主机环境中声明并作为绑定变量传递给PL / SQL的游标变量的名称。不要在冒号（:)和host_cursor_variable之间放置空格。
