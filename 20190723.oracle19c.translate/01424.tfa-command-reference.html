E Oracle Trace File Analyzer安装程序，命令行和Shell选项

跟踪文件分析器控制实用程序TFACTL是Oracle跟踪文件分析器的命令行界面。

TFACTL为Oracle Trace File Analyzer命令提供命令行和shell接口，用于：

  • 管理

  • 总结和分析

  • 诊断收集

您可以运行的tfactl命令取决于您的访问级别。

  • 您需要root访问权限或sudo访问tfactl才能运行管理命令。

  • 运行命令子集：

    • Oracle数据库主所有者或Oracle Grid Infrastructure主所有者

    • OS DBA或ASM组的成员

    通过将命令作为Oracle数据库主所有者或Oracle Grid Infrastructure主所有者运行，您可以访问摘要，分析和诊断收集功能。

授予其他用户访问tfactl ：

tfactl访问

要使用tfactl作为命令行工具：

tfactl command [ options ]

要使用tfactl作为shell接口：

tfactl

shell启动后根据需要输入命令。

$ tfactl tfactl>

将-help选项附加到任何tfactl命令以获取特定于命令的帮助。 

$ tfactl command --help

    使用这些命令可以查看Oracle Trace File Analyzer的部署和状态摘要，以及Oracle Trace File Analyzer检测到的更改和事件。
  • 运行诊断收集命令
    运行诊断收集命令以收集诊断数据。
