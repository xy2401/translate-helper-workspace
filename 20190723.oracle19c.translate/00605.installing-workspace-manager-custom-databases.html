A安装工作区管理器与自定义数据库

默认情况下，Workspace Manager安装在种子数据库和Database Configuration Assistant（DBCA）创建的所有数据库中。但是，在所有其他Oracle数据库（例如使用自定义过程创建的数据库）中，必须先安装Workspace Manager，然后才能使用其功能。

要在自定义数据库中安装Workspace Manager，请执行以下操作：

  1. 在系统命令提示符下，将当前目录更改为包含Workspace Manager安装脚本和程序包的目录，如以下示例所示：

    cd <ORACLE_HOME> / rdbms / admin

  2. 使用以下格式的命令以SYS身份连接到Oracle实例：

    sqlplus sys

    出现提示时，输入SYS帐户的密码。

  3. 运行owminst.plb脚本：

    SQL> @ owminst.plb

  4. 通过在作为任何有效数据库用户连接时输入以下命令来验证Workspace Manager的安装，并确保输出如下所示：

    SQL>从dual中选择dbms_wm.getWorkspace; GETWORKSPACE ------------------------------------------------- --------------------------- LIVE
