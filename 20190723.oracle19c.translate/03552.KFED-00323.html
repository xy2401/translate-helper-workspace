43 KFED-00323至KFED-00403

KFED-00323：设备[ string ]的标头具有无效的块校验和。

原因：磁盘头已损坏数据。块校验和无效。

操作：联系Oracle支持服务。

KFED-00324：设备[ string ]的标头具有无效的标签校验和。

原因：标签块已损坏数据。标签校验和无效。

操作：联系Oracle支持服务。

KFED-00325：磁盘[ string ]属于磁盘组[ string ]。

原因：要标记的磁盘是ASM磁盘组的一部分。

操作：从磁盘组中删除磁盘或使用-force选项覆盖标签。

KFED-00326：磁盘[ string ]有一个标签[ string ]。使用-force覆盖。

原因：磁盘标签不是空白。

操作：使用-force选项覆盖磁盘标签。

KFED-00327：磁盘[ string ]有一个站点标签[ string ]。使用-force覆盖。

原因：网站标签不是空白。

操作：使用-force选项覆盖站点标签。

KFED-00328：磁盘[ string ]有一个失败组标签[ string ]。使用-force覆盖。

原因：失败组标签不为空。

操作：使用-force选项覆盖失败组标签。

KFED-00329：指定的磁盘标签[ string ]大于最大大小[ number ]。

原因：提供的磁盘标签太大。

操作：提供较短的磁盘标签。

KFED-00330：网站标签[ string ]无效。

原因：提供了无效的站点标签。

操作：指定一个站点名称，其长度至少为一个字符但不超过15个字符。站点名称由字母数字字符和连字符（a..z，A ..Z，0..9， - ）。它不能以数字字符开头。它不能以连字符（ - ）开头或结尾。

KFED-00331：指定的失败组标签[ string ]大于最大大小[ number ]。

原因：提供的失败组标签太大。

操作：提供更短的失败组标签。

KFED-00332：设备[ string ]未格式化为ASM磁盘。

原因：提供的设备未被识别为ASM磁盘。

操作：将磁盘添加到ASM发现路径。

KFED-00333：设备[ string ]是Exadata磁盘。

原因：尝试标记Exadata磁盘。

操作：指定不受Exadata管理的设备。

KFED-00334：发现磁盘string出错

原因： kfed无法根据提供的发现字符串完成磁盘发现。

操作：有关详细信息，请参阅随附的错
