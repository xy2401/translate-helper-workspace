前言

Oracle数据库XML C API参考描述了用于C编程语言的Oracle XML Developer's Kits（XDK）和Oracle XML DB API。它主要列出了与这些API相关的函数，方法和过程的语法。

听众

Oracle数据库XML C API参考适用于在Oracle中构建XML应用程序的开发人员。

要使用本文档，您需要对面向对象编程概念有基本的了解，熟悉结构化查询语言（SQL）以及使用C编程语言进行应用程序开发的工作知识。
