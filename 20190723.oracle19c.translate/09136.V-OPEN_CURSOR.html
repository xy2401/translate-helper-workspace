8.103 V $ OPEN_CURSOR

V$OPEN_CURSOR列出了每个用户会话当前已打开并解析或缓存的游标。

柱 数据类型 描述

SADDR

RAW（4 | 8）

会话地址

SID

数

会话标识符

用户名

VARCHAR2（128）

登录会话的用户

地址

RAW（4 | 8）

与HASH_VALUE一起使用以唯一标识会话中正在执行的SQL语句

HASH_VALUE

数

与ADDRESS一起使用以唯一标识会话中正在执行的SQL语句

SQL_ID

VARCHAR2（13）

在会话中执行的SQL语句的SQL标识符

SQL_TEXT

VARCHAR2（60）

解析为打开游标的SQL语句的前60个字符

LAST_SQL_ACTIVE_TIME

日期

上次执行此游标的时间

SQL_EXEC_ID

数

如果正在执行打开的游标，则执行该执行的SQL执行标识符（请参阅V$SQL_MONITOR ）

CURSOR_TYPE

VARCHAR2（64）

游标类型：

  • OPEN PL/SQL - 打开PL / SQL游标

  • OPEN - 其他开放游标

  • SESSION CURSOR CACHED - 游标缓存在通用会话游标缓存中

  • OPEN-RECURSIVE - 打开递归游标

  • DICTIONARY LOOKUP CURSOR CACHED - 游标缓存在字典查找游标缓存中

  • BUNDLE DICTIONARY LOOKUP CACHED - 游标缓存在捆绑的字典查找游标缓存中

  • JAVA NAME TRANSLATION CURSOR CACHED - 游标缓存在Java名称转换游标缓存中

  • REPLICATION TRIGGER CURSOR CACHED - 复制触发器游标缓存中缓存的游标

  • CONSTRAINTS CURSOR CACHED - 游标缓存在约束游标缓存中

  • PL/SQL CURSOR CACHED - 缓存在PL / SQL游标缓存中的游标

CHILD_ADDRESS

RAW（4 | 8）

子光标的地址

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
