安装向导用于创建映像的安装选项

在启动Oracle数据库或Oracle Grid Infrastructure安装的安装向导之前，请确定是否要使用任何可用的映像创建选项。

在基于映像的安装中，您可以分别通过运行安装向导runInstaller和gridSetup.sh来启动Oracle数据库安装或Oracle Grid Infrastructure安装。这两个向导都带有以下图像创建选项。

表2-1安装向导的映像创建选项

选项 描述

-createGoldImage

从当前的Oracle主目录创建黄金映像。

-destinationLocation

指定将在其中创建黄金映像的完整路径或位置。

-exclFiles

指定要从新创建的黄金映像中排除的文件的完整路径。

-救命

显示所有可用选项的帮助。

例如：

./runInstaller-createGoldImage -destinationLocation / tmp / my_db_images
./gridSetup.sh -createGoldImage -destinationLocation / tmp / my_grid_images

哪里：

/tmp/my_db_images是创建映像zip文件的临时文件位置。

/tmp/my_grid_images是创建映像zip文件的临时文件位置。

/u01/app/oracle/product/19.0.0/dbhome_1/relnotes是要在新创建的黄金映像中排除的文件。
