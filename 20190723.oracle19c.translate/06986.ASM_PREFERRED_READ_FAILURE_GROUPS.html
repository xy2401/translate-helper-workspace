1.21 ASM_PREFERRED_READ_FAILURE_GROUPS

ASM_PREFERRED_READ_FAILURE_GROUPS指定包含首选读取磁盘的故障组。 首选磁盘是特定于实例的。

参数类型

串

ASM_PREFERRED_READ_FAILURE_GROUPS ='<diskgroup_name>。<failure_group_name>，...'

空值

可修改

没有

基本

没有

Oracle RAC

不同节点上的值不同

注意：

Oracle Database 12 c第2版（12.2.0.1）中不推荐使用ASM_PREFERRED_READ_FAILURE_GROUPS初始化参数，并且可能在将来的版本中不支持该参数。它由PREFERRED_READ.ENABLED磁盘组属性替换。

有关PREFERRED_READ.ENABLED磁盘组属性的详细信息，请参见“ Oracle自动存储管理管理员指南” 。

要将参数值重置为null，请发出以下语句，该语句将更新服务器参数文件（SP文件），然后重新引导Oracle ASM以使更改生效：

alter system reset asm_preferred_read_failure_groups;
