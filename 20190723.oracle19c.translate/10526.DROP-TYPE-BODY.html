DROP TYPE BODY

目的

对象类型使用PL / SQL定义。有关创建，更改和删除对象类型的完整信息，请参阅Oracle Database PL / SQL语言参考 。

使用DROP TYPE BODY语句删除对象类型，varray或嵌套表类型的主体。删除类型主体时，对象类型规范仍然存在，您可以重新创建类型主体。在重新创建主体之前，您仍然可以使用对象类型，尽管您无法调用成员函数。

先决条件

对象类型主体必须位于您自己的架构中，或者您必须具有DROP ANY TYPE系统特权。

句法

drop_type_body :: =


插图说明drop_type_body.eps

语义

模式

指定包含对象类型的架构。如果省略schema ，则Oracle数据库假定对象类型在您自己的模式中。

TYPE_NAME

指定要删除的对象类型主体的名称。

对跌落型身体的限制

只有在没有类型或表依赖关系的情况下，才可以删除类型主体。

例子

删除对象类型主体：示例

以下语句删除对象类型主体data_typ1 。有关创建此对象类型的示例，请参见Oracle Database PL / SQL语言参考 。 

DROP TYPE BODY data_typ1;
