8.68 V $ MAP_FILE

V$MAP_FILE显示实例共享内存中所有文件映射结构的列表。

柱 数据类型 描述

FILE_MAP_IDX

数

与文件对应的索引

FILE_CFGID

VARCHAR2（256）

配置ID（如果不支持配置ID，则为N / A）

FILE_STATUS

VARCHAR2（7）

映射信息的状态：

  • VALID - 文件映射信息是最新的

  • INVALID - 必须刷新映射

文件名

VARCHAR2（256）

绝对文件名

文件类型

VARCHAR2（11）

文件类型：

  • DATAFILE

  • SPFILE

  • TEMPFILE

  • CONTROLFILE

  • LOGFILE

  • ARCHIVEFILE

FILE_STRUCTURE

VARCHAR2（9）

文件结构：

  • 文件

  • RAWVOLUME

  • 裸设备

  • 没有

文件大小

数

文件大小以HKB（半KB）为单位

FILE_NEXTS

数

文件中的文件范围数（不一定与映射的文件范围数相同）

LIB_IDX

数

声明文件所有权的映射库索引

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
