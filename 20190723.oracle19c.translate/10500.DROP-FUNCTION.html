降功能

目的

使用PL / SQL定义函数。有关创建，更改和删除功能的完整信息，请参阅Oracle数据库PL / SQL语言参考 。

使用DROP FUNCTION语句从数据库中删除独立存储的函数。

注意：

请勿使用此语句删除属于包的函数。相反，要么使用DROP PACKAGE语句删除整个包，要么使用带有OR REPLACE子句的CREATE PACKAGE语句重新定义包而不使用该函数。

先决条件

该函数必须位于您自己的架构中，或者您必须具有DROP ANY PROCEDURE系统特权。

句法

drop_function :: =


说明drop_function.eps的描述

语义

模式

指定包含该函数的模式。如果省略schema ，则Oracle数据库假定该函数位于您自己的模式中。

FUNCTION_NAME

指定要删除的函数的名称。

Oracle数据库使依赖或调用已删除函数的任何本地对象无效。如果随后引用其中一个对象，则数据库会尝试重新编译该对象，如果尚未重新创建已删除的函数，则会返回错误。

如果任何统计类型与该函数关联，则数据库将使用FORCE选项取消关联统计类型，并删除使用统计类型收集的任何用户定义的统计信息。

例子

删除功能：示例

以下语句删除示例模式oe的函数SecondMax ，并使依赖于SecondMax所有对象无效：

DROP FUNCTION oe。SecondMax;
