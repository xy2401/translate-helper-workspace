D.4 chactl配置

使用chactl config命令列出所有受监视的目标，以及每个目标的当前模型。

如果指定的目标是多租户容器数据库（CDB）或集群，则chactl config命令还会显示配置数据状态。

句法

chactl config {cluster | database -db db_unique_name } [ -  help]

例子

要显示监视器配置和每个目标的指定模型：

$ chactl config监控的数据库：prodDB，hrDB
$ chactl config database -db prodDB Monitor：启用模型：GoldDB
$ chactl config cluster监控：启用模型：DEFAULT_CLUSTER
