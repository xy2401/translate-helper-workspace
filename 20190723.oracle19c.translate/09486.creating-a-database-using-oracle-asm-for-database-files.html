使用Oracle ASM为数据库文件创建数据库

在创建使用Oracle自动存储管理（Oracle ASM）的数据库之前，必须运行root.sh脚本。

因此，在静默模式安装期间，无法使用Oracle ASM创建数据库作为数据库文件的存储选项。相反，您可以使用静默模式完成纯软件安装，然后在完成纯软件安装并运行root.sh脚本后以静默方式运行Oracle Net Configuration Assistant和DBCA。

注意：

此限制仅适用于使用Oracle自动存储管理作为数据库文件的存储选项的数据库。您可以在静默模式安装期间创建使用文件系统选项的数据库。
