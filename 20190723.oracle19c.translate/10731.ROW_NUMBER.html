目的

ROW_NUMBER是一个分析函数。它为应用它的每一行（分区中的每一行或查询返回的每一行）分配一个唯一的编号，在order_by_clause指定的有序行序列中，从1开始。

通过在查询中使用ROW_NUMBER嵌套子查询来检索指定范围的ROW_NUMBER值，您可以从内部查询的结果中找到精确的行子集。使用该功能可以实现前N，下N和内N报告。为了获得一致的结果，查询必须确保确定性排序顺序。

例子

以下示例在hr.employees表中查找每个部门中三个最高薪酬的员工。员工少于三人的部门返回的行数少于三行。 

SELECT department_id，first_name，last_name，salary FROM（SELECT department_id，first_name，last_name，salary，ROW_NUMBER（）OVER（PARTITION BY department_id ORDER BY salary desc）rn FROM employees）WHERE rn <= 3 ORDER BY department_id，salary DESC，last_name;
