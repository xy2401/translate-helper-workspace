支持的SPARC Oracle Solaris 11发行版（64位）

检查支持的Oracle Solaris 11发行版和其他操作系统要求。

表4-1适用于SPARC（64位）最低操作系统要求的Oracle Solaris 11发行版

项目 要求

SSH要求

在安装Oracle Solaris时配置Secure Shell。

Oracle Solaris 11操作系统

Oracle Solaris 11.4（Oracle Solaris 11.4.2.0.1.3.0）或更高版本的SRU和更新

Oracle Solaris 11.3 SRU 31（Oracle Solaris 11.3.31.6.0）或更高版本的SRU和更新

适用于Oracle Solaris 11的软件包

必须安装以下软件包：

PKG：//的Solaris /系统/库/ OpenMP的
PKG：//的Solaris /压缩/解压缩
PKG：//的Solaris /开发商/汇编
PKG：//的Solaris /开发/建设/制造
PKG：//的Solaris /系统/ dtrace将
PKG：//的Solaris /系统/报头
PKG：//的Solaris /系统/库
PKG：//的Solaris /系统/接头
pkg：// solaris / system / xopen / xcu4（如果尚未作为标准Oracle Solaris 11安装的一部分安装）
PKG：//的Solaris / X11 /诊断/ X11-信息的客户端
pkg：// solaris / system / kernel / oracka（仅适用于Oracle Real Application Clusters安装）

注意：从Oracle Solaris 11.2开始，如果您已执行标准的Oracle Solaris 11安装，并安装了Oracle数据库先决条件组软件包oracle-rdbms-server-18c-preinstall ，那么您不必安装这些软件包，如oracle-rdbms-server-18c-preinstall会为您安装它们。
