第二部分在Oracle XML DB中操作XML数据

此处介绍了以下内容：XQuery，XMLType操作和XML数据索引。

  • XQuery和Oracle XML DB
    XQuery语言是您与Oracle XML DB中的XML数据交互的主要方式之一。对该语言的支持包括SQL * Plus命令XQUERY和SQL / XML函数XMLQuery ， XMLTable ， XMLExists和XMLCast 。
  • 查询和更新XML数据
    应用程序有许多方法可以查询和更新Oracle数据库中的XML数据，包括基于XML模式和非基于模式的数据。
  • XMLType数据的索引
    您可以在XML数据上创建索引，专注于经常查询的特定部分，从而提高性能。您可以通过多种方式索引XMLType数据，无论是基于XML模式还是非基于模式，无论您使用何种XMLType存储模型。
  • XMLType数据的转换和验证
    有几个Oracle SQL函数和XMLType API，用于使用XSLT样式表转换XMLType数据，以及根据XML模式验证XMLType实例。
