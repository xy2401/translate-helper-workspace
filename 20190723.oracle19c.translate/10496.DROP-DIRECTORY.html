DROP DIRECTORY

目的

使用DROP DIRECTORY语句从数据库中删除目录对象。

也可以看看：

有关创建目录的信息，请创建DIRECTORY

先决条件

要删除目录，您必须具有DROP ANY DIRECTORY系统特权。

注意：

当PL / SQL或OCI程序访问关联文件系统中的文件时，请勿删除目录。

句法

drop_directory :: =


插图说明drop_directory.eps

语义

目录名

指定要删除的目录数据库对象的名称。

Oracle数据库删除目录对象，但不删除服务器文件系统上的关联操作系统目录。

例子

删除目录：示例

以下语句删除目录对象bfile_dir ：

DROP DIRECTORY bfile_dir;
