1.2更改命令行字体和字体大小

在Windows中，从命令提示符打开“命令提示符属性”对话框，以设置SQL * Plus命令行界面中使用的字体和字体大小。

更改命令行界面字体和字体大小

  1. 右键单击命令行界面标题栏。
  2. 单击属性。“窗口预览”框根据您选择的字体和字体大小显示当前窗口在显示器上的相对大小。“选定字体：”框显示当前字体的样本。
  3. 单击“字体”选项卡。
  4. 从“大小”框中选择要使用的字体大小。光栅字体大小显示为高度宽度（以像素为单位）。TrueType字体大小显示为高度（以像素为单位）。
  5. 从“字体”框中选择要使用的字体。
  6. 选中粗体字体复选框以使用粗体版本的字体。

有关更改命令提示符属性的详细信息，请参阅Windows帮助或单击“命令提示符属性”对话框中的“帮助”。
