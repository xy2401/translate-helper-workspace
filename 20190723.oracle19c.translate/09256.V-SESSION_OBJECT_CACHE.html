9.24 V $ SESSION_OBJECT_CACHE

V$SESSION_OBJECT_CACHE显示本地服务器（实例）上当前用户会话的对象缓存统计信息。

柱 数据类型 描述

销

数

缓存中的对象引脚数或查找数

HITS

数

找到已存在于缓存中的对象的对象引脚数

TRUE_HITS

数

找到对象已经在缓存中并处于所需状态的对象引脚数（因此，不需要从数据库刷新）

HIT_RATIO

数

HITS / PINS比率

TRUE_HIT_RATIO

数

TRUE_HITS / PINS比率

OBJECT_REFRESHES

数

使用数据库中的新值刷新的缓存中的对象数

CACHE_REFRESHES

数

刷新整个缓存（所有对象）的次数

OBJECT_FLUSHES

数

刷新到数据库的缓存中的对象数

CACHE_FLUSHES

数

整个缓存（所有对象）刷新到数据库的次数

CACHE_SHRINKS

数

缓存缩小到最佳大小的次数

CACHED_OBJECTS

数

当前缓存的对象数

PINNED_OBJECTS

数

当前固定的对象数

CACHE_SIZE

数

缓存的当前大小（以字节为单位）

OPTIMAL_SIZE

数

缓存的最佳大小（以字节为单位）

MAXIMUM_SIZE

数

最大缓存大小（以字节为单位）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
