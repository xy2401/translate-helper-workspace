演示要求

案例研究假设满足了这些要求：

  • 网关演示表安装在Informix数据库中

  • Oracle数据库有一个名为SCOTT的帐户，密码为TIGER

  • Oracle数据库有一个名为GTWLINK的数据库链接（设置为用户SCOTT公共或私有），它将网关作为SCOTT连接到Informix数据库，密码为TIGER2

    例如，您可以按如下方式创建数据库链接：

    SQL>创建数据库链接GTWLINK连接到SCOTT 2，由TIGER2使用'GTWSID'识别;

  • Oracle Net Services已正确配置并正在运行。

  • Informix环境变量INFORMIXDIR已正确设置。

创建演示表

案例研究基于GTW_EMP ， GTW_DEPT和GTW_SALGRADE表。如果尚未在Informix数据库中创建演示表，请使用用于创建它们的bldifmx.sql脚本，如下所示：

设置环境变量DELIMIDENT 。

如果您有Bourne或Korn Shell，请输入以下内容：

$ DELIMIDENT = y;导出DELIMIDENT

如果您有C Shell，请输入以下内容：

$ setenv DELIMIDENT y $ cd $ ORACLE_HOME / dg4ifmx / demo $ dbaccess database_name bldifmx.sql

该脚本相应地在Informix数据库中创建演示表：

CREATE TABLE GTW_EMP（EMPNO SMALLINT NOT NULL ENAME VARCHAR（10），JOB VARCHAR（9），MGR SMALLINT，HIREDATE DATETIME，SAL NUMERIC（7,2），COMM NUMERIC（7,2），DEPTNO SMALLINT）CREATE TABLE GTW_DEPT（DEPTNO SMALLINT NOT NULL，DNAME VARCHAR（14），LOC VARCHAR（13））创建表GTW_SALGRADE（等级金币，LOSAL NUMERIC（9,4），HISAL NUMERIC（9,4））
