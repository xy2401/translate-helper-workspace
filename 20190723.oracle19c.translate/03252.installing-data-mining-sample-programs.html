A.2安装数据挖掘示例程序

了解如何安装Data Mining示例程序。

该数据挖掘示例程序要求：

  • Oracle数据库企业版用高级分析选项

  • Oracle数据库示例模式

  • Oracle数据库示例

  • 数据挖掘用户帐户

  • 系统管理员执行dmshgrants.sql

  • 数据挖掘用户执行dmsh.sql

请按照以下步骤安装数据挖掘示例程序：

  1. 安装或获取访问权限带有Advanced Analytics选项的Oracle Database 12 c企业版。要安装数据库，请参阅Oracle Database 18c Release适用于您的平台的安装说明。

  2. 确保示例模式安装在数据库中。默认情况下，Oracle Database会安装示例模式。有关示例模式的详细信息，请参阅Oracle数据库示例模式。

  3. 验证 Oracle数据库示例已随数据库一起安装，或在本地安装。 Oracle数据库示例将Database示例程序加载到Oracle主目录下的rdbms/demo目录中。有关安装说明，请参阅Oracle数据库示例安装指南

  4. 验证数据挖掘用户帐户已创建，或者如果您具有管理权限，则自行创建。请参阅“创建数据挖掘用户” 。

  5. 问您的系统管理员要运行dmshgrants.sql ，或者如果您具有管理权限，请dmshgrants.sql运行它。 dmshgrants授予运行示例程序所需的权限。这些包括对SH模式中的表的SELECT访问，如“数据挖掘示例数据”和系统中所述下表中列出的权限。

    将数据挖掘用户的名称传递给dmshgrants 。 

    SQL> CONNECT sys / as sysdba输入密码： sys_password已连接。SQL> @ $ ORACLE_HOME / rdbms / demo / dmshgrants dmuser
