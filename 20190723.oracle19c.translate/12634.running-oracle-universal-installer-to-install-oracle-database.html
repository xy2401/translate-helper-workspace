运行Oracle数据库安装向导以安装Oracle数据库

提取数据库映像文件并使用runInstaller命令开始安装。

在开始安装之前，请准备好有关用户组和存储路径的所有信息。Oracle建议您在安装期间使用My Oracle Support凭据。您还应该准备好运行根脚本或提供自动化根脚本的信息。

  1. 以您希望拥有软件二进制文件的Oracle安装所有者用户帐户（ oracle ）身份登录。
  2. 将Oracle数据库安装映像文件（ db_home.zip ）下载到您选择的目录中。例如，您可以将图像文件下载到/tmp目录。
  3. 创建Oracle主目录并将已下载的映像文件解压缩到此Oracle主目录。例如：
    $ mkdir -p /u01/app/oracle/product/19.0.0/dbhome_1 $ chgrp oinstall /u01/app/oracle/product/19.0.0/dbhome_1 $ cd /u01/app/oracle/product/19.0.0/ dbhome_1 $ unzip -q /tmp/db_home.zip

    注意：

    Oracle建议您创建的Oracle主目录路径符合Oracle Optimal Flexible Architecture建议。此外，仅在您创建的此Oracle主目录中解压缩安装映像文件。
  4. 在Oracle主目录中，运行runInstaller命令以启动Oracle数据库设置向导。
    $ cd /u01/app/oracle/product/19.0.0/dbhome_1 $ ./runInstaller

    注意：

    仅从Oracle主目录运行runInstaller命令。请勿使用驻留在$ORACLE_HOME/oui/bin/或任何其他位置的runInstaller命令来安装Oracle数据库，Oracle数据库客户端或Oracle Grid Infrastructure。
  5. 在“选择配置选项”屏幕中，选择“ 创建并配置单个实例数据库”。
  6. 选择您的安装类型。

    安装屏幕因您选择的安装选项而异。根据需要响应配置提示。

  7. 提供信息以自动执行根脚本，或在设置向导提示时以root身份运行脚本。

    如果为运行根脚本配置自动化，并且根脚本失败，则可以手动修复问题，然后单击“ 重试”以再次运行根脚本。

    注意：

    如果您对安装期间要求提交的信息有任何疑问，请单击“ 帮助” 。
