INS-40102：指定的网格主页无效

原因：集群主目录的Oracle Grid Infrastructure（网格主目录）已放置在安装所有者主目录中。

操作： Oracle建议您不要将Grid主目录放在用户主目录中，因为该目录路径的所有权更改为root。

INS-40103：安装程序检测到指定的软件位置位于OCFS2分区上。

原因： OCFS2分区不支持为集群网格家庭配置Oracle Grid Infrastructure。

操作：指定不是OCFS2分区的网格主目录位置。

INS-40104：安装程序检测到指定的软件位置位于OCFS2分区上。

原因： OCFS2分区不支持为集群网格家庭配置Oracle Grid Infrastructure。

操作：指定不是OCFS2分区的网格主目录位置。

INS-40105：为Oracle Base指定的位置无效。

原因：安装程序检测到您已在此服务器上配置了Oracle Grid Infrastructure软件，并且您为此安装提供的Oracle基础与现有Oracle Grid Infrastructure软件Oracle基础不同。

操作：为此Oracle Grid Infrastructure安装提供与现有Oracle Grid Infrastructure相同的Oracle基本位置。

INS-40106：为Grid home指定的位置无效。

原因：安装程序检测到您提供的Oracle主目录位置位于Oracle ASM群集文件系统（ACFS）挂载点上。您不能将Oracle Grid Infrastructure二进制文件放在ACFS安装点上。

操作：指定位于ACFS装入点以外的位置的网格主位置。

INS-40107：建议您选择具有足够磁盘空间的位置，以适应将来修补程序的安装。

原因：所选的Oracle主目录位于没有足够磁盘空间的卷上，以便将来进行修补。

操作：为Oracle主目录选择具有足够空间（最小字符串 MB）或释放现有卷空间的位置。

INS-40108：安装程序检测到指定的软件位置位于临时文件系统上。

原因：不应在临时文件系统上为群集Grid Home配置Oracle Grid Infrastructure。

操作：指定不是临时文件系统的网格主页位置。
