1.322 SPFILE

此参数的值是正在使用的当前服务器参数文件（ SPFILE ）的名称。

属性 描述

参数类型

串

句法

SPFILE = spfile_name

默认值

ORACLE_HOME /dbs/spfile.ora

可修改

没有

可在PDB中修改

没有

价值范围

任何有效的SPFILE

基本

没有

Oracle RAC

多个实例必须具有相同的值。

可以在客户端PFILE中定义此参数，以指示要使用的服务器参数文件的名称。

当服务器使用默认服务器参数文件时， SPFILE的值由服务器在内部设置。

SPFILE驻留在ORACLE_HOME /dbs目录中;但是，只要在初始化参数文件中指定，用户就可以将其放在系统的任何位置。
