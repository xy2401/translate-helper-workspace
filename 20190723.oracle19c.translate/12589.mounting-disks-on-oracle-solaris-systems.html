在Oracle Solaris系统上安装磁盘

在大多数Oracle Solaris系统上，当您将磁盘插入磁盘驱动器时，磁盘会自动挂载。如果磁盘未自动装入，请按照以下步骤进行装载。

  1. 如有必要，将用户切换到root并弹出当前安装的磁盘，然后将其从驱动器中删除：
    $ sudo sh密码：#eject
  2. 将适当的安装介质插入磁盘驱动器。
  3. 验证磁盘是否已自动装入：
    #ls / dvd / dvd0

    如果此命令无法显示安装介质的内容，请输入与以下命令类似的命令进行安装：

    ＃/ usr / sbin / mount -r -F hsfs / dev / dsk / cxtydzs2 / dvd

    在此示例中， /dvd是光盘安装点目录， /dev/dsk/cxtydzs2是光盘设备的设备名称。
