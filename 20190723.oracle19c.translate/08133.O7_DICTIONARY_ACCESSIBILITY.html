1.215 O7_DICTIONARY_ACCESSIBILITY

O7_DICTIONARY_ACCESSIBILITY控制对SYSTEM权限的限制。

属性 描述

参数类型

布尔

默认值

假

可修改

没有

可在PDB中修改

是

价值范围

是的|假

基本

没有

如果参数设置为true ，则允许访问SYS模式中的对象（Oracle7行为）。false设置确保允许访问“任何模式”中对象的系统特权不允许访问SYS模式中的对象。

例如，如果将O7_DICTIONARY_ACCESSIBILITY设置为false ，则SELECT ANY TABLE权限允许访问除SYS模式之外的任何模式中的视图或表（无法访问数据字典表）。如果将O7_DICTIONARY_ACCESSIBILITY设置为false ，则要访问SYS模式中的对象，用户应具有SELECT ANY DICTIONARY系统特权，或者应该已授予用户对特定对象的SELECT对象特权。系统特权EXECUTE ANY PROCEDURE允许访问除SYS模式之外的任何模式中的过程。

如果此参数设置为false并且您需要访问SYS架构中的对象，则必须为您授予显式对象权限。可以授予数据库管理员的以下角色也允许访问字典对象：

  • SELECT_CATALOG_ROLE

  • EXECUTE_CATALOG_ROLE

注意：

Oracle Database 12 c第2版（12.2.0.1）中不推荐使用O7_DICTIONARY_ACCESSIBILITY初始化参数，并且可能在将来的发行版中不支持该参数。
