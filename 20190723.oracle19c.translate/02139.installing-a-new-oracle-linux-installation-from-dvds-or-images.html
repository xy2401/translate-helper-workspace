从DVD或映像安装新的Oracle Linux安装

使用此过程安装新的Oracle Linux安装并使用Oracle预安装RPM执行系统配置：

  1. 通过从Oracle Store订购Oracle Linux媒体包，或从Oracle Software Delivery Cloud网站下载Oracle Linux和Oracle VM的磁盘映像，获取Oracle Linux。
    • Oracle商店
      https://shop.oracle.com
    • Oracle Software Delivery Cloud网站：
      https://edelivery.oracle.com/linux
  2. 启动Oracle Linux安装并使用适合您环境的值响应安装屏幕。
  3. 查看第一个软件选择屏幕，其中列出了特定于任务的软件选项。在屏幕的底部，有一个选项可以立即自定义或稍后自定义。选择立即自定义 ，然后单击下一步 。
  4. 在Oracle Linux上，选择屏幕左侧的“ 服务器 ”和屏幕右侧的“ 系统管理”工具 （选项可能因版本而异）。
    将打开“系统工具中的包”窗口。
  5. 从包列表中选择Oracle Preinstallation RPM包框。例如，对于Oracle Linux 7，请选择类似于以下内容的包：
    Oracle的数据库预装-19C-1.0-4.el7.x86_64.rpm
    如果您没有适用于Oracle数据库发行版的Oracle预安装RPM软件包选项，因为您使用的是Oracle数据库和Oracle Grid Infrastructure版本之前的Oracle Linux安装，请在安装后手动安装RPM。完成操作系统安装。
  6. 关闭可选包窗口，然后单击“ 下一步” 。
  7. 完成其他屏幕以完成Oracle Linux安装。
    Oracle Linux自动创建标准（非角色分配）Oracle安装所有者和组，并根据Oracle安装的要求设置其他内核配置设置。
  8. 在所有其他集群成员节点上重复步骤2到6。
