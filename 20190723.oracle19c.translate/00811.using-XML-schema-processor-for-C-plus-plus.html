用于C ++的Oracle XML Schema Processor

用于C ++的XML Schema处理器是C ++可扩展标记语言（XML）解析器的配套组件，它允许将简单和复杂的数据类型支持到XML应用程序中。

用于C ++的XML Schema处理器支持万维网联盟（W3C）XML模式建议书。这使得编写处理XML文档的自定义应用程序变得简单，这意味着符合标准的XML Schema处理器是移植Oracle数据库的每个操作系统上XDK的一部分。

用于C ++特性的Oracle XML Schema

描述了用于C ++的Oracle XML Schema处理器的功能。

这些是功能：

  • 支持简单和复杂的类型

  • 基于C ++的XML解析器

  • 支持W3C XML Schema Recommendation

C ++类的XML Schema处理器是OracleXml::Parser::SchemaValidator 。

用于C ++的Oracle XML Schema Processor的标准一致性

列出了XML Schema Processor for C ++符合的标准。

  • W3C推荐的可扩展标记语言（XML）1.0

  • W3C对XML 1.0中命名空间的建议

  • W3C对XML Schema 1.0的建议
