说明fs_arch.png的描述

描述了Real Application Security架构。左边是一个标有“Web客户端”的框。它有3台垂直排列的台式电脑。右侧是标有“Application Server”的框。它有五个子盒，垂直排列。顶部子框标记为“JEE Applications”。第二个子框标记为“Java容器”。第三个子框标记为“Real Application Security Java API”。第四个子框标记为“Real Application Security”，包含两个名为“Application Session Cache”和“ACL Cache”的组件。底部子框标记为“连接池”，包含三个标记为“数据库会话1”，“数据库会话2”和“数据库会话3”的数据库会话。

右侧是标有“Oracle数据库”的圆柱体，其中包含一个标有“真实应用程序安全性”的框。在这个盒子里面是垂直排列的三个部分。第一部分标记为“ACL，应用程序权限，用户和角色”。下面是标有“应用程序会话”的部分，最后是包含四个数据库会话的部分，标记为“数据库会话1”，“数据库会话2”，“数据库会话3”和“数据库会话4”。每个数据库会话都附加到应用程序会话。数据库会话4显示了一个活动的客户端SQL * Plus会话。
