2.199 ALL_LOB_PARTITIONS

ALL_LOB_PARTITIONS显示当前用户可访问的表中的LOB分区。

相关观点

  • DBA_LOB_PARTITIONS显示数据库中的所有LOB分区。

  • USER_LOB_PARTITIONS显示当前用户拥有的LOB分区。此视图不显示TABLE_OWNER列。

柱 数据类型 空值 描述

TABLE_OWNER

VARCHAR2（128）

表的所有者

TABLE_NAME

VARCHAR2（128）

表的名称

COLUMN_NAME

VARCHAR2（4000）

LOB列的名称

LOB_NAME

VARCHAR2（128）

分区LOB项的名称

PARTITION_NAME

VARCHAR2（128）

表分区的名称

LOB_PARTITION_NAME

VARCHAR2（128）

LOB数据分区的名称

LOB_INDPART_NAME

VARCHAR2（128）

相应LOB索引分区的名称

PARTITION_POSITION

数

LOB数据分区在LOB项目中的位置

综合

VARCHAR2（3）

指示分区是否为复合（ YES ）或否（ NO ）

CHUNK

数

LOB数据分区的CHUNK属性的值

PCTVERSION

数

LOB数据分区的PCTVERSION属性的值

CACHE

VARCHAR2（10）

指示是否以及如何在缓冲区高速缓存中缓存LOB数据：

  • YES - LOB数据放在缓冲区缓存中

  • NO - LOB数据要么不被带入缓冲区高速缓存，要么被带入缓冲区高速缓存并放置在最近最少使用的LRU列表末尾

  • CACHEREADS - LOB数据仅在读取操作期间进入缓冲区高速缓存，但在写入操作期间不进入缓冲区高速缓存

在排队

VARCHAR2（3）

指示是否为LOB数据分区启用了STORAGE IN ROW属性（ YES ）（ NO ）（ NO ）

TABLESPACE_NAME

VARCHAR2（30）

包含LOB数据分区的表空间的名称

INITIAL_EXTENT

VARCHAR2（40）

LOB数据分区的初始范围的大小（以字节为单位），或DEFAULT

NEXT_EXTENT

VARCHAR2（40）

MIN_EXTENTS

VARCHAR2（40）

MAX_EXTENTS

VARCHAR2（40）

LOB数据分区的段中允许的最大扩展区数，或DEFAULT

MAX_SIZE

VARCHAR2（40）

VARCHAR2（7）

  • 默认

  • 无效

  • 是

VARCHAR2（40）

  • 没有
