D.9 chactl查询校准

使用chactl query calibration命令查看有关特定目标的校准数据的详细信息。

句法

chactl query calibration {-cluster | -db db_unique_name } [-timeranges'start = time_stamp ，end = time_stamp ，...'] [-kpiset'name = kpi_name min = val max = val ，...'] [-interval val ] [ - 帮助]

以小时为单位指定间隔。

以YYYY-MM-DD HH24:MI:SS格式指定日期和时间。

注意：

如果未指定时间间隔，则chactl query calibration命令将显示为特定目标收集的所有校准数据。

数据库支持以下关键性能指标：

  • CPUPERCENT - CPU利用率 - 百分比

  • IOREAD - 磁盘读取 - Mbyte / sec

  • DBTIMEPERCALL - 每个用户调用的数据库时间 - usec / call

  • IOWRITE - 磁盘写入 - Mbyte / sec

  • IOTHROUGHPUT - 磁盘吞吐量 - IO /秒

群集支持以下关键性能指标：

  • CPUPERCENT - CPU利用率 - 百分比

  • IOREAD - 磁盘读取 - Mbyte / sec

  • IOWRITE - 磁盘写入 - Mbyte / sec

  • IOTHROUGHPUT - 磁盘吞吐量 - IO /秒
