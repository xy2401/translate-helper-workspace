DROP ATTRIBUTE DIMENSION

目的

使用DROP ATTRIBUTE DIMENSION语句删除属性维度。ATTRIBUTE DIMENSION对象是分析视图的一个组件。

先决条件

要在自己的模式中删除属性维，您必须具有DROP ATTRIBUTE DIMENSION系统特权。要在另一个用户的模式中删除分析视图，您必须具有DROP ANY ATTRIBUTE DIMENSION系统特权。

句法

drop_attribute_dimension :: =


插图描述drop_attribute_dimension.eps

语义

模式

指定属性维存在的模式。如果未指定架构，则Oracle数据库会在您自己的架构中查找属性维度。

attr_dimension_name

指定要删除的属性维的名称。

例

以下语句删除指定的属性维对象：

DROP ATTRIBUTE DIMENSION product_attr_dim;
