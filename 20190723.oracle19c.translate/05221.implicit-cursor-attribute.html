13.39隐式游标属性

隐式游标具有返回有关最近运行的SELECT或DML语句的信息的属性，该语句与命名游标无关。

注意：

您只能在过程语句中使用游标属性，而不能在SQL语句中使用游标属性。

语义

％开了

SQL%ISOPEN的值始终为FALSE 。

％FOUND

SQL%FOUND具有以下值之一：

  • 如果没有运行SELECT或DML语句， NULL 。

  • 如果最近的SELECT或DML语句返回一行， TRUE 。

  • 如果最新的SELECT或DML语句未返回行， FALSE 。

％未找到

SQL%NOTFOUND具有以下值之一：

  • 如果没有运行SELECT或DML语句， NULL 。

  • 如果最近的SELECT或DML语句返回一行， FALSE 。

  • 如果最新的SELECT或DML语句未返回行， TRUE 。

％ROWCOUNT

SQL%ROWCOUNT具有以下值之一：

  • 如果没有运行SELECT或DML语句， NULL 。

  • 如果已运行SELECT或DML语句，则到目前为止提取的行数。

SQL％BULK_ROWCOUNT

复合属性，类似于关联数组，其第i个元素是最近完成的FORALL语句中受第i个DML语句影响的行数。有关更多信息，请参阅“ 获取受FORALL语句影响的行数 ” 。

对SQL％BULK_ROWCOUNT的限制

您不能将SQL%BULK_ROWCOUNT( index )的值分配给另一个集合。

SQL％BULK_EXCEPTIONS

复合属性，类似于在最近运行的FORALL语句期间失败的DML语句的关联信息数组。SQL%BULK_EXCEPTIONS 。 COUNT是失败的DML语句数。如果是SQL%BULK_EXCEPTIONS 。 COUNT不为零，然后对于每个索引值i从1到SQL%BULK_EXCEPTIONS 。 COUNT ：

  • SQL%BULK_EXCEPTIONS( i ) 。ERROR_INDEX是失败的DML语句的编号。

  • SQL%BULK_EXCEPTIONS( i ) 。ERROR_CODE是失败的Oracle数据库错误代码。

通常，此属性出现在具有SAVE EXCEPTIONS子句的FORALL语句的异常处理程序中。有关更多信息，请参阅“ FORALL语句完成后处理FORALL异常 ” 。
