Oracle数据库的弃用和不支持的功能

本主题描述Oracle Database 19c的已弃用和不支持的功能。

除了新功能之外，Oracle Database 19c还引入了数据库的行为更改。行为更改包括已弃用和不支持的初始化参数，选项，语法以及功能和组件的弃用和不支持。有关更多信息，请参阅“ Oracle数据库升级指南” 。
