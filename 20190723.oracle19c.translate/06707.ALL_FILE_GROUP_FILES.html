2.140 ALL_FILE_GROUP_FILES

ALL_FILE_GROUP_FILES显示当前用户可访问的每个版本化文件组的文件集。

相关观点

  • DBA_FILE_GROUP_FILES显示数据库中每个版本化组的文件集。

  • USER_FILE_GROUP_FILES显示当前用户拥有的每个版本化组的文件集。此视图不显示FILE_GROUP_OWNER列。
