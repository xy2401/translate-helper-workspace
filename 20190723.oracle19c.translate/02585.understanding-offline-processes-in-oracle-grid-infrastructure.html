了解Oracle Grid Infrastructure中的脱机进程

安装Oracle Grid Infrastructure后，某些组件可能会列为OFFLINE。当您选择添加这些资源时，Oracle Grid Infrastructure会激活这些资源。

Oracle Grid Infrastructure为各种Oracle产品和组件提供所需的资源。其中一些产品和组件是可选的，因此您可以在安装Oracle Grid Infrastructure后安装和启用它们。为简化安装后添加，Oracle Grid Infrastructure预先配置并注册所有可用于这些产品和组件的产品所需的所有资源，但仅在您选择添加它们时才激活它们。因此，在安装Oracle Grid Infrastructure之后，某些组件可能会列为OFFLINE。运行以下命令以查看任何资源的状态：

$ crsctl status resource resource_name -t

列为TARGET的资源：OFFLINE和STATE：OFFLINE不需要监控。它们表示已注册但未启用的组件，因此它们不使用任何系统资源。如果系统上安装了Oracle产品或组件，并且需要特定资源联机，则软件会提示您激活所需的脱机资源。
