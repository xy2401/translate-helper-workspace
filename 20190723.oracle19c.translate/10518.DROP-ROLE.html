DROP ROLE

目的

使用DROP ROLE语句从数据库中删除角色。删除角色时，Oracle数据库会从已授予角色的所有用户和角色中撤消该角色，并将其从数据库中删除。已启用角色的用户会话不受影响。但是，没有新的用户会话可以在删除后启用该角色。

也可以看看：

  • CREATE ROLE和ALTER ROLE，用于创建角色和更改启用角色所需的授权的信息

  • SET ROLE以获取有关禁用当前会话角色的信息

先决条件

您必须已使用ADMIN OPTION授予该角色，或者您必须具有DROP ANY ROLE系统特权。

句法

drop_role :: =


说明drop_role.eps的描述

语义

角色

指定要删除的角色的名称。

例子

删除角色：示例

要删除在“ 创建角色：示例 ”中创建的角色dw_manager ，请发出以下语句：

DROP ROLE dw_manager;
