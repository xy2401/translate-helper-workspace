9.102 V $ SYSTEM_PARAMETER2

V$SYSTEM_PARAMETER2显示有关当前对实例有效的初始化参数的信息，每个列表参数值在视图中显示为一行。新会话从实例范围的值继承参数值。

以此格式显示列表参数值使您可以快速确定列表参数的值。例如，如果参数值为a, b ，则V$SYSTEM_PARAMETER视图不会告诉您参数是否具有两个值（ a和b ）或一个值（ a, b ）。V$SYSTEM_PARAMETER2清除列表参数值之间的区别。

ISSES_MODIFIABLE

VARCHAR2（5）

ISSYS_MODIFIABLE

VARCHAR2（9）

指示是否可以使用ALTER SYSTEM更改参数以及更改何时生效：

  • IMMEDIATE - 无论用于启动实例的参数文件类型如何，都可以使用ALTER SYSTEM更改参数。更改立即生效。

  • DEFERRED - 无论用于启动实例的参数文件类型如何，都可以使用ALTER SYSTEM更改参数。更改将在后续会话中生效。

  • FALSE - 除非使用服务器参数文件启动实例，否则无法使用ALTER SYSTEM更改参数。更改将在后续实例中生效。

ISPDB_MODIFIABLE

VARCHAR2（5）

指示是否可以基于每个PDB修改参数（ TRUE ）（ FALSE ）。

在非CDB中，此列的值为NULL 。

ISINSTANCE_MODIFIABLE

VARCHAR2（5）

对于可以使用ALTER SYSTEM更改的参数，指示参数的值是否可以针对每个实例（ TRUE ）不同，或者参数是否必须对所有Real Application Clusters实例具有相同的值（ FALSE ）。如果ISSYS_MODIFIABLE FALSE ，则此列始终为FALSE 。

ISMODIFIED

VARCHAR2（8）

指示参数的修改方式。如果执行了ALTER SYSTEM ，则该值将为MODIFIED 。

ISADJUSTED

VARCHAR2（5）

指示Oracle是否将输入值调整为更合适的值（例如，参数值应为prime，但用户输入非素数，因此Oracle将值调整为下一个素数）
