安装Oracle Solaris的Oracle数据库先决条件包

使用此过程为Oracle软件安装Oracle数据库先决条件组包。

group/prerequisite/oracle/oracle-rdbms-server-18c-preinstall组软件包将安装Oracle数据库和Oracle Grid Infrastructure安装所需的所有软件包。

要安装oracle-rdbms-server-18c-preinstall组软件包，请以root身份登录，然后在Oracle Solaris 11.2.5.5.0及更高版本的系统上运行以下命令：

#pkg install oracle-rdbms-server-18c-preinstall

注意：

  • 您不必指定整个包名称，只需指定唯一名称的尾部。见pkg(5) 。

  • Oracle建议您安装solaris-minimal-server组软件包，然后安装oracle-rdbms-server-18c-preinstall 。
