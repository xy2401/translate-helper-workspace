12.29列表

句法

L [IST] [ n | n m | n * | n最后| * | * n | *最后|持续]

清单 SQL缓冲区的一行或多行 。

在SQL * Plus命令行中，您还可以使用“;”列出SQL缓冲区中的所有行。

条款

术语 描述 
ñ

列表行 n 。 

为N×M

列出行 从n到m 。 

n *

列出当前行的第n行。 

ñLAST

列出第n行至最后一行。 

*

清单该当前线。 

* n

列出通过第n行的当前行。 

*最后

列出最后一行的当前行。

 
持续

列出最后一个线。

输入没有子句的LIST或“;”列出所有线。 最后列出的行成为新的当前行（用星号标记）。

例子

要列出缓冲区的内容，请输入

LIST

或输入

;
1 SELECT LAST_NAME，DEPARTMENT_ID，JOB_ID 2来自EMP_DETAILS_VIEW 3 WHERE JOB_ID ='SH_CLERK'4 *按DEPARTMENT_ID排序

星号表示第4行是当前行。

要仅列出第二行，请输入

列表2

显示第二行：

2 *来自EMP_DETAILS_VIEW

要从当前行（现在为第2行）列到最后一行，请输入

列表*最后

然后你会看到这个：

2 FROM EMP_DETAILS_VIEW 3 WHERE JOB_ID ='SH_CLERK'4 *按DEPARTMENT_ID排序
