4.1初始化支持Java的数据库

如果使用Oracle JVM选项安装Oracle数据库，则数据库将启用Java。也就是说，它已准备好运行Java存储过程和Java数据库连接（JDBC）。

本节包含以下主题：

  • 在Oracle数据库模板中配置Oracle JVM选件

  • 修改现有Oracle数据库以包含Oracle JVM

4.1.1在Oracle数据库模板中配置Oracle JVM选件

在数据库模板中配置Oracle JVM选项。这是Java安装的推荐方法。

Database Configuration Assistant使您可以创建数据库模板，以定义每个数据库实例安装将包含的内容。选择Oracle JVM选项以在数据库中安装Java平台。

4.1.2修改现有Oracle数据库以包含Oracle JVM

如果您已安装没有Oracle JVM的Oracle数据库，则可以通过Oracle Database 12 c第1版（12.1）的数据库配置助手的修改模式将Java添加到数据库。通过修改模式，您可以选择要在现有Oracle数据库实例之上安装的功能，例如Oracle JVM。
