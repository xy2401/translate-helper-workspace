1.83 DB_INDEX_COMPRESSION_INHERITANCE

DB_INDEX_COMPRESSION_INHERITANCE指示索引创建如何继承压缩属性。

属性 描述

参数类型

串

句法

DB_INDEX_COMPRESSION_INHERITANCE = {TABLESPACE |表|所有|没有 }

默认值

没有

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

Oracle RAC

您必须为每个实例设置此参数，并且多个实例必须具有相同的值。

价值观：

  • TABLESPACE

    索引继承基于表空间属性。

  • 表

    索引继承基于表属性。

  • 所有

    索引继承基于表设置。但是，如果表没有默认压缩属性，则索引继承基于表空间属性。

  • 没有

    表或表空间没有索引继承。

注意：

如果CREATE INDEX语句指定压缩属性，则忽略DB_INDEX_COMPRESSION_INHERITANCE的值。

如果CREATE INDEX语句指定压缩属性，则忽略DB_INDEX_COMPRESSION_INHERITANCE的值，并且表或表空间没有继承。当存在表继承或表空间继承（并且表空间没有显式索引压缩子句）时，则使用以下矩阵：

表/表空间级压缩 索引的继承压缩
OLTP 高级低
QUERY LOW，QUERY HIGH 高级
存档低，存档高 高级
