这是在“选择时间段”字段中选择“实时 - 最后一小时”时，Performance Hub页面的屏幕截图。

Performance Hub标题位于页面的左上角，表示从“选择时间段”字段中选择“实时 - 最后一小时”。“选择时间段”字段右侧是“隐藏时间选择器”图标。隐藏时间选择器图标右侧是PerfHub报告图标。“PerfHub报告”图标右侧是AWR报告图标。最右边是页面刷新字段和页面上次刷新的时间。

时间选择器显示在页面顶部的用户界面元素下方。它由选定时间段的性能数据组成。

时间选择器下方有五个选项卡，分别是Summary，Activity，Workload，Monitored SQL和ADDM选项卡。将显示“摘要”选项卡。

“摘要”选项卡左上角是“主机：可运行进程”图表。它显示CPU核心，负载平均值，其他主机CPU，每秒后台CPU使用率和每秒前景CPU使用率数据。

Host：Runnable Processes图表右侧是Memory：Host Total图表。它显示Total PGA，Streams Pool，Java Pool，Shared Pool和Buffer Cache数据。

在Host：Runnable Processes图表下方，将显示Active Sessions部分。它包括两个选项卡：“活动”选项卡（显示）和“服务”选项卡。

“活动”选项卡可以显示“仅前景”或“前景+背景”数据。选择仅前向选项，并显示CPU内核，平均活动会话，其他（并发，CPU，系统I / O，提交，用户I / O，其他和网络）和应用程序的数据。

“服务”选项卡显示SYS $ USERS的数据。

“活动会话”部分右侧是I / O图表。它包括“请求”，“吞吐量”和“延迟”选项卡。默认情况下显示“请求”选项卡。您可以使用“请求”选项卡上的“I / O类型”和“I / O功能”和“I / O使用者组”选项来显示数据。在此屏幕截图中，选择了I / O类型，并显示Large Writes，Large Reads，Small Writes和Small Reads数据。
