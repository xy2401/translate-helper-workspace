Oracle软件所有者的环境要求

您必须进行以下更改才能配置Oracle软件所有者环境：

  • 在shell启动文件中将安装软件所有者用户（ grid ， oracle ）默认文件模式创建掩码（umask）设置为022。将掩码设置为022可确保执行软件安装的用户创建具有644权限的文件。

  • 为安装软件所有者（ grid ， oracle ）设置文件描述符和进程的ulimit设置。

  • 设置DISPLAY环境变量以准备运行Oracle Universal Installer（OUI）安装。

警告：

如果您使用作为Oracle Grid Infrastructure软件所有者的用户标识安装的现有Oracle安装，则取消设置该用户的所有Oracle环境变量设置。
