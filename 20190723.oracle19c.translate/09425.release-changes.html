Oracle Database Release 19c，版本19.1中的更改

本节列出了Oracle Database Database for 19c版本19.1中Oracle Database Reference的更改。

有关此版本的新功能的完整列表，请参阅“ Oracle数据库新增功能指南” 。

新功能

本节列出了本手册中针对Oracle Database Release 19c，版本19.1的新功能。

以下初始化参数是Oracle Database Release 19c，版本19.1中的新增参数：

  • ADG_REDIRECT_DML

  • DATA_GUARD_MAX_IO_TIME

  • DATA_GUARD_MAX_LONGIO_TIME

  • ENABLE_IMC_WITH_MIRA

  • LOB_SIGNATURE_ENABLE

  • MAX_DATAPUMP_PARALLEL_PER_JOB

注意：

请参见初始化参数有关这些初始化参数的详细信息。

不推荐的功能

Oracle Database 19c版本19.1中不推荐使用以下功能，并且可能在将来的版本中不支持这些功能：

  • CLUSTER_DATABASE_INSTANCES初始化参数

不受支持的功能

Oracle Database 19c版本19.1中不支持以下功能：

  • LOG_ARCHIVE_DEST_ n初始化参数的MAX_CONNECTIONS属性，用于Oracle Data Guard重做传输

  • EXAFUSION_ENABLED初始化参数

  • Oracle Streams不受支持。因此，不支持以下视图：
    • ALL_STREAMS_COLUMNS
    • ALL_STREAMS_MESSAGE_RULES
    • ALL_STREAMS_RULES
    • ALL_STREAMS_UNSUPPORTED
    • DBA_STREAMS_ADMINISTRATOR
    • DBA_STREAMS_COLUMNS
    • DBA_STREAMS_MESSAGE_RULES
    • DBA_STREAMS_RULES
    • DBA_STREAMS_SPLIT_MERGE
    • DBA_STREAMS_SPLIT_MERGE_HIST
    • DBA_STREAMS_STMTS
    • DBA_STREAMS_STMT_HANDLERS
    • DBA_STREAMS_TRANSFORMATIONS
    • DBA_STREAMS_UNSUPPORTED
    • V $ STREAMS_CAPTURE
    • V $ STREAMS_MESSAGE_TRACKING
    • V $ STREAMS_TRANSACTION
