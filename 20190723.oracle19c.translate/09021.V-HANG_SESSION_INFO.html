7.260 V $ HANG_SESSION_INFO

V$HANG_SESSION_INFO显示有关V$HANG_INFO描述的挂起涉及的会话的信息。

具有相同HANG_ID每一行描述了由V$HANG_INFO具有相同HANG_ID的行描述的挂起等待链中的V$HANG_INFO 。该会话被该挂起的受害者或最终阻止者阻止。

柱 数据类型 描述

HANG_ID

数

标识挂起的数字，包括此会话。此列可用于将V$HANG_INFO与V$HANG_SESSION_INFO 。

INSTANCE

数

此会话所在的实例的实例编号

SID

数

此会话的Oracle会话ID

SERIAL＃

数

此会话的Oracle会话序列号

OSPID

VARCHAR2（24）

此会话的操作系统进程ID

FATAL_BACKGROUND

VARCHAR2（1）

Y - 会话是一个关键的后台流程

N - Session不是一个关键的后台进程

根

VARCHAR2（1）

Y - 此会话是挂起的受害者或最终阻止者

N - 此会话不是挂起的受害者或最终阻止者

PNAME

VARCHAR2（5）

受害者或最终阻止程序的名称

WAIT_EVENT_TEXT

VARCHAR2（64）

此会话正在等待的资源或事件;如果会话未在资源上not in a wait则设置为不等待

QOS_PC_RANK

VARCHAR2（12）

会话的Oracle数据库服务质量（QoS）管理性能等级

QOS_PC_ITT

数

会话的Oracle数据库服务质量（QoS）管理性能类别故障阈值

QOS_PC_RTT

数

会话的Oracle数据库服务质量（QoS）管理性能类根终止允许阈值

QOS_PC_KEY

数

会话的Oracle数据库服务质量（QoS）管理性能类密钥

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
