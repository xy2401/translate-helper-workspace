2.195 ALL_JSON_COLUMNS

ALL_JSON_COLUMNS提供有关当前用户可访问的JavaScript Object Notation（JSON）列的信息。 在AND视图中具有IS JSON检查约束并且用户可以访问的每个列都显示在此视图中。此视图使用户能够找到他或她可访问的所有JSON列。

例如，如果检查约束将IS JSON条件与使用逻辑条件OR的另一个条件组合在一起，则该列不会在此视图中列出。在这种情况下，不确定列中的数据是否为JSON数据。例如，以下约束不能确保列jcol中的数据是JSON数据：

jcol是json OR length（jcol）<1000

相关观点

  • DBA_JSON_COLUMNS提供有关所有JSON列的信息。

  • USER_JSON_COLUMNS提供有关用户所有者的JSON列的信息。此视图不显示OWNER列。
