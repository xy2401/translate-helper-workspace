前言

本文档提供有关Oracle Data Guard代理的信息，该代理是一个管理和监视接口，可帮助您配置，监视和控制Oracle Data Guard代理配置。

听众

Oracle Data Guard Broker适用于希望使用Oracle Data Guard代理自动完成配置和监视Oracle Data Guard配置所涉及的许多任务的数据库管理员（DBA）和系统管理员。

本书中提供的信息假定您已熟悉Oracle Data Guard，Oracle企业管理器云控制（云控制），Oracle Real Application Clusters（Oracle RAC）以及Oracle Net Services提供的网络服务。
