2.182 ALL_JAVA_CLASSES

ALL_JAVA_CLASSES显示有关当前用户可访问的存储Java类的类级别信息。

相关观点

  • DBA_JAVA_CLASSES显示有关数据库中所有存储的Java类的类级别信息。

  • USER_JAVA_CLASSES显示有关当前用户拥有的存储Java类的类级别信息。此视图不显示OWNER列。
