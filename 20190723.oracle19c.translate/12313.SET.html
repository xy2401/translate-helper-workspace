12.40 SET

设置系统变量以更改当前会话的SQL * Plus环境设置，例如：

  • 自定义HTML格式

  • 启用或禁用列标题的打印

  • 设置每页的行数

  • 设置数据的显示宽度

句法

SET system_variable 值

其中system_variable和value表示以下SET系统变量摘要表中显示的子句之一。

用法

SQL * Plus维护系统变量（也称为SET命令变量），使您能够为SQL * Plus会话设置特定环境。您可以使用SET命令更改这些系统变量，并使用SHOW命令列出它们。以下各节中为每个系统变量的默认值加下划线。

SET ROLE和SET TRANSACTION是SQL命令（有关更多信息，请参阅SQL语句：MERGE to UPDATE ）。如果没有后跟关键字TRANSACTION或ROLE，则假定SET是SQL * Plus命令。
