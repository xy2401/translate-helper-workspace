7.110 V $ BACKUP_SYNC_IO

V$BACKUP_SYNC_IO显示有关正在进行和最近完成的RMAN备份和还原的性能信息。

对于每个备份，它包含每个输入数据文件的一行，一行用于所有数据文件的总体性能，一行用于输出备份片。此数据不会持久存储，并且在重新启动实例时不会保留。

柱 数据类型 描述

SID

数

执行备份或还原的会话的Oracle SID

SERIAL

数

执行备份或还原的SID的使用计数

USE_COUNT

数

一个计数器，可用于标识来自不同备份集的行

RMAN_STATUS_RECID

数

拥有V$RMAN_STATUS记录ID

RMAN_STATUS_STAMP

数

拥有V$RMAN_STATUS记录戳记

设备类型

VARCHAR2（17）

文件所在的设备类型

类型

VARCHAR2（9）

INPUT ， OUTPUT或AGGREGATE

状态

VARCHAR2（11）

NOT STARTED ， IN PROGRESS ，或FINISHED

文件名

VARCHAR2（513）

要读取或写入的备份文件的名称

SET_COUNT

数

正在读取或写入的备份集的设置计数

SET_STAMP

数

正在读取或写入备份集的设置标记

缓冲区大小

数

用于读/写此文件的缓冲区大小（以字节为单位）

BUFFER_COUNT

数

用于读/写此文件的缓冲区数

TOTAL_BYTES

数

如果已知，将为此文件读取或写入的总字节数。如果未知，则此列将为null。

OPEN_TIME

日期

此文件打开的时间。如果TYPE =' AGGREGATE '，那么这是聚合中第一个文件打开的时间。

CLOSE_TIME

日期

该文件关闭的时间。如果TYPE =' AGGREGATE '，那么这是聚合中最后一个文件关闭的时间。

ELAPSED_TIME

数

文件打开的时间，以百分之一秒为单位

MAXOPENFILES

数

同时打开的DISK文件数。此值仅出现在TYPE =' AGGREGATE '的行中。

BYTES

数

到目前为止读取或写入的字节数

EFFECTIVE_BYTES_PER_SECOND

数

此备份期间使用此设备实现的I / O速率

IO_COUNT

数

对此文件执行的I / O数

IO_TIME_TOTAL

数

用于执行此文件的I / O的总时间（以百分之一秒为单位）

IO_TIME_MAX

数

单个I / O请求所需的最长时间

DISCRETE_BYTES_PER_SECOND

数

此文件的平均传输速率

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
