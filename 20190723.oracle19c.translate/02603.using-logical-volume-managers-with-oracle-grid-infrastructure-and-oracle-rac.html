将逻辑卷管理器与Oracle Grid Infrastructure和Oracle RAC配合使用

Oracle Grid Infrastructure和Oracle RAC仅支持群集感知卷管理器。

使用逻辑卷管理器

Oracle Grid Infrastructure和Oracle RAC仅支持群集感知卷管理器。某些第三方卷管理器不支持群集，因此不受支持。要确认支持您要使用的卷管理器，请单击My Oracle Support上的“ 证书 ”以确定您的卷管理器是否已通过Oracle RAC认证。我的Oracle支持可从以下URL获得：

https://support.oracle.com
