第二部分使用Oracle Label Security功能

第II部分介绍了如何使用Oracle Label Security功能。

  • 注册并登录Oracle Label Security
    在使用Oracle Label Security之前，必须在数据库中注册（配置）它，然后才能登录Oracle Label Security。
  • 创建Oracle标签安全策略
    Oracle Label Security策略是一组命名的实现Oracle Label Security的命令。
  • 使用标记数据
    您可以管理标记数据，查看会话的安全属性数据，以及更改会话属性的值。
  • 使用Oracle Internet Directory的Oracle Label Security
    您可以将Oracle Label Security与Oracle Internet Directory一起使用。
