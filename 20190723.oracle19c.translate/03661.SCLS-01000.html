152 SCLS-01000至SCLS-01005

SCLS-01000：无法为层{0}创建超级集群代表服务器，{1}

原因：无法为给定层创建超级集群代表服务器。

操作：解决消息中声明的原因后重试。

SCLS-01001：无法获得{0}的ONS运行时

原因：无法获取给定服务器的ONS运行时。

操作：解决消息中声明的原因后重试。

SCLS-01002：无法获取层{0}的当前成员列表

原因：无法获取给定层的当前成员列表。

操作：解决消息中声明的原因后重试。

SCLS-01003：层{0}未运行eONS运行时

原因： eONS运行时异常停止。

操作：在集群环境中，尝试srvctl stop nodeapps -n node和srvctl start nodeapps -n node 。在Oracle Restart环境中，尝试srvctl stop eons和srvctl start eons。

SCLS-01004：无法将{1}的侦听端口设置为{0}

原因：无法更新此层的eONS运行时的侦听端口。

操作：确保用户有权更新资源属性。在群集环境中，确保群集堆栈已启动。

SCLS-01005：为eONS指定的侦听端口{0}无效

原因：指定的eONS侦听端口号无效。

操作： Unix上的有效端口号为1025到65535。为eONS提供有效的端口号以供监听。
