Oracle Flex Cluster配置的网络要求

  • 您必须将网格命名服务（GNS）与Oracle Flex Cluster部署配合使用。

  • 您必须将GNS VIP配置为静态IP地址。

  • 在多群集配置中，您必须标识GNS客户端数据文件位置。在开始配置GNS客户端群集之前，将从GNS服务器复制GNS客户端数据文件。

  • 无论是手动分配还是自动分配，所有公共网络地址都必须位于同一子网范围内。

  • 所有Oracle Flex Cluster地址必须是静态IP地址，通过DHCP（IPv4）分配的DHCP地址或通过自动配置服务（IPv6）分配的自动配置地址，通过GNS在群集中注册。
