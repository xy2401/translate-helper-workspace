ORA-12400：设施错误处理的参数无效

原因：设施错误处理功能的参数超出最大限制或被称为无效产品/设施。

操作：指定有效的设施错误处理参数值。

ORA-12401：无效的标签字符串： string

原因：策略无法将标签字符串转换为有效的内部标签。

操作：更正标签字符串的语法。

ORA-12402：格式字符串无效： string

原因：策略不支持格式字符串。

操作：更正格式字符串的语法。

ORA-12403：内部标签无效

原因：无法将内部标签转换为策略的有效标签。

操作：分析错误堆栈上的任何其他消息并查阅策略文档。

ORA-12404：无效的权限字符串： string

原因：策略无法解释权限字符串。

操作：指定策略支持的权限字符串。

ORA-12405：标签列表无效

原因：该策略确定标签列表对其预期用途无效。

操作：检查特定标签列表上的策略约束。

ORA-12406：未经授权的策略string SQL语句

原因：策略未授权数据库会话执行请求的SQL语句。

操作：为用户或程序单元授予必要的策略权限或其他授权。

ORA-12407：对策略string未授权操作

原因：策略未授权数据库会话执行请求的操作。

操作：为用户或程序单元授予必要的策略权限或其他授权。

ORA-12408：不支持的操作： string

原因：指定的策略不支持请求的操作。

操作：查阅策略文档以确定支持的访问中介操作。

ORA-12409： string策略的策略启动失败

原因：策略在启动处理期间遇到错误;禁止访问受策略保护的数据。

操作：检查警报日志以获取其他信息，更正策略错误，然后重新启动实例。
