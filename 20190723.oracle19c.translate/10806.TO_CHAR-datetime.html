TO_CHAR（日期时间）

句法

to_char_date :: =


插图描述to_char_date.eps

目的

TO_CHAR （datetime）将日期时间或间隔值DATE ， TIMESTAMP ， TIMESTAMP WITH TIME ZONE ， TIMESTAMP WITH LOCAL TIME ZONE ， INTERVAL DAY TO SECOND或INTERVAL YEAR TO MONTH数据类型转换为VARCHAR2数据类型的值，格式为日期格式fmt 。如果省略fmt ，则date将转换为VARCHAR2值，如下所示：

  • DATE值将转换为默认日期格式的值。

  • TIMESTAMP和TIMESTAMP WITH LOCAL TIME ZONE值被转换为值在默认时间戳格式。

  • TIMESTAMP WITH TIME ZONE值将转换为具有时区格式的默认时间戳中的值。

  • 间隔值将转换为区间文字的数字表示形式。

有关日期时间格式的信息，请参阅“ 格式模型 ” 。

'nlsparam'参数指定返回月份和日期名称和缩写的语言。这个论点可以有这样的形式：

'NLS_DATE_LANGUAGE = 语言 '

如果省略'nlsparam' ，则此函数使用会话的默认日期语言。

例子

以下示例使用此表：

CREATE TABLE date_tab（ts_col TIMESTAMP，tsltz_col TIMESTAMP WITH LOCAL TIME ZONE，tstz_col TIMESTAMP WITH TIME ZONE）;

该示例显示了将TO_CHAR应用于不同TIMESTAMP数据类型的结果。TIMESTAMP WITH LOCAL TIME ZONE列的结果对会话时区敏感，而TIMESTAMP和TIMESTAMP WITH TIME ZONE列的结果对会话时区不敏感：

ALTER SESSION SET TIME_ZONE =' -  8:00'; INSERT INTO date_tab VALUES（TIMESTAMP'1999-12-01 10:00:00'，TIMESTAMP'1999-12-01 10:00:00'，TIMESTAMP'1999-12-01 10:00:00'）; INSERT INTO date_tab VALUES（TIMESTAMP'1999-12-02 10:00:00 -8：00'，TIMESTAMP'1999-12-02 10:00:00 -8：00'，TIMESTAMP'1999-12-02 10： 00：00-8：00'）; SELECT TO_CHAR（ts_col，'DD-MON-YYYY HH24：MI：SSxFF'）AS ts_date，TO_CHAR（tstz_col，'DD-MON-YYYY HH24：MI：SSxFF TZH：TZM'）AS tstz_date FROM date_tab ORDER BY ts_date，tstz_date ; TS_DATE TSTZ_DATE ------------------------------ ------------------ ------------------- 01-DEC-1999 10：00：00.000000 01-DEC-1999 10：00：00.000000 -08：00 02-DEC-1999 10： 00：00.000000 02-DEC-1999 10：00：00.000000 -08：00 SELECT SESSIONTIMEZONE，TO_CHAR（tsltz_col，'DD-MON-YYYY HH24：MI：SSxFF'）AS tsltz FROM date_tab ORDER BY sessiontimezone，tsltz; SESSIONTIM TSLTZ ---------- ------------------------------ -08：00 01-DEC- 1999年10：00：00.000000 -08：00 02-DEC-1999 10：00：00.000000 ALTER SESSION SET TIME_ZONE ='-5：00'; SELECT TO_CHAR（ts_col，'DD-MON-YYYY HH24：MI：SSxFF'）AS ts_col，TO_CHAR（tstz_col，'DD-MON-YYYY HH24：MI：SSxFF TZH：TZM'）AS tstz_col FROM date_tab ORDER BY ts_col，tstz_col ; TS_COL TSTZ_COL ------------------------------ ------------------ ------------------- 01-DEC-1999 10：00：00.000000 01-DEC-1999 10：00：00.000000 -08：00 02-DEC-1999 10： 00：00.000000 02-DEC-1999 10：00：00.000000 -08：00 SELECT SESSIONTIMEZONE，TO_CHAR（tsltz_col，'DD-MON-YYYY HH24：MI：SSxFF'）AS tsltz_col FROM date_tab ORDER BY sessiontimezone，tsltz_col; 2 3 4 SESSIONTIM TSLTZ_COL ---------- ------------------------------ -05：00 01 -DEC-1999 13：00：00.000000 -05：00 02-DEC-1999 13：00：00.000000

使用TO_CHAR格式化日期和数字：示例

以下语句将日期值转换为TO_CHAR函数中指定的格式：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual）SELECT d“Original Date”， to_char（d，'dd-mm-yyyy'）“日 - 月 - 年”，to_char（d，'hh24：mi'）“24小时格式的时间”，to_char（d，'iw-iyyy'）“ ISO年份和周年“从日期;

以下语句将日期和时间戳值转换为TO_CHAR函数中指定的格式：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual union SELECT timestamp'2015-03- 03 23:44:32来自双联盟SELECT timestamp'2015-04-11 12:34:56'd FROM dual）SELECT d“Original Date”，to_char（d，'dd-mm-yyyy'）“Day -Month-Year“，to_char（d，'hh24：mi'）”24小时格式的时间“，to_char（d，'iw-iyyy'）”ISO年份和周年“，to_char（d，'月'）“月份名称”，to_char（d，'年'）“年”FROM日期;

以下语句从输入datetime表达式中EXTRACT函数中指定的datetime字段：

WITH date AS（SELECT date'2015-01-01'd FROM dual union SELECT date'2015-01-10'd FROM dual union SELECT date'2015-02-01'd FROM dual union SELECT timestamp'2015-03- 03 23:44:32来自双联盟SELECT timestamp'2015-04-11 12:34:56'd FROM dual）SELECT提取物（分钟从d）分钟，提取物（小时从d）小时，提取物（天从d）天，提取物（从d开始的月份），提取物（从d开始的年份）从日期开始;

以下语句按照TO_CHAR函数中指定的格式显示输入数字：

WITH nums AS（SELECT 10 n FROM双联合SELECT 9.99 n FROM双联盟SELECT 1000000 n FROM dual --one million）SELECT n“输入数字N”，to_char（n），to_char（n，'9,999,999.99'）“数字带逗号“，to_char（n，'0,000,000.000'）”零填充数字“，to_char（n，'9.9EEEE'）”科学记数法“来自nums;

TO_CHAR（datetime）功能：示例

以下语句创建一个名为empl_temp的表，并使用员工详细信息填充它：

CREATE TABLE empl_temp（employee_id NUMBER（6），first_name VARCHAR2（20），last_name VARCHAR2（25），email VARCHAR2（25），hire_date DATE DEFAULT SYSDATE，job_id VARCHAR2（10），clob_column CLOB）; INSERT INTO empl_temp VALUES（111，'John'，'Doe'，'example.com'，'10-JAN-2015'，'1001'，'有经验的员工'）; INSERT INTO empl_temp VALUES（112，'John'，'Smith'，'example.com'，'12-JAN-2015'，'1002'，'Junior Employee'）; INSERT INTO empl_temp VALUES（113，'Johnnie'，'Smith'，'example.com'，'12-JAN-2014'，'1002'，'Mid-Career Employee'）; INSERT INTO empl_temp VALUES（115，'Jane'，'Doe'，'example.com'，'2015年1月15日'，'1005'，'执行员工'）;

以下语句使用短格式和长格式显示日期：

SELECT hire_date“Default”，TO_CHAR（hire_date，'DS'）“Short”，TO_CHAR（hire_date，'DL'）“long”FROM empl_temp WHERE employee_id IN（111,112,115）;默认短期长---------- ---------- -------------------------- 10 -JAN-15 2015年1月10日星期六2015年1月10日星期六2015年1月12日星期一2015年1月12日星期一15-JAN-15 2015年1月15日星期四2015年1月15日星期四
