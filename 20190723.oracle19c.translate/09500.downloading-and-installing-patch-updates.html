下载和安装版本更新修补程序

完成安装后，下载并安装Oracle软件的版本更新（RU）和版本更新修订（RUR）修补程序。

从Oracle Database 18c开始，Oracle以Release Updates（RU）和Release Update Revisions（RUR）的形式提供季度更新。Oracle不再发布补丁集。有关更多信息，请参阅My Oracle Support说明2285040.1。
查看My Oracle Support网站，了解安装所需的更新。
  1. 使用Web浏览器查看My Oracle Support网站：

    https://support.oracle.com

  2. 登录My Oracle Support网站。

    注意：

    如果您不是My Oracle Support注册用户，请单击Register for My Oracle Support并注册。
  3. 在My Oracle Support主页面上，单击“ 修补程序和更新” 。
  4. 在“ 修补程序搜索”区域中，选择“ 产品”或“系列（高级）” 。
  5. 在“ 产品或系列（高级）”屏幕上，提供有关要获取修补程序的产品，版本和平台的信息，然后单击“ 搜索” 。
    将打开“ 修补程序搜索”窗格，其中显示了搜索结果。
  6. 选择修补程序编号，然后单击自述文件 。

    将显示README页面。它包含有关修补程序的信息以及如何将修补程序应用于安装。

  7. 解压缩从My Oracle Support下载的Oracle修补程序更新。

    注意：

    即使成功安装了最新版本更新或发行版更新修订补丁，也不要删除或移动.patch_storage目录，因为OPatch将补丁信息存储在$ORACLE_HOME/.patch_storage目录中。
