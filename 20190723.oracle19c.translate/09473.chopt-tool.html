Chopt工具

安装后使用chopt工具添加或删除Oracle数据库选项。

目的

chopt工具是一个命令行实用程序，用于启用和禁用数据库选项。

先决条件

必须先完成安装才能使用chopt工具。

文件路径

该工具位于ORACLE_HOME /bin目录中

句法

chopt [ enable | disable ] db_option

选项

命令选项 描述
OAA

Oracle Advanced Analytics
OLAP

Oracle OLAP
分区

Oracle分区
鼠

Oracle Real Application Testing

例子

要使用chopt工具修改Oracle数据库，必须在运行chopt工具之前关闭数据库，然后在添加或删除数据库选项后启动数据库。

示例4-1使用Chopt启用Oracle数据挖掘

以下示例显示如何使用chopt工具在名为Sales的Oracle数据库中启用Oracle Data Mining选项：

cd $ ORACLE_HOME / bin srvctl stop database -d Sales chopt enable oaa srvctl start database -d Sales
