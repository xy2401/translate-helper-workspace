1.355 XML_DB_EVENTS

XML_DB_EVENTS启用或禁用XML DB事件。

属性 描述

参数类型

串

句法

XML_DB_EVENTS = {enable |禁用}

默认值

启用

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

通过ALTER SESSION语句更改此参数仅影响当前会话。只允许具有XDBADMIN角色的用户在会话中更改此参数。

通过ALTER SYSTEM语句更改此参数将对此参数进行ALTER SYSTEM范围的更改;但是，更改仅在更改后启动的新会话中注册。当前正在运行的会话需要重新启动以获取新的更改。用户必须具有ALTER SYSTEM权限才能进行系统范围的更改。
