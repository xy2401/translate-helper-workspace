创建包

目的

包使用PL / SQL定义。因此，本节提供了一些常规信息，但有关语法和语义的详细信息，请参阅Oracle Database PL / SQL语言参考 。

使用CREATE PACKAGE语句为存储包创建规范，存储包是存储在数据库中的相关过程，函数和其他程序对象的封装集合。包规范声明了这些对象。随后指定的包 体定义了这些对象。

也可以看看：

  • CREATE PACKAGE BODY以获取有关指定包实现的信息

  • 创建功能和创建过程以获取有关创建独立功能和过程的信息

  • 有关修改和删除程序包的信息，请参阅ALTER PACKAGE和DROP PACKAGE

  • “Oracle数据库开发指南”和“ Oracle数据库PL / SQL包和类型参考” ，详细讨论了包及其使用方法

先决条件

要在您自己的架构中创建或替换包，您必须具有CREATE PROCEDURE系统特权。要在另一个用户的模式中创建或替换包，您必须具有CREATE ANY PROCEDURE系统特权。

要在Oracle数据库预编译器程序中嵌入CREATE PACKAGE语句，必须使用关键字END-EXEC终止该语句，然后使用特定语言的嵌入式SQL语句终止符。

语义

或替换

指定OR REPLACE以重新创建包规范（如果已存在）。使用此子句可以更改现有程序包的规范，而无需删除，重新创建和重新授予先前在程序包上授予的对象权限。如果更改包规范，则Oracle Database会重新编译它。

以前被授予重新定义的包特权的用户仍然可以访问该包而无需重新授予特权。

如果任何基于函数的索引依赖于包，则数据库会将索引标记为DISABLED 。
