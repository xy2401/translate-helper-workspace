13.6 Bfile类

Bfile类定义BFILE类型的对象的公共属性。 BFILE是存储在Oracle数据库之外的操作系统文件中的大型二进制文件。Bfile对象包含指向BFILE的逻辑指针，而不是BFILE本身。

Bfile类的方法使您能够执行与Bfile对象相关的特定任务。

ResultSet和Statement类的方法（如getBfile()和setBfile()使您可以访问SQL BFILE值。

NULL Bfile对象上唯一有效的方法是setName（） ， isNull（）和operator =（） 。

Bfile对象可以通过以下方式初始化：

  • setName（）方法。该BFILE然后可以通过将本修改BFILE到表中，然后使用它检索SELECT...FOR UPDATE 。 write()方法修改BFILE ;但是，仅在提交事务时才将刷新的数据刷新到表中。请注意，不需要INSERT操作。

  • 为其分配初始化的Bfile对象。

13.6.7 getStream（）

返回从BFILE读取的Stream对象。如果流是打开的，则不允许在Bfile对象上打开另一个流。必须在执行任何Bfile对象操作之前关闭流。

句法

Stream * getStream（unsigned int offset = 1，unsigned int amount = 0）;
参数 描述 
抵消

从BFILE开始读取数据的起始位置。如果未指定offset ，则从BLOB的开头写入数据。有效值是大于或等于1 。 

量

从BFILE读取的总字节数;如果amount为0 ，则以流传输模式从输入offset读取数据，直到BFILE结束。

13.6.8 getUStringDirAlias（）

返回包含与BFILE关联的目录对象的UString 。请注意， UString对象采用UTF16字符集。与BFILE相关联的环境应与UTF16字符集相关联。

句法

UString getUStringDirAlias（）const;

13.6.9 getUStringFileName（）

返回包含与BFILE关联的文件名的UString 。注意UString对象是UTF16字符集。与BFILE相关联的环境应与UTF16字符集相关联。

句法

UString getUStringFileName（）const;

13.6.12 isOpen（）

测试BFILE是否打开。仅当通过对此Bfile对象的调用打开BFILE ，才认为它是打开的。（不同的Bfile对象可以打开此文件，因为可以通过将文件与不同的Bfile对象相关联来对同一文件执行多个open()调用）。如果BFILE打开，则返回TRUE ;否则，返回FALSE 。

句法

bool isOpen（）const;

13.6.15 operator =（）

参数 描述

13.6.16 operator ==（）

比较两个Bfile对象是否相等。如果Bfile对象都引用相同的BFILE则它们是相等的。如果Bfile对象为NULL ，则返回FALSE 。如果Bfile对象相等，则返回TRUE ;否则，返回FALSE 。

句法

bool operator ==（const Bfile＆srcBfile）const;
参数 描述 
srcBfile

的Bfile对象与当前进行比较Bfile对象。

13.6.17运营商！=（）

比较两个Bfile objects的不等式。如果Bfile对象都引用相同的BFILE则它们是相等的。如果Bfile对象不相等，则返回TRUE ;否则，返回FALSE 。

句法

布尔运营商！=（const Bfile＆srcBfile）const;
参数 描述 
srcBfile

的Bfile对象与当前进行比较Bfile对象。

13.6.18读（）

将部分或全部BFILE读入指定的缓冲区，并返回读取的字节数。

句法

unsigned int read（unsigned int amt，unsigned char * buffer，unsigned int bufsize，unsigned int offset = 1）const;
参数 描述 
AMT

要读取的字节数。有效值是大于或等于1 。 

缓冲

要读取BFILE数据的缓冲区。有效值是大于或等于amt 。 

BUFFSIZE

要读取BFILE数据的缓冲区大小。有效值是大于或等于amt 。 

抵消

从BFILE开始读取数据的起始位置。如果未指定offset ，则从BFILE的开头写入数据。
