1.270 PRIVATE_TEMP_TABLE_PREFIX

PRIVATE_TEMP_TABLE_PREFIX指定数据库用于专用临时表的前缀。

属性 描述

参数类型

串

句法

PRIVATE_TEMP_TABLE_PREFIX =字符串

默认值

ORA $ PTT_

可修改

改变系统......DEFERRED

可在PDB中修改

是

基本

没有

Oracle RAC

必须在每个实例上指定相同的值。

用于PRIVATE_TEMP_TABLE_PREFIX参数的默认值是ORA$PTT_ 。

如果选择指定其他前缀值，则必须以字符串ORA$开头，并且在整个数据库中必须是唯一的。使用以下SQL查询来确定前缀值是否唯一（在查询中，将prefix-value替换为您的实际前缀值）：

从obj $中选择count（*），其名称如'<prefix-value>％';

如果上面的查询返回0，那么您指定的前缀是要设置的适当值。

注意：

设置前缀后，常规表，视图和对象名称不能使用相同的前缀。
