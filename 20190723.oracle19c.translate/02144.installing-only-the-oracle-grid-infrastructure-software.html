仅安装Oracle Grid Infrastructure软件

此安装选项需要手动安装后步骤才能启用Oracle Grid Infrastructure软件。

如果在安装期间使用“ 仅设置软件”选项，则Oracle Universal Installer（OUI）会在多个节点上安装软件二进制文件。然后，您可以执行配置Oracle Clusterware和Oracle ASM的其他步骤。

  • 为群集安装Oracle Grid Infrastructure的软件二进制文件
    您可以一次在多个节点上为群集软件安装Oracle Grid Infrastructure软件二进制文件。
  • 为群集配置Oracle Grid Infrastructure的软件二进制文件
    通过在GUI模式下启动Oracle Grid Infrastructure配置向导来配置软件二进制文件。
  • 使用响应文件配置软件二进制文件
    在任何节点上安装或复制Oracle Grid Infrastructure软件时，可以将配置推迟到以后进行。使用配置向导（ gridSetup.sh ）在节点上安装或复制软件后，请查看此过程以完成配置。
  • 设置网络检查的Ping目标
    通过在Oracle Grid Infrastructure安装期间设置Ping_Targets参数来接收有关网络状态的通知。
