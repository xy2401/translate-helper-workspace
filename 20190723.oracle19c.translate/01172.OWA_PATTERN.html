218 OWA_PATTERN

OWA_PATTERN包提供了一个接口，用于在字符串中定位文本模式，并将匹配的字符串替换为另一个字符串。

218.1 OWA_PATTERN类型

您可以将模式用作输入和输出参数。因此，您可以将相同的正则表达式传递给OWA_PATTERN函数调用，并且只需解析一次。

  • OWA_PATTERN.PATTERN

218.2 OWA_PATTERN操作说明

OWA_PATTERN子程序过载。具体来说， MATCH有六个版本， AMATCH和CHANGE各有四个版本。

子程序使用以下参数：

  • line - 这是要匹配的目标。它可以是多行文本或owa_text.multi_line数据类型。

  • pat - 这是子程序尝试排队的模式。该模式可以包含正则表达式。在owa_pattern.change函数和过程中，此参数从from_str调用。

  • flags - 指定搜索是区分大小写还是全局替换。

将正则表达式与此包中的子程序一起使用。您可以通过创建要与各种通配符标记和量词交叉的字符串来指定正则表达式。

218.3.1 AMATCH功能

此函数指定模式是否出现在字符串中的特定位置。

这个功能有四个版本：

  • 函数的第一个和第二个版本不保存匹配的标记（这些标记保存在第三个和第四个版本的backrefs参数中）。第一个和第二个版本之间的区别是pat参数，可以是VARCHAR2或模式数据类型。

  • 函数的第三个和第四个版本在backrefs参数中保存匹配的标记。第三个和第四个版本之间的区别是pat参数，可以是VARCHAR2或模式数据类型。

    注意：

    如果多个重叠字符串与正则表达式匹配，则此函数采用最长匹配。

句法

OWA_PATTERN.AMATCH（行IN VARCHAR2，from_loc IN INTEGER，pat IN VARCHAR2，标志IN VARCHAR2 DEFAULT NULL）RETURN INTEGER; OWA_PATTERN.AMATCH（行IN VARCHAR2，from_loc IN INTEGER，pat IN OUT PATTERN，标志IN VARCHAR2 DEFAULT NULL）RETURN INTEGER; OWA_PATTERN.AMATCH（行IN VARCHAR2 from_loc IN INTEGER pat in varchar2 backrefs OUT owa_text.vc_arr flags IN VARCHAR2 DEFAULT NULL）RETURN INTEGER; OWA_PATTERN.AMATCH（行IN VARCHAR2 from_loc IN INTEGER pat IN OUT PATTERN backrefs OUT owa_text.vc_arr flags IN VARCHAR2 DEFAULT NULL）RETURN INTEGER;

218.3.2更改功能和程序

此函数或过程搜索并替换字符串或multi_line数据类型。如果多个重叠字符串与正则表达式匹配，则此子程序采用最长匹配。

句法

参数

表218-6 CHANGE过程参数

参数 描述

多线

要搜索的文字。这是owa_text.multi_line datatype 。此参数的输出值是更改的字符串。

旗

搜索是否区分大小写，以及是否要在全局范围内进行更改。如果指定“ i ”，则搜索不区分大小写。如果指定“ g ”，则对所有匹配进行更改。否则，在第一次替换后该函数停止。

返回值

作为一个函数，它返回所做的替换次数。如果未使用标志“g”，则此数字只能为0或1，并且仅替换第一个匹配。标志“g”指定用正则表达式替换所有匹配。

例子

OWA_PATTERN.CHANGE（'穿着睡衣的猫'，'C。+'，'和红色'）

正则表达式匹配子字符串“ Cats in ”。然后用“＆red”替换这个字符串。＆符号“ & ”表示“ Cats in ”，因为它与正则表达式匹配。因此，此过程将字符串“ Cats in pajamas ”替换为“ Cats in red ”如果将其称为函数而不是过程，则返回的值为1，表示已进行单个替换。

218.3.4匹配功能

此函数确定字符串是否包含指定的模式。该模式可以包含正则表达式。如果多个重叠字符串可以匹配正则表达式，则此函数采用最长匹配。

句法

owa_pattern.match（行IN VARCHAR2，pat IN VARCHAR2，标志IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN; owa_pattern.match（行IN VARCHAR2，pat IN OUT PATTERN，标志IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN; owa_pattern.match（行IN VARCHAR2，pat IN VARCHAR2，backrefs OUT owa_text.vc_arr，flags IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN; OWA_PATTERN.MATCH（行IN VARCHAR2，pat IN OUT PATTERN，backrefs OUT owa_text.vc_arr，flags IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN; owa_pattern.match（mline IN owa_text.multi_line，pat IN VARCHAR2，rlist OUT owa_text.row_list，flags IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN; OWA_PATTERN.MATCH（mline IN owa_text.multi_line，pat IN OUT pattern，rlist OUT owa_text.row_list，flags IN VARCHAR2 DEFAULT NULL）RETURN BOOLEAN;

参数

表218-8 MATCH功能参数

参数 描述

线

要搜索的行。

多线

要搜索的文字。这是一个owa_text.multi_line datatype ..

拍

匹配的模式。这是VARCHAR2或OWA_PATTERN.PATTERN DATA TYPE 。它是一种模式，该参数的输出值是匹配的模式。

backrefs

匹配的文本。匹配的每个令牌都放在OWA_TEXT.VC_ARR DATA TYPE PL / SQL表的单元格中。This parameter is a row_list ，用于保存目标中与正则表达式中的一系列标记匹配的每个字符串。

RLIST

包含匹配列表的输出参数。

旗

搜索是否区分大小写。如果此参数的值为“i”，则搜索不区分大小写。否则搜索区分大小写。

返回值

如果找到匹配则为TRUE ，否则为FALSE 。

例子

KAZOO是寻找zoo.*的目标zoo.*正则表达式。句点表示除换行符以外的任何字符，星号与前面的0个或多个字符匹配。在这种情况下，它匹配换行符以外的任何字符。

因此，此正则表达式指定匹配的目标由zoo组成，后跟任何一组字符，既不以结尾也不包括换行符（与句点不匹配）。i标志表示忽略搜索中的大小写。在这种情况下，函数返回TRUE ，表示已找到匹配项。 

boolean foundMatch; foundMatch：= owa_pattern.match（'KAZOO'，'zoo。*'，'i'）;

以下示例搜索字符串“goal”，后跟sometext任意数量的字符。如果找到，

sometext VARCHAR2（256）; pat VARCHAR2（256）; sometext：='目标是什么？'pat：='目标。*“;如果OWA_PATTERN.MATCH（sometext，pat）THEN HTP.PRINT（'Match found'）; ELSE HTP.PRINT（'未找到匹配'）;万一;

操作说明

  • 此函数中的正则表达式可以是VARCHAR2或OWA_PATTERN.PATTERN DATA TYPE.使用OWA_PATTERN.GETPAT过程从字符串创建AN OWA_PATTERN.PATTERN DATA TYPE 。

  • 使用OWA_TEXT.STREAM2MULTI过程从长字符串创建MULTI_LINE DATA TYPE 。如果使用multi_line ，则rlist参数指定找到匹配项的块列表。

  • 如果该行是字符串而不是multi_line ，则可以添加名为backrefs的可选输出参数。此参数是一个row_list ，用于保存目标中与正则表达式中的一系列标记匹配的每个字符串。
