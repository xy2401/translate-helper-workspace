2.21 ALL_ANALYTIC_VIEW_HIERS

ALL_ANALYTIC_VIEW_HIERS描述当前用户可访问的分析视图中的所有层次结构。

相关观点

  • DBA_ANALYTIC_VIEW_HIERS描述数据库中所有分析视图中的层次结构。

  • USER_ANALYTIC_VIEW_HIERS描述当前用户拥有的分析视图中的层次结构。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

分析视图层次结构的所有者

ANALYTIC_VIEW_NAME

VARCHAR2（128）

不是NULL

分析视图的名称

DIMENSION_ALIAS

VARCHAR2（128）

分析视图中属性维度的别名

HIER_OWNER

VARCHAR2（128）

不是NULL

层次结构的所有者

HIER_NAME

VARCHAR2（128）

不是NULL

层次结构的名称

HIER_ALIAS

VARCHAR2（128）

为层次结构指定的别名

IS_DEFAULT

VARCHAR2（1）

如果这是分析视图中分析视图维度的默认层次结构，则为Y，否则为N.

ORDER_NUM

数

不是NULL

分析视图中层次结构列表中的层次结构顺序

ORIGIN_CON_ID

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
