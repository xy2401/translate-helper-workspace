C.1实施例中使用的表

本附录中的示例引用名为GEOD_CITIES，GEOD_COUNTIES和GEOD_INTERSTATES的表，其定义如下：

CREATE TABLE GEOD_CITIES（LOCATION SDO_GEOMETRY，CITY VARCHAR2（42），STATE_ABRV VARCHAR2（2），POP90 NUMBER，RANK90 NUMBER）; CREATE TABLE GEOD_COUNTIES（COUNTY_NAME VARCHAR2（40），STATE_ABRV VARCHAR2（2），GEOM SDO_GEOMETRY）; CREATE TABLE GEOD_INTERSTATES（HIGHWAY VARCHAR2（35），GEOM SDO_GEOMETRY）;

C.2 SDO_WITHIN_DISTANCE示例

SDO_WITHIN_DISTANCE运算符标识在给定对象的某个指定距离内的空间对象集。您可以指示距离是近似的还是精确的。如果指定querytype=FILTER ，则距离是近似值，因为只执行主过滤操作;否则，距离是精确的，因为执行了主要和次要过滤操作。

示例C-1查找公路距离内的所有城市

示例C-1查找州际公路I170的15英里范围内的所有城市。 

SELECT / * + ORDERED * / c.city FROM geod_interstates i，geod_cities c WHERE i.highway ='I170'和sdo_within_distance（c.location，i.geom，'distance = 15 unit = mile'）='TRUE';

示例C-1查找指定高速公路（ i.highway = 'I170' ）的15英里（ 'distance=15 unit=mile' ）内的所有城市，并且默认情况下结果是精确的（因为querytype参数未用于将查询限制为主过滤器操作）。在此示例的WHERE子句中：

  • i.highway指的是INTERSTATES表的HIGHWAY列， I170是HIGHWAY列的值。

  • c.location指定搜索列（ geometry1 ）。这是GEOD_CITIES表的LOCATION列。

  • i.geom指定查询窗口（ aGeom ）。这是GEOD_INTERSTATES表的GEOM列中的空间几何，其HIGHWAY列包含值I170 。

示例C-2查找城市距离内的所有高速公路

示例C-2发现距离坦帕市15英里范围内的所有州际高速公路。 

SELECT / * + ORDERED * / i.highway FROM geod_cities c，geod_interstates i WHERE c.city ='Tampa'和sdo_within_distance（i.geom，c.location，'distance = 15 unit = mile'）='TRUE';

示例C-2查找指定城市（ c.city = 'Tampa' ）15英里（ 'distance=15 unit=mile' ）内的所有高速公路，默认情况下结果是精确的（因为querytype参数未用于将查询限制为主过滤器操作）。在此示例的WHERE子句中：

  • c.city指的是GEOD_CITIES表的CITY列， Tampa是CITY列的值。

  • i.geom指定搜索列（ geometry1 ）。这是GEOD_INTERSTATES表的GEOM列。

  • c.location指定查询窗口（ aGeom ）。这是GEOD_CITIES表的LOCATION列中的空间几何，其CITY列包含值Tampa 。

C.3 SDO_NN示例

SDO_NN运算符确定几何的最近邻几何。除非您在ORDER BY子句中指定SDO_NN_DISTANCE辅助运算符以使结果按距离顺序返回，否则不应对返回结果的顺序进行任何假设。如果未指定可选参数，则返回一个最近邻居几何体。

如果指定可选的sdo_num_res关键字，则可以请求所需的最近邻居数，但不会评估WHERE子句中的其他条件。例如，假设您想要距交叉点最近的五个银行，但仅限于银行名称为CHASE 。如果五个最接近的银行未命名为CHASE ， 则 sdo_num_res=5将不返回任何行，因为sdo_num_res关键字仅考虑接近度，而不是WHERE子句中的任何其他条件。

如果指定可选的sdo_batch_size关键字，则会忽略任何sdo_num_res规范，并且SDO_NN会按距离顺序将相邻几何返回到WHERE子句。如果WHERE子句指定bank_name = 'CHASE' AND rownum < 6 ，则可以返回bank_name = 'CHASE'的五个最接近的bank。

SDO_NN_DISTANCE是SDO_NN运算符的辅助运算符。它返回SDO_NN运算符返回的对象的距离，仅在对SDO_NN运算符的调用中有效。

另请参阅使用分区空间索引中的 示例5-1 。

示例C-3查找最近高速公路的城市

示例C-3找到距离州际公路I170最近的五个城市以及每个城市距离高速公路的英里距离，按英里距离排序。 

SELECT / * + ORDERED * / c.city，sdo_nn_distance（1）distance_in_miles FROM geod_interstates i，geod_cities c WHERE i.highway ='I170'和sdo_nn（c.location，i.geom，'sdo_num_res = 5 unit = mile'， 1）='TRUE'ORDER by distance_in_miles;

在示例C-3中 ，因为使用了/*+ ORDERED*/ optimizer提示，所以在GEOD_INTERSTATES.HIGHWAY列上有一个索引很重要。在此示例中，提示强制查询在尝试查找最近邻居几何之前定位高速公路I170。在此示例的WHERE子句中：

  • i.highway指的是GEOD_INTERSTATES表的HIGHWAY列， I170是HIGHWAY列的值。

  • c.location指定搜索列（ geometry1 ）。这是GEOD_CITIES表的LOCATION列。

  • i.geom指定查询窗口（ geometry2 ）。这是GEOD_INTERSTATES表的GEOM列中的空间几何，其HIGHWAY列包含值I170 。

  • sdo_num_res=5指定要查找的最近邻居几何数。

  • unit=mile指定与SDO_NN_DISTANCE辅助运算符返回的距离关联的度量单位。

  • 1 （在sdo_nn_distance (1)和'sdo_num_res=5 unit=mile', 1 ）是number ，该调用相关联的参数值SDO_NN与呼叫到SDO_NN_DISTANCE 。

在示例C-3中 ， ORDER BY distance_in_miles按距离（英里）对WHERE子句的结果进行排序。

示例C-3中的语句产生以下输出（为了便于阅读，略微重新格式化）：

CITY DISTANCE_IN_MILES ---------------------- -------------------------- ---- St Louis 5.36297295 Springfield 78.7997464 Peoria 141.478022 Evansville 158.22422 Springfield 188.508631

示例C-4查找高速公路附近特定人口以上的城市

示例C-4通过将结果限制为具有超过一定数量的1990人口的城市来扩展示例C-3 。它找到了最接近I170州际公路的五个城市，人口超过300,000，每个城市的1990年人口，以及每个城市距离公路的距离，按英里距离排序。 

SELECT / * + ORDERED NO_INDEX（c pop90_idx）* / c.city，pop90，sdo_nn_distance（1）distance_in_miles FROM geod_interstates i，geod_cities c WHERE i.highway ='I170'和sdo_nn（c.location，i.geom，'sdo_batch_size = 10 unit = mile'，1）='TRUE'且c.pop90> 300000 AND rownum <6 ORDER BY distance_in_miles;

在示例C-4中 ，由于使用了ORDERED优化程序提示，因此在GEOD_INTERSTATES.HIGHWAY列上建立索引非常重要。在此示例中，提示强制查询在尝试查找最近邻居几何之前定位高速公路I170。

要确保正确的结果，请禁用与SDO_NN搜索列（ geometry1 ）相同的表的列上的所有非空间索引。在此示例中， NO_INDEX(c pop90_idx)优化程序提示禁用POP90列上的非空间索引。

在此示例的WHERE子句中：

  • sdo_batch_size=10导致连续返回几何（按距离顺序，以10个几何的批量），以检查它们是否满足WHERE子句中的其他条件。

  • c.pop90 > 300000将结果限制为POP90列值大于300000的行。

  • rownum < 6将返回的结果数限制为五。

在示例C-4中 ， ORDER BY distance_in_miles按距离（英里）对WHERE子句的结果进行排序。

示例C-4中的语句产生以下输出（为了便于阅读，略微重新格式化）：
