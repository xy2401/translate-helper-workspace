测试Oracle自动存储管理安装

在为单个实例安装Oracle Grid Infrastructure之后，使用ASMCMD命令行实用程序来测试Oracle ASM安装。

  1. 打开shell窗口，并临时设置ORACLE_SID和ORACLE_HOME环境变量，以指定要使用的Oracle ASM实例的相应值。

    例如，如果Oracle ASM SID名为+ ASM且Oracle主目录位于ORACLE_BASE目录的grid子目录中，则输入以下命令以创建所需设置：

    Bourne，Bash或Korn shell：

    $ ORACLE_SID = + ASM $ export ORACLE_SID $ ORACLE_HOME = / u01 / app / oracle / product / 19.0.0 / grid $ export ORACLE_HOME

    C shell：

    ％setenv ORACLE_SID + ASM％setenv ORACLE_HOME /u01/app/oracle/product/19.0.0/grid
  2. 使用ASMCMD列出Oracle ASM实例的磁盘组：
    $ ORACLE_HOME / bin / asmcmd lsdg
    如果Oracle ASM实例正在运行，则ASMCMD默认连接为具有SYSASM权限的SYS用户，并且可用。
  3. 如果Oracle ASM实例未运行，则使用以下命令启动Oracle ASM实例：
    $ ORACLE_HOME / bin / srvctl启动asm
