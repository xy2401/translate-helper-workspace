194 DBMS_XDBZ

DBMS_XDBZ包控制Oracle XML DB存储库安全性，该安全性基于访问控制列表（ACL）。

本章包含以下主题：

194.1 DBMS_XDBZ安全模型

由XDB拥有， DBMS_XDBZ包必须由SYS或XDB创建。 EXECUTE特权被授予PUBLIC 。此包中的子程序使用当前用户的权限执行。

194.3.1 CREATENONCEKEY程序

此过程生成用于摘要式身份验证的随机数值。

句法

DBMS_XDBZ.CREATENONCEKEY;

194.3.2 DISABLE_HIERARCHY过程

此过程禁用特定XMLType表或视图的存储库支持。

句法

DBMS_XDBZ.DISABLE_HIERARCHY（object_schema IN VARCHAR2，object_name IN VARCHAR2）;

参数

表194-5 DISABLE_HIERARCHY过程参数

参数 描述

object_schema

XMLType表或视图的模式名称

OBJECT_NAME

XMLType表或视图的名称

194.3.3 ENABLE_HIERARCHY程序

此过程为特定XMLType表或视图启用存储库支持。这允许在存储库中的所有文档中使用基于ACL的统一安全模型。

也可以看看：

有关的更多信息，请参阅Oracle XML DB Developer's Guide

句法

DBMS_XDBZ.ENABLE_HIERARCHY（object_schema IN VARCHAR2，object_name IN VARCHAR2，hierarchy_type IN PLS_INTEGER：= DBMS_XDBZ.ENABLE_CONTENTS）;

参数

表194-6 ENABLE_HIERARCHY过程参数

参数 描述

object_schema

XMLType表或视图的模式名称

OBJECT_NAME

XMLType表或视图的名称

hierarchy_type

如何启用层次结构。

  • ENABLE_CONTENTS - 为内容启用层次结构，也就是说，此表将存储资源库中的资源内容

  • ENABLE_RESMETADATA - 为资源元数据启用层次结构，也就是说，此表将存储基于模式的资源自定义元数据

如果在表上调用此子程序，则另一个调用将不起作用。请注意，您无法为内容和资源元数据启用层次结构。

194.3.4 GET_ACLOID功能

如果存储库路径已知，则此函数检索指定资源的ACL对象ID。

句法

DBMS_XDBZ.GET_ACLOID（aclpath IN VARCHAR2，acloid OUT RAW）RETURN BOOLEAN;

参数

表194-7 GET_ACLOID函数参数

参数 描述

aclpath

存储库的ACL资源路径

acloid

返回的对象ID

返回值

如果成功则返回TRUE 。

194.3.5 GET_USERID函数

此函数检索指定用户名的用户标识。首先搜索本地数据库，如果找到， USERID 4字节数据库格式返回USERID 。否则，搜索LDAP目录（如果可用），如果找到， USERID 4字节数据库格式返回USERID 。

句法

DBMS_XDBZ.GET_USERID（用户名IN VARCHAR2，用户ID OUT RAW，格式IN BINARY_INTEGER：= NAME_FORMAT_SHORT）RETURN BOOLEAN;

参数

表194-8 GET_USERID函数参数

参数 描述

用户名

数据库或LDAP用户的名称。

用户身份

返回匹配用户标识的参数。

格式

指定用户名的格式;有效选项是：

  • DBMS_XDBZ.NAME_FORMAT_SHORT （默认值） - 数据库用户名或LDAP昵称

  • DBMS_XDBZ.NAME_FORMAT_DISTINGUISHIED - LDAP专有名称。

返回值

如果成功则返回TRUE 。

194.3.6 IS_HIERARCHY_ENABLED函数

此函数确定是否启用了对指定XMLType表或视图的存储库支持。

句法

DBMS_XDBZ.IS_HIERARCHY_ENABLED（object_schema IN VARCHAR2，object_name IN VARCHAR2，hierarchy_type IN PLS_INTEGER：= IS_ENABLED_CONTENTS）RETURN BOOLEAN;

参数

表194-9 IS_HIERARCHY_ENABLED函数参数

参数 描述

object_schema

XMLType表或视图的模式名称

OBJECT_NAME

XMLType表或视图的名称

hierarchy_type

要检查的层次结构类型：

  • IS_ENABLED_CONTENTS - 如果为内容启用了层次结构，即，使用hierarchy_type作为ENABLE_CONTENTS调用ENABLE_HIERARCHY过程

  • IS_ENABLED_RESMETADATA - 如果为资源元数据启用了层次结构，即，使用hierarchy_type作为ENABLE_RESMETADATA调用ENABLE_HIERARCHY过程

返回值

如果给定的XMLTYPE表或视图使用指定的类型启用了XDB层次结构，则返回TRUE 。

194.3.7 PURGELDAPCACHE功能

此函数清除LDAP昵称缓存。

如果成功则返回TRUE 。

句法

DBMS_XDBZ.PURGELDAPCACHE RETURN BOOLEAN;
