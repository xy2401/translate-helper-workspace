1.13 APPROX_FOR_COUNT_DISTINCT

APPROX_FOR_COUNT_DISTINCT自动用APPROX_COUNT_DISTINCT查询替换COUNT (DISTINCT expr )查询。

属性 描述

参数类型

布尔

默认值

假

可修改

ALTER SESSION，ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

Oracle RAC

不同的实例可以使用不同的值。

APPROX_COUNT_DISTINCT查询的查询结果的返回速度快于等效的COUNT (DISTINCT expr )查询。APPROX_COUNT_DISTINCT查询对于可接受的容错量是可接受的情况非常有用，以便获得比使用COUNT (DISTINCT expr )查询更快的查询结果。
