创建TABLESPACE SET

注意：

只有在使用Oracle Sharding时，此SQL语句才有效。有关Oracle Sharding的更多信息，请参阅Oracle Database Administrator's Guide 。

目的

使用CREATE TABLESPACE SET语句创建表空间集。表空间集可以在分片数据库中用作一个或多个分片表和索引的逻辑存储单元。

表空间集包含分布在分片空间中的分片中的多个表空间。数据库自动在表空间集中创建表空间。表空间的数量是自动确定的，并且等于相应分片空间中的块数。

表空间集中的所有表空间都是永久的bigfile表空间;表空间集不包含SYSTEM ，undo或临时表空间。数据库自动为每个表空间创建一个数据文件。表空间集中的所有表空间共享相同的属性。您可以使用ALTER TABLESPACE SET语句修改表空间中所有表空间的属性。

语义

tablespace_set

指定要创建的表空间集的名称。该名称必须满足数据库对象命名规则中列出的要求。

在SHARDSPACE

如果使用复合分片，请指定此子句。对于shardspace_name ，请指定要在其中创建表空间集的分片空间的名称。

如果使用系统管理的分片，请省略此子句。在这种情况下，表空间集在分片数据库的默认分片空间中创建。

使用模板

USING TEMPLATE子句允许您为表空间集中的表空间指定属性。

DATAFILE和permanent_tablespace_attrs子句与CREATE TABLESPACE语句具有相同的语义，但有以下例外：

  • 对于DATAFILE file_specification子句，只能指定SIZE子句和autoextend_clause 。

  • 您无法指定MINIMUM EXTENT size_clause 。

  • 对于segment_management_clause ，您只能指定SEGMENT SPACE MANAGEMENT AUTO 。不支持MANUAL设置。
