在使用DBCA创建任何Oracle RAC数据库之前要完成的任务

在使用DBCA创建Oracle RAC数据库之前，如果在Oracle Grid Infrastructure安装期间未完成系统配置，则必须配置系统以满足软件要求。

  • 在启动DBCA之前将SSH密钥加载到内存中
    在Oracle RAC环境中，必须将SSH密钥加载到内存中，以用于启动DBCA的终端会话。如果不这样做，则在尝试启动DBCA时会收到用户等效错误。
  • 确定用于Oracle RAC数据库的命名约定
    有关Oracle RAC数据库的命名约定，请查看此信息。
  • 为Oracle RAC数据库配置共享存储
    在启动DBCA以配置Oracle RAC数据库之前，必须已为群集安装了Oracle Grid Infrastructure，并且必须已为Oracle RAC文件配置了共享存储区域。
