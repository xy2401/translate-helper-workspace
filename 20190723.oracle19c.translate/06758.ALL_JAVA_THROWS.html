2.192 ALL_JAVA_THROWS

ALL_JAVA_THROWS显示有关当前用户可访问的Java类的方法抛出的异常的信息。

相关观点

  • DBA_JAVA_THROWS显示有关从数据库中所有Java类的方法抛出的异常的信息。

  • USER_JAVA_THROWS显示有关当前用户拥有的Java类的方法抛出的异常的信息。此视图不显示OWNER列。
