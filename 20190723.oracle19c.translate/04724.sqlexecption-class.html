13.28.3 getMessage（）

如果使用错误消息字符串创建，则返回此SQLException的错误消息字符串。如果创建的SQLException没有错误消息，则返回NULL 。

句法

string getMessage（）const;

13.28.4 getNLSMessage（）

返回的错误信息string如此SQLException ，如果它是一个错误信息创建string 。通过全球化启用的环境。如果创建的SQLException没有错误消息，则返回NULL string 。错误消息位于与环境关联的字符集中。

句法

string getNLSMessage（Environment * env）const;
参数 描述 
ENV

全球化促成了环境。

13.28.5 getNLSUStringMessage（）

返回的错误信息UString如此SQLException ，如果它是一个错误信息创建UString 。通过全球化启用的环境。如果创建的SQLException没有错误消息，则返回NULL UString 。错误消息位于与环境关联的字符集中。

句法

UString getNLSUStringMessage（Environment * env）const;
参数 描述 
ENV

全球化促成了环境。

13.28.6 getUStringMessage（）

返回的错误信息UString如此SQLException ，如果它是一个错误信息创建UString 。如果创建的SQLException没有错误消息，则返回NULL UString 。错误消息位于与环境关联的字符集中。

句法

UString getUStringMessage（）const;

13.28.7 getXAErrorCode（）

确定抛出的异常是由XA还是SQL错误引起的。

由动态注册的C ++ XA应用程序使用。如果异常是由XA引起的，则返回XA错误代码，否则返回XA_OK 。

句法

int getXAErrorCode（const string＆dbname）const;
参数 描述 
DBNAME

数据库名称;与Open String中提供的可选dbname相同，并在连接到Resource Manager时使用。

13.28.8 isRecoverable（）

确定应用程序是否可以故障转移并从错误中恢复。如果可恢复，则返回TRUE 。

例如，应用程序可以从ORA-03113恢复，但不能从ORA-942恢复。

句法

bool isRecoverable（）;
