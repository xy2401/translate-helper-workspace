关于Oracle ASM和Oracle ASM过滤器驱动程序

在Oracle Grid Infrastructure安装期间，您可以选择安装和配置Oracle自动存储管理筛选器驱动程序（Oracle ASMFD）。Oracle ASMFD有助于防止Oracle ASM磁盘和磁盘组中的文件损坏。

Oracle ASM筛选器驱动程序（Oracle ASMFD）拒绝Oracle软件未发出的写入I / O请求。此写入筛选器有助于防止具有管理权限的用户无意中覆盖Oracle ASM磁盘，从而防止Oracle ASM磁盘和磁盘组中的文件损坏。对于磁盘分区，受保护的区域是Oracle ASMFD管理的磁盘上的区域，假设分区表未被用户保持不变。

Oracle ASMFD通过在每次重新启动系统时无需重新绑定与Oracle ASM一起使用的磁盘设备，从而简化了磁盘设备的配置和管理。

如果Linux系统上存在Oracle ASMLIB，则在安装Oracle Grid Infrastructure之前卸载Oracle ASMLIB，以便您可以选择在Oracle Grid Infrastructure安装期间安装和配置Oracle ASMFD。

警告：

配置Oracle ASM（包括Oracle ASMFD）时，请勿修改或删除Oracle ASM磁盘的内容，也不要修改任何文件，包括配置文件。

注意：

Linux x86-64和Oracle Solaris操作系统支持Oracle ASMFD。
