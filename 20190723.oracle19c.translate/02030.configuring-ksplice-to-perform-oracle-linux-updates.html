配置Oracle Ksplice以执行Oracle Linux更新

如果您拥有Oracle Linux Premier Support订阅和访问密钥（可在Unbreakable Linux Network（ULN）上使用），则可以将Oracle Ksplice配置为对您的操作系统执行零停机更新。

有关Ksplice（包括试用版）的更多信息，请参阅Ksplice站点：

http://ksplice.oracle.com/

要使用Ksplice，您必须注册您的系统并创建Ksplice存储库。

  1. 在以下URL检查您的内核分发：
    http://ksplice.oracle.com/legacy#supported-kernels
  2. 以root.身份登录root.
  3. 确保您可以在要使用Ksplice的服务器上访问Internet。例如，如果您使用的是代理服务器，则使用类似于以下内容的命令在shell中设置代理服务器和端口值：
    #export http_proxy = http：//proxy.example.com:port #export https_proxy = http：//proxy.example.com:port
  4. 下载Ksplice Uptrack存储库RPM包：
    https://www.ksplice.com/yum/uptrack/ol/ksplice-uptrack-release.noarch.rpm
  5. 运行以下命令：
    rpm -i ksplice-uptrack-release.noarch.rpm yum -y install uptrack
