2.186 ALL_JAVA_IMPLEMENTS

ALL_JAVA_IMPLEMENTS描述了由当前用户可访问的存储Java类实现的接口。

相关观点

  • DBA_JAVA_IMPLEMENTS描述了由数据库中所有存储的Java类实现的接口。

  • USER_JAVA_IMPLEMENTS描述由当前用户拥有的存储Java类实现的接口。此视图不显示OWNER列。
