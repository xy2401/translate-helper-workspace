4.293 DBA_HIST_BASELINE

DBA_HIST_BASELINE显示有关系统中采用的基线的信息。

对于每个基线，此视图显示完整的时间范围以及基线是否为默认基线。

柱 数据类型 空值 描述

DBID

数

数据库ID

BASELINE_ID

数

基线的内部ID

BASELINE_NAME

VARCHAR2（64）

用户指定的基线名称

BASELINE_TYPE

VARCHAR2（13）

基线类型如下：

STATIC - 用户手动创建的基线

MOVING WINDOW - 具有动态开始和结束快照ID的基线

GENERATED - 使用模板由系统自动生成的基线

START_SNAP_ID

数

启动基准的快照ID

START_SNAP_TIME

TIMESTAMP（3）

与启动快照ID关联的时间

END_SNAP_ID

数

结束基线的快照ID

END_SNAP_TIME

TIMESTAMP（3）

与结束快照ID关联的时间

MOVING_WINDOW_SIZE

数

如果BASELINE_TYPE是MOVING WINDOW ，则此字段是移动窗口的大小，以天数为单位。

如果为NULL，则窗口大小为AWR保留设置的值。

CREATION_TIME

日期

创建基线的时间

到期

数

保持基线的天数，以天数为单位。NULL值表示基线将永久保留。

TEMPLATE_NAME

VARCHAR2（64）

创建此基准的模板的名称（如果有）。

LAST_TIME_COMPUTED

日期

上次在基线上计算统计数据。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
