CON_NAME_TO_ID

句法


con_name_to_id.eps插图的描述

目的

CON_NAME_TO_ID将容器名称作为其参数，并返回容器ID。对于container_name ，请在任何数据类型中指定字符串或解析为字符串的表达式。该函数返回NUMBER值。

此功能在多租户容器数据库（CDB）中很有用。如果在非CDB中使用此函数，则返回0。

例

以下查询显示CDB中所有容器的ID和名称。显示的示例输出用于此示例的目的。

从CON $ CONTAINERS中选择CON_ID，NAME; CON_ID NAME ---------- ---------- 1 CDB $ ROOT 2 PDB $ SEED 4 SALESPDB

以下语句返回名为SALESPDB的容器的ID：

SELECT CON_NAME_TO_ID（'SALESPDB'）“容器ID”FROM DUAL;容器ID ------------ 4
