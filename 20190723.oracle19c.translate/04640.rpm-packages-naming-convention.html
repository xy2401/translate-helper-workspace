RPM包命名约定

RPM包命名约定包括诸如版本，发行版和体系结构之类的信息。

RPM包的命名约定是name - version - release 。 architecture .rpm

下表描述了RPM oracle-database-ee-19c-1.0-1.x86_64.rpm

表11-2 RPM软件包命名约定示例

变量 例 描述
名称 Oracle的数据库EE-19C

Oracle数据库版本和发行版的名称。
版 1.0

RPM包的版本号。
发布 1

RPM包的版本号。
建筑 x86_64的

RPM包的体系结构。
