B键盘快捷键

键盘快捷键支持Analytic Workspace Manager中的辅助功能。大多数快捷方式适用于所有平台，但Windows为所有平台提供最可靠的结果。如果使用键盘快捷方式进行辅助功能，请在Windows平台上安装Analytic Workspace Manager。

键盘快捷键在用户界面的特定区域内处于活动状态：

  • 菜单栏

  • 导航树

  • 物业表

  • 穿梭钥匙

  • 映射画布

B.1菜单栏

文件菜单：Alt + F.

工具菜单：Alt + T.

帮助菜单：Alt + H.

B.2导航树

要显示所选对象的菜单，请按Shift + F10。这相当于单击鼠标右键。

要关闭所选对象的菜单，请按Esc。

要展开文件夹，请按向右箭头键。

要折叠文件夹，请按向左箭头键。

要将光标向下移动树，请按向下箭头键。

要将光标向上移动树，请按向上箭头键。

要将光标从导航树移动到属性表，请按Tab键。

要将光标从属性表移动到导航树，请按Shift + Tab。

B.3物业单

要将光标从导航树移动到属性表，请按Tab键。

要将光标移动到下一个选项卡，请按向右箭头键。

要将光标移动到上一个选项卡，请按向左箭头键。

要将光标从属性表移动到导航树，请按Shift + Tab。

要在导航树和属性表之间移动拆分器，请按F8向右箭头或向左箭头。

要更改表格中的菜单选项，请按F2向下箭头。

B.4班车钥匙

全部移动：Alt + L.

移动选中：Alt + D.

删除所选：Alt + R.

全部删除：Alt + O.

要选择多个项目，请按Ctrl +箭头，然后按空格键。

B.5映射画布

表映射视图：Ctrl + T.

图形映射视图：Ctrl + G.

自动排列映射：Ctrl + Alt + K.

自动映射星型架构：Ctrl + M.

删除所有映射：Ctrl + D.

架构查看器导航器：

可以使用导航树的所有键盘快捷键，以及表视图的以下附加内容：

要从树中复制所选列的名称：Ctrl + C.要将列名粘贴到所选字段中：Ctrl + V.

要扩展列的宽度：选择标题，然后按Alt +向右箭头。

要减小列的宽度：选择标题，然后按Alt +向左箭头。
