Oracle Grid Infrastructure使用的网络的广播要求

广播通信（ARP和UDP）必须在配置为供Oracle Grid Infrastructure使用的所有公共和专用接口上正常工作。

广播必须在公共或专用接口使用的任何已配置VLAN上工作。

为Oracle RAC配置公用和专用网络接口时，必须启用地址解析协议（ARP）。高可用IP（HAIP）地址在公共网络上不需要ARP，但是对于VIP故障转移，您需要启用ARP。不要配置NOARP 。
