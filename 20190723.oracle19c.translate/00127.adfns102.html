说明adfns102.eps的描述

该过程包括以下步骤：

  1. 用户访问网页，跟随超文本链接或以表单形式提交数据，这使得浏览器将URL的HTTP请求发送到HTTP服务器。

  2. HTTP服务器根据URL中编码的数据在Oracle数据库上调用存储的子程序。 URL中的数据采用要传递给存储的子程序的参数的形式。

  3. 存储的子程序调用PL / SQL Web Toolkit中的子程序。通常，子程序如HTP 。 Print动态生成网页。生成的网页因数据库内容和输入参数而异。

  4. 子程序将动态生成的页面传递给Web服务器。

  5. Web服务器将页面传递给客户端。
