不推荐的功能

此版本中不推荐使用以下功能，并且可能在将来的版本中不支持这些功能。

  • PL / SQL包DBMS_XMLQUERY 。请改用DBMS_XMLGEN包。

  • PL / SQL包DBMS_XMLSAVE 。请改用DBMS_XMLSTORE包。

不受支持的功能

此版本不支持以下功能。

  • PL / SQL子程序包DBMS_XMLSCHEMA generateSchema和generateSchemas 。这些子程序没有替代品。

  • DBMS_XDB包中的以下子程序。请改用包DBMS_XDBCONFIG使用相同名称的子程序。

    • addhttpexpiremapping

    • addmimemapping

    • addschemalocmapping

    • addservlet

    • addservletmapping

    • addservletsecrole

    • addxmlextension

    • cfg_get

    • cfg_refresh

    • cfg_update

    • deletehttpexpiremapping

    • deletemimemapping

    • deleteschemalocmapping

    • deleteservlet

    • deleteservletmapping

    • deleteservletsecrole

    • deletexmlextension

    • getftpport

    • gethttpport

    • getlistenerendpoint

    • setftpport

    • sethttpport

    • setlistenerendpoint

    • setlistenerlocalaccess

  • DBMS_XDB包中的以下常量。请改为使用包DBMS_XDBCONFIG具有相同名称的常量。

    • xdb_endpoint_http

    • xdb_endpoint_http2

    • xdb_protocol_tcp

    • xdb_protocol_tcps

  • 用于更新XML数据的所有Oracle特定SQL函数。请改用XQuery Update。这些是不受支持的XML更新功能：

    • 如updateXML

    • insertChildXML

    • insertChildXMLbefore

    • insertChildXMLafter

    • insertXMLbefore

    • insertXMLafter

    • appendChildXML

    • deleteXML

  • Oracle XQuery函数ora:matches和ora:replace 。使用相应的标准XQuery函数，即具有相同名称但具有名称空间前缀fn ： fn:matches和fn:replace的函数。

  • 不支持以下支持XML转换的Oracle构造。任何这些结构都没有替代品。

    • PL / SQL包DBMS_XMLTRANSLATIONS

    • Oracle XPath函数ora:translate

    • XML Schema annotations xdb:maxOccurs ， xdb:srclang和xdb:translate

  • 不支持以下XML架构注释。任何这些注释都没有替代品。

    • XDB：defaultTableSchema

    • XDB：maintainOrder

    • XDB：mapUnboundedStringToLob

    • XDB：maxOccurs的

    • XDB：SQLCollSchema

    • XDB：SQLSchema

    • XDB：srclang

    • XDB：翻译

  • 值xml_clobs出口参数data_options 。此参数没有替代品。
