了解Oracle Flex Clusters

从Oracle Grid Infrastructure 12c第2版（12.2）开始，Oracle Grid Infrastructure集群配置是Oracle Flex Clusters部署。

安装在Oracle Flex Cluster配置中的Oracle Grid Infrastructure是一个可扩展，动态，强大的节点网络。Oracle Flex Clusters为具有大量节点的Oracle Real Application Clusters数据库提供了一个平台，以支持大规模并行查询操作。Oracle Flex Clusters还为其他服务部署提供了一个平台，需要协调和自动化以实现高可用性。

Oracle Flex Cluster中的所有节点都属于单个Oracle Grid Infrastructure集群。此体系结构集中了基于应用程序需求部署资源的策略决策，以考虑各种服务级别，负载，故障响应和恢复。Oracle Flex Clusters中的节点紧密连接，可直接访问共享存储
