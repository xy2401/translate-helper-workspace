5.12 DBA_HIST_MTTR_TARGET_ADVICE

DBA_HIST_MTTR_TARGET_ADVICE显示对应于每行的MTTR的物理I / O请求数的历史预测。

该数据还包括物理I / O因子，其是在测量间隔期间估计的I / O请求的数量与当前MTTR设置实际执行的I / O请求的数量的比率。该视图包含V$MTTR_TARGET_ADVICE快照。

柱 数据类型 空值 描述

SNAP_ID

数

不是NULL

唯一快照ID

DBID

数

不是NULL

快照的数据库ID

INSTANCE_NUMBER

数

不是NULL

快照的实例编号

MTTR_TARGET_FOR_ESTIMATE

数

模拟MTTR设置（如果这是视图的第一行，则等于当前MTTR设置）

ADVICE_STATUS

VARCHAR2（5）

MTTR仿真的现状：

  • ON SET

  • 准备好了

  • 关闭

DIRTY_LIMIT

数

从模拟的MTTR导出的脏缓冲区限制

ESTD_CACHE_WRITES

数

MTTR下估计的缓存物理写入数

ESTD_CACHE_WRITE_FACTOR

数

估计MTTR下的缓存物理写入比率。它是当前MTTR设置下估计的高速缓存写入次数与高速缓存写入次数的比率。

ESTD_TOTAL_WRITES

数

估计MTTR下的物理写入总数

ESTD_TOTAL_WRITE_FACTOR

数

估计MTTR下的总物理写入比率。它是在当前MTTR设置下估计的物理写入总数与物理写入总数之比。

ESTD_TOTAL_IOS

数

估计MTTR下的I / O请求总数

ESTD_TOTAL_IO_FACTOR

数

估计MTTR下的总I / O比率。它是在当前MTTR设置下估计的I / O请求总数与I / O请求总数之比。

CON_DBID

数

采样会话的PDB的数据库ID

CON_ID

数

CON_DBID标识的容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

也可以看看：

“ V $ MTTR_TARGET_ADVICE ”
