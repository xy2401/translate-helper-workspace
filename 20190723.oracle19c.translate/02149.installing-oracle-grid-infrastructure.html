9安装Oracle Grid Infrastructure

查看有关Oracle Grid Infrastructure的安装和部署选项的此信息。

Oracle数据库和Oracle Grid Infrastructure安装软件有多种介质，可以使用多种选项进行安装。Oracle Grid Infrastructure软件以图像形式提供，可从Oracle Technology Network网站或Oracle Software Delivery Cloud门户下载。在大多数情况下，您使用Oracle Universal Installer提供的图形用户界面（GUI）来安装软件。您也可以使用Oracle Universal Installer完成静默模式安装，而无需使用GUI。您还可以将Oracle Fleet修补和供应用于后续Oracle Grid Infrastructure和Oracle数据库部署。

  • 关于基于映像的Oracle Grid Infrastructure安装
    从Oracle Grid Infrastructure 12c第2版（12.2）开始，通过基于映像的安装简化了Oracle Grid Infrastructure软件的安装和配置。
  • 安装向导用于创建映像的安装选项
    在启动Oracle数据库或Oracle Grid Infrastructure安装的安装向导之前，请确定是否要使用任何可用的映像创建选项。
  • 了解群集配置选项
    查看这些主题以了解Oracle Grid Infrastructure 19c中可用的群集配置选项。
  • 为新群集安装Oracle Grid Infrastructure
    查看这些过程以安装此版本的Oracle Grid Infrastructure中提供的群集配置选项。
  • 使用群集配置文件安装Oracle Grid Infrastructure
    在安装Oracle Grid Infrastructure期间，您可以选择手动提供群集配置信息，也可以使用群集配置文件。
  • 仅安装Oracle Grid Infrastructure软件
    此安装选项需要手动安装后步骤才能启用Oracle Grid Infrastructure软件。
  • 关于使用Oracle Fleet补丁和配置部署Oracle Grid Infrastructure
    Oracle Fleet Patching and Provisioning（Oracle FPP）是一种用于配置和维护Oracle主目录的软件生命周期管理方法。Oracle Fleet Patching and Provisioning支持大规模部署和维护数据库，集群和用户定义的软件类型的标准操作环境。
  • 确认Oracle Clusterware功能
    安装Oracle Grid Infrastructure后，确认已安装并正确运行Oracle Clusterware安装。
  • 确认Oracle Clusterware文件的Oracle ASM功能
    安装Oracle Grid Infrastructure后确认Oracle ASM正在运行。
  • 了解Oracle Grid Infrastructure中的脱机进程
    安装Oracle Grid Infrastructure后，某些组件可能会列为OFFLINE。当您选择添加这些资源时，Oracle Grid Infrastructure会激活这些资源。
