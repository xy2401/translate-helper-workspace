280 MGD_ID包类型

MGD_ID包提供了一个可扩展的框架，支持当前的射频ID（RFID）标签，其标准系列的EPC位编码用于支持的编码类型。MGD_ID包还支持新的和不断发展的标签编码，这些编码不包含在当前的EPC标准（EPC v1.1规范）中。

MGD_ID包包含几种预定义类型 。

280.1 MGD_ID包类型安全模型

您必须运行catmgd.sql脚本以加载DBMS_MGD_ID_UTL包并在MGDSYS模式中创建所需的Identity Code Package模式对象。

MGD_ID是MGDSYS拥有的对象类型。从匿名PL / SQL块调用的任何MGD_ID子程序都使用当前用户的权限运行。

必须为用户授予连接和资源角色才能使用MGD_ID对象类型及其子程序。

为此ADT授予PUBLIC EXECUTE特权： MGD_ID 。

为此ADT创建一个同名的公共同义词： MGD_ID 。

280.3.1 MGD_ID构造函数

此构造函数构造标识代码类型对象MGD_ID 。构造函数已重载。每种语法形式的不同功能与定义一起呈现。

句法

根据类别ID和组件列表构造MGD_ID对象类型。 

MGD_ID（category_id IN VARCHAR2，组件IN MGD_ID_COMPONENT_VARRAY）返回自定为结果确定性;

根据类别ID，标识符字符串以及创建它所需的其他参数列表构造MGD_ID对象类型。 

MGD_ID（category_id VARCHAR2，标识符VARCHAR2，parameter_list VARCHAR2）返回自定为结果确定性;

根据类别名称，类别版本和组件列表构造MGD_ID对象类型。 

MGD_ID（category_name VARCHAR2，category_version VARCHAR2，组件MGD_ID_COMPONENT_VARRAY）返回自身结果确定性;

根据类别名称，类别版本，标识符字符串以及创建它所需的其他参数列表构造MGD_ID对象类型。 

MGD_ID（category_name VARCHAR2，category_version VARCHAR2，标识符VARCHAR2，parameter_list VARCHAR2）返回自定为结果确定性;

参数

表280-6 MGD_ID构造函数参数

参数 描述

CATEGORY_ID

类别标识符

组件名称值对列表

分类名称

类别名称，例如EPC

category_version

分类版本。如果为NULL ，则将使用指定类别名称的最新版本。

识别码

指定类别中任何格式的编码方案的标识符字符串。例如，对于SGTIN-96编码，标识符的格式可以是BINARY ， PURE_IDENTITY ， TAG_ENCODING或LEGACY 。

根据标签数据转换（TDT）标记文件中的相应语法或模式将此标识符表示为字符串。例如，由字符0和1组成的二进制字符串，URI（标记编码或纯标识格式），或表示为输入字符串格式的序列化遗留代码，例如gtin=00037000302414;serial=10419703 for SGTIN编码方案。

parameter_list

  • 使用MGD_ID_UTL.EPC_ENCODING_CATEGORY_ID作为category_id 。

  • 如果bit_length参数为NULL ，则bit_length是bit_encoding的长度的8 *。

  • 如果组件列表不包含所有必需组件，则将抛出异常MGD_ID_UTL.e_LackComponent 。

TRANSLATE（category_name IN VARCHAR2，标识符IN VARCHAR2，parameter_list IN VARCHAR2，output_format IN VARCHAR2）RETURN VARCHAR2 DETERMINISTIC;
