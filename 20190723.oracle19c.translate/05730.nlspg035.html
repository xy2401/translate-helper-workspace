说明的描述

此图显示了3个客户端，其语言环境为英语，日语和希伯来语。英语客户端使用ISO 8859-1字符集通过HTTP连接到应用程序服务器A中具有英语语言环境的单语应用程序。日语客户端使用通过HTTP连接的Shift-JIS字符集到应用程序中具有日语语言环境的单语应用程序服务器A.希伯来语客户端使用ISO 8859-8字符集通过HTTP连接到应用程序服务器B中具有希伯来语语言环境的单语言应用程序。

Application Server A包含以下内容：

  • 单语应用程序：英语语言环境

  • 应用服务器实例1

  • 单语应用：日语语言环境

  • 应用服务器实例2

英语应用程序使用WE8MSWIN1252 Oracle字符集通过Oracle Net连接到Oracle Unicode数据库。日语应用程序将JA16SJIS Oracle字符集通过Oracle Net连接使用到同一Oracle Unicode数据库。

Application Server B包含以下内容：

  • 单语应用：希伯来语语言环境

  • 应用服务器实例3

希伯来语应用程序使用IW8MSWIN1255 Oracle字符集通过Oracle Net连接到同一个Oracle Unicode数据库。
