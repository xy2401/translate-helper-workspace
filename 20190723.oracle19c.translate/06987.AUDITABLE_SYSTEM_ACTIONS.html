3.175 AUDITABLE_SYSTEM_ACTIONS

AUDITABLE_SYSTEM_ACTIONS将可审计的系统操作号映射到操作名称。启用统一审核时，可以配置这些操作以进行审核。

行动包括：

  • 除以下内容之外的所有标准RDBMS操作（来自V$SQLCOMMAND视图），这些操作不可配置用于审计：

    • 更改版

    • ALTER REWRITE EQUIVALENCE

    • 更新摘要

    • 更改跟踪

    • 创建BITMAPFILE

    • 创建控制文件

    • 创建数据库

    • 创建摘要

    • 不要使用184

    • 不要使用185

    • 不要使用186

    • DECLARE REWRITE EQUIVALENCE

    • DROP BITMAPFILE

    • DROP DATABASE

    • DROP REWRITE EQUIVALENCE

    • DROP SUMMARY

    • FLASHBACK数据库

    • 合并

    • NO-OP（无操作）

    • SAVEPOINT

    • 设置约束

    • UNDROP OBJECT

    • 更新指数

    • 更新加入索引

    • 有效指数

  • 其他行为：

    • 所有

    • LOGON

    • 注销

注意：

此视图仅在启用了统一审核的Oracle数据库中填充。

  • 有关统一审计的详细信息，请参见“ Oracle数据库安全指南” 。

  • 有关迁移到统一审计的详细信息，请参阅“ Oracle数据库升级指南” 。
