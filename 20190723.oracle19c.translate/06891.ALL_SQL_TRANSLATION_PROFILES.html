3.75 ALL_SQL_TRANSLATION_PROFILES

ALL_SQL_TRANSLATION_PROFILES描述了用户可访问的所有SQL翻译配置文件。

相关观点

  • DBA_SQL_TRANSLATION_PROFILES描述数据库中的所有SQL转换配置文件。

  • USER_SQL_TRANSLATION_PROFILES描述用户拥有的所有SQL翻译配置文件。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

SQL翻译配置文件的所有者

PROFILE_NAME

VARCHAR2（128）

不是NULL

SQL转换配置文件的名称

译者

VARCHAR2（261）

翻译包

FOREIGN_SQL_SYNTAX

VARCHAR2（5）

指示SQL语法是否为外语。

可能的值：

  • 真正

  • 假

TRANSLATE_NEW_SQL

VARCHAR2（5）

指示是否使用转换程序转换新的SQL语句和错误。

可能的值：

  • 真正

  • 假

RAISE_TRANSLATION_ERROR

VARCHAR2（5）

指示是否引发翻译错误。

可能的值：

  • 真正

  • 假

LOG_TRANSLATION_ERROR

VARCHAR2（5）

指示是否记录转换错误。

可能的值：

  • 真正

  • 假

VARCHAR2（5）

指示是否跟踪转换。

可能的值：

  • 真正

  • 假
