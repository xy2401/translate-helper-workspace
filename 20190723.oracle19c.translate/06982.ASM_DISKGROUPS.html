1.17 ASM_DISKGROUPS

ASM_DISKGROUPS指定在发出ALTER DISKGROUP ALL MOUNT语句时Oracle ASM实例在启动时安装的磁盘组名称列表。

属性 描述

参数类型

串

句法

ASM_DISKGROUPS = diskgroup [, diskgroup ] ...

默认值

没有默认值。

可修改

改变系统

可在PDB中修改

没有

价值范围

以逗号分隔的字符串列表，最多30个字符

Oracle RAC

多个实例可以具有不同的值。

注意：

此参数只能在Oracle自动存储管理（Oracle ASM）实例中指定。

除非指定了NOMOUNT启动选项，否则Oracle ASM实例启动过程将执行ALTER DISKGROUP ALL MOUNT 。

ASM_DISKGROUPS参数是动态的。如果使用服务器参数文件（SPFILE），则在大多数情况下不必手动更改ASM_DISKGROUPS的值。成功创建或装入磁盘组后，Oracle ASM会自动将磁盘组添加到参数中。删除磁盘组时，Oracle ASM还会自动从参数中删除磁盘组。但是，SPFILE不会在手动卸载时更新。

发行ALTER DISKGROUP...ALL MOUNT或ALTER DISKGROUP...ALL DISMOUNT命令不会影响此参数的值。

支持最多511个磁盘组，用于Oracle ASM实例

在Oracle Database 12 c第1版或更高版本中，Oracle ASM配置最多支持511个磁盘组。在12 c版本1之前的Oracle数据库版本的Oracle ASM配置最多只能支持63个磁盘组。
