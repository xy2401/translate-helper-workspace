关于最小的Linux安装

要完成最小的Linux安装，请选择一个最小安装选项。

最小的Linux安装选项是自定义安装，您可以从“包组选择”中选择“ 最小”选项，或者取消选择除基本包之外的所有包。最小的Linux安装缺少数据库安装所需的许多RPM，因此必须使用适用于Oracle Linux发行版的RPM软件包来安装所需的软件包。您使用的软件包取决于您的Linux版本以及您对Unbreakable Linux Network（ULN）的支持状态。

注意：

Oracle Preinstallation RPM安装X11客户端库，但不安装X Window System服务器软件包。要使用OUI，配置助手和Oracle Enterprise Manager等图形用户界面，请将显示设置为具有X Window System服务器软件包的系统。

有关安装减少的软件包的文档，请参阅以下URL：

https://support.oracle.com/rs?type=doc&id=579101.1

如果您不是Unbreakable Linux Network或Red Hat Support网络的成员，并且您是My Oracle Support客户，那么您可以下载说明以配置脚本，该脚本记录从同一My Oracle Support Note安装一组简化的软件包。

您还可以搜索“Linux简化包”以查找说明。
