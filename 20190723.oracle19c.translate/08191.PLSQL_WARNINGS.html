1.268 PLSQL_WARNINGS

PLSQL_WARNINGS启用或禁用PL / SQL编译器报告警告消息，并指定显示为错误的警告消息。

属性 描述

参数类型

串

句法

PLSQL_WARNINGS = ' value_clause ' [, ' value_clause ' ] ...

句法

value_clause :: =

{ENABLE |禁用|错误}：

{全部

|严重

|信息化

|性能

| { integer

| ( integer [, integer ] ...)

}

}

默认值

'禁用所有'

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

例子

 
PLSQL_WARNINGS ='ENABLE：SEVERE'，'DISABLE：INFORMATIONAL'; PLSQL_WARNINGS ='禁用：全部'; PLSQL_WARNINGS ='DISABLE：5000'，'ENABLE：5001'，'ERROR：5002'; PLSQL_WARNINGS ='ENABLE：（5000,5001,5002）'，'禁用：（6000,6001）';

value_clause

可以指定多个值子句，用引号括起来并用逗号分隔。每个值子句都包含一个限定符，冒号的（ : ）和改性剂。

限定符值：

  • ENABLE

    启用特定警告或一组警告

  • DISABLE

    禁用特定警告或一组警告

  • 错误

    将特定警告或一组警告视为错误

修饰符值：

  • 所有

    将限定符应用于所有警告消息

  • 严重

    仅将限定符应用于SEVERE类别中的那些警告消息

  • 信息化

    仅将限定符应用于INFORMATIONAL类别中的那些警告消息

  • 性能

    仅将限定符应用于PERFORMANCE类别中的那些警告消息
