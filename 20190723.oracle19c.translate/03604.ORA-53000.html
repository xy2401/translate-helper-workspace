95 ORA-54000到ORA-54056

ORA-54000：虚拟列功能尚未实现

原因：功能尚未实现。

行动：正在实施功能。

ORA-54001： string ：为虚拟列表达式指定的无效标识符

原因：列表达式引用了表中不存在的列。

操作：重写列表达式以仅引用表中的标量列。

ORA-54002：只能在虚拟列表达式中指定纯函数

原因：列表达式包含一个评估为非确定性的函数。

操作：重写列表达式以仅引用纯函数。

ORA-54003：虚拟列不支持指定的数据类型

原因：虚拟列仅支持标量数据类型。虚拟列不支持LONG，BLOB，REF和BFILE数据类型。

操作：使用支持的标量数据类型指定表达式列。

ORA-54004：不支持虚拟列的结果数据类型

原因：不支持基础表达式的数据类型。虚拟列仅支持标量数据类型。虚拟列不支持LONG，BLOB，REF和BFILE数据类型。

操作：指定虚拟列的表达式以返回支持的标量数据类型。

ORA-54005：此处无法指定关键字VIRTUAL

原因：重复或错误指定关键字VIRTUAL。

操作：从指定的语法中删除关键字。

ORA-54006：此处无法指定关键字VISIBLE

原因：重复或错误指定关键字VISIBLE。

操作：从指定的语法中删除关键字。

ORA-54007：此处无法指定关键字HIDDEN

原因：关键字HIDDEN重复或未正确指定。

操作：从指定的语法中删除关键字。

ORA-54008：索引组织表不支持表达式列

原因：尝试使用表达式列创建/更改索引组织表

操作：不支持这些列，更改DDL。

ORA-54009：外部表不支持表达式列

原因：尝试使用表达式列创建/更改外部表

操作：不支持这些列，更改DDL。

ORA-54010：临时表不支持表达式列

原因：尝试使用表达式列创建/更改临时表

操作：不支持这些列，更改DDL。

ORA-54011：群集表不支持表达式列

原因：尝试使用表达式列创建/更改聚簇表

操作：不支持这些列，更改DDL。

ORA-54012：列表达式中引用了虚拟列

原因：此虚拟列在另一个虚拟列的表达式中引用

操作：确保任何虚拟列的列表达式定义不引用任何虚拟列

ORA-54013：虚拟列上不允许INSERT操作

原因：尝试将值插入虚拟列

操作：重新发出参数，而不提供虚拟列的值

ORA-54014：CTAS操作的结果表包含虚拟列

原因： CTAS操作创建的表包含虚拟列定义

操作：从正在创建的表中删除虚拟列定义

ORA-54015：指定了重复的列表达式

原因：添加/创建的虚拟列的表达式与现有/先前指定的功能索引表达式或虚拟列表达式冲突

操作：更改虚拟列的表达式，没有重复的表达式

ORA-54016：指定了无效的列表达式

原因：虚拟列表达式不是有效的算术表达式。它可能是指表中的另一列

操作：更改虚拟列的表达式

ORA-54017：虚拟列上不允许更新操作

原因：尝试更新虚拟列的值

操作：重新发出参数，而不设置虚拟列的值

ORA-54018：此表达式存在虚拟列

原因：指定的索引表达式与现有虚拟列匹配“

操作：通过使用匹配的虚拟列替换索引表达式来重新发出语句

ORA-54019：无法更改虚拟列表达式，因为它是分区列

原因：尝试修改也是分区列的虚拟列的表达式。

行动：不支持此功能。

ORA-54020：无法更改虚拟列表达式，因为它是子分区列

原因：尝试修改也是子分区列的虚拟列的表达式。

行动：不支持此功能。

ORA-54021：无法在分区或子分区列中使用PL / SQL表达式

原因：尝试在包含PL / SQL表达式的虚拟列上对表进行分区。

行动：不支持此功能。

ORA-54022：无法更改虚拟列表达式，因为在列上定义了索引

原因：尝试更改已编制索引的虚拟列的表达式。

行动：改变指数无法解决。更改表达式，然后重建索引。

ORA-54023：由于在列上定义了约束，因此无法更改虚拟列表达式

原因：尝试更改已在其上定义约束的虚拟列的表达式。

操作：删除约束，然后更改表达式。

ORA-54024：组织多维数据集表不支持表达式列

原因：尝试使用表达式列创建或更改组织多维数据集表

操作：不支持这些列，更改DDL。

ORA-54025：虚拟列不能具有默认值

原因：尝试将虚拟列更改为具有默认值。

操作：这是无效的，更改DDL。

ORA-54026：实列不能有表达式

原因：尝试更改实列以获得表达式。

操作：这是无效的，更改DDL。

ORA-54027：无法修改虚拟列的数据类型

原因：尝试更改虚拟列的数据类型而不修改基础表达式

操作：更改基础表达式以与数据类型更改兼容

ORA-54028：无法更改虚拟列的HIDDEN / VISIBLE属性

原因：尝试更改虚拟列的HIDDEN / VIRTUAL属性

操作：重新发出DDL而不更改虚拟列属性

ORA-54029：无法在触发器主体中更新虚拟列

原因：尝试更改触发器主体中虚拟列的值“

操作：这是无效的，更改触发器定义。

ORA-54030：虚拟列和表达式之间的数据类型不匹配

原因：创建列后更改了虚拟列表达式“

操作：更改基础表达式以返回符合虚拟列的数据类型

ORA-54031：要删除或修改的列用于虚拟列表达式

原因：尝试删除或修改虚拟列表达式中使用的列。

操作：首先删除虚拟列或更改虚拟列表达式以消除对要删除的列的依赖性。

ORA-54032：要重命名的列用于虚拟列表达式

原因：尝试重命名虚拟列表达式中使用的列。

操作：首先删除虚拟列或更改虚拟列表达式以消除对要重命名的列的依赖性。

ORA-54033：要修改的列用于虚拟列表达式

原因：尝试修改虚拟列表达式中使用的列的数据类型。

操作：首先删除虚拟列或更改虚拟列表达式以消除对要修改的列的依赖性。

ORA-54034：功能索引表达式中不允许使用虚拟列

原因：尝试创建具有在一个或多个虚拟列上定义的表达式的功能索引。

操作：仅使用常规列指定索引表达式。

ORA-54035：此处无法指定关键字HIDDEN

原因：尝试为虚拟列指定HIDDEN关键字

行动：不支持此功能。

ORA-54036：无法使用虚拟列上的ON DELETE SET NULL子句定义参照约束

原因：尝试为虚拟列上的参照完整性约束指定ON DELETE SET NULL子句。

操作：重新发出语句而不指定ON DELETE SET NULL子句。

ORA-54037：表必须至少有一列非虚拟列

原因：尝试创建仅包含虚拟列的表。

操作：在正在创建的表中包含至少一个非虚拟列。

ORA-54038：虚拟列无法加密

原因：尝试创建或更改虚拟列以进行加密。

操作：虚拟列不需要加密，因为数据未存储在磁盘上。

ORA-54039：表必须至少有一个不可见的列

原因：尝试创建或更改表以仅包含不可见的列。

操作：包括至少一个在创建或更改的表中不可见的列。

ORA-54040：无法更改系统生成列的可见性。

原因：尝试更改系统生成的列的可见性。

行动：不支持此功能。

ORA-54041：无法更改抽象数据类型列的可见性。

原因：尝试更改抽象数据类型列的可见性。

操作：使用支持的数据类型指定列可见性更改。

ORA-54042：此类表格不支持不可见列。

原因：尝试在不受支持的表类型上创建或更改列的可见性。只有堆和索引组织表（IOT）才支持不可见列。

操作：不支持这些列，更改DDL。

ORA-54043：包括IOT列不能是不可见的列。

原因：尝试更改索引组织表（IOT）的包含列的可见性。

行动：不支持此功能。

ORA-54044：无法更改具有列级对象权限的列的可见性。

原因：尝试使用列级对象权限更改列的可见性。

操作：在使列不可见之前撤消列级对象权限。

ORA-54045：无法为不可见的列授予列级对象权限。

原因：尝试将列级对象权限授予不可见列。

操作：修改列以使其可见，然后授予列级对象权限。

ORA-54046：列可见性修改不能与任何其他已修改的列DDL选项组合使用。

原因：尝试将列的可见性修改与其他已修改的列属性更改组合在一起。

操作：将列可见性修改分离为独立DDL。

ORA-54047：具有对象关系存储的XMLType表不允许使用虚拟列

原因：尝试使用虚拟列创建对象关系XMLType表。

操作：不要为具有对象关系存储的XMLType表创建虚拟列。此功能不可用。

ORA-54048：列表达式修改不能与任何其他列DDL操作组合使用。

原因：尝试将列表达式的修改与其他列操作组合在一起。

操作：将列表达式修改分离为独立的DDL。

ORA-54049：外部表不支持表达式编辑列

原因：尝试使用版本表达式列创建或更改外部表。

操作：不支持这些列;改变DDL。

ORA-54050：编辑列上不支持的类型或属性。

原因：尝试使用编校策略更改列，以便为该列提供不受支持的类型或属性。数据编校不支持虚拟列。数据编校不支持具有INVISIBLE属性的列（这适用于用户指定的隐藏列和系统生成的隐藏列）。数据编校不支持矢量编码列。

操作：指定非编辑列的名称，或使用DBMS_REDACT包中的DROP_POLICY PL / SQL过程在更改之前从列中删除编校策略。查看REDACTION_POLICIES和REDACTION_COLUMNS目录视图，查看所有编校策略和编辑列的列表。

ORA-54051：系统生成的虚拟列不能是主键。

原因：尝试将系统生成的虚拟列*指定为主键。

行动：没有

ORA-54052：无法重命名系统生成的功能索引虚拟列。

原因：尝试重命名系统生成的功能索引虚拟列。

行动：没有

ORA-54053：无法更改SYS用户拥有的表中列的可见性。

原因：尝试更改SYS用户拥有的表中列的可见性。

操作：在不属于SYS用户的表上执行DDL。

ORA-54054：系统生成的虚拟列或保护列不能处于约束中。

原因：尝试添加包含系统生成的虚拟列或保护列的约束。

行动：没有

ORA-54055：内存中虚拟列表达式不能引用系统生成或隐藏的列。

原因：尝试在系统生成或隐藏列上创建虚拟列表达式。

操作：避免在系统生成或隐藏的列上创建虚拟列表达式。

ORA-54056：对敏感列的操作不受支持

原因：尝试更改敏感列的可见性或将敏感列更改为矢量编码。

操作：指定非敏感列的名称，或使用DBMS_TSDP_MANAGE包中的DROP_SENSITIVE_COLUMN PL / SQL过程在更改列之前删除列的灵敏度。
