Oracle数据泵已知错误

这些是Oracle Database 19c中的Oracle Data Pump错误。

  • 错误27133957
  • 错误27092988

父主题： 打开影响所有平台的错误

错误27133957

如果在数据泵加载期间在统一资源标识符（URI）的文件名组件中使用%U或%L通配符，则会收到以下错误：

ORA-31640：无法打开转储文件ORA-17503：ksfdopn：11无法打开文件ORA-17500：ODM错误：ODM HTTP错误请求

解决方法：

不要使用通配符。而是将每个文件名列为DUMPFILE参数的一部分。

父主题： Oracle Data Pump已知错误

错误27092988

在Oracle Database 19c之前的版本中，Oracle Data Pump始终启用授予运行数据泵的用户的安全角色。从Oracle Database 19c开始，默认情况下，数据泵不再启用这些角色。有一个新参数ENABLE_SECURE_ROLES ，用于控制是否应为数据泵作业启用这些角色。

解决方法：

没有。
