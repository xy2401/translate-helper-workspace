23.1具有R可扩展性的Oracle数据挖掘

了解如何使用Oracle Data Mining构建，评分和查看Oracle Data Mining模型以及R模型。

Oracle数据挖掘框架通过开源R生态系统的算法扩展了数据挖掘算法集。Oracle Data Mining在Oracle数据库内核中实现。挖掘模型是数据库模式对象。通过可扩展性增强，数据挖掘框架可以构建，评分和查看Oracle Data Mining模型和R模型。

注册R脚本

数据库服务器上的R引擎执行R脚本以构建，评分和查看R模型。必须事先由具有rqAdmin角色的特权用户向数据库注册这些R脚本。您必须先安装Oracle R Enterprise才能注册R脚本。

Oracle数据挖掘与R模型的作用

R模型支持以下功能：

  • Oracle Data Mining DBMS_DATA_MINING包已得到增强，可支持R模型。例如， CREATE_MODEL和DROP_MODEL 。

  • MODEL VIEW获取有关单个模型和分区模型的R模型详细信息。

  • Oracle Data Mining SQL功能已得到增强，可与R模型功能一起使用。例如， PREDICTION和CLUSTER_ID 。

R模型可扩展性支持以下数据挖掘功能：

  • 协会

  • 属性重要性

  • 回归

  • 分类

  • 聚类

  • 特征提取

23.3关于算法元数据注册

算法元数据注册允许注册新算法函数及其设置的统一且一致的方法。

用户可以通过注册过程添加新的基于R的算法。新算法在Oracle R Enterprise中和适当的挖掘功能中可用。基于注册元数据，动态呈现设置页面。优点如下：

  • 更轻松地管理基于R的算法

  • 易于为模型构建指定基于R的算法

  • 清理JSON结构中的各个属性

  • 在用户之间共享基于R的算法

算法元数据注册扩展了Oracle Data Mining的挖掘模型功能。
