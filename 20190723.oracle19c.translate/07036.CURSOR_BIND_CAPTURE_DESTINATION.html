1.58 CURSOR_BIND_CAPTURE_DESTINATION

CURSOR_BIND_CAPTURE_DESTINATION确定从SQL游标捕获的绑定变量可用的位置。

属性 描述

参数类型

串

句法

CURSOR_BIND_CAPTURE_DESTINATION = {off |记忆|内存+磁盘}

默认值

内存+硬盘

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

值

  • 离

    不从SQL游标捕获绑定变量。

  • 记忆

    绑定变量从SQL游标捕获，仅在内存中可用（ V$视图）。

  • 内存+硬盘

    绑定变量从SQL游标捕获，并且可在内存（ V$视图）和磁盘（自动工作负载存储库表，SQL调整集表等）中使用。

    使用SQL ALTER SESSION或ALTER SYSTEM语句指定此值时，请将值括在单引号或双引号中。例如：

    ALTER SESSION SET CURSOR_BIND_CAPTURE_DESTINATION ='memory + disk'; ALTER SYSTEM SET CURSOR_BIND_CAPTURE_DESTINATION =“memory + disk”;
