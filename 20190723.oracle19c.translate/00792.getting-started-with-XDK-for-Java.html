11 Oracle XML Developer's Kit for Java入门

介绍了如何开始使用XDK for Java。

安装XDK for Java组件

Oracle数据库包含XDK for Java组件。本章假定您已将XDK与Oracle数据库一起安装，并从Oracle数据库示例介质中安装了演示程序。

警告：

使用Oracle XML Developer Kit（XDK）的组件构建软件程序可以实现一些功能强大但具有潜在危险的功能，例如外部实体扩展和递归扩展。有关如何安全使用XDK的信息，请参阅Oracle XML Developer's Kit的安全注意事项 。

有关XDK目录结构的说明，请参阅关于安装XDK 。

例11-1列出了Java主目录下的Java主目录（这是UNIX目录结构。）在示例之后，子目录的内容将单独列出。

bin目录包含以下组件：

orajaxb orapipe oraxml oraxsl transx

lib目录包含以下JAR和ZIP文件：

classgen.jar jdev-rt.zip oraclexsql.jar transx.zip xml.jar xml2.jar xmldemo.jar xmlmesg.jar xmlparserv2.jar xschema.jar xsqlserializers.jar xsu12.jar

设置XDK for Java环境

您可以使用环境变量或命令行选项设置XDK for Java环境。

要设置XDK for Java环境，请执行以下任一操作：

  • 在Oracle数据库安装XDK期间，手动设置$CLASSPATH （UNIX）或%CLASSPATH% （Windows）环境变量。

  • 在命令行编译和运行Java程序时，请设置-classpath选项。
