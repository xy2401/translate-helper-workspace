11.1关于CUR矩阵分解

CUR矩阵分解是低秩矩阵分解算法，其在少量实际列和/或数据矩阵的实际行中明确表示。

CUR矩阵分解是作为奇异值分解 （SVD）和主成分分析（PCA）的替代方法而开发的。CUR矩阵分解选择具有高统计杠杆或来自数据矩阵的大影响的列和行。通过实现CUR矩阵分解算法，可以从原始数据矩阵中识别少量最重要的属性和/或行。因此， CUR矩阵分解是探索性数据分析的重要工具。CUR矩阵分解可以应用于各种领域，并有助于回归 ， 分类和聚类 。

11.4列（属性）选择和行选择

列（属性）选择和行选择是CUR矩阵分解的最后阶段。

属性选择：选择具有高杠杆分数的属性，并报告其名称，分数（按重要性）和等级（按重要性）。

行选择：选择具有高杠杆分数的行并报告其名称，分数（作为重要性）和排名（按重要性）。

  1. CUR矩阵分解首先选择第n列（或属性）A与概率第j p j = min {1, cπ j }对于所有j ε {1,..., n }

  2. 如果用户启用行选择时，选择第i行A的概率为pˊ i = min {1, rπˊ i }对于所有i ε {1,..., m }

  3. 报告所有选定属性的名称（或ID）和杠杆得分（作为重要性）（如果禁用行重要性）或所有选定的属性和行（如果启用了行重要性）。

c是用户想要选择的近似（或预期）列数， r是用户想要选择的近似（或预期）行数。

要实现列和行选择，您需要计算选择每个列和行的概率。

计算每列的概率如下：

计算每行的概率如下：

pˊ i = min{1, c πˊ i } 。

如果概率大于某个阈值，则选择列或行。
