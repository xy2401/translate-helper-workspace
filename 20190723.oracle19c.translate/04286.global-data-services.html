9.5全球数据服务

本节介绍可与通用连接池一起使用的新全局数据服务（GDS）功能：

  • 全球数据服务概述

  • 配置使用GDS的应用程序

9.5.1全球数据服务概述

自Oracle Database 12 c第1版（12.1）以来，全球数据服务（GDS）功能可用。通过此功能，仅在Oracle RAC中可用的快速连接故障转移，运行时连接负载平衡和连接关联功能已扩展为一组提供公共服务的复制数据库。

该组数据库可能包括通过Data Guard，GoldenGate或任何其他复制技术互连的Oracle RAC和单实例Oracle数据库。可以由多个数据库提供的数据库服务称为全局服务，因此可以将其与仅可由单个数据库提供的传统服务区分开来。此组合使服务可以部署在此全局分布式配置中的任何位置，从而支持负载平衡，高可用性，数据库关联性等。

9.5.2配置使用GDS的应用程序

UCP连接到Global Data Services的方式与连接到Oracle RAC上的本地服务的方式相同。连接字符串中的服务名称应该是全局服务的名称。端点应该是GDS侦听器的端点，而不是数据库的本地，远程或SCAN侦听器的端点。

客户端必须在连接字符串的REGION参数中指定其区域。这是GDS的新要求。区域名称是必需的，因为在GDS的情况下，会针对特定区域自定义运行时负载平衡建议。以下是典型连接字符串的示例：

（DESCRIPTION =（ADDRESS =（GDS_protocol_address_information））（CONNECT_DATA =（SERVICE_NAME = global_service_name）（REGION = region_name）））

与本地服务一样，UCP可以在同一连接字符串中指定多个GDS侦听器，以用于侦听器故障转移，负载平衡或两者。

注意：

GDS侦听器不支持SCAN，因此必须指定每个侦听器的端点。

（DESCRIPTION =（ADDRESS_LIST =（LOAD_BALANCE = ON）（FAILOVER = ON）（ADDRESS =（GDS_protocol_address_information））（ADDRESS =（GDS_protocol_address_information）））（CONNECT_DATA =（SERVICE_NAME = global_service_name）（REGION = region_name）））

如果在客户端连接字符串中仅指定了本地区域的全局服务管理器，则REGION参数是可选的。当GDS配置中只有一个区域时，或者当存在多个区域时，可能就是这种情况。但是，更改设计为使用单个数据库的现有客户端的连接字符串是不可行的。如果未指定REGION参数，则假定客户端的区域是用于连接到全局服务的全局服务管理器的区域。

注意：

除非在连接字符串中指定了REGION参数，否则只能使用具有单个区域的GDS配置的12c之前的瘦JDBC客户端。

前面示例中的所有GDS侦听器都属于运行UCP的同一区域，即本地区域。要提供高可用性，当本地区域中的所有GDS都不可用时，您可以在其他ADDRESS_LIST描述符中为伙伴区域指定GDS侦听器。 

（DESCRIPTION =（FAILOVER = on）（ADDRESS_LIST =（LOAD_BALANCE = ON）（ADDRESS =（global_protocol_address_information））（ADDRESS =（global_protocol_address_information）））（ADDRESS_LIST =（LOAD_BALANCE = ON）（ADDRESS =（global_protocol_address_information））（ADDRESS =（ global_protocol_address_information）））（CONNECT_DATA =（SERVICE_NAME = global_service_name）（REGION = region_name）））

您不需要手动ONS配置，因为UCP会自动从GDS检索为UCP区域最佳定制的ONS连接信息。

注意：

  • 要为GDS启用自动ONS配置，必须在UCP上启用快速连接故障转移（FCF）。

  • 自动ONS配置仅适用于Oracle GDS和Oracle RAC。它不适用于单实例Oracle数据库。

    自动ONS配置不支持ONS钱包或密钥库参数。如果您的应用程序需要任何这些参数，则必须以下列两种方式之一显式配置ONS：

    • 调用PoolDataSource.setONSConfiguration(String)方法

    • 在本地ONS配置文件中添加ONS钱包或密钥库参数
