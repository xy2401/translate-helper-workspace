标准SQL的Oracle扩展

Oracle支持许多超出标准SQL的功能。如果您关心应用程序对SQL的其他实现的可移植性，那么使用Oracle的FIPS Flagger来帮助识别嵌入式SQL程序中对SQL-92的Oracle扩展的使用。FIPS Flagger是Oracle预编译器和SQL *模块编译器的一部分。也可以使用ALTER SESSION SET FLAGGER = ENTRY在SQL * Plus中启用FIPS FLAGGER 。虽然SQL-92已被SQL：2016取代，但自SQL-92以来，对于任何版本的SQL都没有一致性测试权限。因此，Entry SQL-92为您提供最可靠的可移植性。
