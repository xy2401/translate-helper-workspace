关于Oracle Base Config

在只读ORACLE_HOME和读/写ORACLE_HOME中，配置文件都位于称为ORACLE_BASE_CONFIG的位置。

在读/写ORACLE_HOME中，ORACLE_BASE_CONFIG路径与ORACLE_HOME路径相同，因为它位于$ ORACLE_HOME。但是，在只读ORACLE_HOME中，ORACLE_BASE_CONFIG路径与ORACLE_BASE相同。

ORACLE_BASE_CONFIG/dbs包含ORACLE_HOME的配置文件。 dbs目录中的每个文件都包含$ ORACLE_SID，因此该目录可以由许多不同的ORACLE_SID共享。

要打印ORACLE_BASE_CONFIG路径，请从$ORACLE_HOME/bin目录运行orabaseconfig命令：
$ setenv ORACLE_HOME /u01/app/oracle/product/19.0.0/dbhome_1 $ cd $ ORACLE_HOME / bin $ ./orabaseconfig

例如：

$ ./orabaseconfig $ / u01 / app / oracle

其中， /u01/app/oracle是ORACLE_BASE。
