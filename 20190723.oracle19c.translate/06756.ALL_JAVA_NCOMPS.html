2.190 ALL_JAVA_NCOMPS

ALL_JAVA_NCOMPS显示有关当前用户可访问的Java类的ncomp相关信息。

相关观点

  • DBA_JAVA_NCOMPS显示有关数据库中所有Java类的ncomp相关信息。

  • USER_JAVA_NCOMPS显示有关当前用户拥有的Java类的ncomp相关信息。此视图不显示OWNER列。
