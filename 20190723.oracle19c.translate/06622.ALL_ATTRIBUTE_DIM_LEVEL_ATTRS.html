2.54 ALL_ATTRIBUTE_DIM_LEVEL_ATTRS

ALL_ATTRIBUTE_DIM_LEVEL_ATTRS描述当前用户可访问的属性维度级别的属性。

相关观点

  • DBA_ATTRIBUTE_DIM_LEVEL_ATTRS描述数据库中的所有属性维级别属性。

  • USER_ATTRIBUTE_DIM_LEVEL_ATTRS描述当前用户架构中属性维度的级别属性。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

属性维度的所有者

DIMENSION_NAME

VARCHAR2（128）

属性维度的名称

LEVEL_NAME

VARCHAR2（128）

属性维度级别的名称

ATTRIBUTE_NAME

VARCHAR2（128）

由级别确定的属性的名称

角色

VARCHAR2（4）

由级别确定的属性的角色

IS_MINIMAL_DTM

VARCHAR2（1）

指示属性是否最小确定（ Y ）或不（ N ）

ORDER_NUM

数

由级别确定的属性列表中的属性顺序

ORIGIN_CON_ID

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
