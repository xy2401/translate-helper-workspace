14.18 DROP PROCEDURE声明

DROP PROCEDURE语句从数据库中删除独立过程。

注意：

请勿使用此语句删除属于包的过程。相反，要么使用“ DROP PACKAGE语句 ”删除整个包，要么使用带有OR REPLACE子句的“ CREATE PACKAGE语句 ”重新定义包而不使用过程。

先决条件

该过程必须位于您的架构中，或者您必须具有DROP ANY PROCEDURE系统特权。

句法

drop_procedure :: =


插图说明drop_procedure.eps

语义

模式

包含该过程的模式的名称。默认值：您的架构。

程序

要删除的过程的名称。

删除过程时，数据库会使依赖于已删除过程的所有本地对象失效。如果随后引用其中一个对象，则数据库会尝试重新编译该对象，如果尚未重新创建已删除的过程，则会返回错误消息。

例

例14-35删除程序

此语句删除用户hr拥有的过程remove_emp ，并使依赖于remove_emp所有对象无效：

DROP PROCEDURE hr.remove_emp;
