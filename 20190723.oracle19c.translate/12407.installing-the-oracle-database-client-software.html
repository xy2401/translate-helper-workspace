安装Oracle数据库客户端软件

这些主题说明如何运行安装向导以执行大多数数据库客户端安装。

  • 运行安装向导以安装Oracle数据库客户端
    使用runInstaller命令启动Oracle Database Client安装。
  • 使用映像文件安装Oracle数据库客户端
    解压缩Oracle Database Client映像文件并使用runInstaller命令启动Oracle Database Client安装。
  • 使用Oracle Net Configuration Assistant
    完成Oracle数据库客户端安装后，以独立模式运行Oracle Net Configuration Assistant，以配置侦听器，命名方法，网络服务名称和目录服务器使用情况。
