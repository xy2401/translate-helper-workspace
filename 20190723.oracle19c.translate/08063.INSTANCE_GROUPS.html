1.149 INSTANCE_GROUPS

与PARALLEL_INSTANCE_GROUP参数一起使用， INSTANCE_GROUPS允许您将并行查询操作限制为有限数量的实例。

属性 描述

参数类型

串

句法

INSTANCE_GROUPS = group_name [, group_name ] ...

默认值

没有默认值。

可修改

没有

可在PDB中修改

没有

价值范围

一个或多个实例组名称，以逗号分隔

基本

没有

Oracle RAC

多个实例可以具有不同的值。

注意：

不推荐使用INSTANCE_GROUPS参数。它仅为了向后兼容而保留。

INSTANCE_GROUPS是Oracle RAC参数，您只能在并行模式下指定。

此参数指定一个或多个实例组，并将当前实例分配给这些组。如果还在PARALLEL_INSTANCE_GROUP参数中指定了其中一个指定的组，则Oracle会为此实例的并行操作分配查询过程。
