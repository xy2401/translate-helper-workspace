1.71 DB_BLOCK_SIZE

DB_BLOCK_SIZE指定（以字节为单位）Oracle数据库块的大小。

属性 描述

参数类型

整数

默认值

8192

可修改

没有

可在PDB中修改

没有

价值范围

2048到32768 ，但您的操作系统可能有更窄的范围

基本

是

Oracle RAC

您必须为每个实例设置此参数，并且多个实例必须具有相同的值。

注意：

在创建数据库时设置此参数。之后不要改变它。

DB_BLOCK_SIZE典型值为4096和8192 。此参数的值必须是设备级别的物理块大小的倍数。

在创建数据库时生效的DB_BLOCK_SIZE的值确定块的大小。该值必须保持设置为其初始值。

对于Oracle Real Application Clusters（Oracle RAC），此参数会影响表和索引的FREELISTS存储参数的最大值。Oracle为每个空闲列表组使用一个数据库块。决策支持系统（DSS）和数据仓库数据库环境倾向于受益于更大的块大小值。

注意：

64位操作系统支持的最大DB_BLOCK_SIZE值为32768
