3.65 ALL_SCHEDULER_WINDOW_GROUPS

ALL_SCHEDULER_WINDOW_GROUPS显示有关当前用户可访问的调度程序窗口组的信息。

相关视图

DBA_SCHEDULER_WINDOW_GROUPS显示有关数据库中所有Scheduler窗口组的信息。

柱 数据类型 空值 描述

WINDOW_GROUP_NAME

VARCHAR2（128）

不是NULL

窗口组的名称

启用

VARCHAR2（5）

指示窗口组是启用（ TRUE ）还是禁用（ FALSE ）

NUMBER_OF_WINDOWS

数

窗口组中的成员数

NEXT_START_DATE

VARCHAR2（64）

如果禁用了一个窗口组，则该列将为NULL。否则，它将从组中已启用的窗口设置为最早的NEXT_START_DATE 。

评论

VARCHAR2（240）

有关窗口组的可选注释
