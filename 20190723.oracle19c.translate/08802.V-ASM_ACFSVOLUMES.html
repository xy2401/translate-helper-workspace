7.52 V $ ASM_ACFSVOLUMES

V$ASM_ACFSVOLUMES显示有关已挂载的Oracle ACFS卷的信息，与V$ASM_FILESYSTEM相关。

柱 数据类型 描述

FS_NAME

VARCHAR2（1024）

文件系统挂载点（ V$ASM_FILESYSTEM外键）

VOL_DEVICE

VARCHAR2（256）

Oracle ADVM设备的名称

VOL_LABEL

VARCHAR2（64）

通过mkfs / acfsformat分配的卷标（可选名称）;如果不存在名称，则为NULL

PRIMARY_VOL

VARCHAR2（5）

指示卷是否是文件系统的主卷（ TRUE ）（ FALSE ）

TOTAL_MB

数

卷设备的大小（以兆字节为单位）

FREE_MB

数

卷设备上的可用空间（以兆字节为单位）

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。

注意：

要显示有关Oracle ACFS文件系统或位于Oracle Flex ASM配置中的节点上的卷的信息，必须连接到Oracle ASM代理实例而不是本地Oracle ASM实例。有关Oracle Flex ASM的信息，请参阅“ Oracle自动存储管理管理员指南” 。
