7.3文本特征的数据准备

文本特征的模型详细信息视图是DM$VX model_name 。

如果存在文本属性，则文本特征视图DM$VX model_name描述提取的文本特征。该视图具有以下架构：

名称类型-------------- --------------------- PARTITION_NAME VARCHAR2（128）COLUMN_NAME VARCHAR2（128）TOKEN VARCHAR2（ 4000）DOCUMENT_FREQUENCY NUMBER

表7-1提取的文本功能的文本功能视图

列名称 描述

PARTITION_NAME

分区模型中的分区以检索详细信息

COLUMN_NAME

标识符列的名称

TOKEN

文本标记通常是单词或词干

DOCUMENT_FREQUENCY

衡量整个训练集中令牌频率的方法
