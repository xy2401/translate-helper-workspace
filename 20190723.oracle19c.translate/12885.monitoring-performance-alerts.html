6监控性能警报

Oracle数据库包含一个内置的警报基础结构，可以通知您数据库即将出现的问题。警报是一种事件，事件包含事件。问题是一种特殊类型的事件。

默认情况下，Oracle数据库会启用一些警报，包括以下警报 ：

  • 存档区域使用

  • 当前公开游标数量

  • 恢复区自由空间

  • 使用的表空间空间

除了这些默认警报之外，您还可以设置度量标准阈值，以便Oracle数据库生成警报，以通知您有关指定的数据库性能条件。

本章包含以下部分：

  • 设置性能警报的度量标准阈值

  • 响应警报

  • 清除警报

也可以看看：

  • “Oracle数据库管理员指南” ，提供有关事件以及如何监视和管理事件的信息

6.1设置性能警报的度量标准阈值

一个 metric是累积统计量的变化率。此费率可以根据各种单位进行衡量，包括时间，交易或数据库调用。例如，每秒数据库调用的数量是一个度量。您可以在度量标准上设置阈值，以便在传递阈值时生成警报。

性能提醒基于与性能相关的指标。这些警报依赖于环境或依赖于应用程序。

与环境相关的性能警报可能与所有系统无关。例如， AVERAGE_FILE_READ_TIME指标在读取文件的平均时间超过指标阈值时生成警报。此警报在仅具有一个磁盘的系统上可能很有用。但是，在具有多个磁盘的系统上，警报可能不相关，因为I / O处理分布在整个子系统中。

依赖于应用程序的性能警报通常与所有系统相关。例如， BLOCKED_USERS度量标准会在特定会话阻止的用户数超过度量标准阈值时生成性能警报。无论环境如何配置，此警报都是相关的。

要从性能警报中获取最相关的信息，请将性能指标的阈值设置为表示系统所需边界的值。然后，您可以随着时间的推移微调这些值，直到您的系统达到或超过您的性能目标。

要为性能指标设置阈值：

  1. 访问数据库主页。

    有关详细信息，请参阅“ 访问数据库主页 ” 。

  2. 从Oracle Database菜单中，选择Monitoring ，然后选择Metric和Collection Settings 。

    如果出现“数据库登录”页面，则以具有管理员权限的用户身份登录。将显示“度量标准设置”页面。

  3. 对于与系统相关的每个性能指标，请单击“ 编辑”图标。

    将出现“编辑高级设置”页面。

  4. 按照向导的步骤设置阈值。

也可以看看：

  • “ 为基线设置度量标准阈值 ”

6.2响应警报

当Oracle数据库生成警报时，它将显示在“数据库主页”的“事件和问题”部分中。

图6-1显示了数据库主页的“事件和问题”部分。该部分位于页面默认布局中的“SQL监视器 - 最后一小时”部分下方。

Oracle Enterprise Manager云控制（云控制）使您可以配置通过电子邮件，寻呼机或文本消息发送的警报。

要回复提醒：

  1. 访问数据库主页。

    有关详细信息，请参阅“ 访问数据库主页 ” 。

  2. 在“事件和问题”部分的表中，找到要调查的警报，然后单击“摘要”列中的链接。

    如果出现“数据库登录”页面，则以具有管理员权限的用户身份登录。将显示“事件管理器”页面，其中包含有关警报的更多信例如，单击图6-1中“摘要”列第一行中的值将导致出现图6-2中所示的“事件管理器问题详细信息”页面。选择“常规”子页面。

    图6-2事件管理器问题详细信息页面


    “图6-2事件管理器问题详细信息页面”的说明

  3. 请执行以下一项或多项操作：

    • 单击其他选项卡以查看子页面的信息。

    • 在“常规”子页面上的“跟踪”部分中执行一个或多个操作。

    • 在“引导分辨率”部分中，单击“ 支持工作台：问题详细信息”查看诊断信息。要将诊断数据打包并上载到Oracle Support，请单击Support Workbench：Package Diagnostic 。

    • 运行自动数据库诊断监视器（ADDM）或其他顾问程序以获取有关系统或对象行为的更详细诊断。

6.3清除警报

当问题原因消失时，大多数警报（例如CPU利用率警报）会自动清除。但是，必须确认其他警报，例如通用警报日志错误或通用事件警报。

采取必要的纠正措施后，您可以清除警报。

要清除警报：

  1. 访问数据库主页。

    有关详细信息，请参阅“ 访问数据库主页 ” 。

  2. 在“事件和问题”部分中，单击表的“摘要”列中的链接。有关该部分的屏幕截图，请参见图6-1 。

    出现事件管理器问题详细信息页面如果可以手动清除事件或问题，则“跟踪”部分中会显示“ 清除”按钮。

    如果未显示“清除”按钮会发生什么。问题是如何解决的？

  3. 在“跟踪”部分中，单击“ 清除” 。
