13.29 StatelessConnectionPool类

此类表示与数据库的无状态，经过身份验证的连接池。

表13-41 StatelessConnectionPool类使用的枚举值

属性 选项 
池类型

  • HETEROGENEOUS是默认状态;可以在同一个池中创建具有不同身份验证上下文的连接。此池类型还支持外部身份验证。

  • HOMOGENEOUS表示池中的所有连接都使用池创建期间提供的用户名和密码进行身份验证。无法创建代理连接。 minConn和incrConn值仅在这些考虑HOMOGENEOUS池。

  • NO_RLB关闭连接池中的运行时负载平衡。可以与HETEROGENEOUS和HOMOGENEOUS池一起使用。

  • USES_EXT_AUTH指示池中的连接支持外部身份验证。只能与HETEROGENEOUS池一起使用。 

BusyOption

  • WAIT表示线程等待并阻塞，直到连接空闲为止。

  • NOWAIT引发错误。

  • FORCEGET表示即使打开了最大连接数且所有连接都忙，也会创建新连接。 

DestroyMode

  • DEFAULT表示如果池中仍然处于活动状态， ORA24422引发ORA24422错误

  • SPD_FORCE表示池中的任何繁忙连接都被强制终止并且池被销毁;用户丢失与强制终止的连接数相对应的内存。

13.29.1 getAnyTaggedConnection（）

返回指向连接对象的指针，不受匹配标记的限制。

此方法适用于启用了数据库驻留连接池的环境。

在执行此调用期间，首先根据提供的标记搜索池。如果存在具有指定标记的连接，则将其返回给用户。如果匹配的连接不可用，则返回经过适当身份验证的无标记连接（带有NULL标记）。如果未注明日期的连接不是免费的，则返回具有不同标签的经过适当认证的连接。

对Connection getTag()调用验证是否已收到连接标记。

句法 描述 
Connection * getAnyTaggedConnection（string＆tag =“”）= 0;

从齐型无状态连接池返回指向连接对象的指针，不受匹配标记的限制; string支持。 

Connection * getAnyTaggedConnection（const UString＆tag）= 0;

从齐型无状态连接池返回指向连接对象的指针，不受匹配标记的限制; UString支持。 

Connection * getAnyTaggedConnection（const string＆userNname，const string＆password，const string＆tag =“”）= 0;

从异构无状态连接池返回指向连接对象的指针，不受匹配标记的限制; string支持。 

Connection * getAnyTaggedConnection（const UString＆userName，const UString＆Password，const UString＆tag）= 0;

从异构无状态连接池返回指向连接对象的指针，不受匹配标记的限制; UString支持。

参数 描述 
用户名

数据库用户名

 
密码

数据库密码。

 
标签

用户定义的连接类型。如果请求默认连接，则可以忽略此参数。

13.29.17 setTimeOut （）

设置连接池中连接的超时时间。OCCI终止与此连接池相关的任何已空闲超过指定超时时间的连接。

如果未设置此属性，则在需要池空间时，最近最少使用的连接将超时。Oracle仅在将连接释放回池时检查超时连接。

句法

void setTimeOut（unsigned int connTimeOut = 0）= 0;
参数 描述 
connTimeOut

超时时间，以秒为单位。

13.29.18 setStmtCacheSize（）

启用或禁用语句缓存。非零值启用语句高速缓存，具有指定大小的高速缓存。零值禁用缓存。

如果用户更改了各个连接的高速缓存大小，并随后使用标记将连接返回到池，则高速缓存大小不会恢复为为池设置的高速缓存大小。如果连接未标记，则重置高速缓存大小以等于为池指定的高速缓存大小。

句法

void setStmtCacheSize（unsigned int cacheSize）= 0;
参数 描述 
CACHESIZE

语句缓存的大小
