CARDINALITY

句法


插图说明cardinality.eps

目的

CARDINALITY返回嵌套表中的元素数。返回类型为NUMBER 。如果嵌套表为空，或者是空集合，则CARDINALITY返回NULL 。

例子

以下示例显示示例表pm.print_media的嵌套表列ad_textdocs_ntab中的元素数：

SELECT product_id，CARDINALITY（ad_textdocs_ntab）基数FROM print_media ORDER BY product_id; PRODUCT_ID CARDINALITY ---------- ----------- 2056 3 2268 3 3060 3 3106 3
