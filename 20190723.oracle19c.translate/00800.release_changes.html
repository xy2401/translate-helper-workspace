本发行版中针对Oracle XML Developer的Kit程序员指南的更改

本前言列出了Oracle XML Developer的Kit程序员指南中的更改。

Oracle XML Developer's Kit Release 18c，Version 1中的更改

不推荐的功能

对于Oracle Database Release 18c，版本1，不推荐使用这些功能，并且可能在将来的版本中不支持这些功能。

DBMS_XMLQUERY PL / SQL包

不推荐使用PL / SQL包DBMS_XMLQUERY 。请改用DBMS_XMLGEN包。

DBMS_XMLSAVE PL / SQL包

不推荐使用PL / SQL包DBMS_XMLSAVE 。请改用DBMS_XMLSTORE包。

Oracle XML Developer Kit 12c第2版（12.2.0.1）中的更改

新功能

以下功能是此版本中的新增功能。

XSL 2.0支持

XDK现在提供对W3C XSL 2.0标准的完全支持。

XQuery 3.0支持

XDK现在为即将推出的XQuery 3.0推荐提供有限的支持。

Oracle XML Developer Kit 12c第1版（12.1.0.1）中的更改

新功能

对于Oracle Database 12c第1版（12.1.0.1）， Oracle XML Developer's Kit程序员指南记录了XDK的这些新功能：

适用于Java的Oracle XQuery Processor

XDK包含一个用于Java的XQuery 1.0处理器，它允许Java应用程序直接在Java虚拟机（JVM）中查询，转换和更新可扩展标记语言（XML）。

有关更多信息，请参阅使用XQuery Processor for Java 。

XDK / J支持Fast Infoset

XDK增加了对Fast Infoset到XDK / J模型的支持。这种支持使开发人员能够在使用Java中的XML内容时使用Fast Infoset技术。Fast Infoset模型是一种在Java中使用XML内容的流行技术。

有关更多信息，请参阅快速信息集的DOM支持 。

XDK / J DOM改进

XDK增加了对万维网联盟（W3C）文档对象模型（DOM）3级核心应用程序编程接口（API）的支持。这种支持使开发人员可以最大限度地利用最新的W3C API进行XML处理，包括定义为DOM Level 3.0 Core规范一部分的API。

有关更多信息，请参阅XDK Java DOM改进 。

XMLDiff支持XDK Java

XDK支持基于Java的XMLDiff，它与C和PL / SQL XMLDiff格式兼容，这是Oracle Database 11g第1版（11.1）中引入的。此支持允许中间层中的纯Java程序与C程序或使用Oracle数据库执行XMLDiff操作的程序交换XMLDiff输出。

有关更多信息，请参阅使用Java确定XML差异 。

不推荐的功能

对于Oracle Database 12c第1版（12.1.0.1），不推荐使用这些功能，并且可能在将来的发行版中不支持这些功能。

XML Developer's Kit JavaBeans

对于Oracle Database 12c第1版（12.1.0.1），不推荐使用这些XDK JavaBeans：

  • DOMBuilder

  • XSLTransformer

  • DBACCESS

  • XMLDBAccess

  • 的xmldiff

  • XMLCompress

  • XSDValidator

有关替代方案，请参阅Oracle XML Developer的Kit JavaBeans（已弃用） 。

XML Developer的Java API工具包

从Oracle Database 12c第1版（12.1.0.1）开始，不推荐使用与已弃用的XDK JavaBeans对应的XDK Java API包和类。

这些是已弃用的包和类：

  • oracle.xml.async

  • oracle.xml.dbaccess

  • oracle.xml。的xmldiff。的BeanInfo

  • oracle.xml.xmlcompl

  • oracle.xml.xmldbaccess

  • oracle.xml.schemavalidator

有关替代方案，请参阅Oracle Database XML Java API Reference 。
