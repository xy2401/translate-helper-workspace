适用于Linux的不受支持的产品

本主题描述Oracle Database 19c不可用的产品或功能。

除了此Oracle Database 19c发行版中的不可用产品或功能列表外，Linux不支持以下产品：

  • IPv6网络支持

    IBM不支持基于IPv6的IP地址来配置集群的专用网络：Linux on System z。目前正在测试中，测试成功完成后将公布支持。

产品支持

本主题描述Oracle Database 19c支持的产品或功能。

支持的产品或功能包括：

  • 数据库智能闪存缓存支持

    Oracle Linux支持Database Smart Flash Cache。

  • Oracle ACFS和Oracle ADVM支持

    虽然Oracle ADVM支持Oracle自动存储管理磁盘组中的原始磁盘，但不支持通过raw创建的Oracle ADVM设备专用文件; Oracle ADVM仅支持块设备专用文件。

    有关支持的平台和发行版的最新信息，请参阅My Oracle Support上的说明1369107.1， 网址为https://support.oracle.com

  • Oracle ASM过滤器驱动程序支持

    仅在Linux x86-64上支持Oracle自动存储管理筛选器驱动程序（Oracle ASMFD）。

  • 基于共享的实例锁定

    Oracle Linux支持基于共享的实例囚笼。

错误26708302

在共享网络连接存储（NAS）上卸载Oracle Real Application Clusters（Oracle RAC）主目录无法删除目录$ORACLE_HOME/deinstall并返回以下错误：

无法删除目录$ ORACLE_HOME / deinstall 。用户无权删除或正在使用中。

解决方法：

手动删除$ORACLE_HOME/deinstall文件夹作为Oracle RAC所有者或root用户。
