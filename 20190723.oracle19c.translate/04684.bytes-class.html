13.8.3 getBytes（）

将Bytes对象中的Bytes复制到指定的字节数组中。

句法

void getBytes（unsigned char * dst，unsigned int count，unsigned int srcBegin = 0，unsigned int dstBegin = 0）const;
参数 描述 
DST

要写入Bytes对象的数据的目标缓冲区。 

计数

要复制的字节数。

 
srcBegin处

从Bytes对象读取数据的起始位置; Bytes对象中第一个字节的位置为0。 

dstBegin

要在目标缓冲区中写入数据的起始位置; dst中第一个字节的位置为0。
