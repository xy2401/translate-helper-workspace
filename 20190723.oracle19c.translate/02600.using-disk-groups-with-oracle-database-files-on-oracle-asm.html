在Oracle ASM上将磁盘组与Oracle数据库文件一起使用

查看此信息以配置Oracle Clusterware和Oracle数据库文件的Oracle自动存储管理（Oracle ASM）存储。

  • 在Oracle ASM上识别和使用现有Oracle数据库磁盘组
    识别现有磁盘组并确定它们包含的可用磁盘空间。（可选）识别Oracle ASM磁盘组设备的故障组。
  • 为Oracle数据库数据文件创建磁盘组
    如果您确定系统上不存在合适的磁盘组，请安装或识别要添加到新磁盘组的相应磁盘设备。
  • 创建Oracle数据库文件的目录
    您可以将Oracle数据库和恢复文件存储在与配置文件不同的文件系统中。
