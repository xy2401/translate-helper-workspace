4.311 DBA_HIST_CR_BLOCK_SERVER

DBA_HIST_CR_BLOCK_SERVER显示缓存融合中使用的全局缓存服务进程（lMS）的历史统计信息。

该视图包含V$CR_BLOCK_SERVER快照。

柱 数据类型 空值 描述

SNAP_ID

数

不是NULL

唯一快照ID

DBID

数

不是NULL

快照的数据库ID

INSTANCE_NUMBER

数

不是NULL

快照的实例编号

CR_REQUESTS

数

由于远程CR块请求而服务的CR块数

CURRENT_REQUESTS

数

由于远程CR块请求而服务的当前块数

CR_REQUESTS + CURRENT_REQUESTS =服务的全局高速缓存CR块（来自V$SYSSTAT ）。

DATA_REQUESTS

数

数据块的当前或CR请求数

UNDO_REQUESTS

数

撤消块的CR请求数

TX_REQUESTS

数

撤消段头块的CR请求数

DATA_REQUESTS + UNDO_REQUESTS + TX_REQUESTS = LMS进程处理的请求总数

CURRENT_RESULTS

数

返回到请求实例的块中没有更改的请求数

PRIVATE_RESULTS

数

返回到请求实例的块的更改请求的数量，并且只有请求的事务可以使用生成的CR块

ZERO_RESULTS

数

返回到请求实例的块的更改请求数。只有零XID事务才能使用该块。

DISK_READ_RESULTS

数

请求实例必须从磁盘读取请求的块的请求数

FAIL_RESULTS

数

失败的请求数;请求交易必须重新发出请求

FAIRNESS_DOWN_CONVERTS

数

接收请求的实例已对块上的X锁进行下转换的次数，因为它没有修改块

刷新

数

LMS进程刷新日志的次数

刷新

数

LMS进程刷新日志的次数

BUILDS

数

服务器必须为其构建CR块的请求数

LIGHT_WORKS

数

引起轻工规则的次数。此规则可防止LMS进程在响应数据，撤消或撤消段头块的CR请求时进入磁盘。此规则可以阻止LMS进程完成对CR请求的响应。

错误

数

LMS进程发出错误信号的次数

CON_DBID

数

采样会话的PDB的数据库ID

CON_ID

数

CON_DBID标识的容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

也可以看看：

“ V $ CR_BLOCK_SERVER ”
