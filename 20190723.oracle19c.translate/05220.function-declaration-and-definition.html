13.36功能声明和定义

在调用函数之前，必须声明并定义它。您可以先声明它（使用function_declaration ），然后在同一个块，子程序或包中（使用function_definition ）定义它，或者同时声明和定义它（使用function_definition ）。

函数是返回值的子程序。值的数据类型是函数的数据类型。函数调用（或调用）是一个表达式，其数据类型是函数的数据类型。

函数声明也称为函数规范或函数规范 。

注意：

本主题适用于嵌套函数。

有关独立函数的信息，请参阅“ CREATE FUNCTION语句 ” 。

有关包函数的信息，请参阅“ CREATE PACKAGE语句 ” 。

语义

function_declaration

声明一个函数，但不定义它。该定义必须稍后出现在与声明相同的块，子程序或包中。

function_heading

函数标题指定函数名称及其参数列表。

FUNCTION_NAME

您声明或定义的函数的名称。

RETURN 数据类型

指定函数返回的值的数据类型，可以是任何PL / SQL数据类型（请参阅PL / SQL数据类型 ）。

对数据类型的限制

您不能约束此数据类型（例如，使用NOT NULL ）。如果datatype是受约束的子类型，则返回的值不会继承子类型的约束（请参阅“ 约束子类型的形式参数 ” ）。

function_definition

定义之前声明的函数或者声明并定义函数。

declare_section

声明函数本地的项，可以在body引用，并在函数完成执行时停止存在。

身体

必需的可执行部分和函数的可选异常处理部分。在可执行部分中，至少一个执行路径必须导致RETURN语句;否则，发生运行时错误。
