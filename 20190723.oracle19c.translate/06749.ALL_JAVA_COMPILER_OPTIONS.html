2.183 ALL_JAVA_COMPILER_OPTIONS

ALL_JAVA_COMPILER_OPTIONS显示有关当前用户可访问的本机编译器选项的信息。

相关观点

  • DBA_JAVA_COMPILER_OPTIONS显示有关数据库中所有本机编译器选项的信息。

  • USER_JAVA_COMPILER_OPTIONS显示有关当前用户拥有的本机编译器选项的信息。此视图不显示OWNER列。
