说明adxdb030.eps的说明

此图显示了Oracle XML DB存储库体系结构。顶部显示虚线框，标记为“Oracle XML DB存储库的应用程序逻辑视图”。在这个框内是一个树，左边是四个文件夹图标和一个文档图标，右边是一个有五列的表。文档图标标记为“abc”。这些列被命名为“表名”，“ACL”，“属性1”，“属性N”和“属性N”。分层列表中的第三个文件夹使用箭头连接到“表名”列中的条目。

虚线框下方是Oracle数据库。名为“Oracle XML DB Repository的数据库视图”的框位于数据库中。这个盒子里面有七个盒子。

这些框中的第一个标记为“RESOURCE_VIEW（XMLType）”。它表示数据库视图，其中包含“Name”，“ACL”，“Property 1”，“Property N”，“Extra”，“Content”（其中包含最后一行中的“LOB”条目）和“Parent”列”。其他框标记为“Path”，“XMLIndex Index”，“B-Tree Index”，“XML Search Index”和“Hierarchical Index”。

名为“Oracle XML DB Repository的数据库视图”的框右侧是一个标有“XMLType Rows”的框。此框下方是“XML或XML视图”。标记为“内容”的列通过双向箭头连接，其中内容标记为“XMLType Rows”。“名称”列是标记为“Oracle XML DB存储库的应用程序逻辑视图”框中“名称”列中不同条目的两个箭头的目标。这两个箭头中的一个标记为“FTP”。另一个标记为“WebDAV”。
