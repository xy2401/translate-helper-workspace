在Oracle数据库上卸载Oracle ASMLIB

如果安装了Oracle ASM库驱动程序（Oracle ASMLIB）但您没有将其用于设备路径持久性，则卸载Oracle ASMLIB：

  1. 停止Oracle ASM和任何正在运行的数据库实例：
    $ srvctl停止数据库
    $ srvctl停止asm
  2. 以root身份登录。
  3. 停止Oracle Restart堆栈：
    #cd Grid_home / bin
    #crsctl停止了

    其中， Grid_home是Oracle Grid Infrastructure主目录路径。

  4. 停止Oracle ASMLIB：
    #etet/init.d/oracleasm禁用
  5. 删除oracleasm库和工具RPM：
    #rpm -e oracleasm-support
    #rpm -e oracleasmlib
  6. 检查是否还有任何oracleasm RPM：
    #rpm -qa | grep oracleasm
  7. 如果剩余任何oracleasm配置文件，请将其删除：
    #rpm -qa | grep oracleasm | xargs rpm -e
    现在删除了Oracle ASMLIB和相关的RPM。

启动Oracle Restart堆栈。（可选）您可以在启动Oracle Restart堆栈之前安装和配置Oracle ASM筛选器驱动程序（Oracle ASMFD）。
