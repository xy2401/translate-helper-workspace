4.247 DBA_FEATURE_USAGE_STATISTICS

DBA_FEATURE_USAGE_STATISTICS显示有关数据库功能使用情况统计信息的信息。

柱 数据类型 空值 描述

DBID

数

不是NULL

正在跟踪的数据库的数据库标识符

名称

VARCHAR2（64）

不是NULL

功能的名称

版

VARCHAR2（17）

不是NULL

跟踪功能的数据库版本

DETECTED_USAGES

数

不是NULL

系统检测到该功能的使用次数

TOTAL_SAMPLES

数

不是NULL

系统已唤醒并检查功能使用情况的次数

CURRENTLY_USED

VARCHAR2（5）

指示系统上次检查时是否检测到使用（ TRUE ）（ FALSE ）

FIRST_USAGE_DATE

日期

系统检测到功能使用的第一个采样时间

LAST_USAGE_DATE

日期

上次采样时间系统检测到该功能的使用情况

AUX_COUNT

数

此列以数字格式存储特定于功能的使用情况数据。

FEATURE_INFO

CLOB

此列以字符格式存储特定于功能的使用情况数据。

LAST_SAMPLE_DATE

日期

系统上次检查使用情况的时间

LAST_SAMPLE_PERIOD

数

最后两次使用采样时间之间的时间量（以秒为单位）

SAMPLE_INTERVAL

数

采样间隔

描述

VARCHAR2（128）

功能和使用检测逻辑的描述

注意：

使用以下SQL查询按字母顺序列出数据库功能及其说明：

SELECT名称，描述FROM dba_feature_usage_statistics ORDER BY名称;
