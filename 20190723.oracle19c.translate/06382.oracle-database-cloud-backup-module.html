D Oracle数据库云备份模块

Oracle数据库云备份模块使您可以将Oracle数据库备份到Oracle数据库备份云服务，这是用于将Oracle数据库备份到Oracle云的存储解决方案。

Oracle数据库云备份模块是一个与Recovery Manager（RMAN）集成的SBT接口。使用RMAN创建数据库备份，Oracle数据库云备份模块通过网络将这些备份发送到Oracle数据库备份云服务以进行异地存储。
