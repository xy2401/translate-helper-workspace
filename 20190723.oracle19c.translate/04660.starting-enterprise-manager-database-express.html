启动Oracle Enterprise Manager Database Express

使用以下步骤登录Oracle Enterprise Manager Database Express（EM Express）。

要启动Oracle Enterprise Manager Database Express，请在数据库安装和创建期间使用Oracle Database Configuration Assistant（Oracle DBCA）提供的EM Express URL。有关登录Oracle Enterprise Manager Database Express的信息，请参阅Oracle Database 2 Day DBA 。
如果Oracle DBCA在数据库安装和创建期间未提供EM Express URL，或者您稍后需要更改EM Express端口，请参阅Oracle Database 2 Day DBA 。
