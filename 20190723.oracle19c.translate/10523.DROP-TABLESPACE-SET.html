DROP TABLESPACE SET

注意：

只有在使用Oracle Sharding时，此SQL语句才有效。有关Oracle Sharding的更多信息，请参阅Oracle Database Administrator's Guide 。

目的

使用DROP TABLESPACE SET语句从shardgroup中删除表空间集。

删除表空间集时，Oracle数据库不会将其放在回收站中。因此，您不能随后清除或取消放置表空间集。

也可以看看：

创建TABLESPACE SET和ALTER TABLESPACE SET

先决条件

您必须作为SDB用户连接到分片目录数据库。

您必须具有DROP TABLESPACE系统特权。如果表空间包含任何包含活动事务的回滚段，则不能删除表空间集。

句法

drop_tablespace_set :: =


插图说明drop_tablespace_set.eps

语义

tablespace_set

指定要删除的表空间的名称。

包括内容

此子句允许您指定数据库如何管理与放置操作期间设置的表空间中的表空间关联的对象和数据文件。INCLUDING CONTENTS子句与DROP TABLESPACE语句具有相同的语义。有关此子句的完整语义，请参阅包含内容 。

例子

删除表空间集：示例

以下语句删除表空间集ts1 ：

DROP TABLESPACE SET ts1;
