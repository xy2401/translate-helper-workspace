关于基于映像的Oracle Grid Infrastructure安装

从Oracle Grid Infrastructure 12c第2版（12.2）开始，通过基于映像的安装简化了Oracle Grid Infrastructure软件的安装和配置。

要安装Oracle Grid Infrastructure，请使用必要的用户组权限创建新的Grid主目录，然后将映像文件解压缩到新创建的Grid主目录中，并运行安装向导以注册Oracle Grid Infrastructure产品。

使用基于映像的安装，您可以执行以下操作：

  • 为新集群配置Oracle Grid Infrastructure。

  • 为独立服务器配置Oracle Grid Infrastructure（Oracle Restart）。

  • 升级Oracle Grid Infrastructure。

  • 仅设置软件。

  • 如果已安装或配置了Oracle Grid Infrastructure软件，则从现有群集添加或删除节点。

此安装功能简化了安装过程，并支持大规模自定义部署的自动化。在使用必要的版本更新（RU）或版本更新修订（RUR）修补基本版本软件之后，您还可以使用此安装方法来部署自定义映像。

注意：

您必须将映像软件解压缩到您希望Grid主目录所在的目录中，然后运行%ORACLE_HOME%\ gridSetup.sh脚本以启动Oracle Grid Infrastructure安装向导。确保您创建的Grid主目录路径符合Oracle Optimal Flexible Architecture建议。
