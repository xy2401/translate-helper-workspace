Patch an unmanaged Oracle Database Software home

patch

/rhp-restapi/rhp/databases

Moves a database from the unamanged Oracle Database Software home to a patched working copy.

Supported Media Types
  • application/json
Header Parameters
  • Authorization: string
    Base64 encoding of the user name and password of the user making the request. For more information, see Authentication for Fleet Patching and Provisioning.
