3.18 SQL

目的

使用SQL命令执行SQL命令和PL / SQL过程。此命令比Oracle Database 11.2及更早版本更易于使用，因为SQL命令不需要用引号括起来，也不需要以“SQL”作为前缀。有关原始语法，请参阅SQL（引用） 。

语义

SqlCommand的:: =

语法元素 描述

SQL CATALOG

在目录数据库中执行SQL命令。

SQL CHANNEL 'channel_id'

在指定通道上执行SQL命令。

目标

在目标数据库中执行SQL命令。

sql语法

适用于指定关键字的SQL语法，RMAN将其发送给SQL进行处理。

开始

表示PL / SQL块的主体。

宣布

表示PL / SQL块的声明部分。

plsql语句

一个PL / SQL语句或块，RMAN将其发送给SQL进行处理。不支持绑定变量并导致执行错误。无法查看任何输出。

孤独/

单斜杠（/）作为一行中的第一个也是唯一的字符。

sqlKeyword :: =

sqlkeyword子句列出了可以在RMAN中执行的SQL命令。有关SQL语法，请参阅Oracle数据库SQL语言参考 。下表描述了例外情况。

语法元素 描述

改变

替换RMAN ALTER DATABASE命令并提供SQL ALTER命令的完整功能。

删除

需要FROM关键字来执行SQL DELETE命令;否则，执行RMAN DELETE命令。

DROP DATABASE

执行RMAN DROP DATABASE命令。

闪回

执行RMAN FLASHBACK DATABASE命令。

选择

使用这些列宽显示返回的行：

  • 数字：10个字符
  • 字符：最大长度
  • 长：80个字符

使用SQL SET命令更改数字和长列的显示大小。没有其他命令或选项可用于格式化输出。

不支持绑定变量并导致执行错误。INTO条款被忽略。

allSqlKeywords :: =

allSqlKeywords子句前面是SQL关键字，它消除了SQL和RMAN命令之间的歧义。

语法元素 描述

删除

执行SQL DELETE命令。

DROP DATABASE

执行SQL DROP DATABASE命令。

FLASHBACK数据库

执行SQL FLASHBACK命令。

FLASHBACK PLUGGABLE DATABASE

执行SQL FLASHBACK PLUGGABLE DATABASE命令。
