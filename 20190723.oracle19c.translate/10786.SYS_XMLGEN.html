SYS_XMLGEN

注意：

不推荐使用SYS_XMLGen函数。它仍然支持向后兼容性。但是，Oracle建议您使用SQL / XML生成函数。有关更多信息，请参见Oracle XML DB Developer's Guide 。

句法


说明sys_xmlgen.eps的说明

目的

SYS_XMLGen接受一个表达式，该表达式求值为数据库的特定行和列，并返回包含XML文档的XMLType类型的实例。expr可以是标量值，用户定义类型或XMLType实例。

  • 如果expr是标量值，则该函数返回包含标量值的XML元素。

  • 如果expr是一个类型，则该函数将用户定义的类型属性映射到XML元素。

  • 如果expr是XMLType实例，则该函数将文档封装在XML元素中，该元素的默认标记名称为ROW 。

默认情况下，XML文档的元素与expr的元素匹配。例如，如果expr解析为列名，则封闭的XML元素将是相同的列名。如果要以不同方式格式化XML文档，请指定fmt ，它是XMLFormat对象的实例。

例子
