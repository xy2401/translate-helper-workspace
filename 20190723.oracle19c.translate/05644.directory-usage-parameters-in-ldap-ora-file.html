9.1目录服务器使用文件概述

ldap.ora文件包含由Oracle Internet Directory配置助手或Oracle Net Configuration Assistant创建的目录使用配置参数。请勿修改这些参数或其设置。

使用Oracle Internet Directory配置助手创建时， ldap.ora位于ORACLE_HOME/ldap/admin目录中。使用Oracle Net Configuration Assistant创建时， ldap.ora文件位于ORACLE_HOME/network/admin目录中。ldap.ora文件也可以存储在LDAP_ADMIN或TNS_ADMIN环境变量指定的目录中。

9.2目录使用参数

本节列出并描述了以下ldap.ora文件配置参数。

  • DEFAULT_ADMIN_CONTEXT
    DEFAULT_ADMIN_CONTEXT ldap.ora文件配置参数指定用于创建，修改或搜索连接标识符的默认目录。
  • DIRECTORY_SERVER_TYPE
    DIRECTORY_SERVER_TYPE是ldap.ora文件的网络参数，它指定正在使用的目录服务器的类型。
  • DIRECTORY_SERVERS
    DIRECTORY_SERVERS是一个目录用法参数，它列出了主LDAP目录服务器和备用LDAP目录服务器的主机名和端口号。

9.2.1 DEFAULT_ADMIN_CONTEXT

DEFAULT_ADMIN_CONTEXT ldap.ora文件配置参数指定用于创建，修改或搜索连接标识符的默认目录。

目的

指定包含Oracle Context的缺省目录条目，可以从中创建，修改或查找连接标识符。

值

有效的专有名称（DN）

例9-1例子

DEFAULT_ADMIN_CONTEXT = “O = OracleSoftware，c =美国”

9.2.2 DIRECTORY_SERVER_TYPE

DIRECTORY_SERVER_TYPE是ldap.ora文件的网络参数，它指定正在使用的目录服务器的类型。

目的

指定正在使用的目录服务器的类型。

值

  • oid for Oracle Internet Directory

  • 适用于Microsoft Active Directory的ad

例9-2例子

DIRECTORY_SERVER_TYPE = OID

9.2.3 DIRECTORY_SERVERS

DIRECTORY_SERVERS是一个目录用法参数，它列出了主LDAP目录服务器和备用LDAP目录服务器的主机名和端口号。

目的

列出主LDAP目录服务器和备用LDAP目录服务器的主机名和端口号。

值

主机：端口[：sslport]

例9-3例子

DIRECTORY_SERVERS =（ldap-server：389，抽奖：400：636）
