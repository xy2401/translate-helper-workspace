1.54 CORE_DUMP_DEST

CORE_DUMP_DEST指定Oracle转储核心文件的目录。

属性 描述

参数类型

串

句法

CORE_DUMP_DEST = directory

默认值

ORACLE_HOME /DBS

可修改

改变系统

可在PDB中修改

没有

基本

没有

CORE_DUMP_DEST主要是UNIX参数，您的平台可能不支持。

注意：

Oracle Database 11g第1版（11.1）中引入的新可诊断性基础结构忽略了此参数，该基础结构将跟踪和核心文件放在由DIAGNOSTIC_DEST初始化参数控制的位置。
