11 Oracle XA库

Oracle XA库是一个外部接口，允许Oracle服务器以外的事务管理器协调全局事务。XA库支持分布式事务中的非Oracle资源管理器。这在多个数据库和资源之间的事务中特别有用。

Oracle XA库的实现符合X / Open分布式事务处理（DTP）软件架构的XA接口规范。Oracle XA库作为Oracle数据库企业版的一部分安装。

11.1使用XA和OCCI开发应用程序

对于Oracle数据库的连接，断开连接和事务控制，应用程序必须与事务管理器连接。OCCI具有用于与XA中的Environment和Connection对象进行交互的API，并使它们可用于Oracle数据库访问，例如SELECT查询，DML语句，对象访问等。

示例11-1如何使用XA使用事务管理器

/ *事务管理器打开与Oracle服务器的连接* / tpopen（“oracle_xa + acc = p / HR / password + sestm = 10”，1，TMNOFLAGS）; / *事务管理器发出XA命令以启动全局事务* / tpbegin（）; / *使用OCCI * / Environment * xaenv = Environment :: getXAEnvironment（“oracle_xa + acc = p / HR / password + sestm = 10”）访问底层Oracle数据库; Connection * xaconn = xaenv-> getXAConnection（“oracle_xa + acc = p / HR / password + sestm = 10”）; / *使用Environment＆Connection对象* / Statement * stmt = xaconn-> createStatement（“Update Emp set sal = sal * 0.2”）; .../ *释放环境和连接对象* / xaenv-> releaseXAConnection（xaconn）;环境:: releaseXAEnvironment（xaenv）;
