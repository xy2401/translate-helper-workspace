DROP MATERIALIZED VIEW LOG

目的

使用DROP MATERIALIZED VIEW LOG语句从数据库中删除LOG化视图日志。

注意：

支持使用关键字SNAPSHOT代替MATERIALIZED VIEW以实现向后兼容。

也可以看看：

  • 创建物化视图和更改物化视图以获取有关物化视图的更多信息

  • 创建物化视图日志以获取有关物化视图日志的信息

  • “Oracle数据库管理员指南” ，了解有关复制环境中的实例化视图的信息

  • Oracle数据库数据仓库指南 ，了解有关数据仓库环境中物化视图的信息

先决条件

要删除实例化视图日志，您必须具有删除表所需的权限。

drop_materialized_view_log :: =

语义

模式

指定包含实例化视图日志及其主表的模式。如果省略schema ，则Oracle数据库假定物化视图日志和主表位于您自己的模式中。

表

指定与要删除的实例化视图日志关联的主表的名称。

删除FOR FAST REFRESH创建的物化视图日志后，无法再快速刷新基于物化视图日志主表的某些物化视图。这些物化视图包括rowid实体化视图，主键物化视图和子查询物化视图。

也可以看看：

“Oracle数据库数据仓库指南”，用于描述这些类型的物化视图

删除FOR SYNCHRONOUS REFRESH创建的SYNCHRONOUS化视图日志（暂存日志）后，无法再刷新基于暂存日志主表的实例化视图。

例子

删除物化视图日志：示例

以下语句删除了oe.customers主表上的oe.customers化视图日志：

DROP MATERIALIZED VIEW LOG ON客户;
