2.19 ALL_ANALYTIC_VIEW_DIMENSIONS

ALL_ANALYTIC_VIEW_DIMENSIONS描述与当前用户可访问的分析视图关联的属性维度。

相关观点

  • DBA_ANALYTIC_VIEW_DIMENSIONS描述与数据库中所有分析视图关联的属性维度。

  • USER_ANALYTIC_VIEW_DIMENSIONS描述与当前用户架构中的分析视图关联的属性维度。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

分析视图的所有者

ANALYTIC_VIEW_NAME

VARCHAR2（128）

不是NULL

分析视图的名称

DIMENSION_OWNER

VARCHAR2（128）

不是NULL

包含属性维度的架构的所有者

DIMENSION_NAME

VARCHAR2（128）

不是NULL

属性维度的名称

DIMENSION_ALIAS

VARCHAR2（128）

分析视图中属性维度的别名

DIMENSION_TYPE

VARCHAR2（8）

属性维度的类型：

  • 时间

  • 标准

ALL_MEMBER_NAME

CLOB

属性维度的ALL成员名称的表达式

ALL_MEMBER_CAPTION

CLOB

属性维度的ALL成员的标题的表达式，如果未指定，则为NULL

ALL_MEMBER_DESCRIPTION

CLOB

属性维度的ALL成员的描述的表达式，如果未指定，则为NULL

REFERENCES_DISTINCT

VARCHAR2（1）

指示事实表键和属性维度属性之间的引用是否指定DISTINCT关键字。可能的值是：

  • Y ：引用指定DISTINCT关键字。

  • N ：引用未指定DISTINCT关键字。

ORDER_NUM

数

不是NULL

分析视图中属性维的订单号

ORIGIN_CON_ID

数

数据源自的容器的ID。可能的值包括：

  • 0 ：该值用于非CDB中的行。该值不用于CDB。

  • n ：此值用于包含源自容器ID为n的容器中的数据的行（如果行源自root，则n = 1）。
