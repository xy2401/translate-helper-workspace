5.291 DBA_ROLES

DBA_ROLES描述数据库中的所有角色。

柱 数据类型 空值 描述

角色

VARCHAR2（128）

不是NULL

角色的名称

ROLE_ID

数

不是NULL

角色的ID号

要求输入密码

VARCHAR2（8）

不推荐使用此列，而推荐使用AUTHENTICATION_TYPE列

AUTHENTICATION_TYPE

VARCHAR2（11）

表示角色的认证机制：

  • NONE - CREATE ROLE role1 ;

  • EXTERNAL - CREATE ROLE role2 IDENTIFIED EXTERNALLY;

  • GLOBAL - CREATE ROLE role3 GLOBAL role3 IDENTIFIED GLOBALLY;

  • APPLICATION - CREATE ROLE role4 IDENTIFIED USING schema IDENTIFIED USING . package ;

  • PASSWORD - CREATE ROLE role5 IDENTIFIED BY role5 IDENTIFIED BY ;

共同

VARCHAR2（3）

指示给定角色是否通用。可能的值：

  • 如果角色很常见，则为YES

  • NO如果角色是本地的（不常见）

ORACLE_MAINTAINED

VARCHAR2（1）

表示角色是由Oracle提供的脚本（例如catalog.sql还是catproc.sql）创建并维护的。除非运行Oracle提供的脚本，否则不得以任何方式更改此列具有值Y的角色。

遗传

VARCHAR2（3）

指示角色是否从其他容器继承（ YES ）（ NO ）

隐

VARCHAR2（3）

指示角色是否是隐式应用程序创建的公共角色（ YES ）否（ NO ）

EXTERNAL_NAME

VARCHAR2（4000）

对于全局角色，外部名称是指映射到全局角色的目录服务中的组的DN。这不适用于本地角色。
