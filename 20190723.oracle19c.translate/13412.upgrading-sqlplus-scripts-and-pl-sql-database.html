升级Oracle数据库后升级SQL * Plus脚本和PL / SQL

要使用新Oracle Database发行版的特性和功能，必须更改现有SQL脚本以使用新Oracle Database发行版的语法。

如果现有SQL脚本不使用新Oracle数据库发行版的特性和功能，则它们在新的Oracle数据库发行版上运行不变，无需修改。

请注意，由于新Oracle Database发行版中的错误检查得到了改进，因此可能会在编译时而不是在运行时识别错误。
