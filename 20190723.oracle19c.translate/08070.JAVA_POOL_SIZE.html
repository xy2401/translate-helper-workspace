1.156 JAVA_POOL_SIZE

JAVA_POOL_SIZE指定（以字节为单位）Java池的大小，Java内存管理器在运行时执行期间从中分配大多数Java状态。 此内存包括Java方法和类定义的共享内存中表示，以及在调用结束时迁移到Java会话空间的Java对象。
属性 描述

参数类型

大整数

句法

JAVA_POOL_SIZE = integer [K | M | G]

默认值

如果设置了SGA_TARGET ：如果未指定参数，则默认值为0 （由Oracle数据库内部确定）。如果指定了参数，则用户指定的值指示内存池的最小值。

如果未设置SGA_TARGET ： 24 MB，则向上舍入到最接近的颗粒大小

可修改

改变系统

可在PDB中修改

没有

价值范围

最小值： 0 （大于零的值向上舍入到最接近的粒度）

最大值：取决于操作系统
