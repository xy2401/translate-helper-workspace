句法

REPH [EADER] [PAGE] [ printspec [ text | 变量 ] ...]| [ON | 关 ]

其中printspec表示用于放置和格式化文本的以下一个或多个子句：

COL n S [KIP] [ n ] TAB n LE [FT] CE [NTER] R [IGHT] BOLD FORMAT text

地点和格式指定的报告标题位于每个报告的顶部，或列出当前的REPHEADER定义。

输入REPHEADER，不带子句列出当前的REPHEADER 定义。

条款

这些条款和条款也适用于REPFOOTER命令。

页

打印指定的报表标题后或打印指定的报表页脚之前开始新页面。 

文本

报告页眉或页脚文本。如果要在一行上放置多个单词，请在单引号中输入文本 。默认值为NULL。

变量

替换变量或以下任何系统维护值。 SQL.LNO是当前行号，SQL.PNO是当前页码，SQL.CODE是当前错误代码，SQL.RELEASE是当前Oracle数据库版本号，而SQL.USER是当前用户名。

要打印其中一个值，请在报表页眉或页脚中引用相应的变量。您可以使用FORMAT子句格式化变量 。 

关闭

打开报告标题或者页脚关闭（抑制其显示）而不影响其定义。

COL n

缩进至当前行的列n （如果已传递列n ，则向后）。此上下文中的列表示打印位置，而不是表格列。 

S [KIP] [ n ]

跳到新行的开头n次;如果省略n ，一次;如果为n输入零，则返回当前行的开头。 

TAB n

向前跳过n列（如果为n输入负值，则向后）。此上下文中的列表示打印位置，而不是表格列。 

LE [FT] CE [NTER] R [IGHT]

左对齐 ，居中和右对齐分别是当前行的数据。SQL * Plus将以下数据项作为一个组对齐，直到printspec的末尾或下一个LEFT，CENTER，RIGHT或COL命令。CENTER和RIGHT使用SET LINESIZE值来计算后面数据项的位置。 

胆大

以粗体打印数据。SQL * Plus通过在三个连续行上重复数据来表示终端上的粗体打印。在某些操作系统上，SQL * Plus可能会指示您的打印机在三个连续行而不是粗体上打印粗体文本。 

格式文字

指定一个格式模型，用于确定直到下一个FORMAT子句或命令结尾的数据项的格式。格式模型必须是文本常量，例如A10或$ 999。有关格式化和有效格式模型的更多信息，请参见COLUMN 。

如果格式模型的数据类型与给定数据项的数据类型不匹配，则FORMAT子句对该项无效。

如果在给定数据项之前没有适当的FORMAT模型，则SQL * Plus将根据SET NUMFORMAT指定的格式打印NUMBER值，如果尚未使用SET NUMFORMAT，则为默认格式。SQL * Plus使用默认格式打印DATE值。

用法

如果在文本或变量之前没有输入printspecact子句，则REPHEADER左对齐文本或变量。

您可以在printspec中使用任意数量的常量和变量。SQL * Plus以您指定的顺序显示常量和变量，定位和格式化每个常量或变量， 如其前面的printspec子句所指定。

例子

要将“员工列表报告”定义为单独页面上的报告标题，并将其居中，请输入：

REPHEADER PAGE CENTER'员工上市报告'TTITLE RIGHT'页面：'格式999 SQL.PNO SELECT LAST_NAME，SALARY FROM EMP_DETAILS_VIEW WHERE SALARY> 12000;
页：1员工上市报告页面：2 LAST_NAME SALARY ------------------------- ---------- King 24000 Kochhar 17000 De Haan 17000 Russell 14000 Partners 13500 Hartstein 13000 ----------总和98500 6行选中。

要在不更改其定义的情况下取消报告标题，请输入：

REPHEADER关闭
