8.201 V $ RULE_SET_AGGREGATE_STATS

V$RULE_SET_AGGREGATE_STATS显示在所有规则集上的所有评估中聚合的统计信息。此视图的每种统计信息都有一行。

柱 数据类型 描述

名称

VARCHAR2（80）

统计名称：

  • rule set evaluations (all) - 所有规则集的评估总数

  • rule set evaluations (first_hit) - stop_on_first_hit设置为TRUE规则集的评估总数

  • rule set evaluations (simple_rules_only) - 将simple_rules_only设置为TRUE规则集的评估总数

  • rule set evaluations (SQL free) - 对内部不发布SQL以评估规则的规则集的评估总数

  • rule set evaluation time (CPU) - 在rule set evaluation time (CPU)总CPU时间（以百分之一秒为单位）

  • rule set evaluation time (elapsed) - 在规则集的评估中花费的总耗用时间（以百分之一秒为单位）

  • rule set SQL executions - 在规则集评估期间执行的SQL语句总数

  • rule set conditions processed - 在规则集评估期间处理的快速（索引）条件的总数

  • rule set true rules - 对规则集的求值期间返回的TRUE规则的总数

  • rule set maybe rules - 在规则集评估期间返回的MAYBE规则总数

  • rule set user function calls (variable value function) - 对用户定义的函数进行的调用总数，用于检索在规则集评估期间生成的变量值（由RE$VARIABLE_TYPE的variable_value_function字段指定）

  • rule set user function calls (variable method function) - 对用户定义的函数进行的调用总数，用于检索在规则集评估期间对变量方法值（由RE$VARIABLE_TYPE的variable_method_function字段指定）

  • rule set user function calls (evaluation function) - 在对规则集进行评估期间对用户定义的评估函数（指定为DBMS_RULE_ADM.CREATE_EVALUATION_CONTEXT过程的evaluation_function参数）进行的DBMS_RULE_ADM.CREATE_EVALUATION_CONTEXT

值

数

统计值

CON_ID

数

数据所属容器的ID。此视图中的CON_ID值始终为0。行属于整个CDB或非CDB。
