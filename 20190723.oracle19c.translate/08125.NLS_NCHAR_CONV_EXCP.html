1.208 NLS_NCHAR_CONV_EXCP

NLS_NCHAR_CONV_EXCP确定在NCHAR / NVARCHAR2和CHAR / VARCHAR2之间的隐式或显式字符类型转换期间数据丢失时是否报告错误。

属性 描述

参数类型

串

句法

NLS_NCHAR_CONV_EXCP = {TRUE |假的}

默认值

假

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

默认值不会导致报告错误。
