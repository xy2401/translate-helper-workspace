6.117 DBA_WORKLOAD_CAPTURES

DBA_WORKLOAD_CAPTURES显示已在当前数据库中执行的所有工作负载捕获。

它还列出了已调用DBMS_WORKLOAD_CAPTURE.GET_CAPTURE_INFO()或DBMS_WORKLOAD_REPLAY.GET_REPLAY_INFO()捕获。每行包含有关一个工作负载捕获的信息。

柱 数据类型 空值 描述

ID

数

不是NULL

工作负载捕获的内部密钥

名称

VARCHAR2（128）

不是NULL

工作负载捕获的名称

DBID

数

不是NULL

捕获工作负载的数据库的ID

DBNAME

VARCHAR2（10）

不是NULL

捕获工作负载的数据库的名称

DBVERSION

VARCHAR2（128）

不是NULL

捕获工作负载的数据库版本

平行

VARCHAR2（3）

指示捕获工作负载的数据库是Oracle RAC数据库（ YES ）还是单实例数据库（ NO ）

目录

VARCHAR2（128）

不是NULL

工作负载捕获的目录对象的名称

状态

VARCHAR2（40）

不是NULL

工作负载捕获的当前状态：

  • IN PROGRESS - 工作负载捕获正在进行中

  • COMPLETED -负载捕获成功完成

  • FAILED - 由于遇到错误而中止了工作负载捕获

开始时间

日期

不是NULL

捕获开始的日期时间

时间结束

日期

捕获完成或失败的日期时间;如果捕获仍在进行中，则为NULL

DURATION_SEC秒

数

工作负载捕获的持续时间（以秒为单位）

START_SCN

数

不是NULL

启动此捕获的SCN值

END_SCN

数

结束此捕获的SCN值;如果捕获仍在进行中，则为NULL

DEFAULT_ACTION

VARCHAR2（30）

不是NULL

应用工作负载捕获过滤器的模式：

  • INCLUDE - 所有捕获过滤器都被视为排除过滤器，用于确定不会捕获的工作负载。

  • EXCLUDE - 所有捕获过滤器都被视为包含过滤器，确定将捕获的工作负载。

FILTERS_USED

数

用于此捕获的过滤器数

CAPTURE_SIZE

数

工作负载捕获的总大小

DBTIME

数

此工作负载捕获中记录的数据库时间总量（以微秒为单位）

DBTIME_TOTAL

数

工作负载捕获期间整个数据库中的数据库时间总量（以微秒为单位），包括未捕获的部分工作负载。

USER_CALLS

数

此工作负载捕获中记录的用户呼叫总数

USER_CALLS_TOTAL

数

在工作负载捕获期间跨整个数据库的用户调用总数，包括未捕获的部分工作负载。

USER_CALLS_UNREPLAYABLE

数

在此工作负载捕获的后续重播中不会重播的用户调用总数

PLSQL_SUBCALL_SIZE

数

从PL / SQL执行的SQL的工作负载捕获总大小

PLSQL_CALLS

数

工作负载捕获中记录的PL / SQL调用总数

PLSQL_SUBCALLS

数

从PL / SQL执行的SQL的工作负载捕获中记录的调用总数

PLSQL_DBTIME

数

来自工作负载捕获中记录的PL / SQL调用的数据库时间总量（以微秒为单位）

交易

数

此工作负载捕获中记录的事务总数

TRANSACTIONS_TOTAL

数

工作负载捕获期间整个数据库中的事务总数，包括未捕获的部分工作负载。

连接

数

此工作负载捕获中记录的会话连接总数

CONNECTS_TOTAL

数

在工作负载捕获期间，整个数据库中的会话总数连接，包括未捕获的部分工作负载

错误

数

此工作负载捕获中记录的错误总数

AWR_DBID

数

与此工作负载捕获对应的AWR快照的数据库ID。对于在当前数据库中执行的捕获，此值等于当前数据库的DBID。对于在其他数据库中执行的捕获，此值将为NULL或将由DBMS_WORKLOAD_CAPTURE.IMPORT_AWR()填充。

AWR_BEGIN_SNAP

数

开始与此工作负载捕获对应的AWR快照的快照ID

AWR_END_SNAP

数

与此工作负载捕获对应的AWR快照的结束快照ID

AWR_EXPORTED

VARCHAR2（12）

指示是否对应于这个工作量捕获的AWR快照一直在使用出口DBMS_WORKLOAD_CAPTURE.EXPORT_AWR() YES ）否（ NO ），或者是否AWR快照无法导出，因为捕获仍在进行中，已完成运行成功，或者是在从它没有导出不同的数据库（做NOT POSSIBLE ）

错误代码

数

此工作负载捕获的错误代码

错误信息

VARCHAR2（300）

此工作负载捕获的错误消息

DIR_PATH

VARCHAR2（4000）

不是NULL

工作负载捕获目录对象的完整目录路径

DIR_PATH_SHARED

VARCHAR2（10）

不是NULL

指示工作负载捕获目录是否由记录数据库的所有实例共享（仅适用于Oracle RAC数据库）

LAST_PROCESSED_VERSION

VARCHAR2（128）

使用DBMS_WORKLOAD_REPLAY.PROCESS_CAPTURE() last预处理此捕获的数据库版本;如果捕获从未被预处理，则为NULL

SQLSET_OWNER

VARCHAR2（128）

SQL调整集所有者的用户名

SQLSET_NAME

VARCHAR2（128）

此工作负载捕获的SQL调整集的名称

PLSQL_MODE

VARCHAR2（12）

捕获PL / SQL调用的选项。可能的值：

  • TOP_LEVEL ：仅限顶级PL / SQL

  • EXTENDED ：从PL / SQL执行的顶级PL / SQL和SQL

加密

VARCHAR2（128）

表示用于给定捕获的加密标准：

  • NULL - 捕获文件未加密

  • AES128 - 使用AES128加密捕获文件

  • AES192 - 使用AES192加密捕获文件

  • AES256 - 使用AES256加密捕获文件
