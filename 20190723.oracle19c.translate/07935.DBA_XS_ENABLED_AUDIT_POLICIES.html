6.149 DBA_XS_ENABLED_AUDIT_POLICIES

DBA_XS_ENABLED_AUDIT_POLICIES描述了为用户启用的Oracle Database Real Application Security特有的所有审核策略。

注意：

此视图在Oracle Database 12 c第1版中称为DBA_XS_ENB_AUDIT_POLICIES 。它已在Oracle Database 12 c第2版（12.2.0.1）中重命名为DBA_XS_ENABLED_AUDIT_POLICIES 。

柱 数据类型 空值 描述

POLICY_NAME

VARCHAR2（128）

审计策略的名称

ENABLED_OPTION

VARCHAR2（15）

已启用审核策略选项。可能的值：

  • BY USER ：用于在用户上启用的策略

  • EXCEPT USERS ：对于在用户上启用的策略

  • BY GRANTED ROLE ：对于在角色上启用的策略

  • INVALID ：对于在角色上启用的策略

实体名称

VARCHAR2（128）

启用审核策略的数据库实体（用户名或角色名称）

ENTITY_TYPE

VARCHAR2（7）

数据库实体类型。可能的值：

  • USER ：表示已对某个用户启用该策略。

  • ROLE ：表示对一个或多个角色启用了策略。

成功

VARCHAR2（3）

指示审核策略是否已启用审核成功事件（ YES ）或否（ NO ）

失败

VARCHAR2（3）

指示是否启用审核策略以审核不成功事件（ YES ）或否（ NO ）
