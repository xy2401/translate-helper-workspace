支持直接升级的Oracle数据库版本

查看直接升级到最新Oracle Database版本的受支持选项。

您可以从以下版本直接升级到新版本：

  • 11.2.0.4

  • 12.1.0.2

  • 12.2.0.1

  • 18

升级到最新Oracle Database版本必须采取的路径取决于当前数据库的版本号。

如果您当前的Oracle数据库是早于11.2.0.4的版本，那么您无法直接将Oracle数据库升级到最新版本。在这种情况下，您需要在升级到Oracle Database 19c之前升级到中间版本。

如果无法进行直接升级，请执行升级到支持直接升级的最新版本。

注意：

对于任何多步升级，如果必须执行两次升级以升级到当前版本，则必须运行两次预升级脚本：首先，完成升级到支持直接升级到目标升级的中间升级版本发布。其次，完成目标升级版本的升级。

例如，如果要升级的数据库正在运行Oracle数据库10g，那么要升级到Oracle Database 19c，请执行以下步骤：

  1. 使用Oracle数据库升级指南12c第1版（12.1）中的说明将版本10.2.0.5升级到版本12.1.0.2（终端补丁集版本） ，包括运行12.1.0.2的预升级脚本。

  2. 将Oracle Database 12c第1版（12.1.0.2）直接升级到Oracle Database 19c。使用本书“ Oracle数据库升级指南”中的说明 ，包括运行Oracle Database 19c的预升级脚本。
