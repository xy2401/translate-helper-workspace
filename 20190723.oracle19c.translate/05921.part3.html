第IV部分管理Oracle Label Security应用程序

第IV部分介绍了如何管理Oracle Label Security应用程序。

  • 实施政策执行选项和标签功能
    您可以自定义Oracle Label Security策略的实施并实现标签功能。
  • 管理和使用受信任的存储程序单元
    您可以使用受信任的存储程序单元来增强系统安全性。
  • 在Oracle Label Security下进行审计
    如果尚未将数据库配置为使用统一审计，则可以使用Oracle Label Security审计。
  • 将Oracle Label Security与分布式数据库配合使用
    您应该了解在分布式配置中使用Oracle Label Security的特殊注意事项。
  • 在Oracle Label Security下执行DBA功能
    Oracle Label Security支持标准的Oracle数据库实用程序，但适用某些限制，这可能需要额外的步骤才能获得预期的结果。
  • 使用反向组的可释放性
    Oracle Label Security可以使用反向组实现可释放性。
