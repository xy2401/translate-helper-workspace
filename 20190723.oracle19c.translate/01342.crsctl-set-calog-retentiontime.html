C.5 crsctl设定calog保留时间

配置群集资源活动日志的保留时间。

句法

crsctl设定calog保留hours

参数

crsctl set calog retentiontime命令需要花费数小时作为参数。

使用说明

指定群集资源活动日志的保留时间的值（以小时为单位）。

例子

以下示例将群集资源活动日志的保留时间设置为72小时：

$ crsctl set calog retentiontime 72
