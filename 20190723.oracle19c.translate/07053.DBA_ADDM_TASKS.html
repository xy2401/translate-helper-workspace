4.9 DBA_ADDM_TASKS

DBA_ADDM_TASKS显示有关数据库中所有ADDM任务的信息。

对于具有ADVISOR_NAME=ADDM和STATUS=COMPLETED的相关DBA_ADVISOR_TASKS视图中的视图，该视图包含一行。

相关视图

USER_ADDM_TASKS显示有关当前用户拥有的ADDM任务的信息。对于具有ADVISOR_NAME=ADDM和STATUS=COMPLETED的相关USER_ADVISOR_TASKS视图中的视图，该视图包含一行。此视图不显示OWNER列。

DB_TYPE_DETECTED 脚1

VARCHAR2（10）

指示ADDM检测到的任务正在分析的数据库的打开模式。可能的值：

  • 只读

  • 读写

DB_TYPE_ANALYZED Footref 1

VARCHAR2（10）

表示任务正在分析的数据库的假定打开模式。如果此列的值与DB_TYPE_DETECTED列的值不同，则ADDM忽略检测到的数据库打开模式，并将该任务分析为在具有假定打开模式的数据库上发生的任务。可能的值：

  • 只读

  • 读写

CDB_TYPE_DETECTED Footref 1

VARCHAR2（25）

指示ADDM检测到的任务正在分析的数据库类型。可能的值：

  • NON-CDB - 非CDB

  • CDB ROOT - CDB的根容器

  • PDB - CDB中的可插拔数据库（PDB）

  • AUTONOMOUS DATA WAREHOUSE - 承载Oracle自治数据仓库云服务的PDB

  • AUTONOMOUS OLTP - 承载Oracle自治事务处理服务的PDB

CDB_TYPE_ANALYZED Footref 1

VARCHAR2（25）

表示任务正在分析的假定数据库类型。如果此列的值与CDB_TYPE_DETECTED列的值不同，则ADDM忽略检测到的数据库类型，并将该任务分析为假定它出现在假定的数据库类型上。可能的值：

  • NON-CDB - 非CDB

  • CDB ROOT - CDB的根容器

  • PDB - CDB中的可插拔数据库（PDB）

  • AUTONOMOUS DATA WAREHOUSE - 承载Oracle自治数据仓库云服务的PDB

  • AUTONOMOUS OLTP - 承载Oracle自治事务处理服务的PDB

脚注1此列从Oracle Database 19c版本19.1开始提供。如果任务是在早于Oracle Database 19c的版本中分析数据库，并且该数据库随后升级到Oracle Database 19c，则此列的值为NULL。
