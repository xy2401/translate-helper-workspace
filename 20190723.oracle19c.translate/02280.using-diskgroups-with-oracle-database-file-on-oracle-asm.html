Using Disk Groups with Oracle Database Files on Oracle ASM

Review this information to configure Oracle Automatic Storage Management (Oracle ASM) storage for Oracle Clusterware and Oracle Database Files.

  • Identifying and Using Existing Oracle Database Disk Groups on Oracle ASM
    Identify existing disk groups and determine the free disk space that they contain. Optionally, identify failure groups for the Oracle ASM disk group devices.
  • Creating Disk Groups for Oracle Database Data Files
    If you are sure that a suitable disk group does not exist on the system, then install or identify appropriate disk devices to add to a new disk group.
  • Creating Directories for Oracle Database Files
    You can store Oracle Database and recovery files on a separate file system from the configuration files.
