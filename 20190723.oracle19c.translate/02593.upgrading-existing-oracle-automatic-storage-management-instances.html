关于升级现有Oracle自动存储管理实例

Oracle自动存储管理（Oracle ASM）升级在Oracle Grid Infrastructure升级期间执行。

如果要从Oracle ASM 11g第2版（11.2.0.4）或更高版本升级，则Oracle ASM始终使用Oracle Grid Infrastructure进行升级，作为升级的一部分，Oracle自动存储管理配置助手（Oracle ASMCA）由根启动升级过程中的脚本随后，您可以使用Oracle ASMCA（位于Grid_home/bin ）配置故障组，Oracle ASM卷和Oracle自动存储管理群集文件系统（Oracle ACFS）。

Oracle ASMCA无法将Oracle ASM从先前版本单独升级到当前版本。

不支持从11g第2版（11.2）之前的版本升级Oracle ASM。
