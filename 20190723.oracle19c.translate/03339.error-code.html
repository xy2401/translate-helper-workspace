9 Oracle Database Provider for DRDA中的错误代码支持

用于DRDA的Oracle数据库提供程序具有错误代码支持，并支持管理由DB2和DRDA开发的应用程序生成的错误和警告。

Oracle错误代码

在DRDA和DB2中，某些类型的操作可能会成功但会生成警告。例如，当操作生成整数溢出情况时，DB2继续并返回其余的行数据，并为包含错误的列值设置指示符。它还会发出警告代码+802 ：

（EXCEPTION ERROR 异常代码在操作 类型操作期间发生数据类型 DATA，POSITION positionnumber ）

在DB2和DRDA中，负SQLCODE是错误，而正SQLCODE是警告。与DRDA（和DB2）不同，Oracle没有警告条件的概念。在Oracle中，调用执行成功或生成错误。例如，Oracle将溢出情况视为错误，并且不会继续。因此，用于DRDA的Oracle数据库提供程序无法精确模拟DB2行为。

但是，可以映射Oracle和DRDA（和DB2）共有的大多数正常错误条件。

“错误代码映射，从Oracle到DRDA”列出了一些常见的Oracle错误代码及其等效的SQL代码和SQL状态。
