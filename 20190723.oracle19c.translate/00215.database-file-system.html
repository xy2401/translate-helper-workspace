第VII部分数据库文件系统（DBFS）

本部分介绍在设计使用数据库文件系统（DBFS）和DBFS内容存储库的应用程序时必须考虑的问题。注意：在大多数情况下，DBFS需要SecureFiles LOB，这在使用Oracle LOB存储中进行了讨论。SecureFiles是从Oracle Database 12 c开始的LOB的默认存储机制。
