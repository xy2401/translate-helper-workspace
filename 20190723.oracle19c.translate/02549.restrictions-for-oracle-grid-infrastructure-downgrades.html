Oracle Grid Infrastructure的限制降级

有关降级Oracle Grid Infrastructure安装的限制和更改，请查看以下信息。

  • 只有在升级后未进行任何配置更改时，才能将Oracle Grid Infrastructure降级到早期版本。

  • 将Oracle成员集群降级到12c版本2（12.2）时，必须首先在最后降级的节点上启动Oracle Clusterware，然后在其他节点上启动。

  • 当您从Oracle Grid Infrastructure 12c第2版（12.2）或更高版本降级到Oracle Grid Infrastructure 11g第2版（11.2）时，您将从Oracle Flex群集配置降级到标准群集配置，因为早于Oracle的版本中的所有群集配置Grid Infrastructure 12c是标准群集配置。

  • 您只能降级到升级后的Oracle Grid Infrastructure版本。例如，如果从Oracle Grid Infrastructure 11g第2版（11.2）升级到Oracle Grid Infrastructure 19c，则只能降级到Oracle Grid Infrastructure 11g第2版（11.2）。
