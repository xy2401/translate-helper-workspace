本手册是Oracle Pro * COBOL和Pro * FORTRAN预编译器的综合用户指南和参考。它逐步向您展示了如何开发使用SQL访问和操作数据的应用程序。它使用明确的示例探索高级编程技术的基本概念。

目标受众

任何开发新应用程序或将现有应用程序转换为在Oracle数据库环境中运行的人都将从阅读本指南中受益。专为程序员编写，Oracle预编译器的这种综合处理对系统分析员，项目经理和其他对嵌入式SQL应用程序感兴趣的人也很有价值。

要有效地使用本指南，您需要具备以下主题的工作知识：

  • 应用程序使用高级语言编程

  • SQL数据库语言

  • Oracle概念和术语

结构体

本指南包含11章和5个附录。第1章和第2章为您提供了方向，然后第3章，第4章，第5章和第6章将介绍嵌入式SQL编程的基本知识。阅读完这些章节后，您将能够编写和运行有用的嵌入式SQL应用程序。第7章，第8章，第9章，第10章和第11章涵盖了高级主题。您将在每章和附录中找到您的简要摘要。

本样本手册包含一个部分，两个章节和一个附录。（将本章，附录和部分作为交叉引用插入，以便链接在HTML中显而易见。）

熟悉起来

本章向您介绍Oracle预编译器。您将了解他们在开发操作Oracle数据的应用程序中的作用，并了解它们允许应用程序执行的操作。

学习基础知识

本章介绍嵌入式SQL程序如何完成其工作。您将检查它们运行的特殊环境，此环境对应用程序设计的影响，嵌入式SQL编程的关键概念以及开发应用程序时所采取的步骤。

满足计划要求

本章介绍如何满足嵌入式SQL程序要求。您将学习用于声明变量，声明通信区域和连接到Oracle数据库的嵌入式SQL命令。您还将了解Oracle数据类型，全球化支持（全球化支持），数据转换以及如何利用数据类型等效。此外，本章还介绍了如何在程序中嵌入Oracle调用接口（OCI）调用以及如何开发X / Open应用程序。

使用嵌入式SQL

本章将向您介绍嵌入式SQL编程的基本知识。您将学习如何使用主机变量，指示符变量，游标，游标变量以及插入，更新，选择和删除Oracle数据的基本SQL命令。

使用嵌入式PL / SQL

本章介绍如何通过在程序中嵌入PL / SQL事务处理块来提高性能。您将学习如何将PL / SQL与主变量，指示符变量，游标，存储的子程序，主机数组和动态SQL一起使用。

运行Oracle预编译器

本章详细介绍了运行Oracle预编译器的要求。您将了解预编译期间发生的情况，如何发出预编译器命令，如何指定许多有用的预编译器选项，如何执行条件和单独的预编译，以及如何在主机程序中嵌入OCI调用。

定义和控制交易

本章介绍事务处理。您将学习保护数据库一致性的基本技术。

错误处理和诊断

本章提供了有关错误报告和恢复的深入讨论。您将学习如何使用状态变量SQLSTATE，SQLCA结构和WHENEVER语句来检测和处理错误。您还将学习如何使用ORACA诊断问题。

使用主机阵列

本章介绍如何使用数组来提高程序性能。您将学习如何使用数组操作Oracle数据，如何使用单个SQL语句操作数组的所有元素，以及如何限制处理的数组元素的数量。

使用动态SQL

本章介绍如何利用动态SQL。您将学习四种方法 - 从简单到复杂 - 用于编写灵活的程序，其中包括让用户在运行时以交互方式构建SQL语句。

编写用户退出

本章重点介绍如何为SQL * Forms或Oracle Forms应用程序编写用户出口。首先，您将学习允许Forms应用程序与用户出口连接的命令。然后，您将学习如何编写和链接Forms用户出口。

新功能

本附录重点介绍了Oracle预编译器1.8版中引入的改进和新功能。

Oracle保留字_关键字_和命名空间

本附录列出了对Oracle具有特殊含义的单词以及为Oracle库保留的名称空间。

性能调优

本附录为您提供了一些简单易用的方法，可以提高应用程序的性能。

句法和语义检查

本附录向您展示了如何使用SQLCHECK选项来控制在嵌入式SQL语句和PL / SQL块上完成的语法和语义检查的类型和范围。

嵌入式SQL命令和指令

本附录包含预编译器指令，嵌入式SQL命令和Oracle嵌入式SQL扩展的说明。这些命令在源代码中以关键字EXEC SQL开头。
