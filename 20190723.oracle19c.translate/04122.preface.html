ORACLE_HOME和ORACLE_BASE

在Oracle8 i版本8.1.3之前的版本中，当您安装Oracle组件时，所有子目录都位于顶级ORACLE_HOME目录下，该目录默认使用以下名称之一：

  • C:\orant for Windows NT

  • C:\orawin98 for Windows 98

此版本符合最佳灵活架构（OFA）指南。所有子目录都不在顶级ORACLE_HOME目录下。有一个名为ORACLE_BASE的顶级目录，默认情况下为C:\oracle 。如果在未安装其他Oracle软件的计算机上安装最新的Oracle版本，则第一个Oracle主目录的默认设置为C:\oracle\ora nn ，其中nn是最新版本号。Oracle主目录直接位于ORACLE_BASE下。

本指南中的所有目录路径示例都遵循OFA约定。

有关OFA合规性的其他信息以及有关在非OFA兼容目录中安装Oracle产品的信息，请参阅适用于Microsoft Windows的Oracle数据库平台指南 。
