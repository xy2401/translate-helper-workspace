确保要安装的Oracle RAC数据库与现有数据库兼容

  • 如果安装了Oracle Clusterware，并且安装了其他Oracle软件的不同版本，则Oracle Clusterware版本必须晚于或等于Oracle Database软件版本。执行Oracle Grid Infrastructure 19c安装时，Oracle Clusterware和Oracle ASM都升级到19c。

  • 如果您有现有的Oracle主目录，则可以创建新的Oracle主目录并将Oracle Database 19c安装到新的Oracle主目录中。确保Oracle Clusterware位于单独的Oracle Grid Infrastructure主目录中。用于群集安装的Oracle Grid Infrastructure无法安装在Oracle数据库的Oracle主目录中。

特定于平台的服务器配置

  • 确保为HP-UX创建了外部作业用户。

  • 配置在Windows平台上的Oracle ASM集群文件系统（Oracle ACFS）上安装Oracle RAC时要使用的Windows域用户帐户。
