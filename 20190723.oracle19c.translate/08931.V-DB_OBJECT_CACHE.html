7.168 V $ DB_OBJECT_CACHE

V$DB_OBJECT_CACHE显示缓存在库高速缓存中的数据库对象。对象包括表，索引，集群，同义词定义，PL / SQL过程和包以及触发器。

柱 数据类型 描述

所有者

VARCHAR2（64）

对象的所有者

名称

VARCHAR2（1000）

对象的名称

DB_LINK

VARCHAR2（64）

数据库链接名称（如果有）

NAMESPACE

VARCHAR2（64）

对象的库缓存命名空间： TABLE/PROCEDURE, BODY, TRIGGER, INDEX, CLUSTER, OBJECT

类型

VARCHAR2（64）

对象的类型： INDEX, TABLE, CLUSTER, VIEW, SET, SYNONYM, SEQUENCE, PROCEDURE, FUNCTION, PACKAGE, PACKAGE BODY, TRIGGER, CLASS, OBJECT, USER, DBLINK

SHARABLE_MEM

数

对象使用的共享池中的可共享内存量

荷载

数

加载对象的次数。当对象失效时，此计数也会增加。

EXECUTIONS

数

不曾用过

另请参阅： “ V $ SQLAREA ”以查看实际执行计数

锁

数

当前锁定此对象的用户数

销

数

当前固定此对象的用户数

KEPT

VARCHAR2（3）

（ YES | NO ）取决于是否使用PL / SQL过程DBMS_SHARED_POOL.KEEP “保留”此对象（永久固定在内存中）

CHILD_LATCH

数

保护对象的子锁存号。此列已过时并维护以便向后兼容。

废票

数

由于修改了从属对象，因此命名空间中的对象被标记为无效的总次数

HASH_VALUE

数

对象的哈希值

LOCK_MODE

VARCHAR2（9）

当前对象的锁定模式

PIN_MODE

VARCHAR2（9）

对象的当前引脚模式

状态

VARCHAR2（19）

对象的状态

TIMESTAMP

VARCHAR2（19）

对象规范的时间戳

PREVIOUS_TIMESTAMP

VARCHAR2（19）

以前的对象规范的时间戳

LOCKED_TOTAL

数

对象被锁定的总次数

PINNED_TOTAL

数

固定对象的总次数

属性

VARCHAR2（80）

对象的属性。可能的值包括HOT或HOTCOPY时库缓存热复制功能是使用用于DBMS_SHARED_POOL.MARKHOT 。

FULL_HASH_VALUE

VARCHAR2（32）

对象的完整哈希值

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

CON_NAME

VARCHAR2（64）

对象的容器名称。在非CDB中，此列的值为NULL 。

ADDR

RAW（8）

此对象的句柄的地址

版

VARCHAR2（138）

版本名称
