6将二进制XML与C一起使用

给出了如何使用C语言的二进制可扩展标记语言（二进制XML）的解释。

C语言二进制XML简介

可扩展标记语言（XML）数据的客户端处理可以使用存储在数据库中的XMLType数据，包括二进制XML格式的数据，或者不在数据库中的瞬态数据。

使用带有C的二进制XML的先决条件

本章假定您熟悉XML的Parser for C，二进制XML的基本概念和OCI（Oracle调用接口）。只有OCI API可用于使用二进制XML在C中编程。

二进制XML存储格式 - C.

二进制XML是XML的优化格式。它包括XML文档的编码和解码，从文本到二进制，二进制到文本。二进制XML是XML Schema感知的，但它也可以用于不基于XML模式的XML数据。

二进制XML处理器是将二进制XML格式处理和转换为文本和XML文本为二进制XML格式的组件。

中间层和客户端层可以以二进制XML格式生成，使用和处理XML。C应用程序从Oracle XML DB Repository中获取数据，使用DOM对XML执行更新，并将其存储回数据库中。或者在客户端上创建或输入XML文档，并在其上使用XSLT，XQuery和其他实用程序。然后输出XML保存在Oracle XML DB中。有关OCI函数的概念和参考页面的更多详细信息，请参阅Oracle调用接口程序员指南。
