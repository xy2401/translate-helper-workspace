28.4.11 OCINumberDec（）

12.1.0.2，增加了一个简短的描述。

目的

减少Oracle NUMBER 。

句法

剑OCINumberDec（OCIError *错误，OCINumber *号码）;

参数

错误（输入/输出）

OCI错误句柄。如果有错误， OCI_ERROR err记录，此函数返回OCI_ERROR 。通过调用OCIErrorGet()获取诊断信息。

号码（IN / OUT）

要减少的正数Oracle NUMBER 。

评论

减少Oracle NUMBER 。假设输入是0到100 ^ 21-2之间的整数。如果输入太大，则将其视为0;结果是Oracle NUMBER 1。如果输入不是正整数，则结果是不可预测的。

返回

如果输入NUMBER为NULL则此函数返回错误。
