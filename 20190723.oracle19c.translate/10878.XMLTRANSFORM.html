XMLTRANSFORM

句法


说明xmltransform.eps的描述

目的

XMLTransform将XMLType实例和XSL样式表作为参数，它本身就是XMLType实例的一种形式。它将样式表应用于实例并返回XMLType 。

当您从数据库中检索数据时，此功能对于根据样式表组织数据非常有用。

有关此功能的更多信息，请参见Oracle XML DB Developer's Guide

例子

XMLTransform函数需要存在XSL样式表。下面是一个非常简单的样式表示例，该样式表按字母顺序排列节点中的元素：

CREATE TABLE xsl_tab（col1 XMLTYPE）; INSERT INTO xsl_tab VALUES（XMLTYPE.createxml（'<？xml版本=“1.0”？> <xsl：stylesheet version =“1.0”xmlns：xsl =“http://www.w3.org/1999/XSL/Transform”> <xsl：output encoding =“utf-8”/> <！ -  alphabetizes xml树 - > <xsl：template match =“*”> <xsl：copy> <xsl：apply-templates select =“* | text（）”> <xsl：sort select =“name（。）”data -type =“text”order =“ascending”/> </ xsl：apply-templates> </ xsl：copy> </ xsl：template> <xsl：template match =“text（）”> <xsl：value- of select =“normalize-space（。）”/> </ xsl：template> </ xsl：stylesheet>'））;已创建1行。

下一个示例使用xsl_tab XSL样式表来对示例表oe.warehouses一个warehouse_spec中的元素进行字母顺序排列：
