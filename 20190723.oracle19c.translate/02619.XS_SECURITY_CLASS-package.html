XS_SECURITY_CLASS包

XS_SECURITY_CLASS包中包含创建，管理和删除安全类及其权限的过程。该程序包还包括用于管理安全性类继承的过程。

本节包括以下主题：

  • XS_SECURITY_CLASS包的安全模型

  • XS_SECURITY_CLASS子程序摘要

XS_SECURITY_CLASS包的安全模型

XS_SECURITY_CLASS包是在SYS架构下创建的。DBA角色被授予ADMIN_ANY_SEC_POLICY ，允许它管理所有模式中的ACL，安全类和安全策略等模式对象。

如果已为模式授予RESOURCE角色，则用户可以在自己的模式中管理模式对象。RESOURCE角色和XS_RESOURCE应用程序角色包括管理模式中的模式对象所需的ADMIN_SEC_POLICY权限，以及在授予的模式中管理策略工件以在应用程序中实现策略管理。

如果已授予APPLY_SEC_POLICY权限，则用户可以管理架构上的策略实施。使用此权限，用户可以在授予的模式中管理策略实施，以在应用程序内实现策略管理。

CREATE_SECURITY_CLASS过程

CREATE_SECURITY_CLASS创建一个新的安全类。

句法

XS_SECURITY_CLASS.CREATE_SECURITY_CLASS（名称IN VARCHAR2，priv_list IN XS $ PRIVILEGE_LIST，parent_list IN XS $ NAME_LIST：= NULL，description IN VARCHAR2：= NULL）;

参数

参数 描述

名称

要创建的安全性类的名称。

该名称是模式限定的，例如， SCOTT.SC1 。如果缺少名称的模式部分，则假定当前会话模式。例如，在同一个示例中，如果名称指定为SC1 ，并且当前架构为SCOTT ，则它将解析为SCOTT.SC1 。

priv_list

要包含在安全性类中的权限列表。

parent_list

从中继承安全性类的父安全性类的列表。这是可选的。

描述

安全类的可选描述。

例子

以下示例创建一个名为HRPRIVS的安全类。安全性类包括priv_list定义的一组权限。安全性类使用DML类作为其父安全性类。 

DECLARE pr_list XS $ PRIVILEGE_LIST; BEGIN pr_list：= XS $ PRIVILEGE_LIST（XS $ PRIVILEGE（name =>'VIEW_SENSITIVE_INFO'），XS $ PRIVILEGE（name =>'UPDATE_INFO'，implied_priv_list => XS $ NAME_LIST（'“UPDATE”'，'“DELETE”'， '“插入”'）））; SYS.XS_SECURITY_CLASS.CREATE_SECURITY_CLASS（ name =>'HRPRIVS'， priv_list => pr_list， parent_list => XS $ NAME_LIST（'DML'））;结束;

DELETE_SECURITY_CLASS过程

DELETE_SECURITY_CLASS过程删除指定的安全性类。

句法

XS_SECURITY_CLASS.DELETE_SECURITY_CLASS（sec_class IN VARCHAR2，delete_option IN NUMBER：= XS_ADMIN_UTIL.DEFAULT_OPTION）;

参数

参数 描述

sec_class

要删除的安全类的名称。

该名称是模式限定的，例如， SCOTT.SC1 。如果缺少名称的模式部分，则假定当前会话模式。例如，在同一个示例中，如果名称指定为SC1 ，并且当前架构为SCOTT ，则它将解析为SCOTT.SC1 。

delete_option

要使用的删除选项。可以使用以下选项：

  • DEFAULT_OPTION：

    默认选项仅允许在其他地方未引用安全类时删除它。如果有其他实体引用安全性类，则无法删除安全性类。

  • CASCADE_OPTION ：

    级联选项会删除安全性类以及对它的任何引用。删除安全类的用户也必须具有删除这些引用的权限。

  • ALLOW_INCONSISTENCIES_OPTION：

    allow inconsistencies选项允许您删除实体，即使其他实体具有对它的后期绑定引用。

例子

以下示例使用默认选项删除HRPRIVS安全类：

BEGIN SYS.XS_SECURITY_CLASS.DELETE_SECURITY_CLASS（'HRPRIVS'，XS_ADMIN_UTIL.DEFAULT_OPTION）;结束;
