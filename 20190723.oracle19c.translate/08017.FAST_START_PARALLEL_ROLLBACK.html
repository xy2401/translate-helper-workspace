1.123 FAST_START_PARALLEL_ROLLBACK

FAST_START_PARALLEL_ROLLBACK指定恢复已终止事务时使用的并行度。

属性 描述

参数类型

串

句法

FAST_START_PARALLEL_ROLLBACK = {HIGH |低|假的}

默认值

低

可修改

改变系统

可在PDB中修改

没有

基本

没有

终止事务是在系统发生故障之前处于活动状态的事务。如果系统在存在未提交的并行DML或DDL事务时失败，则可以使用此参数在启动期间加快事务恢复。

值

  • 假

    并行回滚已禁用

  • 低

    将最大并行度限制为2 * CPU_COUNT

  • 高

    将最大并行度限制为4 * CPU_COUNT

如果更改此参数的值，则将停止并使用新隐含的并行度重新启动事务恢复。
