特定于产品的安装后任务

必须先完成安装后安装任务，然后才能使用某些Oracle数据库功能。

  • 配置Oracle Database Vault
    安装Oracle RAC软件时，OUI默认安装Oracle Database Vault，但您必须在Oracle RAC数据库中注册Oracle Database Vault并创建数据库用户帐户，然后才能使用它。
  • 配置Oracle数据库安全设置
    使用Oracle Database Configuration Assistant（DBCA）命令在安装后更改安全设置。
  • 配置Oracle Label Security
    安装后，必须在使用之前在数据库中配置Oracle Label Security。
  • 配置Oracle XML DB
    Oracle XML DB是Oracle数据库安装的必需组件。但是，您必须手动为Oracle XML DB配置FTP和HTTP端口。
  • 配置外部表，共享文件或目录对象的存储
    如果Oracle RAC数据库使用数据库外部的文件，则在所有节点均可访问的共享存储上找到外部文件。每个节点应使用相同的安装点来访问该文件。
