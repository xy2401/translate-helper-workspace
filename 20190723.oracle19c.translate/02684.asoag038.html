asoag038.eps插图的描述

这是asoag038.gif的文本描述，该图像显示了Oracle安全体系结构以及企业用户安全性所适用的位置。图像描绘了如下三层：

Application Security层位于顶部，包含以下元素：

  • 第三方应用程序，提供授权和审计

  • Oracle电子商务套件，提供职责和角色

  • Oracle Collaboration Suite，提供S-MIME（安全电子邮件），人际权限和文件权限

  • OracleAS Portal和OracleAS Wireless，提供角色和权限组

Oracle Platform Security层位于中间，包含以下元素：

  • Oracle Application Server，提供JAAS角色，Web服务安全性和Java 2权限

  • Oracle数据库，提供企业用户安全性，虚拟专用数据库，加密和标签安全性

Oracle身份管理基础架构层位于底部，包含以下元素：

  • OracleAS证书颁发机构

  • Oracle Delegated Administration Services

  • OracleAS单点登录

  • Oracle目录集成服务

  • Oracle Internet Directory（位于最底层，表示所有其他身份管理元素都使用该目录作为其基础）

Oracle身份管理基础结构与外部安全服务进行通信，外部安全服务位于基础结构附近，表示外部服务位于同一层。外部安全服务包括以下元素：

  • 访问管理

  • 目录服务

  • 供应服务
