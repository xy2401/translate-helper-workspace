afd_configure

目的

配置Oracle ASM筛选器驱动程序。

语法和说明

#asmcmd afd_configure [-d | -e] [-f]

下表列出了afd_configure命令的选项。

表10-30 afd_configure命令的选项

选项 描述

-d

禁用Oracle ASMFD筛选模式。

-e

启用Oracle ASMFD筛选模式。

-F

强制Oracle ASMFD配置。

afd_configure命令必须以root用户非交互方式运行。在节点上运行命令之前，必须在该节点上关闭Oracle Clusterware堆栈。运行afd_configure命令后，在每个节点上重新启动Oracle Clusterware。运行afd_configure ，启用默认过滤状态。

例

以下示例在运行该命令的节点上配置Oracle ASM筛选器驱动程序。示例中的$ORACLE_HOME指的是Oracle Grid Infrastructure主目录。

示例10-33使用afd_configure命令

＃$ ORACLE_HOME / bin / asmcmd afd_configure

afd_filter

目的

在给定磁盘路径上设置Oracle ASM筛选器驱动程序筛选模式。

语法和说明

afd_filter {-e | -d} [ disk_path ]

afd_filter命令的语法选项如下表所述。

表10-34 afd_filter命令的选项

选项 描述

-e

启用Oracle ASM筛选器驱动程序模式。

-d

禁用Oracle ASM筛选器驱动程序模式。

disk_path

指定磁盘的路径。

如果在未指定磁盘路径的情况下执行该命令，则在节点级别设置过滤。

例

以下示例使用afd_filter在指定的磁盘路径上启用Oracle ASM筛选器驱动程序筛选。

示例10-38使用afd_filter命令

ASMCMD [+]> afd_filter -e / dev / sdq

afd_label

目的

将Oracle ASM筛选器驱动程序（Oracle ASMFD）标签设置为指定的磁盘。

语法和说明

afd_label afd_label disk_path [--migrate | --rename] [--init]

afd_label命令的语法选项如表10-35所示 。

表10-35 afd_label命令的选项

选项 描述

afd_label

指定Oracle ASM筛选器驱动程序标签。

disk_path

指定应用标签的磁盘的路径。

- 迁移

指定将Oracle ASM筛选器驱动程序磁盘标签迁移到disk_path所有磁盘。

- 改名

指定标记先前标记的磁盘。

- 在里面

指定在初始化阶段设置Oracle ASMFD标签。如果已加载Oracle ASMFD，则不允许使用此选项。

有关Oracle ASM筛选器驱动程序的信息，请参阅“ 管理Oracle ASM筛选器驱动程序 ” 。

例

以下示例将Oracle ASM筛选器驱动程序标签设置为指定的磁盘。

例10-39使用afd_label命令

ASMCMD [+]> afd_label'disk0''/ dev / rdsk / mydisks / disk0'

afd_refresh

目的

语法和说明

下表列出了afd_refresh命令的选项。

表10-38 afd_refresh命令的选项

选项 描述

- 所有

指定群集范围刷新。

如果Oracle ASM筛选器驱动程序磁盘由root用户拥有，则必须以root身份运行afd_refresh --all ，以确保在V$ASM_DISK视图中显示准确的磁盘大小。

有关Oracle ASM筛选器驱动程序的信息，请参阅管理Oracle ASM筛选器驱动程序 。

例

以下示例使用afd_refresh刷新Oracle ASMFD磁盘

表10-39 afd_scan命令的选项

afd_unlabel

目的

清除指定的Oracle ASM筛选器驱动程序（Oracle ASMFD）标签。

语法和说明

afd_unlabel afd_label { afd_label | ' disk_path '} [-f] [--init]

表10-41中描述了afd_unlabel命令的语法选项。

表10-41 afd_unlabel命令的选项

选项 描述

afd_unlabel

指定要清除的现有Oracle ASM筛选器驱动程序标签。

' disk_path '

指定清除由现有磁盘路径确定的标签。如果在初始化阶段需要取消标记磁盘，则只能使用其磁盘路径来完成。

-F

如果磁盘是Oracle ASM成员磁盘并且没有挂起的磁盘打开，则强制清除标签。

- 在里面

指定在初始化阶段清除Oracle ASMFD标签。如果已加载Oracle ASMFD，则不允许使用此选项。

有关Oracle ASM筛选器驱动程序的信息，请参阅“ 管理Oracle ASM筛选器驱动程序 ” 。

例

以下示例清除现有的Oracle ASM筛选器驱动程序标签。

例10-45使用afd_unlabel命令

ASMCMD [+]> afd_unlabel'disk0'
