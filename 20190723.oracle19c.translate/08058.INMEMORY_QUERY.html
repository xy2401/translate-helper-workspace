1.144 INMEMORY_QUERY

INMEMORY_QUERY用于在会话或系统级别启用或禁用整个数据库的内存中查询。

属性 描述

参数类型

串

句法

INMEMORY_QUERY = {ENABLE |禁用}

默认值

ENABLE

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

Oracle RAC

所有实例都应使用相同的值。

当您想要使用和不使用内存中列存储（IM列存储）来测试工作负载时，此参数很有用。

默认情况下，当指定INMEMORY_SIZE参数时，此参数将启用对整个数据库的内存中查询。

如果要禁用内存中查询，请将此参数设置为DISABLE 。
