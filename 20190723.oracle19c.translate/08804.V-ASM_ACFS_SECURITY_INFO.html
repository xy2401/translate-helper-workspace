7.46 V $ ASM_ACFS_SECURITY_INFO

V$ASM_ACFS_SECURITY_INFO显示每个已安装的Oracle ACFS的安全信息。

柱 数据类型 描述

FS_NAME

VARCHAR2（1024）

文件系统挂载点（ V$ASM_FILESYSTEM外键）

VOL_DEVICE

VARCHAR2（256）

Oracle ADVM设备的名称

PREPARED_STATUS

VARCHAR2（7）

指示文件系统是否已准备好安全性（ YES ）或否（ NO ）; UNKNOWN 。

ENABLED_STATUS

VARCHAR2（8）

指示是否为文件系统ENABLED安全性（ ENABLED ）（ DISABLED ）; UNKNOWN 。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。
