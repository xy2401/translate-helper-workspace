使用ConfigToolAllCommands脚本进行安装后配置

您可以在安装Oracle软件后创建并运行响应文件配置。configToolAllCommands脚本要求用户创建第二个响应文件，其格式与用于安装产品的格式不同。

从Oracle Database 12c第2版（12.2）开始，不推荐使用configToolAllCommands脚本，并且可能在将来的发行版中不支持该脚本。

  • 关于安装后配置文件
    运行静默或响应文件安装时，在响应文件中提供有关服务器的信息，否则在图形用户界面安装期间手动提供这些信息。
  • 创建密码响应文件
    您可以创建密码响应文件，并将其与配置助手一起使用以执行静默安装。
  • 使用密码响应文件运行安装后配置
    完成此过程以使用configToolAllCommands脚本运行配置助手。
