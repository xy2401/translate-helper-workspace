说明sqlta_auto_start.gif的说明

这是EM Express中SQL页面调整结果的屏幕截图。页面左上角包含页面名称，SQL的调优结果。 “调整SQL结果”标题右侧是“保存”按钮。在最右侧，页面刷新字段显示上次刷新页面的时间。

在页面名称左下方是SQL Details部分。这包括以下字段和值：

  • 任务名称：SYS_AUTO_SQL_TUNING_TASK

  • 任务所有者：SYS

  • 执行名称：EXEC_22

  • SQL ID：ffy1dpzphwuud

  • 架构：DWH_TEST

“状态”部分右侧是“SQL文本”部分，其中包含SQL语句的文本。

Select Recommend表将显示在页面底部。在表格上方，出现“只应实施一项建议”的句子。在该句子下方显示“查看详细信息”和“实施”按在右侧，就在表格上方，将显示表格的“搜索”字段。

表中的列（从左到右）是Type，Findings和Benefit（％）。

表的第一行包含以下值：

  • 键入：SQL配置文件

  • 调查结果：此声明找到了一个可能更好的执行计划。

  • 效益（％）：93.14

表的第二行包含以下值：

  • 类型：索引

  • 结果：可以通过创建一个或多个索引来改进此语句的执行计划。

  • 效益（％）：70.88

在此屏幕截图中，选择表格中的第一行。
