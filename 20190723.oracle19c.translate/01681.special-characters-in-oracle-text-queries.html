Oracle Text Queries中的4个特殊字符

本章介绍可在文本查询中使用的特殊字符。此外，它还提供了Oracle Text视为保留字和字符的单词和字符列表。

本章涵盖以下主题：

  • 分组字符

  • 逃脱角色

  • 保留字和字符

4.1分组字符

分组字符通过在查询表达式中对查询术语和运算符进行分组来控制运算符优先级。分组字符如表4-1所示 。

表4-1查询术语分组的字符

分组字符 描述

（）

括号字符用于对字符之间的术语和运算符进行分组

[]

括号字符用于对字符之间的术语和运算符进行分组;但是，它们可以防止扩展操作员的穿透（模糊，soundex，干）。

一组术语和运算符的开头由一组分组字符中的空心字符表示。组的结尾由启动组的开放字符的适当关闭字符的出现表示。在两个字符之间，可能会出现其他组。

例如，左括号表示组的开头。遇到的第一个闭括号是组的结尾。在右括号之前遇到的任何打开括号都表示嵌套组。

4.2逃脱角色

查询在查询表达式中具有特殊含义的单词或符号，例如和＆或|累积 ，你必须逃脱它们。有两种方法可以在查询表达式中转义字符，如表4-2中所述 。

表4-2转义查询术语的字符

逃避角色 描述

{}

使用大括号来转义字符串或符号。在一组括号中的所有内容都被认为是转义序列的一部分。

使用大括号转义单个字符时，转义字符将成为查询中的单独标记。

\

使用反斜杠字符可以转义单个字符或符号。只有反斜杠后面的字符才会被转义。例如， blue \ -green的查询匹配蓝绿色和蓝绿色 。

在以下示例中，转义序列是必需的，因为每个表达式都包含Text运算符或保留符号：

'high \ -voltage''{high-voltage}''XY \＆Z''{XY＆Z}'

在第一个示例中，查询匹配高压或高压。

请注意，在第二个示例中，对XY和Z的查询将返回“XY Z”，“XY-Z”，“XY * Z”等，以及“XY和Z”。这是因为非字母字符被视为空格（因此XY和Z被视为'XY Z'）。要仅匹配XY和Z ，您必须声明＆作为printjoin。（但是，如果这样， XY和Z将不匹配'XY和Z'。）有关printjoins的更多信息，请参阅BASIC_LEXER 。

注意：

如果使用大括号来转义单词中的单个字符，则该字符将被转义，但该单词将被分为三个标记。例如，写为高{ - }电压的查询会搜索高电压 ，并且连字符的两侧都有空格。

查询转义字符

开括号{表示转义序列的开始，而闭括号}表示序列的结束。左大括号和右大括号之间的所有内容都是转义查询表达式的一部分（包括任何打开的大括号字符）。要在转义的查询表达式中包含近括号字符，请使用}} 。要转义反斜杠转义字符，请使用\\ 。
