Description of the illustration racad_vm_038.eps

This illustration shows a four-node cluster organized into two server pools—sales and backoffice—plus the default free server pool. The sales and backoffice server pools each contain two nodes. The values of the MIN_SIZE and MAX_SIZE server pool attributes are 2 and 3, respectively, for the sales server pool, and the value of its IMPORTANCE server pool attribute is 10. The values of the MIN_SIZE and MAX_SIZE server pool attributes are 2 and 2, respectively, for the backoffice server pool, and the value of its IMPORTANCE server pool attribute is 5.

A database named dbapps runs in the both server pools, offering the orderentry and billing services in the sales server pool, and the erp and reports services in the backoffice server pool.
