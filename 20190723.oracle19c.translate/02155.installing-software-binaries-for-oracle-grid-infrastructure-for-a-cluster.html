为群集安装Oracle Grid Infrastructure的软件二进制文件

您可以一次在多个节点上为群集软件安装Oracle Grid Infrastructure软件二进制文件。

使用此过程为集群软件安装Oracle Grid Infrastructure：

  1. 下载Grid主页图像文件。
  2. 运行gridSetup.sh命令并选择“配置选项”作为“仅设置软件” 。

    注意：

    您可以使用带有-applyRU和-applyOneOffs标志的gridSetup.sh命令在Oracle Grid Infrastructure安装或升级期间安装版本更新（RU）和一次性修补程序。
  3. 通过在安装程序屏幕中提供信息以响应您的配置选择，在一个或多个节点上完成Oracle Grid Infrastructure软件的安装。您可以一次在多个节点上安装Oracle Grid Infrastructure软件。
  4. 配置软件后，在出现提示时，在所有节点上运行orainstRoot.sh脚本。
  5. 在所有节点上， root.sh脚本输出提供有关如何继续的信息，具体取决于您计划在此安装中完成的配置。记下这些信息。
  6. 确保您已完成所有存储和服务器预安装要求。
  7. 验证所有群集节点是否满足安装要求：
    runcluvfy.sh stage -pre crsinst -n node_list
  8. 使用Oracle Universal Installer（OUI）配置向导或响应文件配置集群。
