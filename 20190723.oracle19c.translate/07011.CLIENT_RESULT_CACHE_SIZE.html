1.37 CLIENT_RESULT_CACHE_SIZE

CLIENT_RESULT_CACHE_SIZE指定客户端每进程结果集缓存的最大大小（以字节为单位）。

属性 描述

参数类型

大整数

句法

CLIENT_RESULT_CACHE_SIZE = integer [K | M | G]

默认值

0

可修改

没有

可在PDB中修改

没有

价值范围

0到操作系统相关

基本

没有

所有OCI客户端进程都继承此最大大小。设置非零值可启用客户端查询缓存功能。这可以由客户端配置参数OCI_RESULT_CACHE_MAX_SIZE 。
