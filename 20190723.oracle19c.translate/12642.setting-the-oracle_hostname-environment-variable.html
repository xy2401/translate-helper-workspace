设置ORACLE_HOSTNAME环境变量

运行此示例中显示的命令作为Oracle用户帐户以设置ORACLE_HOSTNAME环境变量。

以下示例显示了在Oracle用户帐户上运行的命令，用于设置ORACLE_HOSTNAME环境变量。在此示例中，标准主机名是somehost.example.com 。

Bourne，Bash或Korn Shell

$ ORACLE_HOSTNAME = somehost.example.com $ export ORACLE_HOSTNAME

C Shell

％setenv ORACLE_HOSTNAME somehost.example.com
