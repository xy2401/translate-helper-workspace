7.48 V $ ASM_ACFSREPL

V$ASM_ACFSREPL显示为复制而初始化的Oracle ACFS文件系统的信息。

柱 数据类型 描述

FSNAME

VARCHAR2（1024）

文件系统挂载点（ V$ASM_FILESYSTEM外键）

VOLDEV

VARCHAR2（256）

Oracle ADVM设备的名称

现场

VARCHAR2（7）

复制站点角色：

  • PRIMARY - 文件系统初始化为主要文件系统

  • STANDBY - 文件系统初始化为备用数据库

落后

VARCHAR2（128）

待机时间在主要时间之后，以小时为单位：分钟：秒（仅限主要）

状态

VARCHAR2（12）

复制主要或备用状态：

  • ONLINE - 初始化复制并处理实时更改

  • INITIALIZING - 复制仍在处理复制开始之前存在的主文件上的文件

  • PAUSED - 主要不是将日志发送到备用数据库，或者备用数据库未应用已记录的更改

INITSTATUS

VARCHAR2（11）

复制主要或备用初始化状态：

  • NONE - 复制ONLINE但未初始化

  • DIRECTORIES - 目录正在初始化

  • FILES - 正在初始化文件

DIRSSCAN

数

扫描和记录的目录数（仅限主要）

百分

数

复制开始之前存在的已初始化文件的百分比（仅限主要文件）

LASTSYNC

日期

最后一次在备用站点上应用的时间（仅限备用）

CRSRUN

数

当前正在运行的Cluster Ready Services复制守护程序数

CRSTOTAL

数

预期正在运行的Cluster Ready Services复制守护程序的数量

PMNTPT

VARCHAR2（1024）

复制主站点装入点

SMNTPT

VARCHAR2（1024）

复制备用站点挂载点

PSVCNAME

VARCHAR2（1024）

复制主站点服务名称

SSVCNAME

VARCHAR2（1024）

复制备用站点服务名称

PHOST

VARCHAR2（1024）

复制主群集主机名

SHOST

VARCHAR2（1024）

复制备用群集主机名

REMALIAS

VARCHAR2（4096）

复制远程别名

TAGS

数

如果复制标记文件，则为TRUE （ 1 ）

压缩

VARCHAR2（3）

复制日志压缩状态：

  • OFF - 未压缩复制日志

  • ON - 正在压缩复制日志

DBGLVL

数

复制跟踪日志调试级别

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。

注意：

此视图仅包含Oracle Database 12 c第2版（12.2.0.1）之前的Oracle ASM版本的记录。要显示Oracle Database 12 c第2版（12.2.0.1）或更高版本的Oracle ACFS复制信息，请使用acfsutil repl info命令。
