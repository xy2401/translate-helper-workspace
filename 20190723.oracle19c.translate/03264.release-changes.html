“Oracle Data Mining用户指南”的此发行版中的更改

Oracle Data Mining用户指南的此版本中的更改。

Oracle Data Mining用户指南是此版本中的新增功能

  • 本指南是发行12c中新。Oracle Data Mining用户指南取代了以前版本中提供的两本手册： Oracle Data Mining管理员指南和Oracle Data Mining Application Developer's Guide 。

  • 有关Oracle Data Mining的数据库管理的信息现已合并到Oracle Data Mining的管理任务中 。本指南的其余章节专门用于应用程序开发。

  • 有关数据挖掘示例程序的信息现在位于数据挖掘示例程序中 。

Oracle Data Mining 18c中的更改

Oracle Data Mining用户指南（适用于18c）中记录了以下更改。

18c的新功能

以下内容是此版本中的新增内容：

模型视图

  • 添加了以下新模型详细信息视图。模型详细信息视图优于GET*函数。

    • 随机森林。请参阅随机森林的模型详图视图 。

    • 神经网络。参见神经网络的模型细节视图 。

    • CUR矩阵分解。请参阅CUR矩阵分解的模型详细信息视图 。

    • 指数平滑。请参阅指数平滑模型的模型详细信息视图 。

  • 将“ Oracle数据库PL / SQL包和类型参考 ”中的“ GET_MODEL_DETAILS_GLOBAL函数”主题中的信息移动到“ 模型详细信息视图”下的相应算法。

    以下模型详细信息视图已修改：

    • 关联规则的模型细节视图

    • 显式语义分析的模型细节视图

    • 广义线性模型的模型细节视图

    • k-Means的模型细节视图

    • 非负矩阵分解的模型细节视图

    • 奇异值分解的模型细节观

    • 支持向量机的模型细节视图

导出和导入序列化模型

新添加的EXPORT_SERMODEL和IMPORT_SERMODEL过程。

请参见导出和导入序列化模型 。

不推荐的功能

此版本中不推荐使用以下功能，并且可能在其他版本中不支持这些功能。有关此版本中已弃用功能的完整列表，请参阅“ Oracle数据库升级指南” 。

  • *GET_MODEL_DETAILS已弃用，已替换为“ 模型详细信息视图” 。请参见模型详图视图 。

不受支持的功能

有关此版本中不支持的功能的完整列表，请参阅“ Oracle数据库升级指南” 。
