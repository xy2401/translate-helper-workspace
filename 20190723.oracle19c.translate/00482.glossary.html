词汇表

原子空对象

值为NULL的对象称为atomically null。原子空对象不同于对其所有属性具有空值的对象。

二进制大对象（BLOB）

一种大对象数据类型，其值由原始二进制数据组成。

字符大对象（CLOB）

大对象（LOB）数据类型，其值由对应于数据库字符集的字符数据组成。

列对象

作为关系数据库表的列存储的对象（与对象表相对）。列对象也可以是另一个对象的属性。列对象也称为存储的内联对象或嵌入对象。

嵌入对象属性

列对象的属性。

动态方法调度

在使用覆盖的同一方法的多个实现时，在运行时调度到最近的方法实现的方法调用。

叶级标量对象

不是集合且不由其他类型组成的对象。

叶级标量属性

叶级标量对象的属性 。

文字调用

调用，其中所有参数都是文字或文字方法的调用。参数不能是变量。

物化观点

包含查询及其结果的视图。

多级集合类型

集合类型，其元素是直接或间接的集合类型。

multiset运营商

将两个嵌套表的元素组合到一个嵌套表中的运算符。

嵌套表

相同数据类型的无序数据元素集。

正常化

通过将数据分成多个表来删除数据冗余的过程。

对象列

一列用户定义类型或抽象数据类型（ADT）。

对象标识符

行对象的标识符，可以是系统生成的（默认），也可以是使用CREATE TABLE语句的主键。

对象实例

对象类型的实例。也称为对象。

对象表

一个表，其中每行代表一个对象。请参见行对象 。

对象类型

对象实例的类型。它类似于具有方法的记录。对象类型是用户定义的。

外桌

包含嵌套表的表。

钉住

抓取。

PL / SQL

SQL的Oracle过程语言扩展。

多态性

允许一般使用相同的接口处理数据类型和方法。在多态覆盖中，子类型重新定义了它们继承的方法。在多态重载中，可能存在具有不同参数的相同方法的若干版本。

首要的关键

在表的PRIMARY KEY约束中指定的列或列集。

REF

一种Oracle内置数据类型，它封装对指定对象类型的行对象的引用。

行对象

存储在对象表中的完整行中的对象 。

专业

将新属性或方法添加到父类型不具有的子类型，或更改方法或方法的实现。

存储过程

一个PL / SQL块，存储在数据库中，可以从应用程序执行。

可替代性

如果超类型的一个子类型可以在声明类型为超类型的变量或列中替换或替代它，则该超类型是可替代的。

类型进化

修改类型的子类型，通常使用ALTER语句。

VARRAY

一组有序的数据元素，具有相同的数据类型或声明的数据类型的子类型。
