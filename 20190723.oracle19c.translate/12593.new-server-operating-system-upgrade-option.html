新服务器操作系统升级选项

您可以通过在服务器上安装新操作系统，然后手动迁移数据库或使用导出/导入方法来升级操作系统。

注意：

确认服务器操作系统受支持，并且操作系统的内核和程序包要求满足或超过要迁移到的Oracle数据库发行版的最低要求。

手动，命令行复制，用于迁移数据和升级Oracle数据库

您可以将文件复制到新服务器并手动升级。如果使用此过程，则无法使用Oracle数据库升级助手。但是，如果遇到升级问题，则可以还原到现有数据库。

  1. 将数据库文件从运行以前操作系统的计算机复制到运行新操作系统的计算机。

  2. 在运行新操作系统的计算机上重新创建控制文件。

  3. 使用命令行脚本和实用程序手动升级数据库。

迁移数据和升级Oracle数据库的导出/导入方法

您可以在新服务器上安装操作系统，在新服务器上安装新的Oracle数据库版本，然后使用Oracle Data Pump导出和导入实用程序将数据副本从当前数据库迁移到新版本中的新数据库。建议使用数据泵导出和导入以获得更高的性能并确保支持新数据类型。
