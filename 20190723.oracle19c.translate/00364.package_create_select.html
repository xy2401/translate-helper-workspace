这是“选择包”页面的部分图像。在页面的左上角，显示所选问题的问题键。在右上角有两个按钮：取消和确定。

页面的其余部分提供了两个选项，一个在另一个之上：“创建新包”或“从现有包中选择”。如果选择“创建新包”，则填写此选项下面的以下字段：包名称和包描述。已在“包名称”字段中输入了默认值。如果选择“从现有包中选择”选项，则必须从现有包的表中选择一个条目。表列为：选择，名称，状态，类型，描述，主要问题键和创建。表中有一行，其类型为Main。Created列是一个时间戳。
