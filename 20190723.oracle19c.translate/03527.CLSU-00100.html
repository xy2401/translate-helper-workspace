18 CLSU-00100至CLSU-00915

CLSU-00100：操作系统功能： string失败，错误数据： string

原因：对操作系统相关服务或功能的调用返回错误指示。该消息包括函数的名称和返回的错误数据。后者因平台而异，但通常是数字;在大多数平台上，失败的呼叫后，它是C“errno”的值。

操作：此错误通常伴随有描述受故障影响的操作的其他（更高级别）消息。它还可以包括提供附加错误细节的消息101,103和104中的一个或多个。应检查所有消息以评估错误，该错误可能具有非常普通的原因和更正，例如由于提供的名称拼写错误而导致输入文件无法打开。

CLSU-00101：操作系统错误消息： string

原因：当操作系统相关的错误数据可以转换为文本消息时，此消息伴随上面的消息100。在大多数Oracle平台上，该消息是消息100中报告的C“errno”值的文本表示。

行动：见消息100。

CLSU-00102：操作系统界面检测到错误

原因：当在操作系统特定接口上检测到“逻辑”错误条件（而不是操作系统函数调用失败）时，将发出此消息而不是消息100-101。通过附带的消息进一步描述错误条件，包括103和104。

操作：此错误通常伴随有描述受故障影响的操作的其他（更高级别）消息。在大多数情况下，它还将包括消息103和104，提供有关错误的其他详细信息。应检查所有消息以评估错误，该错误可能具有非常普通的原因和更正，例如缺少必需的环境变量。

CLSU-00103：错误位置： string

原因：此消息伴随消息100,102或105，并标识Oracle程序代码中遇到错误的位置。

操作：请参阅消息100,102或105。仅当将错误情况报告给Oracle作为潜在的代码错误时，此信息通常才有用。

CLSU-00104：其他错误信息： string

原因：此消息伴随消息100,102或105，并提供与错误情况相关的其他信息。单个错误可能包括多行附加信息。

操作：请参阅消息100,102或105。

CLSU-00105：操作系统界面报告内部故障

原因：此消息表明Oracle代码中的操作系统相关接口检测到内部损坏或内部程序失败的其他一些证据。

操作：此消息可能伴有其他（更高级别）消息，指示受故障影响的产品操作。它还可以伴随消息103和104中的任何一个或两者。报告此情况的所有情况都应提交给Oracle支持部门以进行解决。

CLSU-00106：尝试了不正确的操作系统错误显示

原因：在处理错误条件期间，尝试格式化或显示与操作系统相关的错误数据，但发现错误数据结构表示“无错误”。

操作：应将此错误报告给Oracle Support以进行解决。

CLSU-00107：操作系统功能： string ;错误数据失败： string ;在位置： string

原因：对依赖于操作系统的服务或函数的调用返回错误。该消息包括函数的名称，返回的错误数据和错误位置。错误数据因平台而异，但通常为数字。在大多数平台上，失败的呼叫后它是C“errno”的值。

操作：此错误通常伴随有描述受故障影响的操作的其他（更高级别）消息。它还可以包括提供附加错误细节的消息101和104中的一个或多个。应检查所有消息以评估错误，该错误可能具有非常普通的原因和更正，例如由于提供的名称拼写错误而导致输入文件无法打开。

CLSU-00110：程序无法在位置string处分配string字节的内存。

原因：尝试分配内存意外失败。没有足够的内存或根本没有可用的内存。

操作：联系Oracle支持服务。
