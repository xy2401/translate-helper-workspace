完成失败的安装和升级

了解在某些节点上安装或升级失败后如何将节点加入群集。

如果某些节点上的Oracle Grid Infrastructure安装或升级失败，并且安装或升级仅在群集中的成功节点完成，则请按照此过程将故障节点添加到群集。
  1. 从发生故障的节点中删除Oracle Grid Infrastructure软件：
    Grid_home / deinstall / deinstall -local
  2. 以root用户身份，从安装了Oracle Clusterware的节点，使用delete node命令删除发生故障的节点：
    Grid_home / bin / crsctl删除节点-n node_name

    node_name是要删除的节点。

  3. 运行Oracle Grid Infrastructure安装向导，然后按照向导中的步骤添加节点：
    Grid_home /gridSetup.sh
节点将添加到群集中。
