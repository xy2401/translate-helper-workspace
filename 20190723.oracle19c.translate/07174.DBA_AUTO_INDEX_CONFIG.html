4.119 DBA_AUTO_INDEX_CONFIG

DBA_AUTO_INDEX_CONFIG显示自动索引的当前配置参数设置。

您可以使用DBMS_AUTO_INDEX.CONFIGURE过程设置自动索引配置参数。

柱 数据类型 空值 描述

PARAMETER_NAME

VARCHAR2（128）

不是NULL

配置参数的名称

PARAMETER_VALUE

VARCHAR2（4000）

配置参数的值

最后修改

TIMESTAMP（6）

上次修改参数值的时间

MODIFIED_BY

VARCHAR2（128）

最后修改参数值的用户

注意：

从Oracle Database 19c版本19.1开始提供此视图。
