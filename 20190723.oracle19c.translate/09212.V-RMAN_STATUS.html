8.179 V $ RMAN_STATUS

V$RMAN_STATUS显示已完成和正在进行的RMAN作业。 对于正在进行的作业，此视图显示进度和状态。正在进行的作业仅存储在内存中，而已完成的作业存储在控制文件中。
柱 数据类型 描述

SID

数

正在运行此RMAN操作的会话的会话ID

RECID

数

控制文件中行的记录ID

邮票

数

行的时间戳（ RECID + STAMP是唯一的）

PARENT_RECID

数

此行的父行的记录ID（对应的V$RMAN_STATUS行， ROW_LEVEL = ROW_LEVEL - 1 ）

PARENT_STAMP

数

此行的父行的时间戳（对应的V$RMAN_STATUS行， ROW_LEVEL = ROW_LEVEL - 1 ）

SESSION_RECID

数

会话的记录ID（对应的V $ RMAN_STATUS行， ROW_LEVEL = 0 ）

SESSION_STAMP

数

会话的时间戳（对应的V $ RMAN_STATUS行， ROW_LEVEL = 0 ）

ROW_LEVEL

数

行的级别。会话级别为0 。

ROW_TYPE

VARCHAR2（19）

行的类型：

  • SESSION

  • 命令

  • 回收操作

command_id的

VARCHAR2（33）

由RMAN SET COMMAND ID命令SET COMMAND ID 。如果未设置，则RMAN将创建唯一编号。

操作

VARCHAR2（33）

此行说明的执行中的命令名称

状态

VARCHAR2（23）

操作状况：

  • RUNNING

  • 运行警告

  • 跑步与错误

  • 已完成

  • 完成了警告

  • 完成错误

  • 失败

MBYTES_PROCESSED

数

完成工作的百分比;如果不适用于该操作，则返回null

开始时间

日期

工作的开始时间

时间结束

日期

工作结束时间

INPUT_BYTES

数

读取的输入字节数

OUTPUT_BYTES

数

写入的输出字节数

优化

VARCHAR2（3）

是，如果在备份作业期间应用了备份优化。否则，不。

OBJECT_TYPE

VARCHAR2（13）

标识备份的对象类型

OUTPUT_DEVICE_TYPE

VARCHAR2（17）

DISK ， SBT_TAPE或* 。*表示输出已写入多个设备类型。

OSB_ALLOCATED

VARCHAR2（3）

值为YES表示在V$RMAN_STATUS视图标识的指定操作期间分配了Oracle Secure Backup通道。

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
