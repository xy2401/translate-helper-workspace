19 CLSW-00001至CLSW-01000

CLSW-00001：群集钱包操作失败

原因：这是群集钱包的一般内部错误。

操作：联系Oracle客户支持。

CLSW-00002：堆内存不足。

原因：程序耗尽了堆内存。

操作：当有更多内存可用时重试操作。

CLSW-00003：群集注册表（OCR）层出错。 [ number ] [ string ]

原因：执行群集钱包操作时群集注册表层中发生错误。

操作：通过使用ocrcheck和ocrdump检查ocr错误（oerr proc ocrerror ）确保群集注册表的正确操作，然后重试该操作。

CLSW-00004：网络安全层出错。 [ number ] [ string ]

原因：执行群集钱包操作时网络安全层出错。

操作：检查网络安全错误（oerr ora nzerr ），并采取纠正措施以确保正确操作。

CLSW-00005：无法执行群集钱包操作，因为其中一个参数为NULL。

原因：程序尝试使用NULL参数进行群集钱包操作。

操作：检查参数，然后重试该操作。

CLSW-00006：无法执行群集钱包操作，因为上下文无效。

原因：传递的群集钱包上下文无效。

操作：初始化群集钱包并重试该操作。

CLSW-00007：无法执行群集钱包操作，因为其中一个参数无效。 [ number ] [ string ]

原因：程序尝试使用无效参数进行群集钱包操作。

操作：检查参数，然后重试该操作。

CLSW-00008：要创建的群集钱包已存在。

原因：该程序试图创建已存在的群集钱包。

操作：取消设置CLSW_FLAGS_WALLET_CREATE后重试该操作

CLSW-00009：要操作的群集钱包不存在。

原因：程序试图访问不存在的群集钱包。

操作：检查群集钱包是否存在并尝试操作

CLSW-00010：NLS服务出错。

原因：执行群集钱包操作时NLS服务发生错误。

操作：检查消息文件是否存在于$ ORACLE_HOME / has / mesg / clsw * .msb中

CLSW-00011：无法执行群集钱包操作，因为存储在注册表中的数据无效。

原因：存储在注册表中的数据无法转换为钱包。

操作：删除与钱包关联的注册表项并重新创建钱包。

CLSW-00012：用户无权执行群集钱包操作。 [ number ] [ string ]

原因：程序尝试对其没有权限的群集钱包执行操作。

操作：检查程序是否以正确的用户身份运行。

CLSW-00013：集群件操作系统（SCLS）服务出错。 [ string ]

原因：执行群集钱包操作时SCLS服务中发生错误。

操作：检查Unix上的NIS / LDAP和Windows上的Active Directory服务是否已正确配置。

CLSW-00014：此操作系统不支持电子钱包类型“ string ”。

原因：程序尝试使用当前操作系统不支持的钱包类型进行群集钱包操作。

行动：这是一个内部错误。联系oracle支持服务。

CLSW-00015：别名“ string ”已存在于钱包类型“ string ”中。

原因：向钱包添加指定别名的请求失败，因为别名已经存在于钱包中。

操作：添加钱包中尚不存在的别名。可以使用命令'crsctl modify wallet'修改已存在的别名的密码。

CLSW-00016：别名“ string ”在钱包式“未找到string ”。

原因：修改钱包中指定别名的请求失败，因为钱包中不存在别名。

操作：确保别名存在于指定的钱包类型中。可以使用命令'crsctl add wallet'将新别名添加到钱包类型。

CLSW-00017：不允许用户' string '创建' string '类型的钱包。

原因：尝试创建指定类型的钱包被拒绝，因为钱包不应由指定的用户创建。

操作：以正确的用户身份运行程序。“CVUDB”类型的钱包必须由Oracle数据库主目录创建。

CLSW-01000：输入密码：

原因：这是用户输入密码的提示。

操作：输入密码。
