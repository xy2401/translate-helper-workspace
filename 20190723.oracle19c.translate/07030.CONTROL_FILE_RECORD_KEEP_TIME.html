1.51 CONTROL_FILE_RECORD_KEEP_TIME

CONTROL_FILE_RECORD_KEEP_TIME指定可以重用控制文件中可重用记录之前的最小天数。

属性 描述

参数类型

整数

默认值

7 （天）

可修改

改变系统

可在PDB中修改

没有

价值范围

0到365 （天）

基本

没有

如果必须将新记录添加到可重复使用的部分并且最旧的记录未足够老化，则记录部分会扩展。如果此参数设置为0，则可重用节从不展开，并根据需要重复使用记录。

如果控制文件中可重用记录的数量超过循环重用记录限制UB4MAXVAL ，则即使尚未过去CONTROL_FILE_RECORD_KEEP_TIME也将覆盖可重用记录。UB4MAXVAL在定义oratypes 。 h头文件，位于public目录中。其值可能因您使用的操作系统而异。

注意：

此参数仅适用于循环可重用的控制文件中的记录（例如归档日志记录和各种备份记录）。它不适用于数据文件，表空间和重做线程记录等记录，除非从表空间中删除相应的对象，否则永远不会重复使用这些记录。
