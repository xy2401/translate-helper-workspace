工具支持

Oracle提供了许多实用程序来促进您的SQL开发过程：

  • Oracle SQL Developer是一个图形工具，可用于浏览，创建，编辑和删除（删除）数据库对象，编辑和调试PL / SQL代码，运行SQL语句和脚本，操作和导出数据以及创建和查看报告。使用SQL Developer，您可以使用标准Oracle数据库身份验证连接到任何目标Oracle数据库模式。连接后，您可以对数据库中的对象执行操作。您还可以连接到选定的第三方（非Oracle）数据库（例如MySQL，Microsoft SQL Server和Microsoft Access）的模式，查看这些数据库中的元数据和数据，以及将这些数据库迁移到Oracle。

  • SQL * Plus是一个交互式批处理查询工具，随每个Oracle数据库服务器或客户端安装一起安装。它有一个命令行用户界面和一个名为i SQL * Plus的基于Web的用户界面。

  • Oracle JDeveloper是一个多平台集成开发环境，支持Java，Web服务和SQL的完整生命周期。它提供了用于执行和调优SQL语句的图形界面以及可视化架构图解器（数据库建模器）。它还支持编辑，编译和调试PL / SQL应用程序。

  • Oracle Application Express是用于开发和部署与数据库相关的Web应用程序的托管环境。SQL Workshop是Oracle Application Express的一个组件，允许您从Web浏览器查看和管理数据库对象。SQL Workshop提供对SQL命令处理器和SQL脚本存储库的快速访问。

Oracle Call Interface和Oracle预编译器允许您在过程编程语言中嵌入标准SQL语句。

  • Oracle调用接口（OCI）允许您在C程序中嵌入SQL语句。

  • Oracle预编译器Pro * C / C ++和Pro * COBOL解释了嵌入式SQL语句，并将它们分别转换为C / C ++和COBOL编译器可以理解的语句。

大多数（但不是全部）Oracle工具也支持Oracle SQL的所有功能。该参考资料描述了SQL的完整功能。如果您使用的Oracle工具不支持此完整功能，那么您可以在描述该工具的手册中找到对限制的讨论，例如SQL * Plus用户指南和参考 。
