7.75 V $ ASM_USER

V$ASM_USER显示已连接数据库实例和文件所有者的有效操作系统用户名。

柱 数据类型 描述

GROUP_NUMBER

数

Oracle ASM磁盘组编号

USER_NUMBER

数

Oracle ASM内部唯一用户编号

COMPOUND_INDEX

数

一个32位数字，由高8位的磁盘组编号和低位24位的用户编号组成（用于有效访问视图）

INCARNATION

数

用户的化身编号

CLUSTER_ID

VARCHAR2（128）

Oracle ASM集群ID。 CLUSTER_ID和OS_NAME对可用于唯一标识用户。

OS_ID

VARCHAR2（128）

操作系统用户标识

OS_NAME

VARCHAR2（33）

操作系统用户名

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

对于此视图，该值始终为0 。
