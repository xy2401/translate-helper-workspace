2准备升级Oracle数据库

完成预升级任务和检查以帮助您完成成功升级。

本章提供升级前任务的信息和过程，包括规划升级，数据收集，测试，安装新的Oracle软件以进行升级，使用并行升级实用程序执行升级以及执行其他检查和任务。

  • 关于以只读模式配置Oracle Home
    从Oracle Database 18c开始，您可以通过将只读Oracle主目录部署为软件映像来简化跨多个数据库服务器的软件修补和批量部署。
  • 关于基于映像的Oracle数据库安装
    从Oracle Database 18c开始，通过基于映像的安装简化了Oracle数据库软件的安装和配置。
  • 准备Oracle数据库升级的任务
    执行这些任务以准备升级。
  • 升级前信息工具清单
    要获取在物理服务器或虚拟机上升级Oracle数据库之前必须完成的任务清单，请运行升级前信息工具（ preupgrade.jar ）。
  • 为单个实例安装新的Oracle数据库软件
    使用此过程概述可帮助您为单个实例部署安装新Oracle Database版本的软件。
  • 安装适用于Oracle RAC的新Oracle数据库软件
    使用此过程概述可帮助您为Oracle RAC .deployment安装新Oracle Database版本的软件。
  • 在启动Oracle数据库升级之前完成的数据库准备任务
    确保在开始Oracle数据库升级之前已完成这些数据库准备任务。
  • 使用Oracle Data Guard准备数据库滚动升级
    如果将数据库升级助手（DBUA）与Oracle Data Guard一起使用以执行滚动升级，则必须在开始升级之前移动Data Guard代理配置文件。
  • 准备新的Oracle主页进行升级
    备份要升级的数据库后，在新位置准备新的Oracle主目录。将新Oracle Database发行版的软件安装到新位置。
  • 在Windows上准备Oracle Home的先决条件
    在Microsoft Windows平台上升级Oracle数据库之前，您的系统必须满足这些要求。
  • 使用Oracle数据库的升级前信息工具
    查看这些主题以了解并使用升级前信息工具（ preupgrade.jar ）。
  • 测试Oracle数据库的升级过程
    您的Oracle数据库升级测试计划应包括这些测试过程。
  • 升级使用Oracle Label Security和Oracle Database Vault的数据库的要求
    在使用Oracle Label Security或Oracle Database Vault开始使用数据库进行升级之前，您必须完成这些任务。
