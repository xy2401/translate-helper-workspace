前言

“ Oracle Clusterware管理和部署指南”介绍了Oracle Clusterware体系结构，并提供了该产品的概述。本书还介绍了Oracle Clusterware的管理和部署主题。

除非另有说明，否则本手册中的信息适用于Oracle Clusterware，因为它在所有平台上运行。此外，本手册的内容补充了其他Oracle文档中出现的Oracle单实例数据库的管理和部署主题。必要时，本手册涉及特定于平台的文档。本前言包含以下主题：

  • 听众

  • 文档可访问性

  • 相关文件

  • 约定

听众

“ Oracle Clusterware管理和部署指南”适用于执行以下任务的数据库管理员，网络管理员和系统管理员：

  • 安装和配置Oracle Real Application Clusters（Oracle RAC）数据库

  • 管理和管理Oracle RAC数据库

  • 管理使用Oracle RAC的集群和网络并对其进行故障排除

文档可访问性

有关Oracle对辅助功能的承诺的信息，请访问Oracle辅助功能计划网站http://www.oracle.com/pls/topic/lookup?ctx=acc&id=docacc 。

访问Oracle支持

已购买支持的Oracle客户可通过My Oracle Support获得电子支持。有关信息，请访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=info或访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=trs听力受损。
