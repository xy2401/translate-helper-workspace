DROP LIBRARY

目的

使用DROP LIBRARY语句从数据库中删除外部过程库。

也可以看看：

有关创建库的信息，请参阅CREATE LIBRARY

先决条件

您必须具有DROP ANY LIBRARY系统特权。

句法

drop_library :: =


说明drop_library.eps的描述

语义

LIBRARY_NAME

指定要删除的外部过程库的名称。

例子

删除库：示例

以下语句删除了ext_lib库：

DROP LIBRARY ext_lib;
