4.12 DBA_ADVISOR_DEF_PARAMETERS

DBA_ADVISOR_DEF_PARAMETERS显示数据库中的所有默认任务参数及其当前值。

创建任务或对象时，参数及其值将复制到私有参数表中。

柱 数据类型 空值 描述

ADVISOR_NAME

VARCHAR2（128）

不是NULL

支持该参数的顾问程序的名称

PARAMETER_NAME

VARCHAR2（128）

不是NULL

参数名称

PARAMETER_VALUE

VARCHAR2（4000）

不是NULL

参数的值。数字参数值将转换为等效的字符串。

可能的关键字值：

  • 所有

  • 无限

  • 没用过

PARAMETER_TYPE

VARCHAR2（10）

参数的数据类型：

  • NUMBER - 数字值

  • STRING - 字符串值。如果字符串包含特殊字符，则它将用单引号括起来。

  • STRINGLIST - 以逗号分隔的字符串元素列表。如果字符串元素包含逗号或其他特殊字符，则该元素将用单引号括起来。

  • TABLE - 单表引用。引用将包含模式名称，后跟可选的表名称。

    如果省略表名或字符为% ，则表名称将被解释为通配符。支持SQL引用的标识符。

  • TABLELIST - 一个或多个以逗号分隔的表引用的列表。引用将包含模式名称，后跟可选的表名称。

    如果省略表名或字符为% ，则表名称将被解释为通配符。支持SQL引用的标识符。

IS_DEFAULT

VARCHAR2（1）

指示参数值是否设置为顾问程序的默认值（ Y ）或不是（ N ）

IS_OUTPUT

VARCHAR2（1）

指示任务执行过程是否设置参数值（ Y ）（ N ）

IS_MODIFIABLE_ANYTIME

VARCHAR2（1）

指示当任务未处于初始状态（ Y ）或不是（ N ）时是否可以修改参数值

IS_SYSTEM_TASK_ONLY

VARCHAR2（1）

指示任务是否是系统任务（ Y ）（ N ）

描述

VARCHAR2（4000）

参数的可选描述

EXECUTION_TYPE

VARCHAR2（128）

上次执行的类型。对于单次执行任务，此信息是可选的。
