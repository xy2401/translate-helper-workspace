检查共享内存资源控制

使用prctl命令对与系统上的活动进程，任务或项目关联的资源控制进行运行时询问和修改。

要查看项目和系统范围的project.max-shm-memory集的当前值：

#prctl -n project.max-shm-memory -i project default

default是通过运行id -p命令获得的项目ID。

例如，要将project.max-shm-memory的设置更改为6 GB，而不重启项目默认值：
prctl -n project.max-shm-memory -v 6gb -r -i项目默认值
