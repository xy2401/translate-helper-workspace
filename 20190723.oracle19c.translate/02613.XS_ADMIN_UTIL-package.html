XS_ADMIN_UTIL包

XS_ADMIN_UTIL包中包含其他包使用的辅助子程序。

本节包括以下主题：

  • 安全模型

  • 常量

  • 对象类型，构造函数，同义词和授权

  • XS_ADMIN_UTIL子程序摘要

安全模型

XS_ADMIN_UTIL包在SYS架构中创建。呼叫者拥有该套餐的调用者权利。需要SYS权限才能向用户或角色授予或撤消Real Application Security系统特权。

常量

下列常量定义删除选项：

DEFAULT_OPTION常量PLS_INTEGER：= 1; CASCADE_OPTION常数PLS_INTEGER：= 2; ALLOW_INCONSISTENCIES_OPTION常数PLS_INTEGER：= 3;

以下常量定义主体的类型：

PTYPE_XS CONSTANT PLS_INTEGER：= 1; PTYPE_DB CONSTANT PLS_INTEGER：= 2; PTYPE_DN CONSTANT PLS_INTEGER：= 3; PTYPE_EXTERNAL CONSTANT PLS_INTEGER：= 4;

为用户或角色授予Real Application Security系统特权。

本节介绍以下XS_ADMIN_UTIL子程序：

GRANT_SYSTEM_PRIVILEGE过程

GRANT_SYSTEM_PRIVILEGE过程用于向用户或角色授予Real Application Security系统特权或架构特权。只有SYS或具有GRANT ANY PRIVILEGE权限的用户才能执行此操作。

审计操作AUDIT_GRANT_PRIVILEGE审计所有GRANT_SYSTEM_PRIVILEGE调用以授予系统特权或架构特权。

句法

XS_ADMIN_UTIL.GRANT_SYSTEM_PRIVILEGE（priv_name IN VARCHAR2，user_name IN VARCHAR2，user_type IN PLS_INTEGER：= XS_ADMIN_UTIL.PTYPE_DB，schema IN VARCHAR2）;

参数

参数 描述

priv_name

指定要授予的Real Application Security系统特权或模式特权的名称。

用户名

指定要向其授予Real Application Security系统特权或架构特权的用户或角色的名称。

用户类型

用户类型。默认情况下是数据库用户。

模式

授予权限的模式。如果特权是系统特权，则该值为NULL 。

例子

以下示例创建数据库用户dbuser1 ，并向此数据库用户授予Real Application Security特权ADMINISTER_SESSION ，并将user_type指定为XS_ADMIN_UTIL.PTYPE_DB ，但默认情况下，这是默认值，无需指定。 

SQL> CREATE USER dbuser1由password标识;
SQL> EXEC SYS.XS_ADMIN_UTIL.GRANT_SYSTEM_PRIVILEGE（'ADMINISTER_SESSION'，'dbuser1'，XS_ADMIN_UTIL.PTYPE_DB，'HR1'）;

以下示例创建应用程序用户user1 ，并向此应用程序用户授予Real Application Security特权ADMINISTER_SESSION ，将user_type指定为XS_ADMIN_UTIL.PTYPE_XS ，并将架构指定为HR1。 

SQL> EXEC SYS.XS_PRINCIPAL.CREATE_USER（'user1'，'HR1'）;
SQL> EXEC SYS.XS_PRINCIPAL.SET_PASSWORD（'user1'，'password'）;
SQL> EXEC SYS.XS_ADMIN_UTIL.GRANT_SYSTEM_PRIVILEGE（'ADMINISTER_SESSION'，'user1'，XS_ADMIN_UTIL.PTYPE_XS，'HR1'）;

REVOKE_SYSTEM_PRIVILEGE程序

REVOKE_SYSTEM_PRIVILEGE用于从用户或角色撤消Real Application Security ststem权限或架构权限。只有SYS权限或具有GRANT ANY PRIVILEGE权限的用户才能执行此操作。

审计操作AUDIT_REVOKE_PRIVILEGE审计所有REVOKE_SYSTEM_PRIVILEGE调用以撤消系统特权或架构特权。

句法

XS_ADMIN_UTIL.REVOKE_SYSTEM_PRIVILEGE（priv_name IN VARCHAR2，user_name IN VARCHAR2，user_type IN PLS_INTEGER：= XS_ADMIN_UTIL.PTYPE_DB，schema IN VARCHAR2）;

参数

参数 描述

priv_name

指定要撤消的Real Application Security系统特权或架构特权的名称。

用户名

指定要从中撤消Real Application Security系统特权或架构特权的用户或角色的名称。

用户类型

用户类型。默认情况下是数据库用户。

模式

撤消权限的模式。如果特权是系统特权，则该值为NULL 。

例子

以下示例dbuser1数据库用户创建数据库用户dbuser1并撤消Real Application Security特权ADMINISTER_SESSION ，并将user_type指定为XS_ADMIN_UTIL.PTYPE_DB ，但默认情况下，这是默认值，无需指定。 

CREATE USER dbuser1由密码标识;
SYS.XS_ADMIN_UTIL.REVOKE_SYSTEM_PRIVILEGE（'ADMINISTER_SESSION'，'dbuser1'，XS_ADMIN_UTIL.PTYPE_DB，'HR1'）;

以下示例从此应用程序用户创建应用程序用户user1并撤消Real Application Security特权ADMINISTER_SESSION ，并将user_type指定为XS_ADMIN_UTIL.PTYPE_XS 。

SQL> EXEC SYS.XS_PRINCIPAL.CREATE_USER（'user1'，'HR1'）;
SQL> EXEC SYS.XS_PRINCIPAL.SET_PASSWORD（'user1'，'password'）;
SQL> EXEC SYS.XS_ADMIN_UTIL.REVOKE_SYSTEM_PRIVILEGE（'ADMINISTER_SESSION'，'user1'，XS_ADMIN_UTIL.PTYPE_XS，'HR1'）;
