父主题： 打开影响所有平台的错误

错误29414664

命令rhpctl instantiate image的帮助文本未显示-server选项。

解决方法：

在已建立对等关系的两个Fleet Patching and Provisioning（FPP）服务器之间rhpctl instantiate image时，使用命令rhpctl instantiate image和-server选项指定远程FPP服务器，从该服务器实例化给定映像。

父主题： Oracle Grid Infrastructure已知错误

错误29053083

网格命名服务（GNS）清单文件验证失败。

解决方法：

创建成员集群清单文件时不要添加GNS。

父主题： Oracle Grid Infrastructure已知错误

错误21559133

此问题会影响从Oracle Grid Infrastructure 12c第1版（12.1）到Oracle Clusterware标准集群与Oracle ASM的Oracle Grid Infrastructure 19c的滚动升级。第一个节点升级到Oracle Grid Infrastructure 19c后，运行Oracle Grid Infrastructure 12c第1版（12.1）的节点无法加入群集。运行Oracle Grid Infrastructure 12c版本1（12.1）且处于ONLINE状态的节点仍然是群集的成员。

解决方法：

在故障节点上将Oracle Grid Infrastructure从12c版本1（12.1）升级到19c。
