2.5.4查询状态和下一次计划的守护程序运行

查询正在运行的守护程序的状态和下一个自动运行计划。

-d status | info | nextautorun

  • -d status ：检查守护程序是否正在运行。

  • -d info ：显示有关正在运行的守护程序的信息。

  • -d nextautorun [-id ID ] ：显示下一个自动运行时间。

要查询状态和下一个计划的守护程序运行：

  1. 要检查守护程序是否正在运行，请使用–d status ：
    $ ./orachk -d状态
    $ ./exachk -d status

    如果守护程序正在运行，则守护程序将确认并显示PID。

  2. 要查询有关守护程序的更多详细信息，请使用–d info ：
    $ ./orachk -d info
    $ ./exachk -d info

    守护程序响应以下信息：

    • 安装守护程序的节点

    • 版

    • 安装位置

    • 守护程序启动的时间

  3. 要查询下一次计划的运行状况检查运行，请使用–d nextautorun ：
    $ ./orachk -d nextautorun
    $ ./exachk -d nextautorun

    守护进程响应计划的详细信息。

    如果已配置多个守护程序选项配置文件，则输出将显示下一个计划运行的任何一个。

    如果已配置多个守护程序选项配置文件，则使用–id ID -d nextautorun查询特定概要文件的下一个计划运行状况检查运行：
    $ ./orachk -d ID -d nextautorun
    $ ./exachk -d ID -d nextautorun

    守护程序会响应您指定的守护程序选项配置文件ID的计划详细信息。
