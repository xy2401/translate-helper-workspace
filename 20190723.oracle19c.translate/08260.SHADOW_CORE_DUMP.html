1.310 SHADOW_CORE_DUMP

SHADOW_CORE_DUMP指定Oracle是否在SHADOW_CORE_DUMP （客户端）进程的核心文件中包含SGA。

属性 描述

参数类型

串

句法

SHADOW_CORE_DUMP = {partial |满|没有 }

默认值

partial 。在Windows上，默认值为none 。

可修改

改变系统

可在PDB中修改

是

基本

没有

值

  • 局部

    Oracle不在核心转储中包含SGA。

  • 充分

    Oracle在核心转储中包含SGA。

  • 没有

    不会为前台进程生成核心文件。
