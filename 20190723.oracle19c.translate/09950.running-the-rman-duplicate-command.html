运行RMAN DUPLICATE命令

在目标服务器上运行DUPLICATE命令。

例如：
DUPLICATE DATABASE to DUdb FOR STANDBY FROM ACTIVE DATABASE PASSWORD FILE SPFILE PARAMETER_VALUE_CONVERT ='/ app / dbhome1'，'/ app / db_home2'Set db_file_name_convert ='/ app / dbhome1 / dbs'，'/ app / db_home2 / database / dbs'SET log_file_name_convert ='/ app / dbhome1 / log'，'/ app / db_home2 / logfiles'NOFILENAMECHECK;
