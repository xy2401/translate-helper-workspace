解锁和重置Oracle数据库用户密码

安装后，将撤消除SYS，SYSTEM和DBSMP之外的所有Oracle系统管理帐户的密码。在使用锁定帐户之前，您必须解锁并重置其密码。

如果您在安装期间创建了预配置的数据库，但未解锁使用数据库所需的帐户，则必须使用这些过程解锁并重置这些帐户。

  • 数据库密码的要求
    要保护数据库，请使用满足Oracle建议密码要求的密码，甚至是预定义用户帐户的密码。
  • Oracle数据库系统权限帐户和密码
    安装后查看这些系统特权帐户，以准备解锁帐户和更改密码。
  • 更改系统权限帐户密码的准则
    请遵循以下规则来更改Oracle数据库系统权限帐户密码。
  • 锁定和解锁用户帐户
    您可以使用Oracle Enterprise Manager Database Express（EM Express）来锁定和解锁用户帐户。
  • 使用SQL * Plus解锁帐户并重置密码
    使用此SQL * Plus过程可以解锁和重置用户帐户密码。
