1.136 IFILE

使用IFILE在当前参数文件中嵌入另一个参数文件。

属性 描述

参数类型

参数文件

句法

IFILE = parameter_file_name

默认值

没有默认值。

可修改

没有

可在PDB中修改

没有

价值范围

有效参数文件名

基本

没有

Oracle RAC

多个实例可以具有不同的值。

例如：

IFILE = COMMON.ORA

您最多可以有三个嵌套级别。在此示例中，文件COMMON.ORA可以包含文件COMMON.ORA的第二个IFILE参数COMMON2.ORA ，可以包含文件GCPARMS.ORA的第三个IFILE参数。您还可以通过使用不同的值列出IFILE几次，在一个参数文件中包含多个参数文件：

IFILE = DBPARMS.ORA IFILE = GCPARMS.ORA IFILE = LOGPARMS.ORA

注意：

您必须在参数文件的连续行上列出多个条目。
