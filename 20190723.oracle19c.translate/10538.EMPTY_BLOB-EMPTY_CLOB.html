EMPTY_BLOB，EMPTY_CLOB

句法

empty_LOB :: =


插图说明empty_lob.eps

目的

EMPTY_BLOB和EMPTY_CLOB返回一个空的LOB定位符，可用于初始化LOB变量，或者在INSERT或UPDATE语句中，将LOB列或属性初始化为EMPTY 。 EMPTY表示LOB已初始化，但未填充数据。

注意：

空LOB与空LOB不同，空CLOB与包含0长度字符串的LOB不同。有关更多信息，请参阅Oracle Database SecureFiles和Large Objects Developer's Guide 。

也可以看看：

“ 数据库全球化支持指南”中的附录C，用于排序规则派生规则，用于定义分配给EMPTY_CLOB返回值的排序规则

对LOB定位器的限制

您不能将此函数返回的定位符用作DBMS_LOB包或OCI的参数。

例子

以下示例将示例pm.print_media表的ad_photo列初始化为EMPTY ：

更新print_media SET ad_photo = EMPTY_BLOB（）;
