11安装Oracle数据库

Oracle数据库和Oracle Grid Infrastructure安装软件在多种介质中以基于映像的zip文件的形式提供，可以使用多个选项进行安装。

Oracle数据库软件可在安装介质上使用，也可以从Oracle Technology Network网站或Oracle Software Delivery Cloud门户下载。在大多数情况下，您使用Oracle Universal Installer提供的图形用户界面（GUI）来安装软件。但是，您也可以使用Oracle Universal Installer完成静默模式安装，而无需使用GUI。您还可以使用Oracle Fleet Patching and Provisioning for Oracle Database和Oracle Grid Infrastructure（集群件）部署。

注意：

要在Oracle自动存储管理（Oracle ASM）上安装Oracle数据库文件，必须首先为独立服务器安装完成Oracle Grid Infrastructure。用于独立服务器的Oracle Grid Infrastructure包括Oracle Restart和Oracle ASM。

要升级现有Oracle ASM安装，请通过运行Oracle Grid Infrastructure升级来升级Oracle ASM。如果未安装Oracle ASM并且希望使用Oracle ASM作为存储选项，则必须在启动Oracle数据库安装之前为独立服务器安装完成Oracle Grid Infrastructure。

您不能使用早期Oracle版本中的Oracle Universal Installer来安装此版本的组件。
