目的

您可以使用TREAT函数更改表达式的声明类型。

如果希望表达式返回JSON数据，请使用关键字AS JSON 。当您想强制将某些文本解释为JSON数据时，这非常有用。例如，您可以使用它将{}的VARCHAR2值解释为空JSON对象而不是字符串。

您必须具有type的EXECUTE对象特权才能使用此功能。

  • 在expr AS JSON ， expr是包含JSON的SQL数据类型，例如CLOB 。

  • 在expr AS type ， expr和type必须是用户定义的对象类型，不包括顶级集合。

  • type必须是声明的expr类型的某个超类型或子类型。如果最具体的expr type是type （或某种类型的子type ），则TREAT返回expr 。如果最特定类型的expr不是type （或类型的某个子type ），则TREAT返回NULL 。

  • 仅当声明的expr类型是REF类型时，才能指定REF 。

  • 如果声明的类型expr是REF到源类型的expr ，那么type必须是源类型的某些亚型或超expr 。如果最特殊类型的DEREF （ expr ）是type （或类型的子type ），则TREAT返回expr 。如果最特定类型的DEREF （ expr ）不是type （或类型的子type ），则TREAT返回NULL 。
