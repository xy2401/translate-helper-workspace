在Oracle Grid Infrastructure安装或升级期间应用修补程序

从Oracle Grid Infrastructure 18c开始，您可以在Oracle Grid Infrastructure安装或升级期间下载并应用版本更新（RU）和一次性补丁。

  1. 从My Oracle Support下载要应用的修补程序：
    https://support.oracle.com
  2. 选择“ 修补程序和更新”选项卡以找到修补程序。
    Oracle建议您选择Recommended Patch Advisor ，然后输入软件的产品组，发行版和平台。
  3. 将修补程序移动到可访问的目录，如/tmp 。
  4. 切换到Oracle Grid Infrastructure主目录：
    $ cd /u01/app/oracle/product/19.0.0/grid
  5. 在安装或升级过程中应用版本更新（RU）和任何一次性补丁：
    $ ./gridSetup.sh -applyRU patch_directory_location -applyOneOffs comma_seperated_list_of_patch_directory_locations

    注意：

    您可以在同一命令中单独或一起应用RU和一次性补丁。
  6. 完成Oracle Grid Infrastructure配置向导中的其余步骤以完成安装或升级。
