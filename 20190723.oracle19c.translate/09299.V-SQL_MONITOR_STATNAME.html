9.56 V $ SQL_MONITOR_STATNAME

V$SQL_MONITOR_STATNAME提供有关在V$SQL_PLAN_MONITOR公开的计划行统计信息的信息。计划行统计信息由其组ID（列GROUP_ID ）及其ID（列ID ） ID 。

柱 数据类型 描述

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID

ID

数

计划行统计ID

GROUP_ID

数

计划行统计组ID

名称

描述
