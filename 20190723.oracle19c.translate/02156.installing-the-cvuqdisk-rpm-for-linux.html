安装适用于Linux的cvuqdisk RPM

如果不使用Oracle预安装RPM，并且要使用Cluster Verification Utility，则必须安装cvuqdisk RPM。

如果没有cvuqdisk ，Cluster Verification Utility无法发现共享磁盘，并且在运行Cluster Verification Utility时收到错误消息“Package cvuqdisk not installed”。将cvuqdisk RPM用于您的硬件（例如， x86_64 ）。
  1. 找到cvuqdisk RPM软件包，该软件包位于Grid_home /cv/rpm目录中。Grid_home是Oracle Grid Infrastructure主目录。
  2. 将cvuqdisk包复制到群集上的每个节点。您应该确保每个节点都运行相同版本的Linux。
  3. 以root身份登录。
  4. 使用以下命令查找是否具有cvuqdisk软件包的现有版本：
    #rpm -qi cvuqdisk

    如果您有现有版本的cvuqdisk ，请输入以下命令以卸载现有版本：

    #rpm -e cvuqdisk
  5. 将环境变量CVUQDISK_GRP设置为指向拥有cvuqdisk的组，通常是oinstall 。例如：
    ＃CVUQDISK_GRP = oinstall;导出CVUQDISK_GRP
  6. 在保存cvuqdisk RPM的目录中，使用命令rpm -iv package安装cvuqdisk软件包。例如：
    #rpm -iv cvuqdisk-1.0.10-1.rpm
