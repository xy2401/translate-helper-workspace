5 Oracle Map Builder工具

本章简要介绍了地图可视化组件Map Builder工具，也称为Oracle Map Builder。它不提供有关工具界面的详细信息;为此，您应该使用Oracle Map Builder时可以使用的在线帮助。

Oracle Map Builder是一个独立的应用程序，可用于创建和管理存储在数据库中的映射元数据（关于样式，主题和基本映射）。例如，使用此工具创建样式或修改样式的定义。除了处理元数据外，该工具还提供了预览元数据的界面（例如，查看线条样式在地图上的显示方式）以及空间信息。

只要有可能，您应该使用Oracle Map Builder而不是直接修改地图可视化组件元数据视图来创建，修改和删除有关样式，主题和地图的信息。对于在Oracle Map Builder外部进行的任何修改（例如使用SQL语句），您应该在Oracle Map Builder中刷新数据库连接以获取当前项。

要有效地使用Oracle Map Builder，您必须了解Map Visualization Concepts中解释的地图可视化组件概念以及Map Visualization Servers中有关地图请求的信息。

5.1运行Oracle Map Builder

Oracle Map Builder作为JAR文件（ mapbuilder.jar ）提供。

您可以在Java Development Kit（J2SE SDK）1.5或更高版本的环境中将其作为独立的Java应用程序运行，如下所示：

％java -jar mapbuilder.jar [选项]

选项：

-cache <cache_size>指定内存中几何缓存的大小。示例： - cache 64M

-config <config-file>指定包含Map Builder配置和首选项信息的文件的位置。如果未指定此选项，Map Builder将在主Java目录中查找名为oasmapbuilder.xml的文件。有关配置和首选项文件的更多信息，请参阅配置映射可视化组件 。

-connect导致Map Builder在启动时为oasmapbuilder.xml首选项文件或使用-config选项指定的文件中指定的所有数据源注册连接，并自动连接到第一个可用数据源。此选项会增加应用程序启动时间。如果未定义此选项，则启动速度更快，但您必须使用“文件”菜单或图标连接到要使用的任何数据源（请参阅Oracle Map Builder用户界面 ）。

-help显示有关可用选项的信息。

5.1.1使用GDAL和Teradata创建主题的Java库

要使用GDAL-OGR（版本1.8或更高版本）或Teradata（版本13或更高版本）在Map Builder中创建主题，必须使用GDAL-OGR（ gdal.jar ）和/或Teradata（ terajdbc4.jar和tdgssconfig.jar ）启动Map Builder tdgssconfig.jar ）类路径上的jar文件。可以在系统和Teradata网站上的GDAL安装中找到这些Java库。（有关GDAL的信息，请参阅GDAL网站以获取更多信息。）

以下示例命令在类路径中使用其他库启动Map Builder。

Windows： java -Xmx512M -cp .\mapbuilder.jar;.\gdal.jar;.\terajdbc4.jar;.\tdgssconfig.jar oracle.mapviewer.builder.MapBuilder

Linux： java -Xmx512M -cp ./mapbuilder.jar:./gdal.jar:./terajdbc4.jar:./tdgssconfig.jar oracle.mapviewer.builder.MapBuilder

5.2 Oracle Map Builder用户界面

Oracle Map Builder通常使用左侧导航来查找和选择对象，右侧则使用右侧显示有关所选对象的信息。

图5-1显示了Oracle Map Builder的主窗口，左侧是元数据导航树，右侧是选定区域样式的详细信息窗格。

图5-1 Oracle Map Builder主窗口


“图5-1 Oracle Map Builder主窗口”的说明

顶部的菜单包含标准条目，以及Oracle Map Builder特有的功能条目。

您可以使用快捷键访问菜单和菜单项：例如，“文件”菜单的Alt + F和“编辑”菜单的Alt + E;或Alt + H，然后按Alt + A获取帮助，然后关于。

菜单下的图标执行以下操作：

  • 添加新连接会为Oracle Map Builder创建新的数据库连接以供使用。

  • 加载/添加/删除连接加载或添加数据库连接以供Oracle Map Builder使用，或从Oracle Map Builder可以使用的可用连接中删除数据库连接。

  • 创建新元数据可创建新的基本地图，主题或样式。

  • 打开打开基本地图，主题或样式。

  • 保存会保存对当前所选对象的任何更改。

  • 全部保存将所有更改保存到所有打开的对象。

Oracle Map Builder窗口的左侧有Metadata导航器，包括数据库连接选择器，用于执行操作的图标，以及用于当前所选数据库连接可访问的地图可视化组件元数据对象（按对象类型分类）的分层树显示。要选择对象，请展开相应的树节点，然后双击该对象。

Oracle Map Builder窗口的右侧有选项卡和窗格，用于选择或打开的对象的详细视图

要在对象之间切换，请单击所需的选项卡;要关闭选项卡，请单击选项卡中的X.如果对对象进行更改并单击X，则会询问您是否要保存更改。

“消息”区域适用于反馈信息（例如，操作结果或错误或警告消息）。

Oracle Map Builder界面中提供了详细的帮助。有关Oracle Map Builder的更多信息，请参阅联机帮助，包括有关特定窗格和对话框的信息。
