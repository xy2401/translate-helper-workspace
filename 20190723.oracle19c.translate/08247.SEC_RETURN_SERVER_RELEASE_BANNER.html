1.301 SEC_RETURN_SERVER_RELEASE_BANNER

SEC_RETURN_SERVER_RELEASE_BANNER指定服务器是否将完整的数据库软件信息返回给未经身份验证的客户端。

属性 描述

参数类型

布尔

默认值

假

可修改

没有

可在PDB中修改

没有

价值范围

是的|假

基本

没有

值

  • 真正

    将完整的数据库版本信息返回给客户端。

  • 假

    返回客户端的通用版本字符串。
