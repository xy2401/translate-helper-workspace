187 DBMS_XDB_ADMIN

DBMS_XDB_ADMIN包提供了一个用于管理Oracle XML DB存储库的接口。

本章包含以下主题：

  • DBMS_XDB_ADMIN安全模型

  • DBMS_XDB_ADMIN子程序的摘要

也可以看看：

有关Oracle XML DB存储库的信息，请参见Oracle XML DB Developer's Guide

187.1 DBMS_XDB_ADMIN安全模型

由XDB拥有， DBMS_XDB_ADMIN包必须由SYS或XDB创建。 EXECUTE特权被授予SYS或XDB或DBA 。此包中的子程序使用当前用户的权限执行。

187.2.1 CREATENONCEKEY程序

此过程生成用于摘要式身份验证的随机数值。

句法

DBMS_XDB_ADMIN.CREATENONCEKEY;

187.2.2 INSTALLDEFAULTWALLET过程

此过程将默认XDB钱包安装在默认的XDB钱包目录中。

存储XDB钱包的目录名称在定义时由ORACLE_BASE或ORACLE_HOME作为前缀。然后是/admin/ db_name /xdb_wallet ，其中db_name是唯一的数据库名称。

句法

DBMS_XDB_ADMIN.INSTALLDEFAULTWALLET;

使用说明

只有SYS可以安装或替换默认钱包。

187.2.3 MOVEXDB_TABLESPACE过程

此过程将XDB（用户）移动到指定的表空间。

句法

DBMS_XDB_ADMIN.MOVEXDB_TABLESPACE（new_tablespace IN VARCHAR2）;

参数

表187-2 MOVEXDB_TABLESPACE过程参数

参数 描述

new_tablespace

移动XDB的表空间的名称

使用说明

  • 此操作等待所有并发的XDB会话退出。

  • 如果MOVEXDB_TABLESPACE失败，则用户应在发出任何进一步的命令之前重新启动数据库。如果不这样做，将导致数据库出现意外行为。

  • 默认情况下，XDB存储库位于SYSAUX表空间中。使用此过程可以将其移动到另一个表空间。作为最佳实践，我们建议仅为XDB存储库创建专用表空间，而不与其他对象（例如表）共享。不应将包含XDB存储库的表空间设置为READ ONLY因为这可能会影响正在执行的各种XML操作。

187.2.4 REBUILDHIERARCHICALINDEX程序

此过程在导入或导出操作后重建层次结构索引。这是必要的，因为无法从索引表中导出数据。

句法

DBMS_XDB_ADMIN.REBUILDHIERARCHICALINDEX;
