ROUND（数字）

句法

round_number :: =


插图描述round_number.eps

目的

ROUND返回n四舍五入到integer位，小数点右边。如果省略integer ，则n舍入为零。如果integer为负数，则n将舍入到小数点的左侧。

n可以是任何数值数据类型或可以隐式转换为数字数据类型的任何非数字数据类型。如果省略integer ，则该函数返回与数字数据类型n相同的数据类型中的值ROUND （n，0）。如果包含integer ，则该函数返回NUMBER 。

ROUND使用以下规则实现：

  1. 如果n为0，则无论integer如何， ROUND始终返回0。

  2. 如果n为负，则ROUND （n，整数）返回 - ROUND （-n，整数）。

  3. 如果n是正数，那么

    ROUND（n，整数）= FLOOR（n * POWER（10，整数）+ 0.5）* POWER（10，-integer）

应用于NUMBER值的ROUND可能会给出与应用于浮点表示的相同值的ROUND略有不同的结果。不同的结果来自NUMBER和浮点值的内部表示的差异。如果出现差异，则在舍入的数字中差异为1。

例子

以下示例将数字舍入到一个小数点：

SELECT ROUND（15.193,1）“Round”FROM DUAL;回合---------- 15.2

以下示例舍入小数点左侧的第一个数字：

SELECT ROUND（15.193，-1）“Round”FROM DUAL;回合---------- 20
