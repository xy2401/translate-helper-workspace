创造功能

目的

使用PL / SQL定义函数。因此，本节提供了一些常规信息，但有关语法和语义的详细信息，请参阅Oracle Database PL / SQL语言参考 。

使用CREATE FUNCTION语句创建独立存储函数或调用规范。

  • 存储函数 （也称为用户函数或用户定义函数 ）是一组可以按名称调用的PL / SQL语句。存储函数与过程非常相似，只是函数会向调用它的环境返回一个值。用户函数可以用作SQL表达式的一部分。

  • 调用规范声明了Java方法或第三代语言（3GL）例程，以便可以从PL / SQL调用它。您还可以使用CALL SQL语句来调用此类方法或例程。调用规范告诉Oracle数据库在调用时调用哪个Java方法或哪个命名函数来调用共享库。它还告诉数据库为参数和返回值进行的类型转换。

注意：

您还可以使用CREATE PACKAGE语句将函数创建为包的一部分。

先决条件

要在您自己的模式中创建或替换函数，您必须具有CREATE PROCEDURE系统特权。要在另一个用户的模式中创建或替换函数，您必须具有CREATE ANY PROCEDURE系统特权。

语义

或替换

如果已存在，请指定OR REPLACE以重新创建该函数。使用此子句可以更改现有函数的定义，而无需删除，重新创建和重新授予以前在函数上授予的对象特权。如果重新定义函数，则Oracle Database会重新编译它。

以前被授予重新定义的功能权限的用户仍然可以访问该功能而无需重新授予权限。

如果任何基于函数的索引依赖于该函数，则Oracle数据库会将索引标记为DISABLED 。
