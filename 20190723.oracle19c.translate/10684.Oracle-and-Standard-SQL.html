C Oracle和标准SQL

本附录声明Oracle符合美国国家标准协会（ANSI）和国际标准化组织（ISO）制定的SQL标准。

ISO SQL标准由九个部分组成（SQL / Framework，SQL / Foundation，SQL / CLI，SQL / PSM，SQL / MED，SQL / OLB，SQL / Schemata，SQL / JRT和SQL / XML）。ANSI SQL标准由九个部分组成。

SQL的强制部分称为Core SQL，可在SQL：2016第2部分（基础）和第11部分（Schemata）中找到。基础特征在第2部分附录F的“特征分类和强制性特征的定义”表中进行了分析。 Schemata功能在第11部分附录F的“功能分类和强制性功能定义”表中进行了分析。
