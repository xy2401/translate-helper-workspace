1.213 NLS_TIMESTAMP_TZ_FORMAT

NLS_TIMESTAMP_TZ_FORMAT定义具有时区格式的默认时间戳，以与TO_CHAR和TO_TIMESTAMP_TZ函数一起使用。

属性 描述

参数类型

串

句法

NLS_TIMESTAMP_TZ_FORMAT = " format "

默认值

源自NLS_TERRITORY

可修改

改变会话

可在PDB中修改

是

价值范围

任何有效的日期时间格式掩码

基本

没有

该值必须用引号括起，如下所示：

NLS_TIMESTAMP_TZ_FORMAT ='YYYY-MM-DD HH：MI：SS.FF TZH：TZM'

您可以通过在初始化参数文件中设置NLS_TIMESTAMP_TZ_FORMAT来指定它的值。您可以将客户端的值指定为客户端环境变量。

您还可以通过更改初始化参数中的值然后重新启动实例来更改NLS_TIMESTAMP_TZ_FORMAT的值。要在会话期间更改值，请使用ALTER SESSION SET语句。

注意：

此初始化参数NLS_TIMESTAMP_TZ_FORMAT值用于初始化此参数的会话值，该值是SQL查询处理引用的实际值。如果客户端使用Oracle JDBC驱动程序或客户端是基于OCI并且定义了NLS_LANG客户端设置（环境变量），则此初始值将被客户端值覆盖。因此，通常忽略初始化参数值。
