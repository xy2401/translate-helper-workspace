2.202 ALL_LOBS

ALL_LOBS显示当前用户可访问的表中包含的大对象（LOB）。LOB包括二进制大对象（BLOB）和字符大对象（CLOB）。 二进制文件（BFILE）存储在数据库外部，因此该视图或相关视图不会显示它们。

相关观点

  • DBA_LOBS描述数据库中的所有LOB。

  • USER_LOBS描述当前用户拥有的LOB。此视图不显示OWNER列。

数

数

此LOB段的空闲池数

CACHE

VARCHAR2（10）

指示是否以及如何在缓冲区高速缓存中缓存LOB数据：

  • YES - LOB数据放在缓冲区缓存中

  • NO - LOB数据要么不被带入缓冲区高速缓存，要么被带入缓冲区高速缓存并放置在最近最少使用的LRU列表末尾

  • CACHEREADS - LOB数据仅在读取操作期间进入缓冲区高速缓存，但在写入操作期间不进入缓冲区高速缓存

测井

VARCHAR2（7）

指示是否记录对LOB的更改：

  • NONE - 未指定

    另请参见： *_LOB_SUBPARTITIONS视图

  • 是

  • 没有

加密

VARCHAR2（4）

指示LOB是否已加密。

SecureFiles的可能值：

  • 是

  • 没有

BasicFiles的可能值：

  • NONE - 不适用

压缩

VARCHAR2（6）

此LOB使用的压缩级别。

SecureFiles的可能值：

BasicFiles的可能值：

  • NONE - 不适用

VARCHAR2（15）
