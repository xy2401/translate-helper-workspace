5.205 DBA_NETWORK_ACL_PRIVILEGES

DBA_NETWORK_ACL_PRIVILEGES描述了当前分配给网络主机的所有访问控制列表中定义的网络权限。

注意：

Oracle Database 12 c第1版（12.1）中不推荐使用此DBA_NETWORK_ACL_PRIVILEGES视图。Oracle建议您使用DBA_HOST_ACES视图。

柱 数据类型 空值 描述

ACL

VARCHAR2（4000）

访问控制列表的路径

ACLID

RAW（8）

不是NULL

访问控制列表的对象ID

主要

VARCHAR2（128）

授予或拒绝特权的委托人（数据库用户或角色）

特权

VARCHAR2（128）

网络权限

IS_GRANT

VARCHAR2（5）

指示是授予（ true ）还是拒绝（ false ）权限

倒置

VARCHAR2（5）

指示访问控制条目是否包含反转主体（ true ）或不包含（ false ）

开始日期

TIMESTAMP（6）

访问控制条目的开始日期

结束日期

TIMESTAMP（6）

访问控制条目的结束日期

ACL_OWNER

VARCHAR2（128）

访问控制列表的所有者
