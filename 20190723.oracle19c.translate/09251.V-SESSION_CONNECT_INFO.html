9.19 V $ SESSION_CONNECT_INFO

V$SESSION_CONNECT_INFO显示有关所有当前登录会话的网络连接的信息。

柱 数据类型 描述

SID

数

会话标识符（可用于将此视图与V$SESSION ）

SERIAL＃

数

会话序列号。用于唯一标识会话的对象。如果会话结束且另一个会话以相同的会话ID开始，则保证会话级命令应用于正确的会话对象。 （可用于将此视图与V$SESSION 。）

AUTHENTICATION_TYPE

VARCHAR2（26）

用户如何进行身份验证：

  • DATABASE - 用户名/密码验证

  • OS - 操作系统外部用户身份验证

  • NETWORK - 网络协议或ANO身份验证

  • PROXY - OCI代理连接身份验证

  • 服务器

  • 密码

  • 外部适配器

  • 内部

  • 全球

  • 外部

  • 基于密码的全球用户

OSUSER

VARCHAR2（128）

此数据库用户的外部用户名

NETWORK_SERVICE_BANNER

VARCHAR2（4000）

用于此连接的每个Oracle Net服务的产品横幅（每个横幅一行）

在client_charset

VARCHAR2（40）

在NLS_LANG环境变量或OCIEnvNlsCreate()调用中指定的客户端字符集;如果Oracle客户端比版本11.1早，或者连接是通过JDBC瘦驱动程序，则为Unknown

CLIENT_CONNECTION

VARCHAR2（13）

客户服务器连接标志：

  • 异质

  • 同质

CLIENT_OCI_LIBRARY

VARCHAR2（27）

OCI客户端库：

  • 以家庭为基础

  • 完整即时客户端

  • XE Instant Client

  • 轻量级即时客户端

  • OCI

  • 未知

CLIENT_VERSION

VARCHAR2（40）

客户端库版本号

CLIENT_DRIVER

VARCHAR2（30）

客户端驱动名称

CLIENT_LOBATTR

VARCHAR2（23）

客户端LOB标志：

  • 客户端Temp Lob Rfc On

  • 客户端Temp Lob Rfc Off

CLIENT_REGID

数

查询客户端发送的缓存注册ID

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
