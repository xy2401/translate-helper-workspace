4.334 DBA_HIST_IOSTAT_FUNCTION

DBA_HIST_IOSTAT_FUNCTION按功能显示历史I / O统计信息。

该视图包含V$IOSTAT_FUNCTION快照。

柱 数据类型 空值 描述

数

不是NULL

功能ID

FUNCTION_NAME

不是NULL

数

不是NULL

数

数

数

数

CON_DBID

数

采样会话的PDB的数据库ID

CON_ID

数

CON_DBID标识的容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
