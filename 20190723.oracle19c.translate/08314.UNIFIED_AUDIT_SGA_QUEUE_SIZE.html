1.347 UNIFIED_AUDIT_SGA_QUEUE_SIZE

UNIFIED_AUDIT_SGA_QUEUE_SIZE指定用于统一审计的SGA队列的大小。

属性 描述

参数类型

整数

默认值

1 MB

可修改

没有

可在PDB中修改

没有

价值范围

1 MB到30 MB

基本

没有

Oracle RAC

多个实例可以具有不同的值

当排队写入模式用于统一审计时， UNIFIED_AUDIT_SGA_QUEUE_SIZE非常有用。在排队写入模式下，首先将审计记录写入SGA队列。当SGA队列达到特定阈值时，审计记录将刷新到AUDSYS模式表。应根据审计数据生成调整SGA队列大小。如果非常频繁地生成大量审计记录，则可以增加SGA队列的大小，以便可以防止频繁刷新队列数据。同样，如果生成的审计记录较少，则可以将较小的大小用于SGA队列。

注意：

Oracle Database 12 c第2版（12.2.0.1）中不推荐使用UNIFIED_AUDIT_SGA_QUEUE_SIZE初始化参数，并且可能在将来的发行版中不支持该参数。
