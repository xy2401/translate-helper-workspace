C网关术语

以下主题包含与网关及其组件和功能相关的术语和定义列表。

有关与网关关联的其他术语和定义的列表，请参阅“ Oracle数据库网关APPC用户指南”的第1章。

以下列出了本版Oracle Database Gateway for APPC安装和配置指南中使用的一些术语和定义。

网关初始化文件

这个文件被称为 init sid .ora ，它包含控制网关操作的参数。如果您使用的是SNA协议，请参阅SNA协议的网关初始化参数以获取更多信息。如果您的协议是TCP / IP，请参阅TCP / IP通信协议的网关初始化参数 。

网关远程过程

Oracle Database Gateway for APPC提供了预构建的远程过程。一般来说， 使用以下三个远程过程：

  • PGAINIT ，初始化交易

  • PGAXFER ，传输数据

  • PGATERM ，终止交易

有关网关远程过程的详细信息，请参阅本指南中的“ 远程过程调用函数 ”和APPC用户指南的Oracle数据库网关中的附录B“网关RPC接口”。

dg4pwd

dg4pwd是一个实用程序，用于加密通常存储在网关初始化文件中的密码。密码以加密形式存储在密码文件中，使信息更安全。有关dg4pwd实用程序如何工作的详细信息，请参阅“ 网关初始化文件中的密码 ” 。

pg4tcpmap工具

仅当网关对IMS Connect使用TCP / IP支持时，此网关映射工具才适用。其功能是将侧面配置文件名称映射到TCP / IP和IMS Connect属性到PGA_TCP_IMSC表中。

PGA

程序网关管理（PGA）是本指南中对包含Oracle数据库网关APPC的所有或大多数组件的一般参考。当对特定产品或组件的引用太窄时使用该术语。

PGDL

程序网关定义语言（PGDL）是用于为PGAU定义事务和数据的语句集合。

PL / SQL存储过程规范（PL / SQL包）

这是一个存储在Oracle数据库中的预编译PL / SQL过程。

UTL_RAW PL / SQL包（UTL_RAW函数）

网关的这个组件代表PL / SQL RAW变量和远程主机数据的一系列数据转换函数。执行的转换类型取决于远程主机数据的语言。有关详细信息，请参阅“ Oracle数据库网关APPC用户指南 ”附录D中的“UTL_RAW函数”。

UTL_PG PL / SQL包（UTL_PG函数）

网关的这个组件代表一系列COBOL数字数据转换功能。有关支持的数值数据类型转换，请参阅“ Oracle数据库网关APPC用户指南 ”附录D中的“NUMBER_TO_RAW和RAW_TO_NUMBER参数值”。
