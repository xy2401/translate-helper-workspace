B TCP / IP通信协议的网关初始化参数

以下主题描述了具有该功能的网关所需的参数用于IMS Connect通信协议的TCP / IP。它还提供了pg4tcpmap工具的示例输出。此外，主题包含使用TCP / IP的网关的示例listener.ora和tnsnames.ora文件。它包含以下部分：

  • 使用TCP / IP的网关初始化参数文件

  • pg4tcpmap工具的输出

使用TCP / IP的网关初始化参数文件

该使用TCP / IP for IMS Connect的Oracle数据库网关APPC的参数文件位于$ORACLE_HOME/dg4appc/admin目录中并被调用 init sid. ora 。

注意：

init sid. ora文件包含SNA和TCP / IP参数。您需要使用合适的参数修改这些文件。

PGA参数

该 PGA参数控制网关的TCP / IP接口部分。使用SET指定PGA参数网关初始化参数。例如：

SET pga_parm = value

哪里：

  • pga_parm是pga_parm列表中的PGA参数名称之一

  • value是一个字符串，其内容依赖于pga_parm

表B-1提供了PGA参数及其描述的列表。

表B-1用于APPC的Oracle数据库网关的PGA参数使用TCP / IP进行IMS Connect

参数 描述

LOG_DESTINATION = logpath

logpath指定重新打开STDERR的目标。LOG_DESTINATION仅指定目录，并将STDERR重新打开到logpath/ sid_pid .log

哪里：

  • sid是sid的名字

  • pid是分配给网关的进程ID

注意：此参数将用于将Trace Level设置为255时的pg4tcpmap工具。此工具的日志文件将与网关日志文件位于pg4tcpmap_ sid .log中的相同位置。

PGA_CAPABILITY

PGA交易能力。以下是有效值：

READ_ONLY或RO ：只读功能。

SINGLE_SITE或SS ：仅限单站点更新。这表明在分布式环境中，只有网关可以执行更新。Oracle事务中不会发生其他数据库更新。

默认值为SINGLE_SITE 。

PGA_SECURITY_TYPE

TCP / IP会话安全选项。这可以控制将哪些安全参数发送到OLTP。以下是有效值：

NONE ：不发送安全参数

PROGRAM ：发送用户ID和密码

默认值为NONE 。

有关这些选项的更多信息，请参阅安全要求 。

重要提示：您必须指定您的 RACF组名称通过 pg4tcpmap工具，如果您已将PGA安全选项设置为SECURITY=PROGRAM 。有关此问题的更多信息，请参阅“ Oracle Database Gateway for APPC用户指南” 。

如果已加载表pga_tcp_imsc并且未首先指定RACF组名称，请删除该行并使用RACF组名称的值重新插入该行。

PGA_TCP_DB

该 Oracle数据库的Oracle Net服务名称，其中网关接收其TCP / IP以获取IMS Connect信息，例如主机名和端口号。此参数的长度可以是1到255字符。此参数是必需的。

没有默认值。

PGA_TCP_PASS

甲骨文连接到PGA_TCP_DB参数指定的Oracle数据库时网关使用的密码。密码长度可以是1到30字符。此参数是必需的。密码可以加密。有关加密密码的详细信息，请参阅网关初始化文件中的密码 。

没有默认值。

PGA_TCP_USER

连接到PGA_TCP_DB参数指定的Oracle数据库时网关使用的Oracle用户标识。用户ID的长度可以是1到30字符。此参数是必需的。

没有默认值。

TRACE_LEVEL

PGA跟踪级别。这将控制写入STDERR跟踪输出（ LOG_DESTINATION参数的目标）。该值必须是0到255的整数。

默认值为0 ，表示没有跟踪。

注意：此参数用于 pg4tcpmap工具以及网关。
