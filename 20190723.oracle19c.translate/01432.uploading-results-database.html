A.4将结果上传到数据库

使用本节中的命令列表将结果上载到数据库。

句法

[-setdbupload all | list of variable names ] [-unsetdbupload all | list of variable names ] [-checkdbupload] [-getdbupload] [-checkfaileduploads] [-uploadfailed all | list of failed collections ]

参数

表A-4将结果上载到数据库

选项 描述

-setdbupload all | variable_names

设置钱包中的值以将运行状况检查运行结果上载到数据库。

all ：设置钱包中的所有变量。

variable_names ：指定要设置的逗号分隔的变量列表。

-unsetdbupload all | variable_names

取消设置钱包中的值以将运行状况检查运行结果上载到数据库。

all ：取消设置钱包中的所有变量。

variable_names ：指定要取消设置的以逗号分隔的变量列表。

-checkdbupload

检查是否正确设置了变量以将运行状况检查运行结果上载到数据库。

-getdbupload

使用wallet中的值打印变量，以将运行状况检查运行结果上载到数据库。

-checkfaileduploads

报告任何失败的集合上传。

-uploadfailed all | list of failed collections

重新尝试上载一个或多个失败的集合上载。

all ：重新尝试上传所有已归档的收集上传内容。

list of failed collections ：指定要上载的逗号分隔的集合列表。
