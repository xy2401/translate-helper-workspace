设置远程显示和X11转发配置

如果您在远程终端上，并且本地系统只有一个可视（这是典型的），则使用以下语法设置您的用户帐户DISPLAY环境变量：

远程显示

Bourne，Korn和Bash shell
$ export DISPLAY = 主机名 ：0
C shell
％setenv DISPLAY 主机名 ：0

例如，如果您使用的是Bash shell，并且主机名是local_host ，请输入以下命令：

$ export DISPLAY = node1：0

X11转发

要确保X11转发不会导致安装失败，请使用以下过程为Oracle安装所有者用户帐户创建用户级SSH客户端配置文件：

  1. 使用任何文本编辑器，编辑或创建软件安装所有者的~/.ssh/config file 。

  2. 确保~/.ssh/config文件中的ForwardX11属性设置为no 。例如：

    主机* ForwardX11没有

  3. 确保~/.ssh上的权限受到Oracle安装所有者用户帐户的保护。例如：

    $ ls -al .ssh total 28 drwx ------ 2 grid oinstall 4096 Jun 21 2015 drwx ------ 19 grid oinstall 4096 Jun 21 2015 -rw-r  -  r-- 1 grid oinstall 1202 Jun 21 2015 authorized_keys -rwx ------ 1 grid oinstall 668 Jun 21 2015 id_dsa -rwx ------ 1 grid oinstall 601 Jun 21 2015 id_dsa.pub -rwx ------ 1 grid oinstall 1610 Jun 21 2015 known_hosts
