31.3 OCI语言环境映射功能

列出并描述OCI语言环境映射功能。

表31-3列出了本节中描述的OCI语言环境映射功能。

OCI语言环境映射功能执行与Oracle数据库，Internet分配号码机构（IANA）和国际标准化组织（ISO）名称之间的名称映射，以表示字符集名称，语言名称和地域名称。

表31-3 OCI语言环境映射功能

功能 目的

OCINlsNameMap（）

将Oracle数据库字符集名称，语言名称和地域名称映射到Internet Assigned Numbers Authority（IANA）和国际标准化组织（ISO）名称之间

31.3.1 OCINlsNameMap（）

将Oracle数据库字符集名称，语言名称和地域名称映射到Internet Assigned Numbers Authority（IANA）和国际标准化组织（ISO）名称。

目的

将Oracle数据库字符集名称，语言名称和地域名称映射到Internet Assigned Numbers Authority（IANA）和国际标准化组织（ISO）名称。

句法

剑OCINlsNameMap（void * hndl，OraText * buf，size_t buflen，const OraText * srcbuf，uword flag）;

参数

hndl（IN / OUT）

OCI环境或用户会话句柄。如果句柄无效，则该函数返回OCI_INVALID_HANDLE 。

buf（OUT）

指向目标缓冲区。如果函数返回OCI_SUCCESS ，则该参数包含所请求名称的NULL终止字符串。

buflen（IN）

目标缓冲区的大小。建议的大小为OCI_NLS_MAXBUFSZ以保证存储全球化支持名称。如果目标缓冲区的大小小于名称的长度，则该函数返回OCI_ERROR 。

srcbuf（IN）

指向NULL终止的全球化支持名称的指针。如果它不是有效名称，则该函数返回OCI_ERROR 。

标志（IN）

它指定名称映射的方向，并可以采用以下值：

  • OCI_NLS_CS_IANA_TO_ORA ：将字符集名称从IANA映射到Oracle数据库
  • OCI_NLS_CS_ORA_TO_IANA ：将Oracle数据库中的字符集名称映射到IANA
  • OCI_NLS_LANG_ISO_TO_ORA ：将语言名称从ISO映射到Oracle数据库
  • OCI_NLS_LANG_ORA_TO_ISO ：将Oracle数据库中的语言名称映射到ISO
  • OCI_NLS_LOCALE_A2_ISO_TO_ORA ：将区域设置名称从A2 ISO映射到Oracle数据库
  • OCI_NLS_LOCALE_ORA_TO_A2_ISO ：将Oracle数据库中的区域设置名称映射到A2 ISO
  • OCI_NLS_TERR_ISO_TO_ORA ：将地域名称从ISO映射到Oracle数据库
  • OCI_NLS_TERR_ORA_TO_ISO ：将Oracle数据库中的地域名称映射到ISO
  • OCI_NLS_TERR_ISO3_TO_ORA ：将区域名称从3个字母的ISO缩写映射到Oracle数据库
  • OCI_NLS_TERR_ORA_TO_ISO3 ：将Oracle数据库中的地域名称映射到3个字母的ISO缩写

返回

OCI_SUCCESS ; OCI_INVALID_HANDLE ;或OCI_ERROR 。
