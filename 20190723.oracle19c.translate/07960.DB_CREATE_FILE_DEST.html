1.74 DB_CREATE_FILE_DEST

DB_CREATE_FILE_DEST指定Oracle管理的数据文件的默认位置。

属性 描述

参数类型

串

句法

DB_CREATE_FILE_DEST = directory | disk group

默认值

没有默认值。

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

是

如果未指定任何DB_CREATE_ONLINE_LOG_DEST_ n初始化参数，则此位置还将用作Oracle管理的控制文件和联机重做日志的缺省位置。

如果将文件系统目录指定为默认位置，则该目录必须已存在; Oracle不会创建它。该目录必须具有允许Oracle在其中创建文件的适当权限。Oracle为文件生成唯一的名称，因此创建的文件是Oracle托管文件。
