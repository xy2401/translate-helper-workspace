第三部分外部表格

本部分讨论的主题包括成功使用外部表所需的信息。

  • 外部表概念
    外部表功能是对现有SQL * Loader功能的补充。它使您可以访问外部源中的数据，就像它在数据库中的表中一样。
  • ORACLE_LOADER访问驱动程序
    ORACLE_LOADER访问驱动程序提供了一组对ORACLE_LOADER类型的外部表唯一的访问参数。
  • ORACLE_DATAPUMP访问驱动程序
    ORACLE_DATAPUMP访问驱动程序提供一组访问参数，这些参数对于ORACLE_DATAPUMP类型的外部表是唯一的。
  • ORACLE_HDFS和ORACLE_HIVE访问驱动程序
    用于创建使用ORACLE_HDFS或ORACLE_HIVE访问驱动程序的外部表的属性在SQL CREATE TABLE ORGANIZATION EXTERNAL语句中的ACCESS PARAMETERS的opaque_format_spec子句中指定。
