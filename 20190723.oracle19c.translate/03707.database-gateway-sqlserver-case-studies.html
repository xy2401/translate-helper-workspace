演示要求

案例研究假设满足了这些要求：

  • 网关演示表和存储过程安装在SQL Server数据库中

  • Oracle数据库有一个名为SCOTT的帐户，密码为TIGER

  • Oracle数据库有一个名为GTWLINK的数据库链接（设置为用户SCOTT公共或私有），它将网关作为SCOTT连接到SQL Server数据库，密码为TIGER2

    例如，您可以按如下方式创建数据库链接：

    SQL>创建数据库链接GTWLINK连接到SCOTT 2，由TIGER2使用'GTWSID'识别;

  • Oracle Net Services已正确配置并正在运行

创建演示表

案例研究基于GTW_EMP ， GTW_DEPT和GTW_SALGRADE表以及存储过程InsertDept和GetDept 。如果尚未在SQL Server数据库中创建演示表和存储过程，请使用用于创建它们的bldmsql.sql脚本。输入以下内容：

> isql -USCOTT -PTIGER2 -ibldmsql.sql

该脚本相应地在SQL Server数据库中创建演示表和存储过程：

CREATE TABLE GTW_EMP（EMPNO SMALLINT NOT NULL ENAME VARCHAR（10），JOB VARCHAR（9），MGR SMALLINT，HIREDATE DATETIME，SAL NUMERIC（7,2），COMM NUMERIC（7,2），DEPTNO SMALLINT）go CREATE TABLE GTW_DEPT（ DEPTNO SMALLINT NOT NULL，DNAME VARCHAR（14），LOC VARCHAR（13））去创建表GTW_SALGRADE（等级钱，LOSAL NUMERIC（9,4），HISAL NUMERIC（9,4））去DROP PROCEDURE InsertDept go CREATE PROCEDURE InsertDept（ @dno INTEGER，@ DNS VARCHAR（14），@ loc VARCHAR（13））AS插入GTW_DEPT值（@dno，@ dname，@ loc）去DROP PROCEDURE GetDept go CREATE PROCEDURE GetDept（@dno INTEGER，@ name VARCHAR（ 14）OUTPUT）AS SELECT @ dname = DNAME FROM GTW_DEPT WHERE DEPTNO = @dno go
