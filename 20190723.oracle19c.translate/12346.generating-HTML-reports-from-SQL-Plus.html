7从SQL * Plus生成报告

本章介绍如何生成包含查询结果的HTML和CSV报告。本章包含以下主题：

  • 关于使用命令行SQL * Plus创建报告

7.1关于使用命令行SQL * Plus创建报告

除了纯文本输出之外，SQL * Plus命令行界面还允许您生成完整的网页，可以嵌入网页的HTML输出或CSV格式的数据。您可以使用SQLPLUS -MARKUP“HTML ON”或SET MARKUP HTML ON SPOOL ON来生成使用<HTML>和<BODY>标记自动封装的完整HTML页面。您可以使用SQLPLUS -MARKUP“CSV ON”或SET MARKUP CSV ON来生成CSV格式的报告。

默认情况下，使用MARKUP HTML ON检索的数据以HTML格式输出，但您可以选择将输出定向到HTML <PRE> 标记，以便它在Web浏览器中显示与SQL * Plus中显示的完全相同。有关这些命令的更多信息，请参见SQLPLUS MARKUP选项和SET MARKUP命令。

在程序脚本中嵌入SQL * Plus时，SQLPLUS -MARKUP“HTML ON”非常有用。启动时，它会在执行任何命令之前输出HTML和BODY标记。所有后续输出都是HTML格式，直到SQL * Plus终止。

-SILENT和-RESTRICT命令行选项可与-MARKUP有效地一起使用，以抑制SQL * Plus提示和标题信息的显示，并限制某些命令的使用。

SET MARKUP HTML ON SPOOL ON为每个后续假脱机文件生成一个HTML页面。执行SPOOL OFF或SQL * Plus退出时，将关闭假脱机文件中的HTML标记。

您可以使用SET MARKUP HTML ON SPOOL OFF来生成适合嵌入现有网页的HTML输出。以这种方式生成的HTML输出没有<HTML>或<BODY>标记。

通过在SQL * Plus命令行中使用-M [ARKUP] CSV ON选项，可以在登录用户会话时启用CSV标记。有关更多信息，请参阅SQL * Plus程序语法 。登录到用户会话后，可以使用SET MARKUP CSV ON命令启用CSV标记。

您可以使用DELIMITER选项指定分隔符。您还可以使用QUOTE OFF输出不带引号的文本。

您可以使用SET FEEDBACK命令的ONLY选项来禁止显示查询返回的数据。显示查询选择和返回的行数。

7.1.1创建HTML报告

在SQL * Plus会话期间，以交互方式使用SET MARKUP命令将HTML写入假脱机文件。您可以在a中查看输出网页浏览器。

SET MARKUP HTML ON SPOOL ON仅指定SQL * Plus输出将是HTML编码的，它不会创建或开始写入输出文件。必须使用SQL * Plus SPOOL命令开始生成假脱机文件。然后，此文件包含HTML标记，包括<HTML>和</ HTML>。

创建HTML文件时，指定.html或.htm文件扩展名是非常重要和方便的，它们是HTML文件的标准文件扩展名。这使您可以轻松识别输出文件的类型，还可以使Web浏览器识别并正确显示HTML文件。如果未指定扩展名，则使用默认的SQL * Plus文件扩展名。

你用 SPOOL OFF或退出以将最终HTML标记附加到假脱机文件，然后将其关闭。如果输入另一个SPOOL文件名命令，则关闭当前假脱机文件，如SPOOL OFF或EXIT，并创建具有指定名称的新HTML假脱机文件。

您可以使用SET MARKUP命令根据需要启用或禁用HTML输出。

7.1.1.1 HTML实体

某些字符，<，>，“和＆在HTML中具有预定义。在上一个示例中，您可能已经注意到，只要您输入SET MARKUP HTML ON命令，>字符就会被＆gt;替换。要使这些字符能够在Web浏览器中显示，HTML将提供要使用的字符实体。

Web浏览器显示>字符，但HTML编码文件中的实际文本是HTML实体，＆gt;。SET MARKUP选项ENTMAP控制HTML实体的替换。ENTMAP默认设置为ON。它确保字符<，>，“和＆”始终由表示这些字符的HTML实体替换。这可以防止Web浏览器在SQL * Plus命令或查询产生的数据中出现错误解释这些字符。

您可以使用SET MARKUP HTML ENTMAP ON在全局级别设置ENTMAP，也可以使用COLUMN column_name ENTMAP ON在列级别设置ENTMAP。

7.1.2创建CSV报告

通过在SQL * Plus命令行中使用-M [ARKUP] CSV ON选项，可以在登录用户会话时启用CSV标记。有关更多信息，请参阅SQL * Plus程序语法 。登录到用户会话后，可以使用SET MARKUP CSV ON命令启用CSV标记。

您可以使用DELIMITER选项指定分隔符。您还可以使用QUOTE OFF输出不带引号的文本。

有关创建CSV报告的详细信息，请参阅SET MARKUP CSV ON 。

7.1.3关于抑制报表中SQL * Plus命令的显示

当与-MARKUP结合使用时，SQLPLUS -SILENT选项在使用CGI脚本或操作系统脚本生成嵌入式SQL * Plus报表时特别有用。它禁止显示SQL * Plus命令和SQL * Plus横幅。HTML输出仅显示SQL查询产生的数据。

您还可以使用SET ECHO OFF来禁止在使用START命令执行的脚本中显示每个命令。
