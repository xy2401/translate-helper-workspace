11 Oracle数据库Real Application Security PL / SQL包

本章介绍Oracle Database Real Application Security可用的PL / SQL包。

表11-1列出了这些包。每个包的详细信息如下表所示。

表11-1 Oracle数据库实际应用程序安全PL / SQL包

PL / SQL包 描述

DBMS_XS_SESSIONS包

包括用于管理应用程序会话的子程序。

XS_ACL包

包括用于创建，管理和删除访问控制列表（ACL）以及添加和删除参数值的子程序。

XS_ADMIN_UTIL包

包括辅助子程序。

XS_DATA_SECURITY包

包括用于创建，管理和删除数据安全策略，关联数据领域约束，列约束和ACL参数的子程序。

XS_DATA_SECURITY_UTIL包

包括子程序，用于为用户表计划静态ACL的自动刷新，并将ACL刷新模式更改为on-commit或按需刷新。

XS_DIAG包

包括子程序，用于诊断Real Application Security对象中的潜在问题并报告已识别的不一致性。

XS_NAMESPACE包

包括用于创建，管理和删除命名空间模板和属性的子程序。

XS_PRINCIPAL包

包括用于创建，管理和删除应用程序用户和角色的子程序。

XS_SECURITY_CLASS包

包括用于创建，管理和删除安全类及其权限的子程序。还包括用于管理安全性类继承的子程序。
