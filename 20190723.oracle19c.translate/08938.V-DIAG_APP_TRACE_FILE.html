7.177 V $ DIAG_APP_TRACE_FILE

V$DIAG_APP_TRACE_FILE包含有关当前容器（PDB）的自动诊断存储库（ADR）中存在的所有跟踪文件的信息，其中包含应用程序跟踪数据（SQL_TRACE或OPTIMIZER_TRACE事件数据）。此视图还支持GV $全局视图。

柱 数据类型 描述

ADR_HOME

VARCHAR2（444）

当前ADR主页的路径

trace_filename的

VARCHAR2（68）

显示进程跟踪文件的名称

CHANGE_TIME

TIMESTAMP（3）带时区

显示进程跟踪文件的更改时间时间戳

MODIFY_TIME

TIMESTAMP（3）带时区

显示进程跟踪文件的上次修改时间戳

SQL_TRACE

VARCHAR2（1）

标识进程跟踪文件是否包含SQL_TRACE数据。可能的值包括：

  • Y ：当进程跟踪文件包含SQL_TRACE事件数据时，使用此值

  • N ：该值表示进程跟踪不包含SQL_TRACE数据

OPTIMIZER_TRACE

VARCHAR2（1）

标识进程跟踪文件是否包含OPTIMIZER_TRACE数据。可能的值包括：

  • Y ：当进程跟踪文件包含OPTIMIZER_TRACE事件数据时，使用此值

  • N ：该值表示进程跟踪不包含OPTIMIZER_TRACE数据

CON_ID

数

数据所属容器的ID。可能的值包括：

  • 0 ：此值用于包含与整个CDB相关的数据的行。此值还用于非CDB中的行。

  • 1 ：此值用于包含仅与根有关的数据的行

  • n ：其中n是包含数据的行的适用容器ID
