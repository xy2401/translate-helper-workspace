13.26 RefAny Class

RefAny类旨在支持对任何类型的引用。其主要目的是处理泛型引用并允许在类型层次结构中转换Ref 。RefAny对象可以用作任何两种类型之间的中介， Ref< x >和Ref< y > ，其中x和y是不同的类型。
