1.154 JAVA_JIT_ENABLED

JAVA_JIT_ENABLED为Oracle Java虚拟机（OracleJVM）环境启用或禁用实时（JIT）编译器。

属性 描述

参数类型

布尔

默认值

依赖于操作系统

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

是的|假

基本

没有

对于支持JIT编译器的平台，此参数的默认值为true ;否则默认值为false 。尝试在不支持的平台上将此参数设置为true将导致错误。
