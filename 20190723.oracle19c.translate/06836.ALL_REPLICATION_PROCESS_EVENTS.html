3.30 ALL_REPLICATION_PROCESS_EVENTS

ALL_REPLICATION_PROCESS_EVENTS提供有关当前用户可访问的复制过程事件的信息。

相关视图

DBA_REPLICATION_PROCESS_EVENTS提供有关数据库中复制过程事件的信息。

柱 数据类型 空值 描述

STREAMS_TYPE

VARCHAR2（10）

流类型：

  • XStream的

  • 加洲的金门大桥

PROCESS_TYPE

VARCHAR2（27）

流程类型：

  • 捕获

  • 捕获服务器

  • 申请协调员

  • 应用服务器

  • 应用网络接收器

  • 申请读者

  • 应用哈希服务器

STREAMS_NAME

VARCHAR2（128）

不是NULL

流名称

EVENT_NAME

VARCHAR2（128）

活动名称：

  • 开始

  • 停

  • 中止

  • 创建

  • 下降

  • 参数变化

  • HANDLER CREATE

  • HANDLER REMOVE

  • 改变
