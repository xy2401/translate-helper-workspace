手动IP地址配置举例

如果您选择不使用GNS，则在安装之前必须配置公共，虚拟和专用IP地址。

检查ping命令是否可以访问默认网关。要查找默认网关，请使用route命令，如操作系统的帮助实用程序中所述。

您无需为互连提供专用名称。如果要对互连进行名称解析，则可以在hosts文件或DNS中配置专用IP名称。但是，Oracle Clusterware会将安装期间定义的接口上的互连地址分配为专用接口（例如eth1），以及用于私有子网的子网。

SCAN解析的地址由Oracle Clusterware分配，因此它们不会固定到特定节点。要启用VIP故障转移，上表中显示的配置定义SCAN地址以及同一子网192.0.2上两个节点的公共和VIP地址。

注意：

所有主机名必须符合RFC-952标准，该标准允许使用字母数字字符，但不允许使用下划线（“_”）。
