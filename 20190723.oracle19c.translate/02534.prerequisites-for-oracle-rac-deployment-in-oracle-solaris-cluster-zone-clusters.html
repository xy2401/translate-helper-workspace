Oracle Solaris Cluster区域群集中Oracle RAC部署的先决条件

查看在Oracle Solaris Cluster区域群集中部署Oracle Real Application Clusters（Oracle RAC）的先决条件。

  • 安装了带有DNS和NIS名称服务的Oracle Solaris 11。

  • Oracle Solaris Cluster 4随ha-cluster-full软件包一起安装。

  • Oracle Solaris 11内核参数在全局区域的/etc/system文件中配置。

  • 共享磁盘也称为/dev/did/rdsk设备。

  • 已在公共网络上分配Oracle虚拟IP（VIP）和单客户端访问名称（SCAN）IP要求。

  • 对于最高4.2的Oracle Solaris Cluster版本，必须在IPMP组中配置用于数据服务流量的每个公共网络适配器。从Oracle Solaris Cluster 4.3开始，您可以使用任何公共网络管理（PNM）对象配置适配器，该对象包括由链路聚合直接支持的IPMP组，链路聚合和虚拟网络接口卡（VNIC）。

  • 从Oracle Solaris Cluster 4.3开始，用于数据服务流量的公共网络适配器必须属于PNM对象，该对象包括由链路聚合直接支持的IPMP组，链路聚合和VNIC。
