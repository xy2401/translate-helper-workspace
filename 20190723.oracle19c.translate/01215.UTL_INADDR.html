258 UTL_INADDR

UTL_INADDR包提供PL / SQL过程以支持Internet寻址。它提供了一个API来检索本地和远程主机的主机名和IP地址。

本章包含以下主题：

  • 安全模型

  • 例外

  • 例子

  • UTL_INADDR子程序摘要

258.1 UTL_INADDR安全模型

此包是一个调用者权限包，这意味着必须在分配给他或她希望连接的远程网络主机的访问控制列表中授予调用用户connect权限。

注意：

有关管理细粒度访问的详细信息，请参阅“ Oracle数据库安全指南” 。

258.4.1 GET_HOST_ADDRESS函数

此函数检索指定主机的IP地址。

句法

UTL_INADDR.GET_HOST_ADDRESS（主机IN VARCHAR2 DEFAULT NULL）RETURN host_address VARCHAR2;

参数

表258-3 GET_HOST_ADDRESS函数参数

参数 描述

主办

用于检索IP地址的主机的名称。

返回值

表258-4 GET_HOST_ADDRESS函数返回值

参数 描述

host_address

指定主机的IP地址，如果主机为NULL则为本地主机的IP地址。

例外

UNKNOWN_HOST:指定的IP地址未知

使用说明

获取当前主机的主机名或IP地址的权限由LOCALHOST上的resolve特权控制。

258.4.2 GET_HOST_NAME功能

此函数检索给定其IP地址的本地或远程主机的名称。

句法

UTL_INADDR.GET_HOST_NAME（ip IN VARCHAR2 DEFAULT NULL）RETURN host_name VARCHAR2;

参数

表258-5 GET_HOST_NAME函数参数

参数 描述

IP

用于确定其主机名的主机的IP地址。如果ip不为NULL,则返回主机的正式名称及其域名。如果为NULL,则返回本地主机的名称，并且名称不包含本地主机所属的域。

返回值

表258-6 GET_HOST_NAME函数返回值

参数 描述

主机名

指定IP地址的本地或远程主机的名称。

例外

UNKNOWN_HOST:指定的IP地址未知

使用说明

获取当前主机的主机名或IP地址的权限由LOCALHOST上通过DBMS_NETWORK_ACL_ADMIN授予的resolve权限控制。
