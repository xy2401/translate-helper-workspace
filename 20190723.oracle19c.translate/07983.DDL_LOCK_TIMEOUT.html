1.98 DDL_LOCK_TIMEOUT

DDL_LOCK_TIMEOUT指定DDL语句在DML锁定队列中等待的时间限制。

属性 描述

参数类型

整数

默认值

0

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

价值范围

0到1,000,000（以秒为单位）

基本

没有

值为零表示NOWAIT的状态。最大值1,000,000秒将导致DDL语句永远等待获取DML锁。

如果在超时期限到期之前未获取锁定，则返回错误。
