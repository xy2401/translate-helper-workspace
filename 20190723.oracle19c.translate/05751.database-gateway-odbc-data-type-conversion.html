数据类型转换

Oracle将ODBC数据类型映射到支持的Oracle数据类型。返回查询结果时，Oracle会将ODBC数据类型转换为Oracle数据类型。

Oracle Database Gateway for ODBC将ODBC兼容数据源中使用的数据类型映射到支持的Oracle数据类型。返回查询结果时，Oracle数据库会将ODBC数据类型转换为Oracle数据类型。例如，ODBC数据类型SQL_TYPE_TIMESTAMP将转换为Oracle的DATE数据类型。

如果表包含Oracle Database Gateway for ODBC不支持其数据类型的列，则列信息不会返回到Oracle数据库。
