2.90 ALL_CREDENTIALS

ALL_CREDENTIALS列出用户可见的所有凭据。

相关观点

  • DBA_CREDENTIALS列出数据库中的所有凭据。

  • USER_CREDENTIALS列出当前用户拥有的凭据。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

不是NULL

凭证的所有者

CREDENTIAL_NAME

VARCHAR2（128）

不是NULL

凭证的名称

用户名

VARCHAR2（128）

用于登录远程数据库或远程或本地操作系统的用户的名称

WINDOWS_DOMAIN

VARCHAR2（30）

对于Windows目标，登录时使用的Windows域

评论

VARCHAR2（240）

关于凭证的评论

启用

VARCHAR2（5）

指示此凭据是否已启用（ TRUE ）（ FALSE ）

注意：

DBMS_CREDENTIAL列出可用于运行外部过程的凭据，或用于远程或外部作业的DBMS_SCHEDULER ，或用于存储或从操作系统检索文件的凭据。

如果禁用了凭据，则上述尝试使用凭据的任何操作都将失败。
