此图显示了四个分片（从左到右编号为1到4），由一条带有标注“Tablespace Set tbs1”的标注线包围。每个分片都包含一系列的块，每个块都包含一个表空间，而表空间又包含一个分区，例如：

分片1包含块1到120的范围，每个块分别包含表空间tbs_1-1到tbs_1-120，tbs_1-1包含分区P_1，而表空间tbs_1-120包含分区P_120。

碎片2,3和4每个包含120个块的范围，每个块包含一个表空间和具有编号方案的分区：块n（tbs_1-n（P_n））。
