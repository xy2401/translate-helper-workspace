11.5用于中间层路由支持的UCP API

OracleShardRoutingCache类

此类扩展了UCP的内部分片路由缓存，并使基本路由缓存功能可用于WebLogic Server或中间层路由器或负载平衡器。

公共类OracleShardRoutingCache扩展了ShardRoutingCache

该类提供了OracleShardRoutingCache(Properties dataSourceProps)和Set<ShardInfo> getShardInfoForKey(OracleShardingKey key, OracleShardingKey superKey)方法。

ShardInfo接口

ShardInfo接口实例封装了唯一的分片名称和优先级。然后，可以将唯一的分片名称映射到连接到特定分片的中间层服务器。
