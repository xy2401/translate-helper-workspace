6.3应用程序中使用的对象视图

对象视图行中的数据可能来自多个表，但对象视图仍然在一次操作中遍历网络。该实例在客户端对象缓存中显示为C或C ++结构或PL / SQL对象变量。您可以像任何其他本机结构一样操纵它。

您可以像引用对象表一样在SQL语句中引用对象视图。例如，对象视图可以出现在SELECT列表， UPDATE-SET子句或WHERE子句中。

您还可以在对象视图上定义对象视图。

您可以使用用于对象表中对象的相同OCI调用来访问客户端上的对象视图数据。例如，您可以使用OCIObjectPin()来固定 REF和 OCIObjectFlush()用于将对象刷新到服务器。在对象视图中更新或刷新对象到数据库时，数据库将更新对象视图。
