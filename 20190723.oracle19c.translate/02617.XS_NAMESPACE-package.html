XS_NAMESPACE包

XS_NAMESPACE包中包含用于创建，管理和删除命名空间模板和属性的子程序。

本节包括以下主题：

  • 安全模型

  • 常量

  • 对象类型，构造函数，同义词和授权

  • XS_NAMESPACE子程序摘要

安全模型

XS_NAMESPACE包是在SYS架构下创建的。DBA角色被授予ADMIN_ANY_SEC_POLICY ，允许它管理名称空间模板和属性。

常量

下列是属性事件常量：

NO_EVENT CONSTANT PLS_INTEGER：= 0; FIRSTREAD_EVENT CONSTANT PLS_INTEGER：= 1; UPDATE_EVENT CONSTANT PLS_INTEGER：= 2; FIRSTREAD_PLUS_UPDATE_EVENT CONSTANT PLS_INTEGER：= 3;

本节介绍以下XS_NAMESPACE子程序：

CREATE_TEMPLATE过程

CREATE_TEMPLATE过程创建一个新的命名空间模板。

句法

XS_NAMESPACE.CREATE_TEMPLATE（名称IN VARCHAR2，attr_list IN XS $ NS_ATTRIBUTE_LIST：= NULL，schema IN VARCHAR2：= NULL，package IN VARCHAR2：= NULL，function IN VARCHAR2：= NULL，acl IN VARCHAR2：='SYS.NS_UNRESTRICTED_ACL'description IN VARCHAR2：= NULL）;

参数

参数 描述

名称

要创建的命名空间模板的名称。

ATTR_LIST

命名空间模板中包含的属性及其默认值和关联的属性事件，例如UPDATE_EVENT 。

模式

包含命名空间模板的处理函数的模式。

包

包含命名空间模板的处理函数的包。

功能

命名空间模板的处理函数。在发生属性事件时调用处理函数。

ACL

此命名空间模板的ACL名称。如果未提供ACL，则缺省值为预定义的ACL SYS.NS_UNRESTRICTED_ACL ，它允许应用程序用户进行不受限制的属性操作。

描述

命名空间模板的可选描述字符串。

例子

以下示例创建名为POAttrs的命名空间模板。命名空间模板包含attrlist定义的attrlist列表。命名空间模板的处理函数称为Populate_Order_Func 。此处理函数是Orders_Pckg包的一部分，该包包含在SCOTT模式中。命名空间模板在模板上设置了NS_UNRESTRICTED_ACL ，允许对从模板创建的命名空间进行无限制的操作。 

DECLARE attrlist XS $ NS_ATTRIBUTE_LIST; BEGIN attrlist：= XS $ NS_ATTRIBUTE_LIST（）; attrlist.extend（2）; attrlist（1）：= XS $ NS_ATTRIBUTE（'desc'，'general'）; attrlist（2）：= XS $ NS_ATTRIBUTE（name =>'item_no'，attribute_events => XS_NAMESPACE.FIRSTREAD_EVENT）; SYS.XS_NAMESPACE.CREATE_TEMPLATE（'POAttrs'，attrlist，'SCOTT'， 'Orders_Pckg'，'Populate_Order_Func'， 'SYS.NS_UNRESTRICTED_ACL'， '采购订单属性'）;结束;

ADD_ATTRIBUTES程序

ADD_ATTRIBUTES过程将一个或多个属性添加到现有命名空间模板。

句法

XS_NAMESPACE.ADD_ATTRIBUTES（模板IN VARCHAR2，属性IN VARCHAR2，default_value IN VARCHAR2：= NULL，attribute_events IN PLS_INTEGER：= XS_NAMESPACE.NO_EVENT）; XS_NAMESPACE.ADD_ATTRIBUTES（模板IN VARCHAR2，attr_list IN XS $ NS_ATTRIBUTE_LIST）;

参数

参数 描述

模板

要添加属性的命名空间模板的名称。

属性

要添加的属性的名称。

ATTR_LIST

要添加的属性列表。

默认值

属性的默认值。

attribute_events

与属性关联的属性事件，例如update事件。

例子

以下示例将名为item_type的属性添加到POAttrs名称空间。它还为添加的新属性指定默认值和属性事件。 

BEGIN SYS.XS_NAMESPACE.ADD_ATTRIBUTES（template =>'POAttrs'，attribute =>'item_type'，default_value =>'generic'，attribute_events => XS_NAMESPACE.update_event）;结束;

REMOVE_ATTRIBUTES程序

REMOVE_ATTRIBUTES过程从命名空间模板中删除一个或多个属性。如果未指定任何属性名称，则会从命名空间模板中删除所有属性。

句法

XS_NAMESPACE.REMOVE_ATTRIBUTES（模板IN VARCHAR2，属性IN VARCHAR2）; XS_NAMESPACE.REMOVE_ATTRIBUTES（模板IN VARCHAR2，attr_list IN XS $ LIST）; XS_NAMESPACE.REMOVE_ATTRIBUTES（模板IN VARCHAR2）;

参数

参数 描述

模板

要从中删除属性的命名空间模板的名称。

属性

要删除的属性的名称。

ATTR_LIST

要删除的属性名称列表。

例子

以下示例从POAttrs名称空间中删除item_type属性。 

BEGIN SYS.XS_NAMESPACE.REMOVE_ATTRIBUTES（'POAttrs'，'item_type'）;结束;

以下示例从POAttrs命名空间模板中删除所有属性。 

BEGIN SYS.XS_NAMESPACE.REMOVE_ATTRIBUTES（'POAttrs'）;结束;

SET_HANDLER程序

SET_HANDLER过程为指定的命名空间模板分配处理函数。

句法

XS_NAMESPACE.SET_HANDLER（模板IN VARCHAR2，模式IN VARCHAR2，包IN VARCHAR2，函数IN VARCHAR2）;

参数

参数 描述

模板

要为其设置处理函数的命名空间模板的名称。

模式

包含处理程序包和函数的模式。

包

包含处理函数的包的名称。

功能

名称空间模板的处理函数的名称。

例子

以下示例为POAttrs命名空间模板设置一个名为Populate_Order_Func的处理函数。 

BEGIN SYS.XS_NAMESPACE.SET_HANDLER（'POAttrs'，'SCOTT'，'Orders_Pckg'，'Populate_Order_Func'）;结束;

SET_DESCRIPTION过程为指定的命名空间模板设置描述字符串。

句法

XS_NAMESPACE.SET_DESCRIPTION（模板IN VARCHAR2，描述IN VARCHAR2）;

参数

参数 描述

模板

要设置其描述的命名空间模板的名称。

描述

指定命名空间模板的描述字符串。

例子

以下示例为POAttrs命名空间模板设置描述字符串。 

BEGIN SYS.XS_NAMESPACE.SET_DESCRIPTION（'POAttrs'，'采购订单属性'）;结束;

DELETE_TEMPLATE过程

DELETE_TEMPLATE过程删除指定的命名空间模板。

句法

XS_NAMESPACE.DELETE_TEMPLATE（模板IN VARCHAR2，delete_option IN PLS_INTEGER：= XS_ADMIN_UTIL.DEFAULT_OPTION）;

参数

参数 描述

模板

要删除的命名空间模板的名称。

delete_option

要使用的删除选项。对于命名空间模板，以下选项的行为是相同的：

  • DEFAULT_OPTION：

    默认选项允许仅在未在其他位置引用时删除命名空间模板。如果有其他实体引用命名空间模板，则无法删除命名空间模板。

  • CASCADE_OPTION ：

    级联选项删除命名空间模板以及对它的任何引用。删除命名空间模板的用户也会删除这些引用。

  • ALLOW_INCONSISTENCIES_OPTION：

    allow inconsistencies选项允许您删除实体，即使其他实体具有对它的后期绑定引用。如果实体是早期依赖项的一部分，则删除失败并引发错误。

例子

以下示例使用默认删除选项删除POAttrs命名空间模板。

BEGIN SYS.XS_NAMESPACE.DELETE_TEMPLATE（'POAttrs'，XS_ADMIN_UTIL.DEFAULT_OPTION）;结束;
