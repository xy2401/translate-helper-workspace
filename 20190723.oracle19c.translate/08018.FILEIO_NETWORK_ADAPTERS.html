1.125 FILEIO_NETWORK_ADAPTERS

FILEIO_NETWORK_ADAPTERS指定可用于访问磁盘存储的网络适配器列表。

属性 描述

参数类型

串

句法

FILEIO_NETWORK_ADAPTERS = adapter_name [, adapter_name ] ...

默认值

没有默认值。

可修改

没有

可在PDB中修改

没有

价值范围

一个或多个网络适配器名称，以逗号分隔

基本

没有

在数据库文件驻留在网络连接存储中的平台上，此参数为存储访问库提供可用于访问存储的网络适配器列表。

网络适配器名称是可通过主机名数据库或使用网络信息服务访问的网卡的完全限定地址名称。适配器名称的组件由句点分隔。例如，以下是完全限定的适配器名称：

ib1.oracle.com
