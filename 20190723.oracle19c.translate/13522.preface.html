前言

Oracle Database Gateway for WebSphere MQ提供对WebSphere MQ服务的访问。此网关需要一个能够运行64位应用程序的系统。

目标受众

本指南适用于负责安装，配置或管理Oracle Database for WebSphere MQ的任何人。它也适用于编写访问消息队列系统的应用程序的开发人员，特别是那些需要访问WebSphere MQ和其他非Oracle消息队列系统所拥有的队列的开发人员以及Oracle Advanced Queuing（AQ）拥有的队列。

如果您负责以下任务，请阅读本指南：

  • 管理网关

  • 设置网关安全性

  • 使用网关

  • 诊断网关错误

在使用本指南之前，您必须先了解操作系统，Oracle数据库网关，PL / SQL，Oracle数据库和WebSphere MQ软件的基础知识，然后再使用本指南来安装，配置或管理网关。

产品名称

此产品的完整名称是Oracle Database Gateway for WebSphere MQ，也称为DG4MQ。

印刷约定

本指南中使用以下印刷约定：

惯例 描述

等宽

等宽字体类型表示命令，目录名称，用户名，路径名和文件名。

斜体

斜体类型表示变量，包括文件名的可变部分。它也用于强调和书籍。

大写

大写字母表示结构化查询语言（SQL）保留字，初始化参数和环境变量。

胆大

粗体显示屏幕名称和字段。

SQL * Plus提示

SQL * Plus提示符SQL>出现在SQL语句和SQL * Plus命令示例中。在提示符下输入您的回复。请勿在响应中输入提示“SQL>”的文本。

命令语法

命令语法以monospace字体显示。美元字符（$），数字符号（＃）或百分号（％）是UNIX命令提示符。不要将它们作为命令的一部分输入。本指南中使用以下命令语法约定：

惯例 描述

反斜杠

反斜杠是UNIX命令延续字符。它用于命令示例，这些示例太长而无法放在一行上。输入显示的命令（带反斜杠）或在没有反斜杠的单行中输入：

dd if = / dev / rdsk / c0t1d0s6 = = dev / rst0 bs = 10b \ count = 10000

大括号{}

大括号表示所需物品：

。DEFINE {macro1}

括号[]

括号表示可选项：

cvtcrt termname [ outfile ]

省略号...

省略号表示任意数量的类似项目：

CHKVAL fieldname value1 value2 ... valueN

斜体

斜体类型表示变量。替换变量的值：

LIBRARY_NAME

垂直线|

垂直线表示括号或括号内的选择：

文件大小 [K | M]
