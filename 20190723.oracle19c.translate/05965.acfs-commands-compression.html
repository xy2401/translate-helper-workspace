用于压缩的Oracle ACFS命令行工具

本主题提供了管理Oracle ACFS压缩的命令的摘要。

表16-99列出了Oracle ACFS压缩命令以及简要说明。有关Oracle ACFS压缩的概述，请参阅Oracle ACFS压缩 。

有关运行Oracle ACFS acfsutil命令的更多信息，请参阅关于使用Oracle ACFS命令行工具 。

表16-99 Oracle ACFS压缩命令摘要

命令 描述

acfsutil压缩副本

复制并压缩目录中的一个或多个文件。

acfsutil压缩信息

显示有关文件的压缩信息。

acfsutil压缩

启用Oracle ACFS文件压缩。

acfsutil压缩

禁用Oracle ACFS文件压缩。

acfsutil压缩副本

目的

压缩和复制文件。

语法和说明

acfsutil compress copy -h acfsutil compress copy [-v] [-f] {-c size source | [-n] {-r source | 来源 [...]}
                       TARGET_DIR

下表包含acfsutil compress copy命令可用的选项。

表16-100 acfsutil compress copy命令的选项

选项 描述

-v

指定详细模式。

-F

指定覆盖任何现有副本。

—c size source

复制单个源文件并使用压缩单元（CU） 大小进行复制。

大小的格式是n K支持的CU大小为： 4K ， 8K ， 16K ， 32K ， 64K和128K字节。

—n source

指定不压缩创建的副本。

—r source

指定从单个源目录开始递归复制文件和目录。

资源 [...]

指定要复制的源文件。

TARGET_DIR

指定要复制到的目标目录。

除非使用-f选项，否则acfsutil compress copy命令不会复制现有文件。如果未使用-f ，则在找到第一个现有目标时终止复制操作。

除非指定了-n选项，否则该命令不会复制到能够支持压缩的Oracle ACFS文件系统（如果禁用压缩）。

示例16-96使用acfsutil compress copy命令

$ / sbin / acfsutil压缩拷贝-v -f my_file1 my_file2 / my_target_directory

acfsutil压缩

目的

禁用Oracle ACFS压缩。

语法和说明

acfsutil压缩off -h acfsutil压缩mount_point

下表包含acfsutil compress off命令可用的选项。

表16-102 acfsutil压缩关闭命令的选项

选项 描述

装入点

指定安装文件系统的目录。

运行acfsutil compress off ，以下内容适用：

  • 禁用压缩后创建的文件是未压缩创建的。

  • 文件系统中的所有压缩文件都保持压缩状态。

您必须具有root或系统管理员权限才能运行此命令。

例16-98使用acfsutil压缩关闭命令

$ / sbin / acfsutil压缩off / my_mount_point

acfsutil压缩

目的

启用Oracle ACFS压缩。

语法和说明

acfsutil压缩on -h acfsutil压缩[-a algorithm ] mount_point

下表包含acfsutil compress on命令可用的选项。

表16-103 acfsutil compress on命令的选项

选项 描述

- 算法

指定默认压缩算法。目前只有lzo可用，这是使用的默认算法。

装入点

指定安装文件系统的目录。

注意：

Oracle数据库文件不支持Oracle ACFS压缩。用于保存数据库文件的Oracle ACFS文件系统不应使用Oracle ACFS压缩进行压缩。

运行acfsutil compress on ，以下内容适用：

  • 启用压缩后创建的文件默认是压缩的。

  • 文件系统中的所有未压缩文件都保持未压缩状态。

您必须具有root或系统管理员权限才能运行此命令。

例16-99使用acfsutil compress on命令

$ / sbin / acfsutil压缩/ my_mount_point
