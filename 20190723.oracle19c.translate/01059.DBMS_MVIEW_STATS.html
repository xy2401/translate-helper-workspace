109 DBMS_MVIEW_STATS

DBMS_MVIEW_STATS包提供了一个界面，用于管理物化视图刷新操作的统计信息的收集和保留。

109.1 DBMS_MVIEW_STATS概述

您可以使用DBMS_MVIEW_STATS包中包含的过程来管理DBMS_MVIEW_STATS化视图刷新操作的统计信息的收集和保留。这包括收集这些统计信息的级别和粒度以及它们在数据库中保留的持续时间。

您还可以为控制统计信息收集的参数设置数据库级系统默认值。

109.2 DBMS_MVIEW_STATS安全模型

有关使用子程序所需权限的信息，请参阅每个子程序中的“使用说明”部分。

109.3 DBMS_MVIEW_STATS子程序摘要

该表列出了DBMS_MVIEW_STATS子程序并简要描述了它们。

表109-1 DBMS_MVIEW_STATS包子程序

子程序 描述

PURGE_REFRESH_STATS程序

清除早于指定保留期的物化视图刷新操作的统计信息。

SET_MVREF_STATS_PARAMS过程

设置定义物化视图刷新统计信息的集合级别和保留期的参数值。您可以在数据库级别或单个物化视图中设置值。

SET_SYSTEM_DEFAULT过程

设置刷新统计信息参数的系统默认值。两个刷新统计信息参数是收集级别和保留期。

109.3.1 PURGE_REFRESH_STATS程序

此过程将清除早于指定实例化视图的指定保留期的刷新统计信息。

此过程强制清除刷新统计信息，而不会更改为指定的实例化视图定义的保留期。

句法

DBMS_MVIEW_STATS.PURGE_REFRESH_STATISTICS（mv_list IN VARACHAR2，retention_period IN NUMBER）;

参数

表109-2 PURGE_REFRESH_STATS过程参数

参数 描述

mv_list

schema_name形式的现有物化视图的完全限定名称。 mv_name 。使用逗号分隔列表指定多个物化视图。

指定NULL以清除数据库中所有实例化视图的实例化视图刷新统计信息。

retention_period

必须在数据字典中保留刷新统计信息的天数。从数据字典中清除早于保留期的物化视图刷新操作的统计信息。

此过程中指定的保留期将覆盖先前在数据库级别或指定的实例化视图中设置的保留期。

指定NULL以使用自动统计信息清除定义的清除策略。指定-1以清除所有刷新统计信息。

使用说明

要调用此过程，您需要在mv_list指定的每个物化视图上具有SYSDBA权限或特权。

109.3.2 SET_MVREF_STATS_PARAMS过程

此过程设置实例化视图刷新统计信息的集合级别和保留期。您可以为单个实例化视图或数据库中的所有实例化视图设置这些属性。

句法

DBMS_MVIEW_STATS.SET_MVREF_STATS_PARAMS（mv_list IN VARACHAR2，collection_level IN VARCHAR2 DEFAULT NULL，retention_period IN NUMBER DEFAULT NULL）;

参数

表109-3 SET_MVREF_STATS_PARAMS过程参数

参数 描述

mv_list

schema_name形式的现有物化视图的完全限定名称。 mv_name 。使用逗号分隔列表指定多个物化视图。

指定NULL以设置数据库中所有现有实例化视图的属性。

collection_level

指定收集mv_list指定的mv_list化视图的刷新统计信息时使用的详细信息mv_list 。

为collection_level设置以下值之一：

  • NONE：未收集物化视图刷新统计信息。

  • TYPICAL：仅为mv_list指定的mv_list化视图收集和存储基本刷新统计信息。

  • 高级：为mv_list指定的物化视图收集和存储详细的刷新统计信息。

如果此参数设置为NULL ，则使用collection_level的系统缺省值（使用SET_SYSTEM_DEFAULT设置）。

retention_period

指定mv_list指定的mv_list化视图的刷新统计信息的保留期（以天为mv_list 。保留期之前的统计信息将自动从数据字典中清除。

有效值介于1和1365000之间。

如果该参数被设置为NULL ，则对于系统默认值retention_period （使用设置SET_SYSTEM_DEAFULT被使用）。

将retention_period设置为-1可指定永远不能清除mv_list中mv_list化视图的刷新统计信息。

使用说明

要设置一个或多个物化视图的集合级别或保留期，您必须具有这些物化视图的特权。要为数据库中的所有实例化视图设置收集级别或保留期限，您必须对数据库中的每个SYSDBA化视图具有SYSDBA权限或特权。

要为统计信息收集级别和保留期设置系统级缺省值，请使用SET_SYSTEM_DEAFULT过程。

使用DBA_MVREF_STATS_PARAMS视图确定物化视图统计信息收集的当前设置保留期和集合级别。

要为数据库中的所有实例化视图禁用刷新统计信息收集，请使用以下命令：

DBMS_MVIEW_STATS.SET_MVREF_STATS_PARAMS（NULL，'NONE'，NULL）;

请注意，使用SET_MVREF_STATS_PARAMS设置的参数仅影响程序运行时数据库中存在的物化视图。运行此过程后创建的任何新物化视图将使用collection_level和retention_period的系统默认值。

109.3.3 SET_SYSTEM_DEFAULT过程

此过程设置系统范围的默认值，用于管理实例化视图刷新统计信息的收集和保留。所有新创建的物化视图都使用这些默认值，直到使用SET_MVREF_STATS_PARAMS过程显式重置参数。

句法

DBMS_MVIEW_STATS.SET_SYSTEM_DEFAULT（parameter_name IN VARCHAR2，值IN VARCHAR2 DEFAULT NULL）;

参数

表109-4 SET_SYSTEM_DEFAULT过程参数

参数 描述

PARAMETER_NAME

正在设置系统默认值的物化视图刷新统计信息参数的名称。

可以设置的参数是：

  • COLLECTION_LEVEL ：指定COLLECTION_LEVEL化视图刷新统计信息的详细程度。

  • RETENTION_PERIOD ：指定在数据字典中保留刷新统计信息的持续时间（以天为单位）

值

物化视图刷新统计参数的值。

COLLECTION_LEVEL的有效值为：

  • NONE：没有为刷新操作收集刷新统计信息。

  • 典型：仅为刷新操作收集基本刷新统计信息。这是默认设置。

  • 高级：收集刷新操作的详细刷新统计信息。

RETENTION_PERIOD的有效值为：

  • -1

  • 1到1365000之间的数字

retention_period的默认值为31。

如果为任何参数指定NULL ，则使用该参数的系统默认设置。

使用说明

您必须具有SYSDBA权限才能调用此过程。

使用DBA_MVREF_STATS_SYS_DEFAULTS视图显示物化视图刷新统计信息收集的当前默认设置。
