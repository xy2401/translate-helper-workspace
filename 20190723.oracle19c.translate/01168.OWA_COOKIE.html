214.1 OWA_CUSTOM概述

Cookie是发送到浏览器的不透明字符串，用于维护HTTP调用之间的状态。可以在整个客户的会话期间维护状态，如果包含截止日期，则可以更长。系统日期是参考OWA_CUSTOM包中指定的信息计算的。

214.2 OWA_COOKIE类型

此数据类型包含cookie名称 - 值对。

由于HTTP标准允许cookie名称被重载（即，多个值可以与相同的cookie名称相关联），因此有一个PL / SQL RECORD保存与给定cookie名称相关联的所有值。 

TYPE vc_arr是由BINARY_INTEGER表示的VARCHAR2（4000）索引。 TYCO COOKIE是RECORD（名称VARCHAR2（4000），vals vc_arr，num_vals INTEGER）;

214.3 OWA_COOKIE规则和限制

所有HTTP标头必须是英文和ASCII字符集。如果标头是从数据库生成的，请验证它们是否以英语创建。

214.4.1 GET功能

此函数返回与指定cookie关联的值。值以OWA_COOKIE.COOKIE DATA TYPE返回。

句法

OWA_COOKIE.GET（名称IN VARCHAR2）返回COOKIE;

参数

表214-2 GET功能参数

参数 描述

名称

Cookie的名称。

OWA_COOKIE.COOKIE DATA TYPE 。

214.4.2 GET_ALL过程

此过程从客户端的浏览器返回所有cookie名称及其值。值显示在从浏览器发送的顺序中。

句法

OWA_COOKIE.GET_ALL（名称OUT vc_arr，vals OUT vc_arr，num_vals OUT INTEGER）;

参数

表214-3 GET_ALL过程参数

参数 描述

名

Cookie的名称。

瓦尔斯

Cookie的值。

num_vals

cookie值对的数量。

214.4.3删除程序

此过程通过将HTTP标头中Set-Cookie行的“expires”字段设置为“01-Jan-1990”来强制cookie立即过期。

必须在HTTP标头的上下文中调用此过程。

句法

OWA_COOKIE.REMOVE（名称IN VARCHAR2，val IN VARCHAR2，路径IN VARCHAR2 DEFAULT NULL）;

参数

表214-4 REMOVE过程参数

参数 描述

名称

要过期的cookie的名称。

VAL

cookie的价值。

路径

[目前未使用]

214.4.4发送程序

此过程生成Set-Cookie行，该行将cookie传输到客户端。

此过程必须在HTTP标头的上下文中进行。

句法

OWA_COOKIE.SEND（varchar2中的名称，varchar2中的值，日期中的过期DEFAULT NULL，varchar2中的路径DEFAULT NULL，varchar2中的域DEFAULT NULL，varchar2中的安全DEFAULT NULL）;

参数

表214-5 SEND过程参数

参数 描述

名称

Cookie的名称。

值

cookie的价值。

到期

Cookie过期的日期

路径

路径字段的值。

域

域字段的值。

安全

如果此参数的值不为NULL ，则将“安全”字段添加到该行。
