3.150 ALL_XML_SCHEMAS

ALL_XML_SCHEMAS描述了当前用户可访问的已注册XML模式。

相关观点

  • DBA_XML_SCHEMAS描述数据库中所有已注册的XML模式。

  • USER_XML_SCHEMAS描述了当前用户拥有的已注册XML模式。此视图不显示OWNER列。

柱 数据类型 空值 描述

所有者

VARCHAR2（128）

XML模式的所有者

SCHEMA_URL

VARCHAR2（700）

XML模式的模式URL

注意：

有关XML架构的schemaurl属性的更多信息， 请参阅下面的“ 请参阅注释”。

本地

VARCHAR2（3）

指示XML架构是本地（ YES ）还是全局（ NO ）

SCHEMA

XMLTYPE

XML架构文档

INT_OBJNAME

VARCHAR2（4000）

架构的内部数据库对象名称

QUAL_SCHEMA_URL

VARCHAR2（865）

完全限定的架构URL

HIER_TYPE

VARCHAR2（11）

为其启用架构的层次结构类型：

  • 没有

  • RESMETADATA

  • 内容

BINARY

VARCHAR2（3）

指示XML模式是否已注册二进制编码用法（ YES ）或否（ NO ）

SCHEMA_ID

RAW（16）

不透明的模式标识符（16个字节）

隐

VARCHAR2（3）

指示是否已在隐藏模式下删除XML架构（ YES ）或否（ NO ）
