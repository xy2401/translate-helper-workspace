对象访问表达式

对象访问表达式指定属性引用和方法调用。

object_access_expression :: =


插图object_access_expression.eps的描述

column参数可以是对象或REF列。如果指定expr ，则必须解析为对象类型。

在SQL语句的上下文中调用类型的成员函数时，如果SELF参数为null，则Oracle返回null并且不调用该函数。

例子

以下示例基于示例oe.order_item_typ对象类型创建表，然后显示如何更新和从对象列属性中进行选择。 

CREATE TABLE short_orders（sales_rep VARCHAR2（25），item order_item_typ）; UPDATE short_orders s SET sales_rep ='Unassigned'; SELECT o.item.line_item_id，o.item.quantity FROM short_orders o;
