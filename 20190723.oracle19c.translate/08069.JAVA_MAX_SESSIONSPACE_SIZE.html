1.155 JAVA_MAX_SESSIONSPACE_SIZE

JAVA_MAX_SESSIONSPACE_SIZE指定（以字节为单位）可用于服务器中执行的Java程序的最大会话空间量。

属性 描述

参数类型

整数

默认值

0

可修改

没有

可在PDB中修改

没有

价值范围

0到2 GB - 1

基本

没有

Java会话空间是将Java状态从一个数据库调用保存到另一个数据库的内存。当用户的会话持续时间Java状态尝试超过JAVA_MAX_SESSIONSPACE_SIZE指定的JAVA_MAX_SESSIONSPACE_SIZE ，Java虚拟机会因JAVA_MAX_SESSIONSPACE_SIZE而导致会话失败。
