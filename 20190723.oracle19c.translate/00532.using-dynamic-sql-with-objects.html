3.4.1将动态SQL与对象类型和集合一起使用

您可以将动态SQL与对象类型和集合一起使用。

示例3-11说明了使用动态SQL的对象和集合。首先，该示例定义了对象类型person_typ和VARRAY类型hobbies_var ，然后它定义了使用这些类型的包， teams 。

您需要AUTHID CURRENT_USER来执行动态包方法;否则，当您运行示例3-12 （这是一个调用程序包TEAMS的过程的匿名块）时，这些方法会引发权限不足错误。

示例3-11对对象类型和集合使用动态SQL的包

创建或替换TYPE person_typ AS OBJECT（名称VARCHAR2（25），年龄NUMBER）; / CREATE TYPE hobbies_var AS VARRAY（10）OF VARCHAR2（25）; /创建或替换PACKAGE团队
   AUTHID CURRENT_USER AS PROCEDURE create_table（tab_name VARCHAR2）; PROCEDURE insert_row（tab_name VARCHAR2，p person_typ，h hobbies_var）; PROCEDURE print_table（tab_name VARCHAR2）;结束; /创建或替换程序包团队AS PROCEDURE create_table（tab_name VARCHAR2）是BEGIN EXECUTE IMECEDIATE'CREATE TABLE'|| tab_name || '（pers person_typ，hobbs hobbies_var）';结束;程序insert_row（tab_name VARCHAR2，p person_typ，h hobbies_var）是BEGIN EXECUTE IMECEDIATE'INSERT INTO'|| tab_name || 'VALUES（：1，：2）'使用p，h;结束; PROCEDURE print_table（tab_name VARCHAR2）IS TYPE refcurtyp IS REF CURSOR; v_cur refcurtyp; p person_typ; h hobbies_var; BEGIN OPEN v_cur FOR'SELECT pers，hobbs FROM'|| TAB_NAME; LOOP FETCH v_cur INTO p，h;退出时v_cur％NOTFOUND; - 打印'p'的属性和'h'DBMS_OUTPUT.PUT_LINE的元素（'Name：'|| p.name ||' - 年龄：'|| p.age）;对于我。FIRST..h。LAST LOOP DBMS_OUTPUT.PUT_LINE（'Hobby（'|| i ||'）：'|| h（i））;结束循环;结束循环;关闭v_cur;结束;结束; /

3.4.2使用对象类型和集合调用包过程

您可以使用对象类型和集合调用包过程。

从匿名块（ 例3-12）中 ，您可以调用TEAMS包中的过程，如例3-11所示

示例3-12 TEAMS包中的调用过程

DECLARE team_name VARCHAR2（15）; BEGIN team_name：='知名人士'; TEAMS.create_table（TEAM_NAME）; TEAMS.insert_row（team_name，person_typ（'John'，31），hobbies_var（'滑雪'，'硬币收集'，'网球'））; TEAMS.insert_row（team_name，person_typ（'Mary'，28），hobbies_var（'golf'，'quilting'，'rock climbing'，'fencing'））; TEAMS.print_table（TEAM_NAME）;结束; /
