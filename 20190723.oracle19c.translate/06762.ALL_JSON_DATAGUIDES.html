2.197 ALL_JSON_DATAGUIDES

ALL_JSON_DATAGUIDES提供有关启用了数据指南的当前用户可访问的JavaScript Object Notation（JSON）列的信息。

相关观点

  • DBA_JSON_DATAGUIDES提供有关数据库中启用了数据指南的JavaScript Object Notation（JSON）列的信息。它的列与ALL_JSON_DATAGUIDES列相同。

  • USER_JSON_DATAGUIDES提供有关当前用户拥有的启用了数据指南的JavaScript Object Notation（JSON）列的信息。此视图不显示OWNER列。
