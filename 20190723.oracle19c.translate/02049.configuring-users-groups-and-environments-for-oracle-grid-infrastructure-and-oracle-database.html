6为Oracle Grid Infrastructure和Oracle数据库配置用户，组和环境

在安装之前，请创建操作系统组和用户，以及配置用户环境。

  • 为Oracle Grid Infrastructure创建组，用户和路径
    以root身份登录，并使用以下说明查找或创建Oracle Inventory组，并为Oracle Grid Infrastructure创建软件所有者，并为Oracle主目录创建目录。
  • 具有标准和作业角色分离组和用户的Oracle安装
    Oracle数据库和Oracle ASM的作业角色分离配置是一种配置，其中组和用户可以为操作系统身份验证提供单独的组。
  • 创建操作系统权限组
    以下部分介绍如何为Oracle Grid Infrastructure和Oracle数据库创建操作系统组：
  • 创建操作系统Oracle安装用户帐户
    在开始安装之前，请创建Oracle软件所有者用户帐户，并配置其环境。
  • 配置网格基础结构软件所有者用户环境
    在安装Oracle Grid Infrastructure之前，请了解要配置的软件所有者用户环境。
  • 启用智能平台管理界面（IPMI）
    智能平台管理接口（IPMI）为计算机硬件和固件提供了一组通用接口，系统管理员可以使用这些接口来监控系统运行状况并管理系统。
