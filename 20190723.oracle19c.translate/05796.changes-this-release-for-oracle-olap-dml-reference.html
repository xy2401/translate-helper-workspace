此版本的Oracle OLAP DML参考中的更改

此版本中的Oracle OLAP DML参考未进行任何更改。有关早期版本中的更改的信息，请参阅以下主题。

Oracle Database 12 c第2版（12.2）中的Oracle OLAP更改

Oracle OLAP DML参考 Oracle Database 12 c第2版（12.2）中进行了以下更改：

  • OBJ函数具有新关键字CHANGEDPAGES，它返回自上次更新以来已更改的分析工作区中的页数。

此外，Active Data Guard（ADG）只读活动备用实例现在支持只读OLAP应用程序。活动备用实例必须具有实时应用操作。

您现在可以卸载到ADG活动备用实例应用程序，这些应用程序查询使用动态抽象数据类型（ADT）的CUBE_TABLE和OLAP_TABLE函数。初始ADT创建会触发在ADG主数据库上创建它的请求。实时应用将新ADT传播到活动备用实例，这允许查询在短暂延迟后运行。如果ADT未能在合理时间内出现在备用数据库上，则会出现错误情况。

新功能

以下功能是此版本中的新增功能：

  • 增加最大线长

    文本的最大行长度已从4,000更改为32,767。

    请参阅文本数据类型中的TEXT和NTEXT以及EXTCOLS ， INSCOLS ， JOINBYTES ， JOINCOLS ， REMCOLS和REPLCOLS函数的说明。

其他变化

以下是此版本中的其他更改：

  • 性能提升

    适用于价值集的全新高度紧凑的存储格式。

    请参阅DEFINE VALUESET的NOORDER关键字。

    • DEFINE VALUESET
