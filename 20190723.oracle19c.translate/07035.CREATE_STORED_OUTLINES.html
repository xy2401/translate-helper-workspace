1.57 CREATE_STORED_OUTLINES

CREATE_STORED_OUTLINES确定Oracle是否自动为会话期间提交的每个查询创建和存储大纲。

属性 描述

参数类型

串

句法

CREATE_STORED_OUTLINES = { true | false | category_name } [ NOOVERRIDE ]

默认值

没有默认值。

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

值

  • 真正

    为同一会话中的后续查询启用自动大纲创建。这些大纲接收系统生成的唯一名称，并存储在DEFAULT类别中。如果特定查询已在DEFAULT类别中为其定义了轮廓，则该轮廓将保留，并且不会创建新轮廓。

  • 假

    在会话期间禁用自动轮廓创建。这是默认值。

  • 分类名称

    启用与true相同的行为，但会话期间创建的任何轮廓都存储在category_name类别中。

  • NOOVERRIDE

    NOOVERRIDE指定此系统设置不会覆盖显式设置此参数的任何会话的设置。如果未指定NOOVERRIDE ，则此设置将在所有会话中生效。
