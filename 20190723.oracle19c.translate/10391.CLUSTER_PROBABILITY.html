目的

CLUSTER_PROBABILITY返回选择中每行的概率。概率是指最高概率群集或指定的cluster_id 。群集概率返回为BINARY_DOUBLE 。

语法选择

CLUSTER_PROBABILITY可以通过以下两种方式之一对数据进行评分：它可以将挖掘模型对象应用于数据，也可以通过执行构建和应用一个或多个瞬态挖掘模型的分析子句来动态挖掘数据。选择语法或分析语法 ：

  • 语法 - 使用第一种语法使用预定义的模型对数据进行评分。提供聚类模型的名称。

  • 分析语法 - 使用分析语法在没有预定义模型的情况下对数据进行评分。包括INTO n ，其中n是要计算的集群数，以及mining_analytic_clause，它指定是否应为多个模型构建分区数据。mining_analytic_clause支持query_partition_clause和order_by_clause 。（参见analytic_clause :: = 。）

在对分区模型进行评分时， CLUSTER_PROBABILITY函数的语法可以使用可选的GROUPING提示。请参阅GROUPING提示 。

mining_attribute_clause

mining_attribute_clause标识要用作评分预测变量的列属性。使用分析语法调用函数时，这些预测变量也用于构建瞬态模型。mining_attribute_clause行为与PREDICTION函数的描述相同。（参见mining_attribute_clause :: = 。）
