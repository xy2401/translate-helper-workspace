说明adxdb025.eps的说明

该图显示左上角的两个XML文档，如下所示：

XML实例文档：employees.xml

...<employee> <first_name> Shelli </ first_name> <last_name> Baida </ last_name> <email> sbaida </ email> ...<hire_date> 24-DEC-97 </ hire_date> ...<department_id> 30 </ department_id> </ employee> ...

XML模式：employee.xsd

...<sequence> <element name =“first_name”type =“string”/> <element name =“last_name”type =“string”/> <element name =“email”type =“string”/> ...<element name =“hire_date”type =“date”/> ...<element name =“dept_id”type =“integer”/> </ sequence> ...

每个XML文档都有一个箭头，标有“Create XMLTYpe Table”框。在此框中有两个箭头，每个箭头指向一个表示数据库表的框。这些箭头从左到右标记为“二进制XML存储”和“对象关系存储”。表格框从左到右标记为“XML数据存储为二进制XML”和“XML数据存储在对象关系列和表中”。

在每个数据库表框中都有标签“employees Tables”和数据库列的描述。二进制XML存储框显示标记为“XMLType列”的列。此列中的条目显示为“二进制XML”。对象关系存储框显示标记为“first_name”，“last_name”，“email”和“dept_id”的列。显示的行具有相应的条目“shelli”，“baida”，“sbaida”和“30”。
