D.7 chactl查询模型

使用chactl query model命令列出所有Oracle Cluster Health Advisor模型或查看有关特定Oracle Cluster Health Advisor模型的详细信息。

句法

chactl查询模型[-name model_name [-verbose] [ -求助]

例子

  • 列出所有基本Oracle Cluster Health Advisor模型：

    $ chactl查询模型型号：MOD1，MOD2，MOD3，MOD4，MOD5，MOD6，MOD7
    $ chactl查询模型-name weekday模型：工作日目标类型：数据库版本：12.2.0.1_0 OS校准：Linux amd64校准目标名称：prod校准日期：2016-09-10 12:59:49校准时间范围：start = 2016-09-09 16：00：00，end = 2016-09-09 23:00:00校准KPI：未指定

  • 要查看有关特定Oracle Cluster Health Advisor模型的详细信息（包括校准元数据），请执行以下操作：

    $ chactl查询模型-name MOD5 -verbose型号：MOD5 CREATION_DATE：Jan 10,2016 10:10 VALIDATION_STATUS：验证DATA_FROM_TARGET：inst72，inst75 USED_IN_TARGET：inst76，inst75，prodDB，evalDB-evalSP CAL_DATA_FROM_DATE：Jan 05,2016 10:00 CAL_DATA_TO_DATE ：Jan 07,2016 13:00 CAL_DATA_FROM_TARGETS inst73，inst75 ...
