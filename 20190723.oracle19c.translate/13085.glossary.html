词汇表

接受的计划

在SQL计划管理的上下文中，该计划位于SQL语句的SQL计划基准中，因此可供优化程序使用。接受的计划包含一组提示，计划哈希值和其他与计划相关的信息。

父主题： 词汇表

访问路径

数据库从数据库检索数据的方法。例如，使用索引的查询和使用全表扫描的查询使用不同的访问路径 。

父主题： 词汇表

自适应光标共享

一种功能，它允许包含绑定变量的单个语句使用多个执行计划。游标共享是“自适应的”，因为游标会调整其行为，以便数据库不会始终对每个执行或绑定变量值使用相同的计划。

父主题： 词汇表

自适应动态采样

自适应优化器的一项功能，可以自动调整动态统计级别 。

父主题： 词汇表

自适应优化器

优化程序的一项功能，使其能够根据运行时统计信息调整计划。

父主题： 词汇表

自适应查询计划

优化后更改的执行计划，因为运行时条件表明优化程序估计不准确。自适应查询计划具有不同的内置计划选项。在第一次执行期间，在特定子计划变为活动状态之前，优化程序最终决定使用哪个选项。优化程序的选择基于执行期间的观察到目前为止。因此，自适应查询计划使语句的最终计划与默认计划不同。

父主题： 词汇表

自适应查询优化

一组功能，使自适应优化器能够对执行计划进行运行时调整，并发现可以带来更好统计信息的其他信息。当现有统计数据不足以生成最佳计划时，自适应优化很有用。

父主题： 词汇表

ADDM

请参阅自动数据库诊断监视器（ADDM） 。

父主题： 词汇表

反连接

返回无法匹配右侧子查询的行的连接。例如，反连接可以列出没有员工的部门。反连接使用NOT EXISTS或NOT IN结构。

父主题： 词汇表

近似查询处理

一组优化技术，通过计算可接受误差范围内的结果来加速分析查询。

父主题： 词汇表

自动捕获过滤器

SQL计划管理功能，使您可以指定自动初始计划捕获的资格标准。

父主题： 词汇表

自动数据库诊断监视器（ADDM）

ADDM是Oracle数据库内置的自诊断软件。ADDM检查并分析自动工作负载存储库（AWR）中捕获的数据，以确定可能的数据库性能问题。

父主题： 词汇表

自动优化器统计信息收集

自动运行DBMS_STATS包以收集缺少统计信息或过时统计信息的所有模式对象的优化程序统计信息。

父主题： 词汇表

自动初始计划捕获

数据库自动为数据库上执行的任何可重复SQL语句创建SQL计划基准的机制。通过将OPTIMIZER_CAPTURE_SQL_PLAN_BASELINES初始化参数设置为true （默认值为false ）来启用自动初始计划捕获。

请参见可重复的SQL语句 。

父主题： 词汇表

自动重新优化

优化程序在后续执行SQL语句时自动更改计划的能力。自动重新优化可以修复由于不正确的优化器估计而选择的任何次优计划，从次优分布方法到错误选择并行度。

父主题： 词汇表

自动SQL调优

自动SQL调优顾问执行的工作在系统维护窗口中作为自动化任务运行。

父主题： 词汇表

自动SQL Tuning Advisor

SQL Tuning Advisor作为自动维护任务运行时。自动SQL调整在系统维护窗口期间作为自动维护任务运行，搜索改进高负载SQL语句执行计划的方法。

请参阅SQL Tuning Advisor 。

父主题： 词汇表

自动调整优化器

SQL Tuning Advisor调用时的优化程序。在SQL调整模式下，优化程序执行其他分析以检查是否可以进一步改进在正常模式下生成的计划。优化程序输出不是执行计划，而是一系列操作，以及它们的基本原理和产生明显更好的计划的预期收益。

父主题： 词汇表

自动工作负载存储库（AWR）

为Oracle数据库组件提供服务以收集，维护和使用统计信息以进行问题检测和自我调整的基础结构。

父主题： 词汇表

AWR

请参阅自动工作负载存储库（AWR） 。

父主题： 词汇表

AWR快照

用于性能比较的特定时间的一组数据。快照捕获的增量值表示该时间段内每个统计信息的更改。收集的统计信息从内存中查询。您可以在报告和视图中显示收集的数据。

父主题： 词汇表

乐队加入

一种特殊类型的非连接，其中一个数据集中的键值必须落在第二个数据集的指定范围（“波段”）内。

父主题： 词汇表

基数

对于表，表中的总行数。

父主题： 词汇表

底线

在AWR的上下文中，两个AWR快照之间的间隔，表示数据库以最佳级别运行。

父主题： 词汇表

绑定感知游标

绑定敏感游标，有资格对不同的绑定值使用不同的计划。在使游标成为绑定感知之后，优化程序会根据绑定值及其基数估计选择将来执行的计划。

父主题： 词汇表

绑定敏感游标

一个游标，其最佳计划可能取决于绑定变量的值。数据库监视绑定敏感游标的行为，该游标使用不同的绑定值来确定不同的计划是否有益。

父主题： 词汇表

绑定变量

SQL语句中的占位符，必须替换为有效的值或值地址才能成功执行语句。通过使用绑定变量 ，您可以编写一个在运行时接受输入或参数的SQL语句。以下查询使用v_empid作为绑定变量：

SELECT * FROM employees WHERE employee_id =：v_empid;

父主题： 词汇表

绑定变量偷看

优化程序在硬解析期间查看绑定变量中的值的能力。通过查看绑定值，优化器可以确定WHERE子句条件的选择性，就像使用了文字一样 ，从而改进了计划。

父主题： 词汇表

位图连接索引

两个或多个表的连接的位图索引。

父主题： 词汇表

位图片

单个位图索引条目的子组件。每个索引列值可以具有一个或多个位图片段。数据库使用位图片段来分解与块大小相关的索引条目。

父主题： 词汇表

B树索引

一个索引像一个颠倒的树组织。B树索引有两种类型的块：用于搜索的分支块和用于存储值的叶块。叶块包含每个索引数据值和用于定位实际行的相应rowid。“B”代表“平衡”，因为所有叶块自动停留在相同的深度。

父主题： 词汇表

批量装载

CREATE TABLE AS SELECT或INSERT INTO ...SELECT操作。

父主题： 词汇表

浓密的连接树

一种连接树，其中内部节点的左子节点或右子节点可以是连接节点。丛林连接树中的节点可能在其后代中具有递归结构。

父主题： 词汇表

基数

预期由执行计划中的操作返回的行数。

父主题： 词汇表

笛卡尔加入

一个连接，其中一个或多个表与语句中的任何其他表没有任何连接条件。优化器将来自一个数据源的每一行与来自另一个数据源的每一行连接起来，从而创建两组的笛卡尔积。

父主题： 词汇表

儿童游标

包含计划，编译环境以及其文本存储在父光标中的语句的其他信息的游标 。父光标的编号为0 ，第一个子编号为1 ，依此类推。子游标引用与父游标相同的SQL文本，但它们不同。例如，带有文本SELECT * FROM t两个查询在引用两个名为t不同表时使用不同的游标。

父主题： 词汇表

集群扫描

表群集的访问路径。在索引表集群中，Oracle数据库首先通过扫描集群索引来获取所选行之一的rowid。然后，Oracle数据库根据此rowid查找行。

父主题： 词汇表

专栏组

一组被视为一个单元的列。

父主题： 词汇表

列组统计信息

收集在作为一个单元处理的一组列上的扩展统计信息。

父主题： 词汇表

列统计

有关优化程序用于确定最佳执行计划的列的统计信息。列统计信息包括不同列值的数量，低值，高值和空值数。

父主题： 词汇表

复杂的视图合并

合并包含GROUP BY或DISTINCT关键字的视图。

父主题： 词汇表

复合数据库操作

在数据库操作中 ，数据库会话中两个时间点之间的活动，每个会话定义其自己的起点和终点。会话一次最多可以参与一个复合数据库操作。

父主题： 词汇表

并发

许多用户同时访问相同的数据。多用户数据库管理系统必须提供足够的并发控制，以便不能不正确地更新或更改数据，从而危及数据完整性。

父主题： 词汇表

并发统计收集模式

一种模式，使数据库能够同时收集模式中的多个表或表中的多个分区或子分区的优化程序统计信息。并发可以通过使数据库完全使用多个CPU来减少收集统计信息所需的总时间。

父主题： 词汇表

条件

一个或多个表达式和逻辑运算符的组合，返回值TRUE ， FALSE或UNKNOWN 。

父主题： 词汇表

成本

一种数字内部度量，表示执行计划的估计资源使用情况。成本越低，计划的效率越高。

父主题： 词汇表

基于成本的优化器（CBO）

优化程序的旧名称。在早期版本中，基于成本的优化器是基于规则的优化器（RBO）的替代方案。

父主题： 词汇表

成本模型

内部优化程序模型，用于计算查询预计使用的I / O，CPU和网络资源的成本 。

父主题： 词汇表

累积统计

计数，例如块读取的数量。Oracle数据库为系统，会话和单个SQL语句生成许多类型的累积统计信息。

父主题： 词汇表

光标

PGA中私有SQL区域的句柄或名称。由于游标与私有SQL区域紧密相关，因此这些术语有时可互换使用。

游标缓存

组合游标以节省共享SQL区域中的空间。如果优化程序为可识别绑定的游标创建计划，并且此计划与现有游标相同，则优化程序可以合并游标。

父主题： 词汇表

游标持续时间临时表

一个临时的内存表，用于存储游标持续时间的查询结果。对于诸如WITH子句查询和星形转换的复杂操作，此优化增强了重复使用的子查询的中间结果的实现。通过这种方式，游标持续时间临时表可以提高性能并优化I / O.

父主题： 词汇表

并行查询中数据重新分发阶段之间的工作单元。

父主题： 词汇表

数据偏斜

列中重复值的数量变化很大。

父主题： 词汇表

数据库操作

父主题： 词汇表

默认计划

对于自适应计划 ，优化程序最初使用数据字典中的统计信息选择执行计划。默认计划可能与最终计划不同。

父主题： 词汇表

父主题： 词汇表

并行度（DOP）

与单个操作关联的并行执行服务器的数量。并行执行旨在有效地使用多个CPU。Oracle数据库并行执行框架使您能够明确选择特定的并行度，或依靠Oracle数据库自动控制它。

父主题： 词汇表

密密集

一个数字键，存储为本机整数并具有一系列值。

父主题： 词汇表

密集分组密钥

一个键，表示所有分组键，其分组列来自特定的事实表或维度。

父主题： 词汇表

密集连接密钥

一个键，表示其连接列来自特定事实表或维度的所有连接键。

父主题： 词汇表

密度

介于0和1之间的十进制数，用于衡量列的选择性。值接近1表示该列是非选择性的 ，而接近0值表示该列更具选择性。

父主题： 词汇表

直接路径读取

单个或多个块读入PGA，绕过SGA。

父主题： 词汇表

驾驶表

其他表连接的表。编程的类比是包含另一个for循环的for循环。外部for循环是驱动表的模拟，也称为外表 。

父主题： 词汇表

动态性能视图

在动态性能表上创建的视图，它是记录当前数据库活动的虚拟表。动态性能视图称为固定视图，因为数据库管理员无法更改或删除它们。它们也称为V$视图，因为它们以字符串V$开头（Oracle RAC中的GV$ ）。

父主题： 词汇表

动态计划

一组子计划组。子计划组是一组子计划。在自适应查询计划中 ，优化程序根据统计信息收集器获取的统计信息在运行时选择子计划 。

父主题： 词汇表

动态统计

一种优化技术，其中数据库执行递归SQL语句以扫描表块的小随机样本以估计谓词选择性。

父主题： 词汇表

动态统计水平

控制数据库何时收集动态统计信息的级别，以及优化程序用于收集统计信息的样本大小。使用OPTIMIZER_DYNAMIC_SAMPLING初始化参数或语句提示设置动态统计级别。

启用计划

父主题： 词汇表

端点号

一个唯一标识直方图中存储桶的数字。在频率和混合直方图中，端点号是端点的累积频率。在高度平衡直方图中，端点号是桶号。

估计

执行计划

数据库用于执行SQL语句的步骤组合。每个步骤都从数据库中物理地检索数据行，或者为发出语句的会话准备它们。您可以使用提示覆盖执行计划。

父主题： 词汇表

执行树

一个树形图，显示执行计划中从一个步骤到另一个步骤的行源流。

父主题： 词汇表

预期的基数

对于表，在将所有过滤器谓词应用于表之后，表具有的估计行数。

父主题： 词汇表

表达

一个或多个值，运算符和SQL函数的组合，用于求值。例如，表达式2*2计算结果为4 。通常， 表达式假定其组件的数据类型。

父主题： 词汇表

表达统计

一种扩展统计信息，可在WHERE子句具有使用表达式的谓词时改进优化程序估计值。

扩展统计

一种优化程序统计信息 ，可在存在多个谓词或谓词包含表达式时改进基数的估计。

父主题： 词汇表

可扩展的优化器

优化器功能，允许用户定义的函数和索引的作者创建优化程序在选择执行计划时使用的统计信息收集，选择性和成本函数。优化器成本模型扩展为集成用户提供的信息以评估CPU和I / O成本。

延期

父主题： 词汇表

外部表

一个只读表，其元数据存储在数据库中，但其数据存储在数据库外的文件中。数据库使用描述外部表的元数据来公开其数据，就好像它们是关系表一样。

父主题： 词汇表

过滤条件

一个WHERE子句组件，用于从SQL语句中引用的单个对象中删除行。

父主题： 词汇表

最终计划

父主题： 词汇表

固定物体

固定计划

已标记为首选的已接受计划 ，以便优化程序仅考虑SQL计划基准中的固定计划。您可以使用固定计划来影响优化程序的计划选择过程。

父主题： 词汇表

频率直方图

一种直方图，其中每个不同的列值对应于单个桶。一个类比是分拣硬币：所有便士都在桶1中，所有镍币都在桶2中，依此类推。

父主题： 词汇表

全外连接

左外连接和右外连接的组合。除了内部联接之外，数据库还使用空值来保留两个表中尚未在内部联接的结果中返回的行。换句话说，完全外部联接将表连接在一起，但在连接表中显示没有对应行的行。

父主题： 词汇表

全表扫描

扫描表数据，其中数据库按顺序从表中读取所有行，并过滤掉那些不符合选择条件的行。扫描高水位线下的所有数据块。

全球临时表

一个特殊的临时表，用于存储特定持续时间的中间会话私有数据。

硬解析

哈希冲突
