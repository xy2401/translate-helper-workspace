适用于x86-64的受支持的Oracle Linux 7发行版

使用以下信息检查支持的Oracle Linux 7发行版：

表4-1 x86-64 Oracle Linux 7最低操作系统要求

项目 要求

SSH要求

确保在服务器上安装了OpenSSH。OpenSSH是必需的SSH软件。

Oracle Linux 7

订阅Unbreakable Linux Network上的Oracle Linux 7频道，或者从Oracle Linux yum服务器网站配置yum存储库，然后安装Oracle Preinstallation RPM。此RPM安装Oracle Grid Infrastructure和Oracle Database安装所需的所有内核包，并执行其他系统配置。

支持的发行版：

  • 具有Unbreakable Enterprise Kernel 4的Oracle Linux 7.4：4.1.12-124.19.2.el7uek.x86_64或更高版本

  • 具有Unbreakable Enterprise Kernel 5的Oracle Linux 7.4：4.14.35-1818.1.6.el7uek.x86_64或更高版本

  • 带有Red Hat兼容内核的Oracle Linux 7.4：3.10.0-862.11.6.el7.x86_64或更高版本
