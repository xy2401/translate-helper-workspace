更改程序

目的

包使用PL / SQL定义。因此，本节提供了一些常规信息，但有关语法和语义的详细信息，请参阅Oracle Database PL / SQL语言参考 。

使用ALTER PROCEDURE语句显式重新编译独立存储过程。显式重新编译消除了对隐式运行时重新编译的需要，并防止了相关的运行时编译错误和性能开销。

要重新编译作为包的一部分的过程，请使用ALTER PACKAGE语句重新编译整个包（请参阅ALTER PACKAGE ）。

注意：

本声明不会更改现有过程的声明或定义。要重新声明或重新定义过程，请使用带有OR REPLACE子句的CREATE PROCEDURE语句（请参阅CREATE PROCEDURE ）。

ALTER PROCEDURE语句与ALTER FUNCTION语句非常相似。有关更多信息，请参阅ALTER FUNCTION 。

先决条件

该过程必须位于您自己的架构中，或者您必须具有ALTER ANY PROCEDURE系统特权。

模式

指定包含该过程的模式。如果省略schema ，则Oracle数据库假定该过程位于您自己的模式中。

PROCEDURE_NAME

指定要重新编译的过程的名称。
