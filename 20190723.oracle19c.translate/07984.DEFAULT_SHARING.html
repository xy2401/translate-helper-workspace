1.99 DEFAULT_SHARING

DEFAULT_SHARING在创建应用程序根目录中的对象的语句中设置共享子句的值。

属性 描述

参数类型

串

句法

DEFAULT_SHARING = {NONE | METADATA |数据|扩展数据}

默认值

对于支持共享的对象类型， METADATA是默认值。

对于不支持共享的对象类型， NONE是默认值。

可修改

ALTER SESSION ， ALTER SYSTEM

可在PDB中修改

是

基本

没有

Oracle RAC

所有实例必须具有相同的值

在create DDL中指定SHARING=会覆盖DEFAULT_SHARING参数的值。

例

在应用程序根目录中发出以下ALTER SYSTEM语句会在应用程序根目录的服务器参数文件中将sharing子句的缺省值设置为NONE ：

ALTER SYSTEM SET DEFAULT_SHARING = NONE SCOPE = SPFILE;
