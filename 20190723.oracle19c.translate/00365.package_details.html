此图显示“包详细信息”页面。它包括以下部分：

  • 摘要（位于左上角） - 包括包状态，类型（例如Main）和大小，包中的问题列表等。

  • 打包任务（右上角的框） - 包括框右上角的两个按钮：生成上载文件并发送到Oracle。按钮下方是四个链接部分，每个链接代表一个打包任务。这些部分排列成两行两列。在左上角，其标题为“编辑内容”，链接为：添加问题，排除问题和查看包清单。在右上部分，其标题为“Scrub User Data”，其链接为：将文件复制到编辑内容并复制到文件中以替换内容。在左下部分，其标题为“其他诊断数据”，链接为：收集附加转储和添加外部文件。在右下角，其标题为“发送给Oracle支持”，链接为：完成内容准备，生成上载文件和发送上载文件。右下方的链接中，只有“完成内容准备”可以单击。其他人都被禁用了。

  • 一组三个子页面（位于底部） - 子页面选项卡标记为“事件”，“文件”和“活动日志”。将显示“事件”子页面。它包含一个事件表。表列是选择，ID，类型，问题ID，描述，大小（MB）和时间戳。该表有一行显示Main和Description以ORA-700开头。每行都有一个时间戳。在表格的正上方，左侧是一个排除按钮。在表格上方和“排除”按钮的右侧，有两个按钮：添加事件和添加最近的事件。
