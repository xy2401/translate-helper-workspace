1.311 SHARED_MEMORY_ADDRESS

SHARED_MEMORY_ADDRESS和HI_SHARED_MEMORY_ADDRESS指定系统全局区域（SGA）运行时的起始地址。 在链接时指定SGA起始地址的许多平台上，将忽略此参数。
属性 描述

参数类型

整数

默认值

0

可修改

没有

可在PDB中修改

没有

基本

没有

使用此参数指定32位平台上的整个地址，并在64位平台上指定64位地址的低32位。使用HI_SHARED_MEMORY_ADDRESS指定64位平台上64位地址的高32位。如果两个参数均为0或未指定，则SGA地址默认为特定于平台的位置。
