Item

Description

Enable

Select this checkbox to activate the Stylelint support.

Node Interpreter

In this field, specify the Node.js interpreter to use. This can be a local Node.js interpreter or a Node.js on Windows Subsystem for Linux.

Stylelint package

In this field, specify the location of the stylelint package installed globally or in the current project.
