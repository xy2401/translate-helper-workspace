Using RSync for Downloading Remote Gems

This feature is only supported in the Ultimate edition.

The following is only valid when Ruby Plugin is installed and enabled!

By default, IntelliJ IDEA installed on Linux or macOS downloads remote gems using the rsync utility. This speeds up the downloading process. You can change the default behavior and disable the use of rsync by utilizing the IntelliJ IDEA registry. To invoke the Registry dialog, press Ctrl+Shift+A, type Registry…, select it and press Enter. Then, use the following options to enable or disable rsync:

  • Linux and macOS: ide.remote.interpreters.use.rsync

  • Windows: ide.remote.interpreters.rsync.enabled.on.windows
