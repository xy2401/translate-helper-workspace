PyCharm 2019.1 Help

Messages Tool Window

View | Tool Windows | Messages
Alt+0

PyCharm parses the compilation output and displays it in a convenient format in the Messages tool window, enabling you to navigate to the location of relevant problems in the source code.

The Messages tool window is only available if there are messages to show.
