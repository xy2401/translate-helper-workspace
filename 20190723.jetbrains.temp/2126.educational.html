This page is available for the PyCharm Educational Edition or when the EduTools plugin is installed and enabled. The page contains the following controls:

Item

Description

Invitation to log in/log out.

This section depends on whether you are actually logged in. The credentials are automatically saved in this page, when you create an account with Stepik.

Testing from samples in adaptive course

This option enables pre-testing from samples provided in task description. If this option is enabled, the test workflow looks like the following: first solution is tested locally by the input/output from task description; if the local tests pass, the solution is sent for a check to a Stepik server.
If the local tests fail, then PyCharm reports in test result tool window: “Local test failed”, and an error text.

Default description mode

Choose the default format of the task descriptions. To do that, choose one of the available options.
