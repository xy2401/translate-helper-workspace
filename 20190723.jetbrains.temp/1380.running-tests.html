Maven. Running Tests

Use this page to configure settings for running JUnit tests using Maven Surefire plugin configuration.

The configuration parameters are activated by default.

You can use these settings for Maven 2 and Maven 3 versions.

Item

Description

argLine

Select this checkbox to set arbitrary JVM options on the command line.

systemPropertyVariables

Select this checkbox to pass a list of System properties to the JUnit tests.

environmentVariables

Use this checkbox to set additional environment variables on the command line.
