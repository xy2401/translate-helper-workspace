Checkout from TFS Wizard: Checkout Mode

VCS | Checkout from Version Control | TFS

On this page, choose the way to map the files and folders on your server with their local copies. These mappings are referred to as workspaces. You can either specify new mappings and have a new workspace generated or use mappings from an existing workspace.

Item

Description

Create workspace automatically

Choose this option to have WebStorm generate a workspace for you, with the folder mapping based on your choice of the local and remote paths.

Workspace name

In this text box, type the name of the workspace to be generated.

Choose workspace manually

Choose this option to have the data downloaded according to the mappings from an existing workspace.
