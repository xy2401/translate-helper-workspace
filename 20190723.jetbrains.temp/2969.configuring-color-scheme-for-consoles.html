Configuring Color Scheme for Consoles

WebStorm enables you to define your habitual color scheme for the various types of consoles. Thus, you can individually configure all sorts of console output and user input.

Color scheme includes numerous colors for background, user input, system output, and error output.

  1. Make sure you are working with an editable scheme.

  2. In the Settings/Preferences dialog (Ctrl+Alt+S), go to Editor | Color Scheme, and select the setting pages related to consoles:

    • Console Colors

    • Console Font

  3. In the right-hand pane, select the desired component in the list, and change its color settings and font:
