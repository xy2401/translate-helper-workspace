Creating Ruby Class

IntelliJ IDEA enables you to create empty Ruby files or generate classes and modules. To create a new Ruby file, class, or module, do the following:

  1. In the Project tool window, select the directory where you want a class to be created, and press Alt+Insert.

  2. From the popup menu, select Ruby File/Class.

  3. In the New Ruby File/Class dialog, do one of the following:

    • To create a new Ruby file, enter its name in the Name field.

    • To create a Ruby class, switch the Kind option to Class and specify the class name in the Name field.

      Note that names of Ruby classes and modules should start with an uppercase letter.

    • To create a Ruby module, switch the Kind option to Module and specify the module name in the Name field.

    Click OK.
