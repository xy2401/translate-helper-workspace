WebStorm 2019.1 Help

Plugins

File | Settings | Plugins for Windows and Linux
WebStorm | Preferences | Plugins for macOS
Ctrl+Alt+S

Use the Plugins dialog to manage plugins: browse the plugin repository, install, remove, enable, disable, and update plugins.

Main controls

Control

Description

Marketplace

Use this tab to browse and install plugins from the JetBrains Plugin Repository.

If you want to browse your custom plugin repository, type repository: followed by the URL of the repository. For example:

repository:"http://plugins.example.com:8080/" myPlugin
Installed

Use this tab to browse your installed plugins, enable, disable, and remove them if necessary.

Updates

Use this tab to browse the plugins that require an update.

Configure additional plugin settings:

  • Manage Plugin Repositories: Open the Custom Plugin Repositories Dialog. For more information, see Custom plugin repositories.

  • HTTP Proxy Settings: Open the HTTP Proxy settings dialog.

  • Install Plugin from Disk: Select the local archive file (ZIP or JAR) from which to install a plugin.
