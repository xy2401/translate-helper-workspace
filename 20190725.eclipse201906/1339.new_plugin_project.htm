Plug-in Project

The New Plug-in Project creation wizard is available under File > New > Project... > Plug-in Project from the top level menu.

The creation of a plug-in project is a three-step process:

  1. Define the plug-in project structure
  2. Specify the plug-in content
  3. Add template extensions (optional)
