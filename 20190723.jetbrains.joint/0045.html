在你开始之前

确保您的计算机上安装了以下软件：

  • IntelliJ IDEA ULTIMATE版。

  • GlassFish Server，版本3.0.1或更高版本。下载GlassFish 。（您可以使用任何其他支持Java EE的应用程序服务器。这里使用GlassFish作为示例。）

  • 一个Web浏览器。

创建一个项目

  1. 单击欢迎屏幕上的创建新项目 ，或选择文件|新的|项目 。

    “ 新建项目”向导将打开。

  2. 在左侧窗格中，选择Java Enterprise 。

  3. 指定要使用的JDK （ Project SDK字段）：从列表中选择一个，单击New并选择JDK安装文件夹，或单击Download JDK 。

  4. 指定应用程序服务器。（我们将使用GlassFish Server。）

    如果尚未在IntelliJ IDEA中定义GlassFish，请单击Application Server字段右侧的New ，然后选择Glassfish Server 。

    在Glassfish服务器对话框中，指定GlassFish Server安装目录。

  5. 在“ 其他库和框架”下 ，选中“ Web应用程序”复选框。

    单击下一步 。

  6. 指定新项目的名称（例如JavaEEHelloWorld ）。

    单击完成并在IntelliJ IDEA创建项目时等待。

探索项目结构

创建项目后，您将在“ 项目”工具窗口中看到与此类似的内容。

  • JavaEEHelloWorld是一个模块文件夹（在本例中与项目文件夹重合）。.idea文件夹和文件JavaEEHelloWorld.iml包含项目和模块的配置数据。文件夹src用于Java源代码。文件夹web用于应用程序的Web部分。此文件夹包含部署描述符WEB-INF/web.xml和文件index.jsp用作应用程序的起始页面。

  • 外部库包括用于处理GlassFish的JDK和JAR文件。

开发源代码

我们的应用程序将是一个JSP页面应用程序。它唯一的功能就是输出文本Hello，World！

  1. 打开index.jsp进行编辑：在Project工具窗口中选择文件，然后按F4 。

  2. 在<body>和</body>输入Hello, World!

    此步骤的代码已准备就绪。

运行应用程序

在工作区的右上角，单击 。

IntelliJ IDEA编译您的源代码并构建应用程序工件 。

之后，将打开“ 运行”工具窗口。IntelliJ IDEA启动服务器并将工件部署到其上。

最后，您的默认Web浏览器启动，您会看到应用程序输出Hello, World! 那里。

修改代码并观察更改

  1. 在index.jsp ，更改Hello, World! Hello! 。

  2. 在“ 运行”工具窗口中，单击“更新” 。

  3. 在Update对话框中，选择Update resources并单击OK 。（有关更多信息，请参阅应用程序更新选项 。）

  4. 切换到Web浏览器并重新加载页面以查看更改。

Angular是构建跨平台应用程序的流行框架。IntelliJ IDEA为Angular提供支持，并帮助您完成开发过程的每一步 - 从创建新的Angular应用程序和处理组件到调试和测试它。

在你开始之前

  1. 下载并安装Node.js.

创建一个新的Angular应用程序

您也可以在Node.js和NPM页面上安装@angular/cli包，如npm和Yarn中所述 。

您可以使用Angular CLI或创建一个空的IntelliJ IDEA项目并在其中安装Angular。

Angular CLI是开始构建新Angular应用程序的推荐方法。

全局安装Angular CLI

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install -g @ angular / cli

创建应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的新建项目按钮。

  2. 在“ 项目类别和选项”对话框（“ 新建项目”向导的第一页）中，在左侧窗格中选择“ 静态Web ”。

  3. 在右侧窗格中，选择AngularCLI ，然后单击“ 下一步” 。

  4. 在向导的第二页上，指定项目名称和要在其中创建的文件夹。在Node Interpreter字段中，指定要使用的Node.js解释器。从列表中选择已配置的解释器，或选择“ 添加”以配置新的解释器在“ 角度CLI”字段中，指定@angular/cli包的路径。

    （可选 ）在“ 附加参数”字段中，指定要传递给Angular CLI的额外ng新选项 。此字段中提供了代码完成功能：当您开始键入选项名称或按Ctrl + Space时 ，IntelliJ IDEA会显示可用选项及其说明。

  5. 单击“完成”后，IntelliJ IDEA将生成一个包含所有必需配置文件的Angular特定项目。

创建一个空的IntelliJ IDEA项目

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的新建项目按钮。

  2. 在“ 项目类别和选项”对话框（“ 新建项目”向导的第一页）中，在左侧窗格中选择“ 静态Web ”。

  3. 在右侧窗格中，选择Static Web ，然后单击Next 。

  4. 在向导的第二页上，指定项目名称和要在其中创建的文件夹。

  5. 单击“完成”后，IntelliJ IDEA将创建并打开一个空项目。

在空项目中安装Angular

  1. 打开您将使用Angular的空项目。

  2. 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install @ angular / core

    这将使用框架的关键运行时部分安装核心Angular包。

    您可能还需要安装其他作为Angular部分的软件包，请参阅软件包列表 。

  3. 在打开的特定于VCS的对话框中，键入您的凭据和存储库以检出应用程序源。

运行和调试Angular应用程序

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    在Project工具窗口中选择package.json文件或在编辑器中打开它并在上下文菜单中选择Show npm Scripts时 ，将打开工具窗口。一旦调用npm或Yarn，它就会开始构建一个在调用它的package.json文件的scripts属性中定义的scripts树。如果项目中有多个package.json文件，则可以为每个文件构建一个单独的脚本树，并运行脚本而不删除以前构建的树。每棵树都显示在一个单独的节点下。

    如果尚未打开npm工具窗口，则打开它

    • 在Project工具窗口中选择所需的package.json文件，或在编辑器中打开它，然后从上下文菜单中选择Show npm Scripts 。将打开npm工具窗口，其中显示根据所选或已打开的package.json文件构建的脚本树。

    在已打开的npm工具窗口中构建脚本树

    • 在npm工具窗口中，单击在工具栏上，从列表中选择所需的package.json文件。默认情况下，PyCharm在项目的根目录中显示package.json文件。

    重建一棵树

    • 切换到所需的节点，然后单击在工具栏上。

    按名称对树中的脚本进行排序

    • 点击在工具栏上，从菜单中选择排序依据 ，然后选择名称 。
      默认情况下，树按照在package.json中定义的顺序显示脚本（选项定义顺序 ）。

    • 双击脚本。

    • 在树中选择脚本，然后按Enter或在上下文菜单中选择“运行<脚本名称> ”。

    调试一个脚本

    • 在树中选择脚本，然后从上下文菜单中选择Debug <script_name> 。

    运行或调试多个脚本

    • 使用多选模式：按住Shift （对于相邻项）或Ctrl （对于非相邻项）键并选择所需的脚本，然后在选择的上下文菜单中选择“运行”或“ 调试 ”。
