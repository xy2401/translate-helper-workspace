%ExtPoint.quickAccess

Identifier:
org.eclipse.ui.quickAccess

Since:
3.115

Description:
Customize the content of the Quick Access dialog and search field with extra entries.

Configuration Markup:

<!ELEMENT extension (computer+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT computer EMPTY>

<!ATTLIST computer

class            CDATA #REQUIRED

name             CDATA #REQUIRED

requiresUIAccess (true | false) >

  • class -
  • name -
  • requiresUIAccess -

Examples:

API Information:
This extension point leverages APIs provided in the org.eclipse.ui.quickaccess package.

Supplied Implementation:
org.eclipse.debug.ui contains an implementation of this extension point. Some already existing but hardwired QuickAccessProviders from org.eclipse.ui.internal.quickaccess package may be contributed through this extension point in future versions.
