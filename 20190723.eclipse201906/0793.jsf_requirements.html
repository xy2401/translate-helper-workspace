Requirements and Installation


Requirements


The JSF Tools project requires:
  • Java 2 Platform, Standard Edition(J2SE) 1.4 or later
  • Eclipse 3.3
  • Eclipse Web Tools Platform Project(WTP) 2.0
  • JavaServer Faces Implementation v1.1 or later

To execute a JSF page/application, it is required to have a server runtime capable of executing a J2EE v1.4 or better web module.

You must also have:
  • JavaServer Pages Standard Tag Library
