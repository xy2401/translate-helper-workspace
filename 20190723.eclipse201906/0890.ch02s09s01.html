2.9.1. Installing the converter plugin

  1. Start the ITE and select from the main menu Help-->Install new software....

  2. Click the "Add..." button in the Install dialog that appears.

  3. Click the "Archive..." button in the Add Repository dialog that appears

  4. Navigate to the directory where Jubula is installed and select the org.eclipse.jubula.repo.zip in the development folder

  5. Click "OK" to exit the Add Repository dialog. The active dialog should now be Install.

  6. Select the Feature with the name "Jubula Functional Testing - API Converter (experimental)" and click "Next"

  7. Confirm the Installation Details by clicking the "Next" button again.

  8. Accept the license agreement terms and click "Finish".

  9. Click the "Restart Now" button to perform a restart, when a dialog appears to prompt the restart that will safely finish the update / installation. After the restart the converter is installed.
