3.2.2. Deploying the toolkit plug-in

Once the toolkit feature has been created, it must be deployed in the ITE. The toolkit feature must be exported to an update site and then installed via this update site into the ITE. If you want to execute tests on your new components via testexec, it is important that you have write access to the ITE so that the installed feature can be accessed during test execution.

3.2.2.1. Create an update site for the toolkit feature

  1. Create an Update Site Project.

  2. Add a category to your update site that defines its ID and Name.

  3. Add your toolkit feature to the category created in the previous step.

  4. Build the created toolkit update site. The directory of the toolkit update site project serves as an update site, which is later used to install your toolkit feature into the ITE. Remember to build and then synchronize this update site if you have changed the toolkit feature.

3.2.2.2. Install the toolkit feature from the update site

  1. Start the ITE and select from the main menu Help-->Install new software....

  2. Click the ”Add...” button in the Install dialog that appears.

  3. Click the ”Local...” button in the Add Repository dialog that appears

  4. Navigate to the directory that contains your update site and confirm your selection.

  5. Click ”OK” to exit the Add Repository dialog. The active dialog should now be Install.

  6. Ensure that the check box named Group items by category is deselected. Your feature should be visible in the central table of the dialog.

  7. Ensure that the check box next to your feature is selected and click ”Next”.

  8. Confirm the Installation Details by clicking the ”Next” button again.

  9. Accept the license agreement terms and click ”Finish”.

  10. A warning dialog may appear to warn you are installing unsigned content. Click ”OK” in this dialog if the feature comes from a trusted source. This should be the case as long as you install your own feature. Then the installation process begins.

  11. Click the ”Restart Now” button to perform a restart, when a dialog appears to prompt the restart that will safely finish the update / installation. After the restart your ITE toolkit extension feature is installed.
