RWT Theming Reference > CTabFolder

CTabFolder

This widget implements the notebook user interface metaphor. It allows the user to select a notebook page from set of pages.

Available CSS Properties

Properties that can be defined for this element:

  • border-color

    CTabFolder border color.

  • border-radius

    CTabFolder border radius.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to items of a disabled CTabFolder.

CTabItem

Items of a CTabFolder control.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color for the item tabs.

  • background-image

    Background image or gradient for CTabItem.

  • color

    Text color for the item tabs.

  • font

    Font for the item tabs.

  • padding

    Padding for CTabItem

  • spacing

    Spacing between text and image

  • text-decoration

    Decoration for CTabItem text.

  • text-shadow

    Shadow for CTabItem text.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to items of a disabled CTabFolder.

  • :selected

    Applies to the selected item.

CTabFolder-DropDownButton-Icon

The drop-down button icon (chevron).

Available CSS Properties

Properties that can be defined for this element:

  • background-image

    The icon for the CTabFolder drop-down button.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [RIGHT_TO_LEFT]

    Indicates that the CTabFolder. is oriented right to left.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :hover

    Applies while the user hovers over the button.

  • :pressed

    Applies while a button is pressed.
