4.25. Environment

Parameter Name Data Type Possible Values Default
ENVIRONMENT String

  • AUT

  • ITE

AUT
Table 4.26. ENVIRONMENT

  • Set this parameter to AUT when your tests are e.g. executed on only one machine, i.e. ITE/Testexec, AUT Agent, AUT and the external command can all be found on the same machine.

  • Set this parameter to ITE when your external AUT Agent and the AUT run on a different machine than the ITE/Testexec and the external command be found on the machine on which the AUT is running.
