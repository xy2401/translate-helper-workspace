4.5.1. Introduction to writing tests for GEF AUTs

You can write tests for JavaFX AUTs when you select the toolkit concrete or JavaFX in the Project properties. The JavaFX and concrete toolkits actually contain the same actions as each other.

When you select JavaFX as the Project toolkit, the library Project unbound_modules_concrete is automatically reused in your Project. The actions in this library are described in the reference manual.
