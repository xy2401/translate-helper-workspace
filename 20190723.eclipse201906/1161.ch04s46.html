4.46. Margin Right

Parameter Name Data Type Possible Values Default
MARGIN_RIGHT Integer   0
Table 4.47. MARGIN_RIGHT

  • Use this parameter to enter the amount of pixels you wish to include in the screenshot to the right of the dialog

  • You can enter positive and negative values.

  • Enter 0 to neither extend nor reduce the margin.
