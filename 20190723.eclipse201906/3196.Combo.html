Combo

The theming of RWT Combo.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The background color used for the text field and the drop-down list of a Combo.

  • background-image

    Background gradient for combo text field.

  • border

    The border that is drawn around a Combo.

  • border-bottom

    The bottom side border for Combo.

  • border-left

    The left side border for Combo.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for Combo.

  • border-top

    The top side border for Combo.

  • box-shadow

    A shadow effect for combo text field.

  • color

    The text color used for the text field and the drop-down list of a Combo.

  • font

    The font used for the text field and the drop-down list of a Combo.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled Combos.

  • :focused

    Applies to focused Combos.

Combo-Button

The drop-down button of RWT Combo.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The background color of the drop-down button of the Combo.

  • background-image

    Background image or gradient for combo drop-down button.

  • border

    The border of the drop-down button of the Combo.

  • border-bottom

    The bottom side border of the Combo drop-down button.

  • border-left

    The left side border of the Combo drop-down button.

  • border-left

    The left side border of the Combo drop-down button.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border of the Combo drop-down button.

  • border-top

    The top side border of the Combo drop-down button.

  • cursor

    Combo button cursor.

  • width

    The width of the drop-down button of the Combo.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [RIGHT_TO_LEFT]

    Indicates that the combo is oriented right to left.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies when the button is disabled.

  • :hover

    Applies while the user hovers over the combo button.

  • :pressed

    Applies while a button is pressed.

Combo-List-Item

Items of Combo list.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Item background color

  • background-image

    Item background image/gradient

  • color

    Item text color

  • padding

    Defines the padding (i.e. the inner distance between border and content) for a DropDown item.

  • text-decoration

    Decoration for item text.

  • text-shadow

    Shadow for item text.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :even

    Applies to Combo list items with an even index.

  • :hover

    Applies while the user hovers over the Combo list item.

  • :selected

    Applies when Combo list item is selected.

Combo-FocusIndicator

The frame that is drawn when the combo is focused.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The fill-color of the frame.

  • border

    The border of the frame. 3D-styles inset, outset, etc are not supported.

  • border-bottom

    The bottom side border of the frame.

  • border-left

    The left side border of the frame.

  • border-right

    The right side border of the frame.

  • border-top

    The top side border of the frame.

  • margin

    The margin between the combos border and the frame.

  • opacity

    The opacity of the indicator. A value between 0 and 1.

Applicable SWT Styles

none

Applicable States
