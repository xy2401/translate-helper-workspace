4.12. Count

Parameter Name Data Type Possible Values Default
COUNT Integer   none
Table 4.13. COUNT

  • Using this parameter you can define a certain number of expected items in combination with CSS Path Count and XPath Count.

  • E.g.: If you expect the amount of items matching an XPath locator to be less or equal to 4 enter 4 as the parameter value.
