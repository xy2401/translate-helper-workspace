3.9.4. Deleting Test Cases with orphans

  1. To delete a Test Case together with orphans that would remain unused, select it in the browser.

  2. Select Delete with orphans from the context-sensitive menu.

If the item you want to delete is referenced somewhere in your test, you must first remove it from the place where it is reused before deleting it.

Before confirming the operation, make sure that you really want to delete every Test Case that would remain unused. This cannot be undone!
