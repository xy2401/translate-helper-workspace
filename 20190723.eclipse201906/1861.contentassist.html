OT/J content/code assist

The content/code assist (code completions) inherent in the Java editor has been extended in order to provide for Object Teams specific keywords and modifiers (e.g. team, playedBy, callin, before, after, replace etc.) as well. Code completion and quick fixes provide more specific assistance depending on the current context of editing.

OT/J specific content assist comprises:

  • Completion (callout, callin, base class).
  • Quick fixes
  • Quick assists
