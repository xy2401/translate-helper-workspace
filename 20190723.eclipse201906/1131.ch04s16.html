4.16. Delay after closure

Parameter Name Data Type Possible Values Default
DELAY_AFTER_CLOSURE Integer   200
Table 4.17. DELAY_AFTER_CLOSURE

  • Sometimes windows are not ready to receive events (mouse clicks, key presses) until a view moments after the overlapping window is closed.

  • Use this parameter to set the amount of time (in milliseconds) to wait once the window has closed before continuing.
