4.78. Textpath

Parameter Name Data Type Possible Values Default
TEXTPATH String   none
Table 4.79. TEXTPATH

  • Enter the path to an item as a textpath.

  • Use slash '/' as a path separator to separate parent items from child items.

  • For example File/Open/ or Category/Action.

  • Either make sure that your path is written exactly as it appears in the interface or use a regular expression to match the text.

  • Each segment of the path will be used to find a corresponding item, using the operator provided.
