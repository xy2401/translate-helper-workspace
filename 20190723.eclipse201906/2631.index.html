Using the Profiler

The following Profile functionality is available in PDT :

  • Profiling Local PHP Script - Profile files on your workspace using your internal debugger.
  • Profiling PHP Web Page - Profile applications situated on a server with the use of PDT launch configurations.
  • Profiling with Browser Toolbars - Profile applications situated on a server with the use of different browser toolbars.

Once a profile session has been launched, the PHP Profile perspective is used to view and analyze the results.
