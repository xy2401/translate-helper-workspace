5.2. Installation

  1. Start the IDE and select from the main menu Help > Install new software....

  2. Click the ” Add...” button in the Install dialog that appears.

  3. Click the ”Archive” button in the Add Repository dialog that appears.

  4. Navigate to the jubula installation directory and choose the "org.eclipse.jubula.extensions.wizard.repo.zip" file from the development directory.

  5. You can optionally give the repository a local name.

  6. Click ”OK” to exit the Add Repository dialog. The active dialog should now be Install.

  7. Tick the checkbox next to the Jubula Extension Development category or the Jubula Extension Wizard feature and click ”Next”.

  8. Confirm the Installation Details by clicking the ”Next” button again.

  9. Accept the license agreement terms and click”Finish”.

  10. A warning dialog may appear to warn you are installing unsigned content. Click ”OK”. Then the installation process begins.

  11. Click the ”Restart Now” button to perform a restart, when a dialog appears to prompt the restart that will safely finish the update / installation. After the restart the Jubula Extension Wizard will be available in the "New Project" menu.

The Jubula Extension Wizard is not supposed to be installed into the Jubula ITE or into Eclipse for Testers, seeSection 5.1, “Requirements”.
