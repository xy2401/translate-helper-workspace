System Requirements

  • Windows XP/Vista/7 or Linux+GTK2 or OSX
    • 32-bit & 64-bit
    • Linux GTK 2.14 or higher (GTK 2.20 or higher recommended)
  • Eclipse must be launched using JDK 1.6 or higher
  • The Eclipse JDT must be fully installed.
  • The PDE must be installed for SWT/RCP support.
  • WST (or WTP) must be installed for XML/XWT support
    • The Eclipse XML Editors and Tools feature should be loaded for XML editing
  • The product may not work with remote desktop environments (except simple ones like VNC)
  • Any Eclipse 3.5, 3.6, 3.7, 3.8 or 4.2 based IDE

Installing the Update Site or Zip editions for Eclipse 3.5 or above requires the full Eclipse SDK including the JDT (and PDE for SWT/RCP support). Use the Eclipse Classic, Java or RCP/RAP Developers distribution or install the JDT and PDE separately.

When using any XML-based API such as Eclipse XWT, the Eclipse XML Editors and Tools feature (from WST/WTP) should be loaded into Eclipse for the optimal XML editing experience.
