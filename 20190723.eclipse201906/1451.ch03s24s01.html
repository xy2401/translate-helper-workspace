3.24.1. Introduction to external task repositories

The ITE is integrated with the Eclipse Mylyn Project to allow you to:

  • Connect your ITE (via the workspace) to external task repositories.

  • View, work on and edit tasks directly in the ITE.

  • Create tasks directly in the ITE.

  • Integrate your tests with application lifecycle management (ALM) tools to provide information on the automated tests performed for specfic features or releases. We have tested the integration with Jira, Bugzilla and HP ALM.

The Mylyn online help is included in the installation.

Select:

Help-->Help Contents

in the ITE to see the Mylyn help.
