Task Execution

  

The gathered task list is prepared to perform the specified tasks. This list is presented to the user on the setup wizard's progressPage page. Initially, each task in the list is visited to determine if it needs to perform. That determination is generally influenced by the trigger. Tasks that don't need to perform are logically removed from the list to induce the so called needed task list. The progressPage page allows the user to see either the full list or the needed task list. Each task remaining in the reduced list of needed tasks is then performed.
