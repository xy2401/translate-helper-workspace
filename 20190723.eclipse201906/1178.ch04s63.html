4.63. Pre-Ascend

Parameter Name Data Type Possible Values Default
PRE_ASCEND Integer   0
Table 4.64. PRE_ASCEND

  • Use this parameter with a relative path to start the search above the currently selected item.

  • The search will begin n items above the currently selected item, where n is the value of this parameter.

  • For example, a value of 0 will simply begin with the currently selected item, and a value of 2 will begin with the parent of the parent of the currently selected item.

  • A value can be given such that the search begins 1 level above the topmost visible level. This is then equivalent to an absolute path.

  • If the search would begin any higher than this, the action fails.
