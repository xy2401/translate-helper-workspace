4.2. Accessibility plug-in example

A sample accessibility plug-in for certain elements of the Logic Diagram sample plug-in project (contributed from the GEF plug-ins to Eclipse's New Project Wizard) are included in the installation. A default implementation can be found in the directory examples/development/gef . There you will find a zip-file containing a full accessibility plug-in example.

You can import the sample accessibility plug-in into an Eclipse workspace in order to examine the general structure. The Java documentation of the source code, which is included in the target platform definition, contains information about IEditPartIdentifier from the package named org.eclipse.jubula.rc.rcp.gef.identifier.
