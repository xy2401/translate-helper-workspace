Chapter 9. The AppStream repository

Content in the AppStream repository includes additional user space applications, runtime languages, and databases in support of the varied workloads and use cases. Content in AppStream is available in one of two formats - the familiar RPM format and an extension to the RPM format called modules.
For detailed information on the length of support for components included in this repository, see the Red Hat Enterprise Linux 8 Application Streams Life Cycle.
The following table lists all the packages in the AppStream repository. Core packages are explicitly referenced in a comps group and can be selected directly using anaconda or kickstart. For more information, see the Scope of Coverage Details document.
