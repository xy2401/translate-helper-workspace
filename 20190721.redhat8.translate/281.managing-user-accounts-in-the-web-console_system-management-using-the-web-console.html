Chapter 9. 在Web控制台中管理用户帐户

RHEL 8 Web控制台提供用于添加，编辑和删除系统用户帐户的界面。阅读本节后，您将了解：

  • 从现有帐户来自哪里。
  • 如何添加新帐户。
  • 如何设置密码到期时间。
  • 如何以及何时终止用户会话。

先决条件

  • 使用分配了管理员权限的帐户登录RHEL 8 Web控制台。有关详细信息，请参阅登录RHEL Web控制台 。

在Web控制台中管理的系统用户帐户

通过RHEL 8 Web控制台中显示的用户帐户，您可以：

  • 访问系统时对用户进行身份验证。
  • 设置他们对系统的访问权限。

RHEL 8 Web控制台显示系统中的所有用户帐户。因此，您可以在首次登录Web控制台后看到至少一个用户帐户。

如果您已登录RHEL 8 Web控制台，则可以：

  • 创建新用户帐户。
  • 改变他们的参数。
  • 锁定帐户。
  • 终止用户会话。

您可以在“ 帐户”设置中找到帐户管理。

在Web控制台中添加新帐户

下面介绍如何在RHEL 8 Web控制台中添加系统用户帐户以及为帐户设置管理权限。

程序

  1. 登录RHEL Web控制台。
  2. 单击帐户 。
  3. 单击“ 创建新帐户” 。

  4. 在“ 全名”字段中，输入用户的全名。

    RHEL Web控制台自动从全名建议用户名，并在“ 用户名”字段中填写。如果您不想使用由名字的第一个字母和整个姓氏组成的原始命名约定，请更新建议。

  5. 在“ 密码/确认”字段中，输入密码并重新键入密码以验证密码是否正确。放置在字段下方的颜色条显示输入密码的安全级别，这不允许您创建具有弱密码的用户。

  6. 单击“ 创建”以保存设置并关闭对话框。
  7. 选择新创建的帐户。
  8. 在“ 角色”项中选择“ Server Administrator ”。

现在，您可以在“ 帐户”设置中看到新帐户，并且可以使用凭据连接到系统。

在Web控制台中强制密码过期

默认情况下，用户帐户已将密码设置为永不过期。要以管理员身份强制执行密码过期，请将系统密码设置为在指定的天数后过期。

密码过期后，下次登录尝试将提示更改密码。

程序

  1. 登录RHEL 8 Web控制台界面。
  2. 单击帐户 。
  3. 选择要强制密码过期的用户帐户。
  4. 在用户帐户设置中，单击“ 永不过期密码” 。

  5. 在“ 密码过期”对话框中，选择“ 每隔...天需要更改密码 ”，然后输入一个正整数，表示密码过期的天数。

  6. 单击更改 。

要验证设置，请打开帐户设置。RHEL 8 Web控制台显示一个到期日期的链接。

在Web控制台中终止用户会话

用户在登录系统时创建用户会话。终止用户会话意味着将用户从系统中注销。

如果您需要执行对配置更改敏感的管理任务（例如，系统升级），则会很有帮助。

在RHEL 8 Web控制台中的每个用户帐户中，您可以终止该帐户的所有会话，但您当前使用的Web控制台会话除外。这可以防止您将自己从系统中切除。

程序

  1. 登录RHEL 8 Web控制台。
  2. 单击帐户 。
  3. 单击要终止会话的用户帐户。

  4. 单击“ 终止会话”按钮。

    如果“ 终止会话”按钮处于非活动状态，则用户未登录系统。

RHEL Web控制台终止会话。
