Chapter 9. 使用Web控制台选择性能配置文件

Red Hat Enterprise Linux 8包括性能配置文件优化：

  • 使用桌面的系统
  • 延迟表现
  • 网络性能
  • 功耗低
  • 虚拟机

以下过程介绍如何在Web控制台中设置性能配置文件。

RHEL 8 Web控制台配置tuned服务。

有关已tuned服务的详细信息，请参阅监视和管理系统状态和性能 。

先决条件

  • 必须安装和访问Web控制台。

    有关详细信息，请参阅安装Web控制台 。

程序

  1. 登录RHEL 8 Web控制台。

    有关详细信息，请参阅登录Web控制台 。

  2. 单击系统 。

  3. 在“ 性能配置文件”字段中，单击当前性能配置文件。

  4. 在“ 更改性能配置文件”对话框中，根据需要更改配置文件。

  5. 单击更改 。

现在，“ 系统”选项卡中提供了更改。
