⁠Chapter 6. The Resilient Storage add-on

The Resilient Storage add-on enables a shared storage or clustered file system to access the same storage device over a network through a pool of data that is available to each server in the group. The add-on is available with a separate subscription. For details, see Resilient Storage Add-On.
The following table lists all the packages available with the Resilient Storage add-on. Core packages are explicitly referenced in a comps group and can be selected directly using anaconda or kickstart. For more information, see the Scope of Coverage Details document.
