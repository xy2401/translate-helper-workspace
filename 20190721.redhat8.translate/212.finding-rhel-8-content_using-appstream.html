Chapter 9. 查找RHEL 8内容

以下部分描述了如何在Red Hat Enterprise Linux 8中的AppStream和BaseOS存储库中查找和检查内容。

  • “搜索包”一节描述了如何搜索提供所需内容的包。
  • “列出可用模块”一节描述了如何列出可用模块并找出有关它们的详细信息。
  • 名为“示例：查找有关模块的详细信息”的部分包含更详细地检查模块所需步骤的示例。
  • 名为“列出内容的命令”的部分提供了对检查内容有用的命令的参考。

正在搜索包裹

本节介绍查找提供特定应用程序或其他内容的程序包所需的步骤。

先决条件

  • 必须知道所需应用程序或内容的名称

程序

  1. 搜索包含文本字符串的包，例如应用程序名称：

    $ yum search "text string"

  2. 查看有关包的详细信息：

    $ yum info package

列出可用模块

本节介绍了查找可用模块及其详细信息所需的步骤。

程序

  1. 列出系统可用的模块流：

    $ yum module list

    此命令的输出在单独的行中列出具有名称，流，配置文件和摘要的模块流。

  2. 显示有关模块的详细信息，包括说明，所有配置文件的列表以及所有提供的包的列表：

    $ yum module info module-name

  3. 可选：您还可以列出每个模块配置文件中安装了哪些软件包：

    $ yum module info --profile module-name

  4. 显示模块的当前状态，包括已启用的流和已安装的配置文件：

    $ yum module list module-name

示例：查找有关模块的详细信息

    由于未指定流，因此所有流都用于列表。

用于列出内容的命令

本节列出了常用的查找内容及其详细信息的命令。

命令列表

列出可用的包
$ yum list available
使用任意文本字符串搜索包
$ yum search "text string"
显示包的详细信息
$ yum info package
找出哪些模块提供了一个包
$ yum module provides package

如果程序包在任何模块外部可用，则此命令的输出为空。

列出可用模块
$ yum module list
显示模块的详细信息
$ yum module info module-name
使用默认流列出由模块的配置文件安装的包
$ yum module info --profile module-name
使用指定的流显示由模块的配置文件安装的包
$ yum module info --profile module-name:stream
显示模块的当前状态
$ yum module list module-name
