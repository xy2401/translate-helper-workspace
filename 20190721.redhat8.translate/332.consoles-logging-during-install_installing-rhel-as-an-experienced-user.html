Chapter 9. 安装过程中的控制台和日志记录

Red Hat Enterprise Linux安装程序使用tmux终端多路复用器来显示和控制除主界面之外可以使用的多个窗口。这些窗口中的每一个都有不同的用途 - 它们显示多个不同的日志，可用于解决安装过程中的任何问题，其中一个窗口提供具有root权限的交互式shell提示符，除非使用引导专门禁用此提示选项或Kickstart命令。

注意

通常，除非您需要诊断安装问题，否则没有理由离开默认的图形安装环境。

终端多路复用器在虚拟控制台1中运行。要从实际安装环境切换到tmux ，请按Ctrl + Alt + F1 。要返回在虚拟控制台6中运行的主安装界面，请按Ctrl + Alt + F6 。

注意

如果选择文本模式安装，则将在虚拟控制台1（ tmux ）中启动，切换到控制台6将打开shell提示符而不是图形界面。

运行tmux的控制台有5个可用窗口;它们的内容在下表中描述，以及用于访问它们的键盘快捷键。请注意，键盘快捷键分为两部分：首先按Ctrl + b ，然后释放两个键，并按下要使用的窗口的数字键。

您也可以使用Ctrl + b n和Ctrl + b p分别切换到下一个或上一个tmux窗口。

表7.1。可用的tmux窗口

捷径内容

Ctrl + b 1

主安装程序窗口。包含基于文本的提示（在文本模式安装期间或使用VNC直接模式时），以及一些调试信息。

Ctrl + b 2

具有root权限的交互式shell提示符。

Ctrl + b 3

安装日志;显示存储在/tmp/anaconda.log消息。

Ctrl + b 4

存储日志;显示存储在/tmp/storage.log内核和系统服务相关存储设备的消息。

Ctrl + b 5

程序日志;显示来自其他系统实用程序的消息，存储在/tmp/program.log 。
