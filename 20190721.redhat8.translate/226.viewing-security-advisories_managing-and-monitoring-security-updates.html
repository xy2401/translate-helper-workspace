Chapter 9. 查看安全建议

本章介绍了有关Red Hat Security Advisories（RHSA）以及如何显示建议的信息。

在客户门户上显示建议

红帽在红帽客户门户网站上发布安全建议。本节介绍了查找建议的位置，以及如何过滤和显示建议。

程序

  1. 在浏览器中打开https://access.redhat.com/security/security-updates/ 。

    此页面列出了Red Hat发布的所有安全建议。

  2. （可选）筛选特定产品，变体，版本和体系结构。例如，要仅显示Red Hat Enterprise Linux 8服务器的建议，请设置以下过滤器：

    • 产品：红帽企业Linux
    • 变体：红帽企业Linux服务器
    • 版本：8

  3. 要显示特定顾问的详细信息，请单击表中的顾问ID。

使用yum显示特定的顾问

如果尚未安装yum提供的更新，请使用yum实用程序显示通报。

先决条件

  • 已为主机分配有效的Red Hat订阅。
  • 安全建议的ID已知。有关显示主机的已安装和可用安全更新的建议的详细信息，请参阅第2章， 标识安全更新 。
  • 未安装该通报提供的更新。

程序

  1. 显示建议。例如，要显示RHSA-2019:0997咨询的详细信息：

    $ sudo yum updateinfo info RHSA-2019:0997
