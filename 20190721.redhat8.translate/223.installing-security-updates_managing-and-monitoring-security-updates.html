Chapter 9. 安装安全更新

本章介绍如何在Red Hat Enterprise Linux 8上安装安全更新。

先决条件

  • 已为主机分配有效的Red Hat订阅。

安装所有可用的安全更新

本节介绍如何安装主机可用的所有安全更新。

程序

  1. 要安装所有安全更新，请输入：

    $ sudo yum update --security

    请注意，如果没有--security参数， yum也会安装包含错误修复和增强功能的更新。

  2. 按y确认，然后开始安装：

    ...
Transaction Summary
===========================================
Upgrade  ... Packages

Total download size: ... M
Is this ok [y/d/N]: y

  3. （可选）列出安装更新的软件包后需要手动重新启动的进程：

    $ yum needs-restarting

安装特定通报提供的安全更新

在某些情况下，例如，如果可以在不安排停机时更新特定服务，则管理员只希望为此服务安装安全更新，并在以后安装所有其他安全更新。

本节介绍如何安装特定安全通报提供的更新包。

先决条件

  • 已为主机分配有效的Red Hat订阅。
  • 安全建议的ID已知。有关显示主机的已安装和可用安全更新的建议的详细信息，请参阅第2章， 标识安全更新 。

程序

  1. 安装特定安全通报提供的安全更新。例如，要安装RHSA-2019:0997顾问提供的更新，请输入：

    $ sudo yum update --advisory=RHSA-2019:0997

  2. 按y确认，然后开始安装：

    ...

===========================================

Total download size: ... M
Is this ok [y/d/N]: y

  3. （可选）列出安装更新的软件包后需要手动重新启动的进程：

    $ yum needs-restarting
