Chapter 9. 安装Image Builder

在使用Image Builder之前，必须在虚拟机中安装Image Builder。

在虚拟机中安装Image Builder

要在专用虚拟机上安装Image Builder，请按照下列步骤操作：

先决条件

  • 连接到虚拟机。
  • 必须安装，订阅和运行Image Builder的虚拟机。

程序

  1. 在虚拟机上安装Image Builder和其他必需的软件包：

    • 绕口令作曲家
    • 作曲家CLI
    • 驾驶舱作曲家

    • 庆典完成

      # yum install lorax-composer composer-cli cockpit-composer bash-completion

      Web控制台作为cockpit-composer包的依赖项安装。

  2. 启用Image Builder在每次重启后启动：

    # systemctl enable lorax-composer.socket
# systemctl enable cockpit.socket

    lorax-composer和cockpit服务在首次访问时自动启动。

  3. 配置系统防火墙以允许访问Web控制台：

    # firewall-cmd --add-service=cockpit && firewall-cmd --add-service=cockpit --permanent

  4. 加载shell配置脚本，以便composer-cli命令的自动完成功能立即开始工作而无需重新启动：

    $ source  /etc/bash_completion.d/composer-cli
