Chapter 9. 完成安装后任务

本节介绍如何完成以下安装后任务：

  • 完成初始设置
  • 注册您的系统
  • 保护您的系统

完成初始设置

本节包含有关如何在Red Hat Enterprise Linux 8系统上完成初始设置的信息。

重要

如果在安装期间选择了具有GUI基本环境的服务器 ，则在安装过程完成后首次重新引导系统时将打开“ 初始设置”窗口。

“ 初始设置”窗口中显示的信息可能因安装期间配置的内容而异。至少会显示许可和订阅管理器选项。

条件

  • 您已根据“图形安装工作流程”一节中介绍的建议工作流程完成了图形安装。
  • 您有一个活跃的，未评估的Red Hat Enterprise Linux订阅。

程序

  1. 从“ 初始设置”窗口中，选择“ 许可信息” 。

    “ 许可协议”窗口将打开并显示Red Hat Enterprise Linux的许可条款。

  2. 查看许可协议，然后选中“ 我接受许可协议”复选框。

    注意

    您必须接受许可协议。退出初始设置而未完成此步骤会导致系统重新启动。重新启动过程完成后，系统会提示您再次接受许可协议。

  3. 单击“完成”以应用设置并返回“ 初始设置”窗口。

    注意

    如果未配置网络设置，则无法立即注册系统。在这种情况下，单击“ 完成配置” 。Red Hat Enterprise Linux 8启动，您可以登录，激活对网络的访问，以及注册您的系统。有关详细信息，请参阅“使用Subscription Manager用户界面注册系统”一节 。如果您配置了网络设置，如“配置网络和主机名选项”一节中所述 ，您可以立即注册系统，如以下步骤所示：

  4. 从“ 初始设置”窗口中，选择“ 订阅管理器” 。
  5. Subscription Manager图形界面将打开并显示您要注册的选项，即： subscription.rhsm.redhat.com 。
  6. 单击下一步 。
  7. 输入您的登录名和密码详情，然后单击注册 。
  8. 确认订阅详细信息，然后单击“ 附加” 。您必须收到以下确认消息： Red Hat Subscription Management的注册已完成！
  9. 单击“完成” 。初始设置窗口打开。
  10. 单击完成配置 。登录窗口打开。
  11. 配置您的系统。有关更多信息，请参阅配置基本系统设置文档。

其他资源

注册系统有四种方法：

  • 在使用初始设置安装期间。
  • 使用命令行安装后。有关详细信息，请参阅“使用命令行注册系统”一节 。
  • 使用Subscription Manager用户界面安装后。有关详细信息，请参阅“使用Subscription Manager用户界面注册系统”一节 。
  • 使用注册助手安装后。注册助手旨在帮助您为Red Hat Enterprise Linux环境选择最合适的注册选项。有关详细信息，请参阅https://access.redhat.com/labs/registrationassistant/ 。

使用命令行注册系统

本节包含有关如何使用命令行注册Red Hat Enterprise Linux 8系统的信息。

注意

自动附加系统时，订阅服务会检查系统是物理还是虚拟，以及系统上有多少个套接字。物理系统通常消耗两个权利，虚拟系统通常消耗一个权利。系统上每两个插槽消耗一个权利。

先决条件

  • 您有一个活跃的，未评估的Red Hat Enterprise Linux订阅。
  • 您的Red Hat订阅状态已经过验证。
  • 您之前没有收到过Red Hat Enterprise Linux 8订阅。
  • 在尝试从客户门户下载权利之前，您已激活订阅。您需要为计划使用的每个实例授予权利。如果您需要帮助激活订阅，可以使用红帽客户服务。
  • 您已成功安装Red Hat Enterprise Linux 8并已登录系统。

程序

  1. 打开终端窗口并运行以下命令注册订阅：

    # subscription-manager register

  2. 输入您的客户门户凭据：

    # Registering to: subscription.rhsm.redhat.com:443/subscription
# Username: USERNAME
# Password: PASSWORD

  3. 订阅成功注册后，您应该看到类似于以下示例的输出：

    # The system has been registered with ID: 123456abcdef
# The registered system name is: localhost.localdomain

  4. 设置系统预期用途的角色：

    注意

    可用角色取决于组织购买的订阅和RHEL 8系统的体系结构。您可以设置以下角色之一： Red Hat Enterprise Linux Server ， Red Hat Enterprise Linux Workstation或Red Hat Enterprise Linux Compute Node 。 

    # subscription-manager role --set="Red Hat Enterprise Linux Server"

  5. 将系统附加到与主机系统体系结构匹配的权利：

    # subscription-manager attach

  6. 成功附加订阅后，您应该看到类似于以下示例的输出：

    Installed Product Current Status:
Product Name: Red Hat Enterprise Linux for x86_64
Status: Subscribed

注意

您还可以以root用户身份登录系统并使用Subscription Manager图形用户界面注册Red Hat Enterprise Linux 8。

使用Subscription Manager用户界面注册您的系统

本节包含有关如何使用Subscription Manager用户界面注册Red Hat Enterprise Linux 8系统以接收更新和访问软件包存储库的信息。

先决条件

  • 您已按照“图形安装工作流程”一节中所述的建议工作流程完成了图形安装。
  • 您有一个活跃的，未评估的Red Hat Enterprise Linux订阅。
  • 您的Red Hat订阅状态已经过验证。

程序

  1. 登录您的系统。
  2. 从窗口的左上角，单击“ 活动” 。
  3. 从菜单选项中，单击“ 显示应用程序”图标。
  4. 单击Red Hat Subscription Manager图标，或在搜索中输入Red Hat Subscription Manager 。

  5. 在“ 需要身份验证”对话框中输入管理员密码。

    注意

    需要身份验证才能在系统上执行特权任务。

  6. 将打开“ 订阅”窗口，其中显示“订阅”，“系统用途”和已安装产品的当前状态。未注册的产品显示红色X.
  7. 单击“ 注册”按钮。
  8. 将打开“ 注册系统”对话框。输入您的客户门户凭据，然后单击“ 注册”按钮。

“ 订阅”窗口中的“ 注册”按钮更改为“ 取消注册” ，已安装的产品显示绿色X.您可以使用subscription-manager status命令对注册失败进行故障排除。

其他资源

  • 有关Subscription Manager的更多信息，请参阅使用和配置Red Hat Subscription Manager文档。

注册助理

注册助手旨在帮助您为Red Hat Enterprise Linux环境选择最合适的注册选项。有关详细信息，请参阅https://access.redhat.com/labs/registrationassistant/ 。

保护您的系统

安装Red Hat Enterprise Linux后立即完成以下与安全相关的步骤。

先决条件

  • 您已根据“图形安装工作流程”一节中所述的建议工作流程完成了图形安装。

程序

  1. 要更新系统，请以root身份运行以下命令：

    # yum update

  2. 即使防火墙服务firewalld通过安装Red Hat Enterprise Linux自动启用，也有可能明确禁用它的情况，例如在Kickstart配置中。在这种情况下，建议您重新启用防火墙。

    要启动firewalld ，请以root身份运行以下命令：

    # systemctl start firewalld
# systemctl enable firewalld

  3. 要增强安全性，请禁用不需要的服务。例如，如果系统未安装打印机，请使用以下命令禁用cups服务：

    # systemctl mask cups

    要查看活动服务，请运行以下命令：

    $ systemctl list-units | grep service
