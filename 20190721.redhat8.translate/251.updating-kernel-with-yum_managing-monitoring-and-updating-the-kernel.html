Chapter 9. 使用yum更新内核

以下部分介绍了Red Hat（Red Hat内核）提供和维护的Linux内核的信息，以及如何更新Red Hat内核。因此，操作系统将具有所有最新的错误修复，性能增强和补丁，以确保与新硬件的兼容性。

什么是内核

内核是Linux操作系统的核心部分，它管理系统资源，并提供硬件和软件应用程序之间的接口。Red Hat内核是自定义构建的，基于上游Linux内核，专注于稳定性以及与最新技术和硬件的兼容性。

在Red Hat发布新的内核版本之前，内核需要通过一系列严格的质量保证测试。

Red Hat内核以RPM格式打包，因此yum软件包管理器可以轻松升级和验证。

警告

Red Hat 不支持自定义内核。

更新内核

以下过程描述如何使用yum包管理器更新内核。

程序

  1. 要更新内核，请使用以下命令：

    # yum update kernel

    此命令将内核以及所有依赖项更新为最新的可用版本。

  2. 重新启动系统以使更改生效。

重要

从Red Hat Enterprise Linux 7升级到Red Hat Enterprise Linux 8时，Red Hat强烈建议您重新安装整个操作系统。虽然理论上可以更新所有必需的软件包，但很容易导致系统无法使用。

安装内核

以下过程描述如何使用yum软件包管理器更新或安装内核。

程序

  • 要安装特定的内核版本，请使用以下命令：

    # yum install kernel-{version}

注意

yum软件包管理器总是安装新内核而不是替换当前内核，这可能会使系统无法启动。
