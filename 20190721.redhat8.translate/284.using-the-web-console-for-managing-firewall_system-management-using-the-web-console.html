Chapter 9. 使用Web控制台管理防火墙

红帽企业Linux Web控制台使您可以运行，停止和配置系统防火墙。您可以选择服务并将其添加到已启用服务的列表中。

您还可以通过从启用的服务列表中删除此服务来禁用服务。

RHEL 8 Web控制台配置firewalld服务。

RHEL 8 Web控制台仅在默认区域中显示预定义的firewalld服务。

有关firewalld服务的详细信息，请参阅firewalld入门 。

使用Web控制台运行防火墙

本节介绍在Web控制台中运行RHEL 8系统防火墙的位置和方式。

注意

RHEL 8 Web控制台配置firewalld服务。

程序

  1. 登录RHEL 8 Web控制台。

    有关详细信息，请参阅登录Web控制台 。

  2. 打开“ 网络”部分。

  3. 在“ 防火墙”部分中，单击“ 开”以运行防火墙。

    如果未看到“ 防火墙”框，请使用管理权限登录Web控制台。

在此阶段，您的防火墙正在运行。

要配置防火墙规则，请参阅使用Web控制台在Web控制台中添加规则 。

使用Web控制台停止防火墙

本节介绍在Web控制台中停止RHEL 8系统防火墙的位置和方法。

注意

RHEL 8 Web控制台配置firewalld服务。

程序

  1. 登录RHEL 8 Web控制台。

    有关详细信息，请参阅登录Web控制台 。

  2. 打开“ 网络”部分。

  3. 在“ 防火墙”部分中，单击“ 关闭”以停止它。

    如果未看到“ 防火墙”框，请使用管理权限登录Web控制台。

在此阶段，防火墙已停止，并且无法保护您的系统。

使用Web控制台在防火墙上启用服务

以下文本描述了如何使用Web控制台在系统防火墙中添加服务。

RHEL 8 Web控制台在默认区域中显示预定义的firewalld服务。

重要

RHEL 8 Web控制台配置firewalld服务。但是，允许服务适用于所有接口。RHEL 8 Web控制台支持默认区域。它可能导致与低级防火墙规则冲突。

此外，Web控制台不允许Web控制台中未列出的通用firewalld规则。

先决条件

  • 必须启用防火墙。

    有关详细信息，请参阅在Web控制台中运行防火墙 。

程序

  1. 登录RHEL Web控制台。

    有关详细信息，请参阅登录Web控制台 。

  2. 单击网络 。

  3. 单击防火墙框标题。

    如果未看到“ 防火墙”框，请使用管理权限登录Web控制台。

  4. 在“ 防火墙”部分，单击“ 添加服务” 。

  5. 在“ 添加服务”对话框中，找到要在防火墙上启用的服务。

  6. 启用所需服务。

  7. 单击添加服务 。

此时，RHEL 8 Web控制台在允许的服务列表中显示该服务 。
