Chapter 9. Netconsole

netconsole内核模块允许通过网络将内核消息记录到另一台计算机。它允许在磁盘日志记录失败或无法使用串行控制台时进行内核调试。

配置netconsole

此过程描述了如何在Red Hat Enterprise Linux（RHEL）8中配置netconsole。

先决条件

已安装netconsole-service软件包。 

 ~]# yum install netconsole-service

程序

  1. 在设置SYSLOGADDR到的IP地址syslogd服务器/etc/sysconfig/netconsole文件匹配的IP地址syslogd服务器。例如：

    SYSLOGADDR=192.168.0.1

  2. 重新启动netconsole.service 。 

     ~]# systemctl restart netconsole.service

  3. 启用netconsole.service以在重新引导系统后运行。 

     ~]# systemctl enable netconsole.service

  4. 在netconsole文件（默认）或/var/log/messages指定的文件中查看来自客户端的rsyslog.conf消息。 

     ~]# cat /var/log/messages
