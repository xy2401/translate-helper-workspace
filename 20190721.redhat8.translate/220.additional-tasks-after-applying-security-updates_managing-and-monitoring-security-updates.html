Chapter 9. 应用安全更新后的其他任务

在Red Hat Enterprise Linux 8上安装安全更新后，您可能需要执行其他任务。本节介绍这些任务。

应用安全更新后显示需要重新启动的服务

在Red Hat Enterprise Linux 8上更新软件包时，可能需要手动重新启动使用更新库和可执行文件的某些进程。本节介绍如何识别这些过程。

先决条件

  • 已安装Red Hat Enterprise Linux 8更新。有关详细信息，请参阅第4章， 安装安全更新 。

程序

  1. 要列出在更新之前仍使用库或可执行文件的所有进程：

    $ sudo yum needs-restarting
1107 : /usr/sbin/rsyslogd -n
1199 : -bash
...

    yum needs-restarting命令仅列出进程，而不是服务。这意味着您无法使用systemctl实用程序重新启动列出的所有进程。例如，当拥有此进程的用户注销时，将终止输出中的bash进程。
