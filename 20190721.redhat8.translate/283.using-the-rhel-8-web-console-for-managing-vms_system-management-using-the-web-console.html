Chapter 9. 使用RHEL 8 Web控制台管理虚拟机

要在图形界面中管理虚拟机，可以使用RHEL 8 Web控制台中的“ Virtual Machines窗格。

以下部分描述了Web控制台的虚拟化管理功能，并提供了使用它们的说明。

使用RHEL 8 Web控制台进行虚拟机管理概述

RHEL 8 Web控制台是用于系统管理的基于Web的界面。通过安装Web控制台插件，Web控制台可用于管理Web控制台可以连接的服务器上的虚拟机（VM）。它提供了Web控制台可以连接的主机系统上的VM的图形视图，并允许轻松监视系统资源和调整配置。

使用RHEL 8 Web控制台进行VM管理，您可以执行以下操作：

  • 创建和删除VM
  • 在VM上安装操作系统
  • 运行和关闭VM
  • 查看有关VM的信息
  • 创建磁盘并将其附加到VM
  • 配置VM的虚拟CPU设置
  • 管理虚拟网络接口
  • 使用VM控制台与VM交互

注意

RHEL 8仍支持Virtual Machine Manager（ virt-manager ）应用程序，但已弃用。RHEL 8 Web控制台旨在成为后续版本的替代品。因此，建议您熟悉用于在GUI中管理虚拟化的Web控制台。但是，在RHEL 8中，某些功能只能从virt-manager或命令行访问。

设置RHEL 8 Web控制台以管理虚拟机

在使用RHEL 8 Web控制台管理VM之前，必须安装Web控制台虚拟机插件。

先决条件

  • 确保计算机上已安装Web控制台。 

    $ yum info cockpit
Installed Packages
Name         : cockpit
[...]

    如果未安装Web控制台，请参阅使用RHEL 8 Web控制台指南管理系统，以获取有关安装Web控制台的更多信息。

程序

  • 安装cockpit-machines插件。 

    # yum install cockpit-machines

    如果安装成功，则虚拟机将显示在Web控制台侧面菜单中。

使用RHEL 8 Web控制台创建虚拟机并安装客户机操作系统

以下部分提供有关如何使用RHEL 8 Web控制台在VM上创建虚拟机（VM）和安装操作系统的信息。

使用RHEL 8 Web控制台创建虚拟机

要在连接Web控制台的主机上创建VM，请按照以下说明操作。

先决条件

  • 要能够使用RHEL 8 Web控制台管理虚拟机，必须安装Web控制台虚拟机插件。

    如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

  • 在创建VM之前，请考虑分配给VM所需的系统资源量，例如磁盘空间，RAM或CPU。根据VM的预期任务和工作负载，建议值可能会有很大差异。

  • 本地可用的操作系统（OS）安装源，可以是以下之一：

    • 安装介质的ISO映像
    • 现有guest虚拟机安装的磁盘映像

程序

  1. 单击RHEL 8 Web控制台的“虚拟机”界面中的“ 创建VM” 。

    将出现“创建新虚拟机”对话框。

  2. 输入要创建的虚拟机的基本配置。

    • 连接 - 与虚拟机使用的主机的连接。
    • 名称 - 虚拟机的名称。
    • 安装源类型 - 安装源的类型 ：文件系统，URL
    • 安装源 - 指向安装源的路径或URL。
    • OS Vendor - 虚拟机操作系统的供应商。
    • 操作系统 - 虚拟机的操作系统。
    • 内存 - 配置虚拟机的内存量。
    • 存储大小 - 用于配置虚拟机的存储空间量。
    • 立即启动VM - 虚拟机是否在创建后立即启动。

  3. 单击“ 创建”

    虚拟机已创建。如果选中“ 立即启动VM”复选框，VM将立即启动并开始安装客户机操作系统。

您必须在第一次运行虚拟机时安装操作系统。

其他资源

  • 有关在虚拟机上安装操作系统的信息，请参阅“使用RHEL 8 Web控制台安装操作系统”一节 。

使用RHEL 8 Web控制台安装操作系统

加载虚拟机时，必须在虚拟机上安装操作系统。

先决条件

  • 在使用RHEL 8 Web控制台管理虚拟机之前，必须安装Web控制台虚拟机插件。

    如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

  • 用于安装操作系统的VM。

程序

  • 单击安装 。
    操作系统的安装例程在虚拟机控制台中运行。

注意

如果选中“创建新虚拟机”对话框中的“ 立即启动VM”复选框，则会在创建虚拟机时自动启动操作系统的安装例程。

注意

如果安装例程失败，则必须删除并重新创建虚拟机。

使用RHEL 8 Web控制台删除虚拟机

您可以从RHEL 8 Web控制台所连接的主机中删除虚拟机及其关联的存储文件。

先决条件

要能够使用RHEL 8 Web控制台管理虚拟机，必须安装Web控制台虚拟机插件。

如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

程序

  1. 在RHEL 8 Web控制台的“虚拟机”界面中，单击要删除的VM的名称。

    该行将展开以显示“概述”窗格，其中包含有关所选虚拟机的基本信息以及用于关闭和删除虚拟机的控件。

  2. 单击删除 。
    出现确认对话框。

  3. [可选]要删除与虚拟机关联的所有或部分存储文件，请选中要删除的存储文件旁边的复选框。
  4. 单击删除 。
    将删除虚拟机和任何选定的关联存储文件。

使用RHEL 8 Web控制台向VM发送不可屏蔽中断

发送不可屏蔽中断（NMI）可能会导致无响应的正在运行的VM响应或关闭。例如，您可以将Ctrl + Alt + Del NMI发送到无响应的VM。

先决条件

在使用RHEL 8 Web控制台管理VM之前，必须安装Web控制台虚拟机插件。

如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

程序

  1. 单击包含要向其发送NMI的虚拟机名称的行。
    该行将展开以显示“概述”窗格，其中包含有关所选虚拟机的基本信息以及用于关闭和删除虚拟机的控件。
  2. 单击“ 关机”按钮旁边的箭头，然后选择“ 发送不可屏蔽中断” 。
    NMI被发送到虚拟机。

其他资源

  • 有关重新启动虚拟机的信息，请参阅“使用RHEL 8 Web控制台重新启动虚拟机”一节 。
  • 有关关闭虚拟机的信息，请参阅“在RHEL 8 Web控制台中关闭虚拟机”一节 。

使用RHEL 8 Web控制台管理虚拟CPU

使用RHEL 8 Web控制台，您可以管理为Web控制台连接的虚拟机配置的虚拟CPU。您可以查看有关虚拟机的信息。您还可以为虚拟机配置虚拟CPU。

先决条件

要能够使用RHEL 8 Web控制台管理虚拟机，必须安装Web控制台虚拟机插件。

如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

程序

  1. 单击包含要查看的虚拟机名称的行，然后配置虚拟CPU参数。
    该行将展开以显示“概述”窗格，其中包含有关所选虚拟机的基本信息，包括虚拟CPU的数量以及用于关闭和删除虚拟机的控件。

  2. 单击“概述”窗格中的vCPU数。
    将出现“vCPU详细信息”对话框。

    注意

    只有在更改虚拟CPU设置后，才会显示“vCPU详细信息”对话框中的警告。

  3. 配置所选虚拟机的虚拟CPU。

    • vCPU Count - 输入虚拟机的虚拟CPU数。

      注意

      vCPU计数不能大于vCPU最大值。

    • vCPU Maximum - 输入可为虚拟机配置的最大虚拟CPU数。
    • 套接字 - 选择要向虚拟机公开的套接字数。
    • 每个插槽的内核 - 选择要向虚拟机公开的每个插槽的内核数。
    • 每个核心的线程数 - 选择要向虚拟机公开的每个核心的线程数。

  4. 单击“ 应用”

    配置虚拟机的虚拟CPU。

    注意

    虚拟机运行时，虚拟CPU设置的更改仅在虚拟机停止并重新启动后才会生效。

使用RHEL 8 Web控制台将新磁盘添加到虚拟机

您可以通过创建新磁盘（存储池）并使用RHEL 8 Web控制台将其附加到虚拟机来将新磁盘添加到虚拟机。

注意

使用RHEL 8 Web控制台为虚拟机创建新磁盘时，只能使用目录类型存储池。

先决条件

要能够使用RHEL 8 Web控制台管理虚拟机，必须安装Web控制台虚拟机插件。

如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

程序

  1. 单击包含要为其创建的虚拟机名称的行，然后附加新磁盘。
    该行将展开以显示“概述”窗格，其中包含有关所选虚拟机的基本信息以及用于关闭和删除虚拟机的控件。

  2. 单击磁盘 。
    将显示“磁盘”窗格，其中包含有关为虚拟机配置的磁盘的信息。


  3. 单击添加磁盘 。

    出现“添加磁盘”对话框。

  4. 确保选中“ 创建新选项”按钮。

  5. 配置新磁盘。

    • 池 - 选择将从中创建虚拟磁盘的存储池。
    • 目标 - 选择将要创建的虚拟磁盘的目标。
    • 名称 - 输入将要创建的虚拟磁盘的名称。
    • 大小 - 输入大小并选择将要创建的虚拟磁盘的单位（MiB或GiB）。
    • 格式 - 选择将要创建的虚拟磁盘的格式。支持的类型：qcow2，raw

    • 持久性 - 虚拟磁盘是否具有持久性。如果选中，则虚拟磁盘是持久的。如果未选中，则虚拟磁盘不是持久的。

      注意

      瞬态磁盘只能添加到正在运行的VM。

  6. 单击添加 。

    创建虚拟磁盘并将其连接到虚拟机。

使用RHEL 8 Web控制台创建存储池

您可以使用RHEL 8 Web控制台创建存储池。

先决条件

要能够使用RHEL 8 Web控制台管理虚拟机，必须安装Web控制台虚拟机插件。

如果未安装Web控制台插件，请参阅“设置RHEL 8 Web控制台以管理虚拟机”一节，以获取有关安装Web控制台虚拟机插件的信息。

程序

  1. 单击“虚拟机”选项卡顶部的“ 存储池” 。将显示“存储池”窗口，其中显示已配置存储池的列表。

  2. 单击“ 创建存储池” 。将出现“创建存储池”对话框。

  3. 在“创建存储池”对话框中输入以下信息：

    • 连接 - 与存储池使用的主机的连接。
    • 名称 - 存储池的名称。
    • 类型 - 存储池的类型：文件系统目录，网络文件系统
    • 目标路径 - 主机文件系统上的存储池路径。
    • 启动 - 主机启动时是否启动存储池。
  4. 单击“ 创建”创建存储池，关闭“创建存储池”对话框，新存储池将显示在存储池列表中。
