Chapter 9. 使用nmtui配置IP网络

以下部分提供了如何使用NetworkManager's工具nmtui配置网络接口。

nmtui入门

nmtui是基于诅咒简单的文本的用户界面（TUI），用于NetworkManager 。

此过程描述如何启动文本用户界面工具nmtui 。

先决条件

  • nmtui工具用于终端窗口。它包含在NetworkManager-tui包中，但默认情况下不与NetworkManager一起安装。要安装NetworkManager-tui ：

    ~]# yum install NetworkManager-tui
  • 要验证NetworkManager是否正在运行，请参阅“检查NetworkManager的状态”一节

程序

  1. 启动nmtui工具：

    ~]$ nmtui

    出现文本用户界面。

    图5.1。NetworkManager文本用户界面的开始菜单

  2. 要导航，请使用箭头键或按Tab键前进，然后按Shift + Tab键退回选项。按Enter选择一个选项。空格键切换复选框的状态。

使用nmtui编辑连接

先决条件

  • 名为“nmtui入门”的部分

要使用nmtui Edit a connection ，请在NetworkManager TUI菜单中选择Edit a connection选项，然后按Enter键 。

将更改应用于nmtui的已修改连接

要在已经激活的已修改连接之后应用更改，需要重新激活连接。在这种情况下，请按照以下步骤操作：

先决条件

  • 名为“nmtui入门”的部分

程序

  1. 选择“ Activate a connection菜单条目。

    图5.2。使用nmtui激活连接

  2. 选择已修改的连接。在右侧，单击“ Deactivate按钮。

    图5.3。使用nmtui取消激活已修改的连接

  3. 再次选择连接，然后单击“ Activate按钮。

    图5.4。使用nmtui重新激活已修改的连接

以下命令也可用：

~]$ nmtui edit connection-name

如果未提供连接名称，则会显示选择菜单。如果提供并正确识别连接名称，则会显示相关的Edit connection屏幕。 

~]$ nmtui connect connection-name

如果未提供连接名称，则会显示选择菜单。如果提供并正确识别连接名称，则激活相关连接。任何无效命令都会打印一条用法消息

请注意， nmtui不支持所有类型的连接。特别是，您无法使用WPA Enterprise编辑VPN，无线网络连接或使用802.1X以太网连接。
