5.3.3. The Test Case Browser

The Test Case Browser is by default in the lower left-hand corner of the Specification Perspective. In this browser, you can create Test Cases, which are the ”building blocks” for your tests. They are used to group Test Steps and other Test Cases.

More information on working with browsers is available in the Tasks (Section 3.9, “Working with browsers: renaming, deleting, using IDs, multiple browsers”). To make working with larger Projects easier, you can also have the Test Case Browser open more than once (Section 3.9.6, “Opening the Test Case Browser multiple times”).

You can filter in the Test Case Browser using the field at the top. Use star * as a wild card.
