Running and monitoring batch jobs on SLURM ALPS systems

In order to monitor target systems, which run a combination of SLURM and ALPS as batch system, the target system configuration SLURM-ALPS-Generic-Batch has to be chosen. This configuration is derived from the SLURM-Generic-Batch configuration. It only includes additional status information provided by ALPS into the monitoring results of the System Monitoring Perspective. Note that the attempt of monitoring a SLURM ALPS system by using the monitoring type SLURM configured in SLURM-Generic-Batch will fail.

Running batch jobs with this target system configuration is identical to running jobs on a plain SLURM system.
