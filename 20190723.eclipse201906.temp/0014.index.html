What's new

The latest features introduced with Acceleo

Acceleo 3.3

  • Stabilisation
  • Improved maven support
  • Improved support for meta-models in the workspace
  • New documentation
  • New operations in the non-standard library
  • New UML to Java example for Juno

Acceleo 3.2

  • Interpreter view
  • New compiler (more stable and faster)
  • JRE selection during the creation on a new project
  • Improved performances of generators with dynamic modules

Acceleo 3.1

  • Language improvements
  • Real time detection of bad behaviors with warnings
  • New wizard for projects and modules creation
  • Support for the documentation
  • New outline view
  • Basic support for meta-models in the workspace
  • New refactoring options
  • Basic maven support
  • New serialization options with support for binary resources

Acceleo 3.0

  • First major release
  • Improved editor
  • Documentation
