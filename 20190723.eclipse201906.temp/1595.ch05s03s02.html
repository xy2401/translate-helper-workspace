5.3.2. The Test Suite Browser

The Test Suite Browser is by default in the upper left-hand corner of the Specification Perspective. In this browser, you can create Test Suites and Test Jobs for executing tests.

More information on working with browsers is available in the Tasks (Section 3.9, “Working with browsers: renaming, deleting, using IDs, multiple browsers”).

You can filter in the Test Suite Browser using the field at the top. Use star * as a wild card.
