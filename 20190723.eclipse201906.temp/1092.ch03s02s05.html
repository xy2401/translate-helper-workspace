3.2.5. Concrete toolkit menu bar component

A menu bar component is the component typically found at the top of an application window.

It typically contains menus such as "File", "Edit", "Help", etc..

Because the forward slash (/) is a special symbol for menus, if you want to use a slash as part of your parameter value, you have to mask it.

Menu bars do not need to be mapped in the Object Mapping Mode as they are automatically found during test execution.

Actions on menus are not supported in the HTML toolkit.
