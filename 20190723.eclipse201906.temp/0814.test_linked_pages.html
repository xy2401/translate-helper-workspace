Testing Linked Pages

After you link the pages of a Web application and define link properties, you can test the linked Web pages. Before you can test linked pages, you must deploy the Web application project to a Web server.

To test linked Web pages:

  1. Deploy the Web application project.

    The server starts and the embedded Web browser opens.

  2. Navigate through the Web page sequence to ensure it matches your intended navigation rule.
