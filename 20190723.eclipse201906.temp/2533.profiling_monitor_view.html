Profiling Monitor View

The Profiling Monitor view displays a list of previously run Profiling sessions.

Expanding the list under a Profiling session allows you to select a Profiling view with the related data to display.

Toolbar Commands

IconNameDescription
Delete SessionDeletes a Profiling session from the list. This will be enabled if a profiling session is selected.
Sort Profile SessionsClick the arrow next to the Profile Session to sort the Profile Session list by date or time.
