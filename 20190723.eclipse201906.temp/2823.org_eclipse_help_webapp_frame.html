Webapp Frame

Identifier:
org.eclipse.help.webapp.frame

Since:
3.5

Description:
Allows additional frames to be added to the help web application.

Configuration Markup:

<!ELEMENT extension (frame)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT frame EMPTY>

<!ATTLIST frame

class CDATA #REQUIRED>

A frame in the help web application


  • class - A class which has methods which return the path of the jsp file for this frame as well as other information abot the frame.
