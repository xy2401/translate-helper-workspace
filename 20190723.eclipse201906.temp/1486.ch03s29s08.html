3.29.8. Object mapping preferences

Figure 3.34. Object Mapping Preference Dialog

You can access the object mapping preferences from Test - Object Mapping in the preferences dialog.

  1. Select whether you want the ITE to display how many nodes are contained in each category in the Object Mapping Editor.

  2. Choose which keystrokes or mouse buttons you want to use to collect a component from a Java AUT in the Object Mapping Mode.

  3. Choose which keystrokes or mouse buttons you want to use to collect a component and all its supported parent components from the AUT which contain it. This option is currently only available for Object Mapping in the JavaFX toolkit.

If your AUT does not accept keystrokes, set the object collection preference to a mouse button combined with a modifier key.
