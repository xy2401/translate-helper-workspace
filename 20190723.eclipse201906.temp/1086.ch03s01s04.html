3.1.4. Abstract toolkit text input component

This is an abstract component.

It is implemented by all components whose properties or values can be altered.

  • Check Existence (General Components)

  • Click

  • Drag and Drop (General Components)

  • Enter Text (General Components)

  • Select in Context Menus

  • Show text

  • Store Property/Value (General Components)

  • Wait for Component (General)
