2.8.4. Extensions with the embedded Agent

The embedded Agent can also be customized. The extensions are written in the same way as described in Section 3.3, “Remote control extension” and Section 3.4, “Addressing custom controls via the client API”.

To deploy the extension for the embedded AutAgent for the API you must create a "ext" directory into the directory where the used org.eclipse.jubula.autagent.api.jar is located. After that you can put your rc extension into the "ext" directory and the extension will be used.
