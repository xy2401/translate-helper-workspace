Spinner

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Spinner background color

  • background-image

    Background gradient for spinner.

  • border

    The border that is drawn around spinners.

  • border-bottom

    The bottom side border for spinners.

  • border-left

    The left side border for spinners.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for spinners.

  • border-top

    The top side border for spinners.

  • box-shadow

    A shadow effect for spinner text field.

  • color

    Spinner foreground color

  • font

    Text font

  • padding

    Padding for spinner.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Style constant for bordered behavior.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled Spinner controls.

  • :focused

    Applies to focused Spinners

Spinner-UpButton

The spinner up button

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color of spinner up button.

  • background-image

    Background image or gradient for spinner up button.

  • border

    The border that is drawn around up button.

  • border-bottom

    The bottom side border for up button.

  • border-left

    The left side border for up button.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for up button.

  • border-top

    The top side border for up button.

  • cursor

    Spinner up button cursor.

  • width

    Width of the spinner up button.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [RIGHT_TO_LEFT]

    Indicates that the spinner is oriented right to left.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled spinner up button.

  • :hover

    Applies while the user hovers over the button.

  • :pressed

    Applies while a button is pressed.
