3.30.7. Using the search result view

The Search Result View shows any items found during a search – either via the search dialog or the ”Show where used” action.

From the Search Result View, you can:

  • Double-click on an item to open its editor.

  • Re-run the search using the button in the top right-hand corner of the view.

  • Perform various actions on your search results (Section 3.30.7.1, “Using search results to make wide-reaching changes to your Project”)

3.30.7.1. Using search results to make wide-reaching changes to your Project

Once you have performed a search, you can perform specific actions on the results:

  • You can replace a Test Case at multiple places where it has been reused with another Test Case (Section 3.12.8, “Replacing a specific Test Case at places where it has been reused”).

  • You can change the column of a central test data set used by any Test Cases that use this central test data set (Section 3.14.9.7, “Changing the column used in a central test data set for multiple Test Cases”).
