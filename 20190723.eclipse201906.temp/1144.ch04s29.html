4.29. Expected exit code

Parameter Name Data Type Possible Values Default
EXPECTED_EXIT_CODE Integer   0
Table 4.30. EXPECTED_EXIT_CODE

  • Scripts generally have an exit code (for example, 0 usually means that the script was successful).

  • Use this parameter to define the expected exit code.
