Toolbar Button

Identifier:
org.eclipse.help.webapp.toolbarButton

Since:
3.5

Description:
Allows additional buttons to be added to the toolbars of the help web application.

Configuration Markup:

<!ELEMENT extension (button)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT button EMPTY>

<!ATTLIST button

class CDATA #REQUIRED>

  • class - A class which extends AbstractButton and which contains methods which return the image URL, action to be performed and othe attributes of the button.
