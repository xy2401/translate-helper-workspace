3.10.3. Navigating between Editors

Double clicking a Test Case Editor root node helps in navigating to a Test Suite or Test Case referencing the edited Test Case:

  1. If there is a single Editor open which references the Test Case, that Editor is activated.

  2. If there are more than one such Editors open, then a dialog pops where you can choose which one to activate.

  3. If there are no Editors referencing the Test Case, then the list of Test Suites and Test Cases referencing the given Test Case is shown, and you can choose one to open its Editor.

Jubula maintains a list of all previously visited Editors, and it is possible to navigate between them using the Previous / Next Editor toolbar items. The default shortkeys for these are Ctrl + Left and Ctrl + Right, respectively. There are also two similar toolbar items, Previous / Next Edited Editors which navigate between those Editors which have been edited.
