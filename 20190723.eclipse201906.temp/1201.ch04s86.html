4.86. URL

Parameter Name Data Type Possible Values Default
URL String   none
Table 4.87. URL

  • Enter the URL you wish to open, e.g. http://testing.bredex.de.

  • We recommend using single quotes around the URL so that you do not have to escape special characters: 'http://testing.bredex.de'.
