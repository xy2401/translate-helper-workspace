Uninstalling the Product

Uninstall using the Update Manager.

If you installed using the Update Manager, you need to use the Update Manager to uninstall.

For Eclipse 3.4

  1. In Eclipse, select Help > Software Updates... to open the Update Manager
  2. In the Installed Software tab, select all the product components to uninstall then click the Uninstall... button.
  3. Follow the rest of the wizard.
  4. Restart Eclipse when prompted.

For Eclipse 3.5 or 3.6

  1. In Eclipse, select Help > Install New Software... to open the Update Manager.
  2. Click the "What is already installed?" link.
  3. In the Installed Software tab, select all the product components to uninstall then click the Uninstall... button.
  4. Follow the rest of the wizard.
  5. Restart Eclipse when prompted.
