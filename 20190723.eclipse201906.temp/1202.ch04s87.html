Parameter Name Data Type Possible Values Default

Table 4.88. USE_NEW_ENVIRONMENT

  • Use this parameter to either create a whole new environment for the external command or use the environment of the AUT/ITE, i.e. that environment variables of the AUT/ITE can be used or will be ignored
