Sash

Sash control

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color for sash.

  • background-image

    Background image for sash.

  • border

    The border that is drawn around sash.

  • border-bottom

    The bottom side border for sash.

  • border-left

    The left side border for sash.

  • border-right

    The right side border for sash.

  • border-top

    The top side border for sash.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Indicates that the sash should have a border.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :hover

    Applies while the user hovers over a sash.

Sash-Handle

The place in the middle of the sash.

Available CSS Properties

Properties that can be defined for this element:

  • background-image

    Background image for sash handle

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :horizontal

    Applies while a sash is in horizontal state. This state responds to sash SWT.HORIZONTAL style flag.

  • :vertical

    Applies while a sash is in vertical state. This state responds to sash SWT.VERTICAL style flag.
