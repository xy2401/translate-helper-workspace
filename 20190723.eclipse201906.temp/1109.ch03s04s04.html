3.4.4. HTML toolkit locator

Locator

You can address components in HTML tests using the according locator. There are eight locators strategies included:

  • Identifier

  • ID

  • Name

  • Link

  • DOM

  • XPath

  • CSS

  • UI-Element

Available Actions

  • Check CSS Path Count

  • Check Text by Locator

  • Check XPath Count

  • Click Locator

  • Wait for component (based on locator)
