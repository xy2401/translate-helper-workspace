DropDown

DropDown widget. Font, color, backgroundColor and cursor are inherited from the parent widget.

Available CSS Properties

Properties that can be defined for this element:

  • border

    Border for DropDown widget.

  • border-bottom

    The bottom side border for DropDown widget.

  • border-left

    The left side border for DropDown widget.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for DropDown widget.

  • border-top

    The top side border for DropDown widget.

  • box-shadow

    The shadow behind the DropDown.

Applicable SWT Styles

none

Applicable States

none

DropDown-Item

Items of DropDown.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Item background color

  • background-image

    Item background image/gradient

  • color

    Item text color

  • padding

    Defines the padding (i.e. the inner distance between border and content) for a DropDown item.

  • text-decoration

    Decoration for item text.

  • text-shadow

    Shadow for item text.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :hover

    Applies while the user hovers over the DropDown item.

  • :selected

    Applies when DropDown item is selected.
