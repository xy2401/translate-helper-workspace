3.30.6. Searching for tasks in ALM repositories

On the fourth tab of the search dialog, you can search for tasks in any repositories you have referenced in the Project.

You can use this search if you have joined your Project to an external respository.

Click ”Search” to start the search. The results are shown in the Search Result View (Section 3.30.7, “Using the search result view”).
