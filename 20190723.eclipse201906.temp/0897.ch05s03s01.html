5.3.1. Opening the wizard

  1. Select from the main menu File > New > Project....

    You can alternatively select New > Project... from the context menu of the Project Explorer or the Package Explorer.

  2. Expand the Jubula folder in the New Project window that appears.

  3. Select Jubula Extension Project and click the "Next" button.
