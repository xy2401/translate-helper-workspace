PHP Perspectives and Views

PDT incorporates a number of Perspectives and Views for managing all aspects of your PHP code, files, project and application creation.

The following PHP perspectives are used for developing PHP:

  • PHP Perspective
  • PHP Debug Perspective
  • PHP Profile Perspective

See the Workbench User Guide for more on Perspectives and Views .

Note:

Additional user guides can be accessed from inside PDT by going to Help | Help Contents, or from the Eclipse Online Documentation site (http://help.eclipse.org/luna/index.jsp).
