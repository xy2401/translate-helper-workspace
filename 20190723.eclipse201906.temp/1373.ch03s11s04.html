3.11.4. Adding comments to categories

You can add comments to categories in the Test Suite Browser and Test Case Browser to add information on the purpose of the category:

  1. In the browser, select the category you want to add a comment to, or edit a comment on.

  2. From the context-menu, select:

    Edit comment

  3. In the dialog that appears, enter the comment for this category and press ”OK”.

  4. You will be able to see the comment in the Properties View for this category, and as a tooltip (as long as the category has no errors or warnings).

You can also add comments to Test Cases, and Test Suites (Section 3.10.7, “Adding comments to items in editors”) as well as to the Project as a whole (Section 3.6.3, “Editing the Project and AUT properties”).
