在Java模块中管理JRuby Facet

基本

IntelliJ IDEA可以将JRuby facet附加到Java模块。这可能是使用Ruby代码，具有解析和代码完成所必需的。

假设您已经创建了一个包含Java模块的项目，并且您想要将一个facet附加到它上面。您必须在计算机上安装JRuby 。

顺便说一下，如果你创建一个Ruby类型的模块，你可以附加任何可用的Ruby SDK。

添加JRuby方面

安装JRuby（例如， rvm install jruby ）后，请按照下列步骤操作：

  1. 打开“ 项目结构对话框”对话框。

  2. 在此对话框中，选择“ 构面”页面。

  3. 点击 ：

  4. 从出现的可能方面的弹出菜单中，选择JRuby facet：

  5. 如果先前已安装JRuby，则IntelliJ IDEA会识别它，并建议选择应添加构面的Java模块：

    选择所需的模块。

删除JRuby方面

  1. 打开“ 项目结构对话框”对话框。

  2. 在此对话框中，选择“ 构面”页面。
