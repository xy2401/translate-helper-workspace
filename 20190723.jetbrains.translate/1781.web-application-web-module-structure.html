填充Web模块

此功能仅在Ultimate版本中受支持。

在模块中启用Web开发支持后，IntelliJ IDEA将按如下方式设置基本模块结构：

  • 使用index.jsp存根文件创建web文件夹。默认情况下， web文件夹将在部署后成为应用程序的根目录， index.jsp文件将成为其主页。

  • 创建一个WEB-INFO子文件夹，其中包含配置了Action servlet的web.xml描述符。

  • 如果在创建项目时指定了src文件夹，则创建该文件夹。

要填充Web模块，请执行以下常规步骤：

  • 在src文件夹下，创建实现应用程序功能的Java类文件。

  • 配置表示用户界面的静态Web内容资源 。

  • 配置应用程序元素 。

  • 指定程序集描述符引用 。
