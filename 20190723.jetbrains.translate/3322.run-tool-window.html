运行工具窗口

查看|工具窗口|跑
Alt + 4

“ 运行”工具窗口显示应用程序生成的输出。如果您正在运行多个应用程序，则每个应用程序都会显示在应用了运行/调试配置后命名的选项卡中。

每个选项卡的外观取决于正在运行的应用程序的类型，并且可以包括其他工具箱和窗格。

“ 运行”工具窗口的主工具栏允许您重新运行，停止，暂停或终止应用程序。下表包含大多数应用程序常用按钮的说明。

工具提示和快捷方式

重新运行
按Ctrl + F5

单击此按钮可停止当前应用程序并再次运行它。
当应用程序停止时（ ），这个按钮切换到 。

重新运行
按Ctrl + F5

单击此按钮可重新运行当前应用程序。
应用程序停止时出现此按钮（ ）。当应用程序运行时，此按钮切换为 。

停止
按Ctrl + F2

单击此按钮可通过标准shutdown脚本在外部终止当前进程。

单击该按钮一次调用soft kill，允许应用程序捕获SIGINT事件并执行正常终止（在Windows上，模拟Ctrl + C事件）。单击按钮一次后，将替换为表示后续点击将导致强制终止应用程序，例如在Unix上发送SIGKILL 。

恢复布局

单击此按钮可以放弃对当前布局的更改并返回默认状态。

Pin标签

单击此按钮可固定或取消固定当前选项卡。您可能需要固定一个标签，以防止在此窗口中达到最大标签数时自动关闭标签。

关
按Ctrl + Shift + F4

单击此按钮可关闭“ 运行”工具窗口的选定选项卡，并终止当前进程。

与剪贴板比较

打开“ 剪贴板与编辑器”对话框，该对话框允许您查看编辑器中的选择与当前剪贴板内容之间的差异。此对话框是一个常规比较工具 ，使您可以将插入符号行复制到剪贴板，查找文本，在差异之间导航以及管理空白区域。

复制网址

选择此命令可将当前URL复制到系统剪贴板。此命令仅在URL上显示，如果它包含在应用程序的输出中。

创建要点

选择此命令以打开“ 创建要点”对话框。

控制台工具栏

项目

工具提示和快捷方式

描述

向上/向下堆栈跟踪
N / A N / A.

单击此按钮可在堆栈跟踪中向上或向下导航，并使光标跳转到源代码中的相应位置。

使用软包装

单击此按钮可切换输出的软包装模式。

滚动到最后

单击此按钮可导航到堆栈跟踪的底部，并使光标跳转到源代码中的相应位置。

打印

单击此按钮可将控制台文本发送到默认打印机。

全部清除

单击此按钮可从控制台中删除所有文本。此功能也可在控制台的上下文菜单中使用。

Karma Server选项卡

仅当您使用Karma测试运行器运行JavaScript单元测试时，才会显示该选项卡。使用选项卡可以查看和分析来自服务器的信息。

有关详细信息，请参阅Karma 。
