使用此对话框可以定义有助于跟踪源代码中的TODO项目的模式。确保源代码中的TODO项插入到对支持的文件类型有效的注释中。

项目

描述

图案

在此文本框中，键入描述所需TODO模式的正则表达式 。

图标

从此下拉列表中，选择图案的图标。

区分大小写

选中此复选框可使模式区分大小写。

字体类型

选中相应的复选框，使图标文本以粗体或斜体显示。

前景

选中此复选框以启用调色板并选择前景色。

背景

选中此复选框以启用调色板并选择背景颜色。

错误条纹标记

选中此复选框以启用调色板并选择错误条带颜色。

效果

选中此复选框以启用效果（下划线，删除线等）并从调色板中选择它们的颜色。
