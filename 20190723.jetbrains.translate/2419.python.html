要在PyCharm中使用Python ，请至少安装一个Python解释器。

支持的版本

  • Python 2： 2.7版

  • Python 3：从版本3.5到版本3.7

PyCharm集成了Python 3.7的以下功能：

  • PEP-561 - 分发和包装类型信息

  • PEP-544 - 协议：结构子类型（静态鸭子类型）

  • PEP-557 - 数据类

  • PEP-560 - 对键入模块和泛型类型的核心支持

  • PEP-562 - 模块__getattr__和__dir__

  • PEP-563 - 延迟评注注释

PyCharm支持Python 3.5和（自2016.3）Python 3.6，支持以下内容：

  • PEP-484 - 类型提示

  • PEP-448 - 附加拆包概括
  • PEP-492 - 具有异步和等待语法的协同程序
  • PEP-526 - 变量注释的语法

  • PEP-498 - 文字字符串插值

  • PEP-515 - 数字文字中的下划线

  • PEP-525 - 异步发电机

  • PEP-530 - 异步理解
