IntelliJ IDEA允许您使用内置的JSLint工具验证JavaScript代码。请注意，JSLint不完全支持最新的ECMAScript功能。

在IntelliJ IDEA中启用JSLint并配置其行为

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 语言和框架”下的“ JavaScript ”，然后在“代码质量工具”下选择“ JSLint ” 。在打开的JSLint页面上，选中“ 启用”复选框。之后，页面上的其他控件变为可用。

  2. 在“ 容错”区域中，指定您不希望JSLint报告的问题类型。默认情况下，该工具会报告检测到的所有类型的问题。要跳过某种类型的问题，请选中此类型旁边的复选框。

  3. 在“ 假设”区域中，指定要检查代码的目标环境。

  4. 指定的问题报告（最大数量maxerr ），最大行长度（ maxlen通过名称或通过对象键（），以及预定义的全局变量predef ）。

  5. 在Validate also区域中选择JSON复选框以验证json文件。
