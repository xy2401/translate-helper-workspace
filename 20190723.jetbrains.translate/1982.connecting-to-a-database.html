PyCharm 2019.1帮助

连接到数据库

要向数据库发出查询，必须连接到数据源。数据源是数据的位置（可以是服务器，CSV或DDL文件）。数据源连接包括依赖于数据源类型的名称和连接设置。在PyCharm中，您可以在“ 数据源和驱动程序”对话框（“ 文件”|“数据源” ）中管理这些连接。

亚马逊Redshift

  1. 在Redshift仪表板中，创建Redshift集群。

  2. 在Redshift群集的设置中，复制JDBC URL 。

  3. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  4. 在“ 数据源和驱动程序”对话框中，单击“ 添加”图标（ ）并选择Amazon Redshift 。

  5. 在数据源设置区域的底部，单击“ 下载缺少的驱动程序文件”链接。或者，您可以为数据源指定用户驱动程序。有关用户驱动程序的详细信息，请参阅向现有连接添加用户驱动程序 。

  6. 从“ 连接类型”列表中， 仅选择“ URL” 。

  7. 将Redshift群集设置中的JDBC URL粘贴到PyCharm中的URL字段。

  8. 在“ 用户”和“ 密码”字段中，指定Redshift凭据。

  9. 要确保与数据源的连接成功，请单击“ 测试连接” 。如果您以前忘记下载JDBC驱动程序，则可以在连接测试期间单击通知对话框中的“ 下载驱动程序文件”按钮。

有关Amazon Redshift群集的更多信息，请阅读Amazon Redshift入门。

Apache Derby

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 在“ 数据源和驱动程序”对话框中，单击“ 添加”图标（ ）并选择Apache Derby 。

  3. 从“ 驱动程序”列表中，选择要使用的JDBC驱动程序的类型。对于远程连接，请选择Apache Derby（远程） 。对于嵌入式数据库，请选择Apache Derby（Embedded） 。

  4. 在数据源设置区域的底部，单击“ 下载缺少的驱动程序文件”链接。或者，您可以为数据源指定用户驱动程序。有关用户驱动程序的详细信息，请参阅向现有连接添加用户驱动程序 。

  5. 指定数据库连接细节或者，将JDBC URL粘贴到URL字段中。

    要设置空密码，请右键单击“ 密码”字段，然后选择“ 设置为空” 。

  6. 要确保与数据源的连接成功，请单击“ 测试连接” 。如果您以前忘记下载JDBC驱动程序，则可以在连接测试期间单击通知对话框中的“ 下载驱动程序文件”按钮。

Azure SQL数据库

  1. 在Microsoft Azure帐户中，创建SQL数据源。

  2. 在PyCharm中，导航到File |数据源 （ Ctrl + Shift + Alt + S ）。

  3. 在“ 数据源和驱动程序”对话框中，单击“ 添加”图标（ ）并选择Azure SQL数据库 。

  4. 在数据源设置区域的底部，单击“ 下载缺少的驱动程序文件”链接。

  5. 指定数据库连接细节或者，将JDBC URL粘贴到URL字段中。

    要设置空密码，请右键单击“ 密码”字段，然后选择“ 设置为空” 。

    要使用Azure Active Directory身份验证 ，请选择“ 使用Windows域身份验证” 。要使用SQL身份验证，请清除该复选框，并在“ 用户”和“ 密码”字段中指定Microsoft Azure凭据。

  6. 要确保与数据源的连接成功，请单击“ 测试连接” 。如果您以前忘记下载JDBC驱动程序，则可以在连接测试期间单击通知对话框中的“ 下载驱动程序文件”按钮。

“ 使用Windows域身份验证”复选框仅适用于Windows操作系统。

Microsoft Azure支持PostgreSQL，MySQL，MariaDB和其他数据库管理系统（DBMS）。如果要连接到这些数据源，请创建与这些DMBS对应的数据源连接。在PyCharm中，这些连接已包含必要的JDBC驱动程序。因此，如果要连接到PostgreSQL的Azure数据库 ，请创建PostgreSQL连接 。

有关Azure SQL数据库的详细信息，请阅读Azure SQL数据库文档。

    • SID ：使用Oracle实例的唯一名称（在工作站上运行的进程）。

    • 服务名称 ：使用Oracle实例（或许多实例）的别名。

    • TNS ：使用tnsnames.ora配置文件。在TNSADMIN字段中，指定tnsnames.ora文件的路径。在TNS名称中 ，如果您有多个服务，请指定要使用的服务名称（请参阅tnsnames.ora的net_service_name值）。

  5. 指定数据库连接细节或者，将JDBC URL粘贴到URL字段中。

    要设置空密码，请右键单击“ 密码”字段，然后选择“ 设置为空” 。

  6. 从“ 驱动程序”列表中，指定驱动程序类型。有关驱动程序类型的更多信息，请参阅Oracle JDBC FAQ 。

  7. 在数据源设置区域的底部，单击“ 下载缺少的驱动程序文件”链接。或者，您可以为数据源指定用户驱动程序。有关用户驱动程序的详细信息，请参阅向现有连接添加用户驱动程序 。

有关tnsnames.ora详细信息，请参阅tnsnames.ora文件中的本地命名参数。
