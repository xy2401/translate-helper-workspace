结构工具窗口，文件结构弹出窗口

结构工具窗口

此工具窗口显示当前在编辑器中打开并具有焦点的文件的结构，或在“项目”工具窗口中选择的文件。

对于图表，此工具窗口显示图表预览。

查看|工具窗口|结构体
Alt + 7

文件结构弹出窗口

此弹出窗口显示当前在编辑器中打开并具有焦点的文件结构。

导航|文件结构
按Ctrl + F12

两种视图都有助于快速浏览文件的结构 。

本节介绍工具窗口标题栏上的按钮以及标题栏上下文菜单中的选项。打开和关闭这些选项可隐藏或显示某些类型的元素，并配置它们的显示方式。

标题栏上的按钮对于所有语言上下文都是通用的。上下文菜单上的选项集取决于上下文。

  • 标题栏
  • Java的
  • HTML，XML
  • JavaScript，TypeScript，CoffeeScript，ActionScript
  • 属性
  • PHP

请参阅“ 结构”工具窗口中文件树视图中的成员图标的“ 图标参考 ”部分。

标题栏

标题栏上的按钮对于所有语言上下文都是通用的。

项目

提示

描述

展开全部Ctrl + NumPad Plus

单击此按钮可展开工具窗口中的所有节点。

全部折叠Ctrl + NumPad -

单击此按钮可使工具窗口中的所有节点折叠。

单击此按钮可打开上下文菜单，并通过打开或关闭菜单项来配置工具窗口的外观，其查看模式以及呈现当前文件结构的方式。

Shift +逃生使用此图标或快捷方式隐藏工具窗口。

与Alt键组合使用时，单击此图标会隐藏连接到同一工具窗口栏的所有工具窗口 。

Java的

图标

提示

描述

按可见性排序

单击此按钮可按以下顺序按项目可见性对项目进行排序：
public - protected - package local - private。

按字母排序

单击此按钮可以按字母顺序对类中的元素进行排序。

当两个 （按可见性排序）和按（按字母顺序排列）按钮，视图中的项目将根据其可见性级别进行分组。在每个可见性级别组内，项目按字母顺序排序。释放两个按钮时，项目将按照它们在代码中显示的顺序显示。

通过定义类型的组方法

单击此按钮可以覆盖/实现特定类/接口的方法的所有方法，这些方法分组在与此类/接口对应的节点下。

显示属性

单击此按钮可在树视图中显示getter，setter和fields。

显示继承

单击此按钮可显示当前类继承并可从中访问的所有方法和字段。继承的成员显示为灰色，以区别于当前类中定义的成员。

显示匿名

单击此按钮可在树视图中显示内部匿名类。

显示字段

单击此按钮可以在树中显示所有字段（属性）。

显示非公开

单击此按钮可显示所有非公共类成员。
关闭选项以隐藏所有非公开成员。

全部折叠Ctrl + NumPad -

单击此按钮可使工具窗口中的所有节点折叠。

展开全部Ctrl + NumPad Plus

单击此按钮可展开工具窗口中的所有节点。

自动滚动到源

单击此按钮可在焦点切换到编辑器时启用自动导航到与所选节点对应的源代码行。

从Source自动滚动

单击此按钮可使IntelliJ IDEA自动将结构工具窗口中的焦点移动到与光标当前在编辑器中定位的代码对应的节点。
