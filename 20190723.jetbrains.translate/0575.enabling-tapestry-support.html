IntelliJ IDEA 2019.1帮助

启用Tapestry支持

此功能仅在Ultimate版本中受支持。

您可以在创建项目或模块或现有模块时启用Tapestry支持。

  • 在创建项目或模块时启用Tapestry支持
  • 为现有模块启用Tapestry支持

在创建项目或模块时启用Tapestry支持

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目中：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java Enterprise 。在页面的右侧，指定要使用的JDK ，并选择要支持的Java EE版本。

  3. 在Additional Libraries and Frameworks下 ，选择Tapestry复选框。

  4. 您需要一个实现Tapestry的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现Tapestry的库文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

    单击下一步 。

  5. 指定名称和位置设置。

    单击完成 。

  6. 在打开的对话框中指定应用程序筛选器名称和根包 。

因此，Tapestry实现库将添加到模块依赖项中，并创建Tapestry构面。

为现有模块启用Tapestry支持

  1. 打开项目工具窗口（例如View | Tool Windows | Project ）。

  2. 右键单击感兴趣的模块，然后选择“ 添加框架支持” 。

  3. 在打开的“ 添加框架支持”对话框的左侧窗格中，选择“ Tapestry”复选框。

  4. 您需要一个实现Tapestry的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现Tapestry的库文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

  5. 在Add Frameworks Support对话框中单击OK 。

  6. 在打开的对话框中指定应用程序筛选器名称和根包 。

因此，Tapestry实现库将添加到模块依赖项中，并创建Tapestry构面。
