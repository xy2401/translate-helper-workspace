启动调试器会话

在调试之前

  1. 配置调试器 。

  2. 在源代码中设置断点 。

  3. 如有必要，请创建或修改相应的运行/调试配置 。

调试会话以所选的运行/调试配置开始。请注意，可以同时启动多个调试过程。

或者，按Shift + Alt + F9 ，从列表中选择配置，然后按Enter键 。

开始调试应用程序

  • 选择要执行的运行/调试配置，然后执行以下操作之一：

    • 点击左侧排水沟中的图标，然后选择 。

    • 点击在工具栏上。

    • 在主菜单上，转到“运行”|调试'选定配置' 。

    • 按Shift + F9 。

请注意，在您启动调试会话后， 标记调试工具窗口切换到的图标表示调试过程处于活动状态。
