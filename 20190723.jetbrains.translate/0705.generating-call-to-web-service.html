生成对Web服务的调用

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA提供编码帮助，用于开发客户端对Web服务的调用，包括严格类型和松散类型 。对强类型 Web服务的调用的编码辅助是Web服务接口和特定于数据类型，并且基于从WSDL描述符生成的XML-Java绑定 。

IntelliJ IDEA还可以帮助您生成对松散类型的 Apache Axis或XFire Web服务的调用 。在这种情况下，通过专用方面提供基本Web服务客户端编码辅助。

调用生成的强类型Web服务

  1. 在编辑器中打开客户端代码，并将光标定位在应生成调用的位置。

  2. 选择工具| Web服务|主菜单或Web服务上的Web服务客户端代码 |上下文菜单上的WebServices客户端代码 。

  3. 从打开的上下文菜单中，根据目标Web服务的类型选择相关选项。可用选项包括：

    • 为轴生成WS调用

    • 为Axis2生成WS调用

    • 为JAX-WS RI / GlassFish / JWSDP生成WS调用

    • 为XFire生成WS调用

    • 为JAXRPC生成WS调用

    • 生成针对WebSphere的WS调用

  4. 通过在模板输入字段中提供相关数据，使用建议列表和意图操作（如果适用）来完成生成的调用代码存根。活动输入字段以红框突出显示。

    建议列表的内容由WSDL描述符生成的Java类确定。

要生成一个松散类型的Web服务

  1. 在编辑器中打开客户端代码，并将光标定位在应生成调用的位置。

  2. 选择工具| Web服务|主菜单或Web服务上的Web服务客户端代码 |上下文菜单上的WebServices客户端代码 。

  3. 从打开的上下文菜单中，根据目标Web服务的类型选择相关选项。可用选项包括：

    • 为轴生成WS无类型调用

    • 为XFire生成WS无类型调用

  4. 通过在模板输入字段中提供相关数据，使用建议列表和意图操作（如果适用）来完成生成的调用代码存根。活动输入字段以红框突出显示。

    建议列表的内容由通过专用方面提供的库确定。
