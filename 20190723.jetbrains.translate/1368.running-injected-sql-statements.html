运行注入的SQL语句

此功能仅在Ultimate版本中受支持。

您可以将 SQL语句注入字符串文字，然后运行该语句：

  1. 在编辑器中，将光标放在相应的字符串文字中。

  2. 执行以下操作之一：

    • 按Ctrl + Enter 。

    • 按Alt + Enter并选择在控制台中运行查询 。

    • 点击并选择在控制台中运行查询 。

  3. 如果询问，请选择要使用的数据库控制台 。

  4. 如果语句包含参数，请指定参数值。
