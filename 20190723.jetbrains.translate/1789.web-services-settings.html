IntelliJ IDEA |偏好|工具|用于macOS的Web服务
按Ctrl + Alt + S

使用此对话框指定用于生成Web服务的URL地址的默认服务器名称和端口，以及无法通过专用方面直接启用的Web服务引擎的安装目录的路径。

项目

描述

外部引擎

在本节中，指定外部Web服务引擎的安装目录的路径。手动输入路径，或单击“ 浏览” 并在打开的对话框中选择路径。支持以下外部引擎：

  • Glassfish / JAX-WS 2.2 RI / Metro 1。X / JWSDP 2.2

  • Apache Axis 2
  • CXF
  • JBoss的
  • Xml豆
  • WebSphere 6.x

服务器名称

在此文本框中，指定要在URL地址中使用的服务器的默认名称。IntelliJ IDEA建议使用localhost 。

服务器端口

在此文本框中，指定要在URL地址中使用的默认端口号。IntelliJ IDEA建议8080 。

Web服务URL的前缀路径

在此文本框中，指定要在Web服务的URL地址中使用的默认前缀。IntelliJ IDEA建议/services 。

启动工具时的最大VM堆大小（Mb）

在此文本框中，指定启动Web服务时将使用的默认最大堆大小（以Mbs为单位）。
