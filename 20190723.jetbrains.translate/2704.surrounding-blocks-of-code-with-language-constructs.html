环绕代码片段

PyCharm根据源代码的语言为周围的代码片段提供各种构造的标准模板。这包括if...else条件语句， do...while和for循环， try...catch...finally ，XML标签，折叠区域和其他结构的组合。

用语言结构包围代码块

  1. 选择所需的代码片段。

  2. 在代码菜单上，单击环绕 （ Ctrl + Alt + T ）。

  3. 从列表中选择必要的环绕声。

此功能适用于以下语言（上下文）：
