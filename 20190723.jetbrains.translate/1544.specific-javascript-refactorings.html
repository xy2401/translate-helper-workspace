重构JavaScript

此功能仅在Ultimate版本中受支持。

重构意味着更新源代码而不改变应用程序的行为。重构可帮助您保持代码的稳定， 干燥和易于维护。

移动符号重构

除了移动文件和文件夹 ，IntelliJ IDEA还允许您移动JavaScript顶级符号。Move Symbol Refactoring适用于ES6模块中的类，函数和变量。

移动类，函数或变量

  1. 选择要移动的符号。

  2. 按F6或选择Refactor |从上下文菜单或从主菜单中移动 。“ 移动模块成员”对话框打开。

  3. 指定目标文件并选择要移动的成员。

  4. 默认情况下，IntelliJ IDEA会自动将成员的可见性提升到所需级别。如果你想保持能见度水平不变，单击由于是在能见度 。

或者，选择Refactor |重构此按钮或按Ctrl + Shift + Alt + T ，然后从列表中选择“ 移动 ”。

重命名重构

如果您刚刚创建了一个新文件，但在开始在其中键入类或接口时提出了更好的名称，这将非常有用。

要调用意图操作，请将光标放在感兴趣的类的名称上，然后按Alt + Enter 。

除了重命名文件和文件夹 （可在任何语言的上下文中使用）之外，您还可以重命名类，函数，变量和参数。IntelliJ IDEA在其声明中更改符号的名称，默认情况下更改当前项目中的所有用法。

重命名函数，类或变量

  1. 在编辑器中，选择要重命名的符号，然后按Shift + F6或选择Refactor |从上下文菜单或主菜单重命名 。

  2. 在打开的“ 重命名”对话框中，键入符号的新名称。

  3. （可选）选中“ 在注释和字符串中 搜索”和“ 搜索文本实例”复选框，以在注释，字符串文字和文本中重命名函数或类的用法。

  4. 如有必要，请预览并应用更改 。

要重命名参数

  1. 在编辑器中选择参数，然后按Shift + F6或选择Refactor |从上下文菜单或主菜单重命名 。

  2. 在所选参数周围带有红色画布的字段中，键入新参数名称。

  3. 按Enter键运行重构。

保持类的名称和包含文件的符合性

重命名类时，IntelliJ IDEA还建议重命名该文件，如果它具有相同的名称。如果您接受该建议，IntelliJ IDEA将在其他文件的import语句中更新此文件的名称。

如果您拒绝此建议，则可以在以后使用重命名文件...意图操作随时重命名该文件。另一个意图操作建议将类移动到具有相应名称的新文件。建议名称的格式由“ 代码样式：JavaScript”页面上的“ 文件名约定”列表中选择的样式确定。

提取参数

  1. 在编辑器中，将光标放在要转换为参数的表达式中，然后按Ctrl + Alt + P或选择Refactor |。提取|参数从上下文菜单或主菜单。

  2. 如果在当前光标位置检测到多个表达式，请从“ 表达式”列表中选择所需的表达式 。

  3. 如果找到多个选定表达式，请选择仅替换此事件或从找到的多个匹配项列表中替换所有匹配项 。

    最后，出现用于配置重构的弹出窗口。

  4. 选择Generate JSDoc以生成JSDoc注释块。如果您需要指定自定义默认参数值，这可能会有所帮助。从JSDoc官方网站了解更多信息。

  5. 选择新参数的类型（ 可选或必需 ）并指定其默认值（如果适用）：

    • 如果选中Optional parameter复选框，则将使用函数体中的默认值初始化该参数。

    • 如果清除Optional参数复选框，则默认参数值将通过现有函数调用传递。所有函数调用都将根据新的函数签名进行更改，并且参数初始化将添加到函数体中。

    最初，IntelliJ IDEA接受调用重构作为默认值的表达式。在大多数情况下，您无需更改它。如果仍然需要，请以@param <parameter name> - <default value>格式在JSDoc注释中指定另一个默认值。

  6. 通过在列表中双击建议的参数名称，或者在红色画布的字段中指定自定义名称，接受其中一个建议的参数名称。准备好后按Enter键 。

    另请注意，在ES6代码中，将应用新的默认函数参数语法function calculate_sum(i, i2 = 1)而不是i2 = i2 || 1; 。从https://developer.mozilla.org网站了解有关默认功能参数的更多信息。

选择重构模式

您可以如上所述在编辑器中（在就地模式下） 提取参数 ，或使用“ 提取参数”对话框 。这两种方法相似，区别如下：

  • 预览重构的结果 。
    在对话框中，可以单击“ 预览”并在“ 查找”工具窗口的专用选项卡中检查预期的更改。在就地模式下，此功能不可用。

  • 指定默认参数值 。
    在对话框中，IntelliJ IDEA会在“ 值”字段中建议默认参数值，您可以在其中接受建议或指定其他值。在就地模式下，IntelliJ IDEA将调用重构的表达式视为默认参数值。要指定另一个值，必须使用JSDoc注释块。

默认情况下，IntelliJ IDEA在就地模式下运行Extract Parameter重构。要使用“ 提取参数”对话框，请在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中单击“ 编辑器”|一般 。在打开的“ 常规”页面上，清除“ 重构”区域中的“ 启用就地模式”复选框。
