Android开发的先决条件

在开始在IntelliJ IDEA中编写第一个“Hello，world”Android应用程序之前，请执行以下操作：

  1. 下载并安装JDK 7。

  2. 下载 Android SDK工具。

    官方Android下载页面提供了不同的下载选项。默认下载选项是Android Studio，其中包含Android SDK工具。由于您已经拥有带有捆绑Android插件的IntelliJ IDEA，因此您可以选择较小的下载并选择Android SDK和平台工具。滚动到其他下载选项，然后选择适用于您的操作系统的SDK Tools Only软件包。

  3. 运行安装程序以启动Android SDK工具安装向导：

    首先，向导检查是否安装了Java SDK。接下来，选择安装选项并指定目标文件夹。我们强烈建议Android SDK主目录的路径不包含空格。

  4. 安装成功完成后，您需要使用SDK Manager添加SDK包。在Android SDK Tools Setup向导的最后一页上，选择Start SDK Manager选项，然后单击Finish 。

  5. 在Android SDK Manager中 ，选择要安装的软件包（有关详细说明，请参阅添加SDK软件包 ）。

    对于我们教程中介绍的过程，您可以保留默认选择。如果需要安装更多软件包，可以随时启动SDK Manager 。要调用它，请在主菜单中选择“ 工具”| Android | SDK Manager 。
