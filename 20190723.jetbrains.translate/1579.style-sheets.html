IntelliJ IDEA 2019.1帮助

样式表

使用IntelliJ IDEA，您可以用CSS和编入其中的各种语言编写样式定义，例如Sass ， Less ， SCSS或Stylus 。

IntelliJ IDEA识别并处理最常见的样式选择器：通用选择器* ，类型选择器.a ，后代选择器.ab ，子选择器.a .b ，ID选择器#b ，伪类和类选择器DIV.warning 。

IntelliJ IDEA为选择器，属性及其值提供代码完成，并通过使用HTML或JavaScript中的选择器到样式表文件中的定义进行导航。您还可以使用代码完成和导航到变量和mixin的定义。

使用样式声明

IntelliJ IDEA允许您从现有声明创建新规则集，甚至可以使用CSS感知意图操作和重构从其他文件移动整个规则集。

提取规则集

  1. 选择要提取的声明。如果只需要一个声明，只需将光标放在其中即可。

  2. 按Alt + Enter并从列表中选择提取规则集 。

IntelliJ IDEA使用相同的选择器创建一个新的规则集，并将选定的声明移动到它。如果选择包含注释，嵌套选择器等，则它们也会移动到新规则集。

将规则集移动到另一个文件

  1. 将光标放在规则集中的任意位置，然后按F6 。

  2. 在打开的对话框中，指定要将规则集移动到的文件。如果指定的文件不存在，IntelliJ IDEA将建议创建它。

  3. 默认情况下，IntelliJ IDEA会自动打开移动规则集的文件。要更改此行为，请清除“ 打开编辑器”复选框。

完成课程

IntelliJ IDEA首先建议样式标记和链接文件中的类和ID。但如果找不到匹配的结果，IntelliJ IDEA还会建议在项目中的所有样式表文件中定义符号。

要在开始输入之前立即查看项目中定义的所有类和ID，请按两次Ctrl + Space 。

提取变量

使用IntelliJ IDEA，您可以使用.css文件中的var(--var-name)语法轻松引入变量。

  1. 在编辑器中，将光标放在表达式上以转换为变量，然后按Ctrl + Alt + V或选择Refactor |提取|上下文菜单或主菜单中的变量 。

  2. 如果找到多个选定表达式，请选择仅替换此事件或从找到的多个匹配项列表中替换所有匹配项 。

  3. 在带有红色边框的字段中，接受建议的变量名称或指定自定义名称。准备好后按Enter键 。

文档查找

请注意，仅针对Chrome，Chrome Android，Safari，Safari iOS，Firefox，Internet Explorer和Edge检查兼容性。

对于CSS属性和伪元素，IntelliJ IDEA可以显示相应MDN文章的摘要。此摘要显示在“ 文档”弹出窗口中，其中显示了属性及其值的简要说明，以及有关其与各种浏览器的兼容性的信息。

查看插入符号属性的文档

  • 按Ctrl + Q或选择查看|主菜单中的快速文档查找 。IntelliJ IDEA打开文档弹出窗口，其中包含该属性的简要说明。

    • 如果该属性在所有版本的浏览器中都可用，则IntelliJ IDEA不会显示有关其兼容性的任何信息。

    • 否则，“ 文档”弹出窗口还会列出支持该属性的浏览器及其版本。

在浏览器中打开MDN文档

  • 在“ 文档”弹出窗口（ Ctrl + Q ）中，单击底部的链接。

  • 按Shift + F1或选择查看|主菜单中的外部文档 。

检查与浏览器的兼容性

除了查看文档弹出窗口外 ，您还可以检查CSS属性是否与动态特定浏览器兼容。此检查基于MDN浏览器兼容性数据，并在每次目标浏览器不支持属性时向您显示警告。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|检查 。

  2. 展开CSS节点，然后选择“ 浏览器兼容性”复选框 。在“ 选项”区域中，选择要定位的浏览器及其最低版本。

格式化样式表

有关详细信息，请参阅Prettier 。

每次IntelliJ IDEA生成，重构或重新格式化样式表代码时，它都会自动应用特定于语言的代码样式设置 。

或者，您可以使用Prettier格式化程序。

使用更漂亮的格式化样式表

  • 选择要在编辑器中重新格式化的代码片段，或在“ 项目”工具窗口中选择文件或文件夹，然后按N / A.

更改样式表中的颜色值

此功能仅在Ultimate版本中受支持。

使用IntelliJ IDEA，您可以轻松更改样式表中的颜色值，而无需键入十六进制代码。

选择颜色

  1. 打开所需的样式表进行编辑。

  2. 键入color: ，然后按Ctrl + Space 。

  3. 从建议列表中选择所需的颜色值，或选择颜色...以选择自定义颜色值。

改变颜色

  1. 打开所需的样式表进行编辑，然后找到要更改的颜色属性。

  2. 单击装订线中的颜色图标。

    或者，如果未显示图标，请按Alt + Enter并从列表中选择更改颜色 。

  3. 在打开的“选择颜色”对话框中，选择所需的新颜色，然后单击“选择” 。

请参见在装订线中显示颜色图标 。

预览颜色代码

  • IntelliJ IDEA使用相应颜色的装订线图标标记每个color属性。当您将鼠标悬停在颜色图标上时，IntelliJ IDEA会显示一个弹出窗口，其中显示颜色预览及其十六进制代码。

    要使用代码而不是人类可读的颜色名称，请按Alt + Enter并从列表中选择将颜色转换为<颜色代码系统> ，其中<颜色代码系统>是HEX ， HSL ， HWB或RGB

在阴沟中显示颜色图标

默认情况下，IntelliJ IDEA在装订线中显示颜色图标。如果出于某种原因隐藏它们，您可以随时返回它们。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|天沟图标 。将打开“ 装订线图标”页面 。

  2. 在“ 公共”区域中，选择“ 颜色预览”复选框。

查看应用于标记的样式

此功能仅在Ultimate版本中受支持。

对于HTML，XHTML，JSP，JSPX文件，IntelliJ IDEA提出了一种探索应用于任意标记的所有样式的方法。

每个标记的结果显示在“ CSS样式”工具窗口的专用选项卡中。使用此工具窗口，您可以查看应用于标记的样式列表以及这些样式的定义。除此之外，您还可以从样式导航到源代码中的相应标记。

  1. 在编辑器中打开所需文件，然后右键单击要探索应用样式的标记。

  2. 从上下文菜单中，选择“ 显示应用的标签样式” 。

配置语法突出显示

您可以根据自己的喜好和习惯配置支持CSS的语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案| CSS 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
