将构建文件添加到项目中

应将构建文件添加到项目中以启用运行构建，过滤目标或定义属性。一个项目可以有几个构建文件。构建文件按名称在Ant构建工具窗口中分组，在每个构建文件的根元素中指定，例如<project name="acme"> 。

生成构建文件时，它会自动添加到项目中。如果手动创建构建文件或重用现有构建文件，则可能需要将其添加到Ant构建工具窗口 。

将构建文件添加到项目中

  1. 打开Ant构建工具窗口 。为此，请选择“ 查看”| Ant在主菜单上构建 ，或单击工具窗口右侧工具栏中的Ant Build按钮。

  2. 在Ant Build工具窗口中，单击 。

  3. 在“选择路径”对话框中，导航到所需的build.xml文件，然后单击“ 确定” 。

    构建文件应至少具有要添加到Ant构建工具窗口的根元素。
