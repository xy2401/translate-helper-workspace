PyCharm 2019.1帮助

运行和调试

功能

捷径

使用此快捷方式...

跑

按住Shift + F10

运行程序。

选择配置并运行

SHIFT + ALT + F10

快速选择运行/调试配置并运行或编辑它。

重新运行

按Ctrl + F5

使用相同设置重复执行，使用具有焦点的“运行”工具窗口的相同选项卡。

重新运行而不会在编辑器中失去焦点

按住Shift + F10

使用相同的设置重复执行，编辑器的相同选项卡具有焦点。

调试

按住Shift + F9

调试程序。

选择配置和调试

SHIFT + ALT + F9

快速选择运行/调试配置并调试或编辑它。

跳过

F8

转到当前文件中的下一行。请参阅单步执行程序 。

踏入

F7

步骤到下一个执行的行。请参阅单步执行程序 。

聪明的步入

按住Shift + F7

如果当前行包含多个方法调用表达式，请选择要介入的方法。请参阅选择要进入的方法 。

走出去

按住Shift + F8

从当前方法返回后，跳到第一个执行的行。请参阅单步执行程序 。

强行退一步

SHIFT + ALT + F8

运行此方法或文件中的下一行，跳过当前执行点引用的方法并忽略断点。请参阅单步执行程序 。

强行进入

SHIFT + ALT + F7

即使要跳过此方法，也要进入当前执行点中调用的方法。请参阅单步执行程序 。

运行到光标

Alt + F9

运行到插入符所在的行。请参阅单步执行程序 。

强制运行到光标

按Ctrl + Alt + F9

运行到插入符所在的行，忽略断点。请参阅单步执行程序 。

简历程序

F9

恢复程序执行。

停止计划

按住Shift + F2

终止调试会话。

评估表达

Alt + F8

评估任意表达式。

快速评估表达

按Ctrl + Alt + F8

无需调用“ 计算表达式”对话框即可评估任意表达式。

切换断点

按Ctrl + F8

在当前行切换断点。

查看断点

按Ctrl + Shift + F8

查看/管理所有断点。
