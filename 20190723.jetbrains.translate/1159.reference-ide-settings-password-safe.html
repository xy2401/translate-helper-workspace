按Ctrl + Alt + S

指定IntelliJ IDEA是否应保存您的密码 - 您用于访问受密码保护的资源（如版本控制存储库和数据库）的密码。

使用KeePass密码管理器时 ，主密码将用于访问存储个人密码的文件。一旦IntelliJ IDEA记住您的密码，除非您需要访问密码数据库，否则它不会再次要求输入密码，包括主密码。

在保存密码下 ，您可以配置密码设置。

项目

描述

在原生钥匙串

IntelliJ IDEA仅为macOS和Linux显示此选项。
选择此选项可使用本机Keychain存储密码。

在KeePass

选择此选项可使用KeePass密码管理器存储密码。

数据库

此字段显示当前c.kdbx文件的位置。如果您需要选择其他位置，请单击在打开的对话框中，选择适当的对话框 。
如果要导入另一个c.kdbx文件，请单击图标，然后从列表中选择导入 。如果要删除c.kdbx文件中的现有密码，请选择“ 清除” 。

主密码

使用此字段输入要用于访问c.kdbx文件的密码。IntelliJ IDEA第一次自动生成密码。

重启后不要保存，忘记密码

如果要在重新启动后删除包含单个密码的c.kdbx文件，请选择此选项。
