扭曲的测试驱动开发

介绍

在这里，我们将使用位于此地址的教程“Twisted的测试驱动开发”，并尝试使用PyCharm重复它。

创建项目，包和Python文件

在主菜单上，选择“ 文件”| New Project ...并在New Project对话框中，选择创建纯Python项目。我们称之为Calculus 。

创建一个名为calculus的空目录，其中包含一个空的__init__.py文件。为此，请选择“ 文件”|在主菜单上新建... ，然后从“ 新建”弹出菜单中选择“ Python包 ”。

接下来，创建一个Python文件。为此，再次指向File |在主菜单上新建... ，然后在弹出窗口中选择Python文件 。此文件名为base_1.py ，它会立即打开以进行编辑。输入以下代码：

class Calculation （ object ）： def add （ self ， a ， b ）： pass def subtract （ self ， a ， b ）： pass def multiply （ self ， a ， b ）： pass def divide （ self ， a ， b ）： pass

复制此代码是可能的，但我们强烈建议您手动输入 - 这样您就可以亲身体验强大的PyCharm代码完成功能。

接下来，右键单击包calculus并再次创建一个Python包 - 这次应该称为test 。

安装包

您必须安装Twisted包才能执行测试。除此之外，如果您在Windows上工作，还要安装包pypiwin32 。

在PyCharm中安装所需的软件包非常容易：要执行此操作，请转到“设置/首选项”对话框，然后单击“ 项目翻译”页面。在此页面的“ 包”工具栏部分中，单击“确定” 并在“ 可用包”对话框中选择所需的包。有关详细信息，请阅读安装，卸载和升级包部分。

选择所需的测试运行器

再次在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中的“ 工具”节点下，单击“ Python集成工具”页面。

在此页面上，从Default test runner字段的列表中选择测试运行器Twisted Trial 。

应用更改并关闭“设置/首选项”对话框。

创建失败的测试

右键单击类声明旁边的文件base_1.py ，然后按Ctrl + Shift + T或选择Go To |在上下文菜单上测试 。在“ 创建测试”对话框中，您必须对默认设置进行一些更改：

首先，单击Target directory字段旁边的Browse按钮，然后选择test package。

其次，在Test file name字段中输入名称test_base_1.py ，在Test class name字段中输入名称TestBase_1 。最后，选择所有方法 - 因此将包括测试类的所有方法的测试。

您应该获得以下代码：

来自 unittest import TestCase 类 TestBase_1 （ TestCase ）： def test_add （ self ）： self 。 fail （） def test_subtract （ self ）： self 。 fail （） def test_divide （ self ）： self 。 fail （） def test_multiply （ self ）： self 。 失败 （）

这是一个工人阶级，但我们正在为Twisted编写测试...让我们导入包。您最终会得到以下代码：

从 twisted.trial 进口 单元测试 类 TestBase_1（ 单元测试 的TestCase）：DEF test_add（ 个体经营 ）： 自我 。 fail （） def test_subtract （ self ）： self 。 fail （） def test_divide （ self ）： self 。 fail （） def test_multiply （ self ）： self 。 失败 （）

由于TDD（测试驱动开发）的想法是在代码之前编写测试，让我们进行实际测试：

从 进口 twisted.trial 单元测试 类 CalculationTestCase（ 单元测试 测试用例 。）calculus.base_1 进口 计算 的：def test_add（ 个体 ）： 计算值 = 计算 （） 结果 计算值 =。 添加 （3,8） 的自我 。 assertEqual （ result ， 11 ） def test_subtract （ self ）： calc = Calculation （） result = calc 。 减去 （7,3） 的自我 。 assertEqual （ result ， 4 ） def test_multiply （ self ）： calc = Calculation （） result = calc 。 乘 （12，5） 的自我 。 assertEqual （ result ， 60 ） def test_divide （ self ）： calc = Calculation （） result = calc 。 分 （12，4） 的自我 。 assertEqual （ 结果 ， 3 ）

顺便说一下，查看项目工具窗口 - 它显示项目的结构。目前，您有4个文件：
- calculus/__init__.py
- calculus/base_1.py
- calculus/test/__init__.py
- calculus/test/test_base_1.py
