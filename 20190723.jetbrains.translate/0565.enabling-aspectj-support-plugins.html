启用AspectJ支持插件

此功能仅在Ultimate版本中受支持。

为了能够在IntelliJ IDEA Ultimate中使用AspectJ支持，您必须确保启用以下插件 ：

  • Spring AOP / @ AspectJ

  • AspectJ支持

除此之外，您还应该为aspectjrt.jar创建一个库 。

确保启用了必要的插件

  1. 在“ 设置/首选项”对话框中（例如Ctrl + Alt + S ），键入plugins 。

  2. 在搜索框下方的窗格中选择“ 插件 ”。

  3. 在对话框右侧打开的“ 插件”页面上 ，确保在“ 显示”框中选中“ 所有插件 ”，然后在搜索字段中键入aspectj 。

  4. 确保选中了AspectJ支持和Spring AOP / @ AspectJ右侧的复选框。

  5. 在Settings对话框中单击OK 。

  6. 如果询问，请重新启动IntelliJ IDEA。
