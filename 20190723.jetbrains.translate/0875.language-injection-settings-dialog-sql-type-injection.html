键入模式

您可以测试您的模式：单击或按Alt + Enter ，然后选择Check RegExp 。然后在Sample字段中键入要与模式匹配的文本。

语言

要注入相应类型的字符串值的语言。

  • ID。 语言ID或名称。

  • 字首。 要在相应字符串值之前添加的字符序列。

  • 后缀。 要在相应字符串值之后添加的字符序列。
