指定代码清理范围对话框

代码|代码清理

指定代码清理的范围。（代码清理意味着找到可能存在问题的代码片段并立即自动修复它们。）

项目

描述

整个项目

如果要对整个项目执行代码清理，请选择此选项。

未提交的文件

此选项仅适用于版本控制下的项目。

如果您只想对尚未提交到版本控制系统的文件执行代码清理，请从列表中选择更改列表，请选择此选项。

文件<文件路径>

选择此选项可对在编辑器中打开或在“ 项目”工具窗口中选择的文件执行代码清理。

目录<目录路径>

选择此选项可对“ 项目”工具窗口中当前选定的目录执行代码清理。

选定的文件

选择此选项可对“ 项目”工具窗口中选定的文件执行代码清理。

自定义范围

选择此选项可指定自定义范围。从列表中选择一个预定义范围，或单击并在打开的“ 范围”对话框中定义范围。有关如何定义范围的说明，请参阅范围语言语法参考 。

检查简介

选择要使用的检验配置文件。从列表中选择预定义的配置文件，或单击并在打开的“ 检查”对话框中配置配置文件。

您可以打开“ 检查”对话框，以检查在运行代码清理时将对所选范围应用哪些修补程序。
