PyCharm 2019.1帮助

将文件另存为模板对话框

PyCharm可帮助您从现有文件创建文件模板 。

项目

描述

名称

在此处指定新模板将在“ 文件和代码模板”设置的“ 文件”选项卡中显示的名称。默认情况下，使用当前文件的名称。

延期

在此指定此新模板要创建的文件的扩展名。默认情况下，使用当前文件的扩展名。

模板文本

编辑模板内容。您可以使用：

  • 纯文本。

  • 使用模板包含的 #parse指令。

  • 自定义变量。变量的名称可以通过#set指令直接在模板中定义，也可以在文件创建期间定义。

    请注意，PyCharm不会提示使用#set定义的Velocity变量的值。

  • 要变换为${<variable_name>}格式的相应值的${<variable_name>} 。

    可以在文件模板中使用以下预定义变量：

    变量

    描述

    $ {DATE}

    当前系统日期

    $ {DAY}

    当月的当天

    $ {} DS

    美元符号（ $ ）。此变量用于转义美元字符，因此不会将其视为模板变量的前缀。

    $ {} END

    完成编辑变量后的插入位置

    $ {HOUR}

    当前时间

    $ {} MINUTE

    当前分钟

    $ {MONTH}

    这个月

    $ {} MONTH_NAME_FULL

    当月的全名（1月，2月等）

    $ {} MONTH_NAME_SHORT

    当前月份名称的前三个字母（Jan，Feb等）

    $ {NAME}

    新实体的名称（文件，类，接口等）

    $ {PRODUCT_NAME}

    IDE的名称（例如，PyCharm）

    $ {} PROJECT_NAME

    当前项目的名称

    $ {TIME}

    当前系统时间

    $ {USER}

    当前用户的登录名

    $ {YEAR}

    今年

    对待美元符号

    • 您可以防止将模板变量中的美元字符（ $ ）视为前缀。如果您需要一个美元字符（按原样插入$ ，请使用${DS}文件模板变量。应用模板时，此变量的计算结果为普通美元字符（ $ ）。

      例子：

      • 要在默认类模板中使用某些版本控制关键字（例如$Revision$ ， $Date$等），请写入${DS}而不是美元前缀（ $ ）。

      • 模板代码${DS}this将呈现为$this 。

根据风格重新格式化

选中此复选框，让PyCharm根据“ 代码样式”页面上定义的样式重新格式化生成的存根文件。

此选项仅在“ 文件”选项卡中可用。

启用实时模板

选中此复选框以在文件模板中使用实时模板。在这样做时，必须将实时模板片段放入Velocity转义语法中。

例如：

＃[[$ MY_VARIABLE $ $ END $]]＃

因此，可以指定光标位置。请注意，这里需要使用实时模板变量 ！

描述

此只读字段提供有关模板，其预定义变量及其工作方式的信息。
