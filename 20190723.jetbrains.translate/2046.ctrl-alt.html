PyCharm 2019.1帮助

按Ctrl + Alt

本节列出并描述了包含Ctrl + Alt键的键盘快捷键：

  • Ctrl + Alt +字母数字键
  • Ctrl + Alt +导航键
  • Ctrl + Alt +功能（F）键

Ctrl + Alt +字母数字键

捷径

功能

使用此快捷方式...

Ctrl + Alt + B

导航到实施

导航到插入符号项的实现 。

Ctrl + Alt + C

提取常数

用常量 （静态最终字段） 替换所选表达式 （重构）。

按Ctrl + Alt + F

提取字段

将选定的表达式结果放入一个字段 （重构）。

按Ctrl + Alt + H

调用层次结构

浏览所选方法的调用层次结构。请参见页面查看源代码的结构和层次结构

按Ctrl + Alt + I

自动缩进行

根据代码样式设置缩进当前行或选定的块。

Ctrl + Alt + J

环绕现场模板

使用其中一个实时模板环绕选择。

按Ctrl + Alt + M

提取方法

从所选代码创建方法 （重构）。

Ctrl + Alt + N

排队

内联选定的方法/变量 （重构）。

按Ctrl + Alt + P

提取参数

将所选表达式转换为方法参数 （重构）。

Ctrl + Alt + T

环绕着

使用if ， while ， try/catch或其他构造环绕选定的代码片段。

Ctrl + Alt + V

提取变量

将选定的表达式结果放入变量（重构）。请参见提取变量页面。

按Ctrl + Alt + Y

同步

检测所有外部更改的文件并从磁盘重新加载它们。

Ctrl + Alt +导航键

捷径

功能

使用此快捷方式...

按Ctrl + Alt + Enter键

在当前之前开始新行

在当前行之前开始一个新行。

按Ctrl + Alt +向下
按Ctrl + Alt +向上

导航到下一个/上一个出现

导航到下一个/上一个找到的项目。

按Ctrl + Alt +左

背部

撤消上次导航操作。请参阅有关导航项目的页面

在macOS计算机上，您还可以使用三指从右向左轻扫手势。

按Ctrl + Alt +右

向前

重做最后撤消的导航操作。请参阅有关导航项目的页面

在macOS计算机上，您还可以使用三指从左向右滑动手势。

按Ctrl + Alt +首页

导航到相关符号

在具有各种关系的文件之间导航。

Ctrl + Alt +功能（F）键

捷径

功能

使用此快捷方式...

按Ctrl + Alt + F6

切换到另一个覆盖套件。

打开Coverage Suites弹出窗口，然后选择要运行的所需套件。

按Ctrl + Alt + F7

显示用法

在插入符号上显示符号的用法。

参见页面查看符号的用法

按Ctrl + Alt + F8

快速评估表达

在不调用“ 计算表达式”对话框的情况下计算任意表达式 。

按Ctrl + Alt + F9

强制运行到光标

运行到插入符所在的行，忽略断点。参见页面逐步完成程序 。
