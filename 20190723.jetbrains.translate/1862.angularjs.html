AngularJS

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

在开始之前，请确保您的计算机上有Node.js.

AngularJS也称为Angular 1，是用于开发单页Web应用程序的框架。PyCharm为预定义和自定义ng指令以及控制器和应用程序名称建议了AngularJS感知的完成选项，以及大括号表达式{{}}内的数据绑定的代码洞察。您可以使用内置的AngularJS实时模板，并在HTML中的控制器名称及其在JavaScript中的定义或ngView或&routeProvider与模板之间进行ngView 。对于AngularJS实体，请使用“ 转到符号”导航。

创建一个新的AngularJS应用程序

AngularJS依赖项包含AngularJS代码以及支持开发和测试的工具。

有关详细信息，请参阅配置JavaScript库 。

您还可以在“设置/首选项”对话框的“ Bower”页面上安装angular包，如“ 管理Bower包”中所述 。

您可以使用AngularJS种子项目或创建一个空的PyCharm项目，并通过下载AngularJS框架或Bower包管理器手动安装AngularJS。

使用种子项目创建应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择AngularJS 。

  3. 在右侧窗格中，指定将存储与项目相关的文件的文件夹的路径。

  4. 单击“ 创建”时 ，PyCharm将根据AngularJS种子项目生成包含所有必需配置文件的AngularJS特定项目结构

要下载AngularJS依赖项，请执行以下操作之一：

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm安装

  • 从项目根目录中的package.json文件的上下文菜单中选择Run'npm install' 。

创建一个空的PyCharm项目

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Empty Project 。

  3. 指定将存储与项目相关的文件的文件夹的路径。

  4. 单击“ 创建”时 ，PyCharm将创建并打开一个空项目。

手动在空项目中安装和配置AngularJS

  1. 在http://angularjs.org/下载AngularJS框架。

  2. 打开将使用AngularJS的纯Python项目。

  3. 将AngularJS配置为PyCharm JavaScript库，让PyCharm识别AngularJS特定结构并提供完整的编码帮助：

    1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript |图书馆 。将打开“ JavaScript库”页面 。

    2. 在“ 库”区域中，单击“ 添加”按钮。

    3. 在打开的“ 新建库”对话框中，指定库的名称。

    4. 点击在库文件列表旁边，从上下文菜单中选择“ 附加文件”或“ 附加目录 ”，具体取决于您是需要单独的文件还是整个文件夹。

    5. 选择Angular.js或Angular.min.js ，或打开的对话框中的整个目录。PyCharm返回New Library对话框，其中Name只读字段显示所选文件或文件夹的名称。

    6. 在“ 类型”字段中，指定已下载并将要添加的版本。

      • 如果添加了Angular.js ，请选择Debug 。此版本在开发环境中很有用，尤其适用于调试。

      • 如果添加了缩小的 Angular.min.js ，请选择“ 释放” 。此版本在生产环境中很有用，因为文件大小要小得多。

通过Bower在空项目中安装AngularJS

  1. 打开将使用AngularJS的纯Python项目。

  2. 安装鲍尔如描述鲍尔 。

  3. 在嵌入式终端 （ Alt + F12 ）中，键入bower install angular以将程序包安装为项目依赖项。

从现有的AngularJS应用程序开始

如果您的项目中已经有Angular源（例如，在bower_components文件夹中），只需打开项目并开始工作。如果从项目中排除这些源，则只需将AngularJS配置为JavaScript库 。

从版本控制中检出应用程序源

  1. 单击“ 欢迎”屏幕上的“ 从版本控制中签出”或选择“ VCS”从主菜单中查看版本控制 。

  2. 从列表中选择您的版本控制系统。

  3. 在打开的特定于VCS的对话框中，键入您的凭据和存储库以检出应用程序源。
