从TFS向导签出：源服务器

此功能仅在Ultimate版本中受支持。

VCS |从版本控制检出| TFS - 自动创建工作区

在此页面上，选择要从中下载数据的服务器。

项目

描述

团队服务器

在此列表框中，管理您有权访问的服务器列表，然后选择要从中下载数据的服务器。

加

单击此按钮以打开“ 添加Team Foundation Server”对话框，并指定TFS服务器的地址以及要连接到它的凭据。

去掉

单击此按钮可从列表中删除所选服务器。

TFS代理

单击此按钮可打开“ 为服务器设置TFS代理...”对话框，您可以在其中指定通过代理访问所选服务器的参数。
