结构工具窗口，文件结构弹出窗口

结构工具窗口

此工具窗口显示当前在编辑器中打开并具有焦点的文件的结构，或在“项目”工具窗口中选择的文件。

对于图表，此工具窗口显示图表预览。

查看|工具窗口|结构体
Alt + 7

文件结构弹出窗口

此弹出窗口显示当前在编辑器中打开并具有焦点的文件结构。

导航|文件结构
按Ctrl + F12

两种视图都有助于快速浏览文件的结构 。

本节介绍工具窗口标题栏上的按钮以及标题栏上下文菜单中的选项。打开和关闭这些选项可隐藏或显示某些类型的元素，并配置它们的显示方式。

标题栏上的按钮对于所有语言上下文都是通用的。上下文菜单上的选项集取决于上下文。

  • 标题栏
  • 蟒蛇
  • HTML，XML
  • JavaScript，TypeScript，CoffeeScript

请参阅“ 结构”工具窗口中文件树视图中的成员图标符号部分。

标题栏上的按钮对于所有语言上下文都是通用的。

展开全部Ctrl + NumPad Plus

单击此按钮可展开工具窗口中的所有节点。

全部折叠Ctrl + NumPad -

单击此按钮可使工具窗口中的所有节点折叠。

单击此按钮可打开上下文菜单，并通过打开或关闭菜单项来配置工具窗口的外观，其查看模式以及呈现当前文件结构的方式。

Shift +逃生使用此图标或快捷方式隐藏工具窗口。

与Alt键组合使用时，单击此图标会隐藏连接到同一工具窗口栏的所有工具窗口 。

当两个 （按可见性排序）和按（按字母顺序排列）按钮，视图中的项目将根据其可见性级别进行分组。在每个可见性级别组内，项目按字母顺序排序。释放两个按钮时，项目将按照它们在代码中显示的顺序显示。
