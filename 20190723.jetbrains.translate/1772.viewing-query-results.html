使用查询结果

此功能仅在Ultimate版本中受支持。

在控制台中运行查询时，您将收到表格格式的结果。每个语句都显示在Database Console工具窗口的单独选项卡中。例如，如果查询中有两个SELECT语句，您将在工具窗口中看到两个选项卡。

为每个查询打开一个新选项卡

默认情况下，每次运行查询时，IntelliJ IDEA都会更新相同的选项卡并显示结果。您可以更改此行为，并在每次运行查询时创建选项卡。

  1. 在IDE设置（ Ctrl + Alt + S ）中，转到数据库|一般 。

  2. 选择在新标签中打开复选框结果 ，并单击确定 。

编辑结果集中的值

  1. 在结果集中，单击要编辑的单元格值。

  2. 指定新值，然后按Enter键 。

  3. 要将更改提交到数据库，请单击“ 提交”图标（ ），或按Ctrl + Enter 。

与在数据编辑器中一样 ，您可以编辑结果集中的值，自动提交更改，删除行，搜索文本以及浏览数据。

导出到文件

  • 右键单击a，选择Dump Data |要归档 。

  • 右键单击查询，单击“ 执行到文件”，然后选择要用于导出的文件类型（例如， 逗号分隔（CSV） ）。

  • 在工具栏上，单击“ 转储数据”图标（ ）并选择To File 。

导出到剪贴板

  • 在结果集或编辑器中选择数据，然后按Ctrl + C.

  • 在工具栏上，单击“ 转储数据”图标（ ）并选择To Clipboard 。

比较两个结果集

  1. 选择结果集。

  2. 单击Compare with图标（ ），并选择另一个结果集。

  3. 如果需要，请在比较对话框中更改Tolerance参数的值。Tolerance参数定义两个结果集之间允许的最大差异数。例如，如果要将两行视为相等（如果它们的数据在单个列中不同），请在“ 容差”字段中输入1 。

使用查询结果固定选项卡

如果使用同一个选项卡显示查询结果，并且您获得了要保留的结果，则可以将选项卡固定到工具窗口。执行以下操作之一：

  • 右键单击选项卡，然后选择“ 固定活动选项卡” 。

  • 点击在工具栏上。

在行的子集之间导航

根据“ 限制页面大小”参数的值，结果集可能会分为多个页面。例如，您将“ 限制页面大小”设置为参数为100，但查询返回了200行。您将有两页行，每页有100行。要在页面之间导航，请使用以下控件：

  • ：导航到结果集的第一页。

  • ：导航到结果集的上一页。（ Ctrl + Alt + Up ）

  • ：导航到结果集的下一页。（ Ctrl + Alt + Down ）

  • ：导航到结果集的最后一页。

在结果集中设置多个行

默认情况下，发出查询时，返回的行数限制为500。引入此限制是为了避免重载（例如，当SELECT语句返回一百万行时）。

  1. 转到设置/首选项 （ Ctrl + Alt + S ）并导航到工具| 数据库|数据视图 。

  2. 在“ 限制页面大小”字段中指定新数字。要禁用此限制，请清除“ 限制页面大小”复选框。

转到指定的行

要导航到具有指定编号的行，请右键单击该表并选择“ 转到”行 （ Ctrl + G ）。在“ 转到行”对话框中，指定行号，然后单击“ 确定” 。

导航到相关对象

您可以从表中检索所选对象的行，并将焦点移动到该对象。例如，在项目树中选择一个字段，然后按Ctrl + Alt + B.焦点移动到新选项卡中的相应列。

要导航到结果集中的对象，请选择一个对象并转到Navigate |相关符号 （ Ctrl + Alt + B ）。

在相关数据之间导航

您可以在不同类型的相关数据之间导航：

  • 引用数据 ：当前对象引用的数据。

  • 引用数据 ：引用当前对象的数据。

  1. 右键单击表中的列，然后导航到“ 转到”参考数据 （ Ctrl + B ）或转到|引用数据 （ Ctrl + Alt + B ）。

  2. 如果出现“ 选择目标”弹出窗口，请选择表格以及要显示的行：

    • 第一个引用行 ：表中的所有行。选择引用当前行的第一行。

    • 仅引用行 ：仅引用当前行的行。

  3. 按Enter键

排序数据

要对列中的表数据进行排序，请单击标题行中的单元格。

数据按升序排序。标记右侧的数字（图片中的1）是排序级别。您可以按多列排序。在这种情况下，不同的列将具有不同的排序级别。

数据按降序排序。

您可以打开Sort via ORDER BY选项以启用相应DBMS对数据进行排序。

此外，您可以在“ 结构”工具窗口（“ 视图”|“工具窗口”|“结构” ）中按列对数据进行排序。在“ 结构”工具窗口中，右键单击列名称，然后从“ 排序”菜单中选择排序方法。

重新排序列

要重新排序列，请对标题行中的相应单元格使用拖放操作。

隐藏列

要隐藏列，请右键单击相应的标题单元格，然后选择“ 隐藏列” 。

或者，您可以右键单击标题行中的任何单元格，然后选择列列表 （ Ctrl + F12 ）。在“ 列列表”窗口中，选择列名称并按空格键。您可以隐藏所有列。要搜索列列表，只需在“ 列列表”窗口中键入列名即可。

显示列

  1. 右键单击标题行中的任何单元格，然后选择“ 列列表” 。或者，按Ctrl + F12 。

  2. 选择隐藏列（删除线列名称）并按空格键。

您可以在“ 结构”工具窗口（“ 查看”|“工具窗口”|“结构” ）中查看整个表结构。“ 结构”工具窗口显示所有列。

重置初始表视图

单击设置图标（ ）并选择重置视图 。结果，数据变为未排序，列按最初定义的顺序显示，并显示所有列。

结构视图中列的操作

结构视图显示所选表中所有列的列表。结构视图在结构工具窗口（ 视图|工具窗口|结构 ）或弹出窗口（ Ctrl + F12 ）中可用。

您可以使用列执行以下操作：

  • 按升序（ Shift + Alt + Up ）或降序（ Shift + Alt + Down ）对数据进行排序。

  • 重置排序状态（ Ctrl + Shift + Alt + Backspace ）

  • 隐藏列或显示隐藏列（ 空格 ）。显示隐藏列的名称。

右键单击表标题行中的单元格，然后选择“ 列列表” 。

或者，要对列数据进行排序，请使用上下文菜单。

使用快速文档视图

快速文档视图提供有关所选单元格中的值的详细信息。例如，如果单元格包含长文本，您将看到整个文本。如果单元格包含图像，您将看到图像的预览。此外，您还可以查看当前记录中引用的记录以及引用当前记录的记录。

记录的快速文档视图显示当前记录中引用的记录以及引用当前记录的记录。

要转置表格，请单击设置图标（ ）并选择Transpose 。

为列注入语言

您可以为整个列分配正式语言（例如HTML，CSS，XML，RegExp等）。因此，当您在单元格内部有一段代码属于该列时，您可以使用代码生成，代码完成和IDE功能的其他功能。

要为整列注入语言，请右键单击相应的标题单元格，然后选择“ 编辑为” 。在支持的语言列表中，选择要注入的语言。

同时编辑多个单元格值

  1. 选择要编辑的单元格范围。

  2. 开始输入新值。更改将应用于所有选定的单元格。

  3. 要确认该值，请按Enter键 。要取消编辑，请按Escape 。

添加一行

  1. 单击“ 添加新行”图标（ ）在工具栏上。或者，右键单击该表，然后从上下文菜单中选择“ 添加新行 ”。

  2. 按Alt + Insert 。

请注意，上下文菜单克隆行命令（ Ctrl + D ）可用作替代方法。

删除一行

  1. 选择要删除的行。

    通过单击显示行号的列中的单元格来选择行。要选择多行，请使用鼠标单击和Ctrl键组合。

  2. 单击“ 删除行”图标（ ）在工具栏上。或者，按Ctrl + Y或删除 。

手动提交更改

默认情况下，不会自动提交表中的更改。首先，您在表的本地副本中进行更改，然后您需要将所有更改提交到数据库。

要将更改提交到数据库，请单击“ 提交”图标（ ），或按Ctrl + Enter 。

自动提交更改

默认情况下，除非您按“ 提交”图标，否则不会将更改提交到数据库（ ），或按Ctrl + Enter 。您可以启用“ 立即提交更改”选项以自动提交更改。

  1. 在设置（ Ctrl + Alt + S ）中，导航到Database |数据视图 。

  2. 选择立即提交更改 。

  3. 单击确定 。

使用数据库事务提交更改

您可以选择自动或手动提交事务。要更改提交模式，请使用工具栏上的Tx下拉菜单。

如果您向数据库服务器提交更改并且提交模式设置为“ 自动” ，则会隐式提交值，行或列的每次更改，并且无法回滚。

如果提交模式设置为“ 手动” ，则可以通过“ 提交”图标显式提交或回滚提交的更改（ ）或回滚图标（ ）。否则，从上下文菜单中选择Commit （ Ctrl + Shift + Alt + Enter ）或Rollback命令。

Tx开关还可用于选择事务的隔离级别 。

还原更改

在提交更改之前，您可以还原它们。

revert命令的范围由表中的当前选择定义：该命令仅应用于选择内的更改。因此，您可以还原单个更改，一组更改或所有更改。如果当前未选择任何内容，则将revert命令应用于整个表。

要还原未提交的更改，请选择并右键单击一个单元格或多个单元格。选择Revert Selected （ Ctrl + Alt + Z ）。

比较表格

您可以将当前表与在数据编辑器中打开或在数据库控制台工具窗口中显示的任何其他表进行比较。

  1. 选择一张桌子。

  2. 单击Compare with图标（ ），并选择另一个表或结果集。

  3. 如果需要，请在比较对话框中更改Tolerance参数的值。Tolerance参数定义两个结果集之间允许的最大差异数。例如，如果要将两行视为相等（如果它们的数据在单个列中不同），请在“ 容差”字段中输入1 。

将表导出到文件

将表数据复制到剪贴板或将其保存在文件中时，数据将转换为可用的输出格式之一。作为输出格式，您可以选择SQL INSERT或UPDATE语句， TSV或CSV ，HTML表或JSON数据。

打开表格或结果集，单击“ 转储数据”图标（ ）并选择To File 。

将表导出到剪贴板

在工具栏上，单击“ 转储数据”图标（ ）并选择To Clipboard 。

在表之间复制和粘贴单元格

您可以复制（ Ctrl + C ）并粘贴（ Ctrl + V ）所选单元格和单元格范围 - 在同一个表格中或从一个表格复制到另一个表格。粘贴时，IntelliJ IDEA会根据需要自动转换数据类型。

配置数据输出格式和选项

您可以使用预定义的输出格式，自定义格式和脚本配置如何将提取的数据转换为文本。要为提取的数据配置输出，请右键单击表中的任何单元格，然后指向“ 数据提取器” 。

在此菜单中，您可以选择输出格式（例如，作为SQL INSERT语句，HTML表格或CSV格式）或将数据转换为特定格式的脚本。还提供以下附加选项：

跳过计算列（SQL）
允许排除具有计算值的列（影响SQL输出格式）。
跳过生成的列（SQL）
允许排除具有自动增量值的列（影响SQL输出格式）。
添加表定义（SQL）
允许包含CREATE TABLE语句（影响SQL输出格式）。

此外，您还可以配置以下参数：

  • 配置CSV格式。 打开CSV格式对话框，您可以使用该对话框管理分隔符分隔值格式（例如，CSV或TSV）。

  • 转到脚本目录。 打开一个目录，其中包含将表数据转换为各种输出格式的脚本。

将数据导出到数据库

  1. 确保在数据库工具窗口中将数据库添加为数据源 。

  2. 右键单击表中的任何单元格，然后单击“ 导出到数据库” 。

  3. 指定数据库，目标模式（以使用导出的数据创建新表）或表（将导出的数据添加到现有表）。

  4. 配置目标表的数据映射和设置。

将LOB保存在文件中

您可以将二进制大对象 （BLOB或LOB）从单元格保存到文件中。

  1. 右键单击包含LOB的单元格，然后选择“ 保存LOB” 。

  2. 在保存对话框中，指定文件的名称和位置，然后单击“ 确定” 。

更新表视图

如果要将编辑器中显示的数据与数据库的内容同步，则需要更新表视图。或者，如果要在更改后应用页面大小限制设置。

要刷新表视图，请考虑以下操作：

  • 单击“ 同步”图标（ ）在工具栏上。

  • 右键单击该表，然后从上下文菜单中选择“ 重新加载页面 ”。

  • 按Ctrl + F5 。

查看生成表的查询

若要查看用于生成表的查询 ，请单击工具栏上的“ 查看查询 ”。如有必要，您可以选择查询文本并将其复制到剪贴板（ Ctrl + C ）。

要关闭显示查询的窗格，请按Escape 。
