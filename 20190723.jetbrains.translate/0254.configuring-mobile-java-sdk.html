在IntelliJ IDEA中定义JDK和移动SDK

此功能仅在Ultimate版本中受支持。

为了能够开发Java移动应用程序，您应该定义要在IntelliJ IDEA中使用的JDK和Java移动工具包。

在IntelliJ IDEA中定义JDK和移动SDK

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ）。

  2. 在最左侧窗格的“ 平台设置”下 ，单击“ SDK” 。

  3. 在右侧窗格上方，单击并选择JDK 。

  4. 在打开的对话框中 ，选择要使用的JDK的安装目录，然后单击“ 确定” 。

  5. 点击并选择Mobile SDK 。

  6. 在打开的对话框中 ，选择Java移动工具包（WTK或DoJa）的安装目录，然后单击“ 确定” 。

  7. 如有必要，请配置自定义配置文件并在SDK页面上指定preverify选项（在对话框的右侧）。

  8. 在Project Structure对话框中单击OK 。
