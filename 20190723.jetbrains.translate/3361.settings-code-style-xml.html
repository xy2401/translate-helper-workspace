标签和缩进

使用制表符

  • 如果选中此复选框，则选项卡字符用于缩进和代码重新格式化。

  • 清除该复选框后，WebStorm使用空格而不是制表符。

智能标签

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

缩进由两部分组成。一部分来自嵌套代码块，另一部分是通过对齐来确定的。

  • 如果选中此复选框，则嵌套产生的部分包含制表符和空格（如果需要），而由对齐定义的部分仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组符合指定选项卡大小的空格后，将自动替换为选项卡，这可能会导致细微对齐。

标签大小

在此文本框中，指定适合选项卡的空格数。

缩进

在此文本框中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

继续缩进

在此文本框中，指定要在数组元素之间，表达式，方法声明和方法调用之间插入的空格数（或选中“ 使用选项卡字符”复选框）。

在空行上保持缩进

如果选中此复选框，则WebStorm会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，WebStorm将删除空行上的制表符和空格。

描述

右边距

使用这些设置指定元素右侧所需的边距空间。如果选择“ 默认”选项，则将使用全局设置中右边距的值。

打字输入

使用这些设置设置指定编辑的文本在指定的右边距中的拟合方式。您可以选择以下选项之一：

  • 默认值 - 在这种情况下，WebStorm使用全局设置中指定的Wrap on typing选项。

  • 是 - 在这种情况下，使用指定的右边距中的值。

  • 否 - 在这种情况下，此选项将关闭，并且一行可以超过右边距中指定的数字。

保持换行符

选中此复选框可在编辑器中查看XML文件时使WebStorm荣誉换行符。

在文本中保留换行符

选中此复选框可在编辑器中查看XML文件时使属性中的WebStorm荣誉换行符（例如，冗长的描述）。

保持空白

在此文本框中，指定重新格式化后要保留的最小连续空行数。

换行属性

使用此下拉列表确定应如何包装属性行。可用选项包括：

  • 不包装 - 如果选择此选项，则不会对代码应用特殊包装样式。

  • 如果长时间换行 - 选择此选项可使线条超出右边距并用适当的缩进包裹。

  • 如果长时间缩小 - 选择此选项可使列表中的元素超出右边距，使每行一个元素具有适当的缩进。

  • 换行总是 - 选择此选项可以将列表中的所有元素都包装为每行一个元素并使用适当的缩进。

换行文字

选中此复选框可根据代码样式设置包裹长行。

对齐属性

选中此复选框可使顺序行中的属性对齐。

保持白色空间

选中此复选框后，编辑器将保留标记内的所有空格。这同样也指缩进和换行。

空间

在此区域中，定义属性和标记名称的空格用法。

  • 属性中的“=”周围 ：选中此复选框可在属性中的“=”符号周围添加空格。

  • 标记名称后 ：选中此复选框以在标记名称后添加空格。

  • 在空标记中 ：选中此复选框可在空标记中添加空格。

CDATA

在此区域中，定义MXML文件中CDATA部分周围和内部的空格的用法：
  • 周围的空白 ：从下拉列表中选择如何处理CDATA周围的空白。

    • 保留 ：重新格式化后，所有空格都将保持不变。

    • 删除（保留标签） ： CDATA周围的所有空格都将被删除，标签将保留在同一行。

    • 新行 ： CDATA之前和之后将添加新行。

  • 保留内部空格 ：如果选中此复选框，则在CDATA[和之前]]之后将保留空格。

安排

此选项卡允许您定义一组规则，根据您的偏好重新排列代码。

项目

描述

匹配规则

使用此区域将元素顺序定义为规则列表，其中每个规则都有一组匹配项，例如修饰符或类型。

  • - 使用此按钮添加规则。空规则区域打开。

  • - 使用此按钮从列表中删除规则。

  • - 使用此按钮编辑现有规则。要查看此按钮，请导航到要编辑的规则，然后单击按钮。在打开的弹出窗口中，修改规则字段。

  • - 使用这些按钮向上或向下移动所选规则。

空规则

使用此区域可以创建新的匹配规则或编辑现有规则。您可以从以下过滤器中进行选择：

  • 类型 - 使用此过滤器为规则选择构造函数，方法，字段或属性。
    请注意，单击类型关键字两次会使条件无效。

  • 名称 - 使用此字段指定规则中的条目名称。此筛选器仅匹配条目名称，例如字段名称，方法名称，类名称等。筛选器支持正则表达式并使用标准语法 。匹配是针对整个名称执行的。

  • 命名空间 - 使用此字段指定规则中的命名空间。它允许您指定控制命名空间属性位置的规则。

  • 订单 - 使用此下拉列表选择规则的排序顺序。当多个元素使用相同的匹配规则时，此选项很有用。在这种情况下，选择Keep order将保持与重新排列之前设置的顺序相同，并选择Order by Name将按名称对具有相同匹配规则的元素进行排序。

从“ 订单”列表中选择“按名称排序”时，将显示此图标。该图标表示此规则中的项目按字母顺序排序。

其他设置

使用此区域设置其他排列选项。“ 强制重新排列”下拉列表允许您选择“ 重新 格式代码”对话框中影响“ 重新排列条目”复选框的选项。
您可以选择以下选项：

  • 使用当前模式（在“重新格式代码”对话框中切换） - 在这种情况下，“ 重新排列条目”复选框保持活动状态，您可以在“ 重新格式代码”对话框中对其进行修改。

  • 始终 - 在这种情况下， 重新排列条目复选框被选中并变为只读。

  • 从不 - 在这种情况下， 重新排列条目复选框被清除并变为只读。
