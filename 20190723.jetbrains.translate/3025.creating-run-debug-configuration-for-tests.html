为测试创建运行/调试配置

您可以使用运行/调试配置运行测试（测试用例，测试套件等），其方式与运行普通应用程序类似。WebStorm提供了一个框架，用于创建用于测试目的的特殊运行/调试配置，其中可以将测试指定为目标。

创建测试配置

  1. 通过执行以下操作之一打开“ 运行/调试配置”对话框：

    • 在主菜单上，选择“运行”|编辑配置 。

    • 按Shift + Alt + F10并从上下文菜单中选择“ 编辑配置 ”。

  2. 点击在工具栏上，选择所需的配置类型：

  3. 在打开的对话框中，指定测试执行前要执行的测试范围，配置参数和活动。应用更改并关闭对话框

自2018年5月以来，不推荐使用Nodeunit。有关详细信息，请参见Nodeunit官方网站。
