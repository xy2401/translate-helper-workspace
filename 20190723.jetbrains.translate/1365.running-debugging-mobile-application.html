运行和调试Java移动应用程序

此功能仅在Ultimate版本中受支持。

为了能够运行或调试Java移动应用程序，您应首先创建相应的运行/调试配置。然后，您应该在运行或调试模式下启动此运行/调试配置。

  • 创建J2ME运行/调试配置
  • 启动J2ME运行/调试配置

创建J2ME运行/调试配置

  1. 选择运行|编辑配置 。

  2. 在“ 运行/调试配置”对话框中 ，单击“确定” 并选择J2ME 。

  3. 根据需要指定配置设置，然后单击“ 确定” 。有关更多信息，请参阅运行/调试配置：J2ME 。

启动J2ME运行/调试配置

  1. 确保在工具栏上的相应选择器中选择了移动应用程序的运行/调试配置。

  2. 现在，要在运行或调试模式下启动此配置：

    • 选择运行|运行或按Shift + F10 。

    • 选择运行|调试或按Shift + F9 。
