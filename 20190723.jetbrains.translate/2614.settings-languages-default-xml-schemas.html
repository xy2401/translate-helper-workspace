  • 默认HTML语言级别

默认HTML语言级别

通常，HTML或XHTML文件具有<!DOCTYPE>声明，声明源文件中使用的语言级别 。此语言级别用作验证文件内容的标准。如果HTML或XHTML文件没有<!DOCTYPE>声明，将根据默认标准（模式）验证文件的内容。在“ 默认HTML语言级别”区域中，选择默认架构以验证HTML和XHTML文件，而不使用<!DOCTYPE>声明。可用选项包括：

  • HTML 4或HTML 5 ：选择其中一个选项，将文件视为HTML 4或HTML 5，并根据其中一个标准进行验证。

  • 其他doctype ：选择此选项可使HTML文件默认再次验证自定义DTD或模式，并指定要使用的DTD或模式的URL。

    请注意，此字段中提供了代码完成功能：按Ctrl + Space可查看建议的URL列表。

默认的XML模式版本

在此区域中，选择XSD（XML架构定义）架构以验证XML文件。可用选项包括：

  • XML Schema 1.1请参阅W3C XML模式定义语言（XSD）1.1第1部分：结构和W3C XML模式定义语言（XSD）1.1第2部分：数据类型以获取详细信息。

  • XML Schema 1.0有关详细信息，请参阅XML架构第1部分：结构第二版和XML架构第2部分：数据类型第二版 。
