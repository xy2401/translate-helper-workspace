PyCharm 2019.1帮助

Alt键

本节列出并描述了包含Alt键的键盘快捷键：

  • Alt +字母数字键
  • Alt +导航键
  • Alt +功能（F）键

Alt +字母数字键

捷径

功能

使用此快捷方式...

ALT + O

导出到文本文件

将工具窗口的内容导出到文本文件。

ALT + Q

背景信息

不可见时显示当前方法或类声明。

ALT +号码

打开工具窗口

打开带有相应编号的工具窗口。

Alt + /

代码完成/扩展字

将插入符号处的字符串展开为可见范围内以相同字符开头的任何单词。

Alt +`

VCS操作

显示包含最多版本控制命令的快速列表。

Alt +导航键

捷径

功能

使用此快捷方式...

Alt + Delete键

安全删除

删除选中的类/方法/字段，检查其用法。

Alt + Enter键

显示意向行动

显示插入符号当前所在代码的意图操作 （如果有）。

Alt + Home

激活导航栏

将焦点放在导航栏上。

Alt +插入

创建新实体

根据具体情况：

  • 在导航视图中：使用“ 新建”弹出窗口创建新的类，文件或目录。

Alt +向下

导航到下一个方法

导航到活动编辑器选项卡中的下一个方法声明。

Alt +左

选择上一个标签

根据具体情况：

  • 在编辑器或视图中打开多个选项卡时，打开左侧的下一个选项卡（如果当前的选项卡是第一个选项卡，则打开最后一个选项卡）。

  • 在“ 版本控制”工具窗口的“ 更新项目信息”选项卡中调用的文件差异查看器中 ，将以前文件的本地副本与服务器中的更新进行比较。

Alt +右

选择Next Tab

根据具体情况：

  • 在编辑器或视图中打开多个选项卡时，打开右侧的下一个选项卡（如果当前选项卡是最后一个选项卡，则打开第一个选项卡）。

  • 在“ 版本控制”工具窗口的“ 更新项目信息”选项卡中调用的文件差异查看器中 ，将下一个文件的本地副本与服务器中的更新进行比较。

Alt +向上

导航到“上一个方法”

导航到活动编辑器选项卡中的上一个方法声明。

Alt +功能（F）键

捷径

功能

使用此快捷方式...

ALT + F1

选择目标

将焦点从当前文件，类，方法或引用移动到数据源表到“ 选择目标”弹出窗口中建议的视图。请参阅源代码导航 。

Alt + F7

查找用法

开始搜索用法 。

Alt + F8

评估表达

调试器：评估任意表达式。

Alt + F9

运行到光标

调试器：运行到插入符所在的行。
