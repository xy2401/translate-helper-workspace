PyCharm 2019.1帮助

从框架模板生成项目

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

PyCharm可以为您创建新的框架特定的应用程序，设置它们的结构，并在必要时下载所需的包。您也可以使用Yeoman发电机，详情请参阅Yeoman 。

HTML5 Boilerplate，Web Starter Kit，Bootstrap或Foundation应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，根据您的首选项和要在应用程序中实现的功能选择要使用的模板：

    • HTML5 Boilerplate：选择此选项以使用HTML5 Boilerplate模板。这是一个可以轻松适应您需求的初始项目模板。

    • Web Starter Kit：选择此选项以使用Web Starter Kit 。

    • Bootstrap：选择此选项以使用Bootstrap模板，这是一个精心设计的模块化工具包，具有丰富的HTML，CSS和JavaScript支持。

    • Foundation：选择此选项以使用Foundation框架。

  3. 右侧窗格中的控件集取决于所选的模板。

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 从“ 版本”列表中，选择要使用的模板版本，然后单击“ 创建” 。

Node.js Express应用程序

有关详细信息，请参阅Node.js。

在你开始之前：

  1. 安装Node.js并将其配置为本地Node.js解释器 。

  2. 在插件页面上安装并启用NodeJS存储库插件，如管理插件中所述 。

生成Node.js应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Node.js Express App 。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. Node解释器，请参阅配置本地Node.js解释器 。

    3. 包管理器使用- NPM或纱线 ，看到NPM和纱线 。

    4. 要使用的Express模板引擎 。从“ 模板引擎”下拉列表中，选择以下选项之一：

      • 灰尘

      • EJS

      • 把手

      • Hogan.js

      • 帕格 （ 帕格（玉） ）

      • 枝条

      • VASH

      • 静态HTML

    5. 要使用的CSS引擎。从CSS引擎下拉列表中，选择以下选项之一：

      • 简单的CSS

      • 唱针

      • 减

      • 指南针 。

      • 萨斯 。

  4. 准备好后，单击“ 创建” 。PyCharm启动Express Project Generator工具，该工具可下载所有必需的数据，设置项目结构，并根据您的选择在当前窗口或新窗口中打开项目。

流星应用

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Meteor App 。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 指定Meteor可执行文件的位置（请参阅安装Meteor ）。

    3. 从“ 模板”下拉列表中，选择要生成的样本。要生成基本项目结构，请选择“ 默认”选项。

    4. 在“ 文件名”文本框中，键入将生成的相互关联的.js ， .html和.css文件的名称。仅当从“ 模板”下拉列表中选择“ 默认”样本类型时，该字段才可用。

  4. 单击“ 创建”时 ，PyCharm会生成Meteor应用程序的框架，包括HTML文件，JavaScript文件，CSS文件和带有子文件夹的.meteor文件夹。PyCharm自动标记.meteor / local文件夹，用于存储构建的应用程序， 排除但仍显示在项目树中。

    默认情况下， 排除的文件显示在项目树中。要隐藏.meteor/local文件夹，请单击在“ 项目”工具窗口的工具栏上，删除“ 显示排除的文件”选项旁边的勾选。

PhoneGap / Cordova / Ionic应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Cordova App 。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 指定可执行文件phonegap.cmd或cordova.cmd或ionic.cmd （请参阅安装PhoneGap / Cordova / Ionic ）。

  4. 单击“ 创建”时 ，PyCharm将生成具有特定于框架结构的PhoneGap / Cordova / Ionic应用程序的框架。

AngularJS应用程序

PyCharm可以使用AngularJS生成用于开发应用程序的项目存根。要生成Angular存根，请使用Angular CLI ，请参阅创建新的Angular应用程序 。

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择AngularJS 。

  3. 在右侧窗格中，指定将存储与项目相关的文件的文件夹的路径。

  4. 单击“ 创建”时 ，PyCharm将根据AngularJS种子项目生成包含所有必需配置文件的AngularJS特定项目结构

  5. 下载包含AngularJS代码的AngularJS依赖项以及支持开发和测试的工具：在嵌入式终端 （ Alt + F12 ）中，键入：

    npm安装

    在readme.md文件的Install Dependencies部分中了解有关安装依赖项的更多信息。

使用create-react-app反应应用程序

了解有关从React官方网站安装React和创建React应用程序的更多信息。

PyCharm只保证使用react-scripts包运行和调试Jest测试。

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择React App 。

  3. 在右侧窗格中，指定项目文件夹，要使用的Node.js解释器以及create-react-app包的路径。

  4. 可选：
    在项目生成期间指定要使用的自定义程序包而不是react-scripts 。这可以是从react-scripts派生的一个包，例如， react-awesome-scripts ， custom-react-scripts ， react-scripts-ts等等。

  5. 单击“ 创建”时 ，PyCharm将生成一个包含所有必需配置文件的React特定项目。

使用vue-cli的Vue.js应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Vue.js。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 在Node Interpreter字段中，指定要使用的Node.js解释器。从列表中选择已配置的解释器，或选择“ 添加”以配置新的解释器

    3. 在vue-cli或@ vue / cli字段中，指定存储vue-cli或@vue/cli包的文件夹。

    4. 从“ 项目模板”列表中，选择要使用的Vue.js模板 。

  4. 单击“ 创建”时 ，PyCharm会向您询问有关可以添加到项目中的工具（如linters和测试运行器）的其他问题。这些问题取决于您选择的模板。之后，PyCharm生成一个包含所有必需配置文件的Vue.js特定项目。
