重新格式化源代码

PyCharm允许您重新格式化源代码以满足代码样式的要求。PyCharm将列出间距，缩进，关键字等。重新格式化可以应用于选定的文本，整个文件或整个项目。

也可以使用格式标记将重新格式化仅应用于源代码的各个部分。

重新格式化目录的代码

要重新格式化目录的代码，请按照下列步骤操作：

  1. 在“项目”工具窗口中，选择要应用重新格式化的目录。

  2. 选择代码|在主菜单上重新格式化代码或按Ctrl + Alt + L.

    或者，在“ 项目工具”窗口中 ，右键单击该目录，然后从上下文菜单中选择“ 重新格式化代码” 。

  3. 在“重新格式代码”对话框中，为重新格式化指定必要的选项和过滤器，然后单击“运行” 。

重新格式化当前文件的代码

要重新格式化当前文件的代码，请按照下列步骤操作：

  1. 在当前打开的文件的编辑器中，按Ctrl + Shift + Alt + L.

    请注意，如果选择Code |从主菜单重新格式化代码或按Ctrl + Alt + L ，PyCharm将尝试自动重新格式化源代码，而无需打开“ 重新格式化文件”对话框。

  2. 在“ 重新格式化文件”对话框中，指定重新格式化的选项，然后单击“运行” 。

重新格式化源代码时跳过区域

要启用格式化程序标记，请确保在“设置/首选项”对话框的“ 标记”页面中的注释中选中“ 启用格式化程序标记 ”复选框，然后在“ 格式化程序关闭/开启”字段中键入标记。

要在重新格式化时跳过某个区域，请按照下列步骤操作：

  1. 在区域的开头，创建一个行注释（ Ctrl + / ），然后手动键入在Code Style页面的Formatter off字段中指定的标记。

  2. 在区域的末尾，创建一个行注释（ Ctrl + / ），然后手动键入在Code Style页面的Formatter on字段中指定的标记。

  3. 执行代码重新格式化，如上所述。

或者，创建一个实时模板以使用格式化器关闭/打开标记来包围代码块，请参阅实时模板 。
