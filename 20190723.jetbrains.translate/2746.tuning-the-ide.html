PyCharm 2019.1帮助

调整PyCharm

除了可用的标准选项在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，PyCharm还允许您执行低级配置。

请注意，这可能会导致意外问题，并可能导致PyCharm安装无法运行。强烈建议您联系JetBrains支持部门 ，获取有关可能帮助您解决任何问题的选项和值的说明。

配置JVM选项

PyCharm在Java虚拟机（JVM）上运行，它具有控制其性能的各种选项。用于运行PyCharm的默认选项在以下文件中指定：

<IDE_HOME>\bin\pycharm64.exe.vmoptions （对于默认的64位JVM）
<IDE_HOME>\bin\pycharm.exe.vmoptions （用于可选的32位JVM）
PyCharm.app/Contents/bin/pycharm.vmoptions
<IDE_HOME>/bin/pycharm64.vmoptions （对于默认的64位JVM）
<IDE_HOME>/bin/pycharm.vmoptions （用于可选的32位JVM）

不要更改默认文件中的JVM选项，因为更新PyCharm时会替换它。此外，在macOS的情况下，编辑此文件会违反应用程序签名。

要配置JVM选项：

  • 在“ 帮助”菜单上，单击“ 编辑自定义VM选项” 。

如果未打开任何项目，请在“欢迎”屏幕上单击“ 配置” ，然后单击“ 编辑自定义VM选项” 。

PyCharm在配置目录中使用JVM选项创建文件的副本，并在新的编辑器选项卡中打开它。您在此文件中更改的任何值都将覆盖原始默认文件中的值。

如果无法启动PyCharm，可以手动将带有JVM选项的默认文件复制到PyCharm配置目录。

如果您没有对PyCharm配置目录的写访问权限，则可以添加PYCHARM_VM_OPTIONS环境变量以指定具有JVM选项的文件的位置。此文件中的值将覆盖原始默认文件和位于PyCharm配置目录中的副本的相应值。

如果您使用的是Toolbox App，它将管理安装和配置目录，并允许您为每个IDE实例配置JVM选项。打开工具箱应用程序，单击必要实例的螺母图标，然后选择设置 。

常见选项

在大多数情况下，JVM选项的默认值应该是最佳的。以下是最常修改的：

选项

描述

-Xmx

限制JVM可以为运行PyCharm分配的最大内存堆大小。默认值取决于平台。如果您遇到速度减慢，则可能需要增加此值，例如，将值设置为2048兆字节，将此选项更改为-Xmx2048m 。

-Xms

指定JVM为运行PyCharm分配的初始内存。默认值取决于平台。它通常设置为最大允许内存的一半（ -Xmx ），例如-Xms1024m 。

-XX：NewRatio

指定堆的年轻和老一代的大小之间的比率。在大多数情况下，建议使用2到4之间的比例。这将相应地将年轻代的大小设置为旧代的1/2到1/4，这在您经常处理一个项目并且一次只有几个文件时是很好的。但是，如果您不断打开新文件并在多个项目之间切换，则可能需要增加年轻代。在这种情况下，尝试设置-XX:NewRatio=1 ，这将使年轻一代与旧一代一样大，允许对象在年轻一代中保留更长时间。

有关可用JVM选项的更多信息，请参阅Windows或macOS / Linux的java参考。

配置平台属性

PyCharm使您可以自定义各种特定于平台的属性，例如用户安装的插件的路径和支持的最大文件大小。用于运行PyCharm的默认属性在以下文件中指定：

<IDE_HOME> \ BIN \ idea.properties
PyCharm.app/Contents/bin/idea.properties
<IDE_HOME> /bin/idea.properties

不要更改默认文件中的平台属性，因为更新PyCharm时会替换它。此外，在macOS的情况下，编辑此文件会违反应用程序签名。

要配置平台属性：

  • 在帮助菜单上，单击编辑自定义属性 。

如果未打开任何项目，请在“欢迎”屏幕上单击“ 配置” ，然后单击“ 编辑自定义属性” 。

PyCharm在配置目录中创建一个空的idea.properties文件，并在新的编辑器选项卡中打开它。添加到此文件的任何属性都将覆盖原始默认文件中的相应属性。

如果无法启动PyCharm，可以在PyCharm配置目录中手动创建一个空的idea.properties文件。

如果您没有对PyCharm配置目录的写访问权限，则可以添加PYCHARM_PROPERTIES环境变量以指定idea.properties文件的位置。此文件中的属性将覆盖原始默认文件和位于PyCharm配置目录中的相应属性。

共同属性

用户通常会在解决特定问题时更改以下属性：

  • 可能需要移动默认IDE目录的位置，例如，如果用户配置文件驱动器空间不足或位于慢速磁盘上，如果主目录已加密（放慢IDE）或位于网络驱动器上，如果要创建可移植安装或从主目录备份中排除缓存，等等。

    您可以使用带有任何属性名称的宏来插入其值，例如，使用${user.home} （标准Java系统属性）来指定相对于用户主目录的路径。

    属性

    通往...的路径

    idea.config.path配置目录
    idea.system.path系统目录
    idea.plugins.path插件目录
    idea.log.path记录目录

  • 可能影响性能的限制：

    属性

    描述

    idea.max.content.load.filesize

    PyCharm能够打开的文件的最大大小（以千字节为单位）。使用大文件会影响编辑器性能并增加内存消耗。默认值为20000 。

    idea.max.intellisense.filesize

    PyCharm为其提供代码帮助的文件的最大大小（以千字节为单位）。大文件的代码辅助会影响编辑器性能并增加内存消耗。默认值为2500 。

    idea.cycle.buffer

    控制台循环缓冲区的最大大小（以千字节为单位）。如果控制台输出大小超过此值，则删除最旧的行。要禁用循环缓冲区，请设置idea.cycle.buffer.size=disabled 。

    idea.max.vcs.loaded.size.kb

    PyCharm在比较更改时加载以显示过去文件内容的最大大小（以千字节为单位）。默认值为20480 。

所有路径都使用正斜杠（ / ）指定，包括Windows路径（例如， C:/idea/system).

PyCharm提供了许多其他属性来定义与环境的交互（窗口管理器，启动器，文件系统等）。它们中的大多数都像隐藏设置（在某种意义上它们没有明显暴露），在某些情况下可能需要启用或禁用。强烈建议仅在联系JetBrains支持部门寻求帮助后才能更改这些属性。

为PyCharm选择Java运行时

PyCharm包括JetBrains Runtime（基于OpenJDK 8），默认使用。

要切换用于运行PyCharm的Java运行时：

  1. 在“ 帮助”菜单上，单击“ 查找操作” ，或按Ctrl + Shift + A.

  2. 查找并选择Switch Boot JDK操作。

  3. 选择所需的JDK，然后单击“ 确定” 。

所选运行时的路径存储在PyCharm 配置目录的pycharm.jdk文件中。要恢复默认的JetBrains运行时，可以删除此文件，或修改指向另一个Java运行时的路径。

建议使用JetBrains Runtime，它修复了各种已知的OpenJDK和Oracle JDK错误，并提供了更好的性能和稳定性。

您还可以通过将PYCHARM_JDK环境变量与所需JDK主目录的路径相加来覆盖用于PyCharm的运行时。

默认IDE目录

默认情况下，PyCharm将用户特定的文件（配置，缓存，插件，日志等）存储在用户的主目录中。但是，如有必要，您可以更改存储这些文件的位置。

配置目录

PyCharm配置目录包含具有个人设置的XML文件，例如键盘映射，配色方案等。它也是用户定义的VM选项和平台属性文件的默认位置。

句法
％HOMEPATH％\。<产品> <版本> \配置
例
C：\用户\约翰\。PyCharm2019.1 \ CONFIG
句法
〜/库/首/ <产品> <版本>
例
〜/库/首选项/ PyCharm2019.1
句法
〜/ <产品> <版本> /配置
例
〜/。PyCharm2019.1 /配置

您可以使用idea.config.path属性更改PyCharm配置目录的位置。

您可以通过将配置目录中的文件复制到另一个PyCharm安装上的相应文件夹来共享您的个人IDE设置。确保PyCharm未运行以避免在关闭IDE时删除复制的文件。

如果通过IDE Settings Sync同步设置 ，则这些子文件夹位于配置目录中的jba_config下。

系统目录

PyCharm系统目录包含缓存和本地历史文件。

句法
％HOMEPATH％\。<产品> <版本> \系统
例
C：\用户\约翰\。PyCharm2019.1 \系统
句法
〜/库/缓存/ <产品> <版本>
例
〜/图书馆/缓存/ PyCharm2019.1
句法
〜/ <产品> <版本> /系统
例
〜/。PyCharm2019.1 /系统

您可以使用idea.system.path属性更改PyCharm系统目录的位置。

插件目录

PyCharm插件目录包含用户安装的插件。

句法
％HOMEPATH％\。<产品> <版本> \配置\插件
例
C：\用户\约翰\。PyCharm2019.1 \ CONFIG \插件
句法
〜/ Library / Application Support / <product> <version>
例
〜/ Library / Application Support / PyCharm2019.1
句法
〜/ <产品> <版本> /配置/插件
例
〜/。PyCharm2019.1 /配置/插件

您可以使用idea.plugins.path属性更改PyCharm插件目录的位置。

记录目录

PyCharm日志目录包含产品日志和线程转储。

句法

例
C：\用户\约翰\。PyCharm2019.1 \ SYSTEM \日志
句法

句法

例
〜/。PyCharm2019.1 /系统/日志

您可以使用idea.log.path属性更改PyCharm日志目录的位置。
