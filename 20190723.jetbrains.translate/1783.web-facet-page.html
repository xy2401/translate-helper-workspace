使用此页面编辑Web构面的设置。

项目

描述

名称

方面的名称。

部署描述符

形成应用程序的部署描述符列表。

（ Alt + Insert ）。创建web.xml并将其添加到列表中。如果列表中已存在相应的描述符，则该功能无效。

（ Alt + Delete ）。从列表中删除选定的描述符。

（ 输入 ）。将所选描述符替换为相同类型的另一个描述符。

添加Application Server特定描述符。 创建特定于应用程序服务器的部署描述符（例如glassfish-web.xml ， jboss-web.xml ）并将其添加到列表中。

Web资源目录

指定包含Web应用程序资源的目录，例如网页，图像等（ Web资源目录列）及其在相应工件中的位置（“ 路径相对于部署根目录”列）。

对于要包含在工件中的资源目录，应确保工件配置包含Web构面资源元素。（所有资源目录都作为一个整体包含在工件中。）在这种情况下，/在路径相对于部署根列对应于在工件构造元件的根部。

来源根源

选择要包含在工件中的源根 。注意：

  • 如果'<ModuleName>'编译输出元素存在于工件配置中，则所有源根的类都包含在相应的工件中，而不管此处选择哪个源根。

  • 要仅包括此页面上选择的源根中的类，请在配置工件时使用以下内容： | JavaEE Facet类| Web（在<ModuleName>中） 。因此，工件配置中的相应元素将显示为“<ModuleName>”模块：“Web”构面类 。
