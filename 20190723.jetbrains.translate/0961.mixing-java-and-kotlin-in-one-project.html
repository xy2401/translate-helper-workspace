在一个项目中混合使用Java和Kotlin

在本页面：

  • 在Kotlin项目中创建Java文件
  • 在Java项目中创建Kotlin文件

在Kotlin项目中创建Java文件

要在Kotlin项目中创建Java文件，请按照下列步骤操作

  1. 按照创建Kotlin-JVM项目或创建Kotlin-JavaScript项目一节中的描述创建Kotlin项目 。

  2. 在“ 项目工具”窗口中 ，选择要在其中创建Java类的目标包或目录。

  3. 按Alt + Insert ，从弹出菜单中选择Java Class，选择其类型，然后指定其名称。

在Java项目中创建Kotlin文件

要在Java项目中创建Kotlin文件，请按照下列步骤操作

  1. 在Java项目中，选择目标位置，按Alt + Insert并从弹出菜单中选择Kotlin文件 。

  2. 如果这是您第一次在Java项目中创建Kotlin文件，则横幅显示：

    单击配置链接，然后选择所需的配置程序：

    在打开的Create Kotlin Java Runtime Library对话框中，执行以下操作：

    • 选择要配置的模块（如果项目中有多个模块）

    • 指定应使用哪个Kotlin运行时库。

    请注意，将来不需要配置操作。
