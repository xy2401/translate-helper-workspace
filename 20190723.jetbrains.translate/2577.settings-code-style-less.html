标签和缩进

使用制表符

  • 如果选中此复选框，则选项卡字符用于缩进和代码重新格式化。

  • 清除该复选框后，PyCharm使用空格而不是制表符。

智能标签

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

缩进由两部分组成。一部分来自嵌套代码块，另一部分是通过对齐来确定的。

  • 如果选中此复选框，则嵌套产生的部分包含制表符和空格（如果需要），而由对齐定义的部分仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组符合指定选项卡大小的空格后，将自动替换为选项卡，这可能会导致细微对齐。

标签大小

在此文本框中，指定适合选项卡的空格数。

缩进

在此文本框中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

继续缩进

在此文本框中，指定要在属性值之间插入的空格数（或选中“ 使用选项卡字符”复选框的选项卡 ）。

在空行上保持缩进

如果选中此复选框，PyCharm会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，PyCharm将删除空行上的制表符和空格。

空白行

使用此选项卡可以定义PyCharm在重新格式化后保留并插入代码中的空白行的位置和数量。结果显示在“ 预览”窗格中。

保持最大空行

在此区域中，指定重新格式化后要保留的额外空行数。

最小空行

在此区域中，配置是否在顶级块和嵌套选择器周围有额外的空行。在每个选项旁边的文本框中，指定要保留的最小额外空行数。

这些设置不会影响第一个项目之前和之后的空白行数。

其他

在此选项卡中，指定要在重新格式化时应用的对齐，大括号和空格选项。

大括号放置

使用此下拉列表指定PyCharm应放置选择器的开括号的位置。可用选项包括：

  • 在行尾

  • 下一行

对齐值

使用此下拉列表指定PyCharm应如何对齐属性和值。可用选项包括：

  • 不对齐 ：选择此选项可指定属性名称的第一个字符的对齐方式。

  • 在值上 ：选择此选项可指定属性值的第一个字符的对齐方式。

  • 在结肠上

将右大括号与属性对齐

如果选中此复选框，则选择器的右括号将放在属性列表下。
如果未选中此复选框，则选择器的右括号将放置在选择器下方。

保持单行块

如果选中此复选框，则具有单个属性的块将限制为一行。
如果未选中此复选框，则每个属性都将放置在其自己的行中。

空间

选中此区域中的复选框以在冒号分隔键和值之后以及选择器的左括号之前添加空格。

HEX颜色

使用此区域配置十六进制颜色语法。您可以从以下选项中进行选择：

  • 将十六进制颜色转换为 - 选中此复选框以配置十六进制颜色字母大小写。您可以选择小写或大写 。

  • 将十六进制颜色格式转换为 - 选中此复选框以配置十六进制颜色格式长度。您可以选择长格式或短格式 。

在“ 预览”窗格中查看更改。
