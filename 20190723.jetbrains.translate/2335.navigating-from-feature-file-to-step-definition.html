从.feature文件导航到步骤定义

  1. 在编辑器中打开所需的.feature文件。

  2. 执行以下操作之一：
    • 按住Ctrl键，将鼠标指针悬停在一个步骤上。该步骤转为超链接，其参考信息显示在工具提示中：

      单击超链接。步骤定义文件在编辑器中打开，插入符号位于所需的步骤定义中。

    • 在主菜单上，选择“ 导航”|宣言 。

    • 按Ctrl + B.
