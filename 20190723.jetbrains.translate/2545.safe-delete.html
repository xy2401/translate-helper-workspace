安全删除

介绍

安全删除重构可让您安全地从源代码中删除文件。

为了确保删除是安全的，PyCharm会查找要删除的文件的用法。如果找到了这样的用法，您可以在实际删除符号之前浏览它们并在代码中进行必要的更正。

执行重构

  1. 在“ 项目”工具窗口中，选择要删除的文件。

  2. 执行以下操作之一：

    • 按Alt + Delete 。

    • 选择Refactor |安全从主菜单或上下文菜单中删除 。

    • 选择Refactor |从主菜单（ Ctrl + Shift + Alt + T ） 重构此项 ，然后选择安全删除 。

  3. 在“ 安全删除”对话框中 ，选择所需选项，然后单击“ 确定” 。

  4. 如果重构可能不安全，则会打开“ Usages Detected”对话框。
    • 查看用法。 单击此按钮可查看代码中您要删除的项目的位置。结果，将打开“ 查找”工具窗口 。

      分析您的代码并进行必要的更正。然后单击“ 执行重构” 。（如果要从重新开始重新运行重构，请单击“ 重新运行安全删除” 。PyCharm将再次检查重构是否安全。）

    • 无论如何删除。 单击此按钮可删除该项目，而无需查看其用法。
