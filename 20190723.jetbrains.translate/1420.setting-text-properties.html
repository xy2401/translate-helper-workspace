设置文本属性

将组件放在表单工作区后，Text属性的就地编辑器会自动激活。

要稍后激活就地编辑器，请选择所需的组件并双击它，或按F2 。最后，您可以选择一个组件并开始输入 - 在这种情况下，现有文本将丢失。

如果需要输入冗长文本或从资源束中获取值，请参阅以下过程。

编辑文本属性

  1. 单击省略号按钮。在“编辑文本”对话框中，指定是否要将文本编辑为字符串或资源包。

  2. 如果已选择“ 字符串”选项，请在“ 值”字段中键入所需的文本，然后继续执行步骤4。

  3. 如果已选择“ 资源包”选项，请指定所需资源包的名称，键和值。

    • 单击“分发包名称”字段旁边的省略号按钮，或按Shift + Enter 。在“选择属性文件”对话框中，在项目树视图中查找所需的资源包，或单击“按名称搜索”选项卡，键入搜索字符串，然后从建议列表中选择资源包。

    • 单击“ 密钥”字段旁边的省略号按钮，或按Shift + Enter 。在“选择属性键”对话框中，从列表中选择所需的键。

    在“ 值”字段中键入值。

  4. 如果要为属性启用i18n支持 ，请确保清除选项Value不需要国际化 。在这种情况下，建议为此组件使用意图操作i18nize属性文本 。如果选择该选项，则禁止意图操作。

  5. 单击确定 。

当您为JList或JComboBox调用就地编辑器时，将打开一个编辑器对话框，使您可以键入列表的元素。
