IDE组件之间的导航

在本节中，您可以找到用于导航的键盘快捷键：

  • 视图和Windows 。

  • 差异 。

视图和Windows

功能

捷径

使用此快捷方式...

选择目标

ALT + F1

将焦点从当前文件，类，方法或引用移动到数据源，移动到“ 选择目标”弹出菜单中建议的视图。请参阅使用Select In弹出窗口导航 。

全部收缩

Ctrl + NumPad -

折叠树视图中的所有节点。

展开全部

Ctrl + NumPad Plus

展开树视图中的所有节点。

切换器

按Ctrl + Tab键

在编辑器中打开的文件和工具窗口之间导航。

打开工具窗口

ALT +号码

打开具有指定编号的工具窗口。

隐藏活动窗口

Shift +逃生

隐藏当前活动的工具窗口。

跳转到最后一个窗口

F12

激活最后一个聚焦工具窗口。

差异

功能

捷径

使用此快捷方式...

转向下一个差异

F7

导航到视图中的下一个差异。

转到上一个差异

按住Shift + F7

导航到之前的视图差异。
