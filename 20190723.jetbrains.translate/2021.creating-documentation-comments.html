为Python函数创建文档注释

为Python函数创建文档注释

  1. 在声明要记录的功能后放置插入符号。

  2. 输入打开三重引号，然后按Enter键或空格 键 。

  3. 添加有意义的参数描述和返回值。

请注意以下事项：

  • 在键入打开三重引号后按空格键生成文档字符串仅在编辑器设置的页面智能键中清除复选框插入对引用时才有效。

  • 如果重命名函数的参数，PyCharm将相应地更新文档注释中的标记。

使用意图操作为Python函数创建文档注释

  1. 将插入符号放在要记录的函数中的某个位置。

  2. 按Alt + Enter显示可用的意图操作。

  3. 选择插入文档字符串存根 ：

    PyCharm根据docstring格式生成文档注释存根，在Python集成工具页面中选择。

Python注释的示例

考虑以下功能：

def 句柄 （ self ， myParam1 ， myParam2 ）：

在Python Integrated Tools页面中，选择Epytext 。然后键入打开的三重引号，然后按Enter或Space 。PyCharm生成文档注释存根：

''' @ param self： @param myParam1： @param myParam2： @return： '''

然后选择reStructuredText ，键入开头的三重引号，然后按Enter或Space 。PyCharm生成文档注释存根：

''' ：param self :: param myParam1 :: param myParam2 :: return： '''
