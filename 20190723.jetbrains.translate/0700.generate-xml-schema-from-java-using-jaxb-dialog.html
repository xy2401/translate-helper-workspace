使用此对话框可基于现有Java代码配置XML模式生成。

  1. 此功能通过WebServices捆绑插件提供，该插件默认启用。如果没有，请按照管理插件一节中的说明启用它。

  2. 在活动编辑器选项卡中打开Java类时，可以使用菜单项和对话框。
