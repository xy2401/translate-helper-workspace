按Ctrl + Alt + S

使用此页面可搜索所选键盘映射中的快捷方式和操作，创建，编辑和删除自定义键盘映射，以及更改与自定义键盘映射中的操作相关联的快捷方式。

请注意， 预定义的键盘映射不可编辑。当您第一次重新配置快捷方式关联时，PyCharm将自动创建当前所选预定义键映射的副本，然后该键映射变为可编辑

编辑快捷方式

输入

更改所选操作的快捷方式。可以删除现有的快捷方式，并添加新的快捷方式。从列表中选择必要的操作：

  • 选择“ 添加键盘快捷键”选项以打开“ 输入键盘快捷键”对话框，您可以在其中指定要分配给当前键映射中所选操作的击键组合。

  • 选择“ 添加鼠标快捷方式”选项以打开“ 输入鼠标快捷方式”对话框，您可以在其中指定鼠标单击和要分配给当前键映射中所选操作的按钮的组合。

  • 选择“ 删除<快捷方式>”选项以从所选操作中删除所选快捷方式。

这些命令在“ 操作”内容窗格中的操作的上下文菜单中重复。

搜索操作的内容窗格。键入搜索字符串时，将显示与搜索模式匹配的操作。

先前使用的搜索模式存储在搜索历史列表中。要将搜索字符串添加到历史记录列表，请按Enter键 。

点击显示先前搜索的历史列表。

点击从字段中清除当前搜索模式。

请参阅配置键盘快捷键一节以了解如何指定键盘快捷键。

具有与指定条件匹配的快捷方式的操作将显示在操作的内容窗格中。

清除过滤

单击此按钮可还原内容窗格中的初始操作列表。

项目

描述

所有行动

此内容窗格显示PyCharm中当前可用的所有操作。这些操作根据其使用区域分组在节点下方。

  • 请注意，默认键盘映射不可编辑。只要您尝试更改与操作关联的键盘快捷键，就会自动创建相应键映射的副本。

  • 如果某些操作没有映射的键盘快捷键，则仍可以通过“ 转到操作”调用它们。

快捷键

此只读字段显示与当前键映射中所选操作关联的快捷方式列表。

快捷方式取决于平台。但是，某些键盘布局上缺少某些键。例如，笔记本电脑上没有PageUp / PageDown键。这就是为什么人们应该使用Fn加箭头键。

动作的上下文菜单

添加键盘快捷键

打开“ 输入键盘快捷键”对话框，您可以在其中指定要分配给当前键映射中所选操作的击键组合。

添加鼠标快捷方式

打开“ 输入鼠标快捷方式”对话框，您可以在其中指定鼠标单击的组合和要分配给当前键映射中所选操作的按钮。

添加缩写

添加可在“随处搜索”中使用的缩写。

删除<shortcut> / <abbreviation>

删除选定的快捷方式或缩写。
