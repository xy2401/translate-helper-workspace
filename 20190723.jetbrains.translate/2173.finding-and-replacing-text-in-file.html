查找和替换文件中的文本

您可以在编辑器中搜索当前打开的文件中的文本字符串。使用不同的选项，您可以缩小搜索过程范围，在搜索中使用正则表达式，以及管理搜索结果。

  1. 在编辑器中打开您的文件。

  2. 从主菜单中，选择“ 编辑”|找到|查找 （ Ctrl + F ）。

  3. 在打开的搜索字段中，输入搜索字符串。

    PyCharm会在文件中突出显示搜索结果。

将插入符号放在文件中的任何字符串上，然后按Ctrl + F查找其出现位置，或者从主菜单中选择“ 编辑”|找到|在Caret找到单词 。

如果要将目标搜索扩展到当前文件之外，请按Ctrl + Shift + F.

管理您的搜索

PyCharm允许您调整搜索过程并使用显示的搜索结果执行各种操作。

  • 如果要查看以前搜索的列表，请在搜索字段中按Alt +向下键 。

  • 如果要输入多行字符串，请单击搜索字段中的新图标。

  • 使用和箭头导航到上一个或下一个出现。

  • 使用“ 查找”工具窗口中的出现列表（ Alt + F7 ），您可以在其中有其他选项，例如，对结果进行分组或在单独的窗口中打开它们。

  • 点击要么图标以添加下一个匹配项的选择或取消选择上一个匹配项（ Alt + J或Shift + Alt + J ）。

  • 如果要在整个文件中快速替换搜索目标，请按Ctrl + Shift + Alt + J并键入新字符串。

  • 当您在搜索窗格中选择“ 单词” ，“ 匹配大小写”复选框或单击过滤器时，可以缩小搜索范围用于选择搜索范围的图标。

  • 您可以按Alt + X （以前称为Alt + G ）快速选择或清除正则表达式复选框。您可以使用正则表达式来选择更具挑战性的搜索。

  • 按Ctrl + F7查看打开文件中任何元素的用法。

    如果您不希望PyCharm突出显示文件中的所有已找到的用法，请打开“ 设置/首选项”对话框（ Ctrl + Alt + S ）并在编辑器上常规页面清除在插入符选项中突出显示元素的用法 。

    按Alt + F7搜索当前文件之外的用法，或按Ctrl + Alt + F7以单独的弹出窗口打开搜索结果。如果需要在搜索之前配置一些选项，请按Ctrl + Shift + Alt + F7打开“ 查找用法”对话框。
