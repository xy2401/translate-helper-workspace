按名称导航到类，文件或符号

概观

使用导航命令可以快速跳转到名称指定的所需类，文件或符号。PyCharm建议一个匹配名称的查找列表，您可以从中选择所需的名称，然后在编辑器中打开它。此导航表示CamelCase和snake_case大小写。有关可用技术的详细列表，请参阅提示 。

按名称导航

要导航到具有指定名称的类，文件或符号：

  1. 在主菜单上，转到导航并从以下选项中选择一个：

    • 类 （ Ctrl + N ）

    • 文件 （ Ctrl + Shift + N ）

    • 符号 （ Ctrl + Shift + Alt + N ）

  2. 在弹出窗口中，开始键入所需的名称。

    这样，您可以输入位于所需名称内任何位置的字符。在您键入时，建议列表会缩小，仅显示匹配的名称。

    • 符号 ：

      如果有多个具有相同名称的符号，则可以通过指定层次结构路径使搜索更精确。例如，如果要查找属于不同类的特定函数，可以键入函数名称，并查看查找列表中的所有可能位置。另一方面，可以指定类或文件名，后跟一个点，然后指定所需的函数：

  3. 单击建议列表中的所需条目，或使用箭头键选择它，然后按Enter键 。
