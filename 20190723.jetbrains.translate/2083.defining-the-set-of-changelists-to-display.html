定义要显示的更改列表集

与扩展合并信息的功能，你可以限制变更列表设置要显示在变更列表窗格修改列表已经从一个特定分支整合到另一个。这些分支分别称为“源”和“目标”。

要定义要显示的分支集，请执行以下操作：

  1. 打开“ 版本控制”工具窗口并切换到“ 存储库”选项卡 。

  2. 要打开“ 合并信息”窗格，请单击“ 突出显示集成”按钮在工具栏上。

  3. 在“ 发件人”字段中，指定源分支的URL地址。

  4. 在“ 收件人”字段中，指定目标分支的路径。如有必要，请使用“ 浏览”按钮打开“ 选择分支”对话框。

  5. 指定要应用基于所选更改列表创建的修补程序的本地工作副本的路径。如有必要，请使用Browse 打开“ 配置工作副本路径”对话框。
