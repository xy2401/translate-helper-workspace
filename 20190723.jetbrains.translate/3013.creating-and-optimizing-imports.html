WebStorm 2019.1帮助

自动导入

在JavaScript和TypeScript文件中，WebStorm会自动为模块，类，组件以及可以导出的任何其他符号以及XML名称空间添加import语句。从JavaScript中的自动导入，TypeScript中的自动导入和导入XML命名空间了解更多信息。

在代码完成时添加import语句

完成导出的JavaScript或TypeScript符号后，WebStorm会添加import语句。

您可以在完成时禁用自动导入并使用快速修复：

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ”，然后单击“ 自动导入” 。将打开“ 自动导入”页面。

  2. 使用TypeScript / JavaScript区域中的复选框可以在代码完成时启用或禁用导入生成。

使用快速修复

优化进口

WebStorm可帮助您删除未使用的导入，并在当前文件，目录中的所有文件或整个项目中组织import语句。

优化文件中的导入

  • 在编辑器中打开文件或在“ 项目”工具窗口中选择该文件，然后按Ctrl + Alt + O.

重新格式化文件时优化导入

您可以告诉WebStorm在每次重新格式化时优化文件中的导入。

  • 在编辑器中打开文件，按Ctrl + Shift + Alt + L ，并确保在打开的“ 重新格式化文件”对话框中选中“ 优化导入”复选框。之后每次在此项目中按Ctrl + Alt + L ，WebStorm将自动优化其导入。

优化文件夹中的导入

  • 在“ 项目”工具窗口中选择文件夹，按Ctrl + Alt + O ，然后在打开的“ 优化导入”对话框中单击“运行” 。

仅在修改后的代码中优化导入

如果项目受版本控制 ，则只能在更新的代码中优化导入。您还可以指示WebStorm在将修改后的文件提交到VCS之前优化其中的导入。

  • 在“ 项目”工具窗口中，选择一个文件夹，然后按Ctrl + Alt + O.在“ 优化导入”对话框中，选择“ 仅处理VCS已更改的文件” ，然后单击“运行” 。

  • 在“ 提交更改”对话框 （ Ctrl + K ）中，选择“ 优化导入”复选框。

配置导入语句的样式

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器 ”下的“ 代码样式 ” ，然后单击“ JavaScript”或“ TypeScript” 。

  2. 使用“ 导入”选项卡上的控件自定义导入语句的外观。
