附加到流程

PyCharm可以连接到Python本地进程，同时运行从您的操作系统或使用PyCharm终端启动的Python脚本，但不是在调试模式下。请注意，您无法附加到远程进程。

Ubuntu用户的先决条件

如果您在Ubuntu上使用PyCharm（或者可能在某些其他Linux发行版上），则在您第一次尝试附加到进程时可能会出现错误消息ptrace: Operation not permitted 。在这种情况下，请执行以下操作之一：

  • 要临时禁用此限制，请输入命令：

    echo 0 | sudo tee / proc / sys / kernel / yama / ptrace_scope

  • 要永久禁用此限制，请打开/etc/sysctl.d/10-ptrace.conf文件进行编辑，并将kernel.yama.ptrace_scope = 1行更改为kernel.yama.ptrace_scope = 0 。
    要立即应用更改，请输入以上命令echo 0 | sudo tee /proc/sys/kernel/yama/ptrace_scope 。或者，运行sudo service procps restart或只是重新启动系统。

附加到流程

附加到本地进程：

  1. 从主菜单中，选择“运行”|附加到Process或按Ctrl + Alt + F5 。

  2. 从可用的本地进程列表中，选择所需的进程。通过键入其名称或PID的首字母来简化搜索。如果有多个调试器可用于所选进程，则可以在弹出菜单中选择它们。

    您可能只想在“ 附加到进程”列表中显示特定进程。将所需模式添加到Python调试器设置（ 设置/首选项|构建，执行，部署| Python调试器 ）中的相应字段。例如，如果您只想附加与此字符串匹配的进程，请输入“Scientific”：

  3. 继续进行调试 ，就像在PyCharm中一样进行调试 （ 设置断点 ，单步执行 ， 暂停和恢复进程， 计算表达式等等） 请注意 ，如果您附加的进程未提供调试信息，则断点将不可用，并将在反汇编视图中执行步进。

  4. 完成后，分离进程：选择Run |停止或单击“ 停止进程”按钮 调试工具窗口 。
