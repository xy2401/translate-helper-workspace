启动任务

WebStorm |偏好|工具| macOS的启动任务
按Ctrl + Alt + S

在此页面上，创建要在项目启动时自动启动的运行/调试配置列表。如果您定期运行一些Grunt或Gulp.js任务或npm脚本，这可能会有所帮助。您只需将启动此类任务或脚本的运行/调试配置添加到启动任务列表中即可。

项目

提示

描述

运行配置

此只读字段显示要在项目启动时启动的运行配置的名称。

共享

选中此复选框后，其他团队成员可以使用相应的任务。请注意，您只能共享那些已在其定义中标记为已共享的运行配置。

共享运行/调试配置保存在.idea \ runConfigurations文件夹下的单独xml文件中，而本地运行/调试配置保存在.idea \ workspace.xml中 。

加

单击此按钮可将项目中当前定义的运行/调试配置之一添加到要在项目启动时执行的任务列表中。从列表中选择相关配置，或选择“ 编辑配置”以打开“ 运行/调试配置对话框”对话框，并在其中定义所需的配置。

去掉

单击此按钮可从列表中删除所选任务。

编辑

单击此按钮可使用所选配置中的设置打开“ 运行/调试配置对话框”对话框，并根据需要进行更新。
