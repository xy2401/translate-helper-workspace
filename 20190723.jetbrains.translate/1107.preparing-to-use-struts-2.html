在你开始之前

确保已启用Web和Struts2捆绑插件。默认情况下会激活插件。如果禁用了插件，请在“ 插件”页面上启用它们，如管理插件中所述 。

基本

通过Web和Struts 2方面启用与Struts 2的集成。这些专用方面包含设置，配置文件路径和验证规则。此信息确定模块的结构，以便IntelliJ IDEA检测如何处理模块内容。

Struts 2 facet只能作为Web facet的子项添加。请注意，模块中只允许使用一个Struts 2构面。

使用Struts 2 Facet创建模块

当您使用专用Web和Struts 2构面创建Struts 2模块时，IntelliJ IDEA会配置模块，自动下载并添加所有必需的库。

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java 。在页面的右侧，指定要使用的JDK 。

  3. 在“ 其他库和框架”下 ，选中“ Web应用程序”复选框。

    从“ 版本”列表中选择要支持的Servlet规范的版本 。

    如果要创建部署描述符web.xml文件，请选中Create web.xml复选框。

  4. 选择Struts 2复选框。

  5. 你需要一个实现Struts 2的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现Struts 2的库文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

    单击下一步 。

  6. 指定名称和位置设置。

    单击完成 。

IntelliJ IDEA按如下方式配置新模块：

  • 使用Web应用程序描述符web.xml创建Web节点。

  • 创建Struts 2配置文件struts.xml并将其添加到src节点。

  • 添加所需的Struts 2库。

将Struts 2 Facet添加到模块

当您将Struts 2 facet添加到现有模块时，IntelliJ IDEA会自动下载并添加必要的库。

Struts 2 facet只能作为Web facet的子项添加。如果模块没有Web构面，请先添加它。另请注意，模块中只允许有一个Struts 2 facet。

这将改变模块的配置 。

  1. 打开模块设置。

  2. 在“ 模块”节点中，右键单击相关模块。

  3. 要添加Web构面，请从上下文菜单中选择“ 新建 ”。在可用构面的下拉列表中，选择Web 。

  4. 右键单击“ Web Facet”节点，然后在其上下文菜单中选择“ 新建 ”。

  5. 在可用构面的上下文菜单中选择Struts 2 。右侧窗格显示Facet Struts 2对话框。

  6. 在“ 文件集”选项卡中， 指定要验证的配置文件 。

  7. 如果需要一些其他库，则会显示相应的警告消息。单击修复 。必要的库将被下载并自动添加。

创建的Struts 2配置文件struts.xml位于src节点中。

定义验证文件集

启用Struts 2支持后，IntelliJ IDEA可以通过添加，编辑和删除文件集来定义验证列表。

添加验证文件集

  1. 打开Struts 2 Facet页面 。

  2. 切换到“ 文件集”选项卡。

  3. 点击 （ Alt + Insert ）。将打开“ 编辑文件集”对话框，显示模块树。

  4. 在“ 文件集名称”字段中，指定新文件集的名称。

  5. 要将文件添加到集合，请选中其名称旁边的复选框。

  6. 要添加未在树中显示的文件，请单击“ 定位” ，然后在“ 选择路径”对话框中选择所需的文件。

编辑验证文件集

  1. 在“ 文件集”选项卡上选择文件集 ，然后单击 （ 输入 ）。“ 编辑文件集”对话框将打开。

  2. 要将文件添加到文件集，请选中所需文件旁边的复选框，然后单击“ 确定” 。

  3. 要删除文件，请选择该文件并单击 （ Alt + Delete ）。

要删除验证文件集，请在“ 文件集”选项卡上选择它，然后单击 （ Alt + Delete ）。
