创建Grails应用程序元素

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA可以在相应类型的模块（域类，控制器，脚本等）中轻松创建Grails或Griffon Application元素。您可以使用特定于应用程序的Grails或Griffon工具窗口或“项目”工具窗口创建新元素。

在控制台中显示与所选元素类型对应的目标的执行。

添加新的Grails或Griffon元素

  1. 打开Grails工具窗口，具体取决于所需的应用程序类型，然后右键单击应添加新应用程序元素的包。

  2. 在目标包的上下文菜单上，选择“ 新建” ，或按Alt + Insert ，然后选择元素类型。

  3. 输入新元素的名称。这样做，您可能不关心大写：当生成新元素时，其名称会自动大写。单击确定 。在应用程序结构规定的位置创建所选类型的Groovy类。

  4. 使用支持Groovy的编码帮助，重构和意图操作在编辑器中添加必要的源代码。

IntelliJ IDEA提供了一键创建Grails过滤器的可能性。只需按Alt + Insert ，然后在弹出菜单中选择Grails Filter 。执行CreateFilters.groovy目标后，将在grails-app/conf目录下创建带有过滤器定义的存根类。

请注意，IntelliJ IDEA会自动提供过滤器类的正确名称（ *Filters.groovy ），因此您只需指定过滤器类名称的初始字符。
