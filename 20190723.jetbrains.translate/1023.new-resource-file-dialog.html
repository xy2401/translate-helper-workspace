新资源文件对话框

项目工具窗口| res文件夹的上下文菜单|新的| Android资源文件

使用此对话框为Android项目创建资源文件。

项目

描述

文件名

在此文本框中，指定新资源定义文件的名称。

资源类型

从此下拉列表中，选择应用程序资源类型 。

您可以使用向上和向下键盘键从文件名文本框直接滚动资源类型列表。

根元素

根据所选资源类型自动填充此字段。

目录名称

在此文本框中，指定将存储应用程序资源的文件夹。

IntelliJ IDEA可以根据您选择的资源类型和限定符自动组合文件夹名称。有关详细信息，请参阅创建资源 。

可用的限定符

从此列表中，选择确定特定设备配置的资源限定符 。选择相关限定符并单击 。然后在打开的对话框中指定限定符的值。限定符将添加到“ 选择的限定符”列表中。
