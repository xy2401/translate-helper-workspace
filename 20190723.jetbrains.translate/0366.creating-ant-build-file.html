创建Ant构建文件

本节介绍如何手动创建Ant构建文件。

IntelliJ IDEA提供了一个用于编辑构建文件的框架，但开发人员有责任使用目标填充构建文件。

创建Ant构建文件

  1. 在“项目”窗口中，选择应在其中创建构建文件的目录。

  2. 右键单击该目录，然后选择“ 新建”|在上下文菜单中输入文件 ，或按Alt + Insert 。

  3. 在“ 新建文件”对话框中，指定扩展名为.xml的新文件的名称，例如build.xml 。新文件将在编辑器中打开。

  4. 指定构建目标。您可以使用fileset或dirset指令中的类似路径的结构来定义

    您必须在build.xml文件中至少输入一个根标记。否则，该文件将不会添加到Ant Build工具窗口中。

  5. 将构建文件添加到项目中 。
