VCS与问题跟踪器集成

使用IntelliJ IDEA，您可以将提交消息与错误跟踪器或问题数据库连接，并从VCS日志中的提交导航到与这些提交相关的问题。

启用从提交消息到问题的导航

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 版本控制”|问题左侧导航 。

  2. 通过使用引用问题的URL地址映射提交消息中的问题模式来配置问题导航模式列表。

    • 如果您使用的是JIRA或YouTrack ，请单击“ 添加JIRA模式” 或添加YouTrack模式 在工具栏上，键入错误跟踪系统安装的URL。
      IntelliJ IDEA将自动添加定义导航模式的正则表达式。

    • 对于其他问题跟踪系统，请单击“ 添加”按钮创建新条目或选择现有条目，然后单击“ 编辑”按钮。在打开的“ 添加问题导航链接”对话框中，指定以下内容：

      • 用于在提交消息中定义问题模式的正则表达式 。

      • 替换表达式 ，用于定义访问相应引用问题的URL。

例

问题ID模式

正则表达式，用于定义在提交消息中引用问题的格式。

[AZ] + \ - \ d +
此正则表达式匹配由两个以n-dash字符分隔的子字符串组成的所有字符串：

  1. 子串1：无限数量的大写字母字符。

  2. 子串2：无限数量的数字字符。

问题链接模式

问题跟踪系统的URL地址和标识其中问题的正则表达式的组合。

HTTP：// mytracker /问题/ $ 0
这里$0表示对整个比赛的反向引用。这意味着只要IntelliJ IDEA在提交消息中检测到匹配，就会将其添加到跟踪器的URL地址中。

匹配问题ID

IntelliJ IDEA在感兴趣的提交消息中检测以下对问题的引用：

MYPROJECT-110

组成问题链接

根据上述问题导航模式，检测到的匹配参考按原样添加到跟踪器的URL，因此引用问题的链接组成如下：

HTTP：// mytracker /问题/ MYPROJECT-110
