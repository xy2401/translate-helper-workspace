平台属性参考'

idea.properties文件可以包含特定于IntelliJ平台的设置和任何其他相关属性。若要配置平台属性，请在“ 帮助”菜单上单击“ 编辑自定义属性” 。

所有路径都使用正斜杠（ / ）指定，包括Windows路径（例如， C:/idea/system).

您可以使用带有任何属性名称的宏来插入其值，例如，使用${user.home}指定相对于用户主目录的路径。

属性

默认值

描述

idea.chooser.lookup.for.project.dirs真正

定义PyCharm是否应在内部目录中查找有效项目，以使用相应的图标标记此类目录。如果禁用此属性，则不会在用户的主目录之外执行此类查找。

idea.cycle.buffer.size1024

设置控制台循环缓冲区的最大大小（以kylobytes为单位）。如果控制台输出大小超过此值，则删除最旧的行。要禁用循环缓冲区，请设置idea.cycle.buffer.size=disabled 。

idea.dynamic.classpath假

定义是否应将长类路径写入文件并从那里读取而不是在命令行上传递它。默认情况下，禁用此选项，如果命令超出shell允许的字符数限制，则会失败。

idea.max.content.load.filesize20000

设置PyCharm能够打开和处理的文件的最大大小（以kylobytes为单位）。

idea.max.intellisense.filesize2500

设置PyCharm为其提供代码帮助的文件的最大大小（以千字节为单位）。大文件的代码辅助可能会降低编辑器性能并增加内存消耗。

idea.max.vcs.loaded.size.kb20480

设置PyCharm在比较更改时加载以显示过去文件内容的最大大小（以千字节为单位）。

idea.no.launcher假

定义在从PyCharm运行进程时是否应使用特殊的启动程序。启动器默认启用，并提供“软退出”和“线程转储”功能。要禁用它，请设置idea.no.launcher=true

idea.popup.weight重

如果您的窗口管理器配置为使用鼠标更改焦点并自动引发窗口，请将此属性设置为medium ，这将防止某些配置上的弹出菜单出现问题。

理念。ProcessCanceledException

启用

定义在检测到用户活动时PyCharm是否应抛出ProcessCanceledException 。仅在开发插件时禁用它，并且您需要调试IDE本身（与PSI相关的活动，这些活动在后台错误分析线程中执行），因为它可能导致显着的减速和锁定。

Windows用户应使用正斜杠。即c:/idea/system 。

$ {idea.home}宏

使用$ {idea.home}宏指定相对于IDE安装主页的位置。

还可以使用${xxx} ，其中xxx是任何java属性（包括在此文件的前几行中定义）以引用其值。

IDE配置文件夹的路径

如果要自定义IDE配置文件夹的路径，请取消注释此选项。

idea.config.path = $ {的user.home} /。PyCharm /配置

IDE系统文件夹的路径

如果要自定义IDE系统文件夹的路径，请取消注释此选项。

idea.system.path = $ {的user.home} /。PyCharm /系统

用户安装插件的路径

如果要自定义用户安装的插件文件夹的路径，请取消注释此选项。

idea.plugins.path = $ {的user.home} /。PyCharm /配置/插件

IDE日志文件夹的路径

如果要自定义IDE日志文件夹的路径，请取消注释此选项。

idea.log.path = $ {的user.home} /。PyCharm /系统/日志

最大文件大小

最大文件大小（千字节）IDE应为其提供代码帮助。如果启用了代码帮助，则文件越大，编辑器工作越慢，整体系统内存要求越高。 https://youtrack.jetbrains.com/issue/IDEA-72543

如果您需要任何文件的代码帮助，无论其大小如何，请删除此属性或设置为非常大的数字。

idea.max.intellisense.filesize = 2500

文件大小限制很大

IDE接受加载的最大文件大小。例如，当您的文件超过“在路径中查找”操作的指定限制时，IDE将忽略此类文件。您可以通过手动指定大小限制来覆盖默认的最大大小。

idea.max.content.load.filesize = 2000

控制台循环缓冲区

此选项控制控制台循环缓冲区：保持控制台输出大小不高于指定的缓冲区大小（Kb）。较旧的行被删除。要禁用循环缓冲区，请使用idea.cycle.buffer.size=disabled

idea.cycle.buffer.size = 1024

发射台

配置在IDE中运行进程时是否应使用特殊启动程序。使用Launcher启用“软退出”和“线程转储”功能。

idea.no.launcher = FALSE

类路径

避免太长的classpath

idea.dynamic.classpath = FALSE

ProcessCanceledException

取消注释此属性以防止IDE在检测到用户活动时抛出ProcessCanceledException。此选项仅对插件开发人员有用，同时调试在后台错误分析线程中执行的PSI相关活动。

除非你自己调试IDE，否则不要解除这个问题。否则将发生显着的减速和锁定。

理念。ProcessCanceledException =禁用

弹出窗口重量

idea.popup.weight属性有两个可能的值：“heavy”和“medium”。如果您将WM配置为“焦点跟随鼠标自动提升”，则必须将此属性设置为“中”。它可以防止某些配置上的弹出菜单出现问题。

idea.popup.weight =重

系统抗锯齿

在系统中使用默认消除锯齿，即覆盖“设置|编辑|外观|使用消除锯齿字体”选项的值。例如，在使用Windows远程桌面连接时可能很有用。

idea.use.default.antialiasing.in.editor = FALSE

重绘

禁用此属性可能会导致视觉故障，如闪烁，并且无法在某些显示适配器卡上重新绘制。

sun.java2d.noddraw =真

编辑表现

删除此属性可能会导致Windows下的编辑器性能下降。

sun.java2d.d3d = FALSE

慢滚动

JDK6中慢速滚动的解决方法。

swing.bufferPerWindow = FALSE

X Window下的编辑器性能

删除此属性可能会导致X Window下的编辑器性能下降。

sun.java2d.pmoffscreen = FALSE

避免长时间挂起

在macOS下访问剪贴板时避免长时间挂起的解决方法。

ide.mac.useNativeClipboard =真

复制库罐

IDE复制库jar以防止其锁定。如果不希望复制，请指定“true”

idea.jars.nocopy = FALSE

以调试模式启动JVM

用于在调试模式下启动JVM的VM选项值。一些JRE以不同的方式定义它（Oracle VM中的-XXdebug）

idea.xdebug.key = -Xdebug

切换到JMX 1.0兼容模式。

取消注释此选项，以便在使用运行1.4的应用程序服务器（如WebLogic）时使用J2SDK 1.5+运行PyCharm。

jmx.serial.form = 1.0

致命错误通知

如果要接收有关安装的IDE或插件发生的致命错误的即时可视通知，请更改为“已启用”。

idea.fatal.error.notification =禁用
