您可以使用Yeoman工具创建特定于框架的PyCharm项目。

在你开始之前

  1. 确保您的计算机上有Node.js.

  2. 按照管理插件中的说明在Plugins页面上安装Yeoman插件。

您还可以在npm和Yarn中描述的Node.js和NPM页面上安装yo包。

在PyCharm中安装和配置Yeoman

  1. 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install -g哟

  2. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript | Yeoman 。在打开的Yeoman页面上，指定全局安装的yo包的位置，并选择要使用的Node.js解释器。这可以是Windows子系统Linux上的本地Node.js解释器或Node.js。

配置项目存根生成器列表

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Yeoman 。

  3. 右侧窗格显示所有以前安装的Yeoman生成器 。单击安装生成器 。

  4. 在打开的对话框中显示所有可用的生成器包，在左侧窗格中选择所需的包，然后单击右侧窗格中显示的“ 安装生成器”按钮。您可以在不离开对话框的情况下依次安装多个软件包。

    安装结束后，单击“ 关闭”返回已使用新添加的包扩展的生成器列表。

通过生成器创建项目

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Yeoman 。

  3. 右侧窗格显示所有以前安装的Yeoman生成器 。要展开列表，请单击“ 安装生成器” ，请参阅上面的PyCharm中的安装和配置Yeoman 。

    从列表中选择所需的生成器，然后单击下一步 。

  4. 在启动的“ 新建项目”向导中指定所需的设置。页数及其内容取决于所选的发生器。

  5. 在向导的最后一页上，选中或清除Run npm install＆bower install复选框，以指定是否要运行Node Package Manager和Bower来安装开发新项目所需的软件包。

  6. 单击“ 下一步”，然后选择在当前窗口或新窗口中打开新项目。
