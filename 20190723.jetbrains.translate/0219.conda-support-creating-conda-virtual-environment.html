配置Conda虚拟环境

以下内容仅在安装并启用Python插件时有效！

IntelliJ IDEA支持使用Conda为Python创建虚拟环境。在开始之前，请确保已在您的计算机上下载并安装Anaconda或Miniconda 。

创建Conda环境

  1. 导航到文件|项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 项目结构”对话框中，选择“ 平台设置”部分下的SDK ，然后单击 ，从弹出菜单中选择Python SDK 。

  3. 在“ 添加Python解释器”对话框的左侧窗格中，选择“ Conda环境” 。以下操作取决于之前是否存在Conda环境。

    如果选择了新环境 ：

    1. 在文本字段中指定新Conda环境的位置，或单击并在文件系统中查找位置。请注意，新Conda环境所在的目录必须为空！

    2. 从列表中选择Python版本。

    3. 在文本字段中指定Conda可执行文件的位置，或单击并在Conda安装目录中找到位置。

    4. 如果需要， 请选中“ 使所有项目可用”复选框。

    如果选择了现有环境 ：

    1. 指定所需的解释器：使用列表，或单击并在您的文件系统中找到一个（例如， C：\ Users \ jetbrains \ Anaconda3 \ python.exe ）。

    2. 如果需要，选中“ 使所有项目可用 ”复选框。

  4. 单击“ 确定”完成任务。

对于任何配置的Python解释器，您可以：

  • 安装和卸载解释器路径
  • 安装和卸载解释器包
