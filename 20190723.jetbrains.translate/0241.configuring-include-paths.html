配置包含路径

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

包含路径用于保存第三方代码，该代码用于在使用文件路径作为参数的某些函数/方法中完成和引用解析，例如， require()或include() 。

请注意，通过包含路径添加的文件不应在项目范围内进行编辑。如果您具有需要在多个项目中访问和修改的现有代码，请考虑通过其他内容根添加它。

配置包含路径

  1. 请执行以下任何操作：

    • 在“ 项目工具”窗口中 ，右键单击“ 外部库” 并从上下文菜单中选择“ 配置PHP包含路径... ”。

    • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 语言和框架”下的“ PHP ” 。

    • 要配置将应用于所有新创建的项目的包含路径，请使用默认设置 （“ 文件”|“其他设置”|“新项目的设置” ）。

  2. 在打开的PHP页面上，在“ 包含路径”选项卡上配置包含路径列表：

    • 使用和添加和删除路径。

    • 使用和重新排序列表中的项目。

    • 点击使路径按升序顺序排序。
