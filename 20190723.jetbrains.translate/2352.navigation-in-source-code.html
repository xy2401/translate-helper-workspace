PyCharm 2019.1帮助

源代码中的导航

功能

捷径

使用此快捷方式...

文件结构弹出窗口

按Ctrl + F12

显示“ 结构”弹出窗口，以便快速浏览当前文件。

选择目标

ALT + F1

将焦点从当前文件，类，方法或引用移动到数据源表到“ 选择目标”弹出菜单中建议的视图。请参阅使用“选择”弹出窗口导航 。

最近的文件

按Ctrl + E

显示最近打开的文件列表。

最近更改的文件

N / A

显示最近更新的文件列表。

键入层次结构

按Ctrl + H

浏览所选类类的层次结构。

导航到Class

CTRL + N

通过在弹出窗口中指定其名称，直接导航到项目中的类 。

导航到文件

Ctrl + Shift + N

通过在弹出窗口中指定其名称，直接导航到项目中的文件 。

导航到最近打开的文件

按Ctrl + E

显示最近打开的文件列表。

导航到最近更改的文件

N / A

显示最近更新的文件列表。

导航到Line

CTRL + G

通过指定其编号导航到当前文件中的任何行 。

导航到声明

按Ctrl + B

导航到插入符号的符号声明 。

导航到实施

Ctrl + Alt + B

导航到插入符号项的实现。

导航到类型声明

CTRL + SHIFT + B

导航到插入符号的符号类型声明，符号是变量或方法调用。

导航到超级方法

按Ctrl + U

导航到插入符下方法的超级方法声明。

导航到测试/测试主题

CTRL + SHIFT + T

导航到插入符号类的测试（如果有），或从测试导航到测试主题。

导航到相关符号

按Ctrl + Alt +首页

在具有复杂关系的文件之间导航。

例如，使用此快捷方式在视图和模板之间导航。

导航到下一个方法

Alt +向下

导航到活动编辑器选项卡中的下一个方法声明。

导航到“上一个方法”

Alt +向上

导航到活动编辑器选项卡中的上一个方法声明。

导航到Opening Brace

按Ctrl + [

导航到当前代码块的开头。

导航至Closing Brace

按Ctrl +]

导航到当前代码块的末尾。

背部

按Ctrl + Alt +左

撤消上次导航操作。

在macOS计算机上，您还可以使用三指从右向左轻扫手势。

向前

按Ctrl + Alt +右

重做最后撤消的导航操作。

在macOS计算机上，您还可以使用三指从左向右滑动手势。

导航到上一次出现

按Ctrl + Alt +向上

导航到上一个找到的项目。

导航到下一次出现

按Ctrl + Alt +向下

导航到下一个找到的项目。

最后编辑位置

按Ctrl + Shift + Backspace键

浏览最近的更改点。

导航到下一个突出显示的错误

F2

导航到下一个找到的错误/警告。

导航到上一个突出显示的错误

按住Shift + F2

导航到先前找到的错误/警告。
