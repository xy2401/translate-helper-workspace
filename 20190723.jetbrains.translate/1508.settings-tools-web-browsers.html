  • 将Web浏览器的安装与IntelliJ IDEA集成，激活或停用从IntelliJ IDEA启动Web浏览器。

浏览器

在本节中，指定哪些浏览器可用于预览HTML或JSP输出。该部分显示了预定义列表中的浏览器以及先前配置的自定义浏览器安装（如果有）。

IntelliJ IDEA附带了一个预定义的最常用浏览器列表，您可以在运行，调试或预览HTML文件的输出期间自动安装和启动这些浏览器。IntelliJ IDEA假设您根据标准过程安装浏览器，并为每个安装分配一个别名 ，该别名代表浏览器的可执行文件或macOS应用程序的默认路径。如果在实际的浏览器安装中，可执行文件的路径不同，则需要在“ 路径”字段中明确指定它。

除了预定义的浏览器，您还可以使用工具栏上的控件根据需要配置任意数量的自定义浏览器。

活性

选中此复选框以启用IntelliJ IDEA中的相应浏览器。浏览器将添加到“浏览时打开”菜单项的上下文菜单中，其图标将显示在“ 浏览器”弹出窗口中。

如果清除此复选框，则相应的浏览器图标将不会出现在图标工具栏或弹出窗口中。

名称

在此列中，指定浏览器名称。

家庭

在此列中，指定浏览器所属的系列。

路径

在此列中，指定可执行文件的路径。如果根据标准安装过程安装了浏览器，则“ 路径”字段中的别名很可能指向正确的位置。如果没有，请单击并在打开的对话框中选择实际路径。在打开的对话框中 ，选择相应浏览器的可执行文件的路径。

单击此按钮可将自定义浏览器添加到列表中。

单击此按钮可从列表中删除选定的客户浏览器。请注意，您无法从预定义列表中删除浏览器。

在“ Firefox设置”对话框中，指定用于预览输出的Firefox浏览器配置文件 ：

  • “profiles.ini”的路径 ：在此文本框中，指定profiles.ini文件的位置，该文件确定要使用的Firefox配置文件。

  • 命令行选项 ：在此文本框中，输入命令行选项以启动Chrome实例。如果您需要更多空间，请单击 ，或按Shift + Enter打开编辑框。通过在Chrome中打开chrome://flags ，详细了解Chrome命令行选项。

默认浏览器

在本节中，指定默认情况下将用于呈现外部资源和使用Web内容预览文件的默认IntelliJ IDEA浏览器 。选择“ 视图”|时，在上下文菜单中将此浏览器称为“ 默认” 在主菜单上的浏览器中打开，或在文件的上下文菜单中在浏览器中打开 。

默认浏览器

从下拉列表中选择默认浏览器。可能的选择是：

  • 系统默认值 ：选择此选项可接受操作系统默认Web浏览器作为IntelliJ IDEA的默认值。

  • 首先列出 ：选择此选项可让IntelliJ IDEA启动列表中的第一个浏览器。使用更改订单或浏览器和工具栏上的图标。

  • 自定义路径 ：选择此选项可将另一个Web浏览器指定为IntelliJ IDEA的默认浏览器。输入浏览器可执行文件的路径或单击并在打开的对话框中选择路径。

在编辑器中显示浏览器弹出窗口

  • 如果选中此复选框，则启用浏览器的弹出窗口将显示在HTML或JSP文件中。

  • 如果未选中此复选框，则此弹出窗口不会显示，从而帮助您阅读或编辑代码。
