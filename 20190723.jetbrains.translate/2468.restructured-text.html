PyCharm 2019.1帮助

reStructuredText支持

除了支持reStructuredText文档字符串格式（PEP 287）之外 ，PyCharm还为重组文本提供了一个方便的编辑器。

与Markdown编辑器类似，您可以修改reStructuredText文件并在“ 预览”面板中验证结果。您可以使用相应的图标在可用的编辑模式之间切换。编辑和预览模式反映了当前选择的PyCharm 用户界面外观 。

您可以使用重新构造的文本打开任何现有文件，也可以从头开始创建新文件。

创建.rst文件

  1. 在“ 项目”工具窗口中单击鼠标右键，然后选择“ 新建”档案 。在打开的对话框中，指定带有.rst扩展名的任何文件名，然后单击“ 确定” 。

  2. PyCharm识别文件类型并用相应的图标标记它。

  3. 新创建的文件将在“ 编辑器”窗口中打开。未添加任何文本时，将禁用预览 。

让我们添加一些内容，看看PyCharm如何协助编辑重组文本。

编辑文本

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，查看reStructuredText（ 编辑器|颜色方案| reStructuredText ）的默认颜色方案。如果需要，为标记元素设置替代颜色和效果。

    单击“ 确定”保存更改并关闭窗口。

  2. 添加任何部分结构标记并切换到拆分视图模式以预览结果。请注意，PyCharm支持标记检查，因此如果检测到任何错误，则会在“ 编辑器”面板中突出显示有问题的行，并在“ 预览”面板中显示相应的错误消息。

  3. 使用.. code-block ::指令添加代码片段。PyCharm支持指令名称的代码完成。

    您可以通过在指令中指定特定语言来启用代码语法突出显示： .. code-block :: python 。另外，确保您的项目解释器已安装 Pygments包。

  4. 如果需要，您可以快速标记文本中的链接并推迟添加参考目标。PyCharm通知您并突出显示缺少引用的链接。

    您只需要为reStructuredText文件添加正确的目标引用，例如， .. _ pytest：https：//docs.pytest.org/ 。

如果您在预览内容时遇到任何问题，请尝试更改“ 预览”面板的类型。在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，导航到“ 语言和框架”| ReStructrued Text ，然后从“ 预览”面板列表中选择“ JavaFX”或“ Swing ”，单击“ 确定” ，然后继续编辑.rst文件。
