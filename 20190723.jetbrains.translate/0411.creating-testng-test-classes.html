创建TestNG测试类

要创建TestNG测试类，可以使用“ 创建测试”意图操作 。

您还可以为TestNG测试创建一个新类，然后将必要的代码添加到该类。本页描述了这种创建TestNG类的方法。

创建TestNG测试类

  1. 在“ 项目工具”窗口中 ，右键单击要在其中创建新测试类的目录。

  2. 选择新建|上下文菜单中的Java Class 。

  3. 在“ 创建新类”对话框中，指定类名，然后单击“ 确定” 。

  4. 在编辑器中，编写测试类的代码。必要时使用TestNG注释。例如，您可能想要注释整个类或单个方法：
    @Test （） public class testetsng { @DataProvider public Object [] [] data （） { return new String [] [] { new String [] { “data1” }， new String [] { “data2” }}; } @Test （ dataProvider = “data” ） public void test （ String d ） { Assert 。 assertEquals （ “第一行> \ n第二行” ， “第三行\ n第四行” ）; } }
