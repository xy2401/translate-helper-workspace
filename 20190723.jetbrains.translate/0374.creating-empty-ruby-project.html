创建空Ruby项目

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

空项目适用于纯Ruby编程。

创建一个空项目

  1. 执行以下操作之一：

    • 在主菜单上，选择“ 文件”|新的|项目 。

    • 在“ 欢迎”屏幕上 ，单击“ 创建新项目 ”链接。

    “新建项目”对话框打开。

  2. 在“ 新建项目”对话框中，执行以下操作：

    • 在“ 项目类型”部分中，单击所需的项目类型，在本例中为Ruby。

    • 指定项目SDK。如果列表中缺少所需的SDK，请单击“ 新建” 。

    • 单击下一步 。

    • 指定项目名称和位置。

  3. 单击完成 。
