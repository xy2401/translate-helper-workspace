WebStorm 2019.1帮助

重构

在XSLT上下文中，可以使用一些WebStorm内置重构：

  • 重命名和安全删除 XSLT项目，例如模板，变量和参数。

  • 介绍用于从选定的XPath表达式创建XSLT变量的变量。

  • 内联变量。

重命名模板

重构|改名

可以像在任何其他符号中一样在WebStorm中重命名命名模板。所有引用此模板的xsl:call-template调用都将相应更新。

重命名变量和参数

重构|改名

与命名模板一样，可以在使用时或在声明时重命名XSLT变量和模板参数。

安全删除

重构|安全删除

可以使用WebStorm安全删除功能删除命名模板，参数和变量，即如果项目中的其他样式表中没有任何引用，则将删除该项目。

如果样式表可以通过xsl:include或xsl:import包含在其他样式表中，这将特别有用，以确保不会删除任何仍在其他地方使用的内容。

介绍变量

重构|提取

可以提取XPath-Expressions并将它们转换为xsl：variable声明。选中“ 替换所有实例”复选框以替换同一表达式的所有其他实例。

介绍参数

重构|提取

这类似于Introduce Variable ，但它创建了一个新参数而不是变量。它还有一个附加选项Create with default value ，用于确定是否应将所选表达式添加为引入参数的默认值，或者是否应更新对模板的所有调用以传递所选表达式。该选项仅在将参数引入命名模板时可用。

内联变量

重构|排队

这与Introduce Variable相反，它将变量的所有用法替换为变量select -attribute中指定的表达式。不具有此类属性的变量无法内联。也不可能内联解析为参数的变量引用。
