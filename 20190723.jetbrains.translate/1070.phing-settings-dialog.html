项目工具窗口| Phing构建文件的上下文菜单|添加为Phing构建文件
查看|工具窗口| Phing Build

单击“ 设置”按钮后，将打开该对话框在Phing Build工具窗口的工具栏上。

在此对话框中：

  • 在当前项目中启用Phing集成 。

  • 在外部配置Phing构建属性以通过命令行传递它们。

  • 显示/隐藏特定文件中的目标。

在本页面：

  • 一般区域
  • 属性选项卡
  • 隐藏目标选项卡

一般区域

项目

描述

Phing可执行文件的路径

在此文本框中，指定phing.bat文件的位置。手动键入路径或单击“ 浏览” 并在打开的对话框中选择文件位置。

命令行选项

在此文本框中，指定在启动构建时要传递给Phing的其他命令行参数。如有必要，请单击并在“ 命令行选项”对话框中键入所需的选项。在新行上键入每个选项。当您关闭对话框时，它们都显示在命令行选项文本框中，空格作为分隔符。

属性选项卡

使用此选项卡可在外部配置属性元素列表。启动构建执行时，这些属性通过命令行传递给Phing，因此每次动态重新计算它们。

项目

描述

加

单击此按钮可为添加到列表中的新属性元素添加空行。

去掉

单击此按钮可从列表中删除所选元素。

属性

在此文本框中，键入属性元素的名称。

值

在此文本框中，指定在构建执行期间要传递的元素的值。执行以下操作之一：

  • 手动键入值。

  • 要使用IntelliJ IDEA宏 ，请单击“ 插入宏”按钮并在打开的“ 宏”对话框中配置相关宏定义的列表。要添加宏，请在列表中选择它，然后单击“ 插入” 。

隐藏目标选项卡

在此选项卡中，指定要在文件树视图中显示或隐藏的当前所选构建文件中定义的目标。

在构建文件中定义的目标中，您可能有一些仅由其他目标调用的目标，并且永远不会单独运行。您可以通过将它们标记为隐藏来禁止在构建文件树中显示此类目标。

展开特定构建文件的节点或单击“ 全部展开”按钮时， 隐藏目标不会显示 。

项目

描述

目标

此字段显示当前构建文件中定义的所有目标的列表。

隐藏

  • 如果选中目标名称旁边的此复选框，则目标不会显示在文件的树视图中。

  • 清除此复选框后，相应的目标将显示在文件的树视图中。

您可以直接从Phing Build工具窗口或Phing Settings对话框中将目标标记为隐藏 。但是， 只能通过Phing Settings对话框删除隐藏状态。
