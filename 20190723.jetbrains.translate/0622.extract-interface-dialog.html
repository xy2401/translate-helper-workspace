描述

从中提取界面

此只读字段显示包含从中提取接口的类的源包的名称。

提取界面

选择此选项后，IntelliJ IDEA将提取新接口，但不会立即使用它，并且不会更改源代码。

接口名称

输入新接口的名称。如果选择“ 提取”界面选项，则该字段可用。

提取界面并尽可能使用它

选择此选项可以提取界面并立即应用于源代码，建议的更改显示在“ 查找”工具窗口的专用选项卡中。

重命名原始类并尽可能使用接口

使用此选项可重命名原始类，并使其成为新创建的接口的实现。

将实现类重命名为

输入原始类的新名称。如果选择“ 重命名原始类并使用可能的接口选项”，则该字段可用。

用于新界面的包

指定新接口的包。如有必要，单击“ 浏览” （ ）并在打开的对话框中选择目标包。

表格界面的成员

指定类的方法，以及要包含在新接口中的最终静态字段（常量）。要在界面中包含元素，请选中它旁边的复选框。

的JavaDoc / ASDoc的

指定要应用于内联文档的操作。可用选项包括：

  • 按原样 - 选择此选项可使内联文档保留原样。

  • 复制 - 选择此选项可将内联文档复制到提取的界面，而无需将其从当前位置删除。

  • 移动 - 选择此选项可将内联文档移至提取的界面，并将其从当前位置删除。
