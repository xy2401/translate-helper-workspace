查找用法。方法选项

编辑|找到|查找用法设置
Alt + F7

本节介绍在“ 查找用法”对话框中指定“方法使用搜索”选项的控件。

单击时也可以打开此对话框在“ 显示用法”弹出窗口中，其中列出了插入符号中符号的所有出现位置。

当您使用此对话框（ Ctrl + Shift + Alt + F7 ）搜索方法实现的用法时，WebStorm将询问您是否要搜索基本方法。使用Alt + F7或Ctrl + Alt + F7等任何其他查找用法操作时，基本方法将自动包含在搜索结果中。

项目

描述

搜索文本事件

选中此复选框可搜索文本文件中的实例。

使用一种用法跳过结果选项卡

如果只找到一个用法，则选中此复选框以直接导航到找到的用法，而不显示“ 查找”工具窗口。

范围

在此区域中，指定搜索范围。从列表中选择预定义的范围，或单击“ 浏览”按钮打开“ 范围”对话框，您可以在其中定义自定义范围。

在新标签页中打开

选中此复选框，每个搜索的结果都显示在“ 查找结果”窗口的单独选项卡中。如果清除该复选框，搜索结果将覆盖当前选项卡的内容。
