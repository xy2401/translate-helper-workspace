在IntelliJ IDEA中定义应用程序服务器

此功能仅在Ultimate版本中受支持。

要在IntelliJ IDEA中定义服务器，在大多数情况下，您所要做的就是指定相应服务器的安装位置。

您可以定义服务器：

  • 另外，在“ 设置”对话框中： Ctrl + Alt + S |构建，执行，部署|应用服务器| 等

  • 创建项目或模块时。

  • 创建服务器运行/调试配置时 ： 运行|编辑配置| | <服务器名称>等

在“设置”对话框中定义服务器

  1. 打开“ 设置/首选项”对话框（ Ctrl + Alt + S ）。

  2. 在左侧窗格的“ 构建，执行，部署”类别中，选择“ 应用程序服务器” 。

  3. 在对话框右侧打开的“ 应用程序服务器”页面上，单击“确定” 。（或者，按Alt + Insert 。）

  4. 选择您要使用的服务器。

  5. 在打开的对话框中，指定服务器设置，然后单击“ 确定” 。对于大多数服务器，您只需指定服务器主目录，即服务器安装目录。有关更多信息，请参阅Application Servers 。

  6. 在Settings对话框中单击OK 。

在创建项目或模块时定义服务器

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java Enterprise 。

  3. 在页面的右侧，在“ Application Server”字段的右侧，单击“ 新建” 。

  4. 选择您要使用的服务器。

  5. 在打开的对话框中，指定服务器设置，然后单击“ 确定” 。对于大多数服务器，您只需指定服务器主目录，即服务器安装目录。

  6. 根据需要指定其他设置，然后单击下一步

  7. 指定名称和位置设置，然后单击“ 完成” 。

在创建运行/调试配置时定义服务器

  1. 打开“ 运行/调试配置”对话框（例如“运行”|“编辑配置” ）。

  2. 点击 （ Alt + Insert ），选择感兴趣的服务器（例如Tomcat Server ），如果可用，选择Local或Remote 。

  3. 在对话框的右侧，在“ 服务器”选项卡上，单击“ 应用程序服务器”列表右侧的“ 配置 ”。

  4. 在打开的对话框中指定服务器设置，然后单击“ 确定” 。

  5. 根据需要指定其他运行/调试配置设置，然后单击“ 确定”
