此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

使用用于从模板导航到本地化文件的装订线图标。要从本地化文件跳转到相应的模板，请使用Ctrl +单击 。

从模板导航到区域设置

  1. 点击所需标签旁边的装订线中的图标。

  2. 如果从多个区域设置引用标记，请从弹出窗口中选择一个：

    选定的django.po文件在编辑器中打开，插入符号位于对应于相关标记的msgid上。

根据Django文档，每个django.po文件都包含注释，每个msgid上面有一个模板路径。

从区域设置导航到模板

  1. 在所需的django.po文件中，将插入符号放在相关语言环境上方的注释中：

  2. 在主菜单上，选择“ 导航”|声明 （ Ctrl + B ），或使用导航到声明或键入符号声明中描述的任何其他方法。

    相应的模板文件将在编辑器中打开。

  3. 或者，您可以按Ctrl键并将鼠标指针悬停在相关区域设置上方的注释上。评论变成了超链接。单击它以导航到模板文件。
