Lightbend Project Starter入门

IntelliJ IDEA允许您使用Lightbend Project Starter模板。

创建一个项目

在创建项目之前，请确保在IntelliJ IDEA中下载并启用了Scala插件。

  1. 如果IntelliJ IDEA中当前未打开任何项目，请单击“欢迎”屏幕上的“ 创建新项目 ”。否则，请选择文件|新的|项目 。

    结果，“ 新建项目”向导将打开。

  2. 在左侧窗格中，选择Scala 。

  3. 在右侧窗格中，选择Lightbend Project Starter 。

    单击下一步 。

  4. 在向导的下一页上，指定项目的信息，选择要打开的模板应用程序（模板描述也可用），然后单击“ 完成” 。

    IntelliJ IDEA将使用选定的模板应用程序创建基于sbt的项目。
