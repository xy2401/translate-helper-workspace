PyCharm 2019.1帮助

网络框架

PyCharm通过继承IntelliJ IDEA的所有与Web相关的功能来促进Web应用程序的开发，包括对以下技术的支持：

  • JavaScript的
  • Node.js的
  • CoffeeScript的
  • 打字稿
  • 文件监视器
  • 镖
  • RESTful Web服务
  • 蚂蚁
  • HTML
  • 样式表
  • XML

PyCharm专业版还为一些流行的特定于Python的Web框架提供了高级支持：

  • Django的
  • 烧瓶
  • 金字塔
  • 的web2py
  • Google App Engine

Web开发支持可能包括特定框架的专用项目类型，相应的运行/调试配置，特定语言模板，使用装订线图标在视图和模板之间导航，代码完成和检查。有关其他功能和特定过程的详细信息，请参阅联机帮助的相应部分。
