Database Console

概观

数据库控制台允许您为PyCharm中定义的数据库编写和执行SQL语句作为数据源。它们还允许您分析和修改检索到的数据。

数据库控制台的输入窗格将作为单独的编辑器选项卡打开。这是您编写SQL语句的地方。

当你执行你的第一个声明时（ ）， 数据库控制台工具窗口打开。如果执行的语句检索数据（例如SELECT ），则“ 输出”和“ 结果”选项卡上的工具窗口中有两个窗格。（显示检索到的数据的选项卡可能标记为Result＃，或者，如果合适，可能会显示表名。）

否则，仅显示输出窗格。

此外，您可以打开“ 参数”窗格（ ）管理SQL语句中的参数。

输入窗格

使用输入窗格组合和执行SQL语句以及执行其他相关任务。

可通过工具栏图标，键盘快捷键和上下文菜单命令访问可用功能。

输出窗格

此窗格显示您运行的SQL语句以及有关在控制台中执行的其他操作的信息。这些包括打开或关闭自动提交模式，提交或回滚事务等。

此窗格中还显示有关发生的错误的信息。

对于大多数事件，提供以下信息：

  • 时间戳，即事件发生的时间。

  • 对于数据定义和数据操作操作 - 受影响的行数（例如，添加，更改或删除）。对于数据检索操作 - 检索了多少行。

  • 持续时间，以毫秒为单位。

摘要信息也会显示在状态栏上。

使用以下上下文菜单命令：

  • 复制 （ Ctrl + C ）将输出窗格中选定的文本复制到剪贴板。

  • 与剪贴板比较，将输出窗格中选择的文本与剪贴板的内容进行比较。

  • 全部清除以清除输出窗格的所有内容。

结果窗格

此窗格以表格格式显示从数据库检索的数据。您可以对数据进行排序，添加，编辑和删除，以及执行其他相关任务。

主要功能

“结果”窗格中的大多数功能都可通过工具栏上的控件，数据单元格的上下文菜单命令以及相关的键盘快捷键进行访问。

数据提取器：<current_format>

使用此按钮或命令打开一个菜单，您可以在其中选择数据的输出格式。

除输出格式外，还有以下选项和命令：

  • 允许换位。 对于分隔符分隔值格式（TSV，CSV）：如果表格已转置，并且您正在将所选单元格或行复制到剪贴板（例如Ctrl + C ），则选项将被转置（如图所示）否则，不转置（如原始表中）。

  • 跳过生成的列（SQL）。 对于SQL INSERT和UPDATE：复制或保存数据时，不要包含自动增量字段。

  • 添加表定义（SQL）。 对于SQL INSERT和UPDATE：复制或保存数据时，添加表定义（CREATE TABLE）。

  • 配置CSV格式。 打开CSV格式对话框 ，该对话框允许您管理分隔符分隔值格式（例如CSV，TSV）。

  • 转到脚本目录。 切换到存储将表数据转换为各种输出格式的脚本的目录。
