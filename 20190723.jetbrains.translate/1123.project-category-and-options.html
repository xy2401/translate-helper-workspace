项目类别和选项

选择“ 文件”| “ 新建项目”向导的此页面将打开新的|项目主菜单或欢迎屏幕上创建新项目 。

在左侧窗格中，选择项目类别。这可能是您将要使用的技术，您的开发将要定位的平台或运行时，等等。

在页面的右侧，选择其他选项并指定相关设置。

不要担心此刻选择“错误”选项。只需选择您认为最适合您的那些。如有必要，您将能够在以后对项目进行必要的更改。

请注意，您可以选择的选项集取决于IntelliJ IDEA中当前启用的插件 。

指定要用于部署和运行应用程序的应用程序服务器。因此，IntelliJ IDEA将为指定的服务器创建运行/调试配置 。（您可以稍后指定服务器。）

  • 对于托管服务器（Cloud Foundry或CloudBees），请指定您的用户帐户详细信息。

  4. 在“ 文件名”文本框中，键入将生成的相互关联的.js ， .html和.css文件的名称。仅当从“ 模板”下拉列表中选择“ 默认”样本类型时，该字段才可用。

Vue.js

在右侧窗格中：

  1. 指定项目名称以及将存储项目相关文件的文件夹的路径。

  2. 在Node Interpreter字段中，指定要使用的Node.js解释器。从列表中选择已配置的解释器，或选择“ 添加”以配置新的解释器

  3. 在vue-cli或@ vue / cli字段中，指定存储vue-cli或@vue/cli包的文件夹。

  4. 从“ 项目模板”列表中，选择要使用的Vue.js模板 。
