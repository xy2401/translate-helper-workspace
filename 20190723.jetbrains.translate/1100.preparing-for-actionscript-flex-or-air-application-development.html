IntelliJ IDEA 2019.1帮助

准备ActionScript或Flex应用程序开发

此功能仅在Ultimate版本中受支持。

在这个部分：

  • 初步步骤
  • 在IntelliJ IDEA中注册Flex或AIR SDK
  • 配置常规Flex编译器设置
  • 创建Flash模块
  • 配置模块内容

初步步骤

准备ActionScript或Flex应用程序开发：

  1. 确保已启用Flash / Flex支持插件。该插件默认激活。如果插件已禁用，请在“ 插件”页面上启用它，如管理插件中所述 。

  2. 在您的计算机上下载并安装Flex或AIR SDK。IntelliJ IDEA提供与Apache Flex ， Adobe Flex ， Adobe AIR和Adobe Gaming SDK的集成。

  3. 在IntelliJ IDEA中注册Flex或AIR SDK 。（您可以将此步骤推迟到以后的某个时间。您可以在创建Flash模块时指定SDK。）

  4. 检查常规Flex编译器设置 ，如有必要，根据需要进行调整。

  5. 使用Flash模块创建新项目，或将Flash模块添加到现有项目。请参阅创建Flash模块 。

    请注意，您可以通过导入Adobe Flash Builder项目来创建项目或模块。

  6. 检查初始模块配置并进行必要的调整。这些可能包括配置模块内容 ， 添加库 ，修改现有构建配置以及创建其他构建配置等。

    但是，所有这些任务可以在需要时在稍后的时间执行，同时开发源代码 。

在IntelliJ IDEA中注册Flex或AIR SDK

在IntelliJ IDEA中注册Flex或AIR SDK

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ）。

  2. 在“ 平台设置”下 ，单击“ SDK” 。

  3. 单击“ 添加新SDK” 在工具栏上，然后单击 “ 添加新SDK”列表中的Flex / AIR SDK 。

  4. 在打开的对话框中 ，选择安装Flex或AIR SDK的目录，然后单击“ 确定” 。

    因此，SDK配置显示在“ 项目结构”对话框右侧的SDK页面上。

  5. 通常，您不需要对Classpath和Sourcepath选项卡进行任何更改，因为所有必需的库和源都已存在。但是，您可能需要添加外部在线文档，以便在编写代码时能够访问其他参考信息。要做到这一点：

    在“ 文档路径”选项卡上，单击“是” 。在打开的对话框中，单击“ 确定” 。 （IntelliJ IDEA建议的URL是您想要的URL。）

  6. 在Project Structure对话框中单击OK 。

配置常规Flex编译器设置

配置常规Flex编译器设置

  1. 打开“ 设置/首选项”对话框（例如， Ctrl + Alt + S或在工具栏上）。

  2. 在对话框的左侧，在“ 构建，执行，部署”节点下，打开“ 编译器”节点，然后单击“ ActionScript和Flex编译器” 。

  3. 在“ 编译器”>“ActionScript和Flex编译器”页面上，根据需要配置设置。

配置模块内容

配置模块内容

通过添加和删除模块内容根以及将单个文件夹（在内容根中）分配给源文件夹，测试源文件夹以及排除文件夹来配置模块内容。

对于Flash模块，通常，这些任务在“ 项目结构”对话框的“ 模块”页面上执行。

要访问此页面：

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ）。

  2. 在项目设置下 ，选择模块 。

  3. 在该地区下 ，选择所需的Flash模块。
