Struts辅助工具窗口

查看|工具窗口| Struts助理

当您打开包含已启用Struts支持的模块的项目时，将显示Struts Assistant工具窗口。使用此工具窗口可管理 Struts应用程序中的元素 ， 切片和验证器 。

Struts Assistant工具窗口有三个选项卡：

  • Struts选项卡，用于管理 struts-config.xml配置文件中的Struts元素 。

  • Tiles选项卡用于在tiles-defs.xml定义文件中管理Struts切片。

  • Validator选项卡，用于定义特定于操作的验证规则。

Struts Assistant工具窗口显示三个视图：

  • 结构树显示Struts配置文件struts-config.xml的结构，具体取决于当前选项卡。

  • “ 属性表”显示在“结构树”中选择的节点的属性。

  • Struts Web Flow Diagram以图形方式显示应用程序元素之间的关系。

    Struts Web Flow Diagram仅出现在Struts选项卡中。

视图是同步的。如果在结构树中选择了节点，则“属性表”将显示节点的属性。如果节点具有相应的图表对象，则也会选择此图表对象。
