JavaScript的。使用范围

此功能仅在Ultimate版本中受支持。

档案|设置|语言和框架| JavaScript |库 - 管理 Windows和Linux的范围
IntelliJ IDEA |偏好|语言和框架| JavaScript |库 - 管理 macOS的范围
按Ctrl + Alt + S

单击JavaScript库设置页面上的管理范围时，将打开该对话框。

使用此对话框可以定义JavaScript库应用于代码完成，突出显示和导航的范围。范围可能涵盖整个项目，目录甚至单个文件。这有助于使JavaScript代码完成更加精确，并避免使用太长的建议列表。

此列显示文件或目录的库（如果适用）。

如果可以为项目树视图的某个节点指定库，请单击所选文件或目录的“ 库”列，然后从可用库列表中选择所需的库。

如果JavaScript库不适用于特定节点，则“N / A”以灰色字体显示。
