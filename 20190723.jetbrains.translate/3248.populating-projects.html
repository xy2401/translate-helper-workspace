创建文件和目录

要配置现有文件模板或创建新模板，请单击“ 设置/首选项”对话框（ Ctrl + Alt + S ）中“ 编辑器 ”下的“ 文件和代码模板 ” 。有关详细信息，请参阅创建和注册文件类型 。

要更改自动生成的文件扩展名，请使用重命名重构 （ 重构|重命名 ）。

如果新扩展没有适当的文件类型，则很可能需要安装相应的存储库插件，如管理插件中所述 。

创建一个空文件

  1. 在“ 项目工具”窗口中 ，选择要创建文件的目录，按Alt + Insert ，然后从列表中选择“ 文件 ”。

  2. 在打开的“ 新建文件”对话框中，键入文件名和扩展名。您可以在新文件名之前键入整个目录结构。如果嵌套目录尚不存在，则将创建它们。

    如果指定的扩展名与WebStorm中识别的任何文件类型无关，则会打开“ 注册新文件类型关联”对话框，您可以在其中将扩展名与其中一种可识别的文件类型相关联。有关详细信息，请参阅创建和注册文件类型 。

在WebStorm中，您可以创建具有特定语言扩展和一些初始内容的文件 ，这要归功于IDE支持的大多数语言的文件模板 。

创建文件

  1. 在“ 项目工具”窗口中 ，选择要在其中创建文件的目录，按Alt + Insert ，然后选择所需的语言或文件类型。

  2. 在打开的对话框中，键入没有任何扩展名的文件名。您可以在新文件名之前键入整个目录结构。如果嵌套目录尚不存在，则将创建它们：

    根据需要指定其他信息。例如，如果相应的文件模板包含此类变量且当前未设置其值，则可能会要求您定义自定义变量的值。

创建目录

  1. 在“ 项目”工具窗口中，右键单击要在其中创建新目录的目录，然后选择“ 新建”|上下文菜单中的目录 。或者，单击父目录，按Alt + Insert ，然后从列表中选择目录 。

  2. 在打开的对话框中，指定目录名称。如果要创建多个嵌套目录，请指定其名称，使用斜杠作为分隔符。
