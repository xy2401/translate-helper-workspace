导入现有来源。模块结构

此页面允许您选择和配置要包含在项目中的模块。

在左侧窗格中， 模块下，潜在可包含在你的项目中的模块中。在此窗格中突出显示某个模块时，其依赖关系将显示在“ 模块依赖关系”下的右侧窗格中。（如果突出显示多个模块，则会显示其聚合依赖项。）

使用复选框选择要包含在项目中的模块。使用此页面上提供的其他控件来重命名模块，还可以拆分或合并它们（参见下文）。

描述

单击此图标可重命名所选（突出显示的）模块。

单击此图标可将多个选定（突出显示）的模块合并为一个新模块。

单击此图标可将选定（突出显示）模块拆分为两部分。在“ 拆分模块”对话框中，选择要分配给新模块的内容根 。
