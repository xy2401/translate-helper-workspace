文件颜色

档案|设置|外观和行为|适用于Windows和Linux的文件颜色
WebStorm |偏好|外观和行为| macOS的文件颜色
按Ctrl + Alt + S

使用此页面设置不同的背景颜色，以区分项目文件和特定范围的文件夹。这些设置适用于以下UI元素：

  • 编辑器标签的标题。

  • 按名称搜索文件或类时的导航列表

  • 项目工具窗口

常见选项

项目

描述

启用文件颜色

选中此复选框可将背景颜色设置应用于导航列表 。

在编辑器选项卡中使用

选中此复选框可将背景颜色设置应用于编辑器选项卡的标题。

在项目视图中使用

选中此复选框可将背景颜色设置应用于“项目”视图。

管理范围

单击此按钮可打开“ 范围”对话框，您可以在其中为各种操作定义自定义范围。

本地色彩

在此区域中，配置要在本地应用的颜色范围关联。

定义后，无法更改颜色范围关联。要为范围重新分配颜色，请删除现有关联并定义新关联。

项目

提示

描述

范围

此只读字段显示应用颜色设置的范围。

颜色

此只读字段显示要应用于相应范围的颜色。

加

单击此按钮可打开“ 添加颜色标签”对话框，您可以在其中配置新的颜色范围关联。

去掉

单击此按钮可删除选定的颜色范围关联。

要么

上移或下移

使用这些按钮对颜色范围关联进行排序，从而确定它们的应用顺序。

分享

单击此按钮可在团队成员之间共享选定的范围 - 颜色关联。

所选关联将相应地移动到“ 共享颜色”区域中的列表。

共享色彩

使用此区域中的控件配置要在团队的所有成员之间共享的颜色范围关联。

定义后，无法更改颜色范围关联。要为范围重新分配颜色，请删除现有关联并定义新关联。
