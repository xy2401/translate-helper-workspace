从命令行查看文件之间的差异

您可以使用IntelliJ IDEA作为命令行工具来查看文件之间的差异。

  • 在命令行中，使用以下语法运行diff命令：

    <IntelliJ IDEA> diff path1 path2

所选文件将在差异查看器中打开。

命令选项如下：

选项

描述

<IntelliJ IDEA>

特定于平台的IntelliJ IDEA启动器。

在macOS和Linux上，建议使用在IntelliJ IDEA安装期间创建的命令行启动程序，或者通过选择Tools |在IntelliJ IDEA主菜单上创建命令行启动器 。请参阅除了在IntelliJ IDEA中工作之外，您还可以从命令行执行以下操作：有关详细信息。

在Windows上，建议将IntelliJ IDEA可执行文件的Path添加到Path环境变量中。

PATH1

文件本地副本的完整路径。

路径2

文件的存储库版本副本的完整路径。

例子

句法
idea.exe diff path1 path2
例
idea.exe diff C： \ M yProject \ R eadme.md C： \ M yProject \ R eadme.md.bak
句法
想法diff path1 path2
例
想法diff~ / MyProject / Readme.md~ / MyProject / Readme.md.bak
