查看方法参数信息

参数提示显示方法和函数中的参数名称。在TypeScript和JavaScript中，参数提示默认显示在编辑器中，使您的代码更易于阅读。

您可以在编辑器中关闭参数提示并在弹出窗口中查看它们。最后，您可以配置WebStorm以显示工具提示，其中包含从建议列表中选择的方法的参数提示以完成。

在编辑器中配置参数提示

默认情况下，参数提示仅针对文字或函数表达式的值显示，但不针对命名对象显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|外观 。将打开“ 外观”页面 。

  2. 单击“ 显示参数名称提示”复选框旁边的“ 配置” （默认情况下选中该复选框）。

  3. 在打开的“ 配置参数名称提示”对话框中，选中“ 选项”区域中的“ 显示所有参数的名称”复选框。

要在编辑器中关闭参数提示

  • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 外观” 。在打开的“ 外观”页面上，清除“ 显示参数名称提示”复选框。

在弹出窗口中查看方法的参数提示

  • 将插入符放置在方法或函数调用的任何位置，然后按Ctrl + P或选择“ 视图”|主菜单上的参数信息 。

在弹出窗口中配置参数提示的外观和行为

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“代码完成” 。将打开“ 代码完成”页面 。

  2. 要显示完整的方法或函数签名而不是所需类型的列表，请选择参数信息区域中的显示完整签名复选框。

  3. 要在一定延迟后自动显示被调用方法或函数的参数类型列表，请选中自动弹出窗口（以毫秒为单位）复选框，并以毫秒为单位指定时间段。

在代码完成后显示参数提示

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“代码完成” 。将打开“ 代码完成”页面 。

  2. 在“ 参数信息”区域中，选中“ 完成时显示参数名称提示”复选框。
