生成和更新版权声明

从头开始创建版权配置文件

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” ，然后单击“ 版权配置文件” 。将打开“ 版权配置文件”页面。

  2. 单击“ 添加”按钮在工具栏上，在打开的“ 创建新版权配置文件”对话框中指定新配置文件的名称。单击“ 确定”后 ，将返回“ 版权配置文件”页面，其中新配置文件将添加到可用版权配置文件列表中。

  3. 指定要生成的版权声明。执行以下操作之一：

    • 键入所需的纯文本。

    • 定义Velocity模板 ，然后单击“ 验证”按钮以检查是否已正确指定模板。有关支持的Velocity变量列表，请参阅版权配置文件 。

  4. 在Regexp中检测注释中的版权文本框，键入字符串以在版权更新期间将版权声明与其他注释区分开来。

    确保此正则表达式与上面指定的版权声明匹配。否则，WebStorm将不会更新版权声明，而是会插入新的通知。

根据现有配置文件的设置创建版权配置文件

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” ，然后单击“ 版权配置文件” 。将打开“ 版权配置文件”页面。

  2. 选择所需的配置文件以继承设置，然后单击“ 复制”按钮在工具栏上。

  3. 在打开的“ 创建新版权配置文件”对话框中，指定新配置文件的名称，然后单击“ 确定” 。您将返回“ 版权配置文件”页面，其中新配置文件将添加到可用版权配置文件列表中。

  4. 查看和编辑配置文件设置。从头开始创建配置文件时继续。

导入现有版权声明文本

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” ，然后单击“ 版权配置文件” 。将打开“ 版权配置文件”页面。

  2. 单击“ 导入”按钮在工具栏上。

  3. 在打开的对话框中 ，选择引用版权配置文件的%.ipr%文件的位置以及所需的通知定义。

  4. 从出现的选择配置文件导入列表中，选择所需的版权配置文件。

  5. 在打开的“ 创建新版权配置文件”对话框中，指定新配置文件的名称，然后单击“ 确定” 。您将返回“ 版权配置文件”页面，其中新配置文件将添加到可用版权配置文件列表中。

  6. 从头开始创建配置文件时继续。

放弃版权资料

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” ，然后单击“ 版权配置文件” 。将打开“ 版权配置文件”页面。

  2. 选择要放弃的配置文件，然后单击“ 删除”按钮在工具栏上。

添加版权声明

  1. 在编辑器中打开所需的文件。

  2. 按Alt + Insert 。

  3. 从编辑器中显示的列表中，选择“ 版权” 。

  4. 以下步骤取决于文件属于某个范围以及“ 版权”页面中“ 默认项目版权”字段的值 。

    • 如果要生成版权声明的文件属于某个范围，则根据版权配置文件设置生成版权声明。

    • 如果“ 默认项目版权 ”字段的值为“ 无版权”，并且您要生成版权声明的文件不属于任何范围，则WebStorm会提示您配置版权设置：

      单击“ 确定”以打开“ 版权”页面并配置所需的设置。

请注意，如果文件属于多个范围，则会对这些范围进行上下颠倒检查。这样做，使用此文件的第一个范围被声明为正确的范围，并使用其版权配置文件。

更新版权声明

  1. 在“ 项目”工具窗口中，选择要更新版权声明的文件或目录，或者只需在编辑器中打开所需的文件。

  2. 右键单击所选内容，然后在上下文菜单中选择“ 更新版权 ”。

    要正确更新版权声明，请确保版权配置文件中指定的关键字是通知定义的一部分。否则，WebStorm将不会检测版权声明，而是更新现有通知，而不是更新现有通知。

    您还可以在提交更改时更新版权声明。
