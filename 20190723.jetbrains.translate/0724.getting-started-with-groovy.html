IntelliJ IDEA 2019.1帮助

运行，调试和测试Groovy

您可以像创建任何其他项目一样创建，导入，测试和运行Groovy应用程序。

在开始使用Groovy之前，请确保在IntelliJ IDEA中启用了Groovy插件。

创建一个Groovy项目

  1. 在“ 项目向导”中 ，选择“ Groovy” 。

  2. 指定以下设置：

    • Project SDK - 指定项目SDK。

    • Groovy库 - 指定Groovy SDK。 IntelliJ IDEA需要标准的Groovy SDK布局，该布局随http://groovy-lang.org/download.html上提供的官方发行版一起提供。下载SDK，将其解压缩到任何目录中，并将此目录指定为库主目录。

    • 其他库和框架 - 如果需要，请指定Groovy支持的其他库和框架。

    • Java EE版本 - 选择适当的Java EE版本。

  3. 单击下一步 。

  4. 指定项目信息，然后单击“ 完成” 。

要导入Groovy项目，请按照导入项目部分中的步骤操作。

检查项目中的Groovy版本

您可以检查Groovy SDK IntelliJ IDEA的哪个版本在项目中使用。

  1. 从主菜单中选择File |项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 项目结构”对话框的“ 平台设置”下 ，选择“ 全局库” 。

    如果需要检查模块中使用的Groovy SDK版本，请选择模块 ，模块名称，单击“ 依赖关系”选项卡。

更改项目中的现有Groovy SDK版本

  1. 在项目结构 （ Ctrl + Shift + Alt + S ）对话框中，从全局库和模块依赖项（ 模块页面）中删除（ Ctrl + Y ）现有的Groovy SDK版本。

  2. 删除Groovy SDK后，IntelliJ IDEA会显示一个弹出窗口，建议添加Groovy SDK。

    单击该链接并设置SDK。

  3. 如果由于某种原因，IntelliJ IDEA未显示弹出窗口，请在“ 项目”工具窗口中右键单击项目名称，然后在上下文菜单中选择“ 添加框架支持” 。

  4. 在打开的对话框中，从技术列表中选择Groovy并添加新库 。

    IntelliJ IDEA创建一个全局级库。

生成Groovy文档

您可以为构成Groovy语言的Groovy和Java类生成API文档。

  1. 从主菜单中，选择工具|生成GroovyDoc 。

  2. 在打开的对话框中，指定文档和其他可用选项的输出目录，然后单击“ 启动” 。

运行Groovy应用程序

输入代码后，您可以通过IntelliJ IDEA运行它，或使用交互式Groovy控制台进行快速代码评估。

  1. 在编辑器中打开您的应用程序。

  2. 按Shift + F10执行应用程序。或者，在编辑器的左侧排水沟中，单击图标，然后选择运行'名称' 。

  3. 在“ 运行”工具窗口中查看结果。

你也可以点击主工具栏上的图标，用于运行您的应用程序。

调试Groovy应用程序

  1. 在编辑器中打开Groovy应用程序。

  2. 在左侧装订线中，为要调试的代码行设置断点。调试器知道Groovy语法，如果需要，可以在断点上计算表达式。

    有关断点的更多信息，请参阅“ 断点”部分。

  3. 如果需要，可以访问运行/调试配置 （运行|编辑配置）并调整设置，但通常默认设置足以成功启动和完成调试会话。

  4. 按Shift + F9 。或者，在主工具栏上单击图标以启动调试过程。

  5. 在“ 调试”工具窗口中评估结果。

    有关如何在“ 调试”工具窗口中使用选项的信息，请参阅调试工具窗口参考 。

测试Groovy应用程序

您可以使用JUnit 4和JUnit 5测试框架测试Groovy应用程序。

  1. 在编辑器中打开一个类，要为其创建测试并将光标放在包含类声明的行中。

  2. 按Ctrl + Shift + T并选择“ 创建新测试” 。

  3. 在打开的对话框中，指定测试设置，然后单击“ 确定” 。

  4. 在编辑器中打开测试，添加代码并按Ctrl + Shift + F10或右键单击测试类，然后从上下文菜单中选择Run'test name' 。

  5. IntelliJ IDEA会自动为测试创建运行/调试配置，但如果要编辑配置中的设置，请单击“运行”|在主菜单上编辑配置 。

  6. 在“ 运行/调试配置”对话框的右侧，指定测试套件的设置，然后单击“ 确定” 。配置具有标准选项，您可以在“ 执行测试”部分中找到更多详细信息。

  7. 在主工具栏上，单击用于运行测试的图标。

  8. 在“ 运行”工具窗口中评估结果。

您可以在Groovy测试和测试主题之间导航 （ 导航|测试/测试主题 ）。如果测试类尚不存在，IntelliJ IDEA建议创建一个。
