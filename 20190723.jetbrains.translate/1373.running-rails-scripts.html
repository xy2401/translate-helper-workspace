运行Rails脚本

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

IntelliJ IDEA使您可以使用单个菜单命令执行Rails脚本。默认情况下，IntelliJ IDEA建议驻留在Rails应用程序的[<project name>]/script目录中的[<project name>]/script 。

每个脚本的结果都显示在“运行”工具窗口的“ 运行Rails脚本”选项卡中，您可以在其中查看脚本输出，必要时更改参数，然后重新运行脚本。

运行Rails脚本

  1. 在主菜单上，选择“ 工具”|运行Rails脚本 。弹出“运行Rails脚本”对话框。

  2. 在“ 脚本”字段中，指定所需的脚本。如果脚本位于[<project name>]/script目录中，请仅键入其名称。如果脚本位于另一个目录中，请键入其完全限定名称。
    或者，单击“浏览”按钮 （ Shift + Enter ）并在打开的对话框中选择所需位置。请注意，此对话框仅显示[<project name>]/script目录的内容。

  3. 在参数中 ，如果有的话。参数应该用空格分隔。
    或者，按Shift + Enter打开“ 编辑参数”对话框，然后键入参数列表。

  4. 单击运行 。

使用新参数重新运行Rails脚本

  1. 在“运行”工具窗口中，单击所需的“运行Rails脚本”选项卡。

  2. 在“运行”工具窗口的工具栏中，单击 。将显示“ 编辑命令行参数”对话框。

  3. 在“ 编辑命令行参数”对话框中，键入用空格分隔的新参数，然后单击“ 确定” 。

  4. 在“运行”工具窗口的工具栏中，单击 ，或按Ctrl + F5 。
