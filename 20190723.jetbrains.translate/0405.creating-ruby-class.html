创建Ruby类

IntelliJ IDEA使您可以创建空的Ruby文件或生成类和模块。要创建新的Ruby文件，类或模块，请执行以下操作：

  1. 在“项目”工具窗口中，选择要在其中创建类的目录，然后按Alt + Insert 。

  2. 从弹出菜单中，选择Ruby File / Class 。

  3. 在“ 新建Ruby文件/类”对话框中，执行以下操作之一：

    • 要创建新的Ruby文件，请在“ 名称”字段中输入其名称。

    • 要创建Ruby类，请将Kind选项切换为Class，并在Name字段中指定类名。

      IntelliJ IDEA允许您创建模块前面的类。如果需要，请在类名前加上模块名称：

      在这种情况下，创建的文件将如下所示：

      请注意，Ruby类和模块的名称应以大写字母开头。

    • 要创建Ruby模块，请将Kind选项切换为Module，并在Name字段中指定模块名称。

    单击确定 。
