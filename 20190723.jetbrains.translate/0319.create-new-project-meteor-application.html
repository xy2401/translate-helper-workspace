当您在左侧窗格中选择静态Web ，然后从右侧窗格的列表中选择Meteor App时，“ 项目类别和选项”对话框的右侧窗格如下所示。

安装并启用Meteor插件时，仅在Ultimate版本中支持该功能。 该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

项目

描述

项目名

在此字段中，指定项目的名称，默认情况下，它是位置字段中指定的文件夹的名称。

项目位置

在此字段中，指定项目文件夹的路径，其中将存储与项目相关的文件。

流星

在此文本框中，指定Meteor可执行文件的位置（请参阅安装Meteor ）。

模板

从此下拉列表中，选择要生成的样本。要生成基本项目结构，请选择“ 默认”选项。

文件名

在此文本框中，键入将生成的相互关联的.js ， .html和.css文件的名称。仅当从“ 模板”下拉列表中选择“ 默认”样本类型时，该字段才可用。
