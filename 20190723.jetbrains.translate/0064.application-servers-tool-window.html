Application Servers工具窗口

查看|工具窗口|应用服务器

要使此工具窗口可用，项目中必须有服务器运行/调试配置 ，或者必须在IntelliJ IDEA中注册云用户帐户。

Application Servers工具窗口允许您在应用程序服务器和云平台上管理应用程序。您可以启动和停止服务器运行/调试配置并连接到云平台，部署和取消部署应用程序工件以及执行其他相关任务。

通过工具栏图标和上下文菜单命令访问所有可用功能。

  • 用于服务器运行配置的图标和命令
  • 服务器工件的图标和命令
  • 云用户帐户的图标和命令
  • 云应用程序的图标和命令
  • 部署状态图标

另请参阅使用服务器运行/调试配置和使用云平台 。

用于服务器运行配置的图标和命令

图标

命令

描述

运行/连接

在运行模式下启动选定的运行/调试配置。对于本地配置，通常会启动相应的服务器。对于远程配置，IntelliJ IDEA将连接到服务器。

调试

在调试模式下启动选定的运行/调试配置。

停止/断开

停止所选的运行/调试配置。对于本地配置，通常会停止相应的服务器。对于远程配置，IntelliJ IDEA将与服务器断开连接。

部署全部

部署与所选运行/调试配置关联的所有工件。

编辑配置

编辑所选运行/调试配置的设置。

文物

编辑所选运行/调试配置的部署列表。（将打开要部署的工件对话框 。）

图标

命令
