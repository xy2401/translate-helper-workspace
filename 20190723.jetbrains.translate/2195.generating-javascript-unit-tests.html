此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

PyCharm与最流行的JavaScript单元测试框架集成，并为编写JavaScript单元测试提供了丰富的编码帮助。要获得支持框架的编码帮助，请下载所需的框架并将其配置为PyCharm JavaScript库 。

创建JavaScript单元测试

  1. 为src文件夹旁边的测试（ 文件|新建|目录 ）创建单独的文件夹。此文件夹的名称取决于您使用的测试框架。

  2. 在“ 项目结构”页面上，将新文件夹标记为测试文件夹 ，请参阅在内容根 目录中 配置文件夹 。

  3. 对于每个生产文件，通过选择（ 文件夹的上下文菜单上的“ 文件”|“新建”|“JavaScript文件”来创建单独的测试文件以进行测试。测试文件名的格式取决于您使用的测试框架。
