IntelliJ IDEA 2019.1帮助

Python语言支持

以下内容仅在安装并启用Python插件时有效！

要在IntelliJ IDEA中使用Python ，请至少安装一个Python解释器。

支持的版本

  • Python 2： 2.7版

  • Python 3：从版本3.5到版本3.7

IntelliJ IDEA支持Python 3.5和（自2016.3）Python 3.6，支持以下内容：

  • PEP-484 - 类型提示

  • PEP-448 - 附加拆包概括
  • PEP-492 - 具有异步和等待语法的协同程序
  • PEP-526 - 变量注释的语法

  • PEP-498 - 文字字符串插值

  • PEP-515 - 数字文字中的下划线

  • PEP-525 - 异步发电机

  • PEP-530 - 异步理解

IntelliJ IDEA中的Python支持

IntelliJ IDEA提供以下功能来帮助使用Python：

  • 专用模块类型 。

  • 能够配置解释器。 。

  • Python控制台 。

  • 运行/调试Python配置和Python远程调试 。

  • 代码洞察力

  • 代码检查 。

  • 意图行动 。

  • 代码完成并解决。

  • 内置代码格式化程序和单独的Python 代码样式设置 。

  • 在Python代码中查找用法 。

  • 测试框架 。

  • 快速文档 。

  • 识别Python 文档注释 。

  • 配置Python调试器 。

  • Python类的UML类图 。
