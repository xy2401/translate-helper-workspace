选择要包含在传输对象中的访问者字段

EJB工具窗口|实体bean的上下文菜单|新的|转移对象

使用此对话框选择要包含在实体bean的传输对象中的方法。

项目

提示

描述

使用此按钮可以按升序或降序排序方法。

显示类

如果bean引用了祖先类，单击此按钮将显示这些类及其成员。否则，该按钮不可用。

此功能取决于EJB规范。

展开全部

单击此按钮可展开所有节点。

全部收缩

单击此按钮可折叠所有节点。

有问题的实体bean中的成员列表

从列表中，选择所需的方法。使用Ctrl和Shift键进行多项选择。

传输对象类名

在此字段中，指定与特定模式匹配的传输对象类的名称。接受IntelliJ IDEA建议的名称或根据需要进行更改。

在<class name>接口中生成getter / setter

选中这些复选框以获得生成的选定字段的访问方法。您可以接受访问者方法的建议名称，也可以根据需要进行更改。

复制JavaDoc

选中此复选框可对包含的生成的传输对象进行JavaDoc注释（如果有）。
