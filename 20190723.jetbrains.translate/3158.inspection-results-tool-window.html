检验结果工具窗口

查看|工具窗口|检查结果

  • 只有在通过Code |打开工具窗口时，才能以这种方式访问工具窗口检查代码 。

  • 通过单击“ 关闭”按钮手动停用工具窗口后 ，工具窗口只能通过Code |再次使用检查代码 。

“ 检查结果”工具窗口在单独的选项卡上显示检查结果

每个选项卡的左侧窗格显示检查发现问题的树视图。右侧窗格显示左侧窗格中所选项目的摘要信息。

在本页面：

  • 工具栏按钮
  • 上下文菜单命令
  • 检查报告

工具栏按钮

项目

捷径

描述

按Ctrl + F5

单击此按钮以运行检查并在同一选项卡上显示结果。

按Ctrl + Shift + F4

单击此按钮可关闭当前选项卡或工具窗口。

Ctrl + NumPad Plus

展开所有节点

Ctrl + NumPad -

折叠所有节点

按Ctrl + Alt +向下

导航到下一个项目。

按Ctrl + Alt +向上

导航到上一个项目。

如果启用此切换，则问题将分组为错误和警告。否则，问题按检查分组。

打开或关闭目录分组。

启用此切换后，不会显示已解决和已排除的项目。

打开或关闭Autoscroll to source选项。当选项打开并且您选择一个项目时，相应的源文件将在编辑器中打开，并突出显示相应的片段。

单击此按钮可将检查结果导出为XML或HTML格式 。

单击此按钮可编辑当前检查设置 。

Alt + Enter键

单击此按钮可通过从列表中选择一个可用的快速修复来解决所选检验项的问题 。

上下文菜单命令

项目

捷径

描述

跳转到源

F4

在编辑器中打开包含所选问题的文件，并将光标放在相应代码片段的开头。

排除

删除

从进一步检查中排除所选项目。排除的节点显示为删除线。如果过滤器切换如果启用，则排除的节点将被隐藏。

包括

插入

在结果列表中包含以前排除的项目。所有嵌套的子元素也包括在内。

Alt + Enter键

选择一个建议的解决方案。

抑制问题/
解决课堂问题

禁止检查所选问题或所选类别。

编辑设置

在“ 错误”对话框中更改所选检查或检查组的设置。

禁用检查

在结果的活动选项卡中禁用所选检查的警报。如果过滤器切换如果启用，则隐藏禁用检查的节点。

另请参阅禁用和禁止检查 。

运行检查

重新运行所选检查并在新选项卡上显示结果。

检查报告

在左侧窗格中选择检查节点时，检查报告将显示在结果选项卡的右侧窗格中。该报告可能包括以下内容：

  • 问题解决：每个可用解决方案的按钮。单击按钮可调用相应的修复程序。如果没有按钮，则必须自行解决问题。

  • 抑制：单击此按钮可显示检查抑制选项列表。

  • 问题简介：问题的简要说明。

  • 禁用检查：在结果的活动选项卡中禁用所选检查的警报。如果过滤器切换如果启用，则隐藏禁用检查的节点。

    另请参阅禁用和禁止检查 。

  • 在...上运行检查：重新运行选定的检查并在新选项卡上显示结果。
