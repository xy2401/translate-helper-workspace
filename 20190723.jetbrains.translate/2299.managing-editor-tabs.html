管理编辑器选项卡

每次打开文件进行编辑时，都会在编辑器窗口中的活动编辑器选项卡旁边添加一个专用选项卡。

PyCharm可以同时限制编辑器中打开的选项卡数量。当选项卡的数量达到其限制时，编辑器将根据编辑器中定义的选项卡关闭策略关闭选项卡一般| PyCharm设置的编辑器选项卡页面（ Ctrl + Alt + S ）默认情况下，选项卡限制为10，但您可以根据需要进行更改。

与管理编辑器选项卡相关的所有命令均可从以下位置获得：

  • 窗口|编辑器选项卡菜单。

  • 选项卡的上下文菜单。

配置编辑器选项卡的行为

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 编辑器选项卡” 。将打开“ 编辑器选项卡”页面 。

  2. 更改允许的最大选项卡数：在选项卡限制字段中键入要一次打开的编辑器选项卡所需的最大允许数。
    如果选项卡限制等于1，则将禁用选项卡。如果希望编辑器永远不关闭选项卡，请键入一些无法访问的编号。
    使用禁用的选项卡，使用View |最近的文件 （ Ctrl + E ）命令可以在文件之间快速切换。

  3. 默认情况下，选项卡标题显示在编辑器的顶部，但您可以根据需要更改其位置，并在编辑器的底部，左侧或右侧显示标题。请注意，选项卡标题放置是适用于所有项目的全局设置。
    使用“ 放置”选择器选择标签标题的所需位置。
    要完全禁用编辑器选项卡，请从“ 放置”选择器中选择“ 无 ”。

在编辑器选项卡之间导航

从当前选项卡导航到下一个或上一个选项卡

此方法允许在编辑会话需要时从一个选项卡跳转到另一个选项卡。当您在编辑器选项卡之间移动时，PyCharm会记住每个打开的文件中的插入位置。

  • 右键单击当前编辑器选项卡， 然后从上下文菜单中选择“ 选择下一个/上一个选项卡 ”

  • 按Alt + Right或Alt + Left 。这样，焦点移动到位于活动编辑器选项卡右侧或左侧的编辑器选项卡。

  • 按Ctrl + Tab键以使用切换器 。

来回浏览所访问标签的历史记录

这种方法使您可以在导航历史中来回移动，就像在Web浏览器中一样。当您在编辑会话期间从一个文件移动到另一个文件时，PyCharm会跟踪访问过的位置并使您能够返回，使用Navigate |后退/前进命令。

在macOS计算机上，您还可以使用三指从右到左和从左到右的滑动手势。

  • 在主工具栏上，单击要么 。

  • 选择导航|从主菜单返回/前进 。

  • 按Ctrl + Alt + Left或Ctrl + Alt + Right 。

  1. 执行以下操作之一：

    • 选择窗口|编辑器标签|从主菜单中显示隐藏选项卡 。

    • 点击

    将显示已打开但不可见的所有选项卡的列表。因此，在明亮的背景上显示当前可见的选项卡的名称;主窗口外部的选项卡名称显示在较暗的背景上。

  2. 单击所需编辑器选项卡的名称：

命令显示所有选项卡和图标仅在以下情况下可用：

  • 在编辑器中选中“在单行中显示选项卡”复选框一般|编辑器选项卡 PyCharm设置页面（ Ctrl + Alt + S ）。

固定和取消固定标签

达到指定的选项卡限制时，PyCharm将在您打开新文档时关闭旧选项卡。要防止选项卡自动关闭，您可以固定此选项卡。此外，当您关闭编辑器选项卡时，您可以选择打开固定选项卡并仅关闭未固定选项卡。

固定标签时，其上有一个特殊标记：

当您打开文件以供参考而不是编辑时，选项卡的固定状态很有用。

要固定或取消固定选项卡，请在其上单击鼠标右键，然后从上下文菜单中选择“ 固定选项卡”或“ 取消 固定选项 卡 ”。

拆分和拆分编辑器窗口

拆分编辑器窗口将其划分为独立窗格。您可以根据需要将编辑器窗口拆分为多个窗格，每个窗格都包含多个选项卡。

每个窗格都可以垂直或水平分配。因此，拆分有助于创建不同的编辑器布局并将选项卡组织到组中。例如，您可以滚动浏览文件的一部分，同时可以查看其他部分的行。

分割和取消分割编辑器窗口

  1. 切换到所需的选项卡，然后右键单击它。

  2. 要拆分创建文件副本的编辑器选项卡，请从上下文菜单中选择“ 垂直 拆分”或“ 水平拆分” 。

  3. 要分割编辑器选项卡而不复制文件，请从上下文菜单中选择“ 向右移动”或“ 向下移动” 。

  4. 要更改拆分器方向，请从上下文菜单中选择“ 更改拆分器方向”。

  5. 要删除拆分器，请从上下文菜单中选择以下命令之一：

    • 要在活动选项卡中删除拆分，请选择Unsplit 。

    • 要在所有打开的编辑器选项卡中删除拆分，请选择Unsplit All 。

分离编辑器选项卡

您可以分离编辑器选项卡，并将它们移动到单独的框架中。分离的编辑器选项卡可以拆分或拆分 ，您可以在拆分窗格之间移动编辑器选项卡。

要分离编辑器选项卡，请执行以下操作之一：

  • 将此选项卡拖到主窗口之外。您也可以使用Alt +鼠标 。出现预览缩略图：

  • 按Shift + F4 。

编辑器选项卡的内容将在单独的框架中打开。

按字母顺序排列选项卡

无论选项卡位置如何，都可以使用编辑器选项卡的字母排序。但是，当在Window |上选中Show row in single row check命令时，可以使用顶部和底部放置的字母顺序编辑器标签|选项卡放置菜单。

按字母顺序排序编辑器选项卡

  1. 右键单击编辑器选项卡。

  2. 选择按文件名排序选项卡 。

    如果选中此检查命令，则选项卡标题按字母顺序显示。否则，编辑器选项卡标题将显示在相应文件的打开顺序中。
