软件开发工具包

软件开发工具包或SDK是为特定软件框架开发应用程序所需的工具集合。例如，要使用Java开发应用程序，您需要一个Java SDK（JDK）。SDK包含二进制文件，二进制文件的源代码和源代码的文档。对于Java，SDK还包含注释。

通常，SDK是全局的。这意味着可以在多个项目和模块中使用一个SDK。创建新项目并为其定义SDK后，可以在此项目中配置模块以继承其SDK。您还可以单独为每个模块指定SDK。有关更多信息，请参阅更改模块SDK 。

支持的SDK

  • Java SE开发套件（JDK）

  • Java Micro Edition（ME）SDK
  • Android SDK

  • Flex和AIR SDK

  • Google App Engine SDK

  • IntelliJ平台插件SDK（用于开发IntelliJ IDEA插件）

定义SDK

定义SDK意味着让IntelliJ IDEA知道您计算机上的哪个文件夹中安装了必要的SDK版本。该文件夹称为SDK主目录。

管理全局SDK

  1. 从主菜单中，选择File |项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 平台设置”部分下，选择“ SDK” 。

  3. 要添加新SDK或新SDK版本，请单击按钮（ Shift + Enter ），选择SDK类型并指定其安装目录。

  4. 要删除SDK，请在列表中选择它，然后单击按钮（ Alt + Delete ）。

更改项目SDK

  1. 从主菜单中，选择File |项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 项目设置”部分下，选择“ 项目” 。

  3. 从Project SDK列表中，选择另一个SDK或SDK版本。

    如果尚未在IntelliJ IDEA中定义必要的SDK，请单击“ 新建”并指定其主目录。

更改模块SDK

  1. 从主菜单中，选择File |项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 项目设置”部分下，选择“ 模块” 。

  3. 选择所需的模块，然后单击“ 依赖关系” 。

  4. 从“ 模块SDK”列表中，选择要使用的其他SDK或SDK版本。

    如果尚未在IntelliJ IDEA中定义必要的SDK，请单击“ 新建”并指定其主目录。

如果希望模块继承项目SDK，请从“ 模块SDK”列表中选择“ 项目SDK”选项。
