Perforce选项对话框

此功能仅在Ultimate版本中受支持。

档案|设置|版本控制|配置|适用于Windows和Linux的Perforce
IntelliJ IDEA |偏好|版本控制|配置| Perforce for macOS
按Ctrl + Alt + S

在此对话框中，配置与指定Perforce服务器的连接。

项目

描述

Perforce在线

选中此复选框以启用与Perforce服务器的连接。

如果存在连接但服务器没有响应（例如，因为服务器当前正在备份），您可以通过清除此复选框来禁用此类尝试，或者IntelliJ IDEA将建议在超时后执行此操作。之后，将禁用版本控制特定的操作。

使用P4CONFIG或默认连接

使用P4CONFIG环境变量或默认Perforce客户端连接中的服务器信息和用户凭据。否则，请指定端口，用户，客户端和密码。

字符集

选择与服务器上设置的字符集对应的字符集。

转储Perforce命令到IDEA_Home \ bin \ p4.output

在指定文件中记录Perforce命令。

使用登录验证

切换身份验证。

尝试以静默方式登录

跳过提示对话框。当需要登录身份验证时，此选项有效。

使用本机Perforce API

使用特殊库加速与服务器的连接。

P4可执行文件的路径

指定Perforce客户端可执行文件的路径。

测试连接

确保已建立与Perforce服务器的连接。

显示分支历史记录

在显示“文件历史记录”的对话框中查看文件的分支。

使用多个分支时，建议启用此选项以正确显示文件分支。
