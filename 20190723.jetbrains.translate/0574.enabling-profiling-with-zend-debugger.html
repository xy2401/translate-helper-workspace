使用Zend Debugger启用性能分析

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

Xdebug探查器包含在Zend Debugger工具中。因此，您只需要下载，安装和启用Xdebug本身。

要使用Zend Debugger启用分析，请执行以下常规步骤：

  • 配置Zend调试器
  • 配置从浏览器切换探查器的方法

配置Zend调试器

  1. 下载并安装 Zend Debugger工具。

  2. 将Zend Debugger与PHP引擎集成 。

  3. 将Xdebug与IntelliJ IDEA集成 。

配置从浏览器切换探查器的方法

要指定GET / POST或COOKIE参数，请执行以下操作之一：

  • 生成小书签以切换调试器。这些小书签将显示在浏览器的工具栏上。它们提供对调试器cookie的控制，通过它们，您将激活和停用调试器。

    1. 根据浏览器类型，通过执行以下操作之一在浏览器中启用“ 书签”工具栏 ：

      • 在Firefox中 ，选择View |工具栏|书签工具栏 。

      • 在Chrome中 ，选择“ 书签”|显示书签栏 。

    2. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，导航到“ 语言和框架”| PHP |调试 。

    3. 在打开的Debug页面上，单击Use debugger bookmarklets以从您喜欢的浏览器链接启动调试器 。

    4. 在打开的Zend Debugger和Xdebug bookmarklets页面上，检查调试引擎设置，然后单击Generate 。生成列出的与调试相关的操作的书签。

    5. 将生成的链接拖动到浏览器中的书签工具栏。

  • 手动指定值。
