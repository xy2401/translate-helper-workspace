介绍

IntelliJ IDEA Ultimate中的AspectJ支持基于以下与IDE捆绑的插件 ：

  • Spring AOP / @ AspectJ

  • AspectJ支持

您可以添加到Java模块的AspectJ方面允许您在模块级别微调AspectJ编译器的使用。

AspectJ支持概述

IntelliJ IDEA Ultimate中的AspectJ支持包括：

  • 能够以两种形式创建方面：作为.aj文件和包含使用@Aspect注释的类的.java文件。

  • 编写方面代码时的编码辅助（包括代码完成 ）。对于注释风格方面，编码帮助是完整的;对于代码样式方面，协助仅限于类型间声明 。

  • 能够执行基本方面重构，例如重命名和移动 ，以及推送ITDs重构类型间声明。

  • 与AcpectJ编译器ajc集成，您可以直接从IDE运行。（此编译器是AspectJ发行版的一部分，您可以从AspectJ网站下载。）

  • 能够在项目级别配置ajc ，并可选择在单个模块级别微调其使用。

使用AspectJ支持的概述

AspectJ特有的任务概述如下。

  1. 确保：

    • 您正在使用IntelliJ IDEA的终极版。Community Edition不支持AspectJ。

    • Spring AOP / @ AspectJ和AspectJ支持插件已启用。请参阅启用AspectJ支持插件 。

  2. 下载并安装AspectJ。

  3. 创建一个包含aspectjrt.jar的库，并将此库添加到您要在其中开发方面的模块的依赖项中。安装AspectJ后，您可以在<AspectJ installation directory>\lib找到aspectjrt.jar 。请参阅为aspectjrt.jar创建库 。

  4. 创建方面文件并开发代码。请注意，支持代码和注释样式方面。

  5. 如有必要， 重构方面代码 。

  6. 要使用AspectJ编译器，请配置编译器设置 。编译器（ ajc ）位于aspectjtools.jar ，它位于<AspectJ installation directory>\lib 。

  7. 要在各个模块的级别上微调ajc的使用，请将AspectJ方面添加到相应的模块，并相应地调整构面设置 。
