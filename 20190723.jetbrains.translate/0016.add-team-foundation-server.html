IntelliJ IDEA |偏好|版本控制|适用于macOS的TFS
按Ctrl + Alt + S

单击“ 管理TFS服务器和工作空间”对话框中的“ 添加”按钮时，将打开该对话框。使用此对话框指定和编辑用于建立与TFS服务器的连接的参数。

项目

描述

地址

在此文本框中，键入要连接到的TFS服务器的URL地址。

验证

在此字段中，指定用于访问服务器的身份验证协议。TFS使用NTLM身份验证 ，因此本机Windows应用程序（即Microsoft Team Explorer ）使用系统凭据进行静默身份验证。由于Java Runtime构成的限制，IntelliJ IDEA用户必须始终指定其用户名和密码。

要通过OAuth进行身份验证（Windows Live ID ），请从“ 身份 验证”下拉列表中选择“ 备用 ”。

用户名

在此文本框中，键入您的TFS用户名。该字段可用于NTLM）和备用身份验证类型。

域

在此文本框中，键入TFS服务器所在的网络域的名称。该字段可用于NTLM）身份验证类型。

密码

在此文本框中，键入您的TFS密码。该字段可用于NTLM）和备用身份验证类型。

存储密码

选中此复选框可让IntelliJ IDEA记住指定的密码。
