IntelliJ IDEA 2019.1帮助

Vaadin入门

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA允许您使用优化Web应用程序开发的Vaadin框架创建和管理项目。请注意，在开始创建项目之前，您需要确保在计算机上下载了Vaadin SDK，并在IntelliJ IDEA中启用了Vaadin插件。

  • 使用Project Wizard创建Vaadin项目
  • 运行Vaadin应用程序
  • 调试Vaadin应用程序

使用Project Wizard创建Vaadin项目

  1. 打开Project Wizard，在Java Enterprise下选择Vaadin框架。

  2. 指定以下设置：

    • Project SDK - 指定项目SDK。

    • JavaEE版本 - 指定JavaEE的版本或使用默认选项。

    • Application Server - 指定应用程序服务器或使用默认服务器。IntelliJ IDEA为您的Vaadin应用程序创建运行配置。

    • 版本 - 对于IntelliJ IDEA 13及更高版本，您可以选择Vaadin 6或7版本。

    • Vaadin分发 - 在您的机器上输入Vaadin安装位置。
      如果未安装Vaadin，则会显示带有相应链接的警告消息。单击该链接，下载“All-in-One Archive”ZIP文件并将其解压缩到所需位置。

    • 创建示例应用程序 - 选中此复选框以输入示例应用程序的名称或使用默认名称。

  3. 单击下一步 。

  4. 指定项目信息，然后单击“ 完成” 。

IntelliJ IDEA使用Web应用程序和所有必要的配置创建项目。

运行Vaadin应用程序

由于指定了应用程序服务器，因此可以运行您的应用程序并在默认浏览器http://localhost:8080/查看输出。

调试Vaadin应用程序

IntelliJ IDEA可让您轻松启动项目的调试会话。

让我们选择一个调试操作，例如， 切换线断点 。

  1. 将插入符号放在源代码的所需行上。按Ctrl + F8或从主菜单中选择Run |切换线断点 。

  2. 按Shift + Alt + F9开始调试会话。

  3. 在Debugger控制台中查看结果。
