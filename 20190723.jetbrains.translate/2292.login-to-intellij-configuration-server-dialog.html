登录IntelliJ配置服务器对话框

  • 在你开始之前
  • 概观
  • 控制

在你开始之前

按照管理插件中的说明在Plugins页面上安装IntelliJ Configuration Server插件。

概观

IntelliJ配置服务器用于存储团队使用的常用IDE设置。这可确保所有团队成员都具有统一的IDE外观。

对话框打开：

  • 安装插件后第一次启动PyCharm时。在这种情况下，对话框在下次启动时包含一个附加区域。

  • 单击IntelliJ配置服务器状态按钮时在会话期间的状态栏上，然后单击打开的对话框中的“ 登录”按钮。

如有必要，使用此对话框登录IntelliJ配置服务器并指定代理设置以访问服务器。

控制

项目

描述

登录

在此文本框中，键入您的JetBrains帐户登录名。

密码

在此文本框中，键入您的JetBrains帐户密码。

创建或管理您的JetBrains帐户

单击此链接可创建JetBrains帐户或编辑您的帐户（如果已有）。

使用HTTP代理

如果要通过代理服务器访问IntelliJ配置服务器，请选中此复选框。

主机名

在此文本框中，键入代理主机名。

端口号

在此文本框中，键入代理端口号。

代理验证

如果要使用登录名和密码访问HTTP代理服务器，请选中此复选框。

登录

在此文本框中，键入您的代理登录。

密码

在此文本框中，键入您的代理密码。

记住密码

如果选中此复选框，PyCharm将记住您的代理密码，并在将来登录时自动填写密码文本框。

在下一个启动区域

仅当PyCharm第一次启动时，该区域才会显示在对话框中。使用此区域可配置将来登录IntelliJ配置服务器的方式。

可以使用以下控件：

项目

描述

显示登录对话框

选择此选项可在下一次PyCharm启动期间显示“ 登录到IntelliJ配置服务器”对话框。

默默登录

选择此选项以登录服务器而不要求登录和密码。

不要登录

如果您不打算登录IntelliJ配置服务器，请选择此选项。

您可以在会话期间随时重新配置此行为。

登录

单击此按钮登录IntelliJ配置服务器。

跳跃

单击此按钮可启动PyCharm会话，而无需登录IntelliJ配置服务器。

您可以在会话期间随时登录服务器。
