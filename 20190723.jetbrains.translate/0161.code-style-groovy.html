标签和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 按Tab键

    • 缩进

    • 用于代码重新格式化

  • 清除该复选框后，IntelliJ IDEA使用空格而不是制表符。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如果需要）空格组成，而由对齐定义的缩进部分仅由空格构成。

  • 如果清除此复选框，则仅使用选项卡。这意味着符合指定选项卡大小的一组空格会自动替换为选项卡，这可能会导致细微对齐。

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签大小

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

在空行上保持缩进

如果选中此复选框，则IntelliJ IDEA将在空行上保留缩进，就好像它们包含一些代码一样。
如果清除此复选框，IntelliJ IDEA将删除制表符和空格。

标签缩进

在此字段中，指定要在标签语句之前的下一行插入的空格数（或选中“ 使用选项卡字符”复选框）。

绝对标签缩进

如果选中此复选框，则标签缩进将计为绝对空格数。否则，标签缩进相对于先前的缩进级别计算。

空间

使用此选项卡可以指定代码中的空格位置。要让IntelliJ IDEA自动在某个位置插入空格，请选中列表中此位置旁边的复选框。结果显示在“ 预览”窗格中。

包裹和括号

在此选项卡中，自定义代码样式选项，IntelliJ IDEA将在重新格式化源代码时应用这些选项。左侧窗格包含异常列表（ 重新格式化时保留 ），以及各种代码构造（列表，语句，操作，注释等）的放置和对齐选项右侧窗格显示预览。

对齐优先于缩进选项。

右边距（列）

使用Hard wrap at字段指定元素右侧所需的边距空间。如果选择“ 默认”选项，则使用全局设置中右边距的值。

打字输入

使用Wrap on typing设置指定编辑文本在指定的Hard wrap中的拟合方式。您可以选择以下选项之一：

  • 默认 - 在这种情况下，IntelliJ IDEA使用全局设置中指定的Wrap on typing选项。

  • 是 - 在这种情况下，IntelliJ IDEA使用右边距字段中指定的值。

  • 否 - 在这种情况下，此选项将关闭，并且一行可以超过右边距中指定的值。

视觉指南

使用“ 可视指南”字段指定多个右边距。您可以保留默认值或输入保证金的空格数。如果要指定多个边距，请输入以逗号分隔的数字。

重新格式化时保留

使用复选框配置IntelliJ IDEA在重新格式化源代码时将执行的异常。例如，默认情况下，选中“ 换行符”复选框。如果您的代码包含比标准约定更短的行，则可以在重新格式化源代码之前通过禁用换行符复选框来转换它们。

包装选项

包装样式适用于左侧窗格中指定的各种代码构造（例如，方法调用参数或赋值语句）。

项目

描述

包装风格

从此列表中，选择所需的包装样式：

  • 不包装 - 选择此选项时，不应用特殊包装样式。

    选择此选项后，将忽略嵌套的对齐和大括号设置。

  • 如果长时间换行 - 选择此选项可使线条超出右边距并用适当的缩进包裹。

  • 总是换行 - 选择此选项可以将列表中的所有元素都包装起来，这样每行有一个元素并且有适当的缩进。

  • 如果长时间缩小 - 选择此选项可使列表中的元素超出右边距，这样每行有一个元素具有适当的缩进。

项目

描述

多线时对齐

如果选中此复选框，则代码构造将在每个下一行的同一列开始。否则，代码构造的位置由当前缩进级别确定。

下一行<character(s)>

选中此复选框可在换行时将指定的一个或多个字符移动到下一行。

新线上的“其他”

使用此复选框可将相应的语句或字符移动到下一行。

<character>之后的新行

选中此复选框可在指定字符移动到新行后显示代码。

特别是如果治疗

如果选中此复选框， else if语句位于同一行。

否则， else if语句被移动到相应缩进级别的下一行。

缩进案例分支

如果选中此复选框，则case语句位于相应的缩进级别。否则， case语句与switch一起放在相同的缩进级别。

大括号放置方式

使用此列表在类声明 ， 方法声明和其他类型的声明中指定左大括号的位置。可用选项包括：

  • 行尾 - 选择此选项可将开括号放置在声明行末尾。

  • 如果换行，则选择下一行 - 选择此选项可将左大括号放置在多行声明行之后的行的开头。

  • 下一行 - 选择此选项可将开括号放置在声明行之后的行的开头。

  • 下一行移动 - 选择此选项可在声明行移动到相应的缩进级别后将开括号放置在该行上。

  • 每个移位的下一行 - 选择此选项，以便在声明行移动到相应的缩进级别后将开括号放置在该行，并使下一行也移动到下一个缩进级别。

    进口

    此表列出了优化导入时要执行的操作。

    项目

    描述

    一般

    在此区域中，配置常规导入选项。

    选项：

    • 使用单类导入 ：选中此复选框可使IntelliJ IDEA在代码生成或导入优化期间仅从包中导入特定类。否则，将插入导入整个包的语句。

    • 使用完全限定的类名 ：选中此复选框可使IntelliJ IDEA在代码生成或导入优化期间使用要导入的类的完全限定名称。否则，将插入正常的import语句。

    • 插入内部类的导入 ：选中此复选框可使IntelliJ IDEA为代码中引用的内部类创建导入。

    • 在Javadoc中使用完全限定名称 ：选中此复选框以使IntelliJ IDEA在Javadoc中使用完全限定的类名。否则，导入一个类。

    • 要使用带'*'的导入的类计数 ：在此文本字段中，指定要从单个包导入的类的数量，直到导入单个类的所有语句都替换为导入整个包的语句。

    • 名称计数使用带有'*'的静态导入 ：在此字段中，指定要从单个类导入的成员数，直到导入单个成员的所有语句都替换为导入整个类的语句。

    JSP导入布局

    在此区域中，配置应如何在代码中组织JSP导入语句。引入的更改显示在下面的“ 预览”窗格中。

    选项：

    • 首选逗号分隔导入列表 ：选择此选项可导入以逗号分隔列表组织的语句。

    • 首选每页导入语句指令 ：选择此选项可以为每行创建一个导入语句。

    使用'*'导入的包

    在此区域中，配置要始终完全导入的包和类列表。

    选项：

    • 静态 ：如果要为所选类声明静态导入，请选中此复选框。

    • 包 ：在此列的文本字段中，指定要始终完全导入的包和类。

      请注意，一旦您完成输入，IntelliJ IDEA会自动在包名末尾添加.* 。

    • 使用子包：选中此复选框以完全导入所选包的所有子包。

    • 添加包 ：单击此按钮可将新条目添加到包和类列表中。

    • 添加空白 ：单击此按钮可将空分隔符添加到包和类列表中。

    • 删除 ：单击此按钮可从列表中删除选定的包或类。

    导入布局

    在此区域中，配置应在代码中组织import语句的方式。您可以将某些类设置为首先或最后，或一个接一个地定位。导入的类将按其包进行分组，并在包中按字母顺序排序。

    选项：

    • 单独布局静态导入 ：如果选中此复选框，则所有静态导入将保留在单独的部分中。否则，将根据指定的布局规则对所有import语句进行排序。

    • 静态 ：如果要为所选包声明静态导入，请选中此复选框。

    • 包 ：在此列的文本字段中，指定要导入的包。

    • 使用子包 ：选中此复选框以使IntelliJ IDEA将布局规则应用于所选包的所有子包。

    • 添加包 ：单击此按钮可将新条目添加到包列表中。

    • 添加空白 ：单击此按钮可在所选条目后插入空行，表示应在相应的导入语句之间插入空行。

    • 上移/下移 ：单击这些按钮可在列表中向上或向下移动包或空行，从而定义导入语句的顺序。

    • 删除 ：单击此按钮可从列表中删除所选包。
