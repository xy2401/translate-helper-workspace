IntelliJ IDEA 2019.1帮助

准备REST开发

此功能仅在Ultimate版本中受支持。

在IntelliJ IDEA级别，RESTful Web服务开发支持基于Java EE：RESTful Web服务（JAX-RS） 插件 。此插件与IntelliJ IDEA捆绑在一起并默认启用。

在模块级别，可以在创建项目，向现有项目添加新模块以及为现有模块添加服务开发支持。

  • 确保启用了RESTful Web Services插件
  • 在创建项目时启用REST支持
  • 将模块添加到项目时启用REST支持
  • 为现有模块启用REST支持

确保启用了RESTful Web Services插件

尽管默认情况下启用了CDI支持插件，但在开始使用CDI之前，始终确保此插件仍然处于启用状态。

  1. 打开“ 设置/首选项”对话框（ Ctrl + Alt + S ）。

  2. 在对话框的左侧，选择“ 插件” 。

  3. 在对话框的右侧，在“ 插件”页面上 ，在搜索框中键入restful 。因此，只有名称和描述包含restful的插件才会显示在插件列表中。

  4. 如果未选中Java EE右侧的复选框：RESTful Web服务（JAX-RS） ，请选择它。

  5. 在Settings对话框中单击OK 。

  6. 如果建议，请重新启动IntelliJ IDEA。

在创建项目时启用REST支持

  1. 如果IntelliJ IDEA中当前未打开任何项目，请单击“ 欢迎”屏幕上的“ 创建新项目 ”。否则，请选择文件|新的|项目 。

    结果，“ 新建项目”向导将打开。

  2. 在向导的第一页上 ，在左侧窗格中，选择Java Enterprise 。在页面的右侧，指定要使用的JDK ，并选择要支持的Java EE版本。

  3. 在Additional Libraries and Frameworks下 ，选择Restful Web Service复选框。

  4. 您将需要一个实现JAX-RS API的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现JAX-RS API的文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

    单击下一步 。

  5. 指定项目和模块的名称和位置设置。有关更多信息，请参阅项目名称和位置 。

    单击完成 。

将模块添加到项目时启用REST支持

    结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java Enterprise 。在页面的右侧，选择要支持的Java EE版本。

  3. 在Additional Libraries and Frameworks下 ，选择Restful Web Service复选框。

  4. 您将需要一个实现JAX-RS API的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

    单击完成 。

为现有模块启用REST支持

  1. 打开项目工具窗口（例如View | Tool Windows | Project ）。

  2. 右键单击感兴趣的模块，然后选择“ 添加框架支持” 。

  3. 在打开的“ 添加框架支持”对话框的左侧窗格中，选中“ RESTful Web服务”复选框。

  4. 您将需要一个实现JAX-RS API的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现JAX-RS API的文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

  5. 在Add Frameworks Support对话框中单击OK 。
