WebStorm 2019.1帮助

在文件中搜索目标

您可以在编辑器中搜索当前打开的文件中的文本字符串。使用不同的选项，您可以缩小搜索过程范围，在搜索中使用正则表达式，以及管理搜索结果。

查找文本字符串的出现次数

  1. 在编辑器中打开您的文件。

  2. 从主菜单中，选择“ 编辑”|找到|查找 （ Ctrl + F ）。

  3. 在打开的搜索字段中，输入搜索字符串。

    WebStorm突出显示您在文件中搜索的结果。

或者，将光标放在字符串中以在文件中搜索，然后按Ctrl + F或选择编辑|找到|从主菜单中找到Caret的Word 。

要在整个项目中搜索目标，请按Ctrl + Shift + F ，有关详细信息，请参阅在项目中查找目标 。

查看当前文件中符号的用法

要在整个项目中查找符号的用法，请按Alt + F7 。有关详细信息，请参阅搜索用法 。

  • 将光标放在此符号处，然后按Ctrl + F7 。WebStorm突出显示文件中符号的所有用法。

    要关闭此突出显示，请在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中转到“ 编辑器”|一般并清除在插入符号框中突出显示元素的用法 。

管理您的搜索

WebStorm允许您调整搜索过程并使用显示的搜索结果执行各种操作。

  • 要查看以前的搜索列表，请在搜索字段中按Alt +向下键 。

  • 要输入多行字符串，请单击搜索字段中的新图标。

  • 要导航到上一个或下一个出现位置，请使用和箭头。

  • 使用“ 查找”工具窗口中的出现列表（ Alt + F7 ），您可以在其中有其他选项，例如，对结果进行分组或在单独的窗口中打开它们。

  • 要添加下一个匹配项的选择或取消选择上一个匹配项，请单击要么 （ Alt + J或Shift + Alt + J ）。

  • 要在整个文件中快速替换搜索目标，请按Ctrl + Shift + Alt + J并键入替换字符串。

  • 要缩小搜索范围，请选择搜索窗格上的“ 单词”或“ 匹配大小写”复选框，或单击“筛选” 并选择搜索范围。

  • 要快速选择或清除正则表达式复选框，请按Alt + X （以前称为Alt + G ）。使用正则表达式进行更具挑战性的搜
