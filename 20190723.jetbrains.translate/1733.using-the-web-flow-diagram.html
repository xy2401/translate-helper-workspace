使用Web流程图

此功能仅在Ultimate版本中受支持。

使用Web Flow Diagram浏览Struts配置文件struts-config.xml并查看Struts操作，页面和其他元素以及它们之间的交互。该图与结构树和属性表视图协调。

要查看Web流程图

  • 打开Struts配置文件struts-config.xml ，然后单击编辑器窗口底部的Diagram 。

该图的布局对于所有图表都是通用的。
