使用文本编辑器编辑UI布局

现在，当您熟悉使用内置设计器的基本UI布局编辑选项时，让我们通过手动编辑其布局，将更多元素添加到应用程序的主视图中。

1。切换到文本视图

切换到编辑器底部的“ 文本”选项卡。IntelliJ IDEA将显示当前所选布局文件的XML源代码：

2。添加水平标尺

让我们添加一些插入分隔符的标记。添加水平分隔线的最简单方法是在源代码中添加以下内容：

<查看 android：layout_width = “fill_parent” android：layout_height = “5dp” android：layout_marginTop = “60dp” android：background = “＃00ff00” />

分隔符为5个单位厚，涂有绿色背景并放置在最近的元素下方。

3。添加TextView元素

要添加另一个TextView元素，请将以下内容添加到源代码中：

<TextView android：layout_width = “wrap_content” android：layout_height = “wrap_content” android：layout_marginTop = “60dp” android：text = “警告！不要触摸机器人......“ android：id = ”@ + id / message“ android：layout_gravity = ”center_horizontal“ />

新元件将放置在分离器下方60个单元;它水平居中并使用默认字体颜色。与元素关联的文本字符串是显式声明的。

但是，与分隔符不同， TextView元素也具有id属性。其语法指示“/”符号后面的字符串是必须被视为ID资源的字符串，并将用于引用视图元素。Android运行时会适当地处理此信息，使您可以编写与TextView组件交互的Java代码。
