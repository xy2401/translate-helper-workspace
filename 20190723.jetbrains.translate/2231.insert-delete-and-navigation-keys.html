插入，删除和导航键

本节介绍“ 插入” ，“ 删除”和导航键的默认映射。

捷径

功能

使用此快捷方式...

删除

删除

根据具体情况：

  • 在编辑器中：删除所选符号/块。

  • 在“ 查找”工具窗口中 ：从搜索结果中排除项目。

  • 在“ 版本控制”工具窗口中 ：从更改列表中删除项目。

  • 在其他视图中：删除所选的一个或多个项目。

下

向下移动

将插入符号向下移动一行。

结束

移至行尾

将插入符号移到行尾。

家

移至Line Start

将插入符号移动到行首。

插入

切换插入/覆盖

在编辑器中切换插入/覆盖模式。光标的形状根据当前模式而变化。

剩下

向左移动

将插入符号向左移动一个字符。

向下翻页

向下翻页

将插入符号向上移动一页。

Page Up

向上翻页

将插入符号向上移动一页。

对

向右移

将插入符号向右移动一个字符。

标签

在编辑器中：

  • 任何选择：缩进选定的行。

  • 没有任何选择：插入制表符号（或相应数量的space.characters）。

在查找列表中：

  • 编辑器中插入符号后没有代码：选择一个项目（如回车 ）

  • 编辑器中插入符号后的一些代码：选择一个项目并用插入符替换插入符号后的代码。

向上

提升

将插入符号向上移动一行。
