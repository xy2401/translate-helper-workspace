测试::单位特别说明

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

Test :: Unit捆绑了Ruby 1.8.x ，通常不需要额外的调优。如果您使用Rails 3.x或Rails 2.3.x以及没有test-unit gem的IntelliJ IDEA，您可以使用此框架“开箱即用”。

但是，在某些情况下，IntelliJ IDEA将无法将测试运行器UI插入测试引擎：

  • Ruby 1.9.x，带有限量版的Test :: Unit
  • Rails 2.3.x和测试单元gem

Ruby 1.9.x，带有限量版的Test :: Unit

使用完整版本，该版本通过test-unit gem分发。如果缺少此gem，测试结果将显示在控制台中，测试运行器UI将仅显示错误消息“未找到测试”。

如果您希望测试运行器UI显示测试结果的树视图，请在Gemfile中包含test-unit gem。

Rails 2.3.x和测试单元gem
