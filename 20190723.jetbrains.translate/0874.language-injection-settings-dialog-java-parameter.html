语言注入设置对话框：Java参数

档案|设置|语言注入 -适用于Windows和Linux的Java参数
IntelliJ IDEA |偏好|语言注入 - macOS的Java参数
按Ctrl + Alt + S

单击时将打开该对话框在“ 语言注入”页面上，在上下文菜单中选择“ Java参数 ”，或选择一个条目并单击 。

使用此对话框为Java参数配置语言注入。如果由于任何原因无法使用注入注释，该对话框可以使用IntelliLang的功能。这主要适用于配置第三方/库方法以及仍需使用Java 1.4的项目。

项目

描述

语言

在此区域中，指定要注入的语言和注入上下文。

  • ID - 选择要注入的语言的ID。

  • 前缀 - 指定构成注入上下文的前缀。

  • 后缀 - 指定后缀以构成注入上下文。

请注意， 前缀和后缀字段是可选的。

分类方法

点击并选择感兴趣的类。结果，显示了类的方法及其参数。

选择要为其启用进样的参数。

高级

指定其他设置。

  • 值模式 - 键入注入上下文的正则表达式。

  • 单个文件 - 如果选项关闭，则与值模式匹配的片段将作为不同的“文件”单独处理 - 例如从片段编辑器的角度。

    如果选项打开，则相应的片段将合并在一起以形成单个单元或“文件”。

    鉴于价值模式

    xxx（。+）yyy（。+）zzz

    和片段

    xxx select * yyy from family zzz ，

    如果选项关闭， select *和from family将被视为两个独立的片段（或“文件”）。如果选项处于启用状态，则select * from family将被视为单个单元或“文件”。
