运行SQL脚本文件

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

以下内容仅在安装并启用数据库工具和SQL插件时有效！

运行SQL文件

  1. 右键单击必要的数据源。

  2. 选择“运行SQL脚本” 。

  3. 在“ 选择路径”窗口中，导航到要应用的SQL文件。

为多个数据源运行SQL文件

  1. 在“ 项目”工具窗口中选择所需的SQL文件。

  2. 右键单击SQL文件，选择Run <file_name> （或按Ctrl + Shift + F10 ）。

  3. 在“ 执行目标”窗口中，选择必要的数据源。

您一次只能为一个数据源运行单个语句。

语句可以包含参数。在运行这些语句之前，请指定这些参数的值。

运行单个语句

  1. 在PyCharm中打开一个SQL文件。

  2. 单击要执行的语句。如果要执行多个语句，请选择（突出显示）必要的语句。

  3. 按Ctrl + Enter或从上下文菜单中选择“ 执行 ”。

  4. 选择要在其中执行语句的数据库控制台 。

SQL文件大小的限制为20 MB。当您打开一个大于20 MB的文件时，您只能看到该文件的前2.5 MB。

在控制台菜单中，您可以将其他控制台与文件关联，或删除文件与控制台（ Detach控制台 ）之间的关联。否则，您可以通过从文件的上下文菜单中选择“ 附加控制台”来将控制台与文件关联。
