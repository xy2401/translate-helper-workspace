示例：创建新的路径变量

您可能有一个位于项目目录之外的库。此库附加到项目中，此库的路径包含在项目的*.iml文件中。但是，此路径不应该是绝对的，因为此库中的位置可能在您的队友的计算机上不同。在这种情况下，您可以创建PATH_TO_LIB变量：

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 外观和行为”|路径变量 ，然后单击 。

  2. 在“ 添加变量”对话框中，键入PATH_TO_LIB变量及其指向磁盘上库位置的值。

  3. 通过版本控制系统共享*.iml文件。

  4. 在队友从VCS更新项目后，他们将更改PATH_TO_LIB变量值，使其指向计算机上的库位置。
