把手/髭

在此区域中，配置IntelliJ IDEA中Handlebars表达式和Mustache模板的处理。此页面上指定的设置适用于分别具有.hbs或.mustache扩展名的专用Handlebars和Moustache文件。
  • 自动插入结束标记：
    • 选中此复选框后，只要键入第一个结束符号 ，IntelliJ IDEA就会自动插入Handlebars表达式的第二个右大括号（ } ）。

      IntelliJ IDEA还可识别三个stashes （ {{{ ），可防止表达式中的值被转义。在这种情况下，只要您键入第一个结束大括号，IntelliJ IDEA就会自动插入两个右大括号。

    • 清除此复选框后，您必须手动键入结束花括号和三重状态。

  • 启用格式：

    • 选中此复选框可在代码生成，重构或重新格式化期间自动重新格式化Handlebars表达式和Mustache模板 （ Ctrl + Alt + L ）。

    • 清除复选框以保留Handlebars表达式和Mustache模板的原始格式。

  • 打开HTML文件作为Handlebars / Moustache：

    • 选中此复选框后，扩展名为.html的文件将被视为Handlebars / Moustache文件，以便IntelliJ IDEA识别并处理Handlebars表达式和Mustache模板 。文件扩展名保持不变，但文件名随之提供图标。

    • 清除此复选框后，只有具有.hbs扩展名的文件才会被视为Handlebars / Moustache文件，并且会识别并处理其中的Handlebars表达式和Mustache模板 。

  • 注释语言：从此下拉列表中，选择要使用注释样式的语言。当您通过按Ctrl + /或Ctrl + Shift + /输入行或块注释时，IntelliJ IDEA会插入所选语言中使用的注释分隔符，例如：

    • Handlebars {{!----}}

    • /**/用于JavaScript

    • HTML的<!---->

有关Handlebars表达式和Mustache模板的更多详细信息，请参阅http://handlebarsjs.com/ 。

该区域仅在安装并启用Handlebars / Moustache插件时可用。该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

EJS

在此区域中，为代码中的EJS包含指定分隔符。基于这些分隔符，IntelliJ IDEA将正确识别和处理这些包含。一对分隔符也称为执行标记 。

  • EJS open delimiter：在此文本框中，键入将指示EJS包含开头的字符串。默认分隔符为<% 。

  • EJS close delimiter：在此文本框中，键入将指示EJS包含结束的字符串。默认分隔符为%> 。
