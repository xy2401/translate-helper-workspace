运行manage.py实用程序的任务

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Python插件时有效！

概观

使用IntelliJ IDEA，您可以在IDE中运行Django manage.py实用程序。此实用程序的每个任务都在manage.py控制台中执行。

请注意， Run manage.py任务命令可用于本地和远程解释器。

配置manage.py实用程序

值得注意的是， manage.py实用程序的配置是在“ 项目结构”对话框的Django构面页面中完成的。

要配置manage.py实用程序，请按照下列步骤操作

  1. 在Project Structure对话框中 ，单击支持Python的模块，然后添加Django facet。

  2. 单击Django facet。

  3. 在Manage.py任务部分中，指定以下内容：

    • 在“ 管理脚本 ”字段中，指定所需的manage.py脚本。
      请注意，默认情况下，IntelliJ IDEA显示位于Django项目根目录下的manage.py脚本。如果您对此建议不满意，可以通过单击“浏览”按钮选择任何其他manage.py脚本 。

    • 在“ 环境变量”字段中，指定要传递给脚本的环境变量。默认情况下，此字段为空。

      单击“浏览”按钮打开“ 环境变量”对话框。使用工具栏按钮组成变量列表。

      如果要查看系统环境变量，请单击此对话框中的“ 显示链接”。

运行manage.py实用程序

运行manage.py实用程序的任务

  1. 在主菜单上，选择“ 工具”|运行manage.py任务 。

    manage.py实用程序在其自己的控制台中启动。

  2. 输入所需任务的名称。

在manage.py实用程序控制台中工作

在manage.py控制台中，可以：

  • 使用向上和向下箭头键滚动执行命令的历史记录。
