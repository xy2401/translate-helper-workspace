JavaScript中的解构

另请参阅意向操作和重构JavaScript 。

通过解构，您可以轻松地将数组和对象中的值解压缩到变量中。此功能具有非常简洁的语法，当您需要在应用程序中传递数据时，通常会使用该语法。从Exploring ES6了解有关解构的更多信息。

在WebStorm中，您可以使用许多意图操作和快速修复来简化从数组或对象中获取多个值的代码片段。

  • 将光标放在数组或对象的值上，按Alt + Enter ，然后从列表中选择Replace with object destructuring或Replace with array destructuring 。

    如果未使用数组或对象中的某些值，则将跳过这些元素：

保留原始作业

  • 按Alt + Enter ，然后选择Introduce object或array destructuring 。

  • 或者，将光标放在数组或对象的值上，然后选择Refactor |提取|上下文菜单中的对象或数组解构 。有关详细信息，请参阅提取对象或数组解构 。

生成函数的解构参数

  • 将光标放在函数的参数上，按Alt + Enter ，然后从列表中选择“ 将参数转换为对象 ”。
