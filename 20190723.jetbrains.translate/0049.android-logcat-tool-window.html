Android监视器工具窗口

查看|工具窗口| Android监视器

在此工具窗口中，您可以在运行或调试Android应用程序时查看和分析系统调试输出。

工具窗口由工具栏和以下嵌套选项卡组成：

  • Logcat选项卡

  • 监视器选项卡包含以下性能监视器 ：
    • 记忆
    • 中央处理器
    • 网络
    • GPU

从设备下拉列表中选择物理设备或虚拟设备，并从进程下拉列表中选择在所选设备上运行的进程。每个选项卡中的信息将根据您的选择进行过滤。

工具栏

两个嵌套选项卡共有的工具栏可以快速访问以下操作：

图标

提示

描述

屏幕截图

单击此按钮可以显示运行应用程序的虚拟或连接物理设备的屏幕截图。您可以旋转屏幕截图并在打开的对话框中添加一些视觉效果，并保存捕获。

屏幕记录

单击此按钮可开始在物理设备上录制应用程序输出。只有将物理Android设备连接到计算机时，此按钮才可用。

系统信息

单击此按钮可查看Dumpsys工具提供的所选流程的系统信息。日志作为文本文件在单独的编辑器选项卡中打开。从打开的弹出菜单中选择要查看的信息类型：

  • 活动管理器状态 ：选择此选项可显示活动管理器的系统输出 - 活动管理器 - 负责管理应用程序活动堆栈的组件。

  • 包信息 ：选择此选项可在应用程序包上显示系统信息。

  • 内存使用 ：选择此选项可以以kB为单位显示应用程序内存使用情况的详细信息。

  • 内存使用时间 ：选择此选项可显示有关过去24小时和过去3小时内聚合的系统内存使用情况的详细信息，以及当前内存使用情况统计信息。

  • 图形状态 ：选择此选项可显示详细应用程序的图形加速信息。

终止申请

单击此图标可停止应用程序执行。

Android Monitor帮助

单击此图标可打开Android Monitor帮助页面。

Logcat选项卡

此选项卡显示与所选设备上所选进程相关的所有系统调试输出消息。

工具栏

使用工具栏控件和按钮配置日志数据的范围和表示，以及浏览日志。

项目

工具提示和快捷方式

日志级别

N / A

从此下拉列表中，选择要显示的日志消息的优先级 。可用选项包括：

  • 详细

  • 调试

  • 信息

找

使用此文本框搜索消息列表。键入搜索字符串时，将显示与搜索模式匹配的消息，并突出显示匹配的字符串。要完成搜索，请按Enter键 。搜索模式存储在搜索历史列表中。要清除搜索历史记录，请单击按钮。

正则表达式

N / A

如果要使用正则表达式搜索模式，请选择此选项。

过滤器

N / A

从此列表中，选择现有过滤器配置 ，或创建一个新配置 。

过滤器配置是一组过滤参数 。使用过滤器配置可以更灵活地控制所显示的日志数据的类型和数量，而不仅仅是通过在日志级别下拉列表中选择消息优先级来指定信息类型。

  • 要显示与所选流程相关的消息，请选择仅显示所选应用程序 。

  • 如果要显示完整日志数据，请从下拉列表中选择“ 无过滤器 ”。

  • 要应用过滤器，请从下拉列表中选择它。

  • 要创建新的过滤器配置 ，请选择“ 编辑过滤器配置” 。在打开的“ 创建新Logcat筛选器”对话框中，单击“ 添加” 并指定以下过滤参数：

    • 日志标记 ：如果只希望显示来自某个系统组件的消息，请使用此参数。键入正则表达式以指定指示相关系统组件的标记 ，例如ActivityManager ， AudioService等，或用户定义的标记。有关更多详细信息，请参阅筛选日志输出 。

    • 日志消息 ：如果只希望显示包含某些元素或字符串的消息，请使用此参数。键入一个正则表达式，用于定义要检测的字符串。

    • 包名称 ：如果只希望显示引用特定Java包（类路径）的消息，请使用此参数。

    • PID ：如果只希望显示引用特定进程（进程ID）的消息，请使用此参数。

    • 日志级别 ：如果您只希望显示具有特定优先级的消息，请使用此参数。

  • 要更新过滤器，请选择“ 编辑过滤器配置” ，然后在打开的“ 创建新的Logcat过滤器”对话框中从列表中选择过滤器并编辑过滤器值。

  • 要从列表中删除筛选器配置，请选择“ 编辑筛选器配置” 。在打开的“ 创建新Logcat过滤器”对话框中，选择过滤器，然后单击“ 删除” 。

清除logcat

单击此按钮可从所选设备上的先前会话中删除日志数据。

滚动到最后

单击此按钮可将插入符号移动到控制台输出的最后一行。

堆栈跟踪
按Ctrl + Alt +向上

单击此按钮可在堆栈跟踪中向上导航，并使光标跳转到源代码中的相应位置。

沿着堆栈跟踪
按Ctrl + Alt +向下

单击此按钮可在堆栈跟踪中向下导航，并使光标跳转到源代码中的相应位置。

使用软包装

单击此按钮可切换输出的软包装模式。

打印

单击此按钮可打印日志。

重新开始

单击此按钮可重新开始记录。

Logcat标题

单击此图标以配置Logcat标头。您可以在打开的对话框中选择以下选项：

  • 显示日期和时间

  • 显示进程和线程ID（PID_TID）

  • 显示包名称

  • 显示标签

Logcat帮助

单击此图标可打开Logcat帮助页面。

监视器选项卡

“ 监视器”选项卡显示不同的Android性能监视器。

内存监视器

内存监视器实时显示应用程序如何分配内存。它在图形中显示可用和已用内存，这有助于测试目的，并有助于检测性能问题是否与垃圾回收事件有关：

内存监视器工具栏

项目

提示

描述

启用/禁用

单击此图标可启用/禁用Android内存监视器。

启动GC

单击此按钮可启动垃圾回收，然后您可以检查所选进程使用的堆内存量（请参阅查看堆使用情况 ）。

转储Java堆

单击此按钮可将Java堆内存的内容转储到文件中。

开始分配跟踪

单击此按钮开始记录应用程序的内存分配。然后与您的应用程序交互并单击相同的按钮以停止分配跟踪。包含记录数据的窗格将作为单独的编辑器选项卡打开。

分配跟踪有助于识别在短时间内分配和解除分配类似对象类型的位置，并检测代码中可能导致内存使用效率低下的位置。
