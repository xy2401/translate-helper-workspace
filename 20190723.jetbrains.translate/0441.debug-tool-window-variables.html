  • 工具栏

使用“ 变量”窗格可以检查存储在应用程序对象中的值。

在“ 帧”窗格中选择堆栈帧时，“ 变量”窗格将显示其范围内的所有数据（方法参数，本地变量和实例变量）。在此窗格中，您可以为对象设置标签，检查对象，计算表达式，向监视添加变量等。

工具栏

仅当“ 监视”窗格处于隐藏状态时，才会显示此工具栏，以便在“ 变量”窗格中显示已配置的监视 。隐藏/显示手表窗格是通过控制切换按钮：

  • 按下按钮（默认状态）时，将隐藏“ 监视”窗格，工具栏将显示在“ 变量”窗格中。这样做，重点是调试器选项卡 。

  • 释放按钮后，工具栏将移至“ 监视”窗格。

  • 若要单独显示“ 监视”窗格并在其中查看已配置的监视，请释放“ 在变量”选项卡中显示监视切换按钮。将显示“ 监视”窗格，其中显示工具栏上的“ 在变量中显示监视”选项卡切换按钮。

  • 要隐藏“ 手表”窗格并在“ 变量”窗格中查看手表，请按 “ 手表”窗格工具栏上的切换按钮。切换按钮返回到“ 变量”窗格工具栏上的默认位置。

查看为

N / A

使用此命令可以选择要显示变量值的方式。对于数值，您可以在十进制和十六进制格式之间切换。对象通常由其类名和实例标识符表示，但您可以选择以字符串格式显示它们。

  • 自动 ：此选项适用于可以进行不同布局的项目。如果选中，IntelliJ IDEA将查看所有可用的渲染器，并搜索当前所选项目的第一个合适布局。如果未找到，则应用默认布局。

  • 十六进制 ：此选项可用于数字变量。如果选中，则变量以十六进制格式显示。

  • 基元 ：此选项可用于基元变量。它显示适合相应基元类型的值。

  • 二进制 ：此选项可用于二进制数据。如果选中，则变量显示为二进制文字。

  • 时间戳 ：此选项可用于long数据。如果选中，则变量显示为时间戳。

  • Boolean ：可用于布尔数据。如果选中，则变量显示为布尔值（ true或false ）。

  • toString（） ：此选项适用于覆盖toString()所有引用类型toString()数组除外），并在树中显示节点的toString()值。

  • 数组 ：此选项可用于数组。

  • 对象 ：这是所有非原始变量类型可用的默认布局。

  • 地图 ：此选项显示为地图。

  • 集合 ：此选项显示为集合。

  • 文件 ：此选项可用于文件，是java.io.的默认布局。File类型变量。

  • 用户定义的渲染器 ：此选项可用于Debugger中用户创建的渲染器Java类型渲染器 ，或在“ 自定义数据视图”对话框中。相应的渲染器名称显示在菜单中。
