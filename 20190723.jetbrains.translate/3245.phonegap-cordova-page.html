项目

描述

PhoneGap / Cordova可执行路径

在此字段中，指定可执行文件phonegap.cmd ， cordova.cmd或ionic.cml （请参阅安装PhoneGap / Cordova / Ionic ）。

PhoneGap / Cordova版

此只读字段显示WebStorm自动检测到的已安装的PhoneGap / Cordova / Ionic版本。

PhoneGap / Cordova工作目录

在此字段中，指定存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

在打开的项目中自动排除'platforms'目录

选中此复选框可在打开项目时自动将平台目录标记为已排除。因此，WebStorm在索引，解析和代码完成期间忽略它。

插件

在此区域中，通过安装所需的软件包来配置要在开发中使用的插件列表。该列表显示了您的计算机上当前安装的所有PhoneGap / Cordova / Ionic插件，包括全局和项目级别。

  • 要安装插件，请单击 。在打开的“ 可用包”对话框中，选择所需的包。

    要全局安装软件包，请选中“ 选项”复选框， --global在字段中键入--global 。单击“ 安装包” 。

  • 要删除插件，请在列表中选择它，然后单击 。

  • 要将插件升级到最新的可用版本，请在列表中选择插件，然后单击 。

有关插件及其使用的信息，请参阅Apache Cordova插件和PhoneGap插件 。
