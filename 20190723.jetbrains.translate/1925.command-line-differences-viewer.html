从命令行查看文件之间的差异

您可以使用PyCharm作为命令行工具来查看文件之间的差异。

  • 在命令行中，使用以下语法运行diff命令：

    <PyCharm> diff path1 path2

所选文件将在差异查看器中打开。

命令选项如下：

选项

描述

<PyCharm>

特定于平台的PyCharm启动器。

在macOS和Linux上，建议使用在PyCharm安装期间创建的命令行启动程序，或者通过选择Tools |在PyCharm主菜单上创建命令行启动器 。请参阅除了在PyCharm中工作之外，您可以从命令行执行以下操作：有关详细信息。

在Windows上，建议将PyCharm可执行文件的Path添加到Path环境变量中。

PATH1

文件本地副本的完整路径。

路径2

文件的存储库版本副本的完整路径。

例子

句法
pycharm.exe diff path1 path2
例
pycharm.exe diff C： \ M yProject \ R eadme.md C： \ M yProject \ R eadme.md.bak
句法
pycharm diff path1 path2
例
pycharm diff~ / MyProject / Readme.md~ / MyProject / Readme.md.bak
