更改主密码对话框

档案|设置|密码 - Windows和Linux的主密码
PyCharm |偏好|密码 - 更改 macOS的主密码
按Ctrl + Alt + S

如果已设置主密码，则可以通过单击“ 密码”页面上的“ 主密码”打开此对话框。

指定当前的主密码，然后指定新的主密码。（主密码用于访问PyCharm密码数据库，如果这样指定，则存储密码。）

您可以将新密码设置为空。因此，PyCharm在访问密码数据库时不会要求您输入密码。但是，在这种情况下，密码的安全性会降低。

如果您不记得当前密码，可以重置主密码（单击重置 ）。

项目

描述

密码

指定当前主密码。

新密码

指定新的主密码。

确认密码

再次键入新的主密码。

使用OS用户凭据加密

仅适用于Windows：如果选中此复选框，主密码本身将以密码数据库的形式加密存储。（您的Windows登录凭据将用于加密。）

因此，PyCharm在访问密码数据库时不会要求您输入主密码。

重启

如果您不记得当前的主密码，请单击此按钮将其重置。（“ 重置主密码”对话框将打开。）
