使用J2ME模块创建项目

此功能仅在Ultimate版本中受支持。

要开发Java移动应用程序，您需要一个带有J2ME模块的项目 。

使用J2ME模块创建项目

  1. 如果IntelliJ IDEA中当前未打开任何项目，请单击“ 欢迎”屏幕上的“ 创建新项目 ”。否则，请选择文件|新的|项目 。

    结果，“ 新建项目”向导将打开。

  2. 在向导的第一页上 ，在左侧窗格中，选择J2ME 。

    在页面的右侧，指定要使用的Java ME SDK。

  3. 如有必要，请启用SQL支持并选择默认使用的SQL方言。

    单击下一步 。

  4. 在向导的下一页上 ，指定特定于移动SDK的选项，然后单击“ 下一步” 。

  5. 指定项目和模块的名称和位置设置。有关更多信息，请参阅项目名称和位置 。

    单击完成 。
