Gradle工具窗口

查看|工具窗口|摇篮

Gradle工具窗口显示Gradle任务，以及对底层build.gradle文件或IntelliJ IDEA项目结构所做的所有更改。如果没有链接的Gradle项目，则禁用该窗口。您需要重新导入Gradle项目以启用Gradle工具窗口。

在这个部分：

  • Gradle项目
  • Gradle任务
  • 工具栏
  • 上下文菜单
  • Gradle任务的上下文菜单命令

Gradle项目

项目

描述

单击此按钮可打开用于配置当前视图和更改工具窗口查看模式的菜单。
菜单项是您可以打开或关闭的选项。活动项目的名称左侧包含复选标记。
菜单中的可用选项如下：

  • 组任务 - 如果要对任务进行分组，请选择此选项。例如，构建任务将包含在构建目录中。

  • 显示工具栏 - 选择此选项可显示Gradle项目的工具栏。

  • 固定，停靠，浮动，窗口，分割模式

  • 组选项卡 - 如果工具窗口中有多个视图，则取消选择此选项可在单独的选项卡上查看视图。

  • 移至 - 选择此选项可将Gradle工具窗口移至顶部，左侧或右侧。

  • 调整大小 - 选择此选项可调整Gradle工具窗口的大小。

使用此图标或快捷方式隐藏工具窗口（ Shift + Escape ）。

与Alt键组合使用时，单击此图标会隐藏连接到同一工具窗口栏的所有工具窗口 。

工具栏

项目

描述

单击此按钮可在对Gradle脚本进行更改后刷新所有已注册的Gradle项目。

单击此按钮可链接gradle项目。

单击此按钮可分离外部gradle项目。

使用此按钮执行Gradle任务。单击此图标时，将打开“ 运行Gradle任务”对话框。您可以输入要执行的任务的名称。请注意，IntelliJ IDEA支持代码完成。

使用这些按钮可以展开或折叠所有节点。

单击此按钮可通过“要导入的项目数据”对话框将模块或数据导入到gradle项目中。它可能对多模块项目有用。

单击此按钮可在脱机模式下使用Gradle项目。当网络连接速度较慢或需要脱机工作时，它可能会有所帮助。

单击此按钮可在“ Gradle设置”对话框中配置当前Gradle项目的设置 。

上下文菜单

项目

描述

选择此选项或使用快捷方式打开build.gradle文件。

选择此选项可刷新项目。

刷新依赖项

选择此选项可刷新项目的依赖项

选择此选项可分离外部项目。

自动导入

选择此选项可为项目启用自动导入功能。

任务激活

选择此选项可添加，编辑或删除任务的激活阶段。它可能对使用Gradle多模块项目很有用。

Gradle任务的上下文菜单命令

命令

描述

运行<task>

选择此命令以使用特定于任务的运行/调试配置运行所选任务。

调试<task>

选择此命令可使用特定于任务的运行/调试配置调试所选任务。

创建<task>

选择此命令可为选定的Gradle任务创建运行/调试配置 。

选择<任务>

选择此命令可激活所选Gradle任务的配置。如果所选任务没有现有配置，则禁用此选项并替换为“创建”选项。

在同步之前执行，在同步之后执行

选择这些命令以设置所选任务的相应标志。在这种情况下，“ 同步前”和“ 同步后”注释将显示在节点名称旁边。

在Build之前执行，在Build之后执行

选择这些命令以设置所选任务的相应标志。在这种情况下， Before Build和After Build注释显示在节点名称旁边。

在重建之前执行，在重建之后执行

选择这些命令以设置所选任务的相应标志。在这种情况下， 重建前和重建后的注释会出现在节点名称旁边。

分配快捷方式

选择此命令可将选定的相位与键盘快捷键相关联 。在这种情况下，带有快捷方式的注释将显示在节点名称旁边。
