历史标签

在通过特定VCS的菜单调用文件或目录的“ 显示历史记录”命令时，“ 历史记录”选项卡将添加到“ 版本控制”工具窗口中 。将为每个文件或目录创建一个新选项卡，其名称如下： 历史记录：<file_name> 。工具栏按钮集略有不同，具体取决于您的版本控制系统。

工具栏中提供的所有命令也可从选择的上下文菜单中获得。

工具提示和快捷方式

描述

刷新

单击此按钮可刷新当前信息。

显示差异
按Ctrl + d

单击此按钮可将文件的选定版本与差异查看器中的先前版本进行比较。

显示所有受影响的文件
SHIFT + ALT + A

单击此按钮可打开“修订中受影响的路径”对话框，您可以在其中查看在所选修订中修改的所有文件。

显示所有分支

单击此按钮可显示当前分支以外的分支的更改。

显示详细资料

单击此按钮可显示所选修订的提交消息。

在GitHub中打开

单击此按钮可打开与GitHub上所选提交对应的页面。
