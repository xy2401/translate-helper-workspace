PhoneGap ， Apache Cordova和Ionic是用于开发具有单一HTML，CSS和JavaScript / TypeScript代码库的移动应用程序并针对各种移动平台（包括Android）的框架。

在你开始之前

  1. 如管理插件中所述，在插件页面上安装并启用PhoneGap / Cordova存储库插件。

  2. 下载并安装Node.js.

  3. 根据您使用的目标平台和操作系统，下载并安装模拟器工具。从Cordova官方网站了解更多信息。

安装PhoneGap / Cordova / Ionic

或者，按照npm和Yarn中的说明在Node.js和NPM页面上安装软件包。

  • 在嵌入式终端 （ Alt + F12 ）中，键入以下命令之一：

    • npm install - 全球phonegap

    • npm install - 全球cordova

    • npm install - 全球离子

创建一个新的PhoneGap / Cordova / Ionic应用程序

如果您还没有应用程序，可以从样板模板生成具有PhoneGap / Cordova / Ionic特定结构的PyCharm项目。或者，创建一个空的PyCharm项目并在其中配置PhoneGap / Cordova / Ionic支持，如下面的开始使用现有的Meteor应用程序中所述。

从模板创建PhoneGap / Cordova / Ionic应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择PhoneGap / Cordova App 。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 指定可执行文件phonegap.cmd或cordova.cmd或ionic.cmd （请参阅安装PhoneGap / Cordova / Ionic ）。

  4. 单击“ 创建”时 ，PyCharm将生成具有特定于框架结构的PhoneGap / Cordova / Ionic应用程序的框架。

创建一个空的PyCharm项目

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Empty Project 。在右侧窗格中，指定应用程序文件夹，然后单击“ 创建” 。

从现有的PhoneGap / Cordova / Ionic应用程序开始

  1. 选择File |，在PyCharm中打开所需的PhoneGap / Cordova / Ionic应用程序打开主菜单或单击打开的欢迎屏幕 。

  2. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript |科尔多瓦Cordova页面打开。

  3. 检查可执行文件phonegap.cmd或cordova.cmd或ionic.cmd的位置，或者如果PyCharm未自动检测到可执行文件，请指定其路径。

    PyCharm检测已安装的版本并将其显示在PhoneGap / Cordova Version只读字段中，

  4. 在Cordova工作目录字段中，指定存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

  5. 在“ 插件”区域中，通过安装所需的软件包来配置要在开发中使用的插件列表。该列表显示了您的计算机上当前安装的所有PhoneGap / Cordova / Ionic插件，包括全局和项目级别。

    • 要安装插件，请单击 。在打开的“ 可用包”对话框中，选择所需的包。

      要全局安装软件包，请选中“ 选项”复选框， --global在字段中键入--global 。单击“ 安装包” 。

    • 要删除插件，请在列表中选择它，然后单击 。

    • 要将插件升级到最新的可用版本，请在列表中选择插件，然后单击 。

    有关插件及其使用的信息，请参阅Apache Cordova插件和PhoneGap插件 。

创建并启动PhoneGap / Cordova / Ionic运行配置

PhoneGap / Cordova / Ionic应用程序根据专用的运行/调试配置执行。

  1. 在主菜单上，选择“运行”|编辑配置 。在打开的“ 编辑配置”对话框中，单击在工具栏上，从上下文菜单中选择PhoneGap / Cordova 。

  2. 在“ 运行/调试配置：打开的PhoneGap / Cordova”对话框中，指定以下内容：

    1. 配置的名称。

    2. 在PhoneGap / Cordova可执行路径字段中，指定可执行文件phonegap.cmd ， cordova.cmd或ionic.cml （请参阅安装PhoneGap / Cordova / Ionic ）。

    3. 在PhoneGap / Cordova工作目录字段中，指定存储要运行的PhoneGap / Cordova / Ionic应用程序文件的文件夹。

    4. 从“ 命令”列表中，选择要运行的命令。列表的内容取决于实际使用的框架，即PhoneGap / Cordova可执行路径字段中指定的可执行文件。可用选项包括：

      • 对于PhoneGap ：

        • 仿真

        • 跑

        • 准备

        • 服务

        • 远程构建

        • 远程运行

        有关具有说明的PhoneGap特定命令的列表，请参阅PhoneGap CLI 。

      • 对于科尔多瓦 ：

        • 仿真

        • 跑

        • 准备

        • 服务

        有关具有描述的Cordova特定命令的列表，请参阅Cordova CLI 。

      • 对于离子 ：

        • 仿真

        • 跑

        • 准备

        • 服务

        有关具有描述的Ionic特定命令列表，请参阅Ionic CLI 。

    5. 从“ 平台”列表中，选择应用程序所针对的平台。可用选项包括：
      • Android的

      • ios要模拟此平台，您需要全局安装ios-sim命令行工具 。您可以通过Node Package Manager（npm） ，查看npm和Yarn，或者运行sudo npm install ios-sim -g命令，具体取决于您的oeprating系统。

      • 亚马逊fireos
      • firefoxos
      • blackberry10
      • Ubuntu的
      • WP8
      • windows8的
      • 浏览器

      从平台指南了解有关目标平台的更多信息。

    6. 对于Cordova和Ionic ，请指定要在其上运行应用程序的目标虚拟或物理Android设备：选择“ 指定目标”复选框，然后从列表中选择所需的设备。

      该列表显示了我们计算机上当前配置的所有虚拟和物理设备。详情。了解有关从Cordova官方网站设置模拟器的更多信息。

      如果PyCharm显示以下错误消息： 无法检测路径中的ios-sim ，请确保已安装ios-sim ，请参阅开始之前 。

  3. 要运行PhoneGap / Cordova / Ionic应用程序，请从主工具栏上的列表中选择所需的运行配置，然后转到运行|在主菜单上运行<configuration name>或单击 。
