使用调试控制台

使用Debug Console可以查看输出和错误消息。您可以选择使命令提示符可用。

打开调试控制台

如果在调试会话期间，您需要使用交互式调试器控制台，请执行以下操作之一：

  • 在调试控制台的工具栏中，单击按钮 。

  • 在主菜单上，选择“ 工具”|打开调试命令行 。

控制台变为交互式：它显示提示，您可以在其中键入命令，执行命令和查看结果：

交互式控制台中可用的操作

在交互式控制台中，您可以：

  • 在控制台的下部窗格中键入命令，然后按Enter键执行它们。结果显示在上部窗格中。

  • 使用基本代码完成 Ctrl + Space 。

  • 使用向上和向下箭头键滚动命令历史记录，并执行所需的命令。

  • 将源代码从编辑器加载到控制台中。

  • 使用上部窗格的上下文菜单将所有输出复制到剪贴板，与剪贴板的当前内容进行比较，或从控制台中删除所有输出。

  • 使用工具栏按钮控制控制台中的会话。
