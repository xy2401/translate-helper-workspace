提取参数对象对话框

重构|提取|参数对象

使用此重构可以围绕方法的某些选定参数创建包装类，或者使用兼容的现有类作为包装器。

项目

描述

从中提取参数的方法

此只读字段显示所选方法的名称。

参数类

使用此部分指定是要创建新的包装类还是使用现有的包装类。

创建新课程

单击此单选按钮可将方法的参数移动到新类。如果选择此选项，请在下面的字段中指定类和目标包名称。默认情况下，显示当前包名称。您可以在文本字段中键入其他包名称，或单击省略号按钮并从树视图中选择目标包。如果所需的包不存在，请单击创建一个新的。

创建内部类

单击此单选按钮可将方法的参数移动到内部类。如果选择此选项，请在下面的字段中指定类名。

使用现有的课程

单击此单选按钮可将方法的参数移动到您选择的现有类。您可以在文本字段中键入完全限定的类名，或单击省略号按钮并在“ 选择参数类”对话框中选择所需的类。请注意，您可以从项目类和非项目类中选择所需的包装类。

要提取的参数

使用此部分中的复选框选择应将方法的哪些参数提取到单独的对象中。

上移/下移

使用这些按钮可以在列表中移动选定的重新排序参数。

将方法保留为委托

如果选中此复选框，则原始方法将保留为新创建的方法的委托。
