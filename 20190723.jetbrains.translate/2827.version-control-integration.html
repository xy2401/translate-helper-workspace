使用PyCharm进行版本控制

使用VCS Operations Popup （ Alt +`或VCS | VCS Operations Popup ）快速调用任何与VCS相关的命令。弹出窗口中的操作列表取决于当前启用的VCS。

VCS Operations弹出命令列表是可配置的 - 您可以在Appearance and Behavior |上添加或删除它们PyCharm设置的菜单和工具栏页面（ Ctrl + Alt + S ）。
