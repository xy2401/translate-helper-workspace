PyCharm 2019.1帮助

行

添加一行

  1. 单击“ 添加新行”图标（ ）在工具栏上。或者，右键单击该表，然后从上下文菜单中选择“ 添加新行 ”。

  2. 按Alt + Insert 。

请注意，上下文菜单克隆行命令（ Ctrl + D ）可用作替代方法。

删除一行

  1. 选择要删除的行。

    通过单击显示行号的列中的单元格来选择行。要选择多行，请使用鼠标单击和Ctrl键组合。

  2. 单击“ 删除行”图标（ ）在工具栏上。或者，按Ctrl + Y或删除 。

克隆一排

您可以克隆选定的行。该行的副本将添加到表的末尾。

要克隆行，请右键单击该行并选择“ 克隆” 。或者，选择行并按Ctrl + D.

在行的子集之间导航

根据“ 限制页面大小”参数的值，结果集可能会分为多个页面。例如，您将“ 限制页面大小”设置为参数为100，但查询返回了200行。您将有两页行，每页有100行。要在页面之间导航，请使用以下控件：

  • ：导航到结果集的第一页。

  • ：导航到结果集的上一页。（ Ctrl + Alt + Up ）

  • ：导航到结果集的下一页。（ Ctrl + Alt + Down ）

  • ：导航到结果集的最后一页。

使所有行同时可见

如果要同时显示所有行：

  1. 点击在工具栏上，选择设置 。

  2. 切换到数据库| “数据视图”页面，在“ 结果集页面大小”字段中指定0 ，然后单击“ 确定” 。

  3. 点击或按Ctrl + F5刷新表格视图。

也可以看看， 和结果集页面大小 。

转到指定的行

要导航到具有指定编号的行，请右键单击该表并选择“ 转到”行 （ Ctrl + G ）。在“ 转到行”对话框中，指定行号，然后单击“ 确定” 。

导航到相关对象

您可以从表中检索所选对象的行，并将焦点移动到该对象。例如，在项目树中选择一个字段，然后按Ctrl + Alt + B.焦点移动到新选项卡中的相应列。

要导航到结果集中的对象，请选择一个对象并转到Navigate |相关符号 （ Ctrl + Alt + B ）。

在相关数据之间导航

您可以在不同类型的相关数据之间导航：

  • 引用数据 ：当前对象引用的数据。

  • 引用数据 ：引用当前对象的数据。

  1. 右键单击表中的列，然后导航到“ 转到”参考数据 （ Ctrl + B ）或转到|引用数据 （ Ctrl + Alt + B ）。

  2. 如果出现“ 选择目标”弹出窗口，请选择表格以及要显示的行：

    • 第一个引用行 ：表中的所有行。选择引用当前行的第一行。

    • 仅引用行 ：仅引用当前行的行。

  3. 按Enter键

在结果集中设置多个行

默认情况下，发出查询时，返回的行数限制为500。引入此限制是为了避免重载（例如，当SELECT语句返回一百万行时）。

  1. 转到设置/首选项 （ Ctrl + Alt + S ）并导航到工具| 数据库|数据视图 。

  2. 在“ 限制页面大小”字段中指定新数字。要禁用此限制，请清除“ 限制页面大小”复选框。
