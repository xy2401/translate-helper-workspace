使用docstrings指定类型

介绍

您可以永久地调试代码，现在在调试过程中，您还可以收集类型信息并在文档字符串中指定这些类型。

PyCharm提供了一个意图操作，可以在运行时收集类型信息，并定义类型规范。

但是，很可能在没有调试器的情况下手动指定参数类型。

这两种情况都在实例部分中进行了探讨。

条件

确保选中编辑器设置的“ 智能密钥”页面中的“ 插入类型占位符 ”复选框。

另请注意，reStructuredText用于所有后续示例，但可以使用文档字符串的任何支持格式，无论是纯文本，Epytext，Google还是NumPy。有关详细信息，请参阅Python集成工具页面的说明。

参数类型规范

要指定参数类型，请遵循以下常规步骤

  1. 将插入符号放在函数名称上，然后按Alt + Enter 。

  2. 在打开的意图操作列表中，选择“ 插入文档字符串存根” 。PyCharm根据所选的文档字符串格式，在调试器会话期间收集类型规范，创建文档存根。

例

请考虑以下代码：

import math class SimpleEquation ： def demo （ self ， a ， b ， c ）： d = math 。 sqrt （ b ** 2 - 4 * a * c ） root1 = （ - b + d ） / （ 2 * a ） root2 = （ - b - d ） / （ 2 * a ） print （ root1 ， root2 ） SimpleEquation （） 。 演示 （3,2，1）

让我们假设已经在Python集成工具页面上选择了reStructuredText作为文档字符串格式。

关于reStructuredText / Sphinx的重要说明

请注意，对于reStructuredText，可以指定两种格式的类型：

  • :param param_type param_name: parameter description （类型说明与参数说明在同一行）。

  • :type param_name: param_type （类型描述在一个单独的行上）

在调试器的帮助下

现在，在“设置/首选项”对话框的“ Python调试器”页面中，选中“ 收集代码洞察的运行时信息 ”复选框。

调试函数调用，并使用意图操作再次插入文档字符串存根 。有关在调试会话期间获得的参数和返回值的信息将用于预先填充文档字符串中的类型注释。
