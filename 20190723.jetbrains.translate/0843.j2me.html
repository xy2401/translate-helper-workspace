IntelliJ IDEA 2019.1帮助

J2ME

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA支持以下用于开发Java移动应用程序的工具包：

  • J2ME Wireless Toolkit （又名WTK），版本1.x和2.x.

  • DoJa ，版本1.x.

IntelliJ IDEA与这些工具包的集成基于J2ME 插件 。此插件与IDE捆绑在一起并默认启用。

准备Java移动应用程序开发

  1. 下载并安装JDK和一个受支持的Java移动工具包 。

  2. 确保已启用J2ME插件，请参阅确保已启用J2ME插件 。

  3. 在IntelliJ IDEA中定义JDK和移动工具包。（在IntelliJ IDEA中，受支持的Java移动工具包称为移动SDK）。请参阅在IntelliJ IDEA中定义JDK和移动SDK 。

  4. 使用J2ME模块创建项目 ，请参阅使用J2ME模块创建项目 。

  5. 如有必要，请配置特定于Java移动应用程序的编译设置，请参阅配置Java移动特定的编译设置 。

另请参阅运行和调试Java移动应用程序 。

确保已启用J2ME插件

尽管默认情况下启用了J2ME插件，但在开始开发Java移动应用程序之前，始终确保仍然启用此插件。要做到这一点：

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 插件” 。将打开“ 插件”页面 。

  2. 在对话框的右侧，在搜索框中键入j2 。因此，只有名称和描述包含j2的插件才会显示在插件列表中。

  3. 如果未选中J2ME左侧的复选框，请选择它。

  4. 在Settings对话框中单击OK 。

  5. 如果建议，请重新启动IntelliJ IDEA。
