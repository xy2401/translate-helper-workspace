组件属性

Inspector的灰色阴影部分提供了一组属性，这些属性是GUI Designer专有的，并由其代码生成和其他进程使用。

将组件添加到表单时， 可以将其创建为组件或容器 。在后一种情况下，此类组件仅获取特定于容器的某些属性。在下表中，特别注明了与容器有关的属性。

在本节中，您将找到以下属性组的描述：

  • 代码绑定属性

  • 组件大小调整属性

  • 布局和对齐属性

  • 其他属性

属性是特定于布局的。某些布局管理器可能缺少某些属性。

代码绑定属性

本节涉及的属性与GUI表单和组件与源代码的绑定有关。

属性

描述

绑定到类

这仅是表单的属性。它指定包含使表单起作用的逻辑的类的名称。当此属性设置为有效类时，我们说Form已绑定到该类 。

如果还没有目标类，您仍然可以键入此未来类的名称。IntelliJ IDEA将提供快速修复，以便在关注属性时为您创建指定名称的类。只要准备好，您就可以使用“快速修复”来创建类。

请参阅将表单绑定到现有类和绑定表单到新类的部分 。

字段名称

这是组件的属性。它指定组件绑定到的父窗体类中的字段名称。对于大多数组件，将自动输入默认字段名称，并将相应的声明写入表单的绑定源文件。如果需要，可以在检查器中更改默认字段名称，源将自动更新。您可以选择更改源文件中的字段名称 ，并在返回GUI Designer时更改将反映在检查器中。

自定义创建

这是组件的属性。如果选中该选项，则表示您要调用组件的非默认构造函数，而不是让GUI Designer在GUI的运行时构建期间生成默认构造函数。代码生成将忽略该组件并假设您已编写了构造函数方法。请参阅创建表单初始化代码 。

如果尚未存在非默认构造函数，则只要在“检查器”中聚焦“自定义创建”属性，IntelliJ IDEA就会显示“快速修复”。您可以使用它在绑定到父Form的源文件中创建构造函数。

布局和对齐属性

本节中描述的属性控制组件布局和/或对齐的各个方面。

属性

子属性

描述

布局管理器

这只是容器类型组件的属性（例如JPanel，JScrollPane）。该设置控制容器使用的布局管理器。该设置会影响设计时和运行时。在“设置”对话框的“ GUI设计器选项 ”部分中查找支持的布局管理器列表。

边界

定义组件边框和（可选）标题的外观。仅适用于容器类型组件，包括以下子属性：

类型

指定元素边框的斜角特征。

  • 没有。 无边界。如果指定了title ，它将被隐藏。

  • 空。 没有指定边框属性。

  • 斜角降低。 边框斜面会使容器看起来降低。

  • 斜面凸起。 边框斜面会使容器看起来凸起。

  • 蚀刻。 容器看起来是扁平的，带有蚀刻或3D边框。

标题

（可选）指定在运行时显示为容器标题的字符串。您可以直接在编辑字段中输入字符串文字，或单击省略号按钮打开一个对话框，您可以在其中对字符串值进行硬编码，或指定资源的标识符。

标题理由

控制标题文本的合理性。

  • 默认。 理由是在编译时确定的。

  • 剩下。强制左对齐文本。

  • 中央。 强制居中的文字。

  • 对。 强制右对齐文本。

  • 领导。 用于需要主导理由的区域设置。

  • 尾随。 对于需要尾随对齐的区域设置。

头衔位置

控制标题相对于容器边框的位置。文本的水平位置由标题对齐设置控制。默认，顶部，顶部，顶部，顶部，底部，底部，底部，如下图所示：

标题字体

控制用于显示标题文本的字体。

标题颜色

控制用于显示标题文本的字体颜色。省略号按钮启动颜色选择器对话框，您可以在其中从多个调色板（AWT，Swing，System）中选择一种颜色，或指定RGB或HSB颜色值，或从图形颜色样本集中选择颜色。

每个属性控制窗格各边缘的间距。每个属性的值是一个整数，它指定相应间距中的像素数。零意味着没有空间。

水平差距
垂直差距

这仅是JPanel的属性，仅当窗格使用网格类型布局管理器（如默认的GridLayoutManager）时才有效。该属性定义窗格中网格单元格边缘与所包含组件边缘之间的空间插入的像素尺寸（例如，JRadiobutton）。

默认值为-1，表示默认间距。您可以输入零或任何正整数值，并在设计时查看结果。

要使Vertical Gap产生任何效果，布局网格应至少有2行。

水平对齐

此属性确定组件在其容器中的相对水平位置。从下拉列表中选择一个值：

  • 剩下。组件的左侧边缘将捕捉到其容器的左边框。

  • 中央。 组件在其容器内水平居中。

  • 对。 组件的右侧边缘捕捉到其容器的右边框。

  • 填。 该组件完全填充其容器的水平空间。

垂直对齐

此属性确定组件在其容器中的相对垂直位置。从下拉列表中选择一个值：

  • 最佳。 组件的上边缘捕捉到其容器的顶部边框。

  • 中央。 组件在其容器内垂直居中。

  • 底部。 组件的底部边缘捕捉到其容器的底部边框。

  • 填。 该组件完全填充其容器的垂直空间。

缩进

此属性仅适用于GridLayoutManager。有效值为零或任何正整数。所选组件向右移动指定的像素数乘以10。例如，如果输入12，则缩进将为120像素（12 * 10）。
