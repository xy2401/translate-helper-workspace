启用Phing支持

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

在项目级别，您可以通过以下方式启用Phing：

  • 显式地通过项目设置，在“ 设置/首选项”对话框的“ Phing”页面上。

  • 将文件添加到Phing构建文件列表时。正确的Phing构建文件是一个xml文件，其根元素为<project> 。

  • 在运行时，当您尝试从编辑器或项目工具窗口运行构建时。

在任何一种情况下， Phing Build工具窗口都可以从View |中获得工具窗口|在当前项目范围内的Phing Build菜单。

在你开始之前

  1. 确保Phing已下载并在您的计算机上进行设置 。如果您使用AMP ，则该软件包可能已包含Phing。

  2. 按照管理插件中的说明在插件页面上安装Phing Support插件。

启用Phing支持

通过项目设置启用Phing支持

  1. 按Ctrl + Alt + S或选择文件|打开设置/首选项对话框 设置 Windows和Linux或IntelliJ IDEA的| macOS的首选项 。展开“ 语言和框架”节点，然后单击“ PHP下的Phing ”。

  2. 在打开的Phing页面中，在Path to Phing可执行文本框中指定文件phing.bat的位置。手动键入路径或单击“ 浏览” 并在打开的对话框中选择文件位置。

从Phing Build工具窗口启用Phing支持

  1. 在编辑器中打开Phing构建文件，或在“ 项目”工具窗口中选择它。

    正确的Phing构建文件是一个xml文件，其根元素为<project> 。

  2. 在编辑器或选择的上下文菜单上，选择Add as Phing build file 。

  3. 在打开的Phing Build工具窗口中，单击“ 设置”按钮在工具栏上。

    该按钮仅在构建文件列表不为空时可用。

  4. 在打开的Phing Settings对话框中，指定phing.bat可执行文件的位置。

在尝试运行构建文件时，要动态启用Phing

  1. 在编辑器中打开Phing构建文件。

  2. 在编辑器的上下文菜单中，选择“运行构建文件” 。

  3. 在打开的Phing Build工具窗口中，单击“ 设置”按钮在工具栏上。

    该按钮仅在构建文件列表不为空时可用。

  4. 在打开的Phing Settings对话框中，指定phing.bat可执行文件的位置。
