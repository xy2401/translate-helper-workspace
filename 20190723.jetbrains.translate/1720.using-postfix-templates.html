激活后缀完成

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 后缀完成” 。

  2. Postfix Completion页面打开，显示可用的后缀列表，旁边有相应的模板。选择后缀时，“ 描述”窗格会显示相应的转换，显示扩展模板之前和之后的代码段。

  3. 选中“ 启用后缀模板”复选框以及需要激活的后缀旁边的复选框。

要应用后缀模板

  1. 键入表达式后跟一个点。

  2. 键入后缀并按扩展键（默认值为Tab ）或从建议列表中选择后缀。如有必要，请选择要包围或替换的表达式。

创建自定义模板

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ Editor下的Editor 常规 Editor ，然后单击“ 后缀完成” 。

  2. 在打开的Postfix完成页面上，单击 ，并选择模板的工作语言。您可以选择JavaScript或TypeScript，JavaScript模板也适用于TypeScript文件。

  3. 在打开的“ 创建新模板”对话框中，键入模板后缀，选择语言级别，键入模板代码，并在需要插入初始表达式的位置添加$EXPR$ 。在您希望光标位于末尾的位置添加$END$ 。

您还可以创建一个与现有模板略有不同的新模板，例如，在其语言上下文或光标的最终位置。IntelliJ IDEA允许您复制原始模板并在该副本中进行必要的更改。

从现有模板创建新的自定义模板

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ Editor下的Editor 常规 Editor ，然后单击“ 后缀完成” 。

  2. 在打开的Postfix Completion页面上，选择要从中创建新模板的自定义模板，然后单击在工具栏上。

  3. 在打开的“ 编辑模板”对话框中，根据需要编辑模板并键入新的后缀。

要更改默认扩展密钥

  • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ Editor下的Editor 常规 Editor ，然后单击“ 后缀完成” 。在打开的Postfix Completion页面上，从Expand templates with list中选择一个新密钥。

禁用后缀完成

  • 要禁止展开所有已配置的后缀模板，请清除“ 启用后缀模板”复选框。

  • 要禁止展开特定模板，请清除其后缀旁边的复选框。
