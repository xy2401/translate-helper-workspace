在你开始之前

  1. 按照Perforce官方网站上的说明安装Perforce客户端。

  2. 使用Perforce库创建帐户。

从PyCharm中的Perforce开始

  1. 使用Perforce客户端创建客户端规范。

  2. 使用Perforce客户端从Perforce depot获取源文件。

  3. 只要本地工作副本在您的计算机上， 请将本地目录与Perforce关联 。

    之后，您将能够打开源文件进行编辑，并使用PyCharm执行与Perforce相关的常规任务。

Perforce支持

  • 启用Perforce与PyCharm的集成后， Perforce项目将显示在VCS菜单上，以及编辑器和项目工具窗口的上下文菜单中。

  • Perforce控件下的文件夹中的文件将根据其状态突出显示。有关文件状态突出显示约定，请参阅文件状态要点 。

  • 修改结果显示在“ 版本控制”工具窗口中 。

  • 使用Perforce集成时，打开“ 版本控制”工具窗口会很有帮助。“ 控制台”选项卡显示以下数据：

    • 根据您通过PyCharm用户界面指定的设置生成所有命令。

    • 有关执行生成的Perforce命令的结果的信息消息。

    • 错误消息。
