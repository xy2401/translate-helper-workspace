语言注射

语言注入允许您处理字符串文字内的代码片段以及类似XML的语言中的标签或属性。

如果在字符串文字，标记或属性中有一段正式语言（如HTML，CSS，XML，RegExp等），则可以获得全面的语言帮助来编辑该片段。

在开始使用语言注入之前，请确保已启用 IntelliLang插件。

暂时注入一种语言

  1. 将光标放在要注入语言的字符串文字，标记或属性中，然后按Alt + Enter （或使用意图操作图标） ）。

  2. 选择“ 注入语言”或“引用”，然后选择要注入的语言。

在专用编辑器部分打开代码片段

  1. 将插入符号放在注入的代码段中，然后按Alt + Enter （或使用意图操作图标 ）。

  2. 选择编辑<语言ID>片段 。

    IntelliJ IDEA将打开一个专用的编辑器部分，用于使用注入的语言编辑代码。此编辑器提供完整的代码帮助，包括代码完成，检查，意图和代码样式操作。

使用语言注入注释

  • 在目标字符串文字前添加一个空行，然后键入以下注释：

    - language = < language_ID >

    对于注释，请使用要注入的语言的语法。语言ID通常是直观的，例如SQL，RegExp，XML，HTML。

语言片段可以与作为包装器一起起作用的前缀和后缀组合，将片段转换为语法上完整的语言单元。编辑代码时，只能在片段编辑器中看到前缀和后缀。它们不会显示在主编辑器中。

前缀和后缀可以包含在注释注释中，如下所示：

// language = <language_ID> prefix = <prefix> suffix = <suffix>

使用@Language注释

在某些语言中（例如，在Java，Groovy或Kotlin中），您还可以使用@Language("language_ID")注释来注入语言。

  1. 在目标字符串文字之前的空白行上，键入@Language("language_ID") 。

  2. 将光标放在注释上，按Alt + Enter并选择“ 添加'注释'到类路径 。

    IDE将提示您从Maven下载带有注释的库。

  3. 在搜索字段中，如果使用JDK 1.8或更高版本，请键入org.jetbrains:annotations:16.0.2 。

    对于JDK org.jetbrains:annotations-java5:16.0.2或1.7，请键入org.jetbrains:annotations-java5:16.0.2 。

  4. 单击确定 。

  5. 按Alt + Enter并选择导入类 。

配置注入规则

您可以在编辑器上配置语言注入规则IntelliJ IDEA设置的语言注入页面（ Ctrl + Alt + S ）。

所有预定义的注入规则都是为内置范围配置的。换句话说，它们是全局的（因此可用于所有IntelliJ IDEA项目）。可以为IDE或仅为一个项目配置自定义规则。要更改自定义进样的范围，请使用图标。

要配置自定义注入规则，请单击添加新规则，或复制预定义规则并更改其设置。

示例：在Java中注入SQL

您可以配置设置，以便每次使用某种方法时，IDE将注入所选语言，例如SQL。

  1. 在设置/首选项对话框（ Ctrl + Alt + S ）中，转到编辑器|语言注射 。

  2. 点击并选择Java参数 。

  3. 从ID列表中，选择SQL 。

  4. 在“ 类方法”字段中，输入要获取SQL参数的方法。您可以手动键入方法，也可以单击并在“ 选择类”对话框中选择方法。

    如果该方法采用多个参数，它们将显示在对话框中。选择要使用的那些。

  5. 应用更改并关闭对话框。

取消注射

  1. 将光标放在代码片段上，然后按Alt + Enter （或使用意图操作图标 ）。

  2. 选择取消注入语言/参考 。

要取消语言注入，您还可以删除注入注释或注释。
