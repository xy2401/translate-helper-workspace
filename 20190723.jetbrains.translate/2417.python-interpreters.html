项目口译员

此对话框显示选择显示全部... ：

使用此对话框配置可用的Python解释器列表。

项目

工具提示和快捷方式

描述

加
Alt +插入

单击此按钮可从列表中选择Python解释器类型。

去掉
Alt + Delete键

单击此按钮可从可用解释器列表中删除选定的Python解释器。

编辑
输入

单击此按钮可更改所选Python解释器的名称和位置。请参阅“ 可选 ”部分，您可以单击并键入备用解释器名称。在字段中指定的Python解释器名称在可用解释器列表中变为可见。单击以应用更改。 。

显示与其他项目关联的虚拟环境

如果按下此按钮，则已与其他项目关联的虚拟环境将显示在可用解释器列表中。
否则，该列表仅包括其他项目中未使用的虚拟环境。此行为由复选框将此虚拟环境与当前项目关联 。

显示所选解释器的路径

单击此按钮可显示解释器路径列表。

口译员路径

单击时会显示此对话框在“ 项目解释器”对话框中。它显示了外部库所在的路径。如果已添加库，建议单击“ 重新加载”按钮以重新扫描Python安装。

工具提示和快捷方式

描述

加
Alt +插入

单击这些按钮可为选定的Python解释器添加外部库。


去掉
Alt + Delete键

单击这些按钮可删除所选Python解释器的外部库。

重新加载路径列表

单击此按钮可重新扫描所选Python安装的库。例如，如果某些软件包已升级，则单击此按钮可帮助更新外部库。
