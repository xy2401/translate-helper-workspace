Gulp工具窗口

Gulpfile.js的上下文菜单 - 显示Gulp任务
查看|工具窗口|吞

使用此工具窗口运行Gulp.js任务执行任务的结果显示在“ 运行”工具窗口中 。工具窗口显示Gulp.js输出，报告发生的错误，列出尚未找到的包或插件等。最后执行的任务的名称显示在工具窗口的标题栏上。

在你开始之前

  1. 下载并安装Node.js.

  2. 全局安装gulp-cli软件包（Gulp命令行界面）并安装gulp软件包作为开发依赖项。有关详细信息，请参阅安装Gulp.js。

运行任务

在当前WebStorm会话期间第一次构建任务树时， Gulp工具窗口尚未打开。

打开Gulp工具窗口

  • 在Project工具窗口中选择所需的Gulpfile.js文件，或在编辑器中打开它，然后从上下文菜单中选择Show Gulp Tasks 。

默认情况下，WebStorm不承认ES6 Gulpfile.js并没有建立一个任务树。要解决此问题，请更新默认的Gulp.js运行配置 。

从ES6 Gulpfile.js构建任务树

  1. 在主菜单上，选择“运行”|编辑配置 。

  2. 在“ 模板”节点下，单击“ Gulp.js” 。

  3. 在运行/调试配置：Gulp.js对话框打开，键入--harmony在节点选项的文本框，然后单击确定 。

构建任务树

  • 在Gulp工具窗口中，单击在工具栏上，从列表中选择所需的Gulpfile.js文件。默认情况下，WebStorm在项目的根目录中显示Gulpfile.js文件。

  • 如果您有另一个Gulpfile.js文件，请单击Choose Gulpfile.js并在打开的对话框中选择所需的Gulpfile.js文件。WebStorm在其标题上添加一个新节点，其中包含所选Gulpfile.js文件的路径，并在新节点下构建任务树。

重建一棵树

  • 切换到所需的节点，然后单击在工具栏上。

按名称对树中的任务进行排序

  • 点击在工具栏上，从菜单中选择排序依据 ，然后选择名称 。
    默认情况下，树以Gulpfile.js （选项定义顺序 ）中的顺序显示任务。

运行任务

  • 双击任务。

  • 在树中选择任务，然后按Enter键或在上下文菜单中选择“运行<任务名称> ”。

运行默认任务

  • 在树中选择根节点，然后在选择的上下文菜单中选择“运行默认值 ”。

运行几个任务

  • 使用多选模式：按住Shift （对于相邻项）或Ctrl （对于非相邻项）键并选择所需任务，然后在所选内容的上下文菜单中选择“运行”或“ 调试 ”。

导航到任务的定义

  • 在树中选择所需任务，然后在所选内容的上下文菜单中选择“ 跳转到源 ”。

工具栏

添加Gulpfile

单击此按钮可为另一个Gulpfile.js文件创建任务树。从列表中选择所需的Gulpfile.js文件。WebStorm构建任务树并将其显示在单独的节点下。

删除Gulpfile

单击此按钮可删除所选节点下的任务树。

重新加载任务

单击此按钮可重新构建所选节点下的任务树。在更新相应的Gulpfile.js文件后，您可能需要重新构建树，因为Gulp.js不会动态更改树。

全部收缩

单击此按钮可隐藏所有任务树，并仅显示Gulpfile.js节点。

单击此按钮可配置当前视图并更改工具窗口的查看模式 ，有关详细信息，请参阅工具Windows查看模式 。请注意，大多数菜单项都是可以打开或关闭的选项。打开的选项在其名称的左侧有一个复选标记。Gulp特定的选项是：

  • Gulp设置：选择此菜单项以打开Gulp Settings对话框并重新配置Gulp和Node解释器的当前设置，请参阅任务运行器 。

  • 排序方式：选择此菜单项以配置任务在树中显示的顺序。点击在工具栏上，从菜单中选择排序依据 ，然后选择名称 。
    默认情况下，树以Gulpfile.js （选项定义顺序 ）中的顺序显示任务。

隐藏

单击此按钮可隐藏工具窗口。要再次显示它，请选择“ 查看”|工具窗口|在主菜单上喝一杯 。工具窗口再次显示所有先前构建的任务树。

树的上下文菜单

Gulp设置

选择此菜单项以打开Gulp Settings对话框并查看或编辑Node.js配置

跳转到源

选择此菜单项以打开Gulpfile.js构建当前树的Gulpfile.js文件。

重新加载任务

选择此菜单项可重新构建所选节点下的任务树。

复制路径

选择此菜单项以保存Gulpfile.js文件的路径，根据该文件将当前树构建到剪贴板。

删除Gulpfile.js

选择此菜单项可删除所选节点下的任务树。

任务的上下文菜单

运行<任务名称>

选择此菜单项以运行所选任务。

调试<任务名称>

选择此菜单项以调试所选任务。

编辑<任务名称>设置

选择此菜单项以打开“ 运行/调试配置”对话框，然后编辑所选任务的预定义设置。

跳转到源

选择此菜单项以打开Gulpfile.js构建当前树的Gulpfile.js文件，并导航到所选任务的定义。
