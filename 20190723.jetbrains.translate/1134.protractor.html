量角器

在开始之前，请确保您的计算机上有Node.js.

Protractor是Angular和AngularJS应用程序的测试框架。量角器针对在真实浏览器中运行的应用程序执行测试，从而确保测试结果的正确性和可信度。IntelliJ IDEA与Protractor集成，因此您可以从IDE内部运行和调试测试。您可以在树视图中查看测试结果，并从那里轻松导航到测试源。

安装量角器

您也可以在Node.js和NPM页面上安装protractor包，如npm和Yarn中所述 。

您可以在本地，项目或全局安装Protractor。全局安装更可取。

全局安装Protractor

  1. 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install -g量角器

  2. 要下载必要的二进制文件，请键入webdriver-manager update 。另请参阅量角器官方网站上的入门 。

运行测试

或者，在“ 项目”工具窗口中选择一个测试文件，然后从上下文菜单中选择“ 创建<文件名> ”。

量角器测试仅通过运行/调试配置启动。

要创建Protractor运行配置

  1. 打开“ 运行/调试配置”对话框（主菜单上的“运行”|“编辑配置 ”）。点击在工具栏上，从列表中选择量角器 。“ 运行/调试配置：量角器”对话框打开。

  2. 指定要使用的Node.js解释器。这可以是Windows子系统Linux上的本地Node.js解释器或Node.js。

  3. 指定protractor包的位置以及protractor.conf.js配置文件的路径。如果您遵循标准安装，IntelliJ IDEA会检测所有这些路径并将其显示在相应的字段中。

通过运行配置运行测试

  1. 手动启动Selenium Server或将其配置为在protractor.conf.js自动启动。有关详细信息，请参阅量角器官方网站。

  2. 从主工具栏上的列表中选择Protractor运行/调试配置，然后单击在列表的右侧。

  3. 在“运行”工具窗口的<current_run_configuration_name>选项卡中查看和分析来自服务器的消息。

导航

使用IntelliJ IDEA，您可以在文件和相关测试文件之间跳转，或者从Test Runner选项卡中的测试结果跳转到测试。

  • 在编辑器中打开文件，然后选择Go To |测试或转到|从上下文菜单中测试主题 ，或者只需按Ctrl + Shift + T.

  • 要从测试结果跳转到测试，请在“ 测试运行器”选项卡中选择测试名称，然后从上下文菜单中选择“ 跳转到源 ”。测试文件在编辑器中打开，光标位于测试定义中。

测试文件应遵循流行的命名约定，例如，具有.test. ， .spec.或_spec.后缀，应位于源文件旁边或test文件夹中。

调试测试

Protractor测试的调试会话仅通过运行/调试配置启动。

开始调试测试

  1. 如上所述创建Protractor运行/调试配置。

  2. 从主工具栏上的列表中选择Protractor运行/调试配置，然后单击在列表的右侧。

  3. 在打开的调试工具窗口中，像往常一样继续： 逐步执行测试 ， 停止并恢复测试执行， 暂停时检查测试 ，等等。
