配置Python SDK

以下内容仅在安装并启用Python插件时有效！

在IntelliJ IDEA中，您可以定义多个Python SDK。要查看可用SDK列表，请选择“ 文件”|主菜单上的项目结构 （ Ctrl + Shift + Alt + S ）。Python SDK可以在以下级别配置：

IntelliJ IDEA支持：

  • 标准Python解释器

  • IronPython的
  • PyPy
  • Jython的
  • CPython的
选定的SDK可用于所有Python项目。
选定的SDK将用于当前项目及其所有模块。
选定的SDK将用于当前项目模块。

有关SDK配置的更多信息，请参阅SDK 。

无论级别如何，您都可以配置本地，远程或虚拟环境安装的Python解释器：

  • Virtualenv环境
  • 皮彭夫环境
  • 康达环境
  • 系统解释器
  • SSH解释器
  • 流浪汉
  • 搬运工人
  • Docker撰写

对于任何配置的Python解释器，您可以：

  • 安装和卸载解释器路径
  • 安装和卸载解释器包
