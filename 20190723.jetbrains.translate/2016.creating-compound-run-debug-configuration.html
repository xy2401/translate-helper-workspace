创建复合运行/调试配置

假设您想同时启动多个运行/调试配置。例如，您可能希望运行多个测试配置的序列，或运行多个不同类型的配置（例如，JavaScript，HTML等）。您可以使用复合运行/调试配置来配置此行为。复合配置可以包括具有相同或不同指定目标的相同或不同类型的多个配置。

使用复合配置运行或调试代码时，实际上会按顺序启动它所包含的一系列配置。

创建复合运行/调试配置

  1. 打开“ 运行/调试配置”对话框 。您可以从Run |访问它编辑配置主菜单或使用工具栏上的运行配置切换器。

  2. 在打开的对话框中，单击并从“ 添加新配置”菜单中选择“ 化合物 ”。

  3. 在“ 名称”字段中，指定配置的名称。

  4. 选择“ 共享”以使此运行/调试配置可供其他团队成员使用 。

  5. 要将新的运行/调试配置包含到复合配置中，请单击并从列表中选择所需的一个。

  6. 单击“ 应用”和“ 确定”按钮完成。
