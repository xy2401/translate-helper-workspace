启用应用程序服务器集成插件

此功能仅在Ultimate版本中受支持。

默认情况下启用所有可用的服务器集成插件 。但是，在开始使用应用程序服务器之前，始终确保启用必要的插件或插件是值得的。

服务器集成插件名称通常包括服务器名称和集成或支持一词，例如，GlassFish Integration，dmServer支持，Tomcat和TomEE集成等。

  1. 打开“ 设置/首选项”对话框（ Ctrl + Alt + S ）。

  2. 在对话框的左侧，选择“ 插件” 。

  3. 在对话框的右侧，在“ 插件”页面上 ，在搜索框中键入感兴趣的服务器名称（例如tomcat ）。因此，只有名称和描述包含类型文本的插件才会显示在插件列表中。

  4. 如果未选中插件名称右侧的复选框，请选择它。

  5. 在Settings对话框中单击OK 。

  6. 如果建议，请重新启动IntelliJ IDEA。
