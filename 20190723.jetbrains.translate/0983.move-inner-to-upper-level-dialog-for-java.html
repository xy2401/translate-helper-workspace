将内部移动到Java的上层对话框

重构|移动
F6

对于在Structure视图中选择或在编辑器中打开的内部类，将调用Move Inner Class重构对话框。

项目

描述

班级名称

如果需要，使用此字段重命名移动的内部类。

包裹名字

指定要移动到的类的包的名称。

将外部类的实例作为参数传递

如果希望移动的类保留对其以前的外部类的访问权限，请选中此复选框。

参数名称

在此指定作为参数传递的外部类实例的名称。

在评论和字符串中搜索

选择此选项可将更改应用于注释和字符串。

搜索文本事件

选择此选项可将更改应用于项目中包含的文档，HTML，JSP和其他文件。

搜索参考

选中此复选框可将更改应用于对相关文件的引用。此选项仅适用于Move File或Move Package重构。
