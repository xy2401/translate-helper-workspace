间谍js工具窗口

仅当启用Spy-js插件时，工具窗口才可用。该插件不与PyCharm捆绑在一起，但它可以从JetBrains插件存储库安装，如管理插件中所述 。

当您为Node.js启动Spy-js或Spy-js类型的运行配置时，将打开工具窗口。一旦Spy-js工具捕获事件，它就会显示事件本身，其详细信息和堆栈跟踪。通过单击事件，您可以打开跟踪文件 ，该文件是具有事件启动脚本的美化源文件 。

工具窗口包含一个常用工具栏和两个带窗格的选项卡：

  • 跟踪运行选项卡
  • 跟踪代理服务器选项卡

工具提示和快捷方式

描述

重新运行
按Ctrl + F5

单击此按钮可停止当前跟踪会话并再次运行它。通过单击从外部取消会话 ，按钮切换到 。请注意，在会话重新启动后，您必须刷新浏览器中的跟踪页面以开始捕获事件。

重新运行
F9

这个按钮代替了如果通过单击从外部取消上一个跟踪会话 。点击发起新会议。请注意，在会话重新启动后，您必须刷新浏览器中的跟踪页面以开始捕获事件。

停止
按Ctrl + F2

单击此按钮可通过标准shutdown脚本在外部终止当前进程。

单击该按钮一次调用soft kill，允许应用程序捕获SIGINT事件并执行正常终止（在Windows上，模拟Ctrl + C事件）。单击按钮一次后，将替换为表示后续点击将导致强制终止应用程序，例如在Unix上发送SIGKILL 。

恢复布局

单击此按钮可以放弃对当前布局的更改并返回默认状态。

Pin标签

单击此按钮可固定或取消固定当前选项卡。您可能需要固定一个标签，以防止在此窗口中达到最大标签数时自动关闭标签。

关
按Ctrl + Shift + F4

单击此按钮可关闭“ 运行”工具窗口的选定选项卡，并终止当前进程。
