配置数据库连接

使用JDBC驱动程序创建与数据库的连接

如果在数据源列表中找不到数据库供应商的名称，请下载数据库管理系统（DBMS）的JDBC驱动程序，并在PyCharm中创建连接。使用JDBC驱动程序，您可以连接到DBMS并开始工作。

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 在“ 数据源和驱动程序”对话框中，单击“ 添加”图标（ ）并选择驱动程序和数据源 。

  3. 单击驱动程序链接。

  4. 在“ 驱动程序文件”窗格中，单击“ 添加”图标，然后选择“ 自定义JAR” 。

  5. 导航到JDBC驱动程序的JAR文件，选择它，然后单击“ 确定” 。

  6. 在“ 类”字段中，指定要用于驱动程序的值。

  7. 单击“ 应用”

  8. 返回创建的数据源连接。

  9. 指定数据库连接细节或者，将JDBC URL粘贴到URL字段中。

    要设置空密码，请右键单击“ 密码”字段，然后选择“ 设置为空” 。

  10. 要确保与数据源的连接成功，请单击“ 测试连接” 。

自定义连接禁用Сode完成，检查和其他IDE功能。

启用单连接模式

单连接模式意味着数据源和所有控制台使用同一个连接。此模式允许您查看数据库树中的临时对象，或在不同的控制台中使用相同的事务。有关控制台的更多信息，请阅读数据库控制台 。

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 选择要启用单一连接模式的数据源。

  3. 单击“ 选项”选项卡。

  4. 选择单连接模式 。

  5. 单击“ 应用”

为数据源指定颜色

您可以为数据源指定颜色（例如，为了更好地区分生产数据库和测试数据库）。

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 选择要为其指定颜色的数据源。

  3. 单击“ 选择颜色”图标（ ）并选择一种颜色。

  4. 单击确定 。

使用SSL连接到数据库

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 选择要更改连接设置的数据源配置文件。

  3. 单击SSH / SSL选项卡，然后选择使用SSL复选框。

  4. 在CA文件字段中，导航到CA证书文件（例如， ca.pem ）。

  5. 在“ 客户端证书文件”字段中，导航到客户端证书文件（例如， client-cert.pem ）。

  6. 在客户端密钥文件字段中，导航到客户端密钥文件（例如， client-key.pem ）。

  7. 要确保与数据源的连接成功，请单击“ 测试连接” 。

建议使用PEM证书。

对于自签名证书，在某些情况下使用受信任的根实体颁发的证书，在使用最新的JDBC驱动程序版本时可能会遇到错误。如果Java密钥库不接受证书链，则SSL连接可能会失败。作为临时解决方案，尝试降级JDBC驱动程序（例如，对于MySQL连接器，您需要切换到5.1.40版本。）

使用SSH连接到数据库

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 选择要更改连接设置的数据源配置文件。

  3. 单击“ SSH / SSL”选项卡，然后选中“ 使用SSH隧道”复选框。

  4. 在代理主机 ， 代理用户 ， 代理密码和端口字段中，指定连接详细信息。

  5. 从Auth类型列表中，选择一种身份验证方法：

    • 密码 ：使用密码访问主机。指定密码，然后选择保存密码复选框以将密码保存在PyCharm中。

    • 密钥对（OpenSSH或PuTTY） ：使用密钥对的SSH身份验证 。要应用此身份验证方法，您必须在客户端计算机上拥有私钥，并在远程服务器上拥有公钥。PyCharm支持使用OpenSSH实用程序生成的私钥。

      指定存储私钥的文件的路径，并在相应字段中键入密码（如果有）。要让PyCharm记住密码，请选中Save passphrase复选框。

    • OpenSSH配置和身份验证代理 ：使用由凭据帮助应用程序管理的SSH密钥（例如，Windows上的Pageant或Mac和Linux上的ssh-agent ）。

  6. 要确保与数据源的连接成功，请单击“ 测试连接” 。

有关使用SSH密钥的详细信息，请参阅生成新的SSH密钥并将其添加到ssh-agent教程。

将JDBC驱动程序添加到现有连接

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ），然后单击数据源属性图标（ ）。

  2. 在“ 数据源和驱动程序”对话框中，选择要更改驱动程序的数据源，然后单击数据源设置中的“ 驱动程序”链接。

  3. 在“ 驱动程序文件”窗格中，单击“ 添加”图标（ ）并选择Custom JARs 。

  4. 导航到JDBC驱动程序的JAR文件，选择它，然后单击“ 确定” 。

  5. 在“ 类”字段中，指定要用于驱动程序的值。

  6. 单击“ 应用”

记住密码

您可以选择以下选项来存储密码：

  • 从不 ：每次与数据库建立连接时都会提示输入密码。

  • 在重新启动之前 ：仅为当前的PyCharm运行保存密码。如果退出PyCharm并再次打开它，则必须再次提供密码。

  • 对于会话 ：仅为与数据库的当前连接会话保存密码（直到与其断开连接）。您可以通过按Disconnect图标终止连接（ ）或Ctrl + F2 。

  • 永远 ：密码保存在PyCharm存储中。下次打开PyCharm时无需提供密码。
