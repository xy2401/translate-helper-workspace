拉大家，推倒会员

Pull Members Up重构允许您在两个类具有相同字段时移动字段，或者如果您的子类具有执行类似工作的方法，则移动方法。这有助于您摆脱重复的代码。您还可以将类成员移动到超类或接口，或接口到超接口。

Push Members Down重构允许您通过将类成员移动到子类或子接口来清理类层次结构。然后，成员仅重定位到直接子类/接口。这提高了内部类的一致性。

  1. 选择要上拉或下拉的项目。

  2. 在主菜单或上下文菜单中，选择Refactor |拉成员或重构|推送会员 。

  3. 在打开的对话框中，根据您的重构，指定目标对象，要移动的成员以及其他附加信息。

  4. 单击“ 重构” 。
