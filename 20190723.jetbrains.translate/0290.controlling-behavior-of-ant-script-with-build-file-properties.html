用构建文件属性控制Ant脚本的行为

您可以控制IntelliJ IDEA执行Ant脚本的方式。如果要在后台执行执行，更改分配给构建过程的内存量，或者设置其他执行选项，则需要打开“ 构建文件属性”对话框。本节介绍如何执行此操作。

打开“构建文件”属性对话框

  1. 在Ant Build Tool窗口中 ，选择所需的构建文件。

  2. 执行以下操作之一：

    • 在选择的上下文菜单上，选择“ 属性” 。

    • 点击 Ant Build工具栏上的按钮。

    • 按Alt + Enter 。

有关特定任务，请参阅以下过程：

  • 增加构建过程的内存堆 。

  • 在后台执行构建文件 。

  • 定义运行时属性 。

  • 定义其他Ant类路径 。

  • 定义Ant执行选项 。

  • 定义Ant过滤器 。
