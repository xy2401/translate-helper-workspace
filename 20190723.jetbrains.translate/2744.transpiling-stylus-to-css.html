在开始之前，请确保您的计算机上有Node.js.

PyCharm允许您动态地将Stylus代码转换为CSS。

安装手写笔

您也可以在Node.js和NPM页面上安装stylus软件包，如npm和Yarn中所述 。

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install --global stylus

    从Stylus官方网站了解更多信息。

编译Stylus代码

要动态编译代码，需要将编译器配置为PyCharm 文件观察器 。

创建文件监视器

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 工具”下的“ 文件监视器 ”。打开的“ 文件监视器”页面显示已配置的文件监视器列表。

  2. 点击或按Alt + Insert并从列表中选择Stylus预定义模板。

  3. 按照使用文件监视器中的说明继续操作。

编译代码

当您打开文件时，PyCharm会检查当前项目中是否有适用的文件观察程序。如果配置但禁用了此类文件观察程序，PyCharm会显示一个弹出窗口，通知您已配置的文件观察程序并建议启用它。

如果在当前项目中配置并启用了适用的文件观察程序，PyCharm将根据“ 新建观察器”对话框中指定的事件自动启动它。

  • 如果选中“ 自动保存已编辑的文件以触发观察者”复选框，则只要对源代码进行任何更改，就会调用“文件观察器”。

  • 如果清除了自动保存已编辑的文件以触发观察者复选框，则会在保存（ 文件|全部保存 ， Ctrl + S ）或从PyCharm移动焦点时（启用帧停用时）启动文件观察器。

PyCharm使用生成的输出创建一个单独的文件。该文件具有源Stylus文件的名称和扩展名css 。生成的文件的位置在“ 新建观察器”对话框的“ 输出路径刷新”字段中定义。但是，在项目树中 ，它们显示在源文件下，该文件现在显示为节点。

配置语法突出显示

您可以根据自己的喜好和习惯配置手写笔感知语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案|手写笔 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
