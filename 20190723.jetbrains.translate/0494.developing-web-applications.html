介绍

术语Web应用程序通常应用于具有HTML，JSP，JavaScript等文件和资源的分层文件夹集，表示用户界面，以及实现应用程序功能的Java类。这些文件夹根据为部署和执行应用程序的服务器指定的标准进行排列。为确保Web应用程序的成功部署和可访问性，您需要根据开发阶段已有的目标服务器标准设置其结构。

接下来，您很可能需要配置Web应用程序元素，设置初始化参数以及定义与EJB，WebServices等的交互。您将需要各种库来配置这些组件和关系以及包含这些组件的定义的设置描述符，设置和约束。

使用IntelliJ IDEA，您可以根据所需标准自动设置基本应用程序结构，并在Web应用程序开发的所有阶段获得广泛的编码帮助。

开发Web应用程序

要在IntelliJ IDEA中开发Web应用程序，请执行以下常规步骤：

  1. 启用Web应用程序开发支持以获取必要的库，基本文件夹层次结构和应用程序描述符。

  2. 填充Web模块 ：创建所需的Java类，配置servlet，过滤器，侦听器和引用。

  3. 配置所需的静态Web内容资源 。

  4. 配置Web应用程序部署 。

  5. 部署并运行应用程序。
