WebStorm 2019.1帮助

HTML

WebStorm为HTML提供强大的支持，包括语法和错误突出显示，根据代码样式进行格式化，结构验证，代码完成，调试会话期间的实时预览（ 实时编辑 ）等等。

HTML规范可使用语言和框架|上的默认HTML语言级别首选项进行配置WebStorm设置的模式和DTD页面（ Ctrl + Alt + S ）。默认情况下，假定来自W3C的规范HTML 5.0。

创建HTML文件

  • 在主菜单上，转到文件|单击“ 新建” ，然后从列表中选择“ HTML文件 ”。WebStorm基于HTML文件模板创建存根文件，并在编辑器中打开它。

在HTML文件中创建引用

在<script> ， <link>或<img>标记内，WebStorm建议完成您引用的文件的路径。

或者，在“ 项目”工具窗口中，选择要引用的JavaScript，CSS或图像文件，然后将其拖到HTML文件中。WebStorm在<head>生成<script> ， <link>或<img>标记。对于<img>标签，WebStorm还会生成width和height属性。

文档查找

请注意，仅针对Chrome，Chrome Android，Safari，Safari iOS，Firefox，Internet Explorer和Edge检查兼容性。

对于大多数HTML标记和属性，WebStorm可以显示相应MDN文章的摘要。此摘要显示在“ 文档”弹出窗口中，该窗口还显示标记或属性的弃用状态以及与各种浏览器的兼容性信息。

查看插入标记或属性的文档

  • 按Ctrl + Q或选择查看|主菜单中的快速文档查找 。WebStorm打开文档弹出窗口，其中包含标记或属性的简要说明。

    • 如果标签或属性在所有版本的浏览器中都可用，则WebStorm不会显示有关其兼容性的任何信息。

    • 否则，“ 文档”弹出窗口还会列出支持标记或属性的浏览器及其版本。

    • 如果不推荐使用标记或属性，则弹出窗口也会通知您此状态。

在浏览器中打开标记或属性的MDN文档

  • 在“ 文档”弹出窗口（ Ctrl + Q ）中，单击底部的链接。

  • 按Shift + F1或选择查看|主菜单中的外部文档 。

在浏览器中预览HTML文件的输出

要隐藏所有图标或其中一些图标，请清除Web浏览器页面上不必要浏览器的“ 活动”复选框。

  • 选择查看|在主菜单上的“ 浏览器 ”中打开，然后从列表中选择所需的浏览器。要打开WebStorm默认浏览器 ，请选择“ 默认” 。

  • 或者，将鼠标指针悬停在代码上以显示浏览器图标栏，然后单击指示所需浏览器的图标：

  • 如果您只配置了一个浏览器，只需按Alt + F2即可 。

在编辑器中查看网页的HTML源代码

  1. 选择文件|打开网址 。

  2. 在打开的“打开URL”对话框中，键入网页的URL地址，或从列表中选择以前打开的URL。

查看嵌入的图像

WebStorm提供了几种查看HTML文件中嵌入的图像的方法。您可以使用导航来源 ， 在外部图形编辑器中打开图像 ，或者即时预览图像 。

在“ 图像”页面上检查并配置预览外观。

在WebStorm中查看图像

  • 在“ 项目”工具窗口中选择图像文件，然后从选择的上下文菜单中选择“ 跳转到源 ”或按F4 。

  • 或者，将光标放在编辑器中对图像的引用上，然后从上下文菜单中选择“ 跳转到源 ”或按Ctrl + B.

在外部编辑器中查看图像

  1. 在“ 图像”页面上配置外部编辑器的路径（“ 文件”|“设置”|“编辑器”|“适用于Windows和Linux的映像”或“ WebStorm”|“首选项”|“编辑器”|“适用于macOS的映像 ”）。

  2. 在“ 项目”工具窗口中选择图像文件，然后选择“ 在外部编辑器中打开”或按Ctrl + Alt + F4 。

提取包含文件

WebStorm还建议添加引用而不是所选片段的重复。

您可以将HTML代码片段提取到单独的包含文件中。还可以提取<script>标记内的整个JavaScript代码块。

  1. 在编辑器中，选择要提取的代码块，然后选择Refactor |提取|在主菜单上或从选择的上下文菜单中提取包含文件 。

  2. 在打开的“ 提取包含文件”对话框中，指定不带扩展名的包含文件的名称以及存储它的目录。您可以接受预定义目录或选择另一个目录。

  3. 准备好后 ，单击“ 确定” 。WebStorm将选定的源代码提取到目标目录中的指定文件中，并在源文件中生成相应的引用。

配置语法突出显示

您可以根据自己的喜好和习惯配置HTML感知语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案| HTML 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
