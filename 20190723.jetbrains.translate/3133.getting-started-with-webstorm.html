WebStorm入门

WebStorm是用于现代JavaScript开发的强大IDE。WebStorm提供对JavaScript，TypeScript，HTML，CSS以及React，Angular和Vue.js等框架的完全支持，不需要额外的插件。

除了客户端应用程序，WebStorm还可以帮助您使用Node.js开发服务器端应用程序，使用React Native或Cordova开发移动应用程序，使用Electron开发桌面应用程序。请注意，强烈建议使用Node.js，尽管没有它可以使用核心WebStorm功能。

打开，签出或创建项目

您还可以从模板生成项目 ，有关详细信息，请参阅生成特定于框架的项目 。

有关更多详细信息，请参阅创建文件和目录 。

WebStorm中的项目是一个文件夹，其中包含您编辑的源代码，您使用的库和工具（例如，在node_modules子文件夹中），以及各种应用程序配置文件（例如， package.json或.eslintrc ）。

在WebStorm中打开文件夹后，会将.idea子文件夹添加到WebStorm存储其内部配置设置的位置，例如，项目代码样式或版本控制系统。

从代码检查中了解更多信息并获取结果并解决问题 。

重构你的代码

有关详细信息，请参阅重构代码 ， 重构JavaScript和重构TypeScript 。

重构意味着更新源代码而不改变应用程序的行为。重构可帮助您保持代码的稳定， 干燥和易于维护。WebStorm确保在重构代码之后以与之前相同的方式工作，因为对整个项目进行了明智的更改。例如，如果重命名一个类，WebStorm会自动重命名其所有用法和import语句。

  1. 在编辑器或“ 项目”工具窗口中，选择要重构的表达式或符号，然后按Ctrl + Shift + Alt + T.

  2. 从Refactor This列表中，选择所需的重构。

有关更多信息，请参阅使用运行/调试配置 。

在WebStorm中，运行或调试应用程序的入口点是运行/调试配置 。WebStorm附带了许多针对不同类型的应用程序和文件的预定义运行/调试配置模板。您需要在配置中提供的信息取决于其类型，可以是要运行的文件还是测试。某些配置可以附加到已在运行的应用程序，在这种情况下，您需要指定要附加到的URL和端口。

开始调试

使用WebStorm，您可以调试各种应用程序 - 客户端应用程序，Node.js应用程序，测试等。以下是调试在外部开发Web服务器上运行的客户端JavaScript的方法，例如由Node.js提供支持。

  1. 根据需要在JavaScript代码中设置断点 。

  2. 在开发模式下运行应用程序，可能使用如上所述的 npm start ，并在浏览器中复制运行应用程序的URL地址。

  3. 选择运行|在主菜单上编辑配置 ，单击在工具栏上，从列表中选择JavaScript Debug 。在“ 运行/调试配置：打开的JavaScript调试”对话框中，指定运行应用程序的URL地址。可以从浏览器的地址栏复制此URL，如上面的步骤2中所述。

  4. 在工具栏上的“ 选择运行/调试配置”下拉列表中选择新创建的配置，然后单击“确定” 。运行配置中指定的URL地址将在浏览器中打开，并显示“ 调试”工具窗口 。

  5. 在调试工具窗口中，像往常一样继续： 逐步执行程序 ， 停止并恢复程序执行， 暂停时检查它 ， 查看实际的HTML DOM ，等等

遵循您的代码风格标准

这些设置也在重构期间应用。

WebStorm根据每种语言特定的代码样式设置自动格式化所有新代码。您可以自己配置代码样式，也可以应用TSLint或ESLint配置文件中的代码样式规则。通过在WebStorm中激活和配置TSLint以及从ESLint导入代码样式了解更多信息 。

要根据项目代码样式设置重新格式化代码，请选择代码片段或整个文件，然后按Ctrl + Alt + L.

将源代码保留在版本控制下

有关更多信息，请参阅使用WebStorm进行版本控制 。

如果您将源代码保持在版本控制之下，您将很高兴知道WebStorm集成了许多流行的版本控制系统，如Git（或GitHub），Mercurial，Perforce，TFS，Subversion和CVS。在大多数情况下，WebStorm会自动检测您的VCS。要指定凭据和任何设置，请转到“ 设置”对话框的“ 版本控制”页面 （ Ctrl + Alt + S ）。

VCS菜单将为您提供有关可用命令的线索。在版本控制工具窗口 （ Alt + 9 ）中，您可以查看和跟踪您和您的队友所做的更改，创建更改列表，提交和推送您的更改等等。

最常用的操作，如Commit，Push，View History也可以从VCS Operations弹出窗口（ Alt +` ）获得。

自定义您的环境

随意调整WebStorm，以便它完美地满足您的需求，并尽可能提供帮助和舒适。

出现

微调的第一件事是一般的“外观和感觉”。默认的WebStorm主题很轻松。如果你喜欢更暗的环境中，在设置/首选项（ 按Ctrl + Alt + S），单击外观下的外观和行为 ，然后选择Darcula。

编辑

您还可以调整编辑器行为的各个方面，例如，启用或禁用Drag'n'Drop，自定义编辑器选项卡的行为，为每种支持的语言配置突出显示，编辑代码折叠设置，更改代码完成策略，以及多得多。要自定义编辑器的功能，请按Ctrl + Alt + S，然后在“ 设置/首选项”对话框中，选择“ 编辑器”下的相关页面。

按键图

WebStorm是一个以键盘为中心的IDE，这意味着它中几乎所有动作都映射到键盘快捷键 。WebStorm附带一个默认的键盘映射，您可以随时更改它以符合您的习惯，如配置键盘快捷键中所述 ，只需按Ctrl + Alt + S并选择外观和行为下的键盘映射 。
