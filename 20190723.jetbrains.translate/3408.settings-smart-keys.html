家

选中此复选框后，在按Home时 ，插入符号位于当前行的第一个非空格字符处。按Home后，将插入符号从Smart Home位置移动到第一列并返回。

结束（在空白行）

选中此复选框后，在空行中按End键时，插入符号将与缩进一起定位，WebStorm认为该缩进在当前代码点中是合理的（缩进基于当前的代码样式设置 ）。

插入对括号

选中此复选框可让WebStorm分别为每个打字的圆形或方括号自动添加一个闭合圆形或方括号。

插入对报价

选中此复选框可让WebStorm分别为每个打开的单引号或双引号自动添加结束单引号或双引号。

键入“}”时重新格式化块

如果选中此复选框，则在键入结束花括号时，如果此代码块的格式与所选代码样式不匹配，则会自动重新格式化随附的代码块。

使用'CamelHumps'字样

选中此复选框可让WebStorm在CamelHump名称中识别单独的单词。名称中的单词应以大写字母或下划线开头。此选项会影响某些编辑器操作，例如：

  • 插入符号移动 （ Ctrl +向右 / Ctrl +向左 ）

  • 使用选择移动插入符号 （ Ctrl + Shift +向右 / Ctrl + Shift +向左 ）

  • 在Caret中选择Word （ Ctrl + W ）

  • 删除到Word开始/结束 （分别为Ctrl + Backspace和Ctrl + Delete ）

  • 双击 （如果在编辑|常规设置页面上启用了使用双击选择时的荣誉“CamelHumps”字设置 ）。

键入引号或括号的环绕选择

如果选中此复选框，则键入引号，双引号或括号的所选文本将被这些字符包围。如果未选中此复选框，则键入的引号，双引号或大括号将替换选择。

使用箭头键在双Ctrl / Add上添加多个插入符号

如果选中此复选框，则：

  • 按Ctrl键 （对于Windows或* NIX）或⌥ （对于macOS）两次加上/向下箭头键可以创建多个插入符号。

  • 按Ctrl （对于Windows或* NIX）或⌥ （对于macOS）两次加上左/右箭头键或Home / End导致创建选择。

有关更多信息，请参阅“ Multicursor”部分。

跳转到闭合括号/引用Tab

如果选中此复选框，则在键入括号/引号内时按Tab键会将插入符号移到结束括号/引号之外。如果未选中此复选框，则按Tab键将插入Tab字符。

请注意，在初始打字这仅适用于：后续的编辑过程中，按Tab键括号内/引号将插入Tab符。

输入

使用此区域可以按Enter键来定义要调用的操作。
  • 智能缩进 - 选中此复选框可让WebStorm添加新行并将插入符号放入其中，并使用WebStorm假定在当前代码点中合理的缩进（缩进基于当前的代码样式设置）。

    如果清除该复选框，则在空白行中按Enter键时，WebStorm会添加一个新行并将插入符号放在当前的非空格字符列中。

  • 插入对'}' - 选中此复选框可让WebStorm在空行中按Enter键时自动将右括号}定位在正确的列上。在这种情况下，WebStorm向后寻找最近的未闭合的开口支撑{并将关闭的一个放在相应的缩进水平。

  • 插入文档注释存根 - 此复选框定义在打开文档标记后按Enter键的行为。
    • 如果选中此复选框，WebStorm将生成文档注释存根。

      对于函数注释，此存根包含必需的标记（签名中声明的每个参数的@param标记和@return ）。有关详细信息，请参阅创建JSDoc注释 。

    • 如果未选中此复选框，则仅生成结束标记。

    请注意，此复选框指的是JavaScript，以及具有特殊文档注释开头的其他语言。

退格

使用此下拉列表可以通过按Backspace键来定义要调用的操作。可用选项包括：

  • 禁用 - 按退格键一次将插入符号返回一个位置。

  • 到最近的缩进位置
  • 适当的缩进

重新格式化粘贴

使用此下拉列表指定粘贴代码块的放置方式。可用选项包括：

  • 无 - 粘贴的代码作为纯文本插入到插入符号位置，无需重新格式化或缩进。

  • 缩进块 - 根据当前的代码样式设置 ，粘贴的代码块位于正确的缩进级别，但其内部结构不会更改。

  • 缩进每一行 - 根据当前的代码样式设置 ，粘贴的代码块的每一行都位于适当的缩进级别。

  • 重新格式化块 - 根据当前的代码样式设置重新格式化粘贴的代码块。

此功能适用于包含尾随换行符的行。

XML / HTML

在此区域中，定义编辑XML或HTML代码时要自动调用的操作。

  • 在标记完成时插入结束标记 ：选中此复选框可让WebStorm在输入相应的开始标记时自动插入结束XML或HTML标记。

  • 在标记完成时插入必需的属性 ：选中此复选框可让WebStorm显示包含类型标记的所有必需属性的模板。

  • 在标记完成时插入必需的子标记 ：选中此复选框可使WebStorm显示包含所有必需子标记的模板。

  • 标记完成时启动属性 ：选中此复选框可让WebStorm显示带有类型标记的第一个必需属性的模板。

  • 在键入'='时为属性值添加引号 ：选中此复选框可让WebStorm自动为当前键入的属性值添加引号。

  • 键入“</”时自动关闭标记 ：选中此复选框可在输入后自动添加结束标记</。清除此复选框以关闭此类自动完成功能。

  • 同时<tag> </ tag>编辑 ：

    • 选中此复选框并编辑开始标记后，相应的结束标记会自动更改。

    • 如果清除此复选框，则编辑开始标记不会影响保持不变的结束标记。结果，打开和关闭标签不匹配，并且整个构造被加下划线作为错误。

    此复选框控制WebStorm在以下上下文中的行为：

    • HTML文件

    • JavaScript代码中的HTML注入

    • 带有模板Handlebars / Moustache模板的HTML

    • Handlebars / Moustache模板文件，扩展名为.hbs

    • XML，XHTML文件

    • DTD文件

    • JSX文件

    如果未选中此复选框，请双击CSS标识符或类名称选择名称的一部分，直到最近的连字符：

JSON

此区域中的设置控制在JSON上下文中自动添加逗号，引号和空格，符合JSON语法。默认情况下启用所有设置。

AngularJS

使用此区域来定义AngularJS的行为：

  • 在插值中自动插入空格 ：如果选中此复选框，则会在大括号之间自动插入空格： {{ }} 。
    如果未选中此复选框，则不会插入空格： {{}} 。
