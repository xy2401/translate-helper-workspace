PyCharm使用捆绑插件实现Vagrant功能，可以通过清除PyCharm设置的插件页面上的Vagrant复选框（ Ctrl + Alt + S ）来完全禁用。

确保满足以下先决条件（在PyCharm之外）：

  • Oracle的VirtualBox安装在您的计算机上。

  • Vagrant已安装在您的计算机上，并且已创建所有必需的基础结构。

  • 以下可执行文件的父文件夹将添加到系统PATH变量中：

    • 您的Vagrant安装中的vagrant.bat或vagrant 。这应该由安装人员自动完成。

    • Oracle的VirtualBox安装中的VBoxManage.exe或VBoxManage 。

  • 将创建所需的虚拟框。

配置远程Python解释器

  1. 确保您已正确启动并启动Vagrant。基本上，您需要打开终端窗口并执行以下命令：

    $ vagrant init ubuntu / trusty64
    和
    $ vagrant up
    有关更多信息，请参阅Vagrant文档 。

  2. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，打开“ 插件”页面并安装Vagrant插件。

  3. 通过以下任一方式打开“ 添加Python解释器”对话框：

    • 当您在编辑器中时 ，最方便的方法是使用状态栏中的Python解释器小部件。单击小部件并选择Add Interpreter ...

    • 如果您位于“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，请选择“ 项目<项目名称>”项目口译员 。点击图标，然后选择添加 。

  4. 在“ 添加Python解释器”对话框的左侧窗格中，单击“ Vagrant” ：

  5. 单击“浏览”按钮在Vagrant实例文件夹字段旁边，并指定所需的Vagrant实例文件夹。

    这导致显示到Vagrant主机URL的链接。

  6. Python解释器路径字段显示所需Python可执行文件的路径。您可以接受默认值，也可以指定其他默认值。

  7. 单击确定 。配置的远程解释器将添加到列表中。

  8. 回到Project Interpreter页面，如有必要，配置路径映射：

    1. 点击 ：

    2. 在打开的对话框中添加（ ）或删除（ ）根据需要进行路径映射。
