标签和缩进

使用制表符

  • 如果选中此复选框，则选项卡字符用于缩进和代码重新格式化。

  • 清除该复选框后，WebStorm使用空格而不是制表符。

缩进

在此文本框中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。
