集成文件对话框（Perforce）

VCS |整合项目
文件或目录的上下文菜单| Perforce |集成文件/目录

使用此对话框可将更改列表从一个分支规范集成到另一个分支规范。

项目

描述

分支规格

选择将用于更改集成的分支规范。考虑以下：

  • 如果启用了“ 反向”选项，则会将更改从所选分支集成到本地副本。

  • 如果禁用“ 反向”选项，则会将更改从本地副本集成到所选分支。

整合变更清单

使用此选项可调用更改浏览器 ，您可以在其中选择将集成到当前分支/本地副本的更改列表。

将更改存储到更改列表

指定应存储集成更改的更改列表。

在同步之前恢复未更改的文件（p4 revert -a）

选择此选项可还原未更改的文件。

同步后自动运行解析（p4 resolve -am）

选择此选项可自动解析无冲突即可解析的文件。
