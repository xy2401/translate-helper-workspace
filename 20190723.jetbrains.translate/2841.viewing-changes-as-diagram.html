以图表形式查看更改

对于版本控制下的模块，通常可以在“更改”工具窗口中查看已更改文件的列表。如果要评估更改如何影响模型，请使用UML类图。此视图显示了更改的完整图片，包括修改后的类之间的关系。

为此，PyCharm提供了操作Show changed classes ，它可以从编辑器，Project工具窗口和Version Control工具窗口的Local Changes选项卡中获得。

此外，PyCharm可帮助您分析更改如何影响修订版本中的模型，并提供操作比较UML版本中的所有类 。

类图中的更改采用颜色编码：

  • 绿色添加元素。

  • 已删除元素的灰色 。

  • 蓝色为修改过的元素。

查看UML类图中的更改

  1. 在“版本控制”工具窗口的“ 本地更改”选项卡中，从建议的弹出窗口中选择所需的更改列表：

    请注意，如果只有一个更改列表，弹出窗口会赢; t显示。

  2. 执行以下操作之一：

    • 在“ 本地”选项卡的工具栏中，单击 ，或按Ctrl + Shift + Alt + D.

    • 在编辑器或项目工具窗口的上下文菜单中，选择Diagrams |显示更改 ，或按Ctrl + Shift + Alt + D.

    该图以弹出窗口打开。双击节点以查看差异查看器中的更改。

要将修订中的更改视为UML类图

  1. 在“ 版本控制”工具窗口的“ 历史记录”选项卡中，选择所需的修订。

  2. 点击 ，或按Ctrl + Shift + D.该图以弹出窗口打开。
