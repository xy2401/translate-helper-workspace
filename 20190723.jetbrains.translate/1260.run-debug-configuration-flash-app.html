运行/调试配置：Flash App

使用此对话框可以创建或编辑Flash App 运行/调试配置 ，以便您运行和调试ActionScript和Flex应用程序。

请注意，此运行/调试配置类型不适用于其输出为RLM或库的构建配置 。

另请注意，设置取决于关联的构建配置（Web，桌面（AIR）或移动（AIR Mobile））中指定的目标平台。

  • 仅限名称，共享和单个实例
  • 构建配置和覆盖主类
  • 特定于Web目标应用程序的设置
  • 特定于桌面目标应用程序（AIR）的设置
  • 特定于移动设备的应用程序的设置（AIR Mobile）
  • 在启动选项之前
  • 工具栏

仅限名称，共享和单个实例

项目

描述

名称

使用此字段可编辑运行/调试配置的名称。

分享

选中此复选框以通过版本控制共享运行配置。

如果未选中该复选框，则运行配置设置将存储在.idea/workspace.xml或.iws文件中。

如果选中此复选框，则设置将存储在.idea/runConfigurations或.ipr文件中的单独.xml文件中。

见项目 。

仅限单个实例

如果选中此复选框，则一次只运行一个运行配置实例。

特定于Web目标应用程序的设置

项目

描述

什么推出

选择以下选项之一：

  • 构建输出。 使用此选项可以使用关联的HTML包装器运行生成的SWF文件。

  • URL或本地文件。 使用此选项可在Web浏览器中打开指定的URL，或运行指定的本地.swf文件（直接或使用相应的.html包装器）。

    在字段中键入所需的URL。通常，这将类似于http(s)://<host>:<port>/<context-root> 。

    对于本地文件，您可以使用并在打开的对话框中选择必要的.swf或.html文件。

推出

指定用于运行应用程序的程序。您可以使用：

  • 系统默认应用程序，即与操作系统中的目标文件类型（HTML或SWF）关联的程序。通常，这是一个Web浏览器。对于SWF文件，这也可能是一个独立的Flash播放器。

  • 一个Web浏览器。

  • 一个Flash播放器。

要选择所需的程序，请单击在“ 启动时”字段的右侧，然后在“ 启动方式”对话框中指定程序：

  • 要选择系统默认应用程序，只需单击系统默认应用程序 。

  • 要选择浏览器，请单击“ 浏览器”，然后从列表中选择所需的浏览器。

    此外，您可以访问“ Web浏览器”对话框以调整Web浏览器设置。要打开此对话框，请使用在列表旁边。

  • 要选择Flash播放器，请单击“ Flash Player” ，然后单击“确定” 在Flash Player字段旁边。然后，在打开的对话框中指定所需Flash播放器的位置。

    • 新实例 （仅在macOS上可用）。如果要启动新的Flash Player实例，请选中该复选框。否则，如果Flash Player已在运行，则将打开正在运行的实例的新窗口。

使用SDK中的调试器

指定包含应使用的调试器的Flex SDK。（如果您使用Flex SDK 3进行编译，建议使用Flex SDK 4进行调试。）

如果已在IntelliJ IDEA中定义了相应的SDK，请从列表中选择SDK。否则，请单击 （ Shift + Enter ）并在打开的“ 配置SDK”对话框中添加必要SDK的定义。

将SWF文件放在本地受信任的沙箱中

如果选择了“ 构建输出”选项 ：如果要将应用程序SWF文件注册为受信任，请选中此复选框。

受信任的SWF文件可以与任何其他SWF文件进行交互。他们可以从任何地方，远程或本地加载数据。

从技术上讲，可信赖的SWF文件被分配给本地受信任的沙箱 。

项目

描述

AIR Debug Launcher选项

如有必要，请指定AIR Debug Launcher选项。

使用与指定程序参数相同的规则。

程序参数

指定要传递给应用程序的参数。

  • 使用空格分隔各个参数。

  • 如果参数包含空格，请将空格或包含空格的参数括在双引号中，例如， some" "arg或"some arg" 。

  • 如果参数包含双引号（例如，作为参数的一部分），则通过反斜杠转义双引号，例如-Dmy.prop=\"quoted_value\" 。

特定于移动设备的应用程序的设置（AIR Mobile）

项目

描述

运行

指定您是要使用模拟器还是真正的Android或iOS设备来运行或调试应用程序：
  • 模拟器。 选择此选项以使用内置仿真器。从列表中选择目标设备。列表右侧显示的数字是所选目标设备的屏幕特征，仅供您参考。

    • 屏幕是您的应用程序可用的屏幕大小。

    • Full是设备的全屏大小。

    • ppi （每英寸像素数）是屏幕像素密度。

    如果列表中不存在必需的设备，则可以选择“ 其他”并在相应的字段中指定所需的屏幕参数。

  • Android设备。 选择此选项可使用Android移动设备运行或调试您的应用程序。

    • 为了能够使用Android设备运行或调试您的应用程序，必须在相关的构建配置（ Android选项卡上的Enabled选项 ）中启用打包Android应用程序。

    • 通常，使用Android设备运行或调试应用程序时，假定在目标设备上安装应用程序包。要使此操作成功，必须使用USB将移动设备连接到您的计算机。

  • iOS模拟器。 （此选项仅适用于Mac计算机。）选择此选项可使用iOS设备模拟器运行或调试应用程序。在SDK字段中，指定要使用的Apple iOS SDK的路径。（模拟器包含在Apple iOS SDK中。）

    你可以点击 （ Shift + Enter ）并在打开的对话框中选择SDK安装文件夹。

    为了能够使用iOS模拟器来运行或调试应用程序，必须在相关的构建配置（ iOS选项卡上的Enabled选项 ）中启用打包iOS应用程序。

  • iOS设备。 选择此选项可使用iOS移动设备运行或调试应用程序。如有必要，启用快速应用程序打包。（如果启用了快速打包，则会解释ActionScript字节码，而不会将其转换为机器代码。因此，打包速度更快，但代码执行速度较慢。）

    • 为了能够使用iOS设备运行或调试应用程序，必须在关联的构建配置（ iOS选项卡上的Enabled选项 ）中启用打包iOS应用程序。

    • 通常，使用iOS设备运行或调试应用程序时，假定在目标设备上安装应用程序包。要使此操作成功，必须使用USB将移动设备连接到您的计算机。

    • 如果您使用的是AIR SDK 3.4或更高版本，则应用程序包将自动安装在目标设备上。对于早期的SDK版本，您必须手动启动安装。

在设备上调试

对于Android或iOS设备：指定在安装和启动应用程序后设备与计算机的通信方式：
  • 网络。 如果设备要通过网络与您的计算机通信，请选择此选项。

    在设备上安装应用程序包需要USB连接。

  • USB。 如果设备将使用USB连接到您的计算机，请选择此选项。如有必要，请更改IntelliJ IDEA建议的端口。

ADL选项（模拟器）

对于模拟器：如有必要，请指定AIR Debug Launcher （ADL）选项。

使用与指定程序参数相同的规则。

应用程序描述符（模拟器）

对于模拟器：指定要使用的应用程序描述符。可用选项指的是相关构建配置中与描述符相关的设置。

  • 设置为Android表示Android选项卡上的相应设置。

  • 为iOS设置是指iOS选项卡上的设置。

根据构建配置中的设置，可能出现以下情况：

  • 为Android设置：<未启用Android支持>。 生成Android的描述符已禁用。如果要启用相应选项，请使用Android选项卡上的已启用复选框。

  • 为Android设置：生成。 将使用自动生成的描述符。如果要使用基于模板的描述符，请选择“ 自定义模板”选项并指定模板。

  • 为Android设置：<file_name> .xml。 相应的自定义模板将用于生成描述符。在这种情况下， <file_name>是将使用的模板文件的名称。

对于iOS选项，也可以使用相应的情况。

在启动选项之前

指定在启动运行/调试配置之前应执行的任务。指定的任务按它们在列表中出现的顺序执行。

项目

捷径

描述

Alt +插入

单击此图标可将任务添加到列表中。选择要添加的任务：

  • 运行外部工具。 选择此选项可运行IntelliJ IDEA外部的应用程序。在打开的对话框中，选择应运行的应用程序。如果尚未在IntelliJ IDEA中定义必要的应用程序，请添加其定义。有关更多信息，请参阅配置第三方工具和外部工具 。

  • 使。 选择此选项可以编译项目或模块。如果在运行/调试配置中指定了特定模块，则执行Make Module命令 ，否则执行Make Project命令 。

    如果在编译期间发生错误，IntelliJ IDEA将不会尝试启动运行/调试配置。

  • 制作，没有错误检查。 与Make选项相同，但IntelliJ IDEA将尝试启动运行/调试配置，而不管编译结果如何。

  • 构建工件。 选择此选项可构建工件或工件。在打开的对话框中，选择应构建的工件或工件。

    另请参阅使用工件 。

  • 运行另一个配置。 选择此选项可执行另一个运行/调试配置。在打开的对话框中，选择要运行的配置。

    仅当您在当前项目中至少有一个运行/调试配置时，此选项才可用。

  • 运行Ant目标。 选择此选项可运行Ant目标。在打开的对话框中，选择要运行的目标。有关更多信息，请参阅Ant 。

  • 生成CoffeeScript源地图。 选择此选项可以生成CoffeeScript源的源映射。在打开的对话框中，指定CoffeeScript源文件的位置。

    有关更多信息，请参阅CoffeeScript 。

  • 运行Maven目标。 选择此选项可以运行Maven目标。在打开的对话框中，选择要运行的目标。

    有关更多信息，请参阅Maven 。

Alt + Delete键

单击此图标可从列表中删除所选任务。

输入

单击此图标可编辑选定的任务。在打开的对话框中进行必要的更改。

Alt +向上

单击此图标可将所选任务在列表中向上移动一行。

Alt +向下

单击此图标可将所选任务在列表中向下移动一行。

显示此页面
