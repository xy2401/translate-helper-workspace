科学模式

此功能仅在Ultimate版本中受支持。

Python插件中的科学模式为交互式科学计算和数据可视化提供支持。必须正确安装以下软件包：

  • Matplotlib

  • NumPy的

检测到numpy导入时，IntelliJ IDEA会显示建议您启用科学模式的横幅：

科学模式

启用此模式后，UI将引入以下更改：

视图|默认情况下， 未选择“ 科学模式”命令。

  • 将自动选择“ 视图”菜单中的“ 科学模式”检查命令。

  • 显示了Python控制台。

  • 将出现“ 文档”工具窗口 （“快速文档”弹出窗口的固定版本），显示插入符号的内联文档：

  • DataFrame / Array视图出现在Python控制台和Debug工具窗口中 。

在科学模式下，图形在SciView窗口的自己的选项卡中打开，允许您调整其大小，放大和缩小等。您可以通过切换工具窗口中的显示图复选框来更改此行为（ 设置/首选项|工具| Python Scientific ）。

Matplotlib在控制台中

Matplotlib也可在控制台中使用。请参阅Scientific Mode教程中的Running in console部分。

    在“ 格式”字段中，可以指定表示输出的格式。
