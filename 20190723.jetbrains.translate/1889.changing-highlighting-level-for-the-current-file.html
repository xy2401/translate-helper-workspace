更改当前文件的突出显示级别

使用状态栏快速重新配置当前在编辑器中打开的文件的突出显示。使用Hector ，您可以选择突出显示语法问题，检查问题或关闭突出显示。

更改当前文件的突出显示级别

  1. 通过执行以下操作之一打开突出显示级别弹出窗口：

    • 在主菜单上，选择Code |配置当前文件分析 。

    • 按Ctrl + Shift + Alt + H.

    • 单击Hector图标在状态栏上。

    • 右键单击滚动条顶部的代码检查指示器。

  2. 将滑块移动到定义突出显示级别的三个可用位置之一：

    • 无 ：关闭编辑器中突出显示的问题。

    • 语法 ：仅突出显示语法问题。

    • 检查 :(默认）突出显示语法问题和检查问题。
