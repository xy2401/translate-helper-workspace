在PyCharm中使用Vim编辑器仿真（IdeaVim）

以下内容仅在安装并启用IdeaVim插件时有效！

在你开始之前

确保：

  • 您正在使用PyCharm 4.0.0或更高版本。如果您还没有PyCharm，请从此页面下载。要安装PyCharm，请按照说明操作，具体取决于您的平台。

  • 您的计算机上至少安装了一个Python解释器。您可以从此页面下载口译员。

下载并安装IdeaVim插件

在PyCharm主窗口的工具栏上，按Ctrl + Alt + S打开“ 设置/首选项”对话框，然后单击“ 插件” 。

您会看到当前安装在您计算机上的插件列表。但是，IdeaVim插件不在其中。单击市场选项卡，在搜索字段中键入单词“vim”以缩小列表范围，然后选择IdeaVim插件：

安装插件后，它实际上在PyCharm重启后可用。

重启后PyCharm的用户界面会发生什么？

首先，在“ 工具”菜单上，出现一个检查命令Vim Emulator ：

PyCharm重启后，选中此检查命令。您可以通过取消选中此检查命令来禁用Vim。

其次，在“设置/首选项”对话框中，在“ 编辑器”下添加了一个附加页面Vim Emulation 。

配置快捷方式

Vim和PyCharm都是以键盘为中心的。使用IdeaVim插件 ，PyCharm的键映射很可能与Vim键映射发生冲突。这就是为什么PyCharm允许您选择您喜欢哪种键盘快捷键进行特定操作的原因。这就是它的完成方式。

在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”| Vim Emulation 。

在“ 快捷方式”列中，选择要配置的快捷方式。接下来，在Handler列中，单击相应的单元格，然后查看三个可能选项（Undefined，IDE，Vim）的下拉列表：

如果选择IDE ，则表示启用了此特定操作的PyCharm快捷方式。当您按下Ctrl + C时 ，PyCharm会静默执行其操作。

如果您未定义处理程序，则在按下快捷键（例如Ctrl + C）时 ，PyCharm会显示横幅。

要将此快捷方式重新定义为IDE快捷方式，从而接受PyCharm的键映射，请单击IDE快捷方式链接。如果单击Vim Emulation链接，则PyCharm将显示Editor | “ 设置/首选项”对话框的“ Vim仿真”页面。

编辑模式

当你启用Vim时，你会看到光标已经改变了它的形状 - 现在它是一个块，这意味着你处于Normal模式 ：

如果要进入插入模式 ，请按i ，光标将变为一行：

在此模式下，您可以键入新内容或更改现有代码。同样，您可以输入各种Vim模式：例如，按r 替换模式 。

顺便说一下，只要您输入Vim仿真，它也会在状态栏中报告。

要返回“正常”模式，请按Escape键。
