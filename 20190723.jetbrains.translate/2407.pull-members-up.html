Pull Members Up重构允许您将类成员移动到超类。

Pull Member Up重构可以创建抽象方法。如果项目使用解释器Python 2.x，则只能抽象实例方法。如果项目使用Python 3.x，则可以抽象出任何方法。

请注意，PyCharm会自动添加抽象方法所需的import语句。

Push Members Down重构通过将类成员移动到子类来帮助清理类层次结构。然后，成员仅重新定位到直接子类中。

拉动成员

  1. 选择要移动到超类的类。

  2. 在主菜单或上下文菜单中，选择Refactor |拉会员 。将出现“ 拉出成员”对话框。

  3. 选择目标对象（超类）。

  4. 在“ 成员”部分中，选择要移动的成员。

  5. 要将方法移动为抽象，请选中相关方法旁边的Make abstract列中的复选框。

  6. 单击“ 重构”将所选成员拉到目标位置。

打倒会员

  1. 在编辑器中，打开需要按下其成员的类。

  2. 在主菜单或上下文菜单中，选择Refactor |推送会员 。“推送成员”对话框显示要下推的成员列表。

  3. 在要下推的成员区域中，选择要移动的成员。请注意，插入符号中的成员已被选中。

    如果推送会员可能会导致问题，则会通过红色突出显示通知您。这意味着，如果情况无人看管，重构后会出现错误。PyCharm会在“检测到问题”对话框中提示您，您可以选择忽略或修复问题。

  4. 预览并应用更改。
