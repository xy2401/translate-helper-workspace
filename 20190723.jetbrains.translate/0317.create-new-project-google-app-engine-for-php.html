新项目：适用于PHP的Google App Engine

档案|新|项目 - PHP - App Engine项目

使用此对话框配置Google App Engine for PHP项目设置。

安装并启用Google App Engine for PHP插件时，仅在Ultimate版本中支持此功能。 该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

项目

描述

应用程序ID

在此文本框中，键入在“ 创建应用程序”页面上指定的应用程序的标识符。

SDK目录

在此文本框中，指定安装Google App Engine SDK for PHP的文件夹的路径。

Python可执行文件

在此文本框中，指定Python可执行文件的位置。手动键入路径或单击“ 浏览” 并在打开的对话框中选择可执行文件。
