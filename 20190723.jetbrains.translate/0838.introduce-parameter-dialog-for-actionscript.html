为ActionScript提取参数对话框

重构|提取|参数
按Ctrl + Alt + P

使用此对话框指定与ActionScript中的“ 提取参数”重构相关的选项和设置。

项目

描述

类型

指定新参数的类型。通常，您无需更改IntelliJ IDEA建议的类型。

名称

指定新参数的名称。

值

要用新参数替换的表达式。最初，此字段包含您选择的表达式。通常，不需要更改此初始值。

可选参数

如果要将新参数作为可选参数，请选中此复选框。

有关必需参数和可选参数的信息，请参阅Flex / ActionScript文档中对函数参数的讨论。

替换所有出现的事件

选择此选项可替换函数中所选表达式的所有匹配项。
