查看|工具窗口|版本控制 - 日志 - 提交的上下文菜单 - Mercurial - Goto补丁

仅当您的项目位于Mercurial版本控制下时，此选项卡才可用。

它显示所有修补程序，并允许您管理Mercurial队列 。

您可以在队列中拖放修补程序以更改它们的应用顺序。

工具栏和上下文菜单

项目

捷径

描述

可从


从文件重新加载

按Ctrl + F5

使用此命令刷新series文件中的修补程序列表。

请注意，在队列中拖放修补程序时，只有在切换到其他选项卡或执行其他外部操作时，才会保存series文件中的相应更改。这允许您只需从文件重新加载即可还原此类更改。

工具栏


去

SHIFT + ALT + G

使用此命令将选定的修补程序及其上方的所有修补程序应用于队列中。应用的修补程序将在“ 日志”选项卡中显示 ，更改将在存储库的工作副本中注册。

工具栏
上下文菜单


移动和推动

SHIFT + ALT + P

使用此命令应用选定的修补程序，而不应用修补程序堆栈中可能存在的任何其他修补程序。

工具栏
上下文菜单


折

SHIFT + ALT + d

使用此命令将选定的未应用修补程序与最顶层应用的修补程序合并，并将其从列表中删除。

工具栏
上下文菜单

删除

使用此命令从系列文件中删除选定的修补程序

N / A

使用此命令应用选定的修补程序。将显示“ 应用修补程序”对话框，您可以在其中选择要还原的更改。
