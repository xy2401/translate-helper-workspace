Stylelint

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

在开始之前，请确保您的计算机上有Node.js.

PyCharm与Stylelint集成，因此您可以从IDE内部检查CSS代码。编辑CSS文件时，Stylelint会自动唤醒并突出显示检测到的问题 。要在信息弹出窗口中查看问题的描述，请将鼠标指针悬停在验证侧栏中的条带上。要检查错误并应用建议的快速修复 ，请按Alt + Enter 。

安装Stylelint

您还可以在Node.js和NPM页面上安装stylelint包，如npm和Yarn中所述 。

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install -g stylelint

激活和配置Stylelint

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”|样式表| Stylelint 。

  2. 在打开的Stylelint页面上，选中Enable复选框以激活Stylelint。之后，对话框中的控件变为可用。

  3. 在Node Interpreter字段中，指定要使用的Node.js解释器。这可以是Windows子系统Linux上的本地Node.js解释器或Node.js。

  4. 在“ 样式包”字段中，指定全局或当前项目中安装的stylelint包的位置。
