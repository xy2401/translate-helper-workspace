为测试创建运行/调试配置

您可以使用运行/调试配置运行测试（测试用例，测试套件等），其方式与运行普通应用程序类似。PyCharm提供了一个框架，用于创建用于测试目的的特殊运行/调试配置，其中可以将测试指定为目标。

除了创建和编辑运行/调试配置一节中描述的常规过程外，PyCharm还提供了一个快捷方式，允许您为容器中的所有测试，单个测试用例或甚至测试方法创建运行/调试配置。 。

创建测试配置

  1. 右键单击所需目标，例如“ 项目”工具窗口中的目录或单个测试。如果是单独测试，您可以在编辑器中打开它，然后右键单击背景。

  2. 从选择的上下文菜单中，为单个测试选择Create <name> ：

    对于测试方法，在编辑器中打开该类，然后右键单击方法中的任意位置。上下文菜单建议命令Create <method name> 。

  3. 在打开的对话框中，指定运行/调试配置参数，应用更改并关闭对话框。

    Ctrl +空间代码完成在运行/调试配置对话框中有效，例如在工作目录字段中。
