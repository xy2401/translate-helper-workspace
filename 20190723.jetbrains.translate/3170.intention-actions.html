意图行动

当您在编辑器中工作时，WebStorm会分析您的代码，搜索优化代码的方法，并检测潜在和实际问题。

一旦IDE找到改进代码的方法，它就会显示一个黄色的灯泡图标在当前行旁边的编辑器中。通过单击此图标，您可以查看当前上下文中可用的意图操作 。意图行动涵盖从警告到优化建议的各种情况。

一个黄色的灯泡图标还表示WebStorm已检测到问题并提示您快速修复它。对于严重问题，WebStorm显示带有感叹号的红色灯泡而不是浅黄色的 。WebStorm建议在检查工具提示中提供第一个可用的快速修复程序，每次将鼠标悬停在警告或错误上时都会出现。

您可以在“ 设置/首选项”对话框中打开完整的意图操作列表，查看其描述，并按照下面的查看所有意图操作中的说明禁用或启用它们。

有关详细信息，请参阅配置检查严重性和代码检查 。

应用意图行动

  1. 单击灯泡图标（或按Alt + Enter ）以打开建议列表。

  2. 从列表中选择一个操作，然后按Enter键 。例如，您可以使用意图操作将变量的声明与其初始化合并：

应用快速修复

  1. 将鼠标悬停在代码中的错误或警告上。WebStorm显示检查工具提示，并显示问题描述旁边的第一个可用修订。

  2. 要应用建议的修复，请按Alt-Shift-Enter / Option-Shift-Enter 。

  3. 要查看包含所有可用快速修复和意图操作的常用弹出窗口，请按Alt + Enter 。

查看所有意图操作

您可以查看可用意图操作的完整列表，并禁用不需要的操作：

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|用心 。

  2. 对于每个意图操作，您可以看到描述和示例。要禁用意图，请清除它旁边的复选框。

  3. 应用更改并关闭对话框。

即时禁用意图操作

  1. 单击灯泡图标（或按Alt + Enter ）以打开建议列表。

  2. 选择要禁用的操作，然后单击向右箭头。

  3. 选择禁用<意图操作名称> 。
