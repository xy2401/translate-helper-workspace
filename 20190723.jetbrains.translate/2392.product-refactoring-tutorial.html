PyCharm重构教程

本教程的内容

本教程使用一个使用有理数的简单类的例子，展示了PyCharm中可用的一些重构。

先决条件

确保满足以下先决条件：

  • 您正在使用PyCharm版本2016.2或更高版本。

  • 已经创建了一个项目。

准备一个例子

在项目中创建一个Python文件rational.py并添加以下代码：

from collections import namedtuple class Rational （ namedtuple （ 'Rational' ， [ 'num' ， 'denom' ]））： def __new __ （ cls ， num ， denom ）： if denom == 0 ： raise ValueError （ 'Denominator not not null' ） 如果 denom < 0 ： num ， denom = - num ， - denom 返回 super （） 。 __new__ （ cls ， num ， denom ） def __str __ （ self ）： return '{} / {}' 。 格式 （ 自我 。NUM， 自我 。DENOM）

简化有理数

让我们通过将分子和分母除以最大公约数来简化有理数：

from collections import namedtuple class Rational （ namedtuple （ 'Rational' ， [ 'num' ， 'denom' ]））： def __new __ （ cls ， num ， denom ）： if denom == 0 ： raise ValueError （ 'Denominator not not null' ） 如果 denom < 0 ： num ， denom = - num ， - denom x = abs （ num ） y = abs （ denom ）， 而 x ： x ， y = y ％ x ， x factor = y return super （） 。 __new__ （ cls ， num // factor ， denom // factor ） def __str __ （ self ）： return '{} / {}' 。 格式 （ 自我 。NUM， 自我 。DENOM）

提取方法

现在，让我们将搜索最大公约数提取到单独的方法中。为此，请选择语句

x = abs （ num ） y = abs （ denom ）， 而 x ： x ， y = y ％ x ， x factor = y

然后按Ctrl + Alt + M.在打开的对话框中键入方法名称（ gcd ），然后单击“ 确定” ：

@staticmethod def gcd （ denom ， num ）： x = abs （ num ） y = abs （ denom ） 而 x ： x ， y = y ％ x ， x factor = y return factor

内联局部变量并更改方法签名

让我们通过使用内联变量重构来摆脱变量factor 。为此，请将插入符号放在相关变量上，然后按Ctrl + Alt + N.所有检测到的factor变量都是内联的。

接下来，使用Change signature更改参数名称。为此，请将插入符号放在方法声明行中，然后按Ctrl + F6 。在打开的对话框中 ，分别将参数denom和num重命名为x和y ，然后单击改变参数的顺序。

您最终得到以下代码：

@staticmethod def gcd （ x ， y ）： x = abs （ x ） y = abs （ y ） 而 x ： x ， y = y ％ x ， x 返回 y

使用快速修复

现在，让我们将现有的静态方法转换为函数。要执行此操作，请按Alt + Enter ，从建议列表中选择将静态方法转换为功能 ，然后按Enter键 ：

将该函数移动到另一个文件

现在，我们将把函数移动到一个单独的文件并添加一个import语句。为此，请将插入符号放在函数gcd声明中，然后按F6键。在打开的对话框中 ，指定目标文件util.py的完全限定路径。此文件不存在，但会自动创建：

def gcd （ x ， y ）： x = abs （ x ） y = abs （ y ） 而 x ： x ， y = y ％ x ， x 返回 y

import语句也会自动添加。因此， rational.py文件如下所示：

from collections import utiltuple from util import gcd class Rational （ namedtuple （ 'Rational' ， [ 'num' ， 'denom' ]））： def __new __ （ cls ， num ， denom ）： if denom == 0 ： raise ValueError （ 'Denominator不能为空”） 如果 DENOM <0：NUM，DENOM = - NUM， - DENOM 因子 = GCD（NUM，DENOM） 返回 超级 （）。 __new__ （ cls ， num // factor ， denom // factor ） def __str __ （ self ）： return '{} / {}' 。 格式 （ 自我 。NUM， 自我 。DENOM）

提取方法并使用快速修复

接下来，我们将表达式Rational(other, 1)提取到一个单独的方法中。为此，请将插入符号放在上述表达式中，按Ctrl + Alt + M，然后在打开的对话框中键入新方法名称from_int 。

最后，将插入符号放在方法from_int声明中，按Alt + Enter ，从建议列表中选择Make method static ，然后按Enter键 ：

@staticmethod def from_int （ other ）： 返回 Rational （ 其他 ， 1 ）

最后，让我们改变参数的名称other ，以number 。为此，请将插入符号放在参数上，然后按Shift + F6键 。

提取超类

接下来，我们将方法__radd__ ， __sub__和__rsub__的实现移动到超类中。另外，我们将使方法__neg__和__add__抽象的。

这就是它的完成方式......将插入符号放在Rational声明类中，在上下文菜单中指向Refactor |提取并选择Superclass ....接下来，在打开的对话框中 ，指定超类的名称（此处为AdditiveMixin ）并选择要添加到超类的方法。对于方法__neg__和__add__ ，选择Make abstract列中的复选框。
