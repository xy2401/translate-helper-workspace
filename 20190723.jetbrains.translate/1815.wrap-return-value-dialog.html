重构|包裹返回值

使用此重构可以围绕方法的返回值创建包装类，或者使用兼容的现有类作为包装器。

项目

描述

包装返回的方法

此只读字段显示所选方法的名称。

创建新课程

单击此单选按钮以创建新的包装类。如果选择此选项，请在下面的字段中指定类和目标包名称。

班级名称

输入新包装类的名称。

包裹名字

默认情况下，显示当前包名称。您可以在文本字段中键入其他包名称，或单击省略号按钮并从树视图中选择目标包。如果所需的包不存在，请单击创建一个新的。

目标目标目录

使用此字段选择目标目标目录。默认情况下，显示当前目标目录。您通常根据当前包选择目标目标。如果此包存在于多个根中，则可以单击箭头按钮并从列表中选择“ 保留在同一源根 ”。在这种情况下，包装器将放置在初始类附近。
您可以单击省略号按钮以打开“选择目标目录”窗口。
您可以选择“ 目录结构”选项卡以选择另一个目标目录，或者选择“按邻居类别选项卡”将包装器放在邻居类附近，例如，如果要将包装器放在您的Pair或Triple类附近的util目录中而不是记住确切的包，将包装器放在Pair类附近会节省你的时间。

使用现有的课程

单击此单选按钮以使用您选择的现有类作为包装器。

名称

指定所需包装类的名称。请注意，此类应包含一个构造函数，其参数与相关返回值的类型相同。
您可以在文本字段中键入完全限定的类名，或单击省略号按钮并在“ 选择参数类”对话框中选择所需的类。请注意，您可以从项目类和非项目类中选择所需的包装类。

包装场

从指定包装类中遇到的字段下拉列表中选择将存储返回值的字段。

内心阶级

单击此单选按钮以创建内部类。例如，如果您有私有方法，则可能需要这样做。在这种情况下，您可以将所有内容保留在同一个类中。

名称

指定内部类的名称。
