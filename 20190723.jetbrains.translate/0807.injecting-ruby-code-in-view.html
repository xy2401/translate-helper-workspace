在View中注入Ruby代码

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

视图文件包含Ruby代码的标记和孤岛。在以下情况下，视图文件中的任何文本都将被视为Ruby代码：

  • 在*.html.erb文件中，如果此类文本包含<%和%>字符。

  • 在*.html.haml文件中，如果此类文本遵循等号。除此之外，如果以==开头，则正确解释Ruby字符串。

因此，在视图中插入了封闭的字符后，您可以键入Ruby代码，享受Ruby感知语法和错误突出显示以及各种编码帮助：

要在* .html.erb视图中注入Ruby代码，请执行以下操作之一

  • 键入开口尖括号< ，按Ctrl + Space ，然后从建议列表中选择% ：

    IntelliJ IDEA自动完成结束字符。这样做，插入符号位于<%和%>字符中的下一个编辑位置。

  • 按Ctrl + Shift +。插入开始和结束字符。

  • 使用Ctrl + Alt + T应用其中一个环绕声模板 ：
