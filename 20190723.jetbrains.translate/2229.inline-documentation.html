此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

快速查找文档

只有在使用Ctrl + Space显式调用完成时，PyCharm才能自动显示文档。

快速文档查找可帮助您快速获取随适用格式的文档注释提供的任何符号的信息。PyCharm识别根据JavaScript文档工具格式创建的内联文档。

查看插入符号的文档
按Ctrl + Q或选择查看|主菜单上的快速文档查找 。

自动显示文档

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“代码完成” 。将打开“ 代码完成”页面。

  2. 选中Show the documentation popup复选框并指定已用时间。

要更改快速文档的字体大小

  • 点击在快速文档窗口的右上角，然后移动滑块。

  • 另外
    在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规” ，然后选择“ 使用Ctrl +鼠标滚轮更改字体大小（缩放）”复选框。

文档窗口

“ 文档”弹出窗口有助于通过超链接导航到相关符号，并提供一个工具栏，用于在已导航的页面中来回移动，更改字体大小以及在外部浏览器中查看文档。

固定时，“ 文档”弹出窗口将变为“ 文档”工具窗口 ，其中包含相应的侧边栏图标和更多控件。

要在“ 文档”弹出窗口和“ 文档”工具窗口之间切换，请使用Ctrl + Q.

  • 显示字体大小滑块。根据需要移动滑块以在快速文档窗口中增大或减小字体大小。

  • 从其源启用或禁用文档的自动更新。

  • 控制窗口的位置。
