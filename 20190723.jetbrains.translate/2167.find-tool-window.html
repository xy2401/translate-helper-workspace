查找工具窗口

查看|工具窗口|找
ALT + 3

基本

“ 查找”工具窗口显示以下搜索的结果：

  • 在路径中查找/替换
  • 查找用法
  • 结构搜索和替换

  • 重构预览

  • 查找数据源 ，表或列的用法 。

每次搜索的结果都显示在单独的选项卡中，或者替换当前选项卡的内容，具体取决于“ 在新选项卡中打开”对话框设置。默认情况下，窗口显示在屏幕底部。

它有一个工具栏，其中包含一组按钮，一个结果窗格以及“ 路径替换”和“ 重构预览”操作的其他按钮。

除非您找到某些内容，否则此工具窗口在“ 视图”菜单中不可见。

工具栏按钮

项目

工具提示和快捷方式

描述

选项
按Ctrl + Shift + Alt + F7

单击此按钮可打开其中一个“ 查找用法”对话框，该对话框对应于相关符号。您可以编辑搜索设置，然后单击“ 重新运行”按钮以执行修改后的搜索查询。

重新运行
按Ctrl + F5

重复上一次搜索。此按钮不可用于查看代码覆盖率结果 。

关
按Ctrl + Shift + F4

关闭当前选项卡或工具窗口。“在路径中重置”和“重构预览”对话框中不提供此按钮。

Pin标签

单击此按钮可固定或取消固定当前选项卡。您可能需要固定一个标签，以防止在此窗口中达到最大标签数时自动关闭标签。

最近发现用法
按Ctrl + E

显示最近的搜索列表。选择列表中的项目以查看搜索结果。


展开全部
Ctrl + NumPad Plus
全部收缩
Ctrl + NumPad -

使用这些按钮可以展开或折叠所有节点。

/

上一次/下次发生
按Ctrl + Alt +向上
按Ctrl + Alt +向下

导航到结果选项卡中的上一个/下一个元素。

自动滚动到源

打开或关闭Autoscroll to source选项。当选项打开并且您选择搜索结果时，将在编辑器中打开相应的源文件，并在文件中突出显示相应的片段。

最爱

单击此按钮可将找到的用法添加到收藏夹 。

导出到文本文件
ALT + O

保存当前结果选项卡的内容。在“ 导出预览”对话框中，指定目标文件或将信息复制到剪贴板。保存之前，您还可以修改要保存的信息。

救命
F1

使用此图标或快捷方式打开相应的帮助页面。

按使用类型分组
按Ctrl + T

此按钮仅适用于“ 查找用户” 。
如果按下此按钮，搜索结果将按以下类别分组：

  • 评论

  • 未分类的使用与任何类别无关

按测试/生产分组

如果按下此按钮，则会根据生产和测试范围对用法进行分组。

按模块分组
按Ctrl + d

如果按下此按钮，则找到的用法显示在相应的模块或库节点下。

当包在多个模块之间拆分时，这种类型的分组很有用。

压扁模块

如果选择此图标，则所有模块都显示为单级树视图。

按包分组
按Ctrl + P

如果按下此按钮，则找到的所有用法都显示在各自的包装下。

按文件结构分组
按Ctrl + M

如果启用此切换，则找到的用法将显示在相应的方法节点下。

合并来自同一行的用法
按Ctrl + F

如果启用此切换，则会合并在同一行上找到的重复使用。

显示读取权限
CTRL + R

此按钮仅适用于“ 查找用户” 。
如果按下此按钮，搜索结果将包括对读取访问方法的引用。

显示写访问权限
按Ctrl + W

此按钮仅适用于“ 查找用户” 。
如果按下此按钮，搜索结果将包括对写访问方法的引用。

显示导入语句
按Ctrl + I

此按钮仅适用于“ 查找用户” 。
如果按下此按钮，搜索结果将包含import语句中的用法。

预览用法

打开或关闭“ 预览”窗格。

按字母顺序排序成员

单击此按钮可按字母顺序排序所有成员。否则，成员按声明的顺序排序。
