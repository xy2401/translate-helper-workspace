WebStorm 2019.1帮助

Password Manager数据库已更新

当您使用先前WebStorm版本的配置启动新的WebStorm版本时，此对话框仅打开一次。

WebStorm允许您将当前保存的密码数据库转换为新的密码数据库。

项目

描述

密码

使用此字段输入您在以前版本的WebStorm中使用的主密码。

兑换

单击此按钮可将保存的密码存储在新密码数据库中。

清除密码

如果单击此按钮，将不会转换任何内容，并且不会保存旧密码。如果在未输入主密码的情况下关闭此对话框，将执行相同的操作。
