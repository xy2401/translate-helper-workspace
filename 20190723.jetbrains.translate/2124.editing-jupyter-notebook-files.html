编辑Jupyter Notebook文件

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

设置你的项目

  1. 在PyCharm中，创建一个新的Python项目或打开一个现有项目 。

  2. 为选定的解释器安装 jupyter包。

  3. 完成所有索引过程后，您就可以开始使用笔记本文件了。

要打开现有的.ipynb文件，请执行与其他类型的文件相同的步骤 。如果需要，您可以创建一个笔记本文件。

创建一个笔记本文件

  1. 执行以下操作之一：

    • 在“ 项目”工具窗口中右键单击目标目录，然后从上下文菜单中选择“ 新建 ”。

    • 按Alt + Insert

  2. 选择Jupyter Notebook 。

  3. 在打开的对话框中，键入文件名。

笔记本文档具有*.ipynb扩展名，并标有相应的图标： 。

使用相应的图标在可用的编辑模式之间切换。

编辑笔记本中的源代码

您可以将Markdown，raw和code cells添加到笔记本文件中。

  1. 使用以下方法之一添加新单元格：

    • 点击排水沟最低线的图标。选择要添加的单元格类型。

    • 输入相应的评论：

      评论

      细胞类型

      ＃%% md

      Markdown标记语言。

      ＃%%

      Python代码片段

      ＃％％ 生的

      “Raw NBConvert”细胞类型。

    • 对于代码单元格，请按Shift + Alt + A （Windows）或⌥⌥A （macOS）在当前所选单元格上方插入新单元格，或按Shift + Alt + B （Windows）或⌥⌥B （macOS）创建所选单元格下方的新单元格。

  2. 将适当的内容添加到新创建的单元格中。添加Markdown或原始单元格后，其输出将显示在“ 预览”窗格中。要预览代码单元格生成的结果，您需要先执行它。

点击图标可快速访问Jupyter笔记本基本操作的可用快捷方式列表。

您可以借助Python代码洞察来编辑代码，例如语法突出显示，代码完成等。使用用于控制源代码预览可见性的图标。
