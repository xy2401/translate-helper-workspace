用Builder对话框替换构造函数

重构|用Builder替换Constructor

使用Builder重构的Replace Constructor有助于隐藏构造函数，将其用法替换为对新生成的构建器类的引用，或者替换为现有的构建器类。

项目

描述

要传递给构建器的参数

参数

此列显示构造函数中检测到的参数列表，该列表将替换为构建器字段。

字段名称

此可编辑列显示构建器中建议的字段名称列表。

塞特的名字

此可编辑列显示构建器中建议的setter名称列表。

默认值

使用此可编辑列初始化具有默认值的字段。

可选的二传手

如果字段的指定缺省值与构造函数调用中的参数值匹配，则选中此复选框将导致在构建器调用中省略此字段的setter方法。
如果未选中此复选框，则无论如何都将显示相应的setter方法。

生成器名称和位置

创建新的

单击此单选按钮以使用指定的名称和目标包生成新的构建器类。

Builder类名

此可编辑字段显示要生成的新构建器类的建议名称。您可以接受默认值，也可以键入新的默认值。

新建筑商的套餐

键入目标程序包的名称，或单击“浏览”按钮，然后在“ 选择目标程序包”对话框中找到它。
如果所需的包不存在，请单击创建一个新的。

利用现有的

单击此单选按钮以指定现有构建器类。

Builder类名

在此处键入项目中已存在的所需构建器类的完全限定名称，或单击“浏览”按钮并按名称或项目树视图查找。
