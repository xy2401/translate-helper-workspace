评估控制台窗格中的变量

在JavaScript上下文中，您可以在调试会话期间在“ 调试”工具窗口中查看变量的值。这可以通过以下几种方式完成：

  • 在“ 变量”窗格中选择有问题的变量，然后在选择的上下文菜单中选择“ 在控制台中评估 ”。切换到“ 控制台”窗格时，变量名称将以绿色显示> ，其值将以蓝色显示在下方。

  • 另外，您可以在任何类型变量的名称>在控制台手动，然后按Enter已经显示其价值。代码完成随时可用：当您键入变量名称时，WebStorm会显示建议列表。请注意，仅当按下“ 使用控制台输入”切换按钮时，此功能才可用 。

  • 要在“ 变量”窗格中搜索以前评估的变量而不搜索它，请使用键盘上的向上和向下箭头找到变量，然后按Enter键 。
