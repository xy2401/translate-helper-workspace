创建Web2Py项目

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

Web2Py项目旨在用于Web2Py应用程序的高效开发。PyCharm负责创建特定的目录结构和设置。

要创建Web2Py项目，请按照下列步骤操作

  1. 在主菜单上，选择“ 文件”| New Project ... ，或单击Welcome屏幕中的New Project按钮。“新建项目”对话框打开。

  2. 在“ 新建项目”对话框中，执行以下操作：

    • 指定项目类型Web2Py 。

    • 指定项目位置。

    • 接下来，单击通过单击相应的单选按钮，展开Project Interpreter节点，并选择新环境或现有解释器。以下步骤取决于您的选择：

      • 使用新环境 ：如果已选择此选项，请选择用于创建虚拟环境的工具。为此，请单击列表并选择Virtualenv ， Pipenv或Conda 。

        接下来，指定新虚拟环境的位置和基本解释器。如有必要，请单击“ 继承全局站点包”和“ 使所有项目可用”复选框。

      • 现有解释器 ：如果选择了此选项，请从列表中选择所需的解释器，或者（如果找不到所需的解释器），单击并选择口译员。有关详细信息，请参阅配置Python解释器 。

        当PyCharm停止支持任何过时的Python版本时，相应的项目解释器被标记为不受支持。

  3. 点击 （ 更多设置 ），并指定以下内容：

    • 应用名称

    • 如有必要，请选中“ 使用本地Web2Py”复选框。选中此复选框后，此复选框旁边的文本字段将变为启用状态。单击浏览按钮从文件系统中选择Web2Py源文件夹。

  4. 单击“ 创建”

PyCharm创建一个应用程序并生成特定的目录结构，您可以在项目工具窗口中浏览它。如果有不满足的要求 ，PyCharm建议解决或忽略它们。
