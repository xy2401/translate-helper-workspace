科学模式

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

PyCharm中的科学模式为交互式科学计算和数据可视化提供支持。必须正确安装以下软件包：

  • Matplotlib

  • NumPy的

要了解如何将软件包安装到PyCharm解释器，请阅读安装，卸载和升级软件包一节 。

当检测到numpy导入时，PyCharm会显示建议您启用科学模式的横幅：

科学模式

启用此模式后，UI将引入以下更改：

视图|默认情况下， 未选择“ 科学模式”命令。

  • 将自动选择“ 视图”菜单中的“ 科学模式”检查命令。

  • 显示了Python控制台 。

  • 将出现“ 文档”工具窗口 （“快速文档”弹出窗口的固定版本），显示插入符号的内联文档：

  • DataFrame / Array视图出现在Python控制台和Debug工具窗口中 。

在科学模式下，图形在SciView窗口的自己的选项卡中打开，允许您调整大小，放大和缩小等。您可以通过切换工具窗口中的显示图复选框来更改此行为（ 设置/首选项|工具| Python Scientific ）。

科学项目

您可以通过在“ 新建项目”向导中选择相应的项目类型来获取预定义的所有科学模式设置。

选择Scientific项目类型时，您需要确保安装了Conda解释器。完成后，您只需指定项目名称即可。如果需要，您还可以更改数据文件夹的名称。
