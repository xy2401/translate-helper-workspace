PyCharm 2019.1帮助

查看图表

PyCharm使用捆绑插件实现UML图表功能，可以通过清除PyCharm设置的插件页面上的UML支持复选框（ Ctrl + Alt + S ）来完全禁用它。

基本

您可以从不同的地方调用UML类图：

  • 从各种工具窗口。

  • 从导航栏。

  • 从“结构”工具窗口中。

  • 来自编辑。

PyCharm以两种模式显示UML图：

  • 在弹出窗口中。

  • 在单独的编辑器选项卡中。

PyCharm可以选择图表类型：

  • Python类图

  • Django模型依赖图（用于Django项目）。

  • Google App Engine模型依赖关系图（适用于Google App Engine项目）。

打开UML类图

  1. 选择所需的项目，或在编辑器中将其打开。

  2. 执行以下操作之一：

    • 在选择的上下文菜单上，单击Diagram ，然后在子菜单上，选择要查看模型的方式： Show Diagram或Show Diagram Popup 。

    • 按Ctrl + Shift + Alt + U或Ctrl + Alt + U.

  3. 从列表中选择图表类型：

该图显示在编辑器选项卡或弹出窗口中：

技巧和窍门

  • 您可以在不使用指点设备的情况下打开UML类图。考虑这样的工作流程：按Alt + Home ，然后按Ctrl + Alt + U.
