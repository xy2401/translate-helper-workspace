名称

工件配置的名称以及工件的名称。

类型

工件类型。定义工件格式和结构，突出显示工件布局中有问题的部分以及IntelliJ IDEA建议（或假定可接受）与工件组合相关的选项。

输出目录

构建时，工件 （ Build | Build Artifacts ）将放置在此字段中指定路径的目录中。

如果工件本身不会被使用但仅作为其他工件的一部分使用，则该字段可以留空。在这种情况下，选择Build |时 ，工件不会出现在建议列表中构建工件 ，在选择Build |时 ， build.xml不会生成任何目标生成Ant构建 。

建立在制造上

在构建项目时自动构建工件（ Build | Make Project ）。
