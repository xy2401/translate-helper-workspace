介绍

在模块中启用Seam构面可以使用jPDL创建页面流定义。IntelliJ IDEA提供专用的图形设计器，与源代码完全同步。在Designer中，您可以创建存根页面流，该页面流立即反映在页面流文件的源代码中。要使页面流定义可用，您还必须编辑页面流文件的源代码。

正确配置模块的Web构面的Web资源目录非常重要。特别是，应将包含要包含在页面流中的页面的目录添加到Web资源列表中。

创建页面流定义

创建页面流定义

  1. 在“项目”工具窗口中，右键单击要在其中创建页面流定义文件的目录，例如， WEB-INF目录，选择“ 新建”|“新建”。在上下文菜单上查找页面流 ，并指定文件名。IntelliJ IDEA创建页面流定义存根xml文件，并在编辑器的单独选项卡中打开它。

  2. 单击页面流定义文件的Designer选项卡。

  3. 单击Designer工具箱中的所需按钮，然后将其拖到Designer区域。添加其他节点，并根据需要更改其名称。这样，您可以创建开始和结束状态，页面，决策和流程状态。

  4. 绘制节点之间的链接。

  5. 在“ 文本”选项卡中，编辑源代码以将其与特定视图相关联，添加属性和子元素（指定操作和决策表达式，转换名称，事件，将子进程添加到进程状态等）。请注意属性和值的代码完成：

Designer中提供的操作

Designer中提供了以下操作：

  • 导航：您可以从Designer中的页面流的可视化表示跳转到底层源代码。要执行此操作，请右键单击元素，然后在上下文菜单中选择“ 跳转到源 ”，或按F4 。

  • 显示用法和查找用法：选择一个节点，然后按Ctrl + Alt + F7或Alt + F7 。

  • 重命名重构：选择一个节点，按Shift + F6 ，然后在“重命名”对话框中指定新名称。这样做，底层xml文件中的相应引用也会更新。请注意，如果使用就地编辑器重命名节点，则不会更改源代码中的引用，并突出显示错误。

  • 删除节点和链接：从图中删除节点或链接时，将从基础xml文件中删除相应的元素。
