按Ctrl + Alt + S

单击“ 格式”节点下面的文件类型时，将打开该页面。使用此页面可根据目标文件类型定义版权声明的格式。

项目

描述

没有版权

如果选择此选项，则不会在所选类型的文件中更新版权声明。

使用默认设置

如果选择此选项，则将根据新项目的默认设置更新版权声明。

使用自定义格式选项

如果选择此选项，将根据下面定义的自定义设置更新所选类型文件中的版权声明。

评论类型

在此区域中，指定要附加版权声明的评论类型。可用选项包括：

  • 使用块注释 - 选择此选项可将版权声明包含在块注释中。

  • 为每行添加前缀 - 选中此复选框可使版权声明的每一行都附加在“ 分隔符”字符文本框中定义的字符前面。默认情况下，使用星号。

  • 使用行注释 - 选择此选项可将版权声明表示为行注释序列。

国界

在此区域中，定义如何将版权声明与其他评论分开。

  • 分隔符之前 - 选中此复选框后，将在分隔符字符文本框中定义一行前面的版权声明。

  • 分隔符后 - 选中此复选框后有一个版权声明，后跟在分隔符字符文本框中定义的一行字符。

  • 分隔符char - 在此文本框中，键入将在分隔符字符串中使用的字符以及作为块注释中的前缀或结尾字符。

  • 长度 - 在此文本框中，键入spartor行中的字符数。

  • 框 - 选中此复选框以使版权声明的每一行后跟在分隔符字符文本框中定义的字符 。

  • 添加空行后 - 选中此复选框可在版权声明后插入空白行。

相对位置

在此区域中，指定版权声明相对于其他评论的位置。

  • 在其他注释之前 - 选择此选项时，会在其他注释上方插入版权声明。

  • 在其他评论之后 - 选择此选项时，会在其他评论下方插入版权声明。

预习

使用此区域可以查看包含您已定义格式的示例版权声明。

文件中的位置

使用此区域可在HTML和XML文件中指定版权声明的位置。对于其他文件类型，没有可用的选择。版权声明始终插入文件顶部。
