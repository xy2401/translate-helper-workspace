查看Seam组件

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA提供了在Seam工具窗口中查看项目中检测到的所有Seam构面的可能性。此视图显示按模块分组的所有Seam组件的列表。还包括适当的库组件。

对于具有Seam构面的每个模块，您可以使用专用依赖图来探索其结构。要打开此图形，请使用模块上下文菜单中的“ 显示接缝组件依赖关系”命令。

只读依赖关系图在单独的编辑器选项卡上打开。您可以更改图形的显示方式（放大和缩小，显示或隐藏网格，更改布局等）并执行其他基本操作。

查看Seam组件和依赖项

  1. 要打开Seam工具窗口 ，请选择View |工具窗口|缝 。显示了带有Seam构面的模块。

    您可以通过按F4在项目视图中从组件导航到其定义。

  2. 使用工具栏按钮或键盘快捷键展开（ ， Ctrl + NumPad Plus ）或折叠（ ， Ctrl + NumPad - ）节点。

  3. 要打开模块的依赖关系图，请右键单击感兴趣的模块，然后从上下文菜单中选择Show Seam Components Dependencies 。

  4. 要查看或编辑组件的源代码，请在编辑器中打开该组件。为此，请选择感兴趣的组件，然后按F4 。

    要打开作为树的“叶子”的组件（即层次结构底部的组件），您还可以使用双击。
