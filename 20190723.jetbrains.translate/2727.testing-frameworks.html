PyCharm 2019.1帮助

测试框架

PyCharm支持使用以下测试框架：

  • Python单元测试
  • pytest
  • Python测试
  • Python doctests

  • Tox 。

  • TwistedTrial

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

  • BDD框架：

    • 表现 。

    • 生菜 。

    • 黄瓜 。

    有关详细信息，请参阅BDD测试框架部分。

在开始使用您选择的测试框架之前，请确保在您的计算机上安装了所需的框架。有关安装详细信息，请参阅框架文档。

根据特定支持的测试框架，PyCharm提供：

  • 代码完成 ，了解具体的测试框架。

  • 运行/调试配置 。

  • 能够创建测试 。

  • 能够在测试和测试科目之间导航 。

  • 能够在IDE中运行测试，并在测试运行器UI中查看测试结果。测试结果显示在“ 运行”工具窗口的“ 测试运行器”选项卡上。

  • 能够在目录，特定测试类，测试用例或功能，单个测试方法或示例中运行所有测试或功能。

  • 代码检查。

  • 在docstrings中 ：

    • PyCharm识别Python代码，提供语法突出显示，代码完成和解析以及Python检查。

    • 能够识别Python代码的功能可以打开或关闭 。

    • 如果doctests作为单独的文件呈现，PyCharm允许打开带有*.rst扩展名的reStructuredText文件等文件。这些文件标有图标和功能语法突出显示。

有关常见测试过程的详细说明，请参阅执行测试一节。
