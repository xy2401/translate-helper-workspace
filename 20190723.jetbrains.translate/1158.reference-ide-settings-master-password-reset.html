主密码/重置主密码对话框

在“ 输入主密码”或“ 更改主密码”对话框中单击“ 重置”时 ，将打开此对话框 。

如果重置主密码，则当前存储在密码数据库中的所有密码都将丢失。

指定新的主密码。

您可以将密码设置为空。因此，IntelliJ IDEA在访问密码数据库时不会要求您输入密码。但是，在这种情况下，密码的安全性会降低。

要返回上一个对话框，请单击“ 输入”或“更改” 。

项目

描述

新密码

指定新的主密码。

确认密码

再次键入新的主密码。

使用OS用户凭据加密

仅适用于Windows：如果选中此复选框，主密码本身将以密码数据库的形式加密存储。（您的Windows登录凭据将用于加密。）

因此，IntelliJ IDEA在访问密码数据库时不会要求您输入主密码。

输入或更改

如果您决定不重置密码（例如您记住它），请单击此按钮返回上一个对话框。这可以是“ 输入主密码”或“ 更改主密码”对话框 。
