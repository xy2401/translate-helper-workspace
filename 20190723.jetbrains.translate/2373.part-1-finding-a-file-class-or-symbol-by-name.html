最近的文件

大多数情况下，您可以在正在处理的少数文件之间切换。其他文件不太必要。PyCharm具有使这个工作流程非常方便的功能。

假设您正在编辑一个函数，同时您正在为此函数编写测试 - 这意味着您来回切换。

这是导航到最近文件的帮助：按Ctrl + E.请注意，最新的文件首先列出并默认选中 - 您只需按Enter键即可 。

最近的文件弹出窗口中提供了速度搜索功能。只需开始输入您要查找的内容，您就会看到与搜索字符串匹配的结果：

请注意，您也可以跳转到工具窗口（例如，打开新的终端会话）。

如果您对最近编辑过的 （不仅仅是访问过的）文件感兴趣怎么办？在这种情况下，请使用键盘快捷键N / A.请注意，还可以使用速度搜索和导航到工具窗口。

摘要

这个简短的教程结束了。您已掌握以下功能：

  • 按名称查找任何文件，类或符号 ，然后导航到该文件，类或符号 。
