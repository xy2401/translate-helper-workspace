TextMate捆绑包

档案|设置|编辑|适用于Windows和Linux的TextMate Bundles
WebStorm |偏好|编辑| TextMate捆绑包用于macOS
按Ctrl + Alt + S

使用此页面导入TextMate / SublimeText 2包 ，并将WebStorm的颜色方案映射到TextMate的颜色方案。

先决条件

此页面显示在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，启用TextMate包支持插件时。

该插件与WebStorm捆绑在一起，默认情况下处于激活状态。如果禁用，则可以手动启用插件 。

项目

描述

TextMate捆绑包

此添加的包列表包含以下列：

  • 复选框 ：如果选中了添加的包名称左侧的复选框，则包将在相应类型的文件中提供突出显示。

  • TextMate Bundle的名称

  • Bundle location ：对于每个添加的TextMate包，都会显示其位置。

单击此按钮可使用“ 选择路径”对话框找到所需的捆绑包。添加后，捆绑包将显示在TextMate Bundles表中。

单击此按钮可删除选定的包。有问题的捆绑包将从列表中删除，但不会从计算机中物理删除。

IDE颜色方案 - TextMate颜色方案

使用此表建立WebStorm和TextMate的各种颜色方案之间的映射。
