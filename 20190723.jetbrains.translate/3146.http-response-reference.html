HTTP响应参考

response对象包含有关收到的HTTP响应（响应内容，标头，状态等）的信息，并提供对标头和contentType嵌套对象的访问。

属性

属性

描述

body （string | object）

响应内容，如果响应的content-type是application/json ，则为字符串或JSON对象。

headers （ ResponseHeaders ）

响应标头存储对象 。

status （int）

响应状态，例如200或404 。

contentType （ ContentType ）

contentType对象 ，它保存Content-Type响应头值的数据。

标题对象

ResponseHeaders对象用于检索有关响应头的值的数据。

的价值

headerName （string）

检索headerName响应头的第一个值，如果headerName响应头不存在，则返回null 。

valuesOf

headerName （string）

检索包含headerName响应头的所有值的数组。如果headerName响应头不存在，则返回一个空数组。

ContentType对象

ContentType数据对象包含Content-Type响应头中的信息。

属性

属性

描述

mimeType （字符串）

响应的MIME类型，例如text/plain ， text/xml ， application/json 。

charset （字符串）

响应字符集的字符串表示形式，例如utf-8 。
