创建测试对话框

导航|测试 - 创建新测试

CTRL + SHIFT + T

如果此类测试尚不存在，请使用此对话框为类或方法生成测试类。

项目

描述

目标目录

单击“浏览”按钮打开目录选择器对话框，然后选择目标目录，其中将生成所需的测试文件。

测试文件名

在此文本字段中，指定将生成测试类的文件的名称，或接受默认名称。

测试类名称

在此文本字段中，指定测试类的名称，或接受默认名称。

测试方法

此列表显示所讨论的类的所有方法（如果已从类声明中调用该对话框），或者显示唯一方法的名称（如果已在此方法中调用该对话框）。

选中方法名称旁边的复选框，以在生成的测试类中包含这些测试方法。
