项目

描述

自动滚动到新的局部变量

选择此选项可自动滚动到步进时出现在范围中的新变量。

节目

在本节中，选择希望调试器显示的元素：

  • 声明的类型

  • 合成领域

  • $ val字段作为局部变量

  • 完全限定名称

  • 对象ID

  • 静态字段

  • 静态最终字段

显示字符串的类型

如果要显示纯字符串的类型，请选择此选项。

显示基元的十六进制值

如果希望以十六进制格式显示数字变量，请选择此选项。

隐藏空数组元素

如果要省略null数组元素，请选择此选项。

自动填充Throwable对象的堆栈跟踪

选择此选项可在“ 变量”视图中自动显示异常对象的堆栈跟踪。

为Collection类启用备用视图

选择此选项可以更方便的格式显示集合和地图。
