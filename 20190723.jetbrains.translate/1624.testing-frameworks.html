以下内容仅在安装并启用Ruby插件时有效！

  • Test :: Unit用于单元测试，并与Ruby捆绑在一起。

  • Shoulda用于单元测试，并在附加了shoulda gem的Ruby项目中可用。这样做，Shoulda测试被添加到Test :: Unit框架中。

  • RSpec 。该测试工具支持BDD。 RSpec在附加rspec gem的Ruby项目中可用。对于Rails应用程序，还需要rspec-rails gem。

  • MiniTest可用于附加minitest-reporter gem。minitests被添加到Test :: Unit框架中。

  • 黄瓜 。此测试工具支持BDD，并允许使用以人类可读语言编写的功能和方案，使用英语或＃language语言中指定的任何其他# language: comment。在安装和激活cucumber宝石时， cucumber在项目中可用。

在开始使用您选择的测试框架之前，请确保在您的计算机上安装了所需的框架。有关安装详细信息，请参阅框架文档。

对于每个受支持的测试框架，IntelliJ IDEA根据特定的受支持测试框架提供：

  • 代码完成 ，了解具体的测试框架。

  • 运行/调试配置 。

  • 能够创建测试 。

  • 能够在测试和测试科目之间导航 。

  • 能够在IDE中运行测试，并在测试运行器UI中查看测试结果。测试结果显示在“ 运行”工具窗口的“ 测试运行器”选项卡上。

  • 能够在目录，特定测试类，测试用例或功能，单个测试方法或示例中运行所有测试或功能。

  • 代码检查。

有关常见测试过程的详细说明，请参阅执行测试一节。
