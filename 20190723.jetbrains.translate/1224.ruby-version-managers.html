IntelliJ IDEA 2019.1帮助

Ruby版本管理器

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

IntelliJ IDEA使您可以使用由以下版本管理器维护的Ruby解释器：

  • RVM

  • rbenv

  • chruby

  • asdf与asdf-ruby插件

请注意，所有受支持的版本管理器都不兼容，不应同时使用。

所有版本管理器都通过WSL支持Linux，MacOS和Windows 10。

如果您在本地计算机上安装了其中一个版本管理器，IntelliJ IDEA会自动检测它并允许您在可用的Ruby解释器（以及RVM和rbenv的gemsets）之间切换。

rbenv需要rbenv-gemsets插件才能使用gemsets。

IntelliJ IDEA还允许您使用使用这些版本管理维护的远程Ruby解释器 。在这种情况下，您必须在配置远程解释器时指定版本管理器可执行文件的路径。例如，下表列出了Linux的典型路径。

版本管理员

可执行路径

RVM

单用户模式： /home/jetbrains/.rvm/bin/rvm （用于jetbrains用户）

多用户模式： / usr / local / rvm / bin / rvm

混合模式： / usr / share / rvm / bin / rvm

rbenv

/home/jetbrains/.rbenv/bin/rbenv （对于jetbrains用户）

在/ usr / bin中/ rbenv

chruby

/usr/local/share/chruby/chruby.sh

ASDF

/home/jetbrains/.asdf/bin/asdf （对于jetbrains用户）
