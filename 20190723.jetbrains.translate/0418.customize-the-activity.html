Customize the Activity

IntelliJ IDEA lets you customize the activity you have selected in the Add an Activity dialog.

Activity Name

Use this field to specify the name of your activity.

Layout Name

Use this field to specify the layout name.

Title

Use this field to specify the name of the title.

The appearance of other fields, drop-down lists and checkboxes depend on the type of form factors and activities you have previously selected.
