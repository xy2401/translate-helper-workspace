配置检查严重性

检查严重性表示检测到的代码问题对项目的影响程度，并确定如何在编辑器中突出显示检测到的问题。默认情况下，每次检查都具有以下严重性级别之一：

  • 服务器问题

  • 错字

  • 没有突出显示，只有修复

  • 弱警告

  • 警告

  • 错误

您可以增加或减少每次检查的严重性级别。例如，您可以强制PyCharm显示一些警告，如错误或弱警告。以类似的方式，最初被认为是弱警告的内容可以显示为警告，错误或仅作为参考信息。

您还可以配置用于突出显示每个严重性级别的颜色和字体样式。此外，您可以创建自定义严重性级别并将其设置为特定检查。

如有必要，您可以为不同范围内的相同检查设置不同的严重性级别。

上述检查的所有修改都保存在检查设置中当前选择的检查配置文件中，并在使用此配置文件时应用。

更改检查的严重性

  1. 在检查设置中 ，选择所需的配置文件 。与配置文件关联的检查显示在树视图中。

  2. 选择所需的检查。如果禁用此检查，请选中它旁边的复选框。

  3. 从检查的上下文菜单或右侧的Severity选择器中选择所需的严重性：

    请注意，状态相对于默认值更改的检查及其所有分组节点都以蓝色突出显示。

  4. 应用更改。使用此配置文件时，修改后的检查现在将具有新的严重性级别。

更改不同范围的检查的严重性

  1. 选择所需的配置文件和检查。

  2. 单击所有范围中的下拉列表，然后选择要更改检查严重性的范围。

    PyCharm显示两个范围的严重性：对于选定的范围和其他所有范围 。

  3. 单击所选范围的严重性下拉列表，然后从下拉列表中选择相应的严重性级别：

更改特定严重性级别的突出显示样式

  1. 执行以下操作之一：

    • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|颜色和字体 - >常规 ，然后选择与所需严重性级别对应的样式。

    • 在检查设置中 ，选择所需的检查，然后从“ 严重性”选择器中选择“ 编辑严重性 ”。

      接下来，在打开的Severities Editor对话框中，选择所需的严重性级别，然后单击Editor |颜色和字体 。

    无论哪种方式，您都会在“ 配色方案”设置页面中看到与此严重性相关联的样式。

  2. 根据需要配置颜色和字体样式并应用更改。现在，当使用当前配置文件时，将使用修改后的样式在编辑器中突出显示检测到的具有相应严重性的问题。

定义自定义严重性级别

  1. 在检查设置中 ，选择所需的检查，然后从“ 严重性”选择器中选择“ 编辑严重性 ”。

  2. 在打开的Severities Editor对话框中，单击 ：

  3. 在“ 新建突出显示严重性”对话框中键入新严重性的名称。

    自定义严重性将添加到严重性列表中。

  4. 使用严重性列表右侧的控件指定新严重性的颜色和字体设置。

  5. 使用Up 和下来 用于更改新严重性优先级的按钮。

  6. 应用更改。现在，新的严重性级别可用于当前配置文件中的所有检查。您可以将其分配给特定检查，并在编辑器中使用指定样式突出显示相应的代码问题。

    如有必要，可以稍后通过在“ Severities Editor”对话框中选择自定义严重性级别并单击来删除自定义严重性级别 。

    您无法更改预定义严重性级别的优先级，也无法将其删除。
