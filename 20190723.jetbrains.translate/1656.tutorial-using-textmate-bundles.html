使用TextMate Bundles

以下内容仅在安装并启用TextMate Bundles Support Plugin时有效！

项目可以包含IntelliJ IDEA未知的文件类型。虽然IntelliJ IDEA内置了对许多编程和脚本语言的支持，但您可能希望为项目特定语言提供语法突出显示。例如，项目可以包含shell脚本或Perl;配置文件可以存在于项目中，用于基础结构自动化目的。如果您希望对这些情况进行语法突出显示，请使用强大的IntelliJ IDEA与文本编辑器TextMate的集成。

本教程旨在逐步引导您配置IntelliJ IDEA以使用TextMate Bundles，并使用已注册的扩展名编辑文件。

学习TextMate超出了本教程的范围。有关TextMate的信息，请参阅产品文档 。

先决条件

确保：

  • 您已经下载了要使用的软件包。例如，您可以在GitHub或Subversion上找到要安装的软件包。

  • 您正在使用IntelliJ IDEA 13.0（支持TextMate Bundles）或更高版本。

  • 在开始使用TextMate Bundles之前，请确保已安装并启用 TextMate包支持插件。

导入包

假设您希望IntelliJ IDEA突出显示Shell脚本文件的语法。为此，您已经下载了Shell脚本TextMate Bundle 。它现在位于您的硬盘上，您只需将此软件包导入IntelliJ IDEA即可。

在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器” ，然后单击“ TextMate Bundles” 。在TextMate Bundles区域中，单击并在硬盘上找到所需的捆绑包：

单击“ 确定”以应用更改。Shell脚本包出现在已识别的包列表中：

请注意TextMate Bundles页面下部的颜色方案映射部分。默认情况下，IntelliJ IDEA的默认颜色方案映射到Mac Classic。要为包使用不同的颜色方案，请单击映射表中的Mac Classic单元格，然后从可用的方案列表中选择所需的方案。

要使用自定义TextMate颜色方案，请导入带有方案的TextMate包。单击“ 应用”后，它将在TextMate颜色方案列表中显示：
