使用GWT Facet创建模块

此功能仅在Ultimate版本中受支持。

当您使用专用GWT方面创建模块时，IntelliJ IDEA会配置模块并自动添加所有必需的库。

创建具有GWT方面的模块

根据具体情况，您可以选择创建新项目或将模块添加到现有项目。

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目中：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java Enterprise 。在页面的右侧，指定要使用的JDK ，并选择要支持的Java EE版本。

  3. 在“ 其他库和框架”下 ，选中“ Google Web Toolkit”复选框。

    在GWT SDK字段中，指定GWT SDK位置。（要下载最新版本的GWT SDK，请使用Download GWT链接。）

    如果要创建示例GWT应用程序 ，请选中“ 创建示例应用程序”复选框，并在下面的字段中指定应用程序名称。

    单击下一步 。

  4. 指定名称和位置设置。

    单击完成 。
