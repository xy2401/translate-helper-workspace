介绍

IntelliJ IDEA允许您使用Osmorc插件创建基于OSGi的应用程序。IntelliJ IDEA还允许您导入Bnd / Bndtools项目 。您可以使用OSGi支持创建项目，向现有项目添加具有OSGI支持的模块，或者将OSGi支持添加到现有模块。

条件

在开始使用OSGi之前，请确保已启用Osmorc插件。该插件与IntelliJ IDEA捆绑在一起，默认情况下处于激活状态。如果未激活插件 ，请在“ 管理插件 ”中所述的“设置/首选项”对话框的“ 插件”页面上启用它。

使用OSGi支持创建项目或模块

使用OSGi支持创建项目或模块

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目中：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java 。在页面的右侧，指定Project SDK并从列表中选择OSGi 。在Libraries区域中指定OSGi库设置：

    • 使用库 - 选择此选项可指定现有OSGi库。

    • 下载 - 选择此选项以下载库。您可以单击“ 配置”以编辑下载选项。

    • 稍后设置库 - 选择此选项可在以后的项目中配置库。

    单击下一步 。

  3. 指定名称和位置设置。

    单击完成 。

向现有模块添加OSGi支持

  1. 在项目树中，右键单击要添加OSGi支持的模块。

  2. 从下拉列表中选择Add Framework Support 。

  3. 在打开的对话框中，从左侧窗格中选择OSGi 。

  4. 在页面右侧的“ 库”区域中，指定OSGi库设置。
    OSGI库设置如下：

    • 使用库 - 选择此选项可指定现有OSGi库。

    • 下载 - 选择此选项以下载库。您可以单击“ 配置”以编辑下载选项。

    • 稍后设置库 - 选择此选项可在以后的项目中配置库。
