EJB Module Editor - General

EJB Tool Window | context menu of a module with an EJB facet | Jump to Source

Use this tab to configure the beans contained in a module with EJB facet.

Item

Description

New

Click this button to add a new bean to a module with EJB facet. The bean type is selected from the submenu. Refer to the section Creating EJB for the detailed description of procedure.

Remove

Click this button to delete the selected bean from the module.

Edit

Click this button to invoke the EJB editor for the selected bean. Refer to the section Editing EJB for the detailed description of procedure.

Help

Click this button to show reference page.
