档案|设置|工具|适用于Windows和Linux的远程SSH外部工具
PyCharm |偏好|工具|用于macOS的远程SSH外部工具
按Ctrl + Alt + S

将需要SSH访问的远程应用程序定义为外部工具，以便能够从PyCharm运行它们。

您可以将上下文信息（如当前选定的文件或项目源路径）传递给外部工具，查看工具输出等。

此页面上定义的工具在“ 工具”菜单和各种上下文菜单中显示为命令。还可以为它们分配键盘快捷键（请参阅配置键盘和鼠标快捷方式部分）。

工具栏图标

图标

描述

添加外部工具的定义。（“ 创建工具”对话框将打开。）

如果选择了单个工具：删除工具定义。如果选择了工具组：删除所选组中所有工具的定义。

编辑所选工具的定义。（“ 编辑工具”对话框将打开。）

 

在组中向上或向下移动选定工具一行。（工具的顺序定义了相应菜单中项目的顺序。）

创建所选定义的副本，然后编辑该副本。（“ 复制工具”对话框将打开。）

复选框

使用复选框启用或禁用工具和工具组。“ 工具”和“上下文”菜单中未提供当前未选择的项目。
