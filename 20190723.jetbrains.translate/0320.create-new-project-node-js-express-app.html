新项目：Node.js Express App

档案|新的|项目

欢迎屏幕|创建新项目

当您在左侧窗格中选择静态Web ，然后从右侧窗格的列表中选择Node.js Express App时，“ 项目类别和选项”对话框的右侧窗格如下所示。

安装并启用Node.js插件时，仅在Ultimate版本中支持该功能。 该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

项目

描述

项目名

在此字段中，指定项目的名称，默认情况下，它是位置字段中指定的文件夹的名称。

项目位置

在此字段中，指定项目文件夹的路径，其中将存储与项目相关的文件。

节点解释器

在此字段中，指定要使用的Node.js解释器。这可以是Windows子系统Linux上的本地Node.js解释器或Node.js。

Npm可执行文件

在此字段中，指定Node.js包管理器文件npm.cmd 。在大多数情况下，IntelliJ IDEA会检测Node.js可执行文件并自动填写该字段。

版

从此列表中，选择将根据其生成存根的模板版本。点击刷新可用模板版本列表。

模板

从此下拉列表中，选择要使用的模板引擎 。可用选项包括：

  • 灰尘

  • EJS

  • 把手

  • Hogan.js

  • 帕格 （ 帕格（玉） ）

  • 枝条

  • VASH

  • 静态HTML

CSS

从此下拉列表中，选择要使用预处理器的CSS。可用选项包括：

  • 简单的CSS

  • 唱针

  • 减

  • 指南针 。

  • 萨斯 。
