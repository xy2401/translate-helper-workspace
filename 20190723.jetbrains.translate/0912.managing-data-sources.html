管理数据源

此功能仅在Ultimate版本中受支持。

数据源是应用程序的数据位置。

在IntelliJ IDEA中，您可以定义以下数据源：

  • 数据库数据源：包含数据的操作数据库（请参阅连接到数据库 ）。

  • 数据定义语言（DDL）文件：包含数据定义语言语句的SQL文件（请参阅添加DDL数据源 ）。

创建数据源

从数据库文件创建数据库数据源

  1. 如果数据库文件位于项目文件夹中，请打开“ 项目”工具窗口。否则，打开文件管理器。

  2. 打开数据库工具窗口（ 查看|工具Windows |数据库 ）。

  3. 选择要用作数据源的文件，然后将其拖到“ 数据库”工具窗口中。为每个文件创建单独的数据源。

  4. 如果您没有必要的数据库驱动程序文件，请下载它们。点击在数据库工具窗口的工具栏上。或者，从上下文菜单中选择“ 属性 ”。

  5. 在“ 数据源和驱动程序”窗口中，单击“ 下载”链接。

  6. 要确保与数据源的连接成功，请单击“ 测试连接” ，然后单击“ 确定” 。

从项目文件配置数据源

您可以使用包含数据库连接设置的XML文件（例如，Spring，Hibernate，JPA和Tomcat的context.xml ）来创建数据库数据源。

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ）。

  2. 点击在工具栏上，选择从源导入 。或者，右键单击工具栏下或现有数据源下的区域，选择“ 新建” ，然后单击“ 从源导入” 。

    在“ 数据源和驱动程序”窗口中，您可以看到绿色的候选数据源的名称。

  3. 如果缺少驱动程序文件，请下载它们。

    要指定驱动程序，请执行以下步骤之一：

    • 要下载必要的驱动程序，请单击“ 下载”链接。

    • 要指定计算机上已有的驱动程序文件，请单击Driver右侧的<DriverName>链接。

      在显示驱动程序设置的页面上，在JDBC drivers / Additional下 ，单击并选择必要的文件。

      返回包含数据源设置的页面。

  4. 要确保与数据源的连接成功，请单击“ 测试连接” 。

  5. 单击确定 。

创建DDL数据源

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ）。

  2. 单击“ 数据源属性”图标（ ）在工具栏上，选择DDL数据源 。

  3. 指定数据源名称。

  4. 在“ DDL文件”窗体中，单击“ 添加”图标（ ）并选择必要的SQL文件。

此外，您可以从“ 文件”工具窗口或文件管理器将DDL文件拖到“ 数据库”工具窗口。

自定义数据源

配置数据源设置

  1. 在“ 数据库”工具窗口（“ 查看”|“工具Windows”|“数据库” ）中，选择要更改的数据源。

  2. 单击“ 数据源属性”图标（ ）在工具栏上（或按Shift + Enter ）。

  3. 在“ 数据源和驱动程序”窗口中，编辑数据源属性。有关数据源属性的详细信息，请参阅数据源设置和DDL数据源设置 。

组数据源

如果您有许多不同的数据库，则可以在“ 数据库”工具窗口（“ 查看”|“工具Windows”|“数据库” ）中对其数据源进行分组。要对数据源进行分组，请选择所需的数据源，按F6 ，或从上下文菜单中选择“ 移至组 ”。

“ 移至组”菜单中的数据源组可使用以下选项：

  • 为数据源创建一个新组。

  • 将数据源移动到根列表。此选项从组中删除数据源。

  • 将数据源移动到现有组。

删除数据源

要删除数据源，请在“ 数据库”工具窗口（“ 查看”|“工具Windows”|“数据库” ）中，按“ 删除” 。或者，您可以从上下文菜单中选择“ 删除 ”，或单击“ 编辑”|删除 。

此外，要删除数据源，可以使用“ 数据源和驱动程序”窗口（“ 文件”|“数据源” ）。在“ 数据源和驱动程序”窗口中，选择数据源，然后按Delete 。或者，您可以单击在工具栏上，或从上下文菜单中选择“ 删除 ”。
