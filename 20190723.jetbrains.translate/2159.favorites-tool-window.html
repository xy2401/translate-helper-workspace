收藏工具窗口

此功能在PyCharm的教育版中不可用。

查看|工具窗口|最爱
ALT + 2

“ 收藏夹”工具窗口允许您管理以下列表：

  • 喜欢的项目项目 。工具窗口中的每个收藏夹列表都会显示星形图标 。

  • 书签 。书签列表标有 。

  • 断点 。断点列表标有 。

随着新书签或断点的添加，书签和断点列表会自动填写。

要将项目添加到收藏夹，请执行以下操作之一：

  • 在“ 项目工具”窗口或“ 查找”工具窗口中选择一个或多个项目。然后选择File |添加到收藏夹 。

  • 右键单击编辑器选项卡，然后选择“ 添加到收藏夹”或“ 全部 添加到收藏 夹” 。

工具栏按钮

项目

捷径

描述

Alt +插入

使用此按钮可创建喜爱项目的新列表。在“ 添加新收藏夹列表”对话框中，指定新列表的名称，然后单击“ 确定” 。

Ctrl + Enter

单击此按钮可重命名所选列表。

Alt + Delete键

使用此按钮删除选定的列表或列表项。

根据选择，将发生以下行为：

  • 如果是收藏夹列表或者选择其中一个嵌套收藏夹，单击此按钮可删除整个收藏夹列表或所选项目。因此，从“收藏夹”工具窗口中删除项目或列表，但在项目中保持不变。

  • 如果书签列表下有书签如果选中，单击此按钮可从列表和源代码中删除此书签。
    反之亦然，如果从源代码中删除书签，它也会从“收藏夹”工具窗口中删除。

  • 如果断点列表下有断点如果选中，单击此按钮可从列表和源代码中删除选定的断点。
    反之亦然，如果从源代码中删除断点，它也会从“收藏夹”工具窗口中删除。

上下文菜单命令

右键单击内容窗格中的项目时，将显示此项目的上下文菜单。此菜单提供对所选项目可用的所有功能的访问。

命令和功能类似于项目工具窗口中的命令 。

标题栏上下文菜单和按钮

您可以右键单击窗口标题栏并使用上下文菜单配置其查看模式 ，将窗口与其他工具窗口栏关联，或调整窗口大小和隐藏窗口。

使用此按钮可折叠当前视图中的所有展开节点。

项目

描述

显示会员

使用此命令可以打开或关闭“ 显示成员”选项。

如果启用此选项，则会显示类成员（字段，方法等）。

自动滚动到源

使用此命令可以打开或关闭“ 自动滚动到源”选项。

如果启用此选项，PyCharm会自动从“ 收藏夹”工具窗口中选择的文件（或类成员）导航到编辑器中相应的源文件（或其片段）。如果相应的文件当前未打开，则会自动打开。

从Source自动滚动

使用此命令可以打开或关闭“ 自动滚动源”选项。

如果启用此选项，PyCharm会自动从编辑器中选择的文件（或类成员）导航到“ 收藏夹”工具窗口中的相应文件。

显示工具栏

模式

检查工具Windows查看模式的详细说明。

从补充工具栏中删除

选择此选项可关闭窗口并将其从侧栏中删除。要返回它，请按Alt + 2 。

搬去

使用此选项可根据您的喜好移动窗口。

调整

使用此选项可调整窗口大小。您可以向不同方向拉伸或最大化它。

隐藏

使用此选项可关闭窗口。

使用拖放

您可以使用拖放操作：

  • 将项目添加到收藏夹：从“ 项目”工具窗口中拖动感兴趣的项目，然后将其拖放到“收藏夹”工具窗口中的所需收藏夹列表中。

  • 从资源管理器/ Finder中拖动外部项目并将其放到“收藏夹”工具窗口中的所需收藏夹列表中。

  • 在“ 收藏夹”工具窗口中将项目从一个“收藏夹”列表移动到另一个
