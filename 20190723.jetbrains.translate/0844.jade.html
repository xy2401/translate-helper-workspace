IntelliJ IDEA 2019.1帮助

帕格（玉）模板引擎

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA与Pug（Jade）模板引擎集成。

在你开始之前

  1. 下载并安装Node.js.

  2. 如管理插件中所述，在插件页面上安装并启用Pug（以前的Jade）存储库插件。

此外，如果您需要文件观察程序，请确保已安装并启用了文件监视器插件。该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

对UI的更改

Pug（ex-Jade）插件向IntelliJ IDEA UI引入了以下更改：

  • Jade文件项目将添加到“ 新建”菜单中。

  • Pug文件标有 ; Jade文件标有 。

  • Pug（Jade）特定和HTML上下文中提供了编码帮助：

    • 代码格式

    • 语法突出显示

    • 代码完成
    • 配色方案

在Node.js应用程序中使用Pug（Jade）模板

在运行时，Pug（Jade）文件填充转换为HTML页面。

  1. 从头开始创建项目，或者在现有源周围创建项目，或者基于NodeExpress模板创建项目。请参阅项目了解详情。

  2. 创建一个Pug（Jade）文件。跟着这些步骤：

    1. 在“ 项目工具”窗口中 ，选择要在其中创建新文件的目录。为此，例如，选择“ 文件”|新的 。

    2. 从上下文菜单中，选择Jade文件并在打开的对话框中指定文件名。

  3. 创建一个文件监视器，将扩展名为.jade或.pug的文件转换为.html页面：

    1. 单击编辑器右上角的“ 添加观察器”链接。

    2. 在New Watcher Dialog中 ，接受默认的预定义设置。

      请注意，如果可执行文件位于PATH中，则不应显式指定它。根据文件扩展名（ .jade或.pug ），将调用相应的可执行文件。

  4. 在编辑.pug / .jade文件时，IntelliJ IDEA调用文件监视器，该文件监视器创建一个.html文件，其中包含已处理的.pug / .jade文件的名称，并将生成的html代码存储在其中。

有关详细信息，请参阅使用文件监视器 。

配置语法突出显示

您可以根据自己的喜好和习惯配置Pug / Jade感知语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案|帕格/玉 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
