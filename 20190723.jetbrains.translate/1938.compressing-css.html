缩小CSS

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

在开始之前，请确保您的计算机上有Node.js.

YUI Compressor是一种用于缩小CSS代码的工具。缩小或压缩意味着删除所有不必要的字符，例如空格，换行，注释而不更改源代码的功能。在开发和调试期间，这些字符使代码更易于阅读。在生产阶段，他们只增加要传输的代码的大小。

安装和配置YUI Compressor

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm安装yuicompressor

压缩CSS代码

要动态压缩代码，您需要将压缩器配置为PyCharm 文件观察器 。

创建文件监视器

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 工具”下的“ 文件监视器 ”。打开的“ 文件监视器”页面显示已配置的文件监视器列表。

  2. 点击或按Alt + Insert并从列表中选择YUI Compressor CSS预定义模板。

  3. 在“ 程序”文本框中，指定yuicompressor-<version>.jar文件的路径。如果您通过节点包管理器安装该工具，PyCharm将自行定位所需的文件并自动填写该字段。否则，请手动键入路径或单击并在打开的对话框中选择文件位置。

  4. 按照使用文件监视器中的说明继续操作。

启用缩小文件观察程序时，只要更改或保存文件观察程序范围中的CSS文件，就会自动开始缩小。

PyCharm使用生成的输出创建一个单独的文件。该文件具有源CSS文件的名称和扩展名min.css 。生成的文件的位置在“ 新建观察器”对话框的“ 输出路径刷新”字段中定义。但是，在项目树中 ，默认情况下它显示在源CSS文件下，该文件现在显示为节点。要更改默认表示，请在“ 项目”工具窗口中配置文件嵌套 。
