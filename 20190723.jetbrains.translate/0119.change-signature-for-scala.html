更改Scala的签名

重构|更改签名
按Ctrl + F6

使用此对话框执行“ 更改签名”重构 。

使用可用控件更改方法签名。指定应如何处理方法调用。

单击“ 重构”立即执行重构。单击“ 预览”以在实际重构之前查看“查找”工具窗口中的潜在更改。

项目

描述

能见度

从列表中选择方法可见范围（访问级别修饰符）。

返回类型

使用此字段可以修改方法返回类型。

代码完成 （ Ctrl + Space ）在此字段中可用，也可用于指定类型的其他字段。

名称

使用此字段可以修改方法名称。

修改方法调用

修改现有方法调用，以便调用具有新签名的方法。

添加到定义

如果选择此选项，则会将新参数添加到方法定义中。

指定结果类型

使用此选项可配置类型注释设置。

签名预览

在此区域中，显示当前方法签名。（此区域中的信息与您对方法签名所做的更改同步。）

在指定参数的区域中，您可以添加新参数，编辑，重新排序或删除现有参数。重构将相应地分配您的更改。

要开始编辑参数，请单击它。或者，使用向上和向下箭头键导航到感兴趣的参数，然后按Enter键开始修改它。

项目

工具提示和快捷方式

描述

加
Alt +插入

使用此图标或快捷方式开始添加新参数。

在相应字段中指定参数信息。（默认参数值是在方法调用中传递给方法的值（或表达式）。）

去掉
Alt + Delete键

使用此图标或快捷方式删除所选参数。

向上
Alt +向上

使用此图标或快捷方式将所选参数在参数列表中向上移动一行。

下
Alt +向下

使用此图标或快捷方式将所选参数在参数列表中向下移动一行。

添加参数子句

ALT + =

使用此图标或快捷方式添加为附加参数组。

删除参数子句

Alt + -

使用此图标或快捷方式删除现有参数子句。
