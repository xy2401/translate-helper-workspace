从命令行合并文件

您可以使用PyCharm作为合并文件的命令行工具。

大多数情况下，您需要合并同一文件的三个版本：本地版本，存储库或上游版本，以及基本版本，这是两个不同版本的来源。

  • 在命令行中，使用以下语法运行merge命令：

    <PyCharm>合并path1 path2 path3输出

也可以只向merge工具传递三个参数。在这种情况下， output的内容将作为基本修订。

  • 在命令行中，使用以下语法运行merge命令：

    < PyCharm > 合并 path1 path2 输出

所选文件将在“ 合并”对话框中打开，允许您单独接受或放弃每个更改。

命令选项如下：

选项

描述

<PyCharm>

特定于平台的PyCharm启动器。

在macOS和Linux上，建议使用在PyCharm安装期间创建的命令行启动程序，或者通过选择Tools |在PyCharm主菜单上创建命令行启动器 。请参阅除了在PyCharm中工作之外，您可以从命令行执行以下操作：有关详细信息。

在Windows上，建议将PyCharm可执行文件的Path添加到Path环境变量中。

PATH1

文件本地副本的路径。

路径2

文件存储库版本的路径。

path3时，

（可选） path1和path2的基本修订的路径。

产量

保存合并的文件路径导致。

例子

句法
pycharm.exe合并path1 path2 path3输出
例
pycharm.exe合并C： \ M yProject \ R eadme.md.bak C： \ M yProject \ R eadme.md C： \ A rchive \ R eadme.md C： \ N ewProject \ R eadme.md
句法
pycharm合并path1 path2 path3输出
例
pycharm merge~ / MyProject / Readme.md.bak~ / MyProject / Readme.md~ / Archive / Readme.md~ / NewProject / Readme.md
