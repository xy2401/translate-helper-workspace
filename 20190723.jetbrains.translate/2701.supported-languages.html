支持的语言

现代应用程序的开发涉及使用多种语言，这就是PyCharm是多语言编程的IDE的原因。

PyCharm支持以下语言：

  • 蟒蛇

    • 应该在您的计算机上下载并安装 Python。

    • PyCharm支持以下版本的Python：

      • Python 2： 2.7版

      • Python 3：从版本3.5到版本3.7

    • PyCharm支持wxPython ， PyQt4和PyGTK 。

  • JavaScript 。请参阅JavaScript部分。

  • CoffeeScript 。请参阅CoffeeScript部分。

  • 标记语言和样式表：

    • XML

    • HTML / XHTML

    • YAML
    • CSS

    • Sass / Less

    • 唱针

PyCharm支持以下框架，工具和模板语言：

  • Django高达1.3版。

  • Google App Engine 。

  • 烧瓶 。

  • 建设 。

    PyCharm检测现有项目中的buildout支持，并可以打开它们。但是，您必须提前提供构建环境。

  • web2py中。有关详细信息，请参阅Web2Py部分。

  • 金字塔。有关详细信息，请参阅金字塔部分。

  • 玛雅此支持包括Maya中的远程调试和启动脚本。

  • 模板语言：

    • 真子
    • Jinja2的

    • 变色龙

    请参阅模板部分。

  • SQLAlchemy 。这种支持包括：

    • 代码洞察力

    • 可以在图中查看数据库结构。请参阅使用图表部分。

    • 代码完成并解决。

PyCharm中的编码协助包括：

  • 语法和错误突出显示以维护绿色代码

  • 用于创建存根类，脚本等的文件模板

  • 实时模板将缩写扩展为复杂的代码构造

  • 代码完成以根据上下文完成类，方法，字段，参数和其他实体的名称

  • 生成代码以生成特定于语言的代码构造

  • 意图操作和快速修复以优化您的代码

  • 代码重构，用于在整个代码库中重命名，移动，复制和删除实体

  • 导入辅助以导入缺失的库

  • 用于在字符串文字，标签或属性中使用受支持语言的语言注入

  • 用于查看API文档，方法中的参数名称和提示，静态表达式数据等的代码参考信息

除了编码帮助，PyCharm还支持用于构建，运行，测试和部署应用程序的特定于语言的工具。
