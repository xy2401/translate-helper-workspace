家

选中此复选框后，在按Home时 ，插入符号位于当前行的第一个非空格字符处。按Home后，将插入符号从Smart Home位置移动到第一列并返回。

结束（在空白行）

选中此复选框时，在空行中按End键时，插入符号将与缩进一起定位，PyCharm在当前代码点中假定合理（缩进基于当前的代码样式设置 ）。

插入对括号

选中此复选框可让PyCharm分别为每个打字的圆形或方括号自动添加一个闭合圆形或方括号。

插入对报价

选中此复选框可让PyCharm分别为每个打开的单引号或双引号自动添加结束单引号或双引号。请参见页面创建文档注释

在键入打开三重引号后按空格键生成文档字符串仅在编辑器设置的页面智能键中清除复选框插入对引用时才有效。

键入“}”时重新格式化块

如果选中此复选框，则在键入结束花括号时，如果此代码块的格式与所选代码样式不匹配，则会自动重新格式化随附的代码块。

使用'CamelHumps'字样

选中此复选框可让PyCharm在CamelHump名称中识别单独的单词。名称中的单词应以大写字母或下划线开头。此选项会影响某些编辑器操作，例如：

  • 插入符号移动 （ Ctrl +向右 / Ctrl +向左 ）

  • 使用选择移动插入符号 （ Ctrl + Shift +向右 / Ctrl + Shift +向左 ）

  • 在Caret中选择Word （ Ctrl + W ）

  • 删除到Word开始/结束 （分别为Ctrl + Backspace和Ctrl + Delete ）

  • 双击 （如果在编辑|常规设置页面上启用了使用双击选择时的荣誉“CamelHumps”字设置 ）。

键入引号或括号的环绕选择

如果选中此复选框，则键入引号，双引号或括号的所选文本将被这些字符包围。如果未选中此复选框，则键入的引号，双引号或大括号将替换选择。

使用箭头键在双Ctrl / Add上添加多个插入符号

如果选中此复选框，则：

  • 按Ctrl键 （对于Windows或* NIX）或⌥ （对于macOS）两次加上/向下箭头键可以创建多个插入符号。

  • 按Ctrl （对于Windows或* NIX）或⌥ （对于macOS）两次加上左/右箭头键或Home / End导致创建选择。

有关更多信息，请参阅“ Multicursor”部分。

跳转到闭合括号/引用Tab

如果选中此复选框，则在键入括号/引号内时按Tab键会将插入符号移到结束括号/引号之外。如果未选中此复选框，则按Tab键将插入Tab字符。

请注意，在初始打字这仅适用于：后续的编辑过程中，按Tab键括号内/引号将插入Tab符。

输入

使用此区域可以按Enter键来定义要调用的操作。
  • 智能缩进 - 选中此复选框可让PyCharm添加一个新行并将插入符号放入其中，并使用PyCharm在当前代码点假定合理的缩进（缩进基于当前的代码样式设置）。

    如果清除该复选框，则在空白行中按Enter键时，PyCharm会添加一个新行并将插入符号放在当前非空格字符列中。

  • 插入对'}' - 选中此复选框可让PyCharm在空行中按Enter键时自动将右括号}定位在正确的列上。在这种情况下，PyCharm向后寻找最近的未闭合的开口支撑{并将关闭的一个放在相应的压痕水平。

  • 插入文档注释存根 - 此复选框定义在打开文档标记后按Enter键的行为。
    • 如果选中此复选框，PyCharm将生成文档注释存根。

      对于函数注释，此存根包含必需的标记（签名中声明的每个参数的@param标记和@return ）。有关详细信息，请参阅创建文档注释 ， 创建JSDoc注释 。

    • 如果未选中此复选框，则仅生成结束标记。

    请注意，此复选框指的是JavaScript，以及具有特殊文档注释开头的其他语言。

    这个复选框不是指Python。

退格

使用此下拉列表可以通过按Backspace键来定义要调用的操作。可用选项包括：

  • 禁用 - 按退格键一次将插入符号返回一个位置。

  • 到最近的缩进位置
  • 适当的缩进

重新格式化粘贴

使用此下拉列表指定粘贴代码块的放置方式。可用选项包括：

  • 无 - 粘贴的代码作为纯文本插入到插入符号位置，无需重新格式化或缩进。

  • 缩进块 - 根据当前的代码样式设置 ，粘贴的代码块位于正确的缩进级别，但其内部结构不会更改。

  • 缩进每一行 - 根据当前的代码样式设置 ，粘贴的代码块的每一行都位于适当的缩进级别。

  • 重新格式化块 - 根据当前的代码样式设置重新格式化粘贴的代码块。

此功能适用于包含尾随换行符的行。

XML / HTML

在此区域中，定义编辑XML或HTML代码时要自动调用的操作。

  • 在标记完成时插入结束标记 ：选中此复选框可让PyCharm在输入相应的开始标记时自动插入结束XML或HTML标记。

  • 在标记完成时插入必需的属性 ：选中此复选框可让PyCharm显示包含类型标记的所有必需属性的模板。

  • 在标记完成时插入必需的子标记 ：选中此复选框可让PyCharm显示包含所有必填子标记的模板。

  • 标签完成时启动属性 ：选中此复选框可让PyCharm显示带有类型标签的第一个必需属性的模板。

  • 在键入'='时为属性值添加引号 ：选中此复选框可让PyCharm自动为当前键入的属性值添加引号。

  • 键入“</”时自动关闭标记 ：选中此复选框可在输入后自动添加结束标记</。清除此复选框以关闭此类自动完成功能。

  • 同时<tag> </ tag>编辑 ：

    • 选中此复选框并编辑开始标记后，相应的结束标记会自动更改。

    • 如果清除此复选框，则编辑开始标记不会影响保持不变的结束标记。结果，打开和关闭标签不匹配，并且整个构造被加下划线作为错误。

    此复选框控制PyCharm在以下上下文中的行为：

    • HTML文件

    • JavaScript代码中的HTML注入

    • 带有模板Handlebars / Moustache模板的HTML

    • Handlebars / Moustache模板文件，扩展名为.hbs

    • XML，XHTML文件

    • DTD文件

    • JSX文件

JSON

此区域中的设置控制在JSON上下文中自动添加逗号，引号和空格，符合JSON语法。默认情况下启用所有设置。

智能缩进粘贴线

如果选中此复选框，则粘贴的线相对于当前插入符号位置缩进。否则，粘贴的线相对于第一列缩进。

在语句中按Enter键时插入反斜杠

如果选中此复选框，则会在按Enter键时自动插入延续字符，同时保留正确的语法。
如果未选中此复选框，则该行将被破坏，并且将通过即时代码检查报告语法错误。

定义方法时插入'self'

如果选中此复选框，则在键入左括号和结束括号和冒号后会自动插入self参数。

如果清除此复选框，则仅自动生成关闭括号。

AngularJS

使用此区域来定义AngularJS的行为：

SQL

 在Enter上插入字符串连接。 如果您使用的DBMS支持多行字符串文字，则可能需要关闭此选项：

比如，PostgreSQL text值notes有以下片段：

SET notes = '最轻的元素'

并且光标位于单词element前面。

如果该选项已打开，并且按Enter键 ，则片段将更改为：

SET notes = 'Lightest' || '元件'

否则，片段将更改为：

SET notes = '最轻的 元素'
在代码完成时限定对象。 选定的选项定义在使用代码完成建议框时如何在编辑器中插入对象的名称。

  • 总是。 始终使用限定对象名称，即<schema_name>.<object_name> 。

  • 在碰撞。 仅当短名称不明确时才使用限定对象名称，例如，当多个模式中存在具有相同名称的对象时。

  • 决不。 始终使用非限定对象名称。
