SciView

打开“数据视图”工具窗口

在以下情况下会出现此工具窗口：

  • 一个在Debug工具窗口的Variables选项卡中调用View as Array / View as DataFrame 。

  • 一个人在Python控制台中执行Python代码，点击在控制台工具栏中，然后从阵列或DataFrame的上下文菜单中调用命令View as Array / View as DataFrame，或单击链接View as Array / View as DataFrame 。

支持的格式

“数据视图”工具窗口支持以下格式：

  • NumPy数组（ 视图为数组 ）

  • pandas dataframes （ 以DataFrame视图 ）。
