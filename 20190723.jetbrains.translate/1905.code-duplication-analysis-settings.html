  • Anonymize Variables：选中此复选框时，使用不同变量名称的两个相同函数被视为重复，例如：

  • 匿名函数

  • 匿名文字

  • 不要显示比以下更简单的重复项：设置结果窗口中显示的重复语言结构的大小。默认情况下，不包括小于10个单位的构造（并且不能更改此限制）。

  • 匿名化不常见的子表达式比：在可被认为相似的语言结构中设置子元素的值，以在结果窗口中将结构显示为重复。数字越大，PyCharm就越接近相似的结构。

    基于使用加法算法计算的元素大小，将值设置为任意权重。元素越大，计算值越高。
