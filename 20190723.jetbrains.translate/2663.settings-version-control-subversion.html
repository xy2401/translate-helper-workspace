按Ctrl + Alt + S

使用此页面指定要应用于Subversion控件下的项目目录的设置

项目

描述

启用交互模式

如果希望PyCharm在交互模式下直接从终端执行Subversion命令时模拟行为，则选择此选项（将弹出可输入凭据的对话框）。这是处理svn + ssh存储库的密码/密码提示以及信任https存储库的无效服务器证书所必需的。

使用自定义配置目录

如果您不想在系统默认位置存储Subversion配置文件，请选择此选项，并指定自定义目录的路径。

介绍

使用此设置页面配置数据显示设置。

项目

描述

准备合并时，检查目标子树中的svn：mergeinfo

如果希望PyCharm在合并之前检查目标分支的合并跟踪信息以防止重复，请选择此选项。

要在注释中回顾的最大修订数

选择此选项可限制在计算注释时要回顾的修订数，并指定修订数。

在历史记录和注释中显示合并源

如果要在注释和文件历史记录中显示合并源，请选择此选项。

忽略注释中的空格差异

如果要在注释时忽略空格，请选择此选项，从而获得更有意义的注释并抛弃无意义的注释。

网络

使用此设置页面配置连接设置。

项目

描述

使用PyCharm常规代理设置作为Subversion的默认设置

如果希望Subversion使用默认的PyCharm代理设置，请选择此选项。

HTTP超时

指定等待建立HTTP连接的秒数。

SSH连接超时

指定等待建立SSH连接的秒数。

SSH读取超时

指定等待响应的秒数。

SSL协议

在此区域中，选择要使用的SSL协议。可用选项包括：

  • 所有

  • 在SSLv3

  • 使用TLSv1

编辑网络选项

单击此按钮可在与网络层对话框相关的“ 编辑Subversion”选项中更改Subversion运行时配置文件。

SSH

使用此设置页面配置用于通过隧道SSH协议连接SVN服务器的设置。

项目

描述

SSH可执行文件

指定SSH客户端的路径。输入可执行文件的名称，或单击“ 浏览”按钮并在打开的对话框中选择路径。如果未指定，则默认使用'ssh' 。

仅当选择了密码或私钥选项时，此字段才可用。

用户名

指定SSH连接的用户名。如果在存储库URL中显式指定了用户名，则将使用此值，并忽略此设置。

仅当选择了密码或私钥选项时，此字段才可用。

港口

如果您的服务器正在侦听非标准端口（对于svn + ssh：//为22），请修改默认值。

仅当选择了密码或私钥选项时，此字段才可用。

密码

如果要使用密码进行SSH身份验证，请选择此选项。

私钥

如果要使用私钥进行SSH身份验证，请选择此选项。

路径

指定私钥的路径。手动输入路径，或单击“ 浏览”按钮并在打开的对话框中选择路径。

Subversion配置

如果要使用Subversion配置中存储的默认设置进行SSH连接，请选择此选项。

SSH隧道

此字段显示存储在Subversion配置中的SSH隧道设置。您可以修改该值，然后单击“ 更新”按钮将此值写入Subversion配置。

更新

单击此按钮以检查Subversion配置并在必要时更新值，或将您输入的值写入Subversion配置。

SVN_SSH

此字段显示可在隧道配置中使用的环境变量（默认情况下为SVN_SSH ），并存储在Subversion配置中。
