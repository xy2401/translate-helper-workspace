IntelliJ IDEA 2019.1帮助

卸载IntelliJ IDEA

删除IntelliJ IDEA的正确方法取决于您用于安装它的方法。

本指南介绍了在Windows macOS Linux上卸载IntelliJ IDEA的方法。使用顶部的平台切换器选择另一个操作系统。

工具箱应用程序

如果您使用Toolbox App安装了IntelliJ IDEA：

  • 打开Toolbox App，单击必要实例的螺母图标，然后选择卸载 。

独立实例

要删除独立的IntelliJ IDEA实例 ，请使用标准过程卸载操作系统上的应用程序，并删除IDE配置和系统目录 。

  1. 在系统设置对话框中打开应用和功能部分。

  2. 选择IntelliJ IDEA应用程序，然后单击卸载 。

  3. 删除以下目录：

    句法
    ％HOMEPATH％\。<产品> <版本> \配置
    例
    C：\用户\约翰\。IntelliJIdea2019.1 \ CONFIG
    句法
    ％HOMEPATH％\。<产品> <版本> \系统
    例
    C：\用户\约翰\。IntelliJIdea2019.1 \系统

  1. 删除Applications目录中的应用程序 。

  2. 删除以下目录：

    句法
    〜/库/首/ <产品> <版本>
    例
    〜/库/首选项/ IntelliJIdea2019.1
    句法
    〜/库/缓存/ <产品> <版本>
    例
    〜/图书馆/缓存/ IntelliJIdea2019.1

  1. 删除安装目录。

  2. 删除以下目录：

    句法
    〜/ <产品> <版本> /配置
    例
    〜/。IntelliJIdea2019.1 /配置
    句法
    〜/ <产品> <版本> /系统
    例
    〜/。IntelliJIdea2019.1 /系统

以静默方式卸载

如果以静默方式安装IntelliJ IDEA ，则可以使用/S开关作为管理员运行卸载程序。卸载程序位于bin下的安装目录中。

以管理员身份运行cmd （Windows命令提示符），切换到IntelliJ IDEA安装目录，然后运行以下命令：

bin \ uninstall 。 exe / S.

Snap包

如果您将IntelliJ IDEA安装为快照包 ，请使用以下命令将其删除：

$ sudo snap删除intellij-idea-ultimate
$ sudo snap删除intellij-idea-community
