WebStorm 2019.1帮助

查看内联文档

对于项目符号或项目依赖项中的符号，WebStorm从相应的JSDoc注释生成文档。要创建注释，请将光标放在方法/函数或字段的声明之前进行记录，键入空心块注释/** ，然后按Enter键 。有关详细信息，请参阅创建JSDoc注释 。

快速文档查找使您可以查看项目及其依赖项中的符号以及标准JavaScript对象和方法的引用。有关详细信息，请参阅JavaScript Documentation Look-Up和TypeScript文档查找 。

文档显示在“ 文档”弹出窗口中。

当您显式调用代码完成时，可以自动显示在建议列表中选择的条目的快速文档。快速文档查找的行为在“设置/首选项”对话框的“ 代码完成”页面上的“ 显示文档”弹出窗口中进行配置。

查看插入符号的文档

  • 选择查看|主菜单上的快速文档查找 。

  • 按Ctrl + Q.

  • 如果选中了在编辑器设置中鼠标移动时显示快速文档的复选框，只需将鼠标指针移动到所需的符号上即可。

按顺序按Ctrl + Q可切换“快速文档”弹出窗口和“ 文档工具”窗口的焦点。

更改快速文档的字体大小

  • 点击在快速文档窗口的右上角，然后移动滑块。

  • 按住Ctrl键的同时旋转鼠标滚轮。

    请注意，要使此功能起作用，您需要在编辑器设置的“ 常规”页面中启用它。

    有关详细信息，请参阅有用的编辑器配置部分
