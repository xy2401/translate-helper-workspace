先决条件

  • Mercurial安装在您的计算机上。

  • 在“设置/首选项”对话框的“ Mercurial”页面上正确指定了Mercurial可执行文件hg.exe的位置。

    如果您遵循标准安装过程，则默认位置为/opt/local/bin或/usr/local/bin适用于Linux和macOS）和/Program Files/TortoiseHG适用于Windows）。

    建议您将Mercurial可执行文件的路径添加到PATH 变量 。在这种情况下，您只能指定可执行文件名，不需要可执行文件位置的完整路径。

  • 为当前项目根目录或目录启用 Mercurial 集成 。

如果要使用远程存储库，请首先创建Mercurial主机帐户。您可以通过一对ssh keys访问远程存储库，也可以应用Mercurial集成支持的用户名/密码和键盘交互式身份验证方法。

Mercurial支持

  • 启用Mercurial与IntelliJ IDEA集成后， Mercurial项目将显示在VCS菜单上，以及编辑器和项目工具窗口的上下文菜单中。

  • Mercurial控件下文件夹中的文件将根据其状态突出显示。有关文件状态突出显示约定，请参阅文件状态要点 。

  • 修改结果显示在“ 版本控制”工具窗口中 。

  • 使用Mercurial集成时，打开“ 版本控制”工具窗口会很有帮助。“ 控制台”选项卡显示以下数据：

    • 根据您通过IntelliJ IDEA用户界面指定的设置生成所有命令。

    • 有关执行生成的Mercurial命令的结果的信息消息。

    • 错误消息。
