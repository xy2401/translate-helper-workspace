管理项目收藏夹

您可以在收藏项目（收藏夹）列表中排列最常用的文件，文件夹， 书签和断点 。WebStorm有一个专用的收藏夹工具窗口，用于管理您的收藏夹。

最初，有一个（空）收藏夹列表，其名称与项目相同。

您可以根据需要创建更多收藏夹列表并管理其内容。

使用“项目”工具窗口将项目添加到收藏夹

  1. 在项目中 ，选择要添加到收藏夹的项目。

  2. 执行以下操作之一：

    • 按Shift + Alt + F.

    • 选择文件|从主菜单添加到收藏夹 。

    • 从选择的上下文菜单中，选择“ 添加到收藏夹” 。

  3. 要将所选项目添加到现有收藏夹列表，请选择列表名称。

    要创建新的收藏夹列表并将所选项目添加到其中，请选择“ 添加到新收藏夹列表” 。在“ 添加新收藏夹列表”对话框中，指定新列表的名称，然后单击“ 确定” 。

使用拖放操作将项目添加到收藏夹：从“ 项目”工具窗口拖动感兴趣的项目 ，或从“ 资源管理器”或“ Finder”拖动外部文件，然后将其拖放到“收藏夹”工具窗口中的所需收藏夹列表中。

在编辑器中，您可以将一个文件（ Shift + Alt + F ）或所有当前打开的文件添加到收藏夹。

使用编辑器将文件添加到收藏夹

  1. 右键单击感兴趣的编辑器选项卡，然后选择以下选项之一：

    • 如果要将当前文件添加到收藏夹，请选择“ 添加到收藏夹” 。（对于当前文件，还可以使用“ 文件”|“添加到收藏夹” 。）

    • 如果要将编辑器中打开的所有文件添加到收藏夹，请选择“ 全部添加到收藏夹” 。

  2. 要将项目添加到现有收藏夹列表，请选择列表的名称。

    要创建新的收藏夹列表并将项目添加到其中，请选择“ 添加到新收藏夹列表” 。在“ 添加新收藏夹列表”对话框中，指定新列表的名称，然后单击“ 确定” 。

在将项目添加到收藏夹或将收藏夹列表项目移动 到其他列表时，您可以选择创建新的收藏夹列表。

您还可以单独创建一个新的（空）收藏夹列表，作为单独的任务。

创建新的收藏夹列表

  1. 打开“ 收藏夹工具”窗口 （ Alt + 2 ）。

  2. 执行以下操作之一：

    • 点击在工具栏上。

    • 按Alt + Insert 。

  3. 在“ 添加新收藏夹列表”对话框中，指定新列表的名称，然后单击“ 确定” 。

重命名收藏夹列表

  1. 打开“ 收藏夹工具”窗口 （ Alt + 2 ）。

  2. 执行以下操作之一：

    • 右键单击要更改其名称的列表，然后选择“ 重命名收藏夹列表” 。

    • 在“ 收藏夹”工具窗口的工具栏中，单击 。

  3. 在“ 收藏夹列表的新名称”对话框中，根据需要更改列表的名称，然后单击“ 确定” 。

将项目移动到其他列表

  1. 打开“ 收藏夹工具”窗口 （ Alt + 2 ）。

  2. 右键单击要移动的列表项，然后选择“ 发送到收藏夹” 。

  3. 要将所选项目移动到现有收藏夹列表，请选择目标列表的名称。

    要创建新的收藏夹列表并在其中移动项目，请选择“ 发送到新收藏夹列表” 。在“ 添加新收藏夹列表”对话框中，指定新列表的名称，然后单击“ 确定” 。

要将项目从一个收藏夹列表移动到另一个收藏夹列表，请使用拖放操作。

要从收藏夹中删除项目，您可以删除相应的收藏夹列表项目和/或整个收藏夹列表。

从收藏夹中删除项目

  1. 打开“ 收藏夹工具”窗口 （ Alt + 2 ）。

  2. 选择要从收藏夹中删除的项目。请注意，您可以同时选择单独的列表项和整个列表。

  3. 执行以下操作之一：

    • 点击在工具栏上。

    • 在上下文菜单中选择“ 从收藏夹删除” 。（如果当前选择了单个收藏夹列表，请注意还有以下选项： 删除收藏夹列表<list_name>和删除除<list_name>之外的所有收藏夹列表 。）

    • 按删除 。
