Struts 2 Facet

档案|项目结构|模块| <module> | Struts 2
档案|项目结构|方面| Struts 2（<module>）

该页面仅适用于支持Struts 2的模块。

使用此页面配置连接到特定模块的单个Struts 2构面的设置。

可以使用以下选项卡：

  • 文件集选项卡

  • 功能选项卡

文件集选项卡

项目

描述

或Alt + Insert

使用此图标或快捷方式可以创建新文件集或将文件添加到所选文件集。（“ 编辑文件集”对话框将打开。）

或输入

使用此图标或快捷方式编辑选定的文件集。（“ 编辑文件集”对话框将打开。）

或Alt +删除

使用此图标或快捷方式删除选定的文件或文件集。

功能选项卡

项目

描述

禁用基于.properties的I18N支持

禁用对消息资源键的内部化支持，例如，对于validation.xml <message key="..."> ，Struts UI taglib中的key属性等。
