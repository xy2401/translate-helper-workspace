围绕WebStorm用户界面进行导览

在这里，您可以了解WebStorm用户界面的组织方式，以帮助您找到适合您工作环境的方法。

此页面概述了默认（开箱即用）IDE接口布局。请注意，您正在安装的插件可能会更改IDE的外观和行为方式，例如可能有额外的命令按钮或菜单项。

首次运行WebStorm或未打开任何项目时，WebStorm将显示“ 欢迎”屏幕 ，该屏幕可以快速访问主要入口点。打开项目时，WebStorm将显示主窗口。

主窗口包含区域，如下图所示，标有数字标签：

  1. 主菜单和工具栏 - 包含影响整个项目或项目部分的命令，例如打开，创建项目，重构代码，运行和调试应用程序，保持文件受版本控制等等。

    主工具栏复制主菜单的基本命令，以便更快地访问。默认情况下，主工具栏是隐藏的。要显示它，请选择查看|主菜单中的工具栏 。

  2. 导航栏 - 项目工具窗口的快速替代方案。使用它在项目中导航并打开文件进行编辑 。

    使用视图|导航栏隐藏或显示导航栏;按Alt + Home将应用程序焦点置于导航栏。

  3. 状态栏 - 指示项目的状态，整个IDE，并显示各种警告和信息消息。

  4. 编辑器 - 在这里您可以阅读，创建和修改代码。

  5. 工具窗口 - 辅助窗口，提供对项目管理，搜索，运行和调试的访问，与版本控制系统的集成以及其他任务。
