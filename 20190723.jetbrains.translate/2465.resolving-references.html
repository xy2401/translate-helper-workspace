PyCharm 2019.1帮助

解决参考

PyCharm的即时检测可以立即检测到未解析的参考，并用红色曲线突出显示它们。PyCharm建议快速修复以处理源代码中未解析的引用。

应用快速修复

  1. 将插入符号放在未解析的参考位置，PyCharm显示红色灯泡 。

  2. 单击灯泡，或按Alt + Enter以显示可用的快速修复列表：

PyCharm提出了许多解决方案。例如，选择以下选项之一：

  • 导入此名称以添加导入语句。

  • 创建参数以将具有一些初始值的参数添加到函数参数列表中。

    另请参阅提取参数和更改签名 。

  • 忽略未解析的引用<完全限定符号名称> 。符号的完全限定名称将添加到忽略的引用列表中（ 文件|设置/首选项|编辑器|检查 - Python - 未解析的引用 ）。

  • 将<完全限定类型名称>的所有属性标记为已忽略 。在这种情况下，将使用末尾带有通配符（*）的类型的完全限定名称修改忽略的引用列表（ 文件|设置/首选项|编辑器|检查 - Python - 未解析的引用 ）。
