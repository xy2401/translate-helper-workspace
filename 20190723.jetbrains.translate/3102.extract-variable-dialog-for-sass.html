为Sass提取变量对话框

重构|提取变量
Ctrl + Alt + V

使用此对话框将Sass表达式替换为变量。

项目

描述

名称

指定新变量的名称。

申报地点

在源代码中选择将声明新变量的位置。声明可以是全局的 （变量在Sass文件的开头声明并在整个文件中可用）或本地声明（变量在使用前立即声明，并且仅在当前块中可用）。

替换所有出现的事件

选择此选项可替换所选表达式的所有匹配项（如果找到所选表达式的多个匹配项）。
