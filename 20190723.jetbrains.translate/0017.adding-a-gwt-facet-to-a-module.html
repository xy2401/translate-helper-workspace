将GWT Facet添加到模块

此功能仅在Ultimate版本中受支持。

向模块添加专用GWT方面时，IntelliJ IDEA会重新配置模块并自动添加所有必需的库。

将GWT构面添加到现有模块

  1. 打开项目工具窗口 （例如View | Tool Windows | Project ）。

  2. 右键单击该模块，然后选择“ 添加框架支持” 。

  3. 在“ 添加框架支持”对话框的左侧窗格中，选中“ Google Web Toolkit”复选框。

  4. 在GWT SDK字段中，指定GWT SDK位置。（要下载最新版本的GWT SDK，请使用Download GWT链接。）

  5. 如果要创建示例GWT应用程序 ，请选中“ 创建示例应用程序”复选框，并在下面的字段中指定应用程序名称。

  6. 在Add Frameworks Support对话框中单击OK 。
