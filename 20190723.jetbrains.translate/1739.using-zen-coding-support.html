Emmet支持

默认情况下，使用Tab扩展缩写，但您可以选择另一个键，如配置缩写扩展键中所述 。

Emmet和Emmet 1.1版是可以更快地开发HTML，CSS或JSX代码的工具包。您可以使用Emmet代码模板而无需离开IntelliJ IDEA。要将模板扩展为标记，只需键入其缩写并按扩展键或快捷键。例如，在HTML文件中，键入table>tr*3>td*2 ，然后按Tab键以获取3x2表的存根：

<table> <tr> <td> </ td> <td> </ td> </ tr> <tr> <td> </ td> <td> </ td> </ tr> <tr> <td > </ td> <td> </ td> </ tr> </ table>

IntelliJ IDEA支持诸如编写RGBA颜色，隐含，默认和布尔属性的新语法， 更新标记操作等功能。

请注意，您只能在Ultimate版本中使用带有CSS和XSL的Emmet。

启用和配置Emmet

使用IntelliJ IDEA，您可以使用本机Emmet模板以及编辑器上Zen CSS ， Zen HTML和Zen XSL节点下列出的200多个额外的HTML，CSS和XSL实时模板。 IntelliJ IDEA设置的实时模板页面（ Ctrl + Alt + S ）。

在IntelliJ IDEA中启用和配置本机Emmet

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ Emmet ” 。Emmet页面打开。

  2. 选择用于展开Emmet缩写的键，默认选中Tab 。

  3. 要在各种语言环境中启用和配置Emmet，请在编辑器|下的语言特定设置页面上使用相应的设置Emmet ： Emmet - HTML ， Emmet - CSS和Emmet - JSX 。

IntelliJ IDEA允许您使用和自定义Emmet实时模板，或者您可以添加自定义模板。假设您有一个模板条目，其中包含以下模板文本：

< entry type = “$ TYPES $” > $ END $ < entry >
要生成条目列表，只需输入“entry-list<entry[number=$]*5″并按Tab键 。默认情况下，将在type之前生成number属性。要自定义生成它的位置，您需要将ATTRS变量添加到模板中，例如：
< entry type = “$ TYPES $” $ ATTRS $ > $ END $ < entry >
ATTRS变量必须具有空字符串作为默认值，应跳过。

使用Emmet的实时模板

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 实时模板 ” 。

  2. 在Emmet CSS ， Emmet HTML和Emmet JSX页面上，选中要使用的模板旁边的复选框。

  3. 在列表中选择模板时，焦点将移动到“ 模板文本”区域，其中字段显示所选模板的设置。

  4. 在“ 模板文本”字段中，将所需的文本和变量添加到模板主体。

  5. 单击“ 编辑变量”按钮。在打开的“ 编辑模板变量”对话框中，在“ 默认值”字段中指定默认变量值，并在必要时选中“ 如果已定义则跳过”复选框。

配置缩写扩展密钥

默认情况下，按Tab键可扩展Emmet本机缩写和其他实时模板。对于其他实时模板，IntelliJ IDEA允许您重新定义默认扩展密钥。请注意，此自定义设置不会覆盖本机Emmet支持的默认设置;您将能够使用这些键之一扩展模板。

为本机Emmet缩写配置扩展密钥

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ Emmet ” 。Emment页面打开。

  2. 从Expand缩写列表中，选择新的扩展键而不是默认的Tab 。

配置Emmet实时模板的扩展密钥

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 实时模板 ” 。将打开“ 实时模板”页面。

  2. 展开Zen HTML ， Zen CSS或Zen XSL模板组，然后选择所需的模板。焦点移动到“ 模板文本”区域。

  3. 从“ 展开方式”列表中，选择用于展开模板的键。

使用Emmet模板围绕一段代码

  1. 在编辑器中，选择要包围的代码块并按Ctrl + Alt + J ，或选择Code |环绕着|主菜单上的实时模板 。

  2. 从“ 选择模板”列表中，选择“ Emmet” ：

  3. 输入要使用的Emmet缩写，然后按Enter键 。

    请注意右侧的下拉列表。单击向下箭头以查看最近应用的Emmet实时模板的历史记录：

    还要注意颜色指示。如果键入有效的Emmet缩写，则背景为绿色。但是，输入不存在的缩写时，背景变为红色：

在编辑点之间导航

在HTML和XML中，您可以在编辑点之间导航，即在Emmet模板适用的那些代码点之间导航。

  • 要将光标移动到上一个编辑点，请选择“ 导航”|上一个Emmet编辑点 ，或按Shift + Alt + [ 。

  • 要将光标移动到下一个编辑点，请选择“ 导航”|下一个Emmet编辑点 ，或按Shift + Alt +] 。
