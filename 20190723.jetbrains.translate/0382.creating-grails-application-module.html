创建Grails应用程序模块

此功能仅在Ultimate版本中受支持。

您可以使用Grails模块创建新项目，也可以将新Grails模块添加到现有项目中。

创建Grails应用程序模块

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目中：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Grails 。在页面的右侧，指定以下设置：

    • 您将要使用的Project SDK 。

    • Grails SDK Home - 选择由库表示的本地Grails安装。

    • 创建create-app或create-plugin - 根据您要创建的内容选择其中一个选项。

    • 选项 - 指定其他选项。

    单击下一步 。

  3. 指定名称和位置设置。

    单击完成 。
