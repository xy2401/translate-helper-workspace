介绍

IntelliJ IDEA允许您开发，调试和测试RESTful Web服务 。支持REpresentational State Transfer（REST）规范JSR-339规范版本2.0和Jersey参考实现 。

因为RESTful Web服务可以是各种类型的，所以没有明确的工作流程来开发它们。在本节中，我们将概述在IntelliJ IDEA中开发REST应用程序时可以执行的一些特定任务。

开发RESTful Web服务

要开发RESTful Web服务，请遵循以下常规步骤

  1. 确保已启用RESTful Web服务插件。为服务开发创建新项目或模块，或为现有模块启用RESTful Web Services开发支持。

    有关更多信息，请参阅准备REST开发 。

  2. 使用必要的类和方法填充RESTful Web服务模块。

  3. 配置要部署的工件 。

  4. 创建运行配置 。在“ 部署”选项卡上，指定要部署的工件。

  5. 在HTTP服务器上部署Web服务并启动服务器。

  6. 在本地或远程主机上运行应用程序。

  7. 测试RESTful Web服务 。
