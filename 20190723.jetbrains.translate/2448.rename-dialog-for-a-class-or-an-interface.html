除了重命名类本身之外，PyCharm还可以查找类名的用法。如果找到，您对类名所做的更改也可以应用于这些用法。

使用分配给不同的类别，这些类别对应于您可以打开和关闭的选项。

请注意，无论选择哪个选项，搜索范围（搜索名称出现位置）始终限制为当前实体（例如，文件或类）以及当前实体所依赖的实体。
