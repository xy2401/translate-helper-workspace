移动类对话框

重构|移动
F6

将在“项目”视图中选择的类或在编辑器中打开的类调用“ 移动类重构”对话框。

项目

描述

打包

指定目标包。单击省略号按钮，然后在“ 选择目标包”对话框中选择目标包，该对话框显示项目中所有包的树视图。

做内在的课堂

如果要使类成为内部类，请指定目标类。单击省略号按钮，然后在“ 选择类”对话框中选择目标类。

此选项不适用于ActionScript。

在评论和字符串中搜索

选择此选项可将更改应用于注释和字符串。

搜索文本事件

选择此选项可将更改应用于项目中包含的文档，HTML，JSP和其他文件。

搜索参考

选中此复选框可将更改应用于对相关文件的引用。此选项仅适用于Move File或Move Package重构。

移动到另一个源文件夹

如果选择此选项，则可以选择目标程序包所在的目标根目录。如果未选择该选项，则仅使用当前根。

对于包含单个源根的模块，将禁用此选项。

目标目标目录

对话框打开时，该字段显示当前存储实现要移动的类的文件的文件夹的路径。

路径以以下格式显示：

... \ < 项目 根 > < 目前的 命名空间 的文件夹 相 对于 所述 项目 根 > \
在指定要将类移动到的命名空间时，路径会自动更新。但是，如果要将类移动到另一个父命名空间下的不存在的命名空间，则IntelliJ IDEA将不会建议正确的文件夹，除非您通过在“ 目录”页面上将相关文件夹标记为“ 源”来为命名空间结构指定根文件夹设置/首选项对话框（ Ctrl + Alt + S ）。

执行以下操作之一：

  • 接受字段中显示的预选路径。

  • 从列表中选择另一个路径。所有这些都是从命名空间根目录或当前目录进行评估的，因此可以安全地选择其中任何一个。

  • 点击并在打开的对话框中选择一个文件夹。

  • 按F2并编辑预选路径。请记住，这可能会导致将来自动加载出现问题。

打开在编辑器中移动

选中此复选框可在编辑器中打开移动的类。
