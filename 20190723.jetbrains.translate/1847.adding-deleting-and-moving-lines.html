添加，删除和移动代码元素

添加一行

  • 按Shift + Enter可在插入符号当前所在位置之后添加新行，并将插入符号移动到此新行的开头。

    例如，您输入了一些文字。

    按Shift + Enter立即开始下一行。

  • 要在当前行之前添加一行，请按Ctrl + Alt + Enter 。

确保键盘快捷键没有冲突。您可以在PyCharm设置的Keymap页面（ Ctrl + Alt + S ）中执行此操作。

复制行或片段

  1. 将插入符号放在要复制的行上，或选择所需的文本片段。

  2. 按Ctrl + D.

删除一行

  • 按Ctrl + Y删除插入符号处的行。

移动一条线

  1. 将插入符号放在要移动的行上。

  2. 执行以下操作之一：

    • 在主菜单上，选择Code |向上/向下移动线 。

    • 按Shift + Alt +向上 键或Shift + Alt +向下键 。

    PyCharm将所选行向上或向下移动一行，执行语法检查。例如：

上下移动声明

  1. 选择要移动的语句，或者只将插入符号放在多行语句的第一行或最后一行。请注意，如果在当前上下文中不允许移动语句，则将禁用这些命令。

  2. 执行以下操作之一：

    • 在主菜单上，选择Code |将语句向上移动/向下移动语句 。

    • 按Ctrl + Shift +向上或Ctrl + Shift +向下 。

    如果将相同的命令应用于插入符号行或选择符号，则将向上或向下移动一行。

    PyCharm将选定的语句移动到前一个语句之上，或者直接在下一个语句的下方移动，执行语法检查。例如，将插入符号放在方法声明中：

将代码元素向左或向右移动

  1. 将插入符号放在所需的代码元素上，或选择要移动的元素。

  2. 执行以下操作之一：

    • 在主菜单上，选择命令Code |向左移动元素或代码|向右移动元素

    • 按Ctrl + Shift + Alt + Left或Ctrl + Shift + Alt + Right

当前实现此功能的代码元素的示例：

  • XML ：标记属性。

  • HTML ：标记属性。
