生成代码覆盖率报告

PyCharm提出了两种基于代码覆盖率测量结果生成HTML报告的方法：使用菜单命令或使用Coverage工具窗口 。

生成代码覆盖率报告

  1. 执行以下操作之一：

    • 在主菜单上，选择“ 工具”|生成覆盖率报告 。

    • 在Coverage工具窗口的工具栏中，单击 。

  2. 在打开的“ 生成覆盖率报告”对话框中，指定将存储生成的报告的目标目录，并可选择在浏览器中选中“ 打开生成的HTML ”复选框。

  3. 单击保存 。PyCharm会将生成的报告存储到指定位置，如果选中了相应的复选框，也会在默认浏览器中打开它。

请注意，如果要为在同一窗口中打开的多个项目之一保存代码覆盖率报告，请务必检查建议的目标位置，因为PyCharm会建议以前使用的位置。
