  • 工具栏

  • 主要控制

工具栏

项目

工具提示和快捷方式

描述

添加存储库位置

单击此按钮可配置新的存储库位置 。

修改位置网址

单击此按钮可编辑所选存储库的URL地址。

丢弃位置

单击此按钮可放弃所选的存储库位置。

显示/隐藏详细信息

单击此按钮可显示存储库位置下方的每个节点的详细信息（更改列表编号，用户名，上次更改的日期和时间）。

刷新
按Ctrl + F5

单击此按钮可刷新视图。

全部收缩
Ctrl + NumPad -

单击此按钮可使所有存储库位置下的所有节点都折叠。

库

在此区域中，管理可用的存储库并选择要从中检出内容的位置。

项目

描述

库

使用此树视图可以探索和管理可用的存储库位置。如有必要，右键单击节点，然后从其上下文菜单中选择相关项。

查看

单击此按钮可将所选节点的内容签出到指定位置。

上下文菜单

项目

描述

新

选择此菜单项可在所选存储库位置配置新的存储库位置或新的远程文件夹。

查看

选择此菜单项以检出所选节点的内容。

与之比较

选择此菜单项可将选定节点与指定分支进行比较 。

浏览更改

选择此菜单项可查看与指定条件（作者，时间范围和修订版）匹配的更改。

进口

选择此菜单项可将目录导入存储库。您可以从打开的“ 导入目录”对话框中选择要导入的目录 。

出口

选择此菜单项可将选定存储库或文件夹的内容导出到指定目标。导出的内容不受版本控制。

分支或标记

选择此菜单项可创建所选文件夹的分支或标记。

移动或重命名

选择此菜单项可更改所选文件夹的名称。

删除
删除

选择此菜单项可从存储库位置删除所选文件夹。

复制网址
按Ctrl + Alt +插入

选择此菜单项可将URL字符串放入剪贴板。

刷新

选择此菜单项以同步到存储库。

修改位置网址

选择此菜单项以编辑选定的存储库位置。

丢弃位置

选择此菜单项以放弃所选的存储库位置。
