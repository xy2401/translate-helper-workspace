使用结构视图浏览图表

要浏览图表，请按照以下常规步骤操作

  1. 按Ctrl + F12 。弹出图表元素菜单。

  2. 使用箭头键选择所需的图表元素，然后按Enter键 。选定的图节点元素变为活动状态。

  3. 再按一次Enter键 。元素的第一个成员获得焦点。

  4. 可以使用以下操作：

    • 按Enter键可在图元素及其成员之间切换焦点。

    • 当图元素具有焦点时，使用箭头键在元素之间跳转。

    • 当成员具有焦点时，使用垂直箭头键在元素中导航到所需成员。

    • 选择元素或成员后， 导航到基础源代码 。
