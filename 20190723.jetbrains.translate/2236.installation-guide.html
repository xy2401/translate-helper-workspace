安装PyCharm

PyCharm是一个跨平台的IDE，可在Windows，macOS和Linux操作系统上提供一致的体验。

本指南介绍了在Windows macOS Linux上安装PyCharm的方法。使用顶部的平台切换器选择另一个操作系统。

PyCharm有三个版本： 专业版 ， 社区版和教育版（Edu版） 。社区和Edu版本是开源项目，它们是免费的，但功能较少。PyCharm Edu提供课程并帮助您学习Python编程。专业版是商业版，提供了一套出色的工具和功能。有关详细信息，请参阅版本比较矩阵 。

使用Toolbox App安装

JetBrains Toolbox App是安装JetBrains产品的推荐工具。使用它来安装和维护多个版本，包括早期访问程序 （EAP）版本，必要时更新和回滚，并轻松删除任何工具。Toolbox App维护所有项目的列表，以便在正确的IDE和版本中快速打开任何项目。

安装工具箱应用程序

  1. 从Toolbox App网页下载安装程序（ .exe ）。

  2. 运行安装程序并按照向导步骤操作。

运行Toolbox App后，单击通知区域中的图标，然后选择要安装的产品和版本。

安装工具箱应用程序

  1. 从Toolbox App网页下载磁盘映像（ .dmg ）。

  2. 挂载图像并将JetBrains Toolbox应用程序拖动到Applications文件夹。

运行Toolbox App后，单击主菜单中的图标，然后选择要安装的产品和版本。

安装工具箱应用程序

  1. 从Toolbox App网页下载tarball（ .tar.gz ）。

  2. 将tarball解压缩到支持文件执行的目录。

    例如，要将其解压缩到推荐的/ opt目录，请运行以下命令：

    $ sudo tar -xzf jetbrains-toolbox-1.13.4801.tar.gz -C / opt

从解压缩的目录中执行jetbrains-toolbox二进制文件以运行Toolbox App并选择要安装的产品和版本。第一次运行Toolbox App后，它会自动将Toolbox App图标添加到主菜单中。

您可以使用此shell脚本自动使用最新版本的Toolbox App下载tarball，将其提取到推荐的/ opt目录，并在/ usr / local / bin目录中创建符号链接。

从Toolbox App登录JetBrains帐户，它将自动激活您安装的任何IDE的可用许可证。

独立安装

手动安装PyCharm以管理每个实例和所有配置文件的位置。例如，如果您的策略需要特定的安装位置。

  1. 下载安装程序 （ .exe ）。

  2. 运行安装程序并按照向导步骤操作。

用于32位系统的JRE未与PyCharm捆绑在一起。如果您使用的是32位版本的Windows， 请在安装向导中选中“ 下载并通过JetBrains安装JRE x86”复选框。

  1. 下载磁盘映像 （ .dmg ）。

  2. 挂载图像并将PyCharm应用程序拖动到Applications文件夹。

  1. Ubuntu安装：

    对于Ubuntu 16.04及更高版本，您可以使用快照包来安装PyCharm。

    运行以下命令：

    sudo snap install <pycharm-professional，·pycharm-community，或·pycharm-educational> --classic

    接下来，在pycharm-professional运行pycharm-professional ， pycharm-community或pycharm-educational 。

  2. 如果快照包不可用，请将pycharm-*.tar.gz文件解压缩到其他文件夹，如果当前的Download文件夹不支持文件执行：

    tar xfz pycharm - * .tar.gz -C <new_archive_folder>

    根据文件系统层次结构标准（FHS）推荐的安装位置是/opt 。要将PyCharm安装到此目录中，请输入以下命令：

    sudo tar xfz pycharm - * .tar.gz -C / opt /

  3. 切换到bin子目录：

    cd <新存档文件夹> / pycharm - * / bin
    例如，
    cd / opt / pycharm - * / bin

  4. 从bin子目录运行pycharm.sh 。

无声安装

无需任何用户界面即可执行静默安装。网络管理员可以使用它在多台计算机上安装PyCharm，避免中断其他用户。

要执行静默安装，请使用以下开关运行安装程序：

  • /S ：启用静默安装

  • /D ：指定安装目录的路径

  • /CONFIG ：指定静默配置文件的路径

例如：

pycharm-professional.exe / S /CONFIG=d:\temp\silent.config / D = d：\ IDE \ PyCharm
要么
pycharm.exe / S /CONFIG=d:\temp\edu_silent.config / D = d：\ IDE \ PyCharm Edu

要在安装过程中检查问题，请添加带有日志文件路径和名称的/LOG开关。安装程序将生成指定的日志文件。例如：/ /LOG=d:\JetBrains\PyCharm\install.log : /LOG=d:\JetBrains\PyCharm\install.log 。

无提示配置文件

您可以在https://download.jetbrains.com/python/silent.config和https://download.jetbrains.com/python/edu_silent.config下载PyCharm的静默配置文件。

静默配置文件定义了安装PyCharm的选项。使用默认选项，仅对当前用户（ mode=user ）执行静默安装。如果要为所有用户安装PyCharm，请使用文本编辑器打开静默配置文件，更改安装模式选项的值（ mode=admin ）并以管理员身份运行安装程序。

默认的静默配置文件对于每个JetBrains产品都是唯一的。您可以根据需要对其进行修改以启用或禁用各种安装选项。
