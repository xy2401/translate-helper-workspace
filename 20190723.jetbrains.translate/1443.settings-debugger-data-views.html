常见的调试器设置

项目

描述

按字母顺序排序值

选择此选项可对“ 调试工具”窗口的“ 变量”窗格中的值进行排序。

在“变量”视图中启用自动表达式

如果希望IntelliJ IDEA调试器自动计算表达式并在“ 调试工具”窗口的“ 变量”窗格中显示相应的值，请选择此选项。

调试器分析断点附近的上下文（当前语句，前一个语句和后一个语句）。它这样做是为了在源代码中找到各种表达式（如果可用），例如myvar.myfield 。

如果此类表达式不包含显式方法调用，则调试器将对它们进行求值并在“变量”视图中显示相应的值。

在诸如Groovy之类的语言中，在评估表达式时无法判断是否调用了任何方法，并且此类方法调用通常会导致不必要的副作用，因此建议在调试Groovy代码时禁用此选项。

项目

描述

显示内联值

选择此选项可启用内联值视图功能，该功能允许在编辑器中查看其使用旁边的变量值。

显示值工具提示

选择此选项可以自动显示值的工具提示。

此上下文中的工具提示是一个弹出窗口，它在“ 调试工具”窗口的“ 变量”窗格中提供了一种替代方法，有时更方便地显示值。

为了说明，我们假设您的代码中有这样的语句：


String s = “Hello，World！\ n你好，世界！“ ;

在调试器中执行此语句时，您将在“ 变量”窗格中看到与此类似的行：


s = { java 。 郎 。String @ 62 } “你好，世界！\ n你好，世界！“

换行符显示为\n 。

如果启用了“ 显示值工具提示”选项并单击此行然后按住鼠标指针，则会看到一个黄色区域（“工具提示”），其中s的值显示为

你好，世界！
你好，世界！

用真正的换行代替\n 。

如果禁用此选项，请按Alt显示值。

值工具提示延迟（ms）

指定鼠标指针悬停在“ 调试工具”窗口的“ 变量”窗格中的对象上的时刻与显示带有对象值的工具提示之间的延迟（以毫秒为单位）。

在代码选择上显示值工具提示

选择此选项可启用在编辑器中选择代码片段时显示表达式值的工具提示。
