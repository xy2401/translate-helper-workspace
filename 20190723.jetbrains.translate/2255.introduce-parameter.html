Extract Parameter重构用于向函数声明添加新参数并相应地更新函数调用。

在Python中就地提取参数

准备提取参数

默认情况下，PyCharm中启用了就地重构 。因此，如果您尚未更改此设置，则Python的Extract Parameter重构将在编辑器中就地执行：

  1. 在编辑器中，将光标放在要由参数替换的表达式中。

  2. 执行以下操作之一：

    • 按Ctrl + Alt + P.

    • 选择Refactor |提取|主菜单上的参数 。

    • 选择Refactor |提取|上下文菜单中的参数 。

  3. 如果检测到当前插入符号位置有多个表达式，则会出现“ 表达式”列表。如果是这种情况，请单击表达式以选择它。或者，按向上或向下导航到感兴趣的表达式，然后按Enter键将其选中。

  4. 在带红色边框的框中键入参数名称。

  5. 要完成重构，请按Tab键或Enter键 。

    如果您尚未完成重构并想取消所做的更改，请按Escape键。

    请注意，有时您可能需要多次按相应的键。

使用“提取参数”对话框在Python中提取参数

使用“提取参数”对话框提取参数

为了能够使用“ 提取参数”对话框（而不是就地执行重构），请确保在编辑器设置中禁用“ 就地启用重构”选项。

在这种情况下，您执行Extract Parameter重构，如下所示：

  1. 在编辑器中，将光标放在要由参数替换的表达式中。

  2. 执行以下操作之一：

    • 按Ctrl + Alt + P.

    • 选择Refactor |提取|主菜单上的参数 。

    • 选择Refactor |提取|上下文菜单中的参数 。

  3. 如果检测到当前插入符号位置有多个表达式，则会出现“ 表达式”列表。如果是这种情况，请单击表达式以选择它。或者，按向上或向下导航到感兴趣的表达式，然后按Enter键将其选中。

  4. 在打开的“ 提取参数”对话框中：

    1. 在“ 名称”字段中指定参数名称。

    2. 如果在函数体中找到多个表达式，则可以选择仅使用对新参数的引用替换所选的实例或所有找到的实例。使用“ 替换所有实例”复选框指定您的意图。

  5. 预览并应用更改 。
