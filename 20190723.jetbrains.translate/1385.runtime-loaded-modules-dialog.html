运行时加载的模块对话框

使用此对话框可管理运行时加载模块（RLM）的依赖关系。RLM由其主要类别标识。

要添加依赖项，请单击 （ Alt + Insert ）并在打开的“选择运行时加载模块的主类”对话框中选择相应RLM的主类 。

要优化（减少）已编译RLM文件的大小，请选中“ 优化”复选框。

有关更多信息，请参阅下表。

项目

描述

或Alt + Insert

使用此图标或快捷方式添加依赖项。在打开的“选择运行时加载的模块的主类”对话框中选择相应RLM的主类 。

或Alt +删除

使用此图标或快捷方式从依赖项列表中删除选定的RLM。

主类

在此列中，将显示主RLM类的完全限定名称。

要用不同的类替换类，请单击相应的字段（表格单元格）并单击出现在字段中（ Shift + Enter ）。然后在打开的“选择运行时加载的模块的主类”对话框中选择所需的类。

输出文件

在此列中，显示了要生成的RLM SWF文件的路径（只读）。所有路径都相对于构建配置输出文件夹。

优化

选中要优化（减少）的相应RLM SWF文件大小的复选框。
