创建布局对话框

项目工具窗口|包含应用程序类的包的上下文菜单新的 - Android组件

创建活动或片段类型的组件时，将打开该对话框，并选中“ 创建布局文件”复选框。

在此对话框中，指定布局名称，其根元素以及要包含在此布局中的限定符集。

项目

描述

文件名

在此文本框中，指定与新组件关联的布局定义文件的名称。仅支持小写字母和数字。

根元素

在此文本框中，指定布局的根元素。按Ctrl + Space可获取可用值列表。该值必须是View或ViewGroup类型，请访问http://developer.android.com/guide/topics/resources/layout-resource.html了解更多信息。

目录名称

在此文本框中，指定将相对于res文件夹存储布局定义的文件夹。

  • 如果您的应用程序不需要与各种Android设备兼容，因此不需要多屏幕支持，请接受默认layout子文件夹。

  • 要提供备用资源 ，请指定用于确定特定设备配置的资源限定符 。将相关限定符从“ 可用限定符”列表移动到“ 选定限定符”列表并指定其值。IntelliJ IDEA将所有选定的限定符附加到目录名称字段，并使用短划线字符作为分隔符。

有关详细信息，请参阅创建资源

可用的限定符

从此列表中，选择确定特定设备配置的资源限定符 。选择相关限定符并单击 。然后在打开的对话框中指定限定符的值。限定符将添加到“ 选择的限定符”列表中。
