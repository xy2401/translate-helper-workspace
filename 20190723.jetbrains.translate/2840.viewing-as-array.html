以数组或DataFrame的形式查看

要使用命令View as Array / View as DataFrame，请按照下列步骤操作：

  1. 启动调试器会话 。

  2. 在“ 调试”工具窗口的“ 变量”选项卡中 ，选择一个数组或一个DataFrame。

  3. 单击右侧的链接视图为阵列/视图作为DataFrame 。

    或者，您可以从上下文菜单中选择View as Array或View as DataFrame 。

    将出现“ 数据视图”工具窗口。
