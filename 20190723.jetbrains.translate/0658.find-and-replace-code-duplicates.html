查找和替换代码重复项

“ 查找和替换代码重复项”操作允许您查找与所选方法或常量字段类似的代码重复，并将其替换为对原始方法或常量的调用。它还允许您指定搜索范围。

  1. 将光标定位在要搜索其重复项的方法或常量字段中。

  2. 从主菜单或上下文菜单中，选择Refactor |查找和替换代码重复项 。

  3. 在打开的对话框中，选择IntelliJ IDEA查找代码重复的范围。

  4. 对于每个找到的代码副本，IntelliJ IDEA将提示您确认替换。
