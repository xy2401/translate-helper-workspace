本教程的内容

本教程介绍如何使用PyCharm内置的SSH终端和远程工具。

SSH基础知识超出了本教程的范围。

先决条件

在开始之前，请确保：

  • 您正在使用PyCharm 3.0或更高版本。本教程是使用PyCharm 2016.2创建的

  • 您可以访问SSH服务器。

让我们看看我们如何使用PyCharm的内置SSH客户端。

开始连接

在主菜单上，选择“ 工具”|启动SSH会话... ，然后单击编辑凭据 ：

提供连接信息

在“ SSH会话”对话框中，指定连接信息：主机（本地或远程），端口，登录名和密码：

我们在SSH会话中可以做些什么？

像往常一样进行交互式会话...远程运行命令，复制和粘贴，使用向上和向下箭头键滚动命令历史记录。

使用远程SSH外部工具

让我们定义一个外部工具，它将通过SSH运行命令，例如，显示日历。

配置SSH外部工具

打开“设置/首选项”对话框（ 在主工具栏上），在“ 工具”节点下，选择“ 远程SSH外部工具 ”页面。点击要创建新的远程工具，并在“ 创建工具”对话框中指定新的工具设置：

  • 字段名称有助于指定新工具的一些可见名称。接下来的两个字段是可选的：

    • 在Description字段中键入一些有意义的描述。

    • 字段组表示将在“ 工具”菜单中创建的具有指定名称的组，新的SSH外部工具将置于此组下。

  • 在“ 显示位置”区域中，指定要在何处查看新工具。在这种情况下，请仅选中“ 主菜单 ”复选框。

  • 在“ 连接设置”区域中，单击“ 部署服务器 ”单选按钮，然后选择“每次运行时选择服这意味着每次要运行此外部工具时，都必须指定连接设置。

  • 最后，在“ 工具设置”区域中，指定要远程执行的工具。在下面的屏幕截图中，我们运行了一个bash命令;参数和工作目录是可选的。我们还可以使用宏来注入当前的命令名。

单击“ 确定”关闭“ 创建工具”对话框，然后返回“ 远程SSH外部工具 ”页面。我们可以在列表中看到新工具：
