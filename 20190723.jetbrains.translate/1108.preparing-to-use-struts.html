准备使用Struts

此功能仅在Ultimate版本中受支持。

在这个部分：

  • 在你开始之前
  • 基本
  • 使用Struts Facet创建模块
  • 将Struts Facet添加到模块
  • 自定义Struts Facet

在你开始之前

确保已启用Web和Struts1.x捆绑插件。默认情况下会激活插件。如果禁用了插件，请在“ 插件”页面上启用它们，如管理插件中所述 。

基本

通过Web和Struts1.x方面实现与Struts的集成。这些专用方面包含设置，配置文件路径和验证规则。此信息确定模块的结构，以便IntelliJ IDEA检测如何处理模块内容。

Struts1.x facet只能作为Web facet的子项添加。请注意，模块中只允许使用一个Struts1.x构面。

使用Struts Facet创建模块

当您使用专用Web和Struts构面创建Struts模块时，IntelliJ IDEA会配置模块，自动下载并添加所有必需的库。

  1. 执行以下操作之一：

    • 如果要创建新项目：在“ 欢迎”屏幕上单击“ 创建新项目 ”，或选择“ 文件”|新的|项目 。

      结果，“ 新建项目”向导将打开。

    • 如果要将模块添加到现有项目：打开要添加模块的项目，然后选择“ 文件”|新的|模块 。

      结果，“ 新建模块”向导将打开。

  2. 在向导的第一页上，在左侧窗格中，选择Java 。在页面的右侧，指定要使用的JDK 。

  3. 在“ 其他库和框架”下 ，选中“ Web应用程序”复选框。

    从“ 版本”列表中选择要支持的Servlet规范的版本 。

    如果要创建部署描述符web.xml文件，请选中Create web.xml复选框。

  4. 选择Struts复选框。

  5. 从版本列表中选择Struts版本。

  6. 你需要一个实现Struts的库。您可以选择使用现有库，创建和使用新库，下载库文件（如果它们尚未在您的计算机上可用），或推迟设置库直到以后。

    • 使用库 - 从列表中选择要使用的库（如果已在IntelliJ IDEA中定义了相应的库）。

      创建 - 如果计算机上已有相应的库文件（ .jar ），则可以在库中排列这些文件并使用该新库。为此，请单击“ 创建” ，然后在打开的对话框中选择所需的文件。

      （可选）单击“ 配置”以编辑选定的库。

    • 下载 - 选择此选项可下载实现所选Struts版本的库文件。（下载的文件将安排在库中。）

      （可选）单击“ 配置”以编辑库设置和内容。

    • 稍后设置库 - 选择此选项以推迟设置库直到稍后的时间。

    配置 - 单击此按钮可编辑“ 使用库”或即将下载的库旁边所选库的设置。

    单击下一步 。

  7. 指定名称和位置设置。

    单击完成 。

IntelliJ IDEA按如下方式配置新模块

  • 使用Web和WEB-INF节点创建模块的结构。

  • 在WEB-INF节点下为版本2.2 - 2.5创建Web应用程序部署描述符Web.xml 。

  • 在WEB-INF节点下创建Struts配置文件struts-config.xml 。

  • 在Web.xml配置Action servlet。

  • 下载并安装实现Struts的库。

将Struts Facet添加到模块

将Struts构面添加到现有模块时，IntelliJ IDEA会自动下载并添加必要的库。

Struts facet只能作为Web facet的子项添加。如果模块没有Web构面，请先添加它。另请注意，模块中只允许使用一个Struts构面。

这将改变模块的配置 。

  1. 打开模块设置。

  2. 在“ 模块”节点中，右键单击相关模块。

  3. 要添加Web构面，请从上下文菜单中选择“ 新建 ”。在可用构面的下拉列表中，选择Web 。

  4. 右键单击Web节点，然后在其上下文菜单中选择“ 新建 ”。

  5. 在可用构面的上下文菜单中选择Struts 。右侧窗格显示Facet'Struts'对话框。

  6. 在Struts功能和验证选项卡中自定义Struts构面设置。

  7. 如果需要一些其他库，则会显示相应的警告消息。单击修复 。必要的库将被下载并自动添加。

自定义Struts Facet

启用基本Struts支持后，IntelliJ IDEA可以通过自定义Struts专用构面来添加更多Struts功能。

  1. 从相关模块打开Struts facet的设置。

  2. Struts Features选项卡显示了其他Struts组件的列表：

    • Struts taglib

    • Struts El-taglib

    • 瓷砖

    • 验证器

    • Struts的面孔

    • 脚本

    • 附加功能

    要从列表中启用对组件的支持，请选中它旁边的复选框。

  3. 在“ 验证”选项卡中，指定要验证的文件和验证设置：
    • 要启用验证特定组件的配置文件，请选中相关的复选框。IntelliJ IDEA将检查是否可以使用可用资源正确支持所选组件，并建议在必要时下载缺少的库。

      可以为以下组件的配置文件启用验证：

      • 支柱

      • 瓷砖

      • 验证器

    • 指定验证设置：

      • 要了解验证期间检测到的错误，请选中“将错误报告为警告”复选框。

      • 要从验证中排除Struts属性键 ，请选中“ 禁用属性键验证”复选框。
