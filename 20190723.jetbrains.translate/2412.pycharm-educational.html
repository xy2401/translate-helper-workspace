PyCharm教育工具

使用PyCharm，您可以使用PyCharm Educational（Edu）版本中提供的教育功能学习编程语言和相关技术。您可以通过EduTools插件为PyCharm Community和PyCharm Professional启用这些功能。

安装EduTools插件

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 插件”并搜索EduTools插件（如果需要，单击“在存储库中搜索” ）。

  2. 单击“ 安装”以启动安装过程。完成后，系统会要求您重新启动PyCharm。

首次安装EduTools时，您需要选择自己的角色： 学习者或教育者 。

  • 加入公共互动编程课程

  • 根据任务描述填写练习代码中的占位符，开始学习。

  • 使用智能开发工具验证任务，获得即时反馈和快速结果。

  • 使用自定义任务，检查，测试，提示等创建您自己的交互式课程。

  • 与您的学生或同事公开或私下分享您的课程

  • 让学习者在综合测试的帮助下独立完成作业。

无论您选择何种角色，您都需要登录Stepik ，这是一个提供各个科学领域课程的网站。

登录Stepik

  1. 您可以使用以下方法之一：

    • 在PyCharm中 / 状态栏中的图标表示用户是否登录到Stepik。点击按钮登录Stepik。

    • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，导航到“ 工具”|单击“ 教育” ，然后单击“ 登录Stepik” 。

    • 打开“ 浏览课程”对话框（“ 文件”|“浏览课程” ），然后单击“ 登录到Stepik”以查看更多课程 。

    当您从PyCharm登录Stepik时，您会收到授予PyCharm一些权限的请求：

  2. 单击“ 授权”

    之后，返回PyCharm。状态栏中的图标应更改为并且您的课程列表（ 文件|浏览课程 ）与您的Stepik课程完全同步。

现在，您可以根据所选角色继续使用教育工具。请注意，您可以通过选择或取消选择“ 帮助”菜单中的“ 启用课程创建者功能”项来更改您的角色。
