标签和缩进

项目

描述

使用制表符

  • 如果选中此复选框，则使用制表符：

    • 按Tab键

    • 缩进

    • 用于代码重新格式化

  • 清除该复选框后，PyCharm使用空格而不是制表符。

智能标签

  • 如果选中此复选框，则由代码块嵌套定义的缩进部分由制表符和（如果需要）空格组成，而由对齐定义的缩进部分仅由空格构成。

  • 如果清除此复选框，则仅使用选项卡。这意味着符合指定选项卡大小的一组空格会自动替换为选项卡，这可能会导致细微对齐。

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

标签大小

在此字段中，指定选项卡中包含的空格数。

缩进

在此字段中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

继续缩进

在此字段中，指定要在数组元素之间，表达式，方法声明和方法调用之间插入的空格数（或选中“ 使用选项卡字符”复选框）。

在空行上保持缩进

如果选中此复选框，则PyCharm将在空行上保留缩进，就好像它们包含一些代码一样。
如果清除此复选框，PyCharm将删除制表符和空格。

空间

使用此选项卡可以指定代码中的空格位置。要让PyCharm自动在某个位置插入空格，请在列表中选中此位置旁边的复选框。结果显示在“ 预览”窗格中。

包裹和括号

在此选项卡中，自定义代码样式选项，PyCharm将在重新格式化源代码时应用这些选项。左侧窗格包含异常列表（ 重新格式化时保留 ），以及各种代码构造（列表，语句，操作，注释等）的放置和对齐选项右侧窗格显示预览。

对齐优先于缩进选项。

右边距（列）

使用Hard wrap at字段指定元素右侧所需的边距空间。如果选择“ 默认”选项，则使用全局设置中右边距的值。

打字输入

使用Wrap on typing设置指定编辑文本在指定的Hard wrap中的拟合方式。您可以选择以下选项之一：

  • 默认值 - 在这种情况下，PyCharm使用全局设置中指定的Wrap on typing选项。

  • 是 - 在这种情况下，PyCharm使用右边距字段中指定的值。

  • 否 - 在这种情况下，此选项将关闭，并且一行可以超过右边距中指定的值。

视觉指南

使用“ 可视指南”字段指定多个右边距。您可以保留默认值或输入保证金的空格数。如果要指定多个边距，请输入以逗号分隔的数字。

重新格式化时保留

使用复选框配置PyCharm在重新格式化源代码时将进行的异常。例如，默认情况下，选中“ 换行符”复选框。如果您的代码包含比标准约定更短的行，则可以在重新格式化源代码之前通过禁用换行符复选框来转换它们。

空白行

使用此选项卡可以定义PyCharm在重新格式化后保留并插入代码中的空白行的位置和数量。对于每种类型的位置，请指定要插入的空行数。结果显示在“ 预览”窗格中。
