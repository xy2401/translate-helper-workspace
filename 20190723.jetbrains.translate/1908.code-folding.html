折叠代码元素

您可以折叠（折叠）代码片段，将它们缩小为单个可见行。通过这种方式，您可以隐藏目前看起来不重要的细节。如果需要，可以扩展（展开）折叠的代码片段。

折叠的代码片段通常显示为阴影椭圆（ ）。

如果折叠的代码块包含错误，PyCharm会将折叠的片段突出显示为红色。

代码折叠方式

您可以使用以下命令折叠和展开代码片段：

  • 代码折叠切换（ ， 要么 ）。这些切换显示在相应折叠区域左侧的编辑器中。如果一个地区展开， 表示该区域的开始位于其末端。对于折叠区域，切换显示为 。

    如果按住Alt键并单击 ， 要么 ，该区域递归地折叠或扩展，即与其所有从属区域一起。

  • 折叠菜单和相关键盘快捷键的命令。可以从主菜单栏（ 代码|折叠 ）访问折叠菜单，也可以在编辑器中作为上下文菜单访问。快捷方式显示在菜单中。请参阅折叠菜单的命令和关联的快捷方式 。

  • 折叠碎片本身：点击扩展相应的片段。另请参阅查看折叠的代码片段 。

折叠预定义和自定义区域

你可以折叠和展开：

  • 列表和词典。

  • 代码块，即关键字class下的代码， def ， while ， if等。

  • 连续评论的行 。

  • 对应于导入声明，方法体，文档注释等的预定义区域 。预定义区域大致对应于编辑器| 默认情况下在“ 折叠”下列出的区域 一般| PyCharm设置的代码折叠页面（ Ctrl + Alt + S ）。

    对于预定义区域，折叠切换可立即使用，无需执行任何其他操作。

  • 任何选定的代码片段。通过Code |创建和删除选择的自定义折叠区域折叠|折叠选择/删除区域命令（ Ctrl +。 ）。

  • 由相应的注释折叠标记包围的区域（例如//<editor-fold desc="Description">...//</editor-fold> ）。请参阅使用代码折叠注释 。

代码折叠适用于关键字if / while / else / for / try / except / finally / with / Long string literals / Long collection literals / Sequential comments for至少两个语句。

支持折叠评论

您可以使用符合NetBeans样式或Visual Studio样式的注释行包围代码片段来创建自定义折叠区域。

NetBeans风格

对于非Python文件：

// <editor-fold desc = “描述” > ...// </ editor-fold>

对于Python文件：

＃ <editor-fold desc = “描述” > ...＃ </ editor-fold>

Visual Studio风格

对于非Python文件：

//区域描述......// endregion

对于Python文件：

#region说明......#endregion

为文件选择样式后，请勿使用该文件中的其他样式。

用折叠注释围绕片段

  1. 选择感兴趣的代码片段。

  2. 选择代码|环绕或按Ctrl + Alt + T.

  3. 选择要使用的折叠注释。

  4. 指定片段描述。

    现在，如果您折叠片段，则会显示您指定的描述以代替代码。

导航到折叠区域

您可以导航到由周围代码片段与相应的注释折叠标记形成的自定义折叠区域：

  1. 选择导航|自定义折叠或按Ctrl + Alt +。 。

  2. 选择目标折叠区域。（列表中的区域由其描述标识。）
