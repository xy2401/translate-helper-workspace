EJB工具窗口|具有EJB构面的模块的上下文菜单|跳转到源|新 - 会话Bean

使用此对话框可以创建会话Bean 。

项目

描述

<EJB名称>

在此文本框中，键入用于生成EJB组成类的名称的基本名称。生成EJB名称的规则在“ Java EE名称”对话框中配置。

包

在此文本框中，指定bean所在的包的完全限定路径。如有必要，请单击“ 浏览”按钮并在项目树中选择所需的包。

EJB类

在此文本框中，指定bean实现类的名称。如有必要，请单击“ 浏览”按钮打开“ 选择EJB类”对话框，您可以在其中按名称搜索所需的类，或在项目树中选择它。
