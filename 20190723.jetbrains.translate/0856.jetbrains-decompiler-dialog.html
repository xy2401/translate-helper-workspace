JetBrains反编译器对话框

当IntelliJ IDEA尝试反编译代码时，会出现此对话框。

项目

描述

接受

如果您决定同意指定的条款和条件，请单击此按钮。在这种情况下，将来不会显示该对话框，并且将自动反编译相应的代码。

拒绝并重新启动

如果您不同意指定的条款和条件，请单击此按钮。在这种情况下， Java Bytecode Decompiler插件将被禁用，您需要重新启动IntelliJ IDEA才能实现更改。

稍后决定

如果您想推迟您的决定，请单击此按钮。在这种情况下，对话框和带有反编译代码的窗口将关闭。但是，当IntelliJ IDEA尝试反编译另一个源代码时，将再次提示您使用相同的对话框。
