基本

在IntelliJ IDEA中，您可以定义多个Ruby SDK。这样，您可以从计算机上可用的解释器列表中选择要在项目中使用的一个。

IntelliJ IDEA支持：

  • 标准的Ruby SDK

  • IronRuby的

  • MacRuby的

可以在以下级别配置Ruby SDK：

  • 当前项目 ：选定的Ruby解释器将用于当前项目。

    在此级别上配置Ruby SDK 如下所述 。

  • 新项目 ：选定的Ruby解释器将用于新项目而不是默认项目。

    有关详细信息，请参阅创建基于Rails的项目一节。

为所选项目配置Ruby SDK

要为当前项目配置Ruby SDK，请按照下列步骤操作：

  1. 打开“ 项目结构”对话框（ Ctrl + Shift + Alt + S ），然后单击“ Gems”选项卡。

  2. 如果已经安装了所有需要的Ruby SDK并将其添加到项目中，请从Ruby SDK列表中选择一个。为此，请单击所需Ruby SDK左侧的单选按钮，或按空格键：

    如果可用SDK列表中缺少所需的Ruby SDK，请单击 。

  3. 应用更改。
