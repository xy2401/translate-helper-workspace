新的Servlet对话框

档案|新的| Servlet的

指定新servlet的设置。

项目

描述

名称

servlet名称的“根部分”。

此名称（及其前缀和后缀）将在web.xml中的<servlet-name>元素或servlet类@WebServlet批注中使用。

您可以在Settings对话框的Editor | Code Style | Java页面的Java EE Names选项卡上指定servlet名称前缀和后缀（ <servlet-name>标记 Prefix和Suffix字段）。在这种情况下，前缀和后缀会自动添加到指定名称之前和之后。

包

应在其中创建新servlet类的包的名称。

点击 。然后选择现有包或单击创建一个新包。

类

servlet类的完全限定名称。

使用包名，类名前缀，servlet名称和类名后缀生成默认类名。

类名前缀和后缀在“ 设置”对话框中设置 。（ 编辑器|代码样式| Java页面的Java EE名称选项卡上的Servlet类 前缀和后缀字段。）

如果要指定现有类，请单击并在打开的对话框中选择该类。

创建Java EE 6注释类

如果未选中该复选框，则将servlet名称 - 类映射添加到web.xml 。

如果选中该复选框，则新的servlet类将被@WebServlet注释，并且不会对web.xml进行任何更改。

如果项目中没有web.xml ，则默认情况下会选中该复选框，您无法取消选中它。
