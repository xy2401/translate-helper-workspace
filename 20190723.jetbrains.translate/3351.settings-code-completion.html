使用此页面配置代码完成和参数信息 。

相符

如果您希望在建议代码完成匹配时考虑字母大小写，请选择此选项。选择是否要匹配所有字母的第一个字母的大小写。

自动插入单个建议

如果只有一个基本和/或智能完成的代码完成建议，请选择是否要自动完成代码。

按字母顺序排序建议

如果要按字母顺序对建议列表中的项目进行排序，而不是按相关性对其进行排序，请选择此选项。

您可以随时单击更改此行为要么 建议列表中的图标。

键入时显示建议

如果您希望自动调用建议列表，请选中此复选框，而不必显式调用完成。

按空格，点或其他依赖于上下文的键插入所选建议

选择是否要通过键入取决于语言，上下文等的某些键来插入所选建议。

显示文档弹出窗口

选中此复选框可让WebStorm自动为建议列表中的每个项目显示弹出窗口，其中包含查找列表中当前突出显示的类，方法或字段的文档。

在右侧的字段中，指定延迟（以毫秒为单位），之后应显示弹出窗口。

完成时显示参数名称提示

如果要显示参数值的提示，请选择此选项。

显示参数信息弹出窗口

选中此复选框可让WebStorm在编辑器中键入左括号时自动显示包含所有可用方法签名的弹出窗口，或者从建议列表中选择方法。

在右侧的文本字段中，指定弹出窗口出现后的延迟（以毫秒为单位）。

如果未选中此复选框，请使用Ctrl + P显示参数信息。

显示完整方法签名

如果选中此复选框，则参数信息将显示完整签名，包括方法名称和返回的类型。

在完成JOIN时使用别名

在JOIN语句中为表创建别名。

在自动生成的ON子句中反转操作数的顺序

在ON子句中切换操作数。清除该复选框后，FROM表将成为JOIN条件中的第一个。

在完成表名时自动添加别名

为表名创建别名。

在表名后的完成中建议别名

使用代码完成时（ Ctrl + Space ），建议表名的别名。

自定义别名（表）

您可以添加表名以及要用于此表的别名。要添加表别名对，请单击“ 添加别名”按钮（ ）。
