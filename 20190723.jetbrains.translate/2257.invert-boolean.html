反转布尔值

这种重构旨在将布尔变量的值及其所有用法从True翻转为False，反之亦然。

例

之前

后

def foo {}： abc < caret here > = True 返回 abc

def foo {}： abc = False 返回 不是 abc

要反转布尔变量，请按照下列步骤操作

  1. 将插入符号放在布尔变量的名称上。

  2. 执行以下操作之一：

    • 在主菜单上，选择Refactor |反转布尔值 。

    • 在插入符号位置的上下文菜单中，选择“ 重构”|反转布尔值 。

  3. 在打开的对话框中，单击“ 预览”以在“查找”工具窗口中查看建议的更改，或单击“ 重构”以静默方式反转检测到的事件。
