WebStorm 2019.1帮助

在项目中搜索目标和用法

您可以搜索项目中目标的出现次数和用法，并使用不同的范围缩小搜索范围并排除某些项目。

在项目中找到您的目标

  1. 从主菜单中，选择“ 编辑”|找到|在路径中查找 （ Ctrl + Shift + F ）。

  2. 在搜索字段中，键入搜索字符串。要查看以前搜索的列表，请按Alt +向下 。

    如果需要，请指定其他选项。

    WebStorm列出搜索字符串和包含它们的文件。

    要进行多行搜索，请单击用于输入换行的图标，然后按Ctrl + Alt +向下键 / Ctrl + Alt +向上键浏览出现的位置。

  3. 检查对话框的预览区域中的结果，您可以在其中替换搜索字符串或选择其他字符串，再次按Ctrl + Shift + F并开始新的搜索。

  4. 如果需要在单独的工具窗口中查看出现的列表，请单击“ 在查找窗口中打开” 。您可以使用此窗口及其选项对结果进行分组，预览结果，并进一步使用它们。

缩小搜索范围

您可以在“ 在路径中查找”对话框中使用不同的选项来调整搜索过程。

  1. 在“ 在路径中查找”对话框中，选择“ 单词”或“ 匹配大小写”等选项以查找项目中的确切单词，或匹配字母大小写。

  2. 点击用于过滤搜索的图标。例如，您可以过滤搜索以省略注释或仅在注释中搜索。

  3. 选择一个显示的选项（如模块或目录）以限制搜索。此外，您可以选择“ 范围”选项，该选项为您提供搜索的预定义范围列表。

    例如，您可以将搜索范围限制为项目中的测试文件。

    您也可以创建自己的自定义范围，单击用于打开“ 范围”对话框的图标。

有关如何在搜索中使用正则表达式的信息，请参阅使用regex文档进行的搜索 。

搜索特定的文件类型

使用“ 文件掩码”选项将搜索范围缩小到特定的文件类型。您可以从列表中选择现有文件类型，添加新文件类型，或添加其他文件掩码语法以搜索具有特定模式的文件类型。

  1. 在“ 在路径中查找”对话框中，选择“ 文件掩码”复选框，然后从文件类型列表中选择所需的文件类型。

    WebStorm将其搜索范围限制为指定的类型。

  2. 如果在列表中找不到所需的文件类型，请在“ 文件掩码”字段中输入文件类型。

    例如，使用以下语法仅在json文件中搜索： *.json 。

除了*之外，还支持其他通配符 。如有必要，请使用逗号作为分隔符指定多个文件类型。

点击打开“ 查找用法”对话框。

要查看以前的“ 查找用法”操作的结果，请选择“ 编辑”|找到|最近从主菜单中查找用法 ，然后选择使用情况查询。

在项目中搜索用法

您可以在整个项目或您设置的范围中搜索符号的用法。WebStorm在“ 查找”工具窗口或弹出窗口中列出搜索结果，弹出窗口可用于快速导航。

  • 要在“ 查找”工具窗口中显示搜索结果，请将光标放在要查找其用法的符号上，然后选择“ 编辑”|找到|从主菜单中查找用法或按Alt + F7 。

    在“ 查找”工具窗口中，您可以使用“ 预览”区域检查找到用法的位置，查看方法的调用层次结构，字段的数据流等。

  • 要在弹出窗口中显示搜索结果，请将光标放在要查找其用法的符号上，然后选择编辑|找到|从主菜单中查找用法或按Ctrl + Alt + F7 。

    点击将搜索结果移动到“ 查找”工具窗口。

管理“查找用法”范围

  • 如果WebStorm未找到符号的任何用法，则会显示一条消息。

    点击或者按Ctrl + Shift + Alt + F7打开“ 查找用户”对话框并为搜索设置新范围，例如“ 打开文件”或“ 项目测试文件” 。

    要设置自定义范围 ，请单击 。完成新范围设置后，单击“ 查找” 。

禁用自动突出显示用法

将插入符号放在符号上时，IDE将突出显示当前文件中此符号的所有用法。使用F3和Shift + F3快捷方式在突出显示的符号之间导航。

  • 要禁用自动突出显示，请在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中转到“ 编辑器”|一般 ，并清楚在插入复选框中突出显示元素的用法 。

禁用自动突出显示时，将插入符号放在必要的符号上，然后按Ctrl + Shift + F7 。这将突出显示当前文件中符号的所有用法。

在省电模式下，禁用突出显示用法。
