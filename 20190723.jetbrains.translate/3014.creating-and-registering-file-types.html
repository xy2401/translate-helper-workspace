创建和注册文件类型

如果您正在处理WebStorm无法识别的文件类型（例如，如果它是内部开发的专有文件类型），或者您需要使用不受支持的语言编写代码，则可以创建自定义文件类型。

您可以通过定义关键字，注释，数字等的突出显示方案来配置IDE如何解析文件。您还可以将每种文件类型与扩展名相关联，以帮助IDE识别自定义格式的文件。

创建一个新的文件类型

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|文件类型 。

  2. 在“ 识别的文件类型”部分中，单击 ，指定新类型的名称，并提供说明。

  3. 在“ 语法突出显示”部分中，配置区分大小写，大括号匹配设置以及指定定义注释的方法：

    • 行注释 ：指定表示单行注释开头的字符。

    • 仅在行开头 ：如果行注释的开头位于行的开头，则它们将被识别为注释。

    • 阻止评论开始 ， 阻止评论结束 ：指定表示阻止评论开始和结束的字符。

    • 十六进制前缀 ：指定表示后续值为十六进制数字的字符（例如， 0x ）。

    • 数字后缀 ：指定表示使用哪个数字系统或单位的字符。后缀是一个尾随字符串（例如， e-3, kg ）。

    • 支持成对的大括号 ， 支持成对的括号 ， 支持成对的parens ， 支持字符串转义 ：选中这些复选框以突出显示成对的大括号，括号，括号和字符串转义。

  4. 在“ 关键字”部分中，您最多可以指定四个关键字列表。每个列表的关键字将在编辑器中以不同方式突出显示，并将自动完成。

  5. “ 忽略大小写”复选框指示自定义格式的文件中的语言是否区分大小写。

每组关键字都有自己的突出显示，您可以修改。为此，请在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中选择“ 编辑器”|配色方案|自定义 ，然后编辑关键字1， 关键字2，KEYWORD3和Keyword4性能。

注册新的文件类型关联

如果您打开具有已注册扩展名的文件，WebStorm将在编辑器中以静默方式打开此类文件。

如果在关联应用程序类别中打开的文件下注册了文件类型，则会在相关应用程序中打开该文件类型。例如，PDF文件在默认的PDF查看器中打开。

如果WebStorm无法识别文件类型，则会提示您选择是否要注册它，或在其关联的应用程序中打开此类文件。

更改文件类型关联

您可以将文件类型与其他扩展名相关联，也可以删除关联。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|文件类型 。

  2. 从“已识别的文件类型”列表中，选择要与其他扩展名关联的文件类型。

  3. 使用“已注册的模式”部分进行必要的更改。您可以添加新扩展，删除现有扩展或修改现有模式。

忽略文件和文件夹

在WebStorm中，有一个文件和文件夹列表，完全排除在任何类型的处理之外。开箱即用，此列表包括临时文件，与版本控制系统相关的服务文件，等等：

* .hprof ; * .pyc ; * .pyo ; * .rbc ; * .yarb ; *〜 ; 。DS_Store ; .git ; .hg ; .svn ; CVS ; __pycache__ ; _svn ; vssver.scc ; vssver2.scc ;

修改被忽略的文件和文件夹列表

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|文件类型 。

  2. 在“ 忽略文件和文件夹”列表中进行必要的更改。确保使用分号（ ; ）分隔条目。

  3. 应用更改并关闭对话框。
