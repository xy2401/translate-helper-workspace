创建和处理表单的运行时框架

要使表单正常工作，必须为其提供运行时框架。GUI表单的main（）方法负责创建和处理这种框架。

为表单创建main（）方法

  1. 打开以编辑要在其中创建main（）方法的表单的绑定类。

  2. 按Alt + Insert 。

  3. 在弹出菜单上，单击Form main（） 。以下方法添加到源代码中：

    public static void main （ String [] args ） { JFrame frame = new JFrame （ “<class name>” ）; 框架 。 setContentPane （ new < class name >（）。 contentPane ）; 框架 。 setDefaultCloseOperation （ JFrame 。EXIT_ON_CLOSE ）; 框架 。 pack （）; 框架 。 setVisible （ true ）; }
