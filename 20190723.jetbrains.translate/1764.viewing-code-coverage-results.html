查看代码覆盖率结果

查看代码覆盖率可帮助您检测未受模拟影响的源代码片段。

查看代码覆盖率结果

  1. 执行以下操作之一：

    • 使用coverage运行所需的类 ， 选择要显示的套件 ，并在编辑器中打开类。

    • 从主菜单中，选择Analyze |显示代码覆盖率数据 。

    • 按Ctrl + Alt + F6 。

  2. 查看覆盖率结果：

    • 在“ 项目”工具窗口中：

    • 在专用Coverage工具窗口中 ：

  3. 在编辑器中打开要探索的文件。

  4. 使用左侧装订线中的颜色指示器来检测未覆盖的代码行。

  5. 要查看线路被击中的次数，请单击装订线区域中的线条。

    打开的弹出窗口显示插入符号行的统计信息。对于具有条件的行，弹出窗口还提供统计信息：

    使用以下工具栏按钮：

    • ：跳转到下一组/前一组已覆盖或未覆盖的线。

    • ：查看覆盖插入符号行的JUnit测试。

      覆盖插入符号线的测试显示在弹出窗口中：

      • 此按钮仅在“ 跟踪”模式下可用，并且选中“ 每个测试覆盖范围”复选框。

      • 固定时，此弹出窗口将转换为“ 查找”工具窗口 。

    • ：在弹出窗口中显示当前类的字节代码：

      • 此按钮仅在启用了与产品捆绑的字节代码查看器插件时可用。

      • 固定时，此弹出窗口将转换为字节代码查看器。

    • ：打开“ 颜色方案”设置，您必须在其中选择节点“ 线覆盖” 。

    • ：隐藏覆盖信息。

  6. 创建缺少的测试，如创建测试一节中所述。
