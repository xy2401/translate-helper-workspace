创建测试

Alt + Enter - 创建测试

Ctrl + Shift + T - 创建新测试

项目

描述

测试库

使用此下拉列表选择要使用的测试框架。

固定

当缺少所选测试框架的库时，此按钮可用。单击此按钮可将所选测试框架的jar存档添加到模块依赖项和库列表中。

班级名称

输入要生成的测试类的名称，或接受默认值。

超

对于JUnit3，超类junit.framework.自动建议TestCase 。对于其他受支持的框架，此字段为空。

目的地包裹

指定将存储生成的测试类的包的名称。您可以从最近的历史记录下拉列表中选择所需的包，或直接在文本字段中键入它。这样做，如果不存在具有指定名称的包，则将自动创建。点击使您可以选择项目中的一个现有包。

生成
设置（）/ @前
TEARDOWN（）/ @后

选中这些复选框，将测试夹具和注释的存根方法包含到生成的测试类中。

显示继承的方法

选择此选项可显示所有方法，包括继承的方法。

生成测试方法

在表格中，选择要为其生成测试的方法旁边的复选框。
