Running and Debugging Grails Applications

This feature is only supported in the Ultimate edition.

In IntelliJ IDEA, you can launch Grails applications using the regular procedures, with the dedicated the Grails Application run/debug configuration:

  • Running Applications

  • Do not suspend code execution

Note that IntelliJ IDEA enables you to debug *.gsp files: you can set breakpoints at the lines of the views, examine variables, and evaluate expressions.

Besides that, you can execute Groovy scripts that exist in Grails applications. Observe results in the Run and Debug tool windows.
