运行覆盖范围

PyCharm提供了一个专用操作，允许您使用代码覆盖率测量执行运行。根据在“设置/首选项”对话框的“ 覆盖”页面中选择的选项处理代码覆盖率数据。

运行代码覆盖率测量

  1. 执行以下操作之一：

    • 在编辑器中打开所需文件，然后在上下文菜单中选择Run <name> with coverage 。在使用coverage运行测试时，请注意您可以运行整个测试类或每个单独的测试方法，具体取决于插入符号位置。

    • 选择所需的运行/调试配置，然后在主菜单上选择Run |运行带有coverage的<run / debug configuration name> 。

    • 在主工具栏上，单击 。这将启动所选的运行/调试配置。

  2. 如果在“设置/首选项”对话框的“ 覆盖率”页面中选中了“ 将覆盖率应用于编辑器之前显示选项 ”复选框，请选择是要替换活动覆盖套件，还是将收集的数据添加到活动套件，或者不是不想申请覆盖数据。您也可以选择稍后跳过此对话框。

    如果选择了任何其他选项，将以静默方式执行相应的操作。

  3. 在Coverage Tool窗口中 浏览收集的coverage数据 。
