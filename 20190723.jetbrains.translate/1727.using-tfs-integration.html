IntelliJ IDEA 2019.1帮助

TFS

此功能仅在Ultimate版本中受支持。

启用TFS集成后，您可以从IntelliJ IDEA内部执行基本TFS操作。

IntelliJ IDEA支持TFS至TFS 2015。

本节中提供的信息假定您熟悉TFS版本控制系统的基础知识。

在你开始之前

  • TFS集成不需要独立的TFS客户端。您只需要Team Foundation Server中的一个帐户即可。

  • 确保已启用TFS集成插件。

    此插件与IntelliJ IDEA捆绑在一起并默认启用。如果没有，请启用插件 。

  • 确保为整个项目根目录或特定目录启用了TFS集成。有关更多详细信息，请参阅启用版本控制 。

TFS支持

  • 启用TFS与IntelliJ IDEA集成后， TFS项目将显示在VCS菜单上，并显示在编辑器和项目工具窗口的上下文菜单中。

  • TFS控件下文件夹中的文件将根据其状态突出显示。有关文件状态突出显示约定，请参阅文件状态要点 。

  • 修改结果显示在“ 版本控制”工具窗口中 。

  • 使用TFS集成时，打开“ 版本控制”工具窗口会很有帮助。“ 控制台”选项卡显示以下数据：

    • 根据您通过IntelliJ IDEA用户界面指定的设置生成所有命令。

    • 有关执行生成的TFS命令的结果的信息消息。

    • 错误消息。

  • TFS与IntelliJ IDEA集成允许您在提交更改时使用新变更集解决/关联工作项。为此，您可以访问已保存的查询并浏览工作项树。

  • 从TFS存储库签出
