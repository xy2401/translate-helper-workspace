从Java代码生成WSDL文档

此功能仅在Ultimate版本中受支持。

Web服务的可用功能，访问它们的端口，可接受的请求格式，生成的响应的格式等反映在Web服务WSDL描述符中 ，该描述符通常在Web服务部署期间在服务器上生成。Web服务客户端开发的一个主要部分是实现对服务的请求的生成，并根据WSDL描述符设置解析来自它的响应。

假设您已经开发了Web服务，并希望在部署Web服务之前启动客户端开发，即在服务器上生成WSDL描述符之前。使用IntelliJ IDEA，您可以在部署之前生成它。

从Java代码创建WSDL描述符

  1. 在编辑器中选择所需的类名。

  2. 选择工具|网络服务|在主菜单上从Java代码生成WSDL，或选择Web Services |从上下文菜单中生成WSDL From Java Code 。

  3. 在打开的“ 从Java生成WSDL”对话框中，指定以下内容：

    • Web服务的名称和URL地址。

    • 访问Web服务的公共操作时使用的协议和编码样式。

    • 类型信息，包括名称，操作，参数和包含Web服务接口的数据。

    准备好后 ，单击“ 确定” 。
