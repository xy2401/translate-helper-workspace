描述

输入目录

在此文本框中，指定具有要为其生成文档的源的目录的完全限定路径。

输出目录

在此文本框中，键入将存储生成的文档的目录的完全限定路径。或者，单击打开对话框 ，您可以在其中选择所需的路径或单击创建新目录 。

私人的

选中此复选框可将private属性设置为true 。

使用

选中此复选框可将use属性设置为true 。

源包

使用按钮和创建和管理包含源文件的包列表。

在浏览器中打开生成的文档

如果希望IntelliJ IDEA在浏览器中自动打开生成的文档，请选中此复选框。
