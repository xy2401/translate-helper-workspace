Vaadin-Maven项目入门

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA允许您使用Vaadin原型创建和管理Maven项目。

  • 使用Vaadin原型创建Maven项目
  • 使用Maven导入外部Vaadin项目
  • 配置运行/调试设置并运行应用程序

使用Vaadin原型创建Maven项目

  1. 打开Project Wizard，在Maven下，选择Create from archetype复选框，然后单击Add archetype 。

  2. 在打开的对话框中，输入maven GroupId，ArtifactId和Version。

  3. 单击“ 确定”后 ，Vaadin原型将添加到可用原型列表中。单击下一步 。

  4. 在向导的下一页上，输入项目的maven信息，然后单击“ 下一步” 。

  5. 在向导的下一页上，输入项目的maven设置，然后单击“ 下一步” 。

  6. 在向导的下一页上，输入项目的信息，然后单击“ 完成” 。

IntelliJ IDEA使用pom.xml创建maven项目。IntelliJ IDEA还认识到这是一个Web应用程序，并在源目录中创建一个webapp。

使用Maven导入外部Vaadin项目

您可以使用VCS导入外部Vaadin项目，并从例如Git中检出您的项目。在这种情况下，IntelliJ IDEA会自动转换您的项目结构，检测必要的框架并下载适当的依赖项。

  1. 在“项目向导”中，单击“ 从版本控制中签出”，然后从下拉列表中选择“ Git” 。

  2. 指定要从中克隆项目的Git存储库，父目录位置和父目录的名称。单击克隆 。

    IntelliJ IDEA将git项目复制到本地目录，识别POM文件，这意味着它识别出这是一个带有Vaadin框架的Maven项目，并使用所有必要的配置导入项目。

  3. 检查项目结构以确保下载所有必需的依赖项。

    框架是自动配置的，因此您可以排除检测到的框架。

配置运行/调试设置并运行应用程序

IntelliJ IDEA允许您快速配置项目的运行/调试设置。

  1. 在主菜单上，选择“运行”|编辑配置 。

  2. 点击要从列表中添加新配置，请为配置指定适当的参数，然后按确定 。

  3. 在主菜单上按运行应用程序。

您可以在默认浏览器http://localhost:8080/查看输出。
