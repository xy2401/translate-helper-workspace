介绍

IntelliJ IDEA提供数据流分析功能，可帮助您进行代码考古 - 更好地理解继承的项目代码，解释代码的复杂部分，查找源代码中的瓶颈等。

具体来说， Dataflow到/来自这里的功能允许您：

  • 查看分配给变量的值的来源。

  • 找出变量可能具有的所有可能值。

  • 找出表达式\ variable \ method参数可以流入的位置。

  • 显示可能出现潜在NullPointerException的位置。

如果要追溯传递给插入符号参数的值，可以使用Analyze |创建源代码的切片视图。 Dataflow to Here命令。而且，使用Analyze | Dataflow from Here命令可以找到表达式可以流入的位置。每个数据流分析的结果显示在Analyze Dataflow工具窗口的专用选项卡中。

分析数据流

要分析符号的数据流：

  1. 打开所需的文件进行编辑，请参阅查找插入符号，编辑的行或编辑的位置 。

  2. 将插入符号放在要分析的符号上（expression \ variable \ method参数）。

  3. 在主菜单上或上下文菜单中，选择Analyze |数据流到这里或分析|来自此处的数据流取决于您的目的。

  4. 指定分析范围，并选择是否要忽略来自测试代码的所有值。

  5. 单击确定 。在专用的Analyze Dataflow工具窗口中查看分析结果。
