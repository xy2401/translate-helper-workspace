介绍

在探索框架及其内容时，您可能希望自定义数据的显示方式。本节介绍如何设置此类选项。

自定义线程视图

您可以根据需要组织列表中显示线程的方式。

自定义“线程”视图

  1. 右键单击“ 帧”选项卡中的任意位置，然后选择“ 自定义线程视图” 。

  2. 指定查看选项。这里将详细介绍它们。

自定义数据视图

您可以根据需要组织数据在“ 变量”选项卡中的显示方式。

自定义数据视图

  1. 右键单击“ 变量”选项卡中的任意位置，然后选择“ 自定义数据视图” 。

  2. 指定查看选项。这里将详细介绍它们。

自定义类型渲染器

您还可以指定自己的类型渲染器而不是默认类型。它们使您能够自定义对象在调试器中的显示方式，提供“面向逻辑”的数据表示，而不是默认情况下面向结构的表示。

渲染对象

渲染对象视图

  1. 打开“设置/首选项”对话框（ Ctrl + Alt + S ），单击“ 调试器”并选择“ 类型渲染器” 。

  2. 点击创建一个新的渲染器。

  3. 指定渲染器名称，应用它的对象类以及渲染时要使用的表达式。有关渲染选项的详细信息，请参阅选项说明 。

  4. 定义展开节点的外观。

  5. 单击确定 。

您还可以在“ 调试”工具窗口的任何视图中右键单击要为其设置类型渲染器的对象，然后选择“ 视图为|”。在打开的对话框中创建并指定此对象类型的自定义渲染器。

禁用自定义类型渲染器

禁用自定义类型渲染器

  1. 打开“ 设置”对话框（ Ctrl + Alt + S ），单击“ 调试器” ，然后选择“ 类型渲染器” 。

  2. 在列表中选择要禁用的类型渲染器，然后清除列表中其名称旁边的复选框。

    即使禁用了类型渲染器，也可以使用“ 查看为”选项临时切换到程序时切换到该类型。

  3. 单击确定 。

在类型渲染器之间切换

在调试工具窗口中单步执行应用程序时，可以在渲染器方案之间临时切换。

在类型渲染器之间切换

  1. 在“ 调试”工具窗口的“ 变量”或“ 监视”选项卡中右键单击对象实例。

  2. 在上下文菜单中，单击“ 查看方式” ，然后从适用的类型渲染器列表中选择渲染器。
