选择路径对话框

使用此对话框选择所需的文件或文件夹。

对话框名称和可用功能取决于您当前正在执行的任务（通常禁用不适当的功能）。例如，根据具体情况，您可以只选择一个项目或多个项目。可能存在您可以选择一个或多个文件夹但无法选择文件或文件等的情况。

  • 主要功能
  • 路径字段
  • 使用文件浏览器中的拖放操作

主要功能

通过工具栏图标（显示在“ 图标”列中）可以访问此对话框中可用的大多数功能。或者，您可以使用上下文菜单命令（通过右键单击树中的项目;在“ 命令”列中列出）或键盘快捷键（“ 快捷方式”列）来访问。

隐藏或显示路径

按Ctrl + P

使用此命令或快捷方式隐藏或显示路径字段 。（该命令位于对话框右侧的工具栏上，并显示为超链接。）

路径字段

路径字段（如果未隐藏）位于工具栏下方。此字段显示树中所选项目的路径。

通过使用此字段中提供的路径自动完成功能，您可以快速浏览文件系统以选择所需的文件或文件夹。

要激活路径自动完成，请将光标放在该字段中，然后按Ctrl + Space 。开始输入。将出现一个弹出窗口，显示当前目录的内容。从弹出窗口中选择一个项目。继续键入并选择，直到选中所需的项目。

使用向右按钮显示最近条目的历史列表。

使用文件浏览器中的拖放操作

如果将相应的项目从文件浏览器（Explorer，Finder等）拖动到显示树的区域，则可以快速找到并选择所需的文件或文件夹。
