项目

描述

在圆括号之前

  • 数组初始化括号

    选中此复选框可在数组声明中的左括号前插入空格，例如：

    $ array = array （ 0 => “0” ， 1 => “1” ）;

    如果清除该复选框，则不会插入空格，代码如下所示：

    $ array = array （ 0 => “0” ， 1 => “1” ）;

围绕运营商

  • 级联

    选中此复选框以在连接运算符（ . ）之前和之后插入空格，以便代码如下所示：

    回声 “结果是” 。 $ i ;

    清除该复选框后，连接运算符周围不会插入空格，因此代码如下所示：

    回声 “结果是” 。 $ i ;

  • 对象访问运算符（ -> ）

    选中此复选框以在对象访问运算符（ -> ）之前和之后插入空格，因此代码如下所示：

    $ obj - > foo （） - > bar （）;

    清除该复选框后，对象访问运算符周围不会插入空格，因此代码如下所示：

    $ obj - > foo （） - > bar （）;

  • 空结合运算符（??）

    选中此复选框以在空合并运算符之前和之后插入空格（ ?? ）所以代码如下：

    foo （） ?? bar （）;

    清除该复选框后，空的合并运算符周围不会插入空格，因此代码如下所示：

    foo （） ??bar （）;

  • 声明声明中的作业

    选中此复选框以在declare语句中的赋值运算符之前和之后插入空格，以便代码如下所示：

    declare （ strict_types = 1 ）;

    清除该复选框后，在赋值运算符周围不会插入空格，因此代码如下所示：

    declare （ strict_types = 1 ）;

左支撑之前

  • 类左支撑

    选中此复选框可在类声明和类声明中的左大括号之间插入空格：

    class Class1 { function Foo （） }

    清除该复选框后，不会插入任何空格：

    class Class1 { function Foo （） }

    §选择或清除复选框仅在类声明中的大括号放置设置为“ 环绕和大括号”选项卡上的“行尾”时才相关。

内

选中此部分中的复选框可在所选代码构造中自动插入空格。

  • 括号

    如果选中该复选框，则始终插入空格。如果清除该复选框，则永远不会插入空格。

  • 围绕变量/表达式的括号

    仅当括号包含变量或表达式时，选中此复选框才能在重新格式化期间将IntelliJ IDEA插入括号内。仅当您尚未通过选择Spaces选项卡中Within节点下的Brackets复选框配置强制插入括号内的空格时，此设置才会影响重新格式化。

    此选项可帮助您保持代码符合WordPress PHP编码标准 。

    如果清除此复选框，则无论括号内的内容类型如何，空格插入策略都取决于括号设置。

在三元运算符中（ ?: :)

选择此部分中的复选框以在前后自动插入空格? ，之前和之后:和之间? 和:在三元（条件）运算符中 。

其他

  • 打字后

    选中此复选框后，IntelliJ IDEA会在演员 $fst = (string) $foo;括号后自动插入空格： $fst = (string) $foo;

    如果清除该复选框，则不会插入任何空格，并且铸造变量会粘贴到演员表： $fst = (string)$foo;

空规则

使用此区域可以创建新的匹配规则或编辑现有规则。您可以从以下过滤器中进行选择：

  • 类型 - 使用此过滤器为您的规则选择类或方法。
    请注意，单击类型关键字两次会使条件无效。

  • 名称 - 使用此字段指定规则中的条目名称。此筛选器仅匹配条目名称，例如字段名称，方法名称，类名称等。该过滤器支持正则表达式并使用标准语法 。匹配是针对整个名称执行的。

  • 订单 - 使用此列表选择规则的排序顺序。当多个元素使用相同的匹配规则时，此选项很有用。在这种情况下，选择Keep order将保持与重新排列之前设置的顺序相同，并选择Order by Name将按名称对具有相同匹配规则的元素进行排序。

  • 别名 - 此选项显示在“ 规则别名定义”对话框中定义的别名 。您可以删除不需要的那些。
