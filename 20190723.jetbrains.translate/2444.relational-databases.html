数据库和SQL

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

以下内容仅在安装并启用数据库工具和SQL插件时有效！

用于处理数据库和SQL的PyCharm功能包括：

  • 与最流行的数据库管理系统集成，如Oracle ， PostgreSQL ， MySQL ， SQL Server等。为了能够使用您的数据库，您应该将它们定义为数据源。请参见连接到数据库 。

  • 用于管理数据库中数据结构的数据库工具窗口（ 查看|工具Windows |数据库 ）。请参阅使用数据库工具窗口 。

  • 数据库控制台，允许您编写和执行SQL语句以及分析和修改检索到的数据（在数据库工具窗口中按Ctrl + Shift + F10 ）。请参阅使用数据库控制台 。

  • 数据编辑器，提供用于处理表数据的GUI（ 数据库工具窗口中的F4 ）。请参阅使用表格 。

  • 数据库控制台和编辑器中的SQL代码生成和编辑功能，例如

    • 预定义的代码片段（也称为实时模板），例如CREATE TABLE ， SELECT ， INSERT ， UPDATE和其他语句（ Ctrl + J ）。

    • 自动完成和突出显示SQL关键字以及表和列名称。

    • 列的数据类型提示（ Ctrl + P ）。

    支持标准化和DBMS特定的SQL方言。

  • 数据编辑器和数据库控制台工具窗口中的表的结构视图，以及打开的数据库控制台和SQL文件（ Ctrl + F12 ）。见例如 。

  • 数据库对象和表格单元格的快速文档视图（ Ctrl + Q ）。见例如 。

  • 导航功能，例如

    • 从表或列引用到其定义： Ctrl + B.

    • 到数据库工具窗口中的表或列的视图： Alt + F1 |数据库视图 。

    • 通过导航栏： Alt + Home 。

    • 通过切换器： Ctrl + Tab 。

  • 数据库图表（ 数据库工具窗口中的Ctrl + Alt + U或Ctrl + Shift + Alt + U ）。
