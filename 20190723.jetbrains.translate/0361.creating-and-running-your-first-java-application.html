创建，运行和打包Java应用程序

在本教程中，您将学习如何使用编码辅助功能和补充工具创建您的第一个Java应用程序，这些功能旨在使您的工作自动化并帮助您更快地完成开发任务。

创建一个新项目

安装JDK

要在IntelliJ IDEA中开发Java应用程序，您需要Java SDK（JDK）。它是一个软件包，包含库，用于开发和测试Java应用程序的工具（开发工具），以及用于在Java平台上运行应用程序的工具。

  1. 打开jdk.java.net网站。在那里，您将找到适用于Windows，macOS和Linux的Oracle OpenJDK二进制文件，如.tar.gz和.zip档案。

  2. 选择并下载必要的JDK版本。如果您不确定需要哪个版本，请选择最新的稳定版本。

  3. 解压缩归档文件并将带有Java的文件夹放到计算机上的Java主目录中。Windows上的默认路径： C：\ Program Files \ Java ，位于macOS： / Library / Java / JavaVirtualMachines上 。

    在Linux上，Java的默认位置可能因您使用的分发而异。

创建一个新的Java项目

  1. 启动IntelliJ IDEA。如果“欢迎”屏幕打开，请单击“ 创建新项目” 。

    否则，从主菜单中选择File |新的|项目 。

  2. “ 新建项目”向导打开后，请确保在左侧列表中选择了Java 。

  3. 从Project SDK列表中，选择要在项目中使用的JDK。

    如果列表为空：

    • 单击“ 新建”并指定Java主目录的路径。

    • 选择Java目录，例如， jdk-11.0.2.jdk 。

    • 单击打开 。

  4. 在下一步中，您可以使用预定义的main()方法创建Java类。

    在本教程中，我们将从头开始执行所有操作，因此请单击“ 下一步”跳过此步骤。

  5. 为项目命名，例如： HelloWorld 。

  6. 如有必要，请更改默认项目位置 - 计算机中存储项目的文件夹 - 然后单击“ 完成” 。

创建一个包和一个类

包用于将属于同一类别或提供类似功能的类分组在一起，用于构建和组织具有数百个类的大型应用程序。

项目工具窗口
（ Alt + 1 ）显示应用程序的结构并帮助您浏览项目。

在Java中，有一个命名约定 ，在命名包和类时应该遵循这些约定 。

  1. 在Project工具窗口中，选择src文件夹，按Alt + Insert ，然后选择Java Class 。

  2. 在“ 名称”字段中，键入com.example.helloworld.HelloWorld并单击OK 。

    IntelliJ IDEA创建com.example.helloworld包和HelloWorld类。

与文件一起，IntelliJ IDEA自动为您的班级生成一些内容。在这种情况下，IDE已插入package语句和类声明。

这是通过文件模板完成的。根据您创建的文件的类型，IDE会插入预期在该类型的所有文件中的初始代码和格式。有关如何使用和配置模板的更多信息，请参阅文件和代码模板 。

编写代码

使用实时模板添加main（）方法

由于IntelliJ IDEA已自动生成包语句和类声明，因此您可以继续执行下一步并添加main()方法的代码。

  1. 将插入符号放在开始括号后面的类声明字符串{并按Shift + Enter 。

    与Enter相反， Shift + Enter在不破坏当前行的情况下开始新行。

  2. 输入p并按Ctrl + J. IntelliJ IDEA显示以p开头的所有可用实时模板。

  3. 找到psvm - main() method declaration模板，然后按Tab键 。

实时模板是可以插入代码的代码片段。 psvm就是其中一个片段。通常，实时模板包含您最常使用的代码块。使用它们可以节省一些时间，因为您不必反复键入相同的代码。

有关在何处查找预定义的实时模板以及如何创建自己的模板的更多信息，请参阅实时模板 。

您还可以使用实时模板添加语句。键入s ，按Ctrl + J ，然后选择sout - Prints a string to System.out 。

使用代码完成添加语句

使用实时模板插入方法后，IntelliJ IDEA会自动将插入符号放在下一行。

  1. 输入Sy 。IntelliJ IDEA建议您为您竞争代码，并根据您输入的内容向您显示选项列表。此功能称为基本代码完成。

  2. 从建议列表中，按Ctrl +选择System (java.lang) 。 。

  3. 输入o并按Ctrl +。再次。IDE将插入out后跟一个点。

  4. 输入p然后查找并选择println（String x） 。IntelliJ IDEA显示可在当前上下文中使用的参数类型 。此信息仅供参考。

  5. 输入" 。第二个引号自动插入，插入符号位于引号之间。输入Hello, World!

基本代码完成分析当前插入符号位置周围的上下文，并建议您在键入时如何完成代码的选项。IntelliJ IDEA会自动显示完成列表，但您可以通过按Ctrl + Space手动打开它。

有关不同完成模式的信息，请参阅代码完成 。

构建和运行应用程序

现在您拥有符合的代码，因此您可以运行它。使用main()方法的类可以使用绿色箭头标记直接从编辑器运行（ ）在左边的排水沟里。

  1. 点击在装订线中，在弹出窗口中选择Run'HelloWorld.main（）' 。IDE开始编译代码。

  2. 编译完成后，将在屏幕底部打开“ 运行”工具窗口。

    工具窗口显示程序输出： Hello, World! 。最后一行显示该进程已成功退出。您还可以看到IDE用于运行该类的命令。

单击“运行”时 ，IntelliJ IDEA会创建一个执行一系列操作的特殊运行配置。首先，它构建您的应用程序。在这个阶段，javac（作为JDK的一部分下载的主要Java编译器）将源代码转换为字节码。

一旦javac完成编译，它就会将编译后的字节码放到out目录中。您可以在“ 项目”工具窗口中看到它，它以黄色突出显示。

之后，Java虚拟机（JVM） - 也是JDK的一部分 - 运行字节码。

自动创建的运行配置是临时的，但您可以修改并保存它们。

如果要重新打开“ 运行”工具窗口，请按Alt + 4 。

将应用程序打包在JAR中

代码准备就绪后，您可以将应用程序打包到Java归档文件（JAR）中，以便与其他开发人员共享。现成的Java归档文件称为工件 。

为JAR创建工件配置

  1. 从主菜单中，选择File |项目结构 （ Ctrl + Shift + Alt + S ）并单击工件 。

  2. 点击 ，指向JAR并从具有依赖项的模块中选择。

  3. 在Main Class字段的右侧，单击并在打开的对话框中选择HelloWorld（com.example.helloworld） 。

    IntelliJ IDEA创建工件配置并在“ 项目结构”对话框的右侧部分显示其设置。

  4. 应用更改并关闭对话框。

构建JAR工件

  1. 从主菜单中，选择Build |构建工件 。

  2. 指向HelloWorld：jar并选择Build 。

    如果你现在查看out/artifacts文件夹，你会在那里找到你的JAR。

运行打包的应用程序

要确保正确创建JAR工件，您可以运行它。

使用“ 查找操作” （ Ctrl + Shift + A ）搜索整个IDE中的操作和设置。

为打包的应用程序创建运行配置

要运行打包在JAR中的Java应用程序，IntelliJ IDEA允许您创建专用的运行配置。

  1. 按Ctrl + Shift + A ，键入编辑配置 。找到操作后，双击它。

  2. 在“ 运行/调试配置”对话框中，单击“确定” 并选择JAR应用程序 。

  3. 命名新配置： HelloWorldJar 。

  4. 在Path to JAR字段中，单击并指定计算机上JAR文件的路径。

  5. 在启动之前 ，单击 ，选择Build Artifacts | HelloWorld：打开的对话框中的jar 。

    这样做意味着每次执行此运行配置时都会自动构建HelloWorld.jar 。

运行配置允许您定义运行应用程序的方式，以及哪些参数和选项。您可以为同一个应用程序提供多个运行配置，每个配置都有自己的设置。

执行运行配置

  • 在工具栏上，单击在运行配置选择器的右侧。或者，如果您喜欢快捷方式，请按Shift + F10 。

    和以前一样，“ 运行”工具窗口会打开并显示应用程序输出。
