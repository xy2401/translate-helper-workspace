创建事件和事件处理程序类

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA可以为GWT事件类和GWT事件处理程序接口生成存根。

GWT事件类和事件处理程序接口只能在客户端部分中创建。

创建GWT事件和事件处理程序

  1. 在项目视图中 ，右键单击客户端程序包。

  2. 从上下文菜单中，选择New | Google Web Toolkit | GWT事件和处理程序类 。

  3. 在打开的Create GWT Event and Handler Classes对话框中，指定要覆盖的调用者的事件类，事件处理程序接口和抽象“回调”方法的名称。

    实际上，您不需要手动指定所有三个名称。只要在“ 事件类名称”文本框中键入事件类名称 ，IntelliJ IDEA就会自动完成处理程序接口和“回调”方法的名称，并分别在“处理程序” 类名称和“ 处理方法名称”文本框中显示建议。这可确保生成的存根将根据GWT Google API库生成。

  4. 单击“ 确定”后 ，IntelliJ IDEA将为事件类和事件处理程序接口生成存根，并在编辑器中打开它们。根据需要填充生成的存根。
