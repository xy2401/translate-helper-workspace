提取方法

Extract Method重构允许您获取可以组合在一起的代码片段，将其移动到单独的方法中，并通过调用方法替换旧代码。

提取方法时，需要检查变量。如果有一个输出变量，则将其用作提取方法的返回值。如果存在多个输出变量，则可能不会应用“ 提取方法”重构，并显示错误消息。

在这种情况下，有几种解决方法允许Extract Method工作。例如，您可以引入包含所有输出值的特殊数据类。

Extract Method重构具有以下限制：

  • 重构不适用于自动模式下的多个输出值。在应用重构之前，您必须更改代码。

  • 重构不适用于有条件地从包含方法返回的代码片段，并且不会放在它的末尾。

要反转Extract Method重构，请按Ctrl + Alt + N以调用Inline重构 。

提取方法

要提取方法，请按照下列步骤操作

  1. 在编辑器中，选择要转换为方法或函数的代码块。

    形成方法的代码片段不一定必须是一组语句。它也可能是代码中某处使用的表达式。

  2. 在主菜单或选择的上下文菜单上，选择Refactor |提取|方法或按Ctrl + Alt + M.

  3. 在打开的“ 提取方法”对话框中，指定新功能的名称。

  4. 在“ 参数”区域中，执行以下操作：

    • 通过选择/清除相应的复选框，指定要作为方法参数传递的变量。

    • 通过双击相应的参数行并输入新名称，重命名所需的参数。

  5. 在“ 签名预览”窗格中检查结果，然后单击“ 确定”以创建所需的功能。

    选定的代码片段将替换为函数调用。
