此对话框是一个包管理器，它启用以下功能：

  • 查看远程存储库中可用的包

  • 在本地安装选定的包。

  • 添加和删除存储库

项目

描述

搜索字段

在此输入搜索字符串。这样，包列表缩小以显示仅包含输入字符串的包名称。

  • ：单击此按钮以使用搜索历史记录。

  • ：单击此按钮可清除搜索字段。

包

此区域显示驻留在可访问远程存储库中的软件包列表。对于每个包，您可以查看其名称和存储库。其他信息（说明和版本）显示在右侧。

描述

对于任何选定的包，此字段显示简要说明，最新版本，作者姓名，网站和电子邮件地址。

指定版本

如果选中此复选框，则将下载并安装指定版本的软件包。如果未选中此复选框，则将安装最新版本。

选项

选中此复选框以输入pip install命令的选项。

可以通过在命令行中键入pip install - help来探索可能的选项。

安装包

单击此按钮可安装选定的包。

管理存储库

单击此按钮以打开“ 管理存储库”对话框。默认情况下，只有pypi存储库可用。无法删除此存储库。
可以添加更多存储库，并删除选定的存储库。
