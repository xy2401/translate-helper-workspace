从命令行打开文件

您可以从命令行打开任意文件，文件夹或项目以在PyCharm中进行编辑，也可以选择在打开文件后指定插入符应该放在哪里。

  • 在命令行中，运行以下命令：
    <PyCharm> [ - line <number> ]路径

    或者，您可以使用以下语法：

    <PyCharm>路径：<number>

命令选项如下：

选项

描述

<PyCharm>

特定于平台的PyCharm启动器。

在macOS和Linux上，建议使用在PyCharm安装期间创建的命令行启动程序，或者通过选择Tools |在PyCharm主菜单上创建命令行启动器 。请参阅除了在PyCharm中工作之外，您可以从命令行执行以下操作：有关详细信息。

在Windows上，建议将PyCharm可执行文件的Path添加到Path环境变量中。

路径

文件的路径或要打开的文件夹。

  • 如果文件或文件夹属于现有项目，则将打开此项目。

  • 如果该文件不属于任何项目，则将自动创建该文件的临时项目。

  • 如果该文件夹不属于任何项目，则将在此文件夹中创建一个新项目。

--line <number>

（可选）打开文件后插入符应在其上放置的行号。

例子

句法
pycharm.exe --line <number>路径
替代语法
pycharm.exe路径：<number>
例子
pycharm.exe --line 3 C： \ M yProject \ s cripts \ n umbers.js pycharm.exe C： \ M yProject \ s cripts \ n umbers.js：3
句法
pycharm --line <number>路径
替代语法
pycharm路径：<number>
例
pycharm --line 3 ~ / MyProject / scripts / numbers.js pycharm~ / MyProject / scripts / numbers.js：3
