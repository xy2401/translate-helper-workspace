更改类签名对话框

重构|更改签名
按Ctrl + F6

使用“ 更改类签名”对话框可以执行“ 更改类签名”重构。

使用可用控件管理正式类型参数（请参阅下表）。您可以添加和删除参数以及更改其顺序。

单击“ 重构”立即执行重构。单击“ 预览”以在实际执行重构之前查看潜在的更改。（这些将显示在“ 查找”工具窗口中 。）

图标

捷径

描述

Alt +插入

使用此图标或快捷方式添加参数。分别在“ 名称”和“ 默认值”字段中指定参数名称和默认类型。

Alt + Delete键

使用此图标或快捷方式删除所选参数。

Alt +向上

使用此图标或快捷方式将所选参数在列表中向上移动一行。

Alt +向下

使用此图标或快捷方式将所选参数在列表中向下移动一行。
