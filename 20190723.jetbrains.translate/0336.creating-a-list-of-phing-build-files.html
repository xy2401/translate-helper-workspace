创建Phing构建文件列表

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

在这个部分：

  • 介绍
  • 访问Phing Buide工具窗口
  • 管理构建文件列表

介绍

正确的Phing构建文件是一个xml文件，其根元素为<project> 。构建文件在Phing Build工具窗口之外创建，通常存储在PHP项目根文件夹下。了解有关在Phing入门中编写Phing构建文件的更多信息。

访问Phing Buide工具窗口

  1. 在编辑器中打开Phing构建文件或在“ 项目”工具窗口中选择它，然后从选择的上下文菜单中选择“ 添加为Phing构建文件 ”。

  2. 选择查看|工具窗口| Phing Build在主菜单上。在编辑器或“项目”工具窗口中通过Phing构建文件的上下文菜单打开工具窗口后，可以访问该工具窗口。

管理构建文件列表

要配置构建文件列表，请执行以下常规步骤

  1. 打开Phing Build工具窗口。

  2. 要将构建文件添加到列表，请单击“ 添加”按钮在工具栏上，在Select Phing Build File对话框中选择所需的.xml构建文件。

  3. 要从列表中删除构建文件，请选择该文件，然后单击“ 删除”按钮在工具栏上。

  4. 要导航到构建文件的源代码，请选择所需的文件，然后在所选内容的上下文菜单中选择“ 跳转到源 ”。
