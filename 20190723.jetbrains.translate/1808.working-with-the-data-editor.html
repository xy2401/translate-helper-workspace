使用表格

此功能仅在Ultimate版本中受支持。

在关系数据库中，表是数据存储的基本单元。表中的数据存储在一个单元格中，该单元格是垂直列和水平行的交集。该表具有指定的列数，但可以包含任意数量的行。使用IntelliJ IDEA，您可以使用表执行数据操作和数据定义操作。

SQL的代码样式

更改代码样式设置

自定义SQL代码的格式规则：对齐，换行和缩进。

  1. 单击文件|设置|编辑|代码风格| SQL 。

  2. 在可用选项卡上选择所有必需的代码样式设置。

  3. 单击“ 应用”

应用规则时，IntelliJ IDEA会更新SQL代码并突出显示“ 预览”窗格中的所有已修改部分。

为方言创建代码样式

  1. 打开设置|编辑|代码风格| SQL 。

  2. 单击齿轮图标（ ），然后选择复制 。

  3. 键入新样式的名称，然后按Enter键 。

  4. 选择所有必需的代码样式设置，然后单击“ 应用” 。

配置每个数据源的代码样式

为方言创建样式后，可以将此样式应用于使用此方言的数据源。

  1. 右键单击数据源，然后选择“ 属性” 。

  2. 单击“ 选项”选项卡。

  3. 在“ 代码样式”下拉菜单中，选择要应用的样式。

  4. 单击“ 应用”
