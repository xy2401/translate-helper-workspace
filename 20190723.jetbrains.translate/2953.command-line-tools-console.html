命令行工具控制台工具窗口

虽然WebStorm为运行命令提供了此工具窗口，但强烈建议您使用内置本地终端 。

工具|运行命令

N / A

只有在安装并启用命令行工具支持插件时，工具窗口才可用，如管理插件中所述 。

工具窗口标有图标 。使用此工具窗口的选项卡在命令行中键入和运行命令，并验证.xml描述符的结构一致性。

从“ 设置/首选项”对话框的“ 命令行工具支持”页面上的“ 控制台编码”列表中选择要在工具窗口中使用的字符集。

工具窗口包含以下选项卡和区域：

  • “ 输入”窗格，用于以<tool alias> <command>格式键入命令。“输入”窗格（字段或弹出窗口）的位置取决于“ 命令行工具支持”页面上的“ 显示控制台”设置。

  • “ 输出”选项卡显示执行命令的结果。该选项卡以最后一次调用的命令命名。

  • 默认情况下隐藏“ 工具定义文件错误”选项卡。只有在验证期间在.xml描述符中检测到任何结构差异时，才能访问该选项卡。

使用软包装

单击此切换按钮以将软包装模式应用于输出。

滚动到最后

单击此按钮可导航到以上次调用的命令命名的输出选项卡的底部。

打印

单击此按钮可打印出控制台的内容。单击该按钮后，WebStorm将打开“ 打印”对话框，您可以在其中配置打印过程和输出。

全部清除

单击此按钮可从控制台中删除所有文本。此功能也可在控制台的上下文菜单中使用。

导出到文本Alt + O.

单击此按钮可将执行命令的结果保存在文本文件中。在“ 导出预览”对话框中，打开，指定目标文件，然后单击“ 保存” 。

按Ctrl + Shift + F4

单击此按钮可关闭工具窗口。如果仍有一个或多个命令正在运行，则会打开“ 命令行工具”对话框。通过单击以下按钮之一指定是要停止它们还是在后台运行：

  • 终止并关闭
  • 关闭而不终止
