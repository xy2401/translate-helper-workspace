新项目：Yeoman

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Yeoman插件时有效！

档案|新的|项目

欢迎屏幕|创建新项目

当您在左侧窗格中选择静态Web ，然后从右侧窗格的列表中选择Yeoman时，“ 项目类别和选项”对话框的右侧窗格如下所示。

安装并启用Node.js和Yeoman插件时，仅在Ultimate版本中支持该功能。 插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装它们，如管理插件中所述 。启用后，插件可在IDE级别使用，也就是说，您可以在所有IntelliJ IDEA项目中使用它们。

项目

描述

项目名

在此字段中，指定项目的名称，默认情况下，它是位置字段中指定的文件夹的名称。

项目位置

在此字段中，指定项目文件夹的路径，其中将存储与项目相关的文件。

发电机

在此区域中，创建所需的Yeoman生成器列表，并选择在当前项目生成期间使用的生成器。
  • 要将生成器添加到列表：

    1. 单击安装生成器 。

    2. 在打开的对话框中显示所有可用的生成器包，在左侧窗格中选择所需的包，然后单击右侧窗格中显示的“ 安装生成器”按钮。您可以在不离开对话框的情况下依次安装多个软件包。

      安装结束后，单击“ 关闭”返回已使用新添加的包扩展的生成器列表。

  • 使用生成器创建项目：

    1. 在“ 位置”字段中，指定将在其中创建新项目的文件夹。手动键入路径或单击并在打开的对话框中选择文件夹。

    2. 如有必要，在“ 选项”文本框中，指定其他标志以自定义项目创建。可用标志集取决于所选的生成器。

    3. 从列表中选择所需的生成器，然后单击下一步 。

    4. 在启动的“ 新建项目”向导中指定所需的设置。页数及其内容取决于所选的发生器。

    5. 在向导的最后一页上，选中或清除Run npm install＆bower install复选框，以指定是否要运行Node Package Manager和Bower来安装开发新项目所需的软件包。

    6. 单击“ 下一步”，然后选择在当前窗口或新窗口中打开新项目。

选项

在此文本框中，指定其他标志以自定义项目创建。可用标志集取决于所选的生成器。
