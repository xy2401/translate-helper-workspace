模板中的可重用内容

包含模板用于定义可重用的代码片段（例如，标准头文件，版权声明等），以通过#parse指令插入到其他模板中。

#parse指令具有以下语法：

＃ 解析 （ “<include_template_name.extension>”）

例如：... #parse("JavaScript File Header.js") 。

可以在其他模板中以这种方式引用的模板列在编辑器的“ 包含”选项卡上WebStorm设置的文件和代码模板页面（ Ctrl + Alt + S ）。

创建包含模板

  1. 按Ctrl + Alt + S打开“ 设置/首选项”对话框，然后转到“ 编辑器”|文件和代码模板 。

  2. 打开“ 包含”选项卡。

  3. 单击创建模板 在工具栏上，指定包含模板名称，扩展名和正文。

将可重用模板插入另一个模板

  • 添加#parse指令，其中包含要插入的包含模板的名称。有关示例，请参阅模板语法 。
