代码参考信息

本节介绍如何查看符号的定义，显示文档参考以及使用视图参数信息功能。

对于标记语言，PyCharm从指定的DTD或模式中检索符号的定义。有关详细信息，请参阅HTML和XML 。

定义

在PyCharm中，您可以看到项目中定义符号的位置和方式，例如标记，类，字段，方法或函数。为此，IDE具有“ 快速定义”弹出窗口。

  • 要查看符号的定义，请在编辑器中选择它，然后按Ctrl + Shift + I （或单击“ 查看”|“快速定义” ）。

或者， 按住Ctrl / ⌘键，将光标悬停在任何符号上。PyCharm将符号显示为链接，并在工具提示中显示其定义。单击此链接可跳转到符号的定义。

点击右上角的图标用于在“ 查找”工具窗口中打开定义。要打开定义的源代码以进行编辑并关闭弹出窗口，请单击图标（ F4 ）。要在不关闭弹出窗口的情况下打开定义，请单击 （ Ctrl + Enter ）。

参数信息

参数信息弹出窗口显示方法和函数调用中的参数名称。在编辑器中键入左括号后，PyCharm会在1秒（1000毫秒）内自动显示包含所有可用方法签名的弹出窗口，或从建议列表中选择方法。

如果弹出窗口已关闭或者IDE 配置为不自动显示弹出窗口，则可以显式调用弹出窗口 。为此，请按Ctrl + P （或单击“ 查看”|“参数信息” ）。

显示完整的方法或功能签名

默认情况下，参数信息弹出窗口显示简单签名。您可以配置IDE以显示包含方法名称和返回类型的完整签名。

  • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|代码完成 ，然后选中显示完整方法签名复选框。

确保在项目源路径中包含所需的第三方JavaScript库 。否则，将不显示参数名称。

配置参数信息弹出窗口

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|代码完成 。

  2. 在“ 显示参数信息弹出窗口...毫秒”字段中，指定弹出窗口应出现的时间（以毫秒为单位）。

如果您不希望弹出窗口自动出现，请清除显示参数信息弹出窗口...毫秒复选框。

参数提示

参数提示在编辑器中与代码一起显示，以提高其可读性。它们显示传递值为文字或空值的方法参数的名称。

默认情况下启用提示。如果您不希望查看特定方法，可以使用例外配置列表。

配置参数提示例外列表

默认情况下，例外列表包含许多预定义的方法。您可以从列表中删除方法以查看其提示，或添加新方法以在编辑器中隐藏其提示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|外观 。

  2. 在“ 显示参数名称提示”选项旁边，单击“ 配置” 。

  3. 从语言列表中选择所需的语言 。

  4. 添加您不希望看到黑名单部分提示的方法。

  5. 您还可以在对话框底部为提示启用其他选项。然后应用更改并关闭对话框。

快速文档

您可以通过“ 快速文档”弹出窗口获取任何符号或方法签名的快速信息 。

PyCharm识别根据PEP-257创建的内联文档。

  • 要在插入符号中查看符号的文档，请按Ctrl + Q （或单击“ 查看”|“快速文档” ）。

    您可以再次按Ctrl + Q在弹出窗口和工具栏之间切换。

单击文档主题中的链接以预览顶级元素（在此示例中为： math ）或相关类型（在此示例中为float ）的文档。

点击用于更改字体大小的图标，显示快速文档工具栏或转到源代码。

自动显示快速文档

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般 。

  2. 在“ 其他”区域中，选中“ 显示鼠标移动快速文档”复选框，并指定弹出窗口应出现的延迟（以毫秒为单位）。

外部文件

外部文档在Web浏览器中打开必要的信息，以便您可以导航到相关的符号并保留信息以供进一步参考。

PyCharm显示了SciPy，NumPy ， PyGTK以及为Python解释器安装的其他模块的文档。应在Settings / Preferences的Python External Documentation页面中正确配置外部文档 。

查看外部文档

  • 要在Web浏览器中查看插入符号的文档，请按Shift + F1或选择查看|主菜单中的外部文档 。

生产力提示

将工具提示文本复制到剪贴板

要将工具提示文本复制到剪贴板，请按住Alt （在Linux上，按住Ctrl + Alt ）并单击工具提示。

查看上下文信息

如果当前方法或类声明不可见，则可以通过按Alt + Q在工具提示中查看它。
