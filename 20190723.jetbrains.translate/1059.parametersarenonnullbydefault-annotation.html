@ParametersAreNonnullByDefault

@ParametersAreNonnullByDefault注释可帮助您定义类或包中的所有方法参数都具有@NotNull语义，除非使用@Nullable注释明确注释它们。

@ParametersAreNonnullByDefault批注可以与包，类或方法一起使用。

要使用注释，请将jsr305库添加到模块依赖项：

  1. 打开“ 项目结构”对话框（ Ctrl + Shift + Alt + S ），然后转到“ 模块”|依赖性 。

  2. 单击添加 并选择Library | Java 。

  3. 在IntelliJ IDEA主目录中，选择lib \ jsr305.jar。

  4. （ 可选 ）在下一个对话框中，您可以修改库名称和级别。

  5. 应用更改并关闭对话框。

将JAR添加到项目后，可以开始使用@ParametersAreNonnullByDefault批注。例如，请考虑以下代码：

public static < T extends Comparable < T >> List < T > sort （ List < T > list ） { if （ list ！）= null ）{ List < T > copy = new ArrayList < T >（ list ）; 排序 （ 复制 ）; 返回 副本 ; } else { return null ; } }

如果使用@ParametersAreNonnullByDefault注释sort()方法，IntelliJ IDEA会立即识别if语句是无关的，并报告始终为true的条件。

但是，如果将方法sort()的参数注释为可为空，则不会看到任何检查消息。
