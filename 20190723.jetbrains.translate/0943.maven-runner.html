使用此页面配置将用于运行目标的外部Maven的设置。

项目

描述

在后台运行

选中此选项以执行作为后台任务运行。

VM选项

指定将传递给选定JRE的VM选项。

指定JVM选项时，请遵循以下规则：

  • 使用空格分隔各个选项，例如-client -ea -Xmx1024m 。

  • 如果选项包含空格，请将空格或包含空格的参数括在双引号中，例如， some" "arg或"some arg" 。

  • 如果选项包含双引号（作为参数的一部分），请使用反斜杠转义双引号，例如-Dmy.prop=\"quoted_value\" 。

  • 您可以将环境变量值传递给自定义Java属性。例如，如果定义变量MY_ENV_VAR ，则可以将其传递给foo属性，如下所示：

    - Dfoo = $ { MY_ENV_VAR }

JRE

选择将用于运行Maven目标的JRE。

环境变量

此字段允许您在项目级别设置自定义环境变量以运行maven目标。单击“ 浏览” 打开“ 环境变量”对话框，您可以在其中创建变量并指定其值。

属性

指定要传递给Maven的属性及其值。

或Alt + Insert

使用此图标或快捷方式将新属性定义为名称 - 值对。

或输入

使用此图标或快捷方式更改所选属性。

或Alt +删除

使用此图标或快捷方式从列表中删除所选属性。

跳过测试

选中此选项可跳过执行单元测试。这对应于Maven选项-Dmaven.test.skip=true
