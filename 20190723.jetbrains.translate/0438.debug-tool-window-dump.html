调试工具窗口。倾倒

转储按钮时会出现此选项卡单击调试器工具栏上 ，或选择运行|在主菜单上获取线程转储 。

使用此选项卡可以方便地查看外部线程转储或从调试器中获取的转储。标签分为两部分。左边一个显示所有线程，右边一个显示所选线程的堆栈跟踪。

在转储选项卡中，对所有线程进行排序，以便最有意义和最有用的线程位于列表顶部。为方便起见，线程以不同的灰色文本显示，并且死锁以红色突出显示。

过滤
按Ctrl + F

单击此按钮可以通过堆栈跟踪中的单词过滤线程转储。搜索字段显示在线程列表中，您可以在其中键入搜索字符串。

请注意，在所选线程的堆栈跟踪的转储选项卡的右侧部分中也可以使用纯搜索 Ctrl + F.

复制到剪贴板

单击此按钮可将整个线程转储复制到剪贴板。

按名称排序线程/按类型排序线程

单击这些按钮可按字母顺序或按类型在排序线程之间切换。

导出到文本文件
ALT + O

单击此按钮可将当前线程导出到指定的文本文件。
