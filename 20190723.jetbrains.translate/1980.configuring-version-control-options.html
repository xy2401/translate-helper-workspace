配置版本控制选项

配置常规VCS设置

通用版本控制设置适用于与PyCharm集成的所有版本控制系统。常规设置在“设置”对话框的“ 版本控制”页面上指定，包括定义需要确认的操作，后台操作，忽略的文件，问题导航和历史深度。

配置常规版本控制设置

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 版本控制” 。“ 版本控制”页面打开。

  2. 指定哪些版本控制相关操作需要确认 。

  3. 指定应在后台执行哪些操作。

  4. 创建版本控制系统要忽略的文件列表。

  5. 配置历史缓存处理 。

  6. 定义问题导航规则以从签入注释切换到错误跟踪系统中的相应问题。

指定哪些操作需要确认

您可以定义只有在您方确认后才能执行某些与版本控制相关的活动。需要确认的活动在“ 确认设置”页面中指定。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中的“ 版本控制”节点下，选择“ 确认”页面。

  2. 在“ 文件创建/删除”区域中，定义如何在版本控制系统中添加或删除在PyCharm中创建或删除的文件。可用选项包括：

    • 在添加到版本控制之前显示选项

    • 静默添加

    • 不要添加

  3. 在调用这些命令的“ 显示选项”对话框中 ，指定要让PyCharm显示“ 选项”对话框的命令。可用选项包括：

    • 查看

    • 状态

    • 获取最新版本

    • 更新

    • 撤消签出

  4. 通过选中或清除相应的复选框，配置用于使用更改列表的其他设置：

    • 指定文件的只读状态是否应该要求显式取消。

    • 指定是否需要有关将文件提交到存储库的有意义的注释。

    • 指定是否应将未提交的更改移动到另一个更改列表。

    • 如果提交操作失败，请指定是否以及如何创建更改列表。

指定要在后台运行的操作

您可以启用某些版本控制相关活动的后台执行。这些活动在后台设置页面中指定。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）下方的“ 版本控制”节点下，选择“ 背景”页面。

  2. 通过选中相应的复选框，启用必要操作的后台执行。可以为以下操作设置后台执行：
    • 更新
    • 承诺

    • 查看

    • 编辑/结帐

    • 添加和删除

    • 还原

    • 历史缓存处理

    • 检测“服务器上已更改”冲突

定义被忽略文件的列表

有时您可能需要保留某些类型的文件无版本。这些可以是VCS管理文件，实用工具，备份副本等。您可以创建一个全局忽略列表，该列表将存储在工作区文件中并应用于所有受支持的版本控制系统。

您要忽略的文件可以通过带通配符的通名模式名称明确指定。要忽略目录，您需要指定相对于项目根目录的完整路径。

使用“ 忽略的文件”设置页面列出必须从版本控制操作中排除的文件。

如果您使用的版本控制系统具有自己的忽略功能，请使用版本控制集成提供的相应本机命令。

请注意，一旦将文件添加到版本控制系统，忽略它将无效。您需要先从VCS中删除它。

  1. 通过执行以下操作之一打开“ 忽略的文件”设置页面：

    • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中的“ 版本控制”节点下，单击“ 忽略的文件” 。

    • 在“ 版本控制”工具窗口的“ 本地更改”选项卡中，单击在工具栏上，选择“ 配置忽略的文件” 。

  2. 点击 （ Alt + Insert ）创建新条目，或选择现有条目并单击 （ 输入 ）。将打开“ 忽略无版本文件”对话框：

  3. 明确指定要忽略的文件/目录或定义文件名模式。执行以下操作之一：

    • 选择“ 忽略指定的文件”选项并指定相对于项目根目录的文件名，例如my_folder/my_subfolder1/my_subfolder2/my_file 。手动键入名称或单击“ 浏览” 并在“ 选择要忽略的文件”对话框中选择所需的文件。

    • 选择忽略选项下的所有文件，并指定应忽略其内容的目录。键入相对于项目根目录的目录名称，例如my_folder/my_subfolder1/ ，或单击“ 浏览”按钮并在“ 选择要忽略的目录”对话框中选择所需的文件夹。

      该规则以递归方式应用于指定目录的所有子目录。如果某个目录有多个子目录，并且您只想忽略其中一个子目录，请明确指定所需的目录，例如my_folder/my_subfolder1/my_subfolder2/ 。

    • 选择“ 忽略所有文件匹配”选项，然后键入定义要忽略的文件名称的模式。

      定义要忽略的文件的模式使用两个通配符。

  4. 根据需要创建任意数量的条目并关闭对话框。

您还可以使用“ 版本控制”工具窗口的“ 本地更改”选项卡中“无版本 文件”节点下新添加文件的上下文菜单上的“ 忽略”命令，即时将文件添加到忽略列表中。

配置历史记录缓存处理

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中的“ 版本控制”节点下，选择“ 背景”页面。

  2. 设置缓存范围。根据您使用的版本控制系统，执行以下操作之一：

    • 指定要存储在缓存中的最大更改列表数。

    • 指定更改列表存储在缓存中的最大天数。

  3. 指定您希望版本控制系统刷新缓存的频率（以分钟为单位）。

配置特定于VCS的设置

某些设置适用于分配给整个项目或其目录的版本控制系统。其他与所选版本控制系统有关。使用“ 设置”对话框的“ VCS”节点下的相应VCS页面定义VCS特定的设置。

要配置特定于VCS的设置，请执行以下常规步骤

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 版本控制” ，然后单击VCS的页面。

  2. 根据需要设置选项。有关详细信息，请参阅版本控制设置的 VCS特定页面。
