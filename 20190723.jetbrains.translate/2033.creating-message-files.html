PyCharm将语言环境存储在语言locale目录的语言相关子目录中。要创建语言环境，请运行manage.py实用程序的makemessages任务。

创建一个消息文件

  1. 在主菜单上，选择“ 工具”|运行manage.py任务 ，或按Ctrl + Alt + R.

  2. 在manage.py任务窗口中，输入makemessages --locale <locale name >并按Enter键 。

    对要创建的每个区域设置重复此步骤。

    如果有标记为本地化的字符串，PyCharm将为每个语言环境生成一个目录和django.po文件：

    如果没有这样的字符串，则只创建一个空目录结构。
