将项目另存为模板对话框

工具|将项目另存为模板

指定要创建的模板的设置。

项目

描述

保存

如果要保存为模板的项目包含多个模块，则可以选择为整个项目创建模板，或仅包含模板中的一个模块。因此，选择<整个项目>或相应模块的名称。

如果项目仅包含一个模块，则此选项不可用。

名称

指定模板的名称。

描述

输入模板说明。

您可以在<b>和</b>以及<i>和</i>之间包含文本片段，以使相应的片段变为粗体或斜体。例如：

用 <B> 的Java </ b>的 模块 ，并 支持 <I> Web 开发 </ i> 和 <I> Tomcat的 一个 项目 </ I>。

用占位符替换参数

如果选中此复选框（推荐），则在创建基于模板的新项目或模块时，您将有其他选项。例如，您将能够指定基本Java包和不同于当前项目中定义的应用程序服务器。
