介绍

IntelliJ IDEA支持用于创建Java EE和Web应用程序的Struts框架。

通过专用的Web和Struts方面支持Struts支持。包含设置，配置文件路径和验证规则的专用构面确定模块的结构，以便IntelliJ IDEA检测如何处理模块内容。

  • Struts facet只能作为Web facet的子项创建。

  • 每个模块只允许一个Struts方面。

IntelliJ IDEA提供Struts框架的以下支持：

  • 基于专用Web构面自动创建特定模块结构和Web应用程序部署描述符web.xml 。

  • 自动下载并添加必要的库，并基于专用的Struts构面创建Struts应用程序配置文件struts-config.xml 。

  • 编码辅助，包括代码完成，语法和错误突出显示，文档查找和重构。

  • 转到Symbol并支持Actions。

  • 代码检查 。

  • 具有本机Struts应用程序结构导航和Struts操作的结构工具窗口 。

  • Struts Assistant工具窗口 ，提供管理Struts元素的工具 。

使用Struts开发Web应用程序

要使用Struts开发Web应用程序，请遵循以下常规步骤

  1. 启用Struts支持。

  2. 创建和编辑Struts元素及其交互 。

  3. 创建和编辑图块。

  4. 创建和编辑验证器。

  5. 使用必要的类填充您的应用程序。

  6. 部署并运行您的应用程序。
