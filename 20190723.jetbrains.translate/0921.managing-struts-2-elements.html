管理Struts 2元素

支柱2层的元件等的动作，结果，和拦截器以及它们之间的相互作用在被定义<package>所述的部分struts.xml配置文件。为模块启用Struts 2支持时 ，将自动创建该文件。

IntelliJ IDEA为您提供了一个友好的界面，用于在文本视图和图表中编辑struts.xml 。

在视图之间切换

  • 在项目视图中打开struts.xml ，然后单击窗口底部的Graph 。

创建Struts 2应用程序元素

  1. 切换到Project Tool Window并在View as下拉列表中选择Project 。

  2. 右键单击struts.xml节点，然后在上下文菜单中选择“ 新建 ”。然后选择相关的元素类型。将打开“ 新建...”对话框。

  3. 指定元素数据，然后单击“ 确定” 。将在项目树中创建并显示新元素。相应的源代码在struts.xml生成。

要删除Struts应用程序元素，请执行以下操作之一

  • 在“ 项目工具”窗口中 ，选择项目树中的元素。然后在上下文菜单中选择“ 删除 ”。

  • 切换到struts.xml并手动进行必要的更改。
