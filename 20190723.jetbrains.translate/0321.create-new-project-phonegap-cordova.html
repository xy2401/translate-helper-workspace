当您在左侧窗格中选择静态Web ，然后从右侧窗格的列表中选择PhoneGap / Cordova App时，“ 项目类别和选项”对话框的右侧窗格如下所示。

安装并启用PhoneGap / Cordova插件时，仅在Ultimate版本中支持该功能。 该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。
