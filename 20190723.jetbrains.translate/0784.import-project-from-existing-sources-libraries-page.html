此页面允许您选择和配置要包含在项目中的库 。

在左侧窗格的“ 库”下，将显示在指定位置找到的库和存档的列表。当您在此窗格中突出显示某个项目时，其内容将显示在“ 库内容”下的右侧窗格中。

使用复选框选择要包含在项目中的库。使用此页面上提供的其他控件来重命名库，还可以拆分或合并它们（请参阅下文）。

单击此图标可重命名所选（突出显示的）库或存档。

单击此图标可将多个选定（突出显示）的库或存档连接到新库中。

单击此按钮可将选定（突出显示）的库拆分为两部分。在“ 拆分库”对话框中，选择要提取到新库中的文件。
