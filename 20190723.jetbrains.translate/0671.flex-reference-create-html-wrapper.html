创建HTML包装器模板对话框

使用此对话框为面向Web的ActionScript或Flex应用程序配置和创建HTML包装器模板。

生成的模板（ index.template.html ）将包含一组标记，例如${title} ， ${swf}等。在编译期间，这些标记将替换为适当的值。例如， ${swf}将替换为.swf文件名。生成的.html包装器文件将与.swf文件具有相同的名称。

另请参阅使用SWF元数据标记来控制HTML包装器属性 。

如果选择此选项，应用程序将在现有Flash Player中运行SWF文件，以将用户升级到最新版本的播放器。
