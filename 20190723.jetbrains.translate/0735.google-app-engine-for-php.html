IntelliJ IDEA 2019.1帮助

适用于PHP的Google App Engine

所描述的功能由Google App Engine支持PHP插件处理。该插件不再维护，与IntelliJ IDEA 2018.1及更高版本不兼容。

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

使用IntelliJ IDEA，您可以开发一个PHP应用程序，以便在Google PHP Runtime Environment中运行 。IntelliJ IDEA支持所有主要的Google App Engine for PHP开发实践，并提供在将应用程序上载到运行时环境之前在本地运行和调试应用程序的可能性。

在这个部分：

  • 准备为PHP应用程序开发Google App
  • 运行，调试和上传应用程序到Google App Engine for PHP

IntelliJ IDEA中的Google App Engine支持

  • 具有特定目录结构和配置文件的专用项目类型 。

  • 用于PHP运行/调试配置的专用App Engine，用于在PHP开发服务器上运行和调试应用程序，该应用程序与Google App Engine for PHP SDK捆绑在一起。

  • 使用“ 工具”菜单的命令上载应用程序。
