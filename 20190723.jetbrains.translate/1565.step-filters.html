步骤过滤器

档案|设置| PHP |调试|适用于Windows和Linux的步骤过滤器
IntelliJ IDEA |偏好| PHP |调试|用于macOS的步骤过滤器
按Ctrl + Alt + S

该页面仅在启用PHP插件时可用。该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

在此页面上，指定您希望调试器跳过的PHP方法和函数，以及在Stepping页面上定义的要跳过的常用项目列表。

项目

描述

跳过魔术方法

选中此复选框以禁止单步执行PHP魔术方法以及名称以双下划线__开头的任何其他方法或函数。

跳过构造函数

选中此复选框以禁止单步执行构造函数 。

跳过的方法

在此区域中，创建一个您不希望IntelliJ IDEA进入的特定方法的列表。

  • 要向列表中添加方法，请单击在打开的“ 添加方法”对话框中，按以下格式键入所需函数的名称或所需方法的完全限定名称：

    • \Namespace\Class->Method 非静态方法的方法。

    • \Namespace\Class::staticMethod for static methods。

  • 要从列表中删除方法，请选择允许单步执行的方法 。

跳过的文件

在此区域中，创建一个您不希望IntelliJ IDEA进入的特定文件列表。当您使用具有许多核心文件的框架时，这是非常有用的。

  • 要将文件添加到列表，请单击 ，然后点击在新添加的行中，然后在打开的对话框中选择要跳过的文件。

  • 要从列表中删除文件，请选择允许单步执行的文件并单击 。
