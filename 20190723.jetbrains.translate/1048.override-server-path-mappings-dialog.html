覆盖服务器路径映射对话框

单击“ 运行/调试配置：PHP网页”对话框的“ 调试”区域中的“ 服务器路径映射”按钮时，将打开该对话框。

使用此对话框将本地计算机上的文件夹映射到服务器上的文件夹。这些映射在远程调试和测试期间用作在异常或断言的测试或堆栈跟踪与相应源代码之间切换的基础。

IntelliJ IDEA会自动检测这些映射，但仍然为您提供了手动指定它们的可能性。

项目

描述

客户端的本地路径

在此文本框中，指定所需本地文件夹的绝对路径。手动键入路径或单击“ 浏览” 并在打开的对话框中选择所需的文件夹。

服务器上的本地路径

在此文本框中，根据服务器上使用的文件系统指定服务器上相应文件夹的路径。

加

单击此按钮可将新行添加到映射列表中。

去掉

单击此按钮可从列表中删除选定的映射。
