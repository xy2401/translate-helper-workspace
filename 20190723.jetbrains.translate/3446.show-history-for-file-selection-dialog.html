显示文件/选择对话框的历史记录

VCS |当地历史|选择显示历史记录或显示历史记录

使用此对话框可以浏览文件或选择的更改。此对话框中有两个视图：

  • 左侧部分的历史记录视图

  • 右侧部分的差异视图

文件的上下文菜单或编辑器中的选定文本提供相同的对话框。

历史观点

此视图显示文件的修订（状态）列表，以及存储修订的日期和时间。部分修订版附带标签和标签。

修订会自动标记，例如，打开项目，提交更改或执行测试。您也可以设置自己的标签 。

项目

描述

单击此按钮可还原所选操作。

单击此按钮可基于所选本地版本创建修补程序。

单击此按钮可打开相应的帮助主题。

每个修订的上下文菜单中都提供相同的操作。

差异观点

差异视图是一个功能强大的编辑器，支持基本搜索和替换 ， 撤消/重做操作和代码完成 。

如果在“ 历史记录”视图中选择了修订，则“差异”视图的左侧窗格将显示此只读修订版，其中包含与右侧窗格中显示的当前修订版的差异。可以编辑当前版本。

项目

捷径

描述

按Ctrl + C

单击此按钮可将当前行或选定的片段复制到剪贴板。

要么

F7或Shift + F7

使用这些按钮可以移动到下一个或上一个差异。

忽略空格

使用此下拉列表可以定义差异查看器应如何处理文本中的空格。

  • 不要忽略 - 选择此选项时，空格会被视为不重要，并且会突出显示差异。

  • 前导和尾随 - 选择此选项可忽略行末尾和开头的差异。

  • 全部 - 选择此选项时，无论它们在源代码中的位置如何，都认为空格不重要。

， 要么

使用这些按钮应用差异。

传说

此区域显示有关遇到的差异的摘要信息：找到的差异数量和颜色映射。

差异查看器的颜色映射在“ 颜色和字体”页面上配置。
