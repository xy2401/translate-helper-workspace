使用JAXB和XmlBeans

要使用JAXB和XmlBeans，请执行以下预备步骤：

  1. 下载并安装XMLBeans工具。

  2. 在“设置/首选项”对话框的“ 插件”页面上，确保已启用“ Web服务”捆绑插件。

  3. 在“设置/首选项”对话框的“ Web服务”页面上，指定XmlBeans的安装目录以及以下Web服务引擎之一：
    • 地铁

    • GlassFish 。

    • 用于XML Web服务参考实现的Java API（JAX-WS RI）

    • Java Web服务开发包（JWSDP）
