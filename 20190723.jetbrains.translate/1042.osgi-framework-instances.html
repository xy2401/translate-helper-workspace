先决条件

此页面显示在设置/首选项对话框（ Ctrl + Alt + S ）中，启用Osmorc插件时。

该插件与IntelliJ IDEA捆绑在一起，默认情况下处于激活状态。如果禁用，则可以手动启用插件 。

使用此页面为当前运行的IntelliJ IDEA添加和删除IDE级框架实例定义。可用框架定义集确定“ 项目设置”选项卡中OSGi框架下拉列表的内容。

项目

描述

框架实例

该字段显示为当前运行的IntelliJ IDEA定义的所有框架实例的列表。

类型

此只读字段显示当前所选定义的类型。

主目录

此只读字段显示当前所选框架实例的位置。

名称

此只读字段显示当前所选定义的名称。

版

此只读字段显示所选定义所基于的框架版本。

或Alt + Insert

单击此按钮以打开“ 添加OSGI框架”下拉列表，并选择要配置的新框架实例的类型。
将打开指定的OSGI Framework Instance对话框。

或输入

单击此按钮可打开“ FrameWork实例”对话框，并在此处编辑所选框架的定义。

如果“ 框架实例”字段中至少定义了一个框架，则该图标处于活动状态。

或Alt +删除

单击此按钮可从列表中删除所选定义。

单击这些按钮可向上移动或向下移动选定的框架实例。
