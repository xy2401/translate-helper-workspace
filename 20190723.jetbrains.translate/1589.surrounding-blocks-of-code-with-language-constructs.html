环绕代码片段

IntelliJ IDEA为周围的代码片段提供标准模板，其中包含基于源代码语言的各种构造。这包括if...else条件语句， do...while和for循环， try...catch...finally ，XML标签，折叠区域和其他结构的组合。

用语言结构包围代码块

  1. 选择所需的代码片段。

  2. 在代码菜单上，单击环绕 （ Ctrl + Alt + T ）。

  3. 从列表中选择必要的环绕声。

您可以编辑环绕声明中使用的代码模板。在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|文件和代码模板 。

例如，您可以为try...catch条件语句配置final修饰符，名称甚至是异常类型。
