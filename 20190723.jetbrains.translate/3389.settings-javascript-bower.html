依赖

该区域显示了计算机上当前安装的所有与Bower相关的软件包的列表。

  • 包：此只读字段显示包的名称，与在命令行模式下安装包时的名称完全相同。

  • 版本：此只读字段显示计算机上安装的软件包的版本。

  • 最新：此只读字段显示该软件包的最新发布版本。如果包不是最新的，则标记为 。

  • 点击安装新包。在打开的“ 可用包”对话框中，选择相关包。准备好后单击Install Package 。

  • 点击删除选定的包。

  • 点击使用最新发布的版本替换所选软件包的当前版本。仅当所选项目不是最新时，才会启用该按钮。
