从命令行格式化文件

命令行源代码格式化程序是PyCharm中的一项特殊功能，它允许您在项目外部格式化任意文件。

为了能够格式化文件，请确保安装并启用了支持所需文件类型的相应插件。

  • 在命令行中，使用以下语法运行format命令：
    格式[ -h ] [ -r | -R ] [ -s | -settings settingsPath ] [ -m | -mask掩码] path1 [ path2 ... ]

    该命令启动PyCharm，它格式化指定的文件然后退出。

命令选项如下：

选项

描述

格式

format.bat （macOS和Linux上的format.sh ）脚本的完整路径，位于<IDE_HOME> / bin目录中。

-H

（可选）显示帮助信息并退出。

-r | -R

（可选）以递归方式扫描path1,path2...指定的目录。

-s | -settings settingsPath

根据在提供的代码样式设置文件（可选）格式代码settingsPath 。您可以使用以下之一：

  • 具有导出的代码样式设置的文件：在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，打开编辑器|代码样式页面，单击 ，然后选择导出 。

  • 存储在项目目录中的.idea / codeStyleSettings.xml文件（适用于PyCharm版本2017.2及更早版本）。

  • 存储在项目目录中的.idea / codeStyles / codeStyleConfig.xml文件（适用于PyCharm版本2017.3及更高版本）。

如果省略该参数，将使用默认代码样式设置。

-m | - 掩码

（可选）以逗号分隔的文件掩码列表，用于定义要处理的文件。* （任何字符串）和? （支持任何单个字符）通配符。

pathN

要处理的文件或目录的路径。

例子

  1. 使用默认代码样式设置格式化C：\ Data \ src目录中的所有文件，包括所有子目录：

    < f ormat.bat -r C： \ D ata \ s \ rc中的<IDE_HOME> \ b

  2. 使用C：\ Data \ settings.xml中的代码样式设置非递归格式化C：\ Data \ src目录中的所有.xml和.html文件：

    <IDE orHOME> \ b in \ f ormat.bat -s C： \ D ata \ s ettings.xml -m * .xml，*。html C： \ D ata \ s rc

  1. 使用默认代码样式设置格式化〜/ MyProject / src目录中的所有文件，包括所有子目录：

    <IDE_HOME> /bin/format.sh -r~ / MyProject / src

  2. 使用〜/ Data / settings.xml中的代码样式设置非递归地格式化〜/ MyProject / src目录中的所有.xml和.html文件：

    <IDE_HOME> /bin/format.sh -s~ / Data / settings.xml -m * .xml，*。html~ / MyProject / src
