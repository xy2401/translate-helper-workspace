将类层次结构视为类图

PyCharm可帮助您查看所选类型的完整层次结构。

这样，您可以修改UML类图的内容，显示祖先或后代类，方法签名中使用的类型，以及执行可用于UML类图的所有操作。

将类层次结构视为UML类图

  1. 在编辑器中打开所需的类，并将插入符号放在要查看其层次结构的类型中。

  2. 执行以下操作之一：

    • 在上下文菜单上，指向Diagrams ，然后选择Show Diagram或Show Diagram Popup

    • 按Ctrl + Shift + Alt + U或Ctrl + Alt + U.
