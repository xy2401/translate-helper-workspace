从Java代码生成Xml架构

本主题描述如何在Java类的基础上生成XML Schema ，其中包括将Java类的成员映射到XML Schema的元素。使用IntelliJ IDEA，可以使用JAXB完成此转换。

使用JAXB从Java类生成XML Schema

  1. 通过执行以下操作之一打开“ 使用JAXB从Java生成XML架构”对话框：

    • 在“ 项目”工具窗口中，选择所需类的名称，然后选择“ Web服务”在上下文菜单中使用JAXB从Java生成XML模式 。

    • 在活动编辑器选项卡中打开所需的类，选择“ 工具”| JAXB |在主菜单上使用JAXB从Java生成XML模式 。

  2. 指定要在生成的Schema中反映的方法参数和返回类型：

    • 要使用所有类方法，请清除以下方法的Include参数和返回类型复选框。

    • 要选择要涉及的特定方法，请选中包含以下方法的参数和返回类型复选框，然后选中所需方法旁边的添加到JAXB生成复选框。
