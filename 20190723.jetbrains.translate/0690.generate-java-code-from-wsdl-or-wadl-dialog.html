从WSDL或WADL对话框生成Java代码

创建Java模块并在其中启用Web服务客户端开发后，将打开该对话框。要在开发期间随时访问该对话框，请在“ 项目”视图中选择所需的客户端模块，然后选择“ Web服务”在上下文菜单中从Wsdl或Wadl生成Java代码 。

使用该对话框可以根据目标Web服务的所需WSDL描述符生成客户端XML-Java绑定。

从技术上讲，IntelliJ IDEA使用通过命令行控制的第三方库从WSDL生成Java代码。此命令行由您在此对话框的字段中输入的数据组合而成。

Apache Axis

Apache Axis

支持包装文档/文字样式

使用此复选框配置“包装”文档/文字的处理，这是一个文档文字变体，它将参数包装为根元素的子元素。

  • 清除此复选框后，不会对“包装”文档/文字样式操作应用特殊处理。

  • 当选中该复选框时，将考虑一组条件来确定顶层元素是否“展开”，并且元素的每个组件都应被视为操作的参数。考虑以下条件：

    • 输入消息由单个部分组成。

    • 这个单一部分是一个元素。

    • 该元素与操作具有相同的名称。

    • 元素的复杂类型没有属性。

默认情况下，选中该复选框。
