将IntelliJ IDEA项目导出到Eclipse

在本页面：

  • 概观
  • 导出到Eclipse
  • 将IntelliJ IDEA模块转换为Eclipse兼容格式

概观

您可以将IntelliJ IDEA项目导出到Eclipse。这样的导出导致为包含内容根的模块目录中的每个模块文件（ *.iml ）创建Eclipse项目文件（ .project和.classpath ）。将IntelliJ IDEA模块导出到Eclipse项目的另一种方法是将此类模块转换为与Eclipse兼容的格式。

在开始导出项目之前，请确保已启用 Eclipse Integration插件。

导出到Eclipse

要将当前打开的项目导出到Eclipse，请按照下列步骤操作：

  1. 在主菜单上，选择“ 文件”|导出到Eclipse 。“ 导出到Eclipse”对话框显示尚未转换并切换为使用Eclipse格式的模块列表（具有IntelliJ IDEA模块格式.iml模块）。

  2. 选择要导出的模块。

  3. 如有必要，请选择建议的选项。

  4. 单击确定 。

目前，从IntelliJ IDEA迁移到Eclipse受到某些限制：

  • 无法迁移具有多个内容根的IntelliJ IDEA模块。

  • Eclipse中的外部源不会迁移。

  • 导出期间仅自动创建Java模块。但是，任何模块都可以手动转换为Eclipse兼容格式。

  • 默认工作区JRE未转换为项目SDK或从项目SDK转换。

将IntelliJ IDEA模块转换为Eclipse兼容格式

要将IntelliJ IDEA模块转换为Eclipse兼容格式，请按照下列步骤操作：

  1. 在主菜单上，选择“ 文件”|项目结构 ，或按Ctrl + Shift + Alt + S.

  2. 在“ 项目结构对话框”对话框中，选择要转换的模块。

  3. 切换到“ 依赖关系选项卡” 选项卡。

  4. 从Dependencies存储格式下拉列表中，选择Eclipse（.classpath） 。
