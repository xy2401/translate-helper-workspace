提取字段

Extract Field重构允许您声明一个新字段并使用所选表达式对其进行初始化。原始表达式将替换为字段的用法。

使用对话框提取字段

如果在“ 设置/首选项”对话框的常规页面上清除了“ 启用就地模式”复选框（ Ctrl + Alt + S ），则“ 提取字段”重构将通过“ 提取字段”对话框执行 。

  1. 在编辑器中，选择要用字段替换的表达式或变量，或者只将光标放在这样的表达式或变量声明中。

  2. 在主菜单或选择的上下文菜单中，选择Refactor |提取|字段 ，或按Ctrl + Alt + F.

  3. 从“ 表达式”列表中，选择要替换的表达式。请注意，PyCharm会在编辑器中突出显示所选表达式。

  4. 在打开的对话框中，指定新字段的类型和名称。

  5. 在Initialize in部分中，指定新字段的初始化位置。

  6. 要自动替换所选出现的所有表达式（如果多次出现），请选择“ 替换所有出现次数 ”选项。

  7. 单击“ 确定”以创建字段。

要就地提取字段

默认情况下，PyCharm中启用了就地重构 。因此，如果您尚未更改此设置，则会在编辑器中就地执行“引入字段”重构：

  1. 将光标放在要由字段替换的变量的表达式或声明中。

  2. 执行以下操作之一：

    • 按Ctrl + Alt + F.

    • 选择Refactor |在主菜单或上下文菜单中引入Field 。

  3. 如果检测到当前插入符号位置有多个表达式，则会出现“ 表达式”列表。如果是这种情况，请单击表达式以选择它。或者，按向上或向下导航到感兴趣的表达式，然后按Enter键将其选中。

  4. 如果在类中找到多个表达式，请指定是否要仅使用新常量替换所选的实例或所有找到的实例：

  5. 如果相关，请指定新字段的初始化位置 - 在当前方法中，或在类构造函数中。

  6. 指定字段的名称。从列表中选择名称，或在带红色边框的框中键入名称。

  7. 要完成重构，请按Tab键或Enter键 。

    如果您尚未完成重构并想取消所做的更改，请按Escape键。

    请注意，有时您可能需要多次按相应的键。
