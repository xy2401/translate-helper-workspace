查看模型依赖关系图

模型依赖关系图使您可以概览Django应用程序中的模型，并分析它们之间的关系。

模型依赖关系图可用于：

  • Django模型

  • Google App Engine型号

  • SQLAlchemy的

打开项目的模型依赖关系图

  1. 执行以下操作之一：

    • 在主菜单上，选择“ 查看”|显示模型依赖关系图 。

    • 在“项目”工具窗口或编辑器的上下文菜单上，指向“ 图” ，然后选择“ 显示图表”或“ 显示图表弹出窗口” 。

    • 按Ctrl + Shift + Alt + U或Ctrl + Alt + U.

  2. 从弹出窗口中选择图表类型：

如果您为特定模型调用模型依赖关系图，则图表将打开，问题模型居中并具有焦点，并缩放到实际大小。

在模型依赖关系图中，您可以执行以下操作

  • 选择元素 。

  • 添加备注，删除元素 。

  • 更改图表布局 。

  • 更改图表比例 。

  • 导航到源代码 。

  • 使用Structure视图 （ Ctrl + F12 ） 浏览模型 。

  • 查找所选节点元素的用法 。

  • 调用重构命令 。

按住Alt键会调用放大镜工具，这将帮助您更仔细地查看模型依赖关系图中最有趣或最有问题的区域。
