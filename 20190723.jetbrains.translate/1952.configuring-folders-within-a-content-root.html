在内容根中配置文件夹

在内容根目录中 ，PyCharm可以区分包含源代码的文件夹，以及在搜索，解析，观看等时要忽略的文件夹。为此，您可以将内容根目录下的任何文件夹标记为源文件夹，或排除，以使其对PyCharm不可见。

标记目录

您可以通过两种不同的方式将文件夹分配给类别：

  • 使用项目结构 。

  • 使用“ 项目”工具窗口中文件夹的上下文菜单。

通过项目结构标记内容根目录下的目录

  1. 打开“项目结构”设置。

  2. 在“ 项目 结构”页面的“ 项目”窗格中，单击要为其配置内容根的项目。

  3. 在“ 项目结构”页面的“ 内容”根窗格中，单击所需的内容根。此内容根目录下的目录显示为树视图。

  4. 选择要标记的目录，然后执行以下操作之一：

    • 单击顶部的其中一个图标，为此目录指定所需的状态。

    • 在目录的上下文菜单中选择相应的状态命令。

    • 按Alt + <目录状态的第一个字母> （例如， Alt + E表示已排除的根， Alt + R表示资源， Alt + S表示源根）。

在“项目”工具窗口中标记目录

  1. 在“ 项目工具”窗口中右键单击所需目录。

  2. 在上下文菜单上，指向标记目录为节点。

  3. 选择标记为<目录状态> 。

取消标记目录

要将文件夹恢复为常规状态，您需要取消标记 。

  • 在项目结构中 ：

    • 在内容根目录下的文件夹列表中选择相关目录，然后单击 ：

    • 再次单击文件夹的状态图标。

    • 在目录的上下文菜单中选择相应的命令。

  • 在“ 项目工具”窗口中 ，右键单击所需目录，指向“将目录另存为”节点，然后选择“ 取消标记为<目录状态>” 。
