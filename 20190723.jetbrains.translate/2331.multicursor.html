Multicursor

PyCharm支持多个插入符号。大多数编辑器操作（例如键盘导航或文本插入和删除）适用于每个插入符号。还支持实时模板和自动完成功能。

可以添加或删除插入符号;编辑器选项卡中至少存在一个插入符号。

最近添加的插入符被认为是主要的 。在插入符号处突出显示编辑行仅适用于主插入符号。

插入符号的位置和行为取决于：

  • 启用或禁用列选择模式 。

  • 在行结束后启用或禁用插入符号 。

有关其他信息，请参阅编辑器中的选择文本部分。

添加，删除和克隆插入符号

要添加插入符号，请执行以下操作之一

  • 按Alt键并在插入符号位置单击鼠标左键。

  • 按Ctrl （Windows或Linux）/ Alt （macOS）两次，然后不释放它，按向上或向下箭头键。

要删除插入符，请执行以下操作之一

  • 按Esc键删除除主要插入符号以外的所有现有插入符号。

  • 按Shift + Alt并在要删除的插入符号上单击鼠标左键。

复制和粘贴

复制具有多个光标的文本（ Ctrl + C ）或剪切（ Ctrl + X ）时，每个插入符号的选择将放置到剪贴板。在粘贴（ Ctrl + V ）时，剪贴板中的文本将分为行。
