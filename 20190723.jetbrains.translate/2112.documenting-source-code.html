文档源代码

基本

PyCharm提供了用于创建文档注释的便捷功能。

源代码中的文档注释可用于代码参考信息，并在按Ctrl + Q时打开以供查看。

当您创建其他标记时 ，PyCharm提供代码完成，建议可能的标记名称。

Python文档

可以根据在“设置/首选项”对话框的“ Python集成工具”页面中选择的语法创建文档注释，例如reStructuredText或epytext 。
如果此功能适用于某个函数，PyCharm会根据所选的文档字符串格式生成标记，例如：

  • 对于reStructuredText :: :param标记，用于在函数签名中声明的每个参数，以及:return标记。

  • 对于epytext：函数签名中声明的每个参数的@param标记，以及@return标记。

这样做， reStructuredText和epytext标记中的标记相应地突出显示。

如果已配置 ，则可以使用type和rtype标记生成文档注释存根。

在Python文件中，PyCharm识别以Python文档字符串表示的文档注释。

在开始之前 ，请确保在Settings / Preferences对话框的Python Integrated Tools页面中选择了所需的docstring格式，例如epytext或reStructuredText 。

另请注意，PyCharm从conf.py捕获自定义角色。配置包含*.rst文件的目录时，请指向带有conf.py的目录（Python 集成工具|具有* .rst文件的目录的路径 ）。
