安装Python扩展

PyCharm的调试器是用Python编写的。Python是一种解释型语言，这意味着在运行代码时必须将指令转换为机器代码。

这会花费时间 ，因此它没有编译代码那么快。值得庆幸的是，Python通过Cython支持编译的C扩展。PyCharm附带了C版本的调试器代码，您可以编译这些代码以更快地在PyCharm中进行调试。

如果你有一个很大的代码库，使用这个加速扩展是合理的。

获得额外加速所需的操作取决于您的平台：

* NIX

当用户启动调试器会话时，将显示通知弹出窗口。用户只需点击那里的安装链接：

如果在单击“ 安装”链接后重复收到相同的弹出通知，请检查PyCharm使用的目录的权限。您必须具有写入权限才能安装Cython扩展。

macOS和Windows

你不需要做任何事情。我们捆绑了预编译扩展，因此您已经在使用它们。

显示以下错误消息：

Python.h: no such file or directory 。

您缺少Python dev的头文件和静态库。

使用包管理器为系统范围内的Python dev安装头文件和静态库。

例如，您可以使用以下命令：

  • 对于Python 2.x：

    sudo apt-get install python-dev

  • 对于Python 3.5：

    sudo apt-get install python3-dev

  • 对于Python 3.6：

    sudo apt-get install python3.6-dev

显示以下错误消息：

Command 'gcc' failed with exit status 1 。

你没有C编译器。

安装C编译器以便为调试器构建Cython扩展。

在没有PyCharm UI的情况下使用Cython Speedup

如果想要手动编译Cython加速，他/她不应该使用PyCharm UI。

在Ubuntu上，使用以下命令：

/ usr / bin / python3 /<PYCHARM_INSTALLATION_PATH>/helpers/pydev/setup_cython.py build_ext --inplace

请注意，您的路径应根据您的特定本地系统进行调整。
