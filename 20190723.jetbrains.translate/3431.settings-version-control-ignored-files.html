忽略的文件

档案|设置|版本控制| Windows和Linux的忽略文件
WebStorm |偏好|版本控制|忽略的macOS 文件
按Ctrl + Alt + S

使用此对话框可以配置您不希望置于版本控制之下的文件和目录列表。这些文件名可以是与VCS管理相关联的文件名，备份文件以及您希望保持未版本化的任何其他工件。您还可以指定要忽略的文件模式。

您只能忽略未版本控制的文件，这些文件是尚未置于版本控制之下的文件。

项目

捷径

描述

Alt +插入

使用此图标或快捷方式将项目添加到列表中。将打开“ 忽略无版本文件”对话框，您可以在其中键入要忽略的文件或目录的确切路径，或指定用于定义要忽略的文件和目录名称的模式。

输入

使用此图标或快捷方式在“ 忽略无版本文件”对话框中编辑选定的路径或模式。

Alt + Delete键

使用此图标或快捷方式从列表中删除选定的路径或模式。

两个字符可用作通配符：

  • * ：替换任何字符串。

  • ? ：替换单个字符。

例如， *.iml将忽略具有iml扩展名的所有文件; *.?ml将忽略所有扩展名以ml结尾的文件。
