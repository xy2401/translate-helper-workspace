IntelliJ IDEA 2019.1帮助

CoffeeScript的

此功能仅在Ultimate版本中受支持。

使用IntelliJ IDEA，您可以使用编译成JavaScript的CoffeeScript 。IntelliJ IDEA识别*.coffee文件并用它们标记 。您可以直接在CoffeeScript代码中设置断点，IntelliJ IDEA使用编译期间生成的源映射识别断点。

IntelliJ IDEA通过建议完成关键字，标签，变量，参数和功能来帮助您编写CoffeeScript代码。您可以从符号跳转到其声明（ Ctrl + B ）或实现（ Ctrl + Alt + B ），也可以按名称查找任何符号（ Ctrl + Shift + Alt + N ）。

在你开始之前

  1. 下载并安装Node.js.

  2. 如管理插件中所述，在“ 插件”页面上安装并启用“文件监视器”存储库插件。

将CoffeeScript编译为JavaScript

您也可以从命令行使用编译器或将其配置为第三方工具，有关详细信息，请参阅配置第三方工具 。

您还可以在npm和Yarn中描述的Node.js和NPM页面上安装coffeescript软件包。

从CoffeeScript官方网站了解更多信息。

IntelliJ IDEA提供了一个通用的过程和用户界面，用于创建所有类型的文件监视器 。唯一的区别在于您在每种情况下选择的预定义模板。

IntelliJ IDEA与coffee-script编译器集成，可将CoffeeScript代码转换为JavaScript。该工具还会创建源映射 ，以设置CoffeeScript代码和生成的JavaScript代码中的行之间的对应关系，以便IntelliJ IDEA调试器识别CoffeeScript代码中的断点并正确处理它们。要动态编译代码，需要将编译器配置为IntelliJ IDEA 文件监视器 。

安装CoffeeScript编译器

  1. 在嵌入式终端 （ Alt + F12 ）中，键入以下命令之一：

    • npm install --global coffeescript for global installation。

    • npm install --save-dev coffeescript将CoffeeScript作为开发依赖项安装。

创建文件监视器

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 工具”|文件监视器 。

  2. 点击或按Alt + Insert并从列表中选择CoffeeScript预定义模板。您的代码将被翻译为JavaScript并随生成的源地图一起提供 。

  3. 在“ 程序”字段中，指定coffee.cmd文件的路径。手动键入路径或单击并在打开的对话框中选择文件位置。

  4. 按照使用文件监视器中的说明继续操作。

编译代码

当您打开文件时，IntelliJ IDEA会检查当前项目中是否有适用的文件观察程序。如果配置但禁用了此类文件观察程序，IntelliJ IDEA将显示一个弹出窗口，通知您有关已配置的文件观察程序并建议启用它。

如果在当前项目中配置并启用了适用的文件观察程序，则IntelliJ IDEA会在“ 新建观察器”对话框中指定的事件时自动启动它。

  • 如果选中“ 自动保存已编辑的文件以触发观察者”复选框，则只要对源代码进行任何更改，就会调用“文件观察器”。

  • 如果清除了自动保存用于触发观察者的已编辑文件复选框，则会在保存（ 文件|全部保存 ， Ctrl + S ）或从IntelliJ IDEA移动焦点时启动文件观察器（在帧停用时）。

编译器将生成的输出存储在单独的文件中。该文件具有源CoffeeScript文件的名称，扩展名为js或js.map具体取决于编译器类型。生成的文件的位置在“ 新建观察器”对话框的“ 输出路径刷新”字段中定义。根据此设置，IntelliJ IDEA会检测编译器输出。但是，在“ 项目”工具窗口中，它们显示在源coffee文件下，该文件现在显示为节点。

在不运行编译器的情况下预览编译结果

IntelliJ IDEA可以在不实际运行编译器的情况下对CoffeeScript代码执行静态分析，并在专用只读查看器中显示预测的编译输出。

  1. 在编辑器中打开所需的CoffeeScript文件，然后右键单击编辑器背景。

  2. 从上下文菜单中，选择“ 预编译的CoffeeScript文件” 。预览在专用的只读查看器中打开：左侧窗格显示原始CoffeeScript源代码，右侧窗格显示编译器运行时将生成的JavaScript代码。

运行CoffeeScript

此功能仅在Ultimate版本中受支持。

在IntelliJ IDEA中运行CoffeeScript有两种方法：

  • 手动编译CoffeeScript代码，然后运行输出JavaScript代码，就像它是Node.js应用程序一样。

  • 通过Node.js运行配置运行原始CoffeeScript代码，并让IntelliJ IDEA动态编译它。

手动编译CoffeeScript并运行生成的JavaScript代码

  1. 将CoffeeScript代码编译为Javascript 。

  2. 使用以下必需设置开始创建Node.js运行配置 ：

    1. 要使用的Node.js引擎。默认情况下，该字段显示路径上指定的解释的Node.js Node.js的配置过程中的页面。

    2. 在“ 工作目录”字段中，指定从要运行的起始CoffeeScript文件引用的文件的位置，例如， 包括 。如果此文件未引用任何其他文件，请将该字段留空

    3. 在Path to Node App JS File字段中，指定编译期间从原始CoffeeScript文件生成的JavaScript文件的完整路径。

  3. 保存配置并单击在工具栏上。

  4. 继续运行Node.js应用程序时继续。

在运行期间动态编译CoffeeScript

  1. 此模式要求register.js文件（它是coffee-script包的一部分）应位于项目内。因此，您需要在本地安装Node.js页面上的coffee-script包，如npm和Yarn中所述 。

  2. 在编辑器中打开起始的CoffeeScript文件，或在“ 项目”工具窗口中选择， 然后从上下文菜单中选择“ 创建<CoffecScript_file_name> ”。或者，开始创建Node.js运行配置，如创建Node.js运行/调试配置中所述 。在打开的“运行/调试配置：Node.js”对话框中，指定以下必需设置：

    1. 要使用的节点解释器。选择相关解释配置或创建一个新的，看到默认情况下，该字段显示路径上指定的解释的Node.js Node.js的配置过程中的页面。

      对于Linux和macOS，Node.js会从CoffeeScript编译器可执行文件的路径中覆盖此设置。

    2. 在“ 节点参数”字段中，键入--require coffee-script/register 。

    3. 在“ 工作目录”字段中，指定应用程序的工作目录 。默认情况下，该字段显示项目根文件夹 。

    4. 在JavaScript文件字段中，指定要运行的CoffeeScript文件的完整路径。

    请注意，如果直接从所需的CoffeeScript文件创建运行配置，则将自动填写所有必填字段。

  3. 保存配置并单击在工具栏上。

  4. 继续运行Node.js应用程序时继续。

调试CoffeeScript

要在IntelliJ IDEA中调试CoffeeScript，除了JavaScript代码之外，还需要生成源映射 。源映射设置CoffeeScript代码和生成的JavaScript代码中的行之间的对应关系，否则将无法正确识别和处理断点。通过使用CoffeeScript类型的文件监视器手动编译CoffeeScript代码来生成JavaScript和源映射。之后，您可以调试输出JavaScript代码，就像它是Node.js应用程序一样。

调试CoffeeScript仅在本地模式下受支持。这意味着IntelliJ IDEA本身根据运行配置启动Node.js引擎和目标应用程序，并完全控制会话。

有关调试Node.js应用程序的更多详细信息，请参阅运行和调试Node.js。

调试CoffeeScript代码

  1. 必要时在CoffeeScript代码中设置断点 。

  2. 使用CoffeeScript类型的文件监视器将CoffeeScript代码编译为Javascript 。

  3. 使用以下必需设置开始创建Node.js运行配置 ：

    1. 要使用的Node.js引擎。默认情况下，该字段显示路径上指定的解释的Node.js Node.js的配置过程中的页面。

    2. 在“ 工作目录”字段中，指定从要运行的起始CoffeeScript文件引用的文件的位置，例如， 包括 。如果此文件未引用任何其他文件，则只需将该字段留空。

    3. 在Path to Node App JS File字段中，指定编译期间从原始CoffeeScript文件生成的JavaScript文件的完整路径。

  4. 保存配置并单击在工具栏上。

  5. 继续在本地调试Node.js应用程序时继续。

配置语法突出显示

您可以根据自己的喜好和习惯配置CoffeeScript感知语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案| CoffeeScript 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
