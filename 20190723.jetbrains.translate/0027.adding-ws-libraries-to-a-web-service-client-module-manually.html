手动将WS库添加到Web服务客户端模块

此功能仅在Ultimate版本中受支持。

如果先前已下载必要的库，则此方法适用。

将必要的库添加到现有模块

  1. 打开“模块设置”对话框。

  2. 选择所需模块后，单击“ 依赖关系”选项卡。

  3. 再次单击“ 添加”按钮，然后从上下文菜单中选择“ 单条目模块库 ”。在打开的对话框中 ，选择javaee.jar库和所需的特定于Web服务的库。

    1. javaee.jar库的位置是在安装IntelliJ IDEA期间定义的。

    2. Web服务特定库的位置在下载期间定义。

  4. 准备好后单击确定 。
