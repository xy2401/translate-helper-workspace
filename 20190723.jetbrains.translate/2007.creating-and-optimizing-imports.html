自动导入

当您引用尚未导入的类时，PyCharm会帮助您找到此文件并将其添加到导入列表中。您可以导入单个类或整个包，具体取决于您的设置。

import语句将添加到imports部分，但插入符号不会从当前位置移动，并且您当前的编辑会话不会挂起。此功能称为“ 导入助手” 。

要配置自动导入，请在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中转到“ 编辑器”|一般|自动导入 ，并使用相应的XML，Python和TypeScript / JavaScript 选项 。

使用Import Assistant是处理PyCharm中导入的首选方法，因为命令行不支持导入优化。

同样的可能性适用于XML文件。键入带有未绑定名称空间的标记时 ，导入助手会建议创建名称空间并提供适当选择的列表。

优化进口

迟早，一些导入的类或包变得对代码是多余的。PyCharm提供了Optimize Imports功能，只要方便，您就可以从当前文件或当前目录中的所有文件中删除未使用的导入。这有助于您避免项目中未使用，过多和重复的导入。可以删除整个项目或当前文件中未使用的import语句。

除了从未使用的导入中清除代码外，PyCharm还根据Python代码样式指南格式化现有的import语句。在这样做时，PyCharm将import语句拆分为单独的行，并将它们分组到组中（有关详细信息，请参阅Imports部分）。

此外，导入按字母顺序排列，并且区分大小写在相应的组中：

您可以在Python代码样式设置的导入选项卡中修改排序策略（ 设置/首选项|编辑器|代码样式| Python ）。有关更多信息，请参阅Python代码样式设置 。

优化整个项目中的导入

  1. 将焦点切换到“项目”工具窗口，然后执行以下操作之一：

    • 在主菜单上，选择Code |优化进口 。

    • 按Ctrl + Alt + O.

    “ 优化导入”对话框将打开。

  2. 如果项目受版本控制，则启用“ 仅VCS已更改文件 ”选项。根据需要选择或清除此选项。

  3. 单击运行 。

处理未使用的导入的一种方法是使用在突出显示的未使用导入时设置插入符时出现的快速修复。但是，您可以在更大范围内优化导入，如下所述。

优化当前文件中的导入

  1. 在主菜单上，选择Code |优化进口 。

  2. 按Ctrl + Alt + O.

  3. 将插入符号放在import语句中，单击 ，然后选择删除未使用的导入 。

  4. 打开“ 重新格式化文件”对话框 （ Ctrl + Shift + Alt + L ），然后选择“ 优化导入”复选框。

在代码完成时添加import语句

完成导出的JavaScript或TypeScript符号时，PyCharm会添加import语句。

您可以在完成时禁用自动导入并使用快速修复：

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ”，然后单击“ 自动导入” 。将打开“ 自动导入”页面。

  2. 使用TypeScript / JavaScript区域中的复选框可以在代码完成时启用或禁用导入生成。
