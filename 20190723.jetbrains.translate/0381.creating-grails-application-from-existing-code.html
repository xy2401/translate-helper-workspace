从现有代码创建Grails应用程序

此功能仅在Ultimate版本中受支持。

如果您已经有Grails项目，则可以围绕其源代码创建IntelliJ IDEA Grails应用程序。IntelliJ IDEA识别结构并创建Grails项目视图。

如果您使用Grails 3，那么您可以简单地导入Gradle模型的项目，因为Grails 3使用Gradle构建系统。

对于Grails 1/2，请使用以下过程：

  1. 启动“ 新建项目”向导 。如果IntelliJ IDEA中当前未打开任何项目，请单击“欢迎”屏幕上的“ 导入项目 ”。否则，请选择文件|新的|来自主菜单中的现有来源的项目 。

  2. 在打开的对话框中，选择Grails项目，然后单击“ 确定” 。

  3. 在向导的下一页上，选择“ 从现有源创建项目” ，然后单击“ 下一步” 。

  4. 在向导的下一页上，选择“ 从现有源创建项目” ，然后单击“ 下一步” 。

  5. 在向导的下一页上，指定Grails项目的名称和位置，然后单击“ 下一步” 。

  6. 在向导的下一页上，选择要包含在项目中的源根，然后单击“ 下一步” 。

  7. 在向导的下一页上，选择将包含在项目中的库，然后单击“ 下一步” 。

  8. 在向导的下一页上，查看将添加到项目的模块结构和依赖项，然后单击“ 下一步” 。

  9. 在向导的下一页上，选择项目SDK，然后单击“ 下一步” 。

  10. 在向导的下一页上，IntelliJ IDEA会在检测到框架时显示框架。单击完成 。

IntelliJ IDEA以其专用结构打开Grails项目。此时，如果IntelliJ IDEA在项目中检测到Gradle构建工具，则会显示建议导入Gradle项目的弹出消息。
