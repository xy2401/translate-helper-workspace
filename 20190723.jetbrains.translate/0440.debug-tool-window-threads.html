上下文菜单选项

项目

描述

暂停

选择此选项可暂停所选线程。线程挂起时，此菜单选项切换为“ 恢复” 。

打断

使用此命令中断活动线程。

丢帧

使用此命令删除选定的帧 ，即在调试时返回。此选项仅在有两个或更多帧时可用。

添加步进过滤器

使用此命令在打开的对话框中添加步进过滤器。

导出线程

使用此项打开“ 导出线程”对话框 ，该对话框允许您将线程导出到指定的文本文件。

自定义线程视图

使用此项目管理“ 帧”选项卡的内容。例如，您可以选择显示线程组，行号等。有关详细信息，请参阅对话框说明 。
