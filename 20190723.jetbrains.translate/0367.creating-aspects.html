创造方面

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA允许您以两种形式创建方面： .aj文件和包含使用@Aspect注释的类的.java文件。

请注意，在您开始创建方面时，您必须已启用AspectJ支持插件并将aspectjrt.jar定义为库 。

创建一个方面

  1. 在“ 项目工具”窗口中 ，执行以下操作之一：

    • 选择源目录或要在其中创建新方面的包，然后选择“ 文件”|新的 。

    • 选择源目录或包，然后按Alt + Insert 。

    • 右键单击源目录或包以打开上下文菜单，然后选择“ 新建” 。

  2. 选择Aspect 。

  3. 在打开的对话框中：

    1. 在“ 名称”字段中指定方面的名称 。

    2. 从种类列表中选择以下选项之一：

      • Aspect用于创建具有指定名称的.aj文件。

      • @Aspect创建一个包含用@Aspect注释的类的.java文件。

    3. 单击确定 。
