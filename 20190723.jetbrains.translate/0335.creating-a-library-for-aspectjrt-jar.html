为aspectjrt.jar创建库

此功能仅在Ultimate版本中受支持。

启用AspectJ支持插件后 ，添加包含aspectjrt.jar的库 ，然后添加具有相应依赖关系的AspectJ 方面 （如果它们是开发方面所必需的）。

aspectjrt.jar包含在AspectJ发行版中，您可以从AspectJ网站下载它。

要为AspectJ创建库：

  1. 打开“ 项目结构”对话框（ Ctrl + Shift + Alt + S ）。

  2. 根据您是要在项目还是IDE级别创建新库，请选择“ 库”或“ 全局库” 。

  3. 单击“ 添加”按钮（ ）并选择Java 。

  4. 选择aspectjrt.jar文件，该文件应位于<AspectJ installation directory>\lib或您下载的任何位置。

  5. 在“ 选择模块”对话框中，选择要在其中使用此库的模块，然后单击“ 确定” 。

要添加AspectJ依赖项：

  1. 打开“ 项目结构”对话框（ Ctrl + Shift + Alt + S ）。

  2. 选择Facets 。

  3. 单击“ 添加”按钮（ ）并选择AspectJ 。

  4. 在“ 选择模块”对话框中，选择要在其中使用此构面的模块，然后单击“ 确定” 。

  5. 为相关模块选择新的AspectJ方面 ，并将依赖项添加为相应方面（ aspectpath ）的路径。
