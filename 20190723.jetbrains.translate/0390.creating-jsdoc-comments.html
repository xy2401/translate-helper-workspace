创建JSDoc注释

此功能仅在Ultimate版本中受支持。

使用JSDoc 简介中的示例查找JSDoc语法的详细说明。

IntelliJ IDEA可帮助您创建JSDoc注释并使用它们进行JavaScript文档查找，使用Ctrl + Q.IntelliJ IDEA识别文档注释中的TODO模式和Closure Compiler注释，并在代码完成，意图操作和其他类型的编码帮助中使用它们。

启用JSDoc注释

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“ 智能键” 。将打开“ 智能钥匙”页面 。

  2. 在Enter区域中，选择Insert documentation comment stub复选框。

创建JSDoc注释

  • 将光标放在方法/函数或字段的声明之前，键入开始块注释/** ，然后按Enter键 。描述列出的参数，返回值等。

  • 或者，使用专用的“ 修复文档注释”操作。将光标放在要记录的方法/功能或字段上，按Ctrl + Shift + A ，从列表中选择修复文档注释 ，然后按Enter键 。
