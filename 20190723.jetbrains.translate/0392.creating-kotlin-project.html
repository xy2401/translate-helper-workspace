创建Kotlin-JVM项目

要创建Kotlin-JVM项目，请按照下列步骤操作

  1. 在主菜单上，选择“ 文件”|新的|项目 。

  2. 在“ 新建项目”对话框中，选择项目类型Kotlin-JVM 。

  3. 单击下一步 。

  4. 在向导的下一页上，指定以下内容：

    • 项目名称和位置。

    • Kotlin运行时库。从下拉列表中选择所需的库，如果缺少，则单击“ 创建” 。

    如有必要，请单击 （ 更多设置 ）并根据需要指定设置。

  5. 单击完成 。

根据您的选择创建项目并在窗口中打开。
