创建AIR描述符模板对话框

使用此对话框为AIR（桌面或移动）应用程序配置和创建应用程序描述符模板。

在编译或打包期间，模板的<content>元素中的文本将替换为应用程序文件的名称和扩展名（ .swf ）。

  • AIR应用程序描述符
  • 应用属性
  • 移动选项
  • Android标签
  • iOS标签

AIR应用程序描述符

项目

描述

文件名

指定要生成的描述符模板.xml文件的名称。

夹

指定应存储生成的描述符模板的文件夹。

使用 （ Shift + Enter ）选择相应对话框中的文件夹。

AIR版本

指定（选择或键入）目标AIR版本。

应用属性

项目

描述

ID

指定应用程序ID。

名称

指定应用程序名称，即要显示为应用程序标题的文本。

版

以xxx格式指定应用程序版本。

请注意，这不是AIR版本 。

移动选项

项目

描述

移动平台

选择应用程序描述符所针对的平台（Android或iOS）。

常见选项

选择或取消选择以下屏幕选项 ：

  • 自动定向。 如果选中，则对应于<autoOrients>true</autoOrients> 。

  • 全屏。 如果选中，则对应于<fullScreen>true</fullScreen> 。

Android标签

项目

描述

权限

使用复选框启用或禁用相应的Android权限 。

iOS标签

项目

描述

设备

选择支持的设备系列。如果支持iPhone / iPod Touch和iPad，请选择全部 。

高分辨率

如果您的应用程序应使用高分辨率屏幕模式，请选择此选项。
