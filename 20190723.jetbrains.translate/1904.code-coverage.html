基本

PyCharm中的代码覆盖允许您以低运行时开销为代码执行即时线覆盖测量。一般来说，线路覆盖率回答了这样一个问题：“在单元测试模拟过程中是否执行了这一行代码？“

代码覆盖率结果反映在专用Coverage工具窗口 ， Project Tool窗口的Project视图和编辑器中。工具窗口显示以下信息：

  • 对于目录：覆盖的类和行的百分比。

  • 对于文件：包含行的百分比。

在编辑器中打开文件时，每行都会突出显示其代码覆盖状态：

  • 模拟期间执行的行标记为绿色。

  • 模拟期间未执行的行标记为红色。

覆盖测量结果包括覆盖套件 。您可以将新模拟的结果与任何现有套件合并。在这种情况下，如果一条线被至少一个模拟覆盖，则该线将被视为已覆盖。

每次执行具有代码覆盖率测量的测试或应用程序时，都会生成覆盖套件 。可以拥有无限量的覆盖套件。

运行代码覆盖率

要在项目中使用代码覆盖率，请遵循以下常规步骤

  1. 指定您希望如何处理覆盖率结果 。

  2. 如果要测量代码覆盖率以进行测试 ，请为目标代码创建测试。

  3. 在所需的运行/调试配置中配置代码覆盖率测量 。

  4. 使用主菜单Run |上的专用命令运行coverage 使用Coverage运行 ，或 。

  5. 执行覆盖率运行后，您可以执行以下操作：

    • 使用各种覆盖套件 。

    • 查看代码覆盖率数据 。

    • 生成代码覆盖率报告 。
