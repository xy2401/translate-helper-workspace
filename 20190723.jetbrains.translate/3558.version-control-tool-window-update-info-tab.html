范围过滤器

仅当在“ 版本控制”下的“设置/首选项”对话框（ Ctrl + Alt + S ）中选择了“ 按范围筛选更新项目信息”选项时，此操作才可用。单击此图标仅显示所选范围的更新信息。

显示差异

单击此按钮可打开文件的差异查看器 ，您可以在其中将所有项目文件的本地副本一个接一个地与服务器中的更新进行比较。使用比较下一个文件 并比较以前的文件 用于滚动更新文件列表的按钮。

显示差异
按Ctrl + d

部署

N / A

该选项仅在启用了Remote Hosts Access插件且至少定义了一个服务器访问配置时可用。

选择此选项可调用包含与部署相关的操作的弹出菜单：

  • 上传到 ：选择此选项或按Ctrl + Shift + Alt + X ，然后选择要上传项目的服务器或服务器组 。如果指定了默认服务器或服务器组，则还可以使用其他上载到<默认部署服务器或服务器组>操作。

  • 与deploy to比较 ：选择此选项，然后选择服务器以将单个文件与其远程版本进行比较。如果指定了默认服务器或服务器组，则还可以使用“部署到<默认部署服务器>”操作的其他“ 比较” 。

  • 与部署同步 ：选择此选项，然后选择服务器以将本地项目与其远程版本同步。

有关详细信息，请参阅上载和下载文件以及将已部署的文件和文件夹与其本地版本进行比较。
