查找用法。课程选择

编辑|找到|查找用法
Alt + F7

本节介绍在“ 查找用法”对话框中指定“类使用搜索”和“接口使用搜索”选项的控件。

单击时将打开该对话框在“ 显示用法”弹出窗口中，其中列出了插入符号中符号的所有出现位置。

项目

描述

找

在此区域中，指定要搜索的对象。

  • 用法 - 如果选中此复选框， 则按名称对该类的所有引用执行搜索。

  • 方法的用法 - 如果选中此复选框，则对所选类方法的所有调用执行搜索。

  • 字段的用法 - 如果选中此复选框，则会对所选类字段的用法执行搜索。

  • 派生类 - 如果选中此复选框，则会对扩展所选类的所有类执行搜索。

    该复选框仅适用于类使用搜索。

  • 实现类 - 如果选中此复选框，则会对实现所选接口的所有类执行搜索。

    该复选框仅适用于界面使用搜索。

  • 派生接口 - 如果选中此复选框，则会对扩展所选接口的所有接口执行搜索。

    该复选框仅适用于界面使用搜索。

选项

在此区域中，使用以下控件配置搜索过程：

  • 搜索文本出现 - 如果选中此复选框，则在IntelliJ IDEA中注册的文件中执行搜索。

  • 使用一种用法跳过结果选项卡 - 如果只找到一个用法，则选中此复选框以直接导航到找到的用法，而不显示“ 查找”工具窗口。

范围

在此区域中，指定搜索范围 。从下拉列表中选择预定义的范围，或单击“ 浏览”按钮打开“ 范围”对话框，您可以在其中定义自定义范围。

在新标签页中打开

选中此复选框可使每个搜索的结果显示在“ 查找结果”窗口的单独选项卡中。如果清除该复选框，搜索结果将覆盖当前选项卡的内容。
