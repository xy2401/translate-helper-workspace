Ctrl + Alt + C

项目

描述

常数类型

IntelliJ IDEA自动确定字段类型。

名称

在此指定新常量的名称。

上课

选择将引入常量的类。

作为枚举常量引入

如果您在“ 引入类”字段中选择了枚举类，则可以使用此选项选择，是要将此常量作为枚举常量引入，还是作为常规字段引入。否则，此选项自然会被禁用，不会影响任何内容。

能见度

选择新字段的可见范围。

替换所有出现的事件

选中此选项可自动替换所选表达式的所有匹配项（如果在类中多次找到所选表达式）。

删除变量声明

选中此选项可删除变量声明。

将字段注释为@NonNls

选中此选项可避免本地化期间的更改。

  • 仅当项目配置为使用注释时，此字段才可用，即库annotation.jar已添加到项目或模块中。如果注释不可用，则此选项不会出现在对话框中。

  • 项目语言级别应为5.0以支持注释。
