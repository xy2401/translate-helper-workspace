在重构中使用Push ITDs

此功能仅在Ultimate版本中受支持。

对于方面，IntelliJ IDEA支持大多数基本重构，例如Move和Rename 。除此之外，您还可以在重构中执行Push ITDs以进行类型间声明 。

Push ITDs在重构中，您可以将字段和方法的定义从方面移动到相应的类。（重构名称中的ITD代表Inter-Type声明 。）

可以针对单个字段或方法，针对整个方面或针对包中的所有方面执行重构。

如果在执行重构时某些方面变为“空”（当相应方面只包含类型间声明时可能就是这种情况），您可以选择自动删除所有这些方面。

在重构中执行推送ITD

  1. 根据重构的预期范围：

    • 要对单个字段或方法执行重构，请在编辑器中打开感兴趣的方面，并将光标放在字段或方法的声明中。

    • 要对整个方面执行重构，请在“ 项目”工具窗口中选择感兴趣的方面。或者，在编辑器中打开方面并将光标放在单个类型间声明之外的某处（例如，在方面的声明中）。

    • 要对包中的所有方面执行重构，请在“ 项目”工具窗口中选择包。

  2. 选择Refactor |在主菜单或上下文菜单中按ITDs 。

  3. 在打开的Push Inter-Type Declarations In对话框中：

    1. 选中或清除“ 删除空方面”复选框。

    2. 单击“ 重构”立即执行重构，或单击“ 预览”以在实际执行重构之前研究预期的更改。

  4. 如果在上一步中单击了“ 预览” ，则“ 查找”工具窗口将打开，显示将受影响的类型间声明。如果对预期结果满意，请单击“ 执行重构” 。
