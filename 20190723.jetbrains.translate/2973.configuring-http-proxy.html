配置HTTP代理

Subversion将http代理设置存储在用户运行时配置区域中的servers文件中（Unix系统上的~/.subversion/auth/或Windows上的<USER HOME>/Application Data/Subversion/auth/ ）。

您可以通过两种方式配置Subversion代理设置：

  • 手动编辑服务器文件 。

  • 直接从WebStorm配置代理设置。请执行下列操作：

    1. 打开版本控制| WebStorm设置的Subversion页面（ Ctrl + Alt + S ），然后打开网络选项卡。

    2. 单击“ 编辑网络选项”按钮，然后在打开的“ 编辑与网络层相关的子版本选项”对话框中指定代理设置。
