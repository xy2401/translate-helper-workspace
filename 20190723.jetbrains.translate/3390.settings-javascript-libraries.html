项目

描述

加

单击此按钮可在“ 新建库”对话框中创建新的JavaScript库。请参阅配置JavaScript库一节。

编辑

单击此按钮可在“ 编辑库”对话框中更改所选库的名称和内容。

去掉

单击此按钮可删除所选的库。

下载

单击此按钮可打开“ 下载库”对话框，其中包含可用于TypeScript定义文件的存根的库列表。选择所需的库，然后单击“ 下载并安装” 。WebStorm将您带回到JavaScript库页面，其中新库已添加到列表中。单击“ 确定”保存设置。

管理范围

单击此按钮可配置用于JavaScript中特定文件和/或目录的库。使用范围对话框。

项目

描述

名称

指定库名称。

框架类型

从此下拉列表中，选择要配置为库的框架。

版

在此文本框中，指定要使用的所选框架的版本。

能见度

在此区域中，指定希望库可用于与文件和文件夹关联的位置。可用选项包括：

  • 当前项目：选择此选项时，库只能与当前项目中的文件和文件夹关联。如果您稍后尝试将该框架与另一个项目一起使用，则必须重新配置该库。

  • 全局：选择此选项可启用将库与任何WebStorm项目相关联。
