创建EJB

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA提供了以下创建bean的方法：

  • 手动，通过直接编辑 ejb-jar.xml文件，使用IntelliJ IDEA提供的大量编码帮助。

  • 使用上下文菜单命令。

使用编辑器创建Enterprise Java bean

  1. 打开部署描述符文件ejb-jar.xml以进行编辑。

  2. 创建<enterprise-beans>部分。

  3. 在本节中，为所需的bean类型创建标记： <session> ， <entity>或<message-driven >

  4. 通过添加标记<ejb-name>指定bean名称和类。和<ejb-class> 。请注意，您可以在此标记中指定目标包，例如samples.ejb.ManuallyCreatedEntity ，在每个点后使用代码完成。

  5. 如果声明的类尚不存在，IntelliJ IDEA建议快速修复：

    从建议列表中选择Create Class <class name> 。如果未在<ejb-class>标记中定义目标包，请在“ 选择目标目录”对话框中选择所需的包。存根类在指定位置创建。

  6. 继续定义其他bean组件。

要创建Enterprise Java bean，请遵循以下常规步骤

  1. 打开EJB工具窗口 。

  2. 右键单击所需模块，在上下文菜单中选择“ 新建 ”，然后在子菜单上选择bean类型。

    将打开New <bean type> Bean对话框。

    或者，在模块的上下文菜单中选择Jump to Source ，打开专用编辑器，在General选项卡中单击New ，或右键单击EJB Relationships选项卡中的图表背景，然后选择所需的bean类型。

  3. 指定bean名称，类和包 。

  4. 定义特定于bean的设置：

    • 对于Entity bean（1.x和2.x）和会话bean（1.x和2.x）， 配置本地和远程接口 。

    • 对于Entity bean（1.x和2.x）， 配置主键和CMP bean字段 。

    • 对于Message bean， 配置消息侦听器 。

    • 对于会话bean，请配置服务端点 。
