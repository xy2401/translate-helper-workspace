配置库对话框

使用此对话框可以编辑库名称并管理库内容。

可用控件的集合取决于您使用的是Java或ActionScript / Flex库还是JavaScript库。

Java或ActionScript / Flex库的控件

项目

描述

更改版本

在IntelliJ IDEA可以在库中进行特定于版本的文件替换的情况下，此按钮可用于实现特定框架或技术的库（例如，JSF或Spring）。

在这种情况下，单击此按钮时，将打开“ 下载选项”对话框 ，您可以在其中选择所需的库版本以及要下载的文件。

因此，库中的文件将替换为下载的文件。

（ Alt + Insert ）

将项目（类，源，文档等）添加到库中。

在打开的对话框中，选择所需的文件和文件夹。对于Java库，这些可以是单独的.class和.java文件，包含此类文件的目录和存档（ .jar和.zip ）以及包含Java本机库（ .dll ， .so或.jnilib ）的目录。对于ActionScript / Flex库，这些库可能是原始ActionScript 3库， .swc ， .jar和.zip文件，包含此类文件的目录，等等。

IntelliJ IDEA将分析所选文件和文件夹，并自动将其内容分配给相应的库类别（类，源，文档，本机库位置等）。

当IntelliJ IDEA无法猜出类别时（例如，当您选择空文件夹时），将显示一个对话框，您可以在其中自行指定类别。

为了能够使用在线提供的外部文档，请单击此图标并指定外部文档的URL。

排除项目（请参阅排除库项目 ）。在打开的对话框中，选择您希望IntelliJ IDEA忽略的项目（存档中的文件夹，存档和文件夹），然后单击“ 确定” 。

（ 删除 ）

此选项允许您：

  • 从库中删除选定的普通库。

  • 恢复所选的已排除项目。这些物品本身将留在图书馆。

JavaScript库的控件

项目

描述

（ Alt + Insert ）

将项目添加到库中。选择以下选项之一：

  • 附加文件或目录。 选择此选项可添加JavaScript文件。

    在打开的对话框中，选择所需的文件和文件夹。这些可能是单独的JavaScript文件和包含此类文件的目录。

    IntelliJ IDEA将分析选定的文件和文件夹，并自动将JavaScript文件分配给适当的类别。缩小的文件将分配给“发布”类别;普通（未压缩）文件将分配给Debug类别。

    当IntelliJ IDEA无法猜测类别时（例如，当您选择空文件夹时），将显示一个对话框，您可以在其中自行指定类别（发布或调试）。

  • 附加调试版本。 选择此选项可添加单个未压缩的JavaScript文件或包含此类文件的目录。

    请注意，IntelliJ IDEA不会分析所选文件的内容。如果选择缩小的JavaScript文件或包含缩小的JavaScript文件的目录，则相应的文件仍将添加到库中。

  • 附上发行版本。 选择此选项可添加单个缩小的JavaScript文件或包含此类文件的目录。

    请注意，IntelliJ IDEA不会分析所选文件的内容。如果选择普通（未压缩）JavaScript文件或包含此类文件的目录，则相应的文件仍将添加到库中。

  • 指定文档URL。选择此选项可在IntelliJ IDEA中提供外部在线文档。在打开的对话框中指定文档URL。

  • 下载文档。 对于jQuery：选择此选项以下载并在库中包含jQuery文档。

排除项目（请参阅排除库项目 ）。在打开的对话框中，选择您希望IntelliJ IDEA忽略的文件夹，然后单击“ 确定” 。

（ 删除 ）

此选项允许您：

  • 从库中删除选定的普通库。

  • 恢复所选的已排除项目。这些物品本身将留在图书馆。
