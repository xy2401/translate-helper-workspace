跟踪代理服务器选项卡

该选项卡包含一个工具栏和显示系统信息和错误消息的控制台区域，有关Spy-js会话状态的信息，代理服务器的可用性，当前活动的帧（“工作人员”）等。

跟踪代理服务器工具栏

项目

工具提示和快捷方式

描述

停止跟踪代理服务器

单击此按钮可通过标准shutdown脚本在外部终止当前正在运行的进程。

上/下堆栈跟踪
Ctrl + Alt +向上 Ctrl + Alt +向下

使用这些按钮可以将堆栈跟踪从一个报告的错误导航到另一个报告错误。这些按钮仅在报告错误时可用，例如，如果配置文件已损坏。

使用软包装

单击此按钮可切换输出的软包装模式。

滚动到最后

单击此按钮可导航到堆栈跟踪的底部，并使光标跳转到源代码中的相应位置。

打印

单击此按钮可将控制台文本发送到默认打印机。

全部清除

单击此按钮可从控制台中删除所有文本。此功能也可在控制台的上下文菜单中使用。
