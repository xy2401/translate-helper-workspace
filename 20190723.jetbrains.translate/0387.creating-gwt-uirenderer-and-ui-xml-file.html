创建GWT UiRenderer和ui.xml文件

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA可以为您生成GWT UiRenderer和ui.xml文件。在ui.xml定义声明性布局时，可以使用各种类型的编码辅助，例如代码完成和导入语句的生成 。

IntelliJ IDEA支持ui.xml的声明性布局与相应的Java类之间的集成，包括查找用法 。

如果您使用的是ClientBundle接口，则IntelliJ IDEA会使用图标标记其方法在左水沟区域。单击此图标可导航到相应的资源。

GWT UiRenderer只能在客户端部分创建。

生成GWT UiRenderer

  1. 在项目视图中 ，右键单击客户端程序包。

  2. 从上下文菜单中，选择New | Google Web Toolkit | GWT Renderer和ui.xml文件 。

  3. 在打开的Create New GWT Renderer和ui.xml文件对话框中，指定UiRenderer的名称和ui.xml文件。
    单击确定 。
