第一次运行IntelliJ IDEA

当您第一次运行IntelliJ IDEA时，或者从先前版本升级后，需要执行一些步骤来完成安装，自定义实例并开始使用IDE。

以下视频教程将指导您完成以下步骤：

导入IntelliJ IDEA设置

在“ 完整安装”对话框中，选择是否要导入IDE设置：

如果这是IntelliJ IDEA的第一个实例，请选择“不导入设置”选项。

您可以使用File |在以后手动导入和导出设置 导入设置和文件|主菜单上的“ 导出设置”命令。

选择用户界面主题

选择是否要使用默认的Darcula或Light主题。

禁用不必要的插件

IntelliJ IDEA包括提供与不同版本控制系统和应用程序服务器集成的插件，添加对各种框架和开发技术的支持等。要提高性能，可以禁用不需要的插件。如有必要，您可以稍后在“ 插件”下的“设置/首选项”对话框（ Ctrl + Alt + S ）中启用它们。有关更多信息，请参阅管理插件 。

您可以单击每组插件的“ 全部禁用”链接以全部禁用它们，或者单击“ 自定义”以禁用组中的各个插件。

下载并安装其他插件

如有必要，请从IntelliJ IDEA插件存储库下载并安装未与IDE捆绑在一起的其他插件。

在IntelliJ IDEA中启动项目

在“ 欢迎”屏幕上，您可以执行以下操作：

  • 创建一个新项目
  • 导入项目

  • 打开现有项目或文件

  • 从版本控制系统中检出现有项目

也可以将现有项目目录或单独的文件拖到“欢迎”屏幕，然后在IntelliJ IDEA中将其打开。
