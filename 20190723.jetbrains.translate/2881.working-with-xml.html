PyCharm 2019.1帮助

XML

PyCharm带来强大的XML编辑支持，包括：

  • 验证和语法突出显示。

  • 代码完成 （ Ctrl + Space ）。

  • 缩进 （ Ctrl + Alt + I ， Ctrl + Alt + L ）。

  • 根据XML代码样式 格式化 （ Ctrl + Alt + L ）。

  • 意图操作 （ Alt + Enter ）。

  • 导入未绑定的命名空间 。

  • 查看代码结构 （ Alt + 7 ）。

  • 在源代码中导航 （ Ctrl + B ）。

  • 集成文档 （ Ctrl + Q ）。

  • 搜索用法 （ Alt + F7 ）。

  • 注释和取消注释行 （ Ctrl + / ， Ctrl + Shift + / ）。

  • 展开和删除标签 （ Ctrl + Shift + Delete ）。

  • 从实例文档生成模式，反之亦然。

  • 生成DTD文件 。

PyCharm使用Xerces 2.11 ，这是Apache Software Foundation Group开发的XML解析器。

导入XML命名空间

  1. 打开所需的文件进行编辑，然后开始键入标签。如果未绑定命名空间，则会出现以下提示：

  2. 按Alt + Enter 。如果有多个选项，请从列表中选择所需的命名空间。

配置语法突出显示

您可以根据自己的喜好和习惯配置XML感知语法突出显示。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案| XML 。

  2. 选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
