代码风格

如果公司中存在某些编码指南，则在创建源代码时必须遵循这些指导原则。PyCharm可帮助您维护所需的代码样式。

代码样式在项目级别和IDE级别 （全局）定义。

  • 在项目级别，设置按项目方案分组，该方案是预定义的，并以粗体标记。项目样式方案仅适用于当前项目。

    您可以使用“ 复制到IDE ...”命令将项目方案复制到IDE级别。

  • 在IDE级别，设置按预定义的默认方案（以粗体标记）和用户通过“ 复制”命令（标记为纯文本）创建的任何其他方案进行分组。当用户不希望将代码样式设置保留在项目中并共享它们时，将使用全局设置。

    您可以使用“ 复制到项目...”命令将IDE方案复制到当前项目。

配置语言的代码样式

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“代码样式” ，然后单击该语言。

  2. 选择代码样式方案，以用作所选语言的自定义编码样式的基础。

  3. 浏览所选语言页面的选项卡，并为其配置代码样式首选项。

仅显示这些语言的Set from链接，其中可以使用其他语言定义设置。

从其他语言复制代码样式设置

对于大多数受支持的语言，您可以从其他语言或框架复制代码样式设置。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“代码样式” ，然后单击该语言。

  2. 单击右上角的“ 设置自 ”。

  3. 从显示的列表中，选择要从中复制代码样式的语言。

使用EditorConfig配置项目的代码样式

在开始使用EditorConfig之前，请确保已启用EditorConfig插件。该插件与PyCharm捆绑在一起，默认情况下处于激活状态。如果未激活插件 ，请在“ 管理插件 ”中所述的“设置/首选项”对话框的“ 插件”页面上启用它。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|代码样式并确保选中 “ 启用EditorConfig支持”复选框 。

  2. 在项目树中，右键单击要在其中创建EditorConfig设置文件的目录，然后选择“ 新建”|档案 。

  3. 在打开的对话框中，输入.editorconfig并单击“ 确定” 。

    PyCharm创建一个EditorConfig设置文件，并在弹出窗口中显示通知 。

    每次打开文件时，EditorConfig插件.editorconfig在打开的文件的目录和每个父目录中查找名为.editorconfig的文件。如果到达根文件路径或找到root = true的EditorConfig文件，将停止搜索.editorconfig文件。因此，如果要使用IDE设置而不是EditorConfig设置，请清除编辑器中的“ 启用EditConfig支持”复选框。默认选择的代码样式 。

  4. 定义代码样式设置并保存.editorconfig （ Ctrl + S ）以立即将设置应用于项目。EditorConfig代码样式配置会覆盖PyCharm设置中的代码样式配置。

从EditorConfig官方网站了解更多信息。
