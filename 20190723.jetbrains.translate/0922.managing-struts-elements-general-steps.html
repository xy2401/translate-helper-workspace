管理Struts元素 - 一般步骤

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA为管理Struts元素提供了用户友好的可视化界面，包括：

  • 创建 ， 编辑和删除元素。

  • 处理元素的属性 。

  • 指定和编辑元素的属性和属性 。

可用于特定元素的一组操作和属性取决于其性质。

创建元素

所有类型的元素都以相同的方式创建。

或者，右键单击树中的元素，然后选择“ 跳转到源”或按F4 。这将带您到文本视图中的struts-config.xml ，您可以在其中手动指定元素的属性。IntelliJ IDEA显示一个模板，用于指定特定类型元素必需的属性。

  1. 打开struts-config.xml 。

  2. 切换到Struts Assistant工具窗口 ，选项卡Struts 。

  3. 在结构树中右键单击相应的元素类型，然后从上下文菜单中选择“ 添加”操作。

  4. 在“ 属性表”中指定新元素的属性 。

删除元素

所有类型的元素都以相同的方式删除。

  • 在结构树中右键单击它，然后从上下文菜单中选择“ 删除”操作。

查看或编辑元素

  1. 在结构树或Struts Web流程图中选择它，并在属性表中进行必要的更改。

  2. 要向元素添加属性，请右键单击“ 结构树”中的元素，然后在上下文菜单中选择“ 添加 ”。

  3. 在“ 属性表”中指定属性的属性 。

  4. 要删除属性，请在“ 结构树”中右键单击该属性，然后在上下文菜单中选择“ 删除 ”

    删除属性时，也会删除其嵌套属性。
