命令行工具支持

档案|设置| Windows和Linux的命令行工具支持
WebStorm |偏好|命令行工具支持 macOS
按Ctrl + Alt + S

只有在按照管理插件中的说明安装并启用命令行工具支持插件时，此页面才可用。

WebStorm支持运行第三方或用户定义工具的命令。代码完成和参数提示通过特定于工具的XML描述符提供。

该页面显示了与WebStorm集成的所有自定义命令行工具的列表。xml描述符中具有不一致性的工具用。标记 。

项目

工具提示和快捷方式

描述

启用

选中此列中的复选框后，可以从WebStorm执行相应命令行工具中定义的命令。

别号

在此字段中，指定要在命令调用中使用的字符串，而不是该工具的完整路径。

工具路径

在此字段中，指定工具的可执行文件的位置。

类型

此只读字段显示第三方命令行工具的正式名称。仅当选中“ 显示工具类型”复选框时，该列才可用。对于自定义命令行工具，此列中的字段为空。

显示工具类型

如果选中此复选框，则“ 类型”列将添加到可用工具表中。

显示控制台

在此区域中，指定要输入命令的位置。可用选项包括：

  • 弹出窗口 - 选择此选项可在单独的弹出窗口中打开“ 命令行工具输入”窗格，并在其中键入命令。

  • 工具窗口 - 选择此选项可在专用命令行工具控制台工具窗口底部的输入字段中输入命令。

虽然WebStorm提供了运行命令的工具窗口，但强烈建议您使用内置的本地终端 。

控制台编码

从此列表中，选择字符集以在命令行工具控制台工具窗口中显示工具的输出。

加
Alt +插入

单击此按钮可打开“ 创建自定义工具”对话框。单击“ 是”时 ，WebStorm将打开“ 工具设置”对话框，以指定新工具的位置。

去掉
Alt + Delete键

单击此按钮可从列表中删除所选工具。

编辑

单击此按钮可更改所选工具的定义文件。

从可执行文件重新加载命令列表

单击此按钮可刷新所选工具的命令列表。

在编辑器中打开定义

单击此按钮可在编辑器中使用所选工具的命令打开xml文件。
