移动并复制重构

Move重构允许您在项目中移动类，函数，模块，文件和目录。在这样做时，PyCharm跟踪这些移动并自动更正源代码中移动符号的所有引用。

可以使用以下Move重构：

  • 移动文件重构将文件移动到另一个目录。

  • Move Directory重构将目录移动到另一个目录。

  • Move Module Members重构移动Python模块的顶级符号。

  • Make本地函数/方法顶级重构将方法或本地函数转换为顶级函数并将其移动到指定文件。

将文件或目录移动到另一个目录

  1. 在“ 项目”工具窗口中选择文件或目录。

  2. 从主菜单或编辑器上下文菜单中，选择Refactor |移动或按F6 。

  3. 在“ 目录”字段中，指定要将所选文件或文件夹移动到的文件夹。从列表中选择现有文件夹，或键入要创建的父文件夹的完整路径。

  4. 要根据重构结果更新对所选文件或文件夹的引用，请选中“ 搜索引用”复选框。

将功能/方法移动到顶层

此重构通过将对实例属性或变量的引用从封闭范围转换为参数并相应地更新现有用法，将本地函数或方法移动到顶级。

要将函数或方法移动到顶级：

  1. 将插入符号放在本地函数或方法名称中。

  2. 从主菜单或编辑器上下文菜单中，选择Refactor |移动或按F6 。

  3. 在打开的Make Method Top-Level对话框中，指定move的目标。您可以手动键入，也可以单击“浏览”按钮并在“ 选择目标文件”对话框中找到目标文件。

  4. 单击“ 重构”以执行重构，或单击“ 预览” ，以在“查找”工具窗口中显示预览。如果对预览结果满意，请单击“ 执行重构”确认移动。

复制重构

通过复制重构，您可以在不同或相同的目录中创建文件或目录的副本。

要执行复制重构：

  1. 在工具窗口中选择感兴趣的项目 （例如“ 项目”工具窗口）。或者，在编辑器中打开必要的类或文件。

  2. 执行以下操作之一：

    • 选择Refactor |复制主菜单或上下文菜单。

    • 按F5 。

    • 在“ 项目”工具窗口中，按住Ctrl键 ，然后将项目拖动到目标位置。

  3. 在打开的“ 复制”对话框中，指定要创建的副本的名称和位置，然后单击“ 确定” 。
