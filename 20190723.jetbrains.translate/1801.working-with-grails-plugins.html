使用Grails插件

此功能仅在Ultimate版本中受支持。

您可以使用从Grails存储库下载的Grails插件扩展Grails应用程序的功能。下载后，插件位于Grails应用程序的plugins目录下。每个插件都具有与整个Grails应用程序相同的目录结构。已安装插件的源目录包含在源路由中，并在模块树视图中标记为图标。

为了更好地与Grails存储库进行通信，IntelliJ IDEA提供了一个插件管理器，可以查看，安装和卸载插件。

从Grails应用程序与Grails存储库的第一次通信可能需要一些时间。

对于Grails版本2.3.0及更高版本，请参阅Grails文档 。

要打开Grails插件管理器，请执行以下操作之一

  • 在主菜单上选择工具| Grails |插件 。

  • 在“项目”视图的上下文菜单中，选择“ Grails”插件 。

查看可用的插件

  • 打开Grails插件管理器，单击刷新按钮如有必要，请在Grails插件对话框中查看插件列表：

您还可以通过运行list-plugins Grails目标来查看插件list-plugins 。为此，请按Ctrl + Alt + G ，然后在“ 运行Grails目标”对话框中键入list-plugins 。

安装或卸载Grails插件

  1. 打开Grails插件管理器。在“ 启用”列中，选择要安装的插件的复选框，然后清除要卸载的插件的复选框。然后单击Apply Changes 。

  2. 在“ 安装/卸载Grails插件”对话框中，查看插件列表。如有必要，请选择插件版本。单击确定 。

    您还可以使用install-plugin Grails目标install-plugin 。为此，请按Ctrl + Alt + G ，然后在“ 运行Grails目标”对话框中键入install-plugin <plugin name> 。
