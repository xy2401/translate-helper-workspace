启用Scala插件后，此页面可用。有关更多信息，请参阅管理插件 。

编译器

如果Play2编译器在项目导入期间未自动生成，或者项目是使用gen-idea等外部工具创建的，请使用此选项卡手动指定Play2编译器的设置。

项目

描述

使用SBT观察者

选中此复选框以使用~compile:test命令单独执行SBT进程。在这种情况下，IntelliJ IDEA监视项目源并在检测到更改时重新编译项目。

将Play 2编译器用于此项目

选中此复选框以使用Play 2编译器版本显式启用此项目的Play 2编译器。

运行前不要在IDEA中编译项目

如果您不想在运行之前编译项目，请选中此复选框。SBT还将编译必要的源文件。

Play2模块

使用此字段为此项目指定Play 2模块。

项目uri

使用此字段指定SBT项目的根地址。对于导入的项目，SBT项目的根与IntelliJ IDEA项目的内容根相匹配。

其他SBT选项

使用此字段指定可以直接传递给SBT的其他选项。例如， -Xmx2048M 。

路线

使用此选项卡可配置项目中routes文件的设置。

项目

描述

路由格式化的最小空间

使用此字段为路由格式设置最小空间。

忽略路径文件中的URL深度

选中此复选框可忽略路径文件中的URL深度。

在Enter上重新格式化路由文件

选中此复选框可在按Enter键时重新格式化路径文件中的新条目。

其他

使用此选项卡可为Pla2项目配置其他设置。

项目

描述

刷新时排除'目标'目录

默认情况下，刷新项目时会排除“目标”目录。如果要在刷新项目时包含“目标”目录，请清除此复选框。

手动设置模板导入

选中此复选框可将Play2框架的导入手动添加到项目中。
