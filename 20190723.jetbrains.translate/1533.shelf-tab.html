货架标签

查看|工具窗口|版本控制 - 货架

当您搁置更改或更改列表时，此选项卡将添加到“ 版本控制”工具窗口 ，并显示，直到您永久删除所有搁置的更改（包括已经取消的更改）和导入的外部修补程序。

默认情况下，此选项卡显示尚未取消搁置的所有搁置更改。变更分为货架。货架是您搁置更改时创建的更改列表。通过提交消息标识架子。您可以让IntelliJ IDEA显示未经保留的更改。它们可以根据需要多次恢复和重新应用，直到它们被永久删除。

有关详细信息，请参阅搁置和取消搁置更改 。

您可以在“设置”|中为“ 显示立体”操作指定自定义快捷方式键图|版本控制系统 。

显示Diff Ctrl + D.

选择此选项可打开文件的差异查看器 ，并将文件的搁置版本与其当前本地版本进行比较。

默默地取消隐藏Ctrl + Alt + U.

单击此图标可以静默取消更改，而不显示“ 取消保护更改”对话框。

创建补丁

选择此选项可基于搁置的更改创建修补程序文件。在打开的“ 创建修补程序”对话框中，指定要保存修补程序的文件，以及从中创建修补程序的更改。默认情况下，会选择来自工具架的所有更改。要查看包含的更改，请单击“ 选定”链接。

有关详细信息，请参阅使用修补程序

显示/隐藏已经没有人

单击此按钮可使IntelliJ IDEA显示或隐藏所有可用的搁置更改，两者都已应用，而不是已应用。请注意，此按钮在上下文菜单中重复。

预览差异

单击此图标可将所选文件的搁置版本与“ 预览”窗格中的本地版本进行比较。

清洁已经没有了

如果您不打算重复使用，请单击此图标以显式删除所有未经保留的更改。

货架设置

单击此图标可跳转到Shelf设置页面，您可以在其中修改默认的货架位置，并为Git和Mercurial启用文件基本修订的自动搁置 。

上下文菜单

通过右键单击更改，更改列表或选项卡中的任何位置，可以使用上下文菜单。

捷径

描述


取消搁置

按Ctrl + Shift + U

选择此选项可应用所选工具架中的更改。在打开的“ 取消保护更改”对话框中，指定要将更改添加到的更改列表。有关详细信息，请参阅搁置和取消搁置更改 。

恢复

选择此选项可重新激活未保留的更改。默认情况下，未显示的更改不再显示在可用的搁置更改列表中，因此您首先需要让IntelliJ IDEA通过选择Show Already Unshelved来显示它。您可以根据需要多次还原任何更改，直到通过选择“ 删除”永久删除更改为止。有关详细信息，请参阅搁置和取消搁置更改 。


显示差异

按Ctrl + d

选择此选项可打开文件的差异查看器 ，并将文件的搁置版本与其当前本地版本进行比较。

与Local比较

选择此选项可将所选文件的搁置版本与文件差异查看器中的本地版本进行比较 。


创建补丁

选择此选项可基于搁置的更改创建修补程序文件。在打开的“ 创建修补程序”对话框中，指定要保存修补程序的文件，以及从中创建修补程序的更改。默认情况下，会选择来自工具架的所有更改。要查看包含的更改，请单击“ 选定”链接。

有关详细信息，请参阅使用修补程序

导入补丁

选择此选项可应用外部或IntelliJ IDEA创建的修补程序。在打开的对话框中，选择要从中导入修补程序的文件。导入的补丁被视为搁置的变化，并在货架标签，您可以在任何时候取消搁置他们所示。

改名

按住Shift + F6

选择此选项可修改所选搁置更改列表的名称。

删除

删除

选择此选项可永久删除选定的搁置更改。无论是否已经取消搁置，您都可以删除任何更改。有关详细信息，请参阅搁置和取消搁置更改 。

显示已经不受欢迎

选择此选项可让IntelliJ IDEA显示已应用和未应用的所有可用搁置更改。默认情况下，隐藏未更改的更改。有关详细信息，请参阅搁置和取消搁置更改 。


清洁已经没有了

如果您不打算重复使用，请单击此图标以显式删除所有未经保留的更改。
