使用Android UI Designer添加和编辑布局组件

布局定义活动或应用程序窗口小部件 （片段）的用户界面。布局在XML 资源定义文件中声明。有关如何创建资源文件夹和资源定义文件的说明，请参阅创建资源 。

使用IntelliJ IDEA，您可以构建应用程序的设计，而无需手动编辑布局定义文件，并检查应用程序设计在各种目标环境中的呈现方式，而无需在任何物理或虚拟设备上运行应用程序。

或者，可以手动编辑布局定义文件，可能使用IntelliJ IDEA提供的特定于Android的重构，并预览立即反映在专用预览工具窗口中的更改 ，您可以在其中调整布局到各种平台和设备。要切换到手动模式，请单击“ 文本”选项卡，或从“ 设计”窗格的上下文菜单中选择“ 转至声明” ，或单击“确定” 预览窗口工具栏中的图标 。

在可视模式下设计布局是在Android UI Designer工具窗口的“ 设计”面板中执行的。窗格位于UI Designer的中心部分（假设是默认的工具窗口布局）。打开布局定义文件进行编辑时，默认情况下窗格将显示在编辑器选项卡中。如果要手动编辑布局定义文件，然后通过单击“ 设计”选项卡切换到可视模式，则会在打开编辑的布局定义文件的选项卡中打开窗格，

该窗格显示一个矩形画布，该画布与当前布局定义文件和“ 组件树”视图同步，因此对画布的任何更改都会相应地反映在那里。如果IntelliJ IDEA检测到代码中存在任何差异，则会显示警告。

请注意，在设计模式下可以使用与手动编辑布局定义文件时相同的意图操作和快速修复。意图行动图标或快速修复图标显示在画布上，“ 组件树”或“ 属性”窗格中。

要构建Android应用程序的设计，请执行以下基本操作：

  • 从Palette添加预定义组件
  • 从Android SDK添加用户定义的组件和组件
  • 安排组件
  • 指定组件属性
  • 将组件转换为保留公共属性的其他类型

  • 预览布局

向画布添加预定义组件

执行以下操作之一：

  • 在“选用板”窗格中选择所需元素，然后将其拖放到“ 设计”窗格中的画布上。

  • 单击“组件面板”窗格中的所需元素，然后单击画布上的区域。

  • 单击“组件面板”窗格中的所需元素，然后单击要将此元素放置在的组件树 。

以任一方式添加的每个组件也会添加到组件树中，并在布局定义文件中声明。

添加在项目或Android SDK中定义的组件

  1. 在“ 选用板”窗格中展开“ 自定义”区域。

  2. 执行以下操作之一：

    • 要嵌入布局，请单击“ 包含”图标。在打开的“ 资源”对话框中，单击“ 项目”选项卡以在布局定义中搜索，或单击“ 系统”选项卡以在SDK中进行搜索。

    • 要添加组件组合，请单击“ 片段” ，然后选择要打开的“ 资源”对话框中嵌入的片段。

    • 要添加一个用户自定义视图 ，单击CustomView并选择在打开的对话框中插入视图。

将组件放置在正确的位置

画布与组件树同步，因此您可以通过在任一窗格中的任一组中移动它们来排列组件。

在画布上或组件树中选择所需组件并将其拖动到正确位置，或使用上下文菜单复制并粘贴它。

指定组件属性

您可以直接在画布中设置必需组件属性的值，也可以切换到“ 属性”窗格以进行更深入的配置。

  • 要在画布中指定基本属性的值，请双击相关组件并在显示的弹出窗口中指定值。

  • 要深入配置组件属性，请在画布或组件树中选择组件 ，切换到“ 属性”窗格，然后指定所选属性的值。

    单击“ 属性”窗格中的右列以开始编辑属性。你可以点击出现在右侧以选择或投射或系统资源。

    • 默认情况下，窗格仅显示一组标准属性，最常用的属性以粗体显示 。要使窗格显示根据规范为所选组件定义的所有属性，请单击“ 显示专家属性”按钮在工具栏上。

    • 要从Android参考 查看所选属性的简要文档，请单击“ 显示文档”按钮在工具栏上或按Ctrl + Q.

    • 具有更新值的属性以蓝色突出显示。要放弃所做的更改并返回默认值，请选择该属性，然后单击“ 恢复默认值”按钮在工具栏上。

将组件转换为另一种类型

在某些情况下，您可能需要将已完全配置的组件转换为另一种类型的组件。使用IntelliJ IDEA，您可以在不丢失指定属性的情况下执行此操作：两个类型共有的所有属性都将保留在新组件中。此操作称为变形 。

  1. 在画布中或在“ 组件树”中选择要转换的组件，然后在所选内容的上下文菜单中选择“ 变形 ”。

  2. IntelliJ IDEA显示兼容组件类型的列表，即可以转换所选组件的类型。选择目标类型并配置未在原始组件中配置的属性 。
