IntelliJ IDEA 2019.1帮助

Rails框架支持

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

在这个部分：

  • 条件
  • 重要笔记
  • Rails支持
  • 特定于Rails的程序

条件

应该下载 Rails框架并将其安装在您的计算机上。

重要笔记

  • Rails 3.2.x需要Ruby 1.8.7或更高版本。

  • Rails 4需要Ruby 1.9.x或更高版本。

  • Rails 5需要Ruby 2.2.2或更高版本。

Rails支持

IntelliJ IDEA中的Rails支持包括：

  • 专用项目类型，定义了所需的Rails版本。

  • 特殊Rails视图表示Rails应用程序的逻辑结构。

  • Rails生成器 。

  • Rails组件之间的导航方法。

  • 可以运行Rake任务 。

  • 定义对象关系映射的可能性。

  • 模型分析及其与模型依赖关系图的关系。

  • 完整的编辑帮助（语法和错误突出显示，Rails应用程序元素和Rake任务的代码完成）

  • Rails感知重构（ 重命名重构 ，将代码提取到局部视图等）。

  • Rails控制台 ，您可以在不离开IDE的情况下执行Rails命令。

  • 支持链轮 。

特定于Rails的程序

  • 创建基于Rails的项目
  • 创建Rails应用程序元素
  • 在View中注入Ruby代码
  • 将代码提取到局部视图中
  • 生成档案
  • 生成Rails应用程序的测试
  • 运行Rails脚本
  • 运行Rails服务器
  • Rails专用导航
  • 运行Rails控制台
  • 宙斯
  • IntelliJ IDEA中的Rails / Spring支持
