使用JAXB Dialog从Xml Schema生成Java

工具| JAXB |使用JAXB从XML模式生成Java代码

使用此对话框可以通过JAXB数据绑定器基于XML架构配置Java代码存根的生成。

  • 此功能通过WebServices捆绑插件提供，该插件默认启用。如果没有，请按照管理插件一节中的说明启用它。

  • 当在活动编辑器选项卡中打开的文件包含XML架构时，菜单项和对话框可用。

项目

描述

JAXB Schemas / wsdl / dtd路径

在此字段中，指定要用作生成基础的文件。默认情况下，该字段显示当前文件的完整路径。要使用其他架构，请单击“ 浏览” 并在“打开JAXB生成的选择XML架构文件”对话框中选择所需的文件。

输出路径

从此下拉列表中，选择模块源目录以放置生成的Java代码存根。

包前缀

使用此下拉列表指定用于放置生成的Java文件的包。

生成包级别注释

执行以下操作之一：

  • 选中此复选框以生成带有注释的package-info.java 。

  • 清除此复选框可将注释内化到其他生成的类中。

使用“生成”注释标记生成的代码

选中此复选框以使javax.annotation.提供生成的代码javax.annotation.Generated注释。

将生成的文件设为只读

选中此复选框以强制XJC绑定编译器将生成的Java源代码标记为只读 。

添加必要的库以便生成代码编译和工作

选中此复选框可将其他JAXB客户端库自动添加到将放置生成的源代码的模块的类路径中。

这些是生成的存根代码所依赖的库。

不要生成标题

选中此复选框可将no-header参数传递给相应的命令。

添加外部绑定文件/目录

选中此复选框以指定生成的文件所在的外部绑定文件或输出目录。

状态

查看此只读字段中的信息以跟踪和改进配置生成过程时的差异。
