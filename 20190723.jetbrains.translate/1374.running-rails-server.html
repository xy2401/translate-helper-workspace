运行Rails服务器

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

对于每个Rails应用程序项目，IntelliJ IDEA为生产和开发环境提供默认的Rails运行/调试配置。

IntelliJ IDEA支持许多Rails服务器（Mongrel，WEBrick，Phusion Passenger等，具体取决于具体平台）。

Rails服务器在“ 运行”工具窗口的专用选项卡中运行 ，以相应的运行/调试配置命名。这样，服务器输出显示在Console选项卡中，其他信息显示在Server <environment type> log选项卡中：

运行Rails服务器

  1. 按Shift + Alt + F10 ，然后选择所需的Rails运行/调试配置类型。

  2. 如有必要，请更改运行/调试配置设置。为此，请在“ 运行”弹出菜单中选择“ 编辑配置... ” ，然后选择所需的Rails配置类型。

    出现Rails运行/调试配置对话框。

    在此对话框中，修改所需的设置。例如，IntelliJ IDEA建议将WEBrick作为默认的Rails服务器。您可以从支持的服务器列表中选择其他服务器。

  3. 点击主工具栏上的按钮。

您可以选择立即在浏览器中预览Rails应用程序的结果。为此，请更改选定的Rails运行/调试配置以启用内置浏览器。为此，请选中“ 运行浏览器”复选框，然后指定所需的IP地址。
