介绍

使用IntelliJ IDEA，您可以为以下最常见类型的Web服务开发客户端：

  • GlassFish中/ JAXWS2。X RI / Netro 1。X / JWSDP2.0，用于开发JAX-WS Web服务客户端。

  • Apache Axis用于开发Apache Axis Web服务客户端。

  • JAX-RS用于开发RESTful Web服务客户端。

您还可以启用并手动配置以下其他WS引擎的支持 ：

  • Apache Axis2
  • https://jax-rpc.dev.java.net/

  • XFire 1。X / CXF

  • 将JBossWS
  • WebSphere 6。X

  3. 修改生成的代码以使客户端正确初始化。
