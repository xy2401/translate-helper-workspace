查看宝石依赖图

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

IntelliJ IDEA可以基于Gemfile信息可视化gem依赖项。gem依赖关系图不可编辑。

如果缺少Gemfile，则Show Gem Dependency Diagram / Show Gem Dependency Diagram Popup命令不可用。

使用Gemfile打开项目的Gem Dependency图

  1. 执行以下操作之一：

    • 在“项目”工具窗口或编辑器的上下文菜单上，指向“ 图” ，然后选择“ 显示图表”或“ 显示图表弹出窗口”。

    • 按Ctrl + Shift + Alt + U或Ctrl + Alt + U.

  2. 从列表中选择Gem Dependency Diagram ：

在Gem Dependency图中，您可以执行以下操作

  • 选择元素 。

  • 添加或删除备注。

  • 更改图表布局 。

  • 更改图表比例 。

  • 导航到源代码 。

  • 使用Structure视图 （ Ctrl + F12 ） 浏览gem 。

按住Alt键会调用放大镜工具，这将帮助您更仔细地查看模型依赖关系图中最有趣或最有问题的区域。
