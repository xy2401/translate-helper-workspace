配置键盘快捷键

WebStorm包含预定义的键盘映射，可让您自定义常用的快捷键。

要查看键盘映射配置，请打开“ 设置 / 首选项”对话框（ Ctrl + Alt + S ）并选择“ 键盘映射” 。

WebStorm会根据您的环境自动选择预定义的键映射。确保它与您正在使用的操作系统匹配，或者选择与您习惯使用的另一个IDE（例如，Eclipse或NetBeans）匹配的快捷方式。

您无法自定义预定义的键盘图。修改任何快捷方式时，WebStorm会创建当前所选键映射的副本，您可以对其进行配置。点击复制选定的键映射，重命名，删除或将其还原为默认值。

自定义键映射不是其父键映射的完整副本。它从父键映射继承未修改的快捷方式，并仅定义已更改的快捷方式。有关键映射文件的信息，请参阅用户定义的键映射的位置 。

键映射本身是具有相应键盘和鼠标快捷键以及缩写的操作列表。要按名称查找操作，请在搜索字段中键入该操作。如果您知道操作的快捷方式，请单击并在“ 查找快捷方式”对话框中按下组合键 。

添加键盘快捷键

  1. 在“ 键盘映射设置”对话框中，右键单击某个操作，然后选择“ 添加键盘快捷键” 。

  2. 在“ 键盘快捷键”对话框中，按必要的组合键。

  3. （可选）选择“ 第二个笔划”复选框以使用两个连续的键组合定义复杂的快捷方式。

  4. 单击“ 确定”以保存快捷方式。

    使用鼠标指针非常重要，因为在“ 键盘快捷方式”对话框中按Enter 键会被视为要用作快捷方式的组合键。

您按下的组合键将显示在“ 键盘快捷方式”对话框中，如果与现有快捷方式冲突，则会显示警告。

添加鼠标快捷方式

  1. 在“ 键盘映射设置”对话框中，右键单击某个操作，然后选择“ 添加鼠标快捷键” 。

  2. 在“ 鼠标快捷方式”对话框中，将鼠标指针移动到中心区域，然后根据需要单击或滚动。

  3. 单击“ 确定”以保存快捷方式。

执行的鼠标操作显示在“ 鼠标快捷方式”对话框中，如果与现有快捷方式冲突，则会显示警告。

添加缩写

缩写可用于快速查找没有快捷方式的操作。例如，您可以按Ctrl + Shift + A并键入跳转到颜色和字体操作的名称，以快速修改当前插入符号位置下元素的颜色和字体设置。如果为此操作指定缩写（如JCF ），则可以键入它而不是完整的操作名称。

  1. 在“ 键盘映射设置”对话框中，右键单击某个操作，然后选择“ 添加缩写” 。

  2. 在“ 缩写”对话框中，键入所需的缩写，然后单击“ 确定” 。

用户定义的键映射的位置

所有用户定义的键映射都存储在WebStorm配置目录中keymaps子目录下的单独配置文件中：

句法
％HOMEPATH％\。<产品> <版本> \配置\键盘映射
例
C：\用户\约翰\。WebStorm2019.1 \ CONFIG \键盘布局
句法
〜/库/首/ <产品> <版本> /键盘映射
例
〜/库/首选项/ WebStorm2019.1 /键盘布局
句法
〜/ <产品> <版本> /配置/键盘映射
例
〜/。WebStorm2019.1 /配置/键盘布局

每个键映射文件仅包含相对于父键映射的差异。

您可以使用这些文件与团队成员共享自定义键盘映射。要导出密钥映射，请复制相应的文件，然后将其放在另一个WebStorm安装的keymaps目录中进行导入。

与全局操作系统快捷方式冲突

预定义的键盘映射不包括所有可能的平台，版本和配置。某些快捷方式可能与全局系统操作和第三方软件的快捷方式冲突。要修复这些冲突，您可以重新分配或禁用冲突的快捷方式。
