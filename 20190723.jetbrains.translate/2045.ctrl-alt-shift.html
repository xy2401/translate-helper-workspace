按Ctrl + Alt + Shift键

本节列出并描述了包含Ctrl + Shift + Alt键的键盘快捷键。

捷径

功能

使用此快捷方式...

CTRL + SHIFT + Alt + C键

复制相对路径

将符号的引用（相对路径）复制到剪贴板。

Ctrl + Shift + Alt + N键

转到符号

导航到具有指定名称的符号 。

按Ctrl + Shift + Alt键+ H

弹出赫克托耳

打开突出显示级别弹出窗口。

按Ctrl + Shift + Alt键+ U

显示Uml图

打开类或包的UML类图 。

CTRL + SHIFT + Alt + V键

粘贴简单

将剪贴板中的最后一个条目粘贴为纯文本。

Ctrl + Shift + Alt + L键

显示重新格式文件对话框

显示重新格式化对话框 。

按Ctrl + Shift + Alt键+ I

按名称运行检查

按名称执行检查。

按Ctrl + Shift + Alt键+插入

新的Scratch文件

使用所选语言创建新的临时文件 。
