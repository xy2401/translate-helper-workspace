远程创建项目

介绍

如果您的本地开发计算机上没有Python解释器，则可以远程创建项目。使用PyCharm创建一个新项目，将其设置为在远程计算机上运行和调试。

配置远程解释器

配置将用于任何新创建的项目的默认解释器非常重要。

要配置远程解释器，请执行以下常规步骤：

  1. 在主菜单上，选择“ 文件”| “新建项目”的设置 ，或在“欢迎”屏幕上，单击下拉配置并选择“设置” 。

  2. 在项目设置的Project Interpreter页面中，单击 。

  3. 在下拉列表中，选择“ 添加” 。

  4. 根据打开的对话框中的选择，请执行以下过程之一：
    • 使用Docker配置远程解释器
    • 使用Docker Compose配置远程解释器
    • 使用SSH配置远程解释器
    • 使用Vagrant配置远程解释器
    • 使用WSL配置远程解释器

使用远程解释器创建项目

请注意，PyCharm允许创建纯Python和Django项目。

要创建远程项目，请按照以下常规步骤操作：

  1. 执行以下操作之一：

    • 在“ 欢迎”屏幕上 ，单击“ 创建新项目 ”链接。

    • 在主菜单上，选择“ 文件”|新项目 。

  2. 在打开的对话框中，选择Python或Django项目类型。

  3. 在Interpreter字段中，从下拉列表中选择所需的远程解释器。

    请注意，远程解释器无法同步映射文件夹之外的项目。如果项目位于映射文件夹之外，则会显示一条红色消息，直到您在映射文件夹中指定项目位置。

  4. 在“ 更多设置”下指定值。

  5. 单击“ 创建”
