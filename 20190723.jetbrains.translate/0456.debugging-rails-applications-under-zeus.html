在Zeus下调试Rails应用程序

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

要调试Rails应用程序，在使用Zeus时，请遵循以下常规步骤

  1. 确保在要调试的Rails应用程序脚本中设置断点。

  2. 执行以下操作之一：

    • 在主菜单上，选择“ 工具”|运行Zeus Server

    • 在主工具栏上，单击运行/调试配置选择器，然后选择“ 编辑配置” 。然后，在“ 运行/调试配置对话框”对话框中，为Zeus服务器创建运行/调试配置。有关该过程的详细说明，请参阅创建和编辑运行/调试配置一节。

      请注意，必须在“ 命令”字段中指定“ Start ”才能启动服务器。

  3. 在调试模式下启动Zeus。为此，选择Zeus run / debug配置，单击 ，或按Shift + F9 。

  4. 在Zeus服务器以调试模式运行时，运行所需的Rails应用程序脚本。
