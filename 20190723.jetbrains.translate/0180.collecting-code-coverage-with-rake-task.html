使用Rake任务收集代码覆盖率

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

如果通过Rake任务启动测试，则用户有责任提供代码覆盖率测量。

为test :: Unit运行/调试配置启用rcov

假设您有一个特殊的Rake任务，通过rcov收集代码覆盖率信息。

  1. 打开以编辑*.rake文件，并添加以下代码：

    要求 'rcov / rcovtask'ovTask 。 新 做的 | t | s << “test” t_files = FileList [ 'test / ** / * _ test.rb' ] bose = true

  2. 为任务'rcov'创建运行/调试配置并使用coverage运行它。
