方面页面

档案|项目结构|模块| <module> | <框架>
档案|项目结构|方面| <Framework>（<module>）

在元素选择器窗格中选择框架（构面）时，框架的设置将显示在对话框的右侧。

设置因框架（构面类型）而异：

  • Android Facet页面
  • AspectJ Facet
  • Android-Gradle Facet
  • EJB facet页面
  • Google App Engine Facet
  • GWT Facet
  • Hibernate和JPA Facet Pages
  • Java EE Application构面页面
  • JSF Facet
  • OSGi Facet
  • Seam Facet
  • Struts Facet
  • Struts 2 Facet
  • Tapestry Facet
  • Web facet页面
  • Web服务方面
  • Web服务客户端方面
