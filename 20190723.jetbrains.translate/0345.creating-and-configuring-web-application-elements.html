创建和配置Web应用程序元素

此功能仅在Ultimate版本中受支持。

Web应用程序的元素和部署设置在Web模块部署描述符web.xml中定义，该描述符是在为模块启用Web应用程序支持时自动创建的。

为了与用户界面保持一致，模块部署描述符也称为程序集描述符 。

您可以使用编辑器并使用IntelliJ IDEA编码帮助编辑源代码。

根据常见的基本过程 ，Web模块使用类似的元素填充。有关元素的详细信息，请参阅本部分中的相关主题：
