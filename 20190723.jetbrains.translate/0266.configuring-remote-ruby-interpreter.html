配置远程Ruby解释器

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

条件

在开始使用远程解释器之前，请确保已启用SSH Remote Run插件。该插件与IntelliJ IDEA捆绑在一起，默认情况下处于激活状态。如果未激活插件 ，请在“ 管理插件 ”中所述的“设置/首选项”对话框的“ 插件”页面上启用它。

配置远程Ruby解释器

要配置远程Ruby解释器，请按照下列步骤操作：

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ）。

  2. 在“ 项目结构”对话框中，单击“ 模块”节点。

  3. 在Gems选项卡中，单击 ，从弹出菜单中选择New remote ... ：

  4. 在“ 配置远程Ruby解释器”对话框中，选择所需选项（部署配置，SSH凭据等）并指定所需的设置。
