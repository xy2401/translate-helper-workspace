PyCharm 2019.1帮助

按Ctrl

本节列出并描述了包含Ctrl键的键盘快捷键：

  • Ctrl +字母数字键
  • Ctrl +导航键
  • Ctrl +符号键
  • Ctrl + Numpad键
  • Ctrl +功能（F）键

Ctrl +字母数字键

捷径

功能

使用此快捷方式...

按Ctrl + A

全选

在活动编辑器中选择整个文本。

按Ctrl + B

导航到声明

从任何用法直接导航到元素的声明。

按Ctrl + C

复制

将所选文本复制到剪贴板。

按Ctrl + d

重复的行或块

在插入符号处复制选定的块或行。

按Ctrl + E

最近的文件
最近发现用法

显示最近打开的文件列表。

当“ 查找”工具窗口具有焦点时，请使用此快捷方式显示最近查找使用结果的列表。

按Ctrl + F

找

在编辑器中启动文本搜索 。

CTRL + G

导航到Line

导航到当前文件中具有指定编号的行。

按Ctrl + H

键入层次结构

浏览所选类的层次结构。

按Ctrl + J

插入实时模板

显示以指定前缀开头的实时模板列表。

按Ctrl + M

滚动到中心

在插入符号处滚动一行到屏幕中心。

CTRL + N

导航到Class

使用指定的名称跳转到项目中的类。

按Ctrl + O

覆盖方法

覆盖当前类中的基类方法。

按Ctrl + P

参数信息

在插入符处显示方法调用的参数。

按Ctrl + Q
Alt +鼠标Button2

快速文档

在插入符号处显示带有符号文档的弹出窗口。

在“ 数据库”工具窗口中 ：显示一个弹出窗口 ，显示插入符号处的数据库表和表的前10行的create table查询。

CTRL + R

更换

调用“ 替换文本”对话框。

按Ctrl + S

保存全部

保存所有文件和设置。

按Ctrl + U

导航到超级方法

导航到插入符号方法的超级方法声明

按Ctrl + V

糊

从剪贴板粘贴。

按Ctrl + W

在Caret选择Word

从插入符号的单词开始，连续选择扩展的文本块。（反复使用此快捷方式选择表达式。）

按Ctrl + X

切

剪切到剪贴板。

按Ctrl + Y

在Caret删除Line

删除从当前插入符号位置开始直到单词结尾的单词。

按Ctrl + Z

解开

撤消上次操作。

按Ctrl + Shift + Z

重做

重做最后一次撤消操作。

按Ctrl +号码

导航到书签

导航到带有相应编号的编号书签。

Ctrl + Enter

拆分线或未清项目

根据具体情况：

  • 在编辑器中：根据需要智能地将当前行拆分为2行，移动引号等。

  • 在工具窗口中：分别打开所选项目的编辑器选项卡或选项卡。

Ctrl + Home

移至文本开始

跳到文本的开头。

按Ctrl + C

复制

将当前行或选定的代码块复制到剪贴板。

Ctrl +空格键

基本代码完成

Ctrl + Page Down

导航到页面底部

将插入符号向下移动到页面底部。

Ctrl + Page Up

导航到页面顶部

将插入符号移动到页面顶部。

按Ctrl +向下

向下滚动

将插入符号向下移动一行，保持语法正确性。

Ctrl +向左

移至上一个单词

将插入符号移到上一个单词。

按Ctrl +右

转到下一个单词

Ctrl +向上
按Ctrl + Shift +向上

Ctrl + Ctrl + Home / End

功能
