创建和删除Web应用程序元素 - 一般步骤

此功能仅在Ultimate版本中受支持。

本主题描述Web模块中元素的一般过程。

向Web模块添加元素

  1. 选择View |打开Web工具窗口工具窗口|网络 。工具窗口显示具有Web构面的所有模块。

  2. 右键单击所需的Web模块节点，然后选择“ 新建”|上下文菜单中的Servlet / Filter / Listener 。

  3. 在打开的“ 新建Servlet /过滤器/侦听器”对话框中，键入元素的名称。

  4. 指定要在其中创建元素的包。

  5. 指定实现该元素的类。

  6. 单击确定 。在编辑器中创建并打开一个新类。

  7. 按照相关元素特定主题中的描述继续操作。

从Web应用程序中删除元素

  1. 在Web工具窗口中，选择要删除的元素。

  2. 在上下文菜单中，选择“ 删除” ，然后在确认对话框中单击“ 是 ”。
