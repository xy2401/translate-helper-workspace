编辑项目路径映射对话框

单击时将打开该对话框在各种情况下，在Path Mappings字段旁边（例如，在Run / Debug Configuration：Node.js对话框中，或者在配置远程解释器时）。

如果在当前的运行/调试配置中使用可通过SFTP连接访问或位于Vagrant实例上的解释器，则会从相应的部署配置或Vagrantfile中自动检索映射，并在对话框中列出。映射是只读的。

  • 要添加自定义映射，请单击并分别在“ 本地路径”和“ 远程路径”字段中指定项目中的路径和远程运行时环境上的相应路径。手动键入路径或单击并在打开的对话框中选择相关文件或文件夹。

  • 要删除自定义映射，请在列表中选择它，然后单击 。
