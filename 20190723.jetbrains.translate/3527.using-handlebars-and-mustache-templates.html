WebStorm 2019.1帮助

把手和小胡子

使用WebStorm，您可以在hbs或mustache文件中使用Handlebars表达式和Mustache模板 。WebStorm根据“设置/首选项”对话框的“ 模板”页面上指定的默认或自定义设置处理此类文件的内容。

在WebStorm中配置Handlebars和Mustache

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript |模板 。

  2. 在打开的“ 模板”页面上，切换到“ Handlebars / Moustache”区域。

  3. 要在键入第一个结束标记时自动插入 Handlebars表达式的第二个右大括号（ } ），请选中自动插入结束标记复选框。

    WebStorm还识别三个 stashes（ {{{ ），防止表达式中的值被转义。在这种情况下，只要您键入第一个结束大括号，WebStorm就会自动插入两个右大括号。

  4. 要在代码生成，重构或按Ctrl + Alt + L时自动重新格式化Handlebars表达式和Mustache模板 ，请选中“ 启用格式”复选框。否则，WebStorm始终保持其原始格式。

  5. 选择要使用样式进行评论的语言。通过按Ctrl + /或Ctrl + Shift + /输入行或块注释时，WebStorm会插入所选语言中使用的注释分隔符，例如：

    • Handlebars {{!----}}

    • /**/用于JavaScript

    • HTML的<!---->

  6. 要配置Handlebars / Moustache感知语法突出显示，请在Settings / Preferences对话框（ Ctrl + Alt + S ）中转到Editor |配色方案|把手/小胡子 。选择颜色方案，接受从默认值继承的突出显示设置，或按照配置颜色和字体中的说明自定义它们。
