窃取现有锁

选中此复选框以覆盖先前由其他人在所需文件上设置的锁定。

以后不要显示此对话框

选中此复选框以禁止显示此对话框并使文件和文件夹静默锁定。

要在再次锁定文件或文件夹之前让WebStorm显示此对话框：

  1. 打开“ 设置”对话框的“ 版本控制 - 确认”页面。

  2. 在“ 显示选项”对话框中调用这些命令区域时，选中“ 结帐”复选框。
