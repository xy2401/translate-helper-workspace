新的CMP实体Bean对话框

EJB工具窗口|具有EJB构面的模块的上下文菜单|新 - CMP实体Bean
EJB工具窗口|具有EJB构面的模块的上下文菜单|跳转到源|新的 - BMP实体Bean

使用此对话框可以创建容器管理持久性（CMP）实体Bean 。

项目

描述

<EJB名称>

在此文本框中，键入用于生成EJB组成类的名称的基本名称。生成EJB名称的规则在“ Java EE名称”对话框中配置。

包

在此文本框中，指定bean所在的包的完全限定路径。如有必要，请单击“ 浏览”按钮并在项目树中选择所需的包。

EJB类

在此文本框中，指定bean实现类的名称。如有必要，请单击“ 浏览”按钮打开“ 选择EJB类”对话框，您可以在其中按名称搜索所需的类，或在项目树中选择它。

主键类

在此文本框中，指定用于访问与实体bean关联的数据源的主键的类。如有必要，单击“ 浏览” 打开“ 选择EJB主键类”对话框，您可以在其中按名称搜索所需的类，或在项目树中选择它。

CMP版本

从此下拉列表中，选择要使用的Container Managed Persistence（CMP）版本。

远程接口

如果要配置Bean的远程客户端视图，请选中此复选框。

选中此复选框将启用“ 主页”和“ 远程”文本框。

家

在此文本框中，指定远程home接口的实现类。如有必要，单击“ 浏览” 打开“ 选择EJB主界面”对话框，您可以在其中按名称搜索所需的界面，或在项目树中选择它。

远程

在此文本框中，指定远程接口的实现类。如有必要，单击“ 浏览” 打开“ 选择EJB远程接口”对话框，您可以在其中按名称搜索所需的接口，或在项目树中选择它。

本地接口

如果要配置Bean的本地客户端视图，请选中此复选框。

选中此复选框将启用“ 本地主页”和“ 本地”文本框。

当地的家

在此文本框中，指定本地home接口的实现类。如有必要，单击“ 浏览” 打开“ 选择EJB本地主页接口”对话框，您可以在其中按名称搜索所需的接口，或在项目树中选择它。

本地

在此文本框中，指定本地接口的实现类。如有必要，单击“ 浏览” 打开“ 选择EJB本地接口”对话框，您可以在其中按名称搜索所需的接口，或在项目树中选择它。
