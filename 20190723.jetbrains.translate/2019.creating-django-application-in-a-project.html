要将新站点添加到现有Django项目，请使用manage.py实用程序的startapp任务。

将新的Django应用程序添加到现有项目中

  1. 在主菜单上，选择“ 工具”|运行manage.py任务

  2. 在“ 输入manage.py任务名称”对话框中，键入startapp 。注意在输入第一个字母后出现在对话框下方的建议列表，并在键入时缩小以仅显示完全匹配。

  3. 在打开的对话框中，键入新Django应用程序的名称。
