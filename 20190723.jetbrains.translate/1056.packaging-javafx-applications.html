打包JavaFX应用程序

您可以通过构建相应的工件来打包JavaFX应用程序。对于JavaFX应用程序，IntelliJ IDEA提供专用工件类型（JavaFx应用程序）。如果您按照创建JavaFX开发项目中所述创建项目，则IntelliJ IDEA会自动创建一个JavaFx Application工件配置。

或者，您可以生成Ant构建文件，然后使用该构建文件打包您的应用程序。

  • 建造一件神器
  • 生成和使用Ant构建文件

另请参见使用预加载器的应用程序：项目组织和打包 。

建造一件神器

要为JavaFX应用程序构建工件：

  • 选择Build |构建工件 。然后，在Build Artifacts中操作弹出窗口，选择工件并选择构建 。

  • 或者，您可以在工件配置中打开Build on make选项。在这种情况下，在制作项目时将构建工件（例如Build | Make Project或Ctrl + F9 ）。

默认情况下，工件在<project_folder>\out\artifacts\<artifact_name> 。

另请参阅使用工件 ， 工件和Java FX选项卡 。

生成和使用Ant构建文件

  1. 为项目生成Ant构建文件（ Build | Generate Ant Build ）。有关更多信息，请参阅生成Ant构建文件 。

  2. 对生成的构建文件进行必要的更改。

  3. 打开Ant Build工具窗口（例如View | Tool Windows | Ant Build ），指定要使用的构建文件并运行必要的目标。有关更多信息，请参阅执行Ant目标 。

另见Ant 。
