查看符号的用法

使用Show Usages功能，您可以在整个项目中显示符号用法列表。

查看整个项目中符号的用法

  1. 将插入符号放在编辑器中的所需符号上。

  2. 在主菜单上，选择“ 编辑”|找到|显示用法 ，或按Ctrl + Alt + F7 。

  3. 检查并分析检测到的符号出现次数：

    • 使用工具栏按钮以所需方式显示搜索结果。

    • 要从搜索结果跳转到源代码行，请单击所需的条目。

    • 要关闭列表，请按Escape 。

  4. 如有必要，请在搜索选项对话框中自定义搜索选项 。要调用该对话框，请执行以下操作之一：

    • 在“ 显示用法”弹出窗口中，单击“ 设置”按钮（ ）。

    • 按Ctrl + Shift + Alt + F7 。

设置.../ Ctrl + Alt + S.

打开所选符号的“ 查找用法”对话框，您可以在其中更改搜索选项。

打开“查找用法”工具窗口/ Alt + F7

单击此按钮可以在“查找用户”工具窗口中显示“显示用户”弹出窗口并显示用法。

除了查看用法，您还可以使用此功能进行快速导航。
