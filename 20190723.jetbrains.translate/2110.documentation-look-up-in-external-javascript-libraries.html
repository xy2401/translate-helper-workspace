在外部库中查找

从PyCharm，您还可以查看外部JavaScript库和框架中定义的符号的文档。在按住Shift + F1并将光标定位在相关符号处时，PyCharm将调用相应的链接并在默认的PyCharm浏览器中打开文档页面。

启用外部JavaScript库查找功能

  1. 下载所需的库或框架。

  2. 将下载配置为 PyCharm级别的库 。

  3. 指定外部文档的链接。

查看外部JavaScript库中定义的符号的文档
将光标定位在相关符号上，然后按Shift + F1或选择View |主菜单上的外部文档 。
