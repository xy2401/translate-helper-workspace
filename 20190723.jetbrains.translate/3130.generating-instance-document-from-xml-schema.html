从XML模式生成实例文档

  1. 在活动编辑器选项卡中打开所需的Schema（ .xsd ）文件后，选择Tools | XML操作|在主菜单上从XSD Schema生成XML文档 。将打开“从架构生成实例文档”对话框。

  2. 在“ 架构路径”文本框中，指定基于生成XML文档的架构的位置。默认情况下，该字段显示当前文件的完整路径。接受此建议或单击“ 浏览” 并在打开的对话框中选择所需的文件。

  3. 在“ 实例文档名称”文本框中，指定用于放置生成的XML的输出文件的名称。

    WebStorm建议扩展名为.xml的源XML文档的名称。如果键入其他名称，请确保扩展名正确。

  4. 指定生成的文档的位置。默认情况下，它将与源Schema文件放在同一目录中。要指定其他位置，请单击“ 浏览” 并在打开的对话框中选择所需的路径。

  5. 从“ 元素名称”下拉列表中，选择要用作生成的XML文档的根的全局元素的本地名称。

  6. 通过选择相应的复选框，指定是否考虑限制和唯一性粒子。
