在目录对话框上重新格式化代码

代码|重新格式化代码
Ctrl + Alt + L

在目录上选择“ 重新格式化代码”操作时，将显示此对话框。

项目

描述

目录

这是一个只读字段，显示所选目录的名称。

选项

此区域显示代码重新格式化的选项。
您可以选择以下选项：

  • 包含子目录：选中此复选框可以从重新格式化的嵌套目录中的文件中获取源代码。仅当嵌套目录存在时，才会显示此复选框。

  • 优化导入：选中此复选框可从所选范围内的代码中删除未使用的导入语句。

  • 重新排列条目：选中此复选框可重新排序源代码条目。

  • 只有VCS更改了文本：如果选中此复选框，则重新格式化将仅适用于已在本地更改但尚未签入存储库的文件。
    此复选框仅适用于版本控制下的文件。

过滤器

此区域显示可应用于代码重新格式化操作的过滤器选项。
您可以选择以下选项：

  • 范围：选中此复选框，从下拉列表中选择要应用重新格式化选项的范围。如果要配置自定义范围，请单击按钮打开“ 范围”对话框。

  • 文件掩码：选中此复选框，从下拉列表中选择要应用重新格式化选项的文件扩展名。

跑

单击此按钮开始重新格式化指定范围内的源代码。
