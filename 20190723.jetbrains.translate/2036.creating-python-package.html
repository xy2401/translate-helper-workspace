创建Python包

PyCharm可以递归地创建Python包，从而创建整个包结构。

Python包节点标有图标。

创建一个新的Python包

  1. 在“ 项目”工具窗口中，选择目标目录。

  2. 在选择的上下文菜单上，选择New | Python包 ，或按Alt + Insert调用上下文菜单：

  3. 在打开的“ 新建包”对话框中，指定包名称。
    您还可以指定嵌套包;在这种情况下，名称应该用点分隔：


    单击确定 。 PyCharm创建新的包或包结构：
