提取方法对话框

重构|提取方法
按Ctrl + Alt + M

在JavaScript上下文中，对话框的标题是Extract Function 。

项目

描述

名称

在此文本框中，根据所选源代码指定要生成的函数或方法的名称。

声明功能表达

选中此复选框以通过函数表达式定义新函数，例如new_method = function() 。

该复选框仅在JavaScript上下文中调用重构时可用。

参数

在此区域中，选择要传递给新方法/函数的参数。

如果未选择任何对新方法功能至关重要的参数，WebStorm将无法继续进行重构。

上移/下移

使用这些按钮可以更改参数的顺序。

签名预览

在此只读字段中，查看新方法的声明。
