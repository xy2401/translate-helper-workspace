提取Groovy的参数对话框

重构|提取|参数
按Ctrl + Alt + P

使用此对话框指定与Groovy中的Extract Parameter重构相关的选项和设置。

项目

描述

类型

指定新参数的类型。通常，您无需更改IntelliJ IDEA建议的类型。

名称

指定新参数的名称。

宣布决赛

选择此选项可声明参数final 。

通过重载方法委派

选择此选项可保持现有方法调用不变。

因此，将创建一个新的重载方法，并将新参数添加到此方法的定义中。将保留具有旧签名的方法。对新方法的调用将添加到旧方法中。必要的值（或表达式）将传递给此调用中的新方法。

删除不再使用的参数<name>

选中此复选框以删除参数。

使用显式返回语句

如果方法返回值，则此复选框处于活动状态。如果它是方法中的最后一个return语句，则可以省略return关键字。如果选中此复选框，则返回关键字。

参数

在此区域中，选择要传递给新方法/函数的参数。

如果未选择任何对新方法功能至关重要的参数，IntelliJ IDEA将无法继续进行重构。

上移/下移

使用这些按钮可以更改参数的顺序。

签名预览

在此只读字段中，查看新方法/函数的声明。
