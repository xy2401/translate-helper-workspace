将代码公开为Web服务

此功能仅在Ultimate版本中受支持。

假设您有一段实现某种功能的代码，并且您希望通过特定协议在Web上提供此功能。在这种情况下，您需要将现有代码转换为Web服务并进行部署。该操作将更新Web服务描述符，并在需要时生成其他部署代码。每次更改任何Web服务方法签名或添加或删除新方法时都需要执行此操作。

揭露一个班级

  1. 将光标定位在编辑器中的类名称，然后按Alt + Enter或单击黄色灯泡图标 。

  2. 从建议列表中，选择Expose Class as Web Service 。

  3. 在打开的Expose Class As Web Service对话框中，指定以下内容：

    • Web服务的名称和URL地址。

    • 访问Web服务的公共操作时使用的协议和编码样式。

    • 类型信息，包括名称，操作，参数和包含Web服务接口的数据。

    IntelliJ IDEA自动将服务描述添加到server-config.wsdd文件。
