在开始之前，请确保您的计算机上有Node.js.

Bower是客户端库和组件的包管理器，包含HTML，CSS，JavaScript，字体，图像文件等。您可以在不使用WebStorm的情况下，在已下载的Bower页面上或从内置终端的命令行中安装，定位，升级和删除Bower软件包。

安装和配置Bower

您也可以在Node.js和NPM页面上安装bower软件包，如npm和Yarn中所述 。

从Bower官方网站了解更多信息。

安装Bower

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install -g bower

创建bower.json

  1. 在嵌入式终端 （ Alt + F12 ）中，键入：

    cd <你的项目文件夹> bower init

  2. 回答问题以指定以下基本设置：

    • 要使用的测试框架。

    • 要自动捕获的浏览器。

    • 定义测试文件位置的模式，用于测试或从中排除。

在WebStorm中配置Bower

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript |凉亭 。Bower页面打开。

  2. 指定要使用的Node.js解释器。这可以是Windows子系统Linux上的本地Node.js解释器或Node.js。

  3. 指定Bower包和bower.json配置文件的位置。

管理Bower包

可选：

  • 默认情况下，程序包作为项目依赖项安装。要将程序包安装为开发依赖项，请选中“ 选项”复选框，然后键入--save-dev 。

  • 默认情况下，Bower会安装最新版本。要忘记另一个，请选中“ 版本”复选框并指定所需的版本。

Bower仅将软件包安装为项目依赖项或开发依赖项 ，请从Bower官方网站了解更多信息。您可以在Bower页面或Terminal工具窗口中管理Bower软件包。

在终端中安装软件包

  • 在嵌入式终端 （ Alt + F12 ）中，键入以下命令之一：

    • bower install --save <package_name>将软件包安装为项目依赖项。

    • bower install --save-dev <package_name>将软件包安装为开发依赖项。

在Bower页面上处理包

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 语言和框架”| JavaScript |凉亭 。

  2. 在打开的Bower页面上， Packages区域显示项目中当前安装的所有Bower软件包。

    • 要安装软件包，请单击 ，在打开的“ 可用包”对话框中，选择所需的包，然后单击“ 安装包” 。

    • 要升级到最新版本的软件包，请在列表中选择它，然后单击 。

    • 要卸载软件包，请在列表中选择它，然后单击 。
