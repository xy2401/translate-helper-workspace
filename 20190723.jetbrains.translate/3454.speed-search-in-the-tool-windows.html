工具窗口中的速度搜索

在工具窗口的速度搜索可以帮助您查找和导航到Project工具窗口中的文件或文件夹，在结构工具窗口中的一员，在版本控制工具窗口一个变更，在TODO列表中的项目，等等。

请注意，仅在展开的节点上执行速度搜索，如果折叠节点，则不会检测到其下的匹配项。

要搜索工具窗口，请按照下列步骤操作：

  1. 选择所需的工具窗口。

  2. 开始键入项目名称（例如，文件，类，字段等）。键入时，“ 搜索”字段将显示在工具窗口工具栏上，显示输入的字符，元素选择将移至与指定字符串匹配的第一个项目。字符串的匹配部分突出显示。

  3. 如果多个相邻项目与模式匹配，请使用键盘上的向上和向下键在其中导航。

  4. 准备好后按Enter键 。结果，在工具窗口中选择匹配项。按Escape会隐藏“ 搜索”字段。
