配置加载路径

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

在这个部分：

  • 基本
  • 通过项目结构配置加载路径
  • 通过上下文菜单配置加载路径

基本

加载路径是require和load语句查找文件的路径。指定的路径将用于require和load 代码完成 。如果未定义加载路径，则代码完成将仅建议相对于项目根目录的路径。

属于加载路径的目录标记为源根目录在项目视图中。

通过项目结构配置加载路径

通过项目结构的“加载路径”页面定义加载路径

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ），然后单击“ 加载路径” 。

  2. 点击 。

  3. 在“ 选择路径”对话框中找到所需的路径。

通过上下文菜单配置加载路径

通过“项目”工具窗口的上下文菜单定义加载路径

  1. 在“ 项目工具”窗口中右键单击所需目录。

  2. 在上下文菜单上，指向标记目录为节点。

  3. 选择加载路径根 。

    相关目录标有源根图标 。

要取消标记目录，请在上下文菜单中选择“将目录取消标记为载入路径 ”。然后使用常规文件夹图标表示该目录 。
