创建本地和远程接口

此功能仅在Ultimate版本中受支持。

本地和远程接口用于从调用客户端提供对bean的访问，并为以下类型的bean定义：

  • 实体Bean，CMP和BMP版本1.x和2.x.

  • Session Bean 3.0之前的版本

IntelliJ IDEA提供了以下创建本地和远程接口的方法：

  • 手动，通过直接编辑 ejb-jar.xml文件，使用IntelliJ IDEA提供的大量编码帮助。

  • 使用上下文菜单命令。

使用编辑器定义企业bean的本地和远程接口

  1. 打开部署描述符文件ejb-jar.xml以进行编辑。

  2. 在<entity>部分中，创建标记<local>或<remote>然后键入所需接口的名称。请注意，您可以在此标记中指定目标包，例如samples.ejb.ManuallyCreatedEntityLocalInterface ，在每个点后使用代码完成。

  3. 如果声明的类尚不存在，IntelliJ IDEA建议快速修复。从建议列表中选择Create Class <class name> 。如果未在<ejb-class>标记中定义目标包，请在“ 选择目标目录”对话框中选择所需的包。存根类在指定位置创建。

配置本地和远程接口

  1. 打开“新建Bean”对话框 。

  2. 如果需要配置Bean的远程客户端视图，请选中“ 远程接口”复选框。

    • 在Home字段中，指定bean 远程home接口的名称。

    • 在“ 远程”字段中，指定Bean 远程接口的名称。

  3. 如果需要配置Bean的本地客户端视图，请选中“ 本地接口”复选框。

    • 在Home字段中，指定bean 本地home接口的名称。

    • 在“ 远程”字段中，指定Bean 本地接口的名称。
