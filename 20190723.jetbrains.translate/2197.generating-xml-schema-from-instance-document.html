从实例文档生成XML模式

要禁止Schema枚举，请指定0。

在Web内容文件上运行结构验证检查需要XSD（XML架构定义） 。PyCharm可以扫描任何XML文件以查找现有元素和属性，并为其生成Schema。

基于XML文档生成架构

  1. 在活动编辑器选项卡中打开所需的XML文档后，选择“ 工具”| XML操作|在主菜单上从XML文件生成XSD架构 。将打开“ 从实例文档生成架构”对话框。

  2. 在“ 实例文档路径”文本框中，指定要用作架构生成基础的文件的位置。默认情况下，该字段显示当前文件的完整路径。接受此建议或单击“ 浏览” 并在打开的对话框中选择所需的文件。

  3. 在“ 结果架构文件名”文本框中，指定用于放置生成的架构的输出文件的名称。

    PyCharm建议扩展名为.xsd的源XML文档的名称。如果键入其他名称，请确保扩展名正确。

  4. 指定生成的架构的位置。默认情况下，生成的Schema文件将放在与源XML实例文档相同的目录中。要指定其他位置，请单击“ 浏览” 并在打开的对话框中选择所需的路径。

  5. 从“ 设计类型”下拉列表中，选择声明元素和复杂类型的方法。

  6. 从“ 检测简单内容类型”下拉列表中，选择要用于叶子文本的类型。

  7. 在“ 检测枚举限制”文本框中，键入导致模式枚举外观的出现次数。
