编译后编织模式

如果选中此复选框，则使用javac编译源代码。然后， ajc用于编织已编译的类文件。结果，整个过程花费的时间更少。

重要信息：如果模块中有.aj方面，请不要选中此复选框。也就是说，只有在所有方面都定义为@Aspect -annotated Java类（在.java文件中）时，才应选中此复选框。

方面路径

使用可用控件形成模块的aspectpath 。

（ Alt + Insert ）。使用此图标或快捷方式添加库和其他模块。在打开的对话框中选择所需的库和模块。（要选择，建议使用模块的依赖关系。）

（ Alt + Delete ）。使用此图标或快捷方式从列表中删除所选项目。

（ Alt + Up ）。使用此图标或快捷方式将所选项目在列表中向上移动一行。

（ Alt + Down ）。使用此图标或快捷方式将所选项目在列表中向下移动一行。
