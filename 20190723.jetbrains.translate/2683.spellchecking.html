PyCharm 2019.1帮助

拼写检查

PyCharm可帮助您确保所有源代码（包括文本字符串，注释，文字和提交消息）拼写正确。为此，PyCharm提供专用的Typo检查，默认情况下启用。

在文本字符串和注释中，插入符号中单词的拼写可以更改为正确的拼写。在启用重命名重构的上下文中，检查建议重命名符号的所有匹配项。

检查单词的拼写

启用Typo检查后，PyCharm会检测并突出显示任何字典中未包含的字词。您可以提供正确的拼写或按原样接受该单词。如果单词被接受，它将被添加到您选择的自定义词典中，并在将来被拼写检查器跳过。如果错字检查被禁用，所有的错别字被忽略。

纠正一个拼写错误的单词

  1. 将插入符号放在Typo检查突出显示的单词上。

  2. 点击或按Alt + Enter显示可用的意图操作。

  3. 选择更改为意图操作，然后从建议列表中选择所需的拼写。

接受拼写错误的单词

  1. 将插入符号放在Typo检查突出显示的单词上。

  2. 点击或按Alt + Enter显示可用的意图操作。

  3. 选择“ 保存到字典”操作以将该单词添加到用户的字典中，并在以后将其跳过。如果未选择默认字典 ，PyCharm将允许您选择要将单词保存到的字典。

    要删除刚刚添加的单词，请按Ctrl + Z.

配置要使用的词典

根据捆绑的词典和用户定义的自定义词典检查拼写的正确性。由于拼写检查功能不允许您实际更改语言本身，因此添加自定义词典将允许您为PyCharm带来几乎任何语言支持。

字典可以是以下任何一种：

  • 内置的项目级和应用程序级词典，您可以通过手动将单词保存到它们来填充。

  • 带有DIC扩展名的纯文本文件，包含用新行分隔的单词。

  • Hunspell词典包含两个通常共享相同名称的纯文本文件，例如en_GB.dic和en_GB.aff 。DIC文件包含单词列表以及适用的修改规则。AFF文件列表前缀和后缀由特定修改规则调节。字典中的hunspell可以在网上轻易获得的，例如， 在这里 。

Hunspell支持由相应的Hunspell插件提供。该插件不与PyCharm捆绑在一起，但它可以从JetBrains插件存储库安装，如管理插件中所述 。

选择要使用的词典

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”下的“ 拼写 ” 。

  2. 在打开的拼写页面上，切换到词典选项卡。

    在“ 自定义词典”区域中，配置要使用的自定义词典。

    • 要向列表中添加新的自定义词典，请单击或按Alt + Insert并在打开的“ 选择路径对话框”对话框中选择所需文件。所选文件的完整路径将添加到“ 自定义词典”列表中。

    • 要在PyCharm中编辑自定义词典的内容，请选择它并单击或按Enter键 。相应的文件将在新的编辑器选项卡中打开。

    • 要从列表中删除自定义词典，请选择它并单击或按Alt + Delete 。

  3. 在“ Bundled Dictionaries”区域中，通过选中或清除它们旁边的复选框来配置要使用的捆绑字典。

填充内置词典

您可以使用拼写检查期间应跳过的单词手动填充内置项目级和应用程序级词典。

添加一个接受的单词

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”下的“ 拼写 ” 。

  2. 在打开的拼写页面上，切换到“ 接受的单词”选项卡。

  3. 创建一个单词列表 ：

    • 点击打开“ 添加新单词”对话框并在其中指定新条目。不支持CamelCase或snake_case 。如果您尝试添加一个已包含在其中一个拼写词典中的单词，PyCharm会显示一条错误消息： 单词<just typed word>已经在词典中 。

    • 要从列表中删除项目，请选择该项目并单击 。

配置拼写检查选项

设置拼写检查默认值

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”下的“ 拼写 ” 。

  2. 在打开的拼写页面上，切换到词典选项卡。在“ 高级设置”区域中，选择是要将内置项目级别还是应用程序级别字典用作保存单词的单个字典。

    如果没有选择默认字典，PyCharm将允许您选择保存单词的单词，以便在错误输入的单词上调用Change to ... intent操作。

  3. 如有必要，使用旋转框限制拼写检查器建议的单词数。

配置错误检查

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”下的“ 检查 ” 。

  2. 在打开的“ 检查”页面上，展开“ 拼写”节点，然后在中央窗格中单击“ 拼写 错误 ”。

  3. 在右侧窗格中，配置错误检查：

    • 在“ 选项”区域中，通过选择或清除“ 处理代码” ，“ 处理文字 ”和“ 处理注释”复选框来定义要检查的内容类型。

    • 在“ 严重性”区域中，选择检查严重性级别以及应用此级别的范围 。
