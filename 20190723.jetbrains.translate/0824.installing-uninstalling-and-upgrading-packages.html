安装，卸载和升级Python包

以下内容仅在安装并启用Python插件时有效！

IntelliJ IDEA提供了为特定Python SDK安装，卸载和升级Python包的方法。

要管理项目解释器的Python包，请导航到File |项目结构 ，在“ 平台设置”部分下选择SDK ，然后从可用SDK列表中选择目标Python SDK。切换到“ 包”选项卡以预览可用包的列表。

如果选择具有已配置的Conda环境的Python SDK，则“ 包”选项卡工具栏中将显示“ 使用Conda包管理器”切换。

使用此切换来管理Conda环境存储库中的包。默认情况下，此切换关闭，您只能管理所选Python解释器可用的包。

安装包

  1. 单击安装 （ ）。

  2. 在打开的对话框中，预览可用包的列表。

    要指定自定义存储库（包括devpi或PyPi） ，请单击“ 管理存储库” 。
    在打开的“ 管理存储库”对话框中，单击“确定” 添加本地存储库的URL，例如， http：// localhost：3141 / root / pypi / + simple / ，然后单击“ 确定” 。在“ 可用包”对话框中，单击重新加载包列表。

  3. 在“ 搜索”字段中键入要安装的包的名称。该列表缩小以仅显示匹配的包。

  4. 如果需要，请选中以下复选框：

    • 指定版本 ：如果选中此复选框，则可以从可用版本列表中选择所需的版本。默认情况下，采用最新版本。

    • 选项 ：如果选中此复选框，则可以在文本字段中键入选项。

    • 安装到用户的站点包目录<path> ：如果清除此复选框（默认情况下），则包将安装到当前解释器包目录中。如果选中该复选框，则软件包将安装到指定的目录中。此选项不适用于Conda环境。

  5. 选择目标包，然后单击“ 安装包” 。

IntelliJ IDEA提供了一个快速修复程序，可以自动安装您尝试导入的软件包：如果在关键字import后import您的计算机上当前不可用的软件包名称，则快速修复建议忽略未解决的软件包参考，或下载并安装缺少的包：

卸载包

  1. 在“ 包”选项卡中，选择要删除的包。

  2. 单击卸载 （ ）。选定的软件包将从磁盘中删除。

IntelliJ IDEA通过显示当前安装的软件包版本号（列版本 ）和最新可用版本（列最新版本 ），智能地跟踪软件包的状态并识别过期版本。当检测到更新版本的软件包时，IntelliJ IDEA会用箭头符号标记它并建议升级它。

默认情况下，列最新版本仅显示软件包的稳定版本。如果要将最新可用版本的范围扩展到任何预发行版本（例如测试版或候选发布版 ），请单击“ 显示早期版本” 。

升级包

  1. 在“ 包”选项卡中，选择要升级的包。

  2. 点击升级 （ ）。请注意，单击“ 应用 ”不会导致包升级。此按钮仅使用所有已执行的包更改。

    选定的包将升级到最新的可用版本。

您可以一次升级多个包。按住Cmd （macOS）或Ctrl （Unix或Windows），左键单击以选择包列表中的多个项目，然后单击“ 升级” 。
