从Bnd / Bndtools模型导入项目

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA允许您导入Bnd / Bndtools项目。

在导入项目之前，请确保在IntelliJ IDEA中启用了Osmorc插件。该插件与IntelliJ IDEA捆绑在一起并默认激活。如果未启用该插件 ，请启用该插件 。

要将外部Bnd / Bndtools模型导入IntelliJ IDEA，请按照下列步骤操作

  1. 如果IntelliJ IDEA中当前未打开任何项目，请单击“ 欢迎”屏幕上的“ 导入项目 ”。否则，请选择文件|新的|来自现有来源的项目 。

  2. 在打开的对话框中 ，选择包含要导入的项目的目录，或包含相应项目描述的文件。单击确定 。

  3. 在Import Project向导的第一页上，选择Bnd / Bndtools，然后单击Next 。（如果IntelliJ IDEA已猜到您要导入的内容，则不会显示此页面。）

  4. 在向导的下一页上，指定要导入的Bnd / Bndtools项目，然后单击“ 下一步” 。

  5. 在向导的下一页上，指定项目SDK ，然后单击“ 完成” 。
