从表编辑器调用时，“导入文件”对话框

将分隔符分隔值从表编辑器导出到数据库时， 将打开此对话框。

  • 表名，结构和数据映射
  • DDL预览
  • 编码，将错误写入文件并将不可转换的值插入为null

表名，结构和数据映射

项目

描述

表

表的名称。

评论

表评论。

列/键等

列的数据映射，以及列，约束和索引的定义。
  • 要开始编辑列的信息，请双击列列表中的相应行。

    请注意Mapped to字段。此字段允许您指定要导入的文件中的哪个数据列应该用作数据库中相应列的数据源。如果清除此字段，则不会将数据添加到数据库中的目标列。

  • 要删除列，请选择相应的行并单击 。

DDL预览

将要运行的一个或多个语句。您可以在此窗格中编辑语句。

编码，将错误写入文件并将不可转换的值插入为null

项目

描述

编码

源文件中数据的字符编码。

将错误写入文件

如果选中该复选框，则导入期间发生的错误将写入指定的文本文件。对于每个错误，指示它出现的位置。

将不可转换的值插入为null

如果选中该复选框，则会在表中插入NULL以获取无法转换的数据。
