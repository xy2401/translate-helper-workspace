WebStorm 2019.1帮助

重新格式化文件对话框

Ctrl + Shift + Alt + L键

在当前文件的编辑器中按Ctrl + Shift + Alt + L时 ，将出现对话框。如果选择Code |在主菜单上重新格式化代码或按Ctrl + Alt + L ，WebStorm会尝试自动重新格式化指定范围的源代码。

项目

描述

只有VCS更改了文本

如果选中此复选框，则重新格式化将仅适用于已在本地更改但尚未签入存储库的文件。
此复选框仅适用于版本控制下的文件。

选定的文字

选择此选项可重新格式化当前选定的源代码片段。

整个档案

选择此选项可以重新格式化当前文件中的所有源代码。

优化进口

选中此复选框可从所选范围内的代码中删除未使用的import语句。

重新排列代码

选中此复选框可根据“代码样式”设置的“ 排列”选项卡中指定的配置对源代码条目重新排序。

此复选框不适用于Python文件。

跑

单击此按钮开始重新格式化指定范围内的源代码。
