HTTP响应处理程序脚本使用JavaScript ECMAScript 5.1编写，编码帮助和文档由捆绑的HTTP Response Handler库HTTP Response Handler 。该库公开了两个用于编写响应处理程序脚本的对象：

  • 客户端存储会话元数据，可以在脚本内部进行修改。在关闭WebStorm之前， client的状态将一直保留。

  • 响应包含有关收到的响应的信息：其内容类型，状态，响应正文等。
