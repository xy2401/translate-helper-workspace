项目

描述

Docker Machine可执行文件

 docker-machine或docker-machine.exe的实际路径（通常位于Docker Toolbox安装文件夹中）。

在以下情况下，默认的docker-machine值很好：

  • 可执行文件的实际名称是docker-machine 。

  • 文件所在目录的Path包含在Path环境变量中。

Docker撰写可执行文件

docker-compose或实际路径docker-compose.exe或docker-compose.sh （通常位于bin的多克尔安装目录的文件夹）。

如果出现以下情况，默认的docker-compose值就可以了：

  • 可执行文件的实际名称是docker-compose 。

  • 文件所在目录的Path包含在Path环境变量中。
