档案|设置|语言和框架|适用于Windows和Linux的XSLT文件关联
IntelliJ IDEA |偏好|语言和框架| macOS的XSLT文件关联
按Ctrl + Alt + S

使用此页面将XSLT样式表文件与XML文件相关联。这对于在XSLT节点选择中启用元素和属性名称的错误突出显示和增强完成是必要的。

项目

描述

项目XSLT文件

该窗格显示项目树视图中的所有XSLT文件，按模块及其内容根进行分组。

相关文件

该窗格显示与所选XSLT文件关联的所有XML文件。

单击此按钮以选择应与所选XSLT文件关联的XML文件。请参阅打开的对话框的说明 。

只有在“ 项目XSLT文件”窗格中选择了XSLT文件时，该按钮才可用。

单击此按钮可取消在“ 关联文件”窗格中选择的XML文件与“ 项目XSLT文件”窗格中选择的XSLT文件之间的关联。
