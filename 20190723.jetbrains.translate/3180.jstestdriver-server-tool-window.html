JSTestDriver服务器工具窗口

查看|工具窗口| JSTestDriver服务器

- 启动本地服务器链接。

在此工具窗口中，启动和停止JSTestDriver服务器以对其运行单元测试并捕获浏览器以执行单元测试。

先决条件

满足以下先决条件时，工具窗口可用：

  1. JSTestDriver插件已启用。该插件默认激活。如果插件已禁用，请在“ 插件”页面上启用它，如管理插件中所述 。

  2. 当前项目包含至少一个测试运行器配置文件 。

选项

项目

工具提示和快捷方式

描述

运行本地服务器

单击此按钮可让WebStorm启动默认的JSTestDriver服务器。

停止本地服务器

单击此按钮可让WebStorm停止当前运行的JSTestDriver服务器。

使用URL捕获浏览器

此只读字段显示用于访问JSTestDriver的远程控制台的URL地址。复制URL地址并在您选择的浏览器中打开它。

图标表示可用的浏览器。与您刚刚打开的浏览器对应的图标处于活动状态。单击该图标以准备执行测试。
