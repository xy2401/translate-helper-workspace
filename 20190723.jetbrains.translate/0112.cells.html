细胞的基本操作

您可以修改表格单元格中的值，并在适当的情况下上载文件。

行动

捷径

描述

编辑

F2

编辑单元格值。或者，选择一个单元格并开始键入新值。要确认更改，请按Enter键 。

编辑最大化

Shift + Enter键

扩展单元格以进行编辑。要确认更改，请按Ctrl + Enter 。

添加新行

Ctrl + Enter

在常规编辑模式（ F2 ）中添加新行。

还原已选中

按Ctrl + Alt + Z

恢复先前的单元格值。

最大化

按Ctrl + Shift + Alt + M键

展开选定的单元格。

在最大化单元格中添加新行

输入

在最大化编辑模式（ Shift + Enter ）中添加新行。要确认更改，请按Ctrl + Enter 。

设置默认值

按Ctrl + Alt + d

将当前单元格值设置为默认值。有关如何设置默认值的更多信息，请参阅设置单元格的默认值 。

设置NULL值

Ctrl + Alt + N

将当前单元格值设置为NULL值。有关如何设置默认值的更多信息，请参阅设置单元格的默认值 。

加载文件

将文件上传到字段中。右键单击单元格，然后选择“ 加载文件” 。

编辑为

F2或Shift + Enter

在单元格中编辑值，作为SQL，HTML，XML或任何其他受支持语言的片段。

价值完成

Ctrl +空格键

打开建议列表。要调用值完成，请开始编辑单元格并按Ctrl + Space 。该列表包含当前列中与您的输入匹配的值。

将具有相同内容的单元格添加到选择中

Alt键+ J

将具有相同内容的最近单元格添加到选择中。要从选择中删除单元格，请按Shift + Alt + J.

将选区从单元格扩展到表格

按Ctrl + W

将选择从单元格扩展到表格。

  • 按一次：选择扩展为列。

  • 按两次：选择从列切换到一行。

  • 按三次：选择扩展到整个表格。

比较细胞

按Ctrl + Shift + d

比较一个表中两个或三个单元格的内容。最多选择三个单元格，然后按Ctrl + Shift + D.或者，右键单击所选内容，然后选择“ 比较单元格” 。

设置单元格的默认值

创建新行并且没有为某些单元格指定值时，这些单元格将填充各自的默认值（ Ctrl + Alt + D ）或NULL（ Ctrl + Alt + N ）值。

为单元格设置默认值时，可以使用上下文菜单中的相应操作（ Set DEFAULT和Set NULL ）。

  1. 在“ 数据库”工具窗口（“ 查看”|“工具Windows”|“数据库” ）中，展开数据源树。

  2. 右键单击该表，然后选择“修改表”（ Ctrl + F6 ）。

  3. 在“ 列”选项卡上，双击要编辑的列。

  4. 在“ 默认”字段中，指定列中单元格的默认值。要允许NULL值，请清除Not null复选框。

使用快速文档视图

快速文档视图提供有关所选单元格中的值的详细信息。例如，如果单元格包含长文本，您将看到整个文本。如果单元格包含图像，您将看到图像的预览。此外，您还可以查看当前记录中引用的记录以及引用当前记录的记录。

宾语

文档视图

长文

如果单元格包含长文本，您将看到整个文本。

图片

如果单元格包含图像，您将看到图像的预览。

记录

记录的快速文档视图显示当前记录中引用的记录以及引用当前记录的记录。

您可以切换到转置视图。在转置视图中，行和列互换。

要切换到转置视图，请单击“ 转置视图” 。有关表中转置视图的详细信息，请参阅转置表 。

§

同时编辑多个单元格值

  1. 选择要编辑的单元格范围。

  2. 开始输入新值。更改将应用于所有选定的单元格。

  3. 要确认该值，请按Enter键 。要取消编辑，请按Escape 。

将LOB保存在文件中

您可以将二进制大对象 （BLOB或LOB）从单元格保存到文件中。

  1. 右键单击包含LOB的单元格，然后选择“ 保存LOB” 。

  2. 在保存对话框中，指定文件的名称和位置，然后单击“ 确定” 。
