隐含提示

IntelliJ IDEA允许您启用，展开和折叠隐式转换和参数的编辑器提示，以帮助您阅读代码。

  1. 在主工具栏上，选择“ 查看”|显示隐含提示 。

  2. 在编辑器中，右键单击提示，然后从弹出菜单中选择相应的操作以展开现有提示，禁用模式或查看隐式参数 。

或者，在编辑器中，您可以按Ctrl + Shift + Alt ++以启用隐式提示。如果再次按相同的快捷方式，IntelliJ IDEA会扩展隐式提示以显示更详细的信息。按Ctrl + Shift + Alt + -以折叠提示。

隐含的转换

IntelliJ IDEA允许您调用隐式转换方法和参数。

隐式转换方法

  1. 选择一个表达式，然后按Shift + Ctrl + Q （对于macOS，为Ctrl + Q ）以调用适用的隐式转换列表。该列表显示顶部显示的常规范围和列表底部显示的扩展范围。

    IntelliJ IDEA突出显示用于所选表达式的隐式转换。如果IntelliJ IDEA无法找到隐式转换，或者如果找到多个匹配，则会打开Introduce Variable列表。

  2. 如果需要，请隐式转换方法。按Alt + Enter并选择Make explicit或Make explicit（导入方法） 。

    • 如果选择Make explicit，则IntelliJ IDEA将返回带有类名的方法调用。如果您需要确保编译器导入您最初想要的特定隐式转换方法，这可能会有所帮助：

    • 如果选择Make explicit（导入方法），则会静态导入该方法，并且IntelliJ IDEA仅返回其没有类名称的调用。此外，下次打开有用的隐式转换列表时，您将在常规范围内看到此方法：

Java-to-Scala代码转换

IntelliJ IDEA允许您将Java代码转换为Scala：

  1. 复制Java代码（表达式，方法，类）并将其粘贴到Scala文件中。

  2. IntelliJ IDEA显示转换Java对话框，建议转换。单击确定 。

如果您不想使用复制/粘贴操作，可以在编辑器中打开Java文件并选择Refactor |转换为Scala或按Ctrl + Shift + G.

在这种情况下，IntelliJ IDEA将使用转换后的代码创建Scala文件。

将Scala代码反编译为Java

您可以将Scala代码反编译为Java，以查看如何使用Java翻译和实现某段代码。

  1. 在“ 项目”工具窗口中，右键单击要反编译的Scala类。

  2. 从上下文菜单中，选择“将Scala反编译为Java” 。

IntelliJ IDEA将代码转换为Java并在编辑器中打开转换后的文件。

您还可以在编辑器中打开Scala类，并使用其上下文菜单进行转换。
