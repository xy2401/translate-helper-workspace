管理Struts元素

此功能仅在Ultimate版本中受支持。

Struts配置文件struts-config.xml定义了Struts应用程序的结构，包括许多Struts元素以及它们之间的交互 。启用Struts对模块的支持时 ，将自动创建Struts配置文件。

使用IntelliJ IDEA，您可以在Struts Assistant工具窗口中编辑struts-config.xml ，该窗口提供三个同步视图：

  • 结构树

  • 属性表

  • Struts Web流程图

要定义Struts应用程序的结构，请使用Struts Assistant工具窗口选项卡Struts 创建和编辑必要的Struts元素并指定它们之间的交互。此专用工具窗口提供三个同步视图：

  • 用于添加，编辑和删除Struts元素的结构树 。

  • 属性表用于指定元素及其属性的值。

  • Struts Web Flow Diagram用于浏览Struts应用程序结构。

IntelliJ IDEA提供了根据常规过程管理以下Struts元素的工具：

  • 数据源
  • 形式豆

  • 全球例外

  • 全球前锋

  • 动作映射
  • 调节器

    Web应用程序中只能有一个控制器元素。

  • 消息资源

  • 插件

在友好的界面中管理切片和验证器需要您首先启用它们的支持 。
