使用XmlBeans对话框从XML模式生成Java代码

工具| XmlBeans |使用XmlBeans从XML模式生成Java代码

使用此对话框可以通过XmlBeans数据绑定器基于XML架构配置Java代码存根的生成。

访问对话框

  1. 此功能通过WebServices捆绑插件提供，该插件默认启用。如果没有，请按照管理插件一节中的说明启用它。

  2. 当在活动编辑器选项卡中打开的文件包含XML架构时，菜单项和对话框可用。

项目

描述

架构路径

在此字段中，指定要用作生成基础的文件。默认情况下，该字段显示当前文件的完整路径。要使用其他架构，请单击“ 浏览” 并在“打开选择XML架构/ Wsdl文件以生成对话框”中选择所需的文件。

输出路径

使用此字段指定.jar的名称以放置生成和编译的Java代码。默认情况下，IntelliJ IDEA建议创建一个新的types.jar 。要覆盖现有的.jar ，请单击“ 浏览” 并在打开的对话框中选择所需的.jar 。

添加必要的库以便生成代码编译和工作

选中此复选框可以将其他XmlBeans库自动添加到将放置生成的源代码的模块的类路径中。

这些是生成的存根所依赖的库。

状态

查看此只读字段中的信息以跟踪和改进配置生成过程时的差异。
