IntelliJ IDEA 2019.1帮助

建立一个新项目

此功能仅在Ultimate版本中受支持。

本主题描述如何使用Osmorc设置新项目。如果要将现有Eclipse工作空间导入到IntelliJ IDEA中，请查看导入Eclipse工作空间。

在本节中，您将学习如何为模块激活Osmorc。没有关于使用Osmorc的项目和模块的创建是Osmorc特定的。因此，就像创建任何其他IntelliJ IDEA项目和模块一样创建它们。

Osmorc通过专用的Osmorc方面在模块中激活。

将Osmorc facet添加到模块中

  • 明确添加构面 。

    1. 在模块的根文件夹中，创建META-INF文件夹。

    2. 在META-INF文件夹中，创建清单文件MANIFEST.MF 。

    3. 将至少以下两个标头添加到MANIFEST.MF清单文件中：

      • 捆绑，ManifestVersion

      • 捆绑-SymbolicName

      IntelliJ IDEA现在将识别模块的Osmorc方面，并建议添加它。确认添加。

  • 自动检测小平面。

现在您可以开始处理OSGI应用程序了。将清单文件更改为导出和导入包，Osmorc将在IntelliJ IDEA模块之间创建相应的依赖关系。

您可能还需要查看本文档的“设置”部分，以了解如何定义和使用框架实例。
