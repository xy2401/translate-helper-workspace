启用文档注释

请注意，本节涉及Python，JavaScript以及具有特殊文档注释开头的其他语言。

启用文档注释

  1. 打开编辑器|一般| PyCharm设置的智能键页面（ Ctrl + Alt + S ）。

  2. 在“ 输入”部分，选择或清除“ 插入文档注释存根”复选框。

  3. 然后，滚动到文档注释存根选项中的插入类型占位符，并根据需要选中或清除该复选框。有关详细信息，请参阅选项说明 。
