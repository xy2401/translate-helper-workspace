测试Grails应用程序

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA支持创建和运行单元和集成Grails测试，并为每种测试类型提供运行配置。

要创建Grails测试，请执行以下操作之一：

  • 使用要为其创建测试的域类的脚手架 ：

  • 运行Grails目标 ：按Ctrl + Alt + G ，然后在文本字段中键入目标名称。使用代码完成来缩小匹配目标列表：

创建Grails测试运行/调试配置

  1. 在Grails视图中 ，右键单击其中一个测试目录，指向上下文菜单中的Create Run Configuration ，然后选择以下建议选项之一：

    • 测试...

    • Grails测试：......

  2. 在打开的“运行/调试配置”对话框中，指定运行/调试配置设置：测试类型，Grails脚本等。

    您可以指定用空格分隔的多个测试名称，所有这些测试都将在单个运行配置中运行。

运行Grails测试

  1. 在Grails视图中 ，选择所需的测试类。

  2. 在打开的菜单中，选择“运行” ，然后选择所需的配置类型：

    选定的运行/调试配置在“运行”工具窗口中执行。
