Phing构建工具窗口

在本页面：

  • 访问Phing构建工具窗口
  • 工具栏
  • 上下文菜单

访问Phing构建工具窗口

  • 在编辑器中打开Phing构建文件或在“ 项目”工具窗口中选择它，然后从选择的上下文菜单中选择“ 添加为Phing构建文件 ”。

  • 选择查看|工具窗口| Phing Build在主菜单上。在编辑器或“项目”工具窗口中通过Phing构建文件的上下文菜单打开工具窗口后，可以访问该工具窗口。

在此工具窗口中：

  • 指定phing.bat文件的位置。

  • 配置Phing构建文件列表，以便打包，部署或测试PHP应用程序。

  • 根据具体配置，在运行或调试之前指定要执行的目标。

  • 运行整个构建文件和特定的构建目标。

Phing工具窗口的功能可通过工具栏按钮 >和构建文件或目标的上下文菜单获得。

工具提示和快捷方式

描述

加

单击此按钮可将新脚本添加到列表中。在打开的Select Phing Build Script对话框中，选择所需的文件。

去掉

单击此按钮可从列表中删除选定的构建文件。

跑

单击此按钮可运行选定的构建文件或目标。

展开全部
Ctrl + NumPad Plus

单击此按钮可使IntelliJ IDEA显示所选构建文件中定义的目标树。

全部收缩
Ctrl + NumPad -

单击此按钮可使IntelliJ IDEA折叠所选构建文件中的所有目标。

属性

单击此按钮可打开“ Phing属性”对话框。在此对话框中，指定phing.bat文件的位置。

如有必要，请通过在“ 命令行选项”文本框中指定命令行参数来配置构建过程和输出格式。如果参数集太大而无法放在文本框中，请单击按钮或按Shift + Enter并在打开的“ 命令行选项”对话框中键入所需的参数。

该按钮仅在构建文件列表不为空时可用。

建立目标

标记隐藏

选择此选项可隐藏所选目标，从而禁止在构建文件树中显示它。当你有一些只被其他目标调用而且从不单独运行的目标时，这可能会有所帮助。

展开特定构建文件的节点或单击“ 全部展开”按钮时， 隐藏目标不会显示 。要删除目标的隐藏状态：

  1. 选择定义它的构建文件，然后单击“ 设置”按钮在工具栏上。

  2. 在打开的Phing Settings对话框中，切换到Hiding targets选项卡，其中选中隐藏目标名称旁边的隐藏复选框。清除要删除状态隐藏的目标旁边的隐藏复选框。

建立目标

属性

单击此按钮可打开“ Phing属性”对话框。在此对话框中：

  • 指定phing.bat文件的位置。

  • 如有必要，请通过在“ 命令行选项”文本框中指定命令行参数来配置构建过程和输出格式。如果参数集太大而无法放在文本框中，请单击按钮或按Shift + Enter并在打开的“ 命令行选项”对话框中键入所需的参数。

  • 在“ 隐藏目标”选项卡中，标记要隐藏的目标并删除隐藏状态。

    在构建文件中定义的目标中，您可能有一些仅由其他目标调用的目标，并且永远不会单独运行。您可以通过将它们标记为隐藏来禁止在构建文件树中显示此类目标。

    展开特定构建文件的节点或单击“ 全部展开”按钮时， 隐藏目标不会显示 。

该按钮仅在构建文件列表不为空时可用。
