查看源文件的结构

您可以使用“ 结构”工具窗口（ Alt + 7 ）或“ 结构”弹出窗口（ Ctrl + F12 ）检查当前在编辑器中打开的文件的结构 。

默认情况下，IntelliJ IDEA显示当前文件的所有类，方法和其他元素。要切换要显示的元素，请单击“ 结构”工具窗口工具栏上的相应按钮。例如：

  • 点击显示类字段。

  • 点击显示继承的成员。

  • 点击显示包含的文件。

在“项目”工具窗口中显示类成员

  • 右键单击“ 项目”工具窗口标题栏，然后从上下文菜单中选择“ 显示成员 ”。
