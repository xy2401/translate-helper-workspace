自定义插件存储库对话框

单击“ 管理插件存储库”下将打开此对话框在插件中 。

使用此对话框可管理自定义插件存储库列表。存储库由其URL标识。

图标

捷径

描述

Alt +插入

使用此图标或快捷方式将新存储库添加到列表中。在打开的对话框中指定存储库的URL。使用“ 立即检查”按钮确保指定的URL正确：IntelliJ IDEA将尝试连接到存储库。

输入

使用此图标或快捷方式编辑选定的插件存储库URL。

Alt + Delete键

使用此图标或快捷方式从列表中删除所选的URL。
