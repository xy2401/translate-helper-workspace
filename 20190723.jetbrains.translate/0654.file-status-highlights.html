文件状态要点

在IntelliJ IDEA中，每个文件都有自己的状态，标记有特定颜色。文件状态显示实际文件的内容如何与标记为“当前”的内容相对应。

在编辑器中，检查文件中的每一行与“当前”状态的对应关系，并在左侧排水沟区域中标记特定颜色。

您可以自定义默认颜色：

  • 对于文件：在“ 设置/首选项”对话框中（ Ctrl + Alt + S ），在“ 版本控制”|下文件状态颜色 。

  • 对于编辑器中的行：在“ 设置/首选项”对话框中（ Ctrl + Alt + S ），在“ 编辑器”|下配色方案| VCS 。
