运行/调试配置对话框

运行|编辑配置

使用此对话框可以创建，编辑，调整或删除运行/调试配置 ，以及配置将应用于所有新创建的运行/调试配置的默认模板。

项目配置按配置类型分组;默认配置模板分组在对话框左侧的“ 模板”节点下。

模板

在运行配置的树视图中的“ 模板”节点下，您可以选择运行配置模板并编辑其默认设置。这不会影响现有配置，但在创建相应类型的新配置时将用作默认配置。

选择“ 模板”节点本身时，WebStorm将打开“ 运行仪表板隐藏”区域中可用的“配置” 。在那里你可以

“运行仪表板”中可用的配置

在此区域中，创建将在“ 运行仪表板”工具窗口中可用的运行配置列表，您可以在其中执行和管理多个运行/调试配置。

确认在进程终止时重新运行

此复选框的行为取决于是否为特定的运行/调试配置选择了“ 仅实例”选项。

  • 如果选中此复选框，则在单个实例的情况下，启动新进程（例如，通过单击在主工具栏上）当另一个进程仍在运行时，导致显示一个对话框，提示在启动新进程之前终止当前进程。

  • 如果未选中此复选框（或在多个实例的情况下），WebStorm将以静默方式启动新进程。

临时配置限制

指定要存储的最大临时配置数，并在“ 选择运行/调试配置”下拉列表中显示。

工具栏

运行/调试配置的树视图有一个工具栏，可帮助您管理项目中可用的配置以及调整默认配置模板。

描述

Alt +插入

创建运行/调试配置。

Alt + Delete键

删除选定的运行/调试配置。请注意，您无法删除默认配置。

按Ctrl + d

创建所选运行/调试配置的副本。请注意，您可以创建默认配置的副本。

编辑模板

查看和编辑所选运行/调试配置的默认模板。模板用于新创建的配置。

Alt + Up / Alt + Down

在列表中上下移动选定的运行/调试配置。

列表中的配置顺序定义了选择运行/调试配置时配置显示的顺序。

运行/调试配置的默认模板始终按字母顺序排序。

进入新文件夹/创建新文件夹

您可以通过将运行/调试配置放入文件夹来对其进行分组。

要创建文件夹，请选择类别中的配置，单击 ，并指定文件夹名称。如果仅关注某个类别，则会创建一个空文件夹。

然后，要将配置移动到文件夹中，文件夹之间或文件夹之外，请使用拖放或和纽扣。

要删除分组，请选择一个文件夹并单击 。

排序配置

单击此按钮可按字母顺序对配置进行排序。
