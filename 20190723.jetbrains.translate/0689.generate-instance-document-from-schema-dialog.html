从架构对话框生成实例文档

工具| XML操作|从XSD架构生成XML文档

在此对话框中，指定用于基于描述其结构的XSD（XML架构定义） 架构生成XML文件的选项。

当在活动编辑器选项卡中打开的文件包含XML架构时，菜单项和对话框可用。

项目

描述

架构路径

在此字段中，指定.xsd架构文件的位置，以用作XML文档生成的基础。
默认情况下，该字段显示当前文件的完整路径。要使用其他架构，请单击“ 浏览” 并在打开的对话框中选择所需的文件。

实例文档名称

在此文本框中，指定要生成的输出XML文件的名称。默认情况下，生成的XML文件将继承源Schema的名称，并具有.xml扩展名。如果为要生成的文档键入其他名称，请确保扩展名正确。

生成的文档的默认位置是源.xsd架构文件所在的目录。要指定其他位置，请单击“ 浏览” 并在打开的对话框中选择所需的路径。

元素名称

在此下拉列表中，指定要用作生成文档的根的全局元素的本地名称。

启用限制检查

选中此复选框以使IntelliJ IDEA考虑限制粒子（如果指定的模式使用任何）。

启用唯一检查

选中此复选框以使IntelliJ IDEA考虑唯一性粒子（如果指定的Schema使用任何粒子）。

状态

查看此只读字段中的信息以跟踪和改进配置生成过程时的差异。
