Vue.js

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

PyCharm与Vue.js框架集成，旨在开发用户界面和高级单页面应用程序。

在你开始之前

  1. 确保您的计算机上有Node.js.

  2. 如“ 管理插件”中所述，在“ 插件”页面上安装并激活Vue.js存储库插件。

创建一个新的Vue.js应用程序

或者，按照Vue.js安装说明进行操作 。

创建新的Vue.js应用程序的推荐方法是使用vue-cli ，这是脚手架Vue.js应用程序的官方工具。如果使用Vue CLI版本2创建应用程序，则可以选择6个默认项目模板中的一个 ，然后回答有关在开发中使用的其他工具（如linters和test runner）的一些问题。

当然，您也可以创建一个空的PyCharm项目并在其中安装Vue.js.

安装vue-cli包

  • 打开内置终端 （ View | Tool Windows | Terminal ）并在命令提示符下键入npm install --global vue-cli 。

创建应用程序

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Vue.js。

  3. 在右侧窗格中：

    1. 指定将存储与项目相关的文件的文件夹的路径。

    2. 在Node Interpreter字段中，指定要使用的Node.js解释器。从列表中选择已配置的解释器，或选择“ 添加”以配置新的解释器

    3. 在vue-cli或@ vue / cli字段中，指定存储vue-cli或@vue/cli包的文件夹。

    4. 从“ 项目模板”列表中，选择要使用的Vue.js模板 。

  4. 单击“ 创建”时 ，PyCharm会向您询问有关可以添加到项目中的工具（如linters和测试运行器）的其他问题。这些问题取决于您选择的模板。之后，PyCharm生成一个包含所有必需配置文件的Vue.js特定项目。

要下载项目依赖项，请执行以下操作之一：

  • 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm安装

  • 从项目根目录中的package.json文件的上下文菜单中选择Run'npm install' 。

创建一个空的PyCharm项目

  1. 选择文件|新的|从主菜单中选择项目 ，或单击欢迎屏幕上的创建新项目 。“ 新建项目”对话框打开。

  2. 在左侧窗格中，选择Empty Project 。

  3. 指定将存储与项目相关的文件的文件夹的路径。

  4. 单击“ 创建”时 ，PyCharm将创建并打开一个空项目。

在空项目中安装Vue.js.

  1. 打开您将使用Vue.js的空项目。

  2. 在嵌入式终端 （ Alt + F12 ）中，键入：

    npm install vue

从现有的Vue.js应用程序开始

要继续开发现有的Vue.js应用程序，请在PyCharm中打开它并下载所需的依赖项。

打开计算机上已有的应用程序源

  • 单击欢迎屏幕上的打开或选择文件|从主菜单打开 。在打开的对话框中，选择存储源的文件夹。

从版本控制中检出应用程序源

  1. 单击“ 欢迎”屏幕上的“ 从版本控制中签出”或选择“ VCS”从主菜单中查看版本控制 。

  2. 从列表中选择您的版本控制系统。

  3. 在打开的特定于VCS的对话框中，键入您的凭据和存储库以检出应用程序源。

下载依赖项

  • 单击弹出窗口中的“运行'npm install' ：

创建Vue.js组件

Extract Vue组件重构仅在原地工作，因此请确保在Editor：General页面上选中Enable in-place mode复选框。

PyCharm识别.vue文件类型，并为Vue.js组件提供专用的.vue文件模板。

创建Vue.js组件

  • 在“ 项目”工具窗口中，选择新组件的父文件夹，然后从列表中选择“ Vue组件 ”。

您还可以从现有组件中提取新的Vue.js组件，而无需任何复制和粘贴，但使用专门的意图操作或重构。新提取的模板中使用的所有数据和方法都保留在父组件中。PyCharm将它们传递给带有props的新组件并复制相关的样式。

提取组件

  1. 选择要提取的模板片段并调用组件提取：

    • 要使用意图操作，请按Alt + Enter ，然后从列表中选择Extract Vue Component 。

    • 要使用重构，请选择Refactor |提取|从主菜单或选择的上下文菜单中提取Vue组件 。

  2. 输入新组件的名称。如果此名称已被使用或无效，PyCharm会显示警告。否则，将创建一个新的单文件组件并将其导入父组件。

编码协助

在.vue文件，PyCharm识别script ， style和template块。在script和style标记内，PyCharm允许您使用TypeScript，Pug和CSS预处理器而不是lang属性。

在script标记内使用TypeScript时，PyCharm会调用TypeScript语言服务进行类型检查，并在TypeScript工具窗口的“ 错误和编译错误”选项卡中显示检测到的错误。

默认情况下，您可以在script块内部的ECMAScript 6和style块内的CSS中获得代码完成。

在模板中，PyCharm自动添加结束花括号（ } ）并完成Vue.js指令。

PyCharm还为template标记内的Vue组件提供代码完成（ Ctrl + Space ）和导航（ Ctrl + B ）：

代码完成和定义导航也可用于Vue.js属性，数据对象中的属性，计算属性和方法。

使用Vue.js Live模板生成代码

使用PyCharm，您可以使用由Sarah Drasner创建的集合改编的Vue.js的Live模板集合 。

要使用Vue.js Live模板

  1. 键入模板的缩写或按Ctrl + J并从可用模板列表中选择它。

  2. 要展开模板，请按Tab键 。

  3. 要在模板内从一个变量移动到另一个变量，请再次按Tab键 。

用Prettier重新格式化Vue.js代码

有关详细信息，请参阅Prettier 。

漂亮的格式化工具.js ， .ts ， .css ， .less ， .scss ， .vue ，并.json代码。使用PyCharm，您可以使用Reformat with Prettier操作格式化选定的代码片段以及整个文件或目录。只要您将Prettier安装为项目中的依赖项或计算机上的全局依赖项，PyCharm就会添加此操作。

用Prettier重新格式化代码

  • 选择要在编辑器中重新格式化的代码片段，或在“ 项目”工具窗口中选择文件或文件夹，然后按N / A或从上下文菜单中选择“ 使用Prettier重新格式化” 。

  • 或者，按Ctrl + Shift + A并从“ 查找操作”列表中选择“ 使用Prettier重新格式化 ” 。

PyCharm可以将Prettier配置中的密钥代码样式规则应用于PyCharm代码样式设置，以便生成的代码（例如，在重构或快速修复之后）和已经使用Prettier处理的代码的格式一致。

要应用更漂亮的代码样式规则

  • 在启用Prettier的项目中，打开package.json ，然后在选项卡顶部的窗格中单击“ 是 ”。

  • 要重新应用Prettier代码样式（在窗格中单击No或修改代码样式后），请按Ctrl + Shift + A并从Find Action列表中选择Apply Prettier Code Style Rules 。
