命令行工具支持：Zend Framework 2

档案|设置|工具| Windows和Linux的命令行工具支持
IntelliJ IDEA |偏好|工具|命令行工具支持 macOS

单击“ 添加”按钮并在“ 选择要添加的工具”对话框中选择“ Zend Framework 2工具”时 ，将打开该对话框。
使用该对话框在IntelliJ IDEA中配置Zend Framework支持。
