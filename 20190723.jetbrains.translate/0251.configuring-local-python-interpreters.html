配置系统解释器

以下内容仅在安装并启用Python插件时有效！

要配置系统Python解释器，请按照下列步骤操作：

  1. 导航到文件|项目结构 （ Ctrl + Shift + Alt + S ）。

  2. 在“ 项目结构”对话框中，选择“ 平台设置”部分下的SDK ，然后单击 ，从弹出菜单中选择Python SDK 。

  3. 在“ 添加Python解释器”对话框的左侧窗格中，选择“ 系统解释器” 。

  4. 在“ 解释器”字段中，键入所需解释器可执行文件的完全限定路径，或单击在打开的Select Python Interpreter对话框中，选择所需的Python可执行文件，然后单击OK 。请注意，您需要管理员权限才能安装，删除和升级系统解释程序包。尝试通过意图操作安装解释程序包时，您可能会收到以下错误消息：

    根据提示，请考虑为项目使用虚拟环境。

  5. 单击“ 确定”完成任务。

对于任何配置的Python解释器，您可以：

  • 安装和卸载解释器路径
  • 安装和卸载解释器包
