在编辑器中选择文本

选择一段文本的基本方法是使用鼠标光标扩展选择。PyCharm作为以键盘为中心的IDE，建议使用导航键进行选择。

您可以选择选择文本片段，或在列模式中选择矩形片段，扩展和缩小选择，使用多重选择和粘性选择。

选择活动编辑器选项卡中的所有文本

要在当前编辑器选项卡中选择整个文本，请执行以下操作之一：

  • 在主菜单上，选择“ 编辑”|选择全部 。

  • 按Ctrl + A.

使用导航键选择

要从当前插入符号位置选择文本到当前单词的开头/结尾：

  • Ctrl + Shift +向左 ， Ctrl + Shift +向右 。

要从插入符号位置选择文本到当前行的开头/结尾：

  • 双击Ctrl并按Home / End

要从当前插入位置选择文本到屏幕的顶部/底部：

  • 按Ctrl + Shift + Page Up ， Ctrl + Shift + Page Down 。

选择代码的其他方法

要选择一列文本，请执行以下操作之一：

  • 按住Alt键，拖动鼠标指针选择所需的区域。

  • 按住鼠标中键，拖动鼠标指针选择所需区域。

  • 按Shift + Alt和鼠标中键。如果您想避免拖动，这特别有用。

扩大选择范围

要将选择从插入符号位置扩展到它所在的单词，然后再包含逻辑代码块，请执行以下操作之一：

  • 选择编辑|从主菜单中扩展选择 。

  • 按Ctrl + W选择插入符号当前所在的单词。

  • 连续按Ctrl + W将选择范围扩展到下一个包含节点（例如，表达式，配对标记，整个条件块，方法体，类，一组vararg参数等）

    在扩展选择时，请记住：

    • 在纯文本或注释中连续按Ctrl + W会将选择首先扩展到当前句子，然后再扩展到当前段落。

    • 在列表 ， 字典或参数或参数列表中连续按Ctrl + W ，选择列表中的元素，然后选择右边或左边的逗号和相邻空格（如果有），然后选择没有括号的列表内容，最后是括号括起来的。

选择缩小

要以相反的顺序缩小选择（从最外面的容器到插入符当前所在的单词），请执行以下操作之一：

  • 选择编辑|从主菜单收缩选择 。

  • 按Ctrl + Shift + W。

多选

PyCharm支持选择多个文本片段。这样做，可以选择多个单词，行或矩形。

要选择多个单词，请按以下步骤操作

  1. 执行以下操作之一：

    • 按Alt并使用鼠标左键双击要选择的单词。

      请注意，您的初始插入符号位置将被保留，此外，新插入符号将出现在所选单词的末尾。

    • 按Alt + J ，或选择一些文本片段。然后再次按Alt + J ，以查找并选择区分大小写的匹配单词或文本。如果您已经开始选择，那么将选择匹配的子串，否则将选择匹配的单词。

    • 将插入符号设置为单词并按Ctrl + Shift + Alt + J。将选择文档中所有区分大小写的匹配单词，插入符号位置将与初始单词中的相同。

  2. 检查选定的碎片和插入位置。请注意，如果选择了某些全局标识符或其部件，则您的编辑可能会破坏代码。

  3. 执行以下操作之一：

    • 使用箭头键调整多个插入符的位置并开始键入。在这种情况下，您键入的内容将添加到现有内容中

    • 立即开始输入。在这种情况下，所有选定的片段将替换为您键入的内容。

要选择多个文本片段，请按照下列步骤操作

  1. 按Alt键

  2. 拖动鼠标指针。

要选择多个矩形文本片段，请按照下列步骤操作

  1. 按Alt键 （Windows或UNIX）/（macOS）

  2. 拖动鼠标指针。

有关其他信息，请参阅Multicursor一节。

在行和列选择模式之间切换

要在行和列选择模式之间切换，请执行以下操作之一：

  • 从主菜单中，选择“ 编辑”|列选择模式 。

  • 从编辑器的上下文菜单中，选择“ 列选择模式” 。

  • 按Shift + Alt + Insert 。

粘性选择

要切换粘选择，按Ctrl + Shift + A键 ，在弹出的帧类型粘 ，从建议列表中选择切换粘性选择 ：

在Emacs键盘映射中，使用键盘快捷键N / A.

技巧和窍门

  • 粘贴选择打开时，可以通过调用复制或剪切或再次切换来关闭它。

  • 要创建矩形选区，首先使用给定的起点和终点进行正常选择，按Shift + Alt + Insert切换到列选择模式，然后在按住Ctrl的同时使用向上 / 向下键扩展选择。
