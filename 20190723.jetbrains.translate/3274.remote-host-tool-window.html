远程主机工具窗口

工具|部署|浏览远程主机

使用此工具窗口可以查看目标FTP / FTPS / SFTP服务器的文件夹结构以及上载到它们的数据。

访问远程主机工具窗口

  • 查看|工具窗口|远程主机 - 使用工具|打开工具窗口后，可以通过这种方式访问工具窗口部署|浏览远程主机命令。

  • 仅当启用了Remote Hosts Access捆绑插件时，工具窗口才可用。该插件默认处于活动状态。如果没有，请在“ 设置”对话框的“ 插件”页面中将其激活。

Shift + Enter键

上下文菜单

项目

描述

上传到这里

选择此选项可根据所选配置上载当前打开的项目中的文件。如果从文件的上下文菜单调用操作，则上载相应的本地文件。如果从文件夹调用操作，则会上载整个文件夹。

从这里下载

选择此选项可将所选文件或文件夹下载到当前打开的文件或文件夹。将更新现有本地文件，并将创建丢失的文件。

新

选择此选项可在所选文件夹中创建新的远程文件或文件夹。该选项仅在文件夹的上下文菜单中可用。

改名

选择此选项可重命名所选文件或文件夹。

删除

选择此选项可删除所选文件或文件夹。

切

选择此选项可将所选文件或文件夹的名称复制到剪贴板，并从树中删除文件或文件夹。

复制

选择此选项可将所选文件或文件夹的名称复制到剪贴板。

糊

选择此选项可将以前复制的文件名或文件夹插入树中。

编辑本地文件

选择文件并选择此选项时，WebStorm会在编辑器中打开其本地副本，并将焦点移动到相应的位置。仅当具有相应本地文件的项目当前在编辑器中打开时，该选项才可用。

与本地同步

选择此选项可将选定的远程文件夹与其本地版本进行比较。在打开的文件夹差异查看器中 ，浏览差异并同步文件（如果适用）。请参阅差异查看器中的两个文件夹比较 。

有关详细信息，请参阅比较已部署的文件和文件夹及其本地版本 。

与本地版本比较

选择此选项可将选定的远程文件与其本地版本进行比较。在打开的“ 差异查看器文件”对话框中，探索差异并在必要时使用它们应用它们按钮。有关详细信息，请参阅查看文件之间的差异 。

有关详细信息，请参阅比较已部署的文件和文件夹及其本地版本 。

编辑远程文件

选择此选项可在WebStorm编辑器中编辑所选文件，而无需将其添加到当前打开的项目中。

有关详细信息，请参阅编辑远程主机上的单个文件

复制路径

排除路径

选择此选项可从上载/下载中排除所选文件夹。

有关详细信息，请参阅从上载/下载中排除文件和文件夹 。

复制路径

选择此选项可将服务器上所选文件或文件夹的绝对路径复制到剪贴板。
