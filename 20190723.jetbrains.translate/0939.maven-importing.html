保留项目文件

选中此复选框可在导入后指定项目文件的位置。例如，当您导入项目并希望将.iml文件和.idea目录文件保留在特定位置而不是默认位置时。默认情况下，IntelliJ IDEA将项目文件放在pom.xml旁边。

自动导入Maven项目

如果希望每次更改pom.xml时IntelliJ IDEA都自动执行重新导入，请选中此复选框。

为聚合器项目创建IntelliJ IDEA模块（使用'pom'包装）

如果选中此复选框，则将为聚合项目的pom.xml文件中包含的每个模块创建IntelliJ IDEA Maven模块 ，前提是其包装设置为“pom”。

为多模块Maven项目创建模块组

如果选中此复选框，IntelliJ IDEA将从聚合Maven项目创建模块组，其中嵌套模块包含在该组中。

保持源和测试文件夹重新导入

如果选中此复选框，则每次导入时都将保留所有源文件和测试文件夹。
如果清除此复选框，则每次导入时都将删除所有先前配置的源和测试文件夹。

默认情况下，此复选框设置如下：

  • 对于新项目 ：清除复选框。

  • 对于已导入的项目 ：选中该复选框。

排除构建目录
（ PROJECT_ROOT/target ）

选中此复选框可从项目中排除构建目录。如果您想加快项目的导入过程，这可能很有用。如果清除此复选框，则每次导入项目时，IntelliJ IDEA都会对构建目录中的文件编制索引，这可能需要额外的时间。

使用Maven输出目录

如果未选中此复选框，则将在常规IntelliJ IDEA的输出目录USER_HOME\IdeaProjects\<project>\classes\Production\创建构建。
如果选中此复选框，则会在Maven的输出目录中生成构建，并重新使用IntelliJ IDEA的编译结果。但是，IntelliJ IDEA本身不会重用Maven构建结果，而是从头开始编译。

生成的源文件夹

重新导入项目时指定源根目录。
您可以选择以下选项之一：

  • 自动检测这是默认选项。选择此选项时，IntelliJ IDEA会自动检测生成的源的位置。IntelliJ IDEA还会检测要标记为源根目录的目录。但是，IntelliJ IDEA仅在目标/生成源和目标/生成源/ *目录中搜索生成的源 。

  • target / generated-sources此选项使您可以手动将目录标记为源根目录。

  • “target / generated-sources”的子目录此选项使您可以手动将子目录标记为源根目录。

  • 不检测此选项可让您跳过检测过程。

用于文件夹更新的阶段

选择要用于文件夹更新的Maven阶段。如果您调整插件以便在某个阶段加载其他源，这可能很有用。

自动下载

选择相应的复选框以自动下载有关打开Maven项目的源（ 源 ）和文档注释（ 文档 ）。

依赖类型

使用此字段指定重新导入项目时要包括的依赖关系类型。

导入程序的VM选项

使用此字段指定VM选项。默认选项为-Xmx512m 。
指定选项时，请遵循以下规则：

  • 使用空格分隔各个选项，例如-client -ea -Xmx1024m 。

  • 如果选项包含空格，请将空格或包含空格的参数括在双引号中，例如， some" "arg或"some arg" 。

  • 如果选项包含双引号（例如作为参数的一部分），则通过反斜杠转义双引号，例如-Dmy.prop=\"quoted_value\" 。

JDK for importer

使用此下拉列表指定导入maven项目时要使用的JDK。
您可以选择以下选项之一：

  • 内部JRE - 这是一个使用JRE安装目录的默认选项。

  • JDK版本 - 此选项使用项目的JDK。

  • 使用JAVA_HOME - 此选项使用用户环境变量设置中指定的值。

导入Flexmojos项目时生成Flex编译器配置文件

如果选中此复选框，则在导入Flexmojos项目时，将自动生成Flex编译器配置文件。

自动生成配置文件是一个相当耗时的过程，特别是对于大型项目。此外，由于独立于IntelliJ IDEA的原因，有时可能无法自动生成配置文件或可能导致错误的结果。

在所有这些情况下，您可能希望关闭此选项并使用以下命令从命令行生成Flex编译器配置文件：

mvn compile -DconfigurationReport = true

要使此选项可用，IntelliJ IDEA中启用了Flash / Flex支持 。该插件默认激活。如果插件已禁用，请在“ 插件”页面上启用它，如管理插件中所述 。

为Spring Boot应用程序创建运行配置

如果希望IntelliJ IDEA在Maven导入上为Spring Boot项目自动创建运行/调试配置，请选择此选项。
