覆盖超类的方法

您可以通过从预定义模板生成必要的代码来覆盖父类的任何方法。IntelliJ IDEA创建一个包含对超类方法的调用的存根，让开发人员在该方法的主体中提供一些有意义的源代码。

覆盖超类的方法

  1. 在“ 代码”菜单上，单击“ 替代方法” （ Ctrl + O ）。或者，您可以右键单击类文件中的任意位置，然后单击“ 生成 （ Alt + Insert ）”，然后选择“ 覆盖方法” 。

  2. 选择要覆盖的方法。方法列表不包括已被覆盖的方法，或者无法从当前子类访问的方法。

    如有必要，选择Copy JavaDoc复选框以为已实现的接口或抽象类插入JavaDoc注释。

    如有必要，请选中“ 复制JavaDoc”复选框，以便为重写的方法插入JavaDoc注释。

    如有必要，选择Insert @Override复选框以插入@Override注释。

  3. 单击“ 确定”并提供方法主体的源代码。

    将鼠标悬停在在装订线中的图标，以查看基类的名称。单击它以打开重写的方法声明 。

更改方法体

用于覆盖方法的代码模板（ 重写方法体 ）接受来自File Header包含模板的预定义模板变量（例如， ${USER} ， ${DATE}等）。

例如，请考虑以下代码模板：

＃ 如果 （$ RETURN_TYPE！= “void” ） 返回 $ DEFAULT_RETURN_VALUE ; ＃ 结束 // TODO（$ USER，$ DATE）：要更改实施方式的主体，使用文件|设置 - 编辑器 - 文件和代码模板。

如果重写的类包含两个方法，则此模板将扩展为以下代码：

public void breathe （） { // TODO（wombat，9/21/12）：要更改方法体，请使用设置 - 编辑器 - 文件和代码模板。 } public void eat （） { // TODO（wombat，9/21/12）：要更改方法体，请使用设置 - 编辑器 - 文件和代码模板。 }
