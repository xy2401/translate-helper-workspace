从Eclipse导入。第1页

使用此页面可以为导入的Eclipse项目定义源工作空间和目标位置。

项目

描述

选择Eclipse项目目录

指定包含要导入的项目的Eclipse工作空间的路径。您可以键入路径，或单击省略号按钮并在“选择路径”对话框中找到所需的目录。

IntelliJ IDEA项目和模块文件位置

选项

描述

在.classpath文件附近创建模块文件

如果选择此选项，将在相应的项目目录中创建每个Eclipse项目的IntelliJ IDEA模块;将在Eclipse工作区的根目录或Eclipse项目目录中创建指定格式的IntelliJ IDEA项目。

保留项目和模块文件

如果选择此选项，请从导入的Eclipse项目中输入要创建的*.iml文件的目标位置，或单击省略号按钮，然后导航到所需位置。

项目文件格式

选择基于文件或基于目录的保存项目格式 。

将IntelliJ IDEA模块链接到Eclipse项目文件

选中此选项可自动保持Eclipse项目和IntelliJ IDEA模块的同步。

检测测试源

指定根目录列表，其中应寻找测试源。有关通配符语法，请参阅“ 编译器”部分。
