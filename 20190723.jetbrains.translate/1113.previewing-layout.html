预览布局定义文件的输出

使用IntelliJ IDEA，您可以预览当前打开的布局定义文件的输出，而无需启动物理或虚拟设备。更新布局定义文件时，预览会动态更改。使用布局预览，您可以将应用程序调整到不同的Android平台，设备，方向，停靠模式，区域设置等。

根据不同的模式中，您在设计应用程序布局，你可以预览以不同的方式输出：

  • 在专用的预览工具窗口中，手动编辑布局定义文件时。打开布局定义文件并切换到“ 文本”选项卡时，将显示工具窗口。

  • 在使用Android UI设计 器 编辑布局时，在编辑 器中 。

在本页面：

  • 访问布局预览
  • 预览不同环境中的布局
  • 调整预览外观

访问布局预览

  1. 在编辑器中打开所需的布局定义文件。

  2. 选择编辑模式：

    • 要手动编辑布局，请切换到“ 文本”选项卡。预览工具窗口打开。

    • 要在Designer工具窗口中编辑布局，请切换到“ 设计”选项卡。在此选项卡中，您可以在可视模式下编辑布局，而无需编辑文本定义文件。

配置以在IDE中呈现此布局

从此下拉列表中，选择要预览和编辑的布局配置，创建新的布局配置或选择不同的预览选项。可用选项包括：

  • 创建横向变化 ：选择此选项可创建横向版本的横向。将在res\layout-land文件夹中生成相应的布局定义文件。创建此变体后，此菜单选项将替换为“ 切换到layout-land”选项，该选项将打开layout-land\<layout_file_name>.xml文件以进行编辑。

  • 创建layout-xlarge变化 ：选择此选项可为超大屏幕尺寸（至少960x720 dp） 创建布局变体。将在res\layout-xlarge文件夹中生成相应的布局定义文件。创建此变体后，此菜单选项将替换为“ 切换到layout-xlarge”选项，该选项将打开layout-xlarge\<layout_file_name>.xml文件以进行编辑。

  • 切换到布局 ：此选项仅在您创建了多个布局版本时可用。选择它以返回到原始布局定义文件。

  • 创建其他 ：选择此选项可创建布局的另一种变体。在打开的“ 选择布局目录”对话框中，指定将存储布局定义的文件夹，并选择用于确定特定设备配置的资源限定符 。选择相关限定符并单击 。然后在打开的对话框中指定限定符的值。限定符将添加到“ 选择的限定符”列表中。

  • 预览代表示例选择此选项可显示多个设备配置，并在最重要的屏幕尺寸上预览布局。

  • 预览所有屏幕尺寸 ：选择此选项可显示多个设备配置并在所有可用屏幕尺寸上预览布局。

  • 预览所有区域设置 ：选择此选项可在将要使用您的应用程序的所有区域设置中预览布局。

  • 预览从右到左布局 ：选择此选项可以并排预览两个方向（从左到右和从右到左）的布局。

  • 预览Android版本 ：选择此选项可预览所有已安装的Android API版本的布局。

  • 预览包含 ：选择此选项可预览嵌套在另一个布局中的布局。仅当当前布局包含在另一个布局中时，此选项才可用。

  • 预览布局版本 ：选择此选项可显示多个设备配置，并在所有可用变体中预览布局。

  • 无 ：选择此选项可返回默认视图。

  • 切换布局模式 ：选择此选项可在不同的预览选项之间切换。

用于呈现布局的虚拟设备

从此下拉列表中，选择虚拟或物理设备以预览应用程序在此设备上的外观。要添加新虚拟设备，请选择“ 添加设备定义”并在打开的Android虚拟设备（AVD）管理器中配置仿真器（有关说明，请参阅管理虚拟设备 ）。

转到下一个州

从此下拉列表中，选择预览方向（ 纵向或横向 ），UI模式（ 普通，车载底座，桌面底座，电视，设备 ），并在夜间 （暗屏幕）和非夜景 （标准亮度）模式之间切换。有关UI模式的详细信息，请参阅UiModeManager 。

主题

单击此按钮可从“ 选择主题”对话框中选择主题 。

N / A

单击此按钮可将布局与活动相关联。选择与<activity_name>关联以将其与当前活动相关联，或选择与其他活动关联以显示可供选择的可用活动列表。

用于在IDE中呈现布局的区域设置

从此下拉列表中，选择现有区域设置或为应用程序添加区域设置。区域设置是目标国家/地区和语言的组合，以根据当地规则和偏好显示日期和其他一些数据。您还可以在所有可用的区域设置和两个方向（从左到右和从右到左）预览布局。

在IDE中渲染布局时使用的Android版本

从此下拉列表中，选择API版本或使用“ 自动选择最佳”选项以使用最合适的Android版本呈现布局。您还可以在所有已安装的Android API版本上预览布局。

通常，在更新定义文件时，预览会即时更改。如果没有，请单击“ 刷新”按钮让IntelliJ IDEA更新预览。

调整预览外观

在“ 预览”工具窗口中预览布局输出并在编辑器中手动设计布局时，可能需要移动工具窗口的边界，以便在编辑器中有更多可用空间。在这种情况下，压缩预览以使其适合工具窗口大小但仍然反映模拟环境中的布局输出可能会有所帮助。

显示选项菜单

单击此按钮可配置工具窗口的外观和行为。

  • 隐藏非布局文件：选择此选项可使IntelliJ IDEA在编辑器中的焦点切换到非布局文件时暂时关闭工具窗口。只要相同或另一个布局定义文件处于焦点，工具窗口就会自动重新出现。

  • 包括设备框架（如果可用）：选择此选项可使预览看起来像将显示在实际设备上。

  • 显示光照效果：选择此选项可显示光照效果，使预览看起来更自然。

    仅当选择“ 包括设备框架”选项时，此选项才可用。
