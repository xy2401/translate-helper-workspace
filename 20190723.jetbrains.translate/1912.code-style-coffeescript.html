标签和缩进

使用制表符

  • 如果选中此复选框，则选项卡字符用于缩进和代码重新格式化。

  • 清除该复选框后，PyCharm使用空格而不是制表符。

智能标签

如果选中“ 使用选项卡字符”复选框，则“ 智能选项卡”复选框可用。

缩进由两部分组成。一部分来自嵌套代码块，另一部分是通过对齐来确定的。

  • 如果选中此复选框，则嵌套产生的部分包含制表符和空格（如果需要），而由对齐定义的部分仅包含空格。

  • 如果清除此复选框，则仅使用选项卡。这意味着在重新格式化一组符合指定选项卡大小的空格后，将自动替换为选项卡，这可能会导致细微对齐。

标签大小

在此文本框中，指定适合选项卡的空格数。

缩进

在此文本框中，指定要为每个缩进级别插入的空格数（如果选中“ 使用选项卡字符”复选框，则为选项卡 ）。

继续缩进

在此文本框中，指定要在数组元素之间，表达式，方法声明和方法调用之间插入的空格数（或选中“ 使用选项卡字符”复选框）。

在空行上保持缩进

如果选中此复选框，PyCharm会在空行上保留缩进，就好像它们包含一些代码一样。如果清除该复选框，PyCharm将删除空行上的制表符和空格。

缩进链式方法

在函数声明中，链中的第二个和更多方法显示在单独的行上。

  • 选中该复选框后，链中的第二个和其他方法将与第一个调用对齐。

  • 清除该复选框后，链中的第二个和更多方法将与调用它们的对象对齐。

空间

使用此选项卡指定PyCharm自动插入空格的位置。选中相关位置描述旁边的复选框，然后在“ 预览”窗格中检查结果。

包裹和括号

在此选项卡中，自定义PyCharm在重新格式化源代码时将应用于各种代码构造的异常，大括号放置和对齐选项。在“ 预览”窗格中检查结果。

对齐优先于缩进选项。

硬包装
在此字段中，指定元素右侧所需的空格数。如果接受“ 默认”选项，则使用全局设置中的值。

打字输入
在此字段中，指定编辑的文本在指定的Hard wrap at字段中的拟合方式。

  • 默认 - 选择此选项以使用全局设置中的Wrap on键值。

  • 是 - 选择此选项以使用右边距字段中的值。

  • 否 - 如果选择此选项，则行可以超过右边距中指定的值。

可视指南在此字段中，指定多个右边距。您可以保留默认值或输入保证金的空格数。如果要指定多个边距，请输入以逗号分隔的数字。

重新格式化时保留
使用复选框配置PyCharm在重新格式化源代码时将进行的异常。例如，默认情况下，选中“ 换行符”复选框。如果您的代码包含比标准约定更短的行，则可以在重新格式化之前通过禁用换行符复选框来转换它们。

包装选项
包装样式适用于左侧窗格中指定的各种代码构造（例如，方法调用参数或赋值语句）。

不要换行

选择此选项后，不会应用任何特殊的包装样式，将忽略嵌套的对齐和大括号设置。

包裹多久

选择此选项可以使用适当的缩进来包围超出右边距的线条。

永远包裹

选择此选项可将所有元素包装在列表中，以便每行有一个元素并具有适当的缩进。

如果长时间砍下来

选择此选项可以在超出右边距的列表中包装元素，以便每行有一个元素并且具有适当的缩进。

对齐选项

多线时对齐

如果选中此复选框，则代码构造将在每个下一行的同一列开始。否则，代码构造的位置由当前缩进级别确定。

下一行<character(s)>

选中此复选框可在包装线条时将指定的一个或多个字符移动到下一行。

新线上的“其他”

使用此复选框将相应的语句或字符移动到下一行。

<character>之后的新行

选中此复选框可将指定字符后面的代码移动到新行。

特别是如果治疗

如果选中此复选框， else if语句位于同一行。

否则， else if语句被移动到相应缩进级别的下一行。

变量声明

选择以下选项之一以配置相等符号的对齐：

  • 不要对齐 - 等号不对齐。

  • 多行时对齐 - 通过插入额外的空格来对齐多行var语句中的等号。

  • 分组时对齐 - 通过插入其他空格来对齐多个var语句中的相等符号。

ES6导入/导出

 对齐'from'子句：选中此复选框后，PyCharm会自动对齐ECMAScript 6代码中的import和export语句，使您的代码更易于阅读和维护。在“ 预览”窗格中比较代码片段的外观与对齐以及没有它的情况。

启用此选项后，PyCharm将动态对齐新代码。在按Ctrl + Alt + L重新格式化代码后，将对齐现有的import和export语句，有关详细信息，请参阅“ 重新格式化源代码”部分。

大括号放置选项

大括号放置方式

使用此下拉列表在类声明 ， 方法声明 ， 函数声明和其他类型的声明中指定左大括号的位置。可用选项包括：

  • 行尾 - 选择此选项可将开括号放在声明行末尾。

  • 如果换行，则选择下一行 - 选择此选项可将左大括号放在多行声明行之后的行的开头。

  • 下一行 - 选择此选项可将开始括号放在声明行之后的行的开头。

  • 下一行移动 - 选择此选项可在声明行移动到相应的缩进级别后将左大括号放在该行上。

  • 每个移动的下一行 - 选择此选项以在声明行移动到相应的缩进级别后将开始括号放置在行上，并将下一行移动到下一个缩进级别。

强制支撑

从此下拉列表中，为if ， for ， while和do () while语句选择大括号引入方法。可用选项包括：

  • 不要强制 - 选择此选项可禁止自动引入括号。

  • 多行时 - 如果语句占用多行 ，请选择此选项以自动插入大括号。请注意，PyCharm分析整个语句中的行数，但不仅仅是它的条件。

  • 始终 - 选中此复选框后，PyCharm将始终自动插入大括号。

空白行

使用此选项卡可以定义PyCharm在重新格式化后保留并插入代码中的空白行的位置和数量。结果显示在“ 预览”窗格中。

保持最大空行

在此区域中，指定重新格式化后要保留的额外空行数。

这些设置不会影响第一个项目之前和之后的空白行数。
