使用命令行中的PyCharm功能

除了在PyCharm中工作外，您还可以从命令行执行以下操作：

  • 从命令行打开文件
  • 从命令行格式化文件
  • 从命令行启动代码检查
  • 从命令行查看文件之间的差异
  • 从命令行合并文件

在Linux上，将PyCharm安装为快照包会自动创建名为pycharm-professional ， pycharm-community或pycharm-educational的命令行启动pycharm-professional 。工具|因此，无法创建命令行启动器命令。

在Windows上启用命令行启动程序

  • 将PyCharm bin文件夹的Path添加到Path环境变量（例如， C:\Program Files\JetBrains\PyCharm\bin ）。配置Path环境变量后，可以运行PyCharm命令。

在Linux和macOS上启用命令行启动程序

在macOS和Linux上，创建命令行启动程序以将PyCharm与shell集成。确保创建的启动程序脚本位于shell的搜索路径中，以便可以在系统范围内调用它。

  1. 在主菜单中，选择“ 工具”|创建命令行启动器 。在“ 创建启动程序脚本”对话框中，提供启动程序的名称和路径，或接受建议的值。

  2. 在PyCharm之外，将路径和启动器脚本的名称添加到path 。

从命令行启动PyCharm

  • 要从命令行启动PyCharm，请在命令行中键入<PyCharm>命令，然后按Enter键 。 <PyCharm>是特定于平台的启动器。
