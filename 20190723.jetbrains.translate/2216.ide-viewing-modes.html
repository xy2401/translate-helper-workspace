PyCharm查看模式

PyCharm提供以下视图模式：

  • 全屏模式允许您使用整个屏幕进行编码。此模式从视图中删除所有菜单以及操作系统控件。但是，您可以使用上下文菜单和键盘快捷键。将鼠标指针悬停在屏幕顶部时，也可以使用主菜单。

  • 演示模式类似于全屏模式 ，但它用于进行涉及使用PyCharm进行编码的演示。在这种模式下，PyCharm会增加字体大小并隐藏除编辑器之外的所有内容。如有必要，还可以使用View |中的相应项目在此视图中显示工具窗口工具Windows菜单。

  • 无分散模式显示没有工具栏，没有工具窗口，没有编辑器选项卡;代码是中心对齐的，依此类推。

这些操作仅可通过“ 查看”菜单使用。默认情况下，它们不会映射到任何快捷方式，但您可以按照配置键盘快捷方式中的说明创建自己的快捷方式 。

切换全屏模式

在macOS上，您可以使用N / A切换全屏模式。

除了操作工具窗口 （显示/隐藏或调整它们）之外，PyCharm还可以最大化整个产品窗口，隐藏主菜单。

  • 要切换到全屏模式，请选择“ 查看”|在主菜单上输入全屏 。

  • 要退出全屏模式，请选择“ 查看”|在主菜单上退出全屏 。

切换演示模式

在演示模式下，编辑器占据整个屏幕，而所有其他PyCharm组件都被隐藏。

除此之外，此模式下的字体大小比平常大。您可以在“设置/首选项”对话框的“ 外观”页面中定义演示模式的字体大小。

  • 要切换到演示模式，请选择“ 视图”|在主菜单上进入演示模式 。

  • 要退出演示模式，请选择“ 视图”|在主菜单上退出演示模式 。

切换无干扰模式

在无干扰模式下，编辑器占用整个PyCharm框架，没有任何编辑器选项卡和工具窗口按钮。代码是中心对齐的。

  • 要切换到无干扰模式，请选择“ 查看”|在主菜单上输入Distraction-Free Mode 。

  • 要退出无干扰模式，请选择“ 查看”|退出无牵引模式 。

在“切换”列表中切换查看模式

  1. 按Ctrl +`或选择查看|主菜单上的快速切换方案 。

  2. 在打开的“ 切换”列表中，选择“ 查看模式” 。

  3. 在上下文菜单中，选择所需的模式。菜单内容取决于您当前的模式：

    • 进入演示模式 / 退出演示模式

    • 进入无牵引模式 / 退出分心自由模式

    • 进入全屏 / 退出全屏
