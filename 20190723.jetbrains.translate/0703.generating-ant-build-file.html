生成Ant构建文件

IntelliJ IDEA提供了为项目自动生成Ant构建文件的可能性，或者为构成项目的每个模块自动生成多个构建文件。生成构建文件后，可以将其添加到Ant构建工具窗口 。您可以编辑和管理此类构建文件作为常规文件。

生成Ant构建文件

  1. 在主菜单上，选择Build |生成Ant构建 。

  2. 在Generate Ant Build对话框中 ，指定生成选项。

    • 选择生成单文件或多文件构建。

    • 选择保留或覆盖以前生成的文件。

    有关控件的详细说明，请参阅“ 生成Ant构建对话框 ”主题。

  3. 单击确定 。
