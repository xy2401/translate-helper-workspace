将Java文件转换为Kotlin文件

IntelliJ IDEA使您可以将Java文件转换为Kotlin。

将Java文件转换为Kotlin

  1. 在主菜单上，指向代码菜单。

  2. 选择“ 将Java文件转换为Kotlin文件” 。

请注意，生成的Kotlin文件将显示在原始Java文件的位置。
