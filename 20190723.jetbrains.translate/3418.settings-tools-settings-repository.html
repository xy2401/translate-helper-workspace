设置存储库

档案|设置|工具|适用于Windows和Linux的设置存储库
WebStorm |偏好|工具|设置 macOS的存储库
按Ctrl + Alt + S

此页面显示在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，启用“设置存储库”插件时。

该插件与WebStorm捆绑在一起，默认情况下处于激活状态。如果禁用，则可以手动启用插件 。

使用此页面配置“ 设置存储库”功能，该功能允许您在安装在不同计算机上的WebStorm（或其他基于IntelliJ平台的）产品的不同实例之间共享IDE设置。

您要共享的设置必须存储在Git存储库中。

自动同步

如果希望每次执行更新项目或推送操作时，或者当您关闭项目或退出WebStorm时，本地设置都会自动与存储库中存储的设置同步，请选中此复选框。

如果禁用此选项，则可以通过选择VCS |手动更新设置从主菜单中同步设置 。

只读来源

使用此部分配置包含要共享的任何类型设置的其他存储库，包括实时模板，文件模板，方案，部署选项等。

这些存储库不能被覆盖或合并，只能用作设置源。

使用以下控件来管理只读存储库：

单击此按钮可添加包含要共享的设置的GitHub存储库的URL。

单击此按钮可从列表中删除选定的存储库。

单击此按钮可编辑所选源的URL。

使用这些按钮在列表中上下移动。

单击此按钮可克隆选定的URL。
