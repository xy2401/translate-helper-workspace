文档工具窗口

查看|工具窗口|文档
按Ctrl + Q

在以下情况下会出现此工具窗口：

  • 通过单击固定快速文档查找 在查找窗口的右上方。

  • 快速定义工具提示通过单击固定在工具提示的右上角。

  • 对于科学观点 。

因此，工具窗口将显示在“ 视图”菜单中的可用工具窗口列表中。

图标

捷径

描述

 

左或右

切换到上一个或下一个文档页面（例如，在使用超链接后）。

在macOS计算机上，您还可以使用三指从右到左和从左到右的滑动手势。

F4

切换到与当前显示的文档页面对应的项目（例如，源）。

单击此图标可执行以下设置：

  • 显示字体大小滑块。根据需要移动滑块以在快速文档窗口中增大或减小字体大小。

  • 从其源启用或禁用文档的自动更新。

  • 控制窗口的位置。
