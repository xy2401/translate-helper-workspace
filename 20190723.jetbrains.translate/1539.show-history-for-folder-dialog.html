显示文件夹对话框的历史记录

VCS |当地历史|显示历史

使用此对话框可以浏览文件夹的更改。此对话框中有两个视图：

  • 左侧部分的历史记录视图

  • 更改右侧部分的视图

文件夹的上下文菜单中提供了相同的对话框。

历史观点

“历史记录”视图显示文件夹修订（状态）列表，每个文件夹都提供了时间戳，修订号以及导致该状态的操作的指示。

项目

描述

单击此按钮可还原所选操作。

单击此按钮可基于所选本地版本创建修补程序。

单击此按钮可打开相应的帮助主题。

更改视图

“更改”视图显示当前状态与“ 历史记录”视图中选择的状态之间的差异。差异显示为已更改（新，已修改和已删除）文件和子文件夹的树。

项目

捷径

描述

按Ctrl + d

单击此按钮可显示当前本地版本与“历史记录”视图中选定的版本之间的差异 。或者，在“更改”视图中双击当前本地版本。

在“更改”视图中选择文件后，单击此按钮可回滚所选操作。

按Ctrl + P

单击此按钮可将更改的文件显示为文件夹的树视图。如果未按下此按钮，则文件显示为平面列表。

要么

Ctrl + NumPad Plus或Ctrl + NumPad -

单击这些按钮可以展开或折叠所有节点。这些按钮仅在更改的文件显示为树视图时可用。

按Ctrl + A

单击此按钮可选择列表中的所有文件或树视图。

按Ctrl + F

在此区域中，键入搜索字符串。另请注意，“更改”窗格中提供了速度搜索 。

单击此按钮可清除搜索区域。

版本控制视图中提供了速度搜索 。
