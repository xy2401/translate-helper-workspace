定义运行时属性

使用“ 构建文件属性”对话框的“ 属性”选项卡可以在运行时将属性传递给构建脚本。指定的值等效于命令行启动程序的-D选项后定义的值。

除了普通值，您还可以使用在运行时计算的宏。当您必须将特定路径和其他不同信息传递给构建脚本时，此类宏很有用。宏是由美元符号包围的字符串。“构建文件属性”对话框提供了可用于所选构建文件的宏的完整列表，其中显示了预览，以及如何在运行时评估这些宏。

定义运行时属性

  1. 打开“构建文件属性”对话框 。

  2. 在Properties选项卡中，单击Add 。

  3. 在“ 名称”列中，键入属性名称。

  4. 在“ 值”列中，键入所需的值。如果使用宏作为属性值，请键入所需宏的名称。如果您不知道名称，请单击按钮，从“宏”对话框中选择所需的宏，然后单击“ 确定” 。

  5. 使用“ 添加”和“ 删除”按钮组成完整的属性列表。
