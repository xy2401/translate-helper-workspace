新项目：Drupal模块

档案|新的|项目

欢迎屏幕|创建新项目

当您在左侧窗格中选择PHP ，然后从右侧窗格的列表中选择Drupal Module时， Project Category和Options对话框的右侧窗格如下所示。

使用此对话框可以生成和设置符合Drupal要求的模块存根。

安装并启用Drupal Support插件时，仅在Ultimate版本中支持该功能。 该插件未与IntelliJ IDEA捆绑在一起，但可以从JetBrains插件存储库中安装，如管理插件中所述 。

项目

描述

项目名

在此字段中，指定项目的名称，默认情况下，它是位置字段中指定的文件夹的名称。

项目位置

在此字段中，指定项目文件夹的路径，其中将存储与项目相关的文件。

安装路径

在此字段中，指定的根文件夹安装。

设置PHP |包含路径

  • 选中此复选框可让Drupal包含为项目自动配置的路径。

    离开对话框后，以下路径将添加到PHP页面上的Include Paths列表中： <drupal installation root>/includes ， <drupal installation root>/modules和<drupal installation root>/sites/all/modules

  • 清除复选框以手动配置包含路径。

版

从此列表中，选择要使用的Drupal版本，支持的版本为6,7和8。

JConfig的路径

在此字段中，提供Joomla期间创建的Joomla configuration.php的路径！安装 。
