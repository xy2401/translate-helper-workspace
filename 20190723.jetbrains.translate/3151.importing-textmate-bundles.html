WebStorm 2019.1帮助

导入TextMate包

当WebStorm 下载到您的计算机时，它们可供WebStorm使用。

  1. 下载要导入WebStorm的TextMate包。例如，您可以在GitHub上找到所需的TextMate包。

    WebStorm无法读取二进制包。仅使用捆绑源。

  2. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ TextMate Bundles” 。

  3. 点击并选择文件系统中的包。

  4. 如有必要，将所需的文件类型添加到导入的TextMate包中。

    例如，如果您希望WebStorm的TextMate集成支持扩展名为.rb Ruby文件，请打开Ruby.tmbundle \ Syntaxes \ Ruby.plist文件，找到fileTypes部分，然后在array下添加rb 。

  5. 重启WebStorm。
