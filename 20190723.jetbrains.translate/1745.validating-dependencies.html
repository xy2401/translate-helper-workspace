验证依赖关系

分析您可能需要从分析过程中排除其中一些的依赖关系。换句话说，您可以为依赖关系分析创建自己的规则。

验证依赖项

  1. 在依赖关系查看器的工具栏中，单击“编辑规则” 按钮。将打开“ 依赖关系验证”对话框 。

  2. 定义禁止和允许使用的范围：

    1. 单击“拒绝/允许”部分中的“ 添加”按钮以添加默认范围。

    2. 单击Deny / Allow usages of列中的范围条目，然后从下拉列表中选择一个预定义范围，或单击省略号按钮并在“ 范围”对话框中定义范围。

    3. 单击in / only in列中的范围条目，并选择一个预定义的范围，或单击省略号按钮并在“ 范围”对话框中定义范围。

    4. 使用“ 添加”和“ 删除”按钮组成完整列表。

  3. 应用更改。依赖性分析重新运行新规则。
