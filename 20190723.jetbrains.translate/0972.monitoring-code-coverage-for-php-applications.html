监控PHP应用程序的代码覆盖率

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用PHP插件时有效！

PHPUnit的 代码覆盖率向您展示了在编辑器中可视化覆盖的测试和标记以及未覆盖的行覆盖了多少代码。

在这个部分：

  • 先决条件
  • 配置代码覆盖率
  • 测量代码覆盖率

先决条件

  1. PHPUnit工具安装在您的计算机上，并在IntelliJ IDEA中启用 。

  2. 在当前的PHP解释器中配置了Xdebug或Zend Debugger调试引擎。

配置代码覆盖率

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 构建”，“执行”，“部署”下的“ 覆盖范围 ”。

  2. 在Coverage页面上指定coverage选项。

测量代码覆盖率

  1. 手动编写测试。

  2. 创建 PHPUnit类型的运行配置 。

  3. 在主工具栏上，在“ 运行/调试配置”下拉列表中选择PHPUnit运行配置，然后单击“使用覆盖率运行”按钮 。

  4. 在Coverage工具窗口中监视代码覆盖率 。
