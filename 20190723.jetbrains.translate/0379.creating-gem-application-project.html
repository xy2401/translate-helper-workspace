创建宝石项目

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

Gem项目旨在开发，构建和推送gems到github。

创建一个Gem项目

  1. 在主菜单上，选择“ 文件”|新的|单击项目 ，或单击欢迎屏幕上的新建项目 。“新建项目”对话框打开。

  2. 在“ 新建项目”对话框中，执行以下操作：

    • 选择宝石 。

    • 选择Ruby解释器，或接受建议的解释器。如果缺少所需的Ruby解释器，请单击“ 新建” ，然后配置您选择的解释器。

    • 观察建议的Bundler版本。

    • 根据需要单击Minitest或RSpec单选按钮。
      选择或清除下面的复选框。有关详细信息，请参阅Bundler文档 。

    • 单击下一步 。

    • 指定项目名称和位置。

  3. 单击完成 。
