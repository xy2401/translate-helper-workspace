文件缓存冲突

外部进程更改了文件，在PyCharm中打开和未保存，导致文件的两个冲突版本。使用以下选项解决此冲突。

项目

描述

加载FS更改

单击此按钮可加载在PyCharm外部生成的文件版本，并覆盖您的本地更改。

保持记忆变化

单击此按钮可保留PyCharm中生成的版本并存储在缓存中。

显示差异

单击此按钮可调用差异查看器 ，该查看器在左侧显示文件系统中的版本，并在右侧显示PyCharm版本。您可以根据需要合并差异，并将所需的版本加载到PyCharm。默认情况下，将保存缓存版本。
