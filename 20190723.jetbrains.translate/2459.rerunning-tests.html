PyCharm 2019.1帮助

重新测试

您可以重复测试会话或单个测试，而无需离开“运行”工具窗口的测试运行器选项卡 。使用与初始运行中相同的运行配置再次执行测试。

重新运行测试会话

执行以下操作之一：

  • 按Ctrl + F5或单击在“ 测试运行器”选项卡的工具栏上。

请注意，您可以自动重新运行测试 。

重新进行个人测试

  • 在Test Runner选项卡中，单击测试用例节点或测试，然后在其上下文菜单中选择Run <test target> 。

重新尝试失败的测试

  • 在Test Runner选项卡中，单击在工具栏上。

重新运行测试自动

如果源代码已更改，PyCharm允许您自动重新运行测试的运行/调试配置。要启用类似 自动测试的 跑步者设施，请确保切换自动测试按钮在Test Runner选项卡的Run工具栏中按下。

  • 按切换自动测试按钮在“ 测试运行器”选项卡的“运行”工具栏上。

  • 对于Mocha和Jest，分别在Run / Debug Configuration：Mocha或Run / Debug Configuration：Jest对话框的Extra Mocha选项 / Extra Jest选项字段中添加--watch标志。

调试失败的测试

  1. 在“ 测试运行器”选项卡中 ，按Shift并单击“ 重新运行失败的测试” 。

  2. 从Restart Failed Tests弹出窗口中选择Debug 。

已失败的测试将在调试模式下重新运行。
