在此页面上，配置要在代码中的EJS包含内应用的格式选项。WebStorm根据“ 设置”对话框的EJS页面上指定的这些分隔符识别此类内容。默认分隔符为<%和%> 。有关使用EJS的更多详细信息，请参阅http://embeddedjs.com/getting_started.html 。

描述

执行标记内的空格

选中此复选框可在开头（ <% ）和结束（ %> ）分隔符及其中的模板主体之间插入空格。否则，模板主体不与分隔符分离。

执行和获取转义标记内的空格

选中此复选框，一方面在模板主体和另一方面的开始/结束分隔符或转义标记 （ = ）之间插入空格。否则，模板主体紧跟在分隔符和标记之后。

执行中的空格并获取未转义的标签

选中此复选框，一方面在模板主体和另一方面打开/关闭分隔符或unescape标签 （ == ）之间插入空格。否则，模板主体紧跟在分隔符和标记之后。

生成标记内的空格

选中此复选框以在模板主体和开/关分隔符之间插入空格或在另一方面生成标记 （ % ）。否则，模板主体紧跟在分隔符和标记之后。

过滤器标签内的空格

选中此复选框，一方面在模板主体和另一方面的开始/结束分隔符或过滤器标签 （ %=: :)之间插入空格。否则，模板主体紧跟在标签之后。

在'：'之前

选中该复选框，有一个冒号前插入一个空格（ : ）。

在'：'之后

选中该复选框，有一个冒号后插入一个空格（ : ）。

连锁'|'

选中此复选框可在垂直条（ | ）之前和之后插入空格。
