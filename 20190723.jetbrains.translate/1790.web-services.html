介绍

IntelliJ IDEA支持Web服务的高效开发，打包和部署。

支持的标准是：

  • 用于XML Web服务的Java API（JAX-WS）

  • Apache Axis

  • JAX-RS

您还可以启用并手动配置以下其他WS引擎的支持 ：

  • Apache Axis2

  • XFire 1。X / CXF （支持2.7版）

  • 将JBossWS
  • WebSphere 6。X

IntelliJ IDEA提供以下开发和包装设施：

  • 根据您指定的专用Web服务构面设置相关模块结构并下载所有必需资源。

  • “即时”生成必要的部署描述符，映射和清单文件。

  • 使用EAR文件结构打包Web服务文件，遵循Enterprise Web Services 1.1规范定义的规则。

Web服务开发功能是通过Web Services捆绑插件提供的 ，默认情况下启用该插件 。如果没有，请使用“设置/首选项”对话框的“ 插件”页面启用它。

开发Web服务

要开发Web服务，请遵循以下常规步骤：

  1. 在Java模块中， 启用对相关Web服务的支持 。

  2. 使用必要的类和方法填充模块。

  3. 编译已开发的类并将它们作为Web服务公开 。

    要在部署Web服务本身之前启用Web服务的客户端开发，请生成WSDL文档 。

  4. 配置要部署的工件 。

    不要忘记在工件中包含Web和Web服务构面资源。

  5. 创建运行配置 。在“ 部署”选项卡上，创建要部署的工件列表。为每个应用程序指定应用程序上下文。

  6. 运行该应用程序。

  7. 在“ 运行”工具窗口的“ 部署控制台”中查看和管理已部署的Web服务 。
