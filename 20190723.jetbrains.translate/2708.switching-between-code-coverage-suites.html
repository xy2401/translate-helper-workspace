管理代码覆盖范围套件

PyCharm提供了一个工具来选择覆盖套件来显示或隐藏，添加和删除套件。

访问选择覆盖套件

要打开“选择要显示的覆盖范围套件”对话框，请执行以下操作之一

  • 在主菜单上，选择“ 工具”|显示覆盖率数据 。

  • 按Ctrl + Alt + F6 。

选择要显示的coverage套件

选择要显示的覆盖范围套件

  1. 如上所述，打开“ 选择要显示的覆盖范围套件”对话框。

  2. 在“ 选择要显示的覆盖范围套件”对话框中，选中所需套件旁边的复选框。

  3. 单击显示所选 。对话框关闭。

  4. 在编辑器中打开已选择覆盖套件的类，并浏览覆盖结果。

隐藏覆盖套房

隐藏覆盖范围套件

  1. 如上所述，打开“ 选择要显示的覆盖范围套件”对话框。

  2. 在“ 选择要显示的覆盖范围套件”对话框中，选中要隐藏的套件旁边的复选框。

  3. 单击无覆盖 。对话框关闭。未显示目标类的覆盖结果。

从磁盘添加覆盖套件

添加或删除coverage套件

考虑从构建服务器获取包含代码覆盖率信息的文件的情况。您可以从磁盘加载此文件并使其可供查看。此外，您可以带一段时间前制作的报道套件进行检查。

另一方面，PyCharm允许您删除不必要的覆盖套件。

  1. 如上所述，打开“ 选择要显示的覆盖范围套件”对话框。

  2. 添加新套件或删除现有套件：

    • 点击 ，并在文件选择器对话框中选择所需的*.es文件。

    • 在列表中选择一个或多个套件，然后单击 。所选套件将从列表和存储中删除。
