将版权配置文件与范围相关联

将配置文件与范围相关联

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” 。打开的“ 版权”页面显示可用版权配置文件和范围之间的映射。

    请注意，除非您至少拥有一个版权配置文件和范围 ，否则将禁用Scope-profile组合区域的控件。

  2. 点击 （ Alt + Insert ）。创建用于定义新映射的行。

  3. 从“ 范围”下拉列表中，选择所需的范围。

    要定义新范围，请打开“ 范围”页面或单击“ 版权”页面下部的链接。

  4. 从“ 版权”下拉列表中，选择要应用于所选范围的配置文件。

  5. 继续编写范围 - 配置文件映射列表。使用 （ Alt + Insert ）或 （ Alt + Delete ）添加或删除项目。要重新排序映射，请使用 （ Alt + Up ）和 （ Alt + Down ）。

设置项目的默认版权配置文件

默认配置文件的设置将应用于任何未包含在任何范围内的项目文件，或属于没有关联版权配置文件的范围。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 版权 ” 。版权页面打开。

  2. 从默认项目版权下拉列表中选择所需的版权配置文件。

    请注意，您选择无版权 ，任何生成版权声明的尝试都将失败，WebStorm将提示您创建要应用的配置文件。
