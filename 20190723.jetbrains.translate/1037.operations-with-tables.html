基本操作

创建一个表

  1. 在“ 数据库”工具窗口中，选择一个数据源并导航到“ 文件”|新的|表 。

  2. 在“ 创建新表”对话框中，指定表设置（列，键，索引，外键）。

  3. 单击确定 。

删除表格

要删除表，请右键单击该表并选择DROP （ N / A ）。您可以为SQL预览字段中生成的代码设置其他选项：

  • 使用模式名称限定对象 ：将模式名称添加到表名称。如果在不同的方案中有两个或多个具有相同名称的表，则可以限定表。此选项具有以下参数：

    • 自动 ：如果在不同方案中有两个以上相同的表名，则自动限定表名。

    • 从不 ：永远不会限制表名。

    • 始终 ：始终限定表名。

  • 使用IF EXISTS语法 ：确保表存在。

  • 使用DROP CASCADE语法 ：删除依赖于表的对象（例如视图）。

查看表格数据

  1. 打开数据库工具窗口（ 查看|工具Windows |数据库 ）。

  2. 在数据库树中，导航到要打开的表。

  3. 双击该表（ F4 ）。

启用表的只读模式

要保护表不被数据编辑器中的意外修改，您可以将表设置为只读。要启用只读模式，请单击“ 单击以切换只读属性”图标（ ）在编辑器的右下角。

要关闭只读模式，请单击“ 单击”以再次切换只读属性图标。

使用数据库控制台或数据库工具窗口时，仍可以修改数据编辑器中具有只读状态的表。

使用查询过滤表数据

您可以通过在“ 行筛选”字段中编写查询来筛选表中的数据。

  1. 如果行筛选器不可用，请单击设置图标（ ）并选择行过滤器 。

  2. 在“ 行筛选”字段中，键入查询。查询语法与WHERE子句中的查询语法相同，但没有WHERE关键字。例如， first_name LIKE 'Joh%' AND last_name LIKE 'lol%' 。

    您可以在LIKE表达式中使用SQL通配符。例如，单个字符的零个或多个字符的百分号（ % ）和下划线（ _ ）。

  3. 按Enter键 。要重置过滤器，请单击清除图标（ ），或删除行筛选字段的内容，然后按Enter键 。

使用快速过滤选项

您可以手动指定过滤条件或使用快速过滤选项。快速筛选选项是当前列名称的筛选条件。条件取决于当前单元格中的值。

  1. 右键单击一个单元格，然后导航到“ 筛选依据” 。

  2. 选择要应用的选项。

运行搜索时筛选行

您可以在表格上运行搜索（ Ctrl + F ）时过滤行。

  1. 按Ctrl + F并选择Filter Rows 。

  2. 开始输入您的搜索查询（例如， John ）。

比较表格

您可以将当前表与在数据编辑器中打开或在数据库控制台工具窗口中显示的任何其他表进行比较。

  1. 选择一张桌子。

  2. 单击Compare with图标（ ），并选择另一个表或结果集。

  3. 如果需要，请在比较对话框中更改Tolerance参数的值。Tolerance参数定义两个结果集之间允许的最大差异数。例如，如果要将两行视为相等（如果它们的数据在单个列中不同），请在“ 容差”字段中输入1 。

转置一张桌子

您可以将表数据从行旋转到列，从列旋转到行。在转置视图中，行和列互换。

要转置表格，请单击设置图标（ ）并选择Transpose 。

将表导出到文件

将表数据复制到剪贴板或将其保存在文件中时，数据将转换为可用的输出格式之一。作为输出格式，您可以选择SQL INSERT或UPDATE语句， TSV或CSV ，HTML表或JSON数据。

打开表格或结果集，单击“ 转储数据”图标（ ）并选择To File 。

将表导出到剪贴板

在工具栏上，单击“ 转储数据”图标（ ）并选择To Clipboard 。

配置数据输出格式和选项

您可以使用预定义的输出格式，自定义格式和脚本配置如何将提取的数据转换为文本。要为提取的数据配置输出，请右键单击表中的任何单元格，然后指向“ 数据提取器” 。

在此菜单中，您可以选择输出格式（例如，作为SQL INSERT语句，HTML表格或CSV格式）或将数据转换为特定格式的脚本。还提供以下附加选项：

跳过计算列（SQL）
允许排除具有计算值的列（影响SQL输出格式）。
跳过生成的列（SQL）
允许排除具有自动增量值的列（影响SQL输出格式）。
添加表定义（SQL）
允许包含CREATE TABLE语句（影响SQL输出格式）。

此外，您还可以配置以下参数：

  • 配置CSV格式。 打开CSV格式对话框，您可以使用该对话框管理分隔符分隔值格式（例如，CSV或TSV）。

  • 转到脚本目录。 打开一个目录，其中包含将表数据转换为各种输出格式的脚本。

将数据导出到数据库

  1. 确保在数据库工具窗口中将数据库添加为数据源 。

  2. 右键单击表中的任何单元格，然后单击“ 导出到数据库” 。

  3. 指定数据库，目标模式（以使用导出的数据创建新表）或表（将导出的数据添加到现有表）。

  4. 配置目标表的数据映射和设置。

在表之间复制和粘贴单元格

您可以复制（ Ctrl + C ）并粘贴（ Ctrl + V ）所选单元格和单元格范围 - 在同一个表格中或从一个表格复制到另一个表格。粘贴时，IntelliJ IDEA会根据需要自动转换数据类型。

将表复制到另一个模式

您可以在当前数据源的架构中创建表的副本，也可以选择其他数据源并在此处创建副本。

  1. 右键单击表并选择“ 将表复制到” 。或者，按F5 。

  2. 输入架构名称，然后单击“ 确定” 。

  3. （可选）在“ 导入”对话框中，修改表设置。

  4. 按确定 。

更新表视图

如果要将编辑器中显示的数据与数据库的内容同步，则需要更新表视图。或者，如果要在更改后应用页面大小限制设置。

要刷新表视图，请考虑以下操作：

  • 单击“ 同步”图标（ ）在工具栏上。

  • 右键单击该表，然后从上下文菜单中选择“ 重新加载页面 ”。

  • 按Ctrl + F5 。

重置初始表视图

单击设置图标（ ）并选择重置视图 。结果，数据变为未排序，列按最初定义的顺序显示，并显示所有列。

查看生成表的查询

若要查看用于生成表的查询 ，请单击工具栏上的“ 查看查询 ”。如有必要，您可以选择查询文本并将其复制到剪贴板（ Ctrl + C ）。

要关闭显示查询的窗格，请按Escape 。
