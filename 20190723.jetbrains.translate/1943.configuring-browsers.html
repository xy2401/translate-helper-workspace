PyCharm 2019.1帮助

配置浏览器

PyCharm附带了一个预定义的最常用浏览器列表，您可以在运行，调试或预览HTML文件的输出期间自动安装和启动这些浏览器。PyCharm假定您根据标准过程安装浏览器，并为每个安装分配一个别名 ，该别名代表浏览器的可执行文件或macOS应用程序的默认路径。除了默认列表中的浏览器之外，您还可以配置自定义浏览器安装。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 工具”下的“ Web浏览器 ”。打开的Web浏览器页面显示预定义列表中的浏览器以及先前配置的自定义浏览器安装（如果有）。对于每个浏览器，PyCharm显示其名称，它所属的系列，以及浏览器的可执行文件/ macOS应用程序的路径或代表此路径的预定义别名 。

  2. 要激活浏览器，请选中其名称旁边的复选框。浏览器将添加到“浏览时打开”菜单项的上下文菜单中，其图标将显示在“ 浏览器”弹出窗口中。

    如果根据标准安装过程安装了浏览器，则“ 路径”字段中的别名很可能指向正确的位置。如果没有，请单击并在打开的对话框中选择实际路径。

选择View |时在浏览器中打开 ，或按Alt + F2 ，或调用浏览器图标栏，可用浏览器按Web浏览器页面上列出的顺序显示。要更改列表中浏览器的顺序，请使用和纽扣。

配置自定义浏览器

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 工具”下的“ Web浏览器 ”。将打开“ Web浏览器”页面。

  2. 点击在工具栏和新行中指定浏览器名称，系列以及其可执行文件或macOS应用程序的路径。

要使用自定义配置文件

对于Chrome系列的Firefox和浏览器，您可以使用自定义配置文件。

  • 在Web浏览器页面上，选择有问题的浏览器并单击在工具栏上。根据所选浏览器的系列，将打开“ Firefox设置”或“ Chrome设置”对话框。

    • 对于Firefox ，请指定所需profiles.ini文件的路径，然后从下拉列表中选择要使用的profiles.ini文件。在Firefox浏览器配置文件中了解更多

    • 对于Chrome浏览器 ，选择使用自定义用户数据目录复选框并指定位置的用户数据目录中PyCharm设置。

    • 要使用其他选项启动Chrome系列的浏览器，请单击在工具栏上，在打开的“ Chrome设置”对话框的“ 命令行选项”文本框中键入所需的键。通过在Chrome中打开chrome://flags ，详细了解Chrome命令行选项。

从列表中删除浏览器

  • 选择浏览器并单击在工具栏上。请注意，只能删除自定义浏览器。

选择默认的PyCharm浏览器

如果要通过选择“ 视图”在浏览器中预览应用程序输出在主菜单上的浏览器中打开或在文件的上下文菜单中在浏览器中打开，您需要选择浏览器以打开预览。您可以从上下文菜单中使用特定浏览器或选择“ 默认浏览器” 。

PyCharm还打开PyCharm默认浏览器以呈现外部资源。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 工具”下的“ Web浏览器 ”。将打开“ Web浏览器”页面。

  2. 从“ 默认浏览器”下拉列表中，选择默认情况下用于预览页面的浏览器。

    • 要使用默认操作系统浏览器，请选择“ 系统默认” 。

    • 要在列表顶部使用浏览器，请选择“ 首先列出” 。使用更改订单或浏览器和工具栏上的图标。

    • 要将其他浏览器用作默认浏览器，请选择“ 自定义路径”并指定所需浏览器的可执行文件的位置。手动键入路径或使用“ 浏览” ，如有必要。

  3. 指定启动浏览器的方式。

    • 要在HTML文件中显示启用了浏览器的弹出窗口，请选中编辑器中的显示浏览器弹出窗口复选框。

    • 如果清除编辑器中的 “ 显示浏览器”弹出窗口复选框，则只能通过“ 视图”|来预览HTML文件在主菜单上的“ 浏览器”命令中打开，或在文件的上下文菜单中的“ 在浏览器中打开”命令中打开 。
