重构|提取变量
Ctrl + Alt + V

使用此对话框提取：

  • Python或JavaScript上下文中的变量。

  • JavaScript 1.7，JavaScript 1.8.5，ES5和ES6的局部变量。

  • JavaScript常量。

变量声明

选择提取（全局）变量，常量或局部变量：

  • extract var - 选择此选项以提取全局变量。

  • extract常量 - 选择此选项以提取常量。

  • 提取局部变量 - 选择此选项以提取局部变量。

    提取局部变量选项仅适用于JavaScript 1.7，JavaScript 1.8.5，ES5和ES6。
