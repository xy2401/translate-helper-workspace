方法调用

$ Instance $ 。$ MethodCall $ （ $ Parameter $ ）

此模板与方法调用表达式匹配。如果出现次数为零，则意味着可以省略方法调用。

@Deprecated $ Instance $ 。$ MethodCall $ （ $ Parameter $ ）

您可以使用此模板查找已弃用的方法，并将其用作创建其他带注释的方法模板的原型。特别是对于搜索方法调用已弃用的方法，您可以选择已经存在的模板方法调用已弃用的方法 。

有关如何创建搜索模板，请参阅结构搜索和替换 。

搜索JavaScript和Typescript类

如果你有一个JavaScript或Typescript类MyClass ：

class MyClass {}

然后，搜索它的最简单的模板是class $a$ 。

搜索XML和HTML标记，属性及其值

搜索标记的最简单模板是<$a$/>

通过在变量$a$上设置约束，您可以指定要查找的变量。例如，如果指定text / regexp约束app.+ ，则会找到名称以app开头的标记。

用于搜索XML和HTML的更通用的模板是<$tag$ $attribute$="$value$"/>通过使用具有正确指定的搜索设置和约束的模板，您几乎可以找到XML或HTML中可能出现的任何内容。例如，如果为变量$attribute$指定text / regexp约束width ，则会找到具有width属性的所有标记。
