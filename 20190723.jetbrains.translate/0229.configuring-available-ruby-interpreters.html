配置可用的Ruby解释器

此功能仅在Ultimate版本中受支持。

以下内容仅在安装并启用Ruby插件时有效！

在这个部分：

  • 概观
  • 查看可用的解释器列表
  • 配置可用解释器列表
  • 从列表中删除解释器

概观

可用于各种项目的Ruby SDK列表可以包括本地或远程安装的解释器。

可用的Ruby解释器被定义为全局SDK。下面描述的过程假设您的计算机上已经安装了必要的Ruby解释器。

查看可用的解释器列表

查看可用的口译员列表

  1. 执行以下操作之一：

    • 在主菜单上，选择“ 文件”|项目结构

    • 在主工具栏上，单击

    • 按Ctrl + Shift + Alt + S.

  2. 在“ 平台设置”下 ，单击节点SDK ：

配置可用解释器列表

配置可用的Ruby解释器

  1. 打开“ 项目结构”对话框（例如Ctrl + Shift + Alt + S ）。

  2. 在“ 项目结构”对话框中，单击“平台设置”下的“ SDKs”节点。

  3. 点击 ，然后从Add New SDK弹出菜单中选择Ruby SDK 。

  4. 从“ 选择解释器路径”弹出窗口中，选择是要从本地计算机添加解释器，还是从远程位置添加解释器。

    下一步取决于所选的解释器位置。添加解释器时，它将包含在可用解释器列表中。

  5. 重复步骤1 - 4，将更多Ruby解释器添加到列表中。

从列表中删除解释器

从可用解释器列表中删除解释器
