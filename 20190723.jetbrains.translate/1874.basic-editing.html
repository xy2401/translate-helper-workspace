捷径

使用此快捷方式...

输入

输入

根据具体情况：

  • 在查找列表中：选择一个项目。

  • 在编辑器中：输入一个新行并在其开头设置插入符号。

  • 在编辑器中：输入一个新行并在其开头设置插入符号。

    按Enter键时 ，PyCharm会添加反斜杠字符以避免语法错误。

标签

标签

在编辑器中：

  • 选择：缩进选定的行。

  • 没有选择：插入制表符号（或相应的空格字符数）。

在查找列表中：

  • 插入符后没有代码：选择一个项目。

  • 插入符之后的一些代码：选择一个项目并用插入符号替换插入符号之后的代码。

删除

删除

根据具体情况：

  • 在编辑器中：删除所选符号/块。

  • 在使用情况视图中：排除所选项目。

  • 在其他视图中：删除所选项目。

退格

退格

删除插入符号左侧的字符。

解开

按Ctrl + Z

撤消上次操作。

重做

按Ctrl + Shift + Z

重做最后一次撤消操作。

切

按Ctrl + X

将当前行或选定的代码块剪切到剪贴板。

复制

按Ctrl + C

将当前行或选定的代码块复制到剪贴板。

糊

按Ctrl + V

从剪贴板粘贴到插入符号位置。

粘贴历史记录

CTRL + SHIFT + V

将选定条目从剪贴板粘贴到插入符号位置。

向上

向上

将插入符号向上移动一行。

选择

Shift +向上

将插入符号移动一行选择文本。

下

下

将插入符号向下移动一行。

选择下来

Shift +向下

将插入符号向下移动一行选择文本。

剩下

剩下

将插入符号向左移动一个字符。

留下选择

Shift +向左

将插入符号向左移动一个字符，选择文本。

对

对

将插入符号向右移动一个字符。

选择权

Shift +右键

将插入符号向右移动一个字符，选择文本。

转到页面底部

Ctrl + Page Down

将插入符号向下移动到页面底部。

使用选择转到页面底部

Ctrl + Shift + Page Down

将插入符号向下移动到页面底部，选择文本。

转到页面顶部

Ctrl + Page Up

将插入符号移动到页面顶部。

使用选择转到页面顶部

Ctrl + Shift + Page Up

将插入符号向上移动到页面底部，选择文本。

向下翻页

向下翻页

将插入符号向下移动一页。

选择页面向下

Shift + Page Down

将插入符号向下移动一页，选择文本。

Page Up

Page Up

将插入符号向上移动一页。

选择页面向上

Shift + Page Up

将插入符号向上移动一页，选择文本。

向下滚动

按Ctrl +向下

向下滚动文本一行。

滚动到中心

按Ctrl + M

在插入符号处滚动一行到屏幕中心。

向上滑动

Ctrl +向上

将文本向上滚动一行。

移至行尾

结束

将插入符号移到行尾。

使用选择移至行尾

SHIFT + END

将插入符号移动到行尾，选择文本。

移至Line Start

家

将插入符号移动到行首。

使用选择移至行开始

SHIFT + HOME

将插入符号移动到行首，选择文本。

转到下一个单词

按Ctrl +右

将插入符号移动到下一个单词。

使用选择移至下一个单词

按Ctrl + Shift +向右

将插入符号移动到下一个单词，然后选择它。

移至上一个单词

Ctrl +向左

将插入符号移到上一个单词。

使用选择移至上一个单词

按Ctrl + Shift +左

将插入符号移动到上一个单词，然后选择它。

移至文本结束

CTRL + END

将插入符号移动到文本末尾。

使用选择移至文本结束

按Ctrl + Shift + End

将插入符号移动到文本末尾，然后选择它。

移至文本开始

Ctrl + Home

将插入符号移动到文本的开头。

移至文本以选择开始。

按Ctrl + Shift +首页

将插入符号移动到文本的开头，然后选择它。

全选

按Ctrl + A

选择在编辑器中打开的整个文本。

在Caret删除Line

按Ctrl + Y

删除插入符号当前所在的行。

删除到Word End

按Ctrl +删除

删除从当前插入符号位置开始直到单词结尾的单词。

删除到Word开始

按Ctrl + Backspace键

删除从当前插入符号位置开始直到单词start的单词。

切换插入/覆盖

插入

切换插入/覆盖模式。

重复的行或块

按Ctrl + d

复制选定的块或插入符号处的行。

切换案例

按Ctrl + Shift + U

切换所选文本块的大小写。

移至代码块结束

按Ctrl +]

将插入符号移动到当前代码块末尾，突出显示块限制。

使用选择移至代码块结束

CTRL + SHIFT +]

将插入符号移动到当前代码块末尾，从初始插入符号位置开始选择代码。

移至代码块启动

按Ctrl + [

将插入符号移动到当前代码块开始，突出显示块限制。

移至代码块以选择开始

Ctrl + Shift + [

将插入符号移动到当前代码块开始，从初始插入符号位置开始选择代码。

开始新线

Shift + Enter键

在当前根据当前缩进级别定位插入符之后开始新行。

在当前之前开始新线

按Ctrl + Alt + Enter键

在当前行之前开始一个新行。

加入行

CTRL + SHIFT + J

将选定的行连接成一个或将插入符号当前所在的行与下一行连接。

分裂线

Ctrl + Enter

在插入符所在的位置拆分选定的行，将插入符号留在第一行的末尾。

此快捷方式分割行而不添加反斜杠。

在Caret选择Word

按Ctrl + W

从当前插入符号位置开始选择连续增加的代码块。

在Caret取消选择Word

按Ctrl + Shift + W

按顺序删除操作所做的选择。

缩进选择

标签

将所选块移动到下一个缩进级别。

Unindent Selection

Shift + Tab键

将所选块移动到上一个缩进级别。

自动缩进行

按Ctrl + Alt + I

根据“ 代码样式”设置缩进当前行或选定的块。
