PyCharm 2019.1帮助

PyCharm用于存储设置，缓存，插件和日志的目录

IDE文件的位置取决于操作系统和PyCharm版本。

视窗

默认情况下，所有文件都位于此目录下：

Windows Vista，7,8,10

<SYSTEM DRIVE> \ Users \ <USER ACCOUNT NAME> \。<PRODUCT> <VERSION>

Windows XP

<SYSTEM DRIVE> \ Documents and Settings \ <USER ACCOUNT NAME> \。<PRODUCT> <VERSION>

例

  • PyCharm 2019.1

    C：\用户\约翰\。PyCharm2019.1 \

在此目录下，您将找到以下子目录：

  • config ：configuration（idea.config.path）

  • config \ plugins ：plugins（idea.plugins.path）

  • system ：缓存，本地历史（idea.system.path）

  • system \ log ：日志和线程转储（idea.log.path）

请参阅调整PyCharm页面。

Linux和其他UNIX系统

以点开头的产品目录可以在用户主目录中找到。模式是：

〜/ <PRODUCT> <VERSION>

〜是主目录的别名，例如/home/john 。

苹果系统

  • 组态

    〜/库/首/ <PRODUCT> <VERSION>

  • 高速缓存

    〜/库/缓存/ <PRODUCT> <VERSION>

  • 插件

    〜/ Library / Application Support / <PRODUCT> <VERSION>

  • 日志

    〜/库/日志/ <PRODUCT> <VERSION>

其中<PRODUCT>是PyCharm。
