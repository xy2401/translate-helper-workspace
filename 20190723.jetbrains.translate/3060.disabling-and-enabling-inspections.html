禁用和禁止检查

某些检查可能会报告您当前不希望看到的问题。在这种情况下，您可以禁用或禁止它们。

禁用检查时，请将其关闭。这意味着代码分析引擎停止搜索项目文件以查找此检查旨在检测的问题。请注意，禁用检查时，请在当前检查配置文件中将其禁用;它在其他配置文件中保持启用

禁用检查

禁用设置中的检查

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，选择“ 编辑器”|检查 。

  2. 找到要禁用的检查，然后清除它旁边的复选框。

  3. 应用更改并关闭对话框。

在编辑器中禁用检查

您可以直接在编辑器中快速禁用触发检查。

  1. 将插入符号放在突出显示的字符串上，然后按Alt + Enter （或使用意图操作） ）。

  2. 点击在要禁用的检查旁边，然后选择“ 禁用检查” 。

在“结果”工具窗口中禁用检查

  1. 在“ 检查结果”工具窗口中（运行代码分析后），右键单击要禁用的检查，然后选择“ 禁用检查” 。

  2. 点击隐藏已禁用的检查警报。

禁止检查

当您取消检查时，代码分析引擎不会突出显示此检查在特定代码段（类，方法或语句）中发现的问题。您还可以取消当前班级中的所有检查。

在编辑器中禁止检查

  1. 将插入符号放在突出显示的字符串上，然后按Alt + Enter （或使用意图操作） ）。

  2. 点击在您要抑制的检查旁边，并选择必要的抑制操作。

在“结果”工具窗口中禁止检查

  • 在“ 检查结果”工具窗口中（运行代码分析后），右键单击要忽略的检测到的问题，然后从上下文菜单中选择“ 抑制语句 ”。

  • 要禁止在整个文件中进行检查，请右键单击该文件，然后从其上下文菜单中选择Suppress for statement 。

    或者，选择文件，然后从右侧窗格的列表中选择Suppress for statement 。

更改文件的突出显示级别

默认情况下，WebStorm会突出显示所有检测到的代码问题。您可以更改当前使用的文件的此行为。点击在状态栏（右下角）中，选择以下选项之一：

  • 无 - 关闭突出显示。

  • 语法 - 仅突出显示语法问题。

  • 检查 - （默认）突出显示检查发现的语法问题和问题。

您还可以从主菜单更改突出显示级别。选择，或按Ctrl + Shift + Alt + H.
