添加框架和技术

为了开发特定于框架的应用程序，IntelliJ IDEA具有各个方面。构面包含库，依赖项和技术，它们为您提供了用于配置特定于框架的设置的其他UI元素。

IntelliJ IDEA可以识别特定框架的典型文件或目录，并为您添加必要的方面。一旦检测到并添加了构面，IntelliJ IDEA将通知您缺少的配置并将建议必要的操作。

如果未自动检测到构面，则可以手动添加构面。您可以向模块添加多个构面。

手动将构面添加到模块

  1. 在“项目”工具窗口中，右键单击要添加构面的模块，然后选择“ 添加框架支持” 。

  2. 从列表中选择必要的框架。根据您的选择，系统可能会提示您配置其他设置（例如，配置库）。

  3. 应用更改并关闭对话框。

禁用框架自动检测

默认情况下，为所有受支持的框架启用自动检测。您可以完全禁用框架自动检测，也可以从自动检测中排除单个框架。

  1. 点击在工具栏上（ Ctrl + Shift + Alt + S ）并选择Facets 。

  2. 选择检测并单击 （ Alt + Insert ）。

  3. 从Framework to Exclude列表中，选择必要的选项。您可以仅在一个目录或整个项目中禁用特定框架的自动检测。该列表还允许您禁用特定目录中所有框架的自动检测。

  4. 如果要禁用整个项目中所有框架的自动检测，请取消选中“ 启用框架检测”复选框。

在此对话框中，您还可以删除框架或添加新框架。
