Documentation Tool Window

View | Tool Windows | Documentation
Ctrl+Q

This tool window appears in the following cases:

  • Quick Documentation lookup is pinned by clicking in the upper-right side of the lookup window.

  • Quick Definition tooltip is pinned by clicking in the upper-right corner of the tooltip.

As a result, the tool window appears in the list of available tool windows in the View menu.

Icon

Shortcut

Description

 

Left or Right

Switch to the previous or next documentation page (e.g. after using hyperlinks).

On a macOS computer, you can also use the three-finger right-to-left and left-to-right swipe gestures.

F4

Switch to the item (e.g. source) that corresponds to the documentation page currently shown.

Click this icon to perform the following settings:

  • Show the font size slider. Move the slider to increase or decrease the font size in the quick documentation window as required.

  • Enable or disable auto-update of documentation from its source.

  • Control the window's position.
