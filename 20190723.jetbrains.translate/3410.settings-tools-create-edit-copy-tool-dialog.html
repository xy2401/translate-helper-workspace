编辑外部工具的设置。

项目

描述

名称

在“ 工具”菜单和上下文菜单中显示为命令名称的工具的名称。

组

该工具所属的组。工具组对应于“ 工具”菜单和上下文菜单中的子菜单。

从列表中选择现有组。

要创建新组，只需在列表中键入其名称即可。

描述

工具说明（可选）。

工具设置

程序

要运行的可执行文件的路径。使用要选择文件，请插入宏以打开“ 宏”对话框以选择宏。

（宏在运行时解析，并允许您指定上下文信息，如当前选定的文件，项目源路径等）

参数

要在命令行中指定它们的方式传递给程序的参数。使用“ 插入宏”打开“ 宏”对话框以选择宏。

指定参数时，请遵循以下规则：

  • 请注意 ，外部工具提供文件的绝对路径，而不是项目目录相关的路径。

  • 使用空格分隔各个参数。

  • 如果参数包含空格，请将空格或包含空格的参数括在双引号中，例如， some" "arg或"some arg" 。

  • 如果参数包含双引号（例如，作为参数的一部分），则通过反斜杠转义双引号，例如-Dmy.prop=\"quoted_value\" 。

工作目录

程序当前工作目录的路径。使用选择目录， 插入宏以打开宏对话框以选择宏。

高级选项

执行后同步文件

使WebStorm在工具完成执行时了解文件系统中的更改。

打开工具输出控制台

打开控制台以查看工具输出等错误消息等。

使控制台在stdout中的消息处于活动状态

使输出控制台处于活动状态，并在发生相应事件时将其转发。

在stderr中使控制台处于活动状态

与前一个选项相同，但对于stderr。

输出滤波器

输出过滤器用于将工具输出中的绝对文件路径和行号转换为超链接。您将能够使用这些链接在编辑器中打开相应的文件。您可以指定一个或多个过滤器。每一行都是正则表达式。您可以使用以下宏： $FILE_PATH$ ， $LINE$和$COLUMN$ ，例如，

$ FILE_PATH $ ： $ LINE $：$ COLUMN $：。*
