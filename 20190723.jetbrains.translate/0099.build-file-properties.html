构建文件属性

Ant构建工具窗口|
Ant构建工具窗口|构建文件的上下文菜单|属性

使用此对话框配置Ant构建脚本的行为。

项目

描述

最大堆大小

使用此字段可更改分配给Ant构建过程的内存量。如果Ant进程内存不足，请增加此值。

最大堆栈大小（MB）

使用此字段可更改堆栈内存大小。默认值为2 MB。

在后台构建

如果启用此选项，则构建过程将在后台运行。否则，将显示模态进度对话框。

编译需要大量的处理器资源，并且可能导致性能严重下降。当构建过程在后台运行时，您可以找到IntelliJ IDEA无法响应您的操作。

如果没有错误，请关闭消息视图

如果启用此选项，则不会为成功构建打开Ant Messages窗口。

属性选项卡

使用此选项卡指定应传递给构建脚本的运行时属性。这些属性等同于命令行的-D选项后定义的属性。属性被定义为一对“名称 - 值”。该值可以是硬编码的，或者在运行时从宏中动态评估。

加

使用此按钮可在运行时属性列表中创建新条目。

去掉

使用此按钮可从列表中删除所选属性。

执行选项卡

使用此选项卡可配置IntelliJ IDEA如何启动构建脚本。

用Ant运行

在此选项组中，您可以确定要运行所选build.xml的Ant版本。

使用项目默认Ant

单击此单选按钮以运行默认的Ant版本。

使用自定义Ant

单击此单选按钮以运行您选择的Ant版本。在组合框中，您可以选择已使用IntelliJ IDEA配置和注册的任何Ant版本。使用省略号按钮修改Ant配置。

默认设置

单击此按钮可配置默认Ant。

Ant命令行

使用此字段提供命令行参数。除了-logger之外，您可以包含任何参数。参数应以破折号开头，并用空格分隔。

在JDK下运行

使用此字段定义用于运行Ant目标的JDK。默认情况下，您可以选择项目或模块JDK。使用省略号按钮配置JDK。

其他Classpath选项卡

使用此选项卡可添加Ant在运行时加载的库和目录。

加

单击此按钮可将归档目录添加到Classpath。

添加所有目录

单击此按钮可将包含其所有内容的目录添加到Classpath。

上移/下移

使用这些按钮更改类路径条目的顺序。对话框中条目的顺序定义了Ant加载资源的顺序。

去掉

单击此按钮可从列表中删除所选条目。

过滤器标签

使用此选项卡可以在应用过滤时配置Ant构建工具窗口中显示的构建目标。该选项卡显示所选构建文件的构建目标列表。左列显示构建目标的名称，右列包含目标的可选描述。如果选中了目标，则在按下后将显示在Ant构建工具窗口中 ;否则这样的目标将被隐藏。
