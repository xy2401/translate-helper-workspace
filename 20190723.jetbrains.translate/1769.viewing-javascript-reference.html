IntelliJ IDEA 2019.1帮助

JavaScript文档查找

此功能仅在Ultimate版本中受支持。

要在“ 文档”弹出窗口和“ 文档”工具窗口之间切换，请按顺序按Ctrl + Q.

IntelliJ IDEA在默认的IntelliJ IDEA浏览器中打开MDN文章。

IntelliJ IDEA显示标准JavaScript API，项目及其依赖项中的符号以及外部库中定义的符号的参考。您可以在“ 文档”弹出窗口中查看符号的文档 ，获取有关方法参数的信息，并在外部站点上打开文档（如果有可用的链接）。

查看插入符号的文档

  • 按Ctrl + Q或选择查看|主菜单中的快速文档查找 。符号的引用显示在“ 文档”弹出窗口中。

    • 对于项目符号或项目依赖项中的符号，IntelliJ IDEA从相应的JSDoc注释生成文档。

    • 对于标准JavaScript对象或方法，IntelliJ IDEA显示来自内置TypeScript定义文件 （ d.ts ）的相应JSDoc注释。这些文件与IntelliJ IDEA捆绑在一起并定期更新。

    • 如果在d.ts文件中未找到任何注释，则IntelliJ IDEA会显示相应MDN文章的摘要。

查看插入符号的MDN文档

  • 在文档弹出窗口（ Ctrl + Q ）中，单击底部的链接：

  • 按Shift + F1或选择查看|主菜单中的外部文档 。

查看第三方JavaScript库的文档

  1. 下载所需的库或框架，将其配置为外部JavaScript库，并指定其外部文档的链接，有关详细信息，请参阅配置自定义第三方JavaScript库 。

  2. 将光标定位在相关符号上，然后按Shift + F1或选择View |主菜单上的外部文档 。

查看npm包的文档

  • 在package.json文件或require或import语句中，将光标放在包的名称上，然后按Ctrl + Q.

  • 要在浏览器中打开npm官方网站上的文档，请按Shift + F1 。

在工具窗口中查看参考

  • 固定文档弹出窗口。它变成了文档工具窗口 ，带有相应的侧边栏图标和更多控件。

自动显示文档

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规 ” ，然后单击“代码完成” 。将打开“ 代码完成”页面。

  2. 选中Show the documentation popup复选框并指定已用时间。

有关详细信息，请参见有用的编

要更改快速文档的字体大小，请执行以下操作之一：

  • 点击在快速文档窗口的右上角，然后移动滑块。

  • 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 编辑器”下的“ 常规” ，然后选择“ 使用Ctrl +鼠标滚轮更改字体大小（缩放）”复选框。
