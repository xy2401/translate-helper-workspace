禁用和启用检查

如果您认为某些检查报告了您不感兴趣的问题，则可以禁用此类检查。请注意，禁用检查时，会在当前检查配置文件中禁用检查 ;在所有其他配置文件中，它仍保持启用状

有几种方法可以禁用/启用检查：

  • 使用“检查设置”页面 - 这是管理检查的主界面;在这里，您可以立即看到所有检查配置文件中启用或禁用的检查。

  • 使用意图操作 - 这是在编辑器中禁用突出显示的代码问题的方法。

  • 在“检查结果”工具窗口中 - 这是在分析检查结果时禁用不感兴趣问题的快速方法。请注意，此处您只能禁用检查。

请注意禁用和禁止代码检查之间的区别：

在抑制检查时，PyCharm会插入一条特殊注释，告诉代码分析引擎忽略特定代码段中此检查发现的问题。

禁用检查时，只需将其关闭，以便代码分析引擎忽略此检查发现的代码问题。

禁用或启用检查

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，通过展开分组节点或使用搜索字段查找所需的检查。

  2. 使用检查旁边的复选框禁用或启用它。

  3. 应用更改并关闭对话框。

在编辑器中禁用对突出显示的问题的检查

  • 以这种方式禁用检查时，将禁用当前检查配置文件 。

  • 要重新启用以这种方式禁用的检查，请使用上述主要步骤 。

  1. 将插入符号设置为突出显示的问题。

  2. 单击灯泡图标或按Alt + Enter以显示检查警报和建议列表。

  3. 选择要禁用的检查，然后单击向右箭头按钮或只需按向右箭头键。

  4. 在子菜单上，单击“ 禁用<检验名称>” 。

禁用检查结果报告中的检查

  • 以这种方式禁用检查时，将禁用它们用于运行检查的检查配置文件 。您可以在“ 检查结果”窗口的选项卡的标题中看到它。

  • 要重新启用以这种方式禁用的检查，请使用上述主要步骤 。

  1. 在“ 检查结果”工具窗口中 ，右键单击要禁用的检查。

  2. 从上下文菜单中，选择“ 禁用检查” 。

  3. 按下过滤器按钮隐藏已禁用的检查警报。
