按名称运行检查

如果要跟踪特定问题，运行单个检查非常有用。如果在文件中发现警告，则可以检查整个项目或文件的必要范围，以确保代码库中不再存在此类警告。

您可以使用CamelHumps将驼峰大小写单词和空格与单词的首字母相匹配。

  1. 从主菜单中，选择Code |按名称运行检查 （ Ctrl + Shift + Alt + I ）。

  2. 在弹出窗口中键入检查名称。建议列表将显示符合您的搜索请求的检查。

    如果您不确定是否选择了正确的检查，则可以查看其说明。为此，请在弹出窗口中选择一个检查，然后按Ctrl + Q.

  3. 双击必要的检查以打开其设置。

  4. 在打开的对话框中，选择要分析的文件范围。

  5. File Masks（s）选项可帮助您缩小要检查的文件数。

    选中该复选框，并指定与要分析的文件名称匹配的字符和通配符模式。使用逗号分隔多个文件掩码。

  6. 某些检查可能还有其他选项，系统会提示您进行配置。

    这些设置仅适用于此检查运行。在您当前的配置文件中，此检查的配置不会更改。

IDE将在工具窗口中显示检查结果。在那里，您可以检查并修复检测到的问题
