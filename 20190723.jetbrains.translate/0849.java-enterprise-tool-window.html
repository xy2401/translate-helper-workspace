Java企业工具窗口

要打开此工具窗口：
查看|工具窗口| Java Enterprise
另请参见显示工具窗口 。

Java Enterprise工具窗口允许您从Java EE角度查看项目。

工具窗口包括一组窗格，其中所选项目的“Java EE内容”显示在右侧窗格中。如果合适，将在最右侧窗格中显示项目的Javadoc HTML文档（或者有时是图表）。

工具栏按钮用于显示或隐藏项目类别（例如模块或CDI bean），或用于更改项目的显示方式。

上下文菜单包括用于在窗格（ 上一个或下一个 ）之间切换以及在编辑器中打开项目（ 编辑 ）的命令。

您可以通过指定相应的搜索字符串来过滤大多数窗格中的信息。

  • 工具栏图标
  • 上下文菜单命令
  • 过滤信息

工具栏图标

除了极少数例外，工具栏图标是可以打开或关闭的切换。

前两个图标（ 和 ）始终可用。其余图标仅适用于某些Java EE技术和框架（例如CDI，WebSocket）。

图标

描述

图标的状态定义了Java EE技术和框架的显示级别。

启用时，技术显示在第一级（即最左侧窗格中）。在这种情况下，如果如果已启用，并且模块显示在右侧窗格中，您可以选择技术并查看，例如，使用所选技术的模块。

关闭时，技术显示在第二级。也就是说，您的模块显示在最左侧窗格中，右侧窗格中显示了技术。在这种情况下，您可以选择一个模块，并查看所选模块中使用的技术。

如果此图标用于显示或隐藏模块。在这种情况下，模块（如果显示）显示在左侧的第二个窗格中。

对于CDI：使用此图标显示或隐藏CDI bean。

对于CDI：使用此图标可显示或隐藏生产者方法和字段。

取决于技术：
  • 对于CDI：使用此图标可在最右侧窗格中显示图表或文档之间切换。

    如果选中，则显示所选项目的依赖注入图。否则，将显示Javadoc HTML文档。

  • 对于WebSocket：使用此图标可以更改端点类的显示和排序方式。

    如果禁用，则按以下顺序显示类的信息：首先是类名，然后是端点URL（如果可用）。这种情况下的类按名称排序。

    如果选中，则首先显示端点URL，然后是类名。这种情况下的类由端点URL排序。

  • 对于RESTfull Web服务：使用此图标可以更改资源类及其方法的显示和排序方式。

    如果禁用，则按以下顺序显示类和方法的信息：首先是名称，然后是资源URL。这种情况下的类和方法按名称排序。

    如果选中，则首先显示资源URL，并按URL排序信息。

    请注意，可以通过显示或隐藏资源类 。要显示或隐藏方法，请使用 。

对于CDI：使用此图标可显示或隐藏InjectionPoint类型。

请注意，如果如果打开，注入点将显示在最右侧窗格的图表上，此图标将变为非活动状态。

对于WebSocket：使用此图标显示或隐藏使用@ServerEndpoint注释的类。

对于WebSocket：使用此图标显示或隐藏使用@ClientEndpoint注释的类。

对于RESTful Web服务：使用此图标显示或隐藏资源类。

如果禁用，则会显示一个显示类的窗格。选择类时，其方法将显示在右侧窗格中。因此，在这种情况下，方法按定义它们的类进行分组。

如果选中，则不显示类。属于不同类的方法都显示在同一窗格中并同时显示。

对于RESTful Web服务：单击此图标，然后单击必要选项以显示或隐藏使用@GET ， @GET @POST ， @GET @PUT或@DELETE注释的方法。
