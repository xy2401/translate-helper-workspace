工具窗口标有图标

使用此工具窗口将Ant构建脚本添加到项目中，控制构建行为并运行构建目标。

  • Ant Build
  • Ant构建结果

IntelliJ IDEA使用捆绑插件实现Ant构建工具窗口功能，可以通过清除IntelliJ IDEA设置的插件页面上的Ant支持复选框（ Ctrl + Alt + S ）来完全禁用该插件 。

Ant Build

项目

描述

单击此按钮可将Ant构建文件添加到当前项目。

单击此按钮可从当前项目中删除对所选构建文件的引用。

单击此按钮可运行选定的构建目标。

单击此按钮可仅查看具有description属性的目标。

使用这些按钮可以展开或折叠所有节点。

单击此按钮可显示所选Ant构建文件的属性 。

Ant构建结果

运行Ant目标或整个构建的结果显示在“ 消息”工具窗口中 。
