启用GWT支持

此功能仅在Ultimate版本中受支持。

IntelliJ IDEA支持通过专用的GWT方面使用Google Web Toolkit。在启用GWT支持之前，请确保已在计算机上下载并安装GWT SDK 。

下载并安装GWT SDK后，有几种方法可以启用GWT支持，具体取决于您的上下文：

  • 创建具有专用GWT方面的模块 。

  • 将GWT方面附加到现有模块。

要在每次为模块启用GWT支持时让IntelliJ IDEA选择GWT安装文件夹的路径，请在默认GWT构面设置中指定此路径。

配置GWT的缺省路径

  1. 打开“ 项目结构”对话框（ Ctrl + Shift + Alt + S ）。

  2. 转到Facets ，然后选择GWT 。

  3. 在“ 默认值”选项卡中，指定GWT安装文件夹的路径。单击确定 。
