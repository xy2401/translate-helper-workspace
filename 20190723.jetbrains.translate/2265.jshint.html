JSHint

此功能仅在专业版中受支持。此版本是商业版，它提供了一系列出色的功能：有关详细信息，请参阅比较矩阵 。

您可以使用已与PyCharm捆绑的JSHint linter检查您的代码。

在PyCharm中启用JSHint并配置其行为

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，单击“ 语言和框架”下的“ JavaScript ”，然后单击“ 代码质量工具”下的“ JSHint ” 。在打开的JSHint上，选中“ 启用”复选框。之后，页面上的所有控件都可用。

  2. 从“ 版本”下拉列表中，选择要使用的工具的版本。PyCharm捆绑了版本2.9.5 ，默认情况下使用。要下载其他版本，请从列表中选择它。

  3. 配置JSHint的行为：
    • 要根据配置文件中的规则验证代码，请选中“ 使用配置文件”复选框。

      配置文件是扩展名为.jshintrc的JSON文件，用于指定应启用或禁用哪些.jshintrc选项。PyCharm首先在工作目录中查找.jshintrc文件。如果搜索失败，PyCharm将在父文件夹中搜索，然后再在父文件夹中搜索。重复该过程，直到PyCharm找到.jshintrc或到达项目根目录。要在项目中找不到.jshintrc时仍让PyCharm运行验证，请指定要使用的默认配置文件。从JSHint官方网站了解有关JSHint配置文件的更多信息。

    • 要手动配置验证，请清除该复选框并在“ 选项”区域中启用相关验证。从JSHint官方网站了解更多信息。
