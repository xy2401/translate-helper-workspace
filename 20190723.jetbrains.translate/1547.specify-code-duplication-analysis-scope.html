使用此对话框启动搜索指定范围内的重复代码片段。

项目

描述

整个项目

选择此选项可对整个项目执行分析。

模块<名称>

选择此选项可让IntelliJ IDEA分析当前在“ 项目”工具窗口中选择的模块。

文件<名称>

选择此选项可分析当前在“ 项目”工具窗口中选择的文件或在编辑器中打开的文件。

选定的文件

选择此选项可分析“ 项目”工具窗口中当前选定的文件。

未提交的文件

此范围仅适用于版本控制下的项目。
选择此选项可使IntelliJ IDEA仅分析尚未提交到版本控制系统的文件。使用该列表进一步限制分析范围。可用选项包括：

  • 全部 - 选择此选项可以分析所有更改列表中的文件。

  • 默认 -选择此选项，IntelliJ IDEA的从缺省的变更只分析文件。

自定义范围

选择此选项以使用自定义范围。从列表中选择预定义的范围，或单击并在打开的“ 范围”对话框中定义范围。

使用特殊语言定义范围。

包括测试源

选中此复选框以对测试源执行分析。
