WebStorm 2019.1帮助

无障碍

WebStorm允许您启用各种辅助功能以满足您的需求。您可以使用屏幕阅读器或调整字体大小，颜色以及某些UI元素的行为，以便更轻松地使用WebStorm。

设置屏幕阅读器

目前，我们完全支持Windows上的WebStorm屏幕阅读器。您可以在某种程度上在macOS上使用WebStorm的屏幕阅读器。然而，为了充分利用屏幕阅读器的支持，我们建议使用Windows。

对于macOS ，请打开VoiceOver并安装和设置WebStorm 。

为Windows启用屏幕阅读器

  1. 下载并启用您首选的屏幕阅读器。

    检查以下推荐的屏幕阅读器：

    • NVDA - 使用NVDA 2015或更高版本。如果使用32位版本的NVDA，则必须在计算机上安装32位JRE，因为此版本的NVDA需要C：\ Windows \ SysWOW64 \ WindowsAccessBridge-32。DLL与WebStorm一起使用。如果NVDA找不到此文件，NVDA事件日志窗口将显示相应的消息。

    • JAWS - 下载所需的版本并重新启动计算机以启用JAWS屏幕阅读器。

  2. 确保已为屏幕阅读器安装了Java Access Bridge和正确的Java版本，如下所示：

    • 要启用Java Access Bridge ，请打开命令提示符并键入[JRE_HOME]\bin\jabswitch -enable ，其中[JRE_HOME]是计算机上JRE的目录。对于Java版本1.8，Java Access Bridge是JDK的一部分，您无需单独下载或启用它。

    • 如果您的屏幕阅读器是32位，请安装32位JRE 1.7或更高版本。如果您的屏幕阅读器是64位，请安装64位JRE版本1.7或更高版本。

    您的计算机可能具有Java Access Bridge的某些重要组件的多个版本，并且它们可能在不同版本之间不兼容。您需要验证Java Access Bridge配置是否正确。

    如果您的屏幕阅读器是32位，请确保C：\ Windows \ SysWOW64 \ WindowsAccessBridge-32。DLL存在且版本号为7.xxx或更高。该文件的描述应为“Java（TM）Platform SE 7”。

安装并设置WebStorm

  1. 下载并安装 WebStorm。

  2. 要在首次启动WebStorm之前启用屏幕阅读器支持，请执行以下操作：

    • 打开包含具有个人设置的文件的配置目录，例如键盘映射，配色方案等。

      句法
      ％HOMEPATH％\。<产品> <版本> \配置
      例
      C：\用户\约翰\。WebStorm2019.1 \ CONFIG
      句法
      〜/库/首/ <产品> <版本>
      例
      〜/库/首选项/ WebStorm2019.1
      句法
      〜/ <产品> <版本> /配置
      例
      〜/。WebStorm2019.1 /配置

    • 创建一个名为idea.properties的文件。

    • 将ide.support.screenreaders.enabled=true属性添加到您创建的文件中。

  3. 启动WebStorm。支持屏幕阅读器选项位于“设置/首选项”|中外观与行为|外观将启用。

自定义IDE

您可以根据您的可访问性需求自定义IDE。

调整颜色以适应红绿色视觉缺陷

如果您有红绿色视觉缺陷，您可以调整颜色。在这种情况下，代码片段（例如通常以红色突出显示的错误或通常为绿色的字符串）将更改颜色以满足您的需求（红色将更改为橙色，绿色将更改为蓝色）。测试运行器中进度条的颜色也将进行调整，以便轻松识别。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 外观和行为”|外观 。

  2. 从右侧的选项中，选择调整红绿色视力缺陷（protanopia，deuteranopia）选项的颜色，然后单击确定以保存更改。

配置代码元素，编辑器，超链接等的颜色。

您可以调整代码元素，错误，编辑器元素和工具窗口的颜色。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案|一般 。

  2. 从右侧的选项列表中，选择要为其调整颜色的内容。例如，您可以选择“代码”并调整注入语言片段或匹配大括号等的颜色。单击“ 确定”以保存更改。

您还可以调整IDE的调试器，控制台和其他部分的颜色：只需在Settings / Preferences |中的选项列表中选择适当的节点即可。编辑|配色方案 。

覆盖默认的UI字体

您可以覆盖UI元素的默认字体。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 外观和行为”|外观 。

  2. 从右侧的选项中，在UI选项部分中，选择覆盖默认字体（不推荐）并指定字体名称及其大小。单击“ 确定”以保存更改。

调整工具窗口的大小

您可以使用快捷方式垂直或水平调整实际工具窗口的大小。

  1. 要向上或向下垂直调整大小，请按Ctrl + Shift +向上或Ctrl + Shift +向下 。

  2. 要向左或向右水平调整大小，请按Ctrl + Shift +向左或Ctrl + Shift +向右 。

在编辑器中调整文本大小

您可以在编辑器中更改文本的字体和大小。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般 。

  2. 从右侧的选项中，选择使用Ctrl +鼠标滚轮选项更改字体大小（缩放），以便在编辑器中工作时快速更改文本大小（转动鼠标滚轮）。

  3. 如果需要指定确切的字体大小，请选择编辑器|字体 。

  4. 从右侧的选项中，指定字体，大小，行间距和其他可用选项。单击“ 确定”以保存更改。

自定义快捷方式

您可以为经常使用的操作配置自定义快捷方式。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 键盘映射” 。

  2. 从右侧的选项列表（如菜单，操作和工具）中，选择所需的操作。

  3. 右键单击所选项目，然后从上下文菜单中选择要执行的操作，例如添加键盘快捷键 ， 添加鼠标快捷键或添加缩写 。

  4. 在打开的对话框中，指定快捷方式。如果需要，请选择“ 第二个笔划”选项，并为快捷方式指定其他键。单击“ 确定”以保存更改。请注意，您需要使用鼠标单击“ 确定 ”。如果按Enter键， WebStorm会将其视为快捷方式。

请注意，您可以忽略冲突并为多个操作指定快捷方式。但是，强烈建议您避免使用相同的快捷方式绑定两个操作，因为未定义这些操作的优先级。

自定义智能键行为

您可以配置智能钥匙的行为。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|智能钥匙 。

  2. 从右侧的选项中，选择或清除智能键选项，例如，您可以清除插入成对的括号或插入成对的引号选项，该选项会自动插入右括号或引号 ，因为它在您使用屏幕阅读器时可能没用。单击“ 确定”以保存更改。

禁用自动代码完成

您可以禁用自动代码完成，以避免在使用屏幕阅读器的编辑器中工作时自动插入代码元素。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|代码完成 。

  2. 从右侧的选项中，清除“ 智能类型完成”选项。如果需要，请清除基本完成以禁用基本完成。

自定义代码折叠

您可以控制代码折叠行为并指定应折叠或不折叠的内容。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|一般|代码折叠 。

  2. 从右侧的选项中，选择默认情况下应折叠的内容。

自定义代码样式

您可以配置空格，制表符和缩进。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|代码风格| [语言] 。

  2. 从右侧的选项中，单击选项卡和缩进以配置选项卡或空格以配置空格的位置和方式。

  3. 单击“ 确定”以保存更改。

管理鼠标指针焦点模式

您可以配置打开对话框时鼠标指针的行为方式。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 外观和行为”|外观 。

  2. 从右侧的选项中，选中自动将鼠标光标置于默认按钮复选框。打开对话框时，将鼠标指针放在默认按钮上。

  3. 单击“ 确定”以保存更改。

在编辑器中读取装订线图标和行号

您可以配置屏幕阅读器以读取位于编辑器左侧装订线中的行号，VCS注释，调试器和其他图标。

  1. 在编辑器中打开您的文件。

  2. 在编辑器中，按双击快捷键Shift + Alt + 6和F将焦点放在装订线上。WebStorm从您的插入符号当前所在的行开始读取。

  3. 使用向上和向下箭头键在行之间移动。如果需要移动到行中的下一个或上一个gutter元素，请分别使用向右和向左箭头键。

    虽然重点在于装订线，但是如果可用，屏幕阅读器可以读取装订线图标工具提示。

    要访问工具提示，请按双击快捷键Shift + Alt + 6和T.要浏览工具提示的内容（按符号逐个符号），请使用向右和向左箭头键。

  4. 按Escape键将焦点切换回编辑器。

高对比度的主题色彩

您可以设置高对比度主题以在WebStorm中工作。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 外观和行为”|外观 。

  2. 在“ 外观”页面的“ UI选项”区域的“ 主题”下拉列表中，选择“ 高对比度” ，然后单击“ 确定”以应用更改。

高对比度配色方案

您可以为编辑器设置高对比度方案。

  1. 在“ 设置/首选项”对话框（ Ctrl + Alt + S ）中，转到“ 编辑器”|配色方案 。

  2. 在“ 颜色方案”页面上，从“ 方案”下拉列表中选择“ 高对比度” 。

  3. 单击“ 确定”以应用更改。

您可以查看编辑器基础知识和WebStorm用户界面的导览，以熟悉其他有用的快捷方式。
