如果可能，移动为枚举常量

在enum类型具有一个具有适当类型的参数的构造函数的情况下，将常量（ static final字段）移动到enum类型时，此选项很有用。

比如说，我们正在从类Events中移动MOUSE_EVENT 。

class Events { public static final String MOUSE_EVENT = “mouseEvent” ; }

到枚举ActionType

枚举 ActionType { ; String typeName ; ActionType （ String name ） { typeName = name ; } }

如果选项已启用，我们将获得以下结果：

枚举 ActionType { MOUSE_EVENT （ “mouseEvent” ）; String typeName ; ... }

如果该选项关闭，结果将是：

枚举 ActionType { ; public static final String MOUSE_EVENT = “mouseEvent” ; String typeName ; ... }
