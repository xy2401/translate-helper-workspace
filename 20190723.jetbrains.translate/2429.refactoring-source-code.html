重构代码

在这部分你会发现：

  • 一般重构程序

  • 可用重构的程序和示例

要执行重构，请遵循以下常规步骤

  1. 选择（或悬停插入符号）要重构的符号或代码片段。可用的重构集取决于您的选择。您可以在以下PyCharm组件中选择符号：

    • 项目视图

    • 结构工具窗口

    • 编辑

    • UML类图

  2. 执行以下操作之一：

    • 在主Refactor菜单上或从选择的上下文菜单中，选择所需的重构或按相应的键盘快捷键（如果有）。

    • 在主菜单上，选择Refactor |重构这个 ，或按Ctrl + Shift + Alt + T ，然后从弹出窗口中选择所需的重构。

  3. 在打开的对话框中，指定重构选项。

  4. 要立即应用更改，请根据重构类型单击“ 重构”或“ 确定” 。

  5. 对于某些重构，可以选择在实际执行重构之前预览更改。在这种情况下， 预览按钮在相应的对话框中可用。

    要预览潜在更改并进行必要的调整，请单击“ 预览” 。PyCharm显示将在“ 查找”工具窗口的专用选项卡上进行的更改。

    此步骤中的一个可能操作是从重构中排除某些条目。为此，请在列表中选择所需条目，然后按Delete 。

    如果在重构后预计会发生冲突，PyCharm会显示一个对话框，其中包含遇到问题的简要说明。如果是这种情况，请执行以下操作之一：

    • 单击“ 继续”按钮忽略冲突。结果，将执行重构，但是，这可能导致错误的结果。

    • 单击“ 在视图中显示”按钮预览冲突。PyCharm在“ 查找”工具窗口的“ 冲突”选项卡上显示所有冲突条目，使您能够导航到有问题的代码行并进行必要的修复。

    • 取消重构并返回编辑器。

  6. 如果对建议的结果感到满意，请单击“ 执行重构”以应用更改。

PyCharm提供以下常见的重构：

  • 改变签名
  • 转换为包和模块
  • 提取重构
  • 排队
  • 反转布尔值
  • 移动并复制重构
  • 拉大家，推倒会员
  • 重命名重构
  • 安全删除
