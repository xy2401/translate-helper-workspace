听众

监听器是Sun Microsystems Java™Servlet 2.3规范中的新功能。通过定义在生命周期事件发生时调用的方法，可以使用侦听器来监视servlet生命周期中的事件并对其做出反应。Java Servlet规范2.3中定义了对应用程序级事件的支持。

应用程序事件侦听器是实现一个或多个servlet事件侦听器接口的类。Servlet事件侦听器支持ServletContext和HttpSession对象中状态更改的通知，具体为：
  • Servlet上下文侦听器用于管理应用程序在VM级别保存的资源或状态。
  • HTTP会话侦听器用于管理与来自同一客户端或用户的Web应用程序中发出的一系列请求相关联的状态或资源。

有关侦听器的更多信息，请参阅java.sun.com/products/servlet/download.html上的Sun Microsystems Java Servlet 2.3规范 。
