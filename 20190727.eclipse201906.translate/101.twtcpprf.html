定义TCP / IP Monitor首选项

使用“首选项”页面，可以定义TCP / IP Monitor属性。

要定义这些首选项，请完成以下步骤：
  1. 从工具栏中，选择“ 窗口” > “首选项”
  2. 在“首选项”窗口中，展开“运行/调试” ，然后选择“ TCP / IP监视器” 。
  3. 如果要在通过TCP / IP监视服务器存在活动时显示TCP / IP监视器视图，请选中“ 存在活动时显示TCP / IP监视器视图”复选框。
  4. 要添加TCP / IP监视服务器，请完成以下操作：
    1. 在“ TCP / IP监视器”列表下，单击“ 添加” 。 “ 新建监视器”对话框打开
    2. 指定以下设置，然后单击“ 确定” ：
    选项 描述
    本地监控端口 在本地计算机上指定唯一的端口号。
    主机名 指定运行服务器的计算机的主机名或IP地址。
    港口 指定远程服务器的端口号。
    类型 指定Web浏览器中的请求类型是由HTTP还是TCP / IP发送。如果选择了HTTP选项，则会修改来自Web浏览器的请求，以便HTTP标头指向远程计算机，并且如果在同一连接中收到多个HTTP请求，则将其分开。如果选择了TCP / IP选项，则所有请求都将逐字节发送，并且TCP / IP监视器不会转换或修改它转发的任何请求。
    超时（以毫秒为单位） 指定在结束与服务器的TCP / IP连接之前可以等待的时间长度（以毫秒为单位）。
  5. 使用“ 开始”和“ 停止”按钮管理TCP / IP监视服务器。 TCP / IP Monitor表列出了可用的TCP / IP监视服务器。您可以从TCP / IP Monitor表中添加，编辑，删除，启动或停止可用的TCP / IP监视服务器。“ 状态”列显示TCP / IP监视器是启动还是停止。“ 主机名”列显示运行服务器的计算机的主机名或IP地址。“ 类型”列显示Web浏览器中的请求类型是由HTTP还是TCP / IP发送。
  6. 单击“ 应用” ，然后单击“确定”保存更改。
