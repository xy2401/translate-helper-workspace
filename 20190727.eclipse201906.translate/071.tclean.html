清理服务器上发布的项目

在执行完全重新发布之前，您可以使用Servers视图中提供的clean选项从服务器中删除任何无效资源。这有助于删除由于发布过程而生成的副产品。当您发现服务器上运行旧代码或无效代码状态时，请尝试使用此清除选项查看是否有助于从服务器中删除这些无效状态。

要清除服务器上发布的项目，请完成以下步骤：
  1. 在“服务器”视图中，右键单击服务器。
  2. 在弹出菜单中，选择“ 清理” 。
  3. 将显示以下消息对话框： 
    Clean will discard all publish state and republish from scratch.  Are you sure you 
want to clean all published projects?
  4. 选择确定 。
