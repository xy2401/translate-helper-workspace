在HTTP预览服务器上进行测试

如果要快速测试静态Web项目中的资源，请使用HTTP预览服务器。HTTP预览服务器是与工作台集成的超文本传输协议（HTTP）嵌入式测试服务器。它用于初步测试，节省您下载，安装和设置HTTP服务器的时间。但是，随着静态Web项目在其开发周期中的进展，建议您在外部HTTP服务器上进行测试和运行。在外部HTTP服务器上进行最终测试之前，可以使用HTTP预览服务器作为检查错误的便捷方法。

要在HTTP预览服务器上创建和测试以在静态Web项目中预览资源，请完成以下步骤：
  1. 在Servers视图中，右键单击并选择New > Server 。将打开“定义新服务器”向导。
  2. 在“ 选择服务器类型”列表下，展开“ 基本”文件夹，然后选择“ HTTP预览” 。单击下一步 。
  3. 将打开“ 添加和删除”页面。在“可用项目”列表下，选择静态Web项目，然后单击“ 添加”按钮。单击完成 。 HTTP预览服务器在Servers视图中创建。
  4. 在Project Explorer视图中，导航到静态Web项目，然后右键单击要测试的资源。
  5. 选择Run As > Run on Server 。“运行服务器”向导将打开。
  6. 确认已启用“ 选择现有服务器”单选按钮。
  7. 在“ 选择要使用的服务器”列表下，选择最近创建的HTTP预览服务器。单击下一步 。
  8. 将打开“ 添加和删除”页面。验证您正在测试的静态Web项目是否列在Configured项目下 。单击完成 。
内部Web浏览器打开并显示静态Web内容。
