Java EE透视图中的Project Explorer视图

在Java™EE透视图中开发J2EE应用程序时，Project Explorer视图是J2EE项目和资源的主视图。

Project Explorer视图提供了所有项目资源的集成视图，包括J2EE部署描述符，Java工件，资源，Web服务，数据库和动态Web项目工件的模型。

您应该使用此视图来处理J2EE部署描述符及其内容。您可以查看企业应用程序项目并查看与其关联的所有模块。

您还可以过滤在Project Explorer视图中看到的内容，以隐藏您不想看到的项目，文件夹或文件。要启用或禁用过滤器，请从视图右上角的下拉菜单中选择自定义视图....有关更多信息，请参阅Project Explorer视图中的Filters 。

或者，您可以通过显示或隐藏工作集，相关资源组或项目来过滤您看到的内容。有关更多信息，请参阅工作集 。
