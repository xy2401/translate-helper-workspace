监控服务器端口

您可以监视服务器端口以查看通过服务器端口发送和接收的数据。使用服务器的“属性”页面，可以从要监视的可用服务器端口类型列表中进行选择。您可以监视的服务器端口类型的示例包括超文本传输协议（HTTP），服务器的管理端口，代理和许多其他特定于供应商的端口。

先决条件：
  • 启动要监视的服务器。
要定义要监视的端口，请完成以下步骤：
  1. 在Servers视图中，右键单击服务器，然后选择Properties > Monitoring 。
  2. 要添加用于监视服务器的端口，请完成以下步骤：
    1. 单击“ 添加”按钮。
    2. 将打开“ 监视端口”对话框，其中显示可以监视的可用服务器端口列表。在此列表中，选择要监视的服务器端口。
    3. 在Monitor端口中 ，指定本地计算机上的唯一端口号以进行监视。默认情况下，此字段使用数字填写，其值为已分配的端口号，增量为1。
    4. 在“ 类型”下拉列表中，选择要监视的内容类型。如果选择“ 全部” ，则将监视在所选服务器端口上发送和接收的所有数据。如果此下拉列表中还有其他可用选项，则可以筛选要监视的数据类型。
    5. 单击确定 。
  3. 使用“ 开始”和“ 停止”按钮管理服务器端口。 Monitor Port表列出了可用的端口监视服务器。您可以从Monitor Port表中添加，编辑，删除，启动或停止可用的端口监视服务器。“ 状态”列显示端口监视服务器是启动还是停止。“ 类型”列显示您正在监视的服务器端口的协议。“ 服务器端口”列显示端口在服务器上运行的端口号。Monitor Port列显示您正在使用监视目的的端口号。内容类型显示通过服务器端口发送和接收的数据类型。
  4. 单击“ 应用” ，然后单击“确定”保存更改。
在服务器上运行应用程序时，数据将显示在TCP / IP Monitor视图中 。
