TCP / IP Monitor视图

TCP / IP Monitor是一个简单的服务器，它监视Web浏览器和服务器之间的所有请求和响应。如果要在TCP / IP活动时打开TCP / IP监视器视图，请在“TCP / IP监视器”首选项页面上选中“ 有活动时显示TCP / IP监视器”视图复选框（ 窗口 > 运行/调试 > TCP / IP监视器 ）。

您可以使用TCP / IP Monitor视图（“ 窗口” >“ 显示视图” >“ 其他” >“ 调试” >“ TCP / IP监视器” ） 监视TCP / IP活动 。该视图由三个窗格组成：

  • “请求列表”窗格。此窗格位于TCP / IP Monitor视图的顶部，包含发送到应用程序服务器的请求列表。
  • 请求窗格。该窗格位于TCP / IP Monitor视图的左侧，包含有关将请求转发到应用程序服务器的信息。
  • 响应窗格。该窗格位于TCP / IP Monitor视图的右侧，包含有关从应用程序服务器收到响应时的响应的信息。

在“请求列表”窗格中，JSP文件，servlet和URL等请求由主机进行分组和显示。选择请求会在“请求”窗格和“响应”窗格中显示有关请求的相应信息。响应时间是请求到达TCP / IP监视器所花费的时间，直到从服务器转发响应为止。
