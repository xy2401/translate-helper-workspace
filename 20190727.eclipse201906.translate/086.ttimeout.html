设置服务器启动或停止时的超时

您可以指定在工作台取消启动或停止服务器的过程之前可以等待的时间长度。

提示：如果您有快速处理机器，则可以尝试设置短暂的超时间隔。如果您的处理速度较慢，可以尝试设置较长的超时间隔。
要设置启动或停止服务器的超时：
  1. 在“服务器”视图中，右键单击要修改其超时的服务器，然后选择“ 打开” 。 服务器编辑器打开。
  2. 在“ 超时”部分下，您有以下选项：
    • 在工作台取消启动服务器的进程之前，使用“ 开始”控件指定允许经过的秒数。
    • 在工作台提示您执行操作之前，使用“ 停止”控件指定允许经过的秒数。您的提示请求单击“ 确定”以结束服务器，或单击“ 取消”继续等待服务器停止。有关提示的文本，请参阅下面的“ 结果”部分。
结果
如果启动服务器的超时间隔已过，则工作台将显示以下消息：
Server server_name was unable to start within x seconds.  If the server requires more time, try increasing the timeout in the server editor.
其中server_name是服务器的名称，用于在Servers视图中标识其条目， x是Start控件中指定的秒数。
如果停止服务器的超时间隔已过，则工作台将显示以下提示：
Terminate Server
Server server_name is not responding.  Do you want to terminate this server?  Click OK to terminate the server or click Cancel to continue waiting.
提示：如果您经常遇到任何这些消息，请尝试以下本主题中的步骤来设置较长的超时间隔。这可能有助于解决消息。
