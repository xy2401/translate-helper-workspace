在J2EE预览服务器上进行测试

如果要在J2EE Web项目中快速编译，测试和运行资源，请使用J2EE预览服务器。您可以在J2EE预览服务器上测试的J2EE Web项目是静态Web项目，动态Web项目和实用程序项目。此后，术语J2EE Web项目指的是前面提到的任何项目。

J2EE预览服务器是与工作台集成的嵌入式应用程序服务器。它用于初步测试，为您节省下载，安装和设置应用程序服务器的时间。但是，随着J2EE Web项目在其开发周期中的进展，建议您在外部应用程序服务器上进行测试和运行。在外部应用程序服务器上进行最终测试之前，可以使用J2EE预览服务器作为检查错误的便捷方法。

要在J2EE预览服务器上创建和测试以在J2EE Web项目中预览资源，请完成以下步骤：

  1. 在Servers视图中，右键单击并选择New > Server 。将打开“定义新服务器”向导。
  2. 在“ 选择服务器类型”列表下，展开“ 基本”文件夹，然后选择“ J2EE预览” 。单击下一步 。
  3. 将打开“ 添加和删除”页面。在Available projects列表下，选择J2EE Web项目，然后单击Add按钮。单击完成 。 J2EE预览服务器在Servers视图中创建。
  4. 在Project Explorer视图中，导航到J2EE Web项目，然后右键单击要测试的资源。
  5. 选择Run As > Run on Server 。“运行服务器”向导将打开。
  6. 确认已启用“ 选择现有服务器”单选按钮。
  7. 在“ 选择要使用的服务器”列表下，选择最近创建的J2EE预览服务器。单击下一步 。
  8. 将打开“ 添加和删除”页面。验证您正在测试的J2EE Web项目是否列在Configured项目下 。单击完成 。
内部Web浏览器打开并显示Web内容。
