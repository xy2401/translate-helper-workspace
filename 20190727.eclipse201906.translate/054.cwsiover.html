Web服务互操作性（WS-I）

WS-I是一个旨在促进跨平台，操作系统和编程语言的Web服务互操作性的组织。

有关WS-I的更多信息，请访问他们的网站： http ： //www.ws-i.org/ 。此站点包含诸如Web服务互操作性概述，使用方案和规范之类的资源。

WS-I Basic Profile概述了WSDL和Web服务协议（SOAP / HTTP）流量必须遵守的要求才能声明WS-I一致性。Web服务WS-I验证工具目前支持WS-I Simple SOAP Binding Profile 1.0（WS-I SSBP），它构建在WS-I Basic Profile和WS-I Attachments Profile 1.0（WS-I AP）之上。要查看规范，请参阅WS-I Web站点，在Deliverables下选择Basic Profile 。

根据所创建的Web服务的类型，您可能希望或不希望Web服务符合WS-I配置文件。默认的合规性级别是，如果选择了非WS-I SSBP投诉Web服务选项，则生成警告，并忽略任何非WS-I AP兼容选择。您可以在工作空间或项目级别设置WS-I合规性级别。Web服务向导，在WebSphere®运行时环境，WSDL编辑器，和其他Web服务工具提供了支持和鼓励的WS-I合规服务的发展。
