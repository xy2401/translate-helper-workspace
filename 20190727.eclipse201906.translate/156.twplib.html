添加Web库项目

Web库项目允许您将Java™项目与Web项目的WEB-INF / lib目录中的“虚拟”JAR文件相关联。您可以引用存在于包含Web项目的企业应用程序项目中其他位置的JAR文件（如果它们位于Web项目的构建路径中），并且在发布Web应用程序之前，无需将这些JAR文件显式复制到项目的lib文件夹中到服务器。

要设置这些关联：

  1. 右键单击Web项目，然后从弹出菜单中选择“ 属性 ”。
  2. 单击Java EE Module Dependencies
  3. 在Java EE Module Dependencies窗口中设置关联
  4. 完成后单击“ 确定” 。
