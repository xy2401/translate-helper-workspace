编辑服务器

您可以使用服务器编辑器编辑服务器。您更改的字段取决于服务器定义使用的运行时环境，或者是否将其定义为本地或远程服务器。

要编辑服务器：
  1. 打开服务器编辑器，完成以下操作之一：
    • 在“服务器”视图中，双击要编辑的服务器。
    • 在Servers视图中右键单击服务器，从弹出菜单中选择Open 。
  2. 在工作台上打开服务器编辑器时，可以通过从工具栏文件 > 保存中选择或键入CTRL+S来保存更改。
    提示：服务器编辑器中尚未保存的更改将通过位于服务器编辑器顶部的*图标表示。在服务器编辑器上保存更改后，服务器编辑器上将不再显示*图标。
  3. 在启动，重新启动或添加和删除服务器上的模块之前，请关闭所有服务器编辑器。 否则，您可能会提示以下消息：
    File Changed
The server has been changed on the file system.  Do you want to load the changes?  Yes or No.
