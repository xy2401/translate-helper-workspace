创建静态Web项目

在工作台中，您可以为Web项目中的Web应用程序创建和维护资源。如果要创建不包含动态元素的静态，基于内容的Web应用程序（如JSP文件或servlet），请使用“ 新建静态Web项目”向导。

要创建新的静态Web项目，请完成以下步骤：

  1. 打开Java EE透视图
  2. 在Project Explorer中，右键单击Other Projects并从上下文菜单中选择New-> Other-> Web-> Static Web Project 。“新建静态Web项目”向导启动。
  3. 按照项目向导提示进行操作

一般信息

项目方面
facet表示Web项目中的一个功能单元。例如，静态Web模块方面可以将项目部署为静态Web模块。选择项目构面时，会在向导中显示项目构面的简要说明。请注意，在许多情况下，您可以通过右键单击构面并从弹出菜单中选择项目约束来查看项目构面的约束。
目标运行时
使用此字段定义新安装的运行时环境。运行时在构建时用于编译项目。
Web内容文件夹
要存储可发布资源的文件夹。
