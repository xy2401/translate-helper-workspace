Java EE透视图

Java™EE透视图包括在为企业应用程序，EJB模块，Web模块，应用程序客户端模块和连接器项目或模块开发资源时可以使用的工作台视图。

您可以重新排列透视图中视图的位置，平铺和大小。您还可以通过单击“ 窗口” >“ 显示视图”并选择视图，将其他视图添加到Java EE透视图。

工作台提供不同视图和编辑器之间的同步。在J2EE透视图中也是如此。

默认情况下，Java EE透视图包括以下工作台视图：

Project Explorer
Project Explorer视图提供项目及其与J2EE开发相关的工件的集成视图。您可以根据工作集显示或隐藏项目。此视图显示J2EE部署描述符，Java工件（源文件夹，包和类）的可导航模型，可用Web服务的可导航模型以及Web模块的专用视图，以简化动态Web应用程序的开发。此外，还可以轻松获得EJB数据库映射和J2EE应用程序服务器项目的配置。
大纲
Java EE透视图中的Outline视图显示了您正在编辑的文件的大纲。例如，如果使用选项卡式部署描述符编辑器，则“大纲”视图将显示所选页面元素的大纲，如果在“源”选项卡上进行编辑，则会显示XML源的大纲。如果要在Java编辑器中编辑企业bean，则“大纲”视图将显示Java类的大纲。
任务
“任务”视图列出您输入的待办事项。
问题
“问题”视图显示与所选项目关联的问题，警告或错误。您可以双击某个项目以解决相应资源中的特定问题。
属性
“属性”视图提供了在编辑器中打开的文件中对象的属性和关联值的表格视图。
服务器
Servers视图显示所有已创建的服务器实例。您可以从此视图启动和停止每个服务器，然后启动测试客户端。
片段
Snippets视图提供了可以插入源代码中适当位置的分类代码片段。
数据源资源管理器
Data Source Explorer提供已配置的连接配置文件列表。如果启用了类别，则可以看到按类别分组的列表。使用Data Source Explorer连接，导航和与所选连接配置文件关联的资源进行交互。它还提供导入和导出功能，以与其他Eclipse Workbenches共享连接配置文件定义。
状态栏
状态栏提供左侧Project Explorer视图中所选对象位置的描述。打开文件和部署描述符时，状态栏会显示文件的只读状态以及适用的行号和列号。有时，当长时间操作运行时，状态栏中将显示状态监视器以及带有停止符号图标的按钮。单击停止符号可在取消操作时停止操作。
