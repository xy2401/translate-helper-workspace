Java EE应用程序

这些主题涉及Java™Enterprise Edition（Java EE）。

  • 开发Java EE应用程序
    Java EE 5编程模型简化了创建Java应用程序的过程。
  • J2EE架构
    Java 2平台企业版（J2EE）为开发多层企业应用程序提供了标准。
  • Java EE透视图
    Java EE透视图包括在为企业应用程序，EJB模块，Web模块，应用程序客户端模块和连接器项目或模块开发资源时可以使用的工作台视图。
  • Java EE透视图中的Project Explorer视图
    在Java EE透视图中开发J2EE应用程序时，Project Explorer视图是J2EE项目和资源的主视图。
  • 在Project Explorer视图中过滤
    您可以过滤Project Explorer视图以隐藏您不想看到的项目，文件夹或文件。
  • 使用项目
    工作台可以与许多不同类型的项目一起使用。以下主题涉及创建和管理与J2EE开发相关的一些类型的项目。
  • 验证企业应用程序中的代码
    工作台包括验证程序，用于检查企业应用程序模块项目中的某些文件是否存在错误。
