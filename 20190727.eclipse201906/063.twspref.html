Setting Web services preferences

Before you begin developing Web services or clients, you can optimize the workbench for Web services development by setting a variety of preferences.

To set any of the Web services preferences, follow these steps:
  1. Click Window > Preferences to open the Preferences notebook.
  2. Expand Web services or Service policies, and click the preference category that you want to set.
  3. Select the check boxes that you want to set as the default when creating your Web service. If you require more information on any of the options, select the option and press F1.
  4. Click OK to apply the changes and close the Preferences notebook.
For information on individual preferences, refer to the following:
  • Axis emitter preferences
  • Web services wizards basic preferences
  • WSDL file preferences
  • WS-I compliance preferences
  • Axis emitter preferences
    The Axis Emitter preferences page allows you to set the following preferences used when creating or consuming Axis Web services.
  • Web services wizards basic preferences
    The default behaviors of the Web services wizards can be set using the following preference panels. All preferences can be overridden in the wizard.
  • WSDL file preferences
    Use the WSDL Files preferences page to set the default behavior when creating or saving WSDL files.
  • WS-I compliance preferences
    The Service policies preferences page allows you to select your level of compliance with the WS-I Simple SOAP Binding Profile (SSBP) and WS-I Attachement Profile (AP). They can be found on the Service Policies preferences page.
