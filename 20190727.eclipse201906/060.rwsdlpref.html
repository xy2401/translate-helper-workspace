WSDL file preferences

Use the WSDL Files preferences page to set the default behavior when creating or saving WSDL files.

Default target namespace:
When creating a WSDL file in the WSDL editor, you can set the default target namespace. If you do not select to change the default, your WSDL file will have http://tempuri.org as its target namespace.
Regenerate Binding on save
Select this to automatically regenerate the binding every time you save a WSDL file.
Prompt Regenerate Binding on save
Select this to be asked if you want to regenerate the binding every time you save a WSDL file.
