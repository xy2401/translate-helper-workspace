5.297 DBA_ROLLING_STATISTICS

DBA_ROLLING_STATISTICS provides a list of rolling operation statistics.

Column Datatype NULL Description

NAME

VARCHAR2(256)

Name of the statistic

VALUE

VARCHAR2(256)

Value of the statistic

UPDATE_TIME

TIMESTAMP(6)

Time of last update
