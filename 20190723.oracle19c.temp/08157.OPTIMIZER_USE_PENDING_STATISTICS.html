1.238 OPTIMIZER_USE_PENDING_STATISTICS

OPTIMIZER_USE_PENDING_STATISTICS specifies whether the optimizer uses pending statistics when compiling SQL statements.

Property Description

Parameter type

Boolean

Default value

false

Modifiable

ALTER SESSION, ALTER SYSTEM

Modifiable in a PDB

Yes

Range of values

true | false

Basic

No

Oracle Database SQL Tuning Guide for more information on setting this parameter
