Server Hardware Checklist for Oracle Database Installation

Use this checklist to check hardware requirements for Oracle Database.

Table 1-1 Server Hardware Checklist for Oracle Database Installation

Check Task
Server Make and Architecture

Confirm that server make, model, core architecture, and host bus adaptors (HBA) or network interface controllers (NICs) are supported to run with Oracle Database and Oracle Grid Infrastructure. Ensure that the server has a DVD drive, if you are installing from a DVD.
Runlevel 3
Server Display Cards At least 1024 x 768 display resolution, which Oracle Universal Installer requires.
Minimum network connectivity Server is connected to a network
Minimum RAM

  • At least 1 GB RAM for Oracle Database installations. 2 GB RAM recommended.

  • At least 8 GB RAM for Oracle Grid Infrastructure installations.
