6.603 USER_SECONDARY_OBJECTS

USER_SECONDARY_OBJECTS provides information about secondary objects associated with domain indexes owned by the current user. Its columns are the same as those in ALL_SECONDARY_OBJECTS.

This view is only relevant in the context of domain indexes.
