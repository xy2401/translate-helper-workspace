Using Dry-Run Upgrade Mode to Check System Upgrade Readiness

Use dry-run upgrade mode of Oracle Grid Infrastructure installation wizard, gridSetup.sh, to check readiness for Oracle Clusterware upgrades.

  • About Oracle Grid Infrastructure Dry-Run Upgrade Mode
    Oracle Grid Infrastructure dry-run upgrade mode enables you to check system readiness for upgrade.
  • Performing Dry-Run Upgrade Using Oracle Universal Installer
    Run the Oracle Grid Infrastructure installer in dry-run upgrade mode to determine if the system is ready for upgrade.
