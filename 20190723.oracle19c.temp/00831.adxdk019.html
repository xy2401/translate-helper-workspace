Description of the illustration adxdk019.gif

This figure shows XSU running in the middle tier.

A bidirectional arrow labeled “SQL data (via JDBC)” connects a cylinder icon labeled “Any Database”, which encloses a box labeled “SQL Tables and Views”, with a large server icon labeled “Middle Tier Application Server or Oracle Database (Java or PL/SQL front end)”. The large server icon is overlaid with two boxes, labeled “Application Logic” and “XML SQL Utility (Java)”. A bidirectional arrow labeled “XML*” connects the large server icon with a smaller server icon labeled “Web Server”. A bidirectional arrow labeled “XML*” connects the smaller server icon with a cloud icon labeled “Internet”. A bidirectional arrow connects the cloud icon with an icon of a user at a computer, labeled “User”. A bidirectional arrow labeled “XML*” connects the larger server icon with an icon of three cylinders, labeled “Other Database, Messaging Systems,...”. A legend in the corner says “* XML, HTML, XHTML, VML,...”.
