Server Hardware Checklist for Oracle Database Client Installation

Use this checklist to check hardware requirements for Oracle Database Client installations.

Table 1-1 Server Hardware Checklist for Oracle Database Client Installations

Check Task
Server Make and Architecture

Confirm that server make, model, core architecture, and host bus adaptors (HBA) or network interface controllers (NICs) are supported to run with Oracle Database and Oracle Grid Infrastructure. Ensure the server has a DVD drive, if you are installing from a DVD.
Runlevel 3
Server Display Cards

At least 1024 x 768 display resolution, which Oracle Universal Installer requires.
Minimum network connectivity

Client is connected to a network.
Minimum RAM

At least 256 MB of RAM.
