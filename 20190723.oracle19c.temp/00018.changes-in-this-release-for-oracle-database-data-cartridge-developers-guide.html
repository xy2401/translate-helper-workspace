Changes in Oracle Database 19c, Version 19.1

This section lists changes in Oracle Database Data Cartridge Developer's Guide for Oracle Database release 19c, version 19.1.

See Oracle Database New Features Guide for a complete list of new features in this release.

Desupported Features

Oracle Multimedia: Starting in Oracle Database 19c, the Oracle Multimedia feature is desupported. Use Oracle SecureFiles with open source or third-party products for metadata extraction and for image processing and conversion instead of Oracle Multimedia.
