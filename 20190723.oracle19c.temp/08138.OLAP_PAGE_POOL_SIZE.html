1.219 OLAP_PAGE_POOL_SIZE

OLAP_PAGE_POOL_SIZE specifies (in bytes) the size of the OLAP page pool.

Property Description

Parameter type

Big integer

Syntax

OLAP_PAGE_POOL_SIZE = integer [K | M | G]

Default value

0

Modifiable

ALTER SESSION, ALTER SYSTEM ... DEFERRED

Modifiable in a PDB

Yes

Range of values

0 to 2 GB

Basic

No
