An example job, XYZ, shows which AutoUpgrade utility steps are options are run with each AutoUpgrade mode. The sequence of AutoUpgrade steps is, in order, setup, preupgrade scripts, prechecks, prefixups, GRP (guaranteed recovery point), drain, DB Upgrade, the following steps run, in sequence: post checks, postfixup scripts, and postupgrade scripts. In Analyze mode, only Setup and Prechecks run. In Fixups mode, the following steps run: Setup, fixups, and prefixups. In Deploy mode, all steps run in sequence: setup, preupgrade scripts, prechecks, prefixups, GRP, drain, DB upgrade, Post checks, postfixup scripts, and postupgrade. In Upgrade mode, the following steps run: Setup, DB upgrade, post checks, and postfixup scripts.
