Use checklists to review system requirements, and to plan and carry out Oracle Database Client installation.

Oracle recommends that you use checklists as part of your installation planning process. Using checklists can help you to confirm that your server hardware and configuration meet minimum requirements for this release and can help you carry out a successful installation.
