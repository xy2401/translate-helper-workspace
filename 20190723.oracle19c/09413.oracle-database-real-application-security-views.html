2.6 Oracle Database Real Application Security Views

This manual describes these Oracle Database Real Application Security auditing views:

  • DBA_XS_AUDIT_POLICY_OPTIONS

  • DBA_XS_AUDIT_TRAIL

  • DBA_XS_ENB_AUDIT_POLICIES

Descriptions of the other Oracle Database Real Application Security views are provided in Oracle Database Real Application Security Administrator's and Developer's Guide.

Note:

Oracle Database Real Application Security views include _XS_ in the view name.
