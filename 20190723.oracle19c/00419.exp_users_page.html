Description of the illustration exp_users_page.gif

This screenshot of the Users page has several components. At the top of the page is the Users heading, and to the far right is the Page Refreshed field that shows the time that the page was last refreshed. Below the Users heading on the left is the Actions menu, the Create User button, and the Drop User button. To the far right is the Search section. Below these are a table that lists information about the users in the database. The table contains the Name, Account Status, Expiration Date, Default Tablespace, Temporary Tablespace, Profile, and Created columns.
