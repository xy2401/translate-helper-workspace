Accessing the Installation Software

You can download Oracle Database software from the Oracle Technology Network website or the Oracle Software Delivery Cloud portal. In some cases, Oracle Database software may be available on installation media also.

To install the software from the hard disk, you must either download it and unpack it, or copy it from the installation media, if you have it.

  • Downloading Oracle Software
    Select the method you want to use to download the software.
  • Downloading the Installation Archive Files from OTN
    Download installation archive files from Oracle Technology Network (OTN).
  • Downloading the Software from Oracle Software Delivery Cloud Portal
    You can download the software from Oracle Software Delivery Cloud as Media Packs.
  • Copying the Software to the Hard Disk
    Oracle recommends that you copy the installation software to the hard disk to enable the installation to run faster.
