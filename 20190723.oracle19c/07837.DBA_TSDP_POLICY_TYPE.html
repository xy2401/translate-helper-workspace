6.64 DBA_TSDP_POLICY_TYPE

DBA_TSDP_POLICY_TYPE shows the Transparent Sensitive Data Protection policy to sensitive column type mapping.

Column Datatype NULL Description

POLICY_NAME

VARCHAR2(128)

The Transparent Sensitive Data Protection policy name

SENSITIVE_TYPE

VARCHAR2(128)

The sensitive column type name
