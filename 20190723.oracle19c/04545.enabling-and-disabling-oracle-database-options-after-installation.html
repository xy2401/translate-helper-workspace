Enabling and Disabling Oracle Database Options After Installation

The chopt tool changes your database options after installation.

When you install Oracle Database, some options are enabled and others are disabled. To enable or disable a particular database feature for an Oracle home, shut down the database and use the chopt tool.

  • Chopt Tool
    Use the chopt tool after installation to add or remove Oracle Database options.
