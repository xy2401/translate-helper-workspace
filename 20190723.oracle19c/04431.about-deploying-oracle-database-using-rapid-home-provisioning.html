About Deploying Oracle Database Using Oracle Fleet Patching and Provisioning

You can use Oracle Fleet Patching and Provisioning (Oracle FPP) to provision Oracle Database software.

Starting with Oracle Database 19c, Rapid Home Provisioning is renamed to Oracle Fleet Patching and Provisioning (Oracle FPP).

With Oracle Fleet Patching and Provisioning, you create, store, and manage templates of Oracle homes as images (called gold images) of Oracle software, such as databases, middleware, and applications. You can make a working copy of any gold image and then you can provision that working copy to any node in the data center or cloud computing environment.

You can use Oracle Fleet Patching and Provisioning to provision, patch, and upgrade single-instance databases running on Oracle Restart, on clusters, or on single, standalone nodes. These may be on nodes with or without Oracle Grid Infrastructure installed.

Note:

Oracle Fleet Patching and Provisioning is not supported on Microsoft Windows and HP-UX Itanium systems.
