Description of the illustration edit_policy-classifer_ordering

The screenshot shows the Edit Policy Set: Classifier Ordering page. The descriptive text for this page says "Work requests are evaluated against the Classifier's Boolean expression in the order specified below." This text is followed by an information icon, which you can click to open the Help page.

The Edit Policy Set: Classifier Ordering page contains a single table which lists the Performance Classes and their Classifiers. There are three columns in the table: Select, Performance Class, and Classifier. The table shown in the screenshot contains 10 rows, which contain the following values:

  • Selected, salescart_pc, SERVICE_NAME INSET prod:sales,USER INSET SOE5

  • Not selected, employee_pc, SERVICE_NAME INSET prod:employee

  • Not selected, hrcmrpt_pc, SERVICE_NAME INSET prod:hrcmrpt

  • Not selected, manager_pc, SERVICE_NAME INSET prod:manager

  • Not selected, sales_pc, SERVICE_NAME INSET prod:sales

  • Not selected, salesdev_pc, SERVICE_NAME INSET dev:salesdev

  • Not selected, salesrpt_pc, SERVICE_NAME INSET prod:salesrpt

  • Not selected, salestest_pc, SERVICE_NAME INSET test:salestest

  • Not selected, webrpt_pc, SERVICE_NAME INSET webdb:webrpt

  • Not selected, websvc_pc, SERVICE_NAME INSET webdb:websvc

To the right of the table is a list of arrows inside circles. The first shows an arrow pointing upwards, with a bar across the top. The second arrow shows an arrow pointing upwards. The third arrow shows an arrow pointing downwards. The fourth arrow shows an arrow pointing downwards, with a bar across the bottom.

End of description.
