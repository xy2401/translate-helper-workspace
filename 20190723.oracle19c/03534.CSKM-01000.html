25 CSKM-01000 to CSKM-01112

CSKM-01000: invalid command line option {0}

Cause: The requested operation was rejected because of the indicated command line option was invalid.

Action: Review the command usage and retry using valid options.

CSKM-01001: invalid number of arguments for the command line option {0}

Cause: The requested operation was rejected because of the number of arguments supplied was invalid for the specified command line option.

Action: Review the command usage and retry using the correct number of arguments.

CSKM-01002: invalid input arguments for the command line option {0}

Cause: The requested operation was rejected because the arguments supplied were invalid for the specified command line option.

Action: Review the command usage and retry using valid arguments.

CSKM-01003: Unexpected error: {0}

Cause: The requested command execution failed unexpectedly. The appended error from the Java Virtual Machine provides detailed failure information.

Action: Retry the command, and if the error persists, contact Oracle Support Services.

CSKM-01004: Request failed; Cluster Ready Services not available.

Cause: Command execution failed because Cluster Ready Services was not running on the local node.

Action: Start Cluster Ready Services on the local node and retry the command.

CSKM-01005: Failed to create wallet: {0}

Cause: An error occurred while performing wallet operation.

Action: Examine the accompanying error messages for details, resolve the issues raised and retry, and if the error persists, contact Oracle Support Services.

CSKM-01006: Failed to perform operation on wallet: {0}

Cause: An error occurred while performing wallet operation.

Action: Examine the accompanying error messages for details, resolve the issues raised and retry, and if the error persists, contact Oracle Support Services.
