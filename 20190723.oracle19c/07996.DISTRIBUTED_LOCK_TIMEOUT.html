1.106 DISTRIBUTED_LOCK_TIMEOUT

DISTRIBUTED_LOCK_TIMEOUT specifies the amount of time (in seconds) for distributed transactions to wait for locked resources.

Property Description

Parameter type

Integer

Default value

60

Modifiable

No

Modifiable in a PDB

No

Range of values

1 to 231

Basic

No
