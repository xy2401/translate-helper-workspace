1.176 LOG_ARCHIVE_MAX_PROCESSES

LOG_ARCHIVE_MAX_PROCESSES specifies the maximum number of ARCn processes that can be created.

Property Description

Parameter type

Integer

Default value

4

Modifiable

ALTER SYSTEM

Modifiable in a PDB

No

Range of values

1 to 30

Basic

No
