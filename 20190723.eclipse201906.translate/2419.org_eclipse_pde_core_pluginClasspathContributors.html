插件Classpath贡献者

标识符：
org.eclipse.pde.core.pluginClasspathContributors

以来：
3.9

描述：
此扩展点提供了一种机制，可在计算PDE类路径时为插件项目提供其他类路径条目。在计算初始类路径时以及每次向项目添加新的bundle依赖项时，都会查询每个插件类路径贡献者以获取其他条目。只有项目类路径存在时，才会存储其他类路径条目。插件或功能导出期间不会考虑其他条目。

配置标记：

<！ELEMENT 扩展 （ 贡献者 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 贡献者 EMPTY>

<！ATTLIST撰稿人

class CDATA #REQUIRED>

  • class - 实现插件类路径贡献者的类。该类必须实现IClasspathContributor

例子：
以下是类路径贡献者的示例


  <extension
    point="org.eclipse.pde.core.pluginClasspathContributors">
    <contributor
      class="at.bestsolution.efxclipse.tooling.pde.core.JavaFXClassPathExtender">
    </contributor>
  </extension>

API信息：
每个贡献者必须提供一个实现org.eclipse.pde.core.IClasspathContributor
