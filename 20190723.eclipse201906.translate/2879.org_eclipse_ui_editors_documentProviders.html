文件提供者

标识符：
org.eclipse.ui.editors.documentProviders

以来：
3.0（最初名为org.eclipse.ui.documentProviders）

描述：
此扩展点用于定义文件类型和文档提供程序之间的映射，或者编辑器可以使用的编辑器输入和文档提供程序之间的映射。文档提供者必须实现org.eclipse.ui.texteditor接口。IDocumentProvider 。编辑器输入必须是org.eclipse.ui的实例。IEditorInput 。

配置标记：

<！ELEMENT 扩展 （ 提供者 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

扩展CDATA #IMPLIED

inputTypes CDATA #IMPLIED

class CDATA #REQUIRED

id CDATA #REQUIRED>

  • extensions - 以逗号分隔的文件扩展名列表
  • inputTypes - 以逗号分隔的限定编辑器输入类名列表
  • class - 文档提供程序类的限定名称
  • id - 此提供程序的唯一ID

例子：
<extension point = “org.eclipse.ui.editors.documentProviders” > <provider extensions = “jav” class = “org.eclipse.ui.examples.javaeditor。JavaDocumentProvider“ id = ”org.eclipse.ui.examples.javaeditor。JavaDocumentProvider“ > </ provider> </ extension>

此示例注册org.eclipse.ui.examples.javaeditor。JavaDocumentProvider作为扩展名为“.jav”的文件的默认提供程序。 

<extension point = “org.eclipse.ui.editors.documentProviders” > <provider inputTypes = “org.eclipse.ui。IStorageEditorInput“ class = ”org.eclipse.ui.editors.text。FileDocumentProvider“ id = ”org.eclipse.ui.editors.text。FileDocumentProvider“ > </ provider> </ extension>
此示例注册org.eclipse.ui.editors.text。FileDocumentProvider作为org.eclipse.ui实例的所有编辑器输入的默认提供程序。IStorageEditorInput 。

API信息：
为文件扩展名注册的文档提供者优先于为输入类型注册的文档提供者。文档提供者必须实现org.eclipse.ui.texteditor接口。IDocumentProvider 。编辑器输入必须是org.eclipse.ui的实例。IEditorInput 。
