添加服务器位置路径映射

此过程描述如何将路径映射添加到服务器，以便在PHP Web页面调试期间在本地位置搜索从服务器上某个位置调用的文件。

有关详细信息，请参阅路径映射

要将路径映射添加到服务器：

  1. 转到Window |打开PHP Servers首选项页面菜单栏上的首选项并选择PHP |来自首选项列表的PHP服务器。
  2. 选择要在其上创建路径映射的服务器，然后单击“编辑”。
  3. 在Edit Server对话框中，选择Path Mapping选项卡。
  4. 单击添加。
  5. 将出现“添加新路径映射”对话框。
  6. 输入要从中创建路径映射的服务器路径。将在下面指定的路径中搜索从此位置调用的文件。
  7. 选择“工作区中的路径”或“文件系统中的路径”选项，然后单击“浏览”以指定位置。

  8. 单击确定。您的路径图将添加到您的服务器列表中。


    下次从服务器上的路径调用文件时，将在您指定的本地位置搜索该文件。

注意：

路径映射也可以在调试期间自动设置。有关详细信息，请参阅路径映射主题。
