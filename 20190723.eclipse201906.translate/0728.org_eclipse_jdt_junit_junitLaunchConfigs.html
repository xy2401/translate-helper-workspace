JUnit启动配置

标识符：
org.eclipse.jdt.junit.junitLaunchConfigs

以来：
3.0

描述：
扩展点，用于注册需要在重构期间更新的基于JUnit的启动配置。

配置标记：

<！ELEMENT 扩展 （ launchConfigType ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT launchConfigType EMPTY>

<！ATTLIST launchConfigType

configTypeID IDREF #REQUIRED>

  • configTypeID - 启动配置类型的ID（请参阅扩展点org.eclipse.debug.core.launchConfigurationTypes）

例子：
以下是JUnit启动配置贡献的示例：

<extension point = “org.eclipse.jdt.junit.junitLaunchConfigs” > <launchConfigType configTypeID = “com.example。JunitLaunchConfig“ /> </ extension>
