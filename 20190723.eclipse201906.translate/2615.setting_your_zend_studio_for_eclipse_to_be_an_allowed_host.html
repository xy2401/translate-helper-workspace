将您的环境设置为允许的主机

此过程描述如何确保安装PDT的计算机将成为允许主机，以便在远程服务器上启动调试会话。

如果您的服务器上只安装了独立的Zend Debugger：

要配置调试器以允许PDT进行调试：

  1. 打开php.ini文件。
  2. 编辑zend_debugger.allow_hosts参数以包含安装PDT的计算机的IP地址。例如zend_debugger.allow_hosts = 127.0.0.1 / 32
  3. 确保该地址不在zend_debugger.deny_hosts参数列表中。
  4. 通过将zend_debugger.expose_remotely参数设置为Always，将Debug Server设置为将自身公开给远程客户端。 （例如zend_debugger.expose_remotely = always）。
  5. 保存文件。
  6. 重新启动Web服务器以使设置生效。
