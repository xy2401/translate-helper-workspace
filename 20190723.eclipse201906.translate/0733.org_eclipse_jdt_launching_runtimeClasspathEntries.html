Java运行时类路径条目

标识符：
org.eclipse.jdt.launching.runtimeClasspathEntries

以来：
3.0

描述：
这在内部扩展点中允许Java调试器扩展用于启动Java应用程序的运行时类路径条目集。客户端无意使用此扩展点。

配置标记：

<！ELEMENT 扩展 （ runtimeClasspathEntry *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT runtimeClasspathEntry EMPTY>

<！ATTLIST runtimeClasspathEntry

id CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 可用于引用此类型的运行时类路径条目的唯一标识符。
  • class - 实现此运行时类路径条目的类。该类必须实现IRuntimeClasspathEntry2 。

例子：
以下是解析器的示例：

<extension point = “org.eclipse.jdt.launching.runtimeClasspathEntries” > <runtimeClasspathEntry id = “com.example。EnvVarEntry“ class = ”com.example。EnvVarClasspathEntry“ > </ runtimeClasspathEntry> </ extension>

提供的实施：
为Java项目提供了一个实现，提供了org.eclipse.jdt.launching.classpathentry.project的标识符。Java项目类路径条目包括其构建路径上的所有引用。
