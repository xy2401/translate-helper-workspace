文件系统支持

标识符：
org.eclipse.ui.ide.filesystemSupport

以来：
3.2

描述：
filesystemSupport提供IDE贡献，以支持使用org.eclipse.core.filesystem.filesystems扩展点提供的文件系统。

配置标记：

<！ELEMENT 扩展 （ filesystemContributor *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT filesystemContributor EMPTY>

<！ATTLIST filesystemContributor

标签CDATA #REQUIRED

class CDATA #REQUIRED

scheme IDREF #REQUIRED>

  • label - 标签是此文件系统的人类可读标签。
  • class - class是org.eclipse.ui.ide.fileSystem。FileSystemContributor用于处理org.eclipse.core.filesystem的浏览和验证。IFileSystem。
  • scheme - 该方案是org.eclipse.core.filesystems.filesystem的方案，这个贡献对应于。

例子：


 <extension
         point="org.eclipse.ui.ide.filesystemSupport">
      <filesystemContributor       
        class="myFileSystemContributor"
            label="My file system"
            scheme="myFilesystem "/>
   </extension>
上面的示例有一个FileSystemContributor，它映射到它正在为其提供用户界面的org.eclipse.core.filesystems.filesystem。

API信息：
应该是org.eclipse.core.filesystems.filesystem方案与filesystemContributor的一对一映射。

提供的实施：
默认情况下没有文件系统支持。有一个内部实现支持由文件系统支持的IResource。
