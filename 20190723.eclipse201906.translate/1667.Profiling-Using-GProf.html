使用GProf进行性能分析

您也可以使用gprof插件分析应用程序，而不是运行应用程序并双击gmon.out文件。这将运行应用程序并为您显示结果。要使用gprof进行配置，您可以使用Profiling Tools-> Profile Timing并将计时工具设置为gprof。

在首选项中，您还可以转到：C / C ++ - > Profiling-> Categories-> Timing并将默认计时工具设置为gprof。

或者对于项目，您可以使用项目属性 - > C / C ++ General-> Profiling Categories-> Timing覆盖工作区首选项默认值

您还可以使用Profile as ...-> Local C / C ++ Application对您的应用程序进行概要分析，从而将Profiler选项卡中的概要分析工具设置为gprof。
