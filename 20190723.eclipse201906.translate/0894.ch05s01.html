5.1。要求

要安装和运行Jubula Extension Wizard，您需要：

  • Jubula 8.3.0或更高版本

  • Eclipse IDE for Java EE Developers或Eclipse Commiters，Version 4.5.2（Mars.2）或更高版本

  • JDK 8.0

  • 用于安装向导和解析扩展程序目标平台的Internet连接。
