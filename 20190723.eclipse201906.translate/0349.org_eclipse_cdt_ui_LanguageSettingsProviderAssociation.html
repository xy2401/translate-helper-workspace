语言设置提供程序UI关联

标识符：
org.eclipse.cdt.ui。LanguageSettingsProviderAssociation

以来：
CDT 8.1

描述：
此扩展点定义了使用扩展点org.eclipse.cdt.core.LanguageSettingsProvider 。

配置标记：

<！ELEMENT 扩展 （ id-association *， class-association *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

此扩展点用于定义用户界面中语言设置提供程序的外观和行为。


  • id - 未使用的扩展点的ID
  • name - 未使用的扩展点的名称

<！ELEMENT id-association EMPTY>

<！ATTLIST id-association

id CDATA #REQUIRED

icon CDATA #IMPLIED

页CDATA #IMPLIED

ui-edit-entries（true | false）

ui-clear-entries（true | false）>

与元素语言设置提供者相关联的UI元素的定义。


  • id - 正在定义外观的语言设置提供程序的ID。
  • icon - 定义插件中提供者图标的路径，例如icons / obj16 / picture.gif。
  • page - 提供程序的选项页面，显示在“提供程序”选项卡的首选项中。
  • ui-edit-entries - 定义是否允许用户在UI中编辑提供者的条目。
  • ui-clear-entries - 定义是否允许用户在UI中清除提供者的条目。对于某些提供商，例如可能触发自动重新运行的编译器规格检测器。

<！ELEMENT 类关联 EMPTY>

<！ATTLIST类关联

class CDATA #REQUIRED

icon CDATA #IMPLIED

页CDATA #IMPLIED

ui-edit-entries（true | false）

ui-clear-entries（true | false）>

与元素语言设置提供者相关联的UI元素的定义。从此类型继承的提供程序将继承最近的超类型的特征（除非定义了精确的ID关联）。


  • class - 正在定义外观的语言设置提供程序的类。
  • icon - 定义插件中提供者图标的路径，例如icons/obj16/picture.gif 。
  • page - 提供程序的选项页面，显示在“提供程序”选项卡的首选项中。
  • ui-edit-entries - 定义是否允许用户在UI中编辑提供者的条目。
  • ui-clear-entries - 定义是否允许用户在UI中清除提供者的条目。对于某些提供商，例如可能触发自动重新运行的编译器规格检测器。
