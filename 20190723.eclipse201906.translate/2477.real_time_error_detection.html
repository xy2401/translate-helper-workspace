实时错误检测

PDT会自动突出显示PHP脚本中的错误和问题。

错误和警告将显示在“问题”视图中，可从中访问 窗口|显示视图|其他|一般|问题 。

另外，错误图标和警告图标将显示在编辑器窗口左侧的垂直标记栏中，以及Project Explorer视图中相关项目旁边（这些将在PHP项目/文件图标中显示 - 例如 ）。

打开项目中的所有警告，错误和问题都将记录在“问题”视图中，该视图显示以下信息：

  • 描述 - 错误的详细描述。
  • 资源 - 包含问题的资源的名称。
  • Path - 包含资源的Project的名称。
  • 位置 - 文件中错误的行号。
  • 类型 - 发生的错误类型。

“问题”视图根据“错误”，“警告”或“信息”对问题进行分组。

双击“问题”视图中的错误将转到编辑器中的相关位置。

如果未显示“问题”视图，请转到“ 窗口”|显示视图|问题 。
