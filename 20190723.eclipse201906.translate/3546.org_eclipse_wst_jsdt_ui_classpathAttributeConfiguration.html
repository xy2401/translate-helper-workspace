IncludePath属性配置

标识符：
org.eclipse.wst.jsdt.ui.classpathAttributeConfiguration

以来：
3.3

描述：
此扩展点允许指定如何呈现和配置贡献的includepath属性。

配置标记：

<！ELEMENT 扩展 （ classpathAttributeConfiguration +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT classpathAttributeConfiguration EMPTY>

<！ATTLIST classpathAttributeConfiguration

attributeName CDATA #REQUIRED

class CDATA #REQUIRED>

  • attributeName - 标识IClasspathAttribute.getName（）中指定的includepath属性的名称
  • class - 实现此属性配置的类的名称。该类必须是公共的并扩展org.eclipse.wst.jsdt.ui.wizards.ClasspathAttributeConfiguration 。

例子：
以下是includepath条目容器页面的示例：


 <extension point="org.eclipse.wst.jsdt.ui.classpathAttributeConfiguration">
  <classpathAttributeConfiguration
   key="com.example.myplugin.myClasspathAttributeKey"
   name="Test property"
   class="com.example.MyAttributeConfiguration">
  </classpathAttributeConfiguration>
 </extension>
