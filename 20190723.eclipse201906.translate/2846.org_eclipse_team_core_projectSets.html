项目集

不推荐使用此扩展点

标识符：
org.eclipse.team.core.projectSets

以来：
2.0

描述：
此扩展点用于注册用于创建和读取项目集的处理程序。项目集是轻量级，可移植的方法，用于在工作空间中共享特定的团队共享项目阵容。项目集文件可用于为团队成员提供一种简单的方法来创建一个工作区，其中包含一个或多个团队提供者的特定项目阵容。提供商可以为此扩展点提供扩展。

不推荐使用：请参阅org.eclipse.team.core。RepositoryProviderType＃getProjectSetCapability

配置标记：

<！ELEMENT 扩展 （ projectSets *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED>


<！ELEMENT projectSets EMPTY>

<！ATTLIST projectSets

id CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 此处理程序为其创建和读取项目集的提供程序的自然ID。
  • class - 实现org.eclipse.team.core.IProjectSerializer 。

例子：
以下是projectSets扩展的示例：


 <extension point="org.eclipse.team.core.projectSets">
  <projectSets
   id="org.eclipse.team.cvs.core.cvsnature"
   class="org.eclipse.team.cvs.core.CVSProjectSetSerializer">
  </projectSets>
 </extension>
