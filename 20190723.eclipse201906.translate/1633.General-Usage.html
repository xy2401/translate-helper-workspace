一般用法

可以从C / C ++角度访问所有分析插件（包括Eclipse Callgraph）。有两种方法可以通过Profiling Tools项目菜单开始使用Eclipse Callgraph：

使用性能分析工具>函数调用图

或使用Profiling Tools> Profile Timing并将计时工具设置为Callgraph。

要正确设置首选项，您可以转到C / C ++ - > Profiling-> Categories-> Timing并将默认计时工具设置为Callgraph。

或者，您可以使用项目Properties-> C / C ++ General-> Profiling Categories-> Timing来设置项目特定首选项，以覆盖工作区首选项。

您还可以使用Profile as ...-> Local C / C ++ Application对您的应用程序进行概要分析，从而将Profiler选项卡中的概要分析工具设置为Callgraph。

这将打开一个对话框，您可以从中选择要分析的可执行文件。

选择要分析的可执行文件后，Eclipse Callgraph将询问要探测的文件。默认情况下，将选择项目中的所有源文件。
