JavaScript完成提案计算机

标识符：
org.eclipse.wst.jsdt.ui.javaCompletionProposalComputer

以来：
3.2

描述：
此扩展点允许贡献JavaScript完成提议计算机以参与JavaScript编辑器的内容辅助过程。

配置标记：

<！ELEMENT 扩展 （ javaCompletionProposalComputer | proposalCategory ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT javaCompletionProposalComputer （ partition ）*>

<！ATTLIST javaCompletionProposalComputer

class CDATA #REQUIRED

activate（true | false）“false”

categoryId CDATA“org.eclipse.wst.jsdt.ui.defaultProposalCategory”>

提案计算机贡献。如果未指定分区类型，则计算机将添加到所有分区类型。


  • class - 实现贡献计算机的类的名称。该类必须是公共的并实现org.eclipse.wst.jsdt.ui.text.java.IJavaCompletionProposalComputer并且必须具有公共0参数构造函数。
  • activate - 如果该属性设置为“true”，它将强制在内容辅助调用上加载此插件
  • categoryId - proposalCategory的id

<！ELEMENT 分区 EMPTY>

<！ATTLIST分区

type（__dftl_partition_content_type | __java_singleline_comment | __java_multiline_comment | __java_javadoc | __java_string | __java_character）>

  • type - 指定计算机可为其提供完成建议的JavaScript分区类型。见IDocument.DEFAULT_CONTENT_TYPE和IJavaPartitions用于有效值。

<！ELEMENT proposalCategory EMPTY>

<！ATTLIST proposalCategory

icon CDATA #IMPLIED>

提案类别贡献定义用于在UI中对其进行分组的提案类别。


  • icon - 类别的可选图标，可以在用户首选项中显示

例子：
以下是JavaScript完成建议计算机贡献的示例：


 <extension point="org.eclipse.wst.jsdt.ui.javaCompletionProposalComputer"
   id="textual_proposals"
   name="Text Proposals">
   <proposalCategory icon="icons/wordcompletions.png"/>
 </extension>
 <extension point="org.eclipse.wst.jsdt.ui.javaCompletionProposalComputer"
   id="WordCompletionProposalComputer"
   name="Word Completion Proposal Computer">
   <javaCompletionProposalComputer
      class="org.eclipse.wst.jsdt.internal.ui.text.java.HippieProposalComputer"
      categoryId="org.eclipse.ui.texteditor.textual_proposals">
      <partition type="__java_javadoc"/>
   </javaCompletionProposalComputer>
 </extension>

API信息：
贡献的类必须扩展org.eclipse.wst.jsdt.ui.text.java.IJavaCompletionProposalComputer

提供的实施：
请参阅org.eclipse.wst.jsdt.internal.ui.text.java.HippieProposalComputer为例。
