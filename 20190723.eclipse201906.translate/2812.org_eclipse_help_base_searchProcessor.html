搜索处理器

标识符：
org.eclipse.help.searchProcessor

以来：
3.7

描述：
此扩展点允许在用户提交帮助搜索时执行处理步骤。

配置标记：

<！ELEMENT 扩展 （ 处理器 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 处理器 EMPTY>

<！ATTLIST处理器

class CDATA #REQUIRED>

  • 课程 -

例子：
以下是对类'com.test.processor的定向处理示例。SearchProcessor”

<extension point = “org.eclipse.help.searchProcessor” > <processor class = “com.ibm.test.processor。SearchProcessor“ > </ processor> </ extension>

API信息：
提供的搜索处理器类必须扩展抽象类org.eclipse.help。AbstractSearchProcessor 。处理器有机会修改搜索字符串，提供备用搜索表达式或修改搜索结果集。

提供的实施：
没有。
