4.68. Scaling Factor

Parameter Name Data Type Possible Values Default
SCALING_FACTOR Integer   100
Table 4.69. SCALING_FACTOR

  • Use this parameter to scale the image created by the screenshot.

  • The value for this parameter must lie between 1 and 200 (inclusive.)

  • The units for this parameter are percent. This means that a value of 50 scales the image to half of its original size and a value of 200 scales the image to double its original size.
