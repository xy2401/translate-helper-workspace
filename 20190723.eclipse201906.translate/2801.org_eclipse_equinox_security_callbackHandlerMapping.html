JAAS Callback Handler Mapping

Identifier:
org.eclipse.equinox.security.callbackHandlerMapping

Description:
Maps a callback handler to a JAAS login configuration. The platform will automatically provide the specified callback handler to the JAAS login modules defined by the login configuration.

Configuration Markup:

<!ELEMENT extension (callbackHandlerMapping)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT callbackHandlerMapping EMPTY>

<!ATTLIST callbackHandlerMapping

configName        CDATA #REQUIRED

callbackHandlerId CDATA #REQUIRED>

  • configName - The name of the JAAS login configuration that will use this callbackHandler.
  • callbackHandlerId - The id of the callbackHandlers extension that defines the callback hander class.

Examples: 
  <extension
         name="HTTP CallbackHandler Mapping"
         point="org.eclipse.equinox.security.callbackHandlerMapping">
      <callbackHandlerMapping
            callbackHandlerId="org.eclipse.equinox.security.ui.defaultHttpBasicCallbackHandler"
            configName="HTTP"/>
   </extension>
