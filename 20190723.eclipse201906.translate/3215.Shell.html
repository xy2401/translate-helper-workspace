贝壳

SWT Shells（SWT中的窗口）。

可用的CSS属性

可以为此元素定义的属性：

  • 动画

    Shell出现和消失时触发的动画。支持的动画类型是“flyInTop”，“flyInBottom”，“flyInRight”，“flyInLeft”，“flyOutTop”，“flyOutBottom”，“flyOutRight”和“flyOutLeft”。

  • 背景颜色

    贝壳的背景颜色。

  • 背景图片

    背景图像或渐变。

  • 边界

    外窗边框。

  • 下边框

    出售的底边。

  • 左边框

    左侧边框出售。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    出售的右侧边框。

  • 边框顶部

    出售的顶边边框。

  • 箱阴影

    贝壳背后的影子。

  • 不透明度

    shell的不透明度。介于0和1之间的值。

  • 填充

    在shell的客户区域周围填充。

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [APPLICATION_MODAL]

    表示shell是模态的。

  • [边界]

    表示shell应该有边框。

  • [片]

    表示shell应该像表单窗口一样。

  • [标题]

    表示shell应该显示它的标题栏。

  • [工具]

    表示shell应该像工具窗口一样。

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：不活跃

    适用于除活动shell之外的所有shell。

  • ：最大化

    适用于当前最大化的shell。

壳牌标题栏

窗口标题栏。仅在设置TITLE样式时可见。

可用的CSS属性

可以为此元素定义的属性：

  • 背景颜色

    标题栏的背景颜色。分别映射到系统颜色SWT.COLOR_TITLE_BACKGROUND和SWT.COLOR_TITLE_INACTIVE_BACKGROUND。

  • 背景梯度色

    标题栏的第二种背景颜色（当前被忽略）。分别映射到系统颜色SWT.COLOR_TITLE_BACKGROUND_GRADIENT和SWT.COLOR_TITLE_INACTIVE_BACKGROUND_GRADIENT。

  • 背景图片

    标题栏的背景图像。

  • 边界

    Shell标题栏边框。

  • 下边框

    shell标题栏的底边框。

  • 左边框

    shell标题栏的左侧边框。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    shell标题栏的右侧边框。

  • 边框顶部

    shell标题栏的顶部边框。

  • 颜色

    标题栏的文字颜色。

  • 字形

    标题栏的字体。

  • 高度

    标题栏的高度。

  • 余量

    标题栏的保证金。

  • 填充

    填充标题栏。

  • 文字阴影

    标题栏文本的阴影。

壳牌DisplayOverlay

覆盖在（关于z顺序）显示区域和模态外壳之间的覆盖。

可用的CSS属性

可以为此元素定义的属性：

  • 动画

    支持动画类型“fadeIn”和“fadeOut”。其他人被忽略了。

  • 背景颜色

    叠加层的背景颜色。

  • 背景图片

    覆盖的背景图像。

  • 不透明度

    叠加层的不透明度。介于0和1之间的值。

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [边界]

    表示模态shell具有BORDER样式。

  • [片]

    表示模态shell具有SHEET样式。

  • [标题]

    表示模态shell具有TITLE样式。

  • [工具]

    表示模态shell具有TOOL样式。
