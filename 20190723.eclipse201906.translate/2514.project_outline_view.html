PHP项目大纲视图

PHP Project Outline视图显示所选项目中所有文件的Soap Clients，Constants，Classes和Functions列表。

在PHP Project Outline视图中选择元素将在编辑器中打开相关文件。

要访问视图，请转到Window |显示视图|其他| PHP工具| PHP项目大纲。

PHP项目大纲视图工具栏命令

图标 名称 描述
全部收缩 折叠元素列表
与编辑链接 如果选中，单击时元素将立即显示在编辑器中。如果未选中，则双击时元素将显示在编辑器中。
