要运行Fortran应用程序，请在Fortran Projects视图中右键单击该项目，然后单击Run As> Local Fortran Application。这将构建并运行Fortran应用程序，输出将打印到控制台。（请记住，如果您使用的是自定义makefile，则需要为要构建的项目正确编写makefile。）

如果要管理运行配置（例如，如果要将命令行参数传递给Fortran应用程序或设置某些环境变量），请在“Fortran项目”视图中右键单击项目，单击“运行方式”>“运行配置”。 ..在Fortran Local Application下创建新的启动配置（或编辑现有的配置）并根据需要修改设置。

除了可以从上下文菜单中选择Debug As> Local Fortran Application或Debug Configurations之外，调试程序的工作方式类似。

运行和调试Fortran应用程序类似于运行和调试C / C ++应用程序。 有关运行和调试二进制应用程序的更多信息，请单击Eclipse中的“帮助”>“帮助内容”，然后导航到“ C / C ++开发用户指南”>“任务”>“运行和调试项目”。

设置断点

有两种方法可以在编辑器中设置断点。

在编辑器中设置断点的最简单方法是双击要设置断点的行的左边距（灰色垂直条）。在设置断点的行的边距中将出现一个蓝点。在同一位置再次双击将删除断点。

您还可以通过右键单击要设置断点的行的左边距并单击“切换断点”来设置断点。指示断点的蓝点将在编辑器的左边缘出现或消失（如果已设置）。

切换到Debug透视图时，您设置的断点将在Breakpoints视图中列出。

有关调试的更多信息，请单击Eclipse中的“帮助”>“帮助内容”，然后导航到“ C / C ++开发用户指南”>“任务”>“运行和调试项目”>“调试”。
