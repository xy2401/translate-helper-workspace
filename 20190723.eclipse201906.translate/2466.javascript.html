JavaScript支持

JavaScript是一种脚本语言，旨在为HTML页面添加交互性。

PDT为独立JavaScript文件以及PHP文件和项目中的JavaScript功能提供支持。

PHP项目和文件的一些JavaScript功能包括：

  • JavaScript元素大纲 - JavaScript元素将显示在“ 大纲”视图中 。
  • JavaScript库将添加到已启用JavaScript支持的Include Paths of PHP项目中。
  • JavaScript构建路径 - 为项目设置JavaScript构建路径将允许您确定哪些元素将包含在项目的构建过程中。这允许从项目本身以外的位置的资源访问内容辅助选项。
  • JavaScript编辑器中的高级编辑功能。例如：
  • 拖放行为，允许您选择代码块并跨行移动它。
  • JavaScript内容辅助
  • JavaScript语法着色
  • JavaScript 代码评论
  • JavaScript 悬停支持
  • JavaScript Mark Occurrences
  • JavaScript Smart Goto来源

可以将JavaScript支持添加到新的或现有的PHP项目中，以便项目可以使用所有PDT的JavaScript功能。

有关为JavaScript开发启用的功能的完整列表，请参阅JavaScript Development Toolkit用户指南中的功能 - Javascript Developtment Toolkit（JSDT） 。

注意：

通过转到Help |，可以从PDT内部访问其他用户指南帮助内容 ，或者来自Eclipse在线文档站点（ http://help.eclipse.org/luna/index.jsp ）。
