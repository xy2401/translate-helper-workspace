标识符：
org.eclipse.ui.views.properties.tabbed.propertyContributor

描述：
描述工作台部件，该工作台部件是要在选项卡式属性表页面的选项卡中显示的属性的参与者。

配置标记：

<！ELEMENT 扩展 （ propertyContributor ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT propertyContributor （ propertyCategory +）>

<！ATTLIST propertyContributor

contributorId CDATA #REQUIRED

typeMapper CDATA #IMPLIED

labelProvider CDATA #IMPLIED

actionProvider CDATA #IMPLIED

sectionDescriptorProvider CDATA #IMPLIED

tabDescriptorProvider CDATA #IMPLIED

overridableTabListContentProvider（true | false）>

  • contributorId - 属性表页面参与者的唯一标识符。
  • typeMapper - 实现类型映射器的类，即实现org.eclipse.ui.views.properties.tabbed。ITypeMapper
  • labelProvider - 实现属性表页面标题的标签提供程序的类，即实现org.eclipse.jface.viewers。ILabelProvider
  • actionProvider - 实现属性表页面的操作提供程序的类，即实现org.eclipse.ui.views.properties.tabbed。IActionProvider
  • sectionDescriptorProvider - 实现属性表页面的节描述符提供程序的类，即实现org.eclipse.ui.views.properties.tabbed。ISectionDescriptorProvider
  • tabDescriptorProvider - 实现属性表页面的选项卡描述符提供程序的类，即实现org.eclipse.ui.views.properties.tabbed。ITabDescriptorProvider
  • overridableTabListContentProvider - 如果为true，则此参与者将使用扩展org.eclipse.ui.views.properties.tabbed的部分覆盖选项卡列表。AbstractOverridableTabListPropertySection

<！ELEMENT propertyCategory EMPTY>

<！ATTLIST propertyCategory

类别CDATA #IMPLIED>

  • category - 用于对选项卡进行分组的类别。

例子：

   <extension
         point="org.eclipse.ui.views.properties.tabbed.propertyContributor">
      <propertyContributor
            contributorId="org.eclipse.ui.tests.views.properties.tabbed.views.TestsView"
            labelProvider="org.eclipse.ui.tests.views.properties.tabbed.views.LabelProvider"
            typeMapper="org.eclipse.ui.tests.views.properties.tabbed.views.TypeMapper">
         <propertyCategory category="first">
         </propertyCategory>
         <propertyCategory category="second">
         </propertyCategory>
      </propertyContributor>
   </extension>
