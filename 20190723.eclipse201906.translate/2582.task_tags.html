任务标签首选项

“任务标签”首选项页面允许您添加新任务标签并编辑现有标签。任务用作提醒操作，工作或程序员要求的任何其他操作。任务标记是PDT使用的字符串，用于识别脚本中添加任务的时间。注释中的这些字符串之后的任何内容都将被识别为任务。

有关使用任务的详细信息，请参阅“Workbench用户指南”中的“ 任务视图 ”主题。

注意：

通过转到Help |，可以从PDT内部访问其他用户指南帮助内容 ，或者来自Eclipse在线文档站点（ http://help.eclipse.org/luna/index.jsp ）。

可从Window |访问“任务标记首选项”页面偏好| PHP |编辑|任务标签。

默认情况下，列表中包含四个常用字符串。

要添加新的任务标记：

  1. 单击新建。
  2. 输入标签名称和优先级（高/正常/低）。标签可能包含任何字符串。
  3. 单击确定。
  4. 单击“应用”以应用设置。

新标记将添加到列表中，并在插入编辑器时触发任务。

要编辑标签：

  1. 双击标签 - 或 - 选择它，然后单击编辑。
  2. 编辑标签名称或优先级。
  3. 单击确定。
  4. 单击“应用”以应用设置。

选择标记并单击“默认”会将任务标记设置为要在代码模板中使用的默认标记。有关模板首选项的更多信息，请参阅模板首选项

注意：

如果标记了“区分大小写的任务标记名称”复选框，则任务标记名称将区分大小写。

仅将“任务标记”设置应用于特定项目：

  1. 选择标记为“配置项目特定设置”的链接。
  2. 从列表中选择所需的项目。将出现“任务标记属性”对话框。
  3. 选择所需设置，然后单击“应用”。将出现一个提示对话框，指出必须重建项目才能使设置生效。
  4. 单击是以重建项目 - 或 - 单击否以仅在重新启动PDT时执行重建。 - 或 - 单击“取消”取消操作。
