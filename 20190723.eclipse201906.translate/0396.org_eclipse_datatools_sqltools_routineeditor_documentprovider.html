SQL Editor文档提供程序

标识符：
com.sybase.stf.dmp.editor.documentprovider

以来：
0.7.0

描述：
此扩展点允许为SQL Routine编辑器提供文档提供程序。扩展名注册了org.eclipse.datatools.sqltools.routineeditor.IRoutineEditorDocumentProvider类，它将提供原始文档提供程序之外的其他功能。注意：此扩展点仅供DTP SQL调试器使用。

配置标记：

<！ELEMENT 扩展 （ 提供者 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

此扩展点应仅由调试器使用。



<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

id CDATA #IMPLIED

class CDATA #REQUIRED>

  • id - 可选的id
  • class - 实现org.eclipse.datatools.sqltools.routineeditor的Java类的完全限定名称。IRoutineEditorDocumentProvider。

例子：

<extension
          point=
"org.eclipse.datatools.sqltools.routineeditor.documentprovider"
>

       
<provider class=
"org.eclipse.datatools.sqltools.debugger.editorextension.SQLDebuggerDocumentProvider"
/>

    
</extension>
SQLEditorDocumentProvider将使用SQLDebuggerDocumentProvider来管理断点位置。

API信息：
class属性必须是实现org.eclipse.datatools.sqltools.routineeditor的Java类的完全限定名。IRoutineEditorDocumentProvider。公共接口IRoutineEditorDocumentProvider扩展了IDocumentProvider，IDocumentProviderExtension2，IDocumentProviderExtension3 {/ ** *刷新数据库中的例程对象。* @param元素例程对象* @param controlCon控件连接由例程编辑器使用* @param proc例程标识符* / public void refreshFromDatabase（Object元素，IControlConnection controlCon，ProcIdentifier proc）抛出CoreException，SQLException; }
