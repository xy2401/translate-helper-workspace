聊天室消息处理程序

标识符：
org.eclipse.ecf.presence.bot.chatRoomMessageHandler

以来：
1.0.0里程碑6

描述：
此扩展点允许消息处理程序与给定的chatroomrobot关联（通过chatroomrobotid属性）。请注意，chatroomrobotid必须存在或不会调用处理程序。

配置标记：

<！ELEMENT 扩展 （ 处理程序 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED

>


<！ELEMENT 处理程序 EMPTY>

<！ATTLIST处理程序

chatRoomRobotId CDATA #REQUIRED

filterExpression CDATA #IMPLIED

class CDATA #REQUIRED

>

聊天室消息处理程序


  • chatRoomRobotId - 必需的chatRoomRobotId。该字段的值必须与chatRoomRobot id匹配
  • filterExpression - 可选的过滤器表达式。如果提供了filterexpression，则将通过给定的正则表达式过滤聊天室消息。如果匹配，则调用handleChatRoomMessage方法。如果不匹配，则将忽略收到的消息。
  • class - 要创建的消息处理程序实例的类。必须实现org.eclipse.ecf.presence.bot。IChatRoomMessageHandler 。

例子：
<extension point = “org.eclipse.ecf.presence.bot.chatRoomMessageHandler” > <handler chatRoomRobotId = “org.eclipse.ecf.presence.bot.defaultbot” class = “org.eclipse.ecf.presence.bot。DefaultChatRoomMessageHandler“ > </ handler> </ extension>
请注意，必须先通过chatroomrobot扩展点定义给定的chatroomrobotid = org.eclipse.ecf.presence.bot.defaultbot 。这是一个例子：
<extension point = “org.eclipse.ecf.presence.bot.chatRoomRobot” > <chatRoomRobot chat chatRoom = “＃eclipse-ecf” connectId = “irc：//slewis2_@irc.freenode.net” containerFactoryName = “ecf.irc .irclib“ id = ”org.eclipse.ecf.presence.bot.defaultbot“ > </ chatRoomRobot> </ extension>
