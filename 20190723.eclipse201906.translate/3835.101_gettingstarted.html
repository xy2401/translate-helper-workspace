你好，世界

您希望在任何语言中看到的第一件事是Hello World示例。在Xtend中，读作为

class HelloWorld {def static void main（String [] args）{println（“Hello World”）}}

你看到Xtend看起来非常类似于Java。乍一看，主要区别似乎是声明方法的def关键字。与Java类似，必须将类和主方法定义为应用程序的入口点。诚然， Hello World程序并不是Xtend的特别优势。当你做真实的事情时，真正的表现力就会被释放，你会在一秒钟内学会。

Xtend类驻留在一个普通的Eclipse Java项目中。一旦安装了SDK，Eclipse就会自动将所有类转换为Java源代码。默认情况下，您将在源文件夹xtend-gen中找到它。hello world示例转换为以下Java代码：

//生成的Java源代码导入org.eclipse.xtext.xbase.lib。输入输出; public class HelloWorld {public static void main（final String [] args）{InputOutput。<String> println（“Hello World”）; }}

生成的Java代码中唯一令人惊讶的事实可能是引用的库类InputOutput 。它是运行时库的一部分，是一个很好的实用程序，在表达式中使用时非常方便。

您可以将Xtend类放入Eclipse或任何Maven项目中的任何Java项目的源文件夹中。如果项目尚未正确配置，Eclipse将会抱怨缺少的库。xtend.lib必须位于类路径上。IDE将提供快速修复以添加它。

您可能想要做的下一件事是将一个示例项目实现到您的工作区中。右键单击Eclipse中Navigator视图中的任意位置，然后选择New→Example ....

在即将到来的对话框中，您将找到Xtend的两个示例：

  • Xtend Introductory Examples包含几个示例代码片段，说明了Xtend的某些方面和优势。例如，它展示了如何构建一个允许编写如下代码的API：

    assertEquals（42.km / h，（40_000.m + 2.km）/ 60.min）

    下一节中详细解释的电影示例也包含在那里。

  • Xtend Solutions For Euler包含了您在Project Euler中可以找到的一些问题的解决方案。这些例子充分利用了Xtend的整体表现力。例如， Euler Problem 1可以用这个表达式解决：

    （1..999）.filter [i | i％3 == 0 || i％5 == 0] .reduce [i1，i2 | i1 + i2]
