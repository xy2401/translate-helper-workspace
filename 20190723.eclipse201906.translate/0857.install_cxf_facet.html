Install the CXF Project Facet

There are three ways to enable a Dynamic Web Project for CXF Web services development.

  1. Let the Web Services wizards do it for you automatically when creating a Web service with Apache CXF.
  2. Select a CXF project configuration or modify another configuration when creating a new Dynamic Web Project

  3. Right-click on a Dynamic Web Project and choose Properties from the context menu.

    On the the Project Facets properties page select the CXF 2.x Web Services facet in the list and click OK.
