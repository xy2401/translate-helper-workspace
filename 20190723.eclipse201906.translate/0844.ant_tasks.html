使用Apache CXF和Ant任务创建Web服务和客户端

除了使用Web服务向导之外，还可以生成Ant文件，您可以使用它来使用Apache CXF运行时环境创建Web服务和客户端。Ant任务可以使用Run As> Ant Build ...在Eclipse工作区中运行，也可以通过命令行无头运行。

先决条件：

  1. 安装和配置Apache CXF运行时环境。有关详细信息，请参阅“ 要求和安装”页面。
  2. 为Web服务或客户端创建动态Web项目，并按照此处所述为Apache CXF Web服务开发启用它：为CXF Web服务创建项目 。
  3. 将所需的Ant文件导入工作区：
    1. 使用上面创建的现有Dynamic Web Project或创建新项目。
    2. 选择导入Ant文件 文件>新建>其他...> Web服务> Ant顶级菜单中的文件 。
    3. 选择以下内容：
      • Apache CXF作为Web服务运行时。
      • Web服务或客户端类型。
      • 将生成Ant文件的文件夹。

      单击完成 。该向导将在所选文件夹中生成两个文件。

      1. Ant文件： cxf_bujava.xml （自底向上）， cxf_tdjava.xml （自上而下）或cxf_client.xml （客户端）之一。
      2. Ant文件引用的Ant属性文件： cxf_bujava.properties （自下而上）， cxf_tdjava.properties （自上而下）或cxf_client.properties （客户端）之一
