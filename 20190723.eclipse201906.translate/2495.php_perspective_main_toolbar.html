PHP Perspective主工具栏

PHP Perspective的主工具栏提供了常用功能的快捷方式：

快捷方式图标快捷键名称描述
新单击该图标将打开“新建向导”对话框。

单击图标旁边的箭头可以选择创建新的PHP项目，项目，PHP文件，文件夹，CSS，HTML，XML，示例项目或其他资源。
按Ctrl + S保存保存活动文件。
按Ctrl + P打印打印活动文件。
调试单击“调试”按钮将执行上次运行配置。

单击图标旁边的箭头可访问以下选项：
  • 调试先前执行的启动配置。
  • Debug As ... - 将活动文件调试为服务器上的调试，Java Applet，Java应用程序，JUnit插件测试，JUnit测试，PHP脚本或PHP Web页面。
  • “打开调试”对话框 - 打开“调试”对话框。
  • 整理收藏夹 - 允许您选择应将哪些启动配置添加到收藏夹列表中。您最喜爱的启动将首先列在启动配置列表中。
跑单击“运行方式”按钮将执行上次运行配置。

单击图标旁边的箭头可访问以下选项：
  • 运行先前执行的启动配置。
  • Run As ... - 将活动文件作为Run on Server，Java Applet，Java Application，JUnit Plug-in Test，JUnit Test，PHP Script或PHP Web Page运行。
  • “打开运行”对话框 - 打开“运行”对话框。
  • 整理收藏夹 - 允许您选择应将哪些启动配置添加到收藏夹列表中。您最喜爱的启动将首先列在启动配置列表中。
外部工具单击“外部工具”按钮将打开“外部工具配置”对话框。

单击图标旁边的箭头可访问以下选项：
  • 运行方式 - 如果适用，允许您使用外部工具运行该文件。
  • 打开外部工具对话框 - 打开配置对话框，以使用外部工具运行文件。
  • 整理收藏夹 - 打开一个对话框，允许您组织外部工具。
有关详细信息，请参阅“Workbench用户指南”中的“ 外部工具”主题。
按Ctrl + H搜索启动“搜索”对话框。
打开Web浏览器打开内部Web浏览器。
按Ctrl +。按Ctrl +，下一个/上一个注释导航到脚本中的下一个/上一个注释。

可能的注释包括：书签，差异添加，差异更改，错误，信息，搜索结果，拼写错误，任务和警告。

单击工具栏上下一个/上一个注释图标旁边的箭头，以配置应包含哪些类型的注释。
按Ctrl + Q最后编辑位置跳转到已编辑的最后一个位置。
Alt + Left Alt + Right后退/前进到最后编辑的文件滚动当前会话中所有已编辑文件中的上一个/下一个编辑位置。
