属性页 - JPA图编辑器选项

此表列出了JPA Diagram Editor Settings页面中可用的属性。

属性 说明/动作
图表的文件夹 指定将创建diagram.xmi文件的文件夹。
默认的Java包 指定默认情况下将创建持久性类型的Java包。
默认表名前缀 指定默认情况下将用于创建表的前缀。
编辑持久性类型名称 确定图中持久性类型名称的编辑是否应该影响源文件中的类名。
持久访问类型 指定持久访问类型。

  • 要在字段级别添加注释，请选择“ 基于字段” 。

  • 要在属性级别添加注释，请选择“基于属性” 。

默认集合类型 指定默认情况下要创建的关系属性的集合类型：

选择java.util。集合 ，使关系属性为Collection类型。

一对多单向旧式关系 确定是否使用旧式的One-to-Many单向关系（由JPA 1.0支持）。
XML实体映射 确定是否应在指定的orm.xml文件中添加持久性类型。
