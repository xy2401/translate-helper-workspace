代码模板首选项

“代码模板首选项”页面允许您配置为不同类型的元素自动创建的代码和注释。

可以从Window |访问“代码模板首选项”页面偏好| PHP |代码风格|代码模板 。

要编辑注释或代码元素的模式：

  1. 展开列表并选择所需的元素。
  2. 单击编辑 。将显示“编辑模板”对话框。
  3. 根据需要编辑模式，然后单击“ 确定” 。
  4. 如果需要，请标记“自动为新方法和类型添加注释”框，以便在创建新方法和类型时自动生成注释。
  5. 单击“ 应用”以应用更改。

导出和导入代码模板

PDT使您可以导出和导入代码模板，这些模板在XML文件中以以下格式创建：

<？xml version =“1.0”encoding =“UTF-8”standalone =“no”？> <templates> <template autoinsert =“true”context =“php”deleted =“false”description =“description”enabled =“true”id =“org.eclipse.php.ui.editor.templates.php.codetemplates。 xxxcomment“name =”name“> / ** * $ {tags} * / </ template> </ templates>

要导入代码模板：

  1. 单击“导入”以打开“导入模板”浏览器。
  2. 选择包含模板信息的相关XML文件。
  3. 单击打开 。

template.xml文件中包含的模板将导入到模板列表中。

要导出代码模板：

  1. 从“模板”列表中选择要导出的模板。
  2. 单击导出打开“导出模板”对话框。
  3. 选择要将XML文件保存到的位置。
  4. 单击保存 。

将使用模板信息创建XML文件。

要导出所有代码模板：

  1. 单击全部导出 。
  2. 选择要将XML文件保存到的位置。
  3. 单击保存 。

将使用模板信息创建XML文件。

注意：

如果您选择了多个要导出的模板，则所有模板都将出现在导出的XML文件中。每个原始模板都受以下内容限制： <template></template>
