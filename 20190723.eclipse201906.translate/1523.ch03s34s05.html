3.34.5. Changing perspectives

  1. To change between perspectives, select:

    Window-->Open Perspective.

  2. Select which perspective to open.

  3. Alternatively, you can use the icons in the top left-hand corner of the current perspective to toggle between perspectives.

3.34.5.1. Automatically changing perspective

By selecting:

Window-->Preferences,

you can choose to automatically change to the Execution Perspective when test execution starts, or to be asked each time a Test Suite is started.
