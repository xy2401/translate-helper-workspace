来源菜单

“源”菜单允许您通过添加或删除注释以及格式化脚本来组织脚本，以使其更易于查看。

“源”菜单中提供的选项包括：

名称捷径描述
切换评论按Ctrl + /
-要么-
按Ctrl + 7		通过添加或删除“//”字符来注释或取消注释一行。
注释用于向脚本添加文本以解释代码段。评论文本不会作为代码的一部分运行。
要使用此功能，请选择该行并按Ctrl + /。
有关更多信息，请参阅“ 评论代码” 。
添加阻止评论按Ctrl + Shift + /通过在代码的任一侧添加“/ *”和“* /”字符来注释块。
要使用此功能，请选择块并按Ctrl + Shift + /。
有关更多信息，请参阅“ 评论代码” 。
删除块注释按Ctrl + Shift + \删除块注释。
要使用此功能，请将光标放在注释中的任意位置，然后单击Ctrl + Shift + \。
有关更多信息，请参阅“ 评论代码” 。
格式文档按Ctrl + Shift + F自动格式化脚本以将其组织为易于阅读的格式。
要格式化代码，请将光标放在编辑器视图中的任意位置，然后按Ctrl + Shift + F.将添加适当的换行符和缩进。
您可以通过可从Window |访问的Formatter Preferences页面配置自动格式化选项偏好| PHP |格式化。有关更多信息，请参阅格式化代码 。
格式化活动元素按Ctrl + I仅格式化选定的代码。
要格式化活动元素，请选择要格式化的所需代码，然后按Ctrl + I.
将在活动元素中添加适当的换行符和缩进。
您可以通过可从Window |访问的Formatter Preferences页面配置自动格式化选项偏好| PHP |格式化。
有关更多信息，请参阅格式化代码 。
