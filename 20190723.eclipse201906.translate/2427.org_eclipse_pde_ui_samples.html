样品

标识符：
org.eclipse.pde.ui.samples

以来：
3.0

描述：

此扩展点应用于注册Eclipse代码示例。示例是一个或多个Eclipse项目，这些项目以适合后续导入工作空间的方式打包。

每个样本都有一个名称，一个或多个项目元素和最终视角。示例向导将为每个project元素创建一个项目，从归档中导入内容，并在指定时打开最终透视图。项目必须完整，即具有正确的性质和类路径（如果是Java或插件项目）。

如果指定了目标视图并包含资源元素，则在创建样本后，它将用于在视图中选择和显示这些资源。

此扩展点目前仅供内部Eclipse SDK使用。它不应该用于除Eclipse SDK示例之外的任何其他内容。

配置标记：

<！ELEMENT 扩展 （ 样本 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 扩展点属性 - 在本例中为org.eclipse.pde.ui.samples 。
  • id - 未使用。
  • 名称 - 未使用。

<！ELEMENT 样本 （ 描述 ， 项目 +， 活动 *）>

<！ATTLIST样本

id IDREF #REQUIRED

名称CDATA #REQUIRED

perspectiveId IDREF #IMPLIED

发射器CDATA #IMPLIED

targetViewId IDREF #IMPLIED>

表示要导入工作空间的单个样本的元素。


  • id - 样本的唯一标识符
  • name - 样本的可翻译名称
  • perspectiveId - 如果指定，示例向导将切换到操作结束时提供的id的透视图。
  • 启动器 - 用于运行示例的可选启动快捷方式。如果未指定，则假定需要启动新的Eclipse实例才能查看示例。如果指定，则它必须是实现org.eclipse.debug.ui.的Java类的完全限定名称org.eclipse.debug.ui.ILaunchShortcut界面。
  • targetViewId - 一个可选属性，指示工作台中的目标视图，其中包含在创建样本后选择和显示的资源。视图必须实现org.eclipse.ui.part.ISetSelectionTarget接口。将在目标视图中选择并显示project元素中列出的资源。

<！ELEMENT 项目 EMPTY>

<！ATTLIST项目

名称CDATA #REQUIRED

归档CDATA #REQUIRED

共享（true | false）

selectReveal CDATA #IMPLIED>

每个项目元素将导致使用从关联存档导入的内容创建工作空间项目。


  • name - 要在此示例的工作空间中创建的项目的名称
  • archive - 包含要导入的项目内容的ZIP文件的相对路径
  • 共享 - 共享项目可能会列在多个样本中。当此属性为true且工作空间中已存在具有相同名称的项目时，将以静默方式跳过该属性。否则，将提示用户提供覆盖权限。
  • selectReveal - 在创建后在目标视图中选择和显示的资源的项目相对路径。

<！ELEMENT description EMPTY>

<！ATTLIST描述

helpHref CDATA #IMPLIED>

此元素的正文文本应包含示例的描述。


  • helpHref - “帮助”中主题的可选引用，提供有关示例的更多信息。

<！ELEMENT 活动 EMPTY>

<！ATTLIST活动

id CDATA #REQUIRED>

指定需要为此示例启用的活动


  • id - 用于启用此样本的活动的活动标识符

例子：

以下是扩展点的示例：

<extension point = “org.eclipse.pde.ui.samples” > <sample id = “com.example.xyz.sample” name = “My Sample” > <description helpHref = “/ com.example.xyz.doc / samples / com.example.xyz.sample1 / doc-html / doc.html“ >示例的简短描述。</ description> <project name = “com.example.xyz.sample1” archive = “samples / sample1.zip” /> </ sample> </ extension>

API信息：

如果指定，则属性launcher的值必须表示实现org.eclipse.debug.ui.的Java类的完全限定名称org.eclipse.debug.ui.ILaunchShortcut界面。

提供的实施：

PDE UI提供Runtime Workbench启动程序，用于启动另一个Eclipse实例。
