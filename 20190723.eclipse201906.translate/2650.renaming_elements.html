重命名元素

此过程描述如何重命名PHP元素并确保更新对该元素的所有引用。
可以从Project Explorer视图重命名和重构所有PHP元素。以下是适用的PHP元素列表：

  • 类
  • 接口
  • 变量
  • 方法
  • 功能
  • 常量
  • 班级成员

注意：

在应用重构功能之前，请确保保存对文件的任何更改。

如果激活就地重构功能（不显示更改预览）或从“ 重命名元素”对话框中，可以从编辑器本身重命名元素 。

在编辑器中重命名元素

要在编辑器中重命名元素：

  1. 确保“在没有对话框的编辑器中重命名”复选框在代码重构首选项页面中标记（可从Window | Preferences | PHP | Code Refactor访问 ）。
  2. 在编辑器中，将光标放在要重命名的元素上。
  3. 从菜单栏中选择Refactor |重命名 - 或 - 右键单击并选择Refactor |重命名 - 或 - 按Alt-Shift-R 。
    所有出现的元素都放在一个框架中，并显示Refactor弹出窗口。
  4. 在框中键入新元素名称。
    所有出现的元素名称都会自动更新。
    注意：要在应用更改之前预览更改，请单击“重构”弹出窗口右侧的箭头，然后选择“ 预览” 。
  5. 单击Enter以应用重构。
    将显示一个对话框，提示您在应用重构之前保存文件。
  6. 选中“始终在重构之前自动保存所有已修改的资源”复选框，以便不再显示该对话框。
  7. 单击确定 。

将重命名该元素，并且将更新引用该元素的所有实例以反映更改。

通过重构对话框重命名元素

要通过“重构”对话框重命名元素：

  1. 确保“在没有对话框的编辑器中重命名”复选框未在“ 代码重构首选项”页面中标记（可从“ 窗口”|“首选项”|“PHP |代码重构”访问 ）。
  2. 在编辑器中，将光标放在要重命名的元素上，或在Project Explorer中选择它。
  3. 从菜单栏中选择Refactor |重命名 - 或 - 右键单击并选择Refactor |重命名 - 或 - 按Alt-Shift-R 。
    将显示“重命名”对话框。对话框的名称取决于元素类型。
  4. 输入元素的新名称。您必须输入所需元素的有效名称 - 即以字母或下划线开头，后跟任意数量的字母，数字或下划线。
  5. 如果希望在引用它的所有注释和字符串中更新元素的名称，请选中“更新注释和字符串中的文本实例”框。
    这将强制您在应用之前预览更改。
  6. 如果要查看此重构将创建的更改的预览，请单击“ 确定”以应用更改，或单击“ 预览” 。
  7. 如果单击预览，将打开预览窗口，其中的更改树将显示将反映元素重命名的所有更改。
    将根据它们出现的上下文列出更改。因此，您可以展开节点以查看特定文件，类或函数中的所有更改。
  8. 使用“下一个/上一个更改”箭头滚动浏览所有可能的更改。
    取消标记更改旁边的复选框将导致这些更改无法生效。
    请注意，如果将在引用要重构的元素的其他文件中进行更改，则更改也将在此处的文件名下列出。
  9. 要应用的更改将显示在底部窗格中。
    您可以使用滚动箭头滚动浏览不同的更改： - 下一个/上一个差异滚动箭头 - 滚动浏览要在顶部窗格中选定的元素中应用的更改。- 下一个/上一个更改滚动箭头 - 滚动浏览要应用的所有更改。如果在顶部窗格中取消标记更改，则在使用这些箭头时不会显示这些更改。
  10. 对更改感到满意后，单击“ 确定” 。

将重命名该元素，并且将更新引用该元素的所有实例以反映更改。
