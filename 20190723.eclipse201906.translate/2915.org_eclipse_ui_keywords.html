关键词

标识符：
org.eclipse.ui.keywords

以来：
3.1

描述：
关键字扩展点定义关键字和唯一ID以供其他模式引用。请参见propertyPages，newWizards，importWizards，exportWizards和preferencePages。

配置标记：

<！ELEMENT 扩展 （ 关键字 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 关键字 EMPTY>

<！ATTLIST关键字

id CDATA #REQUIRED

标签CDATA #REQUIRED>

  • id - id是用于引用关键字的唯一ID。
  • label - 关键字的人类可读标签

例子：
以下是关键字扩展的示例：


   <extension
     point="org.eclipse.ui.keywords">
    <keyword
            label="presentation tab themes"
            id="com.xyz.AppearanceKeywords"/>
   </extension>

API信息：
目前没有用于检索或创建此扩展点以外的关键字的公共API。

提供的实施：
关键字与工作台向导（新建，导入和导出），首选项和属性页一起使用。请参阅org.eclipse.ui.propertyPages和org.eclipse.ui.preferencePages扩展点的keywordReference元素。
