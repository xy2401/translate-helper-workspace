DTP帮助 - 帮助程序插件概述

自1.5版（2007年6月29日发布）以来，Eclipse DTP项目中提供了Data Tools Platform（DTP）help-helper插件。

DTP help-helper插件（org.eclipse.datatools.help）为动态上下文相关帮助提供了两个关键的启用程序：
  • ContextProviderDelegate实现，其方法允许从用户界面（UI）代码中抽象帮助上下文ID和特定于上下文的帮助搜索表达式
  • 扩展点（org.eclipse.datatools.help.helpKeyProperties），允许其他插件提供ResourceBundle属性文件，这些文件定义抽象helpKey常量（在UI代码中使用）到外部化帮助上下文ID字符串和上下文的映射特定的帮助搜索表达式
