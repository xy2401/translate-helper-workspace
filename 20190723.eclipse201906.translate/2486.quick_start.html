快速开始

以下快速入门页面将帮助新手和（甚至）我们的资深用户熟悉这个新版本。

快速入门中涵盖的功能包括：

  • 工作区
  • 创建PHP项目
  • 创建PHP文件
  • PHP调试

工作台

Workbench是一个窗口，显示透视图 ， 视图和菜单栏，通过它可以执行不同的操作。

有关使用和自定义Workbench的更多信息，请参阅Workbench用户指南中的Workbench主题。

注意：

通过转到Help |，可以从PDT内部访问其他用户指南帮助内容 ，或者来自Eclipse在线文档站点（ http://help.eclipse.org/ ）。

工作区

工作空间是创建和存储所有项目的文件系统位置。

您可以使用PDT创建的默认工作区或从菜单栏转到文件|切换工作区|其他选择不同的工作区。

创建PHP项目

项目是一组文件和资源，它们显示在Project Explorer视图的树中。

要创建一个新的PHP项目：

转到菜单栏，然后选择文件|新的| PHP项目 。

- 或 - 在Project Explorer视图中，右键单击并选择New | PHP项目 。

有关更多信息，请参阅创建PHP项目 。

创建PHP文件

要创建新的PHP文件：

转到菜单栏，然后选择文件|新的| PHP文件 。

- 或者在Project Explorer视图中，右键单击要在其中创建文件的文件夹，然后选择New | PHP文件

- 或 - 单击工具栏上的“新建PHP文件”图标 。这将在项目外部创建一个文件。

有关更多信息，请参阅创建PHP文件 。

PHP调试

调试器检测并诊断位于本地或远程服务器上的PHP代码中的错误：

要调试位于工作区的PHP脚本：

  1. 通过双击相关行上的标记栏（位于编辑器区域左侧），在脚本中的相关位置设置断点。出现蓝色球表示设置了断点。
  2. 从主菜单中，选择“运行”|调试配置 - 或者 - 在Project Explorer视图中右键单击该文件，然后选择Debug As |调试配置 。
  3. 要创建新配置，请双击PHP CLI Application类别。
  4. 输入所有相关信息，然后单击Apply and Debug 。

要调试位于服务器上的PHP Web页面：

  1. 从主菜单中，选择“运行”|调试配置 - 或者 - 在Project Explorer视图中右键单击该文件，然后选择Debug As |调试配置 。
  2. 要创建新配置，请双击“ PHP Web应用程序”类别。
  3. 输入所需信息，然后单击“ 应用并调试” 。

可以从Debug Preferences页面配置调试首选项 ，可以从Window |访问偏好| PHP |调试 。

有关更多信息，请参阅调试 。

利益的视角

要打开透视图，请转到Window |打开透视图 ，选择“其他”以查看完整的透视列表。

  • PHP（默认） - 这是在PDT中默认打开的透视图。它允许您管理和创建PHP项目和文件。
  • PHP Debug - 允许您管理和跟踪调试过程。
  • GIT / CVS / SVN Repository Exploring - 允许您管理和查看源代码管理。

有关更多信息，请参阅PHP透视图和视图 。
