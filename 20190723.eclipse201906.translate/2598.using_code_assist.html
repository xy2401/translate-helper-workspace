使用内容辅助

此过程描述了如何使用内容辅助来快速轻松地将代码元素插入到脚本中：

要使用内容辅助：

  1. 在编辑器中输入所需代码元素的前几个字符。应自动显示“内容辅助”窗口。
  2. 如果内容辅助窗口未自动弹出，请按Ctrl + Space 。
  3. 使用箭头键滚动代码完成选项。右侧的窗口将显示使用所选代码元素的说明和提示。
  4. 双击选择所需选项，然后按Enter键选择所需选项。

所选代码将插入到您的脚本中。

要使内容辅助窗口自动打开，请转到从窗口|访问的内容辅助首选项页面 偏好| PHP |编辑|内容辅助并标记“启用自动激活”复选框。这是默认标记。
