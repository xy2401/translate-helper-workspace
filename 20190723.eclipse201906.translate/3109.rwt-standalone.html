RWT Standalone

可以在没有OSGi的情况下使用RAP。在那种情况下，RWT像普通的Java库一样使用。建议使用RAP工具开发RWT应用程序。尽管使用工具并非绝对必要，但它通过为RWT应用程序和文档量身定制的启动配置简化了开发。

创建并启动应用程序

按照下面列出的步骤，您将在几分钟内启动并运行一个简单的Web应用程序。

  • 如果您愿意，可以创建Java项目 （或插件项目 ，并熟悉插件开发）
  • 配置项目以匹配Web应用程序的布局。如果您使用RAP工具启动应用程序，则可以跳过或推迟此步骤。如果要将项目部署为WAR，则必须进行布局。
    • 创建三个文件夹： WEB-INF ， WEB-INF / lib ， WEB-INF / classes
    • 将项目的输出文件夹更改为WEB-INF / classes 。
  • 将org.eclipse.rap.rwt_ * jar从RAP Runtime复制到WEB-INF / lib文件夹，并将其添加到项目的构建路径中。org.eclipse.rap.rwt.source_ * jar包含RWT源代码。为了能够浏览源代码并读取JavaDoc，请将此jar指定为源附件
  • 实现如下的EntryPoint ：
    public class HelloWorld extends AbstractEntryPoint {public void createContents（Composite parent）{Label label = new Label（parent，SWT.NONE）; label.setText（“Hello RAP World”）; }}

安装RAP工具后，您就可以启动HelloWorld应用程序了。为此，请选择HelloWorld类（即在Package Explorer中 ），然后从上下文菜单中选择Run As > RWT Application 。

部署

如果您希望在外部servlet引擎上部署应用程序，或者由于其他原因需要部署描述符，或者尚未安装RAP工具，则还需要执行一些步骤来运行应用程序。

  • 将部署描述符（ web.xml ）放在WEB-INF文件夹中，其中包含以下内容：
    <？xml version =“1.0”encoding =“UTF-8”？> <web-app xmlns =“http://java.sun.com/xml/ns/j2ee”xmlns：xsi =“http://www.w3.org/2001/XMLSchema-instance”xsi：schemaLocation =“ http://java.sun.com/xml/ns/j2ee http://java.sun.com/xml/ns/j2ee/web-app_2_4.xsd“version =”2.4“> <context-param> <param -name> org.eclipse.rap.applicationConfiguration </ param-name> <param-value> com.example。HelloWorldConfiguration </ param-value> </ context-param> <listener> <listener-class> org.eclipse.rap.rwt.engine。RWTServletContextListener </ listener-class> </ listener> <servlet> <servlet-name> rwtServlet </ servlet-name> <servlet-class> org.eclipse.rap.rwt.engine。RWTServlet </ servlet-class> </ servlet> <servlet-mapping> <servlet-name> rwtServlet </ servlet-name> <url-pattern> / hello </ url-pattern> </ servlet-mapping> </ web -app>
  • 提供应用程序配置以配置您的应用程序，如下所示：
    公共类HelloWorldConfiguration实现ApplicationConfiguration {public void configure（Application application）{application.addEntryPoint（“/ hello”，HelloWorld.class，null）; }}
  • 您可以再次使用RAP工具从刚刚创建的web.xml启动应用程序。为此，请创建一个新的RWT启动配置，然后选择从web.xml运行 。输入web.xml文件的位置，并将hello指定为servlet路径。

您可能还会发现JFace组件很有用。要从RWT独立版中使用它们，您需要从RAP运行时添加以下jar：

  • org.eclipse.rap.jface
  • org.eclipse.core.runtime
  • org.eclipse.core.commands
  • org.eclipse.equinox.common
