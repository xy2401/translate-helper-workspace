完成提案计算机

标识符：
org.eclipse.cdt.ui.completionProposalComputer

以来：
4

描述：
此扩展点允许提供完成提议计算机以参与C / C ++编辑器的内容辅助过程。

配置标记：

<！ELEMENT 扩展 （ completionProposalComputer | proposalCategory ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>

  • point - 目标扩展点的标准标识符
  • id - 扩展实例的标识符，在声明插件中是唯一的（插件的标识符将被添加到形成平台范围的唯一ID）
  • name - 扩展实例的可选名称

<！ELEMENT completionProposalComputer （ 分区 ）*>

<！ATTLIST completionProposalComputer

class CDATA #REQUIRED

activate（true | false）“false”

categoryId IDREF“org.eclipse.cdt.ui.defaultProposalCategory”>

提案计算机贡献。如果未指定分区类型，则计算机将添加到所有分区类型。


  • class - 实现贡献计算机的类的名称。该类必须是公共的并实现org.eclipse.cdt.ui.text.contentassist。ICompletionProposalComputer并且必须具有公共0参数构造函数。
  • activate - 如果该属性设置为“true”，它将强制在内容辅助调用上加载此插件。
  • categoryId - proposalCategory的id

<！ELEMENT 分区 EMPTY>

<！ATTLIST分区

type（__dftl_partition_content_type | __c_multiline_comment | __c_singleline_comment | __c_string | __c_character | __c_preprocessor | __c_multiline_doc_comment | __c_singleline_doc_comment）>

  • type - 指定计算机可为其提供完成建议的AC / C ++分区类型。见IDocument.DEFAULT_CONTENT_TYPE和ICPartitions表示有效值。

<！ELEMENT proposalCategory EMPTY>

<！ATTLIST proposalCategory

icon CDATA #IMPLIED>

提案类别贡献定义用于在UI中对其进行分组的提案类别。


  • icon - 类别的可选图标，可以在用户首选项中显示。

例子：
以下是完成建议计算机贡献的示例：

<extension point = “org.eclipse.cdt.ui.completionProposalComputer” id = “textual_proposals” name = “Text Proposals” > <proposalCategory icon = “icons / wordcompletions.png” /> </ extension> <extension point = “org .eclipse.cdt.ui.completionProposalComputer“ id = ”WordCompletionProposalComputer“ name = ”Word Completion Proposal Computer“ > <completionProposalComputer class = ”org.eclipse.cdt.internal.ui.text.contentassist。HippieProposalComputer“ categoryId = ”org.eclipse.ui.texteditor.textual_proposals“ > <partition type = ”__ c_multiline_comment“ /> </ completionProposalComputer> </ extension>

API信息：
贡献的类必须扩展org.eclipse.cdt.ui.text.contentassist.ICompletionProposalComputer

提供的实施：
请参阅org.eclipse.cdt.internal.ui.text.contentassist.HippieProposalComputer为例。
