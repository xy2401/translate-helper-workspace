使用Inspect Action

在调试PHP脚本时，Inspect Action是一种快速评估表达式的方法。

  1. 通过双击要设置断点的行左侧的垂直标尺来设置代码的断点 - 或 - 选择该行并转到运行|切换断点或按Ctrl + Shift + B.HERE 。将出现一个蓝色球，表示已设置断点。
  2. 在编辑器中选择一个表达式，然后从右键菜单中选择Inspect - 或 - 按Ctrl + Shirft + I.显示评估的表达式。

有关调试方法的更多信息，请参阅调试 。
