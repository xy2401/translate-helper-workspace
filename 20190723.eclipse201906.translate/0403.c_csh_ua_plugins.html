上下文相关帮助UA插件

上下文相关帮助UA插件处理帮助上下文到特定于上下文的帮助内容和相关联机文档主题的映射。

对于使用DTP帮助助手的项目，上下文相关帮助插件有助于：
  • Eclipse上下文XML文件，为每个帮助上下文提供特定于上下文的帮助内容，并指向与帮助上下文相关的其他帮助主题贡献
  • Java属性文件，用于定义将抽象helpKey常量映射到具体帮助上下文ID和特定于上下文的帮助搜索表达式的属性键 - 值对
提示： helpKey属性文件通常成对提供，每个文件用于帮助上下文ID和帮助搜索表达式。

下图显示了专用的上下文相关帮助插件如何在运行时与DTP帮助帮助程序和Eclipse帮助系统交互，以便为UI插件提供动态上下文相关帮助。



在工作台启动期间：
  • DTP帮助助手注册helpKey属性文件贡献，由上下文相关帮助插件声明
  • Eclipse帮助系统注册：
    • IContext内容贡献，由上下文相关帮助插件声明
    • 由文档插件声明的在线帮助（TOC和主题）贡献

初始化工作台之后，将显示GUI，其中包含由UI插件提供的工作台部件和控件。上下文相关的帮助请求通常由UI插件（或其委托的帮助事件侦听器）处理，用于其提供的工作台部件和UI控件。

当用户请求上下文相关帮助时（例如，通过按F1或单击帮助图标），将发生以下序列：
  1. UI插件通过传递抽象的helpKey从DTP help-helper请求具体的帮助上下文ID。
  2. DTP帮助帮助程序在为UI插件注册的helpKey属性文件中通过其关联的helpKey查找帮助上下文ID。
  3. DTP帮助助手将帮助上下文ID返回给UI插件。
  4. UI插件在其关联的工作台部件或UI控件上设置具体的帮助上下文ID，然后将该帮助上下文ID传递给Eclipse帮助系统，以请求显示上下文相关帮助。
  5. Eclipse帮助系统通过帮助上下文ID查找特定于上下文的内容和相关主题，从为UI插件注册的上下文XML文件中检索它们，然后在“帮助”视图中显示它们。
  6. 根据用户请求（通过单击相关主题链接），Eclipse帮助系统将检索文档插件提供的帮助主题内容，并将其显示在“帮助”视图中。
  • Eclipse Context XML文件扩展
    每个提供Eclipse上下文XML文件的插件都必须在其插件清单（plugin.xml文件）中声明org.eclipse.help.contexts的扩展名。
  • helpKey属性文件扩展名
    每个提供helpKey属性文件的插件都必须在其插件清单（plugin.xml文件）中声明org.eclipse.datatools.help.helpKeyProperties的扩展名。
  • 上下文相关帮助插件ID
    上下文相关的帮助插件可以作为专用插件提供，这些插件指向由其他在线文档插件提供的相关帮助主题，但不提供任何帮助主题。
