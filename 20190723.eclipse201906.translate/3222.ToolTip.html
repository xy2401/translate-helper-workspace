ToolTip

ToolTip widgets represent popup windows that are used to inform or warn the user

Available CSS Properties

Properties that can be defined for this element:

  • animation

    Supports animations-types "fadeIn" and "fadeOut". Others are ignored.

  • background-color

    Background color

  • background-image

    Background image or gradient for tooltip widget.

  • border

    The border that is drawn around tooltip widget.

  • border-bottom

    The bottom side border for tooltip widgets.

  • border-left

    The left side border for tooltip widgets.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for tooltip widgets.

  • border-top

    The top side border for tooltip widgets.

  • box-shadow

    The shadow behind the tooltip.

  • color

    Text color

  • cursor

    The cursor for the tooltip

  • font

    Text font for tooltips.

  • height

    Fixed height for the tooltip. If set auto the tooltip height will be calculated automatically.

  • opacity

    The opacity of the tooltip widget. A value between 0 and 1.

  • padding

    Padding for tooltip widget.

  • width

    Fixed width for the tooltip. If set to auto the tooltip width will be calculated automatically.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BALLOON]

    Style constant for balloon behavior.

  • [ICON_ERROR]

    The image used for ToolTips with style SWT#ICON_ERROR.

  • [ICON_INFORMATION]

    The image used for ToolTips with style SWT#ICON_INFORMATION.

  • [ICON_WARNING]

    The image used for ToolTips with style SWT#ICON_WARNING.

Applicable States

none
