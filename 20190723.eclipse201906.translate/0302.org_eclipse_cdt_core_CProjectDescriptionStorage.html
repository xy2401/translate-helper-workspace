描述：
此扩展点允许贡献者提供他们自己的CProjectDescriptionStorage后备存储的实现。

配置标记：

<！ELEMENT 扩展 （ CProjectStorageType ）+>

<！ATTLIST扩展

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT CProjectStorageType EMPTY>

<！ATTLIST CProjectStorageType

class CDATA #REQUIRED

id IDREF #REQUIRED

版本CDATA #REQUIRED

min_version CDATA #IMPLIED

max_version CDATA #IMPLIED

名称CDATA #IMPLIED>

CProjectDescriptionStorageType是AbstractCProjectDescriptionStorage的具体类型，它被委托用于持久化ICStorageElement树


  • class - 实现ICProjectDescriptionStorageType的工厂类，用于为给定的IProjects创建AbstractCProjectDescriptionStorages
  • id - 此特定项目描述存储类型的ID
  • version - 如果指定了min_version和max_version，则此版本的版本号，此版本必须在范围内（min_version，max_version），即min_version <version <= max_version此String值必须与org.eclipse.osgi.framework.version兼容
  • min_version - 此描述存储类型支持的项目描述的最低版本。如果未指定此值，则使用默认的Version.emptyVersion此String值必须与org.eclipse.osgi.framework.version兼容
  • max_version - 此描述存储类型支持的项目描述的最大版本。如果未指定，则使用等于version的值。此String值必须与org.eclipse.osgi.framework.version兼容
  • name - 此项目描述类型的可读名称
