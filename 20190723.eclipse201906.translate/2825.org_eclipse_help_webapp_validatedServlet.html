验证的Servlet

标识符：
org.eclipse.help.webapp.validatedServlet

以来：
3.7

描述：
此扩展点实现了围绕servlet的webapp安全验证程序。在返回客户端之前，将检查对servlet的所有调用是否存在已知的恶意威胁。要点：servlet验证器资源的所有上下文路径必须以/ vs开头。例如：http：// host：port / help / vs / service / toc

配置标记：

<！ELEMENT 扩展 （ servlet ）*>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT servlet EMPTY>

<！ATTLIST servlet

class CDATA #REQUIRED

别名CDATA #REQUIRED>

  • 别名 -

例子：
没有提供。

API信息：
提供的servlet类必须扩展类javax.servlet.http。HttpServlet 。servlet类的功能与普通servlet没有区别。

提供的实施：
对于org.eclipse.help.internal.webapp.services中的任何服务，可以在org.eclipse.help.webapp中找到此点的提供实现。
