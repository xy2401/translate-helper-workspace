使用浏览器工具栏进行分析

此过程描述如何使用Z-Ray或Zend Debugger Toolbar等浏览器工具栏来分析PHP Web应用程序。

使用Z-Ray进行分析

  1. 确保在Zend Server上启用并运行Z-Ray 。
  2. 在浏览器中打开要配置的网页。
  3. 确保您的PDT已打开。
  4. 单击Z-Ray工具栏上的“Bug”图标，然后选择“ 配置文件当前页面”选项以分析当前显示在浏览器中的页面。

相关的概况会议将在PDT中启动。

使用Zend Debugger工具栏进行性能分析

  1. 确保浏览器上安装了Zend Debugger Toolbar 。
  2. 在浏览器中打开要配置的网页。
  3. 确保您的PDT已打开。
  4. 单击Zend Debugger工具栏上的“配置文件”图标，以分析浏览器中当前显示的页面。

相关的概况会议将在PDT中启动。

使用Xdebug工具栏进行性能分析

  1. 确保您的浏览器上安装了Xdebug帮助程序或最简单的Xdebug 。
  2. 确保已启用xdebug.profiler_enable_trigger
  3. 在浏览器中打开要配置的网页。
  4. 确保您的PDT已打开。
  5. 单击Xdebug工具栏上的“配置文件”图标，以分析浏览器中当前显示的页面。

你Xdebug将创建CacheGrind。现在您可以导入配置文件会话
