分析配置

Perf插件有几个配置选项，其中大多数是二进制文件中存在的配置选项的镜像。有关所有这些选项的更多信息，请参阅Perf手册页或https://perf.wiki.kernel.org/ 。

远程分析

可以使用RSE或RemoteTools来配置远程项目。在“配置文件配置”中，在“使用远程Perf配置文件”选项中创建配置后，需要搜索项目二进制文件。用户可以通过选中“复制C / C ++可执行文件”复选框将C / C ++可执行文件从远程服务器复制到本地工作站。但默认情况下它会分析C / C ++可执行文件字段中指定的可执行文件。在这种情况下，您需要浏览二进制文件。工作目录自动设置为基础项目目录，但您可以根据需要进行修改。

Perf选项选项卡

此配置选项卡包含使用Perf进行性能分析的运行时选项，例如加载内核映像文件或隐藏未解析的符号。

Perf Events选项卡

在此选项卡上，可以为分析设置自定义事件数组，或者，如果需要，可以使用默认事件选项，该选项将覆盖所有其他设置，并使用默认事件（循环计数）运行Perf。
