添加评论

PDT允许您通过选择一行或一段文本并将其标记为注释来快速轻松地注释和取消注释代码。

注释可以添加到单行代码（ Ctrl + / ）或代码块（ Ctrl + Shift + / ）。

此外，还可以添加特殊的PHPDocBlock注释。有关更多信息，请参阅“ 添加PHP DocBlock注释 ”。

以下过程描述了如何注释和取消注释代码行和代码块。

评论一条线：

  1. 将光标放在所需的代码行上的任何位置。
  2. 按Ctrl + /两行斜线“//”将添加到行的前面，使其被识别为注释。

要评论多行：

  1. 选择您要评论的所有行。
  2. 按Ctrl + /两行斜线“//”将添加到每行的前面，使它们被识别为注释。

要取消注释一行/一行：

  1. 选择所需的行。
  2. 按Ctrl + /注释格式将从代码中删除。

评论一个块：

  1. 选择所需的代码块。
  2. 按Ctrl + Shift + /开头（/ ）和结束（ /）字符将添加到适当的位置，以便将所选块标记为注释。
