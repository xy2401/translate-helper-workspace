注释处理器工厂

标识符：
org.eclipse.jdt.apt.core.annotationProcessorFactory

以来：
3.2

描述：
贡献JSR-175注释处理器的插件应扩展此扩展点以参与编译。

配置标记：

<！ELEMENT 扩展 （ 工厂 ？， java6processors ？）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 工厂 （ 工厂 *）>

<！ATTLIST工厂

enableDefault（true | false）>

  • enableDefault - 如果默认情况下应启用此处理器， 则为 true。可以通过注释处理配置设置覆盖此值。如果不存在，则此值默认为true。

<！ELEMENT factory EMPTY>

<！ATTLIST工厂

class CDATA #REQUIRED

实现CDATA #IMPLIED>

  • class - 实现com.sun.mirror.apt的类的限定名称。AnnotationProcessorFactory并具有无参数构造函数。
  • 不推荐使用的 工具 - 保留。不要使用此属性。

<！ELEMENT java6processors （ java6processor *）>

<！ATTLIST java6processors

enableDefault（true | false）>

  • enableDefault - 如果默认情况下应启用此处理器， 则为 true。可以通过注释处理配置设置覆盖此值。如果不存在，则此值默认为true。

<！ELEMENT java6processor EMPTY>

<！ATTLIST java6processor

class CDATA #REQUIRED>

  • class - 实现javax.annotation.processing的类的限定名称。处理器并具有无参数构造函数。

例子：
annotationProcessorFactory的声明示例：
<extension point = “org.eclipse.jdt.apt.core.annotationProcessorFactory” > <factories enableDefault = “true” > <factory class = “org.xyz。HelloWorldAnnotationProcessorFactory“ > </ factory> <factory class = ”org.xyz。GoodNightMoonAnnotationProcessorFactory“ > </ factory> </ factories> </ extension>

API信息：
注释处理器必须实现com.sun.mirror.apt。AnnotationProcessorFactory或其子类。
