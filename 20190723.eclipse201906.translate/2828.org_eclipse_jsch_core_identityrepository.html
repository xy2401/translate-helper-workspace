identityrepository

标识符：
org.eclipse.jsch.core.identityrepository

以来：
1.2

描述：
此扩展点允许插件注册身份存储库工厂。该工厂预计将创建com.jcraft.jsch.IdentityRepository ，它使用ssh-agent和Pageant抽象通信。此扩展点必须实现抽象类org.eclipse.jsch.core.AbstractIdentityRepositoryFactory 。

配置标记：

<！ELEMENT 扩展 （ identityrepository ）>

<！ATTLIST扩展

点CDATA #REQUIRED>

  • point - 目标扩展点的完全限定标识符

<！ELEMENT identityrepository （ 运行 ）>



<！ELEMENT 运行 EMPTY>

<！ATTLIST跑

class CDATA #REQUIRED>

  • class - 实现org.eclipse.jsch.core.AbstractIdentityRepositoryFactory

例子：
以下是ssh-agent的身份存储库工厂的示例：


<extension point = "org.eclipse.jsch.core.identityrepository">
     <identityrepository>
       <run
         class="com.jcraft.jsch.agentproxy.eclipse.sshagent.IdentityRepositoryFactory">
       </run>
     </identityrepository>
</extension>

API信息：
贡献的类必须实现org.eclipse.jsch.core.AbstractIdentityRepositoryFactory 。

提供的实施：
Eclipse jsch-agent-proxy插件定义了身份库工厂。
