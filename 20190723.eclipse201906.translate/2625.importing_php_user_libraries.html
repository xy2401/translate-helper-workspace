导入PHP用户库

此过程描述如何导入磁盘或存储库中的现有用户库。这允许您获取已构建的库并在项目中使用它，以及与同一存储库的其他用户共享库。导入库只会以.xml格式导入库的描述，并且不会包含任何库的内容。在导入用户库之前，必须首先访问导出的用户库。有关更多信息，请参阅导出PHP用户库 。

要导入用户库：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，单击Import ... 将打开“导入用户库”对话框。
  3. 要选择要从中导入库的位置，请使用URL填写“文件位置：”文本字段，或单击“ 浏览...”并选择位置。
  4. 从“所选文件中包含的库：”框中选项中选择要导入的库，或按“全 选”或“ 取消全选” 。
  5. 要应用更改，请单击确定 。您的库的.xml格式描述现已导入PDT。

如果导入它的用户的库存储在磁盘上与导出它的用户相同的位置，则PDT将自动查找库内容并相应地存储它。如果用户库存储在两个用户的不同位置，则可以使用“ 编辑...”按钮替换位置URL。有关更多信息，请参阅编辑PHP用户库 。
