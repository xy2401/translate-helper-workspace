JavaScript查询参与者

标识符：
org.eclipse.wst.jsdt.ui.queryParticipants

以来：
3.0

描述：
此扩展点允许客户端为javascript搜索提供结果

配置标记：

<！ELEMENT 扩展 （ queryParticipant +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT queryParticipant EMPTY>

<！ATTLIST queryParticipant

class CDATA #REQUIRED

id CDATA #REQUIRED

自然CDATA #REQUIRED

名称CDATA #REQUIRED>

  • class - 实现此查询参与者的类。该类必须是公共的并实现org.eclipse.wst.jsdt.ui.search.具有零参数构造函数的IQueryParticipant 。
  • id - 此查询参与者的唯一ID
  • nature - 此参与者应该活跃的项目性质ID。如果参与者应该对多个项目性质有效，则必须定义多个参与者。
  • name - 参与者的用户可重命名

例子：
以下是查询参与者贡献的示例：

<extension point = “org.eclipse.wst.jsdt.ui.queryParticipants” > <queryParticipant label = “Example Query Participant” nature = “org.eclipse.wst.jsdt.core.javanature” class = “org.eclipse.wst .jsdt.ui.example。TestParticipant“ id = ”org.eclipse.wst.jsdt.ui.example。TestParticipant“ > </ queryParticipant> </ extension>

API信息：
贡献的类必须实现org.eclipse.wst.jsdt.ui.search.IQueryParticipant

提供的实施：
没有
