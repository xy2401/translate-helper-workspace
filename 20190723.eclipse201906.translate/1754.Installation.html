安装

先决条件

Java的

  • 下载并安装JVM （JRE或JDK）（如果尚未安装）。
    • 对于ATL 2.0或更早版本，JDK 1.4将起作用。
    • 对于ATL 3.0+，您需要JDK 5.0或更高版本

日食

  • 下载并安装Eclipse 。

EMF

  • 安装EMF 。

安装ATL

有两种方法可以安装ATL：

  • 通过更新站点
  • 通过zip：请参阅ATL下载 。下载完zip后，只需将其解压缩即可
    • 使用Eclipse 3.3,3.2：〜/ eclipse / features /和〜/ eclipse / plugins / folders
    • 使用Eclipse 3.4+：〜/ eclipse / dropins /文件夹

最后，重启Eclipse。

ATL SDK构建使：

  • 离线查阅文档
  • 访问ATL示例
  • 访问已安装的ATL插件的源代码

要从CVS安装ATL，请参阅开发人员指南 。
