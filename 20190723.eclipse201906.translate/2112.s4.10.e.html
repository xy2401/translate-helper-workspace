§4.10。（e） 传播类型参数

如果callin绑定绑定到泛型基本方法，则必须通过使用类型参数声明callin绑定，将基本方法的任何类型参数传播到角色方法中。通过将基本方法的类型参数与callin绑定的类型变量匹配，该通用性通过callin绑定传播。

1 
class MyBase {
2 
	<T> T getIt(T it) { return it; }
3 
}
4 
team class MyTeam {
五 
	protected class MyRole playedBy MyBase {
6 
		callin <U> U rm(U a) { return base.rm(a); }
7 
		<U> U rm(U a) <- replace U getIt(U it);
8 
	}
9 
}
说明：
callin绑定声明了一个类型参数<U> ，它用于匹配T签名中所有出现的getIt因此， rm的实现使用类型U ，其方式与getIt使用T完全相同。
