任务编辑器

任务编辑器允许您查看和编辑任务列表中的任务。双击任务列表中的任务以打开编辑器。任务编辑器的功能将根据它是本地任务还是共享存储库任务而有所不同。对于共享存储库任务，根据您使用的存储库（和相应的连接器）的类型（链接：连接器）存在一些差异。

存储库任务详细信息

在本节中，我们将描述Bugzilla存储库中共享错误的任务编辑器。其他存储库类型（如Trac）的任务编辑器提供类似的功能。

编辑器工具栏按钮

  • 同步传入更改 - 更新任务的本地副本以反映服务器上的任何更改。
  • 创建新的子任务 - 创建一个新任务，该任务将被视为完成当前任务的先决条件。子任务显示嵌套在任务列表中的父任务下。在Bugzilla术语中，子任务“阻止”父任务和子任务的ID将出现在父任务的“块：”字段中。
  • 历史记录 - 使用Web界面在内部浏览器中显示任务的更改历史记录。
  • 使用Web浏览器打开 - 在内部Web浏览器中显示任务的Web界面。
  • 查找 - 在评论，说明，摘要和私人笔记中查找文字。
  • 激活 - 切换任务的激活和停用。

属性使用“属性”部分可添加或更新有关任务的结构化信息。

团队规划团队规划部分包含与团队共享的任务的时间相关信息。您可以使用“ 到期”字段为任务设置截止日期。在截止日期，任务将在任务列表中以红色显示。

附件您可以将文件附加到此任务，以便将副本上载到任务存储库，并且可供任何可以访问该任务的人员使用。

附加文件

  • 单击“附加...”按钮，这将打开一个向导。
  • 选择以下之一：
    • 文件 - 从您的系统上传文件。单击右侧的“浏览”以选择文件。
    • 剪贴板 -
    • 工作区 - 从工作区上传文件。从下面的框中选择文件。
  • 单击下一步输入附件详细信息：
    • 描述 - 提供文件的简要说明。此说明将显示在任务编辑器的附件列表中。
    • 注释 - 提供有关该文件的注释。此注释将显示在任务编辑器的注释部分中。
    • 内容类型 - （可选）指定文件的内容类型
    • 补丁 - 如果附件是源代码补丁文件，请检查此项
    • 附加上下文 - 如果您还要附加任务的上下文，请选中此项。此上下文描述与任务最相关的资源。如果附加上下文，其他人可以下载它并将UI集中在与您相关的相同资源上。
    • 单击“下一步”以预览文件。如果您的文件是图像，则会显示该文件的预览。
    • 单击“完成”将文件上载到任务存储库。上传文件时无需单击任务编辑器底部的“提交”。

重复检测提交错误报告时，您可以通过单击“搜索”按钮来避免重复。这将在存储库中搜索与任务的Description字段中的堆栈跟踪匹配的堆栈跟踪。重复检测的结果显示在“搜索”视图中。如果找到匹配项，您可以打开它并发表评论，而不是创建新的错误报告。

注释使用此部分可添加有关任务的新注释并查看以前的所有注释。您之前阅读过的评论已弃用。您可以展开和重新阅读单个评论，或单击右上角的“+”以展开所有评论。

操作使用此部分更改任务的状态或将任务重新分配给其他人。

  • 附加上下文 - 将有关您已与之交互的资源的信息上载到服务器。上下文将显示在附件列表中，以便其他人可以下载它并将其UI集中在您发现与此任务相关的资源上。
  • 提交 - 将任务的所有本地更改提交到团队的共享任务存储库。

人员此部分显示正在协作完成任务的人员。

  • 分配给 - 这是负责完成任务的人
  • 记者 - 这个人创造了这个任务
  • QA联系 -
  • 添加CC - 使用此框将新人添加到“CC”列表。“CC”列表中的人员将收到有关此任务的任何更改的通知。要添加新人，请键入其电子邮件地址的开头，然后按ctrl + space以使用内容辅助完成地址。您可以添加多个地址，用逗号分隔。
  • CC - 此框显示当前位于“CC”列表中的人员。要删除某个人，只需选择他们的电子邮件地址，使其突出显示并单击“提交”。您可以按住ctrl选择多个人。

上下文

上下文选项卡允许您管理与任务关联的资源的上下文。您可以通过在编辑器窗口的左下角选择它来查看上下文选项卡。

分子
本节列出了作为任务上下文一部分的资源。由于元素的数量可能很大，因此您可以使用“ 操作”部分顶部的滑块调整细节级别。将控件一直滑动到左侧将显示任务上下文中的所有元素。向右滑动时，仅显示具有高兴趣度的元素。您可以通过右键单击并选择“从上下文中删除”来手动从任务上下文中删除元素。在将上下文附加到任务之前，您可以选择以这种方式查看所有元素并修剪不相关的项目，以便其他人可以下载它。

操作

  • 元素详细信息滑块 - 调整元素在“ 元素”部分中显示所需的最低兴趣级别。
  • 附加上下文 - 将上下文附加到任务，以便可以从共享任务存储库下载它。上下文由右侧显示的元素组成。
  • 检索上下文 - 将当前任务上下文替换为附加到共享任务存储库中任务的任务上下文。
  • 将上下文复制到... - 将任务上下文复制到另一个任务。然后，该任务将具有与当前任务相同的上下文。
  • 清晰的背景。 - 从任务中删除所有上下文信息。

规划

使用“计划”选项卡可以访问有关工作区专用任务的本地信息。您可以通过在编辑器窗口的左下角选择计划选项卡来查看计划选项卡。此选项卡包含一个大区域，您可以在其中输入有关任务的个人注释。有关“个人计划”部分中字段的详细信息，请参阅本地任务部分。
