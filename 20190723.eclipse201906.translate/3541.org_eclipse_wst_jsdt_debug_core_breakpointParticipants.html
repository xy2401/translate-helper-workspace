断点参与者

标识符：
org.eclipse.wst.jsdt.debug.core.breakpointParticipants

以来：
1.0

描述：
此扩展点允许消费者提供断点参与者，在决定暂停参与者适用的断点时将参考该断点参与者。

配置标记：

<！ELEMENT 扩展 （ 参与者 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 参与者 EMPTY>

<！ATTLIST参与者

class CDATA #REQUIRED

kind（function | line | script | *）>

断点参与者


  • class - class属性是必需的，必须实现org.eclipse.wst.jsdt.debug.core.breakpoints.IJavaScriptBreakpointParticipant
  • kind - 此属性指定此参与者应应用于值的断点类型*表示它适用于所有断点类型。该值必须是函数 ， 行 ， 脚本或*之一 。

例子：
以下示例是Mozilla Rhino参与者对行断点的实际定义。 

<extension point="org.eclipse.wst.jsdt.debug.core.breakpointParticipants">
  <participant
     class="org.eclipse.wst.jsdt.debug.internal.rhino.breakpoints.RhinoBreakpointParticipant"
     kind="*">
  </participant>
</extension>

API信息：
class属性是必需的，必须实现org.eclipse.wst.jsdt.debug.core.breakpoints.IJavaScriptBreakpointParticipant
