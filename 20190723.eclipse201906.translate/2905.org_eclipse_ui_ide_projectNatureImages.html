项目自然图像

标识符：
org.eclipse.ui.ide.projectNatureImages

以来：
3.0（最初在1.0版中添加为org.eclipse.ui.projectNatureImages）

描述：
此扩展点用于将图像与项目性质相关联。所提供的图像用于形成由标准项目图像与其性质的图像组合的合成图像。提供的图像绘制在基本图像的右上角。

配置标记：

<！ELEMENT 扩展 （ 图像 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 图像 （ 描述 ？）>

<！ATTLIST图片

id CDATA #REQUIRED

natureId IDREF #REQUIRED

icon CDATA #REQUIRED>

  • id - 将用于标识此自然图像的唯一名称。
  • natureId - 提供图像的性质的唯一名称。
  • icon - 将与此透视关联的图像的相对名称。

<！元素描述 （#PCDATA）>

这个图像代表什么的简短描述。



例子：
以下是自然图像扩展的示例：


 <extension point="org.eclipse.ui.ide.projectNatureImages">
  <image
    id="org.eclipse.ui.javaNatureImage"
    natureId="Resource"
    icon="icons/javaNature.png">
  </image>
 </extension>

API信息：
natureId属性的值是由创建项目的插件定义的nature id。
