设置远程调试

在使用PHP Web页面调试在服务器上进行调试之前，需要配置某些设置以确保PDT可以与您的服务器通信。

注意：

以下说明介绍了如何使用Zend Debugger设置远程调试。

要在PDT和要调试的服务器之间建立通信，请执行以下操作：

  1. 确保服务器上已安装Zend Debugger。Zend Debugger与Zend Server ， Zend Core和Zend Platform捆绑在一起，但也可以从http://www.zend.com/en/products/studio/downloads作为单独的组件下载。
  2. 确保安装PDT的计算机是调试器的允许主机。有关详细信息，请参阅将PDT设置为允许的主机 。
  3. 在PDT中，根据“ PHP服务器首选项”中的“ 添加服务器”下的说明配置服务器。
  4. 确保在“ 调试首选项”和“ 调试程序首选项”页面中配置了正确的设置。
  5. 确保远程服务器的文档根目录中有dummy.php文件 。
