遥远的Valgrind

您可以像本地项目一样分析远程项目。在为不同的体系结构开发应用程序时，这非常有用。

要远程运行Valgrind，请右键单击远程c / c ++项目，然后导航到“Profiling Tools”，然后“Profile Configurations”。在新窗口中，双击Profile with Valgrind（远程）配置文件配置。

在主选项卡中，单击“C / C ++可执行文件”下的“浏览”以选择将要执行的二进制文件。本地项目的相同工具配置适用于远程项目。

配置配置文件后，单击配置文件按钮以启动配置文件。结果将显示在之前描述的Valgrind视图中。
