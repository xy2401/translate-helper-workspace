参考提供者

标识符：
org.eclipse.ui.workbench.texteditor.quickdiffReferenceProvider

以来：
3.0

描述：
允许贡献者为快速差异显示添加参考提供者。

配置标记：

<！ELEMENT 扩展 （ referenceprovider +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT referenceprovider EMPTY>

<！ATTLIST referenceprovider

class CDATA #REQUIRED

标签CDATA #IMPLIED

id CDATA #REQUIRED

default（true | false）>

快速差异显示的参考提供者的定义。


  • class - 引用提供程序的类，必须实现org.eclipse.ui.texteditor.quickdiff.IQuickDiffReferenceProvider
  • label - 提供程序的显示标签，它将显示在菜单中，允许用户为此提供程序设置快速差异引用
  • id - 唯一标识此引用提供程序的字符串。
  • 不推荐使用 默认值 - 如果此标志设置为true ，则在第一次为文档启用快速diff时，将默认安装此引用提供程序。如果安装了多个提供程序且遇到标记集，则会采用第一个提供程序。

例子：
以下是参考提供程序定义的示例。它提供了一个提供程序，它使用保存在磁盘上的文档版本作为参考。 


 <extension point="quickdiff.referenceprovider">
  <referenceprovider
   id="default"
   name="%LastSavedProvider.name"
   label="%quickdiff.referenceprovider.label"
   class="org.eclipse.ui.internal.editors.quickdiff.providers.LastSaveReferenceProvider">
  </referenceprovider>
 </extension>

API信息：
没有用于管理参考提供程序的其他API。

提供的实施：
org.eclipse.ui.editors插件提供LastSaveReferenceProvider 。以其实现为例。
