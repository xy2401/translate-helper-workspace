EGF的新功能是什么？

EGF路线图可在Wiki上访问

EGF 0.4.0 - Eclipse 3.6.x：

  • 动态执行：管理[workspace / target platform / runtime]优先级
  • 活动建模（Factory component）：支持Ant任务
  • 模式：模式可以应用于文件资源（不限于模型资源）

EGF 0.3.0 - Eclipse 3.6.x：

  • 新一代单位：发电链。通过多个型号的EMF生成快速跟踪示例
  • 常见功能：复制/粘贴，移动EGF元素
  • 投资组合：构建链模型;生成目标：Hudson和Buckminster

EGF 0.2.5 - Eclipse 3.6.x：

  • 产品组合：EMF定制的EMF模式化
  • 模式：先决条件

EGF 0.2.4 - Eclipse 3.5.2：

  • 常见功能：JUnit测试
  • 模式：使用模式可扩展性重用和自定义

EGF 0.2.3 - Eclipse 3.5.2：

  • 常见功能：任务继承

EGF 0.2.2 - Eclipse 3.5.1：

  • EGF结构：活动工作流程 - 输入/输出/输出参数管理完成
  • 模式：模式注入和模型驱动模式策略的介绍
  • 投资组合：EMF包装

EGF 0.2.1 - Eclipse 3.5.1：

  • EGF结构：改进生成工作流程
  • 模式：引入方法模板编辑器和模式策略;模式调用的改进

EGF 0.2.0 - Eclipse 3.5：

  • 常见功能：fcore文件中工厂组件和任务的持久性;动态执行工厂组件和任务; UI改进
  • EGF结构：EGF的可扩展性以支持贡献（例如新观点）;支持输出，工作流定义的输入/输出参数
  • 模式：模式继承，模式调用的介绍;能够支持新引擎的基础设施

EGF 0.1.0 - Eclipse 3.5：

  • 工厂组件的版本和执行
  • 使用Jet-1生成模式版本和执行
