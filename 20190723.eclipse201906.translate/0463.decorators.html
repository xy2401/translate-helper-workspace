覆盖装饰器

注意：默认情况下，这是一项未启用的可选功能。

Eclipse工作台具有所谓的装饰器的概念，它将图形和文本信息添加到工作台视图中显示的元素。EclEmma为当前活动的覆盖会话提供覆盖装饰器：元素图标上有一个绿色/红色小条，名称旁边有一个百分比值。

显示的百分比基于指令计数器计算。覆盖装饰器仅在存在活动的覆盖会话时才可见，并且仅针对包含可执行代码的元素显示，因此例如不用于抽象方法。

可以在Eclipse 首选项对话框中启用此可选功能：

  • 从Window → Preferences打开首选项对话框...
  • 导航到常规 → 外观 → 标签装饰器页面
  • 选择Java Code Coverage并按OK
