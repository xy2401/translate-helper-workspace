工具栏

工具栏控件

可用的CSS属性

可以为此元素定义的属性：

  • 背景颜色

    工具栏的背景颜色

  • 背景图片

    工具栏的背景图像

  • 边界

    围绕工具栏绘制的边框。

  • 下边框

    工具栏的底部边框。

  • 左边框

    工具栏的左侧边框。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    工具栏的右侧边框。

  • 边框顶部

    工具栏的顶部边框。

  • 颜色

    工具栏的文字颜色

  • 字形

    工具栏的字体

  • 不透明度

    工具栏的不透明度。介于0和1之间的值。

  • 填充

    工具栏的填充（内部距离）

  • 间距

    工具项之间的间距

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [边界]

    设置样式BORDER时适用

  • [平面]

    设置样式FLAT时适用

  • [垂直]

    设置样式VERTICAL时适用

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：禁用

    适用于已禁用的工具栏

ToolItem

ToolBar控件的项目

可用的CSS属性

可以为此元素定义的属性：

  • 动画

    支持动画类型“hoverIn”和“hoverOut”。其他人被忽略了。

  • 背景颜色

    工具栏项目的背景颜色

  • 背景图片

    工具栏项目的背景图像

  • 边界

    围绕工具栏项目绘制的边框。

  • 下边框

    工具项的底边。

  • 左边框

    工具项的左侧边框。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    工具项的右侧边框。

  • 边框顶部

    工具项的顶部边框。

  • 颜色

    工具栏项目的文本颜色

  • 不透明度

    项目的不透明度。介于0和1之间的值。

  • 填充

    工具栏项目的填充（内部距离）。

  • 间距

    文本和图标之间的间距

  • 文字阴影

    文字的阴影。

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [平面]

    在父工具栏上设置样式FLAT时应用。

  • [右到左]

    表示工具栏从右到左。

  • [垂直]

    在父工具栏上设置样式VERTICAL时适用。

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：禁用

    适用于已禁用的工具项。

  • ：第一

    适用于第一个工具项。

  • ：徘徊

    适用于鼠标悬停。

  • ：持续

    适用于最后一个工具项。

  • ：按下

    按下ToolItem时应用。

  • ：选择

    选择ToolItem时应用（检查RADIO样式）。

ToolItem分离器

具有SEPARATOR样式的ToolItems

可用的CSS属性

可以为此元素定义的属性：

  • 宽度

    分隔符的宽度

适用的SWT样式

没有

适用国家

没有

ToolItem-DropDownIcon

ToolItem上显示的图标为DROPDOWN样式

可用的CSS属性

可以为此元素定义的属性：

  • 背景图片

    图标

  • 边界

    分隔dropDown-icon的行的边框。

  • 下边框

    分隔线的底边边框。

  • 左边框

    用于分隔线的左侧边框。

  • 右边框

    用于分隔线的右侧边框。

  • 边框顶部

    用于分隔线的顶边边框。

适用的SWT样式

没有

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：禁用

    适用于已禁用的工具项。

  • ：徘徊

    适用于鼠标悬停。

  • ：按下

    按下ToolItem时应用。

  • ：选择

    选择ToolItem时应用（检查RADIO样式）。
