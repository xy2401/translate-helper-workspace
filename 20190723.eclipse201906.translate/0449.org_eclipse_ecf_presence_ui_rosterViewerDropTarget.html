名册查看器删除目标

标识符：
org.eclipse.ecf.presence.ui.rosterViewerDropTarget

以来：
1.0.0

描述：
此扩展点允许为MultiRosterView指定放置目标。

配置标记：

<！ELEMENT 扩展 （ dropTarget +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED

>


<！ELEMENT dropTarget EMPTY>

<！ATTLIST dropTarget

class CDATA #REQUIRED

>

删除目标元素。


  • class - 名册查看器放置目标的类。必须实现org.eclipse.ecf.presence.ui.dnd。IRosterViewerDropTarget 。需要。

例子：
<extension point = “org.eclipse.ecf.presence.ui.rosterViewerDropTarget” > <dropTarget class = “org.eclipse.ecf.presence.collab.ui。TextRosterViewerDropTarget“ > </ dropTarget> </ extension>
