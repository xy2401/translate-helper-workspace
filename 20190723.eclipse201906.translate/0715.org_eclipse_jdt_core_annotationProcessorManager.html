Java 6注释处理器管理器

标识符：
org.eclipse.jdt.core.annotationProcessorManager

以来：
3.3

描述：
此扩展点使JDT可以在可用的平台上访问Java 6注释处理器管理器。最多可以注册一个Java 6注释处理器管理器扩展。使用org.eclipse.jdt.core.compilationParticipants扩展点分别处理Java 5批注处理器。

配置标记：

<！ELEMENT 扩展 （ annotationProcessorManager ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT annotationProcessorManager EMPTY>

<！ATTLIST annotationProcessorManager

class CDATA #REQUIRED

id CDATA #REQUIRED>

Java 6注释处理器负责在IDE构建和协调的上下文中调度Java 6注释处理器。系统范围内最多只有一个Java 6注释处理器管理器。


  • class - 实现Java 6注释处理器管理器的类。该类必须实现org.eclipse.jdt.internal.compiler.的公共子类org.eclipse.jdt.internal.compiler.AbstractAnnotationProcessorManager带有一个公共0参数构造函数。
  • id - 此参与者的唯一标识符

提供的实施：
org.eclipse.jdt.compiler.apt
