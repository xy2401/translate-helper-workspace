自动化开发人员设置

为JGit / EGit做出贡献的最快开发人员设置是使用Eclipse Installer和EGit项目设置为JGit / EGit准备Eclipse IDE：

  • 下载并解压缩Eclipse Installer
  • 启动Eclipse安装程序
  • 选择高级模式

  • 如果安装程序底部工具栏中最右侧的图标旋转，则安装程序可以使用更新，应在继续之前安装
  • 如果您位于代理后面，请从底部的工具栏中更改代理设置

  • 在产品页面上选择“Eclipse IDE for Eclipse Committers”并单击“Next”

  • 在项目页面上选择项目“EGit”并单击“下一步”

  • 在Variables页面上接受默认目标平台，微调变量点击“显示所有变量”，点击“下一步”
  • 在“确认”页面上单击“完成”

  • 安装程序安装所选的IDE并启动它，只要安装程序显示“按完成以关闭对话框”，您就可以关闭安装程序窗口
  • 新安装的IDE将自动克隆JGit和EGit存储库，并为JGit / EGit开发配置工作台。您可以在工具栏中观察设置进度，如有必要，可以通过单击状态栏中的图标重新打开设置向导

  • 安装完成后，IDE看起来应该与此类似

如果要改进EGit项目设置，请检查tools \ oomph \ _ EGit.setup中的设置文件（在新克隆的egit存储库中）。你可以在这里找到更多关于Oomph的信息
