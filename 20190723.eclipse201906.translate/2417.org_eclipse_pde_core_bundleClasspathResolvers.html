捆绑类路径解析器

标识符：
org.eclipse.pde.core.bundleClasspathResolvers

以来：
3.8

描述：
此扩展点允许客户端将其他bundle类路径条目和源查找路径条目添加到插件项目类路径

配置标记：

<！ELEMENT 扩展 （ 解析器 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT resolver EMPTY>

<！ATTLIST解析器

自然CDATA #REQUIRED

class CDATA #REQUIRED>

  • nature - 启用此解析程序的项目的性质 。
  • class - 实现此动态包类路径条目解析程序的类。该类必须实现IBundleClasspathResolver

例子：
以下是类路径提供程序的示例：


  <extension
        point="org.eclipse.pde.core.bundleClasspathResolvers">
     <resolver
           class="org.eclipse.example.pde.SomeResolver"
           nature="org.eclipse.example.projectNature">
     </resolver>
  </extension>

API信息：
每个类路径解析器都必须提供一个实现org.eclipse.pde.core.IBundleClasspathResolver接口。
