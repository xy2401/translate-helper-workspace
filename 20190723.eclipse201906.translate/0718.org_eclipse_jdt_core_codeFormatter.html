代码格式化程序

不推荐使用此扩展点

标识符：
org.eclipse.jdt.core.codeFormatter

以来：
2.0

描述：
此扩展点允许客户端提供新的源代码格式化程序实现。

配置标记：

<！ELEMENT 扩展 （ codeFormatter *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT codeFormatter EMPTY>

<！ATTLIST codeFormatter

class CDATA #REQUIRED>

  • class - 定义代码格式化程序实现的类。该类必须是org.eclipse.jdt.core.的公共实现org.eclipse.jdt.core.带有公共0参数构造函数的ICodeFormatter 。

例子：
ICodeFormatter的实现ICodeFormatter ：
                                                                       
<extension point="org.eclipse.jdt.core.codeFormatter">            
   <codeFormatter                                                                                              
      class="com.example.MyCodeFormatter"/>                           
</extension>
