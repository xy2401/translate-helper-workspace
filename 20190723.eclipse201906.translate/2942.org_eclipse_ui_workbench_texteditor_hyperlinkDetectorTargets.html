超链接检测器目标

标识符：
org.eclipse.ui.workbench.texteditor.hyperlinkDetectorTargets

以来：
3.3

描述：
此扩展点用于插入超链接检测器目标。

配置标记：

<！ELEMENT 扩展 （ 目标 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 目标 （ 上下文 +）？>

<！ATTLIST目标

id CDATA #REQUIRED

名称CDATA #REQUIRED

说明CDATA #IMPLIED>

  • id - 将用于标识此超链接检测器目标的唯一ID
  • name - 允许在UI中标识此超链接检测器目标的名称
  • description - 此超链接检测器目标的可翻译描述。目前，Eclipse SDK中的任何位置都不会显示此描述。

<！ELEMENT context EMPTY>

<！ATTLIST背景

键入CDATA #REQUIRED>

  • type - 可以通过org.eclipse.jface.text.hyperlink.访问的超链接检测器目标上下文org.eclipse.jface.text.hyperlink.AbstractHyperlinkDetector.getAdapter（...）

例子：
以下是超链接检测器目标定义的示例：
<extension point = “org.eclipse.ui.workbench.texteditor.hyperlinkDetectorTargets” > <target id = “org.eclipse.jdt.ui.javaCode” name = “Java Editor” > <context type = “org.eclipse.ui 。文本编辑器。ITextEditor“ /> </ target> </ extension>
