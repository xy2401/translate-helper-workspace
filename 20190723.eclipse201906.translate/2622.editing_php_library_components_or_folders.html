编辑PHP库组件或文件夹

此过程描述如何编辑用户库文件夹。导入原始位置与磁盘上内容位置不同的文件夹时，请编辑用户库文件夹。有关更多信息，请参阅导入PHP用户库 。

要编辑库组件：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，选择要编辑的用户库文件夹，然后单击Edit ... 将打开“编辑外部文件夹”对话框。
  3. 选择包含用户库内容的文件夹，或单击“创建新文件夹”创建新文件夹 。
  4. 要应用更改，请单击确定 。

您现在已经编辑了用户库的URL，该URL将从磁盘上的适当位置导入内容。

注意：

如果您正在编辑的所选文件的导入位置与新位置不同，则新位置将覆盖该位置。
