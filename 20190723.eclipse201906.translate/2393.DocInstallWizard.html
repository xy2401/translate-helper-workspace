安装向导

  

安装向导是魅力的bootstrap-基础上触发 ，自动安装和配置过程。

向导的页脚包含以下内容：

提供这个帮助。
导航到上一页。
导航到下一页。
完成向导，执行任何最终操作。
关闭向导，不采取进一步操作。
打开相当于Eclipse的Window → Preferences ... → General → Network Connections首选项。对于在防火墙后面的环境中工作的用户来说，网络代理的配置是必要的第一步;如果没有这种配置，将无法访问Oomph的互联网资源。所有已配置的网络设置都将传播到已安装的产品。
打开相当于Eclipse的Window → Preferences ... → General → Network Connections → SSH2首选项。在使用基于SSH访问的任何技术时，在初始安装期间确认SSH2设置是个好主意，例如，如果您计划通过公钥加密克隆Git项目。所有已配置的SSH2设置都将传播到已安装的产品。
将安装程序本身更新为最后一个Internet托管版本。该向导始终检查更新是否可用。如果是这样，此按钮是动画的，否则它被禁用。
指示您当前使用的版本以及该版本的版本。这在报告问题时特别有用。它充当了一个带来about对话框的链接。

以下下载适用于此Eclipse RCP应用程序。

  • Windows 64位
  • Windows 32位
  • Mac OS 64位
  • Linux 64位
  • Linux 32位
这个下载可以像Eclipse本身一样“安装”。
