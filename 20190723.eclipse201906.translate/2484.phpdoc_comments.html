phpDoc块评论

PDT提供了一种预设方法，通过在将语句，类，类变量和常量包含在代码中时提供输入行，将phpDoc注释添加到文件中。系统会提示开发人员立即添加描述，以确保添加的元素在其上下文中实时记录。

phpDoc块是描述性注释，是应用程序代码的一部分。它们用于在元素出现的代码中的确切位置描述PHP元素。该块由简短描述，长描述和phpDoc标记组成。

例：

为以下函数创建phpDoc Block注释时：

function add ($a, $b) {
    return $a + $b;
}

将创建以下注释：

/**
 * Enter description here...
 *
 * @param unknown_type $a
 * @param unknown_type $b
 * @return unknown
*/

现在应使用相关描述和参数编辑注释。

为代码元素添加的描述也会自动添加到内容辅助库中，以便下次使用代码元素时，可以从内容辅助列表中轻松获取。元素的描述也将出现在“大纲”视图中。

注意：

PDT为代码注释中声明的魔术成员提供内容辅助支持。有关详细信息，请参阅内容辅助概念。phpDoc块也可以作为创建PHPDoc的输入。
