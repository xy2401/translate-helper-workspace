启动配置类型图像

标识符：
org.eclipse.debug.ui.launchConfigurationTypeImages

描述：
此扩展点提供了一种将图像与启动配置类型相关联的方法。

配置标记：

<！ELEMENT 扩展 （ launchConfigurationTypeImage *）>

<！ATTLIST扩展

点CDATA #REQUIRED>

  • point - 目标扩展点的完全限定标识符

<！ELEMENT launchConfigurationTypeImage EMPTY>

<！ATTLIST launchConfigurationTypeImage

id CDATA #REQUIRED

configTypeID IDREF #REQUIRED

icon CDATA #REQUIRED>

  • id - 指定此启动配置类型映像的唯一标识符。
  • configTypeID - 指定启动配置类型的完全限定ID。（在2.1中，也可以使用“type”属性指定此属性，以与launchConfigurationTabGroups扩展点一致）。
  • icon - 指定图像文件的插件相对路径。

例子：
以下是启动配置类型图像扩展点的示例：


 <extension point="org.eclipse.debug.ui.launchConfigurationTypeImages">
  <launchConfigurationTypeImage
     id="com.example.FirstLaunchConfigurationTypeImage"
     configTypeID="com.example.FirstLaunchConfigurationType"
     icon="icons/FirstLaunchConfigurationType.png">
  </launchConfigurationTypeImage>
 </extension>
