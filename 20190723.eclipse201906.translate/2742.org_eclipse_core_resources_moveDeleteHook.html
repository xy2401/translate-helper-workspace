移动/删除挂钩

标识符：
org.eclipse.core.resources.moveDeleteHook

以来：
2.0

描述：
用于提供要在IResource.move和IResource.delete机制中使用的IMoveDeleteHook的实现。此扩展点最多可容忍一个扩展。

配置标记：

<！ELEMENT 扩展 （ moveDeleteHook ？）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT moveDeleteHook EMPTY>

<！ATTLIST moveDeleteHook

class CDATA #REQUIRED>

  • class - 实现org.eclipse.core.resources.team的类的完全限定名称。IMoveDeleteHook

例子：
以下是使用moveDeleteHook扩展点的示例。（在文件plugin.xml中）
<extension point = “org.eclipse.core.resources.moveDeleteHook” > <moveDeleteHook class = “org.eclipse.team.internal。MoveDeleteHook“ /> </ extension>

API信息：
class属性的值必须表示org.eclipse.core.resources.team的实现。IMoveDeleteHook 。

提供的实施：
Team组件通常会提供move / delete钩子的实现。任何其他客户端都不应使用扩展点。
