添加视图处理程序

view-handler元素包含将在请求处理生命周期的Restore View和Render Response阶段期间调用的具体ViewHandler实现类的完全限定类名。faces实现必须提供此类的默认实现。

要添加视图处理程序：

  1. 在J2EE透视图的Project Explorer中选择faces-config.xml 。

    将打开JSF应用程序配置编辑器。

  2. 选择编辑器的“ 其他”选项卡。

    出现“其他”页面。

  3. 展开“ 视图处理程序”部分。

  4. 单击添加 。

    将出现“查找类型”对话框。

  5. 在“ 选择类型”字段中键入要使用的类型的第一个字母。

    以您键入的字母开头的可用类型列表将显示在“匹配类型”列表中。

  6. 选择要使用的类型，然后单击“ 确定” 。

    视图处理程序已添加到列表中。
