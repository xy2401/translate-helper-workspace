Link

SWT Link widgets.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color for links.

  • background-image

    Background image or gradient for links.

  • background-position

    Background image position for links.

  • background-repeat

    Background image repetition for links.

  • border

    The border that is drawn around a link.

  • border-bottom

    The bottom side border for links.

  • border-left

    The left side border for links.

  • border-right

    The right side border for links.

  • border-top

    The top side border for links.

  • color

    Text color for links.

  • font

    Font for link texts.

  • padding

    Padding (inner distance) for links.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Indicates that the widget should have a border.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies when Link is disabled.

Link-Hyperlink

Embedded hyperlinks in a Link widget.

Available CSS Properties

Properties that can be defined for this element:

  • color

    Text color for hyperlinks in Link widget.

  • text-decoration

    Decoration for hyperlinks text.

  • text-shadow

    Shadow for hyperlinks in Link widget.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to hyperlinks of a disabled Link widget.

  • :hover

    Applies while the user hovers over a hyperlink.
