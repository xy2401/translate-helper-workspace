创建JBoss BPEL服务器

一旦安装了JBoss BPEL Runtime-RiftSaw，就可以为JBoss BPEL运行时创建一个服务器。

  • 选择Windows> Show View> Other ...打开Servers视图。 然后是服务器>服务器 。

  • 右键单击Servers视图，然后选择New> Server以打开New Server向导：

  • 选择JBoss AS 5.1作为服务器类型。

    注意

    请注意，只有JBoss As 5.1或更高版本支持BPEL。

  • 单击“ 下一步”按钮。在下一页上，您应该输入JBoss As™位置 。然后单击下一步 。你会得到这样的页面：

  • 选择HelloWorld ，然后单击“ 添加”按钮将项目添加到服务器。最后，单击Finish 。

    通过右键单击服务器并选择“ 启动”项来启动服务器。

    有关此过程的更多详细信息，请阅读JBoss Server Manager参考指南 。

  • 您可以在Web浏览器中输入http：// localhost：8080 / bpel-console / app.html链接以访问已部署的进程。
