2.9.2. Converting the project and other things to do

After installing the converter, a project must be loaded to start the conversion with "Convert project to java for use in API" . If there are problems during the conversion they will be shown in the console window or an error window will occur.

For compiling the converted project you need two additional classes (RuntimeContext and VariableStore). These classes are used in many places in the converted project, therefore we recommend to use our provided target project.
