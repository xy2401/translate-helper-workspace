系统摘要部分

标识符：
org.eclipse.ui.systemSummarySections

以来：
3.0

描述：
Eclipse UI提供了一个AboutDialog，可以由客户端产品插件进行标记和重用。此对话框包含包含配置详细信息的SystemSummary信息。通过扩展org.eclipse.ui.systemSummarySections扩展点，客户端可以将自己的信息放入日志中。

配置标记：

<！ELEMENT 扩展 （ 部分 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 部分 EMPTY>

<！ATTLIST部分

id CDATA #IMPLIED

sectionTitle CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 可选的唯一名称，用于标识此系统摘要部分
  • sectionTitle - 一个可翻译的名称，将在系统摘要中显示为此部分的标题
  • class - 实现org.eclipse.ui.about.ISystemSummarySection 。该类必须提供默认构造函数。

例子：
以下是systemSummarySections扩展的示例：

<extension
 point="org.eclipse.ui.systemSummarySections">
 <section
  id="RCPBrowser.CookieDetails"
  sectionTitle="Browser Cookies"
  class="org.eclipse.ui.examples.rcp.browser.CookieConfigDetails"/>
</extension>

API信息：
section元素中指定的类必须是org.eclipse.ui.about.ISystemSummarySection 。

提供的实施：
Workbench使用此扩展点在系统摘要对话框中提供以下部分：
  • 系统属性：java.lang返回的属性。System.getProperties（）。
  • 功能：已安装的功能。
  • 插件注册表：已安装的插件及其状态。
  • 用户首选项：已从其默认值修改的首选项。
