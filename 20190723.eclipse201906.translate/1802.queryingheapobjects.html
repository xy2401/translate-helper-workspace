查询堆对象（OQL）

Memory Analyzer允许使用类似SQL的查询查询堆转储。OQL将类表示为表，将对象表示为行，将字段表示为列。

SELECT * FROM [INSTANCEOF] <class name =“name”> [WHERE <filter-expression>] </ filter-expression> </ class>

要打开OQL编辑器，请使用工具栏按钮 ：

OQL编辑器分为两个区域：

  • 要在查询中键入的文本区域（上部区域）
  • 结果区域显示查询执行的结果

使用F5或Ctrl-Enter或工具栏按钮执行查询。

基本的OQL语法如下：

SELECT * FROM [INSTANCEOF] <类名> [WHERE <filter-expression>]

文本区域具有类名称，类名正则表达式，字段名称，属性和方法的自动完成。请参阅OQL自动完成 。

导航历史记录显示以前的OQL查询。单击上一个查询再次执行它。在许多结果窗格中，选择一些对象，然后复制 > OQL查询复制对象的OQL表示。然后可以将此OQL粘贴到OQL编辑器文本区域中。这适用于OQL结果区域本身，还可以复制列详细信息以及对象。
