Includepath变量初始值设定项

标识符：
org.eclipse.wst.jsdt.core。JsGlobalScopeVariableInitializer

以来：
2.0

描述：
此扩展点允许客户端提供自定义includepath变量初始值设定项，用于延迟绑定includepath变量。

配置标记：

<！ELEMENT 扩展 （ JsGlobalScopeVariableInitializer *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT JsGlobalScopeVariableInitializer EMPTY>

<！ATTLIST JsGlobalScopeVariableInitializer

变量CDATA #REQUIRED

class CDATA #REQUIRED

不推荐使用CDATA #IMPLIED

readOnly（true | false）>

  • 变量 - 标识将为其激活此初始值设定项的变量的唯一名称。
  • class - 实现此变量初始化程序的类。该类必须实现org.eclipse.wst.jsdt.core.JsGlobalScopeVariableInitializer 。
  • deprecated - 解释不推荐使用关联变量的原因的字符串
  • readOnly - 表示无法修改关联的变量

例子：
为名为“FOO”的类路径变量声明JsGlobalScopeVariableInitializer示例：
                                                                       
<extension point="org.eclipse.wst.jsdt.core.JsGlobalScopeVariableInitializer">            
   <JsGlobalScopeVariableInitializer                                          
      variable="FOO"                                                        
      class="com.example.CPVInitializer"/>                           
</extension>
