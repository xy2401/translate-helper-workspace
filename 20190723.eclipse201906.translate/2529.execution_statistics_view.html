执行统计视图[PHP Profile Perspective]

“执行统计信息”视图显示在分析过程中调用的文件列表以及有关文件中元素的处理时间的详细信息。

该窗口包含与每个元素相关的统计信息，如下所示

  • 功能 - 功能的名称和位置。
  • Calls Count - 调用函数的次数。
  • 平均拥有时间 - 没有内部呼叫的平均持续时间。
  • Own Time（s） - 没有内部呼叫的净进程持续时间。
  • 其他时间 - 调用其他文件所花费的时间。
  • 总时间（s） - 处理所需的总时间。

注意：

单击工具栏上的“显示为百分比”按钮，将统计信息视为百分比而不是时间。

上下文菜单

右键单击列表中的函数可以选择打开函数调用统计信息 。这将打开一个视图，其中包含有关所选函数的统计信息，调用它的函数以及它调用的函数。
