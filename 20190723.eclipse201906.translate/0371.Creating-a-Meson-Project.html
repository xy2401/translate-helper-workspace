创建一个Meson项目

要创建一个介子项目，使用文件- >新建- > C / C ++项目要么找到介子项目中的所有项目的列表，或在左栏点击介子过滤掉非介子项目。

有两种选择：空项目或Meson项目。由于Meson项目配置它们是否支持C，C ++或两者，因此无需像旧Managed Build项目那样指定语言。Empty Project模板意味着没有文件将提供给新项目，而Meson Project是一个使用Meson配置构建的示例C hello world程序。

要使用已在系统中签出的Meson项目，请选择“空项目”模板，然后将项目的位置更改为指向项目在系统上的位置。

Meson插件将在构建目录中执行构建，每个配置都有一个单独的目录。 （例如{$项目} /编译/默认）。默认情况下，有两种基本配置， 运行和调试 。
