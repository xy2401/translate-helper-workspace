新侦察项目向导

New Scout Project向导可用于从头开始创建新的Scout项目。

要打开向导按Ctrl + N或使用文件 ▸ 新建 ▸ 其他...，然后键入“ 侦察兵 ”到奇才搜索字段。然后，选择条目“ Scout Project ”并单击Next 。这将导致New Scout Project向导的初始对话框， 如图1所示。

图1。新的项目向导

下一节将详细介绍图1中的单个向导字段。

通过单击Finish按钮，将启动向导，并以Maven多模块项目的形式创建新的Scout客户端服务器应用程序。

向导字段和默认值

图1的所有字段最初都填充了默认值。

组ID

Maven groupId用于所有创建的项目。默认值为org.eclipse.scout.apps 。

工件ID

父项目的Maven artifactId。其他项目源自此名称。helloworld是默认值。

显示名称

提交给用户的应用程序的名称。此名称显示在浏览器标题栏中。默认值为“我的应用程序”

使用“ 项目位置”组框，您可以控制项目的创建位置。取消选中使用默认工作区位置复选框，在目标目录字段中输入其他值。浏览按钮可以帮助您找到合适的路径。

创建组件

使用图1向导，可以创建一个完整的Maven多模块项目。使用默认工件Id helloworld创建以下Maven模块。

  • Maven模块helloworld

    • 包含项目的父pom.xml文件

  • Maven模块helloworld.client

    • 包含src/main/java客户端应用程序的模型组件和src/test/java模型测试。

    • 包HelloWorldForm的类org.eclipse.scout.apps.helloworld.client.helloworld是模型类的一个示例。

  • Maven模块helloworld.shared

    • 包含客户端和服务器应用程序中所需的组件。

    • 例如，请参阅IHelloWorldService的src/main/java接口和HelloWorldFormData src/generated/java类。

    • 可以在Scout NLS编辑器中打开的Texts.nls文件。

  • Maven模块helloworld.server

    • 包含src/main/java服务器应用程序的模型组件和src/test/java模型测试。

    • 包HelloWorldService的类org.eclipse.scout.apps.helloworld.server.helloworld就是这种模型类的一个例子。

  • Maven模块helloworld.server.app.dev

    • 包含从Eclipse IDE中运行Scout服务器应用程序的所有组件。

    • 文件config.properties中的文件src/main/resources包含Scout服务器应用程序的开发配置。

    • 文件pom.xml将Jetty Web服务器与服务器应用程序捆绑在一起。

    • 文件[webapp] dev server.launch包含Eclipse IDE的启动配置。

  • Maven模块helloworld.server.app.war

    • 包含创建Scout服务器WAR文件以部署到外部Web服务器的所有组件。

    • 文件config.properties中的文件src/main/resources包含服务器配置。

    • 文件pom.xml用于构建Scout服务器WAR文件。

  • Maven模块helloworld.ui.html

    • 包含servlet过滤器和HTML页面以及Scout UI Server的自定义CSS和JavaScript文件。

    • 请参阅UiServletFilter类src/main/java和WebContent source/main/resources文件夹。

  • Maven模块helloworld.ui.html.app.dev

    • 包含从Eclipse IDE中运行Scout UI应用程序的所有组件。

    • 文件config.properties中的文件src/main/resources包含应用程序的开发配置。

    • 文件web.xml中的文件src/main/webapp包含应用程序的Web配置。

    • 文件pom.xml将Jetty Web服务器与应用程序捆绑在一起。

    • 文件[webapp] dev ui.launch包含Eclipse IDE的启动配置。

  • Maven模块helloworld.ui.html.app.war

    • 包含创建Scout UI WAR文件以部署到外部Web服务器的所有组件。

    • 文件config.properties中的文件src/main/resources包含应用程序配置。

    • 文件web.xml中的文件src/main/webapp包含Web配置。

    • 文件pom.xml用于构建Scout UI WAR文件。
