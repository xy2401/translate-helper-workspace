IBM Platform MPI

本节介绍如何使用IBM Platform MPI将运行配置设置为并行工具平台（PTP）的目标系统配置。

从PTP运行IBM Platform MPI程序时，首先创建一个C / C ++ / Fortran项目，然后构建它。有关更多信息，请参阅

  • 介绍创建MPI项目 ，其中包括有关构建的信息
  • 使用PTP的PLDT设置MPI项目
  • 同步项目 - 在远程计算机上创建和构建项目

如运行并行程序中所述 ，接下来通过运行>运行配置创建运行配置...并创建一个新的并行应用程序 。

使用PTP运行IBM Platform MPI程序的细节在运行配置的“ 资源”选项卡上指定。对于Target System Configuration ，选择IBM Platform MPI

然后指定一个连接。如果您已经指定了连接，例如，对于同步项目或远程项目，它将被选中。如果要启动本地应用程序，请选择“本地”单选按钮。

请参阅创建远程连接和本地连接

注意：要在创建连接后编辑连接信息，例如更改用户名或密码，请参阅编辑远程连接 。

填写您需要的IBM Platform MPI选项。在Basic Options选项卡上，我们指定了4个进程。

在“ 高级选项”选项卡上，可以指定其他参数;使用复选框将填写已检查的参数。

现在继续运行配置设置以运行并行应用程序 ，方法是在“ 应用程序”选项卡上指定应用程序，以及所需的任何其他信息。
