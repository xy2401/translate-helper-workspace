为IBM转储安装IBM DTFJ功能

从IBM下载站点安装IBM Diagnostic Tool Framework for Java功能，以向Eclipse Memory Analyzer添加读取和分析IBM虚拟机为Java生成的转储的功能。
  1. 选择帮助 > 安装新软件...
    出现“ 安装”对话框。
  2. 选择使用： IBM Diagnostic Tool Framework for Java - http://public.dhe.ibm.com/ibmdl/export/pub/software/websphere/runtimes/tools/dtfj/
    在主表中显示以下内容：


    IBM监视和诊断工具

  3. 选择IBM Monitoring and Diagnostic Tools > Diagnostic Tool Framework for Java
  4. 单击下一步
    在对话框的底部显示此进度消息： 计算需求和依赖项 。该对话框将更改为： 安装详细信息查看要安装的项目。


    Java诊断工具框架

  5. 单击下一步
    该对话框将更改为： 查看许可证


    必须先检查并接受许可证，然后才能安装软件。

  6. 选择我接受许可协议的条款
  7. 单击完成
    对话框关闭，然后在Eclipse Memory Analyzer窗口的底部出现消息“ 安装软件”，然后在一段时间后出现一个新的对话框窗口： 您信任这些证书吗？在对话框中有以下证书详细信息： IBM Canada Ltd;空值; IBM Canada Ltd.
  8. 单击选择全部 接受选中
    这表示您信任所提供证书的提供者。
    对话框关闭，安装继续， 安装软件指示进度。安装完成后，将显示一个新对话框： 软件更新是否要重新启动Eclipse Memory Analyzer以应用更改？
  9. 单击立即重启
    Eclipse Memory Analyzer在安装了DTFJ功能的情况下关闭并重新启动。
  10. 使用“ 文件” >“ 打开”检查功能是否已正确安装
    将出现一个文件对话框： 打开快照
  11. 单击“ 所有已知格式”
    以下列表下拉列表：


    所有已知格式
    HPROF二进制堆转储
    IBM 1.4.2 SDFF 1
    IBM Javadumps
    IBM SDK for Java（J9）系统转储
    IBM SDK for Java
    便携式堆转储
    所有的文件

  12. 单击取消
    DTFJ功能的安装已完成。
1 IBM DTFJ功能版本> = 1.12.29003.201808011034没有此选项
