配置隧道调试首选项

在使用隧道启动调试会话之前，必须确保在“调试首选项”页面中配置了正确的端口号。

此过程描述如何配置“调试”首选项以启用隧道。

要为PDT配置调试设置：

1。转到Window |打开PHP Debug Preferences页面偏好| PHP |调试 。

2。在“调试选项”部分中，确保配置了以下默认设置：

  • Zend Debugger端口号必须为10137。
  • 在“高级Zend调试器选项”类别中，广播端口必须设置为20080。
  • 在“客户端主机/ IP”类别中，输入将返回调试结果的客户端主机/ IP。PDT将自动搜索并识别客户端主机/ IP，但输入特定的主机/ IP将加快调试过程并降低会话超时的可能性。

创建隧道连接的下一步是将您的环境设置为允许的主机 。
