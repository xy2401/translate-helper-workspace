删除PHP库或库文件夹

此过程描述如何从PDT中删除用户库或库文件夹。删除用户库或库文件夹意味着其内容将不再在PDT中可用，包括其内容辅助和重构等功能。

重要的提示：

如果您的PHP用户库或库文件夹与特定项目相关联，则删除它也会将其从项目中删除。

要删除PHP库/库文件夹：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，选择要删除的用户库或库文件夹，然后单击Remove 。（删除用户库也会删除其中的所有文件夹。）您的用户库或库文件夹已删除。
  3. 要应用更改，请单击确定 。

您的用户库或库文件夹已被删除。如果要导入库，请参阅导入PHP用户库 。要将文件夹添加到现有库，请参阅将外部文件夹添加到PHP库 。
