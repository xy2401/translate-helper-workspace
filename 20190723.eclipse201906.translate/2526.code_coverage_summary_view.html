代码覆盖率摘要视图[PHP Profile Perspective]

“代码覆盖率摘要”视图显示在分析过程中涵盖了多少行代码。包含覆盖信息的表包含以下列：

  • 元素 - 调用的文件/项目。
  • 覆盖线（访问/重要/总计） - 每个文件中涵盖的线的百分比。（已访问=已覆盖的行数/有效数据=对函数调用有意义的行数/总计=文件中的总行数。）

单击“Covered lines”百分比将打开包含相关文件的Code Coverage视图 ，并突出显示覆盖的行：
