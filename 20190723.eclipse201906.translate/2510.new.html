新子菜单

新子菜单位于File |下菜单栏中的新功能。

注意：

可以通过“窗口菜单”中的“ 自定义透视图”选项配置当前透视图的“新建菜单”中可用的选项。

新子菜单下的可用选项包括：

名称描述
PHP项目在工作区内创建一个新项目，PHP配置设置允许完整的PHP功能。
项目将新项目添加到Navigator / Project Explorer视图中，可在其中创建文件。
PHP文件使用PHP标记创建新文件。允许完整的PHP功能。
夹在项目中创建一个新文件夹。
通过单击新文件夹创建对话框中的“高级”按钮，可以将新文件夹链接到文件系统中的文件夹。使用此选项会将现有文件夹插入工作区文件夹。对工作区中的文件和文件夹所做的任何更改都将自动反映在文件系统中文件的本地版本中。
CSS将级联样式表插入到项目中。
HTML在项目中创建新的HTML文件，允许使用HTML功能。
XML在项目中创建新的XML文件，允许使用XML功能。
例在工作区中创建以下示例项目：
  • GEF - （图形编辑框架）插件
  • XML - 将XML示例项目插入工作空间。
其他允许访问不在主列表中的所有其他类型的项目。
要配置主列表中可用的项目，请转到“窗口”菜单|自定义透视。

  • 连接配置文件 - 连接配置文件，连接配置文件存储库
