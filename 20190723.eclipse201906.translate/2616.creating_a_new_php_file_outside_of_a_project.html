在项目之外创建PHP文件

可以快速，轻松地在项目外部创建单个PHP文件，以便编写以后不需要调试或运行的代码的短片段，并且不需要与其他文件或项目相关联。

要创建与项目无关的新PHP文件：

  1. 从菜单栏中，转到文件|新的|其他| PHP |无标题的PHP文档 。
  2. 默认情况下，一个名为PHPDocument1的新PHP文件将在编辑器中打开。

创建文件后，可以将其保存在项目中。

要将文件保存到项目：

  1. 点击保存在工具栏上。将打开“另存为”对话框。
  2. 选择要与文件关联的项目-Or-要创建新项目：
  3. 单击“ 创建新项目” 。将显示“新建PHP项目”对话框。
  4. 输入项目名称，然后单击“ 完成” 。新项目将添加到列表中。选择它以将文件保存在新项目中。
  5. 编辑文件名。
  6. 单击确定 。

您的文件将保存在所选项目中，并可用于运行，调试和分析操作。
