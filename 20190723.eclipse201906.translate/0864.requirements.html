要求和安装

JAX-WS Tools具有以下最低要求。

  • Eclipse 3.4.0（ http://www.eclipse.org/downloads ）

  • Eclipse Web Tools Platform（WTP）3.0.1（ http://download.eclipse.org/webtools/downloads ）

JDK版本

Java 5.0是所需的最低版本，更新4及更高版本支持JDK 6.0。

http://java.sun.com/javase/downloads/

JAX-WS运行时

在使用JAX-WS Tools创建Web服务应用程序之前，必须在工作台中下载，安装和配置JAX-WS运行时。

目前，我们从2.0.6版本开始支持用于JAX-WS开发的Apache CXF运行时。建议使用CXF 2.1.x或更高版本。

可以从Apache CXF下载页面下载Apache CXF运行时。

有关使用CXF 2.6.0的说明，请参见此处 。
