先决条件

PTP 9.1安装需要以下先决条件。有关更多详细信息（包括安装说明），请参阅http://wiki.eclipse.org/PTP/release_notes

  • Java 1.8或更高版本
  • Eclipse SDK Neon（版本4.6）或更高版本
  • Eclipse C / C ++开发工具（CDT） 9.0或更高版本

PTP客户端 - 并行包

PTP现在作为单个下载包提供

Eclipse用于并行应用程序开发人员
通常称为并行包 - 包括所有必备组件（除Java之外）。这是获得PTP的首选方法。 Parallel包适用于以下平台：
操作系统 建筑
Linux的 x86，x86_64
MacOS X（10.6及更高版本） x86，x86_64
视窗 x86，x86_64

Eclipse支持的任何平台也支持Eclipse PTP客户端，但安装更复杂。有关从更新站点安装PTP的更多信息，请访问此处 。有关Eclipse支持的平台的信息，请访问此处 。

目标系统

PTP支持以下目标系统：

操作系统 建筑 调试
Linux的 x86，x86_64，ppc gdb 6.3及更高版本
MacOS X（10.6及更高版本） x86，x86_64 gdb 6.3及更高版本

PTP还支持许多并行运行时系统和作业调度程序。在使用PTP之前，这些需要在目标系统上可用。
有关当前支持的系统列表，请参阅PTP支持的目标系统配置 。

此外，对于同步项目 ，PTP需要远程计算机上的Git 。
