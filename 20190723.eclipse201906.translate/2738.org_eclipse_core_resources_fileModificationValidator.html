文件修改验证器

标识符：
org.eclipse.core.resources.fileModificationValidator

描述：
用于提供要在validate-edit和validate-save机制中使用的IFileModificationValidator的实现。此扩展点最多可容忍一个扩展。

配置标记：

<！ELEMENT 扩展名 （ fileModificationValidator ？）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT fileModificationValidator EMPTY>

<！ATTLIST fileModificationValidator

class CDATA #REQUIRED>

  • class - 实现org.eclipse.core.resources.IFileModificationValidator 。

例子：
以下是使用fileModificationValidator扩展点的示例：


 <extension point="org.eclipse.core.resources.fileModificationValidator">
  <fileModificationValidator class="org.eclipse.vcm.internal.VCMFileModificationValidator"/>
 </extension>

API信息：
class属性的值必须表示org.eclipse.core.resources.IFileModificationValidator 。

提供的实施：
Team组件通常会提供文件修改验证器的实现。任何其他客户端都应使用扩展点。
