语法着色

PDT可以自动将不同的颜色和字体属性应用于不同的PHP语法元素，以便您的脚本更易于导航，并帮助您快速轻松地查找相关部分。使用语法着色规则，您可以通过“语法着色首选项”页面为代码中的一系列不同元素设置前景色，背景色和字体类型。这样可以快速评估代码，并且可以单独看到不同的元素，从而使视觉上的错误更加清晰，更易于检测。

完成相关元素将导致应用所需的颜色和字体设置。

注意：

所有元素都有默认的白色背景。可以从“ 语法着色”首选项页面配置颜色和字体设置，从Window |访问偏好| PHP |编辑|语法着色 。

注意：

如果为项目启用了JavaScript支持，则语法Coloring也可用于JavaScript元素。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。要配置JavaScript语法着色首选项，请转到Window |偏好| JavaScript |编辑|语法着色 。
