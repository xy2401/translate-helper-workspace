任务6： 在组头中显示信用限额范围

如果报告显示每个组的信用额度范围，则报告更容易导航，而不是显示组标题中每个组的第一个值，如下所示：

0 - 49999
50000 - 99999
100000 - 149999

此过程说明如何编写JavaScript表达式以显示这些信用限额范围。该过程还说明了如何创建用于关联JavaScript表达式的列绑定。

1
选择“布局”以返回布局编辑器。
2
删除组头中的[CREDITLIMIT]数据元素。
3
通过右键单击单元格并选择“插入” →“数据”， 在其位置插入新数据元素 。新数据绑定会提示您为新数据元素创建列绑定。
4
指定列绑定的名称和表达式。使用默认数据类型String。
1
在“列绑定名称”中，键入以下名称：
CREDIT_GRP_HEADER

2
打开表达式构建器以编写JavaScript表达式。
3
在表达式构建器中，键入以下表达式：
for（i = 50000; i <300000; i + = 50000）{
if（row [“CREDITLIMIT”] <i）{
rangeStart = i-50000;
rangeEnd = i-1;
打破;
}
}
displayString = rangeStart +“ - ”+ rangeEnd;
选择确定。表达式出现在New Data Binding的Expression字段中， 如图7-8 所示 。

图7-8 定义了列绑定
4
选择“确定”以保存数据元素的列绑定。在布局编辑器中，数据元素显示列绑定名称[CREDIT_GRP_HEADER]。
五
选择数据元素。在“属性编辑器”的“属性”下的“常规”中，选择“B”将组标题格式化为粗体文本。
6
预览报告。组标题显示信用额度范围。图7-9显示了50000 - 99999组头。
