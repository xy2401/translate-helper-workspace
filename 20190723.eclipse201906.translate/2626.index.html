PHP库首选项

PHP Libraries Preferences页面允许您创建和维护外部代码库。

在项目中启用PHP库允许项目引用库，并使这些资源中的元素可用于内容辅助和重构等操作。

注意：

您可以使用向上和向下按钮更改用户库所在的顺序。在此页面中排列库的顺序定义了它们在PDT功能中可用的顺序，例如内容辅助。

PHP Libraries Preferences页面允许您执行以下操作：

  • 添加PHP库
  • 将外部文件夹添加到PHP库
  • 导出PHP用户库
  • 导入PHP用户库
  • 编辑PHP用户库
  • 编辑PHP库组件或文件夹
  • 删除PHP库或库文件夹

从Window |访问PHP Libraries Preferences页面偏好 | PHP | PHP 库 。

注意：

在PHP首选项页面中添加用户库后，还必须将其添加到项目的PHP包含路径中，以使其可用。有关更多信息，请参阅配置项目的PHP包含路径 。
