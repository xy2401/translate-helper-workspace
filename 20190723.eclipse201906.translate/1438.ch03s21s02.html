3.21.2. Starting the AUT

To start the AUT, you must:

  • have defined an AUT (Section 3.7, “Defining applications under test (AUTs)”)

  • have connected to the AUT Agent (Section 3.2.3, “Connecting to the AUT Agent”)

  • if you want to start your AUT from the ITE, and not using the autrun option, then you must have configured an AUT (Section 3.8.1, “Configuring AUTs to be started from the ITE”)

  1. On the toolbar, click on the arrow next to the ”Start AUT” button .

  2. Select an AUT and the configuration to start it with from the list.

  3. Clicking directly on the ”Start AUT” button will start the most recently started AUT (marked with a tick in the list of AUTs and configurations).

Once the AUT has been started, it will appear in the running AUTs view.

Further information on starting specific types of AUT is available in the toolkit-specific details (Chapter 4, Toolkit-specific information).
