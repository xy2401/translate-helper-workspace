方法细节
{串| $ } attr （属性，值）
一种设置或获取HTML属性值的方法。请注意，属性“id”和“class”不能以这种方式设置。
参数：
{string | Object} 属性
要返回或修改的属性的名称。或者，可以设置具有键值对的普通对象。
{string} value 可选
要将属性设置为的值。
返回：
{串| $ }如果仅使用字符串调用函数，则为给定属性的值。否则引用此对象。
{串| $ } css （属性，值）
设置或获取CSS属性值的方法。请注意，RAP可能会覆盖对窗口小部件样式属性所做的任何更改，并且更改不会应用于Java端窗口小部件实例。修改RAP未使用的属性是安全的。仅支持控件小部件的元素。
参数：
{string | Object} 属性
要返回或修改的属性的名称。或者，可以设置具有键值对的普通对象。
{string} value 可选
要将属性设置为的值。
