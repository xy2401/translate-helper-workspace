根据JavaScript语法着色首选项页面中的配置，JavaScript代码中包含的任何代码都将应用语法着色

JavaScriptSyntax着色配置

JavaScript语法着色选项可以从JavaScript语法着色首选项页面进行配置。

配置JavaScript语法着色选项：

  1. 转到从Window |访问的JavaScript语法着色首选项页面偏好| JavaScript |编辑|语法着色 。
  2. 从“语法元素”列表中选择所需的项。要为元素启用“语法着色”，请确保标记“启用”复选框。
  3. 选择要应用于文本的颜色。
  4. 选择要应用于文本的格式（如果有）（粗体，斜体，删除线，下划线）。

注意：

“示例”文本框显示不同元素的预览。

单击“ 应用”和“ 确定”以应用并保存设置。
