OclComparable

OclComparable类型定义sortedBy迭代使用的compareTo操作。只能对提供派生的compareTo实现的类型进行排序。

符合OclAny

操作

<(that : OclSelf[?]) : Boolean[1]优先级： RELATIONAL

如果self低于that就是真的。

<=(that : OclSelf[?]) : Boolean[1]优先级： RELATIONAL

如果self小于或等于that则为真。

>=(that : OclSelf[?]) : Boolean[1]优先级： RELATIONAL

如果self大于或等于that则为真。

>(that : OclSelf[?]) : Boolean[1]优先级： RELATIONAL

如果self大于that就是真的。

compareTo（即：OclSelf [？]）：整数[1]

根据self是小于，等于还是大于，返回-ve，0，+ ve。

compareTo操作应该是可交换的。
