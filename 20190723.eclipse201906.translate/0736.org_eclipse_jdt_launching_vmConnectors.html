Java VM连接器

标识符：
org.eclipse.jdt.launching.vmConnectors

描述：
此扩展点表示与远程VM的不同类型的连接。每个扩展必须实现org.eclipse.jdt.launching.IVMConnector 。IVMConnector负责与远程VM建立连接。

配置标记：

<！ELEMENT 扩展 （ vmConnector *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT vmConnector EMPTY>

<！ATTLIST vmConnector

id CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 可用于引用此IVMConnector的唯一标识符。
  • class - 实现此连接器的类。该类必须实现IVMConnector 。

例子：
以下是IVMConnector的示例：


 <extension point="org.eclipse.jdt.launching.vmConnectors">
  <vmConnector
   class="com.example.ConnectorImplementation"
   id="com.example.ConnectorId">
  </vmConnector>
 </extension>

提供的实施：
提供了标准插座连接连接器的实现。
