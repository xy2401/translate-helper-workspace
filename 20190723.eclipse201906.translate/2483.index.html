代码评论

评论您的代码涉及添加字符（通常是斜杠和星号），这些字符将某些代码区域标记为“注释”。

注释仅用于参考信息，不会作为代码的一部分运行。在代码中注释所有函数，类和方法是一种很好的编程习惯。这有助于开发人员和可能查看代码的其他人了解其目的。

评论可以是单行或多行：

  • 单行注释将具有以下格式：
// This is a single line comment.
  • 多行注释将具有以下格式：
/* This is
a comment
on multiple
lines
*/

注意：

如果为项目启用了JavaScript支持，则代码注释也将可用于JavaScript元素。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。
