资源

标识符：
org.eclipse.rap.ui.resources

以来：
RAP 1.0

描述：
resources扩展点用于注册静态资源（如图像或JavaScript文件），以便为客户端提供这些资源。可以通过RWT.getResourceManager().getLocation( String )获取的URL访问该资源。

配置标记：

<！ELEMENT 扩展 （ 资源 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 资源 （ dependsOn ）*>

<！ATTLIST资源

id CDATA #IMPLIED

class CDATA #REQUIRED>

  • id - 资源实例的可选标识符。
  • class - org.eclipse.rap.ui.resources.IResource用于定义此资源的位置，选项和其他方面。

<！ELEMENT 依赖于 EMPTY>

<！ATTLIST依赖于

resourceId IDREF #REQUIRED>

  • resourceId - resourceId用于标识此资源所依赖的资源。

例子：

<extension
    id="example.customwidget.customjs"
    point="org.eclipse.rap.ui.resources">
  <resource class="example.customwidget.CustomJavaScriptResource"/>
</extension>

API信息：
除了使用扩展注册静态资源外，您还可以使用资源管理器动态注册新资源。使用RWT.getResourceManager()获取当前资源管理器实例。
