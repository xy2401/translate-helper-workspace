标识符：
org.eclipse.datatools.sqltools.common.ui.preferenceSections

[输入出现此扩展点的第一个版本。]

描述：
此扩展点为数据服务器供应商提供了一种向现有DTP首选项页面提供首选项控制的方法。所有首选项设置都将显示在供应商名称为标题的选项卡中。

配置标记：

<！ELEMENT 扩展 （ 部分 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 部分 EMPTY>

<！ATTLIST部分

dbdefinitionid CDATA #REQUIRED

class CDATA #REQUIRED

页CDATA #REQUIRED

launchConfig（true | false）>

  • dbdefinitionid - 由Model Base项目定义的db定义标识。（目前，客户端应使用productName_version作为此属性。）
  • class - 实现org.eclipse.datatools.sqltools.common.ui.preferences的完全限定类。IDataServerPreferenceSection。
  • page - 首选项页面ID。该页面应扩展AbstractDBPreferenceFieldPage，以便可以显示供应商特定的设置。
  • launchConfig - 此部分是否可以在启动配置对话框中重用。如果为true，“class”应该实现org.eclipse.datatools.common.ui.preferences。IDataServerLaunchPreferenceSection。
