支持Swing和SWT GroupLayout

  • 从Swing Layouts或SWT Layouts面板中选择GroupLayout
  • 使用GroupLayout约束的位置和大小小部件
  • 水平和垂直对齐小部件
  • 智能缩进小部件
  • 智能地插入小部件
  • 沿着文本基线对齐小部件
  • 将小部件与边距对齐
  • 将小部件固定在左侧，右侧或两侧
  • 锚定小部件到顶部，底部或两侧
  • 通过弹出上下文菜单设置锚点
  • 通过上下文菜单设置窗口小部件锚定和调整大小
  • 使用工具栏对齐和居中控件
  • 使用浮动布局助手设置对齐，大小和间距
  • 用于控制反馈颜色，弹出对齐菜单和代码生成的首选项
  • 从Swing GroupLayout移植SWT GroupLayout

支持创建和编辑RCP ActionBarAdvisors

  • 用于创建ActionBarAdvisor类的新向导
  • ActionBarAdvisor调色板包括常见和本地操作
  • 使用拖放操作创建工具栏和按钮
  • 使用“操作”选项卡编辑Coolbar和菜单
  • 预先填充ActionFactory类中定义的操作的操作列表
  • 拖动菜单类别以创建完全填充的菜单
  • 更新关联的WorkbenchWindowAdvisor类
  • 支持ToolBarManager样式属性

新的调色板经理

  • 使用Palette Manager对话框配置调色板类别和条目
  • 使用调色板菜单添加类别，条目和分隔符
  • 从JAR归档导入多个小部件
  • 使用拖放重新排列项目
  • 恢复默认调色板
  • Swing和SWT的新调色板扩展点

增强的NLS支持

  • 在将键重命名为现有键时询问是否使用旧值或新值
  • 改进了对默认语言环境的支持
  • 支持VA Java NLS模式

增强的SWT支持

  • 增强的绝对/空布局的捕捉反馈
  • 支持Eclipse 3.2 ExpandBar和ExpandItem
  • 支持菜单项的“Expose control”
  • 支持GridLayout（cols，equal）构造函数
  • 增强的GridData代码生成
  • 支持Accessibility属性和AccessibleListener
  • 支持将Forms API工具包添加到任何窗口
  • 自动将Forms API JAR添加到.classpath和插件清单
  • 向Shell和Composite向导添加了Superclass字段
  • 对自定义静态工厂的基本支持
  • 将复合/组合变形为SashForm

增强的JFace和RCP支持

  • 支持StructuredViewer
  • 增强了对JFace对话框的解析
  • 支持静态工厂中的观众

增强的Swing支持

  • 增强的绝对/空布局的捕捉反馈
  • 支持默认的GridBagLayout插件
  • 支持在main（）方法中读取/更改LAF
  • 在SpringLayout中显示网格的选项
  • 支持菜单项的“公开组件”
  • 支持JGoodies Forms版本1.0.6
  • 将超类字段改为JPanel向导

代码生成和解析增强功能

  • 在widget中设置变量名称的选项：setData（“name”，“widget-name”）
  • 忽略“if（！）中的代码Beans.isDesignTime（））“在解析期间

编辑和财产窗格增强功能

  • 最大化“设计”页面激活编辑器的选项
  • 在编辑器保存上格式化源代码的选项
  • 在编辑器中没有选择时显示“<No Properties>”消息
  • 在类路径模式下从源文件夹和引用的项目中选择图像
  • 在Designer首选项中保存属性表列宽
