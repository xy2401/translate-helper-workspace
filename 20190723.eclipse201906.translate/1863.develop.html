对象团队开发工具

OTDT是OT / J编程语言的开发环境。

它基于Eclipse Java开发工具（JDT），包括以下主要功能，这些功能将在下一章中介绍。

OTDT的主要特点

更多详细信息可在综合功能列表中找到 。
  • 使用向导创建对象团队项目
  • 使用向导创建团队和角色类
  • 编辑 OT / J：
    • 语法突出显示， 导航 ， 内容/代码辅助 ， 基本代码注释
    • 比较绑定方法以促进角色与其基础的共同演化
    • 重构对象团队计划
  • 增强的结构/导航视图和表格编辑器
    • 基于表的绑定编辑器专门支持创建连接器
    • 包浏览器 ，支持角色文件/团队包
    • 大纲显示所有OT / J元素
    • 键入层次结构 ，支持团队继承
    • 调用层次结构，显示由callout / callin方法绑定引起的控制流
  • 编译对象团队程序
  • 运行对象团队程序
  • 调试对象团队程序

先决条件

如果您之前从未使用过eclipse Java IDE，请考虑熟悉Java Development Tooling ，因为OTDT是建立在JDT之上并扩展了JDT的。

反馈

如果您对OT / J语言有疑问，或者您遇到OTDT的问题或错误，请不要犹豫，发布到对象团队论坛 。
