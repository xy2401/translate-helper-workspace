3.10.2. Opening items in editors

To open an existing Test Case, Test Suite or Test Job in an editor, you can either double-click the item you want to open in its browser or you can select:

Open With-->... Editor

from the context-sensitive menu for the item.

If you double-click on an item in a browser that is reused in another item (e.g. you double-click a Test Case that is reused in another Test Case), the editor for its direct parent will open, and the item you double-clicked is selected.

You can also open an existing Test Case without having to select it from the Test Case Browser using the Open Test Case dialog (Section 3.12.4, “Opening existing Test Cases”).
