搜索结果查看页面

标识符：
org.eclipse.search.searchResultViewPages

以来：
3.0

描述：
此扩展点允许客户端将页面插入搜索结果视图。

配置标记：

<！ELEMENT 扩展 （ viewPage ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符

<！ELEMENT viewPage EMPTY>

<！ATTLIST viewPage

searchResultClass CDATA #REQUIRED

class CDATA #REQUIRED

id CDATA #REQUIRED

标签CDATA #IMPLIED

icon CDATA #IMPLIED

helpContextId CDATA #IMPLIED>

  • searchResultClass - 此搜索结果页面应显示的ISearchResult实现的完全限定类名。
  • class - 实现此搜索结果页面的完全限定类名。该类必须实现org.eclipse.search.ui.ISearchResultPage 。
  • id - id，通常与完全限定的类名相同
  • label - 可在搜索结果页面中使用的可翻译标签，列出所有可用的搜索结果视图。它目前尚未使用，但可能在将来使用。
  • icon - 可在搜索结果页面中的所有可用搜索结果视图列表中使用的图标。它目前尚未使用，但可能在将来使用。
  • helpContextId - 页面的帮助上下文。在3.4中添加。

例子：
例如，这是文件搜索结果页面的标记。 


<extension
 id="FileSearchPage"
    point="org.eclipse.search.searchResultViewPages">
 <viewPage
  id="org.eclipse.search.text.FileSearchResultPage"
  searchResultClass="org.eclipse.search.internal.ui.text.FileSearchResult"
  class="org.eclipse.search.internal.ui.text.FileSearchPage">
 </viewPage>
</extension>

API信息：
贡献的类必须实现org.eclipse.search.ui.ISearchResultPage

提供的实施：
搜索插件提供文件搜索的搜索结果页面。
