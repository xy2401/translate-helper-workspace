调试模型上下文绑定

标识符：
org.eclipse.debug.ui.debugModelContextBindings

以来：
3.0

描述：
此扩展点提供了一种机制，用于指定应与给定调试模型关联的上下文。Debug视图使用这些绑定自动启用上下文。当选择Debug视图中提供IDebugModelProvider适配器或具有指定调试模型标识符的堆栈帧的元素时，将启用具有给定标识符的上下文。

配置标记：

<！ELEMENT 扩展 （ modelContextBinding *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT modelContextBinding EMPTY>

<！ATTLIST modelContextBinding

debugModelId CDATA #REQUIRED

contextId IDREF #REQUIRED>

  • debugModelId - 指定此绑定所针对的调试模型标识符
  • contextId - 指定应与给定调试模型关联的上下文的上下文标识符

例子：
以下是调试模型上下文绑定贡献的示例：
<extension point = “org.eclipse.debug.ui.debugModelContextBindings” > <modelContextBinding contextId = “com.example.myLanguage.debugging” debugModelId = “com.example.myLanguageDebugModel” > </ modelContextBinding> </ extension>
在上面的示例中，当选择具有调试模型标识符“com.example.myLanguageDebugModel”的堆栈帧时，将启用具有标识符“com.example.myLanguage.debugging”的上下文。
