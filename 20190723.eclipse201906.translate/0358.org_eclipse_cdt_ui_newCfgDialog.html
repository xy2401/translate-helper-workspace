newCfgDialog

标识符：
org.eclipse.cdt.ui.newCfgDialog

以来：
4

描述：
对此扩展点的贡献定义了新配置创建的特定对话框。此对话框将从“管理配置”屏幕而不是标准（独立于托管构建系统）对话框中调用。
贡献扩展通常与整个托管构建系统（MBS）一起提供，因此新对话框可能能够处理某些特定于MBS的功能。
为了区分单独的MBS的多个对话框，mbs_id元素应该与相应的MBS Id相同。

配置标记：

<！ELEMENT 扩展 （ 对话框 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 对话框 EMPTY>

<！ATTLIST对话框

class CDATA #REQUIRED

标题CDATA #REQUIRED

mbs_id IDREF #REQUIRED>

  • class - 新配置对话框类。
    应该实现org.eclipse.cdt.ui.newui.INewCfgDialog
  • title - 新配置对话框的初始标题。
  • mbs_id - 相应的托管buld系统（MBS）的ID。
    仅当当前项目的MBS ID等于此属性时，才会显示该对话框。
