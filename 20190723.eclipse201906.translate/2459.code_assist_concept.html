内容辅助

内容辅助功能允许选择和插入现有代码元素以完成部分输入的代码。

根据光标的上下文，可以在文件的相关位置显示可能的代码元素列表，然后您可以选择将其自动输入到代码中。

每种类型的代码元素都有一个唯一的图标。有关每个图标的说明，请参阅PHP图标。

内容辅助使用以下元素：PHP类，函数，变量，常量，关键字，接口，属性，值，嵌套函数，名称，语法和包含调用，以及所有用户定义的类，函数和常量。

注意：

如果为项目启用了JavaScript支持，则内容辅助也将适用于JavaScript元素。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。
