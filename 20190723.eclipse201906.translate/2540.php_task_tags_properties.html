PHP任务标签属性

PHP任务标签属性页面允许您添加新任务标签并编辑现有标签。任务用作提醒操作，工作或程序员要求的任何其他操作。任务标记是PDT使用的字符串，用于识别脚本中添加任务的时间。注释中的这些字符串之后的任何内容都将被识别为任务。

要访问“PHP任务标记属性”页面，请在“项目资源管理器”视图中右键单击PHP项目，然后选择“属性”|资源 - 或 - 选择项目，然后从菜单栏转到项目|属性| PHP任务标签属性。

要为项目配置PHP任务标记属性：

  1. 选中“启用项目特定设置”复选框。
  2. 根据您的喜好配置设置。有关可用设置的详细信息，请参阅任务标签首选项 。
  3. 单击“确定”以应用设置。

默认PHP任务标签可以在任务标签首选项页面中设置所有项目的属性（通过单击属性页面上的配置工作区设置链接，转到窗口|首选项| PHP |编辑器|任务标签 - 或 - 访问）。
