动态字符串替换变量

标识符：
org.eclipse.core.variables.dynamicVariables

以来：
3.0

描述：
此扩展点提供了一种用于定义字符串替换中使用的动态变量的机制。在执行字符串替换时，将使用可选参数解析动态变量的值。

配置标记：

<！ELEMENT 扩展 （ 变量 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 变量 EMPTY>

<！ATTLIST变量

名称CDATA #REQUIRED

解析器CDATA #REQUIRED

说明CDATA #REQUIRED

supportsArgument（true | false）>

  • name - 指定此变量的唯一名称。
  • resolver - 指定实现org.eclipse.core.variables.IDynamicVariableResolver ，用于确定变量的值
  • description - 指定此变量的可读描述
  • supportsArgument - 此变量是否支持参数。未指定时，隐含值为true 。

例子：
以下是动态变量的定义，该变量解析为所选资源的名称：

 <extension point="org.eclipse.core.variables.dynamicVariables">
   <variable 
      name="resource_name"
      expanderClass="com.example.ResourceNameExpander"
      description="The name of the selected resource">
   </variable>
 </extension>

API信息：
属性解析程序的值必须是实现接口org.eclipse.core.variables的Java类的完全限定名称。IDynamicVariableResolver 。
