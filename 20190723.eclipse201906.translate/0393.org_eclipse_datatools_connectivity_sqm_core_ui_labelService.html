labelService

标识符：
org.eclipse.datatools.connectivity.sqm.core.ui.labelService

描述：
此扩展点将为插件提供一种方式来提供自己的图标以及SQLObject元素的displayType。

配置标记：

<！ELEMENT 扩展 （ 贡献者 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

此扩展点将为插件提供一种方式来提供自己的图标以及SQLObject元素的displayType。



<！ELEMENT 贡献者 EMPTY>

<！ATTLIST撰稿人

键入CDATA #REQUIRED

iconLocation CDATA #REQUIRED

displayType CDATA #IMPLIED

选择器CDATA #IMPLIED>

  • type - 由LabelService提供商提供的EMF类型
  • iconLocation - 这是Icon位置，以便core.ui插件可以加载您的图标
  • displayType - 这是在UI中显示的类型，例如属性浏览器。默认情况下，要显示的类型是元类名称eClass（）。getName（）
  • selector - 实现org.eclipse.datatools.connectivity.sqm.core.internal.ui的类的名称。LabelSelector接口。如果提供，则除了类型之外还将使用此类来确定是否应该为给定对象传递标签信息。
