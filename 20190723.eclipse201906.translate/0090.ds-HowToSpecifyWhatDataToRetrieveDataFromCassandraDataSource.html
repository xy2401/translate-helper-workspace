在数据资源管理器中，右键单击“数据集”，然后选择“新建数据集”。
2
在“新数据集”中，指定以下信息：
1
在“数据源选择”中，选择要使用的Cassandra数据源。数据集类型显示HQL选择查询。
2
在“数据集名称”中，键入数据集的名称。
3
选择下一步。
3
在“输出”列中，选择“添加”并添加必要的输出列以从数据源检索。对于每个输出列，您可以在New Script Data Set Column上指定以下值， 如图2-2 所示 ：
名称

分析类型
显示名称
显示名称密钥
完成指定数据集列属性后，选择“确定”。
