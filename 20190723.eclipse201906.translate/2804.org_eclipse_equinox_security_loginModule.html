JAAS登录模块

标识符：
org.eclipse.equinox.security.loginModule

描述：
loginModule扩展点为bundle提供了一种方法，用于向Java身份验证和授权服务（JAAS）注册实现LoginModule的类。

配置标记：

<！ELEMENT 扩展 （ loginModule ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>

扩展点描述符。



<！ELEMENT loginModule EMPTY>

<！ATTLIST loginModule

class CDATA #REQUIRED

说明CDATA #IMPLIED>

定义单个LoginModule的条目，可用于对用户进行身份验证。请参阅javax.security.auth.spi。LoginModule以获取更多信息。


  • class - 实现LoginModule提供程序接口的类的完全限定名称， javax.security.auth.login.spi。LoginModule 。
  • 说明 -

例子：
 <extension
         id="httpBasicLoginModule"
         name="httpBasicLoginModule"
         point="org.eclipse.equinox.security.loginModule">
      <loginModule      class="org.eclipse.equinox.accounts.internal.auth.module.HttpBasicLoginModule"
description="org.eclipse.equinox.accounts.internal.auth.module.HttpBasicLoginModule"/>
   </extension>

API信息：
JAAS API是Java Runtime Environment 1.4及更高版本的一部分。

提供的实施：
平台不提供公共登录模块。
