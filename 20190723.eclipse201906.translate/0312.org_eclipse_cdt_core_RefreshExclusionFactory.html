刷新排除工厂

标识符：
org.eclipse.cdt.core。RefreshExclusionFactory

以来：
5.3.0

描述：
此扩展点允许ISV提供知道如何实例化给定类型的RefreshExclusion的工厂类。在提供新类型的RefreshExclusions时必须使用此扩展点，否则在加载与给定项目的刷新策略相关的持久设置时，CDT无法实例化正确的排除项。

配置标记：

<！ELEMENT 扩展 （ exclusionFactory ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT exclusionFactory EMPTY>

<！ATTLIST exclusionFactory

factoryClass CDATA #REQUIRED

exclusionClass CDATA #IMPLIED

instanceClass CDATA #IMPLIED>

  • factoryClass - 用于创建exclusionClass和/或instanceClass属性中指定类型的对象的工厂。
  • exclusionClass - 可选地指定此工厂为其创建对象的RefreshExclusion的完全限定类型名称。
  • instanceClass - 可选地指定此工厂为其创建对象的RefreshExclusion的完全限定类型名称。
