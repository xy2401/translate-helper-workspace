按键

RWT按钮，包括按钮，切换按钮，单选按钮和复选框。

可用的CSS属性

可以为此元素定义的属性：

  • 动画

    支持动画类型“hoverIn”和“hoverOut”。其他人被忽略了。

  • 背景颜色

    按钮的背景颜色。

  • 背景图片

    按钮的背景图象。

  • 背景位置

    按钮的背景图像位置。

  • 背景重复

    按钮的背景图象重复。

  • 边界

    围绕按钮绘制的边框。

  • 下边框

    按钮的底边框。

  • 左边框

    按钮的左侧边框。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    按钮的右侧边框。

  • 边框顶部

    按钮的顶部边框。

  • 箱阴影

    按钮的阴影效果。

  • 颜色

    按钮的文字颜色。

  • 光标

    控制光标。

  • 字形

    按钮文字的字体。

  • 不透明度

    按钮的不透明度。介于0和1之间的值。

  • 填充

    按钮的填充（内部距离）。

  • 间距

    文本和图像之间的空间。

  • 文字修饰

    按钮文本的装饰。

  • 文字阴影

    文字的阴影。

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [箭头]

    表示该按钮是箭头按钮。

  • [边界]

    表示该按钮应具有边框。

  • [校验]

    表示该按钮是一个复选框。

  • [平面]

    表示该按钮应具有平面外观。

  • [推]

    表示该按钮是按钮。

  • [无线电]

    表示该按钮是单选按钮。

  • [右到左]

    表示按钮从右向左。

  • [TOGGLE]

    表示该按钮是切换按钮。

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：默认

    适用于shell默认按钮。

  • ：禁用

    适用于禁用按钮。

  • ：重点

    适用于聚焦按钮。

  • ：徘徊

    用户将鼠标悬停在按钮上时应用。

  • ：按下

    按下按钮时应用。

  • ：选择

    适用于选定的TOGGLE，CHECK和RADIO按钮。

扣FocusIndicator

按钮聚焦时绘制的框架。

可用的CSS属性

可以为此元素定义的属性：

  • 背景颜色

    框架的填充颜色。

  • 边界

    框架的边框。不支持3D样式插入，开头等。

  • 下边框

    框架的底边边框。

  • 左边框

    框架的左侧边框。

  • 右边框

    框架的右侧边框。

  • 边框顶部

    框架的顶部边框。

  • 余量

    按钮边框和框架之间的边距。仅在填充设置为“0px”时使用。

  • 不透明度

    指标的不透明度。介于0和1之间的值。

  • 填充

    框架和按钮文本之间的填充。如果将其设置为“0px”，则将使用margin来布局FocusIndicator。
