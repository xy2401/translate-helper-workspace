模板首选项

“模板首选项”页面允许您创建，编辑，删除，导入和导出模板。

模板是用于插入预定义代码框架的快捷方式。目的是节省时间并减少标准重复代码单元中出错的可能性。插入模板后，您可以使用手动和自动代码输入方法快速完成代码。

要在代码中插入模板，请输入其名称的前几个字母，然后按Ctrl + Space激活内容辅助功能并打开内容辅助列表。从列表中选择所需的模板。模板标有一个图标。有关使用内容辅助功能的更多信息，请参阅使用内容辅助 。

可以从Window |访问“模板首选项”页面偏好| PHP |编辑|模板 。

要从可用选项列表中删除模板，请在列表中取消选中其复选框。

要编辑现有模板，请从列表中选择它，然后单击“编辑”。

创建一个新模板

此过程描述如何创建要添加到模板列表的新模板。

要创建新模板：

  1. 单击新建。
  2. “新建模板”对话框将打开。

  1. 输入模板的详细信息：
    • 名称 - 用于标识模板的简短名称（例如，用于while循环的模板的“while”）。
    • Context - 模板可用时的代码上下文（PHP，PHPDoc，New PHP）。例如，PHP模板只能在编写PHP代码时使用。
    • 描述 - 模板代码的简短描述。
    • 模式 - 模式是在选择此模板时将插入编辑器的实际代码字符串。使用“插入变量”按钮从公共变量列表中进行选择。
  2. 单击确定。

您的模板将添加到列表中，并可从相关上下文中的内容辅助中获取。

导出和导入模板

PDT使您能够以下列格式导出和导入在XML文件中创建的模板：

<?xml version="1.0" encoding="UTF-8" ?>
<templates>
<template autoinsert="true" context="php" deleted="false" description="description" enabled="true" name="for">
for($$i = 0; $$i < 1; $$i++){ }
</template>

要导入模板：

  1. 单击“导入”以打开“导入模板”的浏览器。
  2. 选择导入包含模板信息的相关XML文件的位置。
  3. 单击打开。

template.xml文件中包含的模板将导入到模板列表中。

要导出模板：

  1. 从“模板”列表中选择要导出的模板。
  2. 单击“导出”以打开“导出模板”对话框。
  3. 选择要将XML文件保存到的位置。
  4. 单击保存。

将使用模板信息创建XML文件。

注意：

如果您选择了多个要导出的模板，则所有模板都将出现在导出的XML文件中。每个原始模板都受以下内容限制：<template> </ template>
