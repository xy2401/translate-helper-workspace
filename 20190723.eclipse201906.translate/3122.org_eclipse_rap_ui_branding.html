品牌

标识符：
org.eclipse.rap.ui.branding

以来：
RAP 1.0

描述：
品牌为RAP应用程序的UI添加自定义和样式。它可用于设置浏览器中显示的标题和图标，或应用自定义主题和样式。品牌与EntryPoint或IApplication实现相关联。

Brandings可以驻留在一个单独的包中，即可以在不修改其包含的包的情况下对应用程序进行标记。

配置标记：

<！ELEMENT 扩展 （ 品牌推广 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 品牌推广 （ additionalHeaders ？， presentationFactory ？）>

<！ATTLIST品牌推广

id CDATA #REQUIRED

themeId IDREF #IMPLIED

标题CDATA #IMPLIED

favicon CDATA #IMPLIED

body CDATA #IMPLIED>

  • id - 品牌标识符。
  • themeId - 与此品牌一起使用的主题。如果省略，则使用默认主题。
  • title - 申请的标题。此标题通常显示为浏览器窗口标题或显示在浏览器选项卡中。

    有效地设置HTML头的<title>元素。

  • favicon - 此应用程序显示的图标。favicon通常显示在浏览器的地址栏旁边或浏览器选项卡中。

    该图标应为.ico格式，以便与所有浏览器兼容。

  • body - 包含要添加到RAP启动页面主体的HTML标记的文件。该文件可能包含任何有效的HTML。

<！ELEMENT additionalHeaders （（ meta | link ））+>

要添加到HTML页面的任何其他标头元素。



<！ELEMENT 元 （ 属性 ）*>

<！ATTLIST meta

名称CDATA #IMPLIED

内容CDATA #IMPLIED>

HTML META元素，用于提供有关网页的结构化元数据，如作者，关键字，描述。


  • name - META元素的name属性。
  • content - META元素的content属性。

<！ELEMENT 链接 （ 属性 ）*>

<！ATTLIST链接

rel CDATA #IMPLIED

href CDATA #IMPLIED>

HTML LINK元素，用于将网页链接到CSS样式表等外部资源。


  • rel - LINK标记的rel属性。
  • href - LINK标记的href属性。

<！ELEMENT 属性 EMPTY>

<！ATTLIST属性

名称CDATA #REQUIRED

值CDATA #IMPLIED>

要添加到HTML元素的其他属性。


  • name - 属性的名称。
  • value - 属性的值。

<！ELEMENT presentationFactory （ stackPresentation *， defaultStackPresentation ？）>

<！ATTLIST presentationFactory

id IDREF #REQUIRED

defaultLayoutId IDREF #IMPLIED

名称CDATA #IMPLIED

viewActionsVisible（true | false）>

  • id - 要加载的PresentationFactory的id。
  • defaultLayoutId - 要加载的默认布局的ID。
  • 名字 -
  • viewActionsVisible - 设置视图的贡献项（操作/命令）的默认可用性。默认值为false。设置为true时，默认情况下所有项都可见。否则，所有项目都是不可见的，可以通过配置菜单进行配置。

<！ELEMENT stackPresentation EMPTY>

<！ATTLIST stackPresentation

id IDREF #REQUIRED

partId IDREF #REQUIRED>

  • id - StackPresentation的id
  • partId - 与StackPresentation的id耦合的部分的id。

<！ELEMENT defaultStackPresentation EMPTY>

<！ATTLIST defaultStackPresentation

id IDREF #REQUIRED>

  • id - StackPresentation的id，如果不存在parts和stackPresentations的映射，则加载该ID。

例子：
<extension point = “org.eclipse.rap.ui.branding” > <branding id = “org.example.tea.teabranding” themeId = “org.example.tea.teatheme” title = “Tea Portal” favicon = “icons /favicon.ico“body = ”body.html“ </ branding> </ extension>

API信息：
没有可用于品牌推广的API。
