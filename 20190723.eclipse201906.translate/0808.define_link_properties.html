定义链接属性

链接Web应用程序的页面后，必须定义链接属性。

要定义链接属性：

  1. 在JSF Application Configuration Editor的Navigation Rule页面上，打开Palette ，在Palette上选择Select 。

    这将激活Select控件，允许您在编辑器中选择一个链接。

  2. 选择要为其定义属性的链接。

    属性的“快速编辑”选项卡显示“链接”属性

  3. 单击省略号在From Outcome字段之后。

    将出现“结果选择”对话框。

  4. 键入结果或从“操作”列表中选择结果，然后单击“ 确定” 。

  5. 单击“ 重定向”字段中的箭头，然后选择“true”或“false”。

  6. 重复步骤2 - 5以定义Web应用程序中的每个链接。

  7. 从主菜单栏中选择File | Save以保存faces-config.xml文件。
