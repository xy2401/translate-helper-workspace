Mylyn GitHub连接器用户指南
更新本文档

概观

EGit Mylyn GitHub连接器支持创建用于处理GitHub问题和Gists的任务存储库。您想要访问问题的每个GitHub项目存储库都需要GitHub问题任务存储库。单个Gist存储库允许您访问http://gist.github.com上的所有Gists。

最新版本的EGit Mylyn GitHub连接器可以从以下更新站点安装：

http://download.eclipse.org/egit/github/updates-nightly

入门

Mylyn GitHub连接器支持批量添加所有GitHub存储库到Mylyn Task Repositorities视图。

  1. 选择文件>导入...
  2. 选择“ 任务”类别下的“ GitHub任务存储库”节点
  3. 选择Next
  4. 在凭据向导页面上输入您的GitHub登录名和密码
  5. 选择Next
  6. 检查要从中访问问题的所有GitHub存储库。
  7. 选择完成
  8. 打开“ 任务存储库”视图以浏览新添加的任务存储库并添加查询或创建新任务。

问题

查询

GitHub问题任务存储库支持创建按以下字段筛选的查询：

  • 国家（开放或封闭）
  • 里程碑
  • 提及
  • 代理人
  • 标签

请注意，在同一查询中使用多个过滤器时，返回的问题将与所有过滤器匹配。有效查询所需的最小值是至少选择一个状态和非空标题。

问题编辑

GitHub问题任务编辑器支持以下内容：

  • 编辑标题和说明
  • 添加新评论
  • 从“ 操作”部分关闭并重新打开该问题
  • 将问题分配给“ 人员”部分中的某人
  • 在“ 属性”部分添加和删除标签
  • 从“ 属性”部分更新里程碑

要旨

创建

可以通过选择编辑器中的文本或浏览器中的文件并选择GitHub > Create Private Gist或GitHub > Create Public Gist上下文菜单来创建Gists 。Gist将在后台作业中创建，通知弹出窗口将显示一个链接以打开新创建的Gist。

查询

Gist查询仅支持按指定的GitHub登录名进行过滤。

要点编辑

Gists的任务编辑器支持以下内容：

  • 编辑说明
  • 添加新评论
  • 在编辑器中打开文件中的文件
  • 将新文件添加到要点
  • 替换gist中当前文件的内容
  • 将gist克隆到本地Eclipse项目

常见问题（FAQ）

  • 连接器是否支持GitHub API令牌？
    • 不 ，连接器用于请求的GitHub v3 API不再支持API令牌。
  • 是否通过HTTPS进行GitHub API请求？
    • 是的 ，通过GitHub API发出的所有请求都是通过HTTPS进行的。
