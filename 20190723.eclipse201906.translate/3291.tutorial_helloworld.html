入门 导入Scout演示应用程序

Hello World Tutorial

通过本教程，您将学习如何创建第一个Scout应用程序。

创建一个侦察项目

启动Eclipse IDE并为工作区选择一个空目录， 如图1所示。然后，此工作空间目录将保存Hello World应用程序的所有项目代码。Eclipse IDE运行后，它将显示Java透视图。

图1。选择一个新的空文件夹来保存项目工作区

要创建一个新的侦察项目在向导中选择搜索栏菜单文件 ▸ 新建 项目▸...，然后键入“侦察项目”。选择Scout Project向导，然后按Next 。然后启动New Scout Project向导， 如图2所示。

图2。新的Scout项目向导。

在New Scout Project向导中，您必须输入Scout项目的group id ， artifact id和display name 。由于创建的项目将使用Apache Maven，请参阅Maven命名约定以选择项目的group id和artifact id 。然后， artifact id也将是Eclipse工作空间中的项目名称。display name用作呈现给用户的应用程序名称（例如，在浏览器标题栏中）。

对于Hello World应用程序，只需使用已预填充的值， 如图2所示 。然后，单击Finish按钮让Scout SDK为您创建初始项目代码。

根据您的Eclipse安装，最初可能会丢失一些Maven plugin connectors 。在这种情况下，可以显示如图3所示的对话框。要继续，请单击“ 完成”以解析所选连接器。然后确认安装，接受许可证以及某些内容尚未签名的消息。最后，安装maven插件连接器需要重新启动Eclipse IDE。

图3。Maven插件连接器安装对话框。

在New Scout Project向导为Hello World应用程序创建了初始Maven模块之后，这些模块由Eclipse IDE编译和构建。如果Eclipse Scout安装成功，Eclipse IDE应该在Package Explorer中显示所有已创建的Maven模块，并且有一个空的Problems视图， 如图4所示。

图4。为Hello World应用程序创建的初始Maven模块集。

运行应用程序

在初始项目创建步骤之后，我们可以首次启动Scout应用程序。为此，需要以下三个步骤

  1. 启动Scout后端服务器

  2. 启动Scout前端服务器

  3. 在浏览器中打开应用程序

要启动Scout后端服务器，我们首先在Eclipse IDE的Package Explorer视图中选择[webapp] dev server.launch文件，然后使用Run As菜单， 如图5所示。

图5。启动Hello World应用程序。

启动Scout前端服务器的工作原理完全相同。但首先在Eclipse IDE中选择[webapp] dev ui.launch文件。此启动文件位于Package Explorer中的模块helloworld.ui.html.app.dev下。

在启动Scout应用程序期间，您应该看到控制台输出提供有关启动的信息。成功启动Scout后端和前端服务器后，可以通过导航到您喜欢的Web浏览器中的http：// localhost：8082 /来访问Hello World应用程序。

然后应在浏览器中启动正在运行的Hello World应用程序， 如图6所示。

图6。浏览器中的Hello World应用程序。
