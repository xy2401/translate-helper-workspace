安全存储模块

标识符：
org.eclipse.equinox.security.secureStorage

描述：
描述Equinox安全存储模块的实现。

配置标记：

<！ELEMENT 扩展 （ 提供者 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>


<！ELEMENT 提供者 （ 提示 *）>

<！ATTLIST提供商

class CDATA #REQUIRED

优先级CDATA #REQUIRED

说明CDATA #IMPLIED>

  • class - 实现密码提供程序接口的类。该类必须扩展org.eclipse.equinox.security.storage.provider。PasswordProvider类。
  • priority - 描述此密码提供程序优先级的整数0到10，其中0表示最低优先级。新创建的安全首选项将使用具有最高优先级的密码提供程序。如果多个提供者具有相同的优先级，则将选择其中的任意提供者。
  • description - 描述此密码提供程序的用户功能的可选文本。

<！ELEMENT 提示 EMPTY>

<！ATTLIST提示

value（AutomaticPasswordGeneration）>

  • value - 为了帮助保护存储优化工作流，在没有用户输入的情况下获取主密码的提供者应指定AutomaticPasswordGeneration提示。

例子：

   <extension
         id="custom"
         point="org.eclipse.equinox.security.secureStorage">
      <provider
            class="com.xyz.PasswordProvider"
            priority="7">
      </provider>
   </extension>

提供的实施：

org.eclipse.equinox.security.ui包提供默认密码提供程序，提示用户输入密码。

org.eclipse.equinox.security.win32.x86片段提供Windows操作系统集成。
