§3.5。（a） 语法

使用其中一个callout修饰符get或set一个角色方法可以绑定到角色基类的一个字段：

1 
getValue -> get value;
2 
setValue -> set value;
3 
int getValue() -> get int value;

其中getValue ， setValue是适当签名的抽象角色方法， value是绑定基类的字段。
也可以使用更长的语法（参见上面第3行），它使用完整的签名。对于左侧§3.1。（c）适用，对于右侧，此较长版本将字段类型添加到字段名称之前。
