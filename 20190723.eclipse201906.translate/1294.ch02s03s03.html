2.3.3. The categories

Executable Test Cases:

This category contains Test Cases which are ready to be executed. Each Test Case corresponds to one Test Suite, and is named accordingly. The executable Test Case contains all the Test Cases necessary for a particular test, and includes all the data.

Tests with the Simple Adder:

This category contains the Test Cases which test the Simple Adder program (Section 2.4, “Adder Tests”).

Tests with the DVD Tool:

This category contains the Test Cases which test the DVD Tool program (Section 2.5, “DVD Tool Tests”).
