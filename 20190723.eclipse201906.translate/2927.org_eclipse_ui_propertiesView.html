属性视图

标识符：
org.eclipse.ui.propertiesView

以来：
3.6

描述：
此扩展点用于修改“属性”视图的行为。

配置标记：

<！ELEMENT 扩展 （ excludeSources *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT excludeSources EMPTY>

<！ATTLIST excludeSources

id IDREF #REQUIRED>

使用此元素可以将工作台部件排除在参与“属性”视图之外。


  • id - 排除的工作台部件的唯一ID。

例子：
以下示例演示了如何排除视图：


   <extension point="org.eclipse.ui.propertiesView">
     <excludeSource id="org.eclipse.help.ui.HelpView"/>
   </extension>
相同的代码模式可用于排除编辑器。
