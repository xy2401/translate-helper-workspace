介绍

Cloudio组件提供专用的标记云视图来呈现文字云。可以从任意文本文件初始化视图的内容，可以通过多个可视化参数灵活地定制可视化。渲染的标签云可以导出为图像。

标签云视图

标签云视图位于“可视化”类别下。

菜单贡献

当它处于打开和活动状态时，它会添加某些菜单贡献，可用于填充其内容或导出图像。用于呈现单词云的单词可以通过'文件 - \>从文件加载单词'从文本文件加载。如果文件包含应从词云中排除的单词，则可以通过“文件 - \>从文件加载停用词”加载停用词列表（黑名单）。在加载单词和/或停用单词后，会自动呈现标签云。渲染的标签云可以通过“文件 - \>导出标签云到图像”导出到PNG文件。

控制面板

Tag Cloud View在其右边界提供了一个控制面板。它可用于调整可视化参数，并根据以下参数重新呈现标记云：

重新定位：允许用户在标签云上重新定位单词。显示的元素只会重新定位，保持字体，颜色和角度不变。

重新布局：允许用户重新布置标签云上的单词。使用控制面板的所有当前值（包括字体，颜色和角度设置）将完全更新显示的元素。

目前，支持以下参数：

字数：允许用户设置将在标签云上显示的最大元素数。预定义值是100,200,300,400,500,600,700,800,900,1000,1100,1200,1300,1400,1500,1600,1700,1800,1900,2000。下面的屏幕截图表示Tag Cloud View仅限于前100个最常出现的单词：

字体：允许用户定义用于标记云的渲染字体的新字体（或在预定义字体之间切换）。

最大字体大小：允许用户设置标签云渲染使用的最大字体大小。请注意，将忽略太大而无法容纳到云区域的单词。默认情况下，此值为100。

最小字体大小：允许用户设置最小字体大小。默认情况下，此值为12。

提升：允许用户启用元素提升（额外放大）。当boost设置为0（默认情况下）时，不会放大任何元素。当增强设置为10时，前10个最常出现的元素被放大。

增强因子：允许用户指定增强（额外放大）使用的增强因子。当增强因子设置为1（默认情况下）时，元素会被放大1倍（无额外放大）。当增强设置为5时，“待增强”元素将比其他元素大5倍。

角度：允许用户设置标签云渲染所使用的单词方向。预定义值为：“仅水平”（单词与x轴平行对齐），“仅垂直”（单词与y轴平行对齐），“水平和垂直”（单词对齐）平行于x轴或平行于y轴），“45度”（单词可以对角排列）“45度，大部分是水平的”和“随机的”。以下屏幕截图表示具有不同角度设置的标签云：

缩放：允许用户设置缩放因子，用于可视化标签云上的差异（以确定每个单词的“绘制重量”）。预定义的值是“线性的”（差异在单词权重之间均匀分布）和“对数”（差异在单词权重之间以对数方式展开）。以下两个屏幕截图分别表示logaritmic和线性比例设置之间的差异。

X轴变化：允许用户设置x轴变化的百分比。默认情况下，在可用区域的中心开始搜索可用空间。通过增加该值，起点将在x轴上移动。

Y轴变化：允许用户设置y轴变化的百分比。默认情况下，在可用区域的中心开始搜索可用空间。通过增加该值，起点将在y轴上移动。

颜色：允许用户定义新的颜色方案（或在预定义颜色方案之间切换），这些颜色方案用作标签云的前景/背景渲染颜色。下面的屏幕截图表示使用不同颜色方案呈现的标签云：
