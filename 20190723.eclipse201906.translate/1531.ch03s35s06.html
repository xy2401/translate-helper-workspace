3.35.6. Additional information for RCP AUTs

3.35.6.1. Keyboard Layout

The Keyboard Layout for the AUT can be defined in the Start AUT Configurations dialog. The entered value must represent a locale and must represent a defined keyboard layout.

3.35.6.2. RCP Remote Control Plug-in

In order to start an RCP AUT, the RCP Remote Control plug-in (org.eclipse.jubula.rc.rcp) must be included in the launch configuration. The list of plug-ins for the launch configuration can be viewed and modified from the Plug-ins tab of the Start AUT Configurations dialog when an Eclipse Application launch configuration is selected.
