新项目布局首选项

New Project Layout Preferences页面允许您配置新PHP项目的默认布局。默认布局将配置项目根目录下的所有文件夹是否将被视为构建过程的 “源”文件夹，或者是否将为构建过程中包含或排除的资源创建单独的文件夹。

“新建项目布局首选项”页面中配置的首选项将影响在“新建PHP项目”向导的“项目布局”类别中选择的默认选项。可以在创建期间按项目修改此设置。

可以从Window |访问“新建项目布局首选项”页面偏好| PHP |新项目布局首选项。

要配置默认的新PHP项目布局：

选择所需选项：

  • 使用项目作为源文件夹 - 默认情况下，在项目根目录下创建的所有文件夹都将被视为“源文件夹”，并将在构建过程中进行扫描。请参阅配置项目的PHP构建路径以配置项目的PHP构建路径。
  • 为源文件和公共资源创建单独的文件夹 - 将在项目根目录下创建两个文件夹 - 一个“源”文件夹用于在构建过程中扫描的资源，一个“公共”文件夹将在构建过程中跳过。您可以通过在“默认源/公用文件夹名称”字段中输入所需名称来更改这些文件夹的默认名称。
