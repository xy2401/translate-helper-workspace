启动连接器

标识符：
org.eclipse.wst.jsdt.debug.core.launchingConnectors

以来：
1.0

描述：
此扩展点允许贡献者指定自己的连接器定义

配置标记：

<！ELEMENT 扩展 （ 连接器 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 连接器 EMPTY>

<！ATTLIST连接器

class CDATA #REQUIRED

id CDATA #REQUIRED>

  • class - 实现org.eclipse.e4.languages.javascript.jsdi.connect的连接器类。连接器
  • id - 此连接器的唯一ID

例子：
以下示例是Mozilla Rhino连接连接器的实际定义。 
<extension point = “org.eclipse.wst.jsdt.debug.core.connectors” > <connector class = “org.eclipse.wst.jsdt.debug.rhino.connect。RhinoAttachingConnector“ id = ”rhino.attaching.connector“ > </ connector> </ extension>

API信息：
class属性是必需的，必须实现org.eclipse.wst.jsdt.debug.core.jsdi.connect.连接器
