5.2.2. The Execution Perspective

In the Execution Perspective, you can see the following (Figure 5.2, “Execution Perspective”):

  • The Test Suite Browser

  • The Test Result View

  • The Properties View

  • The Image View

Figure 5.2. Execution Perspective

You cannot edit in the Execution Perspective, but you can see the results of tests that have been started interactively..
