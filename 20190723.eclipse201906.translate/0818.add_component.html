添加组件

您可以开发新的JavaServer Faces（JSF）组件。使用JSF Application Configuration编辑器Component页面的Component部分，您可以注册组件及其关联的类。通常，JSF组件由一组类组成，例如组件类，其呈现类等。有关组件开发的详细信息，请参阅java.sun.com上的JSF规范和教程。

要添加组件：

  1. 在J2EE透视图的Project Explorer中选择faces-config.xml 。

    将打开JSF应用程序配置编辑器。

  2. 选择编辑器的“ 组件”选项卡。

    将出现“Faces配置组件”页面。

  3. 展开“ 组件”部分。

    定义的组件显示在组件列表中。

  4. 单击添加 。

    新组件x将添加到列表中，并显示“常规”，“构面”，“属性”和“属性”部分。

  5. （可选）在“ 常规”部分的“ 显示名称”字段中将默认名称更改为唯一的描述性名称。

  6. （可选）在“ 描述”字段中键入组件的描述 。

  7. 在“ 组件类型”字段中键入类型 。

  8. 单击“ 浏览”

    将出现“选择类型”对话框。

  9. 在“ 选择条目”字段中键入要使用的类型的第一个字母。

    以您键入的字母开头的可用类型列表将显示在“匹配类型”列表中。

  10. 选择要使用的类型，然后单击“ 确定” 。

  11. 单击“ 构面”部分中的“ 添加 ”。

    将出现“添加/编辑构面”对话框。

  12. 键入构面名称，显示名称和描述，然后单击“ 完成” 。

  13. 单击“ 属性”部分中的“ 添加 ”。

    将出现“添加/编辑属性”对话框。

  14. 在“ 属性名称”字段中键入唯一的描述性名称。

  15. 单击“ 浏览”

  16. 在“ 选择条目”字段中键入要使用的类型的第一个字母。

    以您键入的字母开头的可用类型列表将显示在“匹配类型”列表中。

  17. 选择要使用的类型，然后单击“ 确定” 。

  18. 在“ 默认值”字段中键入值。

  19. （可选）在“ 建议值”字段中键入值。

  20. 单击Properties部分中的Add 。

    将出现“添加/编辑属性”对话框。

  21. 在“ 属性名称”字段中键入唯一的描述性名称 。

  22. 单击“ 浏览”

  23. 在“ 选择条目”字段中键入要使用的类型的第一个字母。

    以您键入的字母开头的可用类型列表将显示在“匹配类型”列表中。

  24. 选择要使用的类型，然后单击“ 确定” 。

  25. 在“ 默认值”字段中键入值。

  26. （可选）在“ 建议值”字段中键入值。
