Sirius - 提供自定义捆绑图像形状

免责声明 - 已弃用

此扩展点已弃用（但仍在使用），不应使用。相反，您应该使用新的扩展点 。

目标

  • Sirius为节点映射提供了基本的图像样式。这种风格提供五种不同形状的选择。此扩展点用于允许说明符提供在SVG文件中定义的自己的形状。

定义捆绑的图像形状扩展名

  • 此扩展点标识为org.eclipse.sirius.diagram.bundledImageShape 。在此扩展中，有三个必填字段：
    • 扩展的ID一次性需要，因为它将用于识别模型中提供的形状（VSM和图形模型）。
    • 图像路径，需要以插件名称开头，因为可以使用不同插件中的图像定义捆绑的图像形状扩展。
    • VSM中显示的图像标签（后缀为声明捆绑图像形状扩展名的插件名称）。
  • 此扩展点中有6个可选参数。这些参数用于提供有关给定SVG文件中使用的颜色，边框颜色和边框大小的标识符或属性名称，如果这些参数不是默认参数（在5个默认形状中使用）。以下是这些参数的列表和默认值。
    • colorIdentifier，定义图形内部颜色的标记的标识符。默认标识符为“stop1”。
    • colorAttribute，定义图形内部颜色的标记的属性。默认标识符为“style”。
    • borderColorIdentifier，定义图形边框颜色的标记的标识符。默认标识符为“elementWithGradient”。
    • borderColorAttribute，定义图形边框颜色的标记的属性。默认标识符为“style”。
    • borderSizeIdentifier，定义图形边框大小的标记的标识符。默认标识符为“externalBorder”。
    • borderSizeAttribute，定义图形边框大小的标记的属性。默认标识符为“style”。

请注意，可以提供非SVG文件（如JPEG），但颜色和边框大小不会与VSM中的选定信息同步。

例

以下示例使用图像文件donut.svg定义新的捆绑图像形状。

<extension id =“org.eclipse.sirius.tests.swtbot.donut”point =“org.eclipse.sirius.diagram.bundledImageShape”> <image imagePath =“/ org.eclipse.sirius.tests.swtbot / images / donut .svg“label =”Donut“> <colorIdentifier colorIdentifier =”stop8224-2“> </ colorIdentifier> <colorAttribute colorAttribute =”style“> </ colorAttribute> <borderColorIdentifier borderColorIdentifier =”path10796-2-6-2-7“ > </ borderColorIdentifier> <borderColorAttribute borderColorAttribute =“style”> </ borderColorAttribute> <borderSizeIdentifier borderSizeIdentifier =“path10796-2-6-2-7”> </ borderSizeIdentifier> <borderSizeAttribute borderSizeAttribute =“style”> </ borderSizeAttribute> < / image> </ extension>

在此示例中，有关内部颜色（colorIdentifier），边框颜色（borderColorIdentifier）和边框大小（borderSizeIdentifier）的标记的标识符不是默认标识符。但是，属性都是默认值。因此，我们可以如下简化扩展。

<extension id =“org.eclipse.sirius.tests.swtbot.donut”point =“org.eclipse.sirius.diagram.bundledImageShape”> <image imagePath =“/ org.eclipse.sirius.tests.swtbot / images / donut .svg“label =”Donut“> <colorIdentifier colorIdentifier =”stop8224-2“> </ colorIdentifier> <borderColorIdentifier borderColorIdentifier =”path10796-2-6-2-7“> </ borderColorIdentifier> <borderSizeIdentifier borderSizeIdentifier =”path10796- 2-6-2-7“> </ borderSizeIdentifier> </ image> </ extension>
