代码重构首选项

Code Refactor Preferences页面允许您启用就地重构选项。此选项允许您使用编辑器中的重构功能重命名元素，而不是通过重构对话框。

有关重构或重命名元素的更多信息，请参阅重构主题。

可以从Window |访问Code Refactor Preferences页面偏好| PHP |代码重构 。

配置代码重构设置

要配置代码重构设置：

  1. 选中“在没有对话框的编辑器中重命名”复选框，可以在编辑器中重命名元素，而不是通过重构对话框。
  2. 单击“应用”以应用更改。
