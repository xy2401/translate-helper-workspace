PHP解释器首选项

PHP Interpreter首选项页面允许您设置要用于项目的PHP版本。这将影响内部调试器，代码分析器和内容辅助。

从Window |访问PHP解释器首选项页面偏好| PHP |口译员 。

要配置PHP版本：

  1. 选择要使用的PHP版本。有关PHP版本设置的效果的更多信息，请参阅PHP支持 。
  2. 选中使用短标签复选框以使用短标签（'<？“）。
  3. 单击“确定”以应用设置。

要为特定项目使用不同的PHP解释器：

  1. 选择“ 配置项目特定设置”链接。
  2. 从列表中选择特定项目。
  3. 将打开另一个PHP Interpreter首选项页面
  4. 选中“启用项目特定设置”复选框。
  5. 选择您的PHP版本。
  6. 单击“应用”
  7. 将出现一个提示对话框，指出必须重建项目才能使设置生效。
  8. 单击是以重建项目。将根据所选的PHP版本执行错误解析。如果单击“否”，将在下次重新启动时执行重建。
