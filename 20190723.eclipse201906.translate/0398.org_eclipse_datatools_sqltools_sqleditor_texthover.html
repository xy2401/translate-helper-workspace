SQL编辑器文本悬停

标识符：
org.eclipse.datatools.sqltools.sqleditor.texthover

以来：
[输入出现此扩展点的第一个版本。]

描述：
此扩展点为SQL编辑器中的文本悬停提供支持。扩展名注册了org.eclipse.datatools.sqltools.sqleditor.sql.AbstractSQLEditorTextHover类，它将提供文本悬停信息。

配置标记：

<！ELEMENT 扩展 （ 悬停 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 悬停 EMPTY>

<！ATTLIST悬停

id CDATA #IMPLIED

class CDATA #REQUIRED>

  • id -
  • class - org.eclipse.datatools.sqltools.sqleditor.sql的子类。AbstractSQLEditorTextHover。当有多个texthover贡献时，sql编辑器将使用第一个可以在当前位置提供悬停信息的编辑器。

例子：

<extension point =
“org.eclipse.datatools.sqltools.sqleditor.texthover”
>

       
<hover class =
“com.sybase.stf.dmp.editorextension。SQLDebugHover”
ID =
“com.sybase.stf.dmp.editor.hover”
/>


</分机>
提供文本悬停，当当前位置在变量名称上时，该悬停将提供变量值。

API信息：
该类必须实现org.eclipse.datatools.sqltools.sqleditor.sql.AbstractSQLEditorTextHover
