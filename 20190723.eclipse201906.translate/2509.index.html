文件菜单

“文件”菜单允许您对活动文件和文件夹执行各种功能，以及组织当前文件和创建新项目。

“文件”菜单中提供的选项包括：

名称捷径描述
新ALT + SHIFT + N创建各种项目和文件类型。
要查看可通过此菜单创建的新项目列表，请参阅“ 新建 ”子主题。
注意：可以通过“窗口菜单”中的“ 自定义透视图”选项配置当前透视图的“新建菜单”中可用的选项。
打开文件在编辑器视图中打开以前创建的文件。如果文件存储在工作区的活动项目中，则该文件将显示在Project Explorer / Navigator视图中。
关按Ctrl + W关闭活动文件的编辑器。
如果文件中有未保存的更改，系统将提示您在关闭之前保存文件。
关闭所有按Ctrl + Shift + W关闭所有打开的编辑器窗口。
如果文件中有未保存的更改，系统将提示您在关闭之前保存它们。
保存按Ctrl + S保存对活动文件所做的更改。
另存为允许您在保存活动文件时指定文件名/位置。
保存全部按Ctrl + Shift + S保存所有打开的文件。
还原将活动文件的状态恢复为上次保存的版本。
移动将当前选定的文件移动到其他文件夹/项目。
在“移动”对话框中标记“更新引用”复选框将应用重构功能，该功能将自动更新其他文件中对该文件的所有引用。
单击“预览”以查看由于移动而将进行的所有更改。
单击“确定”以实施更改并移动文件。
对文件的所有引用都将自动更新以反映其新位置。
有关使用重构功能移动文件的更多信息，请参阅“移动文件”。
改名F2重命名当前选定的文件。
在“移动”对话框中标记“更新引用”复选框将应用重构功能，该功能将自动更新其他文件中对该文件的所有引用。
单击“预览”以查看由于重命名而将进行的所有更改。
单击“确定”以实施更改。对文件的所有引用都将自动更新以反映更改。
有关使用重命名功能重命名文件的更多信息，请参阅“重命名文件”。
刷新F5进行外部更改后刷新导航器视图。
将行间距转换为选择首选行结束样式。选择Windows，Unix和Mac风格。
打印按Ctrl + P打印活动文件。
切换工作区允许您打开备用工作区。
使用此功能将重新启动Navigator / PHP Views中显示的备用工作空间。
如果要打开位于其他位置的文件和项目，或者要将文件和项目保存到其他位置，这将非常有用。
进口将各种类型的项目导入工作区。
有关可导入的不同类型项目的列表，分为类别，请参阅“ 导入 ”。
出口从工作区导出并创建不同类型的项目到不同位置。
有关可以从工作区导出的不同项目的列表（分为类别），请参阅“ 导出 ”。
属性Alt + Enter键显示包含活动文件信息的屏幕，包括其路径，类型，位置，大小以及上次修改的时间。从这里也可以配置文本文件编码类型，并且可以设置文件是只读，归档还是派生。
上次查看的文件列出上次查看的文件以便于访问。
出口关闭 。如果文件中有未保存的更改，系统将提示您在退出之前保存它们。
