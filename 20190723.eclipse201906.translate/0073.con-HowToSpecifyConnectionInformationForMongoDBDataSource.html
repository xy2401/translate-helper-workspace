如何创建MongoDB数据源

1
在“数据资源管理器”中，右键单击“数据源”，然后选择“新建数据源”。
2
在“新建数据源”中，指定以下信息：
1
从数据源类型列表中选择MongoDB数据源。
2
在“数据源名称”中，键入数据源的名称。
3
选择下一步。
3
在New MongoDB数据源配置文件中，指定要连接到MongoDB数据库实例的属性。选择Mongo Database URI Elements以单独指定URI的元素，或选择Mongo Database URI以键入URI字符串。
如果选择Mongo Database URI Elements，请提供以下属性的值：
服务器主机：要连接的数据库服务器的IP地址。
服务器端口：数据库服务器侦听请求的端口。默认值为27017。
数据库名称：从中检索数据的数据库的名称。
用户名和密码：仅当MongoDB进程以安全模式运行时才需要用户身份验证。在受信任环境中未经身份验证的情况下运行MongoDB时，用户名和密码是可选的。
如果选择Mongo Database URI，请按以下格式键入URI字符串：
MongoDB的：// [用户名：密码@]主机1 [：端口1] [，主机2 [：端口2]，... [，hostN [：端口n]]] [/ [数据库] [？选项]
4
（可选）启用以下一个或两个属性：
保持Socket Alive：启用时，MongoDB连接通过防火墙保持活动状态。
一致的请求会话：启用时，ODA连接中的所有数据库操作都使用相同的MongoDB连接，确保它们以固定顺序发生，并具有可预测的结果。

五
选择测试连接。如果连接成功，请选择“完成”。如果连接不成功，则必须为连接属性提供不同的值。
