打开数据访问设计器

标识符：
org.eclipse.datatools.connectivity.oda.design.ui.dataSource

以来：
3.0

描述：
此扩展点为开放数据访问（ODA）Designer UI框架提供了可扩展性，用户可通过该框架定义ODA数据源设计和数据集设计。
实现ODA运行时扩展点的ODA驱动程序可以选择扩展此ODA UI扩展点，以提供自定义ODA设计向导和属性页。

配置标记：

<！ELEMENT 扩展 （ dataSourceUI ， dataSetUI +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT dataSourceUI （ newDataSourceWizard ）>

<！ATTLIST dataSourceUI

id CDATA #REQUIRED>

允许ODA数据提供者为用户定义自定义对话框页面，以创建新的ODA数据源设计实例。


  • id - 在ODA使用者应用程序环境中唯一标识ODA数据源扩展的完全限定ID。其值必须与同一ODA驱动程序的org.eclipse.datatools.connectivity.oda.dataSource扩展中定义的dataSource.id属性的值相匹配。实现org.eclipse.datatools.connectivity.connectionProfile扩展点时，其connectionProfile.id属性也必须与此id值匹配。

<！ELEMENT newDataSourceWizard EMPTY>

<！ATTLIST newDataSourceWizard

windowTitle CDATA #IMPLIED

includesProgressMonitor（true | false）“true”

pageTitle CDATA #IMPLIED

pageClass CDATA #REQUIRED>

定义数据源向导的可自定义行为，该向导允许用户创建新的ODA数据源设计实例。
数据源向导的实现类在org.eclipse.datatools.connectivity.connectionProfile扩展点中定义的newWizard.class属性中指定。
向导类必须使用或扩展ODA框架的向导基类org.eclipse.datatools.connectivity.oda.design.ui.wizards。NewDataSourceWizard 。

此元素包含用于指定扩展为数据源向导提供的自定义向导起始页的属性。


  • windowTitle - 数据源向导窗口的自定义标题。
  • includesProgressMonitor - 指示数据源向导是否应包含进度监视器。默认值为“true”。
  • pageTitle - 数据源向导的自定义起始页面的页面标题。如果指定此值，则覆盖可能由向导页面的构造函数设置的任何标题。
  • pageClass - 一个完全限定的类名，它实现了用于创建新的ODA数据源设计实例的向导起始页。
    此类必须从ODA抽象基类org.eclipse.datatools.connectivity.oda.design.ui.wizards扩展。DataSourceWizardPage 。
    ODA框架中提供了默认的具体实现类org.eclipse.datatools.connectivity.oda.design.ui.pages.impl。DefaultDataSourceWizardPage 。ODA扩展可以选择使用或扩展此默认实现类。

<！ELEMENT dataSetUI （ dataSetWizard ， dataSetPage +）>

<！ATTLIST dataSetUI

id CDATA #REQUIRED

initialPageId CDATA #IMPLIED

supportsInParameters（true | false）“true”

supportsOutParameters（true | false）“false”>

允许ODA数据提供者为用户定义自定义对话框页面，以创建或编辑ODA数据集设计实例。该元素定义数据集向导页面和编辑器页面的子元素。
ODA主机设计者可以使用自定义向导页面的控件来编辑数据集设计实例。


  • id - 唯一标识ODA数据源扩展中的ODA数据集类型的完全限定ID。其值必须与同一ODA驱动程序的org.eclipse.datatools.connectivity.oda.dataSource扩展中定义的dataSet.id属性的值相匹配。
  • initialPageId - 标识应在数据集对话框中选择的特定dataSetPage。
  • supportsInParameters - 表示此类型的数据集设计支持输入参数定义，并且ODA主机设计者可能需要收集有关自定义页面提供的参数定义的更多元数据。
  • supportsOutParameters - 表示此类型的数据集设计支持输出参数定义，并且ODA主机设计者可能需要收集有关自定义页面提供的参数定义的更多元数据。

<！ELEMENT dataSetWizard EMPTY>

<！ATTLIST dataSetWizard

class CDATA“org.eclipse.datatools.connectivity.oda.design.ui.wizards。DataSetWizard”

windowTitle CDATA #IMPLIED>

定义一个数据集向导，允许用户创建或编辑ODA数据集设计实例。


  • class - 数据集向导的实现类。
    向导类必须使用或扩展ODA框架的向导基类org.eclipse.datatools.connectivity.oda.design.ui.wizards。DataSetWizard 。
  • windowTitle - 数据集向导窗口的自定义标题。

<！ELEMENT dataSetPage EMPTY>

<！ATTLIST dataSetPage

id CDATA #REQUIRED

wizardPageClass CDATA #REQUIRED

displayName CDATA #IMPLIED

路径CDATA #IMPLIED

icon CDATA #IMPLIED>

定义扩展有助于ODA主机设计者的数据集对话框的自定义页面。


  • id - 数据集对话框中此页面的唯一名称。
  • wizardPageClass - 一个完全限定的类名，它实现了一个数据集向导页面，用于创建或编辑ODA数据集设计实例。
    此类必须从ODA抽象基类org.eclipse.datatools.connectivity.oda.design.ui.wizards扩展。DataSetWizardPage 。
  • displayName - 将在此页面的UI中显示的自定义页面的页面标题或描述性名称。如果指定此值，则会覆盖页面构造函数可能设置的任何标题或描述性名称。
  • path - 数据集首选项对话框中页面的路径。
    要将页面添加到根节点，请在此值中指定“/”。默认值为“/”。
    要将页面添加为另一个页面节点的子节点，请指定以“/”开头的完整路径;并在层次结构中的每个页面节点之间使用“/”作为分隔符。
    例如，路径值“/ parent”会将此页面作为子节点添加到名为“parent”的现有页面。

    注意：在尝试添加子页面之前，应该添加名为“parent”的页面。
  • icon - 除了页面的显示名称之外，将在UI中使用的图标的相对路径。

例子：

   
<extension
  point=
"org.eclipse.datatools.connectivity.oda.design.ui.dataSource"
>

      
<dataSourceUI id=
"%oda.data.source.id"
>

         
<newDataSourceWizard
               pageClass=
"org.eclipse.datatools.connectivity.oda.flatfile.ui.wizards.FolderSelectionWizardPage"

               pageTitle=
"Select Folder"

               windowTitle=
"New Flat File Data Source Profile"

               includesProgressMonitor=
"false"
/>

      
</dataSourceUI>

      
<dataSetUI
            id=
"org.eclipse.datatools.connectivity.oda.flatfile.dataSet"

            initialPageId=
"oda.flatfile.ui.tablePage"

            supportsInParameters=
"false"

            supportsOutParameters=
"false"
>

         
<dataSetWizard
               class=
"org.eclipse.datatools.connectivity.oda.design.ui.wizards.DataSetWizard"

               windowTitle=
"Select Table"
/>

         
<dataSetPage
               displayName=
"Select Columns"

               icon=
"icons/file.gif"

               id=
"oda.flatfile.ui.tablePage"

               path=
"/"

               wizardPageClass=
"org.eclipse.datatools.connectivity.oda.flatfile.ui.wizards.FileSelectionWizardPage"
/>

      
</dataSetUI>

   
</extension>

API信息：
[ 临时 ]扩展必须实现扩展点模式中定义的相关对话框页面。有关更多信息，请参阅org.eclipse.datatools.connectivity.oda.design.ui项目中的抽象基类的JavaDoc文档。

提供的实施：
随Eclipse DTP连接源提供的插件org.eclipse.datatools.connectivity.oda.flatfile.ui提供了实现简单ODA设计UI扩展的示例。
