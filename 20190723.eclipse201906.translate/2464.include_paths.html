PHP包含路径

PHP Include Path是一组位置，用于查找include / require语句引用的资源。

注意：

将库或外部项目元素添加到项目的“包含路径”中也会使这些资源中定义的元素可用作内容辅助选项。

添加到项目的“包含路径”的元素会影响以下内容：

  • 运行 / 调试 - 将在运行时期间以包含路径中指定的资源和顺序搜索使用“ 相对路径 ”调用的文件。
  • 转到源 - 将根据项目的包含路径中指定的资源和顺序搜索使用“ 相对路径 ”调用的文件。

在'include'/'require'调用中，可以通过三种方式定义文件位置：

  1. 绝对路径 - 指定了确切的文件位置（例如C：\ Documents and Settings \ MyProject \ myfolder \ a.php）。在PHP网页运行/调试期间，将激活路径映射机制。
  2. 相对于当前工作目录 - 文件名前面带有“./”或“../”这些只会相对于PHP“当前工作目录”进行搜索。您可以通过运行命令“echo getcwd（）”找到当前工作目录的位置。

有关PHP的搜索机制的更多信息，请参见http://il2.php.net/manual/en/function.include.php 。
