为CXF Web服务创建项目

要创建动态Web项目并为Apache CXF Web服务开发启用它，请执行以下步骤：

  1. 选择 文件>新建>项目...> Web>动态Web项目从顶级菜单输入以下信息：
    1. 项目名称：输入项目名称。
    2. 目标运行时：选择要将Web服务部署到的服务器。如果未列出，请单击“ 新建”并浏览到安装它的位置。
    3. 动态Web模块版本：选择版本2.5
    4. 默认配置：选择服务器类型的默认配置。
    5. EAR成员身份：如果部署到Apache Tomcat，请确保未选中“ 将项目添加到EAR”复选框。
    6. 单击完成
  2. 安装CXF Project Facet
  3. 在部署时管理CXF库的导出
