3.3.3. Choosing a workspace

  1. When you start the ITE, a dialog appears asking you to choose a workspace (Figure 3.1, “Workspace chooser”).

    Figure 3.1. Workspace chooser

    The workspace is where your preferences are saved.

  2. Select the default workspace offered, or locate new one.

  3. You can choose to always select this workspace using the checkbox.

    You can see and change your workspace options in the preferences (Section 3.29.13, “Workspace preferences”).

  4. Select ”OK”.
