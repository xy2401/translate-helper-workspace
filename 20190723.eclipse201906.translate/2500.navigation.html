导航子菜单

导航子菜单在Window |下可用从菜单栏导航。

导航子菜单下的可用选项包括：

名称捷径描述
显示系统菜单Alt + -显示活动视图的系统菜单。其中包含有关视图显示方式的选项（快速查看，分离，还原，移动，大小，最小化，最大化，关闭）。
显示视图菜单按Ctrl + F10显示活动视图的菜单，其中包含每个视图的功能。
快速访问按Ctrl + 3使您可以快速访问一系列Eclipse选项。
最大化/最小化活动视图或编辑器按Ctrl + M在全屏和最小视图之间切换
激活编辑器F12切换到编辑器视图。
下一个编辑按Ctrl + F6切换到下一个打开的编辑器。
按住Ctrl键并按F6在编辑器之间滚动。
此命令类似于Windows上的Alt + Tab功能。
前编辑按Ctrl + Shift + F6切换到上一个打开的编辑器。
按住Ctrl + Shift键并按F6在编辑器之间滚动。
此命令类似于Windows上的Alt + Tab功能。
切换到编辑器按Ctrl + Shift + E打开一个显示所有打开的编辑器的对话框。允许您选择要打开，关闭或保存的编辑器。
下一个视图按Ctrl + F7切换到下一个打开视图。
按住Ctrl键并按F7在视图之间滚动。
此命令类似于Windows中的Alt + Tab功能。
上一个视图按Ctrl + Shift + F7切换到上一个打开的视图。
按住Ctrl键并按F7在视图之间滚动。
此命令类似于Windows中的Alt + Tab功能。
下一个观点按Ctrl + F8切换到上次打开的Perspective。
以前的观点按Ctrl + Shift + F8切换到下一个开放的Perspective。
