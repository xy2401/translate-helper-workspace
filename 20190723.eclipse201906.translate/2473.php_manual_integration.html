PHP手动集成

PDT可以与PHP手册站点集成，以获取最新的PHP信息。

PHP Functions视图列出了PHP手册中包含的所有PHP函数，可以使用它们轻松地将这些函数添加到脚本中。要向代码添加函数，只需将光标放在编辑器中的所需位置，然后双击列表中的所需元素即可。

您可以通过右键单击PHP Functions视图中的一个函数并选择Open Manual来打开一个PHP手册网站，其中包含有关列表中大多数函数的说明 。

将打开一个新的浏览器窗口，其中包含PHP手册站点中的功能说明。

注意：

某些功能将不会在PHP手册站点中为其分配说明。在这种情况下，浏览器将打开“无法找到服务器”错误消息。

可以从PHP Manual Preferences页面添加和编辑PHP手册站点。

此外，可以使用PHP Functions视图轻松地将功能添加到脚本中。要向代码添加函数，只需将光标放在编辑器中的所需位置，然后双击列表中的所需元素即可。

要打开PHP手册，在编辑器中单击鼠标右键，然后单击打开PHP手册 - 或按Shift + F2 。
