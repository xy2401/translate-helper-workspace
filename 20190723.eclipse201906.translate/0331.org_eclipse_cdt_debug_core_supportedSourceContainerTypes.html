C / C ++支持的源容器类型扩展

标识符：
org.eclipse.cdt.debug.core.supportedSourceContainerTypes

描述：
扩展点提供了一种机制，用于将源容器类型提供给首选项页面“C / C ++，Debug，Common Source Lookup Path”。

配置标记：

<！ELEMENT 扩展 （ sourceContainer +）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT sourceContainer EMPTY>

<！ATTLIST sourceContainer

id IDREF #REQUIRED>

  • id - 此属性的值必须是对扩展点“org.eclipse.debug.core.sourceContainerTypes”的贡献。

例子：
在上面的示例中，C调试器指定它支持absolutePath contaner类型。 

   <extension
         point="org.eclipse.cdt.debug.core.supportedSourceContainerTypes">
      <sourceContainer
            id="org.eclipse.cdt.debug.core.containerType.absolutePath"/>
   </extension>

API信息：
属性id的值必须是对扩展点“org.eclipse.debug.core.sourceContainerTypes”的贡献。
