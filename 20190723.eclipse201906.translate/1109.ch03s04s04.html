定位器

您可以使用相应的定位器来处理HTML测试中的组件。包括八个定位策略：

  • 识别码

  • ID

  • 名称

  • 链接

  • DOM

  • XPath的

  • CSS

  • UI的元素

可用操作

  • 检查CSS路径计数

  • 按定位器检查文本

  • 检查XPath计数

  • 单击定位器

  • 等待组件（基于定位器）
