在PHP项目中启用JavaScript支持

在PHP项目中启用JavaScript支持允许项目引用JavaScript库和外部文件。

注意：

为项目启用JavaScript支持后，您应该设置项目的JavaScript构建路径 ，以便为项目提供所需的资源。

这些过程描述了如何为新的PHP项目启用JavaScript支持 ， 添加对现有项目的 支持或删除JavaScript支持 。

为新的PHP项目启用JavaScript支持

要在新的PHP项目中启用JavaScript支持：

  1. 转到文件菜单，然后选择新建| PHP项目 。 - 或者 - 在Project Explorer视图中，右键单击并选择New | PHP项目 。将启动新的PHP Project向导。
  2. 在各个字段中输入所需信息。
  3. 要启用JavaScript支持，请选中“ 为此项目启用JavaScript支持”复选框。
  4. 单击完成。

将使用完整的JavaScript支持创建一个新的PHP项目。

为现有PHP项目启用JavaScript支持

JavaScript库和功能可以添加到工作区中的现有PHP项目中。

为现有PHP项目启用JavaScript支持：

在Project Explorer视图中，右键单击要为其启用JavaScript支持的项目，然后选择Configure |添加Java脚本支持。将为项目启用JavaScript支持。

删除JavaScript支持

如果您未在项目中使用JavaScript库或文件，则可以删除该项目的JavaScript支持。

要删除现有PHP项目的JavaScript支持：

在Project Explorer视图中，右键单击要为其启用JavaScript支持的项目，然后选择Configure |删除JavaScript支持 。将从项目中删除JavaScript支持，并且项目不会使用任何JavaScript库或外部文件。
