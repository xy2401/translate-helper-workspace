安装RAP目标

默认情况下，Eclipse IDE会针对运行平台（即IDE本身）编译工作空间中的所有项目。但由于必须针对RAP运行时编译RAP应用程序，因此必须将工作空间配置为执行此操作。这是通过将RAP运行时设置为目标平台来完成的 。

使用RAP工具安装最新的目标平台

在Eclipse欢迎屏幕（“ 帮助”>“欢迎” ）上，转至概述>远程应用程序平台 。在下一页上选择Install Target Platform 。在即将出现的对话框中确认默认值，从而启动目标安装过程。

安装完成后，您就可以使用RAP了。对于您的第一步，您可以尝试一些RAP示例和/或编写Hello World应用程序。

替代方案：手动安装目标

您也可以手动安装目标，例如，如果您要针对不同版本的RAP进行开发，或者将RAP与其他捆绑包结合使用。要选择目标平台，您可以使用Target Platform首选项页面（“ 窗口”>“首选项”>“插件开发”>“目标平台” ）或创建目标定义文件 。后者是可取的，因为目标定义文件可以签入到源代码存储库中并在团队成员之间共享。

  1. 首先，您可能希望创建一个项目来托管目标定义文件。普通项目将执行（ 文件>新建>项目>常规>项目 ）。
  2. 在此新项目中，创建新的目标定义： 文件>新建>其他>插件开发>目标定义 。选择一个像rap-custom.target这样的名字然后按Finish 。
  3. 目标编辑器将打开并显示此目标定义的内容。现在，您必须从p2存储库（“软件站点”）添加RAP运行时。按添加 ，选择软件站点 。将显示另一个对话框，名为Add Software Site 。
  4. 再次按Add并输入RAP存储库的URL。您可以在RAP 下载页面上找到这些URL。对于RAP 3.1，URL为http://download.eclipse.org/rt/rap/3.1 。您也可以直接从下载的zip存档中安装RAP。在这种情况下，按Archive ...并选择zip文件。
  5. 选择远程应用程序平台（RAP）类别。此类别包含四个功能。对于最小的RAP目标，您至少需要RAP运行时和RAP基本目标要求 。如果要包含源代码和演示，请选择所有四个功能。
  6. 单击目标定义编辑器右上角的“ 设置为目标平台 ”链接，激活目标平台。
