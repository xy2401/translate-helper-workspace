JavaScript编辑器文本悬停

标识符：
org.eclipse.wst.jsdt.ui.javaEditorTextHovers

描述：
此扩展点用于在JavaScript编辑器中插入文本悬停。

配置标记：

<！ELEMENT 扩展 （ 悬停 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 悬停 EMPTY>

<！ATTLIST悬停

id CDATA #REQUIRED

class CDATA #REQUIRED

标签CDATA #IMPLIED

描述CDATA #IMPLIED

activate（true | false）“false”>

  • id - id，通常与完全限定的类名相同
  • class - 实现接口org.eclipse.wst.jsdt.ui.text.java.hover.的完全限定类名org.eclipse.wst.jsdt.ui.text.java.hover.IJavaEditorTextHover
  • label - 此悬停的可翻译标签
  • description - 此悬停的可翻译描述
  • activate - 如果属性设置为“true”，它将强制在悬停激活时加载此插件

例子：
以下是悬停定义的示例：

<extension point = “org.eclipse.wst.jsdt.ui.javaEditorTextHover” > <hover id = “org.eclipse.example.wst.jsdt.internal.debug.ui。JavaDebugHover“ class = ”org.eclipse.example.wst.jsdt.internal.debug.ui。JavaDebugHover“ label = ”％javaVariableHover“ /> </ hover> </ extension>
