访问现有的CVS结帐

此过程说明如何访问之前从CVS检出的PDT中的项目。这意味着您无需再次将资源签出到新位置即可访问PDT的功能。

要访问以前签出的项目：

  1. 转到File |，启动New PHP Project Wizard 新的| PHP项目 - 或 - 右键单击Project Explorer视图并选择New | PHP项目 。将打开“新建PHP项目”向导。
  2. 在“项目内容”类别中，取消选中“使用默认值”复选框，然后浏览到已签出的CVS资源的位置。

  3. 单击完成 。将使用您签出的项目的内容创建一个新的PHP项目。
  4. 在Project Explorer视图中，右键单击您的项目并选择Team |分享项目 。将打开“共享项目”对话框。
  5. 从存储库列表中，选择CVS，然后单击“ 下一步” 。
  6. 如果尚未配置CVS存储库，则需要输入CVS存储库详细信息。有关更多信息，请参阅配置CVS连接。如果已配置CVS连接，请选中“使用现有存储库位置”选项，然后从列表中选择存储库。
  7. 单击完成 。
  8. 根据您的身份验证设置，可能会出现一个对话框，要求您提供身份验证信息。输入密码并单击“ 下一步” 。（选中“保存密码”复选框以确保不再显示此屏幕。）将打开“提交”对话框。
  9. 根据需要输入注释，然后单击“ 确定” 。

您的项目将上传到CVS存储库。

您的项目将具有存储库图标在Project Explorer视图中旁边，表示它已链接到CVS存储库。您现在可以在此项目上执行所有CVS功能（提交，更新等）。
