JavaScript折叠结构提供程序

标识符：
org.eclipse.wst.jsdt.ui.foldingStructureProviders

以来：
3.0

描述：
对此扩展点的贡献定义了JavaScript编辑器的折叠结构。也就是说，它们定义了可折叠的JavaScript源文件的区域。请参阅org.eclipse.jface.text.source.ProjectionViewer供参考。

扩展可以选择提供一个首选项块，它将出现在JavaScript编辑器首选项页面上。

配置标记：

<！ELEMENT 扩展 （ 提供者 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

id CDATA #REQUIRED

名称CDATA #IMPLIED

class CDATA #REQUIRED

preferencesClass CDATA #IMPLIED>

  • id - 此提供程序的唯一标识符
  • name - 此提供程序的名称。如果没有给出，则使用id代替。
  • class - org.eclipse.wst.jsdt.ui.text.folding.IJavaFoldingStructureProvider
  • preferencesClass - org.eclipse.wst.jsdt.ui.text.folding.IJavaFoldingPreferenceBlock

例子：
请参阅org.eclipse.wst.jsdt.ui.text.folding.DefaultJavaFoldingStructureProvider的示例。

提供的实施：
org.eclipse.wst.jsdt.ui.text.folding.DefaultJavaFoldingStructureProvider为JavaScript编辑器提供默认的折叠结构。 org.eclipse.wst.jsdt.internal.ui.text.folding.DefaultJavaFoldingPreferenceBlock为默认结构提供程序提供首选项块。
