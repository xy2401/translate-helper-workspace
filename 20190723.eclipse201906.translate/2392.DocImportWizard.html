Import Wizard

  

The wizard is the basis for Oomph's manually-triggered updates to the project streams included to the automated installation and provisioning process.

The wizard's footer contain the following controls:

Brings up this help.
Navigates to the previous page.
Navigates to the next page.
Completes the wizard, performing any final actions.
Closes the wizard, taking no further action.

The project wizard is available in any running Eclipse product with Oomph installed via File → Import… → Oomph → Projects into Workspace and the execution tool bar button. Its purpose is to augment the workspace's set of streams.
