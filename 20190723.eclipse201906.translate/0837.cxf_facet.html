CXF Project Facet

Facets can be thought of as units of functionality that can be added and removed from facet based projects in the workspace.

For more information on facets see the WTP Project Facets documentation.

JAX-WS Tools provides the CXF Facet that performs the following when installed on a Dynamic Web Project:

  • Configures the project with the CXF Classpath Container.
  • Configures the project with the CXF JAX-WS endpoint configuration option that is selected in the Endpoint Config preferences.
  • Enables JAX-WS Annotation processing if the option is selected in the JAX-WS preferences.

The CXF Facet has the following facet prerequisites:

  • Dynamic Web Module version 2.5 or greater
  • Java version 5.0 or greater

The CXF Facet and it's prerequisites are grouped into a project configuration that can be selected when creating a Dynamic Web Project.
