CXF项目方面

可以将构面视为可以在工作区中基于构面的项目添加和删除的功能单元。

有关构面的更多信息，请参阅WTP Project Facets文档 。

JAX-WS Tools提供了在动态Web项目上安装时执行以下操作的CXF Facet：

  • 使用CXF Classpath Container配置项目。
  • 使用在Endpoint Config首选项中选择的CXF JAX-WS端点配置选项配置项目。
  • 如果在JAX-WS首选项中选择了该选项，则启用JAX-WS Annotation处理。

CXF Facet具有以下方面的先决条件：

  • 动态Web模块版本2.5或更高版本
  • Java 5.0或更高版本

CXF Facet及其先决条件分为可在创建动态Web项目时选择的项目配置。
