安装和设置

Gcov插件依赖于binutils（例如addr2line，c ++ filt和nm）。只要这些binutils位于PATH中，Gcov就可以在任何平台上使用。例如，您可以在Windows上使用Cygwin。

旧版本配置3.2.0之前

首先，用户必须编译C / C ++程序，并在运行该工具之前使用“-ftest-coverage -fprofile-arcs”选项启用性能分析。这两个选项必须通过项目属性传递给编译器（C / C ++ Build-> Settings-> Tool Settings-> GCC C Compiler-> Miscellaneous-> Other flags）和链接器（C / C ++ Build-> Settings） - >工具设置 - > GCC C链接器 - >其他 - >链接器标志）。对于CDT Autotools项目，在项目属性下有一个复选框（Autotools-> Configure Settings-> configure-> Advanced-> Gcov support）。

较新版本配置3.2。*以后
