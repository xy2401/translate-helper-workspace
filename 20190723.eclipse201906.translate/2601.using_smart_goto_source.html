使用Smart Goto Source

此过程描述如何使用Smart Goto Source函数以便轻松导航到元素的声明。

要使用Smart Goto Source功能：

  1. 将鼠标悬停在要导航到其源声明的元素上。将显示工具提示，显示元素的原始位置。
  2. 按住Ctrl键并移动光标，直到元素加下划线。
  3. 单击元素。

您将自动获取元素的源代码。如果声明位于不同的文件中，则将打开此文件。

注意：

如果为项目启用了JavaScript支持，Smart Goto Source也将可用于JavaScript元素。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。
