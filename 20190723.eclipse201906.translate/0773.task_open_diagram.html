打开JPA Diagram Editor

  1. 在Project Explorer中 ，选择要用于实体的项目。

  2. 从项目上下文菜单中，选择JPA Tools → Open Diagram 。

创建新实体

在Palette的Entities列表中，选择 （ 实体 ） ，然后单击图表区域。

Eclipse IDE使用预定义的名称和主键创建新实体。

创建新的嵌入式

在Palette的Entities列表中，选择 （ 可嵌入 ） ，然后单击图表区域。

Eclipse IDE使用预定义的名称创建新的embeddable。

创建新的映射超类

在Palette的Entities列表中，选择 （ Mapped Superclass ） ，然后单击图表区域。

Eclipse IDE使用预定义的名称创建新的映射超类。

在图上显示项目中的现有持久类型

  • 要显示单个持久性类型，请使用拖放操作将持久性类型从Project Explorer复制到JPA图。

  • 要显示所有持久性类型，请使用鼠标左键单击JPA图，然后选择“ 显示所有持久性类型” 。另一种方法是使用拖放操作将JPA Content节点从项目资源管理器中的项目复制到JPA图表。
