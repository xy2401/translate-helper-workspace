入门

配置解释器

要验证是否为Shell脚本开发正确设置了Eclipse，您需要确保已配置解释器。选择菜单项Window - > Preferences以打开工作台首选项。然后选择Shell Script - > Interpreters首选项页面以显示当前的解释器配置。

如果没有列出解释器，您可以通过按“ 添加”按钮手动添加解释器，或者让ShellEd尝试通过按“ 搜索”按钮自动发现解释器。添加一个或多个解释器后，您可以通过选中解释器名称旁边的框来选择默认使用的解释器。

新Shell脚本项目

要创建新的shell脚本项目，请选择菜单项File - > New - > Project，然后从可用项目类型列表中选择Shell Script Project 。

在下一页中，将要求您选择项目的名称和位置并选择解释器。

如果ShellEd中没有配置解释器，则会说您的默认解释器“当前未定义”。您仍然可以创建项目，但是您将无法运行或调试shell脚本。强烈建议您至少配置一个解释器，此时可以通过单击“ 配置解释器”链接来执行此操作 ，该链接将转到上一节中描述的首选项页面。

单击“ 完成”按钮后，新创建的shell脚本项目将在Project Explorer类型视图中可见。

现有项目

您可能在要运行和调试的其他项目中有shell脚本。在这种情况下，您可以通过右键单击它们并选择Configure - > Convert to Shell Script Project ..菜单项，为现有的非shell脚本项目添加shell脚本支持。
