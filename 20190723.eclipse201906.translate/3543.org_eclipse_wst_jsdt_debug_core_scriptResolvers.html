scriptResolvers

标识符：
org.eclipse.wst.jsdt.debug.core.scriptResolvers

以来：
3.4

描述：
此扩展点用于提供脚本解算器，该解析器在JSDT中的任何位置进行查询，其中必须将ScriptReference解析为工作空间本地IFile 。

典型用法的一个示例是解析断点的工作空间路径是否与给定ScriptReference路径匹配

配置标记：

<！ELEMENT 扩展 （ scriptResolver ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT scriptResolver EMPTY>

<！ATTLIST scriptResolver

class CDATA #REQUIRED>

一个解析器，可以帮助确定工作区本地IFiles和JavaScript ScriptReferences之间的相等性


  • class - 实现org.eclipse.wst.jsdt.debug.core.model的Java类的完全限定名称。ScriptResolver

例子：
以下是默认脚本解析程序的示例：

   <extension point="org.eclipse.wst.jsdt.debug.core.scriptResolvers">
      <scriptResolver
            class="org.eclipse.wst.jsdt.debug.internal.core.model.DefaultScriptResolver">
      </scriptResolver>
   </extension>

API信息：
类字段必须指定扩展org.eclipse.wst.jsdt.debug.core.model.AbstractScriptResolver 。
