创建一个JAX-WS处理程序

JAX-WS处理程序是消息拦截器，可用于对入站和出站消息执行其他处理。

协议处理程序特定于协议（例如SOAP），并且它们可以访问或更改消息的协议特定方面，而逻辑处理程序是协议不可知的并且可以仅作用于消息有效负载。

先决条件：

完成以下任务之一以生成Web服务：

  • 使用Apache CXF从WSDL文档生成SEI和框架实现bean
  • 使用Apache CXF从WSDL文档生成JAX-WS Web服务客户端
要创建JAX-WS处理程序：
  1. 选择从顶级菜单中选择Window> Open Perspective> Java EE切换到Java EE透视图。
  2. 选择打开JAX-WS处理程序向导 文件>新建>其他...> Web服务> JAX-WS>顶级菜单中的JAX-WS Handler 。
  3. JAX-WS Handler页面：

    1. 浏览以选择“ 源”文件夹。
    2. 浏览以选择现有包或输入新包名称。
    3. 输入JAX-WS处理程序的名称 。
    4. 选择处理程序类型 ：
      • 逻辑处理程序
      • 协议处理程序（SOAP）

    在“ 配置”部分中：

    1. 选中“ 将此处理程序添加到处理程序链”复选框。

    2. 选择“ 创建新的处理程序链文件”，然后浏览以选择项目中将生成处理程序链文件的位置。

      注意 ：要将此JAX-WS处理程序添加到项目中的现有处理程序链 ，请选择“ 将此处理程序添加到现有处理程序链文件”并浏览以选择文件位置。

      选择Next将显示一个向导页面，允许您在处理程序链文件中选择新JAX-WS处理程序的位置。

    3. 选择将此处理程序链与Web服务关联，然后浏览以选择在先决条件中创建的Web服务实现或服务端点接口。

      单击完成 。

      该向导将生成一个框架JAX-WS Handler，处理程序链文件，并使用@ javax.jws注释Web服务。HandlerChain注释。
