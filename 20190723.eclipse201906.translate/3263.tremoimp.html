Importing resources from a remote file system

A page in the Import wizard enables you to import resources from a remote file system to the workspace.

To import resources to the workspace:
  1. Select File > Import from the menu bar. The Import wizard opens.
  2. Select Remote file system. Click Next. The Remote file system page opens.
  3. Select or type a source folder, or click Browse to select a folder from the Browse For Folder window. Note that you can create a connection from this window.
  4. Use the left and right panes to select resources to import. Choose individual resources from the right pane, or entire folders from the left one. You can select files from multiple projects.
  5. In the second Folder field, specify the destination for the imported resources.
  6. Optional: Select the check box to overwrite (without prompting) files of the same name in the workspace.
  7. Select one of the radio buttons to either replicate the remote folder structure in the workspace or create only the actual folders you have selected.
  8. When you have specified the source and destination, click Finish to import the resources to the workspace.
