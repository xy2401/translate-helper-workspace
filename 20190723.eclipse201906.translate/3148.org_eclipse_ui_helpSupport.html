帮助支持

标识符：
org.eclipse.ui.helpSupport

以来：
3.0（最初名为org.eclipse.help.support）

描述：
此扩展点用于提供帮助系统UI。该平台应配置不超过一个帮助系统UI。

配置标记：

<！ELEMENT 扩展 （ 配置 ？）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 配置 EMPTY>

<！ATTLIST配置

class CDATA #REQUIRED>

  • class - 用于显示在线和上下文相关帮助的实现类。该类必须实现org.eclipse.ui.help.AbstractHelpUI接口。

例子：
以下是帮助支持扩展点的示例用法：


 <extension point="org.eclipse.ui.helpSupport">
  <config class="com.example.XYZHelpUI"/>
 </extension>

API信息：
提供的类必须实现org.eclipse.ui.help.AbstractHelpUI 。在该类中实现抽象方法可确定当用户请求联机帮助或上下文相关帮助时会发生什么。实现应该使用org.eclipse.help访问提供的帮助信息。帮助系统。

提供的实施：
org.eclipse.help.ui插件包含帮助系统UI的实现。
