工作集

标识符：
org.eclipse.ui.workingSets

以来：
2.0

描述：
此扩展点用于定义工作集向导页面。工作集包含许多IAdaptable类型的元素，可用于将元素分组以呈现给用户或对一组元素进行操作。工作集向导页面用于创建和编辑包含特定类型元素的工作集。updater类能够监视工作集中包含的资源的更改，并且元素适配器类能够在包含在工作集中之前转换资源。

要选择工作集，将向用户显示工作台中存在的工作集列表。从该列表中，可以使用此扩展点定义的向导页面之一选择和编辑工作集。始终使用用于创建现有工作集的向导页面或使用默认的基于资源的工作集页面编辑现有工作集（如果原始页面不可用）。

用户可以从同一工作集选择对话框中定义新的工作集。定义新的工作集时，插件提供的向导页面前面会有一个列出所有可用工作集类型的页面。此列表由每个工作集扩展的名称属性值组成。

视图提供用户界面以打开工作集选择对话框，并且必须存储选定的工作集。

资源导航器使用工作集来过滤导航器视图中的元素。除了工作集元素本身之外，视图中仅显示工作集元素的父元素和子元素。

配置标记：

<！ELEMENT 扩展 （ workingSet *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT workingSet （EMPTY *）>

<！ATTLIST workingSet

id CDATA #REQUIRED

名称CDATA #REQUIRED

描述CDATA #IMPLIED

icon CDATA #IMPLIED

pageClass CDATA #IMPLIED

updaterClass CDATA #IMPLIED

elementAdapterClass CDATA #IMPLIED>

  • id - 可用于标识此工作集对话框的唯一名称。
  • name - 将由工作集页面显示和编辑的元素类型的名称。这应该是一个描述性的名称，如“资源”或“Java元素”。
  • description - 此工作集类型的简要说明。这可能包括可以添加到集合中的一般元素种类。在3.4中添加。
  • icon - 将在工作集创建向导的第一页以及工作集选择对话框中的工作集类型列表中显示的图像的相对路径。
  • pageClass - 实现org.eclipse.ui.dialogs的Java类的完全限定名称。IWorkingSetPage 。
  • updaterClass - 实现org.eclipse.ui的Java类的完全限定名称。IWorkingSetUpdater 。使用此属性不会导致急切的插件激活。只有在激活声明包时才会实例化此类。
  • elementAdapterClass - 实现org.eclipse.ui的Java类的完全限定名称。IWorkingSetElementAdapter 。使用此属性不会导致急切的插件激活。只有在激活声明包时才会实例化此类。在3.3中添加。

例子：
以下是如何定义资源工作集对话框扩展以显示和编辑通用IResource元素的示例。此外，它提供了一个updaterClass，它可以动态地处理资源的移除和添加，以及一个能够使对象适应IResources的elementAdapterClass。 

<extension point = “org.eclipse.ui.workingSets” > <workingSet id = “org.eclipse.ui.resourceWorkingSetPage” name = “Resource” description = “包含基本资源（文件，文件夹和项目）” icon = “ icons / resworkset.png“ pageClass = ”org.eclipse.ui.internal.dialogs。ResourceWorkingSetPage“ updaterClass = ”org.eclipse.ui.internal.workingsets。ResourceWorkingSetUpdater“ elementAdapterClass = ”org.eclipse.ui.internal.workingsets。ResourceWorkingSetElementAdapter“ > </ workingSet> </ extension>

API信息：
pageClass属性的值必须表示实现org.eclipse.ui.dialogs的类。IWorkingSetPage接口。

提供的实施：
工作台提供了一个工作集向导页面，用于创建和编辑基于资源的工作集。
