搜索PHP元素

搜索PHP元素是一种允许您使用定义的规范搜索PHP元素的功能。PHP元素是代码中使用的类，函数，常量，类型，方法和引用/声明。如果要在工作区，项目或单个文件中找到其中一个元素，请使用此选项。

注意：

有关“搜索”对话框中“搜索”选项卡的信息，请阅读以下内容。要了解有关“搜索”对话框中其他选项卡的更多信息，请参阅“ Workbench用户指南 ”中的“ 文件搜索 ” 。如果您使用的是“远程搜索”选项，请注意它不会搜索远程服务器的所有资源。

此过程描述如何在文件和项目中搜索PHP元素（类，函数和常量）。

要搜索PHP元素：

  1. 从菜单栏转到搜索|搜索 - 或 - 按Ctrl + H.将打开“PHP搜索”对话框。
  2. 输入搜索字符串。您可以选择使用通配符 。

您可以通过选择“区分大小写”框来使搜索区分大小写。

  1. 选择是否搜索：
    • Type - A class（此选项将搜索方法和字段。）
    • 方法 - 一个功能
    • 场 - 一个常数
  2. 选择是否将搜索范围限制为：
    • 声明 - 定义类型，方法或字段的位置。
    • 引用 - 引用类型，方法或字段的任何位置。
    • 所有出现 - 引用或声明类型，方法或字段的任何位置。
  3. 要扩展搜索以包括解释器系统库，请选择“搜索解释器系统库”框。这允许PDT在PHP解释器首选项中选择的PHP版本库中进行搜索。
  4. 选择是否在您的搜索中搜索：
    • 工作区 - 整个工作区。
    • 所选资源 - 在打开“搜索”对话框之前，在Project Explorer视图中选择这些资源。所选资源中的所有子文件和子文件夹都将包含在搜索中。
    • 封闭项目 - 所选资源所在的项目。
    • 工作集 - 单击选择...以选择所需的工作集。
  5. 单击搜索

搜索视图将打开，显示搜索结果。

要转到元素，请在搜索视图中双击所需选项。

注意：

默认情况下，“文件搜索”，“远程搜索”，“任务搜索”，“Java搜索”和“JavaScript搜索”对话框将使用“PHP搜索”对话框进行选项卡。要使这些不可用，请单击“PHP搜索”对话框中的“ 自定义... ”并取消标记所需选项。

热键

以下热键可用 ：

行动捷径
Esc / Alt + F4关闭“搜索”对话框。
Alt + A.将光标移动到“搜索字符串”字段。
Alt + I选中/取消选中“区分大小写”复选框。
Alt + T.检查“搜索”框中的“类型”。
Alt + M.检查“搜索”框中的“方法”。
Alt + F.检查“搜索”字段中的“字段”。
Alt + L.检查“限制到”字段中的“声明”。
Alt + R.检查“限制到”字段中的“参考”。
Alt + O.检查“限制到”字段中的“所有出现次数”。
Alt + Y.选中/取消选中“搜索解释器系统库”字段。
Alt + W.在“范围”字段中选择“工作区”。
Alt + D.在“范围”字段中选择“所选资源”。
Alt + J.在“范围”字段中选择“包围项目”。
Alt + K.在“范围”字段中选择“工作集”。
Alt + H.打开“选择工作集”对话框。
Alt + S.执行搜索并在“搜索”视图中显示结果。
Alt + Z.打开“搜索页面选择”对话框，允许您自定义搜索页面。
