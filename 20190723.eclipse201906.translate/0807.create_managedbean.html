创建托管Bean

您可以创建托管bean并将其添加到Web应用程序，以将Bean属性和业务逻辑绑定到Java ServerPage（JSP）的用户界面组件。使用New Managed Bean向导，您可以从现有Java类创建托管bean，也可以创建新的Java类。

要创建托管bean：

  1. 在J2EE透视图的Project Explorer中选择faces-config.xml 。

    将打开JSF应用程序配置编辑器。

  2. 选择编辑器的ManagedBean选项卡。

    将显示ManagedBean页面。

  3. 单击添加 。

    出现“新建托管Bean向导”。

  4. 按照向导说明操作。

    该向导显示帮助，帮助您做出需要做出的决定。

    注意

    项目名称不能包含空格。
