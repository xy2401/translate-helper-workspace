首选项 - 通用>代码解析

此页面用于控制各种代码解析选项。

该工具支持特殊注释标记，可用于隐藏其解析器中的代码。如果您需要向布局中添加一些无法由编辑器解析的复杂动态代码，这可能很有用。您还可以使用特殊的行尾注释来隐藏单行代码（例如，小部件的单个属性）。

开始隐藏代码块标记

此首选项确定用于指定隐藏代码块开头的注释文本。

结束隐藏的代码块标记

此首选项确定用于指定隐藏代码块结尾的注释文本。

在解析后突出显示源中的访问/执行行

此首选项确定是否突出显示在分析期间访问/执行的行。

访问线突出显示颜色

此首选项确定颜色用于突出显示已访问/已执行的行。

单个隐藏代码行标记

此首选项确定用于隐藏单行代码的行尾注释文本。
