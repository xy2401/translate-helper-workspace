Javadoc导出向导

标识符：
org.eclipse.jdt.ui.javadocExportWizardPage

以来：
3.4

描述：
此扩展点允许向Javadoc导出向导添加向导页面。

配置标记：

<！ELEMENT 扩展 （ javadocExportWizardPage +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT javadocExportWizardPage EMPTY>

<！ATTLIST javadocExportWizardPage

id CDATA #REQUIRED

描述CDATA #IMPLIED

class CDATA #REQUIRED>

  • id - 标识页面
  • description - 页面的描述。此属性应为已翻译的字符串。
  • class - 实现此页面的类的名称。该类必须是公共的并扩展org.eclipse.jdt.ui.wizards。带有公共0参数构造函数的JavadocExportWizardPage 。

例子：
以下是Javadoc导出向导页面的示例：

<extension point = “org.eclipse.jdt.ui.javadocExportWizardPage” > <javadocExportWizardPage id = “com.example.myplugin.myJavadocExportPage” description = “My Taglet Configuration” class = “com.example。MyJavadocExportPage“ > </ javadocExportWizardPage> </ extension>
