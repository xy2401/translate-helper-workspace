FileUpload

RWT FileUpload control.

Available CSS Properties

Properties that can be defined for this element:

  • animation

    Supports animations-types "hoverIn" and "hoverOut". Others are ignored.

  • background-color

    Background color for FileUpload controls.

  • background-image

    Background image for FileUpload controls.

  • background-position

    Background image position for FileUpload controls.

  • background-repeat

    Background image repetition for FileUpload controls.

  • border

    Border for FileUpload controls.

  • border-bottom

    The bottom side border for FileUpload controls.

  • border-left

    The left side border for FileUpload controls.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for FileUpload controls.

  • border-top

    The top side border for FileUpload controls.

  • color

    Text color for FileUpload controls.

  • cursor

    Control's cursor.

  • font

    Font for FileUpload controls text.

  • opacity

    The opacity of the FileUpload controls. A value between 0 and 1.

  • padding

    Padding (inner distance) for FileUpload controls.

  • spacing

    Space between text and image.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Indicates that the FileUpload controls should have a border.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies for disabled FileUpload controls.

  • :hover

    Applies while the user hovers over a FileUpload control.

  • :pressed

    Applies while a FileUpload control is pressed.

FileUpload-FocusIndicator

The frame that is drawn when the FileUpload control is focused.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The fill-color of the frame.

  • border

    The border of the frame. 3D-styles inset, outset, etc are not supported.

  • border-bottom

    The bottom side border of the frame.

  • border-left

    The left side border of the frame.

  • border-right

    The right side border of the frame.

  • border-top

    The top side border of the frame.

  • margin

    The margin between the FileUpload control border and the frame. Only used if padding is set to "0px".

  • opacity

    The opacity of the indicator. A value between 0 and 1.

  • padding

    The padding between the frame and the FileUpload control text. If this is set to "0px", margin will be used to layout the FocusIndicator.
