cPropertyTab

标识符：
org.eclipse.cdt.ui.cPropertyTab

以来：
4

描述：
属性/首选项UI元素的实现。有两种显示属性/首选项数据的方法：在带有选项卡的页面上，或在简单页面上。在第一种情况下，每个cPropertyTab扩展类表示tabfolder中的单个选项卡。在第二种情况下，cPropertyTab扩展类表示整个页面内容（除了所有页面通用的标题元素）。请注意，cPropertyTab扩展类（下面的“标签”）不区分这两种情况，而且，它们可以在多标签和单标签模式下显示。可以选择显示模式的页面。在多标签页中，标签按其重量定义的顺序显示。

配置标记：

<！ELEMENT 扩展 （ tab +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 选项卡 EMPTY>

<！ATTLIST标签

icon CDATA #IMPLIED

class CDATA #REQUIRED

名称CDATA #REQUIRED

父CDATA #REQUIRED

工具提示CDATA #IMPLIED

重量CDATA #IMPLIED

helpId IDREF #IMPLIED>

  • icon - 要在tabfolder中显示相应选项卡的图标。单标签页面被忽略
  • class - 实现org.eclipse.cdt.ui.newui的类。ICPropertyTab
  • name - 选项卡文件夹中的选项卡名称。单标签页面被忽略。
  • parent - 容器页面的类名称。通常页面是从org.eclipse.cdt.ui.newui派生的。AbstractPage，但它不是强制性的。
  • tooltip - 选项卡上显示的工具提示文本。
  • weight - 用于对选项卡文件夹内的选项卡进行排序的抽象值。单标签页面被忽略。
  • helpId - 给定选项卡的帮助上下文ID。
