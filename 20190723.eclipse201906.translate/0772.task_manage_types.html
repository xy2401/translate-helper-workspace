管理持久性类型

概观

您可以对特定实体图形模型执行以下操作：

  • 重命名实体

    就源代码而言，此操作仅更改@Entity注释的name属性。

  • 添加，重命名或删除属性

    注意：

    Eclipse IDE自动为新属性生成getter和setter方法。新属性的默认数据类型是java.lang。字符串 。

    在源代码方面，重命名属性会更改实体的字段名称以及字段的getter和setter方法。

    删除属性也会从数据模型中删除它们。

  • 折叠或展开实体或属性组

  • 保存或放弃更改，同时从图中删除实体而不将其从应用程序数据模型中删除

  • 从图中删除实体并将其从应用程序数据模型中删除

程序

要执行上述操作，请按照以下步骤操作：

  1. 将鼠标光标放在要管理的实体上。

    实体显示一个按钮栏。


  2. 选择相关的按钮。每个按钮都有一个信息工具提示，描述相关操作。

  3. 要重命名实体或属性，请双击实体或属性的标题，更改名称，然后按Enter 。

  4. 要删除属性，请单击该属性并按DELETE或从上下文菜单中选择“ 删除 ”。

    您还可以通过将鼠标光标放在属性上并选择“ 删除属性”按钮来删除属性 。

注意：

所有操作都可以在可嵌入的，以及映射的超类图形模型上执行。
