属性选项卡

标识符：
org.eclipse.ui.views.properties.tabbed.propertyTabs

描述：
描述将提供给选项卡式属性表页面的选项卡列表。

配置标记：

<！ELEMENT 扩展 （ propertyTabs ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT propertyTabs （ propertyTab +）>

<！ATTLIST propertyTabs

contributorId CDATA #REQUIRED>

  • contributorId - 属性表页面参与者的唯一标识符。

<！ELEMENT propertyTab EMPTY>

<！ATTLIST propertyTab

标签CDATA #REQUIRED

类别CDATA #REQUIRED

afterTab CDATA #IMPLIED

id CDATA #REQUIRED

缩进（true | false）

image CDATA #IMPLIED>

  • label - 要在选项卡上显示的标签。
  • category - 用于对选项卡进行分组的类别。
  • afterTab - 当类别中有多个选项卡时，选项卡按afterTab属性排序。
  • id - 选项卡的唯一ID。
  • 缩进 -如果true ，则该选项卡中idented。这是为了表示父标签的子标签或类别。
  • image - 如果提供了图像，则当选项卡处于活动状态时，图标图像将显示在选项卡上。

例子：
<extension point = “org.eclipse.ui.views.properties.tabbed.propertyTabs” > <propertyTabs contributorId = “org.eclipse.ui.tests.views.properties.tabbed.views。TestsView“ > <propertyTab category = ”first“ id = ”propertyTab1“ label = ”Name“ > </ propertyTab> <propertyTab afterTab = ”propertyTab1“ category = ”first“ id = ”propertyTab2“ label = ”Project“ > </ propertyTab> <propertyTab category = “second” id = “propertyTab8” label = “Message” > </ propertyTab> </ propertyTabs> </ extension>
