图书馆位置解析器

标识符：
org.eclipse.jdt.launching.libraryLocationResolvers

以来：
3.7

描述：
此扩展点允许解析JRE / SDK中库的其他信息

配置标记：

<！ELEMENT 扩展 （ 解析器 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT resolver EMPTY>

<！ATTLIST解析器

class CDATA #IMPLIED>

  • class - 实现此解析程序的类。该类必须实现org.eclipse.jdt.launching.ILibraryLocationResolver

例子：
以下是解析器的示例：


  <extension
         point="org.eclipse.jdt.launching.libraryLocationResolvers">
      <resolver
            class="com.example.ExtLibraryInfoResolver">
      </resolver>
  </extension>
