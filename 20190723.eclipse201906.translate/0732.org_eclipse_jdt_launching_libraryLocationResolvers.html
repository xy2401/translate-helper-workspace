Library Location Resolver

Identifier:
org.eclipse.jdt.launching.libraryLocationResolvers

Since:
3.7

Description:
This extension point allows to resolve additional information for libraries in a JRE / SDK

Configuration Markup:

<!ELEMENT extension (resolver+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT resolver EMPTY>

<!ATTLIST resolver

class CDATA #IMPLIED>

  • class - The class that implements this resolver. The class must implement org.eclipse.jdt.launching.ILibraryLocationResolver

Examples:
The following is an example of the resolver: 


  <extension
         point="org.eclipse.jdt.launching.libraryLocationResolvers">
      <resolver
            class="com.example.ExtLibraryInfoResolver">
      </resolver>
  </extension>
