5.3.3.1。�等待组件（基于定位器）

  • 使用此操作可基于其定位器等待元素。

  • 该操作不需要组件名称，也不需要对象映射 - 根据您输入的定位器找到AUT中的元素。

  • 例如，您可以输入如下所示的xpath定位器：

    XPath的= ...

  • 您可以输入这样的CSS定位器：

    CSS = ...

    您很可能希望在整个定位器周围使用单引号来取消字符串中的任何特殊字符。

  • 您可以使用不同类型的定位器，并可以使用网站检查工具找到元素的定位器。例如，您可以复制项目的唯一选择器（CSS Locator）。

  • 参数：

    • Millisecs超时

    • 能见度后延迟

    • 定位器
