代码覆盖率首选项

“代码覆盖率首选项”页面显示“代码覆盖率”视图中的代码预览，其中包含当前颜色和字体设置。使用各种功能（如性能分析，单元测试和调试）时，将显示代码覆盖率视图，以显示此功能已涵盖哪些代码行。

可以从Window |访问“代码覆盖率首选项”页面偏好| PHP |代码覆盖范围

要配置代码覆盖颜色和字体：
  1. 单击“颜色和字体”链接。将显示“颜色和字体”首选项页面，并打开“PHP调试”类别。

    。
  2. 通过选择相关选项并在颜色选择框（右上角）中单击所需颜色，为覆盖线和未覆盖线选择所需的背景颜色。
  3. 单击“应用”
  4. 更改将显示在“代码覆盖率”预览页面中。
