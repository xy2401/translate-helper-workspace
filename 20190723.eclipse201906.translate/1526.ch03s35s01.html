3.35.1. Introduction to launch configurations

If you are using the the Eclipse Jubula Feature in your Eclipse installation, you can start AUTs from within Eclipse, similar to the way in which you would debug an application using Eclipse. This section describes how to start your AUT directly from the IDE.
