控制台颜色提供商

标识符：
org.eclipse.debug.ui.consoleColorProviders

以来：
2.1

描述：
此扩展点提供了一种为进程提供控制台文档着色方案的机制。颜色提供程序将用于控制台中的颜色输出。

配置标记：

<！ELEMENT 扩展 （ consoleColorProvider *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT consoleColorProvider EMPTY>

<！ATTLIST consoleColorProvider

id CDATA #REQUIRED

class CDATA #REQUIRED

processType CDATA #REQUIRED>

  • id - 指定此控制台颜色提供程序的唯一标识符。
  • class - 指定实现IConsoleColorProvider的Java类的完全限定名称
  • processType - 指定此颜色提供程序所用的进程类型。该属性对应于进程属性IProcess.ATTR_PROCESS_TYPE 。

例子：
以下是控制台颜色提供程序扩展点的示例：
<extension point = “org.eclipse.debug.ui.consoleColorProviders” > <consoleColorProvider id = “com.example。ExampleConsoleColorProvider“ class = ”com.example。ExampleConsoleColorProvider“ processType = ”ExampleProcessType“ > </ consoleColorProvider> </ extension>
在上面的示例中，贡献的颜色提供程序将用于“ExampleProcessType”类型的进程，该进程对应于进程属性IProcess。ATTR_PROCESS_TYPE。流程类型是客户端定义的，由创建流程的客户端设置。

API信息：
属性类的值必须是实现接口org.eclipse.debug.ui.console的Java类的完全限定名称。IConsoleColorProvider 。
