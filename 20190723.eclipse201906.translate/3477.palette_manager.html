调色板经理

  可以从调色板上下文菜单访问调色板管理器 。

Palette Manager对话框允许完全配置特定于工具包的调色板。可以添加，编辑，重新排列和删除类别和条目，并导入JAR文件。

可以使用向上和向下按钮或通过拖放重新排列项目。“ 全部折叠”和“ 全部 展开”按钮显示或隐藏列表中的所有元素。

通过取消选中其类别或组件名称旁边的复选框，可能会从调色板中隐藏项目。

 

调色板管理器命令

命令

描述
添加类别... 创建一个包含调色板条目的新类别。该类别可以具有名称，描述和初始状态可见和/或默认打开 。可以在任何现有类别之前插入类别，也可以将其插入列表末尾。
添加条目>> 单击“ 添加条目>>”按钮会弹出“ 添加条目”菜单。

选择Component ...命令以创建表示用户指定组件的新调色板条目。该条目必须具有名称，类和描述（在调色板中显示为悬停帮助）。如果窗口小部件在其bean信息中定义了自己的图标，则默认情况下将使用该图标。该组件最初可见或不可见 ，并且可以添加到任何可用类别。

选择Single static factory ...命令以从静态工厂添加单个组件。要添加单个工厂项目，请为其命名，选择工厂类和方法签名，并为其提供说明。该组件最初可见或不可见 ，并且可以添加到任何可用类别。

选择几个静态工厂...命令以从静态工厂添加多个组件。要同时添加多个工厂项，请选择工厂类和方法签名。组件名称和描述将被赋予默认值。组件最初可见 ，可以添加到任何可用类别。

选择Single instance factory ...命令以从实例工厂添加单个组件。要添加单个工厂项目，请为其命名，选择工厂类和方法签名，并为其提供说明。该组件最初可见或不可见 ，并且可以添加到任何可用类别。

选择几个实例工厂...命令以从实例工厂添加多个组件。要同时添加多个工厂项，请选择工厂类和方法签名。组件名称和描述将被赋予默认值。组件最初可见 ，可以添加到任何可用类别。
导入Jar ... 使用JAR文件中的窗口小部件类（从类路径，工作空间或磁盘中选择）创建多个新的调色板条目。（可选）创建新类别以添加条目。类别名称默认为JAR的名称。

 

如果在选择jar文件时没有出现小部件，这通常表明jar文件没有提供Sun的JavaBean清单阅读器可读的正确JavaBean清单。请联系罐子提供商进行更正。只要jar位于项目路径上，您就可以使用Add Entry >>命令手动将各个小部件添加到调色板中。
编辑... 编辑选定的类别或组件。
去掉... 删除所选的类别或条目。
向上 在列表中向上移动选定的组件。组件也可以通过拖放移动。
下 在列表中向下移动选定的组件。组件也可以通过拖放移动。
全部收缩 折叠树中的所有组件。
展开全部 展开树中的所有组件。
