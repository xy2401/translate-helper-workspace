概观

Eclipse Callgraph插件提供了程序的可视功能跟踪。这允许您查看已配置应用程序使用的所选（甚至所有）功能的可视化。

此插件允许您直接在Eclipse IDE中分析C / C ++项目，提供各种运行时详细信息，例如：

  • 函数调用之间的关系
  • 每个函数被调用的次数
  • 每个函数实例所花费的时间（相对于程序的执行时间）
  • 函数的所有实例所花费的时间（相对于程序的执行时间）

Eclipse Callgraph使用SystemTap来执行函数跟踪。有关SystemTap的更多信息，请参阅SystemTap初学者指南 。
