允许将地图文件提供给基于地图文件的CDT CHelpProvider。

标识符：
org.eclipse.cdt.ui。HELPINFO

以来：
CDT 4.0.2

描述：
允许将地图文件提供给基于地图文件的CDT CHelpProvider。

配置标记：

<！ELEMENT 扩展 （ helpInfo +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT helpInfo EMPTY>

<！ATTLIST helpInfo

文件CDATA #REQUIRED

格式CDATA #IMPLIED>

允许定义相对于插件目录的映射文件路径。


  • file - 包含相对于插件目录的映射文件路径。
  • format - 可以包含地图文件格式的ID。目前，只有一种（默认）帮助地图格式。
