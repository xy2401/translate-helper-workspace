Project Explorer视图

注意：

从6.0.0开始，PHP开发工具使用通用的Project Explorer 。

Project Explorer视图是一个内部文件系统浏览器，允许您查看Workspace中的所有PHP项目和文件。它显示了Workbench中PHP项目的PHP元素层次结构，并为您提供了特定于PHP的可用资源视图。在每个项目中，源文件夹和引用的库都显示在树中。此外，该视图还显示了所有PHP代码元素（类，函数，变量等）。单击元素或声明将导致相应的代码显示在PHP编辑器中。

有关Project Explorer视图中显示的图标的说明，请参阅PHP图标 。

Project Explorer视图菜单命令

可以通过视图菜单图标访问视图的菜单 。

图标名称描述

编辑活动工作集允许您编辑选定的工作集。有关更多信息，请参阅PHP工作集 。

注意：

默认情况下，Project Explorer视图显示为PHP Perspective的一部分。要手动打开视图，请转到Window |显示视图|其他| PHP工具| Project Explorer。
