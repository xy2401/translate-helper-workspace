菜单

菜单

可用的CSS属性

可以为此元素定义的属性：

  • 动画

    支持动画类型“slideIn”和“slideOut”。其他人被忽略了。

  • 背景颜色

    背景颜色

  • 背景图片

    背景图像或渐变

  • 边界

    围绕菜单绘制的边框。

  • 下边框

    菜单的底部边框。

  • 左边框

    菜单的左侧边框。

  • 边界半径

    边界半径。如果使用增强边框（如“inset”或“outset”），则边框半径无效。

  • 右边框

    菜单的右侧边框。

  • 边框顶部

    菜单的顶部边框。

  • 箱阴影

    菜单背后的阴影。

  • 颜色

    菜单文字颜色

  • 字形

    菜单字体

  • 不透明度

    菜单的不透明度。介于0和1之间的值。

  • 填充

    边框和文本/图像之间的空间

适用的SWT样式

没有

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：禁用

    适用于禁用的菜单。

菜单项

菜单项

可用的CSS属性

可以为此元素定义的属性：

  • 背景颜色

    背景颜色

  • 背景图片

    背景图片

  • 颜色

    菜单项文本颜色

  • 不透明度

    项目的不透明度。介于0和1之间的值。

  • 填充

    菜单项的填充（内部距离）。

  • 文字阴影

    文字的阴影。

适用的SWT样式

可以在此元素的选择器中引用的SWT样式标志（作为CSS属性）：

  • [级联]

    级联菜单项行为的样式常量。

  • [校验]

    复选框菜单项行为的样式常量。

  • [推]

    推送菜单项行为的样式常量。

  • [无线电]

    单选按钮菜单项行为的样式常量。

  • [分隔器]

    分隔符菜单项行为的样式常量。

适用国家

窗口小部件可以在选择器中引用该元素（作为CSS伪类）：

  • ：禁用

    适用于禁用的菜单项。

  • ：徘徊

    鼠标悬停

  • ：onMenuBar

    适用于菜单栏中的菜单项。

  • ：按下

    在鼠标按下并打开子菜单。仅适用于菜单栏。

菜单项，CheckIcon

复选框的图标。当MenuItem具有SWT.CHECK样式标志时适用。

可用的CSS属性

可以为此元素定义的属性：

  • 背景图片

    图标图像。图像的最大尺寸应为16 x 16像素。
