CLabel

RWT CLabel widgets.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color for clabels.

  • background-image

    Background image or gradient for clabels.

  • background-position

    Background image position for clabels.

  • background-repeat

    Background image repetition for clabels.

  • border

    The border that is drawn around clabels.

  • border-bottom

    The bottom side border for clabels.

  • border-left

    The left side border for clabels.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for clabels.

  • border-top

    The top side border for clabels.

  • color

    Text color for clabels.

  • cursor

    Control's cursor.

  • font

    Text font for clabels.

  • opacity

    The opacity of the CLabel. A value between 0 and 1.

  • padding

    Padding for clabels.

  • spacing

    Space between text and image.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Indicates that the widget should have a border.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled clabels.

  • :hover

    Applies to widgets while the cursor hovers over them.
