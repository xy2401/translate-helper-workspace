嵌入嵌入式

概观

可嵌入类可以固定到实体，也可以固定到另一个嵌入类。

您可以嵌入可嵌入类的单个实例，也可以嵌入可嵌入类的集合。

嵌入单个元素

  1. 在Palette的Composition列表中，选择Embed Single Element 。

  2. 单击源实体或Has-A关系的可嵌入或按ESCAPE取消。

  3. 单击Has-A关系的目标嵌入或按ESCAPE取消。

    Eclipse IDE创建并可视化Has-A关系，以及实体中设置了embeddable的相关嵌入属性。

    该属性将使用@Embedded批注进行映射。

嵌入元素集合

  1. 在Palette的Composition列表中，选择Embed Collection of Elements 。

  2. 单击源实体或Has-A关系的可嵌入或按ESCAPE取消。

  3. 单击Has-A关系的目标嵌入或按ESCAPE取消。

    Eclipse IDE创建并可视化Has-A关系，以及实体中设置了embeddable的相关嵌入属性。

    该属性将与@ElementCollection注释一起映射。

删除Has-A关系

要删除特定的Has-A关系，请在JPA图上选择关系并按DELETE ，或从上下文菜单中选择“ 删除 ”。

通过JPA图删除关系也会将其从应用程序数据模型中删除，并删除嵌入实体的嵌入属性。
