Formatter首选项

Formatter Preferences页面允许您选择高级PHP格式化程序配置文件。

从Window |访问Formatter Preferences Preferences页面偏好| PHP | Formatter首选项 。

仅将Formatter Preferences设置应用于特定项目：

  1. 选择标记为“配置项目特定设置”的链接。
  2. 从列表中选择所需的项目。将出现Formatter Preferences Properties对话框。
  3. 选择所需设置，然后单击“应用”。将出现一个提示对话框，指出必须重建项目才能使设置生效。
  4. 单击是以重建项目。 - 或 - 单击否以仅在重新启动PDT时执行重建。 - 或 - 单击“取消”取消操作。

要更改格式化程序选项，您可以选择其中一个内置配置文件或创建新的配置文件。要创建新的配置文件，请选择新建... 然后，您可以通过选择编辑...来编辑新的配置文件。

注意：

内置配置文件只能预览，使用Show ...查看它们。

每个配置文件都可以导出到XML视图导出....稍后可以通过Import ...对话框导入到不同的工作区。
