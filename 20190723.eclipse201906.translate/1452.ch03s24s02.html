3.24.2。�在工作区中配置任务存储库

您要在ITE中使用的每个存储库都必须在您正在使用的工作区中进行配置。

  1. 选择：

    窗口 - > 显示视图 - > 其他

    从菜单中。

  2. 在Mylyn部分中，选择Task Repositories并单击“ OK ”。将出现“ 任务存储库视图”。Jubula的Bugzilla存储库已预先配置。

  3. 在Task Repositories视图中，右键单击并从上下文菜单中选择Add Task Repository 。

  4. 在出现的对话框中，您将看到ITE的预定义任务存储库。您可以选择其中一个或选择安装其他连接器。根据您要使用的连接器，您可能需要使用Tasktop的其他软件，否则连接器可能会产生许可费用。如果要连接到HP ALM存储库，请按照以下说明进行操作（ Section�3.24.2.1，“将HP ALM存储库添加到工作区” ）。

  5. 选择连接器后，单击“ 下一步 ”。

  6. 在下一页中，您将需要配置任务存储库。有关存储库配置的信息，请参阅Mylyn文档。

  7. 配置存储库后，单击“ 完成 ”。

  8. 要能够查看此存储库中的任务，请选择：

    窗口 - > 显示视图 - > 其他

    从菜单中。

  9. 在Mylyn部分中，选择Task List并单击“ OK ”。将出现任务列表视图。

您现在可以查看此存储库中的项目，在ITE中打开它们，为您的工作区添加查询以及从此存储库处理任务。您还可以在Project属性中选择此存储库作为项目的存储库（ Section�3.24.5.1，“为项目配置任务存储库” ）。

3.24.2.1。�将HP ALM存储库添加到工作区

  1. 要为HP ALM存储库添加连接器，您必须从Tasktop安装和许可软件。

  2. 获得Tasktop帐户和HP ALM许可证后，即可安装并使用HP ALM连接器。

  3. 在www.tasktop.com网站上，选择下载并找到Tasktop Dev产品。

  4. 选择下载插件的选项。

  5. 按照Tasktop网站上的说明将该插件安装到您的ITE中。您将需要Tasktop Dev Enterprise作为HP ALM连接器。

  6. 当系统提示您自动重启ITE时，请说“ 否” ，然后通过菜单手动重启ITE。

  7. 重新启动ITE时，系统将提示您输入Tasktop的帐户详细信息。

  8. 输入信息后，您将能够通过“ 任务存储库”视图配置存储库 。
