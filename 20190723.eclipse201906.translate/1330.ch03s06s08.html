3.6.8. Saving a Project as a new Project

  1. To save the currently opened Project as a new Project, select:

    Test-->Save as....

  2. Enter the new name for the Project in the dialog that appears.

  3. You can also define the version numbers for the Project that should be created or use the suggested version number. See the section on Project versions for more information on version numbers (Section 3.6.11, “Versioning Projects”).

  4. This creates a new Project in the database, saved under the name you entered.

  5. If no changes have been made since the previous Project save, the new Project has the same content as the previous one.

  6. Otherwise, any modifications are saved under the new Project name, and the old Project is kept in the state of its last save.

  7. The new Project becomes active in the ITE.

    Any test result summaries for the Project are not duplicated in the new Project. Tests that ran for previous versions of the Project are, however, still in the database to be used for long term analysis.
