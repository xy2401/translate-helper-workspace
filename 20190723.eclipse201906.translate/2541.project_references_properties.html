项目参考属性

“项目引用属性”页面允许您设置项目引用的项目。这会影响诸如打开和关闭项目之类的操作。 （即，如果您关闭项目并重新打开它，系统将提示您同时打开它引用的项目。）

要访问“项目引用属性”页面，请在“项目资源管理器”视图中右键单击PHP项目，然后选择“属性”|资源 - 或 - 选择项目，然后从菜单栏转到项目|属性|项目参考属性。

要设置项目引用的项目：

  1. 标记项目中引用的项目的复选框。
  2. 单击确定。

现在，所选项目被指定为项目引用。
