JSRuntimeInstallProvider

标识符：
org.eclipse.wst.jsdt.core。JSRuntimeInstallProvider

以来：
2.0

描述：
为JavaScript运行时安装提供动态贡献者。此提供程序提供的所有IJSBaseRuntimeInstall都具有相同的类型（通过其runtimeTypeId属性定义）。为了使这个贡献有效，runtimeTypeId必须匹配通过org.eclipse.wst.jsdt.core注册的一个有效类型id。JSRuntimeType扩展点。

配置标记：

<！ELEMENT 扩展 （ runtimeProvider ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT runtimeProvider EMPTY>

<！ATTLIST runtimeProvider

runtimeTypeId CDATA #REQUIRED

class CDATA #IMPLIED>

  • runtimeTypeId - 此属性必须与通过org.eclipse.wst.jsdt.core提供的有效运行时类型ID匹配。JSRuntimeType扩展点。
