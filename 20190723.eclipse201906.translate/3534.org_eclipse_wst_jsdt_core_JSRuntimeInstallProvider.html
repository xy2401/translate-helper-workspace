JSRuntimeInstallProvider

Identifier:
org.eclipse.wst.jsdt.core.JSRuntimeInstallProvider

Since:
2.0

Description:
Provides a dynamic contributor for JavaScript runtime installs. All the IJSBaseRuntimeInstall delivered by this provider will have the same type (defined through its runtimeTypeId attribute). For this contributions to be valid, the runtimeTypeId MUST match one valid type id registered through org.eclipse.wst.jsdt.core.JSRuntimeType extension point.
