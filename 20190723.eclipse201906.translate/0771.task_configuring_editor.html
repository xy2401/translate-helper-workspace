配置JPA图编辑器

配置JPA图编辑器的全局默认设置

  1. 从主菜单中，选择“ 窗口” → “首选项” 。

  2. 在属性树中，选择常规 → 编辑器 → JPA图编辑器 。

  3. 指定JPA图编辑器设置。要还原全局预定义值，请选择“还原默认值” 。

  4. 选择Apply ，然后单击OK 。

为每个项目配置JPA图编辑器

  1. 从Project Explorer的项目上下文菜单中，选择Properties 。

  2. 选择JPA → JPA Diagram Editor 。

  3. 指定JPA图编辑器设置。要恢复全局默认设置，请选择“恢复默认值” 。

  4. 选择Apply ，然后单击OK 。
