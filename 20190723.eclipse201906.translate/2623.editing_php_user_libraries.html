编辑PHP用户库

此过程描述如何编辑用户库。如果要更改用户库的名称，则需要编辑用户库。更改库的名称允许您在库之间创建描述性差异。要编辑库，必须有可用的库。有关更多信息，请参阅导入用户库或添加PHP库 。

要编辑用户库：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，选择要编辑的用户库，然后单击Edit ...将打开“编辑用户库”对话框。
  3. 在“用户库名称：”文本字段中输入用户库名称。
  4. 要将给定的用户库添加到您的环境，您必须选择“添加到环境”框。否则，您的库将在项目级别添加。
  5. 要应用更改，请单击确定 。

您现在已经编辑了用户库。要编辑库中的文件夹，请参阅编辑库组件 。
