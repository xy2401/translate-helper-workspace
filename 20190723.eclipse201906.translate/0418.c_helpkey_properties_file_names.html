helpKey属性文件名

helpKey属性文件的名称是任意的（只要它们以.properties结尾）。但是，上下文相关的帮助插件可以为多个UI插件提供成对的helpKey属性文件，如果这些文件位于同一目录中，则需要具有唯一的名称。

作为惯例，helpKey属性文件名应与它们所服务的UI插件相对应。例如：
  • org.eclipse.datatools.connectivity.ui.contextIds.properties
  • org.eclipse.datatools.connectivity.ui.searchExpressions.properties

为了允许helpKey属性文件的任意名称，并从UI组件中抽象出helpKey属性文件的名称，DTP帮助助手提供了一个扩展点（org.eclipse.datatools.help.helpKeyProperties）。该扩展点允许任何插件声明UI插件的helpKey属性文件贡献。
