导出PHP用户库

此过程描述如何导出用户库，使其可以访问存储它的存储库的任何人。导出用户库只会以.xml格式导出库的描述，并且不会包含任何库的内容。

要导出用户库：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，选择要导出的用户库，然后单击Export ...将打开“导出用户库”对话框。
  3. 单击旁边的复选框，选择要导出的库。您也可以使用“全选”或“ 取消全选”按钮。
  4. 要选择要将库导出到的位置，请使用URL填写“文件位置：”文本字段，或单击“ 浏览...”并选择位置。
  5. 要应用更改，请单击确定 。

您的用户库现已导出到您指定的位置。

您现在可以从任何有权访问存储位置的位置导入导出的用户库。有关更多信息，请参阅导入PHP用户库 。
