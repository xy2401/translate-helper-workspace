4.45. Margin Left

Parameter Name Data Type Possible Values Default
MARGIN_LEFT Integer   0
Table 4.46. MARGIN_LEFT

  • Use this parameter to enter the amount of pixels you wish to include in the screenshot to the left of the dialog

  • You can enter positive and negative values.

  • Enter 0 to neither extend nor reduce the margin.
