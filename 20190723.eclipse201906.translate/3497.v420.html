  • SWT GridLayout的新“自由单元格模式”编辑策略

  • 支持非可视bean

  • 增强的SWT支持

  • 增强的JFace和RCP支持

  • 增强的Swing支持

  • 代码生成和解析增强功能

  • 编辑和财产窗格增强功能
     

  • V4.1.0

  • V4.0.0

  • V3.0.0

支持SWT可视化继承

  • 从任何Composite或Shell子类继承
  • 所有继承的小部件都是可见的
  • 访问通过公共访问者或受保护访问者公开的继承小部件
  • 继承的小部件显示在带有“i”装饰器的树中

SWT GridLayout的新“自由单元格模式”编辑策略

  • 在任何开放的单元格中自由定位小部件
  • 在现有单元格之间插入新的行和列
  • 通过拖放重新排列行和列
  • 双击或右键单击标题以设置/清除水平/垂直抓取
  • 交互式列和行跨越反馈
  • 右键单击标题区域以访问弹出标题菜单
  • 右键单击以删除行和列
  • 填充标签自动管理
  • 文本小部件的选项可自动水平抓取
  • 选项可自动将标签小部件右对齐到文本小部件的左侧

支持非可视bean

  • 非可视bean放置在“（非可视bean）”类别中
  • 在源代码中标记为“// @wb：location = X，Y”标记
  • 用于将非可视bean创建为字段的选项
  • 通过拖放重新排序非可视bean

增强的SWT支持

  • 增强了对自定义SWT小部件的支持
  • 选项生成Eclipse 3.0样式的GridData
  • 容器上下文菜单上的新“设置选项卡顺序”命令
  • 在null和FormLayout中自动自动调整窗口小部件
  • FormLayout的新“自动调整小部件”命令
  • 重新支持Forms ExpandableComposite和Section
  • FillLayout和GridLayout的增强布局助手
  • 支持SWT中的受保护属性
  • 支持SWT Point属性
  • 支持3.2 CoolBar垂直风格
  • 支持3.2 sortColumn / sortDirection属性

增强的JFace和RCP支持

  • 在弹出窗口中显示字段编辑器的标签和主要控件的事件
  • 支持视图/编辑器的名称/图标属性
  • 支持视图类别编辑和创建
  • 在设计画布上支持工具栏/酷吧管理器
  • 为新ViewPart创建ID字段
  • ViewPart向导在plugin.xml中添加了视图声明
  • EditorPart向导在plugin.xml中添加了编辑器声明
  • 需要时创建视图/编辑器扩展
  • 自动适应非形式控制
  • 使用弹出菜单添加/删除paintBorders / adapt
  • Forms API的“适应控件”弹出项目
  • 支持ExpandableComposite / Section的clientVerticalSpacing / marginWidth / marginHeight属性

增强的Swing支持

  • 在GridBagLayout中显示insets / padding的值反馈
  • 支持Swing的焦点遍历策略
  • 容器上下文菜单上的新“设置选项卡顺序”命令
  • 在null和SpringLayout中自动自动调整小部件
  • Swing容器的新“Flatten Hierarchy”操作
  • 在绝对布局中移动时将大小更改为首选的选项
  • SpringLayout的新“自动调整小部件”命令
  • 当bean描述符中的“isContainer == FALSE”时，不显示子组件
  • 支持自定义布局管理器
  • 支持JBuilder XYLayout和XYConstraints
  • 支持来自字段的Swing组件
  • 在设置焦点策略时将容器标记为焦点循环根

代码生成和解析增强功能

  • 用于指定默认代码生成方法的选项
  • 使用try {} catch {}包围方法调用代码的选项
  • 在字段之前支持“this”前缀
  • 禁用复杂表达式评估的选项
  • 自定义组件更改后启用/禁用重新分析的选项
  • 支持事件处理程序内部类的位置选项
  • 在组件删除期间删除“真实”事件处理程序的选项
  • 根据需要提供JDK 1.5版本的ResourceManager
  • 在解析具有EventObject的单个参数子类的方法时忽略
  • 支持静态工厂的组件
  • 支持静态工厂的边界
  • 理解构造函数中具有两个以上参数的控件的SWT代码
  • 支持使用复杂构造函数添加新控件
  • 支持SWT控件的构造函数参数
  • 支持枚举属性

编辑和财产窗格增强功能

  • 激活时自动下拉组合属性编辑器的选项
  • “Expose property ...”项添加到属性表弹出菜单中
  • 自动指导编辑新窗口小部件的选项
  • 支持使用多选方式编辑布局属性
  • 接收焦点后刷新自定义组件面板
  • 支持首选属性描述符
  • 记住图像选择对话框中的扩展状态
  • 复杂“构造函数”属性中的组构造函数属性
