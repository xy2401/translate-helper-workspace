标识符：
org.eclipse.debug.core.watchExpressionDelegates

以来：
3.0

描述：
此扩展提供了一种机制，用于为委托者在每个调试模型的基础上评估监视表达式。监视表达式委托对监视表达式执行评估并异步报告结果。

配置标记：

<！ELEMENT 扩展 （ watchExpressionDelegate *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT watchExpressionDelegate EMPTY>

<！ATTLIST watchExpressionDelegate

debugModel CDATA #REQUIRED

delegateClass CDATA #REQUIRED>

  • debugModel - 指定此委托为其提供评估的调试模型的ID
  • delegateClass - 指定实现org.eclipse.debug.core.model.IWatchExpressionDelegate ，用于计算表达式的值。

例子：

The following is the definition of a watch expression delegate for the com.example.foo plug-in:
 <extension point="org.eclipse.debug.core.watchExpressionDelegates">
   <watchExpressionDelegate
     debugModel="org.eclipse.jdt.debug"
     delegateClass="org.eclipse.jdt.internal.debug.ui.JavaWatchExpressionDelegate"/>
  </extension>

API信息：
delegateClass属性的值必须是实现接口org.eclipse.debug.core.model的Java类的完全限定名称。IWatchExpressionDelegate 。
