工作集配置

标识符：
org.eclipse.cdt.ui.workingSetConfigurations

以来：
6

描述：

此扩展点是CDT插件系列的内部。

注册项目配置工厂的扩展点，用于在工作集配置中处理自定义项目性质的扩展配置选项。这些工厂负责为“管理工作集配置”对话框创建项目配置元素和UI控制器。

配置标记：

<！ELEMENT 扩展 （ projectConfigurationFactory ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT projectConfigurationFactory （ projectNature ）+>

<！ATTLIST projectConfigurationFactory

id CDATA #REQUIRED

class CDATA #REQUIRED>

在工作集配置中注册工厂的项目配置元素。工厂通过嵌套的<nature>元素注册到项目性质。


  • id - 项目配置工厂的唯一标识符。
  • class - 工厂类的完全限定名称。它必须实现IWorkingSetProjectConfigurationFactory接口。

<！ELEMENT projectNature EMPTY>

<！ATTLIST projectNature

id IDREF #REQUIRED>

对项目性质的引用。


  • id - 引用的项目性质ID。

例子：
<extension point = “org.eclipse.cdt.ui.workingSetConfigurations” > <projectConfigurationFactory id = “org.example.myProject” class = “org.example。MyProjectConfigurationFactory“ > <nature id = ”org.example.myProjectNature“ /> </ projectConfigurationFactory> </ extension>

API信息：
注册的项目配置工厂需要实现org.eclipse.cdt.internal.ui.workingsets。IWorkingSetProjectConfigurationFactory接口。
