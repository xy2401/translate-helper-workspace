创建PHP项目

PHP项目是应该在其中创建所有PHP和其他应用程序文件的容器。

要创建一个新的PHP项目：

  1. 从菜单栏中，转到文件|新的| PHP项目 - 或 - 在Project Explorer视图中，右键单击并选择New | PHP项目 。将显示“新建PHP项目”向导。
  2. 输入以下信息：
    • 项目名称 - 所需的项目名称
    • 内容 - 选择是否：
    • 在工作区中创建一个新项目 - 在工作区目录中创建一个新的PHP项目。
    • 从现有源创建项目 - 创建指向位于工作区外部的文件的PHP项目。单击“ 浏览”以选择所需的源内容。
    • PHP版本 - 选择是否：
    • 使用默认的PHP设置 - 使用默认的PHP解释器设置。
    • 使用项目特定设置 - 选择要用于项目的PHP版本以及ASP标记是否用作PHP标记。有关更多信息，请参阅PHP版本支持
    • 项目布局 - 选择是否：
    • 使用项目作为源文件夹 - 默认情况下，项目中的所有资源都将添加到构建路径中。
    • 为源文件和公共资源创建单独的文件夹 - 将创建单独的文件夹，您可以在其中放置应在构建路径中包含或排除的资源。有关更多信息，请参阅配置项目的PHP构建路径 。可以从“ 新建项目布局首选项”页面配置此选项的默认设置。
    • JavaScript支持 - 标记“为此项目启用JavaScript支持”复选框，以便为项目提供JavaScript功能（例如JavaScript内容辅助选项）。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。
  3. 单击下一步以配置项目的包含路径 （这也可以在项目创建后完成）。
  4. 单击下一步以配置项目的PHP构建路径 （这也可以在项目创建后完成）。
  5. 单击完成 。

新的PHP项目将在您的工作区中创建，并显示在Project Explorer View中 。

您现在可以通过创建PHP文件或向项目添加其他资源来开始开发应用程序。
