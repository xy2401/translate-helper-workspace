设置差异，交集和联盟

除E2之外的表达式E1选择E1中的所有节点，除非它们也在E2中。两个表达式都必须返回节点序列。结果按文档顺序返回。例如，@ *除了@note返回除note属性之外的所有属性。E1与E2相交的表达式选择E1和E2中的所有节点。两个表达式都必须返回节点序列。结果按文档顺序返回。表达式E1 union E2选择E1或E2或两者中的所有节点。两个表达式都必须返回节点序列。结果按文档顺序返回。上述表达式的完整示例如下。考虑一个如下所示的XML文档：

<source lang =“xml”>
<节点>
<A>
<connecteda> A </ connecteda>
<connecteda>乙</ connecteda>
<connecteda> C </ connecteda>
</A>
<B>
<connectedb>乙</ connectedb>
<connectedb> C </ connectedb>
<connectedb> d </ connectedb>
</ B>
</节点>

那么每个表达式的一个例子是：

数据（/ a / *）联合数据（/ b / *）

'结果：'

  1. xs：string：A

  2. xs：string：B

  3. xs：string：C

  4. xs：string：D

数据（/ a / *）与数据相交（/ b / *）

结果：

  1. xs：string：B

  2. xs：string：C

数据（/ a / *），数据除外（/ b / *）

结果：

  1. xs：string：D
