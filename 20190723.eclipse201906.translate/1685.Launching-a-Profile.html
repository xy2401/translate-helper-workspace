启动个人资料

Perf插件在用户友好的界面中提供有用的配置文件信息，从Perf命令行工具收集所需信息并将其挂钩到Eclipse和CDT的工具中。

在配置文件运行期间，Perf插件将根据需要在后台启动，停止和执行其他Perf任务，而正在配置的二进制文件正常运行。

一键启动

Eclipse的Perf插件包括一键启动功能，该功能使用默认选项对二进制文件进行概要分析。要使用一键启动，请右键单击项目，二进制文件或在打开的编辑器中单击属于该项目的源文件。这里有两种方式来启动发布：

导航到性能分析工具并单击“使用Perf配置文件”

或者导航到分析工具，单击Profile Timing并将计时工具设置为Perf

要正确设置首选项，您可以转到C / C ++ - > Profiling-> Categories-> Timing并将默认计时工具设置为Perf。

您还可以使用项目Properties-> C / C ++ General-> Profiling Categories-> Timing来设置项目特定首选项，以覆盖工作区首选项。

您还可以使用Profile as ...-> Local C / C ++ Application对您的应用程序进行概要分析，从而将Profiler选项卡中的概要分析工具设置为perf。

启动自定义配置文件

Perf插件允许您使用多个选项配置配置文件运行。这些选项通过启动配置以用户友好的方式公开，允许更复杂的配置文件。

要自定义配置文件，请右键单击项目，二进制文件或在打开的编辑器中单击属于该项目的源文件。然后，导航到“性能分析工具”>“配置文件配置...”以打开“配置文件配置”菜单。有关配置配置文件的详细信息，请参阅“配置文件配置”部分。

配置配置文件后，单击配置文件按钮以启动配置文件。
