悬停支持

将鼠标悬停在元素上将导致工具提示显示有关该元素的信息，包含其声明的位置以及元素相关PHPdoc注释中包含的任何其他信息（描述，参数等）：

悬停信息还包括其他信息，例如方法，类型和其他元素的覆盖和返回：

在鼠标悬停在元素上时按住Ctrl键也会显示该元素中包含的所有内容：

悬停时，按F2键使悬停工具提示成为焦点。这样可以确保即使不悬停也可以显示，并可以从中选择文本。

在悬停时单击元素将转到该元素的声明。有关更多信息，请参阅使用Smart Goto Source 。

配置悬停首选项

要配置悬停设置，请转到“ 悬停首选项”页面 ，可通过转到“ 窗口”|来访问该页面 偏好| PHP |编辑|盘旋 。

注意：

如果为项目启用了JavaScript支持，则还将为JavaScript元素启用悬停。有关详细信息，请参阅在PHP项目中启用JavaScript支持 。可以从JavaScript Hovers首选项页面配置JavaScript悬停设置，可通过转到Window |来访问偏好|网络| JavaScript |编辑|盘旋 。
