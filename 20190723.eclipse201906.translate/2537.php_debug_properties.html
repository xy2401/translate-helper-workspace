PHP调试属性

PHP Debug Properties页面允许您配置在项目中调试文件时使用的默认设置。

要访问PHP Debug Properties页面，请在Project Explorer视图中右键单击PHP项目，然后选择Properties |资源 - 或 - 选择项目，然后从菜单栏转到项目|属性| PHP调试属性。

要为项目配置PHP调试属性：

  1. 选中“启用项目特定设置”复选框。
  2. 根据您的喜好配置设置。有关可用设置的详细信息，请参阅调试首选项 。
  3. 单击“确定”以应用设置。

所有项目的默认PHP调试属性都可以在“调试首选项”页面中设置（通过单击“属性”页面上的“配置工作区设置”链接转到“窗口”|“首选项”|“PHP”|“调试”或“ - ”）。
