快速链接配置

标识符：
org.eclipse.ui.intro.quicklinks

以来：
快速链接首次出现在Eclipse Platform 4.6中，这是Neon版本系列的一部分。

描述：
此扩展点用于配置Quicklinks Welcome / Intro查看器组件。快速链接是在欢迎/简介中向用户显示的有用操作。Quicklinks使用Eclipse Core Commands框架（请参阅org.eclipse.ui.commands扩展点），但支持覆盖要用于项目的图标和文本。Quicklinks是实验性组件，尚未被视为API。

配置标记：

<！ELEMENT 扩展 （ 命令 | 覆盖 | url ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 命令 EMPTY>

<！ATTLIST命令

id IDREF #REQUIRED

标签CDATA #IMPLIED

描述CDATA #IMPLIED

icon CDATA #IMPLIED

重要性（高|中|低）

分辨率（待机|启动栏|关闭|完整）>

定义Quicklink组件。必须指定“命令”或“url”之一。


  • id - 要调用的命令标识符。该命令也可以是用于编码命令参数的序列化命令（有关详细信息，请参阅ParameterizedCommand＃serialize（））。
  • label - 要显示的标签。如果未指定，则使用命令名称。
  • description - 快速链接的描述。如果未指定，请使用命令的说明。
  • icon - 用于快速链接的图标。如果未指定，则使用命令的关联图像。
  • 重要性 - 定义此项目的重要性级别。如果未指定，则默认为“低”。
  • 解决方案 - 执行命令后欢迎视图应该发生什么。如果“已满”或未指定，欢迎/简介部分将进入完全可视模式。如果“待机”（默认），欢迎/简介部分将进入待机模式。如果是“启动栏”，欢迎/简介将在状态区域中进入“启动栏”模式。如果“关闭”，则欢迎/简介将关闭。

<！ELEMENT url EMPTY>

<！ATTLIST网址

位置CDATA #REQUIRED

标签CDATA #REQUIRED

说明CDATA #REQUIRED

icon CDATA #IMPLIED

重要性（高|中|低）>

定义URL快速链接组件。URL快速链接必须提供标签，建议您提供图标和说明


  • location - 要调用的URL。URL可以是简介URL或外部URL。
  • label - 要显示的标签。如果未指定，则使用命令名称。
  • description - 快速链接的描述。如果未指定，请使用命令的说明。
  • icon - 用于快速链接的图标。如果未指定，则使用命令的关联图像。
  • 重要性 - 定义此项目的重要性级别。如果未指定，则默认为“低”。

<！ELEMENT 覆盖 EMPTY>

<！ATTLIST覆盖

命令IDREF #REQUIRED

主题IDREF #REQUIRED

icon CDATA #REQUIRED>

为主题提供机制以覆盖用于表示快速链接中的命令的图标。


  • command - <quicklink>元素中引用的命令标识符。该命令也可以是用于编码命令参数的序列化命令（有关详细信息，请参阅ParameterizedCommand＃serialize（））。该命令可以包括简单的'*'通配符以匹配任何子字符串。例如， org.eclipse.ui.newWizard*将匹配任何“新建”向导定义。
  • theme - 主题标识符。可以是逗号分隔的值集。
  • icon - 用于表示此命令的图标。

例子：
以下是配置Quicklink的示例：

   <extension
         point="org.eclipse.ui.intro.quicklinks">
      <!-- Reuses command definition label and icons -->
      <quicklink
            command="org.eclipse.epp.mpc.ui.command.showMarketplaceWizard"
            icon="icons/mpcdrag.png">
      </quicklink>
      
      <!-- Invoke a particular new wizard; place the Welcome/Intro in Standby mode on success -->
      <quicklink
            command="org.eclipse.ui.newWizard(newWizardId=org.eclipse.jdt.ui.wizards.JavaProjectWizard)"
            description="Create a new Java Eclipse project"
            label="Create a new Java project"
            standby="true">
      </quicklink>
      
      <!-- Allow overriding theme and icons for all newWizard-related commands -->
      <override
            command="org.eclipse.ui.newWizard*"
            icon="icons/solstice/new-project.png"
            theme="org.eclipse.ui.intro.universal.solstice">
      </override>
   </extension>

使用introContent.xml定义中的Quicklinks Viewer Component显示快速链接 。 


 <extensionContent id="quicklinks" name="Quick Actions" path="root/extra-group2/anchor">
        <contentProvider 
         pluginId="org.eclipse.ui.intro.quicklinks"
         class="org.eclipse.ui.intro.quicklinks.QuicklinksViewer"
         id="quick-links">
        </contentProvider>
    </extensionContent>
