PHP包含路径属性

PHP Include Path Properties页面允许您配置项目的Include Path。要访问PHP Include Path Properties页面，请在Project Explorer视图中右键单击PHP项目，然后选择Properties | PHP Include Path - 或者 - 选择项目，然后从菜单栏转到Project |属性| PHP包含路径属性。

有关更多信息，请参阅配置项目的PHP包含路径 。
