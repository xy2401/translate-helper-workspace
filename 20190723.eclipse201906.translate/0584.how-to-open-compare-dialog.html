如何打开比较对话框

这仅在发布2.1M4后有效。

在此页面中，您将学习如何打开显示比较结果的对话框。

准备输入

首先要做的是选择CompareEditorInput类的EMF Compare子实现。

提供了两种实现：

  • ComparisonEditorInput ，当您想要显示预先计算的比较（EMFCompare的结果）时应该使用。
  • ComparisonScopeEditorInput ，当您想要打开比较编辑器或对话框并让它执行比较时应该使用它。

两者都可以从包org.eclipse.emf.compare.ide.ui.internal.editor中的org.eclipse.emf.compare.ide.ui插件获得。这仍然是临时API，因此我们可能随时打破它。

准备配置

实例化EMF比较CompareEditorInput的特定实现时，必须至少给它三个参数：

  • EMFCompareConfiguration 。这是标准CompareConfiguration的精确配置，因此您只需将其实例化为：
EMFCompareConfiguration配置=新的EMFCompareConfiguration（new CompareConfiguration（））;
  • EMFCompareEditingDomain。它不是EMF中EditingDomain的实现。它共享类似的概念（它有一个命令堆栈，它可以创建一些命令），但更简单。您可以通过工厂方法创建它：
//祖先可能为null ICompareEditingDomain editingDomain = EMFCompareEditingDomain.create（left，right，ancestor）;

如果您需要了解已执行的合并命令，您甚至可以提供自己的命令堆栈。

  • 框架使用的AdapterFactory ，可以很好地显示EObjects。大多数情况下，具有全局注册表的ComposedAdapterFactory就足够了，如下例所示：
AdapterFactory adapterFactory = new ComposedAdapterFactory（ComposedAdapterFactory。描述。注册。实例）;

根据CompareEditorInput的选择子实现，您可能需要提供其他参数。

ComparisonEditorInput

您必须提供比较对象，即EMFCompare的比较计算结果。

EMFCompare comparator = EMFCompare.builder（）。build（）;比较比较= comparator.compare（EMFCompare.createDefaultScope（left，right，ancestor））;

ComparisonScopeEditorInput

您必须提供比较器（EMFCompare的实例）和比较范围。

EMFCompare comparator = EMFCompare.builder（）。build（）; IComparisonScope scope = EMFCompare.createDefaultScope（left，right，ancestor）;

打开比较UI

然后，您可以从Eclipse Compare框架中调用black magic方法。你有两个选择。您可以通过模态对话框或在编辑器中打开比较UI。只需调用以下两种方法之一：

  • CompareUI.openCompareEditorOnPage（CompareEditorInput，IWorkbenchPage） ，打开一个编辑器。
  • CompareUI.openCompareDialog（CompareEditorInput） ，打开模态对话框。

端到端的例子

与预先计算的比较

public void compare（Notifier left，Notifier right，Notifier ancestor）{EMFCompare comparator = EMFCompare.builder（）。build（）;比较比较= comparator.compare（EMFCompare.createDefaultScope（left，right，ancestor））; EMFCompareConfiguration配置=新的EMFCompareConfiguration（new CompareConfiguration（））; ICompareEditingDomain editingDomain = EMFCompareEditingDomain.create（left，right，ancestor）; AdapterFactory adapterFactory = new ComposedAdapterFactory（ComposedAdapterFactory。描述。注册。实例）; CompareEditorInput input = new ComparisonEditorInput（configuration，comparison，editingDomain，adapterFactory）; CompareUI.openCompareDialog（输入）; //或CompareUI.openCompareEditor（输入）; }

与比较范围

public void compare（Notifier left，Notifier right，Notifier ancestor）{EMFCompare comparator = EMFCompare.builder（）。build（）; IComparisonScope scope = EMFCompare.createDefaultScope（left，right，ancestor））; EMFCompareConfiguration配置=新的EMFCompareConfiguration（new CompareConfiguration（））; ICompareEditingDomain editingDomain = EMFCompareEditingDomain.create（left，right，ancestor）; AdapterFactory adapterFactory = new ComposedAdapterFactory（ComposedAdapterFactory。描述。注册。实例）; CompareEditorInput input = new ComparisonScopeEditorInput（configuration，editingDomain，adapterFactory，comparator，scope）; CompareUI.openCompareDialog（输入）; //或CompareUI.openCompareEditor（输入）; }
