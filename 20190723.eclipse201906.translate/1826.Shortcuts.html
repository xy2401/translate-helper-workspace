快捷键

任务列表视图

  • F2：重命名
  • F4：显示工具提示
  • F5：同步选中
  • Alt + Down：转到下一个未读
  • Alt + Up：转到上一个未读状态
  • Alt + Shift + C：标记完成（仅限本地任务）
  • Alt + Shift + I：标记不完整（仅限本地任务）
  • Alt + Shift + R：标记为已读
  • Alt + Shift + U：标记为未读
  • Esc：隐藏工具提示
  • 输入：打开任务
  • 插入：新的本地任务
  • Shift + Insert：新的子任务
  • Alt + N（仅限Mac）：新的本地任务
  • Alt + Shift + N（仅限Mac）：新的子任务
  • Ctrl + C：复制详细信息
  • Ctrl + F：查找
  • 将URL拖动到任务列表时：在Mozilla / Firefox中拖动，在Internet Explorer中Ctrl+drag

任务编辑器

  • Ctrl + F：在评论，说明，摘要和私人笔记中查找文字
  • Alt + Shift + C：标记完成（仅限本地任务）
  • Alt + Shift + I：标记不完整（仅限本地任务）
  • Alt + Shift + R：标记为已读
  • Alt + Shift + U：标记为未读

一般

  • Alt+click或Alt+RightArrow ：显示焦点视图中元素的所有子项，然后单击以选中。按住alt以继续钻取，单击视图中的空白以显示所有根元素。
  • Ctrl+Shift+Alt+RightArrow快速上下文视图
  • Ctrl+F9 ：激活任务对话框
  • Ctrl+Shift+F9 ：停用任务
  • Ctrl+F12 ：打开任务对话框
  • Ctrl+Shift+F12 ：打开存储库任务对话框

利息操纵

  • Ctrl+Shift+Alt+Up ：标记为地标
  • Ctrl+Shift+Alt+Down ：标记不太有趣

有用的Eclipse快捷方式

  • Alt+Shift+Q, K ：显示任务列表视图
  • Ctrl+F10 ：在编辑器中调用视图菜单或标尺菜单
