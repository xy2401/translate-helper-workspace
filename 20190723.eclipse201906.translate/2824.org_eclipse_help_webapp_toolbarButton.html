工具栏按钮

标识符：
org.eclipse.help.webapp.toolbarButton

以来：
3.5

描述：
允许将其他按钮添加到帮助Web应用程序的工具栏。

配置标记：

<！ELEMENT 扩展 （ 按钮 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 按钮 EMPTY>

<！ATTLIST按钮

class CDATA #REQUIRED>

  • class - 扩展AbstractButton的类，它包含返回图像URL，要执行的操作和按钮属性的方法。
