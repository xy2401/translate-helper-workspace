Table4.11。COMMAND

  • 使用此参数指定要执行的命令。

  • 根据本地参数中的定义，在本地计算机或远程计算机上提供命令的路径

  • 可以使用包含的空格简单地写入相对路径。如果使用路径片段（例如./或..），则需要在路径或命令周围使用引号（“）。/）相对路径。

  • 使用绝对路径时，请在包含空格的命令或参数周围使用引号（“）。

  • 您需要为路径片段使用引号，例如，而不是

    C：\ Program Files \ MyProgram \ example.exe -data C：\ Program Files \ MyProgram \ ws

    输入：

    “C：\ Program Files \ MyProgram \ example.exe”-data“C：\ Program Files \ MyProgram \ ws”

  • 在Linux中，可以在命令和参数周围放置引号。Windows cmd.exe只能接受参数或命令中的引号。

  • 请注意，不会检查引号内的字符串是否有效。
