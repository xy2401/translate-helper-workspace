PDOM AST处理器

标识符：
org.eclipse.cdt.core。PDOMASTProcessor

以来：
8.3

描述：
此扩展点允许扩展名对写入.pdom文件的信息做出贡献。在解析AST之后，所有贡献者都有机会在更新.pdom时处理AST的翻译单元。

配置标记：

<！ELEMENT 扩展 （ 处理器 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 处理器 （ 启用 ？）>

<！ATTLIST处理器

class CDATA #REQUIRED>

  • 课程 -

例子：
以下是Tagger贡献的示例：


<extension
    point="org.eclipse.cdt.core.PDOMASTProcessor"
    id="example"
    name="Example PDOM AST Processor">
    <bindingTagger
        local-id="my-processor"
        class="com.example.internal.ExamplePDOMASTProcessor">
        <enablement>
            <with variable="projectNatures">
                <iterate operator="or">
                    <equals value="com.example.my-nature"/>
                </iterate>
            </with>
        </enablement>
    </bindingTagger>
</extension>

API信息：
贡献类必须实现org.eclipse.cdt.core.index.IPDOMASTProcessor接口。
