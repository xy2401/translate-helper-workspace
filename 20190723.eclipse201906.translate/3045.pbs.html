PBS

本节介绍如何使用PBS作为PTP目标系统配置来设置运行配置

  • 有关先前PTP版本中PBS的遗留资源管理器的信息 ，请参阅PBS遗留资源管理器。当前版本不再需要资源管理器。

从PTP运行PBS程序时，首先创建一个C / C ++ / Fortran项目，然后构建它。有关更多信息，请参阅

  • 介绍创建MPI项目 ，其中包括有关构建的信息
  • 同步项目 - 在远程计算机上创建和构建项目

如运行并行程序中所述 ，接下来通过运行>运行配置创建运行配置...并创建一个新的并行应用程序 。

使用PTP运行PBS程序的细节在运行配置的“ 资源”选项卡上指定。对于Target System Configuration ，选择PBS-Generic-Batch

然后指定一个连接。如果您已经指定了连接，例如，对于同步项目或远程项目，它将被选中。如果要启动本地应用程序（在运行Eclipse的同一台计算机上），请为“ 连接类型”选择“ 本地 ”。

请参阅创建远程连接和本地连接

注意：要在创建连接后编辑连接信息，例如更改用户名或密码，请参阅编辑远程连接 。

在“ 基本/高级设置”选项卡中，根据您的系统填写PBS选项。

有关在启动PTP作业时使用Torque PBS的示例，请参阅PTP SC12教程幻灯片

 

现在继续运行配置设置以运行并行应用程序 ，方法是在“ 应用程序”选项卡上指定应用程序，以及所需的任何其他信息。
