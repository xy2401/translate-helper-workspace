GDB Trace Perspective

要打开透视图，请选择Window > Open Perspective > Other ... > GDB Trace 。

默认情况下，透视图包括以下视图：

  • 项目浏览器 ：此视图显示工作区中的项目，用于创建和管理跟踪项目。
  • 调试 ：此视图显示正在运行的C / C ++ Postmortem Debugger实例，并显示与跟踪点关联的线程和堆栈跟踪。
  • 跟踪控制 ：此视图显示调试器的状态，并允许导航跟踪记录。
  • 控制台 ：此视图显示C / C ++ Postmortem Debugger的控制台输出。

编辑器区域包含打开GDB跟踪时的事件和C / C ++编辑器。
