安装对话页面

标识符：
org.eclipse.ui.installationPages

以来：
3.5

描述：

Eclipse UI提供了一个AboutDialog，可以由客户端产品插件进行标记和重用。此对话框通常显示产品启动图像和有关产品的其他描述性信息。about对话框包括一个允许用户启动辅助对话框的按钮，即“ 安装详细信息”对话框。

此扩展点的目的是允许插件将页面添加到安装详细信息对话框中。打开安装详细信息对话框时，以这种方式提供的页面将添加到对话框中。这些页面用于提供有关产品配置的其他详细信息。

这是一个可扩展的对话框的实验扩展点。它尚未准备好进行生产，目前仅供下游客户评估。使用风险由您自己承担。

配置标记：

<！ELEMENT 扩展 （ 第 * 页 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 页面 EMPTY>

<！ATTLIST页面

id CDATA #REQUIRED

class CDATA #REQUIRED

名称CDATA #REQUIRED>

  • id -
  • class - org.eclipse.ui.about.InstallationPage 。
  • 名字 -

例子：
以下是一个示例安装页面：


   <extension point="org.eclipse.ui.installationPages"> 
       <page
            name="XYZ Info"
            class="org.eclipse.ui.internal.XYZInstallInfoPage"
            id="org.eclipse.ui.internal.xyz>
       </page>
    </extension>

API信息：
属性类的值必须表示类的完全限定名，该类是org.eclipse.ui.about.InstallationPage 。

提供的实施：
Workbench使用此扩展点在安装详细信息对话框中提供以下页面：
  • 插件：运行系统中所有插件的列表，包括每个插件的详细信息。
  • 配置：通过org.eclipse.ui.systemSummarySections扩展提供的配置信息。
