§1团队和角色

§1.1团队课程

→语法§A.1.1

使用修改器team声明的类是团队类 （或简称团队）。

1 
public team class MyTeamA {
2 
  ...
3 
}

团队意味着角色的容器，在以下段落中定义。

1 
public team class MyTeamA {
2 
  public class MyRole
3 
    ...
4 
  }
五 
}

团队为包含的角色类引入了一种新的继承变体（参见下面的§1.3.1 ）。团队的其他属性（在后面的部分中定义）包括：

  • 团队激活（ §5 ）
  • 抽象和实例化（ §2.5 ）
  • 宣布以团队方法取消（ §2.3.2 ）
  • org.objectteams.ITeam （ 第6.1节 ）

除了这些差异之外，团队类是具有方法和字段的常规Java类，其实例是常规Java对象。
