RWT Theming Reference > TabFolder

TabFolder

SWT TabFolder controls (don't mix up with CTabFolder).

Available CSS Properties

Properties that can be defined for this element:

  • border

    The border that is drawn around a TabFolder.

  • border-bottom

    The bottom side border for TabFolder.

  • border-left

    The left side border for TabFolder.

  • border-right

    The right side border for TabFolder.

  • border-top

    The top side border for TabFolder.

  • font

    Font for tab items text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Indicates that the TabFolder should have a border.

Applicable States

none

TabItem

Items of TabFolder controls.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color for tab folder headers.

  • background-image

    Background image or gradient for tab folder headers.

  • background-position

    Background image position for tab folder headers.

  • background-repeat

    Background image repetition for tab folder headers.

  • border

    The border that is drawn around a TabFolder.

  • border-bottom

    The bottom side border for tab folder headers.

  • border-left

    The left side border for tab folder headers.

  • border-right

    The right side border for tab folder headers.

  • border-top

    The top side border for tab folder headers.

  • color

    Tab item text color

  • margin

    Margin (outer distance) for tab items.

  • padding

    Padding (inner distance) for tab items.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [RIGHT_TO_LEFT]

    Indicates that the tab folder is oriented right to left.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :bottom

    Applies when TabFolder style flag SWT.BOTTOM is set.

  • :disabled

    Applies to tab items in a disabled tab folder.

  • :first

    Applies to the first tab item.

  • :hover

    On mouse over

  • :last

    Applies to the last tab item.

  • :selected

    On tab item selection

    The bottom side border for content container.

  • border-left

    The left side border for content container.

  • border-right

    The right side border for content container.

  • border-top

    The top side border for content container.
