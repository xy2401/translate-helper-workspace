文本搜索查询提供者

标识符：
org.eclipse.search.textSearchQueryProvider

以来：
3.2

描述：
扩展点允许注册用于生成文本搜索查询的文本搜索查询提供程序。

配置标记：

<！ELEMENT 扩展 （ textSearchQueryProvider +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT textSearchQueryProvider EMPTY>

<！ATTLIST textSearchQueryProvider

id CDATA #REQUIRED

标签CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 用于区分不同的提供者。
  • label - 以人类可读的术语描述查询提供程序。
  • class - 扩展org.eclipse.search.ui.text的类。TextSearchQueryProvider

例子：
作为示例，这是用于提供搜索引擎的标记


<extension id="MyProvider"
      point="org.eclipse.search.textSearchQueryProvider">
   <textSearchEngine
         id="myproduct.textSearchQueryProvider"
         label="Extended Text Search Query"
         class="myproduct.QueryProvider" />
</extension>

API信息：
贡献类必须实现org.eclipse.search.ui.text.TextSearchQueryProvider
