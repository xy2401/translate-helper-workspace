JSF Facets


Facet是一个Eclipse WebTools项目概念，它本质上意味着附加到Eclipse项目的功能。有关更多信息，请参阅WST WTP Facets文档。因此，JSF Facet提供与JSF相关的行为和功能。JSF Tooling项目提供的大多数功能都需要安装JSF Facet。为了使用JSF工具功能，项目必须具有以下先决条件方面：
  • 动态Web模块v2.3或更高版本
  • Java v1.4或更高版本

安装JSF facet时可以使用以下功能：
  • JSF库配置支持
  • 应用配置管理
  • 使用表达式语言（EL）的大多数JSF Core和JSF HTML标记库属性值的JSP源页面语义验证和内容辅助。
  • “运行在服务器上”支持JSF JSP页面

注意：当包含JSF Web模块的WAR文件作为项目导入时，该技术当前无法检测到这一点。有必要手动将JSF Facet添加到导入的项目中。
