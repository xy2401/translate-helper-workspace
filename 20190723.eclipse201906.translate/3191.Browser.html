Browser

The theming of RWT Browser.

Available CSS Properties

Properties that can be defined for this element:

  • border

    The border that is drawn around a Browser.

  • border-bottom

    The bottom side border for Browser.

  • border-left

    The left side border for Browser.

  • border-right

    The right side border for Browser.

  • border-top

    The top side border for Browser.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Style constant for bordered behavior.

Applicable States

none
