内容过滤器

标识符：
org.eclipse.help.webapp.contentFilter

以来：
3.4

描述：
此扩展点允许对帮助系统提供的文档进行额外的预处理，以在用户看到内容之前修改内容。这是通过定义一个输出过滤器来实现的，该过滤器将在读取页面时应用于输出流。可以定义多个过滤器。

配置标记：

<！ELEMENT 扩展 （ 过滤器 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 过滤器 EMPTY>

<！ATTLIST过滤器

class CDATA #REQUIRED

优先级CDATA“1”>

将应用于从帮助服务器提供的输出文件的过滤器，可用于修改发送到客户端的内容。


  • class - 处理输出流的过滤器类。
  • priority - 整数值。在具有较高值的过滤器之前应用具有较低值的过滤器。
