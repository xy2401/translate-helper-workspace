打开PHP元素

此过程描述如何使用Open PHP元素函数导航到打开项目中的PHP元素（类，函数或常量）。

要打开PHP元素：

  1. 从菜单栏中，转到导航|打开PHP元素 - 或者 - 单击工具栏上的“打开PHP元素”图标 。将打开“打开PHP元素”对话框。
  2. 输入要打开的元素的前几个字符。以这些字母开头的资源将显示在“匹配资源”窗格中，按字母顺序列出。
  3. 您可以通过标记/取消标记相关复选框来按元素类型（类，函数或常量）进行过滤。
  4. 选择所需元素，然后单击“ 确定” 。

包含元素声明的文件将在编辑器中打开，并突出显示元素。
