上下文相关的帮助UI实现

视图中的动态帮助上下文实现的Java代码示例以及具有上下文抽象的对话框来自Eclipse Data Tools Platform（DTP）1.6源代码。

  • 查看实施示例
    DataSourceExplorerView（在org.eclipse.datatools.connectivity.ui.dse中）首先实现org.eclipse.help。IContextProvider接口，然后创建org.eclipse.datatools.help的实例。ContextProviderDelegate。
  • 对话实现示例
    对话框必须在createDialogArea方法中设置帮助上下文（为对话框本身提供上下文相关的帮助）。
  • helpKey常量接口类
    要引用在ResourceBundle属性文件中定义的外化帮助上下文ID，UI组件必须实现一个接口类来声明抽象的helpKey常量。
