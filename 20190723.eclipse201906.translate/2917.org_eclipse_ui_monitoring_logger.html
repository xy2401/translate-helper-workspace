UiFreezeEventLogger

标识符：
org.eclipse.ui.monitoring.logger

以来：
1.0.0

描述：
一个扩展点，除了记录到Eclipse错误日志之外，还允许以不同方式处理UiFreezeEvent。

配置标记：

<！ELEMENT 扩展 （ 记录器 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 记录器 EMPTY>

<！ATTLIST记录器

class CDATA #IMPLIED>

  • 课程 -

例子：
接收UiFreezeEvent并将其记录到远程服务器。

API信息：
只要需要记录事件，就会将UiFreezeEvent传递给实现IUiFreezeEventLogger接口的类。

提供的实施：
每次准备好处理UiFreezeEvent时，实现IUiFreezeEventLogger的类都将调用log方法。
