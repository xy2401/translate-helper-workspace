Webapp查看

标识符：
org.eclipse.help.webapp.view

以来：
3.5

描述：
视图扩展点允许将视图提供给帮助系统webapp。webapp的视图区域是选项卡区域（在高级演示文稿中），其中包含目录以及搜索结果和其他视图。每个视图都需要两个jsp文件，一个View.jsp文件和一个Toolbar.jsp文件。此扩展点使用类来获取视图的所有属性。

配置标记：

<！ELEMENT 扩展 （ 查看 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 视图 EMPTY>

<！ATTLIST观点

class CDATA #REQUIRED>

  • class - 一个扩展AbstractView的类，它包含返回图标路径和用于此视图的jsp文件的方法。
