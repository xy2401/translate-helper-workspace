演讲工厂

标识符：
org.eclipse.ui.workbench.presentationFactories

以来：
3.0

描述：
此扩展点用于将演示工厂添加到工作台。演示工厂定义工作台的整体外观，包括视图和编辑器的呈现方式。

配置标记：

<！ELEMENT 扩展 （ 工厂 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT factory EMPTY>

<！ATTLIST工厂

class CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #REQUIRED>

  • class - 指定要用于演示工厂的完全限定类。指定的值必须是org.eclipse.ui.presentations.的子类org.eclipse.ui.presentations.AbstractPresentationFactory 。
  • id - 将用于标识此演示文稿工厂的唯一名称
  • name - 可翻译的名称，可用于在UI中显示此演示文稿工厂

例子：
以下是presentationFactory扩展的示例：
<extension point = “org.eclipse.ui.presentationFactories” > <factory class = “org.eclipse.ui.workbench。ExampleWorkbenchPresentationFactory“ /> </ extension>

API信息：
factory元素中指定的类必须是org.eclipse.ui.presentations.的具体子类org.eclipse.ui.presentations.AbstractPresentationFactory 。

提供的实施：
如果未指定或缺少演示工厂，则在org.eclipse.ui.presentations.执行org.eclipse.ui.presentations.将使用WorkbenchPresentationFactory 。
