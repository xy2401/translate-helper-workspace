什么是新的 - v4.0.0

  • 支持SWT_AWT桥接，允许混合SWT和Swing

  • 支持JGoodies FormLayout

  • 支持新的Eclipse 3.1 SWT元素

  • 增强的SWT支持

  • 增强的JFace支持

  • 增强的Swing支持

  • 增强的NLS支持

  • 编辑和财产窗格增强功能

  • 代码解析增强功能

  • V3.0.0

支持SWT_AWT桥接，允许混合SWT和Swing

  • 新的SWT_AWT调色板
  • 将任何Swing小部件嵌入SWT或RCP窗口

支持JGoodies FormLayout

  • 以图形方式管理JGFL列和行
  • 以图形方式管理单元格对齐和列/行跨越
  • 以图形方式重新排序和调整列和行的大小
  • 自动处理间隙列和行
  • 智能地将现有布局转换为JGFL
  • 支持像素和对话框等不同单位
  • 支持所有预定义的FormLayout类型
  • 支持对列进行分组和取消分组
  • 分组列和行的颜色突出显示
  • 单元格，列和行的浮动布局助手
  • 智能地合并和拆分列和行

支持新的Eclipse 3.1 SWT元素

  • 支持SWT Spinner＆Link控制
  • 支持SWT TreeColumns
  • 支持虚拟表

增强的SWT支持

  • 支持SWT DragSource和DropTarget
  • 支持SWT TableItems和TreeItems
  • 支持SWT切换按钮
  • 支持SWT Cursor属性
  • 支持SWT字体的删除线和下划线属性（仅限Windows）
  • 支持新的3.0 SWT RowLayout属性
  • 增强的SWT FormLayout支持
  • 支持SWT颜色常数源
  • 支持自定义SWT拖放传输类

增强的JFace支持

  • 支持ApplicationWindow coolbars
  • JFace Action的样式属性支持
  • SWT对话框，TitleAreaDialogs和ApplicationWindows的图标支持
  • 支持ApplicationWindow中的Shell事件
  • 支持JFace动作加速器属性

增强的Swing支持

  • 支持自定义Swing属性和编辑器
  • 支持自定义Swing小部件定制程序
  • 支持Swing JFrame向导中的超类
  • 支持实例化自定义JTable / JTree / JComboBox模型
  • 添加了对解析和编辑JTable列的支持
  • GBL选择句柄的工具提示，在首选项页面上有选项
  • 所有Swing布局的键盘导航支持
  • 支持具有自定义默认值的GridBagConstraints子类
  • 支持GridBagConstraints的静态方法工厂
  • 支持树和调色板中的自定义组件图标

增强的NLS支持

  • 添加了用于在所有语言环境中复制默认值的标志
  • 在“新建本地”对话框中添加了标志，以将默认值复制到新属性文件
  • 添加了为新添加的小部件自动外部化字符串的功能

编辑和财产窗格增强功能

  • 使用Ctrl键按名称添加自定义小部件
  • 通过选择Bean自动添加到自定义控件选项板
  • 添加了大多数组件属性的工具提示
  • 当设计画布处于活动状态时，支持属性表中的鼠标滚轮
  • 支持在文本属性编辑器中复制/粘贴（Ctrl + C / X / V）

代码解析增强功能

  • 支持代码隐藏
  • 无法转换表达式时支持默认字符串属性值
