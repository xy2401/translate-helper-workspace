Includepath容器初始化程序

标识符：
org.eclipse.wst.jsdt.core。JsGlobalScopeContainerInitializer

以来：
2.0

描述：
此扩展点允许客户端提供自定义includepath容器初始值设定项，用于将类路径容器延迟绑定到org.eclipse.wst.jsdt.core的实例。IJsGlobalScopeContainer。

配置标记：

<！ELEMENT 扩展 （ JsGlobalScopeContainerInitializer *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT JsGlobalScopeContainerInitializer EMPTY>

<！ATTLIST JsGlobalScopeContainerInitializer

id CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 标识将激活此初始值设定项的所有容器的唯一名称。
  • class - 实现此容器初始值设定项的类。该类必须实现org.eclipse.wst.jsdt.core.JsGlobalScopeContainerInitializer 。

例子：
为名为“JDK”的类路径容器声明JsGlobalScopeContainerInitializer示例：
                                                                       
<extension point="org.eclipse.wst.jsdt.core.JsGlobalScopeContainerInitializer">            
   <JsGlobalScopeContainerInitializer                                          
      id="JDK"                                                        
      class="com.example.MyInitializer"/>                           
</extension>
