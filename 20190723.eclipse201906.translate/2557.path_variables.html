路径变量首选项

“路径变量”首选项页面显示已配置路径变量的列表，并允许您添加或编辑它们。

路径变量是映射到计算机上特定位置的名称。

通过使用路径变量，您可以与团队成员共享包含链接资源的项目，而不需要与文件系统上相同的目录结构。

包含路径变量也可以添加到项目的包含路径中。

可以从Window |访问Path Variables Preferences页面偏好| PHP |路径变量。

  1. 要添加新路径变量：
  2. 单击新建。
  3. 输入路径的名称。
  4. 输入路径应指向的位置。
  5. 单击确定。
  6. 路径变量将添加到列表中。
  7. 单击“确定”以应用设置。
