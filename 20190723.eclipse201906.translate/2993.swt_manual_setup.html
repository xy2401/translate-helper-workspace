SWT独立示例设置

将SWT添加到工作区

  1. 下载SWT以用于独立应用程序。可以在与Eclipse SDK相同的下载页面上获得独立版本的SWT。查找标题为SWT二进制和源的部分。不要提取存档文件，只需将其保存到磁盘即可。
  2. 从文件菜单中选择导入....
  3. 选择Existing Projects into Workspace ，然后单击Next按钮。
  4. 选择选择存档文件：并使用“ 浏览”按钮找到先前下载的SWT独立存档。
  5. 单击“ 完成”按钮。

导入示例源

  1. 下载并安装Eclipse Example Plug-ins。Eclipse示例插件可在与Eclipse SDK相同的下载页面上获得。查找标题为“ 插件示例”的部分。您可以在安装Eclipse的同一位置解压缩示例，也可以选择其他位置。如果将示例安装在与Eclipse相同的位置，则当您选择Windows> Show View ...> Other ...> SWT Examples时，示例视图和编辑器将显示在Eclipse环境中。（如果SWT示例视图没有立即显示在菜单中，请使用-clean参数重新启动eclipse）。
  2. 要导入示例源，请从“ 文件”菜单中选择“ 新建”>“项目... ”。
  3. 选择Java Project并单击Next按钮。
  4. 为java项目指定一个名称，例如“SWT Examples”。
  5. 取消选择使用默认位置 ，然后单击浏览... 按钮，找到以下目录： eclipse/plugins/org.eclipse.sdk.examples.source_3.1.0/src/org.eclipse.swt.examples_3.1.0
  6. 单击“ 下一步”按钮。
  7. 单击Projects选项卡，然后单击Add按钮。
  8. 在org.eclipse.swt旁边放一张支票，然后点击OK按钮。
  9. 单击“ 完成”按钮。

此时应编译SWT示例而不会出现任何错误。检查Problems视图是否有错误。如果你收到像“ java.lang. ”这样的错误java.lang.找不到Object “这表示您尚未配置JRE。转到Window > Preferences ...对话框，然后选择Java > Installed JREs首选项页面。确保安装了JRE并且JRE的路径是正确的。

运行示例

现在您可以运行SWT独立示例。

  1. 打开Java透视图。
  2. 在“包”视图中，选择要运行的主类。例如，Address Book示例的主类是org.eclipse.swt.examples.addressbook.AddressBook 。
  3. 从主菜单中选择Run > Run As ... > Java Application 。
