结合标记

绑定 标记是扩展Java编辑器左侧垂直标尺（标记栏）上的OT / J特定注释。这些标记有三种形式：

  • 一个playingBy标记 使用playedBy表示一个或多个角色绑定到当前类的playedBy （ 2.1 ）。
  • 一个callin标记 表示一个或多个callin绑定（ 4 ）影响给定的基本方法。
  • 标注标记 表示一个或多个标注绑定（ 3 ）影响给定的基本方法，并且这些标注绑定应用解封装（ 3.4 ） - （ 没有解封装的标注绑定按常规方法调用排序，因此不被认为与标尺中的标记相关）。

标注解封标记

除了提高对这些绑定的认识之外，标记还可以用于直接导航到相应的角色或callin / callout绑定。当右键单击绑定标记时，将打开一个弹出菜单，其中包含相应的子菜单OT / J绑定角色 ，或OT / J callin绑定或OT / J callout解封装 。该子菜单依次保存当前位置的所有绑定，按团队排序。
