CommandLauncherFactory

Identifier:
org.eclipse.cdt.core.CommandLauncherFactory

Since:
CDT 9.4.0

Description:
This extension point is used to contribute a Command Launcher factory to CDT. A Command Launcher factory creates a Command Launcher for running commands.

Configuration Markup:

<!ELEMENT extension (factory)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT factory EMPTY>

<!ATTLIST factory

id       CDATA #IMPLIED

priority CDATA "0"

class    CDATA #IMPLIED>

  • id -
  • priority - Integer priority of factory. Default is 0. Use higher values if you want to override an existing factory.
  • class - Command Launcher factory class that implements org.eclipse.cdt.core.ICommandLauncherFactory.
