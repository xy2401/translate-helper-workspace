CommandLauncherFactory

标识符：
org.eclipse.cdt.core。CommandLauncherFactory

以来：
CDT 9.4.0

描述：
此扩展点用于向CDT提供Command Launcher工厂。Command Launcher工厂创建命令启动器以运行命令。

配置标记：

<！ELEMENT 扩展 （ 工厂 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT factory EMPTY>

<！ATTLIST工厂

id CDATA #IMPLIED

优先级CDATA“0”

class CDATA #IMPLIED>

  • id -
  • priority - 工厂的整数优先级。默认值为0。如果要覆盖现有工厂，请使用更高的值。
  • class - 实现org.eclipse.cdt.core的Command Launcher工厂类。ICommandLauncherFactory。
