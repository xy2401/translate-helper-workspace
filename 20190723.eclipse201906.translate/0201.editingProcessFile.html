编辑BPEL流程文件

如果未打开“ 属性”视图和“ 调色板”视图，则可以通过右键单击BPEL编辑器并选择“ 在属性中 显示”或“在调色板视图中显示调色板”选项来打开视图 。
然后你应该有这样的观点：

在Palette视图中，您可以将BPEL元素拖动到BPEL编辑器并将其放在所需的位置。

在“ 属性”视图中，您可以查看BPEL流程中每个元素的信息。
在BPEL编辑器中选择元素时， Properties视图的内容会自动更新。

下表描述了Properties视图中显示的选项卡：

标签 描述

描述

显示有关元素的描述性信息，例如元素的名称。

细节

显示有关元素的详细和重要信息。它是元素中最重要的部分。元素的大多数属性都在此部分中设置。

加入行为

显示元素的Join Failure属性。

文档

显示元素的文档子元素。

其他

每个BPEL元素都有自己的部分：Correlation部分，Message Exchange部分等。这些部分将在后面的章节中介绍。

为了了解简单的BPEL流程如何运作，您应该执行以下步骤：

  • 修改流程的两个变量：

    • 单击输入变量的详细信息选项卡，然后单击“ 浏览...”按钮。

      然后从匹配列表中选择字符串基元。

    • 在弹出菜单中选择xsd作为命名空间。

  • 在receiveInput元素和replyOutput元素之间添加Assign元素。

  • 在BPEL编辑器中选择Assign元素，以便在Properties视图中获取它的属性信息。

  • 在Description选项卡中将其名称设置为assignHelloMesg 。

    在“属性”视图的“ 详细信息”部分中，应单击“ 新建”以向元素添加复制子元素。将“变量赋值”（输入：字符串输出）。此时，将出现“初始化程序”弹出对话框。在对话框中单击“ 是 ”。

    然后，您应该再次单击New并选择Expression to Variable （将concat($input,' World') ）分配给result：string 。
