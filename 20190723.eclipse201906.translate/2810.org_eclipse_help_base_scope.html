帮助范围

标识符：
org.eclipse.help.base.scope

以来：
3.5

描述：
支持帮助目录，索引和搜索结果中的用户定义范围。

配置标记：

<！ELEMENT 扩展 （ 范围 *， scopeProducer ？）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 范围 EMPTY>

<！ATTLIST范围

class CDATA #REQUIRED

id CDATA #REQUIRED>

定义一个范围类，其中包含确定哪些UA元素在范围内的方法。


  • class - 一个类，其中包含确定范围处于活动状态时应显示哪些元素的方法。
  • id - 此范围的id，在范围ID中必须是唯一的。

<！ELEMENT scopeProducer EMPTY>

<！ATTLIST scopeProducer

class CDATA #REQUIRED>

使用scopeProducer在运行时动态生成范围
