比较过滤器

此扩展点是内部的

标识符：
org.eclipse.compare.compareFilters

以来：
[输入出现此扩展点的第一个版本。]

描述：
此扩展点仅供内部使用。
此扩展点允许插件为特定内容类型注册比较筛选器。过滤器将在比较查看器中作为切换操作公开，并可用于自定义比较文档时如何计算差异以及影响结构差异的计算。扩展点必须实现接口org.eclipse.compare.ICompareFilter 。对于影响结构差异计算的过滤器，结构创建者必须扩展org.eclipse.compare.structuremergeviewer.StructureCreator 。

配置标记：

<！ELEMENT 扩展 （ filter *， contentTypeBinding *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

限内部使用


  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 过滤器 EMPTY>

<！ATTLIST过滤器

id CDATA #REQUIRED

definitionId IDREF #IMPLIED

扩展CDATA #IMPLIED

class CDATA #REQUIRED

filter.label CDATA #REQUIRED

filter.tooltip CDATA #REQUIRED

filter.tooltip.checked CDATA #IMPLIED

filter.tooltip.unchecked CDATA #IMPLIED

filter.description CDATA #REQUIRED

filter.description.checked CDATA #IMPLIED

filter.description.unchecked CDATA #IMPLIED

filter.image CDATA #IMPLIED>

限内部使用


  • id - 可用于引用过滤器的唯一标识符。它还将用作添加到比较查看器的切换操作的操作ID
  • definitionId - 添加到比较查看器的切换操作的命令ID
  • extensions - 以逗号分隔的文件扩展名列表，例如“java，txt”
  • class - ICompareFilter的实现。
  • filter.label - 将在此过滤器的UI中使用的可翻译标签。
  • filter.tooltip - 将在此过滤器的UI中使用的可翻译标签。
  • filter.tooltip.checked - 将在此过滤器的UI中使用的可翻译标签。
  • filter.tooltip.unchecked - 将在此过滤器的UI中使用的可翻译标签。
  • filter.description - 将在此过滤器的UI中使用的可翻译标签。
  • filter.description.checked - 将在此过滤器的UI中使用的可翻译标签。
  • filter.description.unchecked - 将在此过滤器的UI中使用的可翻译标签。
  • filter.image - 将在此过滤器的UI中使用的图像。

<！ELEMENT contentTypeBinding EMPTY>

<！ATTLIST contentTypeBinding

contentTypeId IDREF #REQUIRED

filterId IDREF #REQUIRED>

限内部使用。
contentTypeBinding将比较筛选器绑定到内容类型。


  • contentTypeId - 使用org.eclipse.core.contenttype.contentTypes扩展点定义的内容类型的id。
  • filterId - 使用此扩展点的filter元素定义的filter器的id（即org.eclipse.compare.compareFilters ）

例子：
以下是示例文件（扩展名“.example”和“.example2”）和文本内容类型的比较过滤器示例：


<extension
         point="org.eclipse.compare.compareFilters">
      <filter
           id="com.example.IgnorePrefixId"
            class="com.example.IgnorePrefix"
            extensions="example,example2"
            definitionId="com.example.IgnorePrefixCommand"
            filter.description="Ignore Columns 1-6"
            filter.label="Ignore Prefix"
            filter.image="/icons/ignorePrefix.gif"
            filter.tooltip="Ignore Prefix">
      </filter>
      <contentTypeBinding
  contentTypeId="org.eclipse.core.runtime.text"
  filterId="com.example.IgnorePrefixId">
   </contentTypeBinding>
   </extension>

API信息：
贡献的类必须实现org.eclipse.compare.ICompareFilter 。对于影响结构差异计算的过滤器，结构创建者必须扩展org.eclipse.compare.structuremergeviewer.StructureCreator 。
