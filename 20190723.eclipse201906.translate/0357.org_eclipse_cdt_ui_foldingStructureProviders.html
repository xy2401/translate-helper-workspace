C / C ++折叠结构提供程序

标识符：
org.eclipse.cdt.ui.foldingStructureProviders

以来：
3.0

描述：
对此扩展点的贡献定义了C / C ++编辑器的折叠结构。也就是说，它们定义了可折叠的C / C ++源文件的区域。请参阅org.eclipse.jface.text.source.ProjectionViewer供参考。

扩展可以选择提供一个首选项块，它将出现在C / C ++编辑器首选项页面上。

配置标记：

<！ELEMENT 扩展 （ 提供者 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

id IDREF #REQUIRED

名称CDATA #IMPLIED

class CDATA #REQUIRED

preferencesClass CDATA #IMPLIED>

  • id - 此提供程序的唯一标识符。
  • name - 此提供程序的名称。如果没有给出，则使用id代替。
  • class - org.eclipse.cdt.ui.text.folding的实现。ICFoldingStructureProvider
  • preferencesClass - org.eclipse.cdt.ui.text.folding的实现。ICFoldingPreferenceBlock

例子：
请参阅org.eclipse.cdt.ui.text.folding.DefaultCFoldingStructureProvider的示例。

提供的实施：
org.eclipse.cdt.ui.text.folding.DefaultCFoldingStructureProvider为C / C ++编辑器提供默认的折叠结构。 org.eclipse.cdt.internal.ui.text.folding.DefaultCFoldingPreferenceBlock为默认结构提供程序提供首选项块。
