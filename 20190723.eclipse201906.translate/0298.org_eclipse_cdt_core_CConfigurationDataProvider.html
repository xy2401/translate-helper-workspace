CConfigurationData提供程序

标识符：
org.eclipse.cdt.core。CConfigurationDataProvider

描述：
每次更改Build Active配置时，都会通过删除与先前活动配置关联的性质并添加与新活动配置关联的性质来调整Project性质集。这允许调整与项目一起使用的构建器等。

配置标记：

<！ELEMENT 扩展 （ 提供者 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>


<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

自然CDATA #IMPLIED

class CDATA #REQUIRED

conflictingNatures CDATA #IMPLIED>

  • natures - 与此提供程序关联的项目自然ID。ID以分号分隔。
  • class - 扩展org.eclipse.cdt.core.settings.model.extension的类。CConfigurationDataProvider
  • conflictingNatures - 要为此提供程序取消设置的项目性质ID。ID以分号分隔。
