系统要求

  • Windows XP / Vista / 7或 Linux + GTK2或 OSX
    • 32位和64位
    • Linux GTK 2.14或更高版本（建议使用GTK 2.20或更高版本）
  • 必须使用JDK 1.6或更高版本启动Eclipse
  • 必须完全安装Eclipse JDT 。
  • 必须安装PDE才能支持SWT / RCP。
  • 必须安装WST （或WTP ）才能支持XML / XWT
    • 应加载Eclipse XML编辑器和工具功能以进行XML编辑
  • 该产品可能无法与远程桌面环境一起使用（除了像VNC这样的简单环境）
  • 任何基于Eclipse 3.5,3.6,3.7,3.8或4.2的IDE

为Eclipse 3.5或更高版本安装Update Site或Zip版本需要完整的Eclipse SDK，包括JDT（以及支持SWT / RCP的PDE）。使用Eclipse Classic，Java或RCP / RAP Developers发行版或单独安装JDT和PDE。

使用任何基于XML的API（如Eclipse XWT）时，应将Eclipse XML编辑器和工具功能 （来自WST / WTP）加载到Eclipse中，以获得最佳的XML编辑体验。
