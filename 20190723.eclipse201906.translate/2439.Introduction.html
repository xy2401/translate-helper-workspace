Photran包含许多复杂的功能，旨在使编写，修改，搜索和维护Fortran代码变得更加容易。这些包括内容辅助，可以在您键入时“自动完成”变量和函数名称; 声明视图，可以显示所选变量或过程的主要注释; Fortran搜索，它允许您查找模块，变量，过程等的声明和引用;和重构，它们会改变您的源代码以改进其设计，同时保留其行为。

启用高级功能

要使用本文档中描述的任何高级功能，您必须按照以下说明专门启用它们。 执行此操作后，Photran将为您的项目编制索引 ;也就是说，它将构建一个数据库，其中包含项目中每个文件中声明的模块，子程序等。每次保存文件时，都会以增量方式更新此信息。尽管此过程通常相当快，但在处理非常大的项目时可能会中断，因此默认情况下已禁用此过程。请注意，第一次对项目编制索引时，可能需要一段时间，因为Photran必须分析项目中的每个文件;之后，它只会索引已更改的文件（以及依赖于已更改文件的文件），因此通常会更快。

如何启用高级功能

  1. 在Fortran Projects视图中右键单击您的项目
  2. 单击“属性”
  3. 在左侧列表中展开Fortran General，然后单击Analysis / Refactoring（参见下面的屏幕截图）
  4. 选中“启用Fortran分析/重构”复选框
  5. 如果要启用内容辅助，Fortran声明视图等，请同时选中这些框
  6. 您可能还想在此时设置模块和包含路径（参见下文）
  7. 单击确定

设置模块和包含路径

如果源代码包含引用其他文件中的模块的INCLUDE行或USE行，则Photran需要知道在哪里查找这些行。它不会自动解决这个问题。 对于您计划使用重构支持的每个项目，

  1. 在Fortran Projects视图中右键单击项目的文件夹
  2. 单击“属性”
  3. 在左侧列表中展开Fortran General，然后单击Analysis / Refactoring
  4. 列出Photran在重构时应搜索INCLUDE文件和模块的文件夹。将从列出的最后一个文件夹中按顺序搜索它们。不会自动搜索子文件夹;你必须明确地包括它们。
  5. 单击确定

高级功能和C预处理代码

从Photran 6开始，Fortran代码支持C预处理器指令，如Photran用户指南中所述。虽然本手册中描述的大多数高级功能都可以处理C预处理代码，但重构目前还不能。如果您尝试重构C预处理代码，您将收到一条错误消息。
