查看类型层次结构

类型层次可以在任何一个连结类型层次视图或类型分层结构视图来查看。

在快速类型层次结构视图中查看类型

要在“快速类型层次结构”视图中查看类型：

  1. 在PHP编辑器或Project Explorer或大纲视图中选择类型名称。
  2. 按Ctrl + T. - 或 - 从菜单栏转到导航|快速类型层次结构 。“快速类型层次结构”视图将在编辑器中以所选类型显示。

注意：

在显示快速类型层次结构视图时按Ctrl + T将在超类型层次结构和子类型层次结构之间切换。

在类型层次结构视图中查看类型

通过使用“在层次结构中打开类型”对话框或直接在编辑器或项目浏览器中选择元素，可以在“类型层次结构”视图中查看类型。

要在“类型层次结构”视图中查看类型：

通过“Open Type in Hierarchy”向导：

  1. 按Ctrl + Shift + H - 或 - 从菜单栏转到导航|在层次结构中打开类型 。将显示“在层次结构中打开类型”对话框。
  2. 如果先前在编辑器或大纲视图中选择了类型，则会在类型字段中输入该类型。如果未选择类型，请开始键入所需类型/方法的字符串以过滤结果。您可以使用通配符或CamelCase表示法（例如，DTB用于DateTextBox）。
  3. 选择所需类型，然后单击“ 确定” 。

直接来自编辑器或Project Explorer：

  1. 在编辑器或Project Explorer中选择一种类型。
  2. 按F4 。

注意：

如果选择不是可解析元素（即，如果选择不是类名，接口名或类方法，常量或字段），则不会激活该操作。该类型将显示在“类型层次结构”视图中。
