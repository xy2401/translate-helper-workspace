管理实体继承

继承持久类型

  1. 在Palette的继承列表中，选择Inherit Persistent Type 。

  2. 单击源实体或Is-A关系的映射超类，或按ESCAPE取消。

  3. 单击Is-A关系的目标实体或按ESCAPE取消。

    Eclipse IDE创建并可视化Is-A关系。

继承实体

  1. 在Palette的Inheritance列表中，选择Inherited Entity 。

  2. 单击源实体或Is-A关系的映射超类，或按ESCAPE取消。

  3. 单击图表中的某个位置或按ESCAPE取消。

    Eclipse IDE创建并可视化新的继承实体，以及相关的Is-A关系。

删除Is-A关系

要删除特定的Is-A关系，请在JPA图上选择关系并按DELETE ，或从上下文菜单中选择“ 删除 ”。

通过JPA图删除关系也会将其从应用程序数据模型中删除。
