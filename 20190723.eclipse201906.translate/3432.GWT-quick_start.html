GWT Designer快速入门


 

  1. 安装最新的GWT SDK或Google Plugin for Eclipse 。
     
  2. 安装最新的GWT Designer中建立到Eclipse 3.4 ， 3.5或3.6 。
     
  3. 如果已安装GWT SDK ，则需要指定GWT安装目录的路径 。
     
  4. 使用GPE或GWT Designer GWT / Java项目向导创建新的GWT项目。
     
  5. 使用GWT Designer EntryPoint向导创建第一个GWT UI或编辑现有UI 。
     
  6. 为UI选择适当的面板类型 。
     
  7. 从Palette添加各种组件，并使用Property Pane编辑它们的属性。
     
  8. 可视化创建和编辑窗口的菜单栏 。
     
  9. 将事件处理程序添加到各种窗口小部件以提供行为。
     
  10. 如果创建新Panel ，则可以使用“ 选择组件”命令将其嵌入到另一个窗口中。
     
  11. 您可以通过右键单击并选择Run As> GWT Application来测试启动窗口。
     
  12. 国际化您的应用程序并管理多个区域设置。
