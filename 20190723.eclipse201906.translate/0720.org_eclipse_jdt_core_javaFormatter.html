Java Formatter

Identifier:
org.eclipse.jdt.core.javaFormatter

Since:
3.11

Description:
This extension point allows clients to supply their own Java source code formatter. The formatter is expected to use the default formatter's options.

Configuration Markup:

<!ELEMENT extension (javaFormatter)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT javaFormatter EMPTY>

<!ATTLIST javaFormatter

id    CDATA #REQUIRED

name  CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - A unique identifier used to reference this Java formatter.
  • name - The name of this Java formatter, used to present this formatter in the UI.
  • class - The fully-qualified name of the Java class that extends the org.eclipse.jdt.core.formatter.CodeFormatter abstract class.

Examples:
Example of a declaration of a javaFormatter: 

<extension point="org.eclipse.jdt.core.javaFormatter">
   <javaFormatter
         class="myformatter.MyFormatter"
         id="myformatter.javaFormatter"
         name="My Custom Formatter">
   </javaFormatter>
</extension>
