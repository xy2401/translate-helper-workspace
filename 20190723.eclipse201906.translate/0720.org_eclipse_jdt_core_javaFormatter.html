Java格式化程序

标识符：
org.eclipse.jdt.core.javaFormatter

以来：
3.11

描述：
此扩展点允许客户端提供自己的Java源代码格式化程序。格式化程序应使用默认格式化程序的选项。

配置标记：

<！ELEMENT 扩展 （ javaFormatter ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT javaFormatter EMPTY>

<！ATTLIST javaFormatter

id CDATA #REQUIRED

名称CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 用于引用此Java格式化程序的唯一标识符。
  • name - 此Java格式化程序的名称，用于在UI中显示此格式化程序。
  • class - 扩展org.eclipse.jdt.core.formatter的Java类的完全限定名称。CodeFormatter抽象类。

例子：
javaFormatter声明的javaFormatter ：

<extension point="org.eclipse.jdt.core.javaFormatter">
   <javaFormatter
         class="myformatter.MyFormatter"
         id="myformatter.javaFormatter"
         name="My Custom Formatter">
   </javaFormatter>
</extension>
