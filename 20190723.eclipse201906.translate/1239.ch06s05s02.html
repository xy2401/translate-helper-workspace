6.5.2. Supported systems

The ITE uses Eclipse Link as an object relational mapping layer.

The ITE is tested with Oracle. Other database solutions supported by Eclipse Link may also be used as a database, though these have not been tested by our team.

Some databases which limit the row size (like MySQL) have problems with Jubula if they are set to use multi-byte encodings like UTF8. Try to use single-byte encodings like latin1 if you encounter any problems.
