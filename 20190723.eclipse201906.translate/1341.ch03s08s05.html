3.8.5.1。�配置从ITE启动的Java AUT

Java FX的AUT配置对话框有三个不同的详细级别：基本，高级和专家。

有关不同级别的信息，请参阅以下部分。

3.8.5.2。�BasicJavaFXAUT配置

如果可以通过可执行文件（例如.bat，.exe，.cmd，.sh等）启动AUT，则可以使用基本设置来配置AUT。如果您的AUT是JAR，那么您还可以使用可执行文件字段输入Java可执行文件（即java或java.exe），然后在高级配置的AUT Arguments字段中输入JAR的路径（ Section�3.8） .5.3，“高级JavaFX AUT配置” ）。

您必须使用Java 8 JRE才能启动JavaFX AUT。

  1. 如前所述输入基本配置详细信息（ Section�3.8.2，“每个AUT配置所需的基本信息” ）。

  2. 在“ 可执行文件名”字段中输入可执行文件名。如果定义工作目录，则此路径可以是相对的（ Section�3.8.5.3，“高级JavaFX AUT配置” ）。

有关AUT配置的高级属性的信息，请参见下一节Section3.8.5.3“高级JavaFX AUT配置” 。

高级配置对话框允许您为AUT创建工作目录，并添加启动AUT所需的命令行参数。如果在基本配置中输入Java可执行文件作为可执行文件名，则可以在AUT Arguments字段中输入-jar <JARFILE> （图�3.16，“示例JavaFX AUT配置” ）。

3.8.5.4。�ExpertJavaFX AUT配置

您可以使用专家对话框配置有关如何启动AUT的更多详细信息。

  1. 输入任何所需的系统环境变量 ，格式为“ <VARNAME> = <value> ”，即“ PATH = C：∖ ”。按�ENTER�将每个变量用新行分开 。

    请注意，目前不支持将一个变量的内容“嵌入”到另一个变量中。也就是说，如果你有一个名为FOO的变量，其值为“ abc ”，并将第二个变量BAR的值设置为“ ％FOO％def ”，则第二个变量将不包含“ abcdef ”，而是包含确切的文本“ ％FOO％def “，未经评估。
