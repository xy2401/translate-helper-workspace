导出HTML报告

此过程描述如何将PHP配置文件会话数据导出为HTML报告。

  1. 打开分析监视器视图（如果尚未打开）。
  2. 在视图中单击鼠标右键以激活上下文菜单，然后选择“报告...”选项以打开“ 导出HTML报告”向导。

  3. 选择要作为HTML报告导出的配置文件会话。
  4. 通过选中/取消选中选择要导出的信息：组中的选项， 选择要报告的信息类型。
  5. 使用报告数据指定文件的位置。

  6. 按“ 完成”按钮导出报告数据。
