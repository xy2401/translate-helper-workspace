创建HTML文件

这些过程描述了如何创建新的HTML文件，允许实现PDT的完整HTML编辑功能。

要创建新的HTML文件：

  1. 在Project Explorer视图中，选择要在其中创建文件的文件夹，然后从菜单栏转到File |新的| HTML页面 - 或 - 右键单击该文件夹，然后选择“ 文件”|新的| HTML。页面 。将出现新的HTML页面对话框。
  2. 输入文件名，然后单击“ 下一步” 。
  3. 选择要用于新文件的HTML模板，然后单击“ 完成” 。

将创建新的HTML文件，并在标准HTML编辑器中打开。

如果您对在WYSIWYG视图中编辑代码感兴趣，可以安装WST网页编辑器插件。此插件提供了功能，同时允许您在项目完成之前查看与最终结果非常相似的内容。要下载必要的插件，请点击此链接： http ： //kb.zend.com/index.php？View = entry＆EntryID = 419 。
