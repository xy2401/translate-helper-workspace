对象团队项目创建

打开项目向导

创建新的Object Teams项目的最简单方法是使用Object Teams项目向导。可以通过从主工作台窗口中选择文件>新建>项目...或从包浏览器的弹出菜单中选择新建>项目...或从中选择项目...来打开项目向导选择对话框。工作台工具栏中“ 新建”按钮上的下拉菜单。然后，必须选择用于创建对象团队项目的向导。

定义设置

该向导支持New Java Project Wizard的所有功能，并自动为OT / J配置项目。向导中只添加了一个用户可见选项，即。选择“目标编织方案” ：

如果不确定请保留默认值“OTRE”

产生的影响

输入所有信息后，将创建具有所选名称的项目，将创建文件.classpath和.project，并且将更新包浏览器中的树视图并显示新项目。

OT / Equinox：插件项目

如果项目是使用Object Teams（OT / Equinox）开发Eclipse插件，请选择“Object Teams Plugin Project”向导。此向导是常规“插件项目”向导的变体。

添加对象团队支持现有项目

或者，如果您已有一个有效的Java项目或插件项目，则可以使用项目的上下文菜单Configure> Add Object Teams支持添加对象团队支持 。
