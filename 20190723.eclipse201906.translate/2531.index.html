PHP Profile Perspective

运行配置文件会话时，可以自动启动PHP配置文件透视图。它允许您查看与您分析的脚本相关的所有信息。

注意：

要配置在运行配置文件会话时是否自动启动PHP Profile Perspective ，请转到Window |偏好| PHP |在“打开关联接收会话数据的视角”类别中， 配置并选择始终，从不或提示。

默认情况下，PHP Profile Perspective包含以下视图：

  • 分析监视器视图 - 显示以前运行的分析会话的列表。
  • Profiler信息视图 - 提供有关分析会话的一般信息，例如：持续时间，日期，构造请求的URL的文件数等。
  • 执行统计信息视图 - 显示在分析过程中调用的文件列表以及有关文件中元素的处理时间的详细信息。
  • 执行流程视图 - 显示执行流程的流程，并总结每个功能所花费的百分比和时间。

还可以通过对配置文件会话数据项执行的相应操作打开其他视图：

  • 代码覆盖率摘要视图 - 在分析过程中涵盖了多少行代码的摘要。
  • 代码覆盖率视图 - 显示所选PHP脚本的代码覆盖率的图形表示。
  • 函数调用统计信息视图 - 显示有关所选函数的统计信息，调用它的函数以及它调用的函数。
