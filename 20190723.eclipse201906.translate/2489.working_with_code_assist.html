使用内容辅助

本教程的目的是教您如何使用PDT的内容辅助功能，以便快速，轻松，准确地编写代码。

内容：

  • 目的和用法
  • 激活内容辅助
  • 使内容辅助元素在同一范围内可用
  • 功能参数提示
  • 类类型提示
  • 配置内容辅助

目的和用法

内容辅助功能允许选择和插入现有代码元素以完成部分输入的代码。

根据光标的上下文，可能的代码元素列表显示在文件的相关位置，然后您可以选择将其自动输入到代码中。

每种类型的代码元素都有一个唯一的图标：

  1. - 保留的PHP单词
  2. - 功能
  3. - 模板
  4. - 课程
  5. - 接口
  6. - 特质
  7. - 常数
  8. - 变量（公共）

内容辅助使用以下元素：PHP类，函数，变量，常量，关键字，接口，特征，属性，值，嵌套函数，名称和语法，以及所有用户定义的类，函数和常量。

注意：

内容辅助适用于PHP和HTML。

激活内容辅助

默认情况下，一旦输入代码的前几个字符，将自动显示内容辅助选项。

以下过程演示如何使用内容辅助：

  1. 创建一个名为“File1”的新PHP文件。
  2. 在开始PHP标记下方的行上，键入“def”。
  3. 双击Code Completion窗口中的第一个define函数 - 或者 - 选择它并按Enter键 。 “define（）”出现在编辑行上。

注意：

如果内容辅助窗口未自动打开，请将光标放在所需位置，然后按Ctrl + Space 。要使内容辅助窗口自动打开，请转到从窗口|访问的内容辅助首选项页面 偏好| PHP |编辑|内容辅助并标记“启用自动激活”复选框。

功能参数提示

进入函数调用时，将显示一个函数参数提示框，详细说明应在函数调用的括号内输入的参数类型。

以下过程演示如何使用“函数参数提示”功能 ：

  1. 将光标放在上述函数调用的括号之间：“define（）”
  2. 按Ctrl + Shift + Space 。

将显示一个函数参数提示框，指示应在括号之间插入的参数类型。

使内容辅助元素在同一范围内可用

添加代码 - 在同一功能和文件中可用

同一范围内的元素（例如，在同一项目，文件或函数中）将可用于内容辅助。

属于不同范围的变量不能通过内容辅助相互使用。

以下过程演示如何使用Content Assist在同一函数和文件中插入元素：

  1. 编辑PHP文件（'File1'），使其包含以下代码：
<？php define（'continent'，'africa'）; $ control =''; $ mail ='int@eclipse.org'; function media（）{$ music =''; $ messenger =''; $ / * --------- Location_1 * /} $ / * --------- Location_2 * /？>
  1. 将光标放在“Location_1”标记的“$”处。这是在功能“媒体”内。
  2. 输入字母“m”。内容辅助窗口将显示变量“$ messenger”和“$ music”，这些变量在函数中定义。请注意，变量$ mail（不在“media（）”范围内）不可用。
  3. 接下来，将光标放在“Location_2”标记的“$”处。
  4. 输入字母“m”。内容辅助窗口将显示变量$ mail，该变量位于同一文件中。请注意，媒体的变量 - $ music和$ messenger - 不在“媒体”功能范围内，因此不会显示。
  5. 从“内容辅助”窗口中选择“邮件”，将其插入脚本中。

添加代码 - 在同一项目中可用

在一个文件中定义的代码元素也可供同一项目中的其他文件使用。

以下步骤演示了如何使用Content Assist在同一项目中插入元素：

  1. 在与“File1”相同的项目中，创建一个名为“File2”的新PHP文件。
  2. 在打开PHP标记下方的行上，键入“def”并按Ctrl + Space以激活内容辅助。双击其中一个定义选项。
  3. 在括号之间，键入“cont”并按Ctrl + Space以激活内容辅助。“File1”中定义的元素“continent”将可用。
  4. 双击“continent”将其输入到您的代码中。

当元素突出显示时，Content Assist将显示代码元素的原始位置，其值（'africa'）以及所有其他可用信息。

类类型提示

通过使用注释，您可以为变量分配其确切的类值。此分配将相应地影响此变量的内容辅助。

要查看和试用此功能：

  1. 使用以下代码创建一个新的PHP文件：
<？php函数getClass（）{return new Test（）; } class Test {function printValues（$ a，$ b）{echo“值：$ a，$ b”; $ myVar = getClass（）; $ myVar->？>
  1. 将光标放在'$ myVar->'之后（在结束的PHP标记上方的行上），然后按Ctrl + Space激活内容辅助。Content Assist将使用'Test'类中定义的函数打开（printValues（$ a，$ b））。双击它将其输入到您的代码中。

配置内容辅助

内容辅助选项可以通过内容辅助首选项页面进行配置，可从Window |访问偏好| PHP |编辑|内容辅助 。
