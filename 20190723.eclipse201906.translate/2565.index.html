调试首选项

Debug首选项页面允许您配置调试过程的默认设置。

可从Window |访问“调试首选项首选项”页面偏好| PHP |调试首选项。

可以从调试首选项页面配置的设置是：

默认设置

  • PHP服务器 - 选择默认情况下调试器将使用的服务器。单击“PHP Servers”类别以进入PHP Servers管理页面。有关更多信息，请参阅PHP服务器。
  • PHP可执行文件 - 选择所需的默认PHP版本。单击“PHP可执行文件”类别以进入PHP可执行文件管理页面。有关更多信息，请参阅PHP可执行文件。

编码设置

  • 调试传输编码 - 从下拉列表中选择所需的调试传输编码。
  • 调试输出编码 - 从下拉列表中选择所需的调试输出编码。
  • 在第一行中断 - 标记此复选框以强制调试过程默认停在第一行代码处。

注意：

可以从General |下的首选项菜单访问更多PHP编码选项内容类型|文字| PHP内容类型。

仅将“调试首选项”设置应用于特定项目：

  1. 选择标记为“配置项目特定设置”的链接。
  2. 从列表中选择所需的项目。将出现“调试首选项属性”对话框。
  3. 选择所需设置，然后单击“应用”。
  4. 将出现一个提示对话框，指出必须重建项目才能使设置生效。
  5. 单击是以重建项目。单击“否”以仅在重新启动PDT时执行重建。单击“取消”取消操作。
