来源定位器

标识符：
org.eclipse.debug.core.sourceLocators

描述：
此扩展点提供了一种机制，用于指定要与启动配置一起使用的源定位器。

配置标记：

<！ELEMENT 扩展 （ sourceLocator *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT sourceLocator EMPTY>

<！ATTLIST sourceLocator

id CDATA #REQUIRED

class CDATA #REQUIRED

名称CDATA #REQUIRED>

  • id - 指定此源定位器的唯一标识符。
  • class - 指定实现IPersistableSourceLocator的Java类的完全限定名称。
  • name - 一个人类可读的名称，描述了此源定位器的类型。

例子：
以下是源定位器扩展点的示例：


 <extension point="org.eclipse.debug.core.sourceLocators">
  <sourceLocator
   id="com.example.ExampleIdentifier"
   class="com.example.ExampleSourceLocator"
   name="Example Source Locator">
  </sourceLocator>
 </extension>

API信息：
属性类的值必须是实现接口org.eclipse.debug.core.model的Java类的完全限定名称。IPersistableSourceLocator 。
