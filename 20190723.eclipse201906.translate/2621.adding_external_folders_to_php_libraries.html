将外部文件夹添加到PHP库

此过程描述如何将外部文件夹添加到用户库。这允许您使用计算机上不同位置的文件夹编译用户库。要添加外部文件夹，您必须已创建用户库。有关更多信息，请参阅添加PHP库 。

如果您对使用压缩文件感兴趣，请单击添加ZIP ....按钮并选择文件，而不是按照以下步骤操作。

要将外部文件夹添加到用户库：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，单击Add External folder ... 将打开“外部文件夹选择”对话框。
  3. 在“外部文件夹选择”对话框中选择一个文件夹，或单击“创建新文件夹”创建一个新文件夹 。
  4. 要应用更改，请单击确定 。

该文件夹将添加到PHP库中。

用户定义库中包含的文件夹允许您控制其访问规则。访问规则默认设置为“无限制”。有关详细信息，请参阅下面的定义访问规则

定义访问规则

此过程描述如何定义访问规则。访问规则允许您控制库的可用位置。从这里，您可以控制在项目级别的内容辅助等功能中可用的代码完成选项。

要定义访问规则，必须在项目中添加外部PHP文件夹。有关更多信息， 请将外部文件夹添加到PHP库 。

要为用户定义的库定义访问规则：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在“PHP库首选项”页面中，双击位于添加的文件夹下方的分层树中的“访问规则：”。默认设置为“无限制”。将打开“类型访问规则”对话框。
  3. 点击添加... 将打开“添加访问规则”对话框。
  4. 从“解决方案：”下拉菜单中选择以下选项之一：
    • 禁止 - 这将使相关项目中的文件完全无法访问。
    • 气馁 - 这将使文件可用但不强烈建议。
    • 可访问 - 这将允许项目自由访问文件及其内容。
  5. 在“规则模式”字段中，插入规则模式以定义规则的模式。使用通配符创建模式。接受的通配符值为：
    • '*'
    • '**'
    • “？“

这将自动将规则应用于与模式匹配的任何文件。

要应用更改，请单击确定 。

在“类型访问规则”对话框中，您还可以通过单击按钮来编辑，删除或更改已定义访问规则的顺序。在允许文件访问项目之前，它将首先确认没有任何访问规则适用于它。这些规则按照它们放置在这里的顺序进行检查。您的PHP库现在具有在PHP库首选项页面的分层树中定义的访问规则。您可以通过重复此过程随时编辑文件的访问规则。
