3.37.3. Symbols with special meanings for certain parameters

In addition to these general symbols, there are a few symbols which have a special meaning for particular components (Table 3.6, “Symbols for components”). They must be masked if you want to use them without their special function for these components only. Masking these symbols must be done with a double backslash.
