快速辅助处理器

标识符：
org.eclipse.cdt.ui.quickAssistProcessors

以来：
5.1

描述：
此扩展点允许添加快速辅助处理器以在C / C ++编辑器中提供新的快速辅助。此扩展点支持enablement标记。要测试的属性是：
  • translationUnit：输入ITranslationUnit;应用快速助手的翻译单位
  • projectNatures：type Collection;当前项目的所有项目性质

    • 配置标记：

    <！ELEMENT 扩展 （ quickAssistProcessor +）>

    <！ATTLIST扩展

    点CDATA #REQUIRED

    id CDATA #IMPLIED

    名称CDATA #IMPLIED>

    • point - 目标扩展点的完全限定标识符
    • id - 扩展实例的可选标识符
    • name - 扩展实例的可选名称

    <！ELEMENT quickAssistProcessor （ 启用 ？）>

    <！ATTLIST quickAssistProcessor

    id CDATA #REQUIRED

    名称CDATA #IMPLIED

    class CDATA #IMPLIED

    requiredSourceLevel CDATA #IMPLIED>

    • id - 快速辅助处理器的唯一标识符。
    • name - 快速辅助处理器的本地化名称。
    • class - 实现此Quick Assist处理器的类的名称。该类必须是公共的并实现org.eclipse.cdt.ui.text。IQuickAssistProcessor与一个公共0参数构造函数。
    • requiredSourceLevel - 一个可选属性，用于指定此处理器所需的最小源合规性。

    例子：
    以下是快速辅助处理器贡献的示例：

    <extension point = “org.eclipse.cdt.ui.quickAssistProcessors” > <quickAssistProcessor id = “AdvancedQuickAssistProcessor” name = “Advanced Quick Assist Processor” class = “com.example。AdvancedQuickAssistProcessor“ > </ quickAssistProcessor> <enablement> <with variable = ”projectNatures“ > <iterate operator = ”或“ > <equals value = ”org.eclipse.cdt.core.comature“ /> <equals value = ”org。 eclipse.cdt.core.ccnature“ /> </ iterate> </ with> </ enablement> </ extension>

    API信息：
    贡献的类必须实现org.eclipse.cdt.ui.text.IQuickAssistProcessor
