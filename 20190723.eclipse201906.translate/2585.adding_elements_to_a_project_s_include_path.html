配置项目的PHP包含路径

将外部资源添加到项目的包含路径可使项目外部的资源可用于调试和内容辅助等操作。有关更多信息，请参阅PHP包含路径 。

此过程描述如何配置项目的包含路径。

配置项目的包含路径：

  1. 在Project Explorer视图中，右键单击所需项目，然后选择Include Path |配置包含路径... - 或 - 右键单击项目，然后选择属性|包括路径 。将出现项目的PHP Include Path属性页面。


    在此页面中，您可以配置以下内容：
    • 要在“包含路径”上配置源文件夹：
      1. 选择“源”选项卡。默认情况下，“包含路径”中的所有文件夹都将添加到“ 构建路径”中 。

        注意：如果项目根目录位于“包含路径”中，则项目中包含的所有文件夹和资源也将位于“包含路径”中。要仅添加某些文件夹，请从“包含路径”中删除项目根目录，并仅添加所需的文件夹。
      2. 要删除文件夹，请选择该文件夹并单击“ 删除” 。
      3. 要从项目中添加文件夹 ，请单击“ 添加文件夹”按钮，然后选择所需的文件夹。将显示一个提示，询问您是否还要将所选文件夹添加到构建路径 。建议构建路径上的资源与“包含路径”上的资源相匹配。单击是将文件夹（S）添加到您的构建路径或没有找到您的构建路径不会受到影响。
    • 要将工作区中的另一个项目添加到“包含路径”：
      1. 选择“项目”选项卡。
      2. 单击添加 。将出现“所需项目选择”对话框
      3. 选择要添加的项目，然后单击“ 确定” 。所选项目将添加到项目的“包含路径”中。
    • 要将库添加到“包含路径”：
      1. 选择“库选项卡”。
      2. 单击“ 添加库” 。将出现“添加库”对话框。
      3. 选择所需的库（如果可用），然后单击“ 确定” 。
    • 要将外部文件夹添加到“包含路径”：
      1. 选择“库选项卡”。
      2. 单击添加外部文件夹。将出现“添加包含路径”对话框。
      3. 浏览并选择所需的文件夹。
      4. 单击确定 。该文件夹将添加到项目的“包含路径”中。

        注意：该库是只读文件，无法进行编辑。
    • 要配置“包含路径”中元素的顺序：
      1. 选择“订单选项卡”。这决定了在require / include调用中搜索资源的顺序。有关搜索文件的顺序的详细信息，请参阅包含路径 。如果适用，建议元素的显示顺序与php.ini中的顺序相同。
      2. 如有必要，重新排列条目的顺序。为此，请选择一个元素，然后单击向上或向下移动它以在列表中移动它。
  2. 添加完所有元素并按正确顺序排列后，单击“ 确定” 。

所有选定的元素都将添加到项目的包含路径中。
