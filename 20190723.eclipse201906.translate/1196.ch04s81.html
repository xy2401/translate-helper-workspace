4.81. Time per word

Parameter Name Data Type Possible Values Default
Time per Word (in milliseconds) Integer   500
Table 4.82. TIME_PER_WORD

  • Use this parameter to define how log the text should be shown for.

  • The time is calculated per word.

  • Enter the value (in milliseconds) that you want to allow per word.

  • For 3 seconds, enter 3000.

  • The time per word is multiplied by the amount of words in your text to give the total amount of time the text is shown for.
