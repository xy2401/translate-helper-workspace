文件类型

标识符：
org.eclipse.team.core.fileTypes

以来：
2.0

描述：
此扩展点用于注册有关是否应将特定文件类型视为包含文本或二进制数据的信息。此信息对某些存储库提供程序很重要，因为它会影响数据的存储，比较和传输方式。

提供商可以为此扩展点提供扩展。不需要XML扩展声明之外的代码。

配置标记：

<！ELEMENT 扩展 （ fileTypes *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED>


<！ELEMENT fileTypes EMPTY>

<！ATTLIST文件类型

扩展CDATA #REQUIRED

type（text | binary）>

  • extension - 此贡献标识的文件扩展名。
  • type - “text”或“binary”之一，标识与扩展名值匹配的文件的内容。

例子：
以下是fileTypes扩展的示例：

<extension point = “org.eclipse.team.core.fileTypes” > <fileTypes extension = “txt” type = “text” /> </ extension>
