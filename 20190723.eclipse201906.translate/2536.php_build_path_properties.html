PHP构建路径属性

PHP Build Path Properties页面允许您配置项目的PHP Build Path 。

要访问PHP Build Path Properties页面，请在Project Explorer视图中右键单击PHP项目，然后选择Properties | PHP构建路径属性 - 或 - 选择项目，然后从菜单栏转到项目|属性| PHP构建路径属性。

有关更多信息，请参阅配置项目的PHP构建路径 。
