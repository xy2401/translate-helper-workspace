5.3.4. The Component Names Browser

The Component Names Browser (Figure 5.5, “Component Name Browser”) is by default in the lower left-hand corner of the Specification Perspective, behind the Test Case Browser. In this browser, you can see component names used in this Project and in other Projects. You can rename, add, delete and merge component names and can also find where you have used component names in your test.

You can filter in the Component Names Browser using the field at the top. Use star * as a wild card.
