添加默认RenderKit ID

render-kit元素表示应在指定的render-kit-id下注册的具体RenderKit实现。如果未指定render-kit-id，则为默认RenderKit（RenderKitFactory）的标识符。假定DEFAULT_RENDER_KIT）。

要添加默认的RenderKit ID：

  1. 在J2EE透视图的Project Explorer中选择faces-config.xml 。

    将打开JSF应用程序配置编辑器。

  2. 选择编辑器的“ 其他”选项卡。

    出现“其他”页面。

  3. 展开Default RenderKit ID部分。

  4. 单击添加 。

    将出现“添加默认RenderKit ID”对话框。

  5. 键入RenderKit名称或从列表中选择它。

  6. 单击确定 。

    默认的RenderKit ID已添加到列表中。
