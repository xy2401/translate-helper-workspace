构建定义UI扩展

标识符：
org.eclipse.cdt.managedbuilder.ui.buildDefinitionsUI

以来：
8

描述：
此扩展点允许各种UI扩展的贡献，可以从org.eclipse.cdt.managedbuilder.core.buildDefinitions扩展中引用。例如，通过此扩展点，开发人员可以提供自定义JFace字段编辑器，然后可以通过其ID从org.eclipse.cdt.managedbuilder.core.buildDefinitions扩展的<option>元素引用该编辑器。

配置标记：

<！ELEMENT 扩展 （ fieldEditor *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT fieldEditor EMPTY>

<！ATTLIST fieldEditor

id CDATA #REQUIRED

class CDATA #REQUIRED>

此元素可用于提供自定义JFace字段编辑器，以在项目“属性”对话框UI中表示构建选项。通过此元素提供自定义字段编辑器是不够的 - 自定义字段编辑器还必须通过其ID来自org.eclipse.cdt.managedbuilder.core.buildDefinitions扩展的<option>元素。


  • id - 一个唯一标识符，用于从org.eclipse.cdt.managedbuilder.core.buildDefinitions扩展的<option>元素引用此字段编辑器。
  • class - 指定实现自定义字段编辑器的Java类。该类必须扩展JFace的FieldEditor抽象类，并实现ICustomBuildOptionEditor接口。
