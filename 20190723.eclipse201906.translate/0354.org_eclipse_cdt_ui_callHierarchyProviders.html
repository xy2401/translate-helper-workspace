Call Hierarchy Providers

Identifier:
org.eclipse.cdt.ui.callHierarchyProviders

Description:
This extension point allows implementers to add external calling points to the call hierarchy. The additional calling points should belong to the files not indexed by the CDT indexer.

Configuration Markup:

<!ELEMENT extension (provider)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT provider EMPTY>

<!ATTLIST provider

class CDATA #REQUIRED

id    IDREF #REQUIRED>
