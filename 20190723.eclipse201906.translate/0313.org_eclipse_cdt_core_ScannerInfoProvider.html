CExtensionGeneric

标识符：
org.eclipse.cdt.core.unspecified

描述：
ICConfigExtensionReference使用的扩展点的通用定义。此定义适用于一系列扩展点，不适用于任何特定扩展点。

配置标记：

<！ELEMENT 扩展 （ cextension ）>

<！ATTLIST扩展

id CDATA #IMPLIED

名称CDATA #IMPLIED

点CDATA #REQUIRED>

  • id - 扩展点的ID（简单ID）
  • name - 扩展点的名称
