使用JavaScript Mark Occurrences

JavaScript Mark Occurrences功能允许您查看JavaScript文件中引用变量，方法或类型的位置。

要使用Mark Occurrences：

站在文件中的变量，方法或类型。

将突出显示文件中引用元素的所有实例。

要配置标记出现次数设置：

  1. 要切换标记出现次数，请单击“切换标记出现次数”按钮在工具栏上 - 或 - 按Alt + Shift + O.
  2. 要进一步配置Mark Occurences首选项，请转到从Window |访问的JavaScript Mark Occurrences首选项页面偏好|网络| JavaScript |编辑| Mark Occurrences 。
  3. 选中“标记所选元素的出现次数”复选框以启用“标记出现次数”功能。
  4. 通过标记所需元素旁边的复选框，选择要标记的元素。
  5. 标记“选择更改时保留标记”复选框，一旦从所选元素移动光标，标记将继续显示。
  6. 单击“ 应用”以应用更改。

注意：

可以在Annotations首选项页面（ Window | Preferences | General | Editors | Text Editors | Annotations ）中配置标记出现的外观。
