添加PHP库

此过程描述如何将用户库添加到PDT。在您重新使用调试代码时，在项目或环境中包含用户库可以节省编写和调试代码的时间。

要向项目中添加其他PHP库：

  1. 转到窗口 | 偏好 | PHP | PHP库 。
  2. 在PHP Libraries Preferences页面中，单击New 。“新用户库”对话框将打开。
  3. 在“新建用户库”对话框中，输入用户库的名称
  4. 如果您希望将此库添加到整个环境而不是特定项目，请选中“添加到环境”复选框。
  5. 要应用更改，请单击确定 。

新的空库将添加到列表中。括号中的名称旁边表示它是由环境共享还是仅与项目相关。

添加PHP库会创建一个位置文件夹，您可以在其中放置包含预编写代码的外部文件。有关更多信息，请参阅向PHP库添加外部文件夹 。

注意：

在PHP首选项页面中添加用户库后，还必须将其添加到项目的PHP包含路径中，以使其可用。有关更多信息，请参阅配置项目的PHP包含路径 。
