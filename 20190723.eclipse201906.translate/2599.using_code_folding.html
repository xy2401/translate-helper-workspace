使用代码折叠

代码折叠折叠或“折叠”代码块的显示。

要启用代码折叠，请转到可从Window |访问的折叠首选项页面偏好| PHP |编辑|使用代码折叠 。

如果启用了代码折叠，则可以折叠的代码块旁边的注释栏中将显示减号。此外，默认情况下，某些元素将根据“折叠首选项”设置进行折叠。

折叠代码块：

  1. 站在类，函数或PHPDocBlock中。
  2. 单击编辑器左侧标记栏上的减号 。

第一行代码将保持可见，但其他行将不会显示。折叠指示器折叠代码时将出现在行尾，表示存在隐藏代码。

要暂时查看折叠代码，请将鼠标悬停在表示折叠块的加号上。折叠的代码将显示在浮动块中。

展开一段代码：

  1. 单击加号 。
  2. 折叠的代码将再次可见，折叠指示器将消失。

要查看折叠的范围：

  1. 将鼠标悬停在减号上 。
  2. 将从折叠的第一行到最后一行显示垂直线，指示其范围。
展开的功能折叠功能

折叠/展开嵌套函数：

  1. 单击嵌套函数的减号之一。低于此级别的所有级别都将折叠到其中。您可以继续折叠，直到所有级别都折叠到最顶层。
  2. 要展开嵌套函数，请单击加号 。折叠后的代码将以折叠后的顺序打开。

注意：

行号与代码一起折叠。折叠和展开不会更改行号，它只能隐藏/显示它们。

注意：

如果折叠的代码包含错误，则显示的窗口将在左侧和右侧注释栏上突出显示语法。
