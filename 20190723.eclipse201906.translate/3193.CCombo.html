CCombo

The theming of RWT CCombo.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The background color used for the text field and the drop-down list of a CCombo.

  • background-image

    Background gradient for ccombo text field.

  • border

    The border that is drawn around a CCombo.

  • border-bottom

    The bottom side border for CCombo.

  • border-left

    The left side border for CCombo.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for CCombo.

  • border-top

    The top side border for CCombo.

  • box-shadow

    A shadow effect for ccombo text field.

  • color

    The text color used for the text field and the drop-down list of a CCombo.

  • font

    The font used for the text field and the drop-down list of a CCombo.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Style constant for bordered behavior.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled CCombos.

  • :focused

    Applies to focused CCombos.

CCombo-Button

The drop-down button of RWT CCombo.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The background color of the drop-down button of the CCombo.

  • background-image

    Background image or gradient for CCombo drop-down button.

  • border

    The border of the drop-down button of the CCombo.

  • border-bottom

    The bottom side border of the CCombo drop-down button.

  • border-left

    The left side border of the CCombo drop-down button.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border of the CCombo drop-down button.

  • border-top

    The top side border of the CCombo drop-down button.

  • cursor

    CCombo button cursor.

  • width

    The width of the drop-down button of the CCombo.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [FLAT]

    Applies when style FLAT is set on the parent.

  • [RIGHT_TO_LEFT]

    Indicates that the CCombo is oriented right to left.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies when the button is disabled.

  • :hover

    Applies while the user hovers over the CCombo button.

  • :pressed

    Applies while a button is pressed.

CCombo-List-Item

Items of CCombo list.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Item background color

  • background-image

    Item background image/gradient

  • color

    Item text color

  • padding

    Defines the padding (i.e. the inner distance between border and content) for a DropDown item.

  • text-decoration

    Decoration for item text.

  • text-shadow

    Shadow for item text.

Applicable SWT Styles

none

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :even

    Applies to CCombo list items with an even index.

  • :hover

    Applies while the user hovers over the CCombo list item.

  • :selected

    Applies when CCombo list item is selected.

CCombo-FocusIndicator

The frame that is drawn when the ccombo is focused.

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    The fill-color of the frame.

  • border

    The border of the frame. 3D-styles inset, outset, etc are not supported.

  • border-bottom

    The bottom side border of the frame.

  • border-left

    The left side border of the frame.

  • border-right

    The right side border of the frame.

  • border-top

    The top side border of the frame.

  • margin

    The margin between the ccombos border and the frame.

  • opacity

    The opacity of the indicator. A value between 0 and 1.

Applicable SWT Styles

none

Applicable States

none
