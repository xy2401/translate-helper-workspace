增量观众

VIATRA Viewers Framework适用于为CPS演示者创建自定义视图。

使用声明性模式注释定义视图，这些注释指定视图中的项和边。视图可以是图形可视化，但是该技术也支持简单的UI元素，例如列表或树视图。在CPS演示器中，我们使用Zest Graph Visualization框架创建了视图。

可以在model/viewer和deployment/viewer包中的cps.queries项目中找到视图的定义。项与定义@Item注释，边缘与指定@Edge注解，而格式（如颜色）与描述@Format注释。

查看器示例

以下模式定义以下内容：

  • 为每个部署主机创建项目

  • 为每个部署应用程序创建项目

  • 在与部署主机和应用程序对象对应的项之间创建边

@Format（color =“＃FBFE00”）@ Item（item = host，label =“Host $ host.ip $”）pattern deploymentHostItem（host）{DeploymentHost（host）; } @Format（color =“＃996600”）@ Item（item = app，label =“App $ app.id $”）模式deploymentApplicationItem（app）{DeploymentApplication（app）; } @Edge（source = host，target = app，label =“apps”）模式deploymentHostApplicationEdge（host，app）{DeploymentHost.applications（host，app）; }
