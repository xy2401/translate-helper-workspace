浏览器

标识符：
org.eclipse.help.base.browser

以来：
3.0（最初在2.0版中添加为org.eclipse.help.ui.browser）

描述：
用于提供能够在给定URL处显示html文档的Web浏览器。

注意：从3.1开始，帮助系统使用workbench浏览器支持（请参阅org.eclipse.ui.browserSupport ）。本文档中描述的扩展点仅用于独立帮助。需要在工作台模式下工作的任何自定义浏览器适配器都需要移动到工作台扩展点。

配置标记：

<！ELEMENT 扩展 （ 浏览器 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 浏览器 （ 工厂类 ？）>

<！ATTLIST浏览器

id CDATA #REQUIRED

factoryclass CDATA #IMPLIED

名称CDATA #REQUIRED>

  • id - 浏览器的唯一ID。
  • factoryclass - 浏览器工厂的实现类。该类必须实现org.eclipse.help.browser。IBrowserFactory接口。可以省略该属性，并且可以替代地提供嵌套的工厂类元素。
  • name - 浏览器的名称（可翻译）。

<！ELEMENT factoryclass （ 参数 *）>

<！ATTLIST工厂级

class CDATA #REQUIRED>

  • class - 浏览器工厂的实现类。该类必须实现org.eclipse.help.browser。IBrowserFactory接口。

<！ELEMENT 参数 EMPTY>

<！ATTLIST参数

名称CDATA #REQUIRED

值CDATA #REQUIRED>

  • name - 传递给实现类的参数的名称
  • value - 传递给实现类的参数的值

例子：
以下是浏览器扩展点的示例用法：

<extension point = “org.eclipse.help.base.browser” > <browser id = “org.eclipse.myPlugin.myBrowserID” factoryclass = “org.eclipse.myPlugin.myPackage。MyFactoryClass“ name = ”我的浏览器“ > </ browser> </ extension>

API信息：
提供的工厂类必须实现org.eclipse.help.browser。IBrowserFactory接口。该接口中的方法确定工厂是否在给定系统上可用，即能够提供浏览器实例，以及创建实现IBrowser接口的浏览器实例。

提供的实施：
org.eclipse.help.base和org.eclipse.help.ui插件包含常见平台上的浏览器实现。其他插件可以提供不同的实现。在首选项中，用户可以从可用的浏览器中选择默认浏览器。
