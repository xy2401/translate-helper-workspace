测试运行监听器

标识符：
org.eclipse.jdt.junit.testRunListeners

以来：
2.1

描述：
扩展点以注册其他测试运行侦听器。测试运行监听器会收到有关测试运行执行情况的通知。

配置标记：

<！ELEMENT 扩展 （ testRunListener ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

目标扩展点的标准标识符


  • id - 目标扩展点的标准标识符

<！ELEMENT testRunListener EMPTY>

<！ATTLIST testRunListener

class CDATA #REQUIRED>

  • class - 扩展org.eclipse.jdt.junit的测试运行侦听器。TestRunListener（自3.6起）。 注意：也接受org.eclipse.jdt.junit。ITestRunListener，但不推荐使用该接口。

例子：
以下是测试运行监听器贡献的示例：


 <extension point="org.eclipse.jdt.junit.testRunListeners">
  <testRunListener class="com.example.SampleTestRunListener"/>
 </extension>

API信息：
测试运行监听器应该扩展org.eclipse.jdt.junit。TestRunListener（自3.6起）。在以前的版本中，他们也可以实现org.eclipse.jdt.junit。ITestRunListener接口，但现在已弃用。
