新页面向导

New Scout Page向导可用于创建新页面和相关类。要启动向导使用文件 ▸ 新建 ▸ 其他...或按Ctrl + N。

图1。新页面向导

在图1的情况下，在包资源管理器中选择了包org.eclipse.scout.apps.helloworld.client.helloworld 。然后，需要手动填写的唯一向导字段是“ 名称”字段。

通过单击“ 完成”按钮，将启动向导并创建指定的组件。

向导字段和默认值

根据IDE的当前上下文， 图1中的大多数字段将填充默认值。上下文可以从Package Explorer中选择的包或活动Java编辑器中的类派生。

源文件夹

用于创建页面的Maven客户端模块的源文件夹。默认值是Maven客户端模块中的src/main/java文件夹。

包

包含页面类的Java包。Scout SDK将尝试从当前上下文中猜测包名称，并为Maven共享模块派生匹配的包名称。

名称

页面类的名称。根据Scout约定，类名以后缀TablePage （对于AbstractPageWithTable子类）或NodePage （对于AbstractPageWithNodes ）结束。

超级

表格的超类。AbstractPageWithTable是默认值。

共享源文件夹

用于创建页面数据和服务接口的Maven共享模块的源文件夹。默认值是Maven共享模块中的src/main/java文件夹。

服务器源文件夹

用于创建服务实现的Maven服务器模块的源文件夹。默认值是Maven服务器模块中的src/main/java文件夹。

创建组件

在上面显示的图1示例中，Scout SDK将创建以下组件。

  • 在Maven模块helloworld.client中

    • 文件夹src/main/java的MyTablePage页面类和包org.eclipse.scout.apps.helloworld.client.helloworld

  • 在Maven模块helloworld.shared中

    • 文件夹src/main/java和包org.eclipse.scout.apps.helloworld.shared.helloworld的IMyService服务接口

    • 文件夹src/generated/java MyTablePageData页面数据类和包org.eclipse.scout.apps.helloworld.shared.helloworld

  • 在Maven模块helloworld.server中

    • 文件夹src/main/java的MyService实现和包org.eclipse.scout.apps.helloworld.server.helloworld
