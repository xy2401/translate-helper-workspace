RWT主题

标识符：
org.eclipse.rap.ui.themes

以来：
RAP 1.0

描述：
此扩展点用于注册自定义RWT主题。RWT主题用于自定义使用远程应用程序平台构建的应用程序的外观。要创建自定义主题，必须提供CSS文件。使用此文件，可以自定义一个或多个小部件的样式。可以在“RAP主题参考”中找到可主题小部件和属性的列表。也可以注册对现有RWT主题的贡献。主题贡献包含附加到现有主题的CSS规则。如果为同一主题注册了多个贡献，则附加这些贡献的顺序是不确定的。此扩展点的用户必须确保主题贡献中指定的规则不会干扰其他主题贡献中的规则。理想情况下，主题贡献仅包含自定义变体的规则。

配置标记：

<！ELEMENT 扩展 （（ 主题 | 主题 贡献 ）+）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 主题 EMPTY>

<！ATTLIST主题

id CDATA #REQUIRED

名称CDATA #IMPLIED

文件CDATA #IMPLIED>

RWT主题。


  • id - 将用于标识此主题的唯一名称
  • name - 主题的可选名称（当前未使用）
  • file - 包含主题定义的.css文件的相对路径

<！ELEMENT themecontribution EMPTY>

<！ATTLIST主题贡献

themeId IDREF #REQUIRED

文件CDATA #REQUIRED>

对现有主题的贡献。贡献的CSS规则附加到扩展主题的CSS规则。


  • themeId - 此扩展所贡献的主题的ID。使用org.eclipse.rap.rwt.theme。默认为默认主题做出贡献。
  • file - .css文件的相对路径，其中包含要附加到主题的其他规则

例子：
以下是具有新主题的主题扩展的示例：

<extension point="org.eclipse.rap.ui.themes">
  <theme
      id="my.application.aquablue"
      name="Aqua Blue Theme"
      file="aqua-blue.css" />
</extension>
以下是定义对现有主题的贡献的主题扩展的示例：

<extension point="org.eclipse.rap.ui.themes">
  <themeContribution
      themeId="my.application.aquablue"
      file="aqua-blue-additions.css" />
</extension>

提供的实施：
RAP演示插件org.eclipse.rap.design.example包含主题扩展的示例。
