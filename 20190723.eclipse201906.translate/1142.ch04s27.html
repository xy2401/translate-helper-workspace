4.27. Exists

Parameter Name Data Type Possible Values Default
EXISTS Boolean

  • true

  • false

 true
Table 4.28. EXISTS

  • Set this parameter to true if you expect the item/component to exist.

  • Set this parameter to false if you expect the item/component to not exist.
