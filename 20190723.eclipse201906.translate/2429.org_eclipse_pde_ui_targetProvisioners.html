目标供应商

不推荐使用此扩展点

标识符：
org.eclipse.pde.ui.targetProvisioners

以来：
3.3

描述：

此扩展点用于注册新的目标插件供应商。当用户尝试从PDE目标平台的首选项页面添加插件时，会列出每个插件供应商。所选供应商负责提供包含用户想要添加到目标平台的插件的目标位置。

不推荐使用此扩展点并计划删除。请改用org.eclipse.pde.ui.targetLocationProvisioners扩展点。

配置标记：

<！ELEMENT 扩展 （ 供应商 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 供应商 （ 描述 ？）>

<！ATTLIST供应商

id IDREF #REQUIRED

名称CDATA #REQUIRED

icon CDATA #IMPLIED

class CDATA #IMPLIED>

  • id - 供应者的唯一标识符
  • name - 供应商的人类可读名称
  • icon - 将用于直观表示供应商的图标的相对路径。
  • 不推荐使用的 类 - 为用户提供向导以通过供应商添加插件的类。此属性的值是实现org.eclipse.pde.ui.IProvisionerWizard 。

<！元素描述 （#PCDATA）>

供应商的人类可读描述



例子：

以下是扩展点的示例：


   <extension
         point="org.eclipse.pde.ui.targetProvisioners">
      <provisioner
            class="org.eclipse.pde.internal.ui.shared.target.InstallableUnitWizard"
            icon="icons/obj16/metadata_repo_obj.png"
            id="org.eclipse.pde.ui.InstallableUnit"
            name="%targetProvisioner.IU.name">
         <description>
            %targetProvisioner.IU.description
         </description>
      </provisioner>
   </extension>

API信息：
每个模板都必须提供一个实现org.eclipse.pde.ui.IProvisionerWizard界面。
