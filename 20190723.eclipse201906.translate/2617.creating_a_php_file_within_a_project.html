在项目中创建PHP文件

此过程描述如何在现有项目中创建新的PHP文件。

要在项目中创建新的PHP文件：

  1. 在Project Explorer视图中，选择要放置文件的项目。
  2. 右键单击并选择New | PHP文件 - 或 - 转到菜单栏上的文件，然后选择新建| PHP文件 。
  3. 将显示PHP文件创建对话框。
  4. 输入文件名，然后单击“ 下一步” 。
  5. 默认情况下将标记“使用PHP模板”复选框。这将创建带有“<？PHP？>“PHP标签。选中所需模板或取消选中该复选框以创建空白文件。
  6. 单击完成 。

您的文件将在编辑器中打开，并将显示在Project Explorer和Navigator视图中的项目中。
