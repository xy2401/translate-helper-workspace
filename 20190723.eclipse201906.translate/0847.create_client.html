使用Apache CXF从WSDL文档生成JAX-WS Web服务客户端

先决条件：

  1. 安装和配置Apache CXF运行时环境。有关详细信息，请参阅“ 要求和安装”页面。
  2. 创建动态Web项目并为Apache CXF Web服务开发启用它，如下所述： 为CXF Web服务创建项目 。
要从WSDL文档生成JAX-WS客户端：
  1. 选择从顶级菜单中选择Window> Open Perspective> Java EE切换到Java EE透视图。
  2. 选择打开Web服务客户端 文件>新建>其他...> Web服务>顶级菜单中的Web服务客户端 。
  3. Web服务页面：

    1. 输入将用于生成客户端的远程WSDL文件的URI。
    2. 选择Java Proxy作为客户端类型。
    3. 将客户端滑块移动到Develop客户端阶段。
    4. Web服务运行时：选择Apache CXF 2.x.
    5. 服务项目：在工作区中选择Apache CXF将生成代码的项目。
  4. 单击Next>转到第一个Apache CXF wsdl2java配置页面：
    1. 输出目录：指定生成代码的服务项目中的源文件夹。
    2. 包名称：键入要用于生成的代码的Java包名称。
    3. 将WSDL命名空间指定为包名称映射：可选择指定Java包名称的WSDL命名空间映射。
    4. 服务名称：指定从中生成代码的WSDL服务的名称。缺省情况是为WSDL文档中的每个服务生成代码。
    5. 绑定文件：指定一个或多个JAX-WS或JAXB绑定文件。
  5. 单击Next>转到第二个Apache CXF wsdl2java配置页面，该页面允许您覆盖wsdl2java默认首选项：
  6. 单击“ 完成”以生成客户端。

生成的客户端将包含一个main（），它在运行时将连接到远程服务并调用该远程服务上的每个操作。

在运行客户端之前，在Java编辑器中打开客户端，并编辑用于调用服务操作的生成参数。

要运行客户端，请在Project Explorer中右键单击客户端，然后选择Run As> Java Application 。
