编辑菜单

“编辑”菜单包含常规文本编辑功能，以及与编辑代码特别相关的任务和书签等功能。

“编辑”菜单中提供的选项包括：

名称捷径描述
撤消文本更改按Ctrl + Z撤消活动文件中的最后一次文本编辑
重做文本更改按Ctrl + Y重做活动文件中的最后一次文本编辑。
切按Ctrl + X剪切选定的文本部分。
复制按Ctrl + C将选定的文本部分复制到剪贴板。
糊按Ctrl + V粘贴剪贴板中的文本。
删除删除删除选定的文本部分。
全选按Ctrl + A选择文件中的所有文本。
查找/替换按Ctrl + F查找并替换活动文件中的文本。
找下一个按Ctrl + K转到编辑器中所选项目的下一个实例。
找到上一个按Ctrl + Shift + K转到编辑器中所选项目的下一个实例。
增量查找下一个/上一个按Ctrl + J Ctrl + Shift + J.在活动文件中的光标之后/之前查找字符串。
要使用此功能，请按Ctrl + J和所需字符串的前几个字母。
相关文本将在文件中突出显示。
在此模式下，上下光标键可用于在匹配之间导航。
按左，右，回车或退出可以取消搜索。
添加书签...在您的脚本中插入书签。
书签用于轻松导航到脚本中的特定部分。
您可以为每个书签附加一个描述性名称，稍后可以在书签旁边的工具提示中看到该名称。
书签由书签图标表示在左边缘。
打开书签视图（ 窗口|显示视图|其他|一般|书签 ）在现有书签之间导航。
添加任务...将任务插入到脚本中。
任务用作程序员的提醒。为了最大限度地提高效率，任务应放在将要实施操作的代码部分旁边。
打开任务视图（ 窗口|显示视图|任务 ）在现有任务之间导航。
显示工具提示说明显示将出现在当前光标位置的悬停值。显示的对话框是可滚动的，不会缩短说明。
单词完成Alt + /完成输入的单词。输入单词的前几个字母，然后按Alt + /键完成单词。
完成所有当前打开的编辑器或缓冲区中出现的单词的前缀。
快速解决按Ctrl + 1显示“问题”视图中问题的可能快速修复选项。
要使用此选项，请首先在“问题”视图中选择问题。
注意：此选项并非始终可用。
设置编码更改用于在活动编辑器中读取和写入文件的文件编码。
