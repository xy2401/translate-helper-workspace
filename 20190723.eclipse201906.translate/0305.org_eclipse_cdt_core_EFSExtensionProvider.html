EFSFileSystemUtilityProvider

标识符：
org.eclipse.cdt.core。EFSFileSystemUtilityProvider

以来：
5.2

描述：
此扩展点允许提供一个类，该类以一组实用程序方法的提供者的形式为EFS文件系统提供补充支持。特别是，该类可以从文件系统及其资源中提取有意义的信息，并提供与这些资源相关的有用操作。

配置标记：

<！ELEMENT 扩展 （ EFSExtensionProvider ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT EFSExtensionProvider EMPTY>

<！ATTLIST EFSExtensionProvider

方案CDATA #REQUIRED

class CDATA #REQUIRED>

EFS扩展提供程序的实例。EFS扩展提供程序映射到与它们提供支持的EFS文件系统相对应的URI方案。CDT会查询这些扩展，以便从文件系统中提取路径信息并操纵文件系统资源。


  • scheme - 此实用程序提供支持的EFS文件系统的URI方案。
  • class - 从org.eclipse.cdt.core扩展的Java类。EFSFileSystemSupplementarySupportProvider为文件系统提供支持。

提供的实施：
org.eclipse.cdt.core提供了一个默认实现。EFSFileSystemUtilityProvider假定给定文件系统的URI直接映射到物理文件系统中的资源，并且URI的路径组件直接表示物理文件系统中文件的绝对路径。客户端可以扩展此实现并根据需要覆盖其行为。
