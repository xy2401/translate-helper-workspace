帮助服务器

标识符：
org.eclipse.help.base.server

以来：
3.4

描述：
允许Eclipse使用Equinox服务器以外的Web服务器来托管帮助系统。

配置标记：

<！ELEMENT 扩展 （ 服务器 ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 服务器 EMPTY>

<！ATTLIST服务器

class CDATA #REQUIRED>

  • class - 实现应用程序服务器的启动和停止方法的类，负责初始化org.eclipse.help.webapp插件中的servlet和资源。

例子：
[在此输入扩展点用法示例。]

API信息：
必须提供一个扩展HelpServer的类，它将启动和停止应用程序服务器并初始化org.eclipse.help.webapp中的servlet和资源。
