获取脚本工厂

标识符：
org.eclipse.pde.build.fetchFactories

以来：
3.2

描述：
此扩展点提供工厂，用于在PDE构建获取过程（eclipse.fetch Ant任务）期间构建特定于存储库的获取脚本。

配置标记：

<！ELEMENT 扩展 （ 工厂 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT factory EMPTY>

<！ATTLIST工厂

id CDATA #REQUIRED

class CDATA #REQUIRED>

定义eclipse.fetch Ant任务中使用的获取工厂。


  • id - 标识存储库的标识。此值用于映射文件。
  • class - 工厂实现。它必须实现org.eclipse.pde.build.IFetchFactory 。

例子：

<extension point="org.eclipse.pde.build.fetchFactories">
    <factory
        class="org.eclipse.pde.internal.build.fetch.CVSFetchTaskFactory"
        id="CVS"/>
    <factory
        class="org.eclipse.pde.internal.build.fetch.COPYFetchTasksFactory"
        id="COPY"/>
</extension>

API信息：
请参阅org.eclipse.pde.build.IFetchFactory接口。

提供的实施：
PDE Build提供了以下提取任务工厂。
  • CVS获取任务factroy用于从CVS存储库中获取功能和插件。
  • COPY fetch任务工厂，用于从任意文件系统位置复制功能和插件。
