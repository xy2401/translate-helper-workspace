语义突出显示

标识符：
org.eclipse.cdt.ui.semanticHighligher

以来：
8.2

描述：
此扩展点允许扩展有助于语义突出显示。

扩展指定突出显示的优先级，它决定了突出显示的调用顺序。

此扩展点支持enablement标记。要测试的属性是：

  • projectNatures：type Collection;当前项目的所有项目性质
  • languageId：type String;如果ILanguage.getId在令牌的ITranslationUnit上的结果

    • 贡献的突出显示将显示在“C / C ++ - 编辑器 - 语法着色”首选项页面的代码树中。

    配置标记：

    <！ELEMENT 扩展 （ semanticHighlighting +）>

    <！ATTLIST扩展

    点CDATA #REQUIRED

    id CDATA #IMPLIED

    名称CDATA #IMPLIED>

    • point - 目标扩展点的完全限定标识符
    • id - 扩展实例的可选标识符
    • name - 扩展实例的可选名称

    <！ELEMENT semanticHighlighting （ 启用 ？）>

    <！ATTLIST语义高亮显示

    id IDREF #REQUIRED

    名称CDATA #IMPLIED

    class CDATA #REQUIRED

    优先级CDATA #REQUIRED

    preferenceKey CDATA #REQUIRED

    displayName CDATA #REQUIRED

    defaultTextColor CDATA #IMPLIED

    defaultBold（true | false）

    defaultItalic（true | false）

    defaultStrikethrough（true | false）

    defaultUnderline（true | false）

    defaultEnabled（true | false）>

    • id - Quick Fix处理器的唯一标识符。
    • name - Quick Fix处理器的本地化名称。
    • class - 实现此语义突出显示的类的名称。该类必须是公共的并实现org.eclipse.cdt.ui.text.ISemanticHighlighter 。
    • 优先级 - 优先级决定突出显示突出显示令牌的顺序。较低的值更重要。org.eclipse.cdt.internal.ui.editor.SemanticHighlightings.loadBuiltInSemanticHighlightings 。
    • preferenceKey - 唯一标识首选项存储中突出显示设置的键。
    • displayName - “首选项”窗口中为突出显示器显示的名称。
    • defaultTextColor - 提供的突出显示的默认文本颜色。该值必须是以逗号分隔的整数RGB值（0-255）。例如，“127,0,85”。默认为“0,0,0”（黑色）。
    • defaultBold - 默认为false 。
    • defaultItalic - 默认为false 。
    • defaultStrikethrough - 默认为false 。
    • defaultUnderline - 默认为false 。
    • defaultEnabled - 默认为false 。

    例子：
    以下是Semantic Highligher贡献的示例：

    
<extension
    point="org.eclipse.cdt.ui.semanticHighlighting"
    name="%extensionName"
    id="com.example.ui.semanticHighlightings">
    <semanticHighlighting
        id="com.example.ui.keywordHighlighting"
        priority="5"
        class="com.example.internal.ui.ExampleHighlighting"
        preferenceKey="example-keywords"
        displayName="%exampleHighlighting.displayName"
        defaultTextColor="127,0,85"
        defaultBold="true"
        defaultEnabled="true">
        <enablement>
            <with variable="projectNatures">
                <iterate operator="or">
                    <equals value="org.eclipse.cdt.core.ccnature"/>
                </iterate>
            </with>
            <with variable="languageId">
             <or>
              <equals value="org.eclipse.cdt.core.g++"/>
              <equals value="org.eclipse.cdt.core.gcc"/>
             </or>
            </with>
        </enablement>
    </semanticHighlighting>
</extension>

    API信息：
    贡献的类必须实现org.eclipse.cdt.ui.text.ISemanticHighlighter
