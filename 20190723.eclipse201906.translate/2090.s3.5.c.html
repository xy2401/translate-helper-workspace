§3.5。（c） 价值映射

可以映射值，类似于纯方法绑定中的参数映射（第3.2节 ）。这种映射可用于建立上述要求的兼容性。
在get和set绑定中，基本侧值由字段的名称表示（下面的第2行和第4行）。

1 
整数getValue（） - > 获取 int val
2 
  与 {result < - new Integer（val）}
3 
void setValue（Integer i） - > set int val
4 
  与 {i.intValue（） - > val}
