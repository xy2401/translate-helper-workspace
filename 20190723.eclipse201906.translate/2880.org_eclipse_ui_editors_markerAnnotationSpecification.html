标记注释规范

标识符：
org.eclipse.ui.editors.markerAnnotationSpecification

以来：
3.0（最初名为org.eclipse.ui.workbench.texteditor.markerAnnotationSpecification）

描述：
此扩展点用于定义标记的表示属性。可以使用org.eclipse.ui.texteditor.MarkerAnnotationPreferences 。使用org.eclipse.ui.texteditor.AnnotationPreferenceLookup用于获取给定注释的注释首选项。

请注意，如果扩展只包含以下四个属性以及所需的MarkerAnnotationPreferences.getAnnotationPreferences则只会从annotationType （因此包含在首选项页面中）返回：

  • colorPreferenceKey
  • colorPreferenceValue
  • overviewRulerPreferenceKey
  • textPreferenceKey

允许扩展另一个注释首选项的注释首选项类型覆盖已在父首选项规范中定义的属性，但这些属性将无法从首选项页面访问。

配置标记：

<！ELEMENT 扩展 （ 规范 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 规范 EMPTY>

<！ATTLIST规范

annotationType IDREF #REQUIRED

colorPreferenceKey CDATA #IMPLIED

overviewRulerPreferenceKey CDATA #IMPLIED

verticalRulerPreferenceKey CDATA #IMPLIED

textPreferenceKey CDATA #IMPLIED

标签CDATA #IMPLIED

highlightPreferenceKey CDATA #IMPLIED

colorPreferenceValue CDATA #IMPLIED

presentationLayer CDATA #IMPLIED

overviewRulerPreferenceValue（true | false）

verticalRulerPreferenceValue（true | false）

textPreferenceValue（true | false）

highlightPreferenceValue（true | false）

contributionsToHeader（true | false）

showInNextPrevDropdownToolbarActionKey CDATA #IMPLIED

showInNextPrevDropdownToolbarAction（true | false）

isGoToNextNavigationTargetKey CDATA #IMPLIED

isGoToNextNavigationTarget（true | false）

isGoToPreviousNavigationTargetKey CDATA #IMPLIED

isGoToPreviousNavigationTarget（true | false）

icon CDATA #IMPLIED

quickFixIcon CDATA #IMPLIED

symbolicIcon（错误|警告|信息|任务|书签）

annotationImageProvider CDATA #IMPLIED

textStylePreferenceKey CDATA #IMPLIED

textStylePreferenceValue（BOX | DASHED_BOX | IBEAM | NONE | PROBLEM_UNDERLINE | SQUIGGLES | UNDERLINE）

includeOnPreferencePage（true | false）“true”>

  • annotationType - 注释类型
  • colorPreferenceKey - 必须提供颜色首选项键，否则此注释类型将不会包含在从MarkerAnnotationPreferences.getAnnotationPreferences（）返回的List中，因此不会显示在首选项中
  • overviewRulerPreferenceKey - 必须提供概述标尺首选项键，否则此注释类型将不会包含在从MarkerAnnotationPreferences.getAnnotationPreferences（）返回的List中，因此不会显示在首选项中
  • verticalRulerPreferenceKey - 垂直标尺首选项中显示的首选项键。从：3.0
  • textPreferenceKey - 必须提供文本首选项键，否则此注释类型将不会包含在从MarkerAnnotationPreferences.getAnnotationPreferences（）返回的List中，因此不会显示在首选项中
  • label - 要在UI中使用的标签
  • highlightPreferenceKey - 用于在文本中突出显示的首选项键。从：3.0
  • colorPreferenceValue - 必须提供颜色首选项值，否则此注释类型将不会包含在从MarkerAnnotationPreferences.getAnnotationPreferences（）返回的List中，因此不会显示在首选项中
  • presentationLayer - 将在其中绘制标记注释的图层的默认值。注释被认为位于图层，并将从第0层向上绘制。因此，将在层4-0处的所有共同定位的注释之上绘制层5处的注释。
  • overviewRulerPreferenceValue - 默认值，用于指示此标记注释是否显示在概览标尺中
  • verticalRulerPreferenceValue - 在垂直标尺中显示的默认值。从：3.0
  • textPreferenceValue - 表示此标记注释是否显示在文本中的默认值
  • highlightPreferenceValue - 文本中突出显示的默认值。从：3.0
  • contributionsToHeader - 默认值，用于指示此标记注释是否有助于概述标尺的标题摘要
  • showInNextPrevDropdownToolbarActionKey - 下一个/上一个下拉工具栏操作中可见性的首选项键。从：3.0
  • showInNextPrevDropdownToolbarAction - 下一个/上一个下拉工具栏操作中可见性的默认值。从：3.0
  • isGoToNextNavigationTargetKey - 转到下一个导航启用的首选项键。从：3.0
  • isGoToNextNavigationTarget - 转到下一个导航启用的默认值。从：3.0
  • isGoToPreviousNavigationTargetKey - 转到上一个导航启用的首选项键。从：3.0
  • isGoToPreviousNavigationTarget - 转到上一个导航启用的默认值。从：3.0
  • icon - 要为此注释类型的注释绘制的图标的路径。如果快速修复可用且指定了“quickFixIcon”属性，则不会使用此图标。
  • quickFixIcon - 为快速修复的注释绘制快速修复图标的路径。自：3.2
  • symbolicIcon - 应该绘制的图像的符号名称，用于表示此注释类型的注释。仅当没有为此注释类型指定垂直标尺图标时才使用该图像。可能的值有：“错误”，“警告”，“信息”，“任务”，“书签”。
  • annotationImageProvider - 实现org.eclipse.ui.texteditor.IAnnotationImageProvider
  • textStylePreferenceKey - 文本修饰属性的首选项键。从：3.0
  • textStylePreferenceValue - “在文本中显示”装饰样式的默认值。自：3.0（DASHED_BOX自3.3起，PROBLEM_UNDERLINE：自3.4起）
  • includeOnPreferencePage - 定义是否应通过标准注释首选项页面配置此注释类型。默认为true。
