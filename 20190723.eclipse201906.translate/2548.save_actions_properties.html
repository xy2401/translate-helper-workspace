保存操作属性

“保存操作属性”页面允许您在每次保存时从文件中删除尾随空格。

要访问“保存操作属性”页面，请在“项目资源管理器”视图中右键单击PHP项目，然后选择“属性”|资源 - 或 - 选择项目，然后从菜单栏转到项目|属性|保存操作属性。

要为项目配置“保存操作属性”：

  1. 选中“启用项目特定设置”复选框。
  2. 根据您的喜好配置设置。有关可用设置的详细信息，请参阅保存操作首选项 。
  3. 单击“确定”以应用设置。

默认保存操作可以在“保存操作首选项”页面中设置所有项目的属性（通过单击“属性”页面上的“配置工作区设置”链接，转到“窗口”|“首选项”|“PHP”|“编辑器”|“保存操作” - 或 - 进行访问。）
