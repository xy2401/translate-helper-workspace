帮助标准提供商

标识符：
org.eclipse.help.criteriaProvider

以来：
3.5

描述：
支持定义独立于目录的标准。

配置标记：

<！ELEMENT 扩展 （ 提供者 ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 提供商 EMPTY>

<！ATTLIST提供商

class CDATA #IMPLIED>

定义条件提供程序类


  • class - 包含用于定义与主题或目录关联的条件值的方法的类。

API信息：
使用类AbstractCriteriaProvider允许动态定义条件。使用此扩展点定义的标准将与内容列表中定义的标准合并，如果定义了多个标准提供者，则每个标准提供者将对ITopic或IToc的标准做出贡献。
