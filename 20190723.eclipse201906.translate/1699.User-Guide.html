概观

Linux Tools项目为许多足以分析C / C ++应用程序的系统工具提供Eclipse接口。例如，Valgrind插件提供了分析应用程序内存使用情况的功能，而Perf插件允许最终用户获取有关各种功能或方法所用时间的详细信息。从用户的角度来看，用户想要分析C / C ++可执行文件，这可以通过配置文件作为..右键单击上下文菜单或通过配置文件配置...对话框或通过下一个找到的配置文件按钮到Eclipse窗口顶部的Debug和Run按钮。

支持存在：

  • Valgrind（memcheck，massif，cachegrind和helgrind子工具）
  • OProfile的
  • PERF
  • SystemTap（使用systemtap脚本获取调用图）
  • Gcov（代码覆盖率）
  • GProf使用
