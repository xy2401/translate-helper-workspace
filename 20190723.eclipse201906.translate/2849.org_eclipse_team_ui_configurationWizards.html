配置向导

标识符：
org.eclipse.team.ui.configurationWizards

描述：
此扩展点用于注册配置项目的方法。配置涉及项目与团队提供者的关联，包括初始化团队提供者所需的所有信息，包括用户名，密码和定位提供者所需的任何相关信息。

提供者可以为此扩展点提供扩展，并提供org.eclipse.team.ui.IConfigurationWizard收集必要的信息并配置项目。

配置标记：

<！ELEMENT 扩展 （ 向导 ？）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT 向导 EMPTY>

<！ATTLIST向导

名称CDATA #REQUIRED

icon CDATA #REQUIRED

class CDATA #REQUIRED

id CDATA #REQUIRED>

  • name - 配置向导中应显示的配置类型的名称。例子是“CVS”，“WebDAV”。
  • icon - 要在名称旁边的配置向导中显示的图标。
  • class - 实现org.eclipse.team.ui.IConfigurationWizard 。
  • id - 此扩展的唯一标识符。

例子：
以下是配置向导扩展的示例：


 <extension point="org.eclipse.team.ui.configurationWizards">
  <wizard 
   name="WebDAV"
   icon="webdav.gif"
   class="com.xyz.DAVDecorator"
   id="com.xyz.dav">
  </wizard>
 </extension>

API信息：
class属性的值必须表示实现org.eclipse.team.ui.IConfigurationWizard 。此接口支持在给定工作台和项目的情况下配置向导。

提供的实施：
插件org.eclipse.team.provider.examples.ui包含用于WebDAV和文件系统提供程序类型的IConfigurationWizard的示例实现。
