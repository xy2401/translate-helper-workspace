OT / J提升参与者

标识符：
org.eclipse.objectteams.otequinox.liftingParticipant

以来：
基于Eclipse 3.6的OTDT 0.7.0（来自Eclipse.org）。

描述：
允许客户端插件安装提升参与者，以便挂钩OT / J提升过程。这可以用于避免默认的按需角色创建，使得特定于应用程序的策略可以用于代表提升操作创建和初始化角色对象。

配置标记：

<！ELEMENT 扩展 （ liftingParticipant ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT liftingEarticipant EMPTY>

<！ATTLIST提升参与者

class CDATA #REQUIRED>

  • class - 实现org.objectteams.ILiftingParticipant

例子：
最初要求OT / JPA集成“提升参与者”的概念，其中可能需要从持久存储中检索角色，而不是默认的按需创建。

API信息：
在class的类属性命名必须实现org.objectteams.ILiftingParticipant接口。

提供的实施：
没有。
