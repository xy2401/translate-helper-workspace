变量编辑器

标识符：
org.eclipse.debug.ui.variableValueEditors

以来：
3.1

描述：
此扩展点提供了为特定调试模型提供变量值编辑器的机制。变量值编辑器用于编辑/保存变量值。当用户在“变量”视图中对变量调用“更改值...”或“分配值”（详细信息窗格）操作时，将提供贡献的org.eclipse.debug.ui.actions.IVariableValueEditor来更改值。

配置标记：

<！ELEMENT 扩展 （ variableValueEditor ）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT variableValueEditor EMPTY>

<！ATTLIST variableValueEditor

class CDATA #REQUIRED

modelId CDATA #REQUIRED>

  • class - org.eclipse.debug.ui.actions.IVariableValueEditor
  • modelId - 给定变量值编辑器适用的调试模型标识符。

例子：

   <extension
         point="org.eclipse.debug.ui.variableValueEditors">
      <variableEditor
         modelId="com.examples.myDebugModel"
         class="com.examples.variables.MyVariableValueEditor"/>
   </extension>

API信息：
属性类的值必须是实现接口org.eclipse.debug.ui.actions的Java类的完全限定名。IVariableValueEditor 。
