文本搜索引擎

标识符：
org.eclipse.search.textSearchEngine

以来：
3.2

描述：
扩展点允许注册用于搜索工作空间文件中的文本的文本搜索引擎。

配置标记：

<！ELEMENT 扩展 （ textSearchEngine +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT textSearchEngine EMPTY>

<！ATTLIST textSearchEngine

id CDATA #REQUIRED

标签CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - 用于区分不同的搜索引擎。
  • label - 在UI中显示搜索引擎时使用。
  • class - 实现org.eclipse.search.core.text的类。TextSearchEngine。

例子：
作为示例，这是用于提供搜索引擎的标记

<extension id = “MyEngine” point = “org.eclipse.search.textSearchEngine” > <textSearchEngine id = “com.windriver.textSearchEngine” label = “Wind River Search” class = “com.windriver.text。SearchEngine“ /> </ extension>

API信息：
贡献的类必须实现org.eclipse.search.core.text.TextSearchEngine
