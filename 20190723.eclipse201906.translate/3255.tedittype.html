Editing an existing type

To edit an existing type:
  1. In the Remote Systems view, right-click on a file subsystem, and select Work With > Named Types.
  2. On the Work With Named Types window, expand Object or Member.
  3. Select your defined type from the list. You cannot edit any predefined types.
  4. Make the required changes.
  5. Click Apply.
