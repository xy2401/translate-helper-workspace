BirtDateTime类

BirtDateTime类提供函数来确定两个日期或时间之间的时间量，添加或减去时间跨度以及获取日期的一部分。由于日期跨度只是数字，因此应用程序可以对它们使用常规数学运算：乘法，除法，加法，减法等。这个类是静态的。应用程序无法创建该类的实例。

BirtDateTime.addDay

此函数返回一个新日期，该日期是将给定天数添加到日期的结果。添加的天数可能为负数。在这种情况下，结果就好像从日期中减去了该数字。

句法

日期BirtDateTime.addDay（日期日期，编号n）

参数

日期

日期对象。表示开始日期的日期对象。

ñ

数字对象。添加到日期的天数。小数被视为整数，向下舍入到最接近的整数。

返回

日期对象。将指定天数添加到开始日期的日期。

例子

以下示例将日期添加到特定日期：

var startDate =“2008-12-31”;
var endDate;
endDate = BirtDateTime.addDay（startDate，1）; //返回2009年1月1日
endDate = BirtDateTime.addDay（startDate，-1）; //返回2008年12月30日

以下示例为InvoiceDate字段中的每个日期值添加15天：

BirtDateTime.addDay（row [“InvoiceDate”]，15）

BirtDateTime.addHour

此函数返回一个新日期，该日期是通过向日期添加给定的小时数而产生的。添加的小时数可能是负数。在这种情况下，结果就好像从日期中减去了该数字。

句法

日期BirtDateTime.addHour（日期日期，编号n）

参数

日期

表示开始日期的日期对象。如果开始日期没有时间值，则该函数假定时间是午夜（00:00:00）。

ñ

数字对象。要添加到日期的小时数。

返回

日期对象。将指定的小时数添加到开始日期的日期。

例子

以下示例为特定日期和时间添加小时数：

var startDate =“2008-12-31 15:00:00”;
var endDate;
endDate = BirtDateTime.addHour（startDate，8）; //返回2008年12月31日下午11:00
endDate = BirtDateTime.addHour（startDate，-8）; //返回2008年12月31日上午7:00

以下示例为StartTime字段中的每个日期值添加36小时：

BirtDateTime.addHour（row [“StartTime”]，36）

BirtDateTime.addMonth

此函数返回一个新日期，该日期是在给定日期添加给定月数的结果。添加的月数可能是负数。在这种情况下，结果就好像从日期中减去了该数字。

此函数始终返回有效日期。如有必要，将结果日期的日期部分向下调整到结果年份中所得月份的最后一天。例如，如果将1个月添加到2008年1月31日，则addMonth（）将返回2008年2月29日（2008年是闰年），而非2008年2月31日。

句法

日期BirtDateTime.addMonth（日期日期，编号n）

参数

日期

日期对象。表示开始日期的日期对象。

ñ

数字对象。要添加到日期的月数。小数被视为整数，向下舍入。

返回

日期对象。将指定的月数添加到开始日期的日期。

例子

以下示例将特定日期的月份添加到：

var startDate =“2008-12-31”;
var endDate;

endDate = BirtDateTime.addMonth（startDate，-1）; //返回2008年11月30日

以下示例为InvoiceDate字段中的每个日期值添加10个月：

BirtDateTime.diffDay

此函数返回两个日期之间的天数。通过从第二个日期的日期值减去第一个日期的日期值来计算一天。例如，如果第一个日期是2008-12-30 23:59:59而第二个日期是2008-12-31 00:00:00，则该函数返回1天，即使只有一秒将两个日期分开。

句法

整数BirtDateTime.diffDay（Date date1，Date date2）

参数

DATE1

表示时间跨度开始的日期对象。

DATE2

表示时间跨度结束的日期对象。

返回

整数。两个日期之间的天数。

例子

以下示例返回特定日期之间的天数：

BirtDateTime.diffDay（“2009-01-01”，“2009-04-15”）//返回104

以下示例返回从当前日期到2009年底的天数：

BirtDateTime.diffDay（BirtDateTime.today（），“2009-12-31”）

以下示例返回InvoiceDate和PaymentDate字段中值之间的天数：

BirtDateTime.diffDay（row [“InvoiceDate”]，row [“PaymentDate”]）

此函数返回两个日期之间的小时数。通过从第二个日期的小时值减去第一个日期的小时值来计算小时。例如，如果第一个日期是2009-01-31 1:30:00，第二个日期是2009-01-31 2:00:00，则该函数返回1小时，即使只有30分钟将两个日期分开。如果日期没有时间值，则该函数假定时间是午夜（00:00:00）。

整数BirtDateTime.diffHour（Date date1，Date date2）

参数

DATE1

表示时间跨度开始的日期对象。

DATE2

表示时间跨度结束的日期对象。

此函数返回1到4之间的数字，表示给定日期所在的季度。第一季度从1月1日开始。

句法

整数BirtDateTime.quarter（日期日期）

参数

日期

从中获取季度数的日期对象。

返回

整数。给定日期的季度数。

例子

以下示例显示了特定日期返回的季度数：

BirtDateTime.quarter（“2009-02-15”）//返回1
BirtDateTime.quarter（“2009-07-10”）//返回3

以下示例返回CloseDate字段中每个值的季度数：

BirtDateTime.quarter（row [“CloseDate”]）

以下示例返回当前日期的季度编号：

BirtDateTime.quarter（（BirtDateTime.today（））

BirtDateTime.weekDay

此函数返回给定日期的星期几。根据您指定的选项，该函数将星期几作为数字，完整的工作日名称或缩写的工作日名称返回。工作日名称是特定于语言环境的。

句法

string BirtDateTime.weekDay（日期，整数选项）

参数

日期

日期对象，从中获取星期几。

选项

整数。可选的。表示要返回的格式的数字。使用以下值之一：
1表示从1（星期日）到7（星期六）的数字
2从1（星期一）到7（星期日）获取数字
3从0（星期一）到6（星期日）获取数字
4获得完整的工作日名称，例如，星期三
5获得缩写的工作日名称，例如Wed

如果省略此参数，则该函数采用选项1。

返回

串。给定日期的星期几。

例子

以下示例显示特定日期返回的工作日值：

BirtDateTime.weekDay（“2009-02-02”）//返回2
BirtDateTime.weekDay（“2009-02-02”，2）//返回1
BirtDateTime.weekDay（“2009-02-02”，4）//返回星期一

以下示例返回当前日期的星期几：

BirtDateTime.weekDay（（BirtDateTime.today（））
