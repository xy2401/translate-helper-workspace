Swing Designer快速入门


 

  1. 使用Swing Designer 类向导创建第一个GUI窗口或编辑现有窗口 。
     
  2. 为您的窗口选择合适的布局管理器 。
     
  3. 从Palette添加各种组件，并使用Property Pane编辑它们的属性。
     
  4. 可视化创建和编辑窗口的菜单栏 。
     
  5. 将事件处理程序添加到各种窗口小部件以提供行为。
     
  6. 如果创建新的JPanel ，则可以使用“ 选择组件”命令将其嵌入到另一个窗口中。
     
  7. 您可以通过右键单击并选择Run As> Java Application来测试启动窗口。
     
  8. 国际化您的应用程序并管理多个区域设置。
