5.3.2.4。by按定位器检查文本

  • 使用此操作可基于其定位器检查元素的文本。

  • 该操作不需要组件名称，也不需要对象映射 - 根据您输入的定位器找到AUT中的元素。

  • 例如，您可以输入如下所示的xpath定位器：

    XPath的= ...

  • 您可以输入这样的CSS定位器：

    CSS = ...

    您很可能希望在整个定位器周围使用单引号来取消字符串中的任何特殊字符。

  • 您可以使用不同类型的定位器，并可以使用网站检查工具找到组件的定位器。

  • 参数：

    • 定位器

    • 文本

    • 操作者
