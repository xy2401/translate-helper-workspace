登录配置提供程序

标识符：
org.eclipse.equinox.security.loginConfigurationProvider

描述：
登录配置提供程序是一个类，它接受登录配置名称并返回组成该登录配置的登录模块列表，包括与该登录模块关联的各种属性。配置提供程序必须扩展javax.security.auth.login。配置类并实现getAppConfigurationEntry（）方法。

配置标记：

<！ELEMENT 扩展 （ loginConfigurationProvider ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #IMPLIED>

扩展点描述符。



<！ELEMENT loginConfigurationProvider EMPTY>

<！ATTLIST loginConfigurationProvider

class CDATA #REQUIRED>

描述从中加载命名登录配置的特定配置提供程序的条目。默认情况下，平台支持通过扩展点提供的loginmodule。有关更多信息，请参阅org.eclipse.equinox.security.loginConfiguration的定义。


  • class - 登录配置的类，它必须是javax.security.auth.login的子类。配置 。

例子：
 <extension
         id="accountLoginConfigurationProvider"
         point="org.eclipse.equinox.security.loginConfigurationProvider">
      <loginConfigurationProvider class="org.eclipse.equinox.accounts.internal.AccountsLoginConfigProvider"/>
   </extension>

API信息：
JAAS API是Java Runtime Environment 1.4及更高版本的一部分。

提供的实施：
平台中不包含公共登录配置提供程序。
