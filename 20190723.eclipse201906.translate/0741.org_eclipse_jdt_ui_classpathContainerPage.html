Classpath容器条目

标识符：
org.eclipse.jdt.ui.classpathContainerPage

描述：
此扩展点允许添加向导页面以创建或编辑类路径容器条目。

配置标记：

<！ELEMENT 扩展 （ classpathContainerPage +）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT classpathContainerPage EMPTY>

<！ATTLIST classpathContainerPage

id IDREF #REQUIRED

名称CDATA #IMPLIED

class CDATA #IMPLIED>

  • id - 标识将使用此页面的类路径容器。id对应于容器类路径条目的路径的第一个段，并匹配org.eclipse.jdt.core.classpathContainerInitializer引用的id。
  • name - 选择新容器时使用的类路径容器的名称。此属性应为已翻译的字符串。
  • class - 实现此容器页面的类的名称。该类必须是公共的并实现org.eclipse.jdt.ui.wizards.IClasspathContainerPage 。

例子：
以下是类路径条目容器页面的示例：


 <extension point="org.eclipse.jdt.ui.classpathContainerPage">
  <classpathContainerPage
   id="com.example.myplugin.myContainerId"
   name="JRE System Libraries"
   class="com.example.NewJDKEntryPage">
  </classpathContainerPage>
 </extension>
