搜索菜单

允许您在工作区中搜索文本或PHP元素。

“搜索”菜单中提供的选项包括：

名称捷径描述
搜索按Ctrl + H打开PHP搜索对话框。
PHP Search使您能够找到PHP类，函数和常量的声明。
有关更多信息，请参阅搜索PHP元素 。
文件打开“文件搜索”对话框。
文件搜索使您可以在工作区中的所有文件中查找文本。
要运行文件搜索或替换：
  • 输入要搜索的所需文本。
  • 选择要搜索的文件类型。单击“选择”以获取文件类型扩展的完整列表。
  • 选择是否搜索：
    1. 工作区 - 整个工作区。
    2. 所选资源 - 在打开“源”对话框之前，在Project Explorer视图中选择这些资源。
    3. 封闭项目 - 所选资源所在的项目。
    4. 工作集 - 单击“选择”以选择所需的工作集。
  • 要搜索字符串，请单击“搜索”。搜索结果将显示在“搜索”视图中。要替换字符串，请单击“替换”。“替换”对话框将打开。

注意

默认情况下，“PHP搜索”对话框将使用“文件搜索”对话框进行选项卡。要使“PHP搜索”对话框不可用，请单击“文件搜索”对话框中的“自定义”，并取消选中“PHP搜索”对话框选项。

文本Ctrl + Alt + G工作区 - 在工作区中搜索在编辑器中选择的文本。
项目 - 在当前活动项目中搜索在编辑器中选择的文本。
文件 - 在当前活动文件中搜索在编辑器中选择的文本。
工作集 - 搜索当前活动工作集以查找在编辑器中选择的文本。
