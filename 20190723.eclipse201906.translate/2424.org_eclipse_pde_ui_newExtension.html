扩展向导

标识符：
org.eclipse.pde.ui.newExtension

描述：
此扩展点应用于提供将用于在PDE插件清单编辑器中创建和编辑新扩展的向导。向导可以同时创建一个或多个扩展，以及实现这些扩展所需的代码。如果专门为特定扩展点创建了贡献向导，则建议还注册匹配的编辑器向导。在清单文件中创建扩展点后，此向导将用于编辑清单编辑器中的扩展点。

配置标记：

<！ELEMENT 扩展 （ 向导 | 类别 | editorWizard ）*>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 向导 （ 描述 ？）>

<！ATTLIST向导

id IDREF #REQUIRED

名称CDATA #REQUIRED

icon CDATA #IMPLIED

class CDATA #IMPLIED

availableAsShortcut（true | false）

类别IDREF #IMPLIED

模板IDREF #IMPLIED>

一个向导，可用于从插件清单编辑器中创建新扩展


  • id - 将用于标识此向导的唯一名称。
  • name - 将在此向导的UI表示中使用的可翻译名称。
  • icon - 将用于直观表示向导的图标的相对路径。
  • class - 实现org.eclipse.pde.ui的类的完全限定名称。IExtensionWizard界面。此属性与template属性互斥。
  • availableAsShortcut - 如果该项为true ，则此向导将出现在菜单栏和工具栏的快捷菜单中。
  • category - 一个可选的id，使该向导成为先前定义的类别的成员。如果类别是分层的，则应使用“/”作为分隔符指定父类别的完整路径。
  • template - 使用扩展点org.eclipse.pde.ui.templates在其他地方声明的模板的标识符。如果已定义，将找到具有指定标识的模板，并使用模板创建扩展向导。此属性与class属性互斥。

<！ELEMENT editorWizard （ 描述 ？）>

<！ATTLIST编辑向导

id IDREF #REQUIRED

名称CDATA #REQUIRED

icon CDATA #IMPLIED

class CDATA #REQUIRED

点IDREF #REQUIRED>

一个向导，可用于从插件清单编辑器中编辑现有扩展


  • id - 将用于标识此向导的唯一名称。
  • name - 将在此向导的UI表示中使用的可翻译名称。
  • icon - 将用于直观表示向导的图标的相对路径。
  • class - 实现org.eclipse.pde.ui的类的完全限定名称。IExtensionEditorWizard界面。
  • point - 此向导能够编辑的扩展点的完全限定标识符

<！ELEMENT 类别 EMPTY>

<！ATTLIST类别

id IDREF #REQUIRED

名称CDATA #REQUIRED

parentCategory IDREF #IMPLIED>

  • id - 将用于引用此类别的唯一名称
  • name - 将用于此类别的UI表示的可翻译名称
  • parentCategory - 可用于创建类别层次结构的可选属性

<！元素描述 （#PCDATA）>

这个向导的简短描述。



例子：
以下是扩展的示例：

<extension point = “org.eclipse.pde.ui.newExtension” > <category name = “Custom Extensions” id = “custom” > </ category> <wizard availableAsShortcut = “true” name = “Simple Java Editor Extension” icon = “icons / java_edit.png” category = “generic” class = “com.example.xyz。SimpleJavaEditorExtension“ id = ”com.example.xyz.simple“ > <description>该向导创建一个包含所有必需类和清单标记的简单Java编辑器。</ description> </ wizard> </ extension>

API信息：
此扩展点需要一个实现org.eclipse.pde.ui的类。IExtensionWizard界面。

提供的实施：
PDE提供了一个通用向导，可根据扩展点架构信息创建扩展点。此外，使用PDE UI中的org.eclipse.pde.ui.templates扩展点注册的所有模板也被作为单独的扩展向导连接。
