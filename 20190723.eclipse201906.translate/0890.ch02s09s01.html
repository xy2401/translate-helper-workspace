2.9.1。�安装转换器插件

  1. 启动ITE并从主菜单帮助 - > 安装新软件...中选择 。

  2. 单击出现的“ 安装”对话框中的“ 添加... ”按钮。

  3. 单击出现的“ 添加存储库”对话框中的“ 存档... ”按钮

  4. 导航到安装Jubula的目录，然后在开发文件夹中选择org.eclipse.jubula.repo.zip

  5. 单击“ 确定 ”退出“ 添加存储库”对话框。现在，活动对话框应为“ 安装” 。

  6. 选择名称为“Jubula Functional Testing - API Converter（experimental）”的功能，然后单击“ 下一步 ”

  7. 再次单击“ 下一步 ”按钮确认安装详细信息 。

  8. 接受许可协议条款，然后单击“ 完成 ”。

  9. 单击“ 立即重新启动 ”按钮以执行重新启动，此时会出现一个对话框，提示重新启动将安全地完成更新/安装。重启后，安装了转换器。
