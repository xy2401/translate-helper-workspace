Eclipse Context XML文件扩展

每个提供Eclipse上下文XML文件的插件都必须在其插件清单（plugin.xml文件）中声明org.eclipse.help.contexts的扩展名。

例如：

<extension point =“org.eclipse.help.contexts”> <contexts file =“dtp_contexts.xml”plugin =“org.eclipse.datatools.connectivity.ui”/> </ extension>

有关Eclipse上下文XML文件及其贡献的更多信息，请参阅“ Platform Plug-in Developer Guide”中的描述和打包上下文相关帮助内容 。
