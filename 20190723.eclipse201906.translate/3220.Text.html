Text

Single-line and multi-line text widgets

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color

  • background-image

    Background image or gradient for text widget.

  • background-position

    Background image position for text fields.

  • background-repeat

    Background image repetition for text fields.

  • border

    The border that is drawn around text fields.

  • border-bottom

    The bottom side border for text fields.

  • border-left

    The left side border for text fields.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for text fields.

  • border-top

    The top side border for text fields.

  • box-shadow

    Inner or outer shadow around the bounds of a Text widget.

  • color

    Text color

  • font

    Text font

  • padding

    Padding for single-line text fields. Also used for spinner.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Style constant for bordered behavior.

  • [MULTI]

    Style constant for multy line text area behavior.

  • [SINGLE]

    Style constant for single line text field behavior.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disabled Texts.

  • :focused

    Applies to focused Texts.

  • :read-only

    Applies to text fields in read-only mode.

Text-Message

The Text message. The message text is displayed as a hint for the user, indicating the purpose of the field.

Available CSS Properties

Properties that can be defined for this element:

  • color

    Text message color

  • text-shadow

    Shadow for message text.

Applicable SWT Styles

none

Applicable States
