代码模板属性

“代码模板属性”页面允许您配置为所选项目的不同类型元素自动创建的代码和注释。

有关更多信息，请参阅代码模板首选

要访问“代码模板属性”页面，请在“项目资源管理器”视图中右键单击PHP项目，然后选择“属性”|资源 - 或 - 选择项目，然后从菜单栏转到项目|属性|代码风格|代码模板 。

看到

要为项目配置代码模板属性：

  1. 选中“启用项目特定设置”复选框。
  2. 根据您的喜好配置设置。有关可用设置的详细信息，请参阅任务标签首选项 。
  3. 单击“确定”以应用设置。

默认代码模板可以在代码模板首选项页面中设置所有项目的属性（通过单击属性页面上的配置工作区设置链接，转到窗口|首选项| PHP |代码样式|代码模板 - 或 - 访问）。
