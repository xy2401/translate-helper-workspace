安装

使用Eclipse Callgraph执行可视化函数跟踪需要三个组件：

  • Eclipse Callgraph插件
  • SystemTap的
  • SystemTap所需的内核信息包

在Fedora上，您可以通过yum安装所有这些项目。要安装SystemTap，请运行：

yum -y install systemtap systemtap-runtime

所需的内核信息包是您要分析的内核的匹配-devel ， -debuginfo和-debuginfo-common软件包。您可以使用yum或debuginfo-install来安装这些软件包。建议您将用户ID添加到stapdev和stapusr组。如果您看到有关avahi的错误消息，或者您收到消息说“无法找到服务器”，这是因为您的用户ID不属于stapdev组。将用户标识添加到这些特殊组时，您需要注销并再次登录才能使更改生效。

有关安装SystemTap的更多详细信息，请参阅SystemTap入门指南中的“ 安装和设置 ” 。
