变流器

标识符：
org.eclipse.cdt.managedbuilder.core。变流器

以来：
CDT 3.0版

描述：
此扩展点允许工具集成商向MBS提供项目类型/工具链/工具/构建器转换器。

配置标记：

<！ELEMENT 扩展 （ 转换器 +）>

<！ATTLIST扩展

点CDATA #REQUIRED

名称CDATA #IMPLIED

id CDATA #IMPLIED>


<！ELEMENT 转换器 EMPTY>

<！ATTLIST转换器

fromId CDATA #REQUIRED

to CD CDTIREIRIRED

mbsVersion CDATA #REQUIRED

class CDATA #REQUIRED

id CDATA #REQUIRED

名称CDATA #REQUIRED>

此扩展点允许工具集成商向MBS提供项目类型/工具链/工具/构建器转换器。转换器可用于升级到更新版本，更改项目类型或在不同工具链/工具/ buider之间执行转换。


  • fromId - 'fromId'是项目类型/工具链/工具/构建器ID - 正则表达式。
  • toId - 'toId'是项目类型/工具链/工具/构建器ID - 正则表达式。
  • mbsVersion - 编写转换器的MBS版本。
  • class - 实现IConvertManagedBuildObject接口的类
  • id - 转换器的ID
  • name - 用户界面中用于向用户显示的转换器的名称

例子：

<extension point = “org.eclipse.cdt.managedbuilder.core.converter” > <converter fromId = “project-type / tool-chain / tool / builder id” toId = “project-type / tool-chain / tool / builder id“ mbsVersion = ”转换器的MBS版本是为“ class = ”org.eclipse.cdt.managedbuilder.core 编写的 。GccConverter“ > </ converter> </ extension>

API信息：
想要扩展此扩展点的插件必须实现“org.eclipse.cdt.managedbuilder.core”。IConvertManagedBuildObject“界面

提供的实施：
“示例插件”具有转换器的默认实现
