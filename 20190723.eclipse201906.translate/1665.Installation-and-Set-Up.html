安装和设置

Gprof插件依赖于binutils（例如addr2line，c ++ filt和nm）。只要这些binutils在PATH中，Gprof就可以在任何平台上使用。例如，您可以在带有cygwin的窗口上使用它。

旧版本配置3.2.0之前

首先，用户必须在运行该工具之前使用-pg选项编译启用了性能分析的C / C ++程序。这可以通过项目Properties-> C / C ++ Build-> Settings-> Tool Settings-> GCC C Compiler-> Debugging选项卡完成，该选项卡有一个复选框“Generate Gprof Information（-pg）”。类似的复选框可用于CDT Autotools项目。它位于项目Properties-> Autotools-> Configure Settings-> configure-> Advanced下。

较新版本配置3.2。*以后

当用户第一次厌倦运行gprof时，将询问用户是否希望eclipse自动为用户启用gprof。
