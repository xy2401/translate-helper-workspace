资源过滤器

标识符：
org.eclipse.ui.ide.resourceFilters

以来：
3.0（最初在1.0版中添加为org.eclipse.ui.resourceFilters）

描述：
此扩展点用于向显示资源的视图添加预定义过滤器，例如导航器视图。可以选择这些过滤器来隐藏名称与过滤器模式匹配的资源。

此扩展点在版本3.5中已弃用，并由Common Navigator Framework替换。请注意，为了防止必须支持此功能的其他Eclipse插件中的警告，故意不会将其实际标记为已弃用。

配置标记：

<！ELEMENT 扩展 （ 过滤器 *）>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>

  • point - 目标扩展点的完全限定标识符
  • id - 扩展实例的可选标识符
  • name - 扩展实例的可选名称

<！ELEMENT 过滤器 （ 描述 ？）>

<！ATTLIST过滤器

模式CDATA #REQUIRED

选中（true | false）“false”>

  • pattern - 要匹配的模式。可能包含*和？通配符。
  • 选中 - 如果默认情况下应选择模式，则为“true”，否则为“false”或未定义。

<！元素描述 （#PCDATA）>

此过滤器的用途说明。



例子：
以下是资源过滤器扩展的示例，它过滤掉类文件，并且默认情况下已选中：


 <extension point="org.eclipse.ui.ide.resourceFilters">
  <filter pattern="*.class" selected="true"/>
 </extension>
