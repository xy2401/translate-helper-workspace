VM安装页面

标识符：
org.eclipse.jdt.debug.ui.vmInstallPages

以来：
3.4

描述：
此扩展点允许为需要自定义编辑控件的VM安装提供向导页面。

配置标记：

<！ELEMENT 扩展 （ vmInstallPage ）+>

<！ATTLIST扩展

点CDATA #REQUIRED

id CDATA #IMPLIED

名称CDATA #IMPLIED>


<！ELEMENT vmInstallPage EMPTY>

<！ATTLIST vmInstallPage

vmInstallType IDREF #REQUIRED

class CDATA #REQUIRED>

为特定VM安装类型提供向导页面。


  • vmInstallType - 指定此向导页面将用于的VM安装类型。与IVMInstallType的id对应的唯一标识符。
  • class - 向导页面实现。必须是org.eclipse.jdt.debug.ui.launchConfigurations.的子类org.eclipse.jdt.debug.ui.launchConfigurations.AbstractVMInstallPage 。

例子：
以下是VM安装页面的示例定义。 

<extension point = “org.eclipse.jdt.debug.ui.vmInstallPages” > <vmInstallPage vmInstallType = “org.eclipse.jdt.launching。EEVMType“ class = ”org.eclipse.jdt.internal.debug.ui.jres。EEVMPage“ > </ vmInstallPage> </ extension>

API信息：
向导页面必须是org.eclipse.jdt.debug.ui.launchConfigurations.的子类org.eclipse.jdt.debug.ui.launchConfigurations.AbstractVMInstallPage 。

提供的实施：
JDT为没有贡献的虚拟机提供了向导页面的默认实现。
