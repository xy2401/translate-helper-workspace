使用Remote Shell视图运行和查看命令

您可以从Remote System Explorer打开本地和远程系统的shell，

打开Remote Shell视图并启动命令shell

在Remote System Explorer中，通过单击菜单栏中的Window> Show View> Remote Shell切换到Remote Shell视图。要在Remote Shell视图中打开新shell，请单击视图中的向下箭头，然后选择Launch shell 。要为另一个连接启动新shell，需要在视图中选择该连接的现有shell。否则，必须从远程系统，远程系统详细信息或远程Shell视图启动shell。

使用shell

远程Shell视图打开后，您可以使用shell。一个shell，它跨命令保留其环境。shell中所有命令的环境可能会根据您的输入而更改。例如，如果键入“cd ..”，则更改该shell的工作目录。

在编辑器中查看输出

您可以双击一行输出以在编辑器中打开它。例如，如果在运行命令时遇到错误，则可以双击错误，编辑器会将源打开到解析器发现问题的代码行。

关闭壳

当前显示的shell或命令处于活动状态时，将启用“ 取消”外壳工具栏项。按Cancel Shell ，以终止该过程。如果shell处于活动状态，则会在从视图中删除shell之前取消它。

内容辅助

Remote Shell视图支持内容辅助。键入命令时，可以按CTRL +空格键完成文件名或环境变量名。
