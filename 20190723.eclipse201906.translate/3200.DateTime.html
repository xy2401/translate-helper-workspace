RWT Theming Reference > DateTime

DateTime

DateTime controls

Available CSS Properties

Properties that can be defined for this element:

  • background-color

    Background color

  • background-image

    Background gradient for DateTime. Not applied to DateTime Calendar.

  • border

    The border that is drawn around a DateTime.

  • border-bottom

    The bottom side border for DateTime.

  • border-left

    The left side border for DateTime.

  • border-radius

    Border radius. The border radius has no effect if an enhanced border, like "inset" or "outset" is used.

  • border-right

    The right side border for DateTime.

  • border-top

    The top side border for DateTime.

  • box-shadow

    Inner or outer shadow around the bounds of a DateTime widget.

  • color

    Text color for DateTime.

  • font

    Text font for DateTime. Not applied to DateTime Calendar.

  • padding

    Padding for DateTime Date/Time.

  • text-shadow

    Shadow for text.

Applicable SWT Styles

SWT style flags that can be referred to in a selector for this element (as CSS attributes):

  • [BORDER]

    Style constant for bordered behavior.

Applicable States

Widget states that can be referred to in a selector for this element (as CSS pseudo-classes):

  • :disabled

    Applies to disbled DateTime controls.

  • :focused

    Applies to focused DateTime controls.
