PHP版本支持

PDT支持PHP版本5.1 / 5.2 / 5.3 / 5.4 / 5.5 / 5.6 / 7.0 / 7.1 / 7.2 / 7.3（实验）。

PHP版本设置会影响：

  • PHP Functions视图中显示的元素。
  • 内容辅助中提供的选项。
  • 调试功能。

可以从以下位置配置PHP版本设置：

  • 可以从PHP可执行文件首选项页面添加和编辑PHP可执行文件 。
  • 可以在PHP Interpreter Preferences页面中设置用于新项目的默认PHP可执行文件。通过此页面，您还可以为特定项目设置PHP版本。此外，您可以通过在“新建PHP项目”对话框中选中“启用项目设置”复选框来选择在创建新PHP项目时应使用的PHP版本。

  • 可以在Debugging Preferences页面中设置与调试器一起使用的默认PHP可执行文件，从Window |访问偏好| PHP |调试 。通过此页面，您还可以设置用于调试特定项目的PHP可执行文件。此外，还可以在“调试配置”屏幕中配置用于每个调试会话的PHP可执行文件。

注意：

为了最大限度地减少错误，用于调试的PHP可执行文件应与用于项目的PHP版本匹配。
